# 术语与卡片

> 本目录计划用于存放化学竞赛中的核心术语卡片、概念速查卡、公式卡片等。

## 用途（规划中）

- 术语卡片：核心概念的极简定义（适合快速复习）
- 公式卡片：常考公式的整理（带适用条件和单位）
- 反应卡片：重要有机反应的速查（试剂、条件、选择性）

## 与 03-知识点 的区别

| | 03-知识点 | 08-术语与卡片 |
|---|---|---|
| 深度 | 完整解释+例题+拓展 | 极简定义+关键信息 |
| 用途 | 系统学习 | 快速复习/考前突击 |
| 格式 | 长文，14节结构 | 卡片式，1页1概念 |
