## 四、🤖 Agent 资料导入指引（SOP 补充）

> 本段落是给 Agent 的操作说明。任何执行「资料提炼 Agent (Agent 1)」或「知识点填充 Agent (Agent 2)」的会话，在 Phase 0 必须阅读本节。

### 导入资料时的标准动作

1. **Phase 0 必做**：读取本清单，筛选出与当前导入资料 **subject/module 相关** 且状态为「待建」的 KP
2. **Phase 1 计划**：在提炼/填充计划中明确列出：
   - 「本次可覆盖的待建 KP 清单」—— 按频率从高到低排序
   - 「预计新建/填充的知识点文件名」
3. **Phase 2 执行**：优先创建/填充待建 KP，再处理其他知识点
4. **Phase 4 归档**：每完成一个待建 KP，在本清单「关联提炼笔记 / 来源资料」列填入对应的提炼笔记 wiki-link

### 跨资料来源的 KP 合并策略

- 若同一待建 KP 可由多本资料覆盖（如「过渡金属通性」可由 Atkins 主题8 和元素化学教材共同覆盖）：
  - **先选最权威/最匹配考纲的资料**创建骨架
  - **其他资料作为增量补充**，在「关联提炼笔记」列追加 wiki-link
- 若待建 KP 在当前资料中只有片段提及（如某书中只提到「Brook重排」的一个例子）：
  - 在提炼笔记中标注「可派生 KP：[[Brook重排]]（片段，需补充）」
  - **不单独为该片段创建 KP 骨架**，等待更系统的资料来源

### 更新 tag_analysis.csv 后

当用户导入新的真题/习题并更新 `tag_analysis.csv` 时：
- 重新运行 REDLINK 分析脚本
- 将新增的高频标签追加到本清单
- **高优（≥5）标签必须在 1 个会话内处理完毕**
- **中优（3-4）标签纳入下次资料导入计划**
