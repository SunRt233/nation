 $\bigtriangleup _ { r } S _ { m } ^ { \circ } = y - S _ { m } ^ { \circ } ( \mathcal { L } \mathbb { T } \not \equiv \mathcal { H } ) \not \Xi ) { \Rightarrow } y = 1 8 6 ) / \left( \mathsf { m o l } { \cdot } \mathsf { K } \right)$

# 求x值：

$$
\Delta_ {r} G _ {m} ^ {\circ} (2 9 8 K) = 2. 5 - 2 9 8 \times 6. 2 5 \times 1 0 ^ {- 3} = 0. 6 \mathrm{kJ/mol}
$$

 $x \cdot \big ( - 1 0 1 . 7 \big ) = 0 . 6 { \Rightarrow } x \approx - 1 0 1 . 1 \big \mathrm { k J / m o l }$

. 单位换算：注意焓变单位是kJ/mol，熵变单位是J/(mol·K)，计算时需统⼀
