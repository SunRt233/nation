逐级稳定常数(Kf): 指配合物形成过程中某一步配位反应的平衡常数, 如从 $\mathrm{ML}_3$ 到 $\mathrm{ML}_4$ 的配位平衡常数。记作K或 $K_{f}$ 。

\- 累积稳定常数(β): 指从中心离子M直接与n个配体L形成MLn的总平衡常数, 其值为各逐级稳定常数的乘积, 即 $\beta_{n} = \prod_{i=1}^{n} K_{i}$ 。

# - 副反应系数分类:

金属离子副反应( $\alpha_{m}$ ): 中心离子与其他竞争配体发生的副反应，如 $Zn^{2+}$ 在氨性溶液中与 $NH_{3}$ 或 $OH^{-}$ 的配位。  
○ 配体副反应( $\alpha_{v}$ ):

■ 酸效应系数：EDTA受pH影响产生的质子化效应  
■ 共存离子效应：EDTA与其他金属离子(N)的竞争配位

# ● 条件稳定常数计算:

○ 公式： $K_{MY}' = \frac{K_{MY}}{\alpha_M \alpha_Y}$   
○ 对数形式: $\log K_{MY}' = \log K_{MY} - \log \alpha_{M} - \log \alpha_{Y}$   
○ 注意：通常忽略配合物MY的副反应

# 2）配位滴定 07:58

# - 化学计量点pM计算:

○ 公式： $[M]_{sp}=\sqrt{\frac{c_{M}^{sp}}{K_{MY}}}$   
- 关键应用：滴定误差计算的基础公式

# - 滴定终点pM'计算:

\- 公式： $pM' = \log K_{MIn}'$ （MIn为金属-指示剂配合物）

# ● 林邦误差公式:

$\% E_{t} = \frac{10^{\Delta pM} - 10^{-\Delta pM}}{\sqrt{c_{sp}K_{MY}}} \times 100\%$   
○ 规律： $K_{MY}$ 越大误差越小；终点与计量点pM差值越小误差越小

# 2. 准确滴定判别式 10:34

# 1）准确滴定判别式

# ● 基本条件:

○ 假设 $\Delta pM' = \pm 0.2$ ，要求 $E_{t} \leq 0.1\%$   
○ 推导结果： $c_{sp}K_{MY}' \geq 10^{6}$ 或 $\log(c_{sp}K_{MY}') \geq 6$

# - 误差要求变化:

○ 若要求 $E_{t} \leq 1\%$ ，则条件变为 $c_{sp}K_{MY}' \geq 10^{4}$   
- 记忆口诀："六次方保千一，四次方保百分"

# 2）分别滴定判别式 13:04

# - 混合离子滴定条件:

○ 假设 $\Delta pM' = \pm 0.2, E_{t} \leq 0.3\%$   
○ 基本式： $\log(c_{M}^{sp}K_{MY}')\geq5$

# ● 共存离子影响:

○ 修正公式： $\log(c_{M}^{sp}K_{MY}') - \log(c_{N}^{sp}K_{NY}') \geq 5$   
○ 简记： $\Delta\log(cK')\geq5$

# ● 实际应用要点:

○ 当 $\alpha_{Y(H)} < <\alpha_{Y(N)}$ 时，可忽略酸效应  
$\alpha_{Y(N)} = 1 + K_{NY}[N] \approx K_{NY}c_N^{sp}$ （当 $K_{NY}$ 较大时）  
若金属离子有配位副反应，需使用条件稳定常数

# 3. 配位滴定的酸度控制 17:03

# 1）酸度控制原理

$$
\begin{array}{r l}&{\lg k _{M Y}=\lg k _{M Y}- \lg \alpha_ {Y}=\underbrace {\lg k _{M Y}- \lg d _{Y (M)}- \lg d _{Y (C)}}}_{\text{等}}\\ &{\quad \alpha_ {Y (H)} \ll \alpha_ {Y (N)}=1+k _{M Y} C _{N} ^ {S P}}\\ &{\Rightarrow \lg (k _{M Y} \cdot C _{M}^{S P})-\lg (k _{M Y} \cdot C _{N}^{S P})\geqslant5}\\ &{\Rightarrow \Delta \lg (K \cdot C)\geqslant5.\quad\Delta \lg (K ^{\prime}C)\geqslant5}\end{array}
$$

双重影响：酸度不仅影响配体Y的酸效应系数 $\alpha_{Y(H)}$ ，还影响金属离子的水解行为。在pH较高时，某些金属离子会发生水解甚至沉淀。  
- 缓冲体系必要性：由于酸度对滴定过程影响显著，配位滴定通常需要良好的缓冲体系来维持稳定的pH环境。  
- 最低pH确定：当仅考虑Y的酸效应时，满足准确滴定条件的酸度上限由 $\log\alpha_{Y(H)}\leq \log K_{MY}-\log K'_{MY}$ 决定，通过查表可得对应的最低pH值。
