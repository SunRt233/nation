- 显色机制：利用与金属离子生成有色配合物的显色剂（用 $I_{n}$ 表示），在接近化学计量点时与金属离子(M)配位形成显色配合物(MIn)  
● 颜色变化：存在两种颜色状态

○ 游离态：指示剂自身颜色(color A)

○ 配合态：与金属离子结合后颜色(color B)

\- 反应方程式:

○ 初始阶段： $M + I_{n} \rightleftharpoons MIn$ （显color B）  
- 终点阶段： $MIn + Y \rightarrow MY + I_n$ （恢复color A）

3）金属离子指示剂的反应过程 02:39:54

- 滴定前期：溶液中金属离子浓度高，主要形成MIn配合物，呈现color B  
● 接近终点：EDTA(Y)逐渐置换出MIn中的指示剂，颜色由B向A转变  
● 关键要求：置换过程必须完全可逆，否则无法准确指示终点

4）金属离子指示剂的选择要求 02:41:08

● 颜色对比：color A与color B必须显著不同（肉眼可辨）  
● 反应动力学：需要迅速灵敏且可逆

● 稳定性要求：

○ $K_{MIn}$ 应适中：既要比 $K_{MY}$ 小（保证可被 EDTA 置换），又不能太小（避免提前变色）  
- 具体范围需通过计算确定

● pH影响：多数金属指示剂为弱酸，需考虑酸效应

\- 变色点计算：

○ 理论公式： $logK_{MIn}' = pM + log\frac{[MIn]}{[In]}$   
○ 变色点时： $logK_{MIn}' = pM_{EP}$

5）常见问题及解决方法

\- 指示剂封闭：

○ 现象：MIn稳定性过强，EDTA无法置换（如 $Al^{3+}$ 、 $Fe^{3+}$ 对EBT的封闭）  
- 解决：加入掩蔽剂（三乙醇胺、KCN、 $\mathrm{Na_2S}$ ）

\- 指示剂僵化：

○ 现象：MIn难溶于水，显色缓慢（如PAN指示剂）  
- 解决：加热或加有机溶剂（如乙醇）提高溶解度

2. 配位滴定终点误差计算 02:48:27

1）基本原理

误差公式：

$\bullet E_{t} = \frac{10^{\Delta pM^{\prime}} - 10^{-\Delta pM^{\prime}}}{\sqrt{K_{MY}^{\prime}\cdot c_{sp}}}\times 100\%$

○ 其中 $\Delta pM' = pM_{EP}' - pM_{SP}'$

● 关键参数：

$\begin{array}{ll} \circ & pM_{SP}' = \frac{1}{2} (pc_{sp} + logK_{MY}') \\ \circ & pM_{EP}' = logK_{MIn}' \end{array}$

2）例题:配位滴定误差计算 02:53:53

● 题目条件:

○ pH=10.00氨性缓冲溶液   
○ 0.020mol/L EDTA滴定0.020mol/L Ca $^{2+}$ /Mg $^{2+}$   
○ EBT指示剂(pK1=6.3, pK2=11.6)   
○ 已知 $\log K_{CaY}=10.69,\log K_{MgY}=8.7$   
$\circ \log K_{Ca - EBT} = 5.4, \log K_{Mg - EBT} = 7.0$

\- 解题步骤：

○ 计算 $c_{sp}=0.010mol/L$   
○ 求条件稳定常数:

$\text{log}K_{CaY}' = 10.69 - 0.45 = 10.24$

$\text{log}K_{MgY}' = 8.7 - 0.45 = 8.25$

○ 计算 $pM_{SP}'$ :

■ $pCa_{SP}' = \frac{1}{2}(2.00 + 10.24) = 6.12$   
■ $pMg_{SP}' = \frac{1}{2} (2.00 + 8.25) = 5.12$

○ 求EBT酸效应系数:

■ $\alpha_{EBT(H)}=1+\frac{\left[H^{+}\right]}{K_{2}}+\frac{\left[H^{+}\right]^{2}}{K_{1}K_{2}}=40$   
$\text{log}\alpha_{EBT(H)} = 1.6$

○ 计算条件指示剂常数:

$\log K_{Ca-EBT}' = 5.4 - 1.6 = 3.8 \rightarrow pCa_{EP}' = 3.8$   
$\text{log} K_{Mg-EBT}' = 7.0 - 1.6 = 5.4 \rightarrow pMg_{EP}' = 5.4$

o 求 $\Delta pM'$ :

■ $\Delta pCa' = 3.8 - 6.12 = -2.32$   
■ $\Delta pMg' = 5.4 - 5.12 = 0.28$

○ 计算终点误差:

■ $E_{t}(Ca^{2 + }) = -1.5\%$   
■ $E_{t}(Mg^{2 + }) = 0.11\%$

结论分析：

虽然 $K_{CaY}' > K_{MgY}'$ ，但因 $\Delta pCa'$ 偏差过大，导致钙的滴定误差反而更大  
○ 说明pH=10时用EBT指示剂滴定钙不合适，应改用钙指示剂（需在pH12-13条件下）
