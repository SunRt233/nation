# 热力学循环（Thermodynamic Cycle）

## 一、定义

热力学循环是指系统经过一系列状态变化后回到初始状态的过程。由于循环过程的**状态函数变化为零**（$\Delta U = 0$, $\Delta H = 0$），热力学循环是计算难以直接测量的热力学量（如晶格能、键能）的**核心工具**。

## 三、核心原理

### 3.1 Hess 定律

化学反应的热效应只与始态和终态有关，与路径无关：

$$\Delta H_{\text{总}} = \sum_i \Delta H_i$$

### 3.2 Born-Haber 循环

用于计算离子晶体的**晶格能**（$\Delta H_L$）：

以 NaCl 为例：
$$\Delta H_f = \Delta H_{\text{sub}} + \frac{1}{2}D + I + A + \Delta H_L$$

| 步骤 | 过程 | 符号 |
|:---|:---|:---:|
| 1 | Na(s) → Na(g) | 升华焓 $\Delta H_{\text{sub}}$ |
| 2 | Na(g) → Na⁺(g) + e⁻ | 电离能 $I$ |
| 3 | ½Cl₂(g) → Cl(g) | ½解离能 $\frac{1}{2}D$ |
| 4 | Cl(g) + e⁻ → Cl⁻(g) | 电子亲合能 $A$ |
| 5 | Na⁺(g) + Cl⁻(g) → NaCl(s) | 晶格焓 $\Delta H_L$ |

### 3.3 Born-Fajans-Haber 循环的变化

- 可用于计算电子亲合能、质子亲合能
- 关键：气相离子生成焓 + 晶格焓 = 晶体生成焓

## 四、关键结论

### 4.1 Carnot 循环

- **效率**：$\eta = 1 - \frac{T_c}{T_h}$
- **意义**：确定了热机效率的理论上限
- **逆 Carnot 循环**：制冷机和热泵的理想模型

### 4.2 制冷循环

- **蒸发制冷**：利用液体蒸发吸热
- **制冷系数**：$\varepsilon = \frac{Q_c}{W} = \frac{T_c}{T_h - T_c}$

### 4.3 实际循环

- **Otto 循环**：汽油机
- **Diesel 循环**：柴油机
- **Rankine 循环**：蒸汽动力装置

### 4.4 电化学中的热力学循环

$$\Delta G = -nFE$$
$$\Delta S = nF\left(\frac{\partial E}{\partial T}\right)_P$$
$$\Delta H = -nFE + nFT\left(\frac{\partial E}{\partial T}\right)_P$$

通过测定不同温度下的电动势，可同时获得 $\Delta G$、$\Delta S$、$\Delta H$。

## 十一、易错点

1. **晶格能的符号约定**：$U$（晶格能）通常取正值（拆散晶体所需能量），但 Born-Haber 循环中晶格形成焓为负值
2. **Carnot 效率是上限**：实际热机效率一定低于 Carnot 效率
3. **制冷系数可以 > 1**：制冷机移走的热量可以大于输入的功
4. **状态函数在循环中变化为零**：但过程量（Q, W）不为零
