# 离子方程式

- 总览：[[中国化学奥林匹克基本要求-总览]]

## 一、定义

离子方程式是用实际参加反应的离子符号表示离子反应的式子。它比化学方程式更能反映反应的本质，因为它省略了未参与反应的 spectator ions（旁观离子），直接展示了反应的核心过程。

## 三、核心原理

### 3.1 书写步骤

1. **写出完整的化学方程式**
2. **将可溶性强电解质拆成离子形式**
   - 强酸（HCl、H₂SO₄、HNO₃、HClO₄ 等）
   - 强碱（NaOH、KOH、Ba(OH)₂、Ca(OH)₂ 等）
   - 可溶性盐（查溶解度表）
3. **删去方程式两边相同的 spectator ions**
4. **检查守恒**：原子守恒、电荷守恒
5. **化简系数为最简整数比**

### 3.2 可拆 vs 不可拆

| 类别 | 处理方式 | 例子 |
|------|---------|------|
| 强酸 | 拆 | $H^+ + Cl^-$ |
| 强碱 | 拆 | $Na^+ + OH^-$ |
| 可溶性盐 | 拆 | $Na^+ + SO_4^{2-}$ |
| 弱酸/弱碱 | 不拆 | $CH_3COOH$、$NH_3·H_2O$ |
| 难溶物 | 不拆 | $BaSO_4$、$AgCl$ |
| 氧化物 | 不拆 | $CO_2$、$CaO$ |
| 单质 | 不拆 | $Cl_2$、Fe |
| 气体 | 不拆 | $H_2$、$CO_2$ |

### 3.3 常见错误类型

- **拆错**：把弱电解质或难溶物拆了
- **漏写**：微溶物（如 $Ca(OH)_2$）在稀溶液中拆、浓溶液中不拆的处理不当
- **不守恒**：原子数或电荷数不相等
- **不符合事实**：如把 $Cu + H^+$ 写成 $Cu^{2+} + H_2$（稀硫酸不反应）
