# 熵

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[06-热力学初步]]

## 一、定义
**熵（Entropy, S）**：体系**微观状态数**的度量，表征体系的混乱度/无序度。热力学第二定律：孤立体系的熵永不减少（$\Delta S_\text{孤立} \geq 0$）。

## 三、核心原理
- 熵是**状态函数**（广度量）：$\Delta S$ 只取决于始态和终态，与变化路径无关（因此可设计可逆路径来计算实际不可逆过程的熵变）
- 热力学第二定律：$\Delta S_\text{孤立} = \Delta S_\text{体系} + \Delta S_\text{环境} \geq 0$（孤立体系的总熵永不减少；可逆过程取等号，不可逆过程取大于号）
- 热力学第三定律：任何理想晶体在热力学零度（0 K）时熵等于零（$S_0 = 0$），以此为基准可测定物质的标准熵值
- 混乱度与微观状态数：体系的混乱度越大，可能存在的微观状态数 $\Omega$ 越多，熵值越大。例如糖溶于水后，糖分子与水分子均匀混合，微观状态数远大于溶解前的有序排列
- Boltzmann 关系：$S = k \ln W$（W = 微观状态数）
- 热力学定义：$dS = \frac{\delta q_\text{rev}}{T}$
- 标准摩尔熵 $S^\circ$：1 mol 物质在标准态下的熵值

## 四、关键结论
### 熵的定性判断
S 增大（ΔS > 0）的情况：
- s → l → g（相变到更无序的状态）
- 气体分子数增多
- 温度升高
- 分子越大/越复杂 → S 越大
- 刚性分子 > 柔性分子

### 标准摩尔熵的规律（298 K，$p^\ominus$）
由实验数据总结的标准摩尔熵规律（选自教材§5.7 表5.2）：

**（1）同一物质：气态 > 液态 > 固态**
- $\mathrm{H_2O}$：$S^\ominus(\mathrm{g}) = 188.8$，$S^\ominus(\mathrm{l}) = 70.0$ J/(mol·K)
- $\mathrm{Br_2}$：$S^\ominus(\mathrm{g}) = 245.5$，$S^\ominus(\mathrm{l}) = 152.2$ J/(mol·K)

**（2）同类物质：摩尔质量 $M$ 越大，$S^\ominus$ 越大**（原子数、电子数越多，微观状态数越多）
- 卤素单质(g)：$\mathrm{F_2}$（202.8）< $\mathrm{Cl_2}$（223.1）< $\mathrm{Br_2}$（245.5）< $\mathrm{I_2}$（260.7）
- 烷烃(g)：$\mathrm{CH_4}$（186.4）< $\mathrm{C_2H_6}$（229.2）< $\mathrm{C_3H_8}$（270.3）< $\mathrm{C_4H_{10}}$（310.0）

**（3）气态多原子分子 > 单原子**（原子数多，微观状态数多）
- $\mathrm{O}$（161）< $\mathrm{O_2}$（205.2）< $\mathrm{O_3}$（238.9）
- $\mathrm{N}$（153）< $\mathrm{NO}$（210.8）< $\mathrm{NO_2}$（240.1）

**（4）结构越复杂（对称性越低），$S^\ominus$ 越大**
- 乙醇(g) 283 > 二甲醚(g) 267 J/(mol·K)（同分异构体，乙醇对称性更低）

**（5）温度升高，$S^\ominus$ 增大**（动能增加，可及能级增多）
- $\mathrm{CS_2(l)}$：161 K 时 103，298 K 时 150 J/(mol·K)

**（6）压力增大，气态物质 $S^\ominus$ 减小**（固态、液态影响很小）
- $\mathrm{O_2(g)}$：101 kPa 时 205，606 kPa 时 190 J/(mol·K)

### 反应熵变计算
$$\Delta_r S^\circ = \sum \nu S^\circ(\text{产物}) - \sum \nu S^\circ(\text{反应物})$$

**计算示例**（298 K）：
- $\mathrm{3H_2(g) + N_2(g) \rightarrow 2NH_3(g)}$：
  $$\Delta S^\ominus = 2 \times 192.8 - (3 \times 130.7 + 191.6) = -198.1 \text{ J/(mol·K)}$$
  （$\Delta n_\mathrm{g} = -2$，气体分子数减少 → 熵减）

- $\mathrm{NH_4Cl(s) \rightarrow NH_4^+(aq) + Cl^-(aq)}$：
  $$\Delta S^\ominus = (113.4 + 56.5) - 94.6 = +75.3 \text{ J/(mol·K)}$$
  （固体溶解并解离 → 混乱度增大 → 熵增）

**教材表5.3 典型反应 $\Delta S^\ominus$ 数据**：
| 反应 | $\Delta S^\ominus$ (J/(mol·K)) | $\Delta n_\mathrm{g}$ |
|------|--------------------------------|------------------------|
| $\mathrm{2Fe_2O_3(s)+3C(s)\rightarrow 4Fe(s)+3CO_2(g)}$ | +558.7 | +3 |
| $\mathrm{Fe_2O_3(s)+3CO(g)\rightarrow 2Fe(s)+3CO_2(g)}$ | +15.5 | 0 |
| $\mathrm{N_2O_4(g)\rightarrow 2NO_2(g)}$ | +176.0 | +1 |
| $\mathrm{CaO(s)+SO_3(g)\rightarrow CaSO_4(s)}$ | −188.4 | −1 |
| $\mathrm{N_2(g)+3H_2(g)\rightarrow 2NH_3(g)}$ | −198.1 | −2 |

**规律**：$\Delta n_\mathrm{g} > 0$ → $\Delta S^\ominus > 0$（熵增）；$\Delta n_\mathrm{g} < 0$ → $\Delta S^\ominus < 0$（熵减）；$\Delta n_\mathrm{g} = 0$ → $\Delta S^\ominus$ 通常很小。无气体参与的反应，总计量系数增加时一般 $\Delta S^\ominus > 0$。

### 相变熵
$$\Delta S_\text{相变} = \frac{\Delta H_\text{相变}}{T_\text{相变}}$$
- Trouton 规则：多数液体的 $\Delta S_\text{蒸发} \approx 85$ J/(mol·K)

## 五、常见分类或情形

熵按其来源可大致分为以下三类：

### 位置熵（Positional Entropy）
与粒子在空间的排列方式和运动自由度相关。粒子可占据的位置越多、运动范围越大，位置熵越高：
- **固态**：粒子只能在晶格点附近振动，运动自由度最小 → 低位置熵（如 C(金刚石) $S^\ominus = 2.4$，C(石墨) $S^\ominus = 5.7$ J/(mol·K)）
- **液态**：粒子可在液体体积范围内无序运动 → 中等位置熵（如 $\mathrm{H_2O(l)}$ $S^\ominus = 70.0$）
- **气态**：粒子可在更大空间自由运动 → 高位置熵（如 $\mathrm{H_2O(g)}$ $S^\ominus = 188.8$）
- **溶解过程**：糖或盐溶于水，溶质粒子从有序排列（晶体）分散进入溶液 → 位置熵大增

### 热熵（Thermal Entropy）
与温度相关的熵，源于热运动导致的微观状态数变化：
- 温度升高 → 分子动能增加 → 可及的平动、转动、振动能级增多 → 微观状态数 $\Omega$ 增大 → $S$ 增大
- **定量示例**：$\mathrm{CS_2(l)}$ 从 161 K 到 298 K，$S^\ominus$ 从 103 增至 150 J/(mol·K)
- 热力学第三定律的推论：$T \to 0$ K 时，所有粒子处于基态，理想晶体 $\Omega = 1$，$S_0 = k \ln 1 = 0$

### 构型熵（Configurational Entropy）
与分子内部结构复杂性、对称性相关的熵：
- **原子数越多** → 构型熵越大：$\mathrm{O}$(161) < $\mathrm{O_2}$(205.2) < $\mathrm{O_3}$(238.9)
- **摩尔质量越大** → 构型熵越大：$\mathrm{F_2}$(202.8) < $\mathrm{Cl_2}$(223.1) < $\mathrm{Br_2}$(245.5) < $\mathrm{I_2}$(260.7)
- **对称性越低** → 构型熵越大：乙醇(g) 283 > 二甲醚(g) 267 J/(mol·K)
- **含结晶水的盐**：$\mathrm{CuSO_4 \cdot 5H_2O(s)}$ $S^\ominus = 300.4$ $\gg$ $\mathrm{CuSO_4(s)}$ $S^\ominus = 109.2$，因结晶水大幅增加了微观构型数目

## 六、适用条件与限制

### 熵变计算的前提与约束

1. **熵是状态函数**：$\Delta S$ 只取决于始态和终态，与路径无关。因此**可以设计可逆路径**来计算实际不可逆过程的熵变——这是熵变计算的核心策略。

2. **热力学定义要求可逆热**：
   $$dS = \frac{\delta q_\text{rev}}{T}, \quad \Delta S = \int_{T_1}^{T_2} \frac{\delta q_\text{rev}}{T}$$
   只有**可逆过程**的热 $q_\text{rev}$ 才能直接代入上式。不可逆过程的实际热 $q_\text{irrev}$ **不能**直接用于计算 $\Delta S$。

3. **可逆过程**是理想化的极限过程——外压始终比内压小一无穷小量（准静态膨胀/压缩），体系每一步都无限接近平衡。实际过程多为不可逆过程，但因为 $S$ 是状态函数，只需在相同始态、终态间**设计可逆路径**即可。可逆膨胀体系做最大功、吸最多热；可逆压缩环境做最小功。

4. **等温过程简化**：若 $T$ 恒定，$\Delta S = \dfrac{q_\text{rev}}{T}$

5. **相变过程的适用条件**：
   $$\Delta S_\text{相变} = \frac{\Delta H_\text{相变}}{T_\text{相变}}$$
   该式**仅适用于可逆相变**，即在相变温度（如沸点、熔点）下进行的相变。若在非相变温度下进行（如 −10°C 的水结冰），需设计可逆路径分段计算。

6. **标准熵变计算**（最常用方法）：利用标准摩尔熵 $S^\ominus$ 通过 $\Delta_r S^\circ = \sum \nu S^\circ(\text{产物}) - \sum \nu S^\circ(\text{反应物})$ 计算，**不涉及路径问题**，是直接、可靠的方法。

7. **温度影响范围**：物质的 $S$ 随 $T$ 升高而增大（动能增加），但化学反应的 $\Delta S$ 随温度变化很小（反应物和产物的 $S$ 大致同步增大），在一般温度范围内可视为常数。

8. **压力影响**：压力对固、液态物质熵的影响很小，对气态物质影响较大（$p$ 越大 $S$ 越小，如 $\mathrm{O_2(g)}$ 从 101 kPa 到 606 kPa，$S^\ominus$ 从 205 降至 190）。

## 七、常见比较与易混点

### 熵（S）vs 焓（H）

| 对比维度 | 焓（$H$） | 熵（$S$） |
|----------|-----------|-----------|
| 物理本质 | 体系的热含量，$H = U + pV$ | 体系的混乱度/微观状态数的度量 |
| 状态函数 | 是（广度量） | 是（广度量） |
| 绝对值 | 无法测定，只有 $\Delta H$ 有意义 | 可测定（第三定律给出 $S_0 = 0$） |
| 单位 | kJ/mol | J/(mol·K) |
| 对自发性的贡献 | $\Delta H < 0$（放热）有利于自发 | $\Delta S > 0$（熵增）有利于自发 |
| 温度依赖性 | $\Delta H$ 随 $T$ 变化很小 | $\Delta S$ 随 $T$ 变化也很小 |

### 自发过程 ≠ 放热过程

这是最常见的误解。**放热（$\Delta H < 0$）不是自发的必要条件**：

**吸热但自发的典型实例**（熵增驱动）：
- $\mathrm{NH_4Cl(s) \rightarrow NH_4^+(aq) + Cl^-(aq)}$：$\Delta H > 0$（吸热，烧杯外壁可结冰），$\Delta S^\ominus = +75.3$ J/(mol·K)，**室温自发**
- $\mathrm{Ba(OH)_2 \cdot 8H_2O(s) + 2NH_4SCN(s) \rightarrow Ba(SCN)_2(s) + 2NH_3(g) + 10H_2O(l)}$：$\Delta H > 0$（吸热，锥形瓶变凉），**室温自发**（产生气体，熵大增）

**熵减但自发的典型实例**（焓降驱动）：
- $\mathrm{H_2O(l, -10^\circ C) \rightarrow H_2O(s)}$：$\Delta S < 0$（液态 → 固态），但 $\Delta H < 0$（放热），**自发结冰**
- 乙烯聚合：$n\mathrm{CH_2=CH_2 \rightarrow [-CH_2-CH_2-]_n}$：$\Delta S < 0$（单体 → 聚合物），$\Delta H < 0$，自发

**$\mathrm{N_2 + O_2 \rightarrow 2NO}$ 两种因素都不利的实例**：
- $\Delta H^\ominus = +183$ kJ/mol（吸热），$\Delta S^\ominus = +25$ J/(mol·K)（熵增，但 $T\Delta S^\ominus$ 项在常温仅约 7.5 kJ/mol），$\Delta G^\ominus \gg 0$，**常温不自发**，仅雷电高温（$>7.3\times 10^3$ K）才可能反应

### 核心判据：Gibbs 自由能

$\Delta H$ 和 $\Delta S$ **都不能单独**作为反应自发性的判据。只有等温等压条件下 $\Delta G = \Delta H - T\Delta S$ 才是正确判据：
- $\Delta G < 0$：正向自发
- $\Delta G = 0$：平衡
- $\Delta G > 0$：正向不自发（逆向自发）

| 类型 | $\Delta H$ | $\Delta S$ | $\Delta G$ | 自发性 |
|------|------------|------------|------------|--------|
| (−, +) 型 | 负（放热） | 正（熵增） | 恒为 − | **任意温度**正向自发 |
| (+, −) 型 | 正（吸热） | 负（熵减） | 恒为 + | **任意温度**正向不自发 |
| (+, +) 型 | 正（吸热） | 正（熵增） | 高温 −，低温 + | 高温自发（如 $\mathrm{N_2+O_2\rightarrow 2NO}$） |
| (−, −) 型 | 负（放热） | 负（熵减） | 低温 −，高温 + | 低温自发（如 $\mathrm{N_2+3H_2\rightarrow 2NH_3}$） |

**转变温度**：$T_\text{转} = \dfrac{\Delta H^\ominus}{\Delta S^\ominus}$（令 $\Delta G^\ominus = 0$ 求得）
