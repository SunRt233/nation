## 一、定义

**原子结构**：原子由原子核（质子 + 中子）和核外电子组成，电子按特定规律分布在不同能级的原子轨道中。理解原子结构是解释**元素周期律、化学键、光谱性质**的基础。

**核心问题**：
- 电子在哪里？（轨道形状与分布）
- 电子能量是多少？（能级与量子数）
- 电子如何排布？（构造原理、Pauli 原理、Hund 规则）

## 三、核心原理

### 3.1 四个量子数

| 量子数 | 符号 | 物理意义 | 取值范围 | 决定性质 |
|:---|:---:|:---|:---|:---|
| **主量子数** | n | 电子层 | 1, 2, 3, ... | 电子能量主要决定因素 |
| **角量子数** | l | 电子亚层/轨道形状 | 0, 1, ..., n−1 | 轨道形状（s/p/d/f）|
| **磁量子数** | mₗ | 轨道空间取向 | −l, ..., 0, ..., +l | 轨道数目（2l+1 个）|
| **自旋量子数** | mₛ | 电子自旋方向 | +1/2, −1/2 | 电子自旋 |

**轨道符号**：l = 0(s), 1(p), 2(d), 3(f)

**轨道容量**：每个轨道最多容纳 **2** 个电子（Pauli 不相容原理）

### 3.2 能级顺序（构造原理）

**近似顺序**（由低到高）：

```
1s < 2s < 2p < 3s < 3p < 4s < 3d < 4p < 5s < 4d < 5p < 6s < 4f ≈ 5d < 6p < 7s < 5f ≈ 6d < 7p
```

**记忆口诀**：
> "动动屁股，怕死进三弟；四弟怕死，五弟进四弟；六弟进五弟，四胖约五弟"

**n + l 规则**：
- 能级高低主要由 **n + l** 决定
- n + l 相同时，**n 越小**能量越低（如 3d > 4s，因为 3+2=5 > 4+0=4）

### 3.3 电子排布三原则

1. **构造原理（Aufbau Principle）**：电子优先填入能量最低的轨道
2. **Pauli 不相容原理**：同一原子中，没有两个电子具有完全相同的四个量子数 → 每个轨道最多 2 电子，且自旋相反
3. **Hund 规则**：
   - 等价轨道（如三个 p 轨道）上电子优先**单独占据**不同轨道，且自旋平行
   - 全满、半满、全空构型特别稳定（Cr: 3d⁵4s¹，Cu: 3d¹⁰4s¹）

### 3.4 电子排布表示法

| 表示法 | 例子（N, Z=7）| 说明 |
|:---|:---|:---|
| **电子排布式** | 1s² 2s² 2p³ | 按能级顺序书写 |
| **价电子排布** | 2s² 2p³ | 只写最外层 |
| **轨道表示式** | ↑↓ ↑↓ ↑ ↑ ↑ | 用箭头表示电子，体现 Hund 规则 |
| **稀有气体简式** | [He] 2s² 2p³ | 用稀有气体核心简化 |

## 四、关键结论

1. **量子数完全确定电子状态**——n, l, mₗ 确定轨道；mₛ 区分同一轨道中的两个电子
2. **能级交错是周期表结构的基础**——4s < 3d 导致过渡元素出现在第四周期
3. **Hund 规则决定等价轨道电子分布**——单独占据优先，平行自旋
4. **半满/全满稳定化**——Cr(3d⁵4s¹)、Cu(3d¹⁰4s¹)、Mo、Ag、Au 等均为此规则体现
5. **电子排布式 ≠ 电子填充顺序**——书写时按 n 排序（如 Cu: 1s² 2s²... 3d¹⁰ 4s¹），但填充时 4s 先于 3d

## 五、常见分类或情形

### 5.1 特殊电子排布（竞赛常考）

| 元素 | 预期排布 | 实际排布 | 原因 |
|:---|:---|:---|:---|
| **Cr (24)** | [Ar] 3d⁴ 4s² | **[Ar] 3d⁵ 4s¹** | 半满稳定（d⁵）|
| **Cu (29)** | [Ar] 3d⁹ 4s² | **[Ar] 3d¹⁰ 4s¹** | 全满稳定（d¹⁰）|
| **Mo (42)** | [Kr] 4d⁴ 5s² | **[Kr] 4d⁵ 5s¹** | 半满稳定 |
| **Ag (47)** | [Kr] 4d⁹ 5s² | **[Kr] 4d¹⁰ 5s¹** | 全满稳定 |
| **Au (79)** | [Xe] 4f¹⁴ 5d⁹ 6s² | **[Xe] 4f¹⁴ 5d¹⁰ 6s¹** | 全满稳定 |
| **Pd (46)** | [Kr] 4d⁸ 5s² | **[Kr] 4d¹⁰** | 全满稳定，5s⁰ |

### 5.2 离子的电子排布

**阳离子**：先失去最外层（n 最大）电子
- Fe: [Ar] 3d⁶ 4s² → Fe²⁺: [Ar] 3d⁶ → Fe³⁺: [Ar] 3d⁵
- **规律**：先失 ns，再失 (n−1)d

**阴离子**：在原子基础上增加电子
- O: 1s² 2s² 2p⁴ → O²⁻: 1s² 2s² 2p⁶（稀有气体构型）

## 六、适用条件与限制

1. **量子数规则是硬性约束**：
   - l 必须 < n
   - mₗ 必须在 −l 到 +l 之间
   - mₛ 只能是 ±1/2
2. **能级图是近似**——多电子原子中轨道能量随核电荷和电子间相互作用变化
3. **Hund 规则有例外**——某些重元素因相对论效应偏离
4. **离子排布不要死记填充顺序**——阳离子失去的是最高 n 的电子，即使它先被填充

---

## 七、常见比较与易混点

### 7.1 电子排布式书写 vs 填充顺序

| | 书写顺序 | 填充顺序 |
|:---|:---|:---|
| **原则** | 按主量子数 n 递增 | 按能量由低到高 |
| **Cu 例子** | [Ar] **3d¹⁰ 4s¹** | 先填 4s¹，再填 3d¹⁰ |
| **为什么不同** | 书写时 3d 和 4s 同属第四周期 | 填充时 4s 能量略低于 3d |

**考试陷阱**：题目问"Cu 的价电子排布"→答 3d¹⁰ 4s¹；问"Cu 的最后一个电子填入哪个轨道"→答 3d。

### 7.2 等价轨道与电子数

| 亚层 | 轨道数 (2l+1) | 最多电子数 |
|:---:|:---:|:---:|
| s (l=0) | 1 | 2 |
| p (l=1) | 3 | 6 |
| d (l=2) | 5 | 10 |
| f (l=3) | 7 | 14 |

## 十、例题

### 10.1 例题 1（★ 基础）

**题目**：写出 Fe（Z=26）的电子排布式、Fe²⁺ 和 Fe³⁺ 的电子排布式。

**解答**：
- Fe: **[Ar] 3d⁶ 4s²**
- Fe²⁺: **[Ar] 3d⁶**（先失 4s²）
- Fe³⁺: **[Ar] 3d⁵**（再失 1 个 3d 电子，半满稳定）

### 10.2 例题 2（★★ 量子数）

**题目**：判断下列各组量子数是否可能存在于原子中：
(1) n=2, l=2, mₗ=0  (2) n=3, l=1, mₗ=−1  (3) n=4, l=0, mₗ=1

**解答**：
- (1) **不可能**：l 必须 < n，l 最大为 1（当 n=2）
- (2) **可能**：l=1（p 轨道），mₗ=−1, 0, +1 均有效
- (3) **不可能**：l=0（s 轨道），mₗ 只能是 0

### 10.3 例题 3（★★★ 特殊排布）

**题目**：解释为什么 Cr 的电子排布是 [Ar] 3d⁵ 4s¹，而不是 [Ar] 3d⁴ 4s²。

**解答**：
- [Ar] 3d⁵ 4s¹ 中，3d 和 4s 均为**半满**
- 半满构型（d⁵）具有额外的交换能稳定化
- 交换能：平行自旋电子之间的量子力学稳定作用
- 半满时平行自旋电子数最多 → 交换能最大
- 因此 [Ar] 3d⁵ 4s¹ 比 [Ar] 3d⁴ 4s² 更稳定

## 十一、易错点

1. ❌ **离子排布按填充顺序失电子** → 阳离子先失 **n 最大**的轨道电子（如 Fe 先失 4s，不是 3d）
2. ❌ **书写顺序与填充顺序混淆** → 书写时按 n 排序；填充时按能量排序
3. ❌ **忽略 Hund 规则** → 等价轨道上电子应先单独占据，不要配对
4. ❌ **认为全满/半满总是稳定** → 对主族元素影响小，对 d 区显著
5. ❌ **量子数取值范围错误** → l < n；|mₗ| ≤ l
6. ❌ **把激发态当基态** → 基态电子必须先填低能级轨道

### 12.3 入门级例题

**题目**：某元素基态原子价电子排布为 3d² 4s²，问：
(1) 该元素的原子序数是多少？(2) 它位于周期表第几周期、第几族？(3) 它有多少个未成对电子？

**预期解答路径**：
1. [Ar] 核心有 18 个电子 + 3d² 4s² = **22 个电子** → Z = 22（Ti）
2. 最高 n = 4 → **第四周期**；价电子数 = 2+2 = 4 → **IVB 族**
3. 3d²：两个 d 电子分占两个 d 轨道（Hund 规则）→ **2 个未成对电子**

### 12.4 与现实/直觉的连接

- **类比**：电子排布就像学生入住宿舍——低楼层（低能级）先住满，同一楼层的不同房间（等价轨道）先每人住一间（Hund 规则），最后才合住（配对）。
- **反例**：直觉认为"电子排布是固定的"。实际上，电子可以被激发到高能级（光谱吸收），离子化后电子构型也改变——电子排布描述的是**基态**，不是唯一状态。

## 十三、竞赛拓展

1. **光谱项符号**：
   - 基态项符号 ^{2S+1}L_J
   - 用于描述多电子原子的能级精细结构

2. **相对论效应**：
   - 重元素（Au、Hg、Pb）中内层电子速度接近光速
   - 导致 6s 轨道收缩（惰性对效应的部分原因）

3. **径向分布函数**：
   - 描述电子在距核不同距离处出现的概率
   - 解释钻穿效应（ns 电子比 np 更靠近核）
