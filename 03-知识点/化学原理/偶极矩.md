# 偶极矩

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[10-分子结构与化学键]]

## 一、定义
**偶极矩 μ**：衡量分子极性大小的物理量。

$$\boldsymbol{\mu} = q \cdot \boldsymbol{d}$$

- 单位：Debye (D)，1 D = $3.336 \times 10^{-30}$ C·m
- 矢量方向：正电荷中心 → 负电荷中心

## 三、核心原理
- 分子的总偶极矩 = 各键偶极矩的**矢量和**
- 键偶极矩由电负性差决定

## 四、关键结论
### 常见分子偶极矩
| 分子 | μ (D) | 说明 |
|------|:---:|------|
| H₂O | 1.85 | 大偶极矩 |
| NH₃ | 1.47 | |
| HF | 1.82 | |
| HCl | 1.08 | |
| CO | 0.12 | 意外小（C⁻←O⁺ 抵消了电负性差的作用）|
| CCl₄ | 0 | 对称抵消 |
| cis-CHCl=CHCl | 1.90 | 极性 |
| trans-CHCl=CHCl | 0 | 抵消 |

### CO 偶极矩反常
- 按电负性预测：C⁺—O⁻（O 电负性大）
- 实际：C⁻←O⁺（因 O→C 的 π 反馈使电子向 C 偏移）
- 净偶极矩非常小（~0.12 D），方向是 C←O

## 五、常见分类或情形

### 卤化氢（HX）系列的键矩与离子性
| 分子 | 键长 $R$ (pm) | $\Delta\chi$ | 键矩 $\mu$ (D) | $\mu_{\text{计}} = eR$ (D) | 离子性百分数 |
|------|:---:|:---:|:---:|:---:|:---:|
| H$_2$ | 74.6 | 0 | 0.00 | 3.54 | 0% |
| F$_2$ | 142 | 0 | 0.00 | 6.80 | 0% |
| HI | 160 | 0.5 | 0.42 | 7.67 | 5% |
| HBr | 141 | 0.8 | 0.80 | 6.77 | 12% |
| HCl | 127 | 1.0 | 1.08 | 6.10 | 18% |
| HF | 91.7 | 1.8 | 1.91 | 4.41 | 41% |

注：$\mu_{\text{计}}$ 的 SI 值为 $10^{-30}$ C$\cdot$m 单位，1 D = $3.336 \times 10^{-30}$ C$\cdot$m。

**从 HI 到 HF，键长 $R$ 减小，但偶极矩 $\mu$ 增大**，因为电负性差增大使偶极长 $d$ 增大的效应超过 $R$ 减小的效应。

### 常见多原子分子偶极矩
| 分子 | $\mu$ (D) | 构型 | 极性 |
|------|:---:|------|:---:|
| CO$_2$ | 0 | 直线形 | 非极性 |
| CS$_2$ | 0 | 直线形 | 非极性 |
| H$_2$O | 1.85 | V形 | 极性 |
| NH$_3$ | 1.47 | 三角锥 | 极性 |
| SO$_2$ | 1.63 | V形 | 极性 |
| HF | 1.91 | 直线形 | 极性 |
| HCl | 1.08 | 直线形 | 极性 |
| CO | $\sim$0.12 | 直线形 | 弱极性（反常方向） |
| CCl$_4$ | 0 | 正四面体 | 非极性 |
| CHCl$_3$ | 1.04 | 四面体 | 极性（不对称取代） |
| cis-CHCl=CHCl | $\sim$1.90 | 平面 | 极性 |
| trans-CHCl=CHCl | 0 | 平面 | 非极性 |

### 偶极矩的用途
1. **判断分子极性**：$\mu = 0 \to$ 非极性，$\mu \neq 0 \to$ 极性且 $\mu$ 越大极性越强
2. **判断分子构型**：$\mu = 0$ 必须是对称结构；非零 $\mu$ 可验证不对称构型
3. **估算离子性百分数**：$(\mu_{\text{实}}/\mu_{\text{计}}) \times 100\%$
4. **CO 偶极矩反常**：揭示反馈 $\pi$ 键的存在

## 六、适用条件与限制

1. **偶极矩是矢量**：$\boldsymbol{\mu} = q \cdot \boldsymbol{d}$，方向为正电荷重心 $\to$ 负电荷重心（化学约定）。
2. **分子总偶极矩 = 各键矩的矢量和**。
3. **无法单独测量 $d$ 和 $q$**：实验只能测得 $\mu$，不能分别得到偶极长和偶极电荷量。
4. **键长 $R \neq$ 偶极长 $d$**：即使在极性分子中，$d$ 通常也小于 $R$（电子云不完全偏移）。
5. **适用范围**：主要适用于中性分子；离子的偶极矩依赖于参考点选择。

## 七、常见比较与易混点

### 键长 vs 偶极长
| 概念 | 定义 | 测定 |
|------|------|:---:|
| 键长 $R$ | 成键两原子核间平衡距离 | 可直接测定（X 射线、光谱） |
| 偶极长 $d$ | 正负电荷重心间距离 | 不能单独测定 |

### Debye (D) vs SI 单位
$$1 \text{ D} = 10^{-18} \text{ esu}\cdot\text{cm} = 3.336 \times 10^{-30} \text{ C}\cdot\text{m}$$
在竞赛计算中，通常使用 Debye 单位。

### CO 偶极矩反常（高频考点）
- **预测**（基于电负性 $\chi_{\text{O}} > \chi_{\text{C}}$）：C$^+$—O$^-$，$\mu$ 较大
- **实测**：C$^-$$\leftarrow$O$^+$，$\mu \approx 0.12$ D 且方向相反
- **解释**：O 原子上的孤对电子通过 $\pi$ 反馈向 C 的空 $\pi^*$ 轨道转移（反馈 $\pi$ 键），抵消并超过电负性差造成的偶极

### 偶极矩与分子间作用力
- 极性分子（$\mu \neq 0$）：存在取向力（Keesom 力）+ 诱导力 + 色散力
- 非极性分子（$\mu = 0$）：只有色散力
- 取向力 $\propto \mu^2$

## 十一、易错点
- 偶极矩单位换算：1 D = $3.336 \times 10^{-30}$ C$\cdot$m
- $\mu = 0$ 可确认非极性，但 $\mu$ 很小（如 CO 0.12 D）仍为极性
- 键矩矢量和为零并不意味着每个键都是非极性的
- 偶极矩方向是从正到负（化学约定），与物理学约定相反
