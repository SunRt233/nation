# 配合反应

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-化学原理]]
- 对应考纲条目：[[12-配合物]]

## 一、定义
**配合反应（配位反应）**：中心原子/离子与配体结合形成配合物的反应。通常是 Lewis 酸碱加合反应。

$$\ce{M^{n+} + mL -> [ML_m]^{n+}}$$

- 配体取代反应是配合物最常见的反应类型
- 动力学上分为活性（labile）和惰性（inert）配合物

## 二、考纲对应
- 考纲条目：[[12-配合物]]、[[07-化学平衡]]，涉及配位反应及其平衡移动

## 三、核心原理

**配位反应**是中心离子与配体结合形成配离子或配合物的可逆过程：

$$\mathrm{M}^{n+} + x\mathrm{L}^- \rightleftharpoons \mathrm{ML}_x^{(n-x)+}$$

该平衡与其他化学平衡一样，当加入其他试剂（酸、碱、沉淀剂、氧化还原剂、其他配合剂）时，由于这些试剂与 $\mathrm{M}^{n+}$ 或 $\mathrm{L}^-$ 发生反应，将导致**配位平衡移动**。这一过程涉及的是**多重平衡**。

## 四、关键结论

### 1. 配位平衡与酸碱平衡
许多配位体是弱酸根（如 $\mathrm{F}^-$、$\mathrm{CN}^-$、$\mathrm{SCN}^-$、$\mathrm{CO_3^{2-}}$、$\mathrm{C_2O_4^{2-}}$ 等），能与外加酸生成弱酸而使配合物解离。当 $[\mathrm{H_3O^+}] > 0.5\,\mathrm{mol\!\cdot\!dm^{-3}}$ 时，$\mathrm{FeF_3}$ 配合物将解离：
$$\mathrm{Fe^{3+} + 3F^- \rightleftharpoons FeF_3}$$
$$3\mathrm{F^-} + 3\mathrm{H_3O^+} \rightarrow 3\mathrm{HF} + 3\mathrm{H_2O}$$

规律：配合物越不稳定，配位体的酸越弱 → 越容易被加入的酸所解离。

**pH 控制的实例**：
- $\mathrm{Fe^{3+}}$ 与水杨酸根 $\mathrm{sal^-}$ 在不同 pH 下生成不同有色螯合物（pH 2~3 紫红色，pH 4~8 红褐色，pH≥9 黄色）→ 比色分析中用缓冲溶液控 pH
- $\mathrm{ZnY^{2-}}$ ($\lg K_{\text{稳}} = 16.4$) 和 $\mathrm{CaY^{2-}}$ ($\lg K_{\text{稳}} = 11.0$)：控 pH 4~5 时 EDTA 只与 $\mathrm{Zn^{2+}}$ 反应 → 提高选择性

### 2. 配位平衡与沉淀平衡
配离子的 $K_{\text{稳}}$ 越大，才能使 $K_{sp}$ 小的难溶物溶解。

**银盐的溶解实验**：
| 沉淀 | $K_{sp}$ | 可溶于 | 形成配离子 | $K_{\text{稳}}$ |
|:---|:---:|:---|:---|:---:|
| $\mathrm{AgCl}$ (白) | $1.8\!\times\!10^{-10}$ | 氨水 | $\mathrm{Ag(NH_3)_2^+}$ | $1.1\!\times\!10^7$ |
| $\mathrm{AgBr}$ (淡黄) | $5.0\!\times\!10^{-13}$ | $\mathrm{Na_2S_2O_3}$ | $\mathrm{Ag(S_2O_3)_2^{3-}}$ | — |
| $\mathrm{AgI}$ (黄) | $8.3\!\times\!10^{-17}$ | KCN | $\mathrm{Ag(CN)_2^-}$ | $1.3\!\times\!10^{21}$ |
| $\mathrm{Ag_2S}$ (黑) | $6.3\!\times\!10^{-50}$ | 通常配合剂不能 | — | — |

**配溶常数（表 14.13）**：
- $\mathrm{AgCl} + 2\mathrm{NH_3} \rightleftharpoons \mathrm{Ag(NH_3)_2^+ + Cl^-}$，$K = K_{\text{稳}} \times K_{sp} = 2.0 \times 10^{-3}$
- $\mathrm{AgI} + 2\mathrm{CN^-} \rightleftharpoons \mathrm{Ag(CN)_2^- + I^-}$，$K = 1.1 \times 10^5$

### 3. 配位平衡与氧化还原平衡
配位反应可改变金属离子的氧化还原能力。

**王水溶金**（例 14.6）：
- $\mathrm{Au}$ 不溶于 $\mathrm{HNO_3}$：$E^{\ominus}(\mathrm{Au^{3+}/Au}) = 1.52\,\mathrm{V} > E^{\ominus}(\mathrm{NO_3^-/NO}) = 0.96\,\mathrm{V}$
- 但 $\mathrm{Au}$ 可溶于王水（浓 $\mathrm{HNO_3}$ + 浓 $\mathrm{HCl}$ 1:3）：$\mathrm{AuCl_4^-}$ 的 $K_{\text{稳}} = 3 \times 10^{26}$，使 $K_5^{\ominus} = 9 \times 10^{-3}$，浓酸条件下反应可行

### 4. 配合物之间的转化
一种配合试剂有时能同时与两种金属离子生成不同颜色的配离子，利用稳定常数的差异可进行**掩蔽**。

**例**：鉴定 $\mathrm{Co^{2+}}$（生成蓝紫色 $\mathrm{Co(NCS)_4^{2-}}$）时 $\mathrm{Fe^{3+}}$ 会干扰（生成血红色 $\mathrm{Fe(NCS)^{2+}}$）。加入 $\mathrm{NH_4F}$ 使 $\mathrm{Fe^{3+}}$ 生成无色 $\mathrm{FeF_3}$（$K_{\text{稳}} = 1.2 \times 10^{12}$），转化常数 $\frac{K_{\text{稳}}(\mathrm{FeF_3})}{K_{\text{稳}}(\mathrm{Fe(NCS)^{2+}})} = 1.3 \times 10^9$，$\mathrm{Fe(NCS)^{2+}}$ 几乎全部转化为 $\mathrm{FeF_3}$。

## 五、常见分类或情形

| 配位平衡耦合类型 | 关键试剂/条件 | 效果 | 实例 |
|:---|:---|:---|:---|
| 配位-酸碱 | 强酸 | 使弱酸根配体的配合物解离 | $\mathrm{FeF_3} + \mathrm{H^+} \rightarrow \mathrm{Fe^{3+} + HF}$ |
| 配位-沉淀 | 配合剂 | 使沉淀溶解 | $\mathrm{AgCl \xrightarrow{NH_3} Ag(NH_3)_2^+}$ |
| 配位-沉淀（逆） | 沉淀剂 | 使配离子解离 | $\mathrm{Ag(NH_3)_2^+ \xrightarrow{Br^-} AgBr\!\downarrow}$ |
| 配位-氧化还原 | 配合剂 | 改变氧化能力 | 王水溶金 |
| 配位-配位（转化） | 竞争配体 | 配合物相互转化 | $\mathrm{Fe(NCS)^{2+} \xrightarrow{F^-} FeF_3}$ |

## 六、适用条件与限制

- **适用**：分析化学中的掩蔽、解蔽、沉淀溶解
- **适用**：物质的分离提纯（如从阳极泥中回收 Au、Pt）
- **适用**：利用多重平衡常数定量估计反应方向和程度
- **限制**：多重平衡计算时需同时考虑所有共存平衡，精确计算较复杂
- **限制**：实际应用中需考虑动力学因素（反应速率）

## 七、常见比较与易混点

| 对比项 | 配位-沉淀正向 | 配位-沉淀逆向 |
|:---|:---|:---|
| 方向 | 沉淀 + 配合剂 → 配离子 + 阴离子 | 配离子 + 沉淀剂 → 沉淀 |
| 条件 | $K_{\text{稳}} \times K_{sp}$ 足够大 | $1/(K_{\text{稳}} \times K_{sp})$ 足够大 |
| 实例 | $\mathrm{AgCl \xrightarrow{NH_3}}$ 溶解 | $\mathrm{Ag(NH_3)_2^+ \xrightarrow{Br^-} AgBr\!\downarrow}$ |

| 易混点 | 区分 |
|:---|:---|
| 配合物解离 vs 被酸解离 | 配位平衡自身解离（$K_{\text{不稳}}$）vs 外加酸与配体反应导致平衡移动 |
| 掩蔽 vs 解蔽 | 掩蔽：加试剂使干扰离子不参与反应；解蔽：用另一试剂破坏掩蔽 |
| 配溶常数 | $K = K_{\text{稳}} \times K_{sp}$，用于判断沉淀在配合剂中的溶解程度 |
