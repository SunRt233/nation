## 一、定义

物料守恒（Mass Balance / Material Balance）是指在一个封闭化学体系中，某一元素或某一组分的总物质的量在反应前后保持不变。它是化学计算中最基本的守恒关系之一，常与电荷守恒、质子守恒联合使用，解决复杂平衡体系中的浓度计算问题。

## 三、核心原理

### 3.1 基本形式

对于某一元素 X，若其在体系中可能以多种形式存在（如游离离子、配合物、沉淀等），则：

$$c_{\text{X,总}} = \sum_i c_{\text{X}_i}$$

其中 $c_{\text{X}_i}$ 为 X 在第 i 种物种中的浓度。

### 3.2 典型应用场景

| 场景 | 物料守恒表达式 | 用途 |
|------|---------------|------|
| 弱酸溶液 | $c_{\text{HA}} = [\text{HA}] + [\text{A}^-]$ | 计算各型体分布 |
| 多元弱酸 | $c_{\text{H}_n\text{A}} = [\text{H}_n\text{A}] + [\text{H}_{n-1}\text{A}^-] + \dots + [\text{A}^{n-}]$ | 分布系数计算 |
| 缓冲溶液 | $c_{\text{酸}} + c_{\text{碱}} = [\text{HA}] + [\text{A}^-]$ | pH 精确计算 |
| 配合物体系 | $c_{\text{M}} = [\text{M}] + [\text{ML}] + [\text{ML}_2] + \dots$ | 配位平衡计算 |
| 滴定分析 | $n_{\text{被测物}} = n_{\text{标准溶液}} \times \text{化学计量比}$ | 滴定结果计算 |

### 3.3 与电荷守恒、质子守恒的关系

在酸碱平衡计算中，三大守恒常联立使用：

- **电荷守恒（CCE）**：溶液中正负电荷总量相等
  $$[\text{H}^+] + [\text{Na}^+] = [\text{OH}^-] + [\text{A}^-]$$

- **物料守恒（MBE）**：某元素各型体浓度之和等于初始浓度
  $$c = [\text{HA}] + [\text{A}^-]$$

- **质子守恒（PBE）**：得质子产物与失质子产物的浓度关系（由 CCE 和 MBE 推导）
  $$[\text{H}^+] + [\text{HA}] = [\text{OH}^-] + c_{\text{碱}}$$

### 3.4 竞赛中的快速技巧

- **分布系数法**：利用 $\delta_i$ 直接计算各型体浓度，避免解高次方程
- **化学计量点分析**：滴定终点时物料守恒关系最简洁
- **元素质量守恒**：燃烧分析、化学式推断中的核心工具

1. **弱酸各型体浓度计算**：已知 pH 和总浓度，求 [HA] 和 [A⁻]

3. **多平衡综合计算**：酸碱 + 配位 + 沉淀共存体系
4. **燃烧法化学式推断**：由 CO₂、H₂O 质量守恒求实验式
