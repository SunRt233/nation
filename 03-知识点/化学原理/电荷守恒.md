# 电荷守恒

## 一、定义

电荷守恒（Charge Balance / Electroneutrality）是指在任何电解质溶液或晶体中，正电荷总量等于负电荷总量，体系整体呈电中性。它是化学计算中最基本的守恒关系之一，常与物料守恒、质子守恒联合使用。

## 三、核心原理

### 3.1 基本表达式

对于任意电解质溶液：
$$\sum_i z_i [\text{正离子}]_i = \sum_j |z_j| [\text{负离子}]_j$$

或更简洁地：
$$\sum_i z_i c_i = 0$$

其中 $z_i$ 为离子电荷数（含符号），$c_i$ 为离子浓度。

### 3.2 典型应用场景

| 场景 | 电荷守恒式 | 用途 |
|------|-----------|------|
| 一元弱酸溶液 | $[\text{H}^+] = [\text{A}^-] + [\text{OH}^-]$ | 精确计算 pH |
| 含共同离子盐 | $[\text{Na}^+] + [\text{H}^+] = [\text{A}^-] + [\text{OH}^-]$ | 缓冲溶液计算 |
| 多元酸盐 | $[\text{Na}^+] + [\text{H}^+] = [\text{HA}^-] + 2[\text{A}^{2-}] + [\text{OH}^-]$ | 注意系数等于离子电荷绝对值 |
| 配合物体系 | 计入各配离子电荷 | 配位平衡与 pH 联立 |

### 3.3 与物料守恒、质子守恒的关系

在酸碱平衡中，通常联立三大守恒求解：
- **电荷守恒（CCE）**：$[\text{H}^+] + [\text{Na}^+] = [\text{OH}^-] + [\text{A}^-]$
- **物料守恒（MBE）**：$c = [\text{HA}] + [\text{A}^-]$
- **质子守恒（PBE）**：由 CCE 与 MBE 相减得到
