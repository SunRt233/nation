## 一、定义
**电子排布（电子组态）**：原子或离子中电子在各原子轨道上的分布。遵循三条基本原则。

## 三、核心原理

### 三条原则

1. **能量最低原理**：在不违背 Pauli 不相容原理的前提下，核外电子在各原子轨道上的排布方式应使整个原子能量处于最低的状态。

2. **Pauli 不相容原理**：在同一原子中没有 4 个量子数完全相同的电子。等价表述：同一原子轨道仅可容纳 2 个自旋相反的电子。由此，各电子层最多可容纳 $2n^2$ 个电子。

3. **Hund 规则**：电子在能量相同的轨道（简并轨道）上分布时，总是尽可能以自旋相同的方向分占不同的轨道。当轨道被电子半充满（$\mathrm{p}^3$、$\mathrm{d}^5$、$\mathrm{f}^7$）或全充满（$\mathrm{p}^6$、$\mathrm{d}^{10}$、$\mathrm{f}^{14}$）时体系较为稳定。Hund 规则实际上是能量最低原理的补充。

### 屏蔽效应与有效核电荷

在多电子原子中，指定电子 $i$ 受到其他电子的排斥，相当于部分抵消了原子核的吸引。有效核电荷 $Z^{*} = Z - \sigma$，其中 $\sigma$ 为屏蔽常数（Slater 规则估算）。

电子能量公式修正为（单位 eV）：

$$E_{i} = -\frac{(Z - \sigma)^{2}}{n^{2}} \times 13.6 = -\frac{(Z^{*})^{2}}{n^{2}} \times 13.6$$

**Slater 规则要点**（电子分组：$(1\mathrm{s})(2\mathrm{s},2\mathrm{p})(3\mathrm{s},3\mathrm{p})(3\mathrm{d})(4\mathrm{s},4\mathrm{p})(4\mathrm{d})(4\mathrm{f})\dots$）：
- 右边各组对左边电子屏蔽可忽略 $( \sigma = 0)$
- 同组内每个电子屏蔽同组价电子 0.35（1s 层为 0.30）
- 相邻内组对 s/p 电子屏蔽 0.85，对 d/f 电子屏蔽 1.00
- 再向内各组对各电子屏蔽 1.00

**示例**：Na 原子（$1\mathrm{s}^2 2\mathrm{s}^2 2\mathrm{p}^6 3\mathrm{s}^1$），3s 电子 $Z^{*} = 11 - (8 \times 0.85 + 2 \times 1.00) = 2.20$，$E_{3\mathrm{s}} = -13.6 \times 2.20^2 / 3^2 = -7.3\,\mathrm{eV}$

### 钻穿效应

$n$ 相同、$l$ 不同的轨道，电子云径向分布不同，钻穿到核附近的能力为 $n\mathrm{s} > n\mathrm{p} > n\mathrm{d} > n\mathrm{f}$。钻穿越深，受屏蔽越少，能量越低：

$$E_{n\mathrm{s}} < E_{n\mathrm{p}} < E_{n\mathrm{d}} < E_{n\mathrm{f}}$$

此即能级分裂的原因。当 $n$ 和 $l$ 均不同时，钻穿效应可导致能级交错（如 $E_{4\mathrm{s}} < E_{3\mathrm{d}}$）。

### 填充顺序（Pauling 近似能级图）

1s → 2s → 2p → 3s → 3p → 4s → **3d** → 4p → 5s → **4d** → 5p → 6s → **4f** → **5d** → 6p → 7s → **5f** → **6d** → 7p

等价规则：$n+l$ 小者先填；$n+l$ 相同则 $n$ 小者先填。

### 离子电子排布
- **先失去最外层电子**（$n$ 最大者），再失去 d 电子
- Fe: [Ar]3d⁶4s² → Fe²⁺: [Ar]3d⁶（失去 4s！不是 3d）→ Fe³⁺: [Ar]3d⁵
- 原因：电离后有效核电荷增大，钻穿效应减弱，$n$ 对能量的影响成为主导，3d 能量显著低于 4s

### 电子排布书写方式

以 Mn (Z=25) 为例：
- **I 式（填充次序）**：$1\mathrm{s}^2 2\mathrm{s}^2 2\mathrm{p}^6 3\mathrm{s}^2 3\mathrm{p}^6 4\mathrm{s}^2 3\mathrm{d}^5$ 或 $[\mathrm{Ar}] 4\mathrm{s}^2 3\mathrm{d}^5$  
  优点：便于判断分区（最后一个电子填入 3d → d 区）、周期（最高能级组数 = 4 → 第四周期）、族（7 个价电子 → VIIB）
- **II 式（$n,l$ 次序）**：$1\mathrm{s}^2 2\mathrm{s}^2 2\mathrm{p}^6 3\mathrm{s}^2 3\mathrm{p}^6 3\mathrm{d}^5 4\mathrm{s}^2$ 或 $[\mathrm{Ar}] 3\mathrm{d}^5 4\mathrm{s}^2$  
  优点：符合过渡元素轨道能量高低的实际情况

### 价电子判断

| 元素类型 | 价电子范围 | 示例 |
|:---:|------|------|
| 非过渡元素（s区/p区） | 最外层电子 | Ca: $4\mathrm{s}^2$，P: $3\mathrm{s}^2 3\mathrm{p}^3$ |
| 过渡元素（d区/ds区） | 最外层 + 次外层 d 电子 | Mn: $3\mathrm{d}^5 4\mathrm{s}^2$，Zn: $3\mathrm{d}^{10} 4\mathrm{s}^2$ |
| 内过渡元素（f区） | 最外层 + 次外层 + 外数第三层 f 电子 | Ce: $4\mathrm{f}^1 5\mathrm{d}^1 6\mathrm{s}^2$，U: $5\mathrm{f}^3 6\mathrm{d}^1 7\mathrm{s}^2$ |

### 能级组与周期的对应

每个周期对应一个能级组。周期内元素数目 = 能级组内轨道数的 2 倍（每轨道 2 电子）：
- 第一周期（2 种）：1 个 s 轨道 → 2 元素
- 第二、三周期（各 8 种）：1 个 s + 3 个 p = 4 轨道 → 8 元素
- 第四、五周期（各 18 种）：1 个 s + 5 个 d + 3 个 p = 9 轨道 → 18 元素
- 第六周期（32 种）：1 个 s + 7 个 f + 5 个 d + 3 个 p = 16 轨道 → 32 元素

## 五、常见分类或情形

### 按元素分区（由价电子构型决定）

| 区 | 价电子构型 | 包含元素 | 示例 |
|:---:|------|------|------|
| s 区 | $ns^{1\sim 2}$ | IA、IIA 族（碱金属、碱土金属） | Na: $3\mathrm{s}^1$，Ca: $4\mathrm{s}^2$ |
| p 区 | $ns^2 np^{1\sim 6}$ | IIIA~VIIA 族、零族 | Cl: $3\mathrm{s}^2 3\mathrm{p}^5$，Ne: $2\mathrm{s}^2 2\mathrm{p}^6$ |
| d 区 | $(n-1)\mathrm{d}^{1\sim 9}ns^{1\sim 2}$ | IIIB~VIII 族（过渡元素） | Fe: $3\mathrm{d}^6 4\mathrm{s}^2$，Mn: $3\mathrm{d}^5 4\mathrm{s}^2$ |
| ds 区 | $(n-1)\mathrm{d}^{10}ns^{1\sim 2}$ | IB、IIB 族 | Cu: $3\mathrm{d}^{10}4\mathrm{s}^1$，Zn: $3\mathrm{d}^{10}4\mathrm{s}^2$ |
| f 区 | $(n-2)\mathrm{f}^{1\sim 14}(n-1)\mathrm{d}^{0\sim 2}ns^2$ | 镧系、锕系（内过渡元素） | Ce: $4\mathrm{f}^1 5\mathrm{d}^1 6\mathrm{s}^2$ |

### 按周期长度

| 周期类型 | 周期 | 能级组 | 元素数 | 填充轨道 |
|:---:|:---:|------|:---:|------|
| 特短周期 | 一 | $1\mathrm{s}^{1\sim 2}$ | 2 | 1s |
| 短周期 | 二、三 | $2\mathrm{s}2\mathrm{p}$ 或 $3\mathrm{s}3\mathrm{p}$ | 各 8 | ns, np |
| 长周期 | 四、五 | $ns\,(n-1)\mathrm{d}\,n\mathrm{p}$ | 各 18 | ns, (n-1)d, np |
| 特长周期 | 六、七 | $ns\,(n-2)\mathrm{f}\,(n-1)\mathrm{d}\,n\mathrm{p}$ | 32（七待定） | ns, (n-2)f, (n-1)d, np |

### 电子填充顺序（按能级组）

第一能级组：1s（2 个电子）
第二能级组：2s → 2p（8 个电子）
第三能级组：3s → 3p（8 个电子）
第四能级组：4s → 3d → 4p（18 个电子）
第五能级组：5s → 4d → 5p（18 个电子）
第六能级组：6s → 4f → 5d → 6p（32 个电子）
第七能级组：7s → 5f → 6d → 7p（待定）

## 六、适用条件与限制

1. **仅适用于基态中性原子**：电子排布规则针对基态原子。激发态或离子态的电子构型不同，需具体分析。
2. **Pauling 近似能级图不是绝对的**：它只是"电子填充顺序图"，不同元素的轨道能量次序随原子序数而变化。例如 K 中 $E_{4\mathrm{s}} < E_{3\mathrm{d}}$（能级交错），但 Sc 以后 $E_{4\mathrm{s}} > E_{3\mathrm{d}}$。
3. **例外情况不能由简单规则预测**：Pd ($[\mathrm{Kr}]4\mathrm{d}^{10}$ 而非 $4\mathrm{d}^8 5\mathrm{s}^2$)、Pt ($[\mathrm{Xe}]5\mathrm{d}^9 6\mathrm{s}^1$)、Nb ($[\mathrm{Kr}]4\mathrm{d}^4 5\mathrm{s}^1$) 等例外构型由光谱实验确定，不能仅凭 Hund 规则推导。
4. **屏蔽常数计算为近似**：Slater 规则将电子分组为 $(1\mathrm{s})(2\mathrm{s},2\mathrm{p})(3\mathrm{s},3\mathrm{p})(3\mathrm{d})(4\mathrm{s},4\mathrm{p})\dots$，屏蔽常数取 0.30（同组 1s）、0.35（同组其他）、0.85（相邻内组对 s/p）、1.00（更内层及相邻内组对 d/f），这仅是粗略估计。
5. **$n+l$ 规则仅在多电子原子中近似成立**：氢原子中轨道能只取决于 $n$，与 $l$ 无关。

## 七、常见比较与易混点

### 填充顺序 vs 书写顺序

| 方式 | 规则 | 示例 Mn (Z=25) |
|:---:|------|------|
| 填充顺序（I 式） | 按 Pauling 能级图顺序 | $1\mathrm{s}^2 2\mathrm{s}^2 2\mathrm{p}^6 3\mathrm{s}^2 3\mathrm{p}^6 4\mathrm{s}^2 3\mathrm{d}^5$ |
| 书写顺序（II 式） | 按主量子数 $n$ 由小到大 | $1\mathrm{s}^2 2\mathrm{s}^2 2\mathrm{p}^6 3\mathrm{s}^2 3\mathrm{p}^6 3\mathrm{d}^5 4\mathrm{s}^2$ |

竞赛中两种书写方式均可接受，但 I 式有助于判断元素所在分区和周期；II 式符合轨道能量高低的实际情况。

### 原子填充顺序 vs 离子电离顺序

- **填充时**：先填 4s 后填 3d（K、Ca 等）
- **电离时**：先失去最外层（$n$ 最大）电子，即先失去 4s 再失去 3d
- 例：Fe ($[\mathrm{Ar}] 3\mathrm{d}^6 4\mathrm{s}^2$) → $\mathrm{Fe}^{2+}$ ($[\mathrm{Ar}] 3\mathrm{d}^6$，失去的是 4s 电子！)
- 原因：电离后有效核电荷增大，钻穿效应影响减弱，主量子数 $n$ 成为能量主导因素，3d 能量显著低于 4s

### 半满/全满稳定性

- $\mathrm{p}^3$、$\mathrm{d}^5$、$\mathrm{f}^7$（半满）和 $\mathrm{p}^6$、$\mathrm{d}^{10}$、$\mathrm{f}^{14}$（全满）构型特别稳定
- Cr ($3\mathrm{d}^5 4\mathrm{s}^1$) 和 Cu ($3\mathrm{d}^{10} 4\mathrm{s}^1$) 是竞赛必记的典型
- 但并非所有类似位置都出现异常：例如 Tc 是 $[\mathrm{Kr}]4\mathrm{d}^5 5\mathrm{s}^2$（非 $4\mathrm{d}^6 5\mathrm{s}^1$）

### 能级分裂与能级交错

- **能级分裂**：同一 $n$ 层内 $E_{n\mathrm{s}} < E_{n\mathrm{p}} < E_{n\mathrm{d}} < E_{n\mathrm{f}}$（由钻穿效应解释）
- **能级交错**：不同 $n$ 层间 $E_{4\mathrm{s}} < E_{3\mathrm{d}}$、$E_{5\mathrm{s}} < E_{4\mathrm{d}}$、$E_{6\mathrm{s}} < E_{4\mathrm{f}} < E_{5\mathrm{d}}$（出现于 K/Ca、Rb/Sr、Cs/Ba）
