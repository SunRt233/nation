# 电子亲合能

- 总览：[[中国化学奥林匹克基本要求-总览]]

## 一、定义
**电子亲合能 EA**：气态基态原子获得一个电子所释放的能量。

$$\mathrm{X}(g) + e^- \to \mathrm{X}^-(g) \quad \Delta H = -EA$$

- EA > 0 表示放热（倾向得电子）
- Cl 的 EA 最大（349 kJ/mol），F 的 EA 反而小于 Cl（F 太小，电子间排斥大）

## 三、核心原理

### 定义

电子亲合能（electron affinity，符号 $E_{\mathrm{ea}}$）是指一个气态原子得到一个电子形成气态负离子所放出的能量。电子亲合能等于电子亲合反应焓变的负值：

$$E_{\mathrm{ea}} = -\Delta H^{\ominus}$$

$$ \mathrm{X}(g) + \mathrm{e}^{-} \to \mathrm{X}^{-}(g)$$

- $E_{\mathrm{ea}} > 0$ 表示放热（体系倾向于获得电子）
- 单位常用 $\mathrm{kJ\cdot mol^{-1}}$ 或 $\mathrm{eV}$

### 第一 vs 第二电子亲合能

- **第一电子亲合能**：中性原子获得第一个电子，一般 $E_{\mathrm{ea}_1} > 0$（放热），因为电子受核吸引
- **第二电子亲合能**：负离子再获得电子，**始终 $E_{\mathrm{ea}_2} < 0$**（吸热），因为负离子排斥外来电子，需吸收能量克服斥力

实例——氧：
$$\mathrm{O}(g) + \mathrm{e}^{-} \to \mathrm{O}^{-}(g) \quad \Delta H_1^{\ominus} = -141\,\mathrm{kJ\cdot mol^{-1}},\quad E_{\mathrm{ea}_1} = 141\,\mathrm{kJ\cdot mol^{-1}}$$

$$\mathrm{O}^{-}(g) + \mathrm{e}^{-} \to \mathrm{O}^{2-}(g) \quad \Delta H_2^{\ominus} = +780\,\mathrm{kJ\cdot mol^{-1}},\quad E_{\mathrm{ea}_2} = -780\,\mathrm{kJ\cdot mol^{-1}}$$

结论：$\mathrm{O}^{2-}$、$\mathrm{S}^{2-}$ 等二价负离子在气态时极不稳定，只能存在于晶体和溶液中。

### 影响电子亲合能的因素

1. **有效核电荷 $Z^{*}$**：$Z^{*}$ 越大，核对额外电子的吸引越强，$E_{\mathrm{ea}}$ 越大
2. **原子半径**：半径越小，电子-电子排斥越强（对第二周期元素尤为显著）
3. **电子构型**：半满/全满构型加电子困难（如 N 的 $2\mathrm{p}^3$ 半满，$E_{\mathrm{ea}} \approx 0$）
4. **是否有空轨道**：第三周期起有空的 d 轨道可容纳额外电子，减小排斥

### 第二周期异常的原因

第二周期非金属原子半径特别小，电子间排斥力很强，当加和电子形成负离子时放出的能量因排斥而减小。第三周期对应元素原子体积较大，且同一层中有空的 d 轨道能容纳电子，排斥作用减小，故加和电子时放出的能量反而更大。

因此：$E_{\mathrm{ea}}(\mathrm{F}) < E_{\mathrm{ea}}(\mathrm{Cl})$、$E_{\mathrm{ea}}(\mathrm{O}) < E_{\mathrm{ea}}(\mathrm{S})$

### 周期表趋势

- **大体趋势**：右上元素电子亲合能大（非金属性强），与电离能趋势类似
- **全局最大**：Cl（$E_{\mathrm{ea}} \approx 349\,\mathrm{kJ\cdot mol^{-1}}$），而非 F
- **特殊点**：第二周期元素 $E_{\mathrm{ea}}$ 常小于同族第三周期元素

## 四、关键结论
### 某些非金属元素的第一电子亲合能（课本表 11.9，单位 $\mathrm{kJ\cdot mol^{-1}}$）

| 元素 | $E_{\mathrm{ea}_1}$ | 元素 | $E_{\mathrm{ea}_1}$ |
|:---:|:---:|:---:|:---:|
| C | 122 | F | 328 |
| N | $0 \pm 20$ | Cl | **348.6** |
| O | 141 | Br | 324.6 |
| S | 134 | I | 295 |

### EA 异常（竞赛考点）

| 比较 | 原因 |
|------|------|
| EA(Cl) > EA(F) | F 的 2p 轨道太小，电子-电子排斥大；Cl 有 3d 空轨道缓冲 |
| EA(S) > EA(O) | 同上：S 半径大且有空 3d 轨道 |
| EA(N) ≈ 0 | N 的 $2\mathrm{p}^3$ 半满稳定，加电子需进入已有电子的 p 轨道，排斥大 |
| EA(F) = 328 < EA(Cl) = 349 | Cl 是全局最大值 |

### 电子亲合能数值的意义

- **Cl 最大**（$348.6\,\mathrm{kJ\cdot mol^{-1}}$）：表示 Cl 原子获得电子形成 $\mathrm{Cl}^{-}$ 的倾向最强
- **F 虽 EA 不是最大**，但单质 $\mathrm{F}_2$ 在化学反应中最活泼——这说明反应趋势还需考虑键能、晶格能等因素，不能只看单个原子的 EA
- **金属的 EA 一般较小或为负**（实验测定困难，数据不完整）

### 第二电子亲合能：始终为负

- $\mathrm{O}^{-} + \mathrm{e}^{-} \to \mathrm{O}^{2-}$：$E_{\mathrm{ea}_2} = -780\,\mathrm{kJ\cdot mol^{-1}}$（强吸热！）
- 气态 $\mathrm{O}^{2-}$ 极不稳定，仅在离子晶体和溶液中因晶格能/水合能补偿而存在

### EA 周期变化规律

周期表中，非金属原子的电子亲和能越大，表示该原子生成负离子的倾向越大。大体趋势：
- **同周期**：左→右 $E_{\mathrm{ea}}$ 增大（非金属性增强）
- **同族**：第二周期 $E_{\mathrm{ea}}$ 异常偏小；第三周期起上→下 $E_{\mathrm{ea}}$ 减小（但变化不规则，可用数据有限）

## 五、常见分类或情形

### 按电子亲合能级数分类

| 级数 | 过程 | 符号特征 |
|:---:|------|:---:|
| 第一电子亲合能 | $\mathrm{X} + \mathrm{e}^{-} \to \mathrm{X}^{-}$ | 一般为正值（放热） |
| 第二电子亲合能 | $\mathrm{X}^{-} + \mathrm{e}^{-} \to \mathrm{X}^{2-}$ | 始终为负值（吸热，需克服静电排斥） |

### 按元素类型分类

| 元素类型 | $E_{\mathrm{ea}}$ 特征 | 原因/示例 |
|------|:---:|------|
| 卤素 | 数值大（300+ kJ/mol） | 只需 1 个电子即达成稀有气体构型：Cl (349), F (328) |
| 氧族 | 数值中等 | O (141), S (134)，但第二 EA 为负 |
| 氮族 | 数值很小 | N ≈ 0（半满稳定），P 数据有限 |
| 碳族 | 数值较小 | C (122) |
| 金属 | 多接近零或难以实验测定 | 实验数据不完整 |
| 稀有气体 | 接近零或负值 | 全满构型，加电子困难 |

## 六、适用条件与限制

1. **仅适用于气态原子/离子**：电子亲合能是气态过程的能量变化，不同于溶液中或固体中的行为。
2. **实验测定困难**：电子亲合能难以直接测量，许多元素的精确数据仍有较大不确定性（如 N 的 $E_{\mathrm{ea}} = 0 \pm 20\,\mathrm{kJ\cdot mol^{-1}}$）。常见元素的可靠数据限于少数非金属。
3. **不能单独预测反应活性**：F 的 EA 小于 Cl（328 vs 349 kJ/mol），但单质 $\mathrm{F}_2$ 更活泼。化学反应趋势必须综合考虑原子化能、键能、晶格能、水合能等多因素。
4. **$E_{\mathrm{ea}} = -\Delta H^{\ominus}$ 的定义需注意符号**：放热时 $E_{\mathrm{ea}} > 0$。不同教材符号约定可能不同，竞赛中以放热为正。
5. **二价负离子的特殊性**：$\mathrm{O}^{2-}$ 的第二 EA 为强吸热（-780 kJ/mol），因此孤立气态 $\mathrm{O}^{2-}$ 不存在；在晶体中由晶格能稳定，溶液中由水合能稳定。

## 七、常见比较与易混点

### 电离能 vs 电子亲合能

| 项目 | 电离能 $I$ | 电子亲合能 $E_{\mathrm{ea}}$ |
|------|------|------|
| 过程方向 | 失电子（$\to$ 阳离子） | 得电子（$\to$ 阴离子） |
| 放热/吸热 | 始终吸热（$I > 0$，需供能） | 第一 EA 通常放热（$E_{\mathrm{ea}} > 0$） |
| 全部元素可测 | 是 | 否（仅部分元素有可靠数据） |
| 数值范围 | 376~2372 kJ/mol | 约 0~349 kJ/mol |
| 全局最大 | He | Cl（非 F） |

### EA vs Mulliken 电负性中的 $E$

在 Mulliken 电负性定义中 $\chi \propto (I + E_{\mathrm{ea}})$，$I$ 和 $E_{\mathrm{ea}}$ 取相同单位（eV 或 kJ/mol）。此处的 $E_{\mathrm{ea}}$ 是绝对值（放热为正）。

### F vs Cl：电子亲合能反常

| 比较项 | F | Cl |
|------|:---:|:---:|
| 原子半径 / pm | 64 | 99 |
| $E_{\mathrm{ea}}$ / kJ·mol⁻¹ | 328 | **349**（更大！） |
| 反常原因 | 半径过小，电子间排斥极大 | 半径适中，且有空的 3d 轨道可分散电子云 |
| 单质活泼性 | F₂ 更强 | — |

这个反差是竞赛中考察"不能仅凭单一参数判断反应活性"的典型案例。
