# 钛

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]

## 一、概述
- [Ar]3d²4s²，第一过渡系
- 常见氧化态：+4（最稳定），+3
- TiO₂（钛白）：最重要白色颜料

## 二、重要反应
- TiCl₄ + 2H₂O → TiO₂ + 4HCl（TiCl₄ 强烈水解，空气中发烟）
- TiO₂ + 2H₂SO₄(浓热) → Ti(SO₄)₂ + 2H₂O

### Ti³⁺ 的颜色
[Ti(H₂O)₆]³⁺ 紫色（d¹, $t_{2g}^1$，d-d 跃迁）。Ti³⁺ 是强还原剂。

## 三、核心原理

### 3.1 钛的矿石与冶炼
钛铁矿的主要成分是 $\mathrm{FeTiO}_3$，还含有 Si、Ca、Mg 等杂质。矿石用浓 $\mathrm{H}_2\mathrm{SO}_4$ 在 $200^{\circ}\mathrm{C}$ 左右处理（酸法分解）：
$$\mathrm{FeTiO}_3 + 3\mathrm{H}_2\mathrm{SO}_4 \longrightarrow \mathrm{Ti(SO}_4)_2 + \mathrm{FeSO}_4 + 3\mathrm{H}_2\mathrm{O}$$
利用溶解度的不同分离得 $\mathrm{Ti(SO}_4)_2$，水解后得 $\mathrm{H}_2\mathrm{TiO}_3$，灼烧得 $\mathrm{TiO}_2$：
$$\mathrm{Ti(SO}_4)_2 + 3\mathrm{H}_2\mathrm{O} \longrightarrow \mathrm{H}_2\mathrm{TiO}_3 + 2\mathrm{H}_2\mathrm{SO}_4$$
$$\mathrm{H}_2\mathrm{TiO}_3 \xrightarrow{\triangle} \mathrm{TiO}_2 + \mathrm{H}_2\mathrm{O}$$

### 3.2 d 区元素的通性
Ti 是 d 区 IVB 族元素，价电子构型 $3\mathrm{d}^2 4\mathrm{s}^2$。d 区元素的特点：
- $(n-1)\mathrm{d}$ 电子和 $n\mathrm{s}$ 电子均可参与成键，存在连续可变的多种氧化态
- 容易作为中心原子形成配合物（有空的 $(n-1)\mathrm{d}$ 轨道）
- 元素及其化合物呈现多姿多彩的颜色（d-d 跃迁吸收能量在可见光区）
- 多数元素熔点在 $1500^{\circ}\mathrm{C}$ 以上，密度较大

### 3.3 第一过渡系最高氧化态的标准电极电势
$E^{\ominus}$ 依次增大，氧化性依次增强：
$\mathrm{TiO}^{2+} + 2\mathrm{H}^{+} + 2\mathrm{e} \rightleftharpoons \mathrm{Ti}^{3+} + \mathrm{H}_2\mathrm{O}$，$E^{\ominus} = 0.1\mathrm{V}$

## 四、关键结论
- Ti 的主要氧化态为 $+4$（最稳定）和 $+3$
- $\mathrm{TiO}_2$ 是最重要的白色颜料（钛白），熔点高、化学稳定性好
- Ti 在地壳中丰度不低但冶炼困难（比铁昂贵），性能优于铁（耐腐蚀、抗张强度好）
- $\mathrm{TiCl}_4$ 是共价化合物，强烈水解，在空气中发烟
- d 区金属在高温可与 N、B、Si、C 等形成间隙化合物（如 TiN），是耐腐蚀、耐磨的高硬度特种材料

## 五、常见分类或情形
- **钛的氧化态**：$+4$（$\mathrm{TiO}_2$, $\mathrm{TiCl}_4$）、$+3$（$\mathrm{Ti}^{3+}$，紫色，d¹）
- **钛的氧化物**：$\mathrm{TiO}_2$（白色颜料）、$\mathrm{TiO}$、$\mathrm{Ti}_2\mathrm{O}_3$
- **钛的卤化物**：$\mathrm{TiCl}_4$（共价，发烟水解）、$\mathrm{TiCl}_3$（紫色固体）
- **钛的氢氧化物**：$\mathrm{Ti(OH)}_4$（两性，以碱性为主）

## 六、适用条件与限制
- Ti 的冶炼用酸法（浓 $\mathrm{H}_2\mathrm{SO}_4$）分解钛铁矿
- $\mathrm{TiCl}_4$ 强烈水解，需在无水条件下保存和使用
- $\mathrm{Ti}^{3+}$ 是强还原剂，在空气中易被氧化
- 金属钛的冶炼困难、价格昂贵，应用不如铁广泛

## 七、常见比较与易混点
| 对比项 | Ti | Fe |
|------|------|------|
| 地壳丰度 | $10^{-1}$ 量级 | $10^0$ 量级 |
| 冶炼难度 | 困难（需特殊方法） | 较易（热还原法） |
| 性能 | 耐腐蚀、抗张强度好 | 强度一般，易生锈 |
| 应用 | 航天、高端合金 | 基础结构材料 |

- Ti(IV) 的氢氧化物以碱性为主的两性，而 Cr(VI) 的则是强酸性
- d 区元素的最高氧化态一般与其族数相同（Ti 为 $+4$ = IVB）
