## 三、核心原理

### 3.1 Ellingham 图与冶炼方法选择
Ellingham 图（$\Delta G^{\ominus}-T$ 图）以消耗 $1\mathrm{mol\ O_2}$ 生成氧化物为基准，将多个相关反应的 $\Delta G^{\ominus}$ 对 $T$ 作图得到直线。直线的位置决定了金属氧化物被还原的难易程度：
- 位于上方的直线（$\Delta G^{\ominus}$ 代数值大）：氧化物易被还原，对应金属活性低
- 位于下方的直线（$\Delta G^{\ominus}$ 代数值小/负值大）：氧化物难被还原，对应金属活性高
- 下方的单质能还原上方的金属氧化物（如 Al 可还原 FeO、CuO——铝热法）

焦炭作为还原剂的 3 种情况：
1. $\mathrm{C} + \mathrm{O}_2 \to \mathrm{CO}_2$：$\Delta G^{\ominus}$ 几乎不随温度变化（水平线）
2. $2\mathrm{C} + \mathrm{O}_2 \to 2\mathrm{CO}$：$\Delta G^{\ominus}$ 随温度升高而降低（高温还原能力增强）
3. $2\mathrm{CO} + \mathrm{O}_2 \to 2\mathrm{CO_2}$：$\Delta G^{\ominus}$ 随温度升高而增大（高温还原能力减弱）

### 3.2 四种主要冶炼方法
| 方法 | 原理 | 适用 | 实例 |
|------|------|------|------|
| **电解法** | 通电使化合物电解 | 活泼金属（K$\sim$Al） | $\mathrm{2NaCl(l)} \xrightarrow{\text{电解}} 2\mathrm{Na} + \mathrm{Cl_2}$（Downs 法）；$\mathrm{2Al_2O_3} \xrightarrow{\text{电解}} 4\mathrm{Al} + 3\mathrm{O_2}$（Hall-Heroult 法） |
| **热还原法** | 用还原剂（C、CO、Al 等）高温还原 | 中等活泼金属（Zn$\sim$Cu） | $\mathrm{Fe_2O_3} + 3\mathrm{CO} \to 2\mathrm{Fe} + 3\mathrm{CO_2}$（高炉炼铁）；$\mathrm{Cr_2O_3} + 2\mathrm{Al} \to 2\mathrm{Cr} + \mathrm{Al_2O_3}$（铝热法） |
| **热分解法** | 加热使化合物分解 | 不活泼金属 | $2\mathrm{HgO} \xrightarrow{\triangle} 2\mathrm{Hg} + \mathrm{O_2}$；$\mathrm{Ni(CO)_4} \xrightarrow{\triangle} \mathrm{Ni} + 4\mathrm{CO}$（Mond 法） |
| **物理法** | 物理分离 | 天然单质 | Au（淘金）、Pt、稀有气体（分馏） |

### 3.3 矿石分解的三类方法
大多数元素在自然界以化合物形式存在，矿石分解大致分为：
- **酸法**：用浓酸处理矿石。如钛铁矿 $\mathrm{FeTiO_3}$ 用浓 $\mathrm{H_2SO_4}$ 在 $200^{\circ}\mathrm{C}$ 处理得 $\mathrm{Ti(SO_4)_2}$，水解后灼烧得 $\mathrm{TiO_2}$
- **碱法**：用碱熔融。如铬铁矿 $\mathrm{FeCr_2O_4}$ 与 $\mathrm{Na_2CO_3/NaOH}$ 共熔，$\mathrm{Cr(III)}$ 被空气中 $\mathrm{O_2}$ 氧化为 $\mathrm{Cr(VI)}$ 生成 $\mathrm{Na_2CrO_4}$
- **火法**：用 C 等高温处理。如重晶石 $\mathrm{BaSO_4}$ 与 C 在 $800^{\circ}\mathrm{C}$ 加热变为可溶的 BaS

### 3.4 高纯物质的制备
- **电解精炼**：粗铜电解精炼得高纯铜，贵金属（Au、Ag、Pt、Pd）富集在阳极泥中
- **化学气相沉积（CVD）**：先将粗品变成挥发性物质（精馏提纯），再高温逆向分解沉积。如 Si 的纯化（粗 Si $\to$ $\mathrm{SiHCl_3}$ $\to$ 超纯 Si"9 个 9"）；$\mathrm{Zr}$ 的纯化（$\mathrm{Zr} + 2\mathrm{I_2} \xrightarrow{200^{\circ}\mathrm{C}} \mathrm{ZrI_4} \xrightarrow{1300^{\circ}\mathrm{C}} \mathrm{Zr} + 2\mathrm{I_2}$）
- **羰基化合物法**：金属在低温与 CO 反应生成羰基化合物，高温再分解为高纯金属（如 Mond 法提纯 Ni：$\mathrm{Ni} + 4\mathrm{CO} \rightleftharpoons \mathrm{Ni(CO)_4}$）

### 3.5 炼铁、炼铝和炼铜的比较
铁、铝、铜是三种最重要的金属材料，冶炼方法完全不同：
- **炼铁**：焦炭高温热还原法（高炉，主要还原剂为 CO，约 $1500^{\circ}\mathrm{C}$）
- **炼铝**：熔盐电解法（$\mathrm{Al_2O_3}$ 溶于冰晶石 $\mathrm{Na_3AlF_6}$ 中，约 $950^{\circ}\mathrm{C}$ 电解）
- **炼铜**：先用热还原得到粗铜，再用水溶液电解精炼

## 四、关键结论
- Ellingham 图是选择冶炼还原剂和温度条件的理论依据
- 活泼金属（$K \sim Al$）用电解法，中等活泼（$Zn \sim Cu$）用热还原法，不活泼用热分解法
- 焦炭是价廉物美的还原剂：低温 CO 起主要还原作用，高温 C 还原能力更强
- Hall-Heroult 法（电解熔融 $\mathrm{Al_2O_3/Na_3AlF_6}$）使铝成为大众金属
- 高炉炼铁中加入石灰石生成炉渣 $\mathrm{CaSiO_3}$ 与铁水分离
- CVD 法和 Mond 法是制备高纯金属的经典方法

## 五、常见分类或情形
| 方法 | 代表性实例 |
|------|------|
| 电解熔融盐 | Na（Downs 法，电解 $\mathrm{NaCl}$）、Mg（电解 $\mathrm{MgCl_2}$）、Al（Hall-Heroult 法） |
| 电解水溶液 | 电解精炼 Cu、$\mathrm{CuSO_4}$ 溶液镀铜 |
| C/CO 还原 | Fe（高炉：$\mathrm{Fe_2O_3} + 3\mathrm{CO} \to 2\mathrm{Fe} + 3\mathrm{CO_2}$） |
| Al 还原（铝热法） | Cr（$\mathrm{Cr_2O_3} + 2\mathrm{Al} \to 2\mathrm{Cr} + \mathrm{Al_2O_3}$）、Mn |
| H₂ 还原 | W（$\mathrm{WO_3} + 3\mathrm{H_2} \to \mathrm{W} + 3\mathrm{H_2O}$） |
| 热分解 | Hg（$2\mathrm{HgO} \to 2\mathrm{Hg} + \mathrm{O_2}$）、Ni（Mond 法） |
| CVD 法 | 超纯 Si（"9个9"）、纯 Zr |
| 物理法 | Au（淘金）、$\mathrm{O_2}$ 和 $\mathrm{N_2}$（分馏液态空气） |

## 六、适用条件与限制
- Ellingham 图中 $\Delta G^{\ominus}$ 以消耗 $1\mathrm{mol\ O_2}$ 为基准，只适用于比较热力学趋势，实际生产还考虑动力学和成本
- 电解法成本高（耗电大），但能制备最活泼的金属
- 铝热法利用 Al 对氧的极大亲和力（$\mathrm{Al-Al_2O_3}$ 线位于 Ellingham 图最下部）
- 冰晶石 $\mathrm{Na_3AlF_6}$ 在炼铝中起降低熔点和提供导电离子双重作用
- CVD 法先"热合成"后"热分解"的双向反应是获得超高纯度的关键

## 七、常见比较与易混点
| 对比项 | 炼铁 | 炼铝 | 炼铜 |
|------|------|------|------|
| 原料 | 赤铁矿 $\mathrm{Fe_2O_3}$ | 铝土矿 $\mathrm{Al_2O_3}$ | 黄铜矿 $\mathrm{CuFeS_2}$ |
| 方法 | CO 高温还原 | 熔盐电解 | 热还原 $+$ 电解精炼 |
| 温度 | $\sim 1500^{\circ}\mathrm{C}$ | $\sim 950^{\circ}\mathrm{C}$ | — |
| 还原剂 | CO（C） | 电子 | C $\to$ 粗铜 |

- CO 还原 vs C 还原：低温 CO 还原能力强，高温 C 还原能力更强（Ellingham 图中 CO-CO₂ 线和 C-CO 线相交于 $710^{\circ}\mathrm{C}$）
- Alkali metals: 电解熔融盐而非水溶液（因 $\mathrm{H_2O}$ 优先还原）
- 电解精炼：阳极溶解（粗金属），阴极沉积（纯金属），杂质中活泼的留溶液、惰性的成阳极泥
