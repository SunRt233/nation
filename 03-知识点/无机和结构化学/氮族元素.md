# 氮族元素（N, P, As, Sb, Bi）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]

## 一、概述
VA 族：ns²np³。N 和 P 是非金属，As 半金属，Sb 和 Bi 金属。性质递变显著。

- N₂：N≡N 最强键之一，化学惰性
- P：白磷(P₄ 四面体)、红磷、黑磷（竞赛热点：白磷结构和反应）

## 二、重要反应

### 氮的化合物
| 化合物 | 氧化态 | 关键性质 |
|------|:---:|------|
| NH₃ | −3 | 三角锥，Lewis 碱，与 H⁺→NH₄⁺ |
| N₂H₄ (肼) | −2 | 强还原剂，火箭燃料 |
| NO | +2 | 奇电子分子，顺磁性 |
| NO₂ | +4 | 奇电子，红棕色，二聚 N₂O₄（无色，低温） |
| HNO₃ | +5 | 强氧化性酸，王水（3HCl+1HNO₃） |

### 硝酸的反应
- Cu + 4HNO₃(浓) → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O
- 3Cu + 8HNO₃(稀) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O
- 王水：HNO₃ + 3HCl → NOCl + Cl₂ + 2H₂O → 可溶 Au, Pt

### 氨的配合性质
- Ag⁺ + 2NH₃ → [Ag(NH₃)₂]⁺
- Cu²⁺ + 4NH₃ → [Cu(NH₃)₄]²⁺（深蓝色）
- 液态 NH₃ 作为非水溶剂（类似水，自耦电离 2NH₃ ⇌ NH₄⁺ + NH₂⁻）

## 三、核心原理

### 3.1 氮的固定与合成氨
$\mathrm{N}_2$ 分子由叁键结合，键能高达 $945\mathrm{kJ}\cdot\mathrm{mol}^{-1}$。使空气中游离的 $\mathrm{N}_2$ 变为化合态称为固氮。

- $\mathrm{N}_2 + \mathrm{O}_2 \longrightarrow 2\mathrm{NO}$：$\Delta G^{\ominus} = 181 - 0.025T$，转变温度 $T = 7.2 \times 10^{3}\mathrm{K}$（在闪电瞬间或汽车发动机附近发生，不能工业化）
- $\mathrm{N}_2 + 3\mathrm{H}_2 \longrightarrow 2\mathrm{NH}_3$：$\Delta G^{\ominus} = -92.2 + 0.199T$，转变温度 $T = 190^{\circ}\mathrm{C}$。工业条件：压力 $20\sim 30\mathrm{MPa}$，温度 $400\sim 530^{\circ}\mathrm{C}$，催化剂主要成分 $\mathrm{Fe}_2\mathrm{O}_3$ 和 $\mathrm{FeO}$，辅助成分 $\mathrm{Al}_2\mathrm{O}_3$—$\mathrm{CaO}$—$\mathrm{K}_2\mathrm{O}$ 等

### 3.2 氨的性质
- 沸点 $-33^{\circ}\mathrm{C}$，临界温度 $132^{\circ}\mathrm{C}$，加压降温易液化
- 气化热 $23.4\mathrm{kJ}\cdot\mathrm{mol}^{-1}$，可用作制冷剂
- 接受质子能力比 $\mathrm{H}_2\mathrm{O}$ 强，水溶液中部分电离成 $\mathrm{NH}_4^+$ 和 $\mathrm{OH}^-$ 而显碱性
- $\mathrm{NH}_3$ 分子中 N 原子上有孤对电子，容易形成配位化合物
- 液氨是常用的非水溶剂

### 3.3 氨氧化法制硝酸
$4\mathrm{NH}_3 + 5\mathrm{O}_2 \longrightarrow 4\mathrm{NO} + 6\mathrm{H}_2\mathrm{O}$、$2\mathrm{NO} + \mathrm{O}_2 \longrightarrow 2\mathrm{NO}_2$、$3\mathrm{NO}_2 + \mathrm{H}_2\mathrm{O} \longrightarrow 2\mathrm{HNO}_3 + \mathrm{NO}$。$\mathrm{HNO}_3$ 是具有强氧化性的强酸。

### 3.4 王水
1 体积市售浓硝酸 ($70\%$) 和 3 体积市售浓盐酸 ($37\%$) 的混合酸叫"王水"，既有硝酸的氧化性又有盐酸的络合性，能溶解 Au、Pt：$\mathrm{Au} + 4\mathrm{HCl} + \mathrm{HNO}_3 \longrightarrow \mathrm{HAuCl}_4 + \mathrm{NO} + 2\mathrm{H}_2\mathrm{O}$

### 3.5 氮的氧化物
氧化数从 $+1$ 至 $+5$ 齐全：
- $\mathrm{N}_2\mathrm{O}$（+1）：笑气，直线形 N—N—O，有 $\pi_3^4$ 键
- $\mathrm{NO}$（+2）：曾被认为是大气污染元，后发现对血管有调节血压和舒张作用（1992 年 Science 明星分子，1998 年 Nobel 奖）
- $\mathrm{N}_2\mathrm{O}_3$（+3）：$\mathrm{HNO}_2$ 的酸酐
- $\mathrm{NO}_2$（+4）：红棕色，低温二聚为 $\mathrm{N}_2\mathrm{O}_4$
- $\mathrm{N}_2\mathrm{O}_5$（+5）：$\mathrm{HNO}_3$ 的酸酐

### 3.6 VA 族元素的酸碱性递变
N 和 P 的氧化物为酸性，As 和 Sb 的氧化物为两性，Bi 的氧化物为碱性。

## 四、关键结论
- $\mathrm{N}\equiv\mathrm{N}$ 叁键是已知最强的化学键之一（$945\mathrm{kJ}\cdot\mathrm{mol}^{-1}$），导致 $\mathrm{N}_2$ 化学惰性
- 合成氨是涉及化学、化工、电子、机械、冶金等多种行业的综合工业
- $\mathrm{NH}_3$ 产量的约 $90\%$ 用于制化肥；另一个重要用途是制硝酸
- 王水溶解 Au/Pt 的原理：氧化性（$\mathrm{HNO}_3$）+ 络合性（$\mathrm{HCl}$ 提供 $\mathrm{Cl}^-$ 配体）
- $\mathrm{Cu}$ 和稀 $\mathrm{HNO}_3$ 产生 $\mathrm{NO}$，和浓 $\mathrm{HNO}_3$ 产生 $\mathrm{NO}_2$（但实际浓稀无严格界线）
- VA 族自上而下：N(非金属) $\to$ P(非金属) $\to$ As(半金属) $\to$ Sb(金属) $\to$ Bi(金属)

## 五、常见分类或情形
- **氮的氢化物**：$\mathrm{NH}_3$、$\mathrm{N}_2\mathrm{H}_4$（肼）、$\mathrm{NH}_2\mathrm{OH}$（羟氨）、$\mathrm{NaNH}_2$（氨基化钠）、$\mathrm{Li}_3\mathrm{N}$（氮化锂）
- **氮的氧化物**：$\mathrm{N}_2\mathrm{O}$、$\mathrm{NO}$、$\mathrm{N}_2\mathrm{O}_3$、$\mathrm{NO}_2/\mathrm{N}_2\mathrm{O}_4$、$\mathrm{N}_2\mathrm{O}_5$
- **含氧酸**：$\mathrm{HNO}_3$（硝酸）、$\mathrm{HNO}_2$（亚硝酸）
- **氮肥**：$(\mathrm{NH}_4)_2\mathrm{SO}_4$、$\mathrm{NH}_4\mathrm{Cl}$、$\mathrm{CO}(\mathrm{NH}_2)_2$（尿素）

## 六、适用条件与限制
- 合成氨需高压（$20\sim 30\mathrm{MPa}$）、催化剂（铁系）、适当温度（$400\sim 530^{\circ}\mathrm{C}$）
- $\mathrm{H}_2$ 分子很小，在高压下能钻过普通钢材，设备需用特种钢
- $\mathrm{NO}$ 迅速与 $\mathrm{O}_2$ 化合生成 $\mathrm{NO}_2$，汽车尾气的主要有害成分之一
- 液氨作溶剂需在加压或低温条件下
- $\mathrm{Cu}$ 与浓/稀 $\mathrm{HNO}_3$ 的反应产物随浓度连续变化

## 七、常见比较与易混点
| 对比项 | $\mathrm{N}_2$ | $\mathrm{O}_2$ |
|------|------|------|
| 键能 | $945\mathrm{kJ}\cdot\mathrm{mol}^{-1}$（叁键） | $498\mathrm{kJ}\cdot\mathrm{mol}^{-1}$（双键） |
| 沸点 | $-196^{\circ}\mathrm{C}$ | $-183^{\circ}\mathrm{C}$ |
| 空气中含量 | $78\%$ | $21\%$ |

| VA 族 | $\mathrm{N}_2\mathrm{O}_3/\mathrm{HNO}_2$ | $\mathrm{P}_4\mathrm{O}_6/\mathrm{H}_3\mathrm{PO}_3$ | $\mathrm{As}_2\mathrm{O}_3/\mathrm{H}_3\mathrm{AsO}_3$ | $\mathrm{Sb}_2\mathrm{O}_3$ | $\mathrm{Bi}_2\mathrm{O}_3$ |
|:---|:---|:---|:---|:---|:---|
| 酸碱性 | 酸性 | 酸性 | 两性（偏酸） | 两性（偏碱） | 碱性 |
