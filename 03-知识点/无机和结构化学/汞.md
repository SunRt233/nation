# 汞

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]

## 一、概述
- [Xe]4f¹⁴5d¹⁰6s²
- 唯一液态金属（常温），相对论效应 → 6s² 惰性
- 氧化态：+1（Hg₂²⁺，含 Hg—Hg 键！）、+2

## 二、重要反应
### Hg₂²⁺ vs Hg²⁺
- **Hg₂²⁺**：Hg—Hg 键（+0.5 氧化态每个 Hg），直线形配合 [H₂O—Hg—Hg—H₂O]²⁺
- Hg₂Cl₂（甘汞）：白色沉淀，感光变黑（分解为 Hg + HgCl₂）
- 歧化：Hg₂²⁺ ⇌ Hg + Hg²⁺（K 适中，加入沉淀剂或络合剂可拉动）

### 汞的反应
- Hg + S（研磨）→ HgS（除汞泼洒）
- Hg²⁺ + 2I⁻ → HgI₂↓（红色）→ 过量 I⁻ → [HgI₄]²⁻（无色，Nessler 试剂碱液鉴定 NH₄⁺）
- Hg²⁺ + 2OH⁻ → HgO↓(黄) + H₂O（Hg(OH)₂ 不稳定，脱水为 HgO）

### Hg 的有机金属化学
- 甲基汞 CH₃Hg⁺（剧毒，神经毒素，生物累积）

## 三、核心原理

### 3.1 Hg 的独特性质
汞是唯一在室温下呈液态的金属（熔点 $-38.84^{\circ}\mathrm{C}$，沸点 $357^{\circ}\mathrm{C}$）。汞在室温下蒸气压较大，必须储存在密闭容器中或加水封保存。d 区金属 IIB 族熔点从上到下递减：Zn (419.5°C) > Cd (321.1°C) > Hg ($-38.84^{\circ}\mathrm{C}$)。第六周期汞受镧系收缩和相对论效应影响，6s² 电子对极稳定（惰性电子对效应）。

### 3.2 Hg(I) 和 Hg(II)
所谓 Hg(I) 其中含的是 $\mathrm{Hg}_2^{2+}$（含 Hg—Hg 键），每个 Hg 的氧化态为 $+1/2$（形式上）。$\mathrm{Hg}_2^{2+}$ 可发生歧化：$\mathrm{Hg}_2^{2+} \rightleftharpoons \mathrm{Hg} + \mathrm{Hg}^{2+}$，加入沉淀剂或络合剂可拉动平衡。

### 3.3 Nessler 试剂鉴定 NH₄⁺
$\mathrm{Hg}^{2+} + 2\mathrm{I}^{-} \to \mathrm{HgI}_2\downarrow$（红色），过量 $\mathrm{I}^{-}$ 则生成 $\mathrm{[HgI}_4]^{2-}$（无色）。K₂[HgI₄] 的碱性溶液为 Nessler 试剂，遇 $\mathrm{NH}_4^{+}$ 生成红棕色沉淀，用于鉴定微量 $\mathrm{NH}_4^{+}$。

### 3.4 汞的化学反应
- Hg 和 $\mathrm{HNO}_3$ 作用（区别于 Zn 可与稀酸反应）
- Hg 与 S 研磨直接化合生成 HgS（用于清除泼洒的汞）
- $\mathrm{Hg(OH)}_2$ 不稳定，生成即脱水为 HgO（黄色）
- HgO 加热分解：$2\mathrm{HgO} \xrightarrow{\triangle} 2\mathrm{Hg} + \mathrm{O}_2$（热分解法制金属）

## 四、关键结论
- Hg 是唯一室温液态金属，蒸气压大，需密闭或水封保存
- $\mathrm{Hg}_2^{2+}$（亚汞离子）含 Hg—Hg 共价键，不是简单的 $\mathrm{Hg}^{+}$
- Hg 的毒性大，需小心防护（汞中毒危害人体健康）
- IIB 族金属从上到下活泼性递减，Hg 仅与 $\mathrm{HNO}_3$ 反应
- $\mathrm{HgS}$ 是极难溶的硫化物（$K_{sp}$ 极小）

## 五、常见分类或情形
- **Hg(I) 化合物**：$\mathrm{Hg}_2\mathrm{Cl}_2$（甘汞，白色）、$\mathrm{Hg}_2(\mathrm{NO}_3)_2$
- **Hg(II) 化合物**：$\mathrm{HgCl}_2$（升汞，剧毒）、$\mathrm{HgO}$（黄色）、$\mathrm{HgS}$（朱砂，红色）、$\mathrm{HgI}_2$（红色沉淀）
- **Hg 的配合物**：$\mathrm{[HgI}_4]^{2-}$（Nessler 试剂）、$\mathrm{[Hg(CN)}_4]^{2-}$
- **有机汞**：甲基汞 $\mathrm{CH}_3\mathrm{Hg}^{+}$（剧毒，神经毒素，生物累积）

## 六、适用条件与限制
- 汞泼洒时用硫粉覆盖（生成 HgS）或锌粉处理
- $\mathrm{Hg}_2\mathrm{Cl}_2$ 感光变黑（分解为 Hg + $\mathrm{HgCl}_2$）
- 汞在室温蒸气压已较大，使用时要特别小心
- $\mathrm{Hg(NO}_3)_2$ 与 $\mathrm{NH}_3$ 水反应生成 $\mathrm{Hg(NH}_2)\mathrm{NO}_3$（白色沉淀）而非 $\mathrm{Hg(OH)}_2$

## 七、常见比较与易混点
| 对比项 | $\mathrm{Hg}_2^{2+}$（亚汞） | $\mathrm{Hg}^{2+}$（汞） |
|------|------|------|
| 结构 | Hg—Hg 键 | 单个 Hg²⁺ |
| 氧化态 | $+1/2$（形式 $+1$） | $+2$ |
| 代表物 | $\mathrm{Hg}_2\mathrm{Cl}_2$（甘汞） | $\mathrm{HgCl}_2$（升汞） |
| 稳定性 | 可歧化 | 稳定 |

- Zn vs Cd vs Hg 活泼性递减：Zn 与稀酸反应 > Cd 较不活泼 > Hg 仅与 $\mathrm{HNO}_3$ 反应
- $\mathrm{Hg}_2\mathrm{Cl}_2$ vs $\mathrm{AgCl}$：两者都是白色感光沉淀，但 $\mathrm{AgCl}$ 溶于 $\mathrm{NH}_3$，$\mathrm{Hg}_2\mathrm{Cl}_2$ 与 $\mathrm{NH}_3$ 生成 $\mathrm{Hg(NH}_2)\mathrm{Cl} + \mathrm{Hg}$（黑色）
