# 点阵类型（14 种 Bravais 格子）

## 一、定义
| 晶系 | Bravais 格子 | 英文 |
|------|:---:|------|
| 立方 | P, I, F | cubic P/I/F |
| 四方 | P, I | tetragonal P/I |
| 正交 | P, I, F, C | orthorhombic P/I/F/C |
| 六方 | P | hexagonal P |
| 三方 | R | rhombohedral |
| 单斜 | P, C | monoclinic P/C |
| 三斜 | P | triclinic P |

## 三、核心原理
### Bravais 点阵的导出（§13.2）
空间点阵有 3 个基本特征：
1. **周期性**：沿三维方向以 $a, b, c$ 为周期重复
2. **对称性**：必须符合该晶系的特征对称元素要求
3. **点阵点的等效性**：每个点阵点周围的几何环境完全相同

在这 3 个约束下，7 种晶系各自可存在 1 至 4 种不同的点阵型式，总计 **14 种 Bravais 空间点阵型式**（Braivais, 1866 年从点阵对称性推论得出，图 13.3）。

### 14 种 Bravais 点阵符号说明
| 符号 | 英文 | 含义 |
|:---:|------|------|
| P | Primitive | 简单（仅顶点有结点） |
| I | Body-centered (Innenzentriert) | 体心（顶点 + 体心） |
| F | Face-centered | 面心（顶点 + 六个面心） |
| C (A, B) | Base-centered | 底心（顶点 + 一对对面心） |
| R | Rhombohedral | 菱面体（三方晶系特有） |

### 14 种点阵在各晶系中的分布（§13.2 图 13.3）
| 晶系 | 点阵型式 |
|------|------|
| 立方 | P（简单立方）、I（体心立方）、F（面心立方） |
| 四方 | P（简单四方）、I（体心四方） |
| 正交 | P（简单正交）、I（体心正交）、F（面心正交）、C（底心正交） |
| 单斜 | P（简单单斜）、C（底心单斜） |
| 六方 | P（简单六方）——唯一 |
| 三方 | R（菱面体格子），也可取 R 心六方 |
| 三斜 | P（简单三斜）——唯一 |

## 四、关键结论
- 并不是 7 种晶系各有 4 种（P/I/F/C）——每种晶系因对称性约束不同，允许的点阵型式数量不同
- 立方晶系无 C（底心）型式——底心立方会破坏 4 个三重轴的对称性
- 三方晶系情况略复杂：R 心三方可取菱面体格子（$a=b=c, \alpha=\beta=\gamma\neq90^\circ$），也可取 R 心六方
- 体心四方（I）和简单四方（P）可互相转换（取不同的 $a$）

## 六、适用条件与限制
- Bravais 点阵是数学抽象——实际晶体中，原子/离子可以以更复杂的方式排列在点阵点上
- P 简单点阵并非"内容最少"——有些结构虽然看似复杂，实际是简单点阵+复合结构基元
- 点阵型式 ≠ 晶体结构类型：例如 CsCl 型晶体属简单立方点阵（P），而非体心立方（I），因顶点和体心是不同离子

## 七、常见比较与易混点
- **点阵（lattice）vs 结构（structure）**：点阵是几何抽象，结构 = 点阵 + 结构基元
- **CsCl = 简单立方 P，不是 BCC**：体心立方要求顶点和体心占据同种原子/离子
- NaCl = 面心立方 F（Na⁺ 和 Cl⁻ 各自都是面心立方子格子）
- **体心四方和简单四方**可以互相转换（取不同的 $a$ 轴方向），因此四方晶系只有 P 和 I 两种独立型式
