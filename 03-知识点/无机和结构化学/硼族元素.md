# 硼族元素（B, Al, Ga, In, Tl）

- 总览：[[中国化学奥林匹克基本要求-总览]]

## 一、定义
IIIA 族：ns²np¹。B 是非金属（缺电子特征），Al 是两性金属，Tl 有惰性电子对效应。

- B：缺电子化合物核心元素，形成多中心键
- Al：地壳含量最高金属元素
- Tl：Tl⁺(6s²) 比 Tl³⁺ 更稳定（惰性电子对）

## 三、核心原理

### 重要反应

### 硼化学（竞赛热门）
| 化合物 | 结构要点 |
|------|------|
| BF₃ | 平面三角形，Lewis 酸（缺电子 B），BF₄⁻ 四面体 |
| B₂H₆ | **3c-2e B—H—B 桥键**（经典案例） |
| H₃BO₃ | 硼酸，层状结构（氢键），一元弱酸，B(OH)₃ + H₂O ⇌ [B(OH)₄]⁻ + H⁺ |
| NaBH₄ | 强还原剂，选择性还原羰基 |
| Na₂B₄O₇·10H₂O | 硼砂，[B₄O₅(OH)₄]²⁻ |

### 硼砂珠试验
Na₂B₄O₇ + CoO → Co(BO₂)₂（蓝色），用于鉴定金属离子

### 铝
- 两性：2Al + 6H⁺ → 2Al³⁺ + 3H₂↑；2Al + 2OH⁻ + 6H₂O → 2[Al(OH)₄]⁻ + 3H₂↑
- Al(OH)₃ 两性：溶于酸 → Al³⁺，溶于碱 → [Al(OH)₄]⁻
- 铝热反应：2Al + Fe₂O₃ → Al₂O₃ + 2Fe

## 三、核心原理

### 3.1 硼的特殊性（对角规则 B-Si）
B 和同族的 Al 相比，一是非金属，一是金属，物理性质、化学性质都不相似，而与右下角的硅比较相似（对角规则）。B 可以有三配位（"$\mathrm{BO}_3$"平面三角形）和四配位（"$\mathrm{BO}_4$"四面体），类似于 $\mathrm{SiO}_4$ 四面体。

### 3.2 硼砂和硼酸
- 硼砂为四硼酸钠，阴离子为 $[\mathrm{B}_4\mathrm{O}_5(\mathrm{OH})_4]^{2-}$，结构中既有"$\mathrm{BO}_4$"四面体又有"$\mathrm{BO}_3$"平面三角形
- 热的浓硼砂溶液和过量 $\mathrm{HCl}$ 起反应，冷却后析出硼酸 $\mathrm{H}_3\mathrm{BO}_3$：$\mathrm{Na}_2\mathrm{B}_4\mathrm{O}_7 + 2\mathrm{HCl} + 5\mathrm{H}_2\mathrm{O} \longrightarrow 2\mathrm{NaCl} + 4\mathrm{H}_3\mathrm{BO}_3$
- 硼酸形式上似三元酸，实际是一元弱酸：$\mathrm{H}_3\mathrm{BO}_3 + \mathrm{H}_2\mathrm{O} \Longrightarrow \mathrm{B(OH)}_4^{-} + \mathrm{H}^{+}$，$K = 5.4 \times 10^{-10}$

### 3.3 硼烷和硼氮化合物
- 硼能与氢形成一系列硼烷：$\mathrm{B}_2\mathrm{H}_6$、$\mathrm{B}_4\mathrm{H}_{10}$、$\mathrm{B}_5\mathrm{H}_9$、$\mathrm{B}_5\mathrm{H}_{11}$、$\mathrm{B}_6\mathrm{H}_{10}$、$\mathrm{B}_{10}\mathrm{H}_{14}$ 等
- 硼烷在空气中燃烧生成 $\mathrm{B}_2\mathrm{O}_3$ 和 $\mathrm{H}_2\mathrm{O}$ 并大量放热
- 氮化硼 (BN) 的性质与石墨相似；$\mathrm{B}_3\mathrm{N}_3\mathrm{H}_6$ 和 $\mathrm{C}_6\mathrm{H}_6$ 性质相似，被称为"无机苯"

### 3.4 含氧酸的酸性规律（适用于同周期比较）
同一周期的几种最高氧化态含氧酸相比较：非羟基氧越多，R 电负性越大，氧化态越高，对羟基氧吸引力越大，酸性越大。
- $\mathrm{HClO}_4$（$\mathrm{(HO)ClO}_3$）$K_a = 1\times 10^{10}$
- $\mathrm{H}_2\mathrm{SO}_4$（$\mathrm{(HO)}_2\mathrm{SO}_2$）$K_{a_1} = 1\times 10^{3}$
- $\mathrm{H}_3\mathrm{PO}_4$（$\mathrm{(HO)}_3\mathrm{PO}$）$K_{a_1} = 7\times 10^{-3}$
- $\mathrm{H}_4\mathrm{SiO}_4$（$\mathrm{(HO)}_4\mathrm{Si}$）$K_{a_1} = 1\times 10^{-10}$

### 3.5 缩合酸
缩合酸的酸性总是大于原正酸（非羟基氧增多），如焦磷酸 $\mathrm{H}_4\mathrm{P}_2\mathrm{O}_7$ 的 $K_{a_1} = 3.2\times 10^{-2}$，大于正磷酸的 $K_{a_1} = 7\times 10^{-3}$。

## 四、关键结论
- B 是 p 区唯一的非金属（与同族的 Al 金属性质差异大），遵循对角规则与 Si 相似
- $\mathrm{H}_3\mathrm{BO}_3$ 是一元弱酸（非三元），$K = 5.4\times 10^{-10}$
- $\mathrm{B}_2\mathrm{H}_6$ 中存在 3c-2e B—H—B 桥键（缺电子特征）
- $\mathrm{B}_3\mathrm{N}_3\mathrm{H}_6$（无机苯）与 $\mathrm{C}_6\mathrm{H}_6$ 等电子、等结构
- 同周期含氧酸酸性：$\mathrm{HClO}_4 > \mathrm{H}_2\mathrm{SO}_4 > \mathrm{H}_3\mathrm{PO}_4 > \mathrm{H}_4\mathrm{SiO}_4$

## 五、常见分类或情形
- **硼的氢化物（硼烷）**：$\mathrm{B}_2\mathrm{H}_6$、$\mathrm{B}_4\mathrm{H}_{10}$、$\mathrm{B}_5\mathrm{H}_9$ 等
- **硼酸盐**：硼砂 $\mathrm{Na}_2[\mathrm{B}_4\mathrm{O}_5(\mathrm{OH})_4]\cdot 8\mathrm{H}_2\mathrm{O}$
- **硼酸**：$\mathrm{H}_3\mathrm{BO}_3$
- **硼的卤化物**：$\mathrm{BF}_3$（平面三角形，Lewis 酸）
- **硼氮化合物**：BN（类石墨）、$\mathrm{B}_3\mathrm{N}_3\mathrm{H}_6$（无机苯）
- **配位氢化物**：$\mathrm{LiBH}_4$、$\mathrm{Al(BH}_4)_3$

## 六、适用条件与限制
- 硼砂溶于水，重结晶即可纯化（不同温度得不同结晶水合物）
- $\mathrm{H}_3\mathrm{BO}_3$ 的酸性来自与 $\mathrm{H}_2\mathrm{O}$ 的配位而非自身电离 $\mathrm{H}^{+}$
- 硼烷在空气中易燃，需无水无氧处理
- 缩合酸的形成与浓度、酸度、温度有关

## 七、常见比较与易混点
| 易混点 | 区分 |
|------|------|
| B vs Al | B 为非金属（对角规则与 Si 相似），Al 为金属 |
| $\mathrm{H}_3\mathrm{BO}_3$ 是一元还是三元酸 | 形式上三元，实际一元弱酸（加合 $\mathrm{OH}^{-}$ 而非电离 $\mathrm{H}^{+}$） |
| $\mathrm{B}_3\mathrm{N}_3\mathrm{H}_6$ vs $\mathrm{C}_6\mathrm{H}_6$ | 等电子体，分别称为"无机苯"和苯 |
| 缩合酸 vs 原正酸 | 缩合酸非羟基氧更多，酸性更强 |
