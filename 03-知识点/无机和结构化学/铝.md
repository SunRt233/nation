# 铝

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-无机和结构化学]]
- 对应考纲条目：[[13-元素化学]]

## 一、概述
- 地壳含量最高的金属元素（~8.2%），以铝硅酸盐和 Al₂O₃ 存在
- 工业电解制铝：Hall-Héroult 法（Al₂O₃ 溶于冰晶石 Na₃AlF₆ 中电解）

## 二、重要反应
### 两性
Al + 酸 → Al³⁺ + H₂；Al + 碱 → Al(OH)₄⁻ + H₂

### Al³⁺ 的水解
Al³⁺ + 3H₂O ⇌ Al(OH)₃ + 3H⁺（Al³⁺ 溶液呈酸性，pKa₁ ≈ 5.0）

### 铝盐
- 无水 AlCl₃：共价性，二聚体 Al₂Cl₆（2 个 3c-4e Al—Cl—Al 桥），Lewis 酸催化剂
- Al₂(SO₄)₃ + K₂SO₄ → KAl(SO₄)₂·12H₂O（明矾）
- Al(OH)₃ 的两性 → 分离 Al³⁺ 与 Fe³⁺/Cr³⁺

## 三、核心原理

### 3.1 铝的存在与冶炼
铝在地壳中的丰度仅次于氧和硅，居金属之首。自然界大部分铝以铝硅酸盐形式存在（如高岭土 $\mathrm{H}_2\mathrm{Al}_2(\mathrm{SiO}_4)_2\cdot \mathrm{H}_2\mathrm{O}$、正长石 $\mathrm{KAlSi}_3\mathrm{O}_8$），不宜作炼铝原料。单独氧化铝矿石有铝矾土 $\mathrm{AlO(OH)}$、刚玉 $\alpha-\mathrm{Al}_2\mathrm{O}_3$ 等。铝矾土是炼铝的工业原料。

### 3.2 电解制铝（Hall-Heroult 法）
炼铝不能用 C 或 CO 还原（$\Delta G$ 转变温度太高，在 $2000^{\circ}\mathrm{C}$ 以上）。用电解熔盐法：将 $\mathrm{Al}_2\mathrm{O}_3$ 与冰晶石 $\mathrm{Na}_3\mathrm{AlF}_6$ 的混合物在约 $950^{\circ}\mathrm{C}$ 高温下熔化后电解。冰晶石的作用是降低电解液熔化温度（$\mathrm{Al}_2\mathrm{O}_3$ 熔点 $2050^{\circ}\mathrm{C}$），同时提供导电离子 $\mathrm{AlF}_6^{3-}$。液态金属铝沉于电解槽下部。

### 3.3 氧化铝的多晶型
$\mathrm{Al}_2\mathrm{O}_3$ 因制备条件不同有 $\alpha, \beta, \gamma, \delta, \theta$ 等多种形式。刚玉是 $\alpha-\mathrm{Al}_2\mathrm{O}_3$，硬度仅次于金刚石。含少量 $\mathrm{Cr}_2\mathrm{O}_3$ 的 $\mathrm{Al}_2\mathrm{O}_3$ 显红色（红宝石），含少量 $\mathrm{Fe}_2\mathrm{O}_3$ 及 $\mathrm{TiO}_2$ 显蓝色（蓝宝石）。$\gamma-\mathrm{Al}_2\mathrm{O}_3$ 用作催化剂载体或吸附剂，$1\mathrm{g}$ 的小孔表面面积在 $200\sim 600\mathrm{m}^2$ 之间。

### 3.4 铝的化学性质
铝导电能力略小于铜，但质轻价廉。铝与稀酸反应放出 $\mathrm{H}_2$（Cu 除外）。纯铝较软，掺入 $\mathrm{Cu}, \mathrm{Mn}, \mathrm{Mg}, \mathrm{Si}$ 等制成合金可增加强度和耐腐蚀性。铝是两性金属，既可溶于酸也可溶于强碱。

## 四、关键结论
- 铝是地壳中丰度最大的金属元素，以铝硅酸盐和氧化铝形式存在
- 铝的冶炼采用熔盐电解法（Hall-Heroult 法），绝非热还原法（对比炼铁的焦炭热还原法）
- $\alpha-\mathrm{Al}_2\mathrm{O}_3$（刚玉）硬度仅次于金刚石；$\gamma-\mathrm{Al}_2\mathrm{O}_3$ 是多孔材料
- Ellingham 图中 $\mathrm{Al}-\mathrm{Al}_2\mathrm{O}_3$ 线位于最下部，因此 Al 可用于铝热法还原其他金属氧化物
- 历史上曾用 K-Hg 齐、金属 K、金属 Na 等还原 $\mathrm{AlCl}_3$ 获得铝，成本极高

## 五、常见分类或情形
- **氧化铝形态**：$\alpha-\mathrm{Al}_2\mathrm{O}_3$（刚玉、宝石）、$\gamma-\mathrm{Al}_2\mathrm{O}_3$（多孔催化剂载体）
- **铝硅酸盐矿物**：高岭土、正长石等
- **铝盐**：无水 $\mathrm{AlCl}_3$（共价化合物，Lewis 酸催化剂）、明矾 $\mathrm{KAl(SO}_4)_2\cdot 12\mathrm{H}_2\mathrm{O}$
- **铝的合金**：铝合金（掺 Cu, Mn, Mg, Si 等）

## 六、适用条件与限制
- 铝硅酸盐中 Al 和 Si 分离困难，不宜作炼铝原料
- 电解制铝需要高温（约 $950^{\circ}\mathrm{C}$）和冰晶石助熔
- $\gamma-\mathrm{Al}_2\mathrm{O}_3$ 的孔径分布因制备条件不同而异
- 铝与稀酸反应但 Cu 除外（铜不溶于稀 HCl/$\mathrm{H}_2\mathrm{SO}_4$）

## 七、常见比较与易混点
| 对比项 | 炼铁 | 炼铝 |
|------|------|------|
| 方法 | 焦炭高温热还原 | 熔盐电解 |
| 原料 | 赤铁矿 $\mathrm{Fe}_2\mathrm{O}_3$ | 铝矾土 + 冰晶石 |
| 温度 | 高炉 $500\sim 1500^{\circ}\mathrm{C}$ | 约 $950^{\circ}\mathrm{C}$ |
| 还原剂 | CO（高炉上部）/ C（高炉下部） | 电子（电解） |

- $\alpha-\mathrm{Al}_2\mathrm{O}_3$ vs $\gamma-\mathrm{Al}_2\mathrm{O}_3$：前者坚硬致密（刚玉），后者多孔高比表面
- Be-Al 对角线规则：$\mathrm{BeO}$ 和 $\mathrm{Al}_2\mathrm{O}_3$ 均为两性，$\mathrm{BeCl}_2$ 和 $\mathrm{AlCl}_3$ 均为共价氯化物
