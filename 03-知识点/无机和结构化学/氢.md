# 氢

## 一、定义
- 位置：1s¹，周期表特殊位置（常放在 IA 和 VIIA 之上）
- 同位素：¹H(氕), ²H(D, 氘), ³H(T, 氚)
- 唯一氧化态：+1, 0, −1
- 单质 H₂：无色无味，最轻气体，易燃爆

## 三、核心原理

### 重要反应
- 工业制备：CH₄ + H₂O → CO + 3H₂（水煤气）；CO + H₂O → CO₂ + H₂（变换反应）
- 实验室：Zn + 2HCl → ZnCl₂ + H₂↑
- 与 O₂：2H₂ + O₂ → 2H₂O（爆鸣气）
- 金属氢化物：2Na + H₂ → 2NaH（H 为 −1 态）

## 三、核心原理

### 3.1 氢的成键类型
氢的核外电子构型为 $1\mathrm{s}^{1}$，化学性质包括：（i）失去电子成为 $\mathrm{H}^{+}$（"裸露的质子"，半径仅 $10^{-3}\mathrm{pm}$，极化能力极强）；（ii）得到电子成为 $\mathrm{H}^{-}$（半径 $154~\mathrm{pm}$，比 $\mathrm{F}^{-}$ 大）；（iii）形成金属氢化物（H 原子/分子填隙）。

### 3.2 氢化物的三种类型
1. **分子型氢化物**：最常见，如 $\mathrm{HX}(\mathrm{X} = \mathrm{F},\mathrm{Cl},\mathrm{Br},\mathrm{I})$、$\mathrm{H}_2\mathrm{O}$、$\mathrm{NH}_3$、$\mathrm{PH}_3$。最简单的硼烷是 $\mathrm{B}_2\mathrm{H}_6$（乙硼烷），因 $[\mathrm{BH}_3]$ 缺电子而聚合。
2. **负离子型氢化物（盐型）**：$\mathrm{H}_2$ 与活泼金属如 $\mathrm{Li}, \mathrm{Na}, \mathrm{Mg}$ 等形成，$\mathrm{H}$ 为 $-1$ 价。反应：$2\mathrm{Na} + \mathrm{H}_2 \longrightarrow 2\mathrm{NaH}$，$\mathrm{Mg} + \mathrm{H}_2 \longrightarrow \mathrm{MgH}_2$。$\mathrm{H}^{-}$ 是极强的碱，可从 $\mathrm{H}_2\mathrm{O}$ 和 $\mathrm{NH}_3$ 中夺取 $\mathrm{H}^{+}$：$\mathrm{CaH}_2 + \mathrm{H}_2\mathrm{O} \longrightarrow \mathrm{Ca(OH)}_2 + \mathrm{H}_2$。$\mathrm{LiH}$ 和 $\mathrm{AlCl}_3$ 反应形成 $\mathrm{LiAlH}_4$，$\mathrm{NaH}$ 和 $\mathrm{B}_2\mathrm{H}_6$ 反应生成 $\mathrm{NaBH}_4$，都是重要的还原剂。
3. **金属型氢化物**：$\mathrm{H}_2$ 与过渡金属形成，保持金属导电性质。整比化合物如 $\mathrm{YH}_3$、$\mathrm{LaH}_3$，非整比化合物如 $\mathrm{PdH}_{x}(x < 1)$。$\mathrm{LaNi}_5 + 3\mathrm{H}_2 \Longrightarrow \mathrm{LaNi}_5\mathrm{H}_6$，是良好的储氢材料（储氢量可达到 $(6 \sim 7) \times 10^{22}$ 氢原子$\cdot\mathrm{cm}^{-3}$，超过液态氢的 $4.2 \times 10^{22}$）。

### 3.3 氢的制取
工业上主要以煤或天然气为原料制取氢气。在催化剂（如 $\mathrm{TiO_2}$）存在下，利用太阳能光解水是获取 $\mathrm{H}_2$ 的理想途径：$\mathrm{H}_2\mathrm{O} \xrightarrow{h\nu} \mathrm{H}_2 + \mathrm{O}_2$。

## 四、关键结论
- $\mathrm{H}^{+}$ 半径仅 $10^{-3}\mathrm{pm}$，极化能力极强，实际上在水溶液中以 $\mathrm{H}_3\mathrm{O}^{+}$ 等形式存在
- $\mathrm{H}^{-}$ 半径 $154~\mathrm{pm}$，是极强的碱和还原剂
- $\mathrm{LiAlH}_4$、$\mathrm{NaBH}_4$ 是化学中很有用的还原剂，由盐型氢化物与缺电子化合物反应制得
- $\mathrm{LaNi}_5\mathrm{H}_6$ 体系：吸氢时放热，放氢时吸热，可实现可逆储氢
- $\mathrm{CaH}_2$ 可除去有机试剂中的微量水分

## 五、常见分类或情形
- **分子型氢化物**：$\mathrm{HX}$、$\mathrm{H}_2\mathrm{O}$、$\mathrm{NH}_3$、$\mathrm{PH}_3$、$\mathrm{B}_2\mathrm{H}_6$ 等
- **盐型氢化物**：$\mathrm{NaH}$、$\mathrm{CaH}_2$、$\mathrm{LiH}$、$\mathrm{MgH}_2$ 等（与 $\mathrm{IA}$、$\mathrm{IIA}$ 活泼金属形成）
- **金属型氢化物**：$\mathrm{PdH}_x$、$\mathrm{LaNi}_5\mathrm{H}_6$、$\mathrm{SmCo}_5\mathrm{H}_x$ 等
- **配位氢化物**：$\mathrm{LiAlH}_4$、$\mathrm{NaBH}_4$

## 六、适用条件与限制
- 盐型氢化物大多数不稳定，受热易分解
- $\mathrm{H}^{-}$ 遇水/醇等质子溶剂立即反应放出 $\mathrm{H}_2$，不能在水溶液中使用
- 金属型氢化物的储氢性能依赖于温度和压力：低温高压吸氢，高温低压放氢
- $\mathrm{H}_2$ 分子与 $\mathrm{N}_2$ 反应合成 $\mathrm{NH}_3$ 需要高温高压和催化剂（$\mathrm{N}\equiv\mathrm{N}$ 键能高达 $945\mathrm{kJ}\cdot\mathrm{mol}^{-1}$）

## 七、常见比较与易混点
| 对比项 | 盐型氢化物 | 金属型氢化物 | 分子型氢化物 |
|------|------|------|------|
| 成键性质 | 离子键（$\mathrm{H}^{-}$） | 金属键+填隙 | 共价键 |
| 典型例子 | $\mathrm{NaH}$, $\mathrm{CaH}_2$ | $\mathrm{PdH}_x$, $\mathrm{LaNi}_5\mathrm{H}_6$ | $\mathrm{H}_2\mathrm{O}$, $\mathrm{NH}_3$, $\mathrm{CH}_4$ |
| 与水反应 | 剧烈放出 $\mathrm{H}_2$ | 不反应 | 一般不反应（部分水解） |
| 导电性 | 熔融态导电 | 导电 | 不导电 |

- $\mathrm{H}^{-}$ 与 $\mathrm{F}^{-}$ 比较：$\mathrm{H}^{-}$ 半径 ($154~\mathrm{pm}$) 大于 $\mathrm{F}^{-}$ ($133~\mathrm{pm}$)，但 $\mathrm{H}^{-}$ 是更强的碱
- 储氢量比较：$\mathrm{LaNi}_5\mathrm{H}_6$ 单位体积氢含量超过液态氢
