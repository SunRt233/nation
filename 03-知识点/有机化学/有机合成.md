## 一、定义

有机合成是指利用有机化学反应，从简单的起始原料出发，通过一系列官能团转化和碳骨架构建，制备目标有机化合物的过程。竞赛中的有机合成题通常要求设计合理的合成路线，考查对反应类型、选择性控制、官能团保护和逆合成分析（retrosynthesis）的掌握。

## 三、核心原理

### 2.1 逆合成分析（Retrosynthesis）

**基本思想**：从目标分子（Target, TGT）出发，通过逻辑上的"切断"（disconnection），将其分解为更简单的前体分子，直到得到可购买的起始原料。

**常用切断策略**：
- **C-C 键切断**：根据已知反应类型的成键位置进行切断
  - 醇 → 格氏试剂 + 羰基化合物
  - α,β-不饱和羰基 → Aldol 缩合切断
  - 1,3-二羰基 → Claisen 缩合切断
- **官能团转化（FGI）**：将目标官能团转化为更容易引入的前体官能团
- **官能团添加（FGA）**：在分子中引入辅助官能团以便切断

### 2.2 常见合成子与等价试剂

| 合成子 | 类型 | 等价试剂 |
|--------|------|---------|
| R⁺ | 亲电性 | R-X、烯烃 |
| R⁻ | 亲核性 | R-MgBr、R-Li、R₂CuLi |
| acyl⁺ | 亲电性 | RCOCl、(RCO)₂O |
| enolate⁻ | 亲核性 | 烯醇盐、烯醇硅醚 |
| a¹（α-给体） | 双官能团 | 1,3-二羰基化合物 |
| d²（β-受体） | 双官能团 | α,β-不饱和羰基 |

### 2.3 选择性控制

1. **化学选择性**：优先与某一官能团反应而不影响其他官能团
   - 醛 > 酮 > 酯 > 酰胺 的还原活性差异
   - 使用不同活性的试剂（如 NaBH₄ vs LiAlH₄）

2. **区域选择性**：在多个可能位置中选择特定位置反应
   - 不对称酮的烯醇化（动力学 vs 热力学控制）
   - 芳香取代的定位规则

3. **立体选择性**：控制产物的手性和构型
   - 顺式/反式加成（如硼氢化-氧化为反马氏顺式加成）
   - 手性辅助基团、手性催化剂

### 2.4 多取代苯合成逆推四步法（Zchem 高阶工具）

**四步法**：

```
目标分子
    ↓
【Step 1】识别目标分子中的官能团和取代模式
    ↓
【Step 2】判断取代基引入顺序（利用定位效应）
         ├─ 先引入邻对位定位基 → 再引入新基团到邻/对位
         └─ 先引入间位定位基 → 再引入新基团到间位
    ↓
【Step 3】检查是否需要占位基/保护基/氧化还原切换
         ├─ 占位基：-SO₃H（引入后稀酸水解除去）
         ├─ 保护基：-NH₂ → -NHCOCH₃（保护氨基不被氧化）
         └─ 氧化切换：-CH₃ → -COOH（将邻对位定位改为间位定位）
    ↓
【Step 4】检查傅克反应是否会发生重排
         ├─ 傅克烷基化：碳正离子中间体 → 可能重排
         └─ 若会重排 → 改用傅克酰基化 + Clemmensen/Wolff-Kishner 还原
    ↓
写出正向合成路线（以苯为起始原料）
```

**经典案例：对位硝基正丙基苯的合成**

```
错误路线：
苯 + CH₃CH₂CH₂Br/AlCl₃ → 正丙基苯（❌ 实际得到异丙基苯，因碳正离子重排）
正丙基苯 + HNO₃/H₂SO₄ → 邻/对位硝基正丙基苯（比例不理想）

正确路线：
苯 + CH₃CH₂COCl/AlCl₃ → 苯丙酮（傅克酰基化，无重排）
苯丙酮 + HNO₃/H₂SO₄ → 对硝基苯丙酮（羰基是间位定位基，硝基进入相对羰基的间位 = 相对烷基的对位）
对硝基苯丙酮 + Zn-Hg/HCl (Clemmensen) → 对硝基正丙基苯 ✅
```

**占位基策略案例：制备 1,3-二溴-5-硝基苯**

```
目标：1,3-二溴-5-硝基苯
    ↓
分析：硝基是间位定位基，溴是邻对位定位基
    ↓
策略：先引入占位基 -SO₃H 控制位置

苯 + 浓 H₂SO₄ → 苯磺酸
苯磺酸 + Br₂/FeBr₃ → 对溴苯磺酸（-SO₃H 是间位定位基，Br 进入间位 = 对位）
对溴苯磺酸 + Br₂/FeBr₃ → 3,5-二溴苯磺酸（-SO₃H 导向间位，-Br 导向邻对位，两者在 3,5 位重合）
3,5-二溴苯磺酸 + 稀 H₂SO₄/△ → 1,3-二溴苯（水解去除 -SO₃H）
1,3-二溴苯 + HNO₃/H₂SO₄ → 1,3-二溴-5-硝基苯 ✅（两个 -Br 共同导向 5 位）
```

### 2.4 官能团保护策略

| 保护基 | 保护对象 | 脱保护条件 |
|--------|---------|-----------|
| 缩醛/缩酮 | 醛/酮 | 稀酸水解 |
| TMS/THP | 醇 | TBAF / 稀酸 |
| 酯/酰胺 | 羧酸/胺 | 碱水解 / 催化氢解 |
| 苄基醚 | 醇 | H₂/Pd |
| Boc/Cbz | 胺 | TFA / H₂/Pd |

**保护原则**：
- 保护基在后续步骤中稳定
- 脱保护条件不破坏分子其他部分
- 上保护和脱保护的产率高、操作简便
