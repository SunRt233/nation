# 高分子化学

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[决赛要求-有机化学]]

## 一、定义

**高分子（Polymer）**：由大量重复结构单元（单体）通过共价键连接而成的大分子化合物，分子量通常在 10⁴ 以上。

**聚合反应（Polymerization）**：单体分子相互连接形成高分子的化学反应。

## 三、核心原理

### 3.1 聚合反应分类

#### 按机理分类

| 类型 | 机理特征 | 单体要求 | 例子 |
|:---|:---|:---|:---|
| **链式聚合（Chain）** | 活性中心（自由基/离子）快速增长 | 含 C=C、C≡C、C=O 等 | 聚乙烯、聚苯乙烯 |
| **逐步聚合（Step）** | 单体逐步反应，每步类似缩合 | 含双官能团 | 聚酯、聚酰胺 |

#### 按反应类型分类

| 类型 | 特点 | 副产物 | 例子 |
|:---|:---|:---:|:---|
| **加聚（Addition）** | 单体不饱和键打开，直接连接 | 无 | 聚乙烯、聚丙烯 |
| **缩聚（Condensation）** | 单体间脱去小分子（H₂O、HCl 等）| 有 | 聚酯、聚酰胺、酚醛树脂 |
| **开环聚合（ROP）** | 环状单体开环连接 | 无 | 聚环氧乙烷、尼龙-6 |

### 3.2 链式聚合的基元反应

**自由基聚合**：
1. **引发**：I → 2R·；R· + M → R-M·
2. **增长**：R-M· + M → R-M-M· → ... → R-(M)n·
3. **终止**：偶合终止或歧化终止
4. **链转移**：向单体、溶剂、链转移剂转移

**离子聚合**：
- **阳离子聚合**：活性中心为碳正离子，适用于异丁烯等富电子单体
- **阴离子聚合**：活性中心为碳负离子，适用于苯乙烯等缺电子单体

**配位聚合（Ziegler-Natta）**：
- 过渡金属催化剂（TiCl₄/AlEt₃）
- 立体选择性控制：等规（isotactic）、间规（syndiotactic）、无规（atactic）

### 3.3 逐步聚合特征

**特点**：
- 反应早期就有高分子生成
- 单体转化率很快达到高值，但分子量增长缓慢
- 需要高转化率才能获得高分子量
- 官能团等活性假设：不同链长的分子末端官能团反应性相同

**Carothers 方程**：
$$\overline{X}_n = \frac{1}{1-p}$$
其中 p 为反应程度，$\overline{X}_n$ 为数均聚合度。

## 四、关键结论

1. **加聚无副产物，缩聚有小分子副产物**
2. **链式聚合分子量增长快，逐步聚合分子量增长慢**
3. **Ziegler-Natta 催化可实现立体选择性聚合**
4. **自由基聚合易发生链转移，离子聚合活性中心寿命长**
5. **缩聚需要严格控制单体配比（等摩尔比）才能获得高分子量**

## 五、常见分类或情形

### 5.1 常见聚合物

| 聚合物 | 单体 | 聚合类型 | 用途 |
|:---|:---|:---|:---|
| **聚乙烯（PE）** | 乙烯 | 自由基加聚 | 塑料薄膜 |
| **聚丙烯（PP）** | 丙烯 | 配位聚合 | 塑料容器 |
| **聚苯乙烯（PS）** | 苯乙烯 | 自由基加聚 | 泡沫塑料 |
| **聚氯乙烯（PVC）** | 氯乙烯 | 自由基加聚 | 管材 |
| **聚对苯二甲酸乙二醇酯（PET）** | 对苯二甲酸 + 乙二醇 | 缩聚 | 饮料瓶 |
| **尼龙-66** | 己二酸 + 己二胺 | 缩聚 | 纤维 |
| **尼龙-6** | 己内酰胺 | 开环聚合 | 纤维 |
| **聚环氧乙烷（PEO）** | 环氧乙烷 | 阳离子开环聚合 | 保水剂 |
| **Kevlar** | 对苯二甲酰氯 + 对苯二胺 | 缩聚 | 防弹衣 |

## 六、适用条件与限制

1. **自由基聚合需要引发剂**——AIBN、BPO 等
2. **离子聚合对水敏感**——需严格无水无氧
3. **Ziegler-Natta 聚合对立体化学敏感**——催化剂选择决定等规度
4. **缩聚需要等摩尔比**——否则分子量受限
5. **开环聚合需要环张力**——三元环、四元环最容易开环聚合

## 七、常见比较与易混点

### 7.1 加聚 vs 缩聚

| | **加聚** | **缩聚** |
|:---|:---|:---|
| 单体 | 含不饱和键 | 含双官能团 |
| 副产物 | 无 | 有（H₂O、HCl 等）|
| 聚合度 | 可很高 | 受转化率限制 |
| 例子 | 聚乙烯 | 尼龙、聚酯 |

### 7.2 链式 vs 逐步聚合

| | **链式聚合** | **逐步聚合** |
|:---|:---|:---|
| 活性中心 | 自由基/离子 | 官能团 |
| 分子量增长 | 快 | 慢 |
| 转化率-分子量关系 | 高转化率时分子量才高 | 分子量随转化率缓慢增长 |
| 例子 | 自由基聚合 | 缩聚 |

## 十、例题

### 例题 1（★ 基础）

**题目**：写出 Kevlar 的缩聚反应方程式。

**解答**：
$$
n\ \text{对苯二甲酰氯} + n\ \text{对苯二胺} \rightarrow [\text{-CO-C}_6\text{H}_4\text{-CO-NH-C}_6\text{H}_4\text{-NH-}]_n + 2n\ \text{HCl}
$$

### 例题 2（★★ 机理）

**题目**：环氧乙烷在酸催化下如何聚合？

**解答**：
1. 质子化环氧乙烷 → 环氧鎓离子
2. 另一分子环氧乙烷进攻 → 开环 → 醇
3. 醇羟基继续质子化、进攻 → 链增长
4. 阳离子开环聚合机理

## 十一、易错点

1. ❌ **混淆加聚和缩聚** → 加聚无副产物，缩聚有小分子
2. ❌ **忽略逐步聚合的等摩尔比要求** → 不等摩尔比限制分子量
3. ❌ **混淆链式和逐步聚合的特征** → 链式有活性中心，逐步是官能团反应
4. ❌ **认为所有聚合都是自由基机理** → 还有离子、配位、开环等机理

### 12.2 学生易踩的认知误区

| 误区 | 正确认识 |
|---|---|
| "聚合就是加聚" | 还有缩聚、开环聚合等 |

| "所有聚合物都是塑料" | 还有纤维、橡胶、弹性体等 |

### 12.3 入门级例题

**题目**：判断下列聚合反应的类型：
(1) 乙烯 → 聚乙烯
(2) 己二酸 + 己二胺 → 尼龙-66

**解答**：
(1) 加聚（链式聚合）
(2) 缩聚（逐步聚合）

### 12.4 与现实/直觉的连接

- 高分子就像"分子项链"——单体是珠子，聚合就是把珠子串成项链。
- 缩聚就像"串珠时掉下小珠子"——每连接两个大珠子，就掉下一个小分子。
