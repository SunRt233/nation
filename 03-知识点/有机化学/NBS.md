# NBS（N-溴代丁二酰亚胺）

- 总览：[[中国化学奥林匹克基本要求-总览]]
- 所属模块：[[基础要求-有机化学]]
- 对应考纲条目：自由基反应、烯丙位/苄位溴化（APiIC 占位）

## 一、定义

**NBS（N-Bromosuccinimide，N-溴代丁二酰亚胺）** 是一种温和的 **溴化试剂**，主要用于 **烯丙位和苄位的自由基溴化**，也可用于 **烯烃的溴加成** 和 **芳环亲电溴化** 的溴源。

结构：
```
      O
      ||
  N—Br
      |
      CH₂
      |
      CH₂
      ||
      O
```

## 二、考纲对应

- 中国化学奥林匹克考纲未单独列出，属于 **自由基取代** 和 **烯烃反应** 的拓展内容。
- 竞赛中常考查烯丙位/苄位选择性溴化。

## 三、核心原理

### 3.1 烯丙位溴化机理（Wohl-Ziegler 反应）

**条件**：NBS + CCl₄（或 CH₂Cl₂）+ 自由基引发剂（AIBN、过氧化苯甲酰）+ 光照

**链引发**：
$$\mathrm{Initiator \rightarrow 2R\cdot}$$
$$\mathrm{R\cdot + NBS \rightarrow R-Br + 琥珀酰亚胺自由基}$$

**链增长**：
$$\mathrm{Br\cdot + RCH_2CH=CH_2 \rightarrow R\dot{CH}CH=CH_2 + HBr}$$
$$\mathrm{HBr + NBS \rightarrow Br_2 + 琥珀酰亚胺}$$
$$\mathrm{R\dot{CH}CH=CH_2 + Br_2 \rightarrow RCHBrCH=CH_2 + Br\cdot}$$

**关键**：
- NBS 维持体系中 **Br₂ 浓度极低**（HBr 与 NBS 缓慢反应生成 Br₂）
- 低 Br₂ 浓度 → 避免烯烃的 Br₂ 加成 → 选择性发生 **烯丙位取代**

### 3.2 为什么不是双键加成？

| 反应 | 条件 | 结果 |
|:---|:---|:---|
| **烯丙位取代** | NBS + 引发剂/光照 | 烯丙位 H 被 Br 取代 |
| **双键加成** | 纯 Br₂（高浓度）| 邻二溴化物 |

NBS 体系的 **低 Br₂ 浓度** 是选择性的关键。

## 四、关键结论

1. **NBS + 引发剂/光照 → 烯丙位/苄位溴化**（选择性自由基取代）
2. **NBS + 极性溶剂（DMSO/H₂O）→ 溴加成或亲电溴化**
3. **NBS 维持低 Br₂ 浓度** → 避免加成，促进取代
4. **烯丙位自由基特别稳定**（p-π 共轭）→ 反应优先发生在此处

## 五、常见分类或情形

| 反应类型 | 条件 | 产物 |
|:---|:---|:---|
| **烯丙位溴化** | NBS + AIBN + CCl₄ + Δ/光照 | 烯丙基溴 |
| **苄位溴化** | 同上 | 苄基溴 |
| **芳环溴化** | NBS + 酸催化 | 芳基溴 |
| **溴加成** | NBS + DMSO/H₂O | 邻溴醇（类似 HOBr 加成）|

## 六、适用条件与限制

| 条件 | 说明 |
|:---|:---|
| **溶剂** | CCl₄、CH₂Cl₂（非极性，自由基条件）|
| **引发剂** | AIBN、BPO、光照 |
| **温度** | 回流或室温光照 |
| **限制** | 需有烯丙位/苄位 H；过度溴化可能竞争 |

## 七、常见比较与易混点

### NBS 溴化 vs Br₂ 溴化

| 维度 | NBS（自由基条件）| Br₂（亲电条件）|
|:---|:---|:---|
| **机理** | 自由基取代 | 亲电加成 |
| **位置** | 烯丙位/苄位 | 双键碳 |
| **产物** | 烯丙基溴 | 邻二溴化物 |
| **双键保留** | ✅ 保留 | ❌ 消失 |

## 十、例题

### 例题 1：产物预测

**题目**：环己烯与 NBS/AIBN/CCl₄ 反应的主产物是什么？

**解析**：
- 烯丙位氢被选择性取代
- 环己烯有三个等价烯丙位
- 产物：**3-溴环己烯**（双键保留）

### 例题 2：条件对比

**题目**：解释为何丙烯与 Br₂/CCl₄ 得 1,2-二溴丙烷，而与 NBS/AIBN 得 3-溴丙烯。

**解析**：
- Br₂/CCl₄：高浓度 Br₂ → 亲电加成机理 → 邻二溴化物
- NBS/AIBN：低浓度 Br₂（原位生成）→ 自由基机理 → 烯丙位 H 被取代（烯丙自由基稳定）

## 十一、易错点

| 错误 | 纠正 |
|:---|:---|
| NBS 直接与双键加成 | NBS 本身不加成；是原位生成的低浓度 Br₂ 参与自由基取代 |
| 忽略引发剂的必要性 | 无引发剂/光照时反应极慢或不发生 |
| 过度溴化 | 多烯丙位底物可能多取代，需控制当量 |

## 十二、🎯 教学视角

**学习路径**：自由基机理 → 烯丙位稳定性 → NBS 选择性溴化 → 与 Br₂ 加成对比

**建议课时**：0.5-1 课时（作为自由基取代的实例）
