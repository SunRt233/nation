## 一、定义

元素推断题是化学竞赛中常见的大题类型，要求根据题目给出的物理化学性质、结构信息、反应特征等线索，推断出未知元素或化合物。这类题目综合考查元素周期律、原子结构、化学键、配位化学、氧化还原等多模块知识。

## 三、核心原理

### 3.1 推断突破口

| 线索类型 | 典型信息 | 推断方向 |
|---------|---------|---------|
| **原子结构** | 最外层电子数、电子层数、离子电子构型 | 确定主族/周期族位置 |
| **物理性质** | 熔点、沸点、密度、颜色、磁性 | 金属/非金属、过渡金属特征 |
| **化学性质** | 常见氧化态、酸碱性质、与酸碱反应 | 金属活泼性、两性元素 |
| **化合物特征** | 氢化物类型、氧化物酸碱性、最高价含氧酸 | 主族元素位置 |
| **晶体结构** | 配位数、晶胞参数、密度 | 原子半径、堆积方式 |
| **定量数据** | 相对原子质量、质量分数、气体体积 | 精确确定元素 |

### 3.2 常见推断模型

1. **周期表位置推断**
   - 最外层电子数 = 主族序数（主族元素）
   - 次外层电子数异常 = 过渡金属或镧系/锕系
   - 电子层数 = 周期数

2. **定量计算推断**
   - 由摩尔质量或密度推算相对原子质量
   - 由质量分数确定化学式
   - 由晶胞参数和密度计算原子量（$M = \rho N_A V / Z$）

3. **反应特征推断**
   - 与酸反应放氢气 → 活泼金属
   - 两性氢氧化物 → Al、Zn、Be、Sn、Pb、Cr 等
   - 强氧化性含氧酸根 → MnO₄⁻、Cr₂O₇²⁻、ClO₄⁻ 等
   - 特殊颜色变化 → 过渡金属离子

### 3.3 常见陷阱

- **同位素干扰**：相对原子质量非整数时考虑同位素丰度
- **多氧化态**：过渡金属不同氧化态性质差异极大
- **惰性电子对效应**：Pb²⁺ 比 Pb⁴⁺ 稳定，Bi³⁺ 比 Bi⁵⁺ 稳定
- **对角线规则**：Li~Mg、Be~Al、B~Si 的相似性

## 九、典型题型

1. **纯文字描述推断**：根据物理化学性质描述推断元素
2. **框图转化推断**：根据化合物之间的转化关系推断各物质
3. **晶体结构+定量推断**：结合晶胞参数和密度计算
4. **综合推断大题**：元素推断 + 反应方程式书写 + 结构判断
