## 三、核心原理

### 3.1 非共价作用力谱系
| 作用力 | 强度 (kJ/mol) | 方向性 | 示例 |
|-------|:---:|:---:|------|
| 离子-离子 | 100–350 | 无 | NaCl |
| 离子-偶极 | 50–200 | 有 | Na⁺·18-冠-6 |
| 氢键 | 4–120 | 强 | DNA 碱基对 |
| π-π 堆积 | 0–50 | 有 | 石墨层间 |
| 疏水作用 | 与表面积相关 | 无 | 蛋白质折叠 |
| 范德华力 | <5 | 无 | 烷烃结晶 |

### 3.2 协同效应
多种弱作用力在同一体系中协同 → 总结合强度远超单个力之和。

### 3.3 超分子相互作用谱系（Steed & Atwood Ch1 §1.7）

| 作用力 | 本质 | 强度 (kJ/mol) | 方向性 | 典型实例 |
|:---|:---|:---:|:---:|:---|
| **离子-离子** | 库仑静电作用 | 100–350 | 无 | 晶格能、盐桥 |
| **离子-偶极** | 电荷-永久偶极 | 50–200 | 有 | Na⁺·18-冠-6（离子-偶极配位）|
| **偶极-偶极** | 永久偶极间 | 5–50 | 有 | 羰基化合物在极性溶剂中的取向 |
| **氢键** | H给体…H受体（部分共价性）| 4–120 | 强 | DNA碱基对、蛋白质二级结构 |
| **阳离子-π** | 阳离子与芳香π电子云 | 5–80 | 有 | K⁺与苯环、杯芳烃上缘络合Cs⁺ |
| **π-π堆积** | 芳香环间（面对面/边对面）| 0–50 | 有 | 石墨层间、碱基堆积 |
| **范德华力** | 瞬时偶极-诱导偶极（London色散）| <5–40 | 无 | 烷烃结晶、紧密堆积 |
| **疏水效应** | 水分子有序笼破坏 → 熵增 | 与表面积相关 | 无 | 环糊精包结、蛋白质折叠 |

#### 疏水效应的本质

疏水效应**不是**一种独立的"吸引力"，而是水溶液中疏水溶质聚集的**熵驱动**现象：
- 疏水溶质迫使周围水分子形成高度有序的"笼状"结构（类冰结构）→ **熵减**（不利）
- 当疏水溶质聚集进入疏水空腔时，有序水笼被破坏释放为自由水 → **熵增**（有利）
- 同时，空腔内高能水分子（与空腔壁作用弱）被释放到本体相 → **焓有利**

> 在环糊精包结中，疏水效应常是主要驱动力之一（见 [[环糊精]]）。

#### 协同效应（Cooperativity）

多种弱相互作用在同一体系中协同作用 → 总结合能远超单个作用之和。
- **加和作用**：各相互作用稳定能简单相加
- **乘法作用**：协同后总稳定性大于加和（螯合效应、大环效应是大规模协同的经典例子）

## 四、关键结论

### 作用力与主体类型的对应关系

| 主体 | 主要作用力 | 次要作用力 |
|:---|:---|:---|
| 冠醚/穴醚 | 离子-偶极 | 范德华、溶剂重排 |
| 环糊精 | 疏水效应、释放高能水 | 范德华、氢键、偶极-偶极 |
| 杯芳烃（下沿）| 离子-偶极（对Na⁺）| 阳离子-π（对K⁺/Ag⁺/Cs⁺）|
| 杯芳烃（空腔）| 疏水效应、范德华 | C–H···π |
| 脲包合物 | 氢键（主体自组装）| 范德华（主-客体）|

## 九、典型题型

### 竞赛常见考点

1. **判断主要驱动力**：根据主体-客体类型和溶剂极性判断主导作用力
2. **疏水效应 vs 疏水相互作用**：前者是熵驱动的水溶液现象，后者常被误称为"疏水吸引力"
3. **氢键的方向性**：在分子识别中提供精确几何匹配，是预组织设计的关键
4. **阳离子-π的竞赛出现频率**：近年信息题中杯芳烃/冠芳烃络合常涉及此作用力
