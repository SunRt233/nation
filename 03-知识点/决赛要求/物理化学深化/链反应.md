# 链反应

- 总览：[[决赛要求-总览]]

## 一、定义

**链反应（Chain Reaction）**：通过活性中间体（通常是自由基或原子）的反复再生，使反应像链条一样持续进行的反应。每消耗一个活性中间体可产生一个新的活性中间体，形成链式传递。

---

## 二、链反应的三个阶段

| 阶段 | 过程 | 特征 |
|:---|:---|:---|
| **链引发** | 产生初始自由基/原子 | 需断裂化学键，活化能高 |
| **链传递** | 自由基与分子反应，生成新自由基 | 反复进行，决定总反应速率 |
| **链终止** | 自由基结合或失活 | 销毁自由基，结束反应链 |

---

## 三、直链反应与支链反应

| 类型 | 特点 | 实例 |
|:---|:---|:---|
| **直链反应** | 每消耗一个自由基产生一个新自由基 | H₂ + Cl₂ → 2HCl |
| **支链反应** | 每消耗一个自由基产生多个新自由基 | H₂ + O₂ → H₂O（爆炸）|

---

## 四、核心原理深化

### 4.1 直链反应的稳态处理

以 $\mathrm{H_2 + Cl_2 \rightarrow 2HCl}$ 为例：

**引发**：$\mathrm{Cl_2 \xrightarrow{k_i} 2Cl\cdot}$

**传递**：
- $\mathrm{Cl\cdot + H_2 \xrightarrow{k_p} HCl + H\cdot}$
- $\mathrm{H\cdot + Cl_2 \xrightarrow{k_p} HCl + Cl\cdot}$

**终止**：$\mathrm{2Cl\cdot \xrightarrow{k_t} Cl_2}$

对链载体 $\mathrm{Cl\cdot}$ 和 $\mathrm{H\cdot}$ 应用稳态近似：

$$\frac{\mathrm{d}[\mathrm{Cl\cdot}]}{\mathrm{d}t} = 2k_i[\mathrm{Cl_2}] - k_p[\mathrm{Cl\cdot}][\mathrm{H_2}] + k_p[\mathrm{H\cdot}][\mathrm{Cl_2}] - 2k_t[\mathrm{Cl\cdot}]^2 = 0$$

$$\frac{\mathrm{d}[\mathrm{H\cdot}]}{\mathrm{d}t} = k_p[\mathrm{Cl\cdot}][\mathrm{H_2}] - k_p[\mathrm{H\cdot}][\mathrm{Cl_2}] = 0$$

由第二式得 $[\mathrm{H\cdot}] = [\mathrm{Cl\cdot}][\mathrm{H_2}]/[\mathrm{Cl_2}]$，代入第一式解得：

$$[\mathrm{Cl\cdot}] = \left(\frac{k_i}{k_t}\right)^{1/2}[\mathrm{Cl_2}]^{1/2}$$

**总反应速率**：

$$v = k_p[\mathrm{Cl\cdot}][\mathrm{H_2}] + k_p[\mathrm{H\cdot}][\mathrm{Cl_2}] = 2k_p\left(\frac{k_i}{k_t}\right)^{1/2}[\mathrm{Cl_2}]^{1/2}[\mathrm{H_2}]$$

### 4.2 链式聚合动力学

根据 [[化学动力学深化]] 中 Atkins 17F.2 补充：

链聚合的三个特征步骤（引发、增长、终止）与直链反应完全对应。对自由基链载体应用稳态近似可得：

**聚合总速率**：

$$v = k_p[\mathrm{M}]\left(\frac{fk_i[\mathrm{In}]}{k_t}\right)^{1/2}$$

**动力学链长**（一个自由基所消耗的单体平均数）：

$$\nu = \frac{k_p[\mathrm{M}]}{2(fk_i k_t)^{1/2}[\mathrm{In}]^{1/2}}$$

### 4.3 支链反应与爆炸极限

支链反应中，一个自由基反应产生多个自由基，导致链载体数目指数增长。当支链产生速率超过终止速率时，反应速率急剧增加，引发爆炸。

**爆炸极限的影响因素**：
- **温度**：升高温度加速支链步骤
- **压力**：低压下自由基易扩散到器壁终止；高压下气相终止占优
- **容器形状**：表面积/体积比影响壁面终止效率

### 4.4 链反应 vs 催化反应

| 特征 | 链反应 | 催化反应 |
|:---|:---|:---|
| 中间体 | 自由基/活性粒子 | 催化剂-底物复合物 |
| 再生方式 | 传递步骤产生新中间体 | 产物释放后催化剂复原 |
| 计量关系 | 中间体被消耗又再生，净不变 | 催化剂不被消耗 |
| 典型实例 | $\mathrm{H_2 + Cl_2}$ | 酶催化、多相催化 |

## 五、竞赛常见考点

1. **写出链反应机理**：识别引发、传递、终止步骤
2. **利用稳态近似推导速率方程**：对自由基中间体应用稳态近似
3. **爆炸极限的解释**：支链反应与链终止的竞争
4. **与催化反应的区别**：链反应消耗并再生中间体，催化反应再生催化剂
