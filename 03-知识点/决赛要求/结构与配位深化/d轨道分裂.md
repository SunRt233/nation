# d轨道分裂

- 所属模块：[[决赛要求-结构与配位深化]]

## 一、定义

在配合物中，中心金属离子的5个简并d轨道由于配体场的静电排斥作用，能量发生分裂，形成不同能级的d轨道组。这是晶体场理论和配位场理论的核心内容。

## 二、核心原理

### 2.1 八面体场中的分裂

八面体配合物中，配体沿 $x, y, z$ 轴方向接近中心离子：

- **$e_g$ 轨道**（高能组）：$d_{z^2}$ 和 $d_{x^2-y^2}$，电子云直接指向配体，受排斥较强
- **$t_{2g}$ 轨道**（低能组）：$d_{xy}$、$d_{xz}$、$d_{yz}$，电子云位于配体之间，受排斥较弱

**分裂能记号**：$\Delta_o$（或 $10Dq$）

**能量重心守恒**：
$$2E(e_g) + 3E(t_{2g}) = 0 \Rightarrow E(e_g) = +0.6\Delta_o, \quad E(t_{2g}) = -0.4\Delta_o$$

### 2.2 四面体场中的分裂

四面体场可看作立方体中交替四个顶点的配体场：

- **$e$ 轨道**（低能）：$d_{z^2}$、$d_{x^2-y^2}$
- **$t_2$ 轨道**（高能）：$d_{xy}$、$d_{xz}$、$d_{yz}$

分裂方向与八面体相反：$\Delta_t \approx \frac{4}{9}\Delta_o$

### 2.3 光谱化学序列

配体按分裂能力从弱到强排列：

$$\text{I}^- < \text{Br}^- < \text{SCN}^- < \text{Cl}^- < \text{NO}_3^- < \text{F}^- < \text{OH}^- < \text{C}_2\text{O}_4^{2-} < \text{H}_2\text{O} < \text{NCS}^- < \text{NH}_3 < \text{en} < \text{bipy} < \text{phen} < \text{NO}_2^- < \text{PPh}_3 < \text{CN}^- < \text{CO}$$

**规律**：
- 卤素 < 氧配体 < 氮配体 < 碳配体
- $\pi$-受体配体（CO、CN⁻）分裂能力最强

## 三、典型题型

1. **绘制不同配位场的d轨道能级分裂图**
2. **根据光谱化学序列判断分裂能大小**
3. **比较配合物的颜色与分裂能的关系**：$\Delta = hc/\lambda$
