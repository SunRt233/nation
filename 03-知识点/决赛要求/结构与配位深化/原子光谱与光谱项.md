## 一、定义

原子光谱项（Term Symbol）是用一套简洁的符号 $^{2S+1}L_J$ 来描述多电子原子在某一电子组态下可能的能级状态。它综合了电子的总自旋量子数 $S$、总轨道角动量量子数 $L$ 和总角动量量子数 $J$，是连接量子力学计算与原子光谱实验观测的核心桥梁。

在化学竞赛中，光谱项的推导和选律的判断是决赛阶段原子结构部分的常见难点和重点。

## 三、核心原理

### 3.1 光谱项符号的构成

光谱项的一般形式：

$$
^{2S+1}L_J
$$

| 符号 | 名称 | 物理意义 | 取值 |
|-----|------|---------|------|
| $S$ | 总自旋量子数 | 所有电子自旋角动量的矢量和 | $0, 1/2, 1, 3/2, \dots$ |
| $2S+1$ | 自旋多重度 | 能级在磁场中分裂的态数 | 单重态(1)、双重态(2)、三重态(3)… |
| $L$ | 总轨道角动量量子数 | 所有电子轨道角动量的矢量和 | $0, 1, 2, 3, \dots$ |
| $J$ | 总角动量量子数 | $L$ 与 $S$ 的矢量和 | $|L-S|, |L-S|+1, \dots, L+S$ |

**$L$ 的字母表示：**

| $L$ | 0 | 1 | 2 | 3 | 4 | 5 | 6 |
|-----|---|---|---|---|---|---|---|
| 符号 | S | P | D | F | G | H | I |

**例子：**
- $^3P_2$：$S=1$（三重态），$L=1$（P项），$J=2$
- $^2D_{5/2}$：$S=1/2$（双重态），$L=2$（D项），$J=5/2$

### 3.2 从电子组态推导光谱项

#### 步骤1：确定未满亚层的电子组态

对于满壳层（$s^2, p^6, d^{10}$ 等），$L=0, S=0$，只贡献 $^1S$ 项，通常不写出。光谱项主要由未满亚层电子决定。

#### 步骤2：计算可能的 $m_l$ 和 $m_s$ 组合

对未满亚层的每个电子，列出所有可能的磁量子数 $m_l$ 和自旋磁量子数 $m_s$ 组合，满足Pauli不相容原理。

#### 步骤3：用微状态法或行列式法求 $L$ 和 $S$

**微状态法**（适用于等价电子，如 $p^2, d^2$）：

1. 列出所有满足Pauli原理的微状态（$m_l, m_s$ 组合）
2. 计算每个微状态的 $M_L = \sum m_l$ 和 $M_S = \sum m_s$
3. 找出最大的 $M_L$ 值，即为最大的 $L$
4. 在该 $L$ 下找出最大的 $M_S$，即为对应的 $S$
5. 扣除已归属的微状态，重复上述过程直至所有微状态归完

**等价电子的光谱项（竞赛常用，需熟记）：**

| 组态 | 光谱项 |
|------|--------|
| $p^1$ 或 $p^5$ | $^2P$ |
| $p^2$ 或 $p^4$ | $^3P, \, ^1D, \, ^1S$ |
| $p^3$ | $^4S, \, ^2D, \, ^2P$ |
| $d^1$ 或 $d^9$ | $^2D$ |
| $d^2$ 或 $d^8$ | $^3F, \, ^3P, \, ^1G, \, ^1D, \, ^1S$ |
| $d^3$ 或 $d^7$ | $^4F, \, ^4P, \, ^2H, \, ^2G, \, ^2F, \, ^2D(2), \, ^2P$ |
| $d^4$ 或 $d^6$ | $^5D, \, ^3H, \, ^3G, \, ^3F(2), \, ^3D, \, ^3P(2), \, ^1I, \, ^1G(2), \, ^1F, \, ^1D(2), \, ^1S(2)$ |
| $d^5$ | $^6S, \, ^4G, \, ^4F, \, ^4D, \, ^4P, \, ^2I, \, ^2H, \, ^2G(2), \, ^2F(2), \, ^2D(3), \, ^2P, \, ^2S$ |

> **记忆技巧**：等价电子的光谱项具有互补性——$p^n$ 与 $p^{6-n}$ 相同，$d^n$ 与 $d^{10-n}$ 相同。

#### 步骤4：确定 $J$ 值

对每一个 $^{2S+1}L$ 项，$J$ 的取值为：

$$
J = |L-S|, \, |L-S|+1, \, \dots, \, L+S
$$

- 若亚层电子数 $\le$ 半满（如 $p^2, d^3$），最低能级对应最小的 $J$（$J = |L-S|$）
- 若亚层电子数 $\ge$ 半满（如 $p^4, d^7$），最低能级对应最大的 $J$（$J = L+S$）
- 半满时（$p^3, d^5$），$L=0$，只有一个 $J=S$

### 3.3 Hund规则对光谱项能级顺序的应用

Hund规则不仅用于电子排布，也用于确定同一电子组态下各光谱项的能级高低：

**规则1**：自旋多重度 $2S+1$ 最大的光谱项能量最低。

**规则2**：在 $S$ 相同的光谱项中，$L$ 最大的能量最低。

**规则3**：在 $S$ 和 $L$ 都相同的情况下（即同一光谱项的不同 $J$）：
- 电子数 $\le$ 半满：$J$ 最小的能级最低（"正常顺序"）
- 电子数 $\ge$ 半满：$J$ 最大的能级最低（"倒置顺序"）

**例子：$p^2$ 组态（如C原子基态）**

光谱项：$^3P, \, ^1D, \, ^1S$

- 规则1：$^3P$（$S=1$）最低
- 规则2：$^1D$（$L=2$）低于 $^1S$（$L=0$）
- 规则3：$^3P$ 项中，$p^2$ 电子数 $<$ 半满（$p^6$ 的一半），故 $J = |1-1| = 0$ 最低

能级顺序（从低到高）：$^3P_0 < \, ^3P_1 < \, ^3P_2 < \, ^1D_2 < \, ^1S_0$

### 3.4 跃迁选律（Selection Rules）

原子光谱中电偶极跃迁必须满足以下选律：

**主要选律：**
1. **$\Delta S = 0$**（自旋守恒，不同自旋多重度之间的跃迁是禁阻的）
2. **$\Delta L = 0, \pm 1$**（但 $L=0 \leftrightarrow L=0$ 禁阻）
3. **$\Delta J = 0, \pm 1$**（但 $J=0 \leftrightarrow J=0$ 禁阻）
4. **$\Delta l = \pm 1$**（单个电子的轨道角动量变化为±1，即 $s \leftrightarrow p, \, p \leftrightarrow d, \, d \leftrightarrow f$）

**说明：**
- 满足选律的跃迁称为**允许跃迁**，谱线较强
- 违反选律的跃迁称为**禁阻跃迁**，但由于自旋-轨道耦合等微扰，实际中仍可微弱发生（如磷光）
- 自旋禁阻跃迁（$\Delta S \ne 0$）通常比轨道禁阻跃迁更弱

**例子：**
- $^3P_1 \to \, ^1S_0$：$\Delta S = -1$（禁阻，自旋禁阻）
- $^3P_1 \to \, ^3S_1$：$\Delta L = -1, \, \Delta S = 0, \, \Delta J = 0$（允许）
- $^3P_0 \to \, ^3P_1$：$\Delta L = 0, \, \Delta S = 0, \, \Delta J = +1$（允许，但纯转动/自旋重取向，电偶极强度弱）

## 四、关键结论

1. **光谱项是电子组态的"标签"**：同一电子组态可对应多个光谱项，代表不同的能量状态
2. **基态光谱项由Hund规则确定**：对竞赛而言，通常只需写出基态光谱项
3. **选律的本质是对称性要求**：电偶极跃迁要求初态和终态的宇称相反（$g \leftrightarrow u$，$\leftrightarrow$ 表示允许，$\not\leftrightarrow$ 表示禁阻）
4. **$LS$耦合适用于轻原子（$Z < 40$）**：重原子需考虑 $jj$ 耦合，竞赛不涉及
5. **光谱项推导的核心是Pauli原理**：等价电子的光谱项比非等价电子少得多（因受Pauli限制）

## 五、常见等价电子组态的光谱项推导示例

### 示例1：$p^2$ 组态（如基态C原子）

**微状态表（部分）：**

| $m_l = +1$ | $m_l = 0$ | $m_l = -1$ | $M_L$ | $M_S$ |
|-----------|-----------|-----------|-------|-------|
| ↑ | ↑ | | +1 | +1 |
| ↑ | ↓ | | +1 | 0 |
| ↑ | | ↑ | 0 | +1 |
| ↑ | | ↓ | 0 | 0 |
| ... | ... | ... | ... | ... |

**推导过程：**
1. 最大 $M_L = +2$（对应 $L=2$，即D项），但 $M_L=+2$ 时两电子 $m_l$ 均为+1，违反Pauli（除非自旋相反），故 $M_S=0$，得 $^1D$
2. 次大 $M_L = +1$，最大 $M_S = +1$（对应 $S=1, L=1$），得 $^3P$
3. 剩余微状态归为一个 $L=0, S=0$，得 $^1S$

**结果：**$^3P, \, ^1D, \, ^1S$

**基态：**$^3P_0$（$p^2 <$ 半满，$J_{\min}$ 最低）

### 示例2：$d^2$ 组态（如基态Ti原子）

**结果（需记忆）：**$^3F, \, ^3P, \, ^1G, \, ^1D, \, ^1S$

**基态：**$^3F_2$（$d^2 <$ 半满，$J_{\min}=|3-1|=2$ 最低）

## 七、竞赛常见题型

### 题型1：写出基态原子的光谱项

**例**：写出基态N原子（$[He]2s^22p^3$）的光谱项。

**解**：$p^3$ 为半满，$S = 3/2$，$L = 0$，$J = 3/2$

答案：$^4S_{3/2}$

### 题型2：判断跃迁是否允许

**例**：判断 $^3P_2 \to \, ^1D_2$ 是否为电偶极允许跃迁。

**解**：$\Delta S = -1 \ne 0$，违反自旋选律。

答案：禁阻（自旋禁阻）

### 题型3：比较能级高低

**例**：$p^2$ 组态的光谱项 $^3P, \, ^1D, \, ^1S$ 中，哪个能量最低？

**解**：Hund规则1，$^3P$ 的 $2S+1=3$ 最大，故 $^3P$ 最低。

答案：$^3P$
