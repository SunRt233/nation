## 一、定义

**分子轨道理论**（Molecular Orbital Theory, MO）是描述分子电子结构的量子力学模型。与定域的价键理论（VB）不同，MO理论强调电子的离域性——分子中的电子不再属于某个原子，而是属于整个分子。决赛阶段要求掌握LCAO-MO方法、同核/异核双原子分子的电子排布、键级计算，以及MO与VB理论的比较。

## 三、核心原理

### 3.1 LCAO-MO基本方法

**原子轨道的线性组合**（Linear Combination of Atomic Orbitals）：

$$psi = c_A phi_A + c_B phi_B$$

- $c_A, c_B$：组合系数，反映原子轨道对MO的贡献比例
- $|c_A|^2 + |c_B|^2 = 1$（归一化）

**组合原则**（Atkins主题9B）：
1. **能量近似原则**：只有能量相近的原子轨道才能有效组合
2. **最大重叠原则**：轨道重叠越大，成键/反键分裂越大
3. **对称性匹配原则**：只有对称性相同的轨道才能组合

### 3.2 同核双原子分子的MO能级顺序

**第二周期双原子分子**有两种能级顺序：

**顺序A**（Z ≤ 7，N₂及更轻）：
$$sigma_{1s} < sigma^*_{1s} < sigma_{2s} < sigma^*_{2s} < pi_{2p} < sigma_{2p} < pi^*_{2p} < sigma^*_{2p}$$

**顺序B**（Z ≥ 8，O₂及更重）：
$$sigma_{1s} < sigma^*_{1s} < sigma_{2s} < sigma^*_{2s} < sigma_{2p} < pi_{2p} < pi^*_{2p} < sigma^*_{2p}$$

**差异根源**：2s-2p轨道能量差。轻原子（Li-N）2s和2p能量接近，发生s-p混杂，使$sigma_{2p}$能量升高（高于$pi_{2p}$）；重原子（O-F）2s-2p能量差大，s-p混杂弱，$sigma_{2p}$低于$pi_{2p}$。

### 3.3 键级与分子稳定性

$$	ext{键级} = rac{	ext{成键电子数} - 	ext{反键电子数}}{2}$$

| 键级 | 稳定性 |
|:---:|:---|
| > 0 | 稳定（可形成分子） |
| = 0 | 无净成键（He₂不存在） |
| < 0 | 极不稳定 |

**示例**：
- H₂⁺：键级 = (1-0)/2 = 0.5（可存在，但键较弱）
- O₂：键级 = (8-4)/2 = 2，顺磁性（$pi^*_{2p}$上有2个未成对电子）
- N₂：键级 = (8-2)/2 = 3，抗磁性

### 3.4 异核双原子分子

**CO与N₂的等电子性**：
- CO和N₂都是14电子，等电子体
- CO的MO排布与N₂类似，但C和O电负性不同
- HOMO主要在C上（偏C的孤对），LUMO主要在C上 → CO是良好的σ给体和π受体

### 3.5 MO vs VB 理论比较

| 特征 | MO理论 | VB理论 |
|-----|-------|-------|
| 电子定位 | 离域（属于整个分子） | 定域（属于两个成键原子） |
| 物理图像 | 轨道在整个分子上离域 | 电子对在原子间定域 |
| O₂顺磁性 | ✅ 自然解释 | ❌ 无法解释 |
| 激发态 | 可描述 | 难描述 |
| 计算复杂度 | 较高 | 较低 |
| 化学直觉 | 较弱 | 较强（与路易斯结构一致）|

**互补性**：MO更适合描述电子光谱、磁性、离域体系；VB更适合描述定域键、反应机理。

## 四、关键结论

1. **键级是MO理论的核心判据**：键级>0才能形成稳定分子
2. **O₂的顺磁性是MO理论最著名的成功**：VB理论预测O₂抗磁性，与实验矛盾
3. **s-p混杂是理解能级顺序B的关键**：轻原子（N₂）$pi_{2p} < sigma_{2p}$，重原子（O₂）相反
4. **异核分子中MO偏向电负性大的原子**：但HOMO/LUMO的分布决定反应活性位点

## 五、典型竞赛题型

### 题型1：写出双原子分子的MO电子排布

**例**：写出O₂的MO电子排布，判断键级和磁性。

**答**：O₂有16个电子。

顺序B：$(sigma_{1s})^2(sigma^*_{1s})^2(sigma_{2s})^2(sigma^*_{2s})^2(sigma_{2p})^2(pi_{2p})^4(pi^*_{2p})^2$

键级 = (8-4)/2 = 2。$pi^*_{2p}$上有2个未成对电子，顺磁性。

### 题型2：比较分子稳定性

**例**：比较O₂⁺、O₂、O₂⁻、O₂²⁻的键长和键能顺序。

**答**：
- O₂⁺（15e）：键级 = 2.5
- O₂（16e）：键级 = 2.0
- O₂⁻（17e）：键级 = 1.5
- O₂²⁻（18e）：键级 = 1.0

键长：O₂⁺ < O₂ < O₂⁻ < O₂²⁻
键能：O₂⁺ > O₂ > O₂⁻ > O₂²⁻
