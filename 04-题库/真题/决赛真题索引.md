# 决赛真题索引

> 本文件通过 Dataview 动态聚合所有已入库的决赛理论题，支持按年份、模块、题型、难度等多维度筛选。
> 实验题暂不入库（用户决策 2026-05-20）。

---

## 一、按届次汇总

```dataview
TABLE WITHOUT ID
  link(file.name, title) as 题目,
  year as 年份,
  exam_session as 场次,
  module as 模块,
  question_type as 题型,
  difficulty as 难度,
  join(knowledge_points, "、") as 核心知识点
FROM "04-题库/真题"
WHERE exam_stage = "决赛"
SORT year DESC, file.name ASC
```

## 二、按模块统计

```dataview
TABLE WITHOUT ID
  count(file.name) as 题数,
  average(difficulty) as 平均难度,
  min(difficulty) as 最低难度,
  max(difficulty) as 最高难度
FROM "04-题库/真题"
WHERE exam_stage = "决赛"
GROUP BY module
SORT count(file.name) DESC
```

## 三、按难度分布

```dataview
TABLE WITHOUT ID
  count(file.name) as 题数,
  join(map(rows, (r) => link(r.file.name, r.title)), "、") as 题目列表
FROM "04-题库/真题"
WHERE exam_stage = "决赛"
GROUP BY difficulty
SORT difficulty ASC
```

## 四、按知识点交叉筛选

```dataview
TABLE WITHOUT ID
  knowledge_points as 知识点,
  count(file.name) as 出现次数,
  join(map(rows, (r) => link(r.file.name, r.title)), "、") as 关联题目
FROM "04-题库/真题"
WHERE exam_stage = "决赛"
FLATTEN knowledge_points
GROUP BY knowledge_points
SORT count(file.name) DESC
LIMIT 30
```

## 五、最近更新

```dataview
TABLE WITHOUT ID
  link(file.name, title) as 题目,
  year as 年份,
  module as 模块,
  difficulty as 难度
FROM "04-题库/真题"
WHERE exam_stage = "决赛" AND updated = date("2026-05-20")
SORT file.name ASC
```
