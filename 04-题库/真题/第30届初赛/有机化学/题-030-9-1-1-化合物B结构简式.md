# 题-030-9-1-1：化合物B的结构简式

## 参考答案

![[mineru/02-真题解析/30届初赛试题解析_images/fe9ddf047af67199ed14673ac6f8143dfe6eebeb11a3e47dca4e4dcfc9bdc3c8.jpg]]

**B 为 N-羟乙基乙酰胺（N-(2-羟乙基)乙酰胺）**

## 解析

本小题中，底物氨基乙醇含有羟基和氨基两个官能团，均可与乙酸酐发生反应（醇解和胺解）。一般来说，氨基氮原子的亲核性强于羟基氧原子，会优先发生胺解；然而，在盐酸中发生的却是醇解反应。这是由于具有碱性的氨基会被盐酸质子化，生成铵盐从而失去亲核性。因此，反应物中仅有的亲核基团羟基便与乙酸酐发生酰基碳上的亲核取代反应，得到化合物 A。

而在碱性条件 $(\mathrm{K}_2\mathrm{CO}_3)$ 下则不存在上述问题，因而得到的产物 B 就是乙酸酐的胺解产物：N-羟乙基乙酰胺。这里，题目已经明确提示 B 是非环状的，也就排除了羟基和氨基都发生亲核进攻而生成了 B' 这种可能。

![[mineru/02-真题解析/30届初赛试题解析_images/c33109fba7423befd1b93b18eb8d6b3adee73a3665e762f359c8501ff19f4cf1.jpg]]  
B'（环状副产物，非题目所求）

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 胺解 vs 醇解 | 氨基亲核性强于羟基，通常优先发生胺解 |
| 酸碱条件影响 | 酸使氨基质子化失去亲核性，碱中氨基保持亲核性 |
| 乙酸酐反应 | 与胺反应生成酰胺，与醇反应生成酯 |
| 选择性控制 | 通过pH调控氨基和羟基的反应选择性 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 画成环状产物B' | 题目明确提示B是非环状的 |
| 画成酯（A） | 碱性条件下氨基优先反应生成酰胺 |
| 忽略羟基保留 | B中羟基未反应，保留在产物中 |
| 混淆胺解和醇解产物 | 胺解得酰胺，醇解得酯 |
