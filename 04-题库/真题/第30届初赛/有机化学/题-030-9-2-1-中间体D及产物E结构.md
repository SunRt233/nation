## 解析

首先我们要捋清该同学所期望发生的反应的历程。第一步，二级醇进攻二氯亚砜中的硫原子，得到 ROSOCl 和氯化氢。由于反应发生在非质子溶剂乙醚中，体系中的 HCl 本该无法电离出氯离子；但由于吡啶的存在，发生酸碱中和生成 pyH⁺ 和 Cl⁻。第二步，游离的 Cl⁻ 与 ROSOCl 发生 Sₙ2 反应，得到构型翻转的卤代烃。

事实上，反应得到另一个同分异构体，那么某步反应机理势必发生了变化。由于反应物只有羟基一个亲核位点，所以第一步机理发生变化的可能性不大，因此反应会经历化合物 D。而在第二步中，则有如下图所示两个亲电位点可供游离的氯离子进攻（分别用虚线和实线表示）。用实线标明的反应历程叫作"$S_N2'$ 反应"，这样反应就得到了实际产物 E。在官方答案中，还给出了 E 的绝对构型（不作具体要求）。

**知识拓展：**

9-2-1 中涉及的 "$S_N2'$ 反应"，事实上是不太常见的。只有在像乙醚这样的惰性溶剂中，较"软"的亲核试剂对离去基团所在位置空间位阻较大的底物发生进攻时，才能在与 $S_N2$ 的竞争中胜出。显然，在此反应中氯离子从双键处进攻，空间位阻会更小。

下面用 $\tau$ 键模型就9-2-1答案中新生成手性中心的构型作简单解释。该模型认为，碳碳双键是由两个 $\mathrm{sp}^3$ 杂化碳中间形成的两个 $\tau$ 键组成的。在该模型的处理方式下，$\mathrm{S_N2'}$ 反应可以看作是 $\mathrm{S_N2}$ 和 E2 的组合。由于两者都有反式的立体化学要求，所以氯离子将从原来羟基所在的 syn 面进攻得到目标产物。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| SN2反应 | Cl⁻对OSOCl进行背面进攻，构型翻转 |
| SN2'反应 | Cl⁻对烯丙基体系的γ位进攻，伴随双键迁移 |
| 二氯亚砜氯化 | SOCl₂将醇转化为ROSOCl，再被Cl⁻取代 |
| 吡啶作用 | 中和HCl生成Cl⁻，提供亲核试剂 |
| 立体化学 | SN2'反应的立体选择性（syn面进攻） |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 只画SN2产物C | 实际得到的是SN2'产物E |
| D结构错误 | D为ROSOCl（氯亚硫酸酯） |
| 忽略立体化学 | 官方答案给出了E的绝对构型 |
| 未理解为何发生SN2' | 烯丙位空间位阻大，Cl⁻从双键端进攻位阻更小 |
| 混淆SN2和SN2' | SN2'伴随双键位置迁移 |
