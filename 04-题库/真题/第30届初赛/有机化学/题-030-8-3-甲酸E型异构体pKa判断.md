# 题-030-8-3：甲酸E型异构体pKa判断

## 题目
> **来源**：[[07-资料提炼/提炼-第30届初赛试题解析|第30届初赛第8题]]（原始文件：[[mineru/02-真题解析/30届初赛试题解析]]）
> **小问关联**：本文件为第8-3小问；其余小问见 [[题-030-8-1-化合物A到F结构简式]]、[[题-030-8-2-反应条件选择]]

理论计算表明，甲酸的 Z 和 E 型两种异构体存在一定的能量差别。

![[mineru/02-真题解析/30届初赛试题解析_images/1e8efbf98d5c9e512c4023b7b4a3c9c461fce080f56573e2531f1bb40f318d26.jpg]]

已知 $Z$ 型异构体的 $\mathrm{p}K_{\mathrm{a}} = 3.77$，判断 $E$ 型异构体的 $\mathrm{p}K_{\mathrm{a}}$：

(a) 3.77

(b) < 3.77

(c) = 3.77

(d) 无法判断

## 参考答案
**(b) < 3.77**

## 解析

本小题考查同学们对于同种物质不同构型性质的推断。由题目可知，从 $Z$ 型异构体转化为 $E$ 型异构体需要吸收一定量的能量。而两种异构体电离之后的产物相同，均为氢离子和甲酸根离子，没有异构体，因此这是两个产物相同、反应物不同的反应。

而由于 $E$ 型异构体能量更高，它从酸分子转化为离子吸收的能量必然比 $Z$ 型更少。又由热力学基础知识，平衡常数和反应的能变大小息息相关。$E$ 型酸电离吸收的能量更少，那么它相比于 $Z$ 型酸电离的反应平衡常数就更大，对应的 $\mathrm{p}K_{\mathrm{a}}$ 也就更小，因此 $E$ 型酸的 $\mathrm{p}K_{\mathrm{a}}$ 比 $Z$ 型酸更小，即小于3.77，故本题答案选(b)。

这里面套了很多层变量转化，最好先用反应势能图联系甲酸的 Z 型异构体、E 型异构体以及甲酸根和氢离子，比较两个反应的 $\Delta G$ 大小，进一步比较 $K_{\mathrm{a}}$ 的大小，再转化为 $\mathrm{p}K_{\mathrm{a}}$ 之间的比较。

**知识拓展：**

8-3 中 s-cis 和 s-trans 两种甲酸构象的能量差异源于广义的异头碳效应；更具体地说，是羟基氧原子的孤对电子与羰基 $\sigma^{*}$ 轨道间的负超共轭（negative hyperconjugation）效应。在这个例子中，羟基氧原子的 n 轨道起到了与负离子 p 轨道相同的作用。当 n 轨道与 $\sigma^{*}$ 轨道处于反式共平面位置关系时（也即采取 s-trans 构象时），二者重叠更佳，稳定效果更好。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 构型与能量 | E型能量高于Z型，电离时释放能量更少 |
| 热力学循环 | 利用Hess定律比较不同构型的电离自由能 |
| pKa与ΔG关系 | ΔG° = -RTlnKa = 2.303RT·pKa |
| 负超共轭效应 | s-trans构象中n→σ*负超共轭使Z型更稳定 |
| 异头碳效应 | 广义的异头碳效应解释构象能量差异 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 选(a)或(c)认为pKa相同 | 不同构型能量不同，电离自由能不同，pKa不同 |
| 选(d)认为无法判断 | 可通过热力学关系明确判断E型pKa更小 |
| 认为E型pKa更大 | 能量高的酸更容易电离，Ka更大，pKa更小 |
| 混淆Z型和E型的稳定性 | Z型能量低更稳定，E型能量高更活泼 |
| 未画热力学循环 | 建议画反应势能图辅助理解 |
