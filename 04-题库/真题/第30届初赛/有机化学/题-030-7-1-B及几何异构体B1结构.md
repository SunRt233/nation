# 题-030-7-1：B及其几何异构体B1结构示意图

## 题目
> **来源**：[[07-资料提炼/提炼-第30届初赛试题解析|第30届初赛第7题]]（原始文件：[[mineru/02-真题解析/30届初赛试题解析]]）
> **小问关联**：本文件为第7-1小问；其余小问见 [[题-030-7-2-A和D中Rh氧化态及电子数]]、[[题-030-7-3-E生成醋酸反应式]]、[[题-030-7-4-Cativa法Cprime面式结构]]

乙醇在醋酸菌作用下被空气氧化是制造醋酸的有效方法，然而这一传统过程远远不能满足工业的需求。目前工业上多采用甲醇和一氧化碳反应制备醋酸：$CH_{3}OH + CO \rightarrow CH_{3}COOH$。第9族元素（Co、Rh和Ir）的一些配合物是上述反应良好的催化剂。以 $[\mathrm{Rh}(\mathrm{CO})_{2}\mathrm{I}_{2}]$ 为催化剂、以碘甲烷为助催化剂合成乙酸（Monsanto法）的示意图如下：

在催化循环中，A 和碘甲烷发生氧化加成反应，变为 B。画出 B 及其几何异构体 B1 的结构示意图。

## 解析

本小题要求同学们推断反应产物并书写异构体。根据条件，A 和碘甲烷发生了氧化加成反应，产物 B 中心离子 Rh 氧化态上升，因而是 Rh(I) 对碘甲烷中 C-I 键进行了氧化加成；而且，此氧化加成的方式是碘和甲基分别加成在Rh上。

注意：在画结构的时候一定不要忘记标出离子自身所带的电荷数。

本小题另一问为画出 B 的几何异构体 B1。在上述的面式结构中，改变 CO 和 $CH_{3}$ 的位置，不能形成新的异构体。而在经式结构中，有 $CH_{3}$ 和 CO 对位及 CO 和 CO 对位两种可能。综上所述，B1 即为上述两个经式异构体。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 氧化加成反应 | Rh(I)对C-I键进行氧化加成，Rh氧化态升至+3 |
| 面式结构 | fac-异构体，三个相同配体占据八面体的一个三角面 |
| 经式结构 | mer-异构体，三个相同配体占据八面体的一条经线 |
| 几何异构体 | 八面体配合物MA₃B₃型有fac和mer两种几何异构体 |
| Monsanto法 | 工业合成醋酸的经典催化循环 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 忘记标电荷 | B为+1价配阳离子，需标明电荷 |
| 只画一种B1 | B1有两个经式异构体：CH₃与CO对位、CO与CO对位 |
| 混淆氧化加成产物 | I⁻和CH₃⁻分别加成到Rh上，不是其他方式 |
| 画错配位数 | 氧化加成后Rh为六配位八面体构型 |
| 忽略立体化学 | 需画出三维结构的示意图 |
