# 题-030-9-1-2：HCl与K₂CO₃条件下产物差异原因

## 题目
> **来源**：[[07-资料提炼/提炼-第30届初赛试题解析|第30届初赛第9题]]（原始文件：[[mineru/02-真题解析/30届初赛试题解析]]）
> **小问关联**：本文件为第9-1-2小问；其余小问见 [[题-030-9-1-1-化合物B结构简式]]、[[题-030-9-1-3-A转化为B的原因]]、[[题-030-9-2-1-中间体D及产物E结构]]、[[题-030-9-2-2-GHJ结构简式]]、[[题-030-9-2-3-得不到J的原因]]

为什么在 HCl 作用下，氨基乙醇与乙酸酐反应生成化合物 A；而在 $\mathrm{K}_{2}\mathrm{CO}_{3}$ 作用下，却生成化合物 B？

## 参考答案

在盐酸中，氨基乙醇的氨基被质子化生成铵盐，失去亲核性，而羟基保持亲核性，因此乙酸酐与羟基发生醇解反应生成酯（化合物A）。

在碳酸钾碱性条件下，氨基未被质子化，保持较强的亲核性，而氨基的亲核性强于羟基，因此乙酸酐优先与氨基发生胺解反应生成酰胺（化合物B）。

## 解析

本小题中，底物氨基乙醇含有羟基和氨基两个官能团，均可与乙酸酐发生反应（醇解和胺解）。一般来说，氨基氮原子的亲核性强于羟基氧原子，会优先发生胺解；然而，在盐酸中发生的却是醇解反应。这是由于具有碱性的氨基会被盐酸质子化，生成铵盐从而失去亲核性。因此，反应物中仅有的亲核基团羟基便与乙酸酐发生酰基碳上的亲核取代反应，得到化合物 A。

而在碱性条件 $(\mathrm{K}_2\mathrm{CO}_3)$ 下则不存在上述问题，因而得到的产物 B 就是乙酸酐的胺解产物。

**知识拓展：**

为什么在 9-1 中，乙酸酐可以出现在一个水相反应中呢？其实，乙酸酐于水相中反应在合成酰胺中是一个成熟的应用。主要的原因在于胺、醇和水三者竞争时，亲核性呈现一个递减的序列。比如乙酰苯胺的合成，一般的做法是先将苯胺做成盐酸盐水溶液，然后加入等物质的量的乙酸钠水溶液和乙酸酐。因为反应物都溶于水，该反应为均相反应。此反应中，乙酸钠的加入十分重要，它可将苯胺盐酸盐中和，释放出亲核性远强于水的氨基，从而降低乙酸酐水解的概率。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 氨基质子化 | HCl使-NH₂→-NH₃⁺，失去亲核性 |
| 亲核性顺序 | 游离氨基 > 羟基 > 水 |
| 酸碱调控选择性 | 酸性条件保护氨基，碱性条件活化氨基 |
| 乙酸酐醇解 | 与醇反应生成酯和乙酸 |
| 乙酸酐胺解 | 与胺反应生成酰胺和乙酸 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 只说"条件不同"未解释本质 | 本质是氨基在酸中被质子化失去亲核性 |
| 认为盐酸中羟基更活泼 | 是氨基被抑制，而非羟基被活化 |
| 忽略亲核性比较 | 游离氨基亲核性本强于羟基 |
| 未提及乙酸酐的水解竞争 | 水相中乙酸酐水解是竞争反应 |
