# 题-030-9-1-3：化合物A在K₂CO₃作用下转化为B的原因

## 题目
> **来源**：[[07-资料提炼/提炼-第30届初赛试题解析|第30届初赛第9题]]（原始文件：[[mineru/02-真题解析/30届初赛试题解析]]）
> **小问关联**：本文件为第9-1-3小问；其余小问见 [[题-030-9-1-1-化合物B结构简式]]、[[题-030-9-1-2-HCl与K2CO3条件产物差异原因]]、[[题-030-9-2-1-中间体D及产物E结构]]、[[题-030-9-2-2-GHJ结构简式]]、[[题-030-9-2-3-得不到J的原因]]

为什么化合物 A 在 $\mathrm{K}_{2}\mathrm{CO}_{3}$ 作用下转化为化合物 B？

## 参考答案

化合物 A（酯）在碱性条件下经历分子内重排转化为 B（酰胺）。首先，氨基（被碱中和后重新具有孤对电子）对酯羰基进行亲核加成，形成一个稳定的五元环中间体（四面体中间体）。随后该中间体裂解，C-O 键断裂，C-N 键保留，生成热力学上更稳定的酰胺 B。

## 解析

A 如何在碱性条件下转化为 B？首先，可以排除酯先水解后形成酰胺的想法：水解得到的羧酸负离子亲电性非常弱，无法接受氨基的亲核进攻。故此反应应该经历分子内重排过程。

观察产物，最后氮原子取代了氧原子与羰基碳相连，因此在过程中的某一步，必有 C-N 键形成、C-O 键断裂。在铵正离子被碱中和后，氮原子重新具有孤对电子，可以通过亲核加成的方式与羰基碳相连，而形成的恰好是一个稳定的五元环。

我们迫切想知道的中间体结构，原来就是之前想到的 B'（五元环状中间体）。这启发我们，做题的时候多想开去一些，是有益无害的，竞赛考查的要点之一就是同学们的发散性思维。

上已述及两点动力学上的原因，现在还剩最后一个问题：为什么四面体中间体 B' 裂解后主要得到 B？由于 A 经过 B' 得到 B 是一个可逆的平衡，所以平衡偏向于生成 B 应该是一个热力学上的原因。最简单的想法便是酰胺在热力学上比酯更稳定，这反映在前者更不容易受到亲核进攻。更深层的原因是氮原子给电子能力强于氧原子，导致酰基碳亲核性减弱。

**尝试用反应势能图连接 A、B 与 B'。**

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 分子内重排 | 不是简单的水解后再酰胺化 |
| 五元环中间体 | 氨基对酯羰基亲核加成形成稳定的五元环四面体中间体 |
| 酰胺 vs 酯稳定性 | 酰胺热力学上比酯更稳定，平衡偏向酰胺 |
| 亲核加成-消除 | 四面体中间体裂解时C-O键断、C-N键留 |
| 热力学控制 | 最终产物由热力学稳定性决定 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 认为先水解再酰胺化 | 羧酸负离子亲电性太弱，无法被氨基进攻 |
| 未提及五元环中间体 | 五元环中间体的形成是反应的关键步骤 |
| 只从动力学解释 | 最终产物是热力学控制的结果 |
| 未解释为何裂解得B而非A | 酰胺比酯更稳定，平衡偏向B |
| 忽略碱的作用 | K₂CO₃中和铵盐使氨基恢复亲核性 |
