# 题-030-8-2：同时保护氨基和羟基的反应条件选择

## 题目
> **来源**：[[07-资料提炼/提炼-第30届初赛试题解析|第30届初赛第8题]]（原始文件：[[mineru/02-真题解析/30届初赛试题解析]]）
> **小问关联**：本文件为第8-2小问；其余小问见 [[题-030-8-1-化合物A到F结构简式]]、[[题-030-8-3-甲酸E型异构体pKa判断]]

某同学设计了以下反应，希望能同时保护氨基和羟基。

![[mineru/02-真题解析/30届初赛试题解析_images/50957cc2d87a343c2a8079823e6079246f6cd560eb57d287ea881687f5ca7464.jpg]]

请选择最有利于实现该反应的实验条件：

(a) 反应在浓盐酸与乙醇 (1:3) 的混合溶剂中加热进行。

(b) 反应在过量三乙胺中加热进行。

(c) 催化量的三氟化硼作用下，反应在无水乙醚中进行。

(d) 反应在回流的甲苯中进行。

## 参考答案
**(c) 催化量的三氟化硼作用下，反应在无水乙醚中进行。**

## 解析

本题中，一种反应物有酰氨基、氨基和羟基三种官能团，而另外一种反应物为缩酮。氨基、羟基与缩酮的取代反应需要用 Lewis 酸催化，以增强缩酮上烷氧基的离去能力。条件 (c) 给出了适合反应的 Lewis 酸催化剂，即三氟化硼。它能够与缩酮中的氧结合，增强烷氧基的离去能力，从而催化反应。另外，条件 (c) 提供了非质子溶剂乙醚，能够让反应物缩酮稳定存在，使得反应能够顺利进行，故本题的答案为 (c)。

**其他条件分析：**

- **(a) 浓盐酸与乙醇**：条件中有强酸浓盐酸，会使得氨基质子化，不能发生亲核取代反应；此外，浓盐酸中的水也会使得缩酮不稳定，无法进行后续的反应。

- **(b) 过量三乙胺**：三乙胺虽然能够提供亲核反应所需的碱性溶剂环境，但此条件下缩酮的甲氧基很难离去（因其离去只能以甲氧基负离子离去，不能很好地存在于三乙胺中）。

- **(d) 回流甲苯**：虽然有加热促进反应的进行，但没有加入合适的推动反应进行的催化剂，因此反应难以发生。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| Lewis酸催化 | BF₃与缩酮氧配位，增强烷氧基离去能力 |
| 缩酮稳定性 | 缩酮在酸性水溶液中不稳定，在非质子溶剂中稳定 |
| 氨基质子化 | 强酸使氨基质子化，失去亲核性 |
| 亲核取代条件 | 需Lewis酸催化且无水环境 |
| 离去基团活性 | 甲氧基离去能力弱，需活化后才能离去 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 选(a)认为酸催化有利 | 浓盐酸使氨基质子化，且水使缩酮分解 |
| 选(b)认为碱有利 | 三乙胺不能活化缩酮的离去基团 |
| 选(d)认为加热即可 | 无催化剂时缩酮的甲氧基难以离去 |
| 忽略无水条件的重要性 | 缩酮遇水易水解，需在无水条件下反应 |
| 未考虑氨基的保护问题 | 反应中氨基不能被质子化，否则失去亲核性 |
