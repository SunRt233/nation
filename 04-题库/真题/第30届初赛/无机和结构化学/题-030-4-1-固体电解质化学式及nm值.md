# 题-030-4-1：固体电解质化学式及n、m值

## 题目
> **来源**：[[07-资料提炼/提炼-第30届初赛试题解析|第30届初赛第4题]]（原始文件：[[mineru/02-真题解析/30届初赛试题解析]]）
> **小问关联**：本文件为第4-1小问；其余小问见 [[题-030-4-2-立方晶系判断]]、[[题-030-4-3-负离子堆积方式与空隙类型]]、[[题-030-4-4-离子导电性与结构关系]]

固体电解质以其在电池、传感器等装置中的广泛应用而备受关注。现有一种由正离子 $A^{n+}$、$B^{m+}$ 和负离子 $X^{-}$ 组成的无机固体电解质，该物质 50.7 ℃ 以上形成无序结构（高温相），50.7 ℃ 以下变为有序结构（低温相），二者结构示意见下图。图中，浅色球为负离子；高温相中的深色球为正离子或空位；低温相中的大深色球为 $A^{n+}$ 离子，小深色球为 $B^{m+}$ 离子。

高温相

![[mineru/02-真题解析/30届初赛试题解析_images/d079b29d7af346764389874c655bdc8f968afcbfade9cc5d3484074be3a401db.jpg]]  
低温相

推出这种电解质的化学式，写出 n 和 m 的值。

## 参考答案
- **化学式**：$A_2BX_4$
- **n = 1**，**m = 2**

## 解析

本题的主要考点为晶体结构，考查立方晶系和四方晶系的概念、对晶胞示意图的理解和识图能力、晶体空隙及其对导电性的影响等知识。

高温相中深色球代表的是统计平均分布的未知比例的阳离子及空隙，只能从低温相的晶胞着手分析。从题中低温相晶体结构图可以看出，低温相晶体中，阴离子浅色球以立方最密堆积形式排列，共8个 $X^{-}$；阳离子填充在四面体空隙中，共4个 $A^{n+}$、2个 $B^{m+}$。故化合物的最简式为 $A_{2}BX_{4}$。

如题所述，X 为 -1 价阴离子，根据晶体电荷守恒，整数 n、m 满足 $2n + m = 4$。故 n = 1, m = 2。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 晶胞分析 | 从低温相晶胞图统计各离子数目 |
| 立方最密堆积 | 阴离子X⁻呈ccp排列，产生八面体和四面体空隙 |
| 电荷守恒 | 化合物整体电中性：2n + m = 4 |
| 化学式推导 | 根据晶胞中离子数目比确定最简式 |
| 高温相/低温相 | 高温相为无序结构，低温相为有序结构 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 从高温相着手分析 | 高温相阳离子和空位统计平均分布，无法确定比例 |
| 离子数目统计错误 | 需仔细数清晶胞中各离子的位置（顶点、棱心、面心、体内） |
| 忽略n、m为正整数的限制 | 2n+m=4的正整数解只有n=1,m=2和n=2,m=0（后者舍去） |
| 未验证电荷守恒 | 推导后应检验化学式是否满足电中性 |
