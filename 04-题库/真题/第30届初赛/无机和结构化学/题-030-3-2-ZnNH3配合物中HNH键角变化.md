# 题-030-3-2：[Zn(NH₃)₆]²⁺中H-N-H键角变化解释

## 题目
> **来源**：[[07-资料提炼/提炼-第30届初赛试题解析|第30届初赛第3题]]（原始文件：[[mineru/02-真题解析/30届初赛试题解析]]）
> **小问关联**：本文件为第3-2小问；其余小问见 [[题-030-3-1-1-磷氧化合物中间体分子式]]、[[题-030-3-1-2-P4O8结构图键长分组键角比较]]、[[题-030-3-3-1-FeO4中铁价电子组态]]、[[题-030-3-3-2-FeO4分子结构示意图]]

$\mathrm{NH}_{3}$ 分子独立存在时 H-N-H 键角为 $106.7^{\circ}$，下图是 $[\mathrm{Zn}(\mathrm{NH}_{3})_{6}]^{2+}$ 离子的部分结构以及 H-N-H 键角的测量值。解释配合物中 H-N-H 键角变为 $109.5^{\circ}$ 的原因。

![[mineru/02-真题解析/30届初赛试题解析_images/615d7636456629560ef0e2e25dd07f7dd4b5eca2bcb80ec3c71352db149321d5.jpg]]

## 参考答案

配合物中 H-N-H 键角变为 $109.5^{\circ}$ 的原因是：氨分子与 $\mathrm{Zn}^{2+}$ 形成配合物后，N 原子上的孤对电子与 $\mathrm{Zn}^{2+}$ 成键，原来的孤对电子与成键电子对间的作用变成了成键电子对间的相互作用，排斥力减弱，故 H-N-H 键角变大。H-N 成键电子对和 $\mathrm{Zn}^{2+}$-N 成键电子对的排斥力与 H-N 成键电子对间的排斥力十分接近，所以键角为 $109.5^{\circ}$。

## 解析

本小题同样涉及 VSEPR 理论。形式为 $AX_4$ 的分子，例如 $\mathrm{CH}_4$，分子构型为正四面体；而形式为 $AX_3E$ 的分子，例如 $\mathrm{NH}_3$，分子构型为三角锥。由于孤对电子与成键电子对之间的排斥力更大，因此 $AX_3E$ 分子的键角会小于 $109.5^{\circ}$。而当氨分子与 $\mathrm{Zn}^{2+}$ 形成配合物后，孤对电子与 $\mathrm{Zn}^{2+}$ 成键，原来的孤对电子与成键电子对间的作用变成了成键电子对间的相互作用，排斥力减弱，故 H-N-H 键角变大。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| VSEPR理论 | AX₃E型分子键角小于正四面体角109.5° |
| 孤对电子排斥 | 孤对电子-成键电子对排斥力 > 成键电子对-成键电子对排斥力 |
| 配位键形成 | NH₃中N的孤对电子与Zn²⁺形成配位键 |
| 键角变化 | 孤对电子变为成键电子对后，排斥力减弱，键角增大 |
| 正四面体构型 | 当四个电子对均为成键电子对时，键角恢复为109.5° |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 认为键角变小 | 配合物中孤对电子参与成键，排斥力减小，键角应增大 |
| 只提到配位未解释键角变化原因 | 需明确说明孤对电子→成键电子对的转变及排斥力变化 |
| 忽略109.5°与四面体构型的关系 | 键角变为109.5°意味着N原子周围呈正四面体排布 |
| 误认为是Zn²⁺的极化作用 | 主要原因是孤对电子参与配位，而非极化 |
