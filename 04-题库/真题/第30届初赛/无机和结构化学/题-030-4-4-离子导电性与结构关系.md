# 题-030-4-4：离子导电性与结构关系

## 题目
> **来源**：[[07-资料提炼/提炼-第30届初赛试题解析|第30届初赛第4题]]（原始文件：[[mineru/02-真题解析/30届初赛试题解析]]）
> **小问关联**：本文件为第4-4小问；其余小问见 [[题-030-4-1-固体电解质化学式及nm值]]、[[题-030-4-2-立方晶系判断]]、[[题-030-4-3-负离子堆积方式与空隙类型]]

高温相具有良好的离子导电性，这源于哪种离子的迁移？简述导电性与结构的关系。

## 参考答案

源于 **$A^{+}$ 离子**的迁移。

## 解析

离子导电是在外加电场作用下，介质中的阴阳离子发生定向迁移的现象。该现象通常发生于熔融盐及电解质溶液中。通常离子晶体难以导电，将其加热至熔点附近才会引起电导率的上升。但是，少数晶体在相对较低的温度下即可获得较高的电导率，这种现象一般是晶体内存在空隙且至少一种离子可以在其间较为自由地运动引起的，它使晶体产生了离子导电性，这也就是常说的固体电解质的工作原理。体积较大的阴离子通常不易运动，构成刚性的晶格骨架，阳离子在阴离子的空隙间迁移产生离子导电性，如本题中的阳离子在晶体中存在的大量的四面体空隙间迁移从而导电，这类阳离子迁移导电的晶体被称为阳离子固体电解质。

在低温时，该晶体中的离子热运动的动能不足，无法跨越在四面体空隙间迁移的能垒，所以两种阳离子处于固定的位置上。而温度升高时，离子的热运动动能较高，容易跨过能垒，造成四面体空隙中 $A$、$B$ 阳离子分布的统计平均化，形成了新的相，即发生了低温相到高温相的相变。二者的电荷数分别为 +1 和 +2，可以对迁移的活化能和速率造成明显的差异：电荷数较低的原子在晶体静电场中四面体空隙处和三角形空隙处的势能差更低，即迁移能垒更低。而题目中没有强调 $A^{+}$ 和 $B^{2+}$ 的离子半径差异有多大，与电荷数相比并不是迁移速率的主要影响因素。综上所述，$A^{+}$ 更容易迁移。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 固体电解质原理 | 阳离子在阴离子骨架的空隙间迁移导电 |
| 阳离子迁移 | 体积较小的阳离子比阴离子更易迁移 |
| 能垒与温度 | 低温时动能不足，高温时可跨越迁移能垒 |
| 电荷数影响 | +1价离子比+2价离子迁移能垒更低 |
| 相变与导电性 | 相变后阳离子统计平均分布，导电性突增 |
| Ag₂HgI₄ | 类似结构的经典固体电解质，50.7℃相变 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 认为两种阳离子都迁移 | 实验表明主要是A⁺迁移，B²⁺迁移贡献很小 |
| 认为阴离子迁移 | 阴离子体积大，构成刚性骨架，不易迁移 |
| 忽略电荷数对迁移的影响 | +1价离子迁移能垒低于+2价离子 |
| 未联系相变解释导电性 | 需说明低温相→高温相的相变使阳离子可自由迁移 |
| 混淆热力学与动力学 | 相变是热力学平衡问题，迁移优先性是动力学问题 |
