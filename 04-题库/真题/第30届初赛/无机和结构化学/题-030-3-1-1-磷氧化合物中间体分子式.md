# 题-030-3-1-1：磷氧化合物中间体分子式

## 题目
> **来源**：[[07-资料提炼/提炼-第30届初赛试题解析|第30届初赛第3题]]（原始文件：[[mineru/02-真题解析/30届初赛试题解析]]）
> **小问关联**：本文件为第3-1-1小问；其余小问见 [[题-030-3-1-2-P4O8结构图键长分组键角比较]]、[[题-030-3-2-ZnNH3配合物中HNH键角变化]]、[[题-030-3-3-1-FeO4中铁价电子组态]]、[[题-030-3-3-2-FeO4分子结构示意图]]

$P_2O_3$ 和 $P_2O_5$ 是两种经典的化合物，其分子结构已经确定。自然而然会有如下问题：是否存在磷氧原子比介于两者之间的化合物？由此出发，化学家合成并证实了这些中间化合物的存在。

写出这些中间化合物的分子式。

## 参考答案
$P_4O_7$、$P_4O_8$、$P_4O_9$

## 解析
本小题要求同学们正确掌握常见磷氧化合物的实际结构。$P_2O_3$ 和 $P_2O_5$ 为磷氧化合物的最简式，其实际分子式是 $P_4O_6$ 和 $P_4O_{10}$。与 $P_4O_6$ 相比，$P_4O_{10}$ 增加了 4 个磷氧双键，磷的平均氧化态由 +3 变为 +5。磷原子个数不变，氧原子个数介于两者之间的磷氧化合物则应为 $P_4O_7$、$P_4O_8$ 和 $P_4O_9$。它们的结构以 $P_4O_6$ 为基础，分别在 1～3 个磷原子上添加磷氧双键。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 磷氧化物实际分子式 | P₂O₃实际为P₄O₆，P₂O₅实际为P₄O₁₀ |
| 最简式与分子式 | 磷氧化物常以最简式表示，但真实结构为P₄单元 |
| 磷的氧化态 | P(III)和P(V)的混合氧化态化合物 |
| 结构基础 | 以P₄O₆四面体结构为基础，逐步添加P=O双键 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 写成P₂O₄等最简式形式 | 应写实际分子式P₄O₇、P₄O₈、P₄O₉ |
| 遗漏某些中间化合物 | P₄O₆到P₄O₁₀之间氧原子数差4，中间有3种 |
| 混淆P₄O₇/P₄O₈/P₄O₉的顺序 | 按氧原子数递增排列 |
