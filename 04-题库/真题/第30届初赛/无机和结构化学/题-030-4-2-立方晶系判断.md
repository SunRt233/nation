# 题-030-4-2：立方晶系判断

## 题目
> **来源**：[[07-资料提炼/提炼-第30届初赛试题解析|第30届初赛第4题]]（原始文件：[[mineru/02-真题解析/30届初赛试题解析]]）
> **小问关联**：本文件为第4-2小问；其余小问见 [[题-030-4-1-固体电解质化学式及nm值]]、[[题-030-4-3-负离子堆积方式与空隙类型]]、[[题-030-4-4-离子导电性与结构关系]]

温度变化会导致晶体在立方晶系和四方晶系之间转换，上述哪种晶相属于立方晶系？

## 参考答案
**高温相**属于立方晶系。

## 解析

高温相的两种阳离子和空隙平均分布，可从统计学意义上视为等同的。高温相的晶胞为立方 ZnS 型，故高温相为立方晶系。也可通过立方晶系与四方晶系的区别来判断：立方晶系比四方晶系的对称性更高，晶胞的 4 条体对角线分别是 4 根三重旋转轴 $(4C_{3})$。观察晶体结构图可知，高温相满足这一条件。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 立方晶系特征 | 具有4个三重旋转轴（沿体对角线方向） |
| 四方晶系特征 | 只有1个四重旋转轴，无4个三重轴 |
| 对称性比较 | 立方晶系对称性高于四方晶系 |
| 立方ZnS型结构 | 高温相为闪锌矿型结构，属于立方晶系 |
| 统计平均分布 | 高温相中阳离子和空位统计平均，提高对称性 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 认为低温相是立方晶系 | 低温相阳离子有序排列，对称性降低 |
| 混淆立方与四方的判断标准 | 关键看是否具有4个C₃轴 |
| 仅从晶胞形状判断 | 需从对称元素角度严格判断 |
| 忽略统计平均化的影响 | 高温相无序分布使对称性升高 |
