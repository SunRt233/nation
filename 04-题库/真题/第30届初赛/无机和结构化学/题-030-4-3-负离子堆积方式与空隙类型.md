# 题-030-4-3：负离子堆积方式、空隙类型与占有率

## 题目
> **来源**：[[07-资料提炼/提炼-第30届初赛试题解析|第30届初赛第4题]]（原始文件：[[mineru/02-真题解析/30届初赛试题解析]]）
> **小问关联**：本文件为第4-3小问；其余小问见 [[题-030-4-1-固体电解质化学式及nm值]]、[[题-030-4-2-立方晶系判断]]、[[题-030-4-4-离子导电性与结构关系]]

写出负离子的堆积方式及形成的空隙类型。指出正离子占据的空隙类型及占有率。

## 参考答案

- **负离子堆积方式**：立方最密堆积（ccp，ABCABC...堆积）
- **形成的空隙类型**：八面体空隙和四面体空隙
- **正离子占据的空隙类型**：四面体空隙
- **正离子对空隙的占有率**：$3/8 = 37.5\%$

## 解析

晶体中阴离子堆积方式如图 4.1。晶体由不同颜色标明的 A、B 和 C 三种堆积层以 …ABCABC… 的形式堆积，故阴离子呈立方最密堆积。阴离子的堆积产生了两种空隙：八面体空隙和四面体空隙。八面体空隙位于棱心和体心，共 4 处；四面体空隙位于晶胞的 $(\pm1/4, \pm1/4, \pm1/4)$，共 8 处，正离子占据的是四面体空隙。

![[mineru/02-真题解析/30届初赛试题解析_images/5f114989b6278f7364bdcf6e359691b72da23bb0258e9788d01d1be6d01822da.jpg]]  
图4.1

图 4.1 的立方体骨架中包含 4 个 $X^{-}$，其间存在 8 个四面体空隙，而 3 个正离子占据了其中的 3 个空隙，故正离子对空隙的占有率为 $3/8 = 37.5\%$。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 立方最密堆积 | ccp，堆积序列为ABCABC... |
| 八面体空隙 | 位于棱心和体心位置 |
| 四面体空隙 | 位于(±1/4, ±1/4, ±1/4)位置 |
| 空隙数目比 | ccp中：四面体空隙数=2×原子数，八面体空隙数=原子数 |
| 空隙占有率 | 被占据空隙数/总空隙数 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 写成六方最密堆积 | 应为立方最密堆积（ccp） |
| 空隙类型遗漏 | 需同时写出八面体和四面体两种空隙 |
| 空隙数目计算错误 | 4个X⁻对应8个四面体空隙、4个八面体空隙 |
| 占有率计算错误 | 3个正离子/8个四面体空隙=37.5% |
| 混淆正离子占据的空隙类型 | 正离子占据四面体空隙而非八面体空隙 |
