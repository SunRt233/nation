# 题-033-8-3-2：L-山梨糖和化合物 C 的 Fischer 投影式

8-3-2 画出 L-山梨糖和化合物 C 的 Fischer 投影式。

## 参考答案

**得分点**：
1. **正确画出 L-山梨糖的 Fischer 投影式**（3分）
2. **正确画出化合物 C 的 Fischer 投影式**（3分）

一旦我们有了 L-呋喃山梨糖的结构，就可以倒推出 L-山梨糖的结构，它是一种多羟基酮。C5 氧化的逆过程推出 D-山梨醇应为饱和的六元醇。但是 D-山梨醇 C5 的构型是无法确定的。

最后，写出 L-山梨糖和 C 的 Fischer 投影式。注意这里是如何将链式结构的伞形式分步改写为 Fischer 投影式的，不要跳步！同时也要注意 Fischer 投影式的书写规范：高氧化态的原子处于上方，低氧化态的原子处于下方。由于 C 是由 L-山梨糖氧化而来，因而羧酸的 $\alpha$ 碳应为羰基，而 $\beta$ 碳应为醇：

## 解题思路

1. **读题定位**：题目要求画出 L-山梨糖和化合物 C 的 Fischer 投影式。
2. **🔑 关键转换**：从合成路线倒推 L-山梨糖的结构（多羟基酮），然后将其伞形式按规范转换为 Fischer 投影式；C 是 L-山梨糖氧化后的羧酸，同样画出其 Fischer 投影式。
3. **书写要点**：Fischer 投影式中高氧化态原子（醛基/羧基）在上方，低氧化态原子（CH₂OH）在下方；水平键朝前，竖直键朝后。

## 易错分析

| 错误 | 原因 | 纠正 | 课堂提问 |
|---|---|---|---|
| 高氧化态原子在下方 | 未遵循 Fischer 投影式规范 | 高氧化态原子必须在上方 | Fischer 投影式的书写规范是什么？ |
| 手性碳构型错误 | 伞形式转换为 Fischer 投影式时出错 | 水平键朝前，竖直键朝后 | 如何将伞形式正确转换为 Fischer 投影式？ |
| L-山梨糖画成醛糖 | 未识别山梨糖是酮糖 | L-山梨糖是多羟基酮（酮糖） | 山梨糖是醛糖还是酮糖？ |
| C 的结构画错 | 未理解 C 是氧化产物 | C 是 L-山梨糖氧化得到的羧酸 | 山梨糖氧化后得到什么产物？ |
