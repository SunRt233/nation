初步研究表明，嫦娥石中含有 ${}^{3}He$ ，$FeO_6$ 和 $PO_4$ 可能是其吸附位置。${}^{3}He$ 已用于中子探测，也很可能成为人类未来的能源。${}^{3}He$ 与中子作用产生质子和氚，写出核反应方程式。

## 参考答案

书写核反应方程式时，要注意将所有粒子物种的核电荷数和质量数标出，同时保证反应前后核电荷数和质量数守恒：

将 $^{3}_{1}H$ 写为 $^{3}_{1}T$ 也可以。

$$
{ } _ { 2 } ^ { 3 } \mathrm{He} + { } _ { 0 } ^ { 1 } \mathrm{n} \longrightarrow { } _ { 1 } ^ { 3 } \mathrm{H} + { } _ { 1 } ^ { 1 } \mathrm{H}
$$

## 解题思路

1. 确定反应物：$^{3}_{2}\mathrm{He}$ 和 $^{1}_{0}\mathrm{n}$
2. 确定产物：$^{3}_{1}\mathrm{H}$（氚）和 $^{1}_{1}\mathrm{H}$（质子）
3. 验证质量数守恒：3 + 1 = 3 + 1
4. 验证核电荷数守恒：2 + 0 = 1 + 1

## 易错分析

- **未标注质量数和核电荷数**：核反应方程式必须标注
- **守恒验证错误**：未验证质量数和核电荷数是否守恒
- **产物判断错误**：将氚误写为其他粒子
