## 题目

> 关联小问：[[题-036b-2-2-E的异构体数目及结构|2-2]]
>
> 来源：[[07-资料提炼/提炼-第36届初赛试题解析（第二场）|提炼笔记]] | [[mineru/02-真题解析/36届初赛试题解析2|原始解析]]

电解质是电池的重要组分之一。锂离子电池中，最常用的电解质为锂盐 X。X 中阴离子为正八面体构型，其组成元素均为短周期元素。X 的合成方法之一如下：具有三角锥形分子结构的化合物 A 与化合物 B 发生复分解反应，生成 C 和 HCl，C 与黄绿色气体 D 按照 1:1 的比例反应生成 E，E 具有数种异构体。E 进一步与 B 反应，获得具有三角双锥结构的 F 和 HCl，F 和 LiF 反应得到 X。

写出 A ~ F 以及 X 的化学式。

## 参考答案

根据题意，A 的构型为三角锥形，F 的构型为三角双锥，在短周期元素中只有磷元素可以同时形成三配位和五配位化合物。而三角双锥构型的 F 与 LiF 反应，生成正八面体构型的 X 阴离子，则可以确定 F 为 PF₅，而 X 为 LiPF₆。通过 F 产生过程中的副产物 HCl，可以确定 B 为无机酸且含氟，即 HF。

因此，根据三角锥形分子 A 与 HF 发生复分解反应生成化合物 C 和 HCl，可以推出 A、C 分别为 P(Ⅲ) 的氯化物与氟化物，即 $PCl_{3}$ 和 $PF_{3}$ 。同时根据颜色也不难确定，黄绿色气体 D 为氯气。而 $PF_{3}$ 与氯气发生 1:1 反应，得到的产物即为 $PF_{3}Cl_{2}$ （化合物 E）。

| 化合物 | 化学式 |
|:---:|:---:|
| A | $\mathrm{PCl_3}$ |
| B | $\mathrm{HF}$ |
| C | $\mathrm{PF_3}$ |
| D | $\mathrm{Cl_2}$ |
| E | $\mathrm{PF_3Cl_2}$ |
| F | $\mathrm{PF_5}$ |
| X | $\mathrm{LiPF_6}$ |

## 解题思路

1. 从 X 的构型入手：正八面体阴离子 + 短周期元素 → 推断为 $\mathrm{PF_6^-}$
2. F 为三角双锥 + 与 LiF 反应生成 X → F 为 $\mathrm{PF_5}$
3. B 含 F 且反应生成 HCl → B 为 HF
4. A 为三角锥形 + 与 HF 复分解生成 C 和 HCl → A 为 $\mathrm{PCl_3}$，C 为 $\mathrm{PF_3}$
5. D 为黄绿色气体 → D 为 $\mathrm{Cl_2}$
6. C 与 D 1:1 反应 → E 为 $\mathrm{PF_3Cl_2}$

## 易错分析

- **元素推断错误**：未考虑到磷元素可同时形成三配位和五配位化合物
- **忽略副产物信息**：未利用 HCl 推断 B 为 HF
- **构型与化学式对应错误**：三角双锥、正八面体等构型与具体化合物的对应关系不清
