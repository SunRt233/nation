# 题-031-10-3：产物C结构推断

## 题目
> **来源**：[[07-资料提炼/提炼-第31届初赛试题解析|第31届初赛第10题]]（原始文件：[[mineru/02-真题解析/31届初赛试题解析]]）
> **小问关联**：本文件为第(10-3)小问；其余小问见 [[题-031-10-1-产物A结构推断]]、[[题-031-10-2-产物B结构推断]]

10-3 当 R = OEt 时，产物为 C，分子式为 $C_{14}H_{10}O_{5}$ 。参照以上实验结果，画出 C 的结构简式。

## 参考答案

10-3 首先进行与 10-1、10-2 相似的 Michael 加成与双键移位反应。在 10-2 的基础上，根据分子式推断，减少的片段为 HOEt 和吡啶。酯基不能发生类似羰基的缩酮化反应，故此时反应后半部分历程必定不可能按照 10-1 进行。

此时再留意到分子内有两个羟基，可以发生形成内酯的反应。由于六元环稳定性高于七元环，容易推测此处形成了六元环内酯。可以写出互为互变异构体的两个合理答案：

## 解析

本题考查酯基取代基对反应路径的影响。关键思路：
1. 与10-1、10-2类似的前半部分：Michael加成和双键移位
2. 酯基不能发生缩酮化反应（不同于10-1的甲基）
3. 分子内两个羟基可形成内酯
4. 六元环内酯比七元环更稳定
5. 产物存在互变异构体

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 酯基反应性 | 酯基不能发生缩酮化反应 |
| 内酯形成 | 分子内羟基与酯基/羧基形成内酯 |
| 六元环优势 | 六元环内酯比七元环更稳定 |
| 互变异构 | 烯醇式与酮式互变异构 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 类比10-1写缩酮 | 酯基不能缩酮化 |
| 形成七元环 | 六元环稳定性更高 |
| 忽略互变异构 | 产物存在互变异构体 |
| 分子式验证 | 需与$C_{14}H_{10}O_5$吻合 |
