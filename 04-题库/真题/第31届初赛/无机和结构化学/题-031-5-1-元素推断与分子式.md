# 题-031-5-1：元素推断与分子式

## 题目
> **来源**：[[07-资料提炼/提炼-第31届初赛试题解析|第31届初赛第5题]]（原始文件：[[mineru/02-真题解析/31届初赛试题解析]]）
> **小问关联**：本文件为第(5-1)小问；其余小问见 [[题-031-5-2-制备A方程式]]、[[题-031-5-3-B与浓硫酸反应]]

由元素 X 和 Y 形成的化合物 A 是一种重要的化工产品，可用于制备润滑剂、杀虫剂等。A 可由生产 X 单质的副产物 $Fe_{2}$ 与黄铁矿反应制备，同时得到另一个二元化合物 B。B 溶于稀硫酸放出气体 C，而与浓硫酸反应放出二氧化硫。C 与大多数金属离子发生沉淀反应。纯净的 A 呈黄色，对热稳定，但遇潮湿空气极易分解而有臭鸡蛋味。A 在乙醇中发生醇解，得到以 X 为单中心的二酯化合物 D 并放出气体 C，D 与 $Cl_{2}$ 反应生成制备杀虫剂的原料 E、放出刺激性的酸性气体 F 并得到 Y 的单质（产物的摩尔比为 1:1:1）。A 与五氧化二磷混合加热，可得到两种与 A 结构对称性相同的化合物 G1 和 G2。

写出 A、C、D、E、F、G1 和 G2 的分子式。

## 参考答案

这是一道综合性的元素推断题，考查了氧化还原反应、溶剂解反应，以及磷、硫元素的相关性质，具有一定的难度。

黄铁矿的化学式为 Fe S₂。A 可由 Fe P₂ 与 Fe S₂ 反应制备，且由两种元素 X 和 Y 组成。根据题目 "A 是一种重要的化工产品，可用于制备润滑剂、杀虫剂" 等信息进行推断：常见的有机杀虫剂通常含有 P、S 和 Cl 元素，如敌敌畏、对硫磷、甲胺磷等；而润滑剂通常为层状化合物，如石墨、滑石、二硫化钼、氮化硼。综上，A 不太可能是 Fe 与 S 或 P 形成的离子化合物，而是 P 和 S 两种元素组成的二元化合物。根据生产 X 单质的副产物为 Fe P₂ 可知：X 为 P 元素，Y 为 S 元素。

Fe $P_{2}$ 与 Fe $S_{2}$ 的反应中铁元素必然进入了二元化合物 B，另一元素可能为硫或磷，与冷酸反应生成对应的氢化物。考虑到磷化氢还原性较强，且 "C 与大多数金属离子发生沉淀反应"，故 B 应为硫化铁，与稀硫酸反应产生的气体 C 应为 $H_{2}S$ 。

接下来的部分涉及很多未知反应，如硫化磷醇解生成二酯类化合物，这不能作为推断的人手点，故暂时跳过此部分。注意题目最后一句话"A与五氧化二磷混合加热，可得到两种与A结构对称性相同的化合物 G1 和 G2"，这说明 A 具有某种值得题目提及的对称性。我们知道另一反应物五氧化二磷实际上是十氧化四磷，是一种高对称性（ $T_{\mathrm{d}}$ 点群）的分子，故可以合理猜测这种硫化磷可能是类似的十硫化四磷 $\mathrm{P}_{4} \mathrm{~S}_{10}$ ，如下图（ $\mu_{2}$ 代表与两个中心原子相连的桥连配体）：

![[mineru/02-真题解析/31届初赛试题解析_images/d95c36f61216015541b25b37151417a5cec713950e28845d36da6c518e586060.jpg]]

![[mineru/02-真题解析/31届初赛试题解析_images/ccf362793a7ef63b235bf912022c232877019ad7be02589c5645321c2db18ce7.jpg]]

在此猜测的基础上进行前面的推断：A 遇潮湿空气分解产生有臭鸡蛋味的气体，应是水解生成偏磷酸和硫化氢。A 的乙醇解反应可以类比为水解，得到的二酯化合物 D 应为硫代磷酸酯，可以类比为磷酸，这个反应类似于 $P_{4}O_{10}$ 吸水转化为酸的反应。这种二酯化合物含有两个 P-OEt 单键，考虑到醇解反应的价态应该没有改变，磷仍形成五根键，应该还有一个双键和一个单键。由于乙醇中氧和乙基的键在该反应中应该不会断裂，所以剩余两个位置不会是 =O 或 -OH，只可能是 =S 和 -SH。故 D 为 $S=P(OEt)_{2}(SH)$ 。D 与 $Cl_{2}$ 反应可认为是 -Cl 取代了 -SH，方程式为

$$
\mathrm{S} = \mathrm{P} (\mathrm{OEt}) _ {2} (\mathrm{SH}) + \mathrm{Cl} _ {2} \longrightarrow \mathrm{S} = \mathrm{P} (\mathrm{OEt}) _ {2} \mathrm{Cl} + \mathrm{HCl} + \mathrm{S}
$$

即 $\mathsf{F}$ 为HCl， $\mathsf{E}$ 为 $S = P(OEt)_2Cl$ 。 $\mathsf{E}$ 的结构为

![[mineru/02-真题解析/31届初赛试题解析_images/76fc0969648427cea0c095fe7bfd78cd24d9cd7b0278338d2be87be1f2eb4df3.jpg]]

最后考虑 A 与 $P_{4}O_{10}$ 混合加热发生的反应。由于反应前后对称性始终是相同的，说明物质在整体结构上没有发生显著的变化，只发生了硫原子与氧原子的简单交换。注意到 $P_{4}S_{10}$ 分子中的磷硫键有两种，如果 P-S 单键发生断裂，需要破坏整个分子的结构，比较困难；更可能发生的是端基的 P=S 双键断裂，与 $P_{4}O_{10}$ 端基的 P=O 双键发生交换。这样可以得到如下两个化合物，恰好生成对称性相同的 G1 和 G2，分子式分别为 $P_{4}S_{6}O_{4}$ 和 $P_{4}O_{6}S_{4}$ ，如下图所示：

![[mineru/02-真题解析/31届初赛试题解析_images/485f3354bca41adab5e68ad48284d7e7c3b01ca865c6c1942dda9f68307642bc.jpg]]
$\mathrm{P_{4}S_{4}(\mu_{2}-O)_{6}}$

![[mineru/02-真题解析/31届初赛试题解析_images/37c0418e3cbfa8c844716b5389e47a1a407feb66c53e0a7265fb10e04aa4b189.jpg]]
$\mathrm{P_{4}O_{4}(\mu_{2}-S)_{6}}$

**答案：**
- A：$P_4S_{10}$
- C：$H_2S$
- D：$S=P(OEt)_2(SH)$
- E：$S=P(OEt)_2Cl$
- F：HCl
- G1：$P_4S_6O_4$
- G2：$P_4O_6S_4$

## 解析

本题是第5题的核心，综合性元素推断。关键思路：
1. 从"生产X单质的副产物FeP₂"推断X为P
2. A为P和S的二元化合物，结合杀虫剂、润滑剂等用途
3. 从"与$P_4O_{10}$反应保持对称性"推断A为$P_4S_{10}$
4. B为FeS，C为$H_2S$
5. 醇解反应类比水解，推断D的结构
6. $Cl_2$取代-SH，推断E和F
7. 与$P_4O_{10}$的配体交换推断G1和G2

| 知识点 | 说明 |

| 溶剂解反应 | 醇解类比水解 |
| 配体交换 | 与$P_4O_{10}$的S/O交换反应 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| A推断为其他硫化磷 | 需满足与$P_4O_{10}$反应保持对称性的条件 |
| D的结构错误 | 醇解后应有=S和-SH |
| G1/G2写错 | 端基P=S与P=O交换，不是桥连键交换 |
