# 题-031-2-4：多硫离子浓度

## 题目
> **来源**：[[07-资料提炼/提炼-第31届初赛试题解析|第31届初赛第2题]]（原始文件：[[mineru/02-真题解析/31届初赛试题解析]]）
> **小问关联**：本文件为第(2-4)小问；其余小问见 [[题-031-2-1-氨晶体氢键]]、[[题-031-2-2-P4S5结构]]、[[题-031-2-3-水煤气反应热]]

硫粉和 $S^{2-}$ 反应，可以生成多硫离子。在 10 mL $S^{2-}$ 溶液中加入 0.080 g 硫粉，控制条件使硫粉完全反应，检测到溶液中最大聚合度的多硫离子是 $S_{3}^{2-}$ ， $S_{n}^{2-}$ ( $n=1,2,3,\ldots$ ) 离子浓度之比符合等比数列 $1,10,\cdots,10^{n-1}$ 。若不考虑其他副反应，计算反应后溶液中 $S^{2-}$ 的浓度 $c_{1}$ 和其起始浓度 $c_{0}$ 。

## 参考答案

本小题考查水溶液中的物料守恒和电荷守恒。根据题意，以反应前后溶液中 $\mathrm{S}^{2-}$ 的浓度 $c_{0}$ 和 $c_{1}$ 列出电荷守恒方程和物料守恒方程，即

$$
\begin{array}{r l} & 2 c _ {1} + 2 0 c _ {1} + 2 0 0 c _ {1} = 2 c _ {0} \\ & 1 0 c _ {1} + 2 \times 1 0 0 c _ {1} = \frac {0 . 0 8 0 \mathrm{g}}{3 2 \mathrm{g} \mathrm{mol} ^ {- 1} \times 0 . 0 1 0 \mathrm{L}} = 0. 2 5 \mathrm{mol} \mathrm{L} ^ {- 1} \end{array}
$$

由此解得

$$
c _ {1} = 1. 2 \times 1 0 ^ {- 3} \mathrm{mol} \mathrm{L} ^ {- 1}
$$

$$
c _ {0} = 0. 1 3 \mathrm{mol} \mathrm{L} ^ {- 1}
$$

## 解析

本题考查多硫离子体系的物料守恒和电荷守恒。关键思路：
1. 多硫离子浓度比为等比数列：$c(S^{2-}):c(S_2^{2-}):c(S_3^{2-}) = 1:10:100$
2. 设 $c(S^{2-}) = c_1$，则 $c(S_2^{2-}) = 10c_1$，$c(S_3^{2-}) = 100c_1$
3. 电荷守恒：反应前后总负电荷相等 $2c_1 + 2(10c_1) + 2(100c_1) = 2c_0$
4. 物料守恒（硫原子）：加入的硫粉转化为多硫离子中的额外硫原子
5. 联立求解

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 多硫离子 | 硫粉与$S^{2-}$反应生成$S_n^{2-}$ |
| 等比数列 | 各多硫离子浓度比为1:10:100 |
| 物料守恒 | 硫原子总数守恒 |
| 电荷守恒 | 溶液电中性 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 电荷守恒列错 | 每个多硫离子带2个负电荷，需乘2 |
| 物料守恒列错 | 注意多硫离子中硫原子数为n，$S_2^{2-}$含2个S，$S_3^{2-}$含3个S |
| 单位换算错误 | 体积为10 mL = 0.010 L |
