## 题目

> 来源：[[mineru/02-真题解析/37届初赛试题解析|37届初赛试题解析]] · [[07-资料提炼/提炼-第37届初赛试题解析|提炼笔记]]

为理解金属离子在相关过程的作用，开展了如下研究：空气中，在配离子 A（$[Ru(EDTA)(OH)]^{2-}$，EDTA 为乙二胺四乙酸根）的溶液中，加入 NaHS 溶液并充分搅拌，溶液变为蓝绿色，对应于双核配离子 B 的生成（反应 4）。分别取 A 和 B 的溶液，在惰性气体保护下，加入被 NO 饱和的磷酸盐缓冲液 (pH = 8.2)，充分反应，A 转化为 C，B 转化为 D（反应 5）。借助于质谱分析，计算出四个配离子的式量分别为 A: 406.3, B: 842.7, C: 419.3, D: 451.4。磁性测量发现，C 为抗磁性，D 为顺磁性。

写出配离子 B、C 和 D 的结构简式。

## 参考答案

**B 的结构简式**：$[(EDTA)Ru-S-S-Ru(EDTA)]^{4-}$

**C 的结构简式**：$[Ru(EDTA)(NO)]^{-}$

**D 的结构简式**：$[(EDTA)Ru(SNO)]^{2-}$

**推导过程**：

1. **分析 A → C**：
   - $M_C - M_A = 419.3 - 406.3 = 13.0$，对应 N - H = 14 - 1 = 13。
   - C = A + NO - OH，即 $[Ru(EDTA)(NO)]^{-}$。
   - NO 为三电子配体，配位形式为 $NO^{+}$，Ru 为 +2 价。
   - C 为抗磁性，Ru(II) 低自旋，与结构一致。

2. **分析 A → B**：
   - $M_B = 2M_A + 30.1$，B 含两个 Ru 中心。
   - 空气参与反应，$S^{2-}$ 被氧化生成过硫键。
   - B = $[(EDTA)Ru-S-S-Ru(EDTA)]^{4-}$，Ru 为 +3 价。

3. **分析 B → D**：
   - $M_D - M_C = 451.4 - 419.3 = 32.1$，对应 S。
   - D = C + S，或 D = B/2 + NO。
   - D = $[(EDTA)Ru(SNO)]^{2-}$，Ru 为 +3 价。
   - D 为顺磁性，与 Ru(III) $d^{5}$ 低自旋一致。

## 解题思路

1. **计算式量差**：分析A→C、A→B、B→D的式量变化，推断原子变化。
2. **确定C的结构**：A + NO → C，OH⁻被NO取代。
3. **确定B的结构**：空气氧化S²⁻生成过硫键，双核结构。
4. **确定D的结构**：B与NO反应，过硫键断裂生成SNO配体。
5. **验证磁性**：C抗磁性（Ru(II)低自旋），D顺磁性（Ru(III)低自旋）。

## 易错分析

- **NO配体形式**：NO一般为三电子配体$NO^{+}$，不是中性NO。
- **Ru氧化态判断**：A中Ru为+3，C中Ru为+2（被NO还原），B中Ru为+3，D中Ru为+3。
- **磁性判断错误**：需结合d电子数和配位场判断高/低自旋。
