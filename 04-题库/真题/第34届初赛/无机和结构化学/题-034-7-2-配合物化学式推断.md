# 题-034-7-2：配合物化学式推断

## 题目

> **来源**：[[07-资料提炼/提炼-第34届初赛试题解析|第34届初赛第7题]]（原始文件：[[mineru/02-真题解析/34届初赛试题解析]]）
> **小问关联**：本文件为第(2)小问；其余小问见 [[题-034-7-1-咔咯环芳香性判断]]、[[题-034-7-3-金属离子价电子组态]]、[[题-034-7-4-配合物制备反应方程式]]、[[题-034-7-5-电化学转化与光谱变化]]

本题干见 [[题-034-7-1-咔咯环芳香性判断]]

7-2 写出 B、C 和 D 的化学式（要求：配体均用简写符号表示）。

## 参考答案

**得分点**：
1. **B 的化学式**（2分）：$\mathrm{CrO(tpfc)}$
2. **C 的化学式**（2分）：$\mathrm{Cr(OPPh_{3})_{2}(tpfc)}$
3. **D 的化学式**（2分）：$\mathrm{Cr(py)_{2}(tpfc)}$

从本小题开始，题目正式进入推断部分。本题题干较长，但每一个有效信息都不容错过。在未进入反应部分时，题目已经告诉我们，咔咯"可实现过渡金属离子的非常规氧化态"，即本题中 Cr 元素可能出现 +2、+4、+5 等非常规价态。接下来提到，咔咯"四个氮原子均参与配位"，而 $H_{3}(tpfc)$ 中连有氢原子的氮，其孤对电子参与共轭，无法进行配位，故若要满足条件，咔咯必须以去掉 3 个质子的 $(tpfc)^{3-}$ 形式进行配位。

下面进入反应部分。注意到 B 的制备在空气中进行，因此可以推测制备 B 的反应有氧气参与。B 为四方锥配位，有效磁矩为 1.72 $\mu_{\mathrm{B}}$ ，由磁矩的计算公式 $\mu = \sqrt{n(n + 2)}\mu_{\mathrm{B}}$ ，B 中单电子数目为 1。四方锥的晶体场与平面四方场比较相似，有简并度为 2 的 $d_{xz}$ 与 $d_{yz}$ 两个能量最低的轨道。在 B、C 和 D 中，为平衡 (tpfc) $^{3-}$ 的负电荷，Cr 的氧化数至少为 +3。对于 B，结合晶体场和单电子条件，可以推断 Cr 的氧化数为 +3 或 +5。又由 B、C 和 D 的转化关系，B 中 Cr 的氧化态比 C 和 D 中 Cr 的氧化态高，因此 B 中 Cr 的氧化数只能为 +5，那么其配体只能为 O 原子，即 B 为 CrO(tpfc)。

由B制备C的过程中，反应物 $\mathrm{PPh}_3$ 与 $\mathrm{OPPh}_3$ 似乎分别可作还原剂与氧化剂。然而观察上下文可以发现，与B不同的是，制备C时采用了惰性气氛保护，相当于阻止了空气中的氧气干扰反应，由此可以推测该反应为还原反应。因此 $\mathrm{PPh}_{3}$ 转化为 $\mathrm{OPPh}_{3}$ ，加上参与反应的一当量，共有两当量 $\mathrm{OPPh}_{3}$ 配体对 $\mathrm{Cr}$ 进行配位。则可以推出 **C** 的化学式: $\mathrm{Cr(OPPh_{3})_{2}(tpfc)}$ 。接下来 **D** 的化学式无论从其制备反应或是从 **C** 与 **D** 的相互转化入手，结合"八面体配位"条件，均可推出为 $\mathrm{Cr(py)_{2}(tpfc)}$ 。

最后即可利用所有已知条件验证答案。从 C 与 D 向 B 的转化看，仅有 $O_{2}$ 参与反应，说明 B 中配体仅含氧元素，可能为 O 或 $O_{2}$ ；按当量比写出 C 的制备反应方程式后，发现只有 O 原子符合要求。此外，配合物的颜色也暗示了 Cr 的价态，C 和 D 的绿色即 +3 价 Cr 的颜色。

## 解题思路

1. **读题定位**：根据题干中的反应条件、磁矩数据、配位几何和颜色信息，推断B、C、D三个配合物的化学式。
2. **🔑 关键转换**：
   - 咔咯配体必须以 $(tpfc)^{3-}$ 形式配位，Cr氧化态至少为+3
   - B在空气中制备，有O₂参与；磁矩1.72 $\mu_{\mathrm{B}}$ 对应1个单电子
   - 四方锥场中，结合单电子条件，Cr为+5价，配体为O²⁻
   - C的制备为还原反应，PPh₃被氧化为OPPh₃，两分子OPPh₃配位
   - D为八面体配位，py为配体

## 易错分析

| 错误 | 原因 | 纠正 | 课堂提问 |
|---|---|---|---|
| B中Cr氧化态判断为+3 | 未考虑B、C、D转化关系中B氧化态最高 | B的氧化态应高于C和D，结合单电子条件应为+5 | 如何从转化关系判断各物种的相对氧化态？ |
| C的化学式写错 | 未注意到PPh₃转化为OPPh₃，加上原有OPPh₃共两分子 | 反应中PPh₃被氧化，产物中有两分子OPPh₃配体 | PPh₃和OPPh₃在该反应中分别起什么作用？ |
| 忽略(tpfc)³⁻形式 | 未理解咔咯四个N均参与配位的条件 | 连H的N孤对电子参与共轭，必须以去质子形式配位 | 为什么咔咯要以(tpfc)³⁻形式配位？ |
