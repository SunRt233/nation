# 题-034-1-3-2：氢键与热加工性能

## 题目

> **来源**：[[07-资料提炼/提炼-第34届初赛试题解析|第34届初赛第1题]]（原始文件：[[mineru/02-真题解析/34届初赛试题解析]]）
> **小问关联**：本文件为第(3-2)小问；其余小问见 [[题-034-1-1-C3F4的Lewis结构与杂化]]、[[题-034-1-2-铝镁合金组成计算]]、[[题-034-1-3-1-可循环聚合物结构式]]

单体 A、C 和 D 按物质的量比例 6:3:2 进行共聚合，得到的聚合物不能进行热加工和循环利用；但若在共聚合时加入一定量 ($\approx$ 10%) 的 B，得到的聚合物又具备了可热加工和循环利用的性能。简述原因。

（本题干及单体结构见 [[题-034-1-3-1-可循环聚合物结构式]]）

## 参考答案

**得分点**：
1. **分析A、C、D共聚形成共价交联网络**（3分）：三官能度单体导致体形聚合物
2. **解释加入B后引入四重氢键**（3分）：氢键的动态可逆性
3. **阐明氢键断裂-重组实现热加工**（2分）：加热时氢键可逆断裂，冷却后重新形成

本小问进一步探讨聚合物的性能，特别是"可热加工和循环利用"的能力。热塑性或者热固性来自热加工时是否形成牢固的化学键。氢键是一种较弱的相互作用力，强度远低于共价键。由于氢键的可逆性，加入 B 后形成的四重氢键在加热条件下能够动态地断裂和重新形成。而本小问"可热加工和循环利用的性能"，也反过来暗示着 1-3-1 中的聚合方式并非传统的共价作用，而是可逆的分子间弱相互作用（此处为氢键）。

对于简答题，答题时需注意逻辑以确保答案的完整性。首先要明确加入单体 B 后的变化：加入一定量的 B 后，聚合物结构中引入了四重氢键连接基元。这些氢键的存在使得聚合物在加热时可以通过断裂和重新形成氢键，实现链段的重新排列，从而具备可热加工的能力。

为什么单体 A、C 和 D 进行聚合时，它们的物质的量之比为 6:3:2？这与单体的官能度以及平均官能度的概念有关。官能度只有 1 的分子只能生成二聚物，官能度为 2 的单体可以生成线形聚合物，官能度大于 2 的分子可以生成体形聚合物。通常体形聚合物比线形聚合物有更高的强度。

其次, 我们要对比掺入前后的聚合物性能: 掺入前的聚合物（仅由 A、C 和 D 共聚合）形成的是共价交联网络结构，这种结构由于共价键的稳定性，无法进行热加工和循环利用。掺入 B 后，A、B、C 和 D 共聚合的结构中由于四重氢键的引入，使得聚合物具备了动态可逆的连接方式，因而能够在加热条件下进行加工和循环利用。

完整答案示例：在 A、C 和 D 共聚合后，形成了共价交联网络结构，这种结构由于共价键的稳定性，无法进行热加工和循环利用。而在 A、B、C 和 D 共聚合的结构中，存在四重氢键的连接基元。由于氢键是较弱的相互作用力，在加热条件下，四重氢键能够动态地断裂与重组，从而使得聚合物材料能够具备可热加工和循环利用的性能。

## 解题思路

1. **读题定位**：对比两组实验——不加B时不能热加工，加B后可以。需要从单体结构和聚合机理角度解释。
2. **🔑 关键转换**：A是二异氰酸酯(官能度2)，C和D的结构决定它们是多官能度单体（6:3:2的比例暗示平均官能度>2，形成共价交联网络）。B引入UPy四重氢键，氢键在加热时可逆断裂、冷却时重新形成，赋予材料热塑性和可循环利用性。这是自修复/本征型可逆材料的典型特征。
3. **逻辑链**：不加B → 纯共价交联 → 热固性 → 不能热加工；加B → 共价键+氢键混合网络 → 氢键可逆 → 热塑性 → 可热加工循环利用。
