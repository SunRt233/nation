## 参考答案

**得分点**：
1. **8-1 正确画出 A 和 B**（1.5分）：A 为三元环或四元环碳正离子；B 为四元环溴代产物
2. **8-2 正确选择**（1分）：**B（环丙基取代的苯甲酸）** 酸性最弱
3. **8-3 正确选择**（1分）：**B** 的 $\alpha$-H 酸性较弱
4. **8-4 正确选择**（1分）：**A（OTs 处于直立键）** 溶剂解速率较快
5. **8-5 正确选择**（1.5分）：**C** 形成环氧速率最快

**完整推导**：

### 8-1 碳正离子扩环重排

反应机理：
1. 烯烃与质子发生反应得到碳正离子（Markovnikov 规则，二级碳正离子）
2. 三元环发生扩环重排，得到四元环中间体
3. 碳正离子与溴离子结合，得到最终产物

- **中间体 A**：三元环碳正离子（或四元环碳正离子，扩环后）
- **产物 B**：四元环溴代产物

> 若题目要求立体化学，最终产物 B 的结构需要考虑碳正离子的平面构型和 Br⁻ 从两侧进攻的概率。

### 8-2 羧酸酸性比较

苯环上的羧基酸性与苯环的富电子程度是**负相关**的：
- 吸电子基团 → 稳定羧酸根负离子 → 酸性增强
- 给电子基团 →  destabilize 羧酸根负离子 → 酸性减弱

比较异丙基和环丙基的电子效应：
- **异丙基**：具有给电子超共轭效应
- **环丙基**：具有**给电子共轭效应**（环丙烷的 Walsh 轨道中 HOMO 是 $\pi$ 轨道，可与苯环共轭）

环丙基的给电子能力比异丙基**更强**。故酸性大小关系为：

$$\mathbf{A} > \mathbf{C} > \mathbf{B}$$

**B（环丙基取代的苯甲酸）酸性最弱。**

### 8-3 羰基 $\alpha$-H 酸性比较

羰基化合物的 $\alpha$-H 酸性与羰基的活性（亲电能力）息息相关：
- $\alpha$-碳负离子可以被羰基的吸电子共轭效应所稳定
- 另一侧连有的给电子取代基会削弱羰基的吸电子能力，导致酸性下降

比较 A 和 B：
- **A** 中多出了四个甲基，带来的斥力使六元环需要旋转一定的角度，偏离羰基所在的平面
- 这会**破坏酰胺 N 原子与羰基的共轭**，给电子共轭效应的消失导致 A 中羰基的活性增强
- 故 **A 的 $\alpha$-H 酸性强于 B**

**B 的 $\alpha$-H 酸性较弱。**

### 8-4 溶剂解反应速率

本小题比较的是 $\mathrm{S_N2}$ 反应的速率快慢。$\mathrm{S_N2}$ 反应要求亲核试剂从离去基团的**背面进攻**。

反应底物都是反式十氢化萘环系，不存在可以翻转的构象异构体，只需分析处于直立键和平伏键的离去基团 -OTs 的区别：

- **A（-OTs 处于直立键）**：亲核试剂的进攻方向与直立氢原子近乎平行，位阻大大降低
- **B（-OTs 处于平伏键）**：亲核试剂的进攻路线被直立 H 原子所阻碍

故反应速率 **A > B**。

> 注意：很多同学会默认溶剂解反应一定是 $\mathrm{S_N1}$ 反应，这是错误的。本小题中的溶剂解体系中含有乙酸根负离子，同时亲核取代反应在二级碳上发生，故应采取 $\mathrm{S_N2}$ 反应过程。

### 8-5 环氧形成速率

三个底物都是 $\beta$-氯代醇，在碱性条件下发生**分子内 $\mathrm{S_N2}$** 反应，得到环氧化合物。

随着成环反应体系中引入的取代基逐渐增多，成环反应的速率加快、平衡常数增大，这种效应称为 **Thorpe-Ingold 效应**（也称**偕二甲基效应**）。

对于生成小环结构（如本题形成的三元环体系）：
- 理想的饱和碳原子链，键角为 $109.5^{\circ}$
- 引入取代基后（逐渐引入两个和四个甲基），由于取代基之间的排斥作用导致主链键角减小
- 在底物分子的构象中会使得亲核基团与离去基团距离减小，最终导致分子内 $\mathrm{S_N2}$ 反应的速率加快

故反应速率 **A < B < C**。

**C 形成环氧乙烷衍生物的反应速率最快。**

## 解题思路

1. **8-1 碳正离子重排**：烯烃质子化 → 二级碳正离子 → 三元环扩环为四元环 → 与 Br⁻ 结合
2. **8-2 酸性比较**：
   - **🔑 关键转换**：环丙基具有给电子共轭效应（Walsh 轨道），比异丙基的超共轭效应更强
   - 给电子效应越强，羧酸根越不稳定，酸性越弱
3. **8-3 α-H 酸性**：
   - 四个甲基的位阻破坏了酰胺 N 与羰基的共轭
   - 共轭破坏 → 羰基活性增强 → α-H 酸性增强
4. **8-4 溶剂解速率**：
   - 本题为 SN2 机理（乙酸根负离子作为亲核试剂）
   - 直立键 -OTs 有利于背面进攻，位阻小
5. **8-5 环氧形成**：
   - Thorpe-Ingold 效应：取代基越多，主链键角越小，亲核基团与离去基团越接近，成环越快

## 知识拓展

**环丙基的给电子共轭效应**：

从分子轨道的角度理解环丙烷的成键过程：C 原子采取 $\mathrm{sp}^2$ 杂化，三条 $\mathrm{sp}^2$ 杂化轨道向心形成碳骨架，余下的三条 $2\mathrm{p}$ 轨道形成离域体系（Walsh 轨道）。

环丙烷的分子轨道中 HOMO（最高占有轨道）是两条简并的 $\pi$ 轨道，而非 $\sigma$ 轨道。正因如此，环丙基应具有的是**给电子共轭效应**，其给电子能力比异丙基要强。

**Thorpe-Ingold 效应（偕二甲基效应）**：

对于生成较大环系（$\geq 5$）的反应，一般通过过渡态理论中活化熵的变化来解释：
- 成环反应之前，底物分子可能采取的优势构象非常多，达到成环反应过渡态结构的概率很低
- 活化熵是相当大的负值
- 当侧链引入取代基后，减少了底物分子可能采取的优势构象，分子达到过渡态结构的概率上升
- 反应活化熵的绝对值减小（仍为负数），活化 Gibbs 自由能降低，更有利于成环反应的进行
