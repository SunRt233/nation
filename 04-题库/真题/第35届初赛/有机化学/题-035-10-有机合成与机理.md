## 参考答案

**得分点**：
1. **10-1 正确画出共振式**（2分）：N 的孤对电子与羰基 $\pi$ 轨道共轭的共振式
2. **10-2-1 正确判断**（1分）：**Z = F** 时端基效应最强
3. **10-2-2 正确判断并简述理由**（2分）：**降低**；磺酰基吸电子能力更强，逆转 N-Z 键极性，Z 以正离子离去
4. **10-3 正确画出两个关键中间体**（3分）：N-N 键中间体（苄胺 N 进攻酰胺 N）和氮宾中间体
5. **10-4 正确完成反应式并标明立体化学**（7分）：四小题各 1.5~2 分

**完整推导**：

### 10-1 酰胺的共振式

酰胺中的氨基具有给电子共轭效应。从结构的视角来看，N 原子采取 $\mathrm{sp^2}$ 杂化，孤对电子所处的 $\mathrm{p}$ 轨道与羰基的 $\pi$ 轨道共轭，因此有共振式：

$$
\mathrm{R-C(=O)-NR_2 \longleftrightarrow R-C(-O^-)=NR_2^+}
$$

主要共振极限式包括：
- 酰胺式：$\mathrm{R-C(=O)-NR_2}$（主要贡献者）
- 烯醇胺式：$\mathrm{R-C(-O^-)=NR_2^+}$（次要贡献者，但解释了 C-N 键的部分双键性质）

### 10-2 端基效应（anomeric effect）

#### 10-2-1 最强端基效应

当酰胺 N 原子上连有两个电负性较大的杂原子时，具有「端基效应」。对于均属于第二周期的元素 N、O 和 F：

同一周期从左到右电负性逐渐增加，与 N 原子成键的反键轨道能级逐渐降低；另一方面，杂原子电负性的增大，导致 N-Z 键的 $\sigma$ 成键轨道向 Z 原子方向极化，$\sigma^*$ 反键轨道向 N 原子方向极化。故 $n_\mathrm{O} \rightarrow \sigma^*_{\mathrm{N-F}}$ 的轨道重叠程度最好、体系的能量降低幅度最大。

**答案：Z 为氟原子取代的酰胺具有最强的端基效应。**

#### 10-2-2 磺酰胺的端基效应

**答案：降低。**

理由：磺酰基与羰基相比，具有更好的吸电子诱导效应和吸电子共轭效应，这导致逆转了 N-Z 键的极性，使得基团 Z 不再以负离子的形式离去。事实上，Z 反而更容易以正离子 $\mathrm{Z^+}$ 的形式离去，端基效应被大幅削弱。

### 10-3 分子编辑：氮原子删除

根据题目提示「删除氮原子，放出氮气，形成 C-C 键」，可知反应的关键中间体必须含有 **N-N 键**。这个 N-N 键可由酰胺 N 原子和苄胺 N 原子构建：

**中间体 1**：酰胺 N 原子上连有三个吸电子诱导取代基，是缺电子原子，取代基中 $\mathrm{PivO^-}$（叔戊酸根）是好的离去基团，故富电子的苄胺 N 原子亲核进攻酰胺 N 原子，发生亲核取代反应，构建 **N-N 键**。

**中间体 2（氮宾）**：得到的中间体 1 仍可看作具有端基效应的酰胺。用切断法进行分析：OBn 可以以 $\mathrm{BnO^-}$ 的形式离去，酰胺可以以 $\mathrm{ArCO^+}$ 的形式离去。由于这两个基团离去能力都不强，因此它们最好**同步离去**，并在离去的同时形成 $\mathrm{ArC(O)-OBn}$ 酯键，这样就能有效降低反应的活化能，得到**氮宾中间体**。

最后通过**紧密自由基对**过程，氮气离去，得到 C-C 键偶联产物。

### 10-4 Dyotropic 重排

双重重排反应本质上属于**周环反应**，具有高度的立体专一性。其前线轨道分析的结论：
- 至少有一个迁移基团需要具有孤对电子，或拥有符合 $\pi$ 对称性的轨道
- 基团 X 和 Y 分别进行**同面迁移**
- 若 A 和/或 B 是手性碳原子，则反应后手性发生**翻转**
- 基团 X 和 Y 中的手性碳构型**不变**

**10-4-1**：$\gamma$-内酯在 Lewis 酸的作用下发生双重重排反应进行扩环，反应的驱动力为**小环张力的释放**。在重排过程中，两个迁移基团呈**反式共平面**的空间关系，以满足过渡态中最大化的轨道重叠。产物为**五元环内酯**。

**10-4-2**：与 10-4-1 类似，仍为 $\gamma$-内酯的扩环重排。但需要注意除 C-O 键迁移外，另一个迁移的是 **C-H 键**而非 C-C 键。若 C-C 键迁移，则六元环结构会扩环为七元环结构，稳定性降低。

**10-4-3**：二甲基铜锂与二氢呋喃锂反应，是一个配位（转金属化）过程，得到四配位的 Cu(I) 负离子中间体；随后在 C-Cu 键上发生双重重排反应，转化为「铜(I)酸盐负离子」中间体；最后进行水解。反应过程中**双键保持 E 构型不变**。

**10-4-4**：起始反应物为钯催化偶联反应的中间体。由于 Pd 含有 d 轨道，虽无孤对电子，也能满足双重重排反应对轨道对称性的要求。与 C-Pd 键处于**反式共平面**位置的是苯基，故双重重排反应模型为「Pd-C-C-$\mathrm{C_{Ph}}$」，重排后得到中间体。随后发生**还原消除**反应，得到最终产物。还原消除反应不影响手性碳原子的绝对构型。

## 解题思路

1. **10-1 共振式**：
   - N 的孤对电子与 C=O 的 $\pi$ 轨道共轭
   - 画出酰胺式和烯醇胺式两个主要共振极限式
2. **10-2 端基效应**：
   - **🔑 关键转换**：端基效应的本质是 $n_\mathrm{Y} \rightarrow \sigma^*_{\mathrm{N-Z}}$ 的轨道重叠
   - 同周期元素中，F 的电负性最大，$\sigma^*_{\mathrm{N-F}}$ 能级最低，重叠最好
   - 磺酰基吸电子能力更强，逆转 N-Z 键极性，端基效应降低
3. **10-3 氮原子删除**：
   - 第一步：苄胺 N 亲核进攻酰胺 N，构建 N-N 键
   - 第二步：同步离去形成氮宾（nitrene）
   - 第三步：紧密自由基对，N₂ 离去，C-C 键形成
4. **10-4 Dyotropic 重排**：
   - 识别「A-B 骨架单元」
   - 反式共平面要求
   - 手性碳构型翻转；迁移基团中的手性碳构型不变

## 知识拓展

**具有端基效应的酰胺 vs 经典酰胺**：

| 特征 | 具有端基效应的酰胺 | 经典酰胺 |
|---|---|---|
| N 原子杂化 | $\mathrm{sp^3}$ | $\mathrm{sp^2}$ |
| N 孤对电子 | 不与羰基共轭 | 与羰基共轭 |
| N 原子性质 | 可发生亲核取代 | 亲核性弱 |
| 结构稳定性 | 依赖 $n \rightarrow \sigma^*$ 相互作用 | 依赖 $\pi$ 共轭 |

**Dyotropic 重排反应的研究进展**：

双重重排反应（Dyotropic rearrangement）是指相邻两个原子上的 $\sigma$ 键发生**协同重排**的反应类型。近年来，该反应在复杂分子合成中展现出巨大潜力：

- **C-C 键活化**：2021 年，Cao 等人报道了通过 Pd(IV) 物种的 Dyotropic 重排实现 C-C 键活化（*Nat. Chem.* **2021**, 13, 671-676）
- **立体专一性合成**：由于反应的协同机理，Dyotropic 重排具有高度的立体专一性，可用于手性中心的精准构建
- **机理研究**：前线轨道分析表明，至少有一个迁移基团需要具有孤对电子或符合 $\pi$ 对称性的轨道，否则反应因轨道对称性不匹配而禁阻
