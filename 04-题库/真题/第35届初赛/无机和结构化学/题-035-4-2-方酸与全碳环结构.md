# 题-035-4-2：方酸与全碳环结构

## 题目

> **来源**：[[07-资料提炼/提炼-第35届初赛试题解析|第35届初赛第4题]]（原始文件：[[06-外部资料导入/真题/第35届初赛试题解析]]）
> **小问关联**：本文件为第 4 题第(2)小问；第(1)小问见 [[题-035-4-1-干冰晶胞分析]]

有趣的碳氧物种：存在一类通式为 $(\mathrm{C_nO_n})^{2-}$（$n = 3, 4, 5, 6$）的环状阴离子，呈平面结构。这些物种在合成化学中起着重要的作用。

**4-2-1** 画出 $\mathrm{H_2C_4O_4}$（方酸）的结构。

**4-2-2** 画出方酸根的结构，要求给出所有稳定的共振式。

**4-2-3** 探索新的碳单质：全碳环 [18]（记为 **O**）是一种环状结构的碳单质。20 世纪 60 年代就已提出，在 2019 年由科学家借助原子力显微镜（AFM）技术，通过对碳氧化物 $\mathrm{C_{24}O_6}$（记为 **P**）进行原子尺度的操作而得以实现。在全碳环 [18] 的探索中，正是借助于方酸的衍生物 $(\mathrm{CH_3})_2\mathrm{C_4O_4}$ 以及 $\mathrm{Li-C \equiv C-SiR_3}$（$\mathrm{R = iPr, Me}$）等试剂，科学家巧妙设计反应路线，制得环状分子 **P**。**P** 分子中有三重旋转轴，在 **P** 转化为 **O** 的过程中，主要有 4 种途径，涉及两个中间体 **X** 和 **Y**，**X** 的质量比 **P** 减少 $14.6\%$，**Y** 则减少 $29.2\%$。画出 **O**、**P**、**X** 和 **Y** 的结构。

## 参考答案

**得分点**：
1. **4-2-1 正确画出方酸结构**（1分）：四元碳环，两个对位 C 各连两个 OH，两个对位 C 各连两个 =O（或两个对位 C 连一个 =O 和一个 -OH，经烯醇-酮互变异构）
2. **4-2-2 正确画出方酸根共振式**（2分）：给出所有 C-C 键等长、C-O 键等长的共振式，至少3种
3. **4-2-3 正确画出 O、P、X、Y 的结构**（4分）：O 为单叁键交替的18元碳环；P 为含3个方酸单元的环状分子；X 为脱去1个方酸单元（2CO）的结构；Y 为脱去2个方酸单元（4CO）的结构

**完整推导**：

### 4-2-1 方酸的结构

方酸分子 $\mathrm{H_2C_4O_4}$ 的不饱和度为 4，其中四元碳环占 1 个不饱和度。

两个羟基同时连在同一 C 原子上的结构（偕二醇）不稳定，故 4 个 C 原子各连一个 O 原子。为满足剩下的 3 个不饱和度，方酸的结构只能是如下两个可能结构之一：

```
      HO          O
       |          ||
   HO--C          C
       ||        ||
       C==C  或   C--C
       |          |
       C==C       C--C
       ||        ||
       C          C
      / \        / \
     O   OH     O   OH
```

实际上，方酸以烯醇-酮互变异构的形式存在，其稳定结构为：

```
         O                O⁻
         ||              //
     HO-C                C
         ||              ||
         C======C   ↔    C------C
         |               |   |
         C======C        C------C
         ||              ||
     HO-C                C
         ||              \\
         O                O⁻
    （方酸）         （方酸根的一种共振式）
```

俗称「方酸」，也意味着其酸性较强。实验测定方酸双负离子 $\mathrm{C_4O_4^{2-}}$ 的结构中，所有的 C-C 键等长、C-O 键等长，证实了其非常稳定的共振结构。

### 4-2-2 方酸根的共振式

方酸根 $\mathrm{C_4O_4^{2-}}$ 为含有 $\pi_4^6$ 体系的平面闭合环状体系，满足 Hückel 芳香性条件（$4n+2$，$n=1$），因而具备一定的稳定性。

稳定的共振式需满足：
- 所有 C-C 键等长（键级 1.5）
- 所有 C-O 键等长（键级 1.5）
- 负电荷离域分布在 O 原子上

主要共振式包括（至少给出以下3种）：

```
    O⁻          O           O⁻
    ||          ||          |
    C           C           C
   /  \        //  \\       /  \
  C====C  ↔  C----C  ↔   C====C
   \  /        \\  //       \  /
    C           C           C
    ||          ||          |
    O⁻          O⁻          O
```

（注：实际答题需画出完整的带电荷的 Lewis 结构）

### 4-2-3 O、P、X、Y 的结构推断

**P 的结构**：

根据题意，P 的分子式为 $\mathrm{C_{24}O_6}$，分子中有三重旋转轴。方酸衍生物 $(\mathrm{CH_3})_2\mathrm{C_4O_4}$ 中的两个甲基只能取代两个羟基上的 H 原子，故其为方酸二甲酯。

由于化合物 P 中不含有 H 原子，故 $\mathrm{Li-C \equiv C-SiR_3}$ 应作为亲核试剂，反应最终将方酸二甲酯中的甲氧基取代，使方酸结构单元与 $-\mathrm{C \equiv C-}$ 相连。

最后根据「P 分子中有三重旋转轴」，可推出化合物 **P** 的结构为：由 3 个方酸单元通过 6 个 $-\mathrm{C \equiv C-}$ 链连接形成的环状分子。

**X 和 Y 的结构**：

由前体化合物 P 得到全碳环 [18] **O** 的过程，仅需要将环上的三个方酸单元「去掉」即可。

合理推测中间体 **X** 和 **Y** 对应「去掉」一个和两个方酸单元的结构：
- **X** 的分子式为 $\mathrm{C_{22}O_4}$（脱去 2 个 CO）
- **Y** 的分子式为 $\mathrm{C_{20}O_2}$（脱去 4 个 CO）

用题干中的失重数据验算：
- 化合物 P 的相对分子质量为 $24 \times 12.01 + 6 \times 16.00 = 384.24$
- 中间体 X 的相对分子质量比 P 少 $384.24 \times 14.6\% = 56.1$，对应 **2 分子 CO**
- 中间体 Y 的相对分子质量比 P 少 $384.24 \times 29.2\% = 112.2$，对应 **4 分子 CO**

故推测的结果符合题意。

**O 的结构**：

全碳环 [18]（O）是 18 元碳环且含有叁键，因此它必为**单、叁键交替结构**。其结构为：

$$
\mathrm{-C \equiv C - C \equiv C - C \equiv C - \cdots}
$$

（共 9 个 $-\mathrm{C \equiv C}-$ 和 9 个 $-\mathrm{C}-$ 单键交替形成 18 元环）

> 也可以通过逆合成分析和对称性分析推断出 Y、X 和 P 的结构。

## 解题思路

1. **读题定位**：看到「方酸」「全碳环」「质量减少百分比」→ 识别为 **结构推断 + 共振式 + 新物质合成** 综合题
2. **4-2-1 方酸结构**：
   - 不饱和度 = 4，四元环占 1
   - 偕二醇不稳定 → 4 个 C 各连 1 个 O
   - 剩余 3 个不饱和度由 C=O 和 C=C 提供
3. **4-2-2 方酸根共振式**：
   - **🔑 关键转换**：方酸根为 $\pi_4^6$ 体系，满足 Hückel 规则，所有 C-C 键等长、C-O 键等长
   - 画出负电荷离域在 O 上的共振式
4. **4-2-3 全碳环推断**：
   - P（$\mathrm{C_{24}O_6}$）含 3 个方酸单元，通过 $-\mathrm{C \equiv C-}$ 连接成环
   - X 减少 14.6% = 2CO（56），Y 减少 29.2% = 4CO（112）
   - O 为 18 元碳环，单叁键交替

## 知识拓展

**全碳环（cyclo[n]carbon）的研究进展**：

在碳的同素异形体家族中：
- 金刚石：$\mathrm{sp}^3$ 杂化，三维网络
- 石墨烯、富勒烯、碳纳米管：$\mathrm{sp}^2$ 杂化，平面环系
- 全碳环：$\mathrm{sp}$ 杂化，环状分子

自 1966 年 Hoffmann 首次预言 $\mathrm{C_{18}}$ 的存在以来，其合成始终是碳材料科学领域的「圣杯」级难题。2019 年，IBM-牛津大学联合团队通过精密设计的脱羧反应策略，在 $5\,\mathrm{K}$ 的超低温环境下，利用原子力显微镜（AFM）首次捕获到 $\mathrm{C_{18}}$ 分子的真实形貌。

**全碳环的双芳香性**：

环系上所有的 C 原子均存在 2 条正交的 p 轨道，形成了垂直于环平面与朝向环平面内的 2 个正交的离域 $\pi$ 体系。这种罕见的双芳香性导致理论化学界对全碳环的键型产生重大分歧——键长平均化（累积双键）与键长交替（单、叁键交替）两种模型长期对峙。

AFM 成像显示 $\mathrm{C_{18}}$ 存在约 $15\,\mathrm{pm}$ 的键长差异（典型 C-C 单键约 $154\,\mathrm{pm}$，C≡C 叁键约 $120\,\mathrm{pm}$），而 $\mathrm{C_{10}}$ 和 $\mathrm{C_{14}}$ 则呈现更平均化的键长分布。这说明 $\mathrm{C_{18}}$ 实际处于两种极限结构的中间态，正是量子化学中电子离域与环张力动态平衡的直观体现。
