# 题-035-1-2：二氯二甲基硅烷水解

## 题目

> **来源**：[[07-资料提炼/提炼-第35届初赛试题解析|第35届初赛第1题]]（原始文件：[[06-外部资料导入/真题/第35届初赛试题解析]]）
> **小问关联**：本文件为第 1 题第(2)小问；第(1)小问见 [[题-035-1-1-GaAs气相外延制备]]；第(3)小问见 [[题-035-1-3-Fe3+氧化硫化物矿]]；第(4)小问见 [[题-035-1-4-钢铁发蓝处理]]

二氯二甲基硅烷水解，得到线性聚合物。写出该反应的化学方程式。

## 参考答案

**得分点**：
1. **正确推断产物**（1分）：聚二甲基硅氧烷（线性）和 HCl
2. **正确配平**（1分）：$n\mathrm{Cl-Si(CH_3)_2-Cl} + (n+1)\mathrm{H_2O} \longrightarrow \mathrm{HO[Si(CH_3)_2-O]_nH} + 2n\mathrm{HCl}$

**完整推导**：

该反应是聚合反应：首先发生 $\mathrm{Si-Cl}$ 键的水解，水分子亲核取代 Si 原子上的 Cl 原子，得到 $\mathrm{Si-OH}$ 键；随后，$\mathrm{Si-OH}$ 的羟基再对另一分子中的 $\mathrm{Si-Cl}$ 键进行同样的亲核取代，得到以 $\mathrm{Si-O-Si}$ 键相连的双分子聚合中间体。重复这个过程，即可得到聚甲基硅醚产物。

**封端方程式**：

$$
n \mathrm{Cl-Si(CH_3)_2-Cl} + (n+1)\mathrm{H_2O} \longrightarrow \mathrm{HO\left[Si(CH_3)_2-O\right]_nH} + 2n\mathrm{HCl}
$$

**不封端方程式（亦可接受）**：

$$
n \mathrm{Cl-Si(CH_3)_2-Cl} + n\mathrm{H_2O} \longrightarrow \left[\mathrm{Si(CH_3)_2-O}\right]_n + 2n\mathrm{HCl}
$$

> ⚠️ 注意：封端和不封端的两个方程式配平后的系数不同，应避免在这种问题上犯错。若体现封端基团，则必须以 $\mathrm{Si-OH}$ 形式进行封端。

## 解题思路

1. **读题定位**：看到「二氯二甲基硅烷」「水解」「线性聚合物」→ 识别为 **主族元素水解聚合** 题
2. **确定产物**：Si-Cl 键水解 → Si-OH；两分子 Si-OH 缩合 → Si-O-Si + H₂O；最终形成线性聚硅氧烷
3. **🔑 关键转换**：Si 原子上的亲核取代与 C 原子不同——Si 有可利用的 3d 轨道，可形成五配位中间体，亲核取代更容易发生
4. **验证**：配平后检查左右两边各原子数目是否守恒

## 易错分析

| 错误 | 原因 | 纠正 | 课堂提问 |
|---|---|---|---|
| 产物写成环状聚合物 | 题干明确要求「线性聚合物」 | 注意审题，环状聚合物不符合题意 | 「线性聚合物和环状聚合物的结构有什么区别？」 |
| 方程式系数错误 | 未区分封端与不封端 | 封端：n 单体需 (n+1) H₂O；不封端：n 单体需 n H₂O | 「如果聚合物两端是 OH，需要多少个水分子？」 |
| 漏写 HCl | 忽略了 Cl 的水解产物 | Si-Cl 水解必然产生 HCl | 「Si-Cl 键水解后 Cl 去哪里了？」 |
| 写成取代反应机理式 | 题干要求写总反应方程式 | 只需写出起始物与最终产物的总方程式 | 「反应方程式和反应机理式有什么区别？」 |

## 知识拓展

**Si 与 C 亲核取代机理的差异**：

- C 原子上的亲核取代（$\mathrm{S_N2}$）：背面进攻，Walden 翻转，过渡态为五配位碳（能量很高）
- Si 原子上的亲核取代：Si 有空的 3d 轨道可参与成键，容易形成稳定的五配位或六配位中间体，亲核取代反应活性远高于相应的碳化合物
