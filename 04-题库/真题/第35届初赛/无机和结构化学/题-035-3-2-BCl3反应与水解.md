# 题-035-3-2：BCl₃反应与水解

## 题目

> **来源**：[[07-资料提炼/提炼-第35届初赛试题解析|第35届初赛第3题]]（原始文件：[[06-外部资料导入/真题/第35届初赛试题解析]]）
> **小问关联**：本文件为第 3 题第(2)小问；第(1)小问见 [[题-035-3-1-硫代硼酸二聚三聚结构]]；第(3)小问见 [[题-035-3-3-卟吩类似物芳香性]]

A（即题-035-3-1中的二聚体 $(\mathrm{HBS_2})_2$）和 $\mathrm{BCl_3}$ 按计量系数 $1:1$ 反应，得到相同计量系数的 $\mathrm{H_2S}$ 和另一产物 **C**（反应1），C 受热分解为 **D** 和 $\mathrm{BCl_3}$（反应2）。在低温惰性介质中 D 可以单分子形式存在，结构中有且仅有二重旋转轴以及过二重轴的两个镜面。

**3-2-1** 写出 **C** 生成（反应 1）和分解（反应 2）的反应方程式。

**3-2-2** C 易水解。写出 C 发生水解的反应方程式。

**3-2-3** 画出 **C** 和 **D** 的结构。

## 参考答案

**得分点**：
1. **3-2-1 反应1正确**（1分）：$(\mathrm{HBS_2})_2 + \mathrm{BCl_3} \longrightarrow \mathrm{H_2S} + \mathrm{B_3S_3Cl_3}$
2. **3-2-1 反应2正确**（1分）：$\mathrm{B_3S_3Cl_3} \longrightarrow \mathrm{BCl_3} + \mathrm{B_2S_3}$
3. **3-2-2 水解方程式正确**（1.5分）：$\mathrm{B_3S_3Cl_3} + 9\mathrm{H_2O} \longrightarrow 3\mathrm{B(OH)_3} + 3\mathrm{H_2S} + 3\mathrm{HCl}$
4. **3-2-3 C 的结构正确**（1分）：三个 Cl 取代三聚体 B 中三个端基 -SH 的结构
5. **3-2-3 D 的结构正确**（1.5分）：折线形 $\mathrm{B_2S_3}$（$C_{2v}$ 点群）

**完整推导**：

### 3-2-1 反应方程式

由于题目中给出了二聚体 A 与 $\mathrm{BCl_3}$ 反应（反应1）的计量系数，合理推测1当量的 $\mathbf{A} = (\mathrm{HBS_2})_2 = \mathrm{H_2B_2S_4}$ 与1当量的 $\mathrm{BCl_3}$ 反应，生成1当量的 $\mathrm{H_2S}$。

根据原子守恒，另一产物 **C** 的化学式应为 $\mathrm{B_3S_3Cl_3}$。

$\mathrm{B_3S_3Cl_3}$ 受热分解，生成 $\mathrm{BCl_3}$ 和另一产物 **D**（反应2）。假设反应物 $\mathrm{B_3S_3Cl_3}$ 中的1个B原子和3个Cl原子变为 $\mathrm{BCl_3}$，剩下的原子构成产物 **D**，即 **D** 分子式为 $\mathrm{B_2S_3}$。

$$
(\mathrm{HBS_2})_2 + \mathrm{BCl_3} \longrightarrow \mathrm{H_2S} + \mathrm{B_3S_3Cl_3} \quad (\text{反应 1})
$$

$$
\mathrm{B_3S_3Cl_3} \longrightarrow \mathrm{BCl_3} + \mathrm{B_2S_3} \quad (\text{反应 2})
$$

### 3-2-2 水解方程式

对比 **C**（$\mathrm{B_3S_3Cl_3}$）与硫代硼酸三聚体 B（$\mathrm{HBS_2})_3 = \mathrm{B_3S_6H_3}$ 的分子式，可推测 **C** 是 3 个 Cl 原子取代了 B 中三个巯基的结构。

**C** 中的 **B-Cl 键**极易水解（可以类比 $\mathrm{BCl_3}$ 极易水解为硼酸），但这里容易忽略的是 **B-S 键也会发生水解**：

- B 原子半径较小，S 原子半径较大，形成 B-S 键的轨道并不能很好地重叠
- O 原子的半径与 B 原子半径匹配，形成的 B-O 键更强
- 从软硬酸碱的角度来理解：B 是硬酸，O 是硬碱，S 是软碱；B-O 键比 B-S 键更稳定

若未考虑到 B-S 键的水解，只写出 B-Cl 键水解的部分水解方程式，只能拿到一半的分数。**C** 彻底水解的反应方程式为：

$$
\mathrm{B_3S_3Cl_3} + 9\mathrm{H_2O} \longrightarrow 3\mathrm{B(OH)_3} + 3\mathrm{H_2S} + 3\mathrm{HCl}
$$

> 如果将含硼产物写为三聚硼酸 $\mathrm{B_3O_3(OH)_3} = \mathrm{B_3O_6H_3}$，也算正确：
> $$\mathrm{B_3S_3Cl_3} + 6\mathrm{H_2O} \longrightarrow \mathrm{B_3O_3(OH)_3} + 3\mathrm{H_2S} + 3\mathrm{HCl}$$

### 3-2-3 C 和 D 的结构

**C 的结构**：根据前一小问的分析，C 的结构是容易画出的——三个 Cl 原子取代三聚体 B 中三个端基 -SH 的结构，保持 $\mathrm{B_3S_3}$ 六元环骨架不变。

**D 的结构**：题目中给出了 $\mathbf{D} = \mathbf{B_2S_3}$ 的对称性信息：「有且仅有二重旋转轴以及过二重轴的两个镜面」。

对常见化合物对称性比较熟悉的同学应很快意识到，这与 $\mathrm{H_2O}$ 分子具有完全相同的对称元素（$C_{2v}$ 点群）。故 **D** 也应为**折线形结构**；其中的一个 S 原子作为中心原子，在中心原子的两侧对称地分布两个 $[\mathrm{BS}]$ 单元。

为使 S 原子满足 8 电子结构，两个 $[\mathrm{BS}]$ 单元应为 $\mathrm{B=S}$ 双键。B 原子虽然是 6 电子的缺电子中心，但可以通过形成离域 $\pi$ 键来削弱其缺电子性质。

**C 的结构**：六元环 $\mathrm{B_3S_3}$，每个 B 上连一个 Cl，每个 S 为桥连 S（保持与三聚体相同的骨架）。

**D 的结构**：
```
    S=B=S=B=S
      折线形，中心S
```
（实际为：中心S原子两侧对称分布 B=S 单元）

> 思考题：D 中的 B 原子能否画成折线形？给出理由。D 中形成了哪种离域 $\pi$ 键？

## 解题思路

1. **读题定位**：看到「按计量系数1:1」「受热分解」「对称性」→ 识别为 **原子守恒 + 结构推断** 题
2. **3-2-1 推导C和D的分子式**：
   - A ($\mathrm{H_2B_2S_4}$) + $\mathrm{BCl_3}$ → $\mathrm{H_2S}$ + C
   - 原子守恒：C = $\mathrm{B_3S_3Cl_3}$
   - C 分解 → $\mathrm{BCl_3}$ + D，D = $\mathrm{B_2S_3}$
3. **3-2-2 水解方程式**：
   - **🔑 关键转换**：C 中既有 B-Cl 键又有 B-S 键，两者都会水解
   - B-Cl → B-OH + HCl；B-S → B-OH + H₂S
   - 彻底水解需 9 H₂O，生成 3 B(OH)₃ + 3 H₂S + 3 HCl
4. **3-2-3 D的结构**：
   - 对称性信息：仅有 $C_2$ 轴 + 2个镜面 → $C_{2v}$ 点群
   - 类比 H₂O → 折线形结构
   - 中心 S 原子，两侧对称分布 B=S 单元

## 知识拓展

**B-S 键与 B-O 键的稳定性比较**：

从软硬酸碱理论（HSAB）来看：
- B³⁺ 是 **硬酸**（电荷密度高、极化性小）
- O²⁻ 是 **硬碱**（电负性高、极化性小）
- S²⁻ 是 **软碱**（电负性较低、极化性大、易变形）

根据「硬亲硬，软亲软」原则，B-O 键比 B-S 键更稳定。这也解释了为什么含 B-S 键的化合物（如本题中的 C）在水中容易发生 B-S → B-O 的转换。

> 也可以从轨道重叠的角度理解：B 原子半径较小，S 原子半径较大，形成 B-S 键的轨道并不能很好地重叠；而 O 原子的半径与 B 原子半径匹配，形成的 B-O 键更强。
