# 题-035-5：元素推断与反应方程式

## 题目

> **来源**：[[07-资料提炼/提炼-第35届初赛试题解析|第35届初赛第5题]]（原始文件：[[06-外部资料导入/真题/第35届初赛试题解析]]）

$4.38\,\mathrm{g}$ 元素 A 的单质在 $\mathrm{HNO_3}$ 中加热，完全反应后得到 $6.16\,\mathrm{g}$ 固体 B（A 的氧化物），并放出红棕色的气体（反应1）。A 也可以与氢氧化钠反应，发生歧化，生成两种盐 C 和 D（反应2），C 中 A 元素的氧化数与其在 B 中相同。B 与邻苯二胺反应生成 E，E 含元素 A 且具有特征的荧光光谱，据此可进行苯二胺异构体共存下邻苯二胺的含量分析。

**5-1** 通过计算和分析，确定元素 A。

**5-2** 分别写出 A 在酸和碱中反应的方程式。

**5-3** 画出化合物 E 的结构简式。

## 参考答案

**得分点**：
1. **5-1 正确确定元素**（3分）：**Se（硒）**
2. **5-2 正确写出两个方程式**（2分）：酸性中生成 $\mathrm{SeO_2}$ 和 $\mathrm{NO_2}$；碱性中歧化为 $\mathrm{Na_2Se}$ 和 $\mathrm{Na_2SeO_3}$
3. **5-3 正确画出 E 的结构**（2分）：苯并硒二唑

**完整推导**：

### 5-1 确定元素 A

根据题目所给条件，整理出相关信息：
- 单质 A 被 $\mathrm{HNO_3}$ 氧化，得到的氧化物 B 中 A 元素应为较高氧化态
- 单质 A 可以在碱中歧化，故 A 元素应为**非金属元素**
- 设氧化物 B 的化学式为 $\mathrm{A_2O_n}$，根据题干中给出的数据可列出不定方程：

$$
n_\mathrm{A} = \frac{4.38\,\mathrm{g}}{M_\mathrm{A}} = 2 \times \frac{6.16\,\mathrm{g}}{2M_\mathrm{A} + 16.00n}
$$

整理得：

$$
M_\mathrm{A} = 19.7n\,\mathrm{g\,mol^{-1}}
$$

分类讨论：

| $n$ | $M_\mathrm{A}$ | 可能元素 | 是否符合 |
|:---:|:---:|---|:---:|
| 1 | 19.7 | 无 | ✗ |
| 2 | 39.4 | 无 | ✗ |
| 3 | 59.1 | Co (58.9)、Ni (58.7) | ✗（金属，不能歧化） |
| **4** | **78.8** | **Se (79.0)** | **✓** |
| 5 | 98.5 | Tc (98) | ✗ |
| 6 | 118.2 | Sn (118.7) | ✗（金属，不能歧化） |
| 7 | 137.9 | Ba (137.3) | ✗ |
| $\geq 8$ | — | 氧化数 $\geq 8$，不存在 | ✗ |

综上，元素 A 为 **Se（硒）**，氧化物 B 为 $\mathrm{SeO_2}$。

### 5-2 反应方程式

**反应1（酸性条件）**：

$$
\mathrm{Se + 4HNO_3 \longrightarrow SeO_2 + 4NO_2 + 2H_2O}
$$

**反应2（碱性条件，歧化）**：

$$
\mathrm{3Se + 6NaOH \longrightarrow 2Na_2Se + Na_2SeO_3 + 3H_2O}
$$

验证：
- 反应1：Se $0 \rightarrow +4$，失 $4\mathrm{e^-}$；N $+5 \rightarrow +4$，每个 N 得 $1\mathrm{e^-}$；4 个 N 共得 $4\mathrm{e^-}$ ✓
- 反应2：Se 歧化，$0 \rightarrow -2$（得 $2\mathrm{e^-}$）和 $0 \rightarrow +4$（失 $4\mathrm{e^-}$）；2 个 Se 得 $4\mathrm{e^-}$，1 个 Se 失 $4\mathrm{e^-}$ ✓

### 5-3 化合物 E 的结构

邻苯二胺与 $\mathrm{SeO_2}$ 反应，得到的产物 E 具有特征的荧光光谱，合理推测其结构中具有较大的共轭体系。而除邻苯二胺以外的苯二胺异构体无法发生类似的反应，说明产物 E 的生成与两个**邻位的氨基**高度相关。

结合基础有机化学中的相关知识：$\mathrm{SeO_2}$ 常作为氧化剂，在反应机理分析中，通常将其结构描述为两个 $\mathrm{Se=O}$ 键，故它与邻苯二胺反应可看作与邻位的两个氨基发生**缩合**的过程，得到**苯并硒二唑**产物：

```
      __________
     /          \
    |    N===Se  |
    |    |       |
    |    N       |
     \__________/
    （苯并硒二唑）
```

> 两种共振极限式均合理，但后者能更好地刻画氧化还原过程。

## 解题思路

1. **读题定位**：看到「质量数据」「红棕色气体」「歧化」「荧光光谱」→ 识别为 **元素推断 + 方程式书写** 综合题
2. **突破口**：利用质量数据建立 $M_\mathrm{A}$ 与 $n$ 的关系式
3. **🔑 关键转换**：$M_\mathrm{A} = 19.7n$，结合「非金属」「能歧化」筛选
   - $n = 4$ 时 $M_\mathrm{A} \approx 79$，对应 **Se**
   - Se 为非金属，可在碱中歧化（$0 \rightarrow -2$ 和 $+4$）
4. **验证**：
   - 酸性：Se 被 $\mathrm{HNO_3}$ 氧化为 $\mathrm{SeO_2}$，红棕色气体为 $\mathrm{NO_2}$ ✓
   - 碱性：歧化为 $\mathrm{Na_2Se}$（Se 为 $-2$）和 $\mathrm{Na_2SeO_3}$（Se 为 $+4$）✓
   - C（$\mathrm{Na_2SeO_3}$）中 Se 为 $+4$，与 B（$\mathrm{SeO_2}$）中相同 ✓
5. **E 的结构**：邻苯二胺 + $\mathrm{SeO_2}$ → 缩合 → 苯并硒二唑（含 N-Se-N 五元杂环）

## 易错分析

| 错误 | 原因 | 纠正 | 课堂提问 |
|---|---|---|---|
| 判断为 Co 或 Ni | 未利用「能在碱中歧化」的关键信息 | Co、Ni 是金属，不能在 NaOH 中歧化 | 「哪些元素能在碱中歧化？金属还是非金属？」 |
| 碱性方程式产物写错 | 未注意 C 中 Se 的氧化数与 B 相同 | C 为 $\mathrm{Na_2SeO_3}$（+4），D 为 $\mathrm{Na_2Se}$（-2） | 「歧化反应中 Se 的氧化数怎么变？C 和 D 分别是什么？」 |
| E 的结构画错 | 未注意「邻苯二胺」的邻位关系 | 只有邻位的两个氨基才能与 Se 形成五元环 | 「为什么间位和对位苯二胺不能发生类似反应？」 |
| 方程式未配平 | 电子转移不守恒 | 用电子守恒验证：歧化反应中得失电子必须相等 | 「3个 Se 原子歧化，电子怎么转移？得失电子守恒吗？」 |

## 知识拓展

**SeO₂ 在有机合成中的应用**：

$\mathrm{SeO_2}$ 是一种重要的选择性氧化剂，常用于：
1. **烯丙位氧化**：将烯烃的烯丙位氧化为醇或醛
2. **邻二醇氧化**：将 1,2-二醇氧化为醛或酮
3. **与邻二胺缩合**：形成苯并硒二唑类杂环化合物

苯并硒二唑类化合物具有特征的荧光光谱，常被用于荧光探针的设计。本题中利用这一性质进行邻苯二胺的选择性检测，是分析化学中有趣的应用实例。

**Se 的歧化反应**：

Se 与 S 类似，在碱性条件下发生歧化：
$$\mathrm{3Se + 6OH^- \longrightarrow 2Se^{2-} + SeO_3^{2-} + 3H_2O}$$

这与 S 的歧化反应类似：
$$\mathrm{3S + 6OH^- \longrightarrow 2S^{2-} + SO_3^{2-} + 3H_2O}$$

体现了氧族元素化学性质的相似性。
