# 题-035-1-4：钢铁发蓝处理

## 题目

> **来源**：[[07-资料提炼/提炼-第35届初赛试题解析|第35届初赛第1题]]（原始文件：[[06-外部资料导入/真题/第35届初赛试题解析]]）
> **小问关联**：本文件为第 1 题第(4)小问；第(1)小问见 [[题-035-1-1-GaAs气相外延制备]]；第(2)小问见 [[题-035-1-2-二氯二甲基硅烷水解]]；第(3)小问见 [[题-035-1-3-Fe3+氧化硫化物矿]]

「发蓝」是钢铁表面处理的一种方法：将洁净的铁件浸入一定浓度的 $\mathrm{NaOH}$ 和 $\mathrm{NaNO_2}$ 溶液中（必要时加入其他辅助物质），加热到适当温度并保持一定时间，在铁件的表面形成均匀稳定且防锈的氧化膜 $\mathrm{Fe_3O_4}$，显现棕黑或蓝黑色光泽，处理过程有**无色刺激性气体**放出。

写出该反应的化学方程式。

## 参考答案

**得分点**：
1. **正确推断产物**（1.5分）：$\mathrm{Fe_3O_4}$、$\mathrm{NH_3}$（或结合 OH⁻）
2. **正确配平**（1.5分）：$9\mathrm{Fe} + 8\mathrm{H_2O} + 4\mathrm{NO_2^-} \longrightarrow 3\mathrm{Fe_3O_4} + 4\mathrm{OH^-} + 4\mathrm{NH_3}$

**完整推导**：

「发蓝」反应是一种钢铁表面处理的方法，通过生成致密的氧化物保护膜来防止金属被水氧腐蚀。

在该反应中：
- **Fe 单质**被氧化为混合价态的 $\mathrm{Fe_3O_4}$（可看作 $\mathrm{FeO} \cdot \mathrm{Fe_2O_3}$，含 Fe²⁺ 和 Fe³⁺）
- **氧化剂**是 $\mathrm{NO_2^-}$
- **还原产物**的判断是本题的易错点：N 元素氧化态低于 $+3$ 时，符合「无色刺激性气体」的还原产物仅有 $\mathrm{NH_3}$（NO 为无色无味气体；$\mathrm{N_2O}$ 为无色、略带甜味的气体）

反应方程式为：

$$
9\mathrm{Fe} + 8\mathrm{H_2O} + 4\mathrm{NO_2^-} \longrightarrow 3\mathrm{Fe_3O_4} + 4\mathrm{OH^-} + 4\mathrm{NH_3}
$$

**电子转移验证**：
- Fe: $0 \rightarrow +8/3$（平均），9 个 Fe 共失 $24\mathrm{e^-}$
- N: $+3 \rightarrow -3$，4 个 N 共得 $24\mathrm{e^-}$
- 电子守恒 ✓

## 解题思路

1. **读题定位**：看到「发蓝」「表面处理」「Fe₃O₄」「无色刺激性气体」→ 识别为 **氧化还原方程式书写 + 产物推断** 题
2. **确定氧化剂还原剂**：
   - 还原剂：Fe（0 价）→ Fe₃O₄（Fe 平均 +8/3 价）
   - 氧化剂：NO₂⁻（N +3 价）→ ?
3. **🔑 关键转换**：通过「无色刺激性气体」推断还原产物为 NH₃
   - NO：无色无味 ✗
   - N₂O：无色、略带甜味 ✗
   - NH₃：无色刺激性气体 ✓
4. **配平**：用电子守恒法，注意反应在碱性条件（NaOH 溶液）中进行

## 易错分析

| 错误 | 原因 | 纠正 | 课堂提问 |
|---|---|---|---|
| 还原产物写成 NO 或 N₂ | 未准确理解「无色刺激性气体」的描述 | NO 无味，N₂O 有甜味，只有 NH₃ 是刺激性气味 | 「NO、N₂O、NH₃ 的气味分别是什么？」 |
| 产物写成 Fe₂O₃ 或 FeO | 未注意题干明确给出 Fe₃O₄ | 发蓝处理生成的是 Fe₃O₄（黑色），不是 Fe₂O₃（红棕色） | 「Fe₃O₄ 和 Fe₂O₃ 的颜色分别是什么？」 |
| 未写 OH⁻ | 未注意反应在碱性溶液中进行 | NaOH 参与反应，产物应有 OH⁻ | 「反应体系中有 NaOH，产物中应该有什么离子？」 |
| 电子转移计算错误 | Fe₃O₄ 中 Fe 为混合价态 | Fe₃O₄ 可看作 FeO·Fe₂O₃，平均氧化数为 +8/3 | 「Fe₃O₄ 中 Fe 的氧化数是多少？怎么计算？」 |

| 处理方法 | 成膜成分 | 颜色 | 原理 |

$\mathrm{Fe_3O_4}$ 膜致密、附着力强，能有效隔绝氧气和水，是理想的防锈保护层。
