# 4-4，本小题总共4分

首先可以计算每个氨基酸在 pH = 7.40 条件下带电荷数（可忽略水的电离的影响，和形成多肽对侧链 pKa 的影响），由 $\mathrm { p H } = \mathrm { p } K _ { \mathrm { a } } + \mathrm { l g } { \frac { \mathrm { [ B a s e ] } } { \mathrm { [ A c i d ] } } }$ （或者配分函数），可得电荷数

$\cdot$ 1+10PH-pK （H、R、K的正电荷数）或 $\mathrm { ( D , ~ E , ~ C ) }$ 1+10PKa-pH 的负电荷数）。计算可知在保留小数点后两位有效数字的前提下，精氨酸(R)、赖氨酸(K)、天冬氨酸(D)与谷氨酸(E)分别为 $+ 1 . 0 0 \mathrm { { \ : \ : + 1 . 0 0 \mathrm { { \ : \ : - 1 . 0 0 \mathrm { { \ : \ : - 1 . 0 0 } } } } } }$ ，所以它们的数量将不改变总电荷数的小数点后两位。因此，组氨酸与半胱氨酸的数量决定了总电荷数的小数点后两位。

计算可知每个组氨酸平均带电荷数为+0.042，每个半胱氨酸平均带电荷数为−0.15，总数为12的条件下各可能的组合产生的小数点后两位（总电荷是负电荷的情况下，组氨酸的数目大于5 个）如下：

因为总电荷数为−7.88，因此可知，组氨酸 $\pmb { n } _ { \mathbf { H } }$ 与半胱氨酸 $\pmb { n } _ { \mathbf { C } }$ 分别为 10 与 2。

根据组氨酸 $\pmb { n } _ { \mathbf { H } }$ 与半胱氨酸 $\pmb { n } _ { \mathbf { C } }$ 的值，以及赖氨酸数目为 20个，天冬氨酸数目为 18个，谷氨酸数目为16个，总电荷数的计算：

$$
(2 0 + n _ {\mathbf {R}} - 1 8 - 1 6) + (0. 0 4 2 \times 1 0 - 0. 1 5 \times 2) = - 7. 8 8
$$

可得 ${ \pmb n } _ { \bf R } = 6$
