# 第 7 题 有机酸性与极性 (10 分)

7-1酸碱理论对有机化学的发展具有重要的意义。

7-1-1 苯甲酸衍生物的酸性与其取代基的性质紧密相关。如下所示，化合物 B 的 $\mathsf { p } K _ { \mathrm { a } }$ 值显著小于其他三个化合物。对此结果给出合理解释。

![[46188ff2fb4634390bf8dc276ed8b858cdc155ad81643edd1e237b739c8e21f2.jpg]]

7-1-2 二元羧酸的酸性与二个羧基的距离密切相关。例如，草酸的 $\mathsf { p } K _ { \mathsf { a l } }$ 小于丙二酸的 $\mathsf { p } K _ { \mathsf { a l } }$ ，邻苯二甲酸的 $\mathsf { p } K _ { \mathsf { a l } }$ 小于间苯二甲酸的 $\mathsf { p } K _ { \mathsf { a l } }$ 。对此结果给出合理解释。

7-1-3 2,6-二甲基-4-硝基苯酚的 $\mathsf { p } K _ { \mathsf { a } }$ 小于3,5-二甲基-4-硝基苯酚的 $\mathsf { p } K _ { \mathsf { a } }$ ，对此结果给出合理解释。

7-1 答案：

7-1-1 由于邻位酚羟基对羧酸根负离子的分子内氢键作用，稳定了负离子。

![[622c115609934a62e5645ff5ea2552b0dafa152e6ffab46fd1bdd8d445614737.jpg]]

7-1-2 由于羧基的吸电子诱导效应随着碳链的延长变弱。

$$
\mathrm{HO} _ {2} \mathrm{C} ^ {-} \mathrm{CO} _ {2} \mathrm{H} \quad v s \quad \mathrm{HO} _ {2} \mathrm{C} ^ {\wedge} \mathrm{CO} _ {2} \mathrm{H}
$$

![[bbd1f03f8985d40b7ab4b8eca45d9c0d5f6287384b91a5aa02a49ee156f5916e.jpg]]

7-1-3 由于 3 和 5 位甲基的取代导致硝基与苯环的共平面程度降低，导致酚氧负电荷离域分散能力减弱，从而使其酸性降低， $\mathsf { p } K _ { \mathrm { a } }$ 变大。

![[9d4a87fac8f1ec47231fcc28164b73918a62a042a113b0a512542022669efabb.jpg]]

7-2 2015年，英国的科研团队首次合成了全顺式的1,2,3,4,5,6-六氟环己烷（1）。这是迄今已知极性最大的非离子型有机化合物。1的合成路线如下图所示：以肌醇（myo-inositol，2）为起始原料，通过6步转化合成中间体3，再经过以脱氧氟化和亲核氟化为关键步骤的 6步反应制备1。（提示：DeoxoFluor是一种常用的脱氧氟化试剂，可将羟基转化为氟，并生成 $\mathrm { S O F } _ { 2 }$ 。Et N•3HF 是一种常用的亲核氟化试剂）

![[a4fb34c5d62f6687d0c1345040f8ac66bd8cb616403db5408fb16ecae2d8e61b.jpg]]

7-2-1 1,2,3,4,5,6-六氟环己烷有多少个立体异构体（仅需数目）？画出其中一对对映体的结构简式。

7-2-2 画出化合物4的结构简式。

7-2-3 检测表明化合物 1 的 19F NMR 谱图只有–212.8 ppm 和–217.3 ppm 两种信号。画出 1 的稳定构象，并解释它具有大极性的原因。

7-2-4 简要阐述最后一步反应（由 8合成1）产率低的主要原因。

7-2 答案：

7-2-1 9个立体异构体；2个对映异构体结构。

7-2-3 化合物 1 具有环己烷的经典椅式构象；氟所在的一面则具有负电性，氢所在的一面具有正电性，正、负电荷中心分离明显。因此具有很大的偶极矩。
