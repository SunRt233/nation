# 第 1 题 硒的提取与同位素化学 (14 分)

1-1 以阳极泥中的 $\mathsf { A g } _ { 2 } \mathsf { S e }$ 和 $\mathrm { C u } _ { 2 } \mathrm { S e }$ 为例，写出反应 1 中的化学反应方程式。

1-1 $Ag_{2}Se + Na_{2}CO_{3} + O_{2} \rightarrow 2Ag + Na_{2}SeO_{3} + CO_{2}$ $Cu_{2}Se + Na_{2}CO_{3} + 2O_{2} \rightarrow 2CuO + Na_{2}SeO_{3} + CO_{2}$

1-2 写出反应2 中的化学反应方程式。

1-2 $H_{2}SeO_{3} + 2SO_{2} + H_{2}O \rightarrow Se + 2H_{2}SO_{4}$

1-3 双β衰变是原子核的一种稀有衰变方式，可近似看作连续的两次β衰变。第一个成功观测到双β衰变现象的原子核是 $^ { 8 2 } _ { 3 4 } \mathrm { S e }$ ，写出 Se 82 发生双β衰变的反应方程式。

1-3 $_{34}^{82}Se \rightarrow _{36}^{82}Kr + 2e^{-\Box}$

1-4 $\mathrm { G a C l } _ { 3 }$ 和 $\mathrm { { \bf S e C l } _ { 4 } }$ 可在室温下发生反应，生成离子型化合物 R。然而，将等摩尔数的 $\mathrm { S e O } _ { 2 } , \mathrm { S e C l } _ { 4 }$ 和GaCl3 于 $5 0 ^ { \circ } \mathrm { C }$ 反应，得到等摩尔数的无色晶体 P和液态物质 Q。单晶 X-射线衍射测试表明，无色晶体P 中，Ga 中心为四面体配位几何构型，Se 的配位几何为三角锥形。Q 的分子几何构型为三角锥形。在$8 0 0 ^ { \circ } \mathrm { C }$ 惰性气氛中，P发生热分解，剩余残渣的质量百分含量约为 11%。

1-4-1 写出能表明 R结构特征的化学式，分别写出 P和 Q的化学式。

1-4-1 R: $[SeCl_{3}][GaCl_{4}]$ P: $GaSeCl_{5}O$ Q : $SeOCl_{2}$

1-4-2 根据 Se、Ga 氯化物和氧化物的相关性质，推导出 P的热分解产物，并写出化学反应方程式。

1-4-2
计算：因为 $GaSeCl_{5}O$ 中 Se:Cl 为 1:5，故产物中必定同时有 $SeCl_{4}$ 和 $GaCl_{3}$ 。
因为 $GaSeCl_{5}O$ 中 Ga:O 为 1:1，故产物中必定同时有 $SeO_{2}$ 和 $Ga_{2}O_{3}$ 。
因此分解产物只可能为 $SeCl_{4}$ 、 $GaCl_{3}$ 、 $SeO_{2}$ 和 $Ga_{2}O_{3}$ $SeCl_{4}$ 、 $GaCl_{3}$ 和 $SeO_{2}$ 高温时为气体，故最后剩下的固体只能是 $Ga_{2}O_{3}$ $m_{Ga2O3}\% = \frac{2 \times 69.72 + 3 \times 16.00}{5 \times (69.72 + 78.96 + 5 \times 35.45 + 16.00)} \times 100\% = 10.96\%$ ，和实验数据接近，因此，可认为 $1\ mol\ Ga_{2}O_{3}$ 由 $5\ mol\ GaSeCl_{5}O$ 分解而得。
方程式： $5\ GaSeCl_{5}O \rightarrow 4\ SeCl_{4}\uparrow + 3GaCl_{3}\uparrow + SeO_{2}\uparrow + Ga_{2}O_{3}$
