4-5 测定过程中可利用光将受体(Acceptor)的电子转变至激发态(Acceptor\*)，给体(Donor)的电子将会沿着 DNA 传递至受体(Acceptor\*)，以填充受体中空缺出来的轨道。给体由于电子的缺失转变为正离子 Donor+。电子传递过程如下：

$$
\text { Acceptor } ^ {*} + \text { Donor } \rightarrow \text { Acceptor } ^ {-} + \text { Donor } ^ {+}
