# 题-029-11-2：硝基甲烷负离子氧化反应

## 题目
> **来源**：[[07-资料提炼/提炼-第29届初赛试题解析|第29届初赛第11题]]（原始文件：[[mineru/02-真题解析/29届初赛试题解析]]）
> **小问关联**：本文件为第(11-2)小问；其余小问见 [[题-029-11-1-托品酮与溴苄反应]]

高效绿色合成一直是有机化学家追求的目标，用有机化合物替代金属氧化剂是重要的研究方向之一。硝基甲烷负离子是一种温和的有机氧化剂。画出硝基甲烷负离子的共振式（氮原子的形式电荷为正），并完成以下反应（写出所有产物）：

![[mineru/02-真题解析/29届初赛试题解析_images/bdc8c050b42fca9db71d0ebc9f640553fe2b4e0e21823133b5a9a89e967f0d06.jpg]]

## 参考答案

硝基甲烷负离子的共振式：

$$
\mathrm{H}_{2}\mathrm{C}=\mathrm{N}^{+}\left(\mathrm{O}^{-}\right)_{2} \longleftrightarrow ^{-}\mathrm{CH}_{2}-\mathrm{N}^{+}(=\mathrm{O})\mathrm{O}^{-} \longleftrightarrow \mathrm{H}_{2}\mathrm{C}^{-}-\mathrm{N}(=\mathrm{O})_{2}
$$

（氮原子的形式电荷为正）

## 解析

这是一个氧化反应。底物含有良好的离去基团 -OTs。从价态上，我们可以将底物等价为一级醇；从反应性上，我们知道它是一个亲电试剂，可以类比为上一题的溴苄。

硝基甲烷的甲基氢由于硝基的吸电子效应而具有一定的酸性。其共轭碱上的负电荷可以在两个氧与一个碳上共振，这也是第一个问题的答案。

更进一步可以看出，硝基甲烷负离子是一个两位负离子。其中碳的可极化能力强，亲核性更强；氧上负电荷密度更大，碱性更强。由软硬酸碱理论，O原子端较硬，C原子端较软。那么在反应中，到底是C原子端进攻还是O原子端进攻？

我们以烯醇负离子为例进行讨论。当烯醇负离子与 $\mathrm{Br}_{2}$ 发生反应时，是较软的C原子端进攻软酸 $\mathrm{Br}_{2}$，软亲软，生成C-溴化产物。而在烯醇硅醚的制备中，是较硬的O原子端进攻硬酸TMSCl，硬亲硬，形成稳定的O-Si键。

再回到题目本身，底物由一个活性很高的苄基和一个离去能力很好的-OTs组成，-OTs的吸电子性使得C-O键上的 $\sigma$ 电子比原先更多地向O原子偏移，C原子的硬度增强。因此这里应该由氧去进攻。

当然这里 $d-p\pi$ 相互作用也起了贡献。

其实仅就题目而言，即使考虑不到软硬酸碱理论的层面，我们也可以通过推断法得到结果。题目已经告诉我们：硝基甲烷负离子是一种氧化剂，如果是碳去进攻，接下来便无法写出合理的氧化反应。

得到了中间体之后，我们来分析下一步反应应如何进行。由于底物与一级醇同类，那么便可以很自然地联想到该反应的氧化产物应该是醛。生成醛需要脱氢，在这里，我们可以选择用氧拔氢，得到甲醛肟；或是用碳拔氢，得到亚硝基甲烷。实际上，在亚硝基甲烷与甲醛肟的互变异构中，平衡有利于甲醛肟，但在本题中，这两个答案都是可接受的。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 共振式 | 硝基甲烷负离子的共振结构，氮原子形式电荷为正 |
| 软硬酸碱理论 | O原子端较硬，C原子端较软 |
| 两位负离子 | 硝基甲烷负离子可在O或C上发生反应 |
| 有机氧化剂 | 硝基甲烷负离子作为温和的有机氧化剂 |
| 氧化反应 | 一级醇等价物被氧化为醛 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 共振式画错 | 氮原子形式电荷必须为正 |
| 进攻位点选错 | 由软硬酸碱理论，O原子端进攻较硬的苄基碳 |
| 氧化产物写错 | 产物为醛（甲醛肟或亚硝基甲烷） |
| 忽略互变异构 | 甲醛肟与亚硝基甲烷存在互变异构 |
| 未写出所有产物 | 需写出中间体和最终产物 |
