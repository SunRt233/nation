# 题-029-10-2：反应势能图与中间体

## 题目
> **来源**：[[07-资料提炼/提炼-第29届初赛试题解析|第29届初赛第10题]]（原始文件：[[mineru/02-真题解析/29届初赛试题解析]]）

在室温下，A 可以缓慢地转化为 B。画出此转换过程的反应势能示意图和中间体的结构简式。

## 解析

A 到 B 的转化通过正离子中间体来完成，该正离子的结构可表示为烯丙基正离子：

![[mineru/02-真题解析/29届初赛试题解析_images/5d98180034184a77567c2f4d45bd64e29f7ea585c08d64423af6f991c285aa1f.jpg]]

这里为了方便观察，也将A和B的结构画出。

A 的势能要高于 B 的势能，但是 A 到中间体的活化能应低于中间体到 B 的活化能，同时，中间体的势能要高于 A 和 B 的势能。反应势能图如下：

![[mineru/02-真题解析/29届初赛试题解析_images/364b2f6790c77a7c59313955f4ccff0b4a50ca9dbd8abb55161ddb40de5add42.jpg]]

A（1,2-加成产物，动力学产物）势能较高，B（1,4-加成产物，热力学稳定产物）势能较低。A 可以缓慢转化为 B，说明 A 到中间体的能垒较低，而中间体到 B 的能垒也较低，使得转化可以进行。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 反应势能图 | 画出反应物、中间体、产物的相对能量和活化能 |
| 烯丙基正离子 | 1,3-丁二烯加成的中间体为烯丙基正离子 |
| 动力学产物 | A为1,2-加成产物，势能较高 |
| 热力学产物 | B为1,4-加成产物，势能较低 |
| 活化能 | A→中间体的活化能低于中间体→B的活化能 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 势能高低画反 | A（动力学产物）势能高于B（热力学产物） |
| 中间体能量位置错误 | 中间体能量应高于A和B |
| 活化能大小错误 | A→中间体的活化能应低于中间体→B |
| 忽略中间体结构 | 中间体为烯丙基正离子，需画出其结构 |
| 未标注A和B | 势能图上需明确标注A和B对应的位置 |
