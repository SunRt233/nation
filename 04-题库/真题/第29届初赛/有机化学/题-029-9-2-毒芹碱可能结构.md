# 题-029-9-2：毒芹碱可能结构

## 题目
> **来源**：[[07-资料提炼/提炼-第29届初赛试题解析|第29届初赛第9题]]（原始文件：[[mineru/02-真题解析/29届初赛试题解析]]）
> **小问关联**：本文件为第(9-2)小问；其余小问见 [[题-029-9-1-毒芹碱结构推断]]

从 Hofmann 消除反应的结果可以确定 coniine 具有哪些结构特征？并以此结果画出你所推断的 coniine 所有可能的结构简式。

## 参考答案

Coniine 的结构特征：**环状的二级胺**。

Coniine 所有可能的结构共有 3 个：

![[mineru/02-真题解析/29届初赛试题解析_images/efdd9abb9fb2b3b7d880baf28d3d4c703fa37a621882902b85e1a347afbe09e0.jpg]]

图中的两个五元环结构并不互为对映异构体，不能用（±）同时表示两个五元环的结构。

## 解析

Hofmann 消除反应是一个四级铵盐在碱性条件下发生消除的反应。4S-(N,N-二甲基)-辛-7-烯-4-胺的化学式为 $\mathrm{C}_{8}\mathrm{H}_{17}\mathrm{N}$，与 coniine 的化学式对比可以得知，coniine 经历了两次甲基化得到四级铵盐，而发生 Hofmann 消除反应后得到的产物中双键和氨基位于同一分子内，可以判断出 coniine 是一个**环状的二级胺**。

再根据 Hofmann 消除反应逆推，氮原子有两个可能的连接位点，所以化合物 coniine 的结构可能有**五元环和六元环两类**，其中五元环产生了一个手性碳。

如下图所示，coniine 所有可能的结构共有 3 个：
- 一个六元环结构
- 两个五元环结构（不互为对映异构体）

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| Hofmann消除 | 四级铵盐在碱性条件下发生消除反应 |
| 结构推断 | 通过消除产物逆推原始结构 |
| 环状胺 | coniine为环状二级胺 |
| 逆合成分析 | 从产物结构逆推反应物可能的结构 |
| 手性 | 五元环结构产生手性碳 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 误认为coniine为一级胺 | 经历两次甲基化，说明原为二级胺 |
| 忽略环状结构 | 消除产物中双键和氨基在同一分子内，说明为环状胺 |
| 五元环结构数错误 | 氮原子有两个连接位点，产生两个不同的五元环结构 |
| 混淆对映异构体 | 两个五元环结构不互为对映异构体 |
| 六元环结构遗漏 | 氮原子也可形成六元环结构 |
