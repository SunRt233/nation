## 解析

不对称的 1,3-二烯和氯化氢的加成反应要稍微复杂一点，但是其原理并没有改变，只需在原有的基础上多考虑一个马氏规则（本质是正离子的稳定性）即可。其反应历程如下图所示：

![[mineru/02-真题解析/29届初赛试题解析_images/5552f15e72d485466dd700e7f54726d326052c59fccee8ed67550d91d9c2a088.jpg]]

从结果可以看出，产物 2 和 3 是 1,2-加成产物，产物 1 和 4 是 1,4-加成产物。这里我们还要注意到产物 1 和产物 2 是同一个化合物，因此在不考虑手性的情况下该反应的产物只有 3 个。

除此之外，甲基取代对反应结果也会产生一定的影响。甲基有给电子能力，这导致有甲基取代的双键的电子云密度更高，和 $\mathrm{H}^{+}$ 结合更快，因此中间体 b 的比例会略高于 a。有些同学提出：a 是次要中间体，那么化合物 1 和 2 应该和反马氏规则的产物一样被忽略。在这里需要注意：并不是所有的次要产物都应该被忽略，这要视该次要产物的含量甚至是重要性而定。本题中，我们可以通过已学的知识来简单判断一下：马氏产物中间体是一个烯丙基正离子，我们知道烯丙基正离子的稳定性远大于普通碳正离子，所以产物几乎都是马氏产物，或者说几乎没有反马氏产物；甲基是一个给电子基团，但只是一个比较弱的给电子基团，它并不能显著提高双键的电子云密度，因此 a 的比例低于 b，但也仅仅是略低于 b，产物 1 和 2 自然也就不能忽略。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 马氏规则 | H⁺优先加成到电子云密度更高的双键碳上 |
| 1,2-加成 | 动力学产物，H和Cl加成在相邻碳上 |
| 1,4-加成 | 热力学产物，H和Cl加成在1,4位 |
| 烯丙基正离子 | 加成反应的中间体，可发生1,2或1,4加成 |
| 区域选择性 | 甲基的给电子效应影响加成位置的选择性 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 忽略产物1和2是同一化合物 | 1和2是同一化合物，产物总数为3个 |
| 忽略次要产物 | 次要产物a的比例虽低，但不能完全忽略 |
| 混淆马氏和反马氏产物 | 甲基是弱给电子基，主要产物仍为马氏产物 |
| 未考虑1,4-加成 | 除1,2-加成外，还需考虑1,4-加成产物 |
| 立体化学考虑过多 | 题目要求不考虑立体化学 |
