# 题-029-9-1：毒芹碱结构推断

## 题目
> **来源**：[[07-资料提炼/提炼-第29届初赛试题解析|第29届初赛第9题]]（原始文件：[[mineru/02-真题解析/29届初赛试题解析]]）
> **小问关联**：本文件为第(9-1)小问；其余小问见 [[题-029-9-2-毒芹碱可能结构]]

Coniine 是一种有毒生物碱，可以通过麻痹呼吸系统导致死亡，致死量小于 0.1 g。公元前 399 年苏格拉底就是由于饮用了含 coniine 的混合物而被毒死的。Hofmann 在 1881 年确定 coniine 分子式为 $\mathrm{C}_{8}\mathrm{H}_{17}\mathrm{N}$。将 coniine 彻底甲基化后接着进行 Hofmann 消除反应的产物为 4S-(N,N-二甲基)-辛-7-烯-4-胺。

画出 $4S$-(N,N-二甲基)-辛-7-烯-4-胺及其对映体的结构简式。

## 解析

本小题考查基本的有机化合物命名和旋光性判断。4S-(N,N-二甲基)-辛-7-烯-4-胺的主链上有8个碳，其中7、8号位为双键，4号位取代有二甲氨基。根据Cahn-Ingold-Prelog规则判断其4号碳的手性，可以得coniine与其对映体的结构简式。

Coniine，中文名毒芹碱，以盐的形式存在于毒芹等植物中，为剧毒品，但在小量使用时具有抗痉挛的生理作用。Coniine 是第一个人工合成的生物碱，最初得到的是 coniine 的外消旋混合物，可用 (D-) 酒石酸重结晶分离为光学纯的产物。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 有机化合物命名 | 根据IUPAC命名规则写出结构简式 |
| 手性判断 | 利用Cahn-Ingold-Prelog规则判断R/S构型 |
| 对映体 | 画出4S构型的对映体（4R构型） |
| 生物碱 | 毒芹碱的结构和性质 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 主链选择错误 | 主链为辛烯，双键在7号位 |
| 手性判断错误 | 根据CIP规则正确判断4号碳的S构型 |
| 对映体画错 | 对映体为4R构型，所有手性中心同时反转 |
| 忽略双键位置 | 双键在7-8号碳之间 |
| 氨基位置错误 | 二甲氨基在4号位 |
