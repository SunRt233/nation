# 题-029-11-1：托品酮与溴苄反应

## 题目

## 解析

本小题题目所给的条件不多，但对于解题来说已经非常充分。首先对底物进行分析，托品酮含有一个三级氨基，而溴苄则是一个常见的亲电试剂。因此这里发生 $S_N2$ 反应：三级胺进攻溴苄，产物是四级铵盐。但是，为什么产物却有 A 和 B 两个呢？观察一下托品酮的结构：

![[mineru/02-真题解析/29届初赛试题解析_images/3ad389fb71f0ba6379c0e97f033a9038d768f881b0ab60e05634da56370a2669.jpg]]

虽然托品酮分子的两个构象都具有对称面，没有手性，但反应中心三级胺两侧的化学环境是不同的。由于胺的孤对电子可以自由翻转，因此托品酮在进攻溴苄时便会有两个方向，自然产物会有 A 和 B 两个。

下一问："A 和 B 两个结构可以在碱性条件下相互转化"。我们很自然地联想到这可以类比于一些光学纯产物外消旋化的过程，其基本原理无非有一个共同的中间体，并与两个对映异构体形成可逆平衡。

因此，我们需要找到这样一个中间体，与两个产物都能形成平衡。分析 A 和 B 的结构，与托品酮相比，酮羰基没有变化，而三级胺变成了四级铵盐。由于反应条件是碱性而无亲核试剂，因此 A 和 B 难以通过脱去苄基回到托品酮。我们需要探索其他路径。观察酮羰基 $\alpha-H$：由于同时受羰基和四级铵盐的影响，羰基 $\alpha-H$ 的酸性大大增强。若消除这个氢，便可得到一个 $\alpha,\beta-$ 不饱和酮——这就是连接 A 和 B 的共同中间体。

本题中反应并不复杂，主要考查的是同学们是否具有基本的立体化学概念以及对反应过程的立体思考能力。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| SN2反应 | 三级胺对溴苄的亲核取代 |
| 立体化学 | 托品酮胺基两侧化学环境不同，产生两个产物 |
| 四级铵盐 | 产物为四级铵盐，具有不同的立体构型 |
| 烯醇负离子 | 碱性条件下羰基α-H被拔除，形成烯醇负离子 |
| 互变异构 | 烯醇负离子与α,β-不饱和酮形成平衡 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 认为发生C-烷基化 | 在碱性条件下，N-烷基化优先于C-烷基化 |
| 中间体找错 | 中间体为α,β-不饱和酮，不是托品酮本身 |
| 忽略胺的翻转 | 胺的孤对电子可自由翻转，导致两个进攻方向 |
| A和B结构画错 | A和B是N-烷基化的非对映异构体 |
| 未考虑α-H酸性 | 四级铵盐和羰基共同增强α-H的酸性 |
