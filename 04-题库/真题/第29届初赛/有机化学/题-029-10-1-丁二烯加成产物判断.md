# 题-029-10-1：丁二烯加成产物判断

## 题目
> **来源**：[[07-资料提炼/提炼-第29届初赛试题解析|第29届初赛第10题]]（原始文件：[[mineru/02-真题解析/29届初赛试题解析]]）
> **小问关联**：本文件为第(10-1)小问；其余小问见 [[题-029-10-2-反应势能图与中间体]]、[[题-029-10-3-不对称二烯加成产物]]

当溴与 1,3-丁二烯在环己烷溶液中发生加成反应时，会得到两个产物 A 和 B（不考虑立体化学）。在 $-15^{\circ}C$ 时 A 和 B 的比例为 62:38；而在 $25^{\circ}C$ 时 A 和 B 的比例为 12:88。

画出化合物 A 和 B 的结构简式；并确定哪个为热力学稳定产物，哪个为动力学产物。

## 参考答案

A 为 **1,2-加成产物**（动力学产物），B 为 **1,4-加成产物**（热力学稳定产物）。

## 解析

溴与 1,3-丁二烯在环己烷溶液中发生加成反应时，会得到两个产物：1,2-加成产物和 1,4-加成产物。其反应机理如下：

![[mineru/02-真题解析/29届初赛试题解析_images/b3ddf18ba5d84068cc550a7d003da2e4ec37de3a49ce002d1f9c07af6d6cdc52.jpg]]

1,4-加成产物更加稳定，但是反应的活化能较 1,2-加成产物更高。在这个反应中，1,2-加成产物是动力学产物，而 1,4-加成产物是热力学稳定产物。在较低温度时，反应的活化能不容易克服，动力学产物比例相对较高；而高温时，活化能容易克服，更易生成热力学稳定产物。因此 A 是动力学产物，为 1,2-加成产物；B 是热力学稳定产物，为 1,4-加成产物。同时，从 10-2 的 "A 可以缓慢地转化为 B" 也可以看出 A 是动力学产物，B 是热力学稳定产物。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 1,2-加成 | Br₂加成在相邻的两个碳上，生成动力学产物 |
| 1,4-加成 | Br₂加成在1,4位，生成更稳定的热力学产物 |
| 动力学控制 | 低温有利于动力学产物（活化能低） |
| 热力学控制 | 高温有利于热力学产物（产物更稳定） |
| 烯丙基正离子 | 加成反应的中间体为烯丙基正离子 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| A和B对应关系错误 | 低温比例高的A是动力学产物（1,2-加成），高温比例高的B是热力学产物（1,4-加成） |
| 忽略温度影响 | 温度是判断动力学/热力学产物的关键 |
| 混淆产物稳定性 | 1,4-加成产物更稳定（共轭体系），但活化能更高 |
| 未考虑烯丙基正离子 | 中间体为烯丙基正离子，可发生1,2或1,4加成 |
