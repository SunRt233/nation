# 题-029-7-1：钒配合物氧化数与配位数

## 题目
> **来源**：[[07-资料提炼/提炼-第29届初赛试题解析|第29届初赛第7题]]（原始文件：[[mineru/02-真题解析/29届初赛试题解析]]）
> **小问关联**：本文件为第(7-1)小问；其余小问见 [[题-029-7-2-铬氧乙酸配合物结构]]

在如下反应中，反应前后钒的氧化数和配位数各是多少？N-N键长如何变化？

![[mineru/02-真题解析/29届初赛试题解析_images/c4c1a14f39aab7f42b32ec98d2c8f1d41c2da2ed5621bd8a20825628180dae7a.jpg]]

## 参考答案

**氧化数：**
- 底物分子中：钒的氧化数为 **+3**
- 产物分子中：钒的氧化数为 **+1**

**配位数：**
- 底物分子中：配位数为 **7**（2个Cl、1个N、1个P各提供1对电子，环戊二烯负离子Cp提供3对电子）
- 产物分子中：配位数为 **6**（1个N、1个P各提供1对电子，Cp提供3对电子，炔烃提供1对电子）

**N-N键长变化：**
N-N键长**增长**（键级降低）。

## 解析

此题有一定难度，主要考查同学们对配位化学一些基础知识的掌握及根据对称性和一些其他条件绘出分子结构的推演能力。

**氧化数：**
底物分子中，胺和膦是中性配体，Cl 和环戊二烯是 -1 价配体，所以氧化数为：$1 + 1 + 1 = +3$。产物分子中，炔烃和 $\mathrm{N}_{2}$ 都是中性配体，所以氧化数为 +1。

**配位数：**
底物分子中 2 个 Cl 原子、1 个 N 原子和 1 个 P 原子各提供 1 对电子和钒配位，6 π 电子的环戊二烯负离子可以提供 3 对电子，故配位数为：$2 + 1 + 1 + 3 = 7$。

产物分子中，N 和 P 原子各提供 1 对电子给金属配位，环戊二烯负离子提供 3 对电子，炔烃提供一组 π 键中的电子去配位，而另一组 π 键由于对称性不匹配，即无法"头碰头"地去和金属的轨道成键，因此炔烃只提供一对电子。故配位数为：$1+1+1+3=6$。

**N-N键长变化：**
判断 N-N 键长变化从键级入手：键级增大，键变强，键长缩短；键级减小，键变弱，键长增大。金属钒除了提供空轨道来和 $\mathrm{N}_{2}$ 形成配位键外，它本身还可以用含有电子的 d 轨道与 $\mathrm{N}_{2}$ 的反键轨道形成反馈 π 键。所以 $\mathrm{N}_{2}$ 的成键轨道电子云密度降低，反键轨道上电子云密度升高，N-N 键的总键级降低，故其键长应该增长。

反应产物的结构由单晶 X 射线衍射确定。配位后的 N-N 键长为 121.2(8) pm，相比于单纯的氮气分子以及端基配位的 V-N₂ 中的 N-N 键长 [109.77 pm 与 111.11(17) pm] 有所增加，同时 V-N 键长介于单键和双键之间，这些都有力地证明了反馈 π 键的存在。

## 知识点映射

| 知识点 | 说明 |
|:---|:---|
| 氧化数 | 根据配体电荷确定金属氧化数 |
| 配位数 | 计算配体提供的电子对数确定配位数 |
| 反馈π键 | 金属d电子向N₂反键轨道反馈，削弱N-N键 |
| 氮气活化 | 通过配位和反馈π键活化惰性的N₂分子 |
| 环戊二烯配体 | Cp⁻作为5配位（提供3对电子）的配体 |

## 易错分析

| 常见错误 | 正确思路 |
|:---|:---|
| 氧化数计算错误 | 注意Cl⁻和Cp⁻为-1价配体，胺和膦为中性配体 |
| 配位数计算错误 | Cp⁻提供3对电子（不是1对或5对） |
| 炔烃配位电子数错误 | 炔烃侧向配位只提供1对电子 |
| N-N键长变化判断错误 | 反馈π键使N-N键级降低，键长增长 |
| 忽略对称性 | 炔烃的另一组π键因对称性不匹配不能配位 |
