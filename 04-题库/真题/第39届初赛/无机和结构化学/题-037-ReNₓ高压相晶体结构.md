# 题-037：高压相 $\mathrm{ReN}_x\cdot y\mathrm{N_2}$ 的晶体结构分析

## 题目

> 来源：[[07-资料提炼/提炼-第39届初赛试题解析|第39届初赛第7题]]（原始文件：[[06-外部资料导入/真题/39初带解析]]）

在高压下，激光加热金属 Re 与 $\mathrm{N_2}$ 反应生成正交晶系化合物 $\mathrm{ReN}_x\cdot y\mathrm{N_2}$，晶胞参数 $a = 3.4475\ \mathrm{Å}$，$b = 6.4911\ \mathrm{Å}$，$c = 6.0483\ \mathrm{Å}$。

结构特点：
- N1：形成共平面一维链状结构，与 Re 配位形成四边形孔道
- N2：以 $\mathrm{N_2\!-\!N_2}$ 二聚体形式填充于孔道中
- N2 之一坐标为 (0.336, 0.5, 0)，N2–N2 键长与 $\mathrm{N_2}$ 分子接近

（⚠️ 晶体结构图片依赖原题）

![[06-外部资料导入/真题/images/3f97c278d3045d4a18b1549008e400832ca18508c284e3b6c62434e87edd7839.jpg]]

![[06-外部资料导入/真题/images/38ce6a6b649cc3cf6df159b9c20adfe32d6aec335ad38cf7826e7b0d42342ca3.jpg]]

**7.3.1** 写出 $\mathrm{ReN}_x\cdot y\mathrm{N_2}$ 中 $x$ 和 $y$ 的数值。

**7.3.2** 写出 Re 的配位数。

**7.3.3** 该结构中 Re 的价态为 +4，画出最符合图中结构特点的 N1 一维链的 Lewis 结构式。

⚠️ 画图子题。

**7.3.4** 通过计算确定 N2–N2 二聚体的键长为多少 Å（保留两位小数）。

## 参考答案

**7.3.1** **$x = 8$，$y = 1$** → $\mathrm{ReN_8\cdot N_2}$

- 每个 Re 被 8 个 N1 包围（形成两个相互垂直的矩形网格）
- 每个 N1 仅与 1 个 Re 相连 → Re:N1 = 1:8
- 每个孔道放 1 个 $\mathrm{N_2\!-\!N_2}$ 二聚体（含 2 个 N2）→ $y=1$

![[06-外部资料导入/真题/images/22f2ee9e3f8b0918339905a0c9608fe5cd22507bf4d483b2c1181612109aedcd.jpg]]

**7.3.2** Re 的配位数为 **8**。

**7.3.3** ⚠️ 依赖图片。

N1 平均氧化数为 −1/2（Re 为 +4，N2 为 0 → 4+ / 8 = 1/2），形式电荷 −1/2 → 一半 N 带 −1，一半 N 带 0。

N1 链为 π 电子离域的共轭体系，Lewis 结构交替出现 $\mathrm{N\!=\!N}$ 和 $\mathrm{N^-\!-\!N}$：

$$-\![\ddot{\mathrm{N}}\!=\!\ddot{\mathrm{N}}\!-\!\ddot{\mathrm{N}}^-\!-\!\ddot{\mathrm{N}}]_n\!- \longleftrightarrow -\![\ddot{\mathrm{N}}^-\!-\!\ddot{\mathrm{N}}\!=\!\ddot{\mathrm{N}}\!-\!\ddot{\mathrm{N}}]_n\!-$$

**7.3.4** N2 坐标 (0.336, 0.5, 0)，对称操作给出另一 N2 坐标 (−0.336, 0.5, 0) = (0.664, 0.5, 0)。

两种可能的 N2–N2 距离：
- 沿 $a$ 方向（较短）：$d = (1 - 2 \times 0.336)a = 0.328 \times 3.4475 = 1.13\ \mathrm{Å}$
- 沿 $a$ 方向越过晶胞边界：$d = 2 \times 0.336 \times 3.4475 = 2.32\ \mathrm{Å}$

$\mathrm{N_2}$ 分子键长 ~1.10 Å → 取 **1.13 Å**。

## 解题思路
1. 7.3.1：从配位关系推导 Re:N1 = 1:8（每个 Re 被 8 个 N1 包围，每个 N1 只连 1 个 Re）
2. 7.3.2：直接读取配位数 = 8
3. 7.3.3：由价态守恒（Re⁴⁺ + N2⁰ → N1⁻¹/²）推断 Lewis 结构
4. 7.3.4：利用原子坐标和晶胞参数计算键长（注意两种可能，取较短的）

## 易错分析
- 把 $y$ 算成 2（二聚体含 2 个 N2 原子，但 N2 本身是分子）
- 把 N2 当成 N₂（分子），键长计算按分子间距离
- N1 平均氧化数算错（Re:+4，N2:0 → N1 平均 −4/8 = −1/2）
- 键长计算忘记检查两种对称等效位置（晶胞内 vs 跨晶胞）
