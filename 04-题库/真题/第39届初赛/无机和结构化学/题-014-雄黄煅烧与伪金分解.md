# 题-014：雄黄煅烧制砷酸钾与伪金分解温度

## 题目

> 来源：[[07-资料提炼/提炼-第39届初赛试题解析|第39届初赛第1题]]（原始文件：[[06-外部资料导入/真题/39初带解析]]）

**1.2** 南北朝时期苏元明将雄黄（$\mathrm{As_4S_4}$）与草木灰（主要成分为 $\mathrm{K_2CO_3}$）在空气中一同煅烧，得到熔融的砷酸钾和四硫代砷酸钾。写出此反应方程式。

**1.3** 孙思邈《太清丹经要诀》描述雄黄"变"锡的过程：温度过低雄黄不能熔化，温度过高伪金 A 发生分解，形成非金属单质 B 和化合物 C（各物质计量比相同）。

**1.3.1** 写出 A、B 和 C 的化学式。

**1.3.2** 根据下表数据，通过计算确定分解反应的温度范围（设 $\Delta_{\mathrm{f}}H_{\mathrm{m}}^{\ominus}$ 和 $S_{\mathrm{m}}^{\ominus}$ 不随温度变化）。

已知：$\mathrm{As_4S_4(s) \to As_4S_4(l)}:\ \Delta_{\mathrm{r}}H_{\mathrm{m}}^{\ominus} = 12.14\ \mathrm{kJ\ mol^{-1}},\ \Delta_{\mathrm{r}}S_{\mathrm{m}}^{\ominus} = 20.92\ \mathrm{J\ mol^{-1}\ K^{-1}}$

| 分解温度附近 | A | B | C |
|:---|:---|:---|:---|
| $\Delta_{\mathrm{f}}H_{\mathrm{m}}^{\ominus}$ / (kJ mol⁻¹) | -116.14 | 76.84 | -75.69 |
| $S_{\mathrm{m}}^{\ominus}$ / (J mol⁻¹ K⁻¹) | 160.32 | 135.36 | 136.10 |

## 参考答案

**1.2** As 从 +2 氧化为 +5（$\mathrm{K_3AsO_4}$ 和 $\mathrm{K_3AsS_4}$ 中均为 +5），$\mathrm{O_2}$ 为氧化剂：

$$\mathrm{As_4S_4} + 6\mathrm{K_2CO_3} + 3\mathrm{O_2} \longrightarrow 3\mathrm{K_3AsO_4} + \mathrm{K_3AsS_4} + 6\mathrm{CO_2}$$

**1.3.1** Sn 的最高价硫化物 $\mathrm{SnS_2}$ 俗称"金粉"（彩金）；高温分解为 $\mathrm{SnS}$ 和 $\mathrm{S}$：

$$\mathbf{A} = \mathrm{SnS_2},\quad \mathbf{B} = \mathrm{S},\quad \mathbf{C} = \mathrm{SnS}$$

**1.3.2** 分解反应：$\mathrm{SnS_2 \longrightarrow SnS + S}$

$$\begin{aligned}
\Delta_{\mathrm{r}}H_{\mathrm{m}}^{\ominus} &= (-75.69) + 76.84 - (-116.14) = 117.29\ \mathrm{kJ\ mol^{-1}} \\
\Delta_{\mathrm{r}}S_{\mathrm{m}}^{\ominus} &= 136.10 + 135.36 - 160.32 = 111.14\ \mathrm{J\ mol^{-1}\ K^{-1}}
\end{aligned}$$

$T > \dfrac{\Delta_{\mathrm{r}}H_{\mathrm{m}}^{\ominus}}{\Delta_{\mathrm{r}}S_{\mathrm{m}}^{\ominus}} = \dfrac{117.29}{0.11114} = 1055\ \mathrm{K}$

温度下限（雄黄熔化）：$T' > \dfrac{12.14}{0.02092} = 580\ \mathrm{K}$

**温度范围：580 K ~ 1055 K**

## 知识点映射
- [[砷]] — 雄黄氧化煅烧、砷的 +2→+5 氧化
- [[锡]] — SnS₂（金粉）的合成与热分解
- [[平衡常数]] — $\Delta G = \Delta H - T\Delta S$ 判断分解温度
- [[氧化还原反应]] — As 的氧化数变化

## 解题思路
1. 1.2：确定 As 价态变化（+2→+5），O₂ 为氧化剂，配平
2. 1.3.1：联想 Sn 的硫化物颜色——SnS₂ 金黄色（金粉）
3. 1.3.2：计算 $\Delta_r H$ 和 $\Delta_r S$，求 $T = \Delta H/\Delta S$ 临界温度

## 易错分析
- 1.2 中 As 氧化数判断错误（As₄S₄ 中 S 为 -2，则 As 为 +2）
- 忘记雄黄熔化给出的是温度下限数据
- 混淆 $\Delta_f H$ 和 $\Delta_r H$ 的计算方向
