# 题-019：XeFₓ 晶体的化学组成推导

## 题目

> 来源：[[07-资料提炼/提炼-第39届初赛试题解析|第39届初赛第5题]]（原始文件：[[06-外部资料导入/真题/39初带解析]]）

惰性气体在特定条件下可以形成各种化学键，其中以 Xe 形成的化合物最为常见。三种 $\mathrm{XeF}_x$（A、B 和 C）的晶胞结构如下图所示，其中大球为 Xe，小球为 F。

**写出三种晶体的化学组成。**

## 参考答案

根据原子在晶胞中的位置计数（顶点 1/8、棱 1/4、面 1/2、体心 1）：

- **A（$\mathrm{XeF_2}$）**：晶胞中含 2 个 $\mathrm{XeF_2}$ 分子，化学组成 $\mathrm{XeF_2}$
- **B（$\mathrm{XeF_4}$）**：晶胞中含 2 个 $\mathrm{XeF_4}$ 分子，化学组成 $\mathrm{XeF_4}$
- **C（$\mathrm{XeF_3}$）**：晶胞中含 2 个 $\mathrm{XeF_4}$ 和 2 个 $\mathrm{XeF_2}$ 分子，化学组成可写为 $\mathrm{XeF_3}$ 或 $\mathrm{XeF_2 \cdot XeF_4}$

对应已知化合物：A = $\mathrm{XeF_2}$（直线形），B = $\mathrm{XeF_4}$（平面正方形），C = $\mathrm{XeF_2}$ 与 $\mathrm{XeF_4}$ 的共晶

- [[VSEPR理论]] — Xe 氟化物的几何构型

## 解题思路
1. 识别晶胞中的分子类型（$\mathrm{XeF_2}$ 直线形 vs $\mathrm{XeF_4}$ 平面正方形）
2. 按原子位置类型计数：顶点×1/8 + 棱×1/4 + 面×1/2 + 内部×1
3. 注意 C 晶胞中含两种不同分子

## 易错分析
- 不同位置的占据率混淆（顶点 vs 棱 vs 面 vs 体心）
- 分子晶体不跨晶胞形成多聚结构，计数的原子即属同一分子
- C 中未注意存在两种不同分子
