# 题-036：全氮正离子 $\mathrm{N_5^+}$ 的合成与结构分析

## 题目

> 来源：[[07-资料提炼/提炼-第39届初赛试题解析|第39届初赛第7题]]（原始文件：[[06-外部资料导入/真题/39初带解析]]）

多氮化合物是重要的高含能材料。

**7.1** 常压下合成全氮正离子盐 B：
- $\mathrm{N_2F_2}$（顺式）与 $\mathrm{AsF_5}$ 按 1:1 混合 → 白色固体 A
- A 溶于无水 HF，降温，通 $\mathrm{HN_3}$ 气体 → 缓慢升温至 −78°C 反应 3 天 → B
- B 中含 V 形全氮正离子，遇水剧烈反应，放出两种气体，摩尔比 10:1

**7.1.1** 写出 A 和 B 中正离子的化学式。

**7.1.2** 画出 B 中全氮正离子的 Lewis 结构式，标出每个氮原子的形式电荷。

⚠️ 画图子题。

**7.1.3** 写出 B 中全氮正离子与水反应的化学方程式。

**7.2** 另一条合成路线：
- $\mathrm{NF_3} + \mathrm{O_2}$ 放电 → C
- C + $\mathrm{SbF_5}$ → D
- D + 过量 $\mathrm{HN_3}$ → 与 B 相同的全氮正离子
- D + $\mathrm{HN_3}$（1:1） → E（正离子有顺反异构体）
- E 加热至 50–60°C → 白色固体 + 两种气体（摩尔比 1:4）

**7.2.1** 写出 C 和 D 的化学式及含氮部分的几何结构。

**7.2.2** 画出 E 中正离子反式异构体的 Lewis 结构式。

⚠️ 画图子题。

**7.2.3** 写出 D 与过量 $\mathrm{HN_3}$ 反应生成全氮正离子的反应方程式。

**7.2.4** 写出 E 在 50–60°C 下受热分解的反应方程式。

## 参考答案

**7.1.1**
- A 中正离子：**$\mathrm{N_2F^+}$**（$\mathrm{AsF_5}$ 夺取 $\mathrm{N_2F_2}$ 中的 $\mathrm{F^-}$ → $[\mathrm{N_2F}][\mathrm{AsF_6}]$）
- B 中正离子：**$\mathrm{N_5^+}$**（V 形全氮正离子）

**7.1.2** ⚠️ 依赖图片。$\mathrm{N_5^+}$ 共 24 个价电子，8 根键（4σ + 4π），4 对孤对。V 形骨架，端位 N 带 −1 形式电荷，中心附近 N 带 +1。

**7.1.3**
$$4\mathrm{N_5^+} + 2\mathrm{H_2O} \longrightarrow 10\mathrm{N_2} + \mathrm{O_2} + 4\mathrm{H^+}$$

或含对离子：
$$4\mathrm{N_5AsF_6} + 2\mathrm{H_2O} \longrightarrow 10\mathrm{N_2} + \mathrm{O_2} + 4\mathrm{HAsF_6}$$

**7.2.1**
- C：**$\mathrm{NOF_3}$**（N:+3 → N:+5 氧化），含氮部分为**四面体**
- D：**$[\mathrm{NOF_2}][\mathrm{SbF_6}]$**，含氮部分为**平面三角形**（$\mathrm{NOF_2^+}$）

**7.2.2** ⚠️ 依赖图片。E = $[\mathrm{N_3NOF}][\mathrm{SbF_6}]$。正离子含 $\mathrm{N=N}$ 双键 → 顺反异构。反式：F 与 $\mathrm{N_3}$ 分处双键两侧。

**7.2.3**
$$[\mathrm{NOF_2}][\mathrm{SbF_6}] + 2\mathrm{HN_3} \longrightarrow [\mathrm{N_5}][\mathrm{SbF_6}] + \mathrm{N_2O} + 2\mathrm{HF}$$

**7.2.4**
$$3[\mathrm{N_3NOF}][\mathrm{SbF_6}] \xrightarrow{50-60^\circ\mathrm{C}} 3[\mathrm{NO}][\mathrm{SbF_6}] + 4\mathrm{N_2} + \mathrm{NF_3}$$

## 知识点映射
- [[杂化轨道理论]] — Lewis 结构书写、形式电荷计算
- [[氧化还原反应]] — $\mathrm{NF_3 \to NOF_3}$ 的氧化（N: +3 → +5）
- [[VSEPR理论]] — $\mathrm{NOF_3}$（四面体）、$\mathrm{NOF_2^+}$（平面三角形）

## 解题思路
1. **7.1**：从产物逆推——B 与水生成 $\mathrm{N_2} + \mathrm{O_2}$（10:1）→ 推测 $\mathrm{N_5^+}$
2. $\mathrm{AsF_5}$ 是 Lewis 酸 → 夺 $\mathrm{F^-}$ → $\mathrm{N_2F^+}$（A 的正离子）
3. **7.2**：$\mathrm{NF_3 \xrightarrow{O_2}}$ 氧化 → $\mathrm{NOF_3}$ → 再夺 $\mathrm{F^-}$ → $\mathrm{NOF_2^+}$
4. E 的顺反异构来自 $\mathrm{N=N}$ 双键（CIP: F > O, N > lp）

![[06-外部资料导入/真题/images/af15a0980b6e8e5e3c1472764ec4bb1f0eba5bd06a5629f8312626bc9947fff1.jpg]]

5. E 分解 → $\mathrm{NO^+}$（稳定正离子）+ $\mathrm{N_2} + \mathrm{NF_3}$

## 易错分析
- $\mathrm{N_5^+}$ 错写为直线形（实际 V 形，N–N–N 角 ~168°）
- 将 A 的正离子写成 $\mathrm{N_2^+}$（忽略 $\mathrm{F^-}$ 转移后的 $\mathrm{N_2F^+}$）
- C 写成 $\mathrm{NOF}$ 而非 $\mathrm{NOF_3}$（$\mathrm{O_2}$ 应氧化到 +5）
- 7.2.4 分解产物计量比不匹配
