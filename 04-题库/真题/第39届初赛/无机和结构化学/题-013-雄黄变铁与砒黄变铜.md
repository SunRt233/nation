# 题-013：雄黄"变"铁与砒黄"变"铜

## 题目

> 来源：[[07-资料提炼/提炼-第39届初赛试题解析|第39届初赛第1题]]（原始文件：[[06-外部资料导入/真题/39初带解析]]）

雄黄为 $\mathrm{As}_4\mathrm{S}_4$，雌黄为 $\mathrm{As}_2\mathrm{S}_3$，砒黄为 $\mathrm{As}_2\mathrm{O}_3$，硫黄为硫单质。

**1.1.1** 写出雄黄"变"铁的反应方程式。

**1.1.2** 写出在木炭存在下，砒黄"变"铜的反应方程式（提示：含砷 $5\%$ 的砷铜合金是黄色的）。

## 参考答案

**1.1.1** 雄黄与 Fe 反应得到 $\mathrm{FeS}_2$（黄铁矿，愚人金）和 As 单质：

$$\mathrm{As_4S_4} + 2\mathrm{Fe} \longrightarrow 4\mathrm{As} + 2\mathrm{FeS_2}$$

**1.1.2** 木炭为还原剂，将 $\mathrm{As_2O_3}$ 还原为 As，自身氧化为 CO；As 与 Cu 形成含砷 5% 的砷铜合金 $\mathrm{Cu}_x\mathrm{As}$：

$$\mathrm{As_2O_3} + 2x\mathrm{Cu} + 3\mathrm{C} \longrightarrow 2\mathrm{Cu}_x\mathrm{As} + 3\mathrm{CO}$$

由 $\omega(\mathrm{As}) = \dfrac{74.92}{74.92 + 63.55x} = 5\%$，解得 $x \approx 22.4$，化为最简整数比：

$$5\mathrm{As_2O_3} + 224\mathrm{Cu} + 15\mathrm{C} \longrightarrow 2\mathrm{Cu_{112}As_5} + 15\mathrm{CO}$$

## 知识点映射
- [[砷]] — 雄黄(As₄S₄)、砒黄(As₂O₃)的化学反应
- [[锡]] — 黄铁矿 FeS₂（愚人金）的生成
- [[氧化还原反应]] — 贱金属置换 + 碳热还原

## 解题思路
1. 识别矿物别名与化学式
2. 1.1.1：Fe 的硫化物中呈金色的是 FeS₂（黄铁矿），As 还原为单质
3. 1.1.2：C 作还原剂（高温产物 CO），As 与 Cu 形成合金，合金组成由质量分数反推

## 易错分析
- 雄黄写为 $\mathrm{AsS}$ 而非 $\mathrm{As_4S_4}$
- 1.1.2 中不体现砷铜合金的形成，而将 As 写为单质
- C 的氧化产物在高温下应写 CO 而非 CO₂
