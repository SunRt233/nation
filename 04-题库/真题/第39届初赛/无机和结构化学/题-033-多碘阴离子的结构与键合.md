# 题-033：多碘阴离子的结构与键合分析

## 题目

> 来源：[[07-资料提炼/提炼-第39届初赛试题解析|第39届初赛第4题]]（原始文件：[[06-外部资料导入/真题/39初带解析]]）

碘离子可与 $\mathrm{I_2}$ 结合生成 $\mathrm{I_3^-}$，继续加合 $\mathrm{I_2}$ 生成 $\mathrm{I_5^-}$、$\mathrm{I_7^-}$ 等多碘阴离子。

**4.2.1** 直线形的 $\mathrm{I_3^-}$ 是最简单的多碘阴离子。写出位于中心的碘原子杂化形式。

**4.2.2** $\mathrm{I_3^-}$ 的 I–I 键长为 $2.91\ \mathrm{Å}$，而 $\mathrm{I_2}$ 的 I–I 键长为 $2.71\ \mathrm{Å}$。利用某一个典型的有机反应过渡态解释 $\mathrm{I_3^-}$ 键长变长的原因。

**4.2.3** 画出 $\mathrm{I_5^-}$ 所有可能的结构式，并标出每一个碘的形式电荷。

**4.2.4** 按相同方法继续加合 $\mathrm{I_2}$，写出 $\mathrm{I_7^-}$ 可能的异构体数目（不考虑顺反异构或构象异构）。

## 参考答案

**4.2.1** **sp³d 杂化**。中心碘原子周围有 5 个价层电子对：2 个成键对（与端位 I）和 3 个孤对（位于三角双锥的平面位置）→ 直线形。

**4.2.2** 类比 **$\mathrm{S_N2}$ 反应过渡态**：

$\mathrm{I_3^-}$ 可看作"亲核试剂" $\mathrm{I^-}$ 进攻 $\mathrm{I_2}$——$\mathrm{I^-}$ 的电子进入 $\mathrm{I_2}$ 的 $\sigma^*$ 反键轨道（LUMO），形成三中心四电子（3c-4e）键。$\mathrm{S_N2}$ 过渡态是五配位高能态，而 $\mathrm{I_3^-}$ 中 I 原子半径大，可稳定容纳更多配体，使此结构成为稳定物种。

反键轨道被占据 → I–I 键级降低 → **键长变长**。

**4.2.3** ⚠️ 涉及图片，摘要：$\mathrm{I_5^-}$ = $\mathrm{I_3^-}$ + $\mathrm{I_2}$，有两个进攻位点：

- **V 构型**：$\mathrm{I_2}$ 进攻 $\mathrm{I_3^-}$ 的端基碘孤对电子
- **T 构型**：$\mathrm{I_2}$ 进攻 $\mathrm{I_3^-}$ 的中心碘孤对电子

形式电荷标注遵循：端基连接 1 根键 + 3 对孤对 → FC = 0；中心原子连接 2 根键 + 3 对孤对 → FC = −1（依此类推）。

*注：实际已知的 $\mathrm{I_5^-}$ 结构只有 V 构型。*

**4.2.4** **6 种**。

- V 构型 $\mathrm{I_5^-}$（3 个进攻位点）→ A、B、C 三种 $\mathrm{I_7^-}$
- T 构型 $\mathrm{I_5^-}$（4 个进攻位点）→ D、E、F、G 四种 $\mathrm{I_7^-}$
- B 与 D 等同 → 共 **6 种**

*注：实际已知的 $\mathrm{I_7^-}$ 结构只有 C，尽管从经典价键理论看 C 的中心碘带 +2 形式电荷（"最不稳定"），但在实际中它是最稳定的——这说明超价化合物需要分子轨道理论解释。*

## 易错分析
- $\mathrm{I_3^-}$ 中心碘杂化写成 sp³（忽略 5 对电子 → sp³d）
- $\mathrm{I_3^-}$ 中 I–I 键长比 $\mathrm{I_2}$ 长 → 认为"排斥增大"而非"反键轨道被占"
- $\mathrm{I_7^-}$ 数错异构体（忘记去重：V 构型的 B = T 构型的 D）
