# 题-031：聚乙烯亚胺（PEI）的聚合机理

## 题目

> 来源：[[07-资料提炼/提炼-第39届初赛试题解析|第39届初赛第3题]]（原始文件：[[06-外部资料导入/真题/39初带解析]]）

聚乙烯亚胺（PEI）是一种富含氨基的高分子材料。PEI 可以通过氮杂环丙烷在酸性条件下阳离子开环聚合得到，但得到的是非线性的支化分子。

![[06-外部资料导入/真题/images/28b616a436f722146be1b824e977b0dc388b1632b0ae4e895b8423b5241e26e9.jpg]]

![[06-外部资料导入/真题/images/bb4739c86b99bc0ea385d2e2017903d2ec7f1ef4be7bc51834f09dcb9220872f.jpg]]

**3.1** 解释该方法为何不能得到线性 PEI，只能得到支化 PEI？

**3.2** PEI 与核酸分子容易形成外层 PEI、内层核酸的复合物结构，可跨过细胞膜递送核酸。核酸和 PEI 形成复合物主要依靠哪种分子间作用力？

**3.3** ⚠️ 画结构式子题（依赖图片）——线性 PEI 通过噁唑啉在酸性条件下阳离子开环聚合得到。画出反应式中 $\mathbf{A\sim E}$ 的结构式（A、B、C 带正电荷）。

## 参考答案

**3.1** 氮杂环丙烷开环后生成的二级胺 N 原子上仍有质子（与环氧乙烷不同——醚 O 无质子），可继续发生烷基化反应，生成支化结构。此外，氮杂环丙烷 C–N 键中 N 的 p 成分高于 75%（sp³ 杂化），孤对电子 s 成分高于 25%，亲核性弱于常规二级胺 → 聚合物链中的二级胺更容易被烷基化 → 进一步促进分支。

![[06-外部资料导入/真题/images/50f750c8d0d6e3b28738dab121cfa5353d29dc2faabb27cbde6453fa67db7f44.jpg]]

**3.2** **静电相互作用**（非氢键）。

PEI 富含一/二/三级氨基，在生理 pH 下质子化带正电荷；核酸主链磷酸基带负电荷 → 强烈的离子-离子静电吸引（几十到上百 kJ/mol）。氢键能量较弱（几到几十 kJ/mol）、方向性强，非主导。

**3.3** ⚠️ 涉及多步反应式图片，详见原题解析。关键步骤：
1. 噁唑啉 N 结合质子 → 亚胺正离子 **A**（非氧正离子共振式）
2. A 被另一噁唑啉亲核进攻开环 → 二聚体正离子 **B**
3. B 继续被噁唑啉进攻 → 线性聚合物 **C**（有 C1/C2 两种中性结构）
4. C 水解 → **D**（酰胺+酯中间体）
5. D 水解 → **E**（线性 PEI）

## 知识点映射
- [[高分子重复单元]] — 阳离子开环聚合机理
- [[亲核加成]] — 酰基中心的亲核取代（缩聚型聚合）

## 解题思路
1. 3.1：对比环氧乙烷聚合（醚 O 无质子 → 线性）vs 氮杂环丙烷（NH 仍有质子 → 可继续反应）
2. 3.2：判断作用力强度（离子-离子 >> 氢键）
3. 3.3：从噁唑啉质子化开始，逐步追踪亲核开环+水解

## 易错分析
- 3.2 答"氢键"（应得"静电相互作用"，质子化氨基+磷酸根是离子-离子作用）
- 3.3 中 A 写成氧正离子共振式（能量更高，非最优答案）
- 忽略氮杂环丙烷 N 的孤对电子亲核性差异
