# 题-030：动力学同位素效应与自由基合成应用

## 题目

> 来源：[[07-资料提炼/提炼-第39届初赛试题解析|第39届初赛第10题]]（原始文件：[[06-外部资料导入/真题/39初带解析]]）

**10.6** 在使用非手性 NHC–BH₃ 催化剂作用下，底物中含 C–H 和 C–D 两种可选攫氢位点。判断以下哪一个是主产物。

![[06-外部资料导入/真题/images/185f4dc4a1598121c20099220b6834f73c654d4124fee02fa709c9acc8126749.jpg]]

**10.7** 依据本题所提供的信息，完成以下反应式（提示：产物为六元环衍生物，并含有两个芳环取代基）：
 
$$\equiv\mathrm{COOR} + \mathrm{F}\!-\!\!\bigcirc \xrightarrow[\mathrm{NHC\!-\!BH_3\ (10\ equiv)}]{\mathrm{ACCN}}\ ?$$

## 参考答案

**10.6** C–H 键优先断裂 → 主产物为 **B**（攫 H 路径的产物）。

![[06-外部资料导入/真题/images/6462c33a77adbb41548e8eb6bccd641f5b043a9b2b89d5aaeb26b2ba1f510e2c.jpg]]

**原因**：C–H 键解离能 < C–D 键解离能（差值 ~12 kJ mol⁻¹），源于**零点能差异（ZPE）**。

$$E_{\mathrm{ZPE}} = \frac{1}{2}h\nu,\quad \nu = \frac{1}{2\pi}\sqrt{\frac{k}{\mu}}$$

- C–H 约化质量 μ 更小 → ν 更大 → ZPE 更高
- ZPE 越高 → 基态能量越高 → "更接近被拉断" → 键解离能更低
- 典型：$\mathrm{H_3C\!-\!H}: 431\ \mathrm{kJ\ mol^{-1}}$，$\mathrm{D_3C\!-\!D}: 443\ \mathrm{kJ\ mol^{-1}}$

此即**动力学同位素效应（KIE）**。同位素标记实验还用于验证机理：将 N 邻位 α-H 全部替换为 D 后，产物中 D 被完全保留，排除了 $\mathbf{B}^{\bullet*}$ 直接攫 α-H 的路径。

![[06-外部资料导入/真题/images/2a4d3d61605415fc44f63b42c45162a9999be1c54d736bdf2dca848ae3e3b0ae.jpg]]

**10.7** 反应过程：$\mathbf{B}^{\bullet*}$ 对炔加成 → 烯基自由基 → 氢迁移（此处为分子间攫 H 或分子内适当位置攫 H） → 苄基自由基 → 与芳环相连的不饱和键发生自由基环化（6 元环） → β-碎裂消除 → 产物。

产物为含两个芳环取代基的六元环衍生物（$\mathrm{COOR}$ 基团和 $\mathrm{F\!-\!C_6H_4\!-}$ 基团位于环上）。

![[06-外部资料导入/真题/images/2a839b3fa04062e0e66eb8c28bca74518281ccb38ffb57e35d107ee4270ef904.jpg]]

## 知识点映射
- [[自由基]] — 自由基攫氢选择性、C–H vs C–D 键解离能差异
- [[动力学同位素效应]] — ZPE 差异导致 KIE、同位素标记法验证机理
- [[有机合成]] — 自由基环化策略在合成中的应用

## 解题思路
1. 10.6：比较 C–H 和 C–D 的键解离能（ZPE 差异 → C–H 键更弱）
2. 10.7：按照 10.2-10.5 确立的机理模式推断
   - $\mathbf{B}^{\bullet*}$ 加成 → 自由基中间体 → 环化 → β-消除
   - 六元环暗示 6-endo-trig 或 6-exo-trig 环化模式
   - 两个芳环取代基体现于底物的酯基芳环和氟代芳环

## 易错分析
- 混淆 KIE 的两种应用场景（速率差异 vs 标记追踪）
- 将同位素效应归因于键能本身而非 ZPE
- 10.7 中氢迁移路径判断错误（需注意底物中可攫氢的位置）
