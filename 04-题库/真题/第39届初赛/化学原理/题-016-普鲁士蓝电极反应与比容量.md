# 题-016：普鲁士蓝电极反应、颜色变化与比容量

## 题目

> 来源：[[07-资料提炼/提炼-第39届初赛试题解析|第39届初赛第2题]]（原始文件：[[06-外部资料导入/真题/39初带解析]]）

普鲁士蓝类化合物 (PBA) 是由 $\mathrm{CN}^{-}$ 与金属离子配位形成的立方体网络结构。某种 PBA（PBA-1）的化学式为 $\mathrm{KFe_2(CN)_6}$，式量 306.92，具有简单立方结构，相邻 Fe-Fe 距离 $a = 0.510\ \mathrm{nm}$。

**2.1** PBA-1 为蓝色晶体，通过电极反应吸收足量的 $\mathrm{K}^{+}$ 后，转变成白色的普鲁士白。

**2.1.1** 写出生成普鲁士白的电极反应方程式。

**2.1.2** 解释电极反应中化合物颜色变化的原因。

**2.1.3** 计算 PBA-1 作为正极材料的理论比容量，用 $\mathrm{mA\ h\ g^{-1}}$ 表示。

## 参考答案

**2.1.1** PBA-1 为 $\mathrm{KFe^{III}Fe^{II}(CN)_6}$（混合价态），嵌入 1 当量 $\mathrm{K}^{+}$ 同时接受 1 当量电子：

$$\mathrm{KFe^{III}Fe^{II}(CN)_6} + \mathrm{K}^{+} + \mathrm{e}^{-} \longrightarrow \mathrm{K_2Fe_2^{II}(CN)_6}$$

![[06-外部资料导入/真题/images/2d2014e3938e086570129546f8ba4c5a6e613cc8c5b8293456872e94bb96da4a.jpg]]

**2.1.2** 颜色源自 $\mathrm{Fe^{II} \to C\!\equiv\!N \to Fe^{III}}$ 的**电荷转移（CT）跃迁**：
- $\mathrm{CN}^{-}$ 的 C 端是强场配体 → $\mathrm{Fe^{II}}$：$t_{2g}^6 e_g^0$
- N 端是中强场配体 → $\mathrm{Fe^{III}}$：$t_{2g}^3 e_g^2$
- $\mathrm{Fe^{II}}$（给体）→ $\mathrm{Fe^{III}}$（受体）通过氰根形成高效 CT 通道
- CT 跃迁对称性允许，吸收截面强，在红光区宽带吸收 → 呈深蓝色

被还原后全部为 $\mathrm{Fe^{II}}$，CT 通道破坏，不再吸收可见光 → 呈白色。

**2.1.3** 可逆储钾过程：$\mathrm{Fe_2(CN)_6 + 2K^{+} + 2e^{-} \rightleftharpoons K_2Fe_2(CN)_6}$

理论比容量按 $2\ \mathrm{mol\ K^{+}/mol\ Fe_2(CN)_6}$ 计算：

$$\begin{aligned}
C &= \frac{nF}{M} = \frac{2 \times 96485}{306.92} = 628.74\ \mathrm{A\ s\ g^{-1}} \\
&= 628.74 \times \frac{1000}{3600} = 174.65\ \mathrm{mA\ h\ g^{-1}}
\end{aligned}$$

## 知识点映射
- [[Nernst方程]] — 电池比容量计算公式 $C = nF/M$
- [[配合物]] — 普鲁士蓝的混合价态配合物结构
- [[晶体场理论]] — CT 跃迁（电荷转移）解释颜色、强场/中强场配体

## 解题思路
1. 判断 PBA-1 中铁的价态（混合价态 $\mathrm{Fe^{III}Fe^{II}}$）
2. 颜色：d-d 跃迁 vs CT 跃迁 → CT 跃迁对称性允许，吸收更强
3. 比容量：找准可逆电子数（2 mol e⁻/mol 单元）

## 易错分析
- 比容量只算 1 个 $\mathrm{K}^{+}$（得 $87.3\ \mathrm{mA\ h\ g^{-1}}$，部分得分）
- 混淆式量（应使用 $\mathrm{KFe_2(CN)_6}$ 的式量，而非 $\mathrm{K_2Fe_2(CN)_6}$）
- 颜色解释为 d-d 跃迁（实际是 CT 跃迁占主导）
