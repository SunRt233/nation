# 题-018：普鲁士蓝颗粒接触不良比例计算

## 题目

> 来源：[[07-资料提炼/提炼-第39届初赛试题解析|第39届初赛第2题]]（原始文件：[[06-外部资料导入/真题/39初带解析]]）

将质量共计 $0.3198\ \mathrm{ng}$、边长为 $180\ \mathrm{nm}$ 的立方体 PBA-1 颗粒均匀铺洒在电极表面，颗粒与电极接触的情况仅有两种：① 接触良好（扩散控制）、② 接触不良（接触电阻控制）。

在恒定电压 $0.500\ \mathrm{V}$、$t = 10.00\ \mathrm{s}$ 时刻，测得总电流为 $1.38576\ \mathrm{nA}$。

**计算与电极表面接触不良的 PBA-1 颗粒的含量百分比。**

提示：$\dfrac{\mathrm{d}}{\mathrm{d}x}\mathrm{e}^{kx} = k\mathrm{e}^{kx}$

（若前述结果未得出，可使用：$Q_0 = 3.60 \times 10^{-12}\ \mathrm{C}$，$D = 3.00 \times 10^3\ \mathrm{nm^2\ s^{-1}}$，$R = 2.00 \times 10^{13}\ \Omega$）

## 参考答案

**（1）计算颗粒总数**

单颗粒质量：

$$m_{\mathrm{p}} = (L/a)^3 \cdot \frac{M_{\mathrm{r}}/2}{N_{\mathrm{A}}} = (180/0.510)^3 \times \frac{306.92/2}{6.022 \times 10^{23}} = 1.12 \times 10^{-14}\ \mathrm{g}$$

总颗粒数：$N = \dfrac{0.3198 \times 10^{-9}}{1.12 \times 10^{-14}} = 2.85 \times 10^4$ 个

**（2）单颗粒电流**

扩散控制：$I_{\mathrm{diff}} = \left|\dfrac{\mathrm{d}Q}{\mathrm{d}t}\right| = \dfrac{\pi^2 D}{4L^2}Q_0 \exp\!\left(-\dfrac{\pi^2 D}{4L^2}t\right)$

代入 $D = 3.40 \times 10^3\ \mathrm{nm^2\ s^{-1}}$，$L = 180\ \mathrm{nm}$，$Q_0 = 3.52 \times 10^{-12}\ \mathrm{C}$，$t = 10.00\ \mathrm{s}$：

$$I_{\mathrm{diff}} = 0.0684\ \mathrm{pA}$$

接触电阻控制：$I_{\mathrm{ohm}} = \dfrac{U}{R} = \dfrac{0.500}{1.57 \times 10^{13}} = 0.0318\ \mathrm{pA}$

**（3）联立求解**

$$\begin{cases}
x + y = 2.85 \times 10^4 \\
0.0684x + 0.0318y = 1.39 \times 10^3
\end{cases}$$

解得：$x = 1.31 \times 10^4$（接触良好），$y = 1.54 \times 10^4$（接触不良）

接触不良比例：$\dfrac{y}{N} \times 100\% = \dfrac{1.54 \times 10^4}{2.85 \times 10^4} \times 100\% = 54.1\%$

## 知识点映射
- [[Nernst方程]] — 电容量、电流与法拉第常数
- [[扩散动力学]] — 两种控制机制的线性叠加模型

## 解题思路
1. 从晶胞参数求单颗粒质量 → 总颗粒数
2. 扩散控制：$I = \mathrm{d}Q/\mathrm{d}t$（指数函数求导）
3. 接触控制：欧姆定律 $I = U/R$
4. 建立二元方程组解两类颗粒数
5. 求百分比

## 易错分析
- 单颗粒质量计算：化学式中有 2 当量 Fe，式量应除以 2
- 单位换算：$\mathrm{ng \to g}$、$\mathrm{nA \to pA}$
- 扩散电流公式未对指数函数正确求导
