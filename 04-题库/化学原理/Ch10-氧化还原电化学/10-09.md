# 10.9 通过ΔG间接计算EΘ

## 题目

已知 $\mathrm{MnO}_4^- +8\mathrm{H}^{+} + 5\mathrm{e}\Longrightarrow \mathrm{Mn}^{2 + } + 4\mathrm{H}_{2}\mathrm{O}$ $E^{\ominus} = 1.51\mathrm{V}$

$$
\mathrm {M n O} _ {2} + 4 \mathrm {H} ^ {+} + 2 \mathrm {e} \rightleftharpoons \mathrm {M n} ^ {2 +} + 2 \mathrm {H} _ {2} \mathrm {O} \quad E ^ {\ominus} = 1. 2 2 \mathrm {V}
$$

求： $\mathrm{MnO_4^- + 4H^+ + 3e\rightleftharpoons MnO_2 + 2H_2O}$ 的 $E^{\ominus} = ?$

## 参考答案

① $\mathrm { M n O } _ { 4 } ^ { - } + 8 \mathrm { H } ^ { + } + 5 \mathrm { e } = \mathrm { M n } ^ { 2 + } + 4 \mathrm { H } _ { 2 } \mathrm { O }$ $E_{1}^{\ominus} = 1.51\mathrm{V}$   
② $\mathrm { M n O } _ { 2 } + 4 \mathrm { H } ^ { + } + 2 \mathrm { e } = \mathrm { M n } ^ { 2 + } + 2 \mathrm { H } _ { 2 } \mathrm { O }$ $E_{2}^{\ominus} = 1.22\mathrm{V}$

①式一②式，即为所求反应式，则有

$$
\begin{array}{l} \Delta G ^ {\ominus} = \Delta G _ {1} ^ {\ominus} - \Delta G _ {2} ^ {\ominus}, \quad n F E ^ {\ominus} = n _ {1} F E _ {1} ^ {\ominus} - n _ {2} F E _ {2} ^ {\ominus} \\ E ^ {\ominus} = \frac {n _ {1} E _ {1} ^ {\ominus} - n _ {2} E _ {2} ^ {\ominus}}{n} = \frac {(5 \times 1 . 5 1 - 2 \times 1 . 2 2) \mathrm {V}}{3} = 1. 7 0 \mathrm {V} \\ \end{array}
$$
