# 10.1 氧化还原方程式配平

## 题目

配平下列各氧化还原反应方程式，并注明有关元素氧化数的变化。

（1） $\mathrm{SO}_2 + \mathrm{MnO}_4^-\longrightarrow \mathrm{Mn}^{2 + } + \mathrm{SO}_4^{2 - }$ （酸性溶液）  
(2) $\left(\mathrm{NH}_{4}\right)_{2} \mathrm{Cr}_{2} \mathrm{O}_{7} \longrightarrow \mathrm{N}_{2} + \mathrm{Cr}_{2} \mathrm{O}_{3}$   
(3) $\mathrm{HNO}_3 + \mathrm{P} \longrightarrow \mathrm{H}_3\mathrm{PO}_4 + \mathrm{NO}$   
(4) $\mathrm{H}_{2} \mathrm{O}_{2} + \mathrm{PbS} \longrightarrow \mathrm{PbSO}_{4}$   
（5） $\mathrm{I}^{-} + \mathrm{IO}_{3}^{-}\longrightarrow \mathrm{I}_{2}$ （酸性溶液）  
(6) $\mathrm{MnO}_2 + \mathrm{KClO}_3 + \mathrm{KOH} \longrightarrow \mathrm{K}_2\mathrm{MnO}_4 + \mathrm{KCl}$

## 参考答案

（1） $5\mathrm{SO}_2 + 2\mathrm{MnO}_4^- + 2\mathrm{H}_2\mathrm{O} \longrightarrow 2\mathrm{Mn}^{2+} + 5\mathrm{SO}_4^{2-} + 4\mathrm{H}^+$

（2） $(\mathrm{NH}_4)_2\mathrm{Cr}_2\mathrm{O}_7\longrightarrow \mathrm{N}_2 + \mathrm{Cr}_2\mathrm{O}_3 + 4\mathrm{H}_2\mathrm{O}$   
（3） $5\mathrm{HNO}_{3} + 3\mathrm{P} + 2\mathrm{H}_{2}\mathrm{O}\longrightarrow 3\mathrm{H}_{3}\mathrm{PO}_{4} + 5\mathrm{NO}$   
（4） $4\mathrm{H_2O_2} + \mathrm{PbS}\longrightarrow \mathrm{PbSO_4} + 4\mathrm{H_2O}$   
（5） $5\mathrm{I}^{-} + \mathrm{IO}_{3}^{-} + 6\mathrm{H}^{+}\longrightarrow 3\mathrm{I}_{2} + 3\mathrm{H}_{2}\mathrm{O}$   
（6） $3\mathrm{MnO_2} + \mathrm{KClO_3} + 6\mathrm{KOH} \longrightarrow 3\mathrm{K_2MnO_4} + \mathrm{KCl} + 3\mathrm{H_2O}$
