# 2.7 在时用排水集气法要收集氧气，至

## 题目

在 $20^{\circ}\mathrm{C}, 99.0\mathrm{kPa}$ 时用排水集气法要收集 $1.50\mathrm{dm}^3$ 氧气，至少要取多少克 $\mathrm{KClO}_3$ （用 $\mathrm{MnO}_2$ 作催化剂）进行热分解？已知：在 $20^{\circ}\mathrm{C}, p(\mathrm{H}_2\mathrm{O}) = 2.34\mathrm{kPa}$ 。

## 参考答案

$n(\mathrm{O}_2) = \frac{p(\mathrm{O}_2)V}{RT} = \frac{(99.0 - 2.34)\mathrm{kPa}\times1.50\mathrm{dm}^3}{8.31\mathrm{kPa}\cdot\mathrm{dm}^3\cdot\mathrm{mol}^{-1}\cdot\mathrm{K}^{-1}\times293\mathrm{K}} = 5.95\times 10^{-2}\mathrm{mol}$

$$
\begin{array}{l} 2 \mathrm {K C l O} _ {3} \longrightarrow 2 \mathrm {K C l} + 3 \mathrm {O} _ {2} \\ 2. 0 0 \times 1 2 3 \mathrm {g} \quad 3. 0 0 \mathrm {m o l} \\ m \\ 5. 9 5 \times 1 0 ^ {- 2} \mathrm {m o l} \\ m \left(\mathrm {K C l O} _ {3}\right) = \frac {2 . 0 0 \times 1 2 3 \mathrm {g} \times 5 . 9 5 \times 1 0 ^ {- 2} \mathrm {m o l}}{3 . 0 0 \mathrm {m o l}} = 4. 8 8 \mathrm {g} \\ \end{array}
$$
