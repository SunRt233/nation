# 11.14 元素电子构型推断

## 题目

| 元素符号 | 原子序数Z | 电子构型 |
|---|---|---|
| (1)Na | 11 | $1s^22s^22p^63s^1$ |
| (2)- | - | $1s^22s^22p^63s^23p^3$ |
| (3)Zr | 40 | [Kr]4d( )5s2 |
| (4)- | - | [Kr]4d( )5s25p4 |
| (5)- | - | [Kr]4d( )5s( )5p5 |
| (6)Bi | 83 | [Xe]4f( )5d( )6s( )6p( ) |

| 元素符号 | 原子序数Z | 电子构型 |

| (1)Na | 11 | $1s^2 2s^2 2p^6 3s^1$ |
| (2)P | 15 | $1s^2 2s^2 2p^6 3s^2 3p^3$ |
| (3)Zr | 40 | [Kr]4d(2)5s2 |
| (4)Te | 52 | [Kr]4d(10)5s25p4 |
