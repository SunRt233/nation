# 7.18 反应机理推导速率方程

## 题目

$2\mathrm{ICl} + \mathrm{H}_{2}\longrightarrow 2\mathrm{HCl} + \mathrm{I}_{2}$ 的反应历程若是：

① $\mathrm{H}_{2} + \mathrm{ICl}\longrightarrow \mathrm{HI} + \mathrm{HCl}$ （慢）
② $\mathrm{ICl + HI\longrightarrow HCl + I_2}$ （快）

试推导速率方程式。

## 参考答案

反应速率由较慢的一步所决定，即

$$
\frac {- \mathrm {d} c (\mathrm {H} _ {2})}{\mathrm {d} t} = k _ {1} c (\mathrm {H} _ {2}) c (\mathrm {I C l})
$$
