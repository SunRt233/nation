# 8.15 缓冲溶液判别

## 题目

下列各组水溶液等体积相混合时，哪些可以作为缓冲溶液，为什么？

(1) $\mathrm{NaOH}(0.100\mathrm{mol}\cdot \mathrm{dm}^{-3})$ 和 $\mathrm{HCl}(0.200\mathrm{mol}\cdot \mathrm{dm}^{-3})$
(2) $\mathrm{HCl}(0.100\mathrm{mol}\cdot \mathrm{dm}^{-3})$ 和 $\mathrm{NaAc}(0.200\mathrm{mol}\cdot \mathrm{dm}^{-3})$
(3) $\mathrm{HCl}(0.100\mathrm{mol}\cdot \mathrm{dm}^{-3})$ 和 $\mathrm{NaNO}_2(0.050\mathrm{mol}\cdot \mathrm{dm}^{-3})$
(4) $\mathrm{HNO}_2(0.300\mathrm{mol}\cdot \mathrm{dm}^{-3})$ 和 $\mathrm{NaOH}(0.150\mathrm{mol}\cdot \mathrm{dm}^{-3})$

## 参考答案

（1）混合后：$c(\mathrm{NaOH}) = 0.0500\mathrm{mol}\cdot \mathrm{dm}^{-3}$，$c(\mathrm{HCl}) = 0.100\mathrm{mol}\cdot \mathrm{dm}^{-3}$

中和后：$c(\mathrm{HCl}) = 0.050 \mathrm{mol}\cdot \mathrm{dm}^{-3}$，无共轭酸碱对，不能作为缓冲溶液。

（2）混合后：$c(\mathrm{HCl}) = 0.050\mathrm{mol}\cdot \mathrm{dm}^{-3}$，$c(\mathrm{Ac}^-) = 0.100\mathrm{mol}\cdot \mathrm{dm}^{-3}$

中和后生成HAc，剩余Ac⁻：$c(\mathrm{HAc}) = 0.050 \mathrm{mol}\cdot \mathrm{dm}^{-3}$，$c(\mathrm{Ac}^-) = 0.050 \mathrm{mol}\cdot \mathrm{dm}^{-3}$

有 $\mathrm{HAc}$、$\mathrm{Ac}^-$ 共存，浓度相等、浓度大小适当，能用做缓冲溶液。

（3）混合后：$c(\mathrm{HCl}) = 0.050\mathrm{mol}\cdot \mathrm{dm}^{-3}$，$c(\mathrm{NO}_2^-) = 0.025\mathrm{mol}\cdot \mathrm{dm}^{-3}$

中和后：$c(\mathrm{HCl}) = 0.025 \mathrm{mol}\cdot \mathrm{dm}^{-3}$，$c(\mathrm{HNO}_2) = 0.025 \mathrm{mol}\cdot \mathrm{dm}^{-3}$

为强酸（HCl）与弱酸（$\mathrm{HNO}_2$）的混合液，不能用做缓冲溶液。

（4）混合后：$c(\mathrm{HNO}_2) = 0.150\mathrm{mol}\cdot \mathrm{dm}^{-3}$，$c(\mathrm{NaOH}) = 0.075\mathrm{mol}\cdot \mathrm{dm}^{-3}$

中和后：$c(\mathrm{HNO}_2) = 0.075 \mathrm{mol}\cdot \mathrm{dm}^{-3}$，$c(\mathrm{NO}_2^-) = 0.075 \mathrm{mol}\cdot \mathrm{dm}^{-3}$

有 $\mathrm{HNO}_2$、$\mathrm{NO}_2^-$ 共存，浓度相等、浓度大小适当，能用做缓冲溶液。
