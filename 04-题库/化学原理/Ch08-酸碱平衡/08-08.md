# 8.8 Na2CO3和Na3PO4溶液pH

## 题目

（1）每 $100\mathrm{cm}^3$ 纯碱溶液中含 $5.7\mathrm{gNa_2CO_3}\cdot 10\mathrm{H_2O}$ 时，溶液的 $[\mathrm{CO}_3^{2-}]$ 和 $\mathrm{pH}$ 是多少？
（2）$0.10 \mathrm{~mol} \cdot \mathrm{dm}^{-3} \mathrm{Na}_{3} \mathrm{PO}_{4}$ 溶液中，$[\mathrm{PO}_{4}^{3-}]$ 和 $\mathrm{pH}$ 各是多少？

## 参考答案

（1）$c(\mathrm{CO}_3^{2-}) = \frac{5.7\mathrm{g} / (286\mathrm{g}\bullet\mathrm{mol}^{-1})}{0.100\mathrm{dm}^3} = 0.20\mathrm{mol}\bullet \mathrm{dm}^{-3}$

$\mathrm{CO}_3^{2-} + \mathrm{H}_2\mathrm{O} \rightleftharpoons \mathrm{HCO}_3^{-} + \mathrm{OH}^{-}$，$K_{\mathrm{b}}(\mathrm{CO}_3^{2-}) = \frac{K_{\mathrm{w}}}{K_{\mathrm{a}_2}} = \frac{1.0\times10^{-14}}{4.7\times10^{-11}} = 2.1\times10^{-4}$

平衡浓度: 0.20-x, x, x

$c/K_{\mathrm{b}} = 0.20/(2.1\times10^{-4}) = 9.5\times10^2 > 400$，可简化计算。

$x = \sqrt{2.1 \times 10^{-4} \times 0.20} = 6.5 \times 10^{-3}$，$[\mathrm{OH}^-] = 6.5 \times 10^{-3} \mathrm{mol} \cdot \mathrm{dm}^{-3}$

$\mathrm{pOH} = 2.19$，$\mathrm{pH} = 11.81$，$[\mathrm{CO}_3^{2-}] = 0.19 \mathrm{mol} \cdot \mathrm{dm}^{-3}$

（2）$\mathrm{PO}_4^{3-} + \mathrm{H}_2\mathrm{O} \rightleftharpoons \mathrm{HPO}_4^{2-} + \mathrm{OH}^-$，$K_{\mathrm{b}}(\mathrm{PO}_4^{3-}) = \frac{1.0\times10^{-14}}{4.8\times10^{-13}} = 0.021$

平衡浓度: 0.10-x, x, x

$c/K_{\mathrm{b}} = 0.10/0.021 = 4.8 < 400$，不可简化计算。

$\frac{x^2}{0.10-x} = 0.021$，$x = 0.037$，$[\mathrm{OH}^-] = 0.037 \mathrm{mol} \cdot \mathrm{dm}^{-3}$

$\mathrm{pOH} = 1.43$，$\mathrm{pH} = 12.57$，$[\mathrm{PO}_4^{3-}] = 0.063 \mathrm{mol} \cdot \mathrm{dm}^{-3}$
