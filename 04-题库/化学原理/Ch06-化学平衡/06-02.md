# 6.2 平衡常数与计量系数关系

## 题目

已知 $\mathrm{N}_2 + 3\mathrm{H}_2 \rightleftharpoons 2\mathrm{NH}_3$ $K_{p}(673\mathrm{K}) = 6.1 \times 10^{-4}$ 。问此 $K_{p}$ 与下述3个 $K$ 的意义是否相同？在进行合成氨反应的平衡计算时，是否可用任意一种 $K_{p}$ ？

(1) $\frac{1}{2}\mathrm{N}_{2} + \frac{3}{2}\mathrm{H}_{2} \rightleftharpoons \mathrm{NH}_{3}$ $K_{p}^{\prime} = ?$

(2) $\frac{1}{3} \mathrm{~N}_{2} + \mathrm{H}_{2} \rightleftharpoons \frac{2}{3} \mathrm{NH}_{3}$ $K_{p}^{\prime \prime} = ?$

(3) $2\mathrm{NH}_{3} \rightleftharpoons \mathrm{N}_{2} + 3\mathrm{H}_{2}$ $K_{p}^{\prime \prime} = ?$

## 参考答案

（1）$K_{p}^{\prime} = \sqrt{K_{p}} = 0.025$

（2）$K_{p}^{\prime \prime} = \sqrt[3]{K_{p}} = 0.085;$

（3）$K_{p}^{\prime \prime \prime} = \frac{1}{K_{p}} = 1.6 \times 10^{3}$

这4个 $K$ 意义相同，讨论合成氨反应时，可用任一种 $K_{p}$ 。注意，当选定一种 $K_{p}$ 后，必须按照它所对应的化学方程式进行计算。
