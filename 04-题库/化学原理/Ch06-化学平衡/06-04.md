# 6.4 FeO与CO反应的平衡浓度

## 题目

已知 $\mathrm{FeO(s) + CO(g)}\rightleftharpoons \mathrm{Fe(s) + CO_2(g)}$ 的 $K_{c}(1273\mathrm{K}) = 0.50$ 。若起始浓度 $c(\mathrm{CO}) = 0.050\mathrm{mol}\cdot \mathrm{dm}^{-3},c(\mathrm{CO}_2) = 0.010\mathrm{mol}\cdot \mathrm{dm}^{-3}$ ，问：

（1）反应物、生成物的平衡浓度各是多少？
（2）CO的转化率是多少？
（3）增加FeO的量，对平衡有何影响？

## 参考答案

（1）$\mathrm{FeO(s) + CO(g)}\rightleftharpoons \mathrm{Fe(s) + CO_2(g)}$ $K_{c}(1273\mathrm{K}) = 0.5$

起始浓度(mol·dm⁻³): 0.050, 0.010

平衡浓度(mol·dm⁻³): 0.050-x, 0.010+x

$$
\begin{array}{l} K _ {c} = \frac {[ \mathrm {C O} _ {2} ]}{[ \mathrm {C O} ]} = \frac {0 . 0 1 0 + x}{0 . 0 5 0 - x} = 0. 5 0, \quad x = 0. 0 1 0 \\ \left[ \mathrm {C O} \right] = 0. 0 4 0 \mathrm {m o l} \cdot \mathrm {d m} ^ {- 3}, \quad \left[ \mathrm {C O} _ {2} \right] = 0. 0 2 0 \mathrm {m o l} \cdot \mathrm {d m} ^ {- 3} \\ \end{array}
$$

（2）CO转化率为 $\frac{0.010\mathrm{mol}\cdot\mathrm{dm}^{-3}}{0.050\mathrm{mol}\cdot\mathrm{dm}^{-3}}\times 100\% = 20\%$

（3）增加FeO的量，对平衡无影响。
