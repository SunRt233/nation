# 6.17 PCl5分解率与平衡常数

## 题目

$\mathrm{PCl}_5$ 遇热按式 $\mathrm{PCl}_5(\mathrm{g})\rightleftharpoons \mathrm{PCl}_3(\mathrm{g}) + \mathrm{Cl}_2(\mathrm{g})$ 分解。 $2.695\mathrm{g}$ $\mathrm{PCl}_5$ 装在 $1.00\mathrm{dm}^3$ 的密闭容器中，在 $523\mathrm{K}$ 达平衡时总压力为 $100\mathrm{kPa}$ 。求 $\mathrm{PCl}_5$ 的分解率及平衡常数 $K_{p}, K_{c}$ 。

## 参考答案

$n(\mathrm{PCl}_5) = \frac{2.695\mathrm{g}}{208.2\mathrm{g}\cdot\mathrm{mol}^{-1}} = 0.0129\mathrm{mol}$

$$
\mathrm {P C l} _ {5} (\mathrm {g}) \quad \rightleftharpoons \mathrm {P C l} _ {3} (\mathrm {g}) + \mathrm {C l} _ {2} (\mathrm {g})
$$

起始 0.0129 mol 0 0

平衡 $(0.0129 - x)\mathrm{mol}$ $x\mathrm{mol}$ $x\mathrm{mol}$

平衡时 $n_{\text{总}} = \frac{pV}{RT} = \frac{100\mathrm{kPa}\times 1.00\mathrm{dm}^3}{8.31\mathrm{kPa}\cdot\mathrm{dm}^3\cdot\mathrm{mol}^{-1}\cdot\mathrm{K}^{-1}\times 523\mathrm{K}} = 0.0230\mathrm{mol}$

$$
p _ {\text {总}} = 1 0 0 \mathrm {k P a} = 1. 0 0 \mathrm {b a r}
$$

$$
(0. 0 1 2 9 - x) \mathrm {m o l} + x \mathrm {m o l} + x \mathrm {m o l} = 0. 0 2 3 0 \mathrm {m o l}, \quad x = 0. 0 1 0 1
$$

$\mathrm{PCl}_5$ 的分解率 $= \frac{0.0101\mathrm{mol}}{0.0129\mathrm{mol}}\times 100\% = 78\%$

$$
p \left(\mathrm {P C l} _ {3}\right) = p \left(\mathrm {C l} _ {2}\right) = 1 0 0 \mathrm {k P a} \times \frac {0 . 0 1 0 1 \mathrm {m o l}}{0 . 0 2 3 0 \mathrm {m o l}} = 4 4 \mathrm {k P a} = 0. 4 4 \mathrm {b a r}
$$

$$
p \left(\mathrm {P C l} _ {5}\right) = 1 0 0 \mathrm {k P a} \times \frac {0 . 0 0 2 8 \mathrm {m o l}}{0 . 0 2 3 0 \mathrm {m o l}} = 1 2 \mathrm {k P a} = 0. 1 2 \mathrm {b a r}
$$

$$
K _ {p} = \frac {(0 . 4 4) ^ {2}}{0 . 1 2} = 1. 6, \quad K _ {c} = 1. 6 \times (0. 0 8 3 \times 5 2 3) ^ {- 1} = 0. 0 3 7
$$
