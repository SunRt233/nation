# Diels-Alder 反应 endo 选择性判断

## 题目

**(1)** 环戊二烯在室温下自发二聚。写出 dimerization 的两种可能立体化学产物（endo 和 exo），并判断哪个是动力学优势产物（主要产物）。用轨道相互作用解释 endo 选择性。

**(2)** 马来酸酐（maleic anhydride）与环戊二烯发生 Diels-Alder 反应。画出 endo 和 exo 两种过渡态，预测主产物，并解释为什么马来酸酐是极好的亲二烯体（dienophile）。

**(3)** 下列反应中 Diels-Alder 加成的区域选择性如何？预测产物：
$$
\mathrm{CH_2\!=\!CHCHO + CH_2\!=\!C(CH_3)CH\!=\!CH_2 \xrightarrow{\Delta}}
$$

## 参考答案

### (1) 环戊二烯的二聚

环戊二烯（CP）既是二烯也是亲二烯体。室温下发生 [4+2] 环加成：

**Endo 过渡态**：一个 CP 分子的"内侧"朝向另一个 CP 的 π 体系，形成立体化学上更拥挤但轨道重叠更好的排列。

**Exo 过渡态**：取代基向外侧伸展，空间位阻最小，但轨道重叠不如 endo。

**两种产物**：

```
    endo-二聚环戊二烯          exo-二聚环戊二烯
    (endo-dicyclopentadiene)   (exo-dicyclopentadiene)
```

在室温下（动力学控制），**endo 产物是主要产物**（>95%）。endo 选择性由 Alder 规则决定。

**Alder Endo 规则**：在 Diels-Alder 反应的过渡态中，亲二烯体的不饱和取代基（吸电子基团）倾向于位于二烯 π 体系的上方（endo 取向），以便亲二烯体的吸电子基团与二烯的 π 轨道发生**次级轨道相互作用**（secondary orbital interaction）。

**次级轨道相互作用**：二烯的 C2-C3 π 轨道与亲二烯体上吸电子取代基的 π*（或 p 轨道）在过渡态中有净的成键相互作用，这些作用不直接参与 σ 键的形成，但稳定了 endo 过渡态，降低了活化能。

在环戊二烯的二聚中，一个 CP 的二烯部分与另一个 CP 的亲二烯体部分发生 endo 取向的次级轨道相互作用（两个 CP 的剩余双键之间的 π-π 重叠）。

### (2) 马来酸酐 + 环戊二烯

马来酸酐是**极强的亲二烯体**，因为：
- 两个吸电子羰基通过 −I 和 −M 效应降低 C=C 的 LUMO 能量，使其与二烯的 HOMO 能量更匹配（正常电子需求的 D-A 反应中，亲二烯体 LUMO 降低 → 活化能降低）。
- 环状酸酐结构使 C=C 处于 s-cis 构象，无需额外构象调整。

**Endo 过渡态**：酸酐环朝向二烯的 π 体系上方（次级轨道相互作用：羰基的 π* 与二烯 C2-C3 π 轨道重叠）。

**Exo 过渡态**：酸酐环向外侧伸展。

**主产物**：**endo 加成产物**（动力学控制）。endo 产物中酸酐环与环己烯环上的双键同侧。

### (3) 丙烯醛 + 异戊二烯的区域选择性

丙烯醛（$\mathrm{CH_2=CHCHO}$）为亲二烯体，异戊二烯（$\mathrm{CH_2=C(CH_3)CH=CH_2}$）为二烯。

对于**不对称二烯 + 不对称亲二烯体**的 D-A 反应，区域选择性由前线轨道系数决定：

**二烯（异戊二烯）的 HOMO**：C1 的系数 > C4 的系数（因为给电子甲基在 C2 位，push 电子使 C1 系数增大）。
```
CH2=CMe-CH=CH2
 1    2   3   4
```
HOMO 系数：C1 大，C4 小。

**亲二烯体（丙烯醛）的 LUMO**：CHO 的吸电子效应使 Cβ（远离 CHO 的 C）的系数 > Cα（靠近 CHO 的 C）的系数。
```
CH2=CH-CHO
 β    α
```
LUMO 系数：Cβ 大，Cα 小。

**区域选择性规则**：HOMO 最大系数与 LUMO 最大系数结合（系数匹配 + 轨道对称性允许）。

- 二烯 C1（HOMO 大系数） + 亲二烯体 Cβ（LUMO 大系数）→ 成键。
- 二烯 C4（HOMO 小系数） + 亲二烯体 Cα（LUMO 小系数）→ 成键。

**主产物**：在环己烯环上，CHO 位于 C2 的邻位（即 1-CHO, 3-CH₃ 的取代模式，或根据 IUPAC 编号的具体结构）。

实际上产物为：

$$
\text{异戊二烯 + 丙烯醛} \xrightarrow{\Delta} \text{4-甲基-3-环己烯-1-甲醛（邻位产物）}
$$

CHO 和 CH₃ 在环己烯环上相互处于邻位（1,2-关系），这是由前线轨道系数匹配决定的**邻位选择**性。

### 总结：Diels-Alder 反应的三大选择性

| 选择性类型 | 规则 | 例子 |
|:---|:---|:---|
| **Endo 选择性** | Alder Endo 规则——次级轨道相互作用 | CP 二聚、马来酸酐 + CP |
| **区域选择性** | 前线轨道最大系数配对（HOMO-LUMO 系数匹配） | 异戊二烯 + 丙烯醛 |
| **立体专一性** | 协同反应——二烯保持原有构型（cis 二烯 → cis 产物） | (E,E)-二烯 → 反式取代产物 |
