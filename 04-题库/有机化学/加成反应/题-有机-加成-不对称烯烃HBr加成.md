# 不对称烯烃 HBr 加成：马氏规则与过氧化物效应

## 题目

3-甲基-1-丁烯（$\mathrm{(CH_3)_2CHCH=CH_2}$）与 HBr 在下列两种条件下反应：

**(1)** 无过氧化物，暗处，$25^\circ\mathrm{C}$

**(2)** 有过氧化物（如 $\mathrm{(PhCOO)_2}$），光照或加热

对每种条件：
1. 写出反应机理（含箭头推电子）；
2. 给出主要产物并命名；
3. 解释区域选择性的原因；
4. 说明条件 (1) 和 (2) 产物不同的本质原因。

## 参考答案

### 条件 (1)：无过氧化物——离子型加成（马氏规则）

**机理（Ad_E 亲电加成）：**

1. HBr 的 H⁺ 被烯烃 π 键进攻，π 电子对流向 H，H—Br 键异裂。
2. 形成碳正离子中间体。**区域选择性**取决于碳正离子稳定性：
   - 若 H⁺ 加在 C1（末端），形成 C2 上的**二级碳正离子**。
   - 若 H⁺ 加在 C2（内部），形成 C1 上的伯碳正离子（极不稳定）。
   - C2 碳正离子更稳定 → H⁺ 选择性加在 C1 上。
3. Br⁻ 进攻碳正离子，生成产物。

**产物：2-溴-3-甲基丁烷**（符合马氏规则：H 加在含 H 较多的 C 上，Br 加在含 H 较少的 C 上）。

$$
\mathrm{(CH_3)_2CHCH\!=\!CH_2 + HBr \xrightarrow{暗处,\;25^\circ C} (CH_3)_2CHCHBrCH_3}
$$

反应中可能伴随少量 1,2-甲基迁移重排产物（二级碳正离子邻位有叔碳），但主要产物仍为上述马氏加成产物。

### 条件 (2)：有过氧化物——自由基加成（反马氏规则，Kharasch 效应）

**机理（自由基链式反应）：**

**链引发**：
$$
\mathrm{(PhCOO)_2 \xrightarrow{h\nu\;or\;\Delta} 2\;PhCOO\cdot \xrightarrow{-CO_2} 2\;Ph\cdot}
$$
$$
\mathrm{Ph\cdot + HBr \longrightarrow PhH + Br\cdot}
$$

**链传递**：
1. Br· 加到烯烃上（**区域选择性取决于自由基稳定性**）：
   - Br· 加在 C1（末端）→ C2 上的**二级自由基**。
   - Br· 加在 C2（内部）→ C1 上的伯自由基（不稳定）。
   - C2 自由基更稳定 → Br· 选择性加在 C1 上。
2. C2 自由基从 HBr 夺取 H，完成加成，同时再生 Br· 继续链反应。

**产物：1-溴-3-甲基丁烷**（反马氏规则！Br 加在含 H 较多的 C 上）。

$$
\mathrm{(CH_3)_2CHCH\!=\!CH_2 + HBr \xrightarrow{过氧化物,\;h\nu} (CH_3)_2CHCH_2CH_2Br}
$$

### 两种区域选择性对比

| | 离子型加成 (1) | 自由基加成 (2) |
|:---|:---|:---|
| 关键中间体 | 碳正离子 | 碳自由基 |
| 进攻试剂 | H⁺（亲电试剂） | Br·（自由基） |
| 第一步加成 | H⁺ → 生成更稳定的碳正离子 | Br· → 生成更稳定的自由基 |
| 碳正离子/自由基稳定性 | $2^\circ > 1^\circ$ | $2^\circ > 1^\circ$ |
| H 最终位置 | 加在含 H 多的 C | 加在含 H 少的 C |
| Br 最终位置 | 加在含 H 少的 C | 加在含 H 多的 C |
| 产物 | 2-溴-3-甲基丁烷 | 1-溴-3-甲基丁烷 |
| 规则 | Markovnikov | 反 Markovnikov |

### 本质原因

两种条件下区域选择性的不同，不是因为碳正离子和碳自由基的稳定性顺序相反（实际上两者都是 $3^\circ > 2^\circ > 1^\circ$），而是因为**第一步加成的是不同的物种**：

- 离子型：第一步加 H⁺，H 的定位决定 Br 的最终位置。
- 自由基型：第一步加 Br·，Br 的定位决定 H 的最终位置。

两者都选择能生成更稳定中间体的方向，但中间体结构决定了产物的区域化学不同。

### 重要注意事项

1. **过氧化物效应仅对 HBr 有效**。HCl、HI、HF 因键能原因不适合自由基链式反应（HCl 的 H—Cl 键太强，HI 的 C—I 加成可逆）。
2. 过氧化物效应在有微量 O₂ 或过氧化物存在时发生，因此市售 HBr 试剂中常含稳定剂以防止自由基反应。
3. NBS（N-溴代丁二酰亚胺）在光照下也是自由基溴代试剂，常用于烯丙位溴代。
