# SN1/SN2/E1/E2 四选一判断

## 题目

判断下列各组反应的主要反应路径（SN1 / SN2 / E1 / E2），给出主要有机产物，并简要说明判断理由。

| 编号 | 底物 | 试剂/溶剂 | 温度 |
|:---:|:---|:---|:---:|
| (a) | $\mathrm{(CH_3)_3CBr}$ | $\mathrm{CH_3OH}$（溶剂解） | $25^\circ\mathrm{C}$ |
| (b) | $\mathrm{CH_3CH_2CH_2Br}$ | $\mathrm{NaOCH_3}$ / $\mathrm{CH_3OH}$ | $25^\circ\mathrm{C}$ |
| (c) | $\mathrm{(CH_3)_3CBr}$ | $\mathrm{NaOCH_2CH_3}$ / $\mathrm{CH_3CH_2OH}$ | 回流 |
| (d) | $\mathrm{CH_3CHBrCH_3}$ | $\mathrm{NaCN}$ / DMSO | $25^\circ\mathrm{C}$ |
| (e) | 溴代环己烷 | $\mathrm{NaOH}$ / $\mathrm{H_2O}$ | $80^\circ\mathrm{C}$ |

## 参考答案

### 判断四步法

对于 RX + Nu⁻/Base 体系：

1. **底物结构**：$3^\circ$ 倾向 SN1/E1，$1^\circ$/$2^\circ$ 倾向 SN2/E2；
2. **试剂性质**：强碱/大位阻碱促进 E2，好亲核试剂促进 SN2；
3. **溶剂效应**：极性非质子溶剂（DMSO、DMF、丙酮）有利 SN2/E2，极性质子溶剂有利 SN1/E1；
4. **温度**：高温偏向消除反应（熵增有利）。

### 各题分析

**(a) $\mathrm{(CH_3)_3CBr + CH_3OH}$，25°C**

- 底物：$3^\circ$ 卤代烷 → 排除 SN2（位阻太大）。
- 试剂：$\mathrm{CH_3OH}$ 是弱亲核试剂/弱碱 → 不利 E2。
- 溶剂：$\mathrm{CH_3OH}$（极性质子）→ 有利 SN1/E1。
- 温度：室温 → E1 较慢。

**判断：SN1 为主**。叔丁基溴在甲醇中溶剂解，生成 $\mathrm{(CH_3)_3COCH_3}$（甲基叔丁基醚）为主产物，伴随少量 $\mathrm{(CH_3)_2C=CH_2}$（E1 消除产物）。

**(b) $\mathrm{CH_3CH_2CH_2Br + NaOCH_3 / CH_3OH}$，25°C**

- 底物：$1^\circ$ 卤代烷 → 排除 SN1/E1（伯碳正离子极不稳定）。
- 试剂：$\mathrm{CH_3O^-}$ 是中等强度碱和好亲核试剂。
- 溶剂：$\mathrm{CH_3OH}$（极性质子）。
- 温度：室温。

**判断：SN2 为主**。伯卤代烷 + 好亲核试剂，SN2 非常顺利。$\mathrm{CH_3O^-}$ 的碱性不足以在室温下对伯卤代烷产生显著的 E2 竞争。产物：$\mathrm{CH_3CH_2CH_2OCH_3}$（甲基正丙基醚）。

**(c) $\mathrm{(CH_3)_3CBr + NaOCH_2CH_3 / CH_3CH_2OH}$，回流**

- 底物：$3^\circ$ 卤代烷 → 排除 SN2。
- 试剂：$\mathrm{CH_3CH_2O^-}$ 是强碱。
- 溶剂：$\mathrm{CH_3CH_2OH}$（极性质子）。
- 温度：**回流（高温）**→ 强烈偏向消除反应。

**判断：E2 为主**。叔卤代烷 + 强碱 + 高温 → E2 是主导路径。$\mathrm{CH_3CH_2O^-}$ 进攻 β-H，发生反式共平面消除。产物：$\mathrm{(CH_3)_2C=CH_2}$（异丁烯）为主。

**(d) $\mathrm{CH_3CHBrCH_3 + NaCN / DMSO}$，25°C**

- 底物：$2^\circ$ 卤代烷 → 可走 SN2/E2。
- 试剂：$\mathrm{CN^-}$ 是**极好的亲核试剂**（$\mathrm{C}$ 端的软碱性质），且碱性较弱（$\mathrm{HCN}$ 的 $\mathrm{p}K_\mathrm{a} \approx 9.2$）。
- 溶剂：**DMSO（极性非质子溶剂）**→ 强烈促进 SN2。在 DMSO 中，$\mathrm{CN^-}$ 几乎不被溶剂化（裸阴离子），亲核性极强。
- 温度：室温。

**判断：SN2 为主**。DMSO + CN⁻ 是经典的 SN2 条件。产物：$\mathrm{CH_3CH(CN)CH_3}$（异丁腈），构型翻转。

**(e) 溴代环己烷 + $\mathrm{NaOH / H_2O}$，80°C**

- 底物：$2^\circ$ 卤代烷（环己基）。
- 试剂：$\mathrm{OH^-}$ 既是亲核试剂又是强碱。
- 溶剂：$\mathrm{H_2O}$（极性质子）。
- 温度：80°C（较高）。

**判断：E2 与 SN2 竞争，E2 有利**。OH⁻ 在高温水溶液中以消除为主。溴代环己烷的 H 和 Br 可处于反式共平面（a,a 构象），E2 条件完美。产物：环己烯为主，环己醇为次。

### 总结表

| 因素 | SN1 | SN2 | E1 | E2 |
|:---|:---|:---|:---|:---|
| 底物 | $3^\circ > 2^\circ$ | $\mathrm{CH_3} > 1^\circ > 2^\circ$ | $3^\circ > 2^\circ$ | $3^\circ > 2^\circ > 1^\circ$ |
| 试剂 | 弱 Nu/弱碱 | 好 Nu/弱碱 | 弱碱 | 强碱 |
| 溶剂 | 极性质子 | 极性非质子 | 极性质子 | 皆可 |
| 温度 | 中等 | 中等 | 高温 | 高温有利 |
| 立体化学 | 外消旋化 | Walden 翻转 | — | 反式共平面 |
