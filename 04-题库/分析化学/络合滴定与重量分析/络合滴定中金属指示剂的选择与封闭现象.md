# 络合滴定中金属指示剂的选择与封闭现象

## 题目

用 EDTA 滴定 $\mathrm{Mg^{2+}}$ 时，以铬黑 T（EBT）为指示剂。

已知：
- $\lg K_{\mathrm{Mg-EBT}} = 7.0$
- $\lg K_{\mathrm{MgY}} = 8.69$
- $\mathrm{pH}=10$ 时 $\lg \alpha_{\mathrm{Y(H)}} = 0.45$，$\lg \alpha_{\mathrm{EBT(H)}} \approx 1.6$

(1) 判断 EBT 作为指示剂是否合适（通过比较化学计量点 $\mathrm{pMg_{sp}}$ 与指示剂变色点 $\mathrm{pMg_{ep}}$）；

(2) 若待测溶液中含有微量 $\mathrm{Fe^{3+}}$（已知 $\lg K_{\mathrm{Fe-EBT}} = 25.0$），滴定过程中会出现什么现象？如何消除？

## 参考答案

### (1) 指示剂合适性判断

**计算化学计量点的 $\mathrm{pMg_{sp}}$：**

首先计算条件稳定常数：

$$
\lg K'_{\mathrm{MgY}} = \lg K_{\mathrm{MgY}} - \lg \alpha_{\mathrm{Y(H)}} = 8.69 - 0.45 = 8.24
$$

设 $\mathrm{Mg^{2+}}$ 初始浓度与 EDTA 均为 $0.010\ \mathrm{mol\cdot L^{-1}}$，化学计量点时浓度减半（$c_{\mathrm{Mg}}^{\mathrm{sp}} = 0.005\ \mathrm{mol\cdot L^{-1}}$）：

$$
\mathrm{pMg_{sp}} = \frac{1}{2}(\mathrm{p}c_{\mathrm{Mg}}^{\mathrm{sp}} + \lg K'_{\mathrm{MgY}}) = \frac{1}{2}(2.30 + 8.24) = 5.27
$$

**计算指示剂变色点的 $\mathrm{pMg_{ep}}$：**

EBT 在 $\mathrm{pH}=10$ 时，条件稳定常数为：

$$
\lg K'_{\mathrm{Mg-EBT}} = \lg K_{\mathrm{Mg-EBT}} - \lg \alpha_{\mathrm{EBT(H)}} = 7.0 - 1.6 = 5.4
$$

指示剂变色点为 $\mathrm{pMg_{ep}} = \lg K'_{\mathrm{Mg-EBT}} = 5.4$。

**判断：** $\mathrm{pMg_{ep}} = 5.4$ 与 $\mathrm{pMg_{sp}} = 5.27$ 非常接近，$\Delta \mathrm{pM} = 0.13 < 0.2$，终点误差小，EBT 是合适的指示剂。

### (2) 干扰现象与消除

**现象：封闭现象**

$\mathrm{Fe^{3+}}$ 与 EBT 形成极稳定的配合物（$\lg K_{\mathrm{Fe-EBT}} = 25.0 \gg \lg K_{\mathrm{Mg-EBT}} = 7.0$）。在滴定过程中，即使到达化学计量点以后，EDTA 也无法从 $\mathrm{Fe-EBT}$ 配合物中夺取 $\mathrm{Fe^{3+}}$ 而置换出游离 EBT，因此指示剂始终呈现 $\mathrm{Mg-EBT}$ 的红色，溶液无法变蓝——即指示剂被"封闭"。

**消除方法（任选其一或组合使用）：**

1. **配位掩蔽法**：加入三乙醇胺，使其与 $\mathrm{Fe^{3+}}$ 形成更稳定的配合物，阻止 $\mathrm{Fe^{3+}}$ 与 EBT 反应。
2. **氧化还原掩蔽法**：加入抗坏血酸将 $\mathrm{Fe^{3+}}$ 还原为 $\mathrm{Fe^{2+}}$（$\lg K_{\mathrm{Fe^{2+}-EBT}} \ll \lg K_{\mathrm{Fe^{3+}-EBT}}$），大幅降低其与 EBT 的络合能力。
3. **沉淀掩蔽法**：在氨性溶液中，$\mathrm{Fe^{3+}}$ 可生成 $\mathrm{Fe(OH)_3}$ 沉淀而除去。
4. **氰化物掩蔽法**（需注意安全）：加 $\mathrm{KCN}$ 使 $\mathrm{Fe^{3+}}$ 形成 $[\mathrm{Fe(CN)_6}]^{3-}$。

> 实际分析中常采用三乙醇胺掩蔽法，操作简便且安全。
