# 题-007：SN2 vs E2 竞争判断

## 题目

> 来源：经典例题

预测叔丁醇钾（$\mathrm{t-BuOK}$）与 2-溴丙烷在叔丁醇中反应的主产物，并说明理由。

## 参考答案
**主产物：丙烯（E2 消除）**

**理由：**
1. $\mathrm{t-BuO^-}$ 是**强碱**、位阻极大的**弱亲核试剂**
2. 2° 底物 + 大位阻强碱 → E2 消除为主导
3. 叔丁醇是极性质子溶剂 → 不利于 SN2（有利于 E2）
4. $\mathrm{t-BuO^-}$ 进攻 $\beta$-H（消除）比进攻 $\alpha$-C（取代）空间上更可行

副产物：少量 2-叔丁氧基丙烷（SN2 产物）

**对比：** 若用 $\mathrm{NaOH/DMSO}$ → SN2 会显著增加（非质子溶剂 + OH⁻ 位阻小）

## 知识点映射
- [[SN2反应]] — 位阻对 SN2 的限制
- [[E2反应]] — 强碱促进消除
- [[SN1反应]] — 3° 底物才走 SN1

## 解题思路
1. 分析亲核试剂性质：强碱 + 大位阻
2. 分析底物：2° 卤代烷
3. 分析溶剂：极性质子 → 不利于 SN2
4. 结论：E2 为主

## 易错分析
- 写出 SN2 主产物（忽略 t-BuO⁻ 的位阻）
- 写出 SN1 产物（2° 底物 + 非极性溶剂 ≠ SN1 条件）
