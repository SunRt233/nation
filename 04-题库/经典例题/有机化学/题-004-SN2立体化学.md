# 题-004：SN2 立体化学预测

## 题目

> 来源：经典例题

$(R)$-2-溴丁烷在 $\mathrm{NaOH/DMSO}$ 中发生 $\mathrm{S_N2}$ 反应。写出产物的结构和绝对构型，并用楔形/虚线键表达式画出。

## 参考答案
$\mathrm{S_N2}$ 机理 → **Walden 翻转**（构型完全反转）。

DMSO 是极性非质子溶剂，促进 $\mathrm{S_N2}$。$\mathrm{NaOH}$ 提供 $\mathrm{OH}^-$ 亲核试剂。

底物为 $(R)$-2-溴丁烷 → 产物为 $(S)$-2-丁醇。

```
     CH₃           CH₃
      |              |
Br — C — H  →  HO — C — H
      |              |
   CH₂CH₃        CH₂CH₃
   (R)             (S)
```

## 知识点映射
- [[SN2反应]] — 背面进攻、Walden 翻转
- [[R-S构型]] — Cahn-Ingold-Prelog 次序规则

## 解题思路
1. 识别机理为 $\mathrm{S_N2}$（DMSO + 强亲核 + 2° 底物）
2. $\mathrm{S_N2}$ = 构型完全翻转
3. 原底物 R → 产物 S

## 易错分析
- 写出构型保持的产物（混淆 SN1 和 SN2）
- 忽略溶剂效应对机理的影响（DMSO → SN2 主导）
