# 题-012：pH 对电极电势的影响

## 题目

> 来源：经典例题

计算 pH = 5 时，$\mathrm{MnO_4^-/Mn^{2+}}$ 的电极电势。已知 $E^\circ(\mathrm{MnO_4^-/Mn^{2+}}) = +1.51\text{ V}$，设 $[\mathrm{MnO_4^-}] = [\mathrm{Mn^{2+}}] = 1.0\text{ mol/L}$。

## 参考答案
半反应：$\mathrm{MnO_4^-} + 8\mathrm{H^+} + 5e^- \rightarrow \mathrm{Mn^{2+}} + 4\mathrm{H_2O}$

Nernst 方程：
$$E = E^\circ + \frac{0.0592}{5}\lg\frac{[\mathrm{MnO_4^-}][\mathrm{H^+}]^8}{[\mathrm{Mn^{2+}}]}$$

$[\mathrm{MnO_4^-}] = [\mathrm{Mn^{2+}}] = 1.0$，$[\mathrm{H^+}] = 10^{-5}$：
$$E = 1.51 + \frac{0.0592}{5}\lg(10^{-5})^8$$
$$= 1.51 + \frac{0.0592}{5} \times (-40)$$
$$= 1.51 - 0.474 = 1.04\text{ V}$$

**结论：** pH = 5 时 MnO₄⁻ 的氧化能力显著下降（$E$ 降低了约 0.47 V）。这是因为 H⁺ 参与半反应，降低 H⁺ 浓度使氧化型（MnO₄⁻）更难获得电子。

## 知识点映射
- [[Nernst方程]] — pH 对含 H⁺ 半反应电势的影响
- [[标准电极电势]] — E° 的使用条件

## 解题思路
1. 写出半反应，确认 H⁺ 参与（系数为 8）
2. 注意 H⁺ 在分子（氧化型一侧）
3. 代入 Nernst 方程，$[\mathrm{H^+}] = 10^{-\mathrm{pH}}$
4. 计算 E 的变化幅度

## 易错分析
- H⁺ 写在分母而非分子
- 忘记 $[\mathrm{H^+}]$ 有指数 8（系数决定指数）
- 混淆 pH 与 $[\mathrm{H^+}]$ 的关系
