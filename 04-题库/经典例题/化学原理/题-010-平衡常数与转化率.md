# 题-010：平衡常数与转化率

## 题目

> 来源：经典例题

反应 $\mathrm{N_2O_4(g)} \rightleftharpoons 2\mathrm{NO_2(g)}$，某温度下 $K_p = 0.14$（atm）。若起始 $\mathrm{N_2O_4}$ 压力为 1.00 atm，求平衡转化率 $\alpha$。

## 参考答案
设转化率为 $\alpha$：

| | $\mathrm{N_2O_4}$ | $\mathrm{NO_2}$ |
|:---:|:---:|:---:|
| 起始 (atm) | 1.00 | 0 |
| 平衡 (atm) | $1-\alpha$ | $2\alpha$ |

$$K_p = \frac{(P_{\mathrm{NO_2}})^2}{P_{\mathrm{N_2O_4}}} = \frac{(2\alpha)^2}{1-\alpha} = 0.14$$

$$\frac{4\alpha^2}{1-\alpha} = 0.14$$
$$4\alpha^2 = 0.14 - 0.14\alpha$$
$$4\alpha^2 + 0.14\alpha - 0.14 = 0$$

解二次方程：
$$\alpha = \frac{-0.14 + \sqrt{0.14^2 + 4\times4\times0.14}}{2 \times 4} = \frac{-0.14 + 1.50}{8} = 0.17$$

平衡转化率为 **17%**。

## 知识点映射
- [[平衡常数]] — $K_p$ 的定义与应用
- [[Le Châtelier原理]] （此反应吸热，升温 → 转化率↑）

## 解题思路
1. 设转化率 $\alpha$，列平衡浓度（压力）表
2. 代入 $K_p$ 表达式
3. 解二次方程（舍去负根或大于 1 的根）

## 易错分析
- 忘记压力单位统一（本题全用 atm）
- NO₂ 的平衡分压写成 $\alpha$ 而非 $2\alpha$（系数为 2）
- 二次方程舍根时选错
