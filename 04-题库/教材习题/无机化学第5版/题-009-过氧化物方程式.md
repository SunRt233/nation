# 题-009：过氧化物和超氧化物反应

## 题目

> 来源：《无机化学》第五版 习题12-2

完成并配平下列化学方程式：
1. $\mathrm{Na_2O_2} + \mathrm{H_2O} \rightarrow$
2. $\mathrm{Na_2O_2} + \mathrm{CO_2} \rightarrow$
3. $\mathrm{KO_2} + \mathrm{H_2O} \rightarrow$
4. $\mathrm{KO_2} + \mathrm{CO_2} \rightarrow$
5. $\mathrm{Fe_2O_3} + \mathrm{Na_2O_2} \xrightarrow{\text{共熔}}$

## 参考答案
1. $\mathrm{Na_2O_2} + 2\mathrm{H_2O} = \mathrm{H_2O_2} + 2\mathrm{NaOH}$
2. $2\mathrm{Na_2O_2} + 2\mathrm{CO_2} = 2\mathrm{Na_2CO_3} + \mathrm{O_2}$
3. $2\mathrm{KO_2} + 2\mathrm{H_2O} = \mathrm{O_2}\uparrow + \mathrm{H_2O_2} + 2\mathrm{KOH}$
4. $4\mathrm{KO_2} + 2\mathrm{CO_2} = 2\mathrm{K_2CO_3} + 3\mathrm{O_2}$
5. $\mathrm{Fe_2O_3} + 3\mathrm{Na_2O_2} \xrightarrow{\text{共熔}} 2\mathrm{Na_2FeO_4} + \mathrm{Na_2O}$

## 知识点映射
- [[碱金属]] — 碱金属含氧化合物的反应
- [[碱土金属]] — 过氧化物化学
- [[氧化物]] — 普通氧化物/过氧化物/超氧化物分类

## 解题思路
1. 过氧化物含 [O-O]²⁻：与水生成 H₂O₂ + 碱；与 CO₂ 生成碳酸盐 + O₂
2. 超氧化物含 O₂⁻：与水生成 O₂ + H₂O₂ + 碱；与 CO₂ 生成碳酸盐 + O₂
3. Na₂O₂ 熔矿时强氧化性，将 Fe³⁺ 氧化为 FeO₄²⁻（高铁酸盐）

## 易错分析
- Na₂O₂ + H₂O 产物写成 NaOH + O₂（忘记中间产物 H₂O₂）
- KO₂ 与 CO₂ 的配平系数（注意 O 原子守恒）
- 熔矿反应中忘记 Na₂O 副产物
