# 题-008：碱土金属推断题

## 题目

> 来源：《无机化学》第五版 习题12-7(1)

某碱土金属 (A) 在空气中燃烧时火焰呈橙红色，反应产物为 (B) 和 (C) 的固体混合物。该混合物与水反应生成溶液 (D)，并放出气体 (E)。气体 (E) 可使红色石蕊试纸变蓝。将 $\mathrm{CO_2}$ 气体通入溶液 (D) 中有白色沉淀 (F) 生成。试给出 (A)~(F) 所代表物质的化学式。

## 参考答案
- (A) = $\mathrm{Ca}$（焰色橙红）
- (B) = $\mathrm{CaO}$（钙在空气中燃烧产生普通氧化物）
- (C) = $\mathrm{Ca_3N_2}$（钙在空气中部分与 $\mathrm{N_2}$ 反应）
- (D) = $\mathrm{Ca(OH)_2}$ 溶液
- (E) = $\mathrm{NH_3}$（$\mathrm{Ca_3N_2}$ 水解产物，使红色石蕊试纸变蓝）
- (F) = $\mathrm{CaCO_3}$（$\mathrm{Ca(OH)_2}$ 吸收 $\mathrm{CO_2}$）

**关键反应：**
- $2\mathrm{Ca} + \mathrm{O_2} \rightarrow 2\mathrm{CaO}$
- $3\mathrm{Ca} + \mathrm{N_2} \rightarrow \mathrm{Ca_3N_2}$
- $\mathrm{Ca_3N_2} + 6\mathrm{H_2O} \rightarrow 3\mathrm{Ca(OH)_2} + 2\mathrm{NH_3}\uparrow$
- $\mathrm{Ca(OH)_2} + \mathrm{CO_2} \rightarrow \mathrm{CaCO_3}\downarrow + \mathrm{H_2O}$

## 解题思路
1. 焰色橙红 → Ca（Sr 是洋红，Ba 是黄绿）
2. Ca 在空气中同时与 O₂ 和 N₂ 反应
3. 氮化物水解 → NH₃ 是关键线索
4. Ca(OH)₂ + CO₂ → CaCO₃↓

## 易错分析
- 只写 CaO 而遗漏 Ca₃N₂（碱土金属在空气中可与 N₂ 反应）
- 焰色橙红误解为 Sr（Sr 是洋红/深红）
