## 二、分维度深度分析

### 2.1 KP 深度（均值 3.2/5）——最大短板在元素化学

**强项（4-5★）**：
- 原子结构 KP（Koopmans 能量分解、Slater 规则、相对论效应）— 竞赛级
- 缓冲溶液 KP（实验数据表、配制方法、H-H 定量推导）— 竞赛级
- Gibbs 自由能 KP（Ellingham 图、偶联反应、CaCO₃ 多温分解）— 竞赛级
- 分析化学 KP 群（滴定误差、林邦公式、Lambert-Beer）— 普遍有例题

**弱项（2★ 及以下）**：
- **反应速率 KP**：§十(例题)=[待补充]、§十一(易错点)=[待补充]、§十三(竞赛拓展)=[待补充] — 全库最差的 KP
- **主族元素 KP 群**：氢、卤素、氮族、氧族、碳族、硼族、碱土金属 — §十~§十二 系统性空白
- **元素推断 KP**：§十二=[待填充]，核心方法论只在专题页

**模式**：
- 物理化学/分析化学 KP > 无机结构 KP >> 元素化学 KP
- 这与化竞实际权重**相反**（元素化学 ~30-40% 初赛分值）

### 2.2 真题覆盖（均值 2.4/5）——全库最大结构性缺口

**核心发现**：
1. `05-真题库/` **不存在**
2. `04-题库/` 仅含 2 个索引文件，无题目数据
3. 全部 51 个专题页 §八 使用 `FROM "04-题库"` Dataview 查询 → **全部返回空集**
4. "Dataview 驱动的真题推荐"机制是**虚设的**

**部分补强**：
- 若干专题页手动编写了真题清单（氢与卤素列出了 8 道 31-39 届真题，电化学计算 §六 有 5 道策展）
- 专题页 §五 的"例题"大多是教学例题，不是真实初赛/决赛真题
- 无任何一道真题有**完整题目正文 + 解析 + 答案**存在库中

### 2.3 讲义深度（均值 3.9/5）——最强维度

**强项**：
- 所有备课大纲均有标准化的 §0.1A（提炼→课堂使用映射）
- 所有备课大纲均有 `depth_boundary`、`deferred_topics` 字段
- 新授课正文普遍有课堂例题 + 易错点预警 + 课后分层作业
- 速查卡（33 张）作为课堂投影入口质量高

**弱项**：
- 化学动力学基础班讲义大量内容 deferred 到第三轮（积分方程、半衰期、机理、催化全部延迟）
- 小专题（SN1/SN2、取代/消除）无正式备课大纲，只有嵌入式 45 分钟教学计划

### 2.4 跨模块连接（均值 3.6/5）

**强项**（5★）：
- 原子结构与元素周期律 → 分子结构 → 晶体结构 → 配位化学 链条完整
- 酸碱理论 → 化学平衡 → 沉淀溶解平衡 → 氧化还原 → 配位化学 多向链接
- §零点五进阶导航机制标准化

**弱项**：
- 元素化学专题页倾向于只链接"上一族/下一族"，跨模块链接少（如卤素未链接到电化学 Nernst 应用）
- KP→KP 层级的双向链接不如专题页层级丰富

## 四、全库系统性模式

### 4.1 三层结构失衡（"头重脚轻"）

```
专题页/速查卡    ████████████████████  4-5★  （教学工具层）
备课大纲/新授课  ██████████████████    4-5★  （教学执行层）
底层 KP         ████████              2-3★  （知识数据层）  ← 瓶颈
真题库           (空)                   0★    （训练验证层）  ← 断裂
```

### 4.2 学科间不平衡

| 学科 | KP 深度 | 化竞权重 | 匹配度 |
|:---|:---:|:---:|:---|
| 物理化学/分析化学 KP | 4★ | ~25% | ✅ 偏强 |
| 无机结构 KP | 3★ | ~20% | ⚠️ 及格 |
| 元素化学 KP | 2★ | ~35% | 🔴 严重不匹配 |

### 4.3 备课大纲标准化程度高

所有备课大纲均有：
- §0.1A（提炼→课堂使用映射）✅
- `depth_boundary` / `deferred_topics` ✅
- 认知台阶表 ✅
- 这是系统性优势，不是缺口

### 4.4 KP 模板填充不完整

v1.3 KP 模板规定的 §十(例题)、§十一(易错点)、§十二(教学视角)、§十三(竞赛拓展) 四个教学转化节，在大量 KP 中为 `[待补充]` 或 `[待填充]`。核心知识（§一~§七）填充率高，教学转化层（§十~§十三）填充率低。
