# 问答与答疑归档

> 本目录存放基于知识库的答疑记录和学生提问归档。
>
> **产出来源**：Agent 11（答疑 Agent）基于 KB 回答后，将问答对归档于此。
>
> **与 04-题库/ 的区别**：题库是正式竞赛题目；本目录是师生问答对话的沉淀。

---

## 目录结构

```
05-问答/
├── README.md（本文件）
├── 按知识点分类/    # 按 KP 组织常见问题
└── 按日期分类/      # 按会话日期归档
```

---

## 归档规范

1. **命名**：`YYYY-MM-DD-<核心KP>-<问题摘要>.md`
2. **frontmatter** 必填：
   - `related_kp`: 关联知识点 wikilink
   - `question_type`: 概念理解 / 计算过程 / 竞赛策略 / 其他
   - `source`: 学生提问 / 教师自查 / Agent生成
3. **正文结构**：
   - `## 问题`
   - `## 解答`（引用 KB wikilink）
   - `## 延伸`（关联 KP / 真题）

---

## 与知识库的联动

- 答疑中发现 KP 内容缺失/错误 → 挂到 `[[活跃任务]]`，并在当日日志补过程与证据，由 Agent 7 优化
- 高频问答 → 提炼后回流到对应 KP 的「教学视角」段
