# AI工作区

> 本目录用于存放 Agent 执行过程中的中间产物、草稿、待确认内容。

## 用途（规划中）

- Agent 产出的临时文件（待用户确认后迁移到正式区）

## 使用规则

1. 本目录下的文件**不保证长期保留**
2. 确认有效的内容应及时迁移到 `03-知识点/`、`04-题库/` 或 `04-专题与题型/`
3. 定期清理（建议每月一次）
