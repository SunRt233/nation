## 一、当前总判断：这页现在应该回答什么

旧版这页最大的问题，不是“分析得不认真”，而是它把 `2026-06-15` 之前的阶段性事实继续当成当前事实在说，导致出现几条明显失真的判断：

- `第三轮有机仅 14.7%` 已不再符合当前 KB 实情
- `第三轮能否开课取决于有机填充速度` 已不再成立
- `第四轮依赖第三轮产出后启动` 也不再成立，因为第四轮主线已经闭环
- 这页不应该继续承担“实时库存报表”职责，那是 [[备课思路/备课与教学思路待办]] 的工作

## 三、轮次与真题任务的更真实映射

### 3.1 第一轮：不是“浅入门”，而是初赛普化骨架

第一轮真正对应的真题任务，不是“只会一点基础概念”，而是：

- 基础定量计算与规范表达
- 平衡、电化学、热力学的一步到两步调用
- 原子结构、键型、Lewis/VSEPR、杂化的首轮结构判断
- 从宏观现象切到微观模型的第一次成型

对应真题可用作各类课堂角色的典型例子：

| 任务类型 | 代表真题 | 第一轮承担方式 |
|:---|:---|:---|
| 入门计算 | 34届 1-2 | 建立单位、守恒、气体/计量规范 |
| 热力学/自发性 | 39届 Q1 | 建立 `ΔG / 判据 / 平衡` 首轮语言 |
| 电化学基础 | 37届 Q3 | 先讲电池语言与电势直觉，不急着一步顶到压轴层 |
| 结构基础判断 | 34届 1-1、35届 Q4、39届 Q5 部分小问 | 建 Lewis/VSEPR/杂化/晶体入口 |
| 方程式与现象 | 35届 Q1、34届 Q2 | 从第一轮起贯穿，不后置 |

**当前判断**：第一轮的矛盾已不在“能不能开课”，而在于要继续维持它的 `普化骨架` 身份，不要被后续轮次挤回成“零散基础专题集合”。

### 3.2 第二轮：不是“元素 + 几节滴定”，而是信号驱动的系统轮

第二轮在真题中主要承接三类任务：

- `元素事实 -> 性质 -> 现象 -> 推断 -> 方程式`
- `分析流程 -> 平衡/误差/滴定 -> 计算与判断`
- `颜色 / 沉淀 / 气体 / 配位 / 价态变化` 这些信号的系统解释

| 任务类型 | 代表真题 | 第二轮承担方式 |
|:---|:---|:---|
| 情境化元素推断 | 38届 Q1、39届 Q1 | 从“背性质”升级到“读题取信号、回调元素事实、写方程式” |
| 过渡金属与配位信号 | 35届 Q7、38届 Q6、39届 Q2 部分小问 | 把颜色、价态、配位稳定性和结构直觉压回元素线 |
| 滴定与分析计算 | 37届 Q2、38届 Q2 | 把四大滴定、误差、条件平衡组织成可做题方法系统 |
| 元素与分析耦合 | 38届 Q2、39届 Q4 | 训练“元素性质”和“分析测量”不是两门平行课，而是同一题里的两种语言 |

**当前判断**：第二轮当前也已不是“有没有材料”的问题，而是要继续把它稳定解释为 `元素系统 + 分析测量 + 信号推断` 三条线共同推进。

### 3.3 第三轮：当前最值得继续打磨的主战场不是“补齐”，而是“桥梁压实”

第三轮在当前框架里，已经不应再被理解成“第三轮有机缺件很多，所以先补目录”。  
它当前更真实的角色是：**把初赛和部分决赛基础题真正难啃的解释层系统搭起来。**

| 任务类型 | 代表真题 | 第三轮承担方式 |
|:---|:---|:---|
| 有机结构与选择性 | 38届 Q8、39届 Q8 | 以基础有机 + Clayden 主线压实稳定性、立体电子效应、选择性语言 |
| 机理与合成 | 34届 Q9/Q10、37届 Q9/Q10、39届 Q9/Q10 | 把取代/加成/消除/重排/自由基/周环/金属有机系统讲完 |
| 晶体与 XRD | 35届 Q6、37届 Q6、38届 Q5、39届 Q5 | 把晶胞投影、坐标、密堆积、XRD 入口从“会算”提升到“会解释” |
| 配位光谱与颜色 | 39届 Q2、37届 Q5 部分小问 | 把 `d-d / CT / 允许性 / 场强 / 对称性` 串成课堂链 |
| 谱图与结构判定 | 各类有机结构题与结构深化小问 | 让 IR / NMR / UV-Vis 不再是第四轮才第一次出现的语言 |
| 物化深化 | 平衡/热力学/动力学综合题 | 把第一轮的公式语言提升为可解释、可迁移的定量框架 |

**当前最值得继续推进的第三轮优化点**：

- 波谱与表征线程要继续显式化，不要再退回“有空再补”
- 配位光谱与对称桥梁要继续和元素线、结构线互相回调
- 物化深化要继续保持“解释力提升”，而不是只补公式
- 有机线后续更该做的是 `教学调用链与讲义厚度微调`，而不是再按旧叙事回到“大面积缺件”

### 3.4 第四轮：不再负责第一次系统教基础，而负责把长题真正缝起来

第四轮现在更适合这样理解：

- 它承接的是 `跨模块长题整合`
- 它负责 `讲评顺序、答题组织、压轴体验`
- 它也可以补少量桥梁，但不应再承担“大面积第一次新授”

| 任务类型 | 代表真题 | 第四轮承担方式 |
|:---|:---|:---|
| 化学原理综合 | 38届 Q7 | 多平衡/热力学/电化学/计算链整合 |
| 结构与配位综合 | 39届 Q2、37届 Q6、38届 Q5 | 把结构、光谱、推断、计算揉成完整长题体验 |
| 有机压轴综合 | 34届 Q10、37届 Q10 | 机理、选择性、逆合成与过程分组织 |
| 元素深度推断 | 38届 Q1、39届 Q1 | 快速提取信号、搭建化学叙事、书写关键反应 |

**当前判断**：第四轮当前的主要工作已经不是“启动”，而是后续如需再做，就做讲评与整合体验优化。

## 四、当前真正剩下的“缺口”是什么

这里的“缺口”不再主要指“缺文件”，而更应指**缺哪些高价值桥梁层**。

### 4.1 仍值得继续压实的高价值桥梁

| 线程 | 当前状态 | 还值得做什么 |
|:---|:---|:---|
| 实验与测量 | 第一轮起已进入，第二轮已系统化 | 继续把数据解释语言稳定挂回第三轮与第四轮 |
| 波谱与表征 | 已建立第三轮显式入口 | 继续补 `IR / NMR / UV-Vis / 配位光谱 / XRD` 的课堂调用链 |
| 结构-性质-反应桥梁 | 已成为新口径核心 | 继续把元素线、结构线、有机线之间的回调写得更顺手 |
| 生化与材料低频线 | 已有位置，但不是主推进线 | 暂以“有入口、不强行做重”处理 |

### 4.2 当前不再属于主缺口的旧叙事

- “第三轮有机大面积缺件”
- “第四轮要等第三轮完成后才能启动”
- “本页主要职责是触发批量补建”

这些说法保留在历史日志和旧阶段报告中还可以，但不应继续作为当前决策依据。

## 六、后续优化建议（按当前目标排序）

### 6.1 高优先级：继续服务 `备课大纲 + 学生讲义`

- 把第三轮的波谱/表征/配位光谱语言继续回流到大纲和讲义

- 继续给 `晶体/XRD -> 配位光谱 -> 对称桥梁` 这条线挂更明确的题型入口

- 决赛/IChO 方向仍值得单开一条后续线
- 但它不应打断当前 `备课大纲 + 学生讲义` 主线收束
