## 一、总判断

这份 `化学竞赛初赛讲义` 不是零散题单，也不是单纯教材摘抄，而是一份已经具有**完整教学骨架**的初赛总讲义：

1. 有完整目录，覆盖 `13 讲 + 2 个附录 + 答案与提示`。
2. 章节跨度横跨 `无机 / 结构 / 元素 / 物化 / 分析 / 有机 / 波谱 / 高分子`。
3. 叙述风格不是纯定义型，而是明显偏向：
   - 题型进入
   - 例题串讲
   - 解释与推断
   - 章节后习题
4. OCR 已经把正文和图片拆开，具备继续结构化提炼的条件。

因此，这份资料**不适合只做“读后摘要”**，更适合做成三层产物：

1. `资料提炼层`：把每一讲压成可引用的系统页。
2. `课堂回流层`：把高价值内容补进现有知识点、专题页、备课大纲和学生讲义。
3. `图片资产层`：把真正有教学价值的结构图、谱图、晶体图、推断图抽成可复用图片入口。

## 五、提炼标准：这次要“提细一点”到什么程度

本轮提炼不能只停在“这一讲讲了什么”，至少要细到以下四层：

### 5.1 细到“知识组织方式”

例如不是只写“第3讲讲分子结构”，而是要拆成：

- Lewis 适合解决什么题
- VSEPR 适合解决什么题
- 哪些地方要切到 MO / 对称 / 离域键
- 初赛层面哪些地方不值得过深展开

### 5.2 细到“例题能提供什么方法”

例如：

- 哪一道例题对应“存在形体判断”
- 哪一道例题对应“通过元素分析反推结构”
- 哪一道例题对应“波谱综合分析而不是单一峰指认”

### 5.3 细到“和现有知识库怎么分工”

例如：

- 已有专题页足够完整时，本讲只补“初赛实用层”
- 已有知识点偏定义时，本讲负责补“题型进入层”
- 已有网课资料偏课堂顺序时，本讲负责补“教材式归纳层”

### 5.4 细到“是否值得抽图”

只有满足以下条件的图，才建议进入精选图片层：

1. 结构/构型/对称信息高度依赖图形
2. 文字复述效率明显低
3. 未来会在多个知识点/专题页重复使用
4. 图本身清晰，OCR 后不需要大规模修图

## 六、图片处理方案

### 6.1 当前判断

图片数量很大，且文件名为哈希式命名。这说明：

1. 图片已经完成了机器切图
2. 但当前没有“图名 -> 教学用途”的人工索引

因此图片层不能直接全量搬运，必须先做**筛选索引**。

### 6.2 建议的图片分级

#### A 类：必须抽取

- 晶体结构 / 晶胞 / 堆积示意图
- 配合物几何构型图
- 分子对称元素示意图
- 有机反应机理关键骨架图
- 波谱图（MS / IR / NMR）综合判断题
- 高分子拓扑 / 胶束 / 嵌段聚合图

#### B 类：可选抽取

- 例题中能显著降低理解成本的结构图
- 章节总结图
- 复习附录中的总览图

#### C 类：通常不抽

- 纯文本公式截图
- 普通反应箭头图
- 没有后续复用价值的单题小图

### 6.3 图片落点建议

图片不建议直接散落挂到很多正文里，建议先做一层中间索引：

1. 每讲提炼页里记录“建议抽图清单”
2. 后续再统一把高价值图转到：
   - 相关知识点
   - 专题页
   - 学生讲义
3. 对波谱、晶体、对称、配位图，可以考虑单独做“图像索引页”

## 八、每一批完成后要做的回流动作

每完成一讲，不是只新增一份资料提炼页，还要做四个判断：

1. 哪些内容回流 `03-知识点`
2. 哪些内容回流 `04-专题与题型/专题`
3. 哪些内容适合进入 `04-课件/备课大纲` 或 `06-学生侧材料`
4. 哪些例题 / 图可以进入后续题型页或图片索引

建议默认回流节奏如下：

- `先提炼页`
- `再改现役专题页`
- `最后再决定是否进入学生讲义`

这样可以避免一开始就把原始资料直接塞进学生侧材料，导致口径不稳。

### 8.5 以后更适合采用的“回流三层法”

经过第1/6/8/9/10/11讲的实际推进，现在更适合把“提炼完成”定义得更具体一些。建议以后每讲都默认做下面三层回流，而不是只停在提炼页：

> [!note]
> 在正式进入三层回流前，先做一次“高复用缺页点”检查。
> 如果某讲反复调用某个概念、公式或桥梁词，但知识库里还没有稳定落点，就应先补一个紧凑的桥梁页，再进入三层回流。  
> 第7讲暴露出来的 `[[Bragg方程]]` 就属于这种情况。

#### 第一层：知识点层（事实与概念入口）

目标：把讲义里真正高价值的**概念组织方式**和**判断按钮**回流到 `03-知识点`。

建议动作：

1. 在对应知识点的 `sources / source_notes / source_extracts` 中补入该讲提炼页。
2. 只补“新增解释力”，不重复已有教材正文。
3. 优先补：
   - 定义容易混的概念
   - 一条更清晰的判断顺序
   - 结构到性质的桥梁句

#### 第二层：专题页层（题型与课堂入口）

目标：把提炼页里的**方法树、题簇、讲评顺序**回流到 `04-专题与题型/专题`。

建议动作：

1. 每讲至少指定 `1-3` 张主专题页作为首要落点。
2. 回流时优先补：
   - `零点六 课堂投影速查卡`
   - 一屏判断卡
   - 代表题簇与讲评顺序
   - 上下游导航
3. 不要把整讲内容平铺到很多专题页，而是让每讲有明确的“主承接页”。

#### 第三层：备课/学生材料层（可直接进课堂的素材）

目标：把提炼页里的**例题簇、图示和课堂提醒**回流到大纲、讲义和学生材料。

建议动作：

1. 抽 `3-5` 个最值得保留的例题簇，而不是整讲全搬。
2. 抽 `1-3` 张真正高复用的图片，而不是全量图片。

## 十、当前结论

> 这份 `化学竞赛初赛讲义` 值得提，而且应该**提得比普通资料更细**。  
> 但它的最佳用法不是“照讲义重写一遍”，而是：
> - 先做一套规整的讲次提炼页；
> - 再把最有价值的初赛实用层回流到现有知识点、专题页、备课大纲和学生讲义；
> - 最后再处理图像与学生侧呈现。

换句话说，后续的目标不是“把这份讲义存进库里”，而是让它变成：

1. 初赛实用题型的稳定补料源
2. 现有四轮体系的桥梁加厚源
3. 学生讲义与课堂例题的现实来源
