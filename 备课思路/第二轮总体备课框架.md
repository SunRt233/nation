# 第二轮总体备课框架

> 📌 **定位**：在第一轮化学原理基础上，系统建立"元素直觉"和"分析思维"。核心任务是掌握主族/过渡金属的重要性质、反应方程式，以及容量分析的定量计算。
>
> 🎯 **总目标**：学生能根据元素位置快速回忆其特征反应；能独立完成氧化还原/酸碱/沉淀/络合滴定的平衡计算；能通过实验现象推断未知元素。
>
> 📚 **KB状态**：无机和结构化学80/80 ✅，分析化学待进一步充实。

## 二、各专题详细说明

### 专题1：氢与卤素（2课时）

**前置知识**：第一轮氧化还原与电化学、原子结构与周期律

**核心内容**：
- 氢的特殊性：H⁺/H⁻/H· 三种形态，氢键与氢桥键
- 卤素通性：氧化性递减（F₂ > Cl₂ > Br₂ > I₂），歧化反应
- 卤素含氧酸：HClO → HClO₄ 酸性增强，氧化性减弱
- 卤素互化物与拟卤素：ICl、BrF₃、CN⁻、SCN⁻
- 重要方程式：Cl₂ + NaOH（冷/热）、卤素与水的歧化

**课堂例题**：
- ⭐⭐ 巩固：37届Q1-1（GaN刻蚀方程式，涉及Cl₂/Br₂）
- ⭐⭐⭐ 提高：卤素含氧酸的酸性/氧化性比较

**易错点预警**：
- F₂ 与水反应生成 O₂ 和 HF（不是歧化）
- 卤素含氧酸的酸性 ≠ 氧化性（HClO₄ 酸性最强但氧化性弱于 HClO）
- ICl 中 I 显正价（电负性 Cl > I）

**下节衔接**：氧族元素的硫化物与卤素反应（如 Cl₂ + H₂S）

---

### 专题2：氧族与氮族元素（3课时）

**核心内容**：
- 硫的化合物：H₂S、SO₂、SO₃、H₂SO₄、硫代硫酸盐、过二硫酸盐
- 硫的含氧酸根结构：SO₃²⁻（三角锥）vs SO₄²⁻（正四面体）
- 氮的化合物：NH₃、N₂H₄、HN₃、NOₓ、HNO₃
- 磷的化合物：P₄、PH₃、P₄O₁₀、磷的含氧酸（正/偏/焦磷酸）
- 重要方程式：Cu + 浓HNO₃ / 稀HNO₃ 产物差异、NH₄NO₃ 分解

**课堂例题**：
- ⭐⭐⭐ 提高：37届Q5（NO/H₂S反应，13分）—— 氧化还原+元素推断综合
- ⭐⭐⭐⭐ 拔高：38届Q1（炼丹元素推断，22分）—— 情境化推断实战

**易错点预警**：
- 浓HNO₃还原产物是NO₂，稀HNO₃是NO（极稀可生成NH₄⁺）
- S₂O₃²⁻ 与酸反应生成 S + SO₂（歧化）
- N₂H₄ 是二元弱碱，也是强还原剂

**下节衔接**：碳族元素的Si/P化合物与氧族/氮族的联系

---

### 专题3：碳族与硼族元素（2课时）

**核心内容**：
- 碳：CO/CO₂、碳酸盐热稳定性、氰化物
- 硅：硅酸盐结构（岛状/链状/层状/架状）、SiO₂与HF反应
- 锡和铅：Sn²⁺的还原性、PbO₂的氧化性、惰性电子对效应
- 硼：硼烷结构（3c-2e键）、硼酸的两性（Lewis酸）、B₂H₆的水解
- 铝：Al³⁺的水解、Al的两性、铝热反应

**课堂例题**：
- ⭐⭐⭐ 提高：35届Q4（碳化合物结构，13分）—— 结合第一轮Lewis结构/VSEPR
- ⭐⭐⭐ 提高：37届Q1（Ga的低价化合物、多重键化合物）—— 碳族同族类比

**易错点预警**：
- Sn²⁺ 极易被氧化，配制溶液需加 Sn 粒防止氧化
- PbO₂ 是强氧化剂（惰性电子对效应），而 Pb²⁺ 稳定
- 硼酸 H₃BO₃ 是一元弱酸（Lewis酸，接受OH⁻的O孤对电子）

**下节衔接**：碱金属/碱土金属与硼族、碳族的对比（对角线规则）

---

### 专题4：碱金属、碱土金属与稀有气体（2课时）

**核心内容**：
- 碱金属：过氧化物/超氧化物/臭氧化物，焰色反应，离子型氢化物
- 碱土金属：Be/Mg的轻质性，Ca/Sr/Ba的硫酸盐/碳酸盐溶解度趋势
- 对角线规则：Li-Mg、Be-Al、B-Si 的相似性
- 稀有气体：Xe的氟化物/氧化物（XeF₂/XeF₄/XeF₆/XeO₃），VSEPR应用
- 重要方程式：Na₂O₂ + CO₂/H₂O、XeF₆ 水解

**课堂例题**：
- ⭐⭐⭐ 提高：39届Q5（Xe化合物晶胞+VSEPR综合，18分）
- ⭐⭐⭐ 提高：38届决理-1（铍类化合物的结构和性质）—— 对角线规则深化

**易错点预警**：
- Na₂O₂ 与酸反应生成 H₂O₂（不是直接放出 O₂）
- XeF₆ 水解剧烈生成 XeO₃（易爆）
- Be²⁺ 毒性源于其小半径强极化能力，与 Al³⁺ 类似

**下节衔接**：过渡金属的d电子与主族元素的s/p电子对比

---

### 专题5：过渡金属（一）：钛、钒、铬、锰（2课时）

**核心内容**：
- 钛：TiO₂（金红石型）、TiCl₄（易水解）、钛白粉的制备
- 钒：V₂O₅（催化氧化SO₂）、钒的多价态颜色（V²⁺紫→V³⁺绿→VO²⁺蓝→VO₂⁺黄）
- 铬：Cr³⁺（稳定，绿色）、Cr₂O₇²⁻（强氧化性，橙色）、CrO₄²⁻（黄色）
- 锰：Mn²⁺（稳定，淡粉）、MnO₄⁻（强氧化性，紫色）、MnO₂（催化）
- 重要方程式：KMnO₄ 在不同pH下的还原产物差异

**课堂例题**：
- ⭐⭐⭐ 提高：37届Q3（LiMn₂O₄合成方程式）—— Mn的多价态应用
- ⭐⭐⭐ 提高：36届Q5（铁化合物ABCD推断）—— 过渡金属颜色信号

**易错点预警**：
- KMnO₄ 在酸性中还原为 Mn²⁺，中性为 MnO₂，碱性为 MnO₄²⁻
- Cr(VI) 有毒，废液需用 FeSO₄ 还原处理
- TiCl₄ 遇水剧烈水解产生白色烟雾（TiO₂·nH₂O）

**下节衔接**：Fe/Co/Ni的磁性、配合物颜色与Ti/V/Cr/Mn的对比

---

### 专题6：过渡金属（二）：铁、钴、镍、铜、锌（2课时）

**核心内容**：
- 铁：Fe²⁺/Fe³⁺的氧化还原，普鲁士蓝/滕氏蓝，Fe的配合物
- 钴：Co²⁺（粉红）vs Co³⁺（强氧化性，配合物中稳定），维生素B₁₂
- 镍：Ni²⁺（绿色）的配合物，镍镉/镍氢电池
- 铜：Cu⁺（易歧化）vs Cu²⁺（稳定），铜的冶炼（火法/湿法）
- 锌：Zn²⁺的两性（弱），Zn作为牺牲阳极，Zn-I₂电池
- 重要方程式：Fe³⁺ + Cu → Fe²⁺ + Cu²⁺（印刷电路板蚀刻）

**课堂例题**：
- ⭐⭐⭐ 提高：39届决理-2（Co的配合物）—— 配合物结构+价态
- ⭐⭐⭐ 提高：39届Q2（普鲁士蓝电化学基础部分，23分）—— Fe的变价应用

**易错点预警**：
- Cu⁺ 在水溶液中自发歧化为 Cu²⁺ + Cu（除非生成沉淀或配合物）
- Co³⁺ 在水溶液中是强氧化剂，但在配合物中可稳定存在
- Fe³⁺ 与 SCN⁻ 的血红色反应是特征鉴定，但F⁻、PO₄³⁻会干扰

**下节衔接**：Ag/Au/Hg的贵金属化学与Mo/W的多酸化学

---

### 专题7：过渡金属（三）：银、金、汞、钼、钨（1-2课时）

**核心内容**：
- 银：Ag⁺的氧化性（适中），AgX的感光性，银镜反应
- 金：Au³⁺的强氧化性，王水溶金，金的配合物
- 汞：Hg₂²⁺（稳定，不歧化）vs Hg²⁺，汞的毒性（Hg蒸气、有机汞）
- 钼和钨：多酸化学（同多酸/杂多酸），Mo/W的催化应用

**课堂例题**：
- ⭐⭐⭐ 提高：34届Q7（配合物电化学转化）—— Ag/Pt系配合物
- ⭐⭐⭐⭐ 拔高：多酸结构的基础认识（Mo₆O₁₉²⁻等）

**易错点预警**：
- Hg₂²⁺ 不歧化（与 Cu⁺ 相反），Hg₂Cl₂ 是甘汞（稳定）
- Ag⁺ 与 NH₃ 先生成 AgOH 沉淀，过量 NH₃ 生成 [Ag(NH₃)₂]⁺
- 王水：HNO₃ + HCl（3:1），HCl 提供 Cl⁻ 配位降低 Au³⁺/Pt⁴⁺ 电势

**下节衔接**：元素推断综合训练（整合所有元素性质）

### 专题8：情境化元素推断综合训练（2课时）

**定位**：第二轮的核心输出能力检验。不再孤立记性质，而是训练"实验现象→推元素→写方程式"的完整推理链。

**核心内容**：
- 颜色信号法：蓝色溶液→Cu²⁺；紫色→MnO₄⁻/Ti³⁺；绿色→Cr³⁺/Ni²⁺/Fe²⁺
- 沉淀信号法：白色沉淀溶于氨→AgCl；白色→灰绿→红棕→Fe²⁺→Fe³⁺
- 气体信号法：无色气体遇空气变红棕→NO；臭鸡蛋味→H₂S
- 价态变化法：某元素既可升价又可降价→中间价态歧化
- 配合物生成法：加入某配体后颜色突变→配合物生成指示

**课堂例题**：
- ⭐⭐⭐⭐ 拔高：38届Q1（炼丹元素推断，22分）—— 完整的"现象→推断→验证"链
- ⭐⭐⭐⭐ 拔高：39届Q1（砷化学，15分）—— 含剧毒元素的推断逻辑
- ⭐⭐⭐ 提高：37届Q5（NO/H₂S反应，13分）—— 氧化还原+元素推断

**训练方式**：
1. 教师给出一段"实验报告式"描述（颜色、沉淀、气体、溶解性）
2. 学生独立推断元素并写出关键方程式（10-15分钟）
3. 分组讨论不同推断路径的合理性
4. 教师展示标准推理链，对照过程分评分表

**与知识库联动**：
- 训练中发现学生常忘记的性质 → 即时补充到对应元素KP的「易错点」段
- 高频推断错误 → 记入 `12-复盘/元素推断常见错误追踪`

### 专题9：容量分析基础与酸碱滴定（2课时）

**前置知识**：第一轮酸碱理论、化学平衡

**核心内容**：
- 基准物质与标准溶液：定义、要求、配制方法
- 指示剂：酸碱指示剂（酚酞/甲基橙）变色原理与选择
- 酸碱滴定曲线：强酸-强碱/弱酸-强碱/多元酸的曲线特征
- 终点误差：指示剂变色点与化学计量点不重合导致的误差
- 双指示剂法：Na₂CO₃/NaHCO₃ 混合物的分别测定

**课堂例题**：
- ⭐⭐ 巩固：37届Q2（复盐组成计算，涉及滴定）
- ⭐⭐⭐ 提高：弱酸滴定的pH突跃范围计算

**易错点预警**：
- 指示剂变色范围是 pH = pKa ± 1，不是恰好等于 pKa
- 弱酸滴定突跃范围窄，需选变色范围窄的指示剂
- 双指示剂法中，酚酞变色时 Na₂CO₃ → NaHCO₃，甲基橙变色时 NaHCO₃ → H₂CO₃

**下节衔接**：氧化还原滴定中电极电势与指示剂选择

### 专题10：氧化还原滴定与沉淀滴定（1-2课时）

**核心内容**：
- 碘量法：直接碘量法（I₂ 氧化）与间接碘量法（I⁻ 还原 + I₂ 滴定）
- KMnO₄法：酸性条件下滴定，自指示剂（紫色→无色/淡粉）
- 莫尔法（Mohr）：K₂CrO₄ 作指示剂，AgNO₃ 滴定 Cl⁻
- Volhard法：Fe³⁺ 作指示剂，NH₄SCN 返滴定 Ag⁺
- 重要方程式：I₂ + 2S₂O₃²⁻ → 2I⁻ + S₄O₆²⁻

**课堂例题**：
- ⭐⭐⭐ 提高：39届Q2（碘量法测定KI浓度）—— 真实竞赛情境
- ⭐⭐⭐ 提高：KMnO₄ 滴定 Fe²⁺ / H₂C₂O₄ 的计算

**易错点预警**：
- 间接碘量法中，I₂ 与 Na₂S₂O₃ 的反应比例是 1:2（不是 1:1）
- KMnO₄ 滴定需在酸性条件下进行，否则生成 MnO₂ 沉淀
- Volhard法需在酸性条件下进行，防止 Fe³⁺ 水解

**下节衔接**：络合滴定中稳定常数与竞争配位

---

### 专题11：络合滴定与重量分析（1-2课时）

**核心内容**：
- EDTA滴定：EDTA 的六齿配位、条件稳定常数 K'f
- 金属指示剂：EBT（铬黑T）、XO（二甲酚橙）的封闭与僵化
- 掩蔽与解蔽：利用 KCN、三乙醇胺等掩蔽干扰离子
- 重量分析原理：沉淀形式→称量形式，换算因数
- 沉淀条件：晶形沉淀（陈化）vs 无定形沉淀（热/稀/慢/搅）

**课堂例题**：
- ⭐⭐⭐ 提高：36届b-Q4（碘化物滴定离子方程式）—— 沉淀+氧化还原综合
- ⭐⭐ 巩固：EDTA 滴定水中总硬度的计算

**易错点预警**：
- 指示剂封闭：指示剂与干扰离子生成更稳定配合物，无法变色
- 重量分析中，称量形式的摩尔质量应尽量大（减小称量误差）
- 陈化可使小晶体溶解、大晶体长大，提高沉淀纯度

**下节衔接**：误差分析中系统误差与随机误差的区分

---

### 专题12：误差处理与分光光度法（1-2课时）

**核心内容**：
- 误差分类：系统误差（可校正）vs 随机误差（统计处理）vs 过失误差
- 准确度与精密度：定义、关系、计算公式
- 有效数字运算回顾（第一轮）：加减 vs 乘除
- Lambert-Beer定律：A = εbc，适用范围与偏离原因
- 分光光度法应用：标准曲线法、差示分光光度法

**课堂例题**：
- ⭐⭐ 巩固：38届Q3（DNA荧光力学计算）—— 结合物理测量的化学情境
- ⭐⭐ 巩固：由标准曲线求未知浓度（作图+计算）

**易错点预警**：
- 准确度 ≠ 精密度：高精密度不一定高准确度（系统误差存在时）
- Lambert-Beer定律仅在稀溶液中成立（高浓度时分子间作用导致偏离）
- 分光光度计选择波长时应选最大吸收波长（灵敏度最高）

**下节衔接**：第三轮有机化学中的光谱分析（UV-Vis、IR、NMR）

### 专题13：方程式书写专项（元素化学版，贯穿全程）

**训练策略**：
- 每周固定时间进行元素化学方程式听写/默写
- 按元素族分类：主族元素→过渡金属
- 重点训练：非水体系反应、高温固相反应、配合物生成反应

**素材来源**：

- 34届Q2风格：给出情境，要求写出并配平方程式
- 元素推断题中的关键方程式（如38届炼丹、39届砷化学）
