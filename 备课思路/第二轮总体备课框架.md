## 零、第二轮深度边界总原则

> 第二轮的核心任务是**两条系统线的完整建设**：元素系统线（把化学原理压回元素事实上使用）和分析测量线（把滴定/误差/光度分析串成可做题的方法系统）。第二轮不讲有机化学（全部后置到第三轮），也不展开元素化学的决赛级深化（多酸/簇合物/生物无机/稀土系统化学/铂系系统配位催化放第三轮及以后）。

| 原则 | 第二轮讲到哪里为止 | 第二轮禁止提前展开 | 后续承接 |

### 专题1：氢与卤素（2课时）

**前置知识**：第一轮氧化还原与电化学、原子结构与周期律

**本轮深度边界**：系统讲完卤素及其化合物的结构-价态-反应性关系，并把第一轮学到的周期律/电化学/平衡语言用于解释卤素性质的递变规律。**不展开**：卤素互化物的完整结构化学、拟卤素的配位化学细节。

**核心内容**：
- 氢的特殊性：H⁺/H⁻/H· 三种形态，氢键与氢桥键
- 卤素通性：氧化性递减（F₂ > Cl₂ > Br₂ > I₂），歧化反应
- 卤素含氧酸：HClO → HClO₄ 酸性增强，氧化性减弱
- 卤素互化物与拟卤素：ICl、BrF₃、CN⁻、SCN⁻
- 重要方程式：Cl₂ + NaOH（冷/热）、卤素与水的歧化

**课堂例题**：
- ⭐⭐ 巩固：37届Q1-1（GaN刻蚀方程式，涉及Cl₂/Br₂）
- ⭐⭐⭐ 提高：卤素含氧酸的酸性/氧化性比较

**易错点预警**：
- F₂ 与水反应生成 O₂ 和 HF（不是歧化）
- 卤素含氧酸的酸性 ≠ 氧化性（HClO₄ 酸性最强但氧化性弱于 HClO）
- ICl 中 I 显正价（电负性 Cl > I）

- 🧪 元素系统线：卤素氧化性递变→周期律在元素化学中的第一次系统应用
- 🏷️ 信号推断线：卤化银沉淀颜色序列、卤素单质颜色→推断信号

### 专题2：氧族与氮族元素（3课时）

**前置知识**：第一轮分子结构（VSEPR）、化学平衡、电化学

**本轮深度边界**：系统讲完氧族（O/S/Se）和氮族（N/P/As/Sb/Bi）的核心化合物，把第一轮的分子结构语言（VSEPR/杂化/离域π）和热力学/电化学工具用于解释含氧酸的结构-酸性-氧化性关系。**不展开**：Se/Te 的系统化学、磷的有机金属化学。

**核心内容**：
- 硫的化合物：H₂S、SO₂、SO₃、H₂SO₄、硫代硫酸盐、过二硫酸盐
- 硫的含氧酸根结构：SO₃²⁻（三角锥）vs SO₄²⁻（正四面体）
- 氮的化合物：NH₃、N₂H₄、HN₃、NOₓ、HNO₃
- 磷的化合物：P₄、PH₃、P₄O₁₀、磷的含氧酸（正/偏/焦磷酸）
- 重要方程式：Cu + 浓HNO₃ / 稀HNO₃ 产物差异、NH₄NO₃ 分解

**课堂例题**：
- ⭐⭐⭐ 提高：37届Q5（NO/H₂S反应，13分）—— 氧化还原+元素推断综合
- ⭐⭐⭐⭐ 拔高：38届Q1（炼丹元素推断，22分）—— 情境化推断实战

**易错点预警**：
- 浓HNO₃还原产物是NO₂，稀HNO₃是NO（极稀可生成NH₄⁺）
- S₂O₃²⁻ 与酸反应生成 S + SO₂（歧化）
- N₂H₄ 是二元弱碱，也是强还原剂

### 专题3：碳族与硼族元素（2课时）

**前置知识**：第一轮分子结构（VSEPR、杂化）、专题2 含氧酸结构

**本轮深度边界**：系统讲完碳族（C/Si/Sn/Pb）和硼族（B/Al/Ga）的核心化学，重点突破惰性电子对效应、硼烷缺电子结构和硅酸盐结构。**不展开**：Ga/In/Tl 的系统化学、有机硅化学、碳纳米材料。

**核心内容**：
- 碳：CO/CO₂、碳酸盐热稳定性、氰化物
- 硅：硅酸盐结构（岛状/链状/层状/架状）、SiO₂与HF反应
- 锡和铅：Sn²⁺的还原性、PbO₂的氧化性、惰性电子对效应
- 硼：硼烷结构（3c-2e键）、硼酸的两性（Lewis酸）、B₂H₆的水解
- 铝：Al³⁺的水解、Al的两性、铝热反应

**课堂例题**：
- ⭐⭐⭐ 提高：35届Q4（碳化合物结构，13分）—— 结合第一轮Lewis结构/VSEPR
- ⭐⭐⭐ 提高：37届Q1（Ga的低价化合物、多重键化合物）—— 碳族同族类比

**易错点预警**：
- Sn²⁺ 极易被氧化，配制溶液需加 Sn 粒防止氧化
- PbO₂ 是强氧化剂（惰性电子对效应），而 Pb²⁺ 稳定
- 硼酸 H₃BO₃ 是一元弱酸（Lewis酸，接受OH⁻的O孤对电子）

### 专题8：情境化元素推断综合训练（2课时）

### 专题9：容量分析基础与酸碱滴定（2课时）

**前置知识**：第一轮酸碱理论、化学平衡、专题10 定量化学分析入门

**本轮深度边界**：以《分析化学》第六版相关内容为上限，系统讲完酸碱滴定的完整流程——从基准物质选择到终点误差分析。**不展开**：非水溶剂酸碱滴定、电位滴定法。

**核心内容**：
- 基准物质与标准溶液：定义、要求、配制方法
- 指示剂：酸碱指示剂（酚酞/甲基橙）变色原理与选择
- 酸碱滴定曲线：强酸-强碱/弱酸-强碱/多元酸的曲线特征
- 终点误差：指示剂变色点与化学计量点不重合导致的误差
- 双指示剂法：Na₂CO₃/NaHCO₃ 混合物的分别测定

**课堂例题**：

- 弱酸滴定突跃范围窄，需选变色范围窄的指示剂
- 双指示剂法中，酚酞变色时 Na₂CO₃ → NaHCO₃，甲基橙变色时 NaHCO₃ → H₂CO₃

### 专题10：氧化还原滴定与沉淀滴定（1-2课时）

**前置知识**：第一轮电化学（Nernst方程）、专题9 容量分析基础

**本轮深度边界**：系统讲完碘量法、KMnO₄法和三种沉淀滴定法（莫尔法、Volhard法、**Fajans法**），以《分析化学》第六版相关内容为上限。**不展开**：电化学分析法（电位滴定/库仑法）。

**核心内容**：
- 碘量法：直接碘量法（I₂ 氧化）与间接碘量法（I⁻ 还原 + I₂ 滴定）
- KMnO₄法：酸性条件下滴定，自指示剂（紫色→无色/淡粉）
- 莫尔法（Mohr）：K₂CrO₄ 作指示剂，AgNO₃ 滴定 Cl⁻
- Volhard法：Fe³⁺ 作指示剂，NH₄SCN 返滴定 Ag⁺
- **Fajans法（法扬司法）**：吸附指示剂法——利用沉淀表面吸附指示剂变色判断终点（如荧光黄指示 Cl⁻ 滴定）；适用条件：沉淀比表面积大（胶状沉淀）、pH 控制
- 重要方程式：I₂ + 2S₂O₃²⁻ → 2I⁻ + S₄O₆²⁻

**课堂例题**：
- ⭐⭐⭐ 提高：39届Q2（碘量法测定KI浓度）—— 真实竞赛情境
- ⭐⭐⭐ 提高：KMnO₄ 滴定 Fe²⁺ / H₂C₂O₄ 的计算

**易错点预警**：
- 间接碘量法中，I₂ 与 Na₂S₂O₃ 的反应比例是 1:2（不是 1:1）
- KMnO₄ 滴定需在酸性条件下进行，否则生成 MnO₂ 沉淀
- Volhard法需在酸性条件下进行，防止 Fe³⁺ 水解
- Fajans法需控制 pH 使指示剂以离子形式存在（荧光黄 pKa~7，需中性/弱碱性）

**资源引用**：
- 提炼：[[提炼-分析化学第六版-第7-10章-氧化还原沉淀重量与光度法]] — 氧化还原+沉淀滴定系统内容
- 提炼：[[提炼-化学竞赛初赛讲义-第9讲-溶液与化学分析]] — 滴定分析入门
- 网课：[[教学逻辑提炼-分析化学网课-配位氧化还原重量与光度法]] — 氧化还原+沉淀滴定
- KP：[[沉淀滴定]]、[[氧化还原方程式配平]]
- 考纲：18-容量分析（氧化还原滴定、沉淀滴定）

**三条线程落点**：
- 🧪 元素系统线：KMnO₄/I₂ 的氧化性与过渡金属/卤素元素化学的对接
- 🏷️ 信号推断线：三种沉淀滴定法的指示剂原理→信号产生的化学本质

**下节衔接**：络合滴定中稳定常数与竞争配位

### 专题11：络合滴定与重量分析（1-2课时）

**前置知识**：第一轮配合物基础（CFT/稳定常数）、专题9 容量分析基础

**本轮深度边界**：系统讲完 EDTA 滴定（条件稳定常数、金属指示剂、掩蔽）和重量分析基本原理。**不展开**：条件稳定常数的完整数学推导、复杂样品的前处理方案。

**核心内容**：
- EDTA滴定：EDTA 的六齿配位、条件稳定常数 K'f
- 金属指示剂：EBT（铬黑T）、XO（二甲酚橙）的封闭与僵化
- 掩蔽与解蔽：利用 KCN、三乙醇胺等掩蔽干扰离子
- 重量分析原理：沉淀形式→称量形式，换算因数
- 沉淀条件：晶形沉淀（陈化）vs 无定形沉淀（热/稀/慢/搅）

**课堂例题**：
- ⭐⭐⭐ 提高：36届b-Q4（碘化物滴定离子方程式）—— 沉淀+氧化还原综合
- ⭐⭐ 巩固：EDTA 滴定水中总硬度的计算

**易错点预警**：
- 指示剂封闭：指示剂与干扰离子生成更稳定配合物，无法变色
- 重量分析中，称量形式的摩尔质量应尽量大（减小称量误差）
- 陈化可使小晶体溶解、大晶体长大，提高沉淀纯度

**资源引用**：

- 网课：[[教学逻辑提炼-分析化学网课-配位氧化还原重量与光度法]] — 络合+重量分析
- KP：[[络合滴定]]、[[EDTA]]
- 考纲：18-容量分析（络合滴定、重量分析）

**三条线程落点**：
- 🧪 元素系统线：EDTA 与不同金属离子的络合选择性→与元素化学中的配合物稳定性对接
- 🏷️ 信号推断线：金属指示剂的变色原理→配合反应作为信号来源

### 专题12：误差处理与分光光度法（1-2课时）

**前置知识**：第一轮专题0 有效数字、专题10 定量化学分析入门

**本轮深度边界**：系统讲完分析化学中的误差评价体系（系统/随机/过失、准确度/精密度）和分光光度法的基本原理与定量应用。**不展开**：统计检验（t/F/Q/Grubbs 检验）、复杂光谱分析（导数光谱/荧光光谱）。

**核心内容**：
- 误差分类：系统误差（可校正）vs 随机误差（统计处理）vs 过失误差
- 准确度与精密度：定义、关系、计算公式
- 有效数字运算回顾（第一轮）：加减 vs 乘除
- Lambert-Beer定律：A = εbc，适用范围与偏离原因
- 分光光度法应用：标准曲线法、差示分光光度法

**课堂例题**：
- ⭐⭐ 巩固：38届Q3（DNA荧光力学计算）—— 结合物理测量的化学情境
- ⭐⭐ 巩固：由标准曲线求未知浓度（作图+计算）

**易错点预警**：
- 准确度 ≠ 精密度：高精密度不一定高准确度（系统误差存在时）
- Lambert-Beer定律仅在稀溶液中成立（高浓度时分子间作用导致偏离）
- 分光光度计选择波长时应选最大吸收波长（灵敏度最高）

**资源引用**：

- 提炼：[[提炼-分析化学第六版-第7-10章-氧化还原沉淀重量与光度法]] — 分光光度法系统内容
- 网课：[[教学逻辑提炼-分析化学网课-配位氧化还原重量与光度法]] — 分光光度法
- KP：[[误差]]、[[有效数字]]

**三条线程落点**：

- 🏷️ 信号推断线：分光光度法→把颜色信号定量化→从"看到颜色"升级为"测出吸光度"
