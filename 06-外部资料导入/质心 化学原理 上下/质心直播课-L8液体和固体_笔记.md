# 8）热稳定性 02:53:27

#  极化作⽤与热稳定性关系

o 极化能⼒定义: 阳离⼦使阴离⼦电⼦云变形的能⼒，取决于离⼦半径和电荷数  
o 热稳定性规律: 阳离⼦极化能⼒越强，含氧酸盐热稳定性越差   
典型示例:

碳酸钠 $( N a _ { 2 } C O _ { 3 } )$ 热稳定性⼩于碳酸钾 $( K _ { 2 } C O _ { 3 } )$   
 原因： $N a ^ { + }$ 离⼦半径⼩于 $K ^ { + }$ ，极化能⼒更强，更容易将氧拽向⾃⼰导致分解

#  阴离⼦被极化能⼒

o 影响因素: 离⼦半径越⼤、电荷越多，被极化能⼒越强  
o ⽐较规律:

$B r ^ { - } > C l ^ { - }$ （溴离⼦被极化能⼒⼤于氯离⼦）  
 应⽤实例：溴化铜 $( C u B r _ { 2 } )$ 热稳定性差于氯化铜 $( C u C l _ { 2 } )$

#  相互极化作⽤

o 反向极化现象: 当阳离⼦半径特别⼤时（如 $H g ^ { 2 + } )$ ），会产⽣电⼦云反向诱导效应  
o 共价性增强: 相互极化导致共价成分升⾼，热稳定性降低  
C 典型对⽐:

氧化锌 $( Z n O )$ 热稳定性 > 氧化汞 $( H g O )$   
碘化锌 $( Z n I _ { 2 } )$ )熔点 > 碘化汞 $( H g I _ { 2 } )$

# 熔点影响因素

o 离⼦半径匹配原则: 正负离⼦半径相近时晶格能最⼤（如 $C s C l$ 푙结构）  
共价成分影响: 相互极化作⽤增强会导致熔点降低  
记忆要点:

极化能⼒：电荷⾼、半径⼩的阳离⼦更强   
被极化能⼒：半径⼤、电荷多的阴离⼦更强  
相互极化：⼤半径阳离⼦特有的反向诱导效应
