# 1. 热⼒学基础概念

. 焓的定义: 焓ℎ等于内能푢加上压强与体积的乘积푝푣，即 $h = u + p v$   
 内能变化公式 $: \Delta \mathsf { u } = \mathsf { q } + \mathsf { w }$ ，其中푞为热量变化，푤为做功  
 恒容条件: 恒容情况下没有体积功 $( w = 0 )$ ，因此 $\mathcal { \Delta } \cup = q _ { v }$   
 恒压条件: 焓变Δh等于恒压热 $\cdot q _ { p }$ ，即 $\Delta { \sf h } = q _ { p }$

. 1850年能源结构: 主要燃烧⽊材，少量使⽤煤炭  
 1900年转变: 开始⼤规模使⽤煤炭  
 20世纪中期: ⽯油和天然⽓成为主要能源  
 2018-2020年现状: ⽯油和天然⽓仍占主导，但核能和氢能等新能源使⽤逐渐增加
