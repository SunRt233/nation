o 表示⽅法：第⼀电离能记作 $\cdot I _ { 1 }$ ，第⼆电离能记作 $\mathbf { \dot { \Omega } } _ { 2 }$ （如钠离⼦再失去⼀个电⼦）

#  影响因素 02:37:48

o 主要因素：原⼦核对电⼦的约束⼒

原⼦半径越⼩，约束⼒越强，电离能越⼤  
 电⼦构型影响：失去电⼦后达到稳定结构（全满/半满）时电离能会降低

# o 反常现象：

铝 $( 3 s ^ { 2 } 3 p ^ { 1 } )$ )电离能偏低：失去⼀个电⼦后形成 $3 s ^ { 2 }$ 全满稳定结构  
 硫 $( 3 s ^ { 2 } 3 p ^ { 4 } )$ 电离能偏低：失去⼀个电⼦后形成 $3 p ^ { 3 }$ 半满稳定结构

#  周期性规律

o 同周期趋势：从左到右电离能总体增⼤（核电荷增加）  
o 同族趋势：从上到下电离能总体减⼩（电⼦层增加）  
o 钻穿效应：内层电⼦靠近原⼦核时可能需要更⼤电离能（需实验验证）

# 3）电⼦亲和能

#  定义与表示

o 定义：基态⽓态原⼦获得⼀个电⼦时的能量变化

# o 表示⽅法：

放热为负值（多数情况）  
吸热为正值（少数情况）

#  ⻧素族规律 02:45:51

o 氯到碘趋势：放出能量依次减少 $( C l ^ { - } > B r ^ { - } > I ^ { - } )$

原因：原⼦半径增⼤，核对外层电⼦约束⼒减弱

o 氟的反常：放出能量( - 327.8푘퐽 / 푚표푙)⼩于氯( - 348.7푘퐽 / 푚표푙)

 解释：氟原⼦外层电⼦云密度⾼，新电⼦受到内层电⼦排斥作⽤抵消部分核吸引⼒

# 4）电负性

# . 基本概念

o 定义：原⼦在化合物中对键合电⼦的吸引能⼒（⼈为定义值）  
表示⽅法：⽤푋表示，氟定义为4.0，氢为1.0

#  与电⼦亲和能区别

# C 物理意义：

电⼦亲和能：明确的⽓态原⼦物理过程  
电负性：化合物中的定性⽐较

# o 应⽤范围：

电⼦亲和能：研究孤⽴原⼦  
电负性：分析化学键性质（判断电⼦云偏移⽅向）

#  应⽤价值

o 键极性判断：电负性差越⼤，键极性越强   
o 反应预测：电负性⼤的原⼦更易吸引电⼦参与反应  
o 结构分析：解释分⼦构型和性质

# 三、课后总结 02:49:10

# 1. 原⼦结构

# 1）量⼦数与原⼦轨道表征

 核⼼能⼒: 给定量⼦数 ${ \cdot n , l , m _ { l } }$ 时，能准确判断对应的原⼦轨道符号 $( \hbar \mathbb { A } 2 p _ { x } \frac { \sharp \mathtt { A } } { \overline { { \mathbf { \Gamma } } } \mathbf { \mathbf { \Gamma } } } )$ ）

 知识关联: 该知识点是理解后续电⼦排布和元素周期律的基础性内容

# 2）电⼦排布规则

#  三⼤原则:

o 能量最低原理：电⼦优先占据能量最低的轨道  
泡利不相容原理：同⼀轨道最多容纳2个⾃旋相反的电⼦   
o 洪特规则：简并轨道上电⼦优先单独占据且⾃旋平⾏

 特殊修正：全空 $( s ^ { 0 } , p ^ { 0 } , d ^ { 0 } , f ^ { 0 } )$ 、半满 $( p ^ { 3 } , d ^ { 5 } , f ^ { 7 } )$ 、全满 $( p ^ { 6 } , d ^ { 1 0 } , f ^ { 1 4 } )$ )状态具有额外稳定性

# 2. 元素周期律

# 1）本质理解

 实验基础: 源于对元素性质的系统观察和规律总结   
理论解释: 需⽤原⼦结构知识（特别是电⼦排布）解释周期性变化  
 应⽤⽬标: 掌握原⼦得失电⼦能⼒的判断⽅法

# 2）关键参数

 电离能(I): ⽓态原⼦失去电⼦所需能量，反映失电⼦难度  
 电⼦亲和能(E\_a): ⽓态原⼦获得电⼦释放的能量，反映得电⼦能⼒   
 电负性(): 综合衡量原⼦在分⼦中吸引电⼦的能⼒  
 原⼦半径: ⽤于合理化解释上述参数的变化趋势

# 3. 典型例题解析

# 1）电离能分析题

 解题关键: 关注电离能的突变点 $( \mathtt { A H } I _ { 2 } \to I _ { 3 }$ 剧增提示失去2个电⼦后达到稳定结构）  
 典型结论: 第2主族元素 $( n s ^ { 2 } )$ )在失去2个电⼦后形成稀有⽓体构型，导致 ${ { I } _ { 3 } }$ 显著升⾼

# 2）镁钾性质对⽐题

# . 电⼦亲和能:

半径⽐较: ${ \sf K } > { \sf N a } > { \sf M g }$ （同周期从左到右减⼩，同主族从上到下增⼤）  
 电离能⽐较: $\mathsf { M g } > \mathsf { K }$ （K失去1个电⼦即可达稳定结构）

o 本质规律：原⼦半径影响>轨道填充效应，故Mg实际更易得电⼦
