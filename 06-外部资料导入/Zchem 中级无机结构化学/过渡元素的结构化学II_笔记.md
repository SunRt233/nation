基本分类：金属有机卡宾配合物分为两大类：Fischer型卡宾和Schrock型卡宾

# - 结构特征：

- Fischer卡宾：单线态(singlet)卡宾，碳为 $sp^{2}$ 杂化，含空p轨道  
○ Schrock卡宾：三线态(triplet)卡宾，碳近似sp杂化，含两个单电子

# - 电子特性：

○ Fischer卡宾：碳带 $\delta^{+}$ ，是亲电试剂  
○ Schrock卡宾：碳带 $\delta^{-}$ ，是亲核试剂

# ● 配位特点：

- Fischer卡宾：金属→碳反馈键较弱，通常为18电子体系  
○ Schrock卡宾：形成两根共价键，通常为18-10电子体系

6. 等瓣相似 02:18:38

1）定义 02:19:23

● 核心思想：将主族元素的八隅体规则与过渡金属的18电子规则（或16电子规则）建立对应关系  
● 判断标准：

○ 电子数：前线轨道电子数相似（不要求完全相同）  
- 对称性：分子轨道对称性匹配  
○ 形状：轨道空间分布相似  
○ 能量：前线轨道能量相近

● 符号表示：用共轭符号加小圆圈标记（\~°）  
● 理论特点：霍夫曼在诺奖报告中提出的经验规则，适用于金属簇合物研究但存在大量例外

2）应用举例 02:24:21

\- 主族-过渡金属对应：

○ $CH_{4}$ (8e) \~°Cr(CO)\_6\$ (18e)   
○ $CH_{3}$ (7e) \~°Mn(CO)\_5\$ (17e)   
○ $CH_{2}$ (6e) \~°Fe(CO)4\$ (16e)   
○ CH (5e) \~°Co(CO)\_3\$ (15e)

● 拓展规则：

○ 配位数相同但电荷不同的等电子体（如 $Mn(CO)_{5}$ 与 $Fe(CO)_{5}$ ）  
○ 同时得失相同数量电子仍保持等瓣关系（如 $CH_{3}^{-}$ 与 $Mn(CO)_{5}^{-}$ ）  
○ 配位方式和电子数相同的配体可互换（如 $CO$ 与 $PR_{3}$ ， $Cp$ 与 $C_{6}H_{6}$ ）

3）金属簇合物 02:28:42

8. 金属硼烷 02:43:59

1）等瓣替换

● 等价关系推导：三羰基合铁 $(Fe(CO)_{3})$ 具有14个电子，离18电子规则差4个电子，因此等价于一个BH基团。同理：

# 9. 金属多重键 03:01:39

# 1）键型分类 03:02:20

● 键型层级：金属-金属键可分为单键（single bond）、双键（double bond）、三键（triple bond）、四重键（quadruple bond）和五重键（quintuple bond）

# - 轨道组成：

- σ键：头碰头（head to head）重叠，永远只有一种  
- $\pi$ 键：肩并肩（shoulder to shoulder）重叠，理论上可形成两种  
○ δ键：面贴面（face to face）重叠，涉及 $d_{x^{2}-y^{2}}$ 和 $d_{xy}$ 轨道相互作用

● 键能排序： $\sigma$ 键 $>\pi$ 键 $>\delta$ 键，其中 $\delta$ 键非常弱

# 2）四重键 03:05:14

\- 经典案例： $[Re_2Cl_8]^{2-}$ 是典型的四重键体系 $(\sigma + 2\pi + \delta)$

# ● 电子排布：

○ 先填 $\sigma$ 轨道，再填 $\pi$ 轨道，最后填 $\delta$ 轨道  
○ $d_{x^{2}-y^{2}}$ 轨道通常不参与成键，导致 $\delta$ 键实际只有一个

# - 结构证据：

○ 全重叠构型（ $D_{4h}$ 点群）使 $d_{xy} - d_{xy}$ 轨道重叠最大化  
○ 交叉构型 $(D_{4d})$ 会破坏δ键形成

# ● 键长特征：

○ 四重键键长约为单键的80%  
- 需要通过X射线衍射数据对比验证

# 3）五重键争议 03:13:39

# ● 理论可能性：

- 理论上可形成 $\sigma + 2\pi + 2\delta$ 的五重键  
- 但早期文献普遍认为最多只存在四重键

# - 争议焦点：

○ $d_{x^{2}-y^{2}}$ 轨道是否真正参与成键  
- 2000年后报道的Mn(I)化合物键长缩短现象

# ● 验证难点：

○ 键长变化微小（如2.215Å vs 2.218Å）在实验误差范围内   
○ 配体效应对金属间距的影响难以排除

# ● 研究现状：

部分教科书已收录五重键概念  
- 但学术界仍存在争议，尚无定论

# 2）配体场理论 03:17:26

● 核心概念: angular overlap 模型分析轨道相互作用

# ● 配体分类:

- $\pi$ acceptor: 接受π电子

# - 轨道分析:

○ 需明确配体与金属离子的作用方式

# - 应用技巧:

○ 能根据配体类型预测其对金属轨道的影响  
◦ 掌握配体在坐标系中的定位方法

# 3）电子光谱 03:18:59

# 4）金属有机化合物 03:19:58
