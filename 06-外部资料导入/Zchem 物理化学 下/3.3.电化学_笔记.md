# 5. 极化 02:30:56

# 1）基本概念

- 极化定义：电极电势偏离平衡值的现象称为极化。对于电解池而言，分解电压总是大于计算得到的可逆电池电压。  
● 超电势来源：极化现象本质上来源于超电势（过电位），即实际分解电压与可逆电压的差值。

# 2）超电势的组成

● 电极表示：阳极(anode)记作a，阴极(cathode)记作c  
- 计算公式： $\eta = E$ 可逆 - $E$ 不可逆（定义式），其中 $E$ 可逆可通过能斯特方程计算  
● 影响因素：包含体系内电阻(IR降)和电极极化两部分

# 3）超电势公式

● 塔菲尔公式： $\eta = a + b\ln j$   
○ j: 电流密度, $j = I / A$ (I为电流, A为电极截面积)  
○ a,b: 有单位的经验常数（单位：V）

● 单位电流密度：公式中常使用 $j_{0}$ 作为参考电流密度

# 4）分解电压与电流关系

总关系式： $E$ 分解 $= \sum \eta + E$ 可逆 $+IR$

○ $\sum\eta$ ：阴阳极超电势之和

\- IR: 体系内阻压降

● 电流影响：电流越大，超电势和IR降都越大，导致实际电势与理论值偏差越大

# 5）可逆电势测量方法

● 补偿法原理：通过调节并联电路电阻使I=0，此时测得 $E_{x}=E_{n}$ 即为可逆电动势  
● 实验装置：使用可调标准电源 $E_{n}$ 与待测电池组成补偿电路  
● 关键操作：必须确保测量时回路电流为零

# 6）工业应用案例

\- 氯碱工业

- 工艺特点：电解浓NaCl溶液制取 $H_{2}$ 、 $Cl_{2}$ 和 $\mathrm{NaOH}$   
- 阳极选择：采用石墨而非金属，因为氧气在石墨上的超电势比在金属上大  
副反应：石墨阳极可能被氧化为 $CO_{2}$ ，进而与NaOH反应生成 $Na_{2}CO_{3}$

\- 氢氧燃料电池

- 电动势特性：标准电动势与介质pH无关  
■ 原因：总反应方程中 $H^{+}$ 在两边计量数相同，能斯特方程中pH项相消  
- 计算要点：只需关注标准电极电势差和反应商Q

# 7）重要提醒

● 核心计算能力：能斯特方程的熟练运用是本章最关键技能  
● 考题特点：电解计算题可能涉及复杂体系中离子放电顺序、剩余组分计算等  
● 极化本质：只要有电流通过就会产生极化，导致系统不可逆

# 6. 电化学腐蚀与防腐 02:47:59

1）电化学腐蚀 02:48:55

- 吸氢腐蚀：金属被氧化过程中，还原反应以 $H^{+}$ 作为氧化剂，反应式为 $H^{+}\rightarrow H_{2}$ （氢离子还原为氢气），典型特征是腐蚀过程中会放出氢气。

● 共同点：两种腐蚀类型均需要氧化剂参与，区别在于氧化剂种类不同（ $H^{+}$ 或 $O_{2}$ ）。

# 2）防腐方法 02:49:48

● 保护层法

○ 实现方式：
