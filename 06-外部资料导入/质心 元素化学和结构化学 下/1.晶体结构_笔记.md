# 1）均匀性 01:58

● 定义：理想晶体中任意部分的物理化学性质完全相同  
● 特点：适用于宏观尺度观察，微观上可能存在周期性差异

# 2）各向异性

● 表现：不同晶向上的物理性质（如导电性、光学性质等）存在差异  
● 原因：原子排列的周期性导致不同方向原子间距不同

# 3）自范性

● 特征：具有规则几何外形  
● 关联：与晶体对称性密切相关

# 4）固定熔点 02:30

● 现象：晶体在熔点温度保持恒温直至完全熔化  
● 原理：所有晶格同时破坏需要相同能量

● 基本定义：点阵单位是点阵的基本重复单位，能够代表点阵的平移周期性。

\- 与结构基元的区别：结构基元是晶体结构的最小重复单位，而点阵单位不一定是最小的重复单位，只要能通过平移复原整个点阵即可。

- 素单位：只包含1个点阵点的单位（最小重复单位）  
复单位：包含≥2个点阵点的单位（非最小但能代表周期性）

\- 选取原则：人们会选择便于研究的点阵单位，因此可能选择包含多个点阵点的复单位。

# 1. 干冰结构

● 结构特征：干冰（ $CO_{2}$ ）晶体为简单立方结构，每个晶胞包含4个取向不同的 $CO_{2}$ 分子

○ 局部对称性：单个晶胞内仅存在一条有效的 $C_{3}$ 轴（沿特定体对角线）

# - 空间想象要点：

○ 每个新晶胞的体对角线即为对应分子的 $C_{3}$ 轴方向

● 关键结论：干冰属于立方晶系，特征对称元素为四条不共点的 $C_{3}$ 轴
