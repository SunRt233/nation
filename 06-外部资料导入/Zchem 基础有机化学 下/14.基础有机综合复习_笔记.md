# - 反应机理:

☐ 第一步：过量的碘甲烷与胺发生 $S_{N}2$ 反应，经过质子转移生成四级铵盐。由于产物亲核性比原料更强，反应会持续进行直到完全生成四级铵盐（氮上连四个取代基带正电荷）  
- 第二步：湿润的氧化银（等价于氢氧化银）进行负离子交换，生成氢氧根离子。这是因为碘化银的 $K_{sp}$ 远小于氢氧化银，反应倾向于生成更难溶的碘化银

# - 消除特点:

E2消除：氢氧根进攻β氢，由于四级铵盐位阻极大，会优先进攻位阻最小的β氢（动力学控制）  
○ 产物判断：通过纽曼投影分析过渡态构型，反式共平面的β氢中位阻最小者被进攻，生成霍夫曼产物（取代基最少的烯烃）

\- 环状化合物应用：在环状体系中，消除需要满足反式共平面要求，通常需要在直立键位置进行。三取代氮的大位阻会使环上消除较困难，需要加热促进

# ● 制备条件：

○ 芳香胺（ $R = Ar$ ）在亚硝酸钠和盐酸作用下生成重氮盐（ $Ar - N_{2}^{+}$ ）  
- 烷基重氮盐极不稳定，会立即分解为正碳离子并发生重排

# ● 主要反应类型:

Sandmeyer反应：与铜盐（CuX， $X = Cl / Br / I / CN$ ）反应，在芳环上引入相应取代基。注意氟化亚铜不存在，因此不能直接引入氟  
○ Schiemann反应：与 $HBF_{4}$ 反应可引入氟原子（Balz-Schiemann反应）
