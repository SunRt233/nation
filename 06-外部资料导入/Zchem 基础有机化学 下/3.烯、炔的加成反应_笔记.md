# 1. 例题: SN2 反应机理判断 03:37

# 1）反应机理判断要点

● 底物结构: 一级卤代烃更倾向于发生 $S_{N}$ 2反应，三级卤代烃更倾向于发生E1或 $S_{N}$ 1反应  
- 试剂性质：强碱强亲核试剂（如 $OH^{-}$ ）同时促进取代和消除反应；大位阻碱主要促进消除反应  
- 溶剂影响：极性非质子溶剂（如DMSO）促进 $S_{N}$ 2反应；质子溶剂（如醇类）可能促进 $S_{N}$ 1反应  
● 温度影响: 低温有利于取代反应，高温有利于消除反应

# 2）区域选择性与立体选择性

● 扎伊采夫规则: 消除反应主要生成热力学稳定的多取代烯烃（位阻较大的β-H被消除）  
- 霍夫曼规则: 使用大位阻碱时, 主要消除位阻较小的 $\beta$ -H  
● 立体专一性: 反式消除产物通常占优势，特别是当有两个β-H可供选择时  
● 构型翻转: $S_{N}$ 2反应必然导致构型翻转； $S_{N}$ 1反应可能得到外消旋混合物

# 3）二级卤代烃的特殊性

● 竞争反应: 同时发生 $S_{N}$ 2和E2反应，通常E2略占优势  
- 溶剂效应: 在质子溶剂中, $S_{N}1$ 机理也可能少量发生  
● 产物多样性: 可能生成取代产物（构型翻转/保持）和多种消除产物（顺/反式）

# 4）威廉姆森醚合成法

- 反应特点: 一级卤代烃与醇盐（如 $RO^{-}$ ）主要发生取代生成醚  
● 位阻影响: 大位阻醇盐会抑制 $S_{N}2$ ，使消除反应成为主要路径  
- 离去基团:碘离子是优良离去基团，促进取代反应发生

# 5）反应机理判断流程图

● 一级卤代烃: $S_{N}$ 2占优势，强碱时E2增加  
● 二级卤代烃:E2略优于 $S_{N}$ 2，可能伴随少量 $S_{N}$ 1  
- 三级卤代烃: $E1$ 占优势，高温时更显著

# 6）关键记忆点

- 口诀: "一级取代多, 三级消除强, 二级两边倒, 溶剂温度帮"  
● 易错点: 忽略二级卤代烃在质子溶剂中可能的 $S_{N}$ 1路径  
● 考点: 综合判断反应主要产物及次要产物，包括区域和立体选择性  
● 复习建议: 结合T6讲内容系统掌握卤代烃反应机理判断体系

# 1）反应机理判断

● SN2反应特征：强亲核试剂+弱碱条件（如a选项），位阻小的底物（一级卤代烃）

# E2反应特征：

○ h选项：仅左侧β-H满足反式→生成取代度较低的烯烃

# 13. 例题:重排路径判断 40:44

# 1）碳正离子重排机理

\- 三元环开环迁移：碳正离子通过三元环开环进行迁移是合理的，因为这种迁移可以释放环张力，使体系能量降低。

- 进一步重排可能性：理论上碳正离子可以继续重排形成更稳定的烯丙基正离子，通过质子转移(proton transfer)实现。  
- 产物导向性：实际反应中是否发生二次重排取决于最终产物的结构，需要根据实验观测到的产物来反推反应路径。

# 2）竞争反应分析 41:20

# - 两种可能路径：

◦ 仅发生第一次碳正离子重排后即被亲核试剂捕获  
○ 在重排的同时发生亲核试剂进攻（协同过程）

# - 反应条件影响：

- 高温/长时间反应：有利于重排产物生成  
- 低温/短时间反应：倾向于直接捕获产物

# ● 机理合理性判断：

○ 机理没有绝对的对错，只有合理性的区别  
- 最终以实验观测到的产物为准来反推可能机理

# 3）关键注意事项

● 氢迁移优先性：第一步碳正离子重排通常优先考虑氢迁移而非三元环开环。  
● 竞争反应速率：当存在竞争反应时，反应速率快的路径往往主导产物分布。  
● 实验证据优先：所有机理推导都必须以实验观测结果为依据，不能脱离实际产物。

# 1）酸催化醚的制备 47:13

- 反应机理：在酸性条件下，醇羟基（R-OH）质子化形成好的离去基团（ $R-OH_{2}^{+}$ ），随后另一个醇分子作为亲核试剂进攻，通过 $SN_{1}$ 或 $SN_{2}$ 机理生成醚（ $R-O-R'$ ）。  
● 适用性：仅适用于制备对称醚（R - O - R），若制备不对称醚（R - O - R'）会产生混合产物（R - O - R 和 R' - O - R'）。  
● 关键步骤：羟基质子化→离去基团形成→亲核进攻→产物生成。

# 2）碱催化醚的制备（Williamson合成法）48:33

# - 反应步骤：

- 醇与强碱（如NaH或NaOH）反应生成醇钠（ $R-O-Na^{+}$ ）

● 碱选择：常用氢化钠（NaH）或氢氧化钠（NaOH），醇钠为中间体。
