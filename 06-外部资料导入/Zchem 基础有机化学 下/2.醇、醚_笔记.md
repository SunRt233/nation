# 1）基团大小比较规则

- 原子序数优先: 直接连接原子的原子序数越大, 基团越大。例如: $O > N > C$   
- 次级原子比较：当第一个原子相同时（如都是碳），比较连接的下一个原子中原子序数最大的。例如：

○ 乙基 $(CH_{2}CH_{3})>$ 甲基 $(CH_{3})$ ，因为乙基第二个原子有碳（连接H和C），而甲基只有氢

○ 甲氧基 $(OCH_{3})>$ 乙基，因为氧原子序数大于碳

# 2）Z/E命名法判定

# ● 判定步骤:

○ 比较双键一侧的a和b基团大小  
○ 比较另一侧的c和d基团大小

# ● 构型判定:

○ Z式: 当a>b且c>d时，大基团在双键同侧  
○ E式: 当a > b但c < d时，大基团在双键异侧

● 记忆技巧: "Z"像"同"字的起笔，表示同侧；"E"像"异"字的起笔，表示异侧

# 3）烯烃稳定性规律

● 取代度原则: 双键碳上连接的取代基越多，烯烃越稳定（most stable）  
例如：四取代 > 三取代 > 二取代 > 一取代

● 应用提示: 该规律在判断反应产物稳定性时非常重要

# ○ 碱的位阻决定产物类型:

■ 大位阻碱(如t-BuOK、LDA、三乙胺)优先进攻位阻小的β-H，生成动力学产物(霍夫曼产物)  
■ 小位阻碱(如NaOH、EtONa)优先进攻位阻大的β-H，生成热力学更稳定的扎伊采夫产物

# - 稳定性原理:

■ 扎伊采夫产物更稳定是因为双键上取代基更多   
■ 霍夫曼产物是动力学控制产物，活化能较低但热力学稳定性较差

# ○ 常见碱分类:

■ 小位阻碱：氢氧化钠、乙醇钠等  
■ 大位阻碱：叔丁醇钾、LDA(二异丙基胺锂)、三乙胺等

# ○ 实际反应特点:

■ 定性做题时通常主副产物都需要考虑  
■ 实际反应中常得到混合物，需要纯化分离  
■ 大位阻碱的选择性更好，因为活化能差异 $(\Delta E_{a})$ 更大

# E2反应的立体化学 14:35

# ○ 立体选择性与立体专一性

# 立体选择性:

- 生成立体异构体混合物  
- 其中一个产物占优势  
● 例子：β-碳上有两个H时，通常得到反式烯烃为主产物

# ■ 立体专一性:

● 由于反应过程的空间专一性，只生成唯一立体构型的产物  
● 例子：β-碳上只有一个H时，必须反式共平面消除，得到唯一产物

# ■ 关键区别:

● 立体选择性：产物是混合物但有主次之分  
- 立体专一性：产物唯一

# 3）例题:伊万消除反应 28:44

# - 反应机理特征

- 反应条件：浓硫酸溶液加热是三级醇发生E1消除的经典条件  
- 两步过程：先质子化羟基形成好的离去基团，再发生碳正离子中间体形成  
平面构型：碳正离子为 $sp^{2}$ 杂化平面结构，导致位阻影响减弱

# - 产物选择性

热力学主导：由于平面结构消除位阻因素，主要生成取代度更高的稳定产物  
- 弱碱作用：仍需路易斯碱（如水或醇）参与拔氢过程，氢不能自动离去  
○ 立体无差别：平面构型使碱可从任意方向进攻，不产生特定立体选择性

# 4）例题:反应机理书写 33:02

# ● 常见错误点

- 碱的遗漏：即使弱碱（如水）也必须明确写出参与拔氢过程  
- 重排完整性：需注意多步碳正离子重排的可能性，特别是邻位有三级碳时  
- 质子化步骤：在酸性条件下需先明确羟基的质子化过程

# - 反应条件解析

- 浓硫酸环境：实际参与反应的碱可能是微量水或硫酸根上的氧  
- 重排优先级：碳正离子重排时，小基团（如甲基）优先迁移  
- 产物验证：最终需检查双键位置是否符合热力学最稳定构型（大基团反式）

- 溶剂化效应(Solvation): 在液相中, 甲醇( $CH_{3}OH$ )酸性>乙醇(EtOH)>叔丁醇(t - BuOH), 因为小体积的甲氧基负离子更容易被溶剂分子稳定; 但在气相中顺序相反, 由于甲基的吸电子诱导效应导致甲醇酸性最弱。
