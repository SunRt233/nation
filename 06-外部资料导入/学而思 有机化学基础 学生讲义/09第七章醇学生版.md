# 第七讲 醇

醇是指含有-OH 连接在饱和的 $sp^{3}$ 杂化的碳上的化合物。醇是水的有机衍生物，其中水的一个氢被有机基团所取代：H-O-H 与 R-O-H。

![[09第七章醇学生版_images/79586dd153e8317468eaf53ab8f0bf1369c0fb8c3f2373b9cce89262f76bce03.jpg]]

![[09第七章醇学生版_images/22f6ac9eb590026bd907b22fde4e0201e9a0f9d7c371fc19ab323bb065a6c398.jpg]]

根据羟基所连碳原子的级数，醇可以被分为一级醇(伯醇)、二级醇（仲醇）和三级醇（叔醇）；根据所含羟基数目来分，含一个羟基的称为一元醇，含两个羟基的称为二元醇，二元以上的醇统称为多元醇。

醇在自然界广泛存在，并有许多工业和药物用途。例如甲醇是最为重要的工业产品之一。历史上，甲醇是通过在隔绝空气下加热木头而获得的，因此又被称为木醇。如今全世界每年大约生产四千万吨（一百三十亿加仑）甲醇，大多数是通过一氧化碳和氢气的催化还原制备的。甲醇对人体有毒，少量（15 mL）造成失明，大量（100-250 mL）致死。工业上，它被用作生产甲醛和醋酸的溶剂与原料。

乙醇是第一个被制备和纯化的有机化学品。从谷物和糖类的发酵生产乙醇已被进行了大约9000年，通过蒸馏对其纯化可以追溯到十二世纪。如今全世界每年大约生产五千五百万吨（一百八十亿加仑）乙醇，大多数是通过玉米、大麦、高粱和其他植物原料的发酵制备的。全部基本上都用来做汽车燃料了。

在工业上被用作溶剂或化学中间体的乙醇是通过乙烯在高温下酸催化水合而大量制备的。

# 三、醇的酸碱性

醇与水的另一个相似性是既有弱酸性又有弱碱性。作为弱碱，它们被强酸可逆地质子化为氧鎓离子 $ROH_{2}^{+}$ 。

![[09第七章醇学生版_images/a14865e42d43cb71bcfac9c180bdebdf85f8495cff1d376270b644be4e4e1f29.jpg]]

作为弱酸，它们的稀水溶液微弱地通过将质子给水生成 $H_{3}O^{+}$ 与烷氧负离子 $RO^{-}$ 。

![[09第七章醇学生版_images/8cd0c3e793c51c650cc2cedc7310b6f17f3700ec93f6ecb0e077165a86b77a55.jpg]]

简单的醇如甲醇和乙醇酸性和水差不多，但更高取代的醇如叔丁醇就更弱一些。取代基也有着显著的影响：例如 2,2,2-三氟乙醇酸性比乙醇强大约 3700 倍。酚与醇的含硫类似物硫醇酸性比水强得多。

液相中醇上烷基取代基的影响主要是由于酸电离时形成的烷氧负离子的溶剂化。烷氧负离子越容易被水溶剂化，它就越稳定，它的形成能量上就越有利，母体醇的酸性就越强。例如，像在甲醇形成的不存在位阻的烷氧负离子中，氧原子在立体上是可以接近并容易被溶剂化的。而像在叔丁醇形成的存在位阻的烷氧负离子中，氧原子是不容易被溶剂化，因此是更不稳定的。而各类醇的共轭酸在水中酸性的强弱也由位阻决定，共轭酸的空间位阻小，与水形成氢键而溶剂化的程度愈大，这个共轭酸就越稳定，质子不易离去，酸性就较低。

诱导效应在决定醇的酸性时也是很关键的。例如吸电子的卤素取代基通过将电荷分散到更大的体积而稳定烷氧负离子，因此令醇的酸性更强。

由于醇是弱酸，它们不与弱碱如胺和碳酸氢根反应，与金属氢氧化物如 NaOH 也只发生有限程度的反应。然而醇与碱金属和强碱，如氢化钠（NaH）、氨基钠（ $NaNH_{2}$ ）和格氏试剂（RMgX）反应。

# 四、醇的制备

醇在有机化学中占据核心地位。它既能从许多其他类型的化合物制备（烯烃、卤代烃、酮、酯、醛还有其他），又能被转化为同样多类型的化合物。

1、醇能够通过烯烃的水合来制备。由于用水直接水合在实验室中反应通常不好，常用两种间接的方法：硼氢化-氧化和羟汞化-脱汞。注意二者产物的区别。

在实验室与生命体中最为普遍的制备醇的方法是通过羰基化合物的还原。就像烯烃的还原是氢加到 C=C 键上生成烷烃一样，羰基化合物的还原是氢加到 C=O 键上生成醇。所有的羰基化合物都能被还原，包括醛、酮、羧酸和酯。

# 2、醛和酮的还原

醛很容易被还原为一级醇，而酮被还原生成二级醇。

实验室中取决于反应环境, 使用许多种试剂来还原醛和酮, 但通常选用硼氢化钠, $NaBH_{4}$ , 因为它安全又容易操作。硼氢化钠是可以在空气中称量的白色晶体, 可以在水或醇溶液中使用。

氢化铝锂， $\mathrm{LiAlH_4}$ 是另一种常用于还原醛和酮的试剂。氢化铝锂是溶于乙醚和四氢呋喃的浅灰色粉末，反应活性比 $\mathrm{NaBH_4}$ 高，但危险性也更大。其与水剧烈反应，加热到 $120^{\circ}\mathrm{C}$ 时爆炸性分解。

这些还原的机理涉及亲核性的氢负离子（H: $^{-}$ ）对羰基碳原子的正极性的亲电性碳的加成。最初的产物是烷氧负离子，它在第二步被加入的 $H_{3}O^{+}$ 质子化生成产物醇。

# 3、羧酸和酯的还原

羧酸和酯被还原产生一级醇。

这些反应并不像醛和酮的还原那样快。 $NaBH_{4}$ 还原酯非常慢，根本不能还原羧酸。羧酸和酯通常是被活性更强的试剂 $LiAlH_{4}$ 所还原的。所有的羰基化合物，包括羧酸、酯、酮和醛都被 $LiAlH_{4}$ 所还原。注意在醛和酮的还原中只有一个氢被转移到羰基碳上，但在羧酸和酯的还原中有两个氢被转移到羰基碳上。

# 五、醇的反应

# 1、醇到卤代烷的转化

三级醇与 HCl 或 HBr 在 $0^{\circ}$ C 通过 $S_{N}1$ 机理经过碳正离子中间体进行反应。然而一级和二级醇对酸抵抗力更强，转化为卤代烃最好是用 $SOCl_{2}$ 或 $PBr_{3}$ 处理通过 $S_{N}2$ 机理进行。

当酸对羟基氧原子质子化时，三级醇与 HX 的反应通过 $S_{N1}$ 机理发生，水离去生成碳正离子，碳正离子和亲核性的卤素负离子反应生成产物卤代烷。

![[09第七章醇学生版_images/2fc15823f2f145abf77fa575ef3e153da938f88d12836528d7efb9dc5f9f3108.jpg]]

<details>

</details>

一级和二级醇与 $\mathrm{PBr}_3$ 的反应通过 $\mathbf{S}_{\mathrm{N}}2$ 机理发生。 $\mathbf{S}_{\mathrm{N}}2$ 反应中羟基自己是不好的离去基团，但醇与 $\mathrm{PBr}_3$ 的反应将-OH变为更好的离去基团，二溴磷酸酯（-OPBr2）很容易被背后的亲核取代所推走。碘代烷可由P和I代替三碘化磷，氯代烷常用五氯化磷与醇反应制备。

醇与亚硫酰氯（ $\mathrm{SOCl}_2$ ）反应，可直接得到氯代烷，同时生成二氧化硫和氯化氢两种气体，在反应过程中这些气体都离开了反应体系，有利于反应向生成物方向进行，该反应不仅速率快，条件温和，产率高，无其他副产物。一般用过量的亚硫酰氯保持微沸，是一个很好的制备氯代烷的方法：

$$
R O H + S O C l _ {2} \longrightarrow R C l + S O _ {2} \uparrow + H C l \uparrow
$$

# 4、醇到酯的转化

醇与羧酸反应生成酯，该反应在实验室与生命体中都很常见。在实验室中，如果用强酸做催化剂反应可以一步进行。但更为常见的是先通过将羧酸转化为酰氯来增强其反应活性，之后再与醇反应。

# 5、醇的氧化

醇最有价值的反应大概就是氧化生成羰基化合物——与羰基化合物还原为醇正好相反。一级醇生成醛或羧酸，二级醇生成酮，但三级醇通常和大多数氧化剂不反应。

# 六、醇的保护

分子中的一个官能团对想要让该分子中另一个官能团发生的反应有干扰这样的事情经常发生，尤其是在复杂分子的合成中。例如我们在本章前面见过，格氏试剂不能从卤代醇制备，因为 C-Mg 键与同一个分子中存在的酸性-OH 基团不能相容。

当这种不相容产生时, 有时能够通过保护干扰的官能团而避免这个问题。保护涉及到三个步骤: (1) 引入一个保护基来限制干扰作用; (2) 进行想要的反应; (3) 除去保护基。

最常见的醇保护方法是通过与三烷基氯硅烷, $\mathrm{Cl - SiR_3}$ 反应生成三烷基硅醚, $\mathrm{R}'\text{-O-SiR}_3$ 。经常使用的是三甲基氯硅烷, 反应在碱如三乙胺的存在下进行, 以帮助醇形成烷氧基负离子并从反应中除去副产物 $\mathrm{HCl}$ 。

![[09第七章醇学生版_images/d84473941eb538f58872971dfd56e3d3355b26b1dfff6a6448d28f665108a2a0.jpg]]

形成醚的步骤是硅原子上的一个烷氧负离子类似于 $S_{N}2$ 的反应，同时伴随着离去基团氯负离子的脱去。但不像大多数 $S_{N}2$ 反应，该反应发生在三级中心上——三烷基取代的硅原子。由于第三周期的硅原子比碳大，形成的键更长，令反应可以发生。因此与硅相连的三个甲基取代基位阻更小，导致其能够反应，而类似的叔丁基氯不反应。

像我们下一章将要学的其他醚一样，TMS 醚相对是不活泼的。它们没有活泼氢也不与氧化剂、还原剂或格氏试剂反应。然而，它们与酸的水溶液或氟离子反应再次生成醇。

# 七、醇的鉴别

浓盐酸和无水氯化锌的混合物称为 Lucas 试剂。可用来鉴别六碳和六碳以下的一级、二级、三级醇，将三种醇分别加入盛有 Lucas 试剂的试管中，经振荡后可发现，三级醇立即反应，生成油状氯代烷，不溶于酸中，溶液浑浊后分两层，反应放热；二级醇 2\~5 min 反应，放热不明显，溶液分两层；一级醇经室温放置 1 h 仍无反应，必须加热。

一些一级醇如烯丙醇和苯甲型醇也可以很快反应，因为 $p-\pi$ 共轭，很容易形成碳正离子进行 $S_{N}1$ 反应。

各类醇与 Lucas 试剂的反应速率为:

烯丙型醇，苯甲型醇，三级醇 > 二级醇 > 一级醇
