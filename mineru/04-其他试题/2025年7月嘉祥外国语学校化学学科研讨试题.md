线性自由能关系(linear free energy relationship, LFER)是物理有机化学中用来描述反应速率常数或平衡常数随取代基结构变化的经验关系。此题中我们讨论反应速率常数 $\mathbf { k }$ 的线性自由能关系，其核心思想是：结构的微小变化(如芳环上的取代基)会以可预测的方式影响反应活性。这种结构与反应性之间的线性关系，有助于推测机理，设计更有效的反应体系。

经典的线性自由能关系模型由哈米特方程描述: $\mathrm { l g } \left( k _ { \mathrm { R } } / k _ { \mathrm { H } } \right) { = } \mathrm { p o }$ 其中 $k _ { \mathrm { R } } / k _ { \mathrm { H } }$ 后表示含取代基R和不含取代基的化合物反应速率常数之比。
