# 离子半径比与配位数-判断CsCl/NaCl/ZnS结构类型

## 题目

已知下列离子半径（单位：pm）：

| 离子 | 半径 |
|------|------|
| $\mathrm{Cs^+}$ | 167 |
| $\mathrm{Na^+}$ | 102 |
| $\mathrm{Zn^{2+}}$ | 74 |
| $\mathrm{Cl^-}$ | 181 |
| $\mathrm{O^{2-}}$ | 140 |

(1) 根据离子半径比规则，分别计算 $\mathrm{CsCl}$、$\mathrm{NaCl}$、$\mathrm{ZnS}$（实际为 $\mathrm{Zn^{2+}/S^{2-}}$，$r_{\mathrm{S^{2-}}} = 184\ \mathrm{pm}$）的 $r_+/r_-$ 值，并判断各自预期的阳离子配位数。

(2) 半径比规则预测的配位数与实际晶体结构是否一致？简要说明半径比规则的适用范围和局限性。

(3) 在高压下，$\mathrm{CsCl}$ 会从 CsCl 型转变为 NaCl 型，还是 NaCl 型转变为 CsCl 型？从离子半径的角度简要解释。

## 解析

### 分析

1. 半径比规则：$r_+/r_-$ 决定阳离子在阴离子密堆积中的配位数和占据的空隙类型
2. 临界值：$r_+/r_- < 0.225$（CN=3-4），$0.225{-}0.414$（CN=4, ZnS型），$0.414{-}0.732$（CN=6, NaCl型），$>0.732$（CN=8, CsCl型）
3. 高压使离子压缩，$r_-$ 比 $r_+$ 更易压缩（阴离子电子云更弥散），$r_+/r_-$ 增大

### 解答

**(1) 半径比计算**

$$\mathrm{CsCl}:\ \frac{r_+}{r_-} = \frac{167}{181} = 0.923 > 0.732$$
预测配位数：**CN = 8**（CsCl 型结构）✅

$$\mathrm{NaCl}:\ \frac{r_+}{r_-} = \frac{102}{181} = 0.564 \in [0.414, 0.732]$$
预测配位数：**CN = 6**（NaCl 型结构）✅

$$\mathrm{ZnS}:\ \frac{r_+}{r_-} = \frac{74}{184} = 0.402 \in [0.225, 0.414)$$
预测配位数：**CN = 4**（ZnS 型结构，闪锌矿或纤锌矿）✅

**(2) 半径比规则的评价**

上述三个例子中预测与实际**一致**。

半径比规则是**经验的半定量规则**，适用范围和局限性：
- 适用于**纯离子晶体**（碱金属卤化物、碱土金属氧化物等），假设离子为刚性球
- 对于**共价成分显著**的晶体（如 $\mathrm{AgI}$、$\mathrm{HgS}$），预测可能失准
- 忽略了**极化效应**——大阳离子与小阴离子间极化作用会降低有效配位数
- 只考虑了阳离子配位数，对复杂结构（如钙钛矿、尖晶石）需结合其他规则

**(3) 高压下的相转变**

在高压下，$\mathrm{CsCl}$ 保持 CsCl 型（CN=8），而**NaCl 型可能在极高压力下转变为 CsCl 型**。因为高压使阴离子（$\mathrm{Cl^-}$）比阳离子更易被压缩，$r_+/r_-$ 增大，当超过 0.732 阈值时 NaCl 型 → CsCl 型。

反过来，$\mathrm{CsCl}$ 在高压下不会变为 NaCl 型（$r_+/r_-$ 本已 > 0.732，高压使其更大）。

> 补充：实际实验中，NaCl 在约 30 GPa 压力下确实转变为 CsCl 型结构。

### 反思

半径比规则是最基础的结构预测工具，但需记住它只是经验规则。竞赛中的常见陷阱：(1) 将 $r_+/r_-$ 颠倒计算；(2) 忘记 ZnS 有两种变体（闪锌矿 CN=4 和纤锌矿 CN=4，配位数相同但堆积方式不同）；(3) 忽略压力对 $r_+/r_-$ 的影响方向。

## 答案

(1) CsCl: 0.923（CN=8）；NaCl: 0.564（CN=6）；ZnS: 0.402（CN=4）。预测与实际结构一致。

(2) 对纯离子晶体预测良好，局限性在于忽略共价性和极化效应。

(3) NaCl 型在高压下可能转变为 CsCl 型，因 $r_-$ 压缩比例大于 $r_+$，$r_+/r_-$ 增大。
