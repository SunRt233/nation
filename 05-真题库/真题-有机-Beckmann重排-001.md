# Beckmann重排-环己酮肟经酸催化生成己内酰胺

## 题目

环己酮肟在浓 $\mathrm{H_2SO_4}$ 或 $\mathrm{PCl_5}$ 作用下发生 Beckmann 重排，生成己内酰胺——尼龙 6 的单体。

(1) 写出环己酮肟的结构式（标明 E/Z 异构）。环己酮肟存在两种几何异构体，哪一种在 Beckmann 重排条件下能发生反应？为什么？

(2) 写出 Beckmann 重排的完整机理，画出关键步骤中基团迁移的立体电子效应。

(3) 工业上利用 Beckmann 重排年产数百万吨己内酰胺。简述在工业催化方面的主要进展（从传统的浓硫酸法到现代催化方法）。

(4) 若使用不对称酮的肟（如苯乙酮肟 $\mathrm{PhC(=NOH)CH_3}$）进行 Beckmann 重排，如何预测哪个基团发生迁移？写出苯乙酮肟重排的两种可能产物并判断主产物。

## 解析

### 分析

1. Beckmann 重排：肟在酸性条件下，与氮上 OH 处于**反式**（anti）关系的基团发生 1,2-迁移至氮上，同时 N-OH₂⁺ 作为离去基团离去 → 生成酰胺（或经水解得到胺/酸）。
2. 立体化学关键：只有与离去基团（-OH₂⁺）处于**反式**的基团能迁移。
3. 工业应用：从传统浓硫酸催化到现代固体酸催化剂（如 TS-1 分子筛等）。
4. 不对称肟的迁移选择性：取决于肟的 E/Z 构型和迁移基团的迁移倾向（芳基 > 烷基）。

### 解答

**(1) 环己酮肟的异构**

环己酮肟有两种几何异构体（E/Z 异构体，对肟而言通常用 syn/anti 表示 OH 与 α-碳上基团的关系）：

- **(E)-环己酮肟**（anti）：OH 与环的 C-C 键在异侧
- **(Z)-环己酮肟**（syn）：OH 与环的 C-C 键在同侧

在 Beckmann 重排中，**与 OH 处于反式（anti）的基团才迁移**。

对于环己酮肟，两种异构体都可以发生 Beckmann 重排，因为无论 OH 在哪一侧，与之呈 anti 关系的总是环上的碳（环己酮的两个 α-碳对称等价）→ 迁移基团始终是环亚甲基 → 产物均为己内酰胺。

若环己酮肟以 Z 式存在（OH 与环的 α-亚甲基同侧），则 α-亚甲基中与 OH 呈 anti 关系的碳（另一端）发生迁移；E 式同理。由于环己酮的两个 α-位等价，两种异构体重排得到**同一产物**——己内酰胺。

**(2) 完整机理**

**步骤 1**：肟的 OH 质子化（活化离去基团）

$$\mathrm{\!-\!C=N\!-\!OH + H^+ \longrightarrow \!-\!C=N\!-\!OH_2^+}$$

**步骤 2**：水分子离去，同时与 OH₂⁺ 处于 anti 的基团发生 1,2-迁移

$$\mathrm{\!-\!C=N\!-\!OH_2^+ \longrightarrow [\!-\!C^+\!\!-\!N] \longrightarrow \!-\!C(=NH^+)\!-\!}$$

这是 Beckmann 重排的核心步。在 OH₂⁺ 离去的同时（或之后），处于 anti 位的 R 基团带着自己的 C 从碳迁移到缺电子的 N 上。这一步中 N₂ 的离去和迁移是协同的（但异步——N₂ 离去在先，迁移在后）。

**步骤 3**：水解（或与水反应）

对环己酮肟：迁移后生成的腈正离子 $\mathrm{-\!C(=NH^+)\!-\!}$ 加水 → 酰胺

$$\mathrm{-\!C(=NH^+)\!-\! + H_2O \longrightarrow \!-\!C(=O)\!-\!NH\!-\! + H^+}$$

最终产物：己内酰胺（ε-己内酰胺，七元环内酰胺）。

**(3) 工业催化进展**

**传统法（浓硫酸法）**：
- 环己酮肟 + 发烟硫酸 → 己内酰胺
- 产生大量低价值的硫酸铵副产物（每吨己内酰胺约 2-3 吨硫酸铵）
- 腐蚀性强，设备要求高

**现代催化方法**：
- **气相 Beckmann 重排**：使用固体酸催化剂（如高硅 MFI 型分子筛、B-MFI 等），环己酮肟在气相通过催化剂床层发生重排
- 代表性的催化剂：高硅 ZSM-5（Si/Al 比极高）或全硅沸石 Silicalite-1，其弱酸性位点足够催化重排但不会导致深度副反应
- 优点：无硫酸铵副产物、连续操作、环境友好
- **Sumitomo 工艺**（已工业化）：使用流化床反应器 + 高硅 MFI 分子筛催化剂

**(4) 不对称酮肟的迁移选择性**

苯乙酮肟 $\mathrm{PhC(=NOH)CH_3}$：

- **(E)-苯乙酮肟**（OH 与 Ph 在 anti 位）→ **Ph 迁移** → $\mathrm{CH_3CONHPh}$（乙酰苯胺）
- **(Z)-苯乙酮肟**（OH 与 CH₃ 在 anti 位）→ **CH₃ 迁移** → $\mathrm{PhCONHCH_3}$（N-甲基苯甲酰胺）

**判断主产物的规则**：
1. 与 OH 处于 anti 位的基团优先迁移（立体化学控制）
2. 当两种异构体并存时，迁移倾向：芳基 > 烷基 > H

因此，对于 $\mathrm{PhCOCH_3}$ 的肟，anti 位有 Ph 的异构体 → 主产物为**乙酰苯胺**（Ph 迁移产物）。实际上，苯乙酮肟通常以 E 式为主（Ph 与 OH 在 anti 侧，因为 Ph 和 OH 之间的氢键排斥使得 Z 式能量较高）。

### 反思

Beckmann 重排的三要素：(1) **立体化学决定迁移基团**（与 OH 反式的基团迁移）；(2) **同步但异步**（N₂ 的离去和基团迁移协同但 timing 不同）；(3) **迁移倾向**：芳基 > 烷基 ~ H。与 Beckmann 重排类似的"缺电子氮上的重排"还有 Curtius 重排（酰基叠氮）、Hofmann 重排（酰胺 + Br₂/NaOH）、Schmidt 反应等——它们都属于"氮烯/nitrenium 化学"的范畴。
