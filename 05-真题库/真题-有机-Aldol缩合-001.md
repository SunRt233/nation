# Aldol缩合-乙醛在碱催化下的缩合与脱水

## 题目

乙醛 $\mathrm{CH_3CHO}$ 在稀 $\mathrm{NaOH}$ 溶液中发生 Aldol 缩合反应。

(1) 写出碱催化下 Aldol 缩合反应的通用机理（三步）。以乙醛自身缩合为例，写出每一步的化学方程式。

(2) 反应初期在低温（5°C）下得到一种 β-羟基醛中间体 C₄H₈O₂；加热后该中间体失水得到不饱和醛 C₄H₆O。写出这两种化合物的结构式，并说明加热脱水产物的立体化学特征（E/Z）。

(3) 若将乙醛与苯甲醛 $\mathrm{PhCHO}$ 置于稀碱条件下进行交叉 Aldol 缩合，可能得到几种产物？哪个是主要的？应采取什么实验操作来提高交叉缩合的选择性？

## 解析

### 分析

1. Aldol 缩合核心三步：**烯醇负离子生成 → 亲核加成 → 脱水**。
2. 乙醛自身缩合：一分子乙醛形成烯醇负离子作为亲核试剂，进攻另一分子乙醛的羰基碳。
3. 交叉 Aldol 缩合需考虑哪种醛优先形成烯醇负离子：$\mathrm{PhCHO}$ 无 α-H，不能形成烯醇负离子，只能做受体；乙醛有 α-H 可形成烯醇负离子做供体。

### 解答

**(1) 通用机理（以乙醛自身缩合为例）**

**第一步——烯醇负离子生成**：

$$\mathrm{CH_3CHO + OH^- \rightleftharpoons \overset{-}{C}H_2CHO + H_2O}$$

乙醛的 α-H 酸性较强（$\mathrm{p}K_\mathrm{a} \approx 17$），在 $\mathrm{OH^-}$ 作用下脱去质子生成烯醇负离子（被 $\ce{C=O}$ 共轭稳定）。

**第二步——亲核加成**：

$$\mathrm{CH_3CHO + \overset{-}{C}H_2CHO \longrightarrow CH_3CH(O^-)CH_2CHO}$$

烯醇负离子作为亲核试剂（碳端）进攻另一分子乙醛的羰基碳 → 生成烷氧负离子中间体。

**第三步——质子化**：

$$\mathrm{CH_3CH(O^-)CH_2CHO + H_2O \longrightarrow CH_3CH(OH)CH_2CHO + OH^-}$$

得到 β-羟基醛（Aldol，醛+醇）。

**(2) 中间体与脱水产物**

低温产物：$\mathrm{CH_3CH(OH)CH_2CHO}$（3-羟基丁醛，"Aldol"）

加热脱水产物：$\mathrm{CH_3CH=CHCHO}$（2-丁烯醛 / 巴豆醛）

脱水经历 E1cb 机理（烯醇负离子消除 OH⁻），受热力学控制，主要生成更稳定的 **(E)-2-丁烯醛**（两个大基团 $\mathrm{CH_3}$ 和 $\mathrm{CHO}$ 在双键异侧）。

**(3) 交叉 Aldol 缩合**

苯甲醛 $\mathrm{PhCHO}$ **无 α-H** → 不能形成烯醇负离子，只能作为**亲电受体**。乙醛有 α-H → 是唯一的烯醇负离子供体。

若简单混合，可能的反应：
- 乙醛自身缩合（副反应）
- 乙醛（供体）+ 苯甲醛（受体）= $\mathrm{PhCH=CHCHO}$（肉桂醛，目标产物）

**提高选择性的方法**：将乙醛的稀碱溶液**缓慢滴加**到过量苯甲醛中。这样：
- 反应体系中苯甲醛浓度始终远大于乙醛 → 生成的烯醇负离子优先与苯甲醛反应
- 同时滴加速率慢，防止局部乙醛浓度过高发生自身缩合

主产物：**肉桂醛**（$\mathrm{PhCH=CHCHO}$），因为 $\mathrm{PhCHO}$ 的羰基碳比乙醛更缺电子（苯环的吸电子共轭效应使羰基碳更亲电），更容易被烯醇负离子进攻。

### 反思

Aldol 缩合的关键是区分**供体**和**受体**：有 α-H 的羰基化合物可做供体（形成烯醇负离子），无 α-H 的只能做受体。交叉 Aldol 缩合中选择性控制的黄金法则：**慢加快搅**（将供体缓慢滴入受体）或使用预形成的烯醇锂（如 LDA 定量生成）。
