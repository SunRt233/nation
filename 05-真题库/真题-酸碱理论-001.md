# 酸碱质子理论-缓冲溶液pH计算

## 题目

欲配制 pH = 5.00 的缓冲溶液，需在 50.0 mL 0.100 mol·L⁻¹ HAc 溶液中加入多少克 NaAc·3H₂O？已知 HAc 的 $K_a = 1.76 \times 10^{-5}$，NaAc·3H₂O 的摩尔质量为 136.1 g·mol⁻¹。

## 解析

### 分析

1. HAc-NaAc 是典型的**共轭酸碱对**缓冲体系
2. 用 **Henderson-Hasselbalch 方程**：$\mathrm{pH} = \mathrm{p}K_a + \lg\frac{[\mathrm{A^-}]}{[\mathrm{HA}]}$
3. $\mathrm{p}K_a = -\lg(1.76 \times 10^{-5}) = 4.755$
4. 注意加入 NaAc·3H₂O 后总体积仍是 50.0 mL（近似），直接由所需浓度比求物质的量

### 解答

**步骤 1**：计算所需 $[\mathrm{Ac^-}]/[\mathrm{HAc}]$ 比值

$$\mathrm{pH} = \mathrm{p}K_a + \lg\frac{[\mathrm{Ac^-}]}{[\mathrm{HAc}]}$$

$$5.00 = 4.755 + \lg\frac{[\mathrm{Ac^-}]}{[\mathrm{HAc}]}$$

$$\lg\frac{[\mathrm{Ac^-}]}{[\mathrm{HAc}]} = 0.245$$

$$\frac{[\mathrm{Ac^-}]}{[\mathrm{HAc}]} = 10^{0.245} = 1.76$$

**步骤 2**：计算所需 NaAc·3H₂O 的质量

HAc 的物质的量：$n_{\mathrm{HAc}} = 0.100 \times 0.0500 = 0.00500\ \mathrm{mol}$

所需 Ac⁻ 的物质的量：$n_{\mathrm{Ac^-}} = 1.76 \times n_{\mathrm{HAc}} = 1.76 \times 0.00500 = 0.00880\ \mathrm{mol}$

NaAc·3H₂O 的质量：$m = 0.00880 \times 136.1 = 1.20\ \mathrm{g}$

## 答案

需加入 **1.20 g** NaAc·3H₂O。
