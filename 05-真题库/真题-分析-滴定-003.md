# 氧化还原滴定-KMnO4法测定H2O2含量

## 题目

过氧化氢（$\mathrm{H_2O_2}$）是常用的消毒剂和氧化剂，其含量可用高锰酸钾法测定。反应在酸性介质中进行：

$$\mathrm{2MnO_4^- + 5H_2O_2 + 6H^+ \longrightarrow 2Mn^{2+} + 5O_2 + 8H_2O}$$

准确量取某市售双氧水样品 2.00 mL，置于 100 mL 容量瓶中，加水稀释至刻度，摇匀。移取该稀释液 25.00 mL 于锥形瓶中，加入适量 $\mathrm{H_2SO_4}$ 酸化，用 0.02000 mol·L⁻¹ 的 $\mathrm{KMnO_4}$ 标准溶液滴定至溶液呈稳定的微红色（30 s 内不褪色），消耗 $\mathrm{KMnO_4}$ 溶液 18.50 mL。

已知：$M(\mathrm{H_2O_2}) = 34.02\ \mathrm{g \cdot mol^{-1}}$。

(1) 写出滴定反应中氧化剂和还原剂的半反应，标出电子转移数。

(2) 计算原双氧水样品中 $\mathrm{H_2O_2}$ 的质量体积百分浓度（% w/v，即 g/100 mL）。

(3) 本滴定实验中，为何 $\mathrm{KMnO_4}$ 标准溶液自身可作为指示剂？滴定开始时溶液褪色缓慢，但随后逐渐加快，试解释这一现象。

## 解析

### 分析

1. 氧化还原滴定的计量关系由电子转移数确定：$\mathrm{MnO_4^-}$ 中 Mn 由 +7 降至 +2（5e⁻），$\mathrm{H_2O_2}$ 中每个 O 由 −1 升至 0（共 2e⁻）
2. 稀释倍数的处理：原样 2.00 mL → 100 mL（稀释 50 倍），每次取 25.00 mL 滴定
3. KMnO₄ 自身有深紫色，在酸性溶液中还原产物 Mn²⁺ 为近无色，终点时过量一滴 KMnO₄ 即显粉红色——**自身指示剂**
4. 反应速率逐渐加快是典型的**自催化**现象：产物 $\mathrm{Mn^{2+}}$ 催化了 $\mathrm{MnO_4^-}$ 与 $\mathrm{H_2O_2}$ 的反应

### 解答

**(1) 半反应与电子转移**

氧化剂（$\mathrm{MnO_4^-}$）半反应：

$$\mathrm{MnO_4^- + 8H^+ + 5e^- \longrightarrow Mn^{2+} + 4H_2O} \quad (\text{得 } 5e^-)$$

还原剂（$\mathrm{H_2O_2}$）半反应：

$$\mathrm{H_2O_2 \longrightarrow O_2 + 2H^+ + 2e^-} \quad (\text{失 } 2e^-)$$

为配平电子数，还原剂 ×5，氧化剂 ×2，得题目中总反应式。

计量关系：每 2 mol $\mathrm{MnO_4^-}$ 与 5 mol $\mathrm{H_2O_2}$ 反应，即：

$$n(\mathrm{H_2O_2}) = \frac{5}{2} \times n(\mathrm{KMnO_4})$$

**(2) H₂O₂ 含量计算**

滴定消耗 $\mathrm{KMnO_4}$ 的物质的量：

$$n(\mathrm{KMnO_4}) = 0.02000 \times 18.50 \times 10^{-3} = 3.700 \times 10^{-4}\ \mathrm{mol}$$

滴定液（25.00 mL 稀释液）中 $\mathrm{H_2O_2}$ 的物质的量：

$$n(\mathrm{H_2O_2}) = \frac{5}{2} \times 3.700 \times 10^{-4} = 9.250 \times 10^{-4}\ \mathrm{mol}$$

100 mL 稀释液（= 原 2.00 mL 样品）中 $\mathrm{H_2O_2}$ 的物质的量：

稀释因子 = 100 mL / 25.00 mL = 4

$$n_{\text{总}}(\mathrm{H_2O_2}) = 9.250 \times 10^{-4} \times 4 = 3.700 \times 10^{-3}\ \mathrm{mol}$$

质量：

$$m(\mathrm{H_2O_2}) = 3.700 \times 10^{-3} \times 34.02 = 0.1259\ \mathrm{g}$$

原样品中质量体积百分浓度：

$$\text{w/v}\% = \frac{0.1259\ \mathrm{g}}{2.00\ \mathrm{mL}} \times 100\% = 6.29\ \mathrm{g/100mL}$$

即约 **6.3%**（市售双氧水常见浓度）。

**(3) 自身指示剂与自催化**

**自身指示剂**：$\mathrm{KMnO_4}$ 水溶液呈深紫色（$\mathrm{MnO_4^-}$ 的特征颜色），而还原产物 $\mathrm{Mn^{2+}}$ 在稀溶液中近无色。化学计量点前，滴入的 $\mathrm{KMnO_4}$ 立即被 $\mathrm{H_2O_2}$ 还原褪色；化学计量点后，半滴过量 $\mathrm{KMnO_4}$ 不被消耗，溶液即呈现稳定的微红色，指示终点到达。无需额外加入指示剂。

**自催化现象**：反应 $\mathrm{MnO_4^- + H_2O_2}$ 的产物中包含 $\mathrm{Mn^{2+}}$，而 $\mathrm{Mn^{2+}}$ 对该反应有催化作用。滴定开始时溶液中尚无 $\mathrm{Mn^{2+}}$，反应速率较慢，滴入的 $\mathrm{KMnO_4}$ 褪色也慢；随着反应进行，$\mathrm{Mn^{2+}}$ 逐渐积累，催化效应增强，反应加速；因此在滴定操作中，**开始时应缓慢滴加并充分摇匀**，待生成一定量 $\mathrm{Mn^{2+}}$ 后反应明显加快，方可适当加快滴定速度。

## 答案

(1) 半反应见解答，计量关系：$n(\mathrm{H_2O_2}) : n(\mathrm{KMnO_4}) = 5:2$

(2) $\mathrm{H_2O_2}$ 含量 = $6.29\ \mathrm{g/100mL}$（约 6.3% w/v）

(3) KMnO₄ 自身显色作指示剂；反应初始慢、随后加快是由于产物 Mn²⁺ 的自催化作用
