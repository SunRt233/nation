# Fe³⁺氧化硫化物矿——辉银矿与磁黄铁矿

## 题目

研究发现，在酸性溶液中，$\mathrm{Fe}^{3+}$ 离子在溶解多种硫化物矿的过程中起着重要作用。以下给出两个例子：

**1-3-1** $\mathrm{Fe}^{3+}$ 将辉银矿（$\mathrm{Ag_2S}$）中的硫氧化成硫磺并使银离子转入溶液。

**1-3-2** $\mathrm{Fe}^{3+}$ 将磁黄铁矿（用 $\mathrm{Fe_7S_8}$ 表示）中的硫直接氧化成硫酸根离子。

分别写出两个反应的化学方程式。

## 参考答案

### 1-3-1 辉银矿的氧化

酸性溶液中 $\mathrm{Fe}^{3+}$ 有一定的氧化性，将 -2 价的 S 氧化为硫单质，其余元素不发生变价：

$$
\mathrm{Ag_2S} + 2\mathrm{Fe^{3+}} \longrightarrow 2\mathrm{Ag^+} + 2\mathrm{Fe^{2+}} + \mathrm{S}
$$

考虑到常温常压下硫单质常见的形态为 $\mathrm{S_8}$ 分子，亦可写成：
$$8\mathrm{Ag_2S} + 16\mathrm{Fe^{3+}} \longrightarrow 16\mathrm{Ag^+} + 16\mathrm{Fe^{2+}} + \mathrm{S_8}$$

### 1-3-2 磁黄铁矿的氧化

磁黄铁矿 $\mathrm{Fe_7S_8}$ 组成复杂，可采用**三种方法**配平：

**方法一：表观氧化态法**。将 $\mathrm{Fe_7S_8}$ 中 S 视为 -2 价，Fe 为 +16/7 价。氧化剂不仅有 $\mathrm{Fe^{3+}}$，还有 $\mathrm{Fe_7S_8}$ 中氧化态高于 +2 的 Fe。

**方法二：混合氧化态法**。将 $\mathrm{Fe_7S_8}$ 写作 $5\mathrm{FeS} \cdot \mathrm{Fe_2S_3}$（类比 $\mathrm{Fe_3O_4}$），再进行配平。

**方法三：巧设氧化数法（推荐）**。将 $\mathrm{Fe_7S_8}$ 中的 Fe 与 S 均视为 0 价。产物中 Fe 为 +2 价，S 为 +6 价，每分子 $\mathrm{Fe_7S_8}$ 全部转化的失电子数为 $6 \times 8 + 2 \times 7 = 62$，故 $\mathrm{Fe^{3+}}$ 的系数为 62。

反应还需要 $\mathrm{H_2O}$ 参与（将 S²⁻ 氧化为 $\mathrm{SO_4^{2-}}$ 需要 O 原子），在酸性溶液中用 $\mathrm{H_2O}$ 和 $\mathrm{H^+}$ 配平：

$$
\mathrm{Fe_7S_8} + 32\mathrm{H_2O} + 62\mathrm{Fe^{3+}} \longrightarrow 69\mathrm{Fe^{2+}} + 64\mathrm{H^+} + 8\mathrm{SO_4^{2-}}
$$

**验证**：电子守恒——失电子 $62$ = 得电子 $62 \times 1$ ✓

**得分要点**：
- 1-3-1 正确推断产物并配平（1.5分）
- 1-3-2 正确推断产物并配平（2.5分）

**解题关键**：
- 1-3-1：S²⁻ → S⁰，Fe³⁺ → Fe²⁺，Ag⁺ 进入溶液
- 1-3-2：巧设氧化数法——将 Fe、S 均视为 0 价，总失电子 = 6×8 + 2×7 = 62

**易错点**：
- 1-3-1 产物写成 SO₄²⁻（注意题干要求"氧化成硫磺"）
- 1-3-2 Fe 氧化态判断错误（Fe₇S₈ 中 Fe 也是还原剂，被氧化到 Fe²⁺）
- 漏写 H₂O 或 H⁺（反应在酸性溶液中进行，需用水和氢离子配平 O 和 H）
