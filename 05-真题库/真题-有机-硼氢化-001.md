# 硼氢化-氧化-1-甲基环己烯的反马氏水合

## 题目

1-甲基环己烯与 $\mathrm{BH_3\cdot THF}$ 反应，再在碱性条件下用 $\mathrm{H_2O_2}$ 氧化，得到主要产物 A。

(1) 写出产物 A 的结构式（标明立体化学），并说明其区域选择性和立体选择性。

(2) 解释为什么硼氢化-氧化反应给出**反马氏规则**的加成产物，而不是马氏规则产物。

(3) 若在相同条件下用 1,2-二甲基环己烯反应，会得到什么产物？写出结构式。

(4) 比较硼氢化-氧化与酸催化水合反应在**立体选择性**上的区别。

## 解析

### 分析

1. 硼氢化-氧化是醇的合成方法之一，与酸催化水合相反，给出**反马氏**加成产物（OH 加在取代较少的碳上）。
2. 硼氢化反应的本质：B-H 键对 C=C 的**顺式加成**，B 加在取代较少的碳上（位阻效应 + 电子效应）。随后的氧化保留构型。
3. 因此，整体为**顺式、反马氏**水合。对于 1-甲基环己烯，环上两个面不等价，需考虑硼烷从位阻较小的面进攻。
4. 酸催化水合为**非立体选择性**（经过平面碳正离子），而硼氢化-氧化为**顺式立体专一**加成。

### 解答

**(1) 产物 A 的结构**

1-甲基环己烯 + BH₃ → 烷基硼（硼加在位阻较小的 C2 上，即取代较少的一端）→ H₂O₂/OH⁻ 氧化 → OH 取代 B，保留构型。

产物 A：**反-2-甲基环己醇**（OH 和 CH₃ 在环的两侧，呈反式关系）。

机理：BH₃ 从环上位阻较小的方向（与甲基相反的一面）顺式加成 → B 和 H 加在同一面 → 氧化后 OH 与 B 在同一位置 → OH 和 CH₃ 在环的异侧（反式）。

**(2) 反马氏规则的原因**

硼氢化反应不经过碳正离子。决定区域选择性的是：
- **位阻效应**：硼原子体积大，优先加在取代较少、空间位阻较小的碳上
- **电子效应**：在四中心过渡态中，B 带有 δ⁺ 电荷（因为 B-H 键中 H 的电子云更多偏向 B）。过渡态中部分正电荷更倾向于在取代较多的碳上积累 → B 最终加在取代较少的碳上

因此得到反马氏产物：H 加在取代较多的碳，B（进而被 OH 取代）加在取代较少的碳。

**(3) 1,2-二甲基环己烯的产物**

底物：1,2-二甲基环己烯（两个双键碳都连有取代基）。

硼加在位阻较小的 C2 上（C1 和 C2 都连有 CH₃，但 C2 的 CH₃ 为平伏键/a 键取向导致位阻较小的一方）。氧化后得到：

**反-1,2-二甲基环己醇**（两个 CH₃ 和 OH 的相互空间关系为反式）。

**(4) 硼氢化-氧化 vs 酸催化水合**

| 特征 | 硼氢化-氧化 | 酸催化水合 |
|:---|:---|:---|
| 区域选择性 | 反马氏 | 马氏 |
| 立体选择性 | 顺式加成（syn） | 非立体专一（碳正离子平面） |
| 重排 | 不重排 | 可能重排 |
| 中间体 | 四中心过渡态 | 碳正离子 |

### 反思

硼氢化-氧化是**同时具有区域选择性和立体选择性**的反马氏水合法——这使其成为醇合成的有力工具。常见的陷阱是混淆硼氢化-氧化的选择性方向：**顺式**（syn）+ **反马氏** = B/H 从同一面加，B 在取代少的碳上。与之对比的另一种反马氏加成是 HBr + 过氧化物（自由基机理），但无立体选择性。
