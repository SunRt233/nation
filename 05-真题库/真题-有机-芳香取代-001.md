# 芳香亲电取代-甲苯硝化定位效应与主产物判断

## 题目

甲苯在混酸（浓 $\mathrm{HNO_3}$ + 浓 $\mathrm{H_2SO_4}$）条件下发生硝化反应。

(1) 写出反应的**三种**主要单硝化产物的结构式，并标明它们的相对比例（主产物 > 次产物 > 微量）。

(2) 从共振论和诱导效应的角度，解释甲苯中甲基的定位效应。为什么邻/对位产物占主导而间位产物极少？

(3) 若要进一步提高对位产物的选择性，可以采取什么措施？说明原理。

## 解析

### 分析

1. 甲苯的 -CH₃ 是**邻/对位定位基**——通过超共轭效应和给电子诱导效应活化苯环，使邻位和对位的电子云密度增加。
2. 硝化反应中，三硝基的位置比例由定位效应和位阻效应共同决定：邻位虽活化但受位阻影响抑制，对位活化且无位阻 → 通常对位产物最多。
3. 提高对位选择性的常见策略：使用大位阻的亲电试剂（如采用乙酰硝酸酯）、低温、或使用对位选择性催化剂。

### 解答

**(1) 三种单硝化产物**

甲苯的单硝化得到邻硝基甲苯、对硝基甲苯和间硝基甲苯的混合物。

| 产物 | 结构 | 大致比例 |
|:---|:---|:---|
| 对硝基甲苯 | $p\text{-}\mathrm{CH_3C_6H_4NO_2}$ | ~58% |
| 邻硝基甲苯 | $o\text{-}\mathrm{CH_3C_6H_4NO_2}$ | ~38% |
| 间硝基甲苯 | $m\text{-}\mathrm{CH_3C_6H_4NO_2}$ | ~4% |

**(2) 甲基定位效应的解释**

-CH₃ 的定位效应来自两个因素：

**超共轭效应（主导）**：甲基的 C-H σ 键与苯环 π 体系发生 σ-π 超共轭 → 电子密度向苯环邻位和对位传递。这可以从共振结构式中体现——邻位和对位可以写出额外的共振贡献结构（带负电荷在邻/对位），而间位不能。

**诱导效应**：C（电负性 2.5）> H（电负性 2.1），C-CH₃ 键的偶极方向为 C δ⁻ — CH₃ δ⁺，即甲基向苯环给电子（+I 效应）。但这一效应较弱，超共轭效应是主导。

邻/对位电子云密度增加 → 亲电试剂（NO₂⁺）更易进攻邻/对位 → 邻/对位产物为主；间位电子云密度增加很少 → 间位产物极少。

**(3) 提高对位选择性的措施**

- **利用位阻效应**：使用体积更大的亲电试剂（如采用 $\mathrm{CH_3COONO_2}$ 乙酰硝酸酯 + 催化剂），大体积亲电试剂难以进攻受甲基位阻影响的邻位，从而提高对位比例。
- **低温**：低温下反应趋于动力学控制，反应速率差异被放大，有利于电子效应更有利的对位加成。
- **对位选择性催化剂**：使用环糊精、分子筛等具有特定孔道结构的催化剂，利用形状选择效应提高对位产物比例。

### 反思

芳香亲电取代的定位效应是初赛必考内容。记忆口诀："邻对位定位基"包括 -R（烷基）、-OH、-OR、-NH₂、-NHR、-NR₂、-X（卤素）等；"间位定位基"包括 -NO₂、-CN、-COOH、-SO₃H、-CHO、-COR 等。注意卤素虽为邻对位定位基，但其强吸电子诱导效应使其使苯环**钝化**——这是常考的例外。
