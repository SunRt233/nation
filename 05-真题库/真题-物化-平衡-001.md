# 化学平衡-已知Kp计算转化率

## 题目

五氯化磷的分解反应是研究化学平衡的经典体系：

$$\mathrm{PCl_5(g) \rightleftharpoons PCl_3(g) + Cl_2(g)}$$

在 523 K 时，该反应的 $K_p = 1.78\ \mathrm{atm}$。

(1) 将纯 $\mathrm{PCl_5}$ 注入一真空密闭容器中，523 K 下平衡时总压为 2.00 atm。求 $\mathrm{PCl_5}$ 的分解率 $\alpha$。

(2) 若上述容器中在注入 $\mathrm{PCl_5}$ 的同时，已含有 $\mathrm{Cl_2}$ 使其初始分压为 0.50 atm，平衡时 $\mathrm{PCl_5}$ 的分解率 $\alpha'$ 为多少？

(3) 比较 (1) 和 (2) 的结果，用平衡移动原理加以解释。

## 解析

### 分析

1. 用分压表示 $K_p$：设初始 $\mathrm{PCl_5}$ 压力为 $p_0$，分解率 $\alpha$，则平衡时各组分分压用 $p_0$ 和 $\alpha$ 表达
2. 由总压条件可求 $p_0$，再代入 $K_p$ 表达式求解 $\alpha$
3. 加入产物 $\mathrm{Cl_2}$ 后，根据 **Le Chatelier 原理**，平衡向逆反应方向移动，$\alpha$ 减小
4. 注意单位一致性：$K_p$ 以 atm 为单位

### 解答

**(1) 纯 PCl₅ 分解**

设初始 $\mathrm{PCl_5}$ 分压为 $p_0$，分解率为 $\alpha$：

|  | $\mathrm{PCl_5}$ | $\mathrm{PCl_3}$ | $\mathrm{Cl_2}$ |
|:---:|:---:|:---:|:---:|
| 初始/atm | $p_0$ | 0 | 0 |
| 变化/atm | $-p_0\alpha$ | $+p_0\alpha$ | $+p_0\alpha$ |
| 平衡/atm | $p_0(1-\alpha)$ | $p_0\alpha$ | $p_0\alpha$ |

平衡总压：$p_{\text{总}} = p_0(1-\alpha) + p_0\alpha + p_0\alpha = p_0(1+\alpha)$

已知 $p_{\text{总}} = 2.00\ \mathrm{atm}$：

$$p_0(1+\alpha) = 2.00 \quad\Rightarrow\quad p_0 = \frac{2.00}{1+\alpha}$$

平衡常数：

$$K_p = \frac{p_{\mathrm{PCl_3}} \cdot p_{\mathrm{Cl_2}}}{p_{\mathrm{PCl_5}}} = \frac{(p_0\alpha)^2}{p_0(1-\alpha)} = \frac{p_0\alpha^2}{1-\alpha} = 1.78$$

代入 $p_0$ 表达式：

$$\frac{2.00}{1+\alpha} \cdot \frac{\alpha^2}{1-\alpha} = 1.78$$

$$\frac{2\alpha^2}{(1+\alpha)(1-\alpha)} = \frac{2\alpha^2}{1-\alpha^2} = 1.78$$

$$2\alpha^2 = 1.78(1-\alpha^2)$$

$$2\alpha^2 = 1.78 - 1.78\alpha^2$$

$$3.78\alpha^2 = 1.78$$

$$\alpha = \sqrt{\frac{1.78}{3.78}} = 0.686$$

$$\alpha = 68.6\%$$

**(2) 含初始 Cl₂**

设初始 $\mathrm{PCl_5}$ 分压为 $p_0$，初始 $\mathrm{Cl_2}$ 分压为 0.50 atm：

|  | $\mathrm{PCl_5}$ | $\mathrm{PCl_3}$ | $\mathrm{Cl_2}$ |
|:---:|:---:|:---:|:---:|
| 初始/atm | $p_0$ | 0 | 0.50 |
| 平衡/atm | $p_0(1-\alpha')$ | $p_0\alpha'$ | $0.50 + p_0\alpha'$ |

总压：$p_{\text{总}} = p_0(1-\alpha') + p_0\alpha' + 0.50 + p_0\alpha' = p_0(1+\alpha') + 0.50 = 2.00$

$$p_0 = \frac{1.50}{1+\alpha'}$$

代入 $K_p$：

$$K_p = \frac{p_0\alpha' \cdot (0.50 + p_0\alpha')}{p_0(1-\alpha')} = \frac{\alpha'(0.50 + p_0\alpha')}{1-\alpha'} = 1.78$$

由 $p_0(1+\alpha') = 1.50$ 得 $p_0$ 代入后用数值求解。为简化，利用 $p_0\alpha'$ 整体处理。

设 $x = p_0\alpha'$，则 $p_0 = \frac{1.50}{1+\alpha'}$，$x = \frac{1.50\alpha'}{1+\alpha'}$

$$\frac{\alpha'(0.50 + x)}{1-\alpha'} = 1.78$$

用试差法（逐次逼近）或用总压关系 $p_0 + x + (0.50+x) = 2.00 \Rightarrow p_0 + 2x = 1.50$：

代入 $x = p_0\alpha'$，$p_0(1+2\alpha') = 1.50$

由 $K_p = \frac{p_0\alpha'(0.50+p_0\alpha')}{p_0(1-\alpha')} = 1.78$

令 $\alpha' \approx 0.50$ 试算：
- $1+2 \times 0.50 = 2.00$，$p_0 = 1.50/2.00 = 0.75$
- $x = 0.75 \times 0.50 = 0.375$
- $K_p = 0.50 \times (0.50+0.375)/(1-0.50) = 0.50 \times 0.875/0.50 = 0.875$（偏小）

令 $\alpha' \approx 0.58$：
- $1+2 \times 0.58 = 2.16$，$p_0 = 1.50/2.16 = 0.694$
- $x = 0.694 \times 0.58 = 0.403$
- $K_p = 0.58 \times (0.50+0.403)/(1-0.58) = 0.58 \times 0.903/0.42 = 1.247$（仍偏小）

令 $\alpha' \approx 0.63$：
- $1+2 \times 0.63 = 2.26$，$p_0 = 1.50/2.26 = 0.664$
- $x = 0.664 \times 0.63 = 0.418$
- $K_p = 0.63 \times (0.50+0.418)/(1-0.63) = 0.63 \times 0.918/0.37 = 1.563$（偏小）

令 $\alpha' \approx 0.66$：
- $1+2 \times 0.66 = 2.32$，$p_0 = 1.50/2.32 = 0.647$
- $x = 0.647 \times 0.66 = 0.427$
- $K_p = 0.66 \times (0.50+0.427)/(1-0.66) = 0.66 \times 0.927/0.34 = 1.80 \approx 1.78$

$$\alpha' \approx 66\%$$

验证：$\alpha' = 0.657$ 时更精确，$K_p \approx 1.78$。

$$\alpha' \approx 65.7\%$$

**(3) 结果比较**

| 条件 | 分解率 $\alpha$ |
|:---|:---:|
| 纯 PCl₅ | 68.6% |
| 含 0.50 atm Cl₂ | ~65.7% |

加入产物 $\mathrm{Cl_2}$ 使分解率**略有降低**。根据 **Le Chatelier 原理**：增加产物的浓度（分压），平衡向消耗该产物的方向——即逆反应方向移动，因此 $\mathrm{PCl_5}$ 的分解受到抑制。

注意此体系中 $K_p = 1.78$ 并不大，加 0.50 atm Cl₂ 的影响较显著（约 3 个百分点），定量反映了平衡移动的程度。

## 答案

(1) $\alpha = 68.6\%$

(2) $\alpha' \approx 65.7\%$

(3) 加入 Cl₂ 使分解率降低，符合 Le Chatelier 原理——增加产物抑制正反应
