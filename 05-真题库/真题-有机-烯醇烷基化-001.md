# 烯醇负离子烷基化-环己酮经LDA与CH3I的区域选择性

## 题目

环己酮在 THF 中，-78°C 下与二异丙基氨基锂（LDA）反应，随后加入过量的碘甲烷 $\mathrm{CH_3I}$，室温搅拌反应。

(1) 写出主产物的结构式和名称。

(2) 解释为什么使用 LDA 而不是 $\mathrm{NaOH}$ 或 $\mathrm{NaOEt}$ 作为碱。LDA 与后两者在去质子化行为上有何本质区别？

(3) 该反应在 -78°C 下进行有何重要意义？若在室温下进行会有什么问题？

(4) 若使用 2-甲基环己酮代替环己酮进行同样的反应，产物中会得到几个区域异构体？各自的相对比例如何（动力学产物 vs 热力学产物）？

## 解析

### 分析

1. 环己酮的 α-位去质子化方向：环己酮是非对称酮，但在 LDA 条件下主要生成**动力学烯醇锂**（取代较少的烯醇负离子）。
2. LDA 是大位阻强碱，在低温下进行**不可逆的动力学去质子化**，而非可逆的热力学平衡去质子化。
3. 室温下烯醇负离子可互变 → 热力学控制 → 产物选择性降低，且可能有自身 Aldol 缩合等副反应。
4. 2-甲基环己酮有两个不同的 α-位 → 生成两种烯醇负离子 → 两种烷基化产物。

### 解答

**(1) 主产物**

**2-甲基环己酮**（α-甲基环己酮）

$$\mathrm{环己酮 \xrightarrow[2) CH_3I]{1) LDA, THF, -78^\circ C} 2\text{-}甲基环己酮}$$

LDA 在低温下优先夺取位阻较小的 α-H（在 C2/C6 位），生成动力学烯醇锂，随后 $\mathrm{CH_3I}$ 进行 SN2 烷基化。

**(2) LDA vs NaOH/NaOEt**

| 特征 | LDA | NaOH / NaOEt |
|:---|:---|:---|
| 碱性强度 | pK_a ≈ 36（极强碱） | pK_a ≈ 15.7 / 16 |
| 去质子化方式 | **不可逆、定量** | 可逆（平衡） |
| 位阻 | 极大（i-Pr 基团） | 较小 |
| 亲核性 | 极弱（无亲核性） | 中等/较强 |
| 产物控制 | 动力学控制 | 热力学平衡 |

LDA 的三大特征：**强碱性**（完全定量去质子化）、**大位阻**（只能夺取位阻最小的 α-H）、**非亲核性**（不会与底物的羰基发生亲核加成，也不会直接与 CH₃I 反应）。

NaOH/NaOEt 的缺点：(1) 去质子化可逆 → 热力学控制 → 烯醇负离子互变导致选择性差；(2) 碱本身也是亲核试剂 → 可能与 CH₃I 或羰基发生副反应；(3) 碱性不够强 → 去质子化不完全。

**(3) -78°C 低温的重要性**

-78°C（干冰/丙酮浴）下的两个关键作用：

1. **动力学控制**：在低温下，LDA 的去质子化是**不可逆**的——一旦夺取 α-H 生成烯醇锂，烯醇锂不会再质子化回到酮。此时去质子化速率由位阻决定（夺取位阻最小的 α-H 最快），产物为**动力学烯醇负离子**。
2. **抑制副反应**：低温下烯醇负离子的自身缩合（Aldol）、多烷基化等副反应被极大抑制。

若在室温下进行：(1) 烯醇负离子可互变 → 热力学控制 → 两种烯醇负离子并存；(2) 烯醇负离子可能与未反应的酮发生 Aldol 缩合；(3) 多烷基化风险增大。

**(4) 2-甲基环己酮的反应产物**

2-甲基环己酮有三个不同的 α-位：C2（已有 CH₃）、C6（CH₂-）、和 C2 的甲基（CH₃）。

LDA 在 -78°C 下：
- C6 位（=CH₂-）：位阻最小 → **动力学烯醇锂**（主要）
- C2 位（=C(CH₃)-）：位阻较大 → 较少

产物：
- **6,2-二甲基环己酮**（C6 位烷基化产物，**主产物**，约 80-90%）
- **2,2-二甲基环己酮**（C2 位烷基化产物，**次产物**）

动力学 vs 热力学产物的区别：低温 + LDA → 动力学产物（C6 烷基化为主）；室温或使用 NaOEt → 烯醇负离子可互变 → 热力学控制下 C2 位烷基化产物更多（因为更稳定的四取代烯醇负离子占主导）。

### 反思

烯醇负离子烷基化中最容易混淆的概念是**动力学控制 vs 热力学控制**。LDA/-78°C → 动力学烯醇 → 位阻最小的 α-位去质子化。NaOEt/室温 → 热力学烯醇 → 取代更多的 α-位去质子化（形成更稳定的烯醇负离子）。这是两种烯醇负离子化学的基础分水岭。
