# 自由基反应的KIE与合成应用

## 题目

**10.6** 在使用非手性 NHC–BH₃ 催化剂作用下，底物中含 C–H 和 C–D 两种可选攫氢位点。判断以下哪一个是主产物。

（原题含反应式选项图片 A 和 B，参见 39 届初赛试题解析第 10.6 题图）

**10.7** 依据本题所提供的信息，完成以下反应式（提示：产物为六元环衍生物，并含有两个芳环取代基）：

$$\equiv\mathrm{COOR} + \mathrm{F}\!-\!\!\bigcirc \xrightarrow[\mathrm{NHC\!-\!BH_3\ (10\ equiv)}]{\mathrm{ACCN}}\ ?$$

## 参考答案

**10.6** C–H 键优先断裂 → 主产物为 **B**（攫 H 路径的产物）。

**原因**：C–H 键解离能 < C–D 键解离能（差值约 12 kJ mol⁻¹），源于**零点能差异（ZPE）**。
$$E_{\mathrm{ZPE}} = \frac{1}{2}h\nu,\quad \nu = \frac{1}{2\pi}\sqrt{\frac{k}{\mu}}$$

- C–H 约化质量 $\mu$ 更小 → $\nu$ 更大 → ZPE 更高
- ZPE 越高 → 基态能量越高 → "更接近被拉断" → 键解离能更低
- 典型：$\mathrm{H_3C\!-\!H}: 431\ \mathrm{kJ\ mol^{-1}}$，$\mathrm{D_3C\!-\!D}: 443\ \mathrm{kJ\ mol^{-1}}$

此即**动力学同位素效应（KIE）**。同位素标记实验还用于验证机理：将 N 邻位 $\alpha$-H 全部替换为 D 后，产物中 D 被完全保留，排除了 $\mathbf{B}^{\bullet*}$ 直接攫 $\alpha$-H 的路径。

**10.7** 反应过程：$\mathbf{B}^{\bullet*}$ 对炔加成 → 烯基自由基 → 氢迁移（分子间攫 H 或分子内适当位置攫 H） → 苄基自由基 → 与芳环相连的不饱和键发生自由基环化（6 元环） → $\beta$-碎裂消除 → 产物。

产物为含两个芳环取代基的六元环衍生物（$\mathrm{COOR}$ 基团和 $\mathrm{F\!-\!C_6H_4\!-}$ 基团位于环上）。

（产物结构图参见 39 届初赛试题解析图片）

**解题思路**：
1. 10.6：比较 C–H 和 C–D 的键解离能（ZPE 差异 → C–H 键更弱）
2. 10.7：按机理模式推断：$\mathbf{B}^{\bullet*}$ 加成 → 自由基中间体 → 环化 → $\beta$-消除。六元环暗示 6-endo-trig 或 6-exo-trig 环化模式，两个芳环取代基体现于底物的酯基芳环和氟代芳环

**易错点**：
- 混淆 KIE 的两种应用场景（速率差异 vs 标记追踪）
- 将同位素效应归因于键能本身而非 ZPE
- 10.7 中氢迁移路径判断错误（需注意底物中可攫氢的位置）
