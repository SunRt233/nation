# Diels-Alder反应-1,3-丁二烯与乙烯的立体选择性

## 题目

1,3-丁二烯与乙烯在加热条件下发生 Diels-Alder 反应。

(1) 写出反应产物的结构式（含立体化学信息）。

(2) 该反应在常温下速率极慢，但在 200°C 下能顺利进行。请从**分子轨道对称性**角度解释：为什么加热条件对该反应是"允许"的（Woodward-Hoffmann 规则）？

(3) 判断以下亲二烯体与 1,3-丁二烯反应的相对速率快慢，排序并说明理由：

$$\mathrm{CH_2=CH_2},\quad \mathrm{CH_2=CHCHO},\quad \mathrm{CH_2=CHCN},\quad \mathrm{CH_2=C(CN)_2}$$

(4) 1,3-丁二烯与顺丁烯二酸酐在室温下即可迅速反应，而与乙烯需要高温。解释原因。

## 解析

### 分析

1. Diels-Alder 是 **[4+2] 环加成**反应：二烯（4π 电子）+ 亲二烯体（2π 电子）。在加热条件下是"对称性允许"的（同面-同面加成）。
2. 反应具有**顺式加成**的立体化学特征——二烯体和亲二烯体的取代基立体化学保留在产物中。
3. 亲二烯体的反应活性受电子效应控制：连接吸电子基越多，LUMO 能量越低，与二烯的 HOMO 能隙越小 → 反应越快。
4. 顺丁烯二酸酐是极好的亲二烯体（两个羰基吸电子 + 环状结构使其处于 s-cis 构象约束）。

### 解答

**(1) 产物结构**

1,3-丁二烯 + 乙烯 → **环己烯**

$$\mathrm{CH_2=CH-CH=CH_2 + CH_2=CH_2 \longrightarrow}$$

产物为最简单的 Diels-Alder 加合物——环己烯。注意：两个亲二烯体上的 H 是顺式加成的（都在产物的同一面）。

**(2) 加热条件与 Woodward-Hoffmann 规则**

1,3-丁二烯（4π 电子体系）的 HOMO 与乙烯（2π 电子体系）的 LUMO 在**加热条件**下发生 [4+2] 同面-同面环加成。

根据 Woodward-Hoffmann 规则：
- 参与反应的 π 电子总数 = 4 + 2 = 6（$4n+2$，$n=1$）
- 同面-同面（suprafacial-suprafacial）加成
- $4n+2$ 体系在**加热**条件下是"对称性允许"的（热允许）

这是"芳香过渡态"概念的体现——过渡态具有 6 个 π 电子的环状共轭体系（类似于苯），因此加热条件下是允许的。

**(3) 亲二烯体活性排序**

排序（从快到慢）：
$$\mathrm{CH_2=C(CN)_2 > CH_2=CHCN > CH_2=CHCHO > CH_2=CH_2}$$

理由：Diels-Alder 反应为**正常电子需求**型（二烯 HOMO + 亲二烯体 LUMO 相互作用）。亲二烯体上连接的吸电子基（-CN, -CHO）降低 LUMO 能量，减小与二烯 HOMO 的能隙 → 反应加速。-CN 的吸电子能力 > -CHO。-C(CN)₂ 有两个氰基，效果最强。乙烯无吸电子基，LUMO 最高，反应最慢。

**(4) 顺丁烯二酸酐的高活性**

顺丁烯二酸酐比乙烯快得多的原因有三：
1. **电子效应**：两个羰基（强吸电子基）大幅降低了亲二烯体的 LUMO 能量
2. **构象约束**：酸酐的环状结构强制双键处于 s-cis 构象，与二烯的轨道重叠最佳
3. **应变释放**：反应后环张力部分释放，热力学驱动力大

### 反思

Diels-Alder 反应是初赛周环反应的核心考点。三个关键要素：**热允许的 [4+2] 环加成**（WH 规则）、**正常电子需求下的 FMO 控制**（吸电子基加速）、**顺式立体专一性**。切记，简单的二烯 + 简单亲二烯体需要高温，但引入恰当的电子效应和环张力可大幅降低反应温度。
