# Nernst方程计算浓差电池电动势

## 题目

浓差电池是一种仅因电极活性物质浓度不同而产生电动势的特殊原电池。考虑以下由铜电极和硫酸铜溶液构成的浓差电池（298 K）：

$$\mathrm{Cu(s) \,|\, CuSO_4}(0.0100\ \mathrm{mol \cdot L^{-1}}) \,\parallel\, \mathrm{CuSO_4}(1.00\ \mathrm{mol \cdot L^{-1}}) \,|\, \mathrm{Cu(s)}$$

已知 $E^\ominus(\mathrm{Cu^{2+}/Cu}) = +0.337\ \mathrm{V}$。

(1) 写出该电池的电极反应和电池总反应，指出正负极（假定活度系数均为 1）。

(2) 计算 298 K 时该浓差电池的电动势。

(3) 若将右侧溶液中通入 $\mathrm{H_2S}$ 至饱和（使 $\mathrm{Cu^{2+}}$ 以 CuS 沉淀析出，残留 $[\mathrm{Cu^{2+}}] = 1.0 \times 10^{-18}\ \mathrm{mol \cdot L^{-1}}$），而左侧浓度不变，此时的电动势为多少？电池的正负极是否发生变化？

## 解析

### 分析

1. 浓差电池的标准电动势 $E^\ominus_{\text{cell}} = 0$（两电极相同），电动势完全来自**浓度差引起的 Nernst 电势差**
2. 浓度高的一侧 Cu²⁺ 被还原的趋势更强，作正极；浓度低的一侧 Cu 被氧化的趋势更强，作负极
3. 电池反应实质是 Cu²⁺ 从浓溶液"转移"到稀溶液的过程
4. 通入 H₂S 后，右侧 Cu²⁺ 浓度极低，正负极发生互换

### 解答

**(1) 电极反应与电池总反应**

负极（氧化，左侧稀溶液）：$\mathrm{Cu(s) \longrightarrow Cu^{2+}}(0.0100\ \mathrm{M}) + 2e^-$

正极（还原，右侧浓溶液）：$\mathrm{Cu^{2+}}(1.00\ \mathrm{M}) + 2e^- \longrightarrow \mathrm{Cu(s)}$

电池总反应：$\mathrm{Cu^{2+}}(1.00\ \mathrm{M}) \longrightarrow \mathrm{Cu^{2+}}(0.0100\ \mathrm{M})$

**(2) 浓差电池电动势**

用 Nernst 方程分别计算两个半电池的电极电势：

左侧（负极，$[\mathrm{Cu^{2+}}] = 0.0100\ \mathrm{M}$）：

$$E_- = E^\ominus + \frac{0.0592}{2}\lg(0.0100) = 0.337 + 0.0296 \times (-2.00) = 0.337 - 0.0592 = 0.278\ \mathrm{V}$$

右侧（正极，$[\mathrm{Cu^{2+}}] = 1.00\ \mathrm{M}$）：

$$E_+ = E^\ominus + \frac{0.0592}{2}\lg(1.00) = 0.337 + 0 = 0.337\ \mathrm{V}$$

电动势：

$$E_{\text{cell}} = E_+ - E_- = 0.337 - 0.278 = 0.059\ \mathrm{V}$$

也可直接用浓差电池公式（$n=2$）：

$$E_{\text{cell}} = \frac{0.0592}{2}\lg\frac{[\mathrm{Cu^{2+}}]_{\text{右}}}{[\mathrm{Cu^{2+}}]_{\text{左}}} = \frac{0.0592}{2}\lg\frac{1.00}{0.0100} = 0.0296 \times 2.00 = 0.0592\ \mathrm{V}$$

**(3) 加入 H₂S 后**

右侧 $[\mathrm{Cu^{2+}}]$ 降至 $1.0 \times 10^{-18}\ \mathrm{M}$：

$$E_{\text{右}} = 0.337 + \frac{0.0592}{2}\lg(1.0 \times 10^{-18}) = 0.337 + 0.0296 \times (-18.0) = 0.337 - 0.533 = -0.196\ \mathrm{V}$$

$$E_{\text{左}} = 0.278\ \mathrm{V}\ (\text{不变})$$

此时 $E_{\text{右}} < E_{\text{左}}$，**正负极互换**：

- 新正极：左侧（$E = 0.278\ \mathrm{V}$）
- 新负极：右侧（$E = -0.196\ \mathrm{V}$）

电动势：

$$E_{\text{cell}} = E_{\text{左}} - E_{\text{右}} = 0.278 - (-0.196) = 0.474\ \mathrm{V}$$

用浓差电池公式验证（高浓度侧为正）：

$$E_{\text{cell}} = \frac{0.0592}{2}\lg\frac{0.0100}{1.0 \times 10^{-18}} = 0.0296 \times \lg(1.0 \times 10^{16}) = 0.0296 \times 16.0 = 0.474\ \mathrm{V}$$

## 答案

(1) 负极（左）：$\mathrm{Cu \to Cu^{2+} + 2e^-}$；正极（右）：$\mathrm{Cu^{2+} + 2e^- \to Cu}$；总反应：$\mathrm{Cu^{2+}}(1.00\ \mathrm{M}) \to \mathrm{Cu^{2+}}(0.0100\ \mathrm{M})$

(2) $E_{\text{cell}} = 0.0592\ \mathrm{V}$

(3) $E_{\text{cell}} = 0.474\ \mathrm{V}$，正负极互换（左侧为正、右侧为负）
