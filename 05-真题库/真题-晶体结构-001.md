# 晶体结构-Bragg方程与晶胞参数计算

## 题目

NaCl 晶体属于立方晶系，面心立方结构。使用 $\mathrm{Cu\ K\alpha}$ X 射线（$\lambda = 154.18\ \mathrm{pm}$）进行粉末衍射，测得 (200) 晶面的衍射角 $2\theta = 31.70^\circ$。

(1) 计算 NaCl 晶胞参数 $a$（单位 pm）。

(2) NaCl 的实测密度为 $2.165\ \mathrm{g\cdot cm^{-3}}$，计算每个晶胞中含有的 NaCl 化学式数目 $Z$，并验证上述晶胞参数的合理性。

已知：$M(\mathrm{NaCl}) = 58.44\ \mathrm{g\cdot mol^{-1}}$，$N_A = 6.022 \times 10^{23}\ \mathrm{mol^{-1}}$。

## 解析

### 分析

1. **Bragg 方程**：$n\lambda = 2d\sin\theta$，其中 $\theta = 2\theta/2$ 是 Bragg 角
2. 立方晶系：(200) 晶面间距 $d_{200} = \frac{a}{\sqrt{h^2+k^2+l^2}} = \frac{a}{\sqrt{4+0+0}} = \frac{a}{2}$
3. 由 $d_{200}$ → $a$ → 晶胞体积 $V = a^3$ → 理论密度 → 验证 $Z$

### 解答

**(1) 计算晶胞参数 $a$**

Bragg 角：$\theta = 31.70^\circ/2 = 15.85^\circ$

一级衍射（$n=1$）：
$$d_{200} = \frac{\lambda}{2\sin\theta} = \frac{154.18}{2 \times \sin 15.85^\circ} = \frac{154.18}{2 \times 0.2731} = \frac{154.18}{0.5462} = 282.3\ \mathrm{pm}$$

$$a = 2 \times d_{200} = 2 \times 282.3 = 564.6\ \mathrm{pm}$$

**(2) 验证 $Z$**

晶胞体积：$V = a^3 = (564.6 \times 10^{-10}\ \mathrm{cm})^3 = (5.646 \times 10^{-8})^3 = 1.800 \times 10^{-22}\ \mathrm{cm^3}$

密度公式：$\rho = \frac{Z \cdot M}{N_A \cdot V}$

$$Z = \frac{\rho \cdot N_A \cdot V}{M} = \frac{2.165 \times 6.022 \times 10^{23} \times 1.800 \times 10^{-22}}{58.44}$$

$$Z = \frac{2.165 \times 6.022 \times 1.800 \times 10^{1}}{58.44} = \frac{2.165 \times 108.4}{58.44} = \frac{234.7}{58.44} = 4.02 \approx 4$$

NaCl 是面心立方结构，每个晶胞预期含有 $Z = 4$ 个 NaCl 化学式（Na⁺: 8×1/8 + 6×1/2 = 4; Cl⁻: 12×1/4 + 1 = 4）。

**验证通过** ✅ $Z$ 的计算值与理论值一致，晶胞参数合理。

## 答案

(1) $a = 565\ \mathrm{pm}$（564.6 pm）

(2) $Z \approx 4$，与 NaCl 面心立方结构的理论值一致。
