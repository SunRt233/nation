# 亲电加成-碳正离子重排与3,3-二甲基-1-丁烯+HBr反应

## 题目

3,3-二甲基-1-丁烯 $(\mathrm{CH_3})_3\mathrm{CCH=CH_2}$ 与 $\mathrm{HBr}$ 反应，得到两种主要产物 A 和 B，其中 A 为正常的马氏加成产物，B 为经过碳正离子重排的产物。

(1) 写出 A 和 B 的结构式。

(2) 分别画出生成 A 和 B 的完整机理，标出每一步中碳正离子的结构变化。

(3) B 的产率随温度的升高而增加。请从热力学角度解释原因。

(4) 若改用 3,3-二甲基-1-丁烯与 $\mathrm{HI}$ 反应，产物组成与 $\mathrm{HBr}$ 相比有何变化？为什么？

## 解析

### 分析

1. 正常加成：H⁺ 加在=CH₂上 → 生成 2° 碳正离子 → Br⁻ 捕获 → 产物 A。
2. 但此 2° 碳正离子邻位有一个叔碳（C(CH₃)₃），可发生**甲基的 1,2-迁移（Wagner-Meerwein 重排）**，将 2° 碳正离子重排为更稳定的 3° 碳正离子 → Br⁻ 捕获 → 产物 B。
3. 温度升高更有利于热力学更稳定的产物（经由更稳定的中间体）。
4. HI 的反应性不同于 HBr。

### 解答

**(1) 产物结构**

**产物 A**（正常马氏加成）：$\mathrm{(CH_3)_3CCHBrCH_3}$（2-溴-3,3-二甲基丁烷）

**产物 B**（重排产物）：$\mathrm{(CH_3)_2CBrCH(CH_3)_2}$（2-溴-2,3-二甲基丁烷）

**(2) 机理**

**生成 A 的路径**：

$$\mathrm{(CH_3)_3CCH=CH_2 + H^+ \longrightarrow (CH_3)_3C\mathop{CH}\limits^{+}CH_3}$$

（2° 碳正离子，邻位有叔丁基）

$$\mathrm{(CH_3)_3C\mathop{CH}\limits^{+}CH_3 + Br^- \longrightarrow (CH_3)_3CCHBrCH_3}$$

**生成 B 的路径（重排路径）**：

初始生成的 2° 碳正离子 $\mathrm{(CH_3)_3C\mathop{CH}\limits^{+}CH_3}$ 发生**甲基 1,2-迁移**（其中一个 CH₃ 从 C3 迁移到 C2⁺）：

$$\mathrm{(CH_3)_3C\mathop{CH}\limits^{+}CH_3 \longrightarrow (CH_3)_2\mathop{C}\limits^{+}CH(CH_3)_2}$$

重排后得到 3° 碳正离子（与苯环共轭的类似体系更稳定）。然后：

$$\mathrm{(CH_3)_2\mathop{C}\limits^{+}CH(CH_3)_2 + Br^- \longrightarrow (CH_3)_2CBrCH(CH_3)_2}$$

**(3) 温度效应**

B 经由更稳定的 3° 碳正离子（相较于 A 的 2° 碳正离子），是热力学更稳定的产物。根据过渡态理论，较高温度使体系有足够能量跨越重排势垒 → B 的相对产率增加。低温时反应以动力学控制为主（产物 A 为主），高温时热力学控制（产物 B 增加）。

**(4) HI 与 HBr 的比较**

HI 中的 $\mathrm{I^-}$ 是比 $\mathrm{Br^-}$ 更好的离去基团，也是更软、更大的亲核试剂。I⁻ 对碳正离子的捕获更快，碳正离子寿命更短 → 重排的时间窗口更窄 → 重排产物（B）的比例较 HBr 条件下更低。$\mathrm{HBr}$ 中 $\mathrm{Br^-}$ 亲核性相对较弱，碳正离子寿命更长，有更多时间发生重排。

### 反思

碳正离子重排是亲电加成中最容易出错的考点。关键判断点：生成的碳正离子**邻位是否有能迁移的基团（H⁻、CH₃⁻、Ph⁻ 等）**且重排后是否得到更稳定碳正离子。迁移驱动力永远是：$1^\circ \to 2^\circ \to 3^\circ$（或从简单碳正离子到苄基型/烯丙基型）。
