# SN2立体化学-Walden翻转与手性中心构型反转

## 题目

$(R)$-2-溴辛烷在丙酮中与过量 $\mathrm{NaI}$ 加热回流，发生 Finkelstein 反应，得到主要产物 A。

(1) 写出产物 A 的结构式，并用楔形/虚线表示其绝对构型（给出 R/S 标示）。

(2) 从 SN2 反应机理的角度解释构型变化的原因。画出过渡态的立体化学示意图。

(3) 若改用 $(R)$-2-溴辛烷的纯对映体在 $\mathrm{NaOH}$ 水溶液中加热，除取代产物外还得到一种烯烃混合物。写出所有可能产物的结构式。

## 解析

### 分析

1. Finkelstein 反应是 $\mathrm{I^-}$ 作为亲核试剂进攻 $2^\circ$ 溴代烷的经典 SN2 反应，$\mathrm{I^-}$ 在极性非质子溶剂（丙酮）中亲核性极强。
2. SN2 反应必定伴随 **Walden 翻转**——亲核试剂从离去基团的反面进攻，产物构型与底物完全相反。
3. $\mathrm{NaOH}$ 水溶液加热条件下，$\mathrm{OH^-}$ 既是亲核试剂又是强碱 → SN2 与 E2 竞争，产生取代产物与多种消除产物。

### 解答

**(1) 产物 A 的结构**

SN2 反应发生 Walden 翻转，$(R)$-2-溴辛烷 → $(S)$-2-碘辛烷：

$$\mathrm{CH_3(CH_2)_5\mathop{CH}\limits_{Br}^{H}CH_3 \xrightarrow{NaI/acetone} CH_3(CH_2)_5\mathop{CH}\limits_{H}^{I}CH_3}$$

即：$(R)\text{-}C_8H_{17}Br \to (S)\text{-}C_8H_{17}I$

**(2) 机理与过渡态**

SN2 是**协同的一步反应**。$\mathrm{I^-}$ 从 C-Br 键的反面（180°方向）接近底物，经过一个平面三角形的过渡态（碳上连接五个基团中三个在平面、$\mathrm{I}$ 和 $\mathrm{Br}$ 在轴线两侧）：

$$[\mathrm{I^{\delta -}\cdots C\cdots Br^{\delta -}}]^{\ddagger}$$

离去基团 $\mathrm{Br^-}$ 脱离后，碳中心发生彻底的构型反转——像一把雨伞被风吹翻（"伞效应"）。

过渡态中碳为 $sp^2$ 杂化，三个取代基在同一平面。

**(3) NaOH 水溶液加热条件下的产物**

条件：$(R)$-2-溴辛烷 + $\mathrm{NaOH/H_2O}$，加热。

- **SN2 产物**：$(S)$-2-辛醇（构型翻转的取代产物）
- **E2 产物**：反式共平面消除，得到2-辛烯（三种异构体：顺-2-辛烯、反-2-辛烯 + 少量1-辛烯）

根据 Zaitsev 规则，E2 主产物为更稳定的取代更多的烯烃——2-辛烯为主（其中反-2-辛烯因热力学更稳定占优），1-辛烯为次。

### 反思

SN2 反应的核心特征是**构型完全翻转**（Walden 翻转）——这是 SN2 与 SN1（外消旋化）最关键的鉴别点。需要注意的是，若底物本身不含手性中心，翻转也无从观察；而当强碱存在且加热时，E2 将与 SN2 竞争，这是常见的干扰因素。

## 答案

(1) $(S)$-2-碘辛烷，$\mathrm{CH_3(CH_2)_5CHI(CH_3)}$

(2) $\mathrm{I^-}$ 从 C-Br 反面进攻，协同经过 $[\mathrm{I\cdots C\cdots Br}]^{\ddagger}$ 过渡态，碳中心构型翻转

(3) 取代：$(S)$-2-辛醇；消除：反-2-辛烯（主）+ 顺-2-辛烯 + 1-辛烯
