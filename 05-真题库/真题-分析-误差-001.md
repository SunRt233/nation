# 误差分析-测定结果的精密度和准确度评价

## 题目

某标准铁矿石样品中铁含量的标准值为 $\mu = 62.50\%$。甲、乙两位分析人员分别独立对此样品进行 5 次平行测定，结果如下：

**甲**：62.48%, 62.52%, 62.45%, 62.50%, 62.55%

**乙**：62.65%, 62.35%, 62.80%, 62.20%, 62.50%

(1) 分别计算两人测定结果的平均值 $\bar{x}$ 和标准偏差 $s$。

(2) 分别计算两人测定结果的相对标准偏差（RSD）。

(3) 比较两人的**精密度**和**准确度**：谁的测定结果更好？为什么？

(4) 甲的第五次测定结果 62.55% 是否属于异常值？试用 **Q 检验法**判断（置信度 90%，$n = 5$ 时 $Q_{\text{表}} = 0.64$）。

## 解析

### 分析

1. **准确度**衡量系统误差，用平均值与标准值之差（绝对误差）或相对误差来评价
2. **精密度**衡量偶然误差，用标准偏差 $s$ 或相对标准偏差 RSD 来评价
3. 异常值的 Q 检验：$Q = \frac{|x_{\text{可疑}} - x_{\text{最近}}|}{x_{\max} - x_{\min}}$，若 $Q > Q_{\text{表}}$ 则舍去

### 解答

**(1) 平均值和标准偏差**

**甲的测定**：

$$\bar{x}_{\text{甲}} = \frac{62.48 + 62.52 + 62.45 + 62.50 + 62.55}{5} = 62.500\%$$

$$s_{\text{甲}} = \sqrt{\frac{\sum (x_i - \bar{x})^2}{n-1}}$$

各偏差：$(−0.020), (0.020), (−0.050), (0.000), (0.050)$

$$s_{\text{甲}} = \sqrt{\frac{0.00040 + 0.00040 + 0.00250 + 0.00000 + 0.00250}{4}} = \sqrt{\frac{0.00580}{4}} = \sqrt{0.00145} = 0.0381\%$$

**乙的测定**：

$$\bar{x}_{\text{乙}} = \frac{62.65 + 62.35 + 62.80 + 62.20 + 62.50}{5} = 62.500\%$$

各偏差：$(0.150), (−0.150), (0.300), (−0.300), (0.000)$

$$s_{\text{乙}} = \sqrt{\frac{0.0225 + 0.0225 + 0.0900 + 0.0900 + 0.0000}{4}} = \sqrt{\frac{0.2250}{4}} = \sqrt{0.05625} = 0.237\%$$

**(2) 相对标准偏差 (RSD)**

$$\mathrm{RSD_{\text{甲}}} = \frac{s_{\text{甲}}}{\bar{x}_{\text{甲}}} \times 100\% = \frac{0.0381}{62.500} \times 100\% = 0.061\%$$

$$\mathrm{RSD_{\text{乙}}} = \frac{s_{\text{乙}}}{\bar{x}_{\text{乙}}} \times 100\% = \frac{0.237}{62.500} \times 100\% = 0.379\%$$

**(3) 精密度与准确度比较**

| 评价指标 | 甲 | 乙 |
|:---|:---:|:---:|
| 平均值 $\bar{x}$ | 62.500% | 62.500% |
| 绝对误差 $\bar{x} - \mu$ | 0.000% | 0.000% |
| 标准偏差 $s$ | 0.038% | 0.237% |
| RSD | 0.061% | 0.379% |

**准确度**：两组平均值恰好都等于标准值 62.500%，绝对误差均为零——两人准确度相同（均为无系统误差）。

**精密度**：甲的 $s_{\text{甲}} = 0.038\%$ 远小于乙的 $s_{\text{乙}} = 0.237\%$（约 6 倍），甲的 RSD 也远小于乙。

**结论**：甲的测定结果**精密度显著优于乙**，数据更加集中、可信。乙的数据虽然均值准确，但各次测定值波动很大（62.20%–62.80%），说明偶然误差大，操作稳定性差。

**(4) Q 检验判断异常值**

甲的测定值排序：62.45%, 62.48%, 62.50%, 62.52%, **62.55%**（可疑）

$$Q = \frac{|62.55 - 62.52|}{62.55 - 62.45} = \frac{0.03}{0.10} = 0.30$$

$Q = 0.30 < Q_{\text{表}}(0.64)$，故 62.55% **不是异常值**，应予以保留。

## 答案

(1) $\bar{x}_{\text{甲}} = 62.500\%$，$s_{\text{甲}} = 0.038\%$；$\bar{x}_{\text{乙}} = 62.500\%$，$s_{\text{乙}} = 0.237\%$

(2) $\mathrm{RSD_{\text{甲}}} = 0.061\%$；$\mathrm{RSD_{\text{乙}}} = 0.379\%$

(3) 准确度相同（均无系统误差），但甲的**精密度显著优于乙**（标准偏差小约 6 倍）

(4) $Q = 0.30 < 0.64$，62.55% 不是异常值，应保留
