# 钢铁发蓝处理——碱性氧化生成Fe₃O₄保护膜

## 题目

「发蓝」是钢铁表面处理的一种方法：将洁净的铁件浸入一定浓度的 $\mathrm{NaOH}$ 和 $\mathrm{NaNO_2}$ 溶液中（必要时加入其他辅助物质），加热到适当温度并保持一定时间，在铁件的表面形成均匀稳定且防锈的氧化膜 $\mathrm{Fe_3O_4}$，显现棕黑或蓝黑色光泽，处理过程有**无色刺激性气体**放出。

写出该反应的化学方程式。

## 参考答案

**反应方程式**：

$$
9\mathrm{Fe} + 8\mathrm{H_2O} + 4\mathrm{NO_2^-} \longrightarrow 3\mathrm{Fe_3O_4} + 4\mathrm{OH^-} + 4\mathrm{NH_3}
$$

**推导过程**：

1. **确定氧化剂和还原剂**：
   - 还原剂：Fe 单质（0 价）→ $\mathrm{Fe_3O_4}$（Fe 平均 +8/3 价）
   - 氧化剂：$\mathrm{NO_2^-}$（N +3 价）

2. **推断还原产物（关键步骤）**：
   N 元素氧化态低于 +3 时，符合"无色刺激性气体"的还原产物仅有 $\mathrm{NH_3}$：
   - NO：无色无味 ✗
   - N₂O：无色、略带甜味 ✗
   - NH₃：无色刺激性气体 ✓

3. **电子转移验证**：
   - Fe：$0 \rightarrow +8/3$（平均），9 个 Fe 共失 $24\mathrm{e^-}$
   - N：$+3 \rightarrow -3$，4 个 N 共得 $24\mathrm{e^-}$
   - 电子守恒 ✓

4. **碱性条件配平**：反应在 NaOH 溶液中进行，产物应有 OH⁻

**得分要点**：
- 正确推断产物：$\mathrm{Fe_3O_4}$、$\mathrm{NH_3}$，碱性条件下有 OH⁻（1.5分）
- 正确配平（1.5分）

**关于 Fe₃O₄**：
$\mathrm{Fe_3O_4}$ 可看作 $\mathrm{FeO} \cdot \mathrm{Fe_2O_3}$，含 Fe²⁺ 和 Fe³⁺ 的混合价态，平均氧化数为 +8/3。$\mathrm{Fe_3O_4}$ 膜致密、附着力强，能有效隔绝氧气和水，是理想的防锈保护层。

**钢铁表面处理技术对比**：

| 处理方法 | 成膜成分 | 颜色 | 原理 |
|---|---|---|---|
| 发蓝（发黑） | $\mathrm{Fe_3O_4}$ | 蓝黑/棕黑 | 碱性氧化 |
| 磷化 | 磷酸盐膜 | 灰黑 | 化学转化 |
| 钝化 | 氧化铬膜 | 银白 | 电化学 |
| 电镀 | 金属镀层 | 各异 | 电沉积 |

**易错点**：
- 还原产物写成 NO 或 N₂（未准确理解"无色刺激性气体"的描述）
- 产物写成 Fe₂O₃ 或 FeO（未注意题干明确给出 Fe₃O₄）
- 未写 OH⁻（未注意反应在碱性溶液中进行）
- 电子转移计算错误（Fe₃O₄ 中 Fe 为混合价态，平均氧化数为 +8/3）
