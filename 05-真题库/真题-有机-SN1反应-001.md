# SN1反应-叔丁基溴溶剂解与碳正离子中间体

## 题目

叔丁基溴 $(\mathrm{CH_3})_3\mathrm{CBr}$ 在乙醇-水混合溶剂中发生溶剂解反应。已知该反应速率只与叔丁基溴浓度成正比，与亲核试剂浓度无关。

(1) 写出反应的总速率方程，判断反应机理。

(2) 写出反应的完整机理，包括碳正离子中间体的生成与捕获步骤。

(3) 产物的立体化学特征是什么？若原碳中心为手性中心会怎样？请说明理由。

(4) 若在反应体系中加入适量的 $\mathrm{NaN_3}$，会得到哪些产物？为什么？

## 解析

### 分析

1. 速率只与 $[\mathrm{RX}]$ 成正比 = **一级动力学** → 决速步为底物单分子解离 → SN1 反应。
2. 三级碳正离子 $(\mathrm{CH_3})_3\mathrm{C^+}$ 稳定性高（3个甲基的超共轭效应），是合格的中间体。
3. SN1 经过平面碳正离子 → 亲核试剂从两侧进攻概率相等 → **外消旋化**。
4. 加入 $\mathrm{N_3^-}$ 会与溶剂竞争捕获碳正离子，得到竞争产物。

### 解答

**(1) 速率方程与机理判断**

速率方程：$v = k[(\mathrm{CH_3})_3\mathrm{CBr}]$（一级反应）

机理：**SN1 反应**（单分子亲核取代）。决速步是 C-Br 键异裂生成叔丁基碳正离子和 $\mathrm{Br^-}$，此步不涉及亲核试剂。

**(2) 完整机理**

**第一步（慢，决速步）**：C-Br 异裂

$$(\mathrm{CH_3})_3\mathrm{C} \!-\! \mathrm{Br} \xrightarrow{\text{慢}} (\mathrm{CH_3})_3\mathrm{C^+} + \mathrm{Br^-}$$

在极性质子溶剂中，$\mathrm{Br^-}$ 和碳正离子均被充分溶剂化稳定。

**第二步（快）**：碳正离子被亲核试剂捕获

$$(\mathrm{CH_3})_3\mathrm{C^+} + \mathrm{C_2H_5OH} \xrightarrow{\text{快}} (\mathrm{CH_3})_3\mathrm{COC_2H_5} + \mathrm{H^+}$$

$$(\mathrm{CH_3})_3\mathrm{C^+} + \mathrm{H_2O} \xrightarrow{\text{快}} (\mathrm{CH_3})_3\mathrm{COH} + \mathrm{H^+}$$

同时部分碳正离子发生 E1 消除：

$$(\mathrm{CH_3})_3\mathrm{C^+} \longrightarrow (\mathrm{CH_3})_2\mathrm{C=CH_2} + \mathrm{H^+}$$

**(3) 立体化学**

碳正离子为 $sp^2$ 杂化、**平面构型**。亲核试剂可从平面上下两侧等概率进攻 → 若原中心为手性碳，产物为**外消旋体**（部分或完全外消旋化）。

这是 SN1 与 SN2（构型完全翻转）最重要的区别之一。

实际上，离子对效应可能使外消旋化不完全——在紧密离子对阶段，离去基团仍部分遮蔽一侧，翻转产物略占优势（通常翻转:保持 ≈ 60:40 到 80:20），但远比 SN2 的彻底翻转来得弱。

**(4) 加入 NaN₃ 的影响**

$\mathrm{N_3^-}$ 是强亲核试剂。在 SN1 条件下，$\mathrm{N_3^-}$ 与溶剂（$\mathrm{C_2H_5OH}$、$\mathrm{H_2O}$）竞争捕获碳正离子中间体。产物为混合物：

- $(\mathrm{CH_3})_3\mathrm{CN_3}$（叠氮化叔丁烷）
- $(\mathrm{CH_3})_3\mathrm{COC_2H_5}$（叔丁基乙基醚）
- $(\mathrm{CH_3})_3\mathrm{COH}$（叔丁醇）
- $(\mathrm{CH_3})_2\mathrm{C=CH_2}$（异丁烯，E1 产物）

### 反思

SN1 反应的三个关键特征：**一级动力学**、**碳正离子中间体**、**外消旋化**。鉴别 SN1 vs SN2 最可靠的方法：手性底物做反应 → 外消旋化（SN1）vs 构型完全翻转（SN2）。需注意 SN1 与 E1 常共存——两者共享同一碳正离子中间体。
