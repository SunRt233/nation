# 钙钛矿结构-CaTiO3中各离子的配位数和占据位置

## 题目

钙钛矿（$\mathrm{CaTiO_3}$）是矿物钙钛矿的代表，其结构是众多功能材料（如铁电体 $\mathrm{BaTiO_3}$、高温超导相关结构等）的母结构。钙钛矿属于立方晶系，空间群 $Pm\bar{3}m$。

(1) 在理想立方钙钛矿结构中，指出 $\mathrm{Ca^{2+}}$、$\mathrm{Ti^{4+}}$、$\mathrm{O^{2-}}$ 各占据晶胞中的什么位置（写出分数坐标）。

(2) 分别指出 $\mathrm{Ca^{2+}}$ 和 $\mathrm{Ti^{4+}}$ 的氧配位数（即与几个 $\mathrm{O^{2-}}$ 相邻），以及 $\mathrm{O^{2-}}$ 的阳离子配位环境（与几个 $\mathrm{Ca^{2+}}$ 和几个 $\mathrm{Ti^{4+}}$ 相邻）。

(3) Goldschmidt 容忍因子 $t = \dfrac{r_A + r_X}{\sqrt{2}(r_B + r_X)}$ 是判断钙钛矿结构稳定性的经验参数。已知 $r_{\mathrm{Ca^{2+}}} = 100\ \mathrm{pm}$，$r_{\mathrm{Ti^{4+}}} = 61\ \mathrm{pm}$，$r_{\mathrm{O^{2-}}} = 140\ \mathrm{pm}$，计算 $\mathrm{CaTiO_3}$ 的容忍因子 $t$。钙钛矿结构的稳定范围通常为 $0.8 < t < 1.0$，$\mathrm{CaTiO_3}$ 是否在此范围内？

## 解析

### 分析

1. 钙钛矿 $\mathrm{ABX_3}$ 结构：A 为大阳离子（12 配位），B 为小阳离子（6 配位），X 为阴离子
2. 理想立方结构中 A 在顶点，B 在体心，X 在面心
3. 容忍因子 $t=1$ 对应理想立方钙钛矿结构；$t$ 偏离 1 时结构发生畸变（四方、正交等）

### 解答

**(1) 各离子的分数坐标**

理想立方钙钛矿结构（空间群 $Pm\bar{3}m$，$Z = 1$）：

| 离子 | Wyckoff 位置 | 分数坐标 |
|------|-------------|---------|
| $\mathrm{Ca^{2+}}$（A位） | $1a$ | $(0, 0, 0)$ |
| $\mathrm{Ti^{4+}}$（B位） | $1b$ | $\left(\frac{1}{2}, \frac{1}{2}, \frac{1}{2}\right)$ |
| $\mathrm{O^{2-}}$（X位） | $3d$ | $\left(\frac{1}{2}, \frac{1}{2}, 0\right)$，$\left(\frac{1}{2}, 0, \frac{1}{2}\right)$，$\left(0, \frac{1}{2}, \frac{1}{2}\right)$ |

**(2) 配位数**

- **$\mathrm{Ca^{2+}}$（A位）**：被 12 个 $\mathrm{O^{2-}}$ 包围（$\mathrm{Ca—O}$ 距离 $= \frac{a}{\sqrt{2}}$），**CN = 12**（立方八面体配位）

- **$\mathrm{Ti^{4+}}$（B位）**：被 6 个 $\mathrm{O^{2-}}$ 包围（$\mathrm{Ti—O}$ 距离 $= \frac{a}{2}$），**CN = 6**（正八面体配位）

- **$\mathrm{O^{2-}}$（X位）**：每个 $\mathrm{O^{2-}}$ 被 4 个 $\mathrm{Ca^{2+}}$（以共面方式）和 2 个 $\mathrm{Ti^{4+}}$（线性 $\mathrm{Ti—O—Ti}$ 排列）包围，即 **CN = 4（Ca）+ 2（Ti）= 6** 个阳离子。

可以理解为 $\mathrm{O^{2-}}$ 处于 $\mathrm{Ti^{4+}}$ 八面体的顶点 = $\mathrm{Ca^{2+}}$ 立方八面体的棱中心。

**(3) 容忍因子**

$$t = \frac{r_{\mathrm{Ca}} + r_{\mathrm{O}}}{\sqrt{2}(r_{\mathrm{Ti}} + r_{\mathrm{O}})} = \frac{100 + 140}{\sqrt{2} \times (61 + 140)}$$

$$= \frac{240}{\sqrt{2} \times 201} = \frac{240}{284.3} = 0.844$$

$t = 0.844 \in (0.8, 1.0)$，在钙钛矿结构的稳定范围内。但由于 $t < 1$，$\mathrm{CaTiO_3}$ 不是理想立方结构，实际在室温下为正交晶系（$\mathrm{GdFeO_3}$ 型畸变，$\mathrm{Ti—O—Ti}$ 键弯曲）。

### 反思

钙钛矿结构是竞赛高频考点。核心记忆点：A 位顶点（12 配位）、B 位体心（6 配位八面体）、X 位面心。容忍因子是连接离子半径与结构稳定性的重要概念——$t < 1$ 意味着 A 位阳离子"太小"，$\mathrm{BO_6}$ 八面体发生协同旋转以填充空间（导致对称性降低）。

## 答案

(1) $\mathrm{Ca}$: $(0,0,0)$；$\mathrm{Ti}$: $(\frac{1}{2},\frac{1}{2},\frac{1}{2})$；$\mathrm{O}$: $(\frac{1}{2},\frac{1}{2},0)$，$(\frac{1}{2},0,\frac{1}{2})$，$(0,\frac{1}{2},\frac{1}{2})$。

(2) $\mathrm{Ca^{2+}}$: CN = 12；$\mathrm{Ti^{4+}}$: CN = 6；$\mathrm{O^{2-}}$: 被 4 个 $\mathrm{Ca^{2+}}$ + 2 个 $\mathrm{Ti^{4+}}$ 包围。

(3) $t = 0.844$，在稳定范围内（$0.8{-}1.0$），但因 $t<1$ 实际结构发生正交畸变。
