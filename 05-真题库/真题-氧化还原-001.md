# 氧化还原-原电池电动势与Nernst方程

## 题目

已知 $E^\circ(\mathrm{Fe^{3+}/Fe^{2+}}) = +0.771\ \mathrm{V}$，$E^\circ(\mathrm{MnO_4^-/Mn^{2+}}) = +1.507\ \mathrm{V}$（在酸性条件下）。

(1) 写出以下原电池的电池符号并计算标准电动势：
$$\mathrm{Pt}|\mathrm{Fe^{2+}}(0.10\ \mathrm{mol\cdot L^{-1}}),\mathrm{Fe^{3+}}(0.010\ \mathrm{mol\cdot L^{-1}}) \parallel \mathrm{MnO_4^-}(0.10\ \mathrm{mol\cdot L^{-1}}),\mathrm{Mn^{2+}}(0.010\ \mathrm{mol\cdot L^{-1}}),\mathrm{H^+}(1.0\ \mathrm{mol\cdot L^{-1}})|\mathrm{Pt}$$

(2) 写出电池反应方程式并计算该电池在 298 K 时的电动势。

(3) 判断在给定条件下反应的自发方向。

## 解析

### 分析

1. 这是典型的**有液体接界的原电池**，需分别计算两极的**实际电极电势**再求电动势
2. 使用 Nernst 方程：$E = E^\circ - \frac{RT}{nF}\ln Q$，298 K 时简化为 $E = E^\circ - \frac{0.0592}{n}\lg Q$
3. MnO₄⁻ 在酸性条件下的半反应：$\mathrm{MnO_4^- + 8H^+ + 5e^- \rightleftharpoons Mn^{2+} + 4H_2O}$

### 解答

**(1) 电池符号**

负极（氧化，电势低的一极）：$\mathrm{Fe^{2+} \to Fe^{3+} + e^-}$

正极（还原，电势高的一极）：$\mathrm{MnO_4^- + 8H^+ + 5e^- \to Mn^{2+} + 4H_2O}$

电池符号已给出在题目中 ✓

标准电动势：$E^\circ_{\text{cell}} = E^\circ_{\text{正}} - E^\circ_{\text{负}} = 1.507 - 0.771 = 0.736\ \mathrm{V}$

**(2) 电池反应与电动势**

电池反应（负极 ×5 + 正极）：
$$\mathrm{5Fe^{2+} + MnO_4^- + 8H^+ \longrightarrow 5Fe^{3+} + Mn^{2+} + 4H_2O}$$

Nernst 方程计算各电极电势：

负极（Fe³⁺/Fe²⁺）：
$$E_- = 0.771 - \frac{0.0592}{1}\lg\frac{[\mathrm{Fe^{2+}}]}{[\mathrm{Fe^{3+}}]} = 0.771 - 0.0592\lg\frac{0.10}{0.010} = 0.771 - 0.0592 = 0.712\ \mathrm{V}$$

正极（MnO₄⁻/Mn²⁺）：
$$\begin{aligned} E_+ &= 1.507 - \frac{0.0592}{5}\lg\frac{[\mathrm{Mn^{2+}}]}{[\mathrm{MnO_4^-}][\mathrm{H^+}]^8} \\ &= 1.507 - \frac{0.0592}{5}\lg\frac{0.010}{0.10 \times 1.0^8} \\ &= 1.507 - 0.0118 \times \lg(0.10) \\ &= 1.507 - 0.0118 \times (-1) = 1.519\ \mathrm{V} \end{aligned}$$

电动势：$E_{\text{cell}} = E_+ - E_- = 1.519 - 0.712 = 0.807\ \mathrm{V}$

**(3) 反应自发性**

$E_{\text{cell}} = 0.807\ \mathrm{V} > 0$，$\Delta G = -nFE_{\text{cell}} < 0$

反应**正向自发**进行（Fe²⁺ 被 MnO₄⁻ 氧化为 Fe³⁺）。

## 答案

(1) $E^\circ_{\text{cell}} = 0.736\ \mathrm{V}$

(2) $E_{\text{cell}} = 0.807\ \mathrm{V}$

(3) 正向自发（Fe²⁺ → Fe³⁺，被 MnO₄⁻ 氧化）
