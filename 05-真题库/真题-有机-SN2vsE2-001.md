# SN2 vs E2竞争-1-溴丙烷在不同条件下的产物切换

## 题目

1-溴丙烷 $\mathrm{CH_3CH_2CH_2Br}$ 分别在下述四种条件下反应。请预测每种条件下的主要有机产物（写出结构式或名称），并简要说明判断理由。

| 编号 | 试剂/溶剂 | 温度 |
|:---:|:---|:---:|
| (a) | $\mathrm{NaCN}$ / DMSO | $25^\circ\mathrm{C}$ |
| (b) | $\mathrm{NaOCH_3}$ / $\mathrm{CH_3OH}$ | $25^\circ\mathrm{C}$ |
| (c) | $\mathrm{NaOCH_3}$ / $\mathrm{CH_3OH}$ | 回流 |
| (d) | $\mathrm{KOt\text{-}Bu}$ / $\mathrm{t\text{-}BuOH}$ | 回流 |

附加问题：在条件 (c) 中得到的产物比例约为 丙基甲基醚:丙烯 = 7:3；而在条件 (d) 中该比例为 1:9。请从**碱的亲核性与位阻**的角度解释这一巨大差异。

## 解析

### 分析

1. 本题的核心是 SN2 vs E2 竞争受四个因素影响：**底物结构**（1° 有利 SN2）、**试剂亲核性 vs 碱性**、**溶剂效应**、**温度**。
2. $\mathrm{CN^-}$ 是极好的亲核试剂、弱碱 → SN2 绝对主导。
3. $\mathrm{CH_3O^-}$ 是中等亲核试剂 + 中等碱 → 室温 SN2 为主，回流时 E2 增强。
4. $t\!\text{-BuO}^-$ 是大位阻强碱 → 亲核性极弱，几乎只走 E2 路径。

### 解答

**(a) NaCN / DMSO, 25°C**

- 底物：1° 卤代烷 → SN2 极有利
- 试剂：$\mathrm{CN^-}$ 是非常好的亲核试剂（C 端软碱），$\mathrm{HCN}$ 的 $\mathrm{p}K_\mathrm{a} \approx 9.2$，碱性很弱
- 溶剂：DMSO（极性非质子）→ "裸阴离子"，亲核性大幅增强
- **主要产物**：$\mathrm{CH_3CH_2CH_2CN}$（丁腈），纯 SN2

**(b) NaOCH₃ / CH₃OH, 25°C**

- $\mathrm{CH_3O^-}$ 是中等强度碱和中等亲核试剂
- 室温下 E2 较慢，SN2 主导但已有部分 E2 竞争
- **主要产物**：$\mathrm{CH_3CH_2CH_2OCH_3}$（甲基正丙基醚），少量 $\mathrm{CH_3CH=CH_2}$（丙烯）

**(c) NaOCH₃ / CH₃OH, 回流**

- 高温偏向消除反应（消除反应产生 3 个分子 → 熵增更大）
- SN2 与 E2 实质竞争，SN2 仍略占优（伯卤代烷 + 中等强度碱）
- **产物比例**：约 70% SN2 产物（丙基甲基醚）+ 30% E2 产物（丙烯）

**(d) KOt-Bu / t-BuOH, 回流**

- $t\!\text{-BuO}^-$ 位阻极大 → O⁻ 难以接近碳背面进行 SN2
- $t\!\text{-BuO}^-$ 是强碱 → 易夺取 β-H 发生 E2
- 回流条件进一步偏向 E2
- **主要产物**：$\mathrm{CH_3CH=CH_2}$（丙烯），→90% E2

**附加解释**：$\mathrm{CH_3O^-}$（甲氧基离子）位阻较小，既可以做亲核试剂又可以做碱，在 1° 底物上以 SN2 为主。而 $t\!\text{-BuO}^-$（叔丁氧基离子）三个甲基的巨大位阻使其几乎不能从碳背面进攻发生 SN2，但作为碱夺取 β-H 不受位阻限制。因此 $t\!\text{-BuO}^-$ 是典型的 **Hofmann 型碱**——以消除为主。

### 反思

判断 SN2 vs E2 竞争的黄金法则是四个维度：**底物（1°/2°/3°）、试剂（亲核性 vs 碱性 vs 位阻）、溶剂（质子 or 非质子极性）、温度（高温→消除）**。伯卤代烷 + 大位阻强碱 + 高温 = E2 主导（Hofmann 消除），这是本题目最容易考到的"极限条件"。
