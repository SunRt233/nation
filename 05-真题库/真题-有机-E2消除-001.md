# E2消除-2-溴丁烷消除与Zaitsev规则

## 题目

2-溴丁烷 $\mathrm{CH_3CH_2CHBrCH_3}$ 在热乙醇溶液中与 $\mathrm{KOH}$ 作用，发生消除反应。

(1) 写出所有可能的消除产物的结构式，并指出主产物和次产物。

(2) 从 E2 反应的立体化学要求（反式共平面消除）出发，画出生成主产物所对应的最稳定构象（Newman 投影式）。

(3) 在此基础上，解释为什么消旋-2-溴丁烷消除生成的反-2-丁烯与顺-2-丁烯的比例受底物构型影响。若使用纯 $(2S,3R)$-2-溴-3-氘代丁烷消除，主要得到哪种烯烃？

## 解析

### 分析

1. 2-溴丁烷有两个不同的 β-碳：C1（甲基端）和 C3（亚甲基端），可给出两种烯烃：1-丁烯和2-丁烯。
2. **Zaitsev 规则**：碱性消除中，取代更多的烯烃更稳定 → 2-丁烯为主产物。
3. E2 有严格的**反式共平面**要求——被消除的 H 与 Br 必须成 180° 二面角。底物的构型决定了能消除到哪个烯烃。

### 解答

**(1) 消除产物**

| 产物 | 结构式 | 类型 |
|:---|:---|:---|
| 2-丁烯 | $\mathrm{CH_3CH=CHCH_3}$ | 主产物（Zaitsev） |
| 1-丁烯 | $\mathrm{CH_3CH_2CH=CH_2}$ | 次产物 |

2-丁烯又分为顺式和反式两种。在热力学控制下，**反-2-丁烯**比顺式更稳定（两个甲基在双键异侧，位阻较小），通常为主。

**(2) 生成反-2-丁烯的 Newman 投影**

E2 消除要求 H-C3 与 C2-Br 成**反式共平面**（二面角 = 180°）。2-溴丁烷绕 C2-C3 旋转到如下构象（Newman 投影沿 C2→C3 方向看）：

```
        CH₃
       /
  H──C2
       \
        Br
        
        H
       /
  CH₃──C3
       \
        H
```

此时 C2-Br 与 C3-H（来自 CH₃-基团）呈反式共平面关系 → E2 消除直接得到反-2-丁烯。

**(3) 不同构型底物的产物差异**

消旋-2-溴丁烷有两个对映体 $(2R)$-2-溴丁烷和 $(2S)$-2-溴丁烷，每种构型绕 C2-C3 旋转后，能与 Br 成反式共平面的 H 不同。

具体到 $(2S,3R)$-2-溴-3-氘代丁烷（已知 $2S$ 构型的布氏排列）：

- 在反式共平面构象中，Br 与 C3 上的 H（而非 D）处于反式 → 消除 H 而非 D
- 得到含 D 的烯烃：$\mathrm{CH_3CD=CHCH_3}$（D 留在双键上）

若使用相反构型 $(2R,3R)$，则 Br 与 D 反式共平面 → 消除 D 而非 H → 得到 $\mathrm{CH_3CH=CDCH_3}$。

这正是 **E2 立体专一性**的经典证明实验。

### 反思

E2 机理看似简单，但其**反式共平面**的立体化学要求使得反应具有高度的立体专一性——这个特征在初赛中常考构象分析与产物预测。Zaitsev 规则给出热力学主产物方向，但大位阻碱（如 $t\!\text{-BuO}^-$）时 Hofmann 规则介入，需分场景分析。
