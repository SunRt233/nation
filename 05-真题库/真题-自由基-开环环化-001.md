# 自由基开环环化反应

## 题目

以下反应是天然产物顺式茉莉酮全合成中的一步。仔细分析反应转换过程中三元环的作用，完成反应式，画出主要产物的结构简式（不考虑产物的立体化学）。

反应条件：Bu₃SnH、AIBN、甲苯、加热。底物含三元环和双键。

（原题含反应式图片，参见 34 届初赛试题解析第 10-3-2 题图）

## 参考答案

三元环在高温下打开，生成五元环产物。由于反应在非极性溶剂下进行，自由基机理比较合适。

反应机理分析：
1. **引发**：AIBN 热分解产生自由基，夺取 Bu₃SnH 的氢生成 Bu₃Sn·
2. **链转移**：Bu₃Sn· 引发底物自由基链反应
3. **开环**：自由基进攻三元环，或三元环张力驱动均裂打开
4. **环化**：开环后新生成的自由基进攻双键，环化形成五元环（遵循 Baldwin 规则）
5. **终止**：从 Bu₃SnH 夺取氢，完成还原

（产物结构图参见 34 届初赛试题解析图片）

**得分要点**：
- 正确画出产物结构（3分）：自由基引发三元环开环，生成五元环产物

**关键判断**：
- Bu₃SnH、AIBN、甲苯、加热是典型的自由基反应条件
- 非极性溶剂 + 高温有利于自由基机理（区别于 10-3-1 的离子型机理）
- 自由基环化通常优先生成五元环（Baldwin 规则）

**易错点**：
- 按离子型机理推导（忽略了 Bu₃SnH/AIBN/甲苯/加热是典型自由基条件）
- 产物环大小错误（对自由基环化的区域选择性不清楚）
- 忽略立体化学提示（题目明确说不考虑立体化学）
