# 元素推断-从反应现象推断卤素化合物Cl/Br/I

## 题目

有三种无色钠盐 A、B、C，分别溶于水后进行以下实验：

- **A 溶液**：加入 $\mathrm{AgNO_3}$ 溶液产生白色沉淀，沉淀不溶于稀 $\mathrm{HNO_3}$，但溶于浓 $\mathrm{NH_3\cdot H_2O}$。A 的固体与浓 $\mathrm{H_2SO_4}$ 加热产生无色有刺激性气味的气体 D。
- **B 溶液**：加入 $\mathrm{AgNO_3}$ 溶液产生淡黄色沉淀，沉淀几乎不溶于稀 $\mathrm{HNO_3}$，也几乎不溶于浓 $\mathrm{NH_3\cdot H_2O}$。B 的固体与浓 $\mathrm{H_2SO_4}$ 加热产生红棕色气体 E 和棕色溶液。
- **C 溶液**：加入 $\mathrm{AgNO_3}$ 溶液产生黄色沉淀，沉淀不溶于稀 $\mathrm{HNO_3}$，也不溶于浓 $\mathrm{NH_3\cdot H_2O}$。C 的固体与浓 $\mathrm{H_2SO_4}$ 加热产生紫色蒸气 F。

(1) 推断 A、B、C、D、E、F 各是什么物质，写出化学式。

(2) 写出 B 与浓 $\mathrm{H_2SO_4}$ 反应的化学方程式。

(3) 将 C 的水溶液与氯水混合，溶液变为棕黄色；再加少量 $\mathrm{CCl_4}$ 振荡，$\mathrm{CCl_4}$ 层呈紫色。解释现象并写出离子方程式。

## 解析

### 分析

1. 通过银盐沉淀颜色判断卤素：AgCl（白）→ A 含 $\mathrm{Cl^-}$；AgBr（淡黄）→ B 含 $\mathrm{Br^-}$；AgI（黄）→ C 含 $\mathrm{I^-}$
2. 沉淀在 $\mathrm{NH_3\cdot H_2O}$ 中的溶解性：AgCl 溶（形成 $[\mathrm{Ag(NH_3)_2}]^+$），AgBr 微溶，AgI 几乎不溶——与实验吻合
3. 与浓 $\mathrm{H_2SO_4}$ 的反应：$\mathrm{Cl^-}$ → HCl↑；$\mathrm{Br^-}$ → $\mathrm{Br_2 + SO_2}$（氧化还原）；$\mathrm{I^-}$ → $\mathrm{I_2 + H_2S}$（更强还原性）

### 解答

**(1) 物质推断**

| 代号 | 物质 | 化学式 | 理由 |
|------|------|--------|------|
| A | 氯化钠 | $\mathrm{NaCl}$ | AgCl 白↓，溶于浓氨水 |
| B | 溴化钠 | $\mathrm{NaBr}$ | AgBr 淡黄↓，不溶于浓氨水 |
| C | 碘化钠 | $\mathrm{NaI}$ | AgI 黄↓，不溶于浓氨水 |
| D | 氯化氢 | $\mathrm{HCl}$ | NaCl + 浓 $\mathrm{H_2SO_4}$ → HCl↑ |
| E | 溴（+ $\mathrm{SO_2}$） | $\mathrm{Br_2}$ | NaBr 被浓 $\mathrm{H_2SO_4}$ 氧化 |
| F | 碘 | $\mathrm{I_2}$ | NaI 被浓 $\mathrm{H_2SO_4}$ 氧化，紫色蒸气 |

**(2) NaBr 与浓 $\mathrm{H_2SO_4}$ 的反应**

$\mathrm{Br^-}$ 的还原性强于 $\mathrm{Cl^-}$，能被浓 $\mathrm{H_2SO_4}$ 氧化：

$$2\mathrm{NaBr} + 3\mathrm{H_2SO_4} \xrightarrow{\Delta} 2\mathrm{NaHSO_4} + \mathrm{Br_2} + \mathrm{SO_2}\uparrow + 2\mathrm{H_2O}$$

或写成：
$$\mathrm{2NaBr + 2H_2SO_4 \xrightarrow{\Delta} Na_2SO_4 + Br_2 + SO_2\uparrow + 2H_2O}$$

红棕色气体为 $\mathrm{Br_2}$ 蒸气，同时有 $\mathrm{SO_2}$ 产生。

**对比**：
- $\mathrm{NaCl + H_2SO_4 \to NaHSO_4 + HCl\uparrow}$（非氧化还原，仅复分解）
- $\mathrm{NaI + H_2SO_4 \to I_2 + H_2S\uparrow}$（$\mathrm{I^-}$ 还原性更强，$\mathrm{H_2SO_4}$ 被还原到 $\mathrm{H_2S}$）

**(3) NaI 与氯水的反应**

氯水将 $\mathrm{I^-}$ 氧化为 $\mathrm{I_2}$：

$$2\mathrm{I^-} + \mathrm{Cl_2} \to \mathrm{I_2} + 2\mathrm{Cl^-}$$

生成的 $\mathrm{I_2}$ 在水中呈棕黄色，在 $\mathrm{CCl_4}$ 中溶解度更大且呈紫色（$\mathrm{I_2}$ 在非极性溶剂中为紫色，在极性溶剂中因形成电荷转移配合物而呈棕色）。$\mathrm{CCl_4}$ 层萃取 $\mathrm{I_2}$ 后呈紫色，这是鉴定 $\mathrm{I_2}$ 的经典方法。

### 反思

卤素推断题的线索链：银盐颜色与氨溶性 → 卤素离子身份；浓 $\mathrm{H_2SO_4}$ 反应产物 → 验证并区分氧化性差异（$\mathrm{Cl^-} < \mathrm{Br^-} < \mathrm{I^-}$）。常见易混点：(1) AgBr 为淡黄色（非纯黄），AgI 为黄色；(2) $\mathrm{Br^-}$ 被浓 $\mathrm{H_2SO_4}$ 氧化产物同时有 $\mathrm{Br_2}$ 和 $\mathrm{SO_2}$；(3) $\mathrm{I_2}$ 在 $\mathrm{CCl_4}$ 中为紫色（非棕色）。

## 答案

(1) A = NaCl, B = NaBr, C = NaI, D = HCl, E = Br₂, F = I₂。

(2) $2\mathrm{NaBr} + 3\mathrm{H_2SO_4} \xrightarrow{\Delta} 2\mathrm{NaHSO_4} + \mathrm{Br_2} + \mathrm{SO_2}\uparrow + 2\mathrm{H_2O}$。

(3) $\mathrm{Cl_2 + 2I^- \to I_2 + 2Cl^-}$；$\mathrm{I_2}$ 被 $\mathrm{CCl_4}$ 萃取呈紫色。
