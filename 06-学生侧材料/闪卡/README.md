# 闪卡

本目录用于存放可打印、可课堂发放、可进一步导出到 Anki 的闪卡材料。

> **页面边界（2026-06-15）**：本页负责“闪卡文件本身放哪里、按什么模板写”；批量 CSV 导出规则不在这里，统一去 [[06-学生侧材料/闪卡/anki-export/README]]。

- 推荐来源：KP 关键结论、常见陷阱、专题页对比表
- 推荐模板：[[11-模板/模板-学生闪卡]]
- Anki 导出说明：[[06-学生侧材料/闪卡/anki-export/README]]

## 推荐拆分

- `课堂即发型`：单节课后可直接发放，强调 1 个概念 / 1 个陷阱 / 1 个决策
- `专题复盘型`：围绕一个专题做成成组卡片，便于阶段性复习
- `导出来源型`：由 KP `review_cycle` 批量筛出的候选卡，后续再人工筛质
