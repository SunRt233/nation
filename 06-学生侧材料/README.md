# 学生侧材料

> 本目录承接学生可直接使用的学习材料。
> 当前作为学生可交付材料的总入口，后续按材料类型分目录维护。

> **页面边界（2026-06-15）**：本页负责“学生最终会拿到什么材料、材料该落在哪个目录”。它不负责安排知识点复习节奏，也不负责讲 Anki 导出脚本细节。
> - 复习排程 / `review_cycle`：去 [[00-首页/KP复习清单]]
> - 闪卡落点与组织方式：去 [[06-学生侧材料/闪卡/README]]
> - Anki CSV 导出 / 导入细节：去 [[06-学生侧材料/闪卡/anki-export/README]]

## 目录结构

- `闪卡/`：概念卡、公式卡、对比卡、信号卡等记忆层材料
- `练习卷/`：专题练习卷、限时小测、配套详解
- `策略卡/`：方法策略卡、口袋卡、决策树类材料

## 使用规则

1. 新建学生侧材料时，优先落到对应子目录。
2. 试点材料文件名建议包含日期、专题、轮次或班型。
3. 先决定材料形态，再决定是否需要回写课堂页、专题页或知识点页。
4. 纸质版验证、Anki 导出、学生反馈，分别在对应活跃任务卡中继续推进。
5. 若需要从知识点侧反推“这周该复习什么”，优先查看 [[00-首页/KP复习清单]]。
6. 若要把现有 Markdown 讲义正式导出为 `.docx` 或与投影 PPT 配套，先去 [[04-课件/加工页/README]] 建加工页，不要直接跳过加工决策层。

## 推荐入口（先用这里）

```text
想给学生新增一份材料
  ↓
先判断它属于 闪卡 / 练习卷 / 策略卡 哪一类

进入对应 README 看模板与落点
  ↓
若要按复习周期筛选知识点，再回 [[KP复习清单]]
  ↓
若要导出 Anki，再去 anki-export/README
```
