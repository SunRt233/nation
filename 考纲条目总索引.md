# 考纲条目总索引

> 本页自动列出全部考纲条目及其覆盖状态。
> 点击条目名可进入对应考纲条目页。
> 数字由 Dataview 自动计算。

---

## 一、基础要求

```dataview
TABLE
  syllabus_module AS 所属模块,
  coverage_status AS 覆盖状态,
  importance AS 重要性,
  length(knowledge_notes) AS 关联KP数
FROM "02-考纲条目/基础要求"
WHERE type = "考纲条目"
SORT file.name ASC
```
