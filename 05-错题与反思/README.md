# 错题与反思区

> 本目录用于收录竞赛学习过程中的错题、复盘记录和反思笔记。

## 用途

- 按科目/模块分类整理错题
- 记录错题的错因分析、正确思路、关联知识点
- 定期回顾，形成个人薄弱环节图谱

## 文件命名规范

- `错题-{科目}-{日期}-{主题}.md`
- `复盘-{课程/考试名称}-{日期}.md`

## 与 04-题库 的区别

| | 04-题库 | 05-错题与反思 |
|---|---|---|
| 内容来源 | 真题/教材/经典题 | 个人做错/卡住的题 |
| 所有权 | 知识库公共资产 | 个人学习记录 |
| 目的 | 供所有人使用 | 供自己回顾提升 |
