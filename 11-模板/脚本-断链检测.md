# 断链检测脚本

> 扫描 vault 中所有 Markdown 文件的 wikilink，检测指向不存在的文件的断链。
>
> **运行频率**：建议每月一次，或大批量修改后运行。
> **执行方式**：在 PowerShell 中运行下方脚本，或交给 Agent 9（健康检查）执行。

---

## 使用方法

1. 打开 PowerShell（vault 根目录）
2. 复制下方脚本内容
3. 查看输出的 `broken_links_report_YYYYMMDD.txt`

---

## 检测脚本

```powershell
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
$vaultRoot = "C:\Obsidion\妙妙屋"

# 收集 vault 中所有 .md 文件 + .pdf 文件（wikilink 可指向 PDF）
$allMdFiles = Get-ChildItem $vaultRoot -Filter "*.md" -Recurse
$allPdfFiles = Get-ChildItem $vaultRoot -Filter "*.pdf" -Recurse
$allAssetFiles = $allMdFiles + $allPdfFiles

$allPaths = $allAssetFiles | ForEach-Object { $_.FullName.Substring($vaultRoot.Length + 1) -replace '\', '/' }
$allNames = $allMdFiles | ForEach-Object { [System.IO.Path]::GetFileNameWithoutExtension($_.Name) }

# PDF 文件名（含扩展名，用于直接匹配 PDF wikilink）
$allPdfNames = $allPdfFiles | ForEach-Object { $_.Name }

# 建立文件名 -> 路径列表 的映射（处理同名文件）
$nameToPaths = @{}
foreach ($f in $allMdFiles) {
    $name = [System.IO.Path]::GetFileNameWithoutExtension($f.Name)
    if (-not $nameToPaths.ContainsKey($name)) { $nameToPaths[$name] = @() }
    $relPath = $f.FullName.Substring($vaultRoot.Length + 1) -replace '\', '/'
    $nameToPaths[$name] += $relPath
}

# 预期的占位链接（这些不是真正的断链）
$placeholderPatterns = @(
    '^\s*$',           # 空
    '^\s*-\s*$',       # "-"
    '^题型-\s*$',       # "题型- "
    '^专题-\s*$',       # "专题- "
    '^\d+$',           # 纯数字（通常是考纲编号占位）
    '^.*?待建.*$',     # 含"待建"
    '^.*?待补充.*$',   # 含"待补充"
    '^知识点-\s*$',    # "知识点- "
    '\.base$',         # .base 占位文件
    '^https?://'       # 误写成 wikilink 的 URL
)

function Is-Placeholder($text) {
    foreach ($pat in $placeholderPatterns) {
        if ($text -match $pat) { return $true }
    }
    return $false
}

$brokenLinks = @()
$totalLinks = 0

foreach ($mdFile in $allMdFiles) {
    $content = Get-Content $mdFile.FullName -Encoding utf8 -Raw
    if (-not $content) { continue }

    $matches = [regex]::Matches($content, '\[\[(.*?)\]\]')
    foreach ($m in $matches) {
        $rawLink = $m.Groups[1].Value
        $totalLinks++

        # 去掉别名（处理转义的 \|）
        $target = $rawLink.Split('|')[0]
        # 如果原始文本使用 \| 转义，Split 后末尾会残留反斜杠
        $target = $target.TrimEnd('\')
        # 去掉块引用/标题锚点
        $target = $target.Split('#')[0]
        $target = $target.Trim()

        # 跳过占位符
        if (Is-Placeholder $target) { continue }

        # 标准化路径分隔符
        $target = $target -replace '\', '/'

        $exists = $false

        if ($target -match '/') {
            # 带路径的链接，先检查精确路径（支持 .md 和 .pdf）
            $testPathMd = $target + ".md"
            $testPathPdf = $target  # 可能已含 .pdf 扩展名
            if ($allPaths -contains $testPathMd -or $allPaths -contains $testPathPdf) {
                $exists = $true
            } else {
                # 再检查文件名是否存在于 vault 中（Obsidian 回退搜索）
                $fileName = [System.IO.Path]::GetFileNameWithoutExtension($target.Split('/')[-1])
                if ($allNames -contains $fileName) {
                    $exists = $true
                }
                # 检查是否为 PDF 文件名（含扩展名）
                $fileNameWithExt = $target.Split('/')[-1]
                if ($allPdfNames -contains $fileNameWithExt) {
                    $exists = $true
                }
            }
        } else {
            # 纯文件名链接
            if ($allNames -contains $target) {
                $exists = $true
            }
        }

        if (-not $exists) {
            $relFile = $mdFile.FullName.Substring($vaultRoot.Length + 1)
            $brokenLinks += [PSCustomObject]@{
                SourceFile = $relFile
                LinkText   = "[[$rawLink]]"
                Target     = $target
            }
        }
    }
}

# 输出报告
$reportPath = Join-Path $vaultRoot "11-模板/broken_links_report_$(Get-Date -Format 'yyyyMMdd').txt"
$reportLines = @("# 断链检测报告", "生成时间: $(Get-Date -Format 'yyyy-MM-dd HH:mm:ss')", "", "统计:", "- 扫描文件数: $($allMdFiles.Count)", "- 总链接数: $totalLinks", "- 断链数: $($brokenLinks.Count)", "- 占位跳过数: $($totalLinks - $brokenLinks.Count - ($allMdFiles.Count))", "")

if ($brokenLinks.Count -gt 0) {
    $reportLines += "## 断链清单"
    $reportLines += ""
    $reportLines += "| 源文件 | 链接文本 | 目标 |"
    $reportLines += "|--------|---------|------|"
    foreach ($bl in $brokenLinks) {
        $reportLines += "| $($bl.SourceFile) | $($bl.LinkText) | $($bl.Target) |"
    }
} else {
    $reportLines += "✅ 未发现断链。"
}

$reportLines += ""
$reportLines += "---"
$reportLines += "*报告由断链检测脚本自动生成*"

[IO.File]::WriteAllLines($reportPath, $reportLines, [Text.UTF8Encoding]::new($false))
Write-Host ""
Write-Host "=== 断链检测完成 ==="
Write-Host "扫描文件: $($allMdFiles.Count)"
Write-Host "总链接: $totalLinks"
Write-Host "断链数: $($brokenLinks.Count)"
Write-Host "报告已保存: $reportPath"
if ($brokenLinks.Count -gt 0) {
    Write-Host ""
    Write-Host "断链预览:"
    $brokenLinks | Select-Object -First 10 | ForEach-Object { Write-Host "  $($_.SourceFile) -> $($_.LinkText)" }
}
```

## 维护说明

- 检测到断链后，判断类型：
  - **待建链接**（如专题总索引中的 🔴 待建）→ 正常，无需处理
  - **误删/移动导致** → 修正链接或恢复文件
  - **拼写错误** → 修正 wikilink 文本
- 建议每月由 Agent 9（健康检查）运行一次，结果追加到工作日志。
