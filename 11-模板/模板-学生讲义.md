# 模板：学生讲义

> 📋 **使用方式**：为每个专题/知识点生成**教材级自修材料**，落到 `04-课件/学生讲义/<日期>-<主题>.md`。
>
> 🎯 **职责定位**：**学生课后可独立阅读的教材级参考书**。包含完整的理论推导、知识脉络、丰富的例题和分层练习，供课前预习、课后深化、考前系统复习使用。
>
> ⚠️ **与其他课件的区别**：
> - **备课大纲**（[[模板-备课大纲]]）是**教师教学设计**，不包含知识内容本身。
> - **新授课**（[[模板-新授课]]）是**课堂跟随讲义**，适配 90 min 课时，精炼为主。
> - **本模板（学生讲义）**是**教材级参考书**，篇幅允许 500-800 行，包含完整推导、大量例题、分层练习和跨专题联系。
>
> **使用顺序**：教师先用备课大纲完成教学设计 → 新授课用于课堂 → 学生讲义作为预习/复习/自学材料（可提前发给学生，亦可课后作为深化阅读）。

---

## 三件套课件体系

- **备课大纲**（教师：教学设计的施工蓝图）
  - **新授课**（课堂：90min 精炼跟随讲义）
  - **学生讲义**（自学：教材级参考书）← 本模板
  - **速查卡**（快速：决策树/公式速查）

## 写作指南

### 1. 篇幅控制

| 层级 | 预计行数 | 说明 |
|:---|:---:|:---|
| 核心概念（§1-2） | ~200 行 | 展开推导 + 表格 |
| 典型例题（§3） | ~150 行 | 3-5 道，带完整解析 |
| 练习题（§6） | ~100 行 | 10-15 道，分层 |
| 其他 | ~100 行 | 导入/总结/拓展/答案 |
| **合计** | **~550 行** | 教材一章的典型长度 |

### 2. 公式要求

- 全部使用 LaTeX 行内/行间公式
- 关键推导步骤保留，不要只抛结论
- 每个公式标注符号含义

### 3. 表格要求

- 对比表优先使用 Markdown 表格（简洁）
- 复杂表格（含合并单元格）使用 HTML `<table>`

### 4. 例题要求

- 每道题标记难度 ⭐~⭐⭐⭐⭐⭐
- 包含「思路分析」和「完整解答」
- 好题标注来源（教材/真题/自编）

### 5. 与知识库的对接

- 所有 wikilink 指向 KB 中实际存在的文件
- 每篇讲义更新后，在对应备课大纲的 §0.1A 添加引用行
- 在对应专题页的参考资料章节添加链接
