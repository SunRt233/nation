# 1. 过渡金属无机化学 00:47

● 课程定位：本课程为ZCHEM500的L5/6/7部分，属于过渡金属无机化学（Chemistry of Transition Metals）范畴  
● 与前序关系：承接葛瑞老师讲授的主族元素无机物结构化学内容  
● 核心特征：相比本科阶段接触的描述性无机化学，本课程更强调理论体系

# 2. 配位化学简介 01:02

● 基础地位：在ZCHEM300中已有配位化学简介（Brief Introduction of Coordination Chemistry）

● 课程深度：当前讲解的配位化学基础知识已足够应对国际化学奥林匹克竞赛要求  
● 学科发展：配位化学是标准的研究生课程，本科化学教育通常不系统涉及

# 3. 金属有机化学 01:43

● 定义特征：含有至少一根金属-碳键的配位化合物

# - 学科演变：

- 早期：作为配位化学的子集（无机化学家研究）  
现代：发展为与配位化学并列的交叉学科（50%无机化学+50%有机化学）

# ● 研究视角：

- 无机视角：关注结构（Structure）  
- 有机视角：关注反应性（Reactivity），如催化循环、偶联反应等

\- 经典案例：Grubbs催化剂（烯烃复分解反应），由Berkeley有机化学教授与MIT无机化学教授共同获得诺奖

# 4. Cluster化学 02:13

● 典型代表：羰基金属原子簇化合物  
● 扩展内容：可能补充讲解硼烷、碳硼烷等（视课时情况而定）

水合异构(hydrate isomer): 水分子在配合物内界或外界的位置不同

● 电离异构(ionization isomer): 内外界离子交换导致性质差异，如 $[CoCl(NH_{3})_{5}]SO_{4}$ 与 $[CoSO_{4}(NH_{3})_{5}]Cl$

● $\sigma$ 给体( $\sigma$ donor): 仅提供 $\sigma$ 电子对与金属形成配位键  
- $\pi$ 受体(πacceptor): 既能提供σ电子对, 又能通过空 $\pi^{*}$ 轨道接受金属d电子  
● $\pi$ 给体( $\pi$ donor): 提供 $\sigma$ 电子对的同时，还能通过填充的 $\pi$ 轨道向金属反馈电子

# 2）配体场理论应用

● 主要解释对象: 特别适合解释光谱化学序列中不同配体场强差异  
● 理论优势: 能完整解释 $\pi$ 作用对分裂能的影响（如CN^-作为强场配体因其 $\pi$ 受体特性）

# 7. 分裂能影响因素 03:10:53

# 1）配体类型对分裂能的影响

● 强场配体特征：

○ 强的σ给体：使分裂能Δ上升（如氢负离子H⁻）  
○ 强的 $\pi$ 受体：同样使分裂能 $\Delta$ 上升（如CN $^{-}$ 、CO、三烷基磷）  
○ 实际应用中强场配体多为 $\pi$ 受体配体

# - 弱场配体特征：

○ 强的 $\pi$ 给体：使分裂能 $\Delta$ 下降（典型代表为氟离子 $F^{-}$ ）  
- 这类配体同时具有 $\sigma$ 给电子能力，但 $\pi$ 相互作用主导场强

# 2）典型配体分类

# ● 强场配体实例：

- 无机配体：CN $^{-}$ 、CO  
- 有机配体：含大 $\pi$ 体系的配体  
○ 磷配体：三烷基磷（通过d轨道π反键作用）

# ● 中强场配体：

- $\mathrm{NH}_{3}$ : 纯 $\sigma$ 给体, 非强 $\sigma$ 给体  
○ $H_{2}O$ : $\sigma$ 给体能力中等

# - 弱场配体实例：

- 卤素离子（特别是 $F^{-}$ ）：兼具 $\sigma$ 给体和 $\pi$ 给体特性  
- $\pi$ 给体作用显著降低分裂能

# 3）作用机制说明

# ● 作用顺序:

○ 先形成 $\sigma$ 键，再考虑 $\pi$ 相互作用  
- $\pi$ 效应可能增强或削弱原有 $\sigma$ 键的影响

# ● 记忆要点：

○ 强σ给体与强π受体都会增大Δ值   
◦ 实际化学体系中强 $\pi$ 受体更常见  
- 氟的特殊性：同时具有 $\sigma$ 给体和 $\pi$ 给体特性，后者起主导作用

# 4）应用注意事项

# ● 预测方法：

○ 判断配体 $\sigma$ 给电子能力强度  
- 分析π相互作用的性质（给电子/受电子）  
- 综合评估对分裂能的净影响

# ● 考试重点：

○ 掌握典型配体的分类（强/中/弱场）  
○ 理解 $\pi$ 效应对分裂能的调控机制  
○ 能预测未知配体对分裂能的影响趋势
