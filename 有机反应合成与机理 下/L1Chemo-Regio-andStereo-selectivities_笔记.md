# 2）保护基知识总结 02:16:45

# ● 保护基的类型

○ 硅类的保护基

# ■ 常见类型:

- TMS（三甲基硅基）： $Si(CH_{3})_{3}$   
TES（三乙基硅基）  
TBS（叔丁基二甲基硅基）  
● TIPS（三异丙基硅基）  
● TBDPS（叔丁基二苯基硅基）

位阻排序：TMS(2) < TES(3) < TBS(4) ≈ TBDPS(5) < TIPS(1)

# ■ 保护方法：

● 碱性条件下与氯硅烷反应  
● 常用碱：三乙胺（中和生成的HCl）

# ■ 脱保护方法：

● 使用氟离子源（如TBAF）  
● 原理：F - Si键能（\~465kJ/mol）强于O - Si键

○ 成醚的保护基 02:24:21

# ■ MOM保护基:

● 保护条件：碱性条件下与MOM chloride反应

● 脱保护：酸性水溶液（HCl/MeOH-H2O）

# ■ 芋基（Bn）保护基：

● 保护条件：碱性条件下与Bn chloride反应  
● 脱保护：氢解 $(H_{2}/\mathrm{Pd}-\mathrm{C}$ 催化）

# ■ 缩醛类保护基：

如THP保护基  
● 保护条件：干酸条件下反应  
● 脱保护：含水酸处理

# ● 保护基应用实例

\- 反应顺序原则：先保护活性基团（如羟基），再进行其他转化反应

# ○ 典型案例：

■ 溴代物转化为羟基的反应路径:

- 先用TIPS保护羟基  
- 再进行溴代物到羟基的转化

● 关键步骤：mCPBA氧化引发的重排反应（类似Dakin反应）
