# 1）反应机理与有机合成的思路差异 20:54

● 机理特点：反应机理题更注重理论自圆其说，考察对有机反应逻辑的理解，只要符合化学原理即可认为正确  
● 合成特点：有机合成具有结果导向性，即使相似反应条件也可能产生不同产物，需要事后分析原因  
● 本质区别：机理题是理论验证，合成题是实验现象解释，后者存在更多不确定性

# 2）有机合成结果的不确定性 21:19

● 实验现象：在特定条件下"获得"产物（而非必然生成），可能忽略副产物  
● 成功归因：即使操作规范，反应成败原因往往难以完全明确  
● 经验认知：有机合成中许多预设理论可能被实验证伪，需保持开放思维  
● 可靠结论：唯一确定的是"在X条件下分离得到Y产物"这一事实

# 20. 二碘化三山的应用 02:12:49

# 1）反应机理与过程

# 二、直接芳环金属化反应02:58:58

# 1. DMG定位基团

● 定义：DMG（Directing Metalation Group）指能引导金属试剂（如丁基锂）在芳环特定位置发生金属化的官能团

# ● 常见类型:

- 主要含氧基团（如甲氧基）  
○ 含氟基团（如三氟甲基）  
- 少量含氮基团

● 强度分类：分为强、中、弱三类，但具体强弱顺序不需记忆

# 2. 反应特点

● 竞争关系：当存在多个可能反应位点时，需强DMG才能有效引导反应

○ 示例：甲氧基（中等强度）vs. MOM 基团（双氧配位，强 DMG）

# ● 影响因素：

- 温度（如负78℃→室温）  
- 辅助配体（如TMEDA）  
- 反应时间（如24小时）

# 3. 区域选择性案例

● 案例1：甲氧基定位时，金属化发生在邻位  
● 案例2：含氟化合物中，氟与锂有强配位能力（氟化锂晶格能高）  
● 案例3：双氧配位的MOM基团比单氧配位的甲氧基定位效应更强

# 4. 分子内反应

# ● 特征产物：

○ 含羟基和羰基（IR: $3300 \, cm^{-1}$ 和 $1600 \, cm^{-1}$ ）  
- 羰基双键键能较弱（供电子效应强）

● 反应路径：芳基锂负离子进攻分子内羰基
