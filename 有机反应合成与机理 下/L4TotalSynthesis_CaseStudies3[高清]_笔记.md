# - Bn与Bz区别:

○ Bn: 苄基 $C_{6}H_{5}-CH_{2}-$   
○ Bz: 苯甲酰基 $C_{6}H_{5}-C(=O)$ -（多一个氧原子）

● Cp的特殊性: 在金属有机化学中通常表示环戊二烯负离子 $C_{5}H_{5}^{-}$ ，具有芳香性

# Ms与Mes区别:

○ Ms: 甲磺酰基 $CH_{3}-SO_{2}$ -（含氧的磺酰基）  
○ Mes: 2,4,6-三甲基苯基（不含氧的三甲基苯基）

● Piv: 新戊酰基 $(CH_{3})_{3}C - C(=O)$ - （与Boc类似但少一个氧）

SEM: 2-三甲基硅乙氧甲基 $Me_{3}SiCH_{2}CH_{2}OCH_{2}-$

\- Tf: 三氟甲磺酰基 $CF_{3}-SO_{2}-$

● Tol: 甲苯基 $p-CH_{3}-C_{6}H_{4}$ -（也可直接表示甲苯溶剂）

\- Tr: 三苯甲基 $(C_6H_5)_3C-$

● Tris: 2,4,6-三异丙基苯磺酰基

● Ts: 对甲苯磺酰基 $p-CH_{3}-C_{6}H_{4}-SO_{2}-$

# ● 缩写使用规范:

- 吡啶基应写作Py而非Pyp   
- Tol多数情况下指甲苯本身而非对甲基苯基  
- 某些缩写（如Tr、Tris等）不常见，考试中可能不会要求

○ 强调创新性和探索性

# 2. 工业合成反应 10:04

# 工业特点:

注重实用性和经济性  
○ 主要进行大规模反应(>100克)  
◦ 关注成本控制、环境友好和工艺优化  
- 反应步骤需考虑可放大性  
现代药物合成最小规模也需达到克级

# ● 与学术差异:

- 工业界更倾向使用已验证的可靠反应  
◦ 学术圈部分重要反应在工业中应用有限  
- 工业合成更注重原子经济性和绿色化学  
- 反应选择需考虑原料可获得性和成本
