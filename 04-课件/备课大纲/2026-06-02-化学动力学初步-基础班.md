## 二、教学目标

### 2.1 知识目标
1. 理解反应速率的定义和表示方法
2. 掌握速率方程的实验确定方法（初始速率法）
3. 理解反应级数的概念（0、1、2级）
4. 掌握Arrhenius方程的两点法计算活化能
5. 理解基元反应与复杂反应的区别
6. 了解催化剂对反应速率的影响本质

### 2.2 能力目标
1. 能根据初始速率实验数据确定速率方程
2. 能利用Arrhenius方程进行活化能计算
3. 能区分基元反应和非基元反应的速率方程写法
4. 能判断反应级数的基本方法（概念层面）

### 2.3 素养目标
1. 建立"动力学与热力学是不同维度"的认知
2. 理解实验在确定反应机理中的核心地位
3. 培养从数据归纳规律的能力

## 三、教学重难点

### 重点
- 速率方程的实验确定（初始速率法）
- 反应级数的概念理解
- Arrhenius方程的应用
- 基元反应与复杂反应的区别

### 难点
- 非基元反应速率方程与计量系数的关系

## 四、教学过程设计

### 4.1 课堂引入（10 min）

**情境导入：**
- 展示两个反应：
  - 铁生锈（慢，数年）
  - 火药爆炸（快，毫秒级）
  - 食物腐败（中等，数天）
- 提问：热力学告诉我们反应能否发生，但为什么有的反应快有的慢？

**学生已有认知诊断：**
- 学生已学化学平衡，知道K和ΔG
- 可能混淆：ΔG < 0 是否意味着反应一定快？
- 可能混淆：催化剂是否改变平衡常数？

**本节要解决的问题：**
1. 如何定量描述反应快慢？
2. 反应速率受哪些因素影响？
3. 如何从实验数据得到速率方程？
4. 温度如何影响反应速率？

**深度边界声明：**
- 只学概念和简单计算
- 积分方程推导放到第三轮
- 复杂反应机理放到第三轮

### 4.2 知识内容（45 min）

#### 4.2.1 反应速率与速率方程（15 min）

**反应速率定义：**
- 平均速率 vs 瞬时速率
- 用不同物质表示时的关系（除以计量系数）
- 单位：mol·L⁻¹·s⁻¹

**速率方程：**
- 一般形式：$v = k[A]^m[B]^n$
- 强调：速率方程必须由实验确定
- 基元反应：可直接由计量系数写速率方程
- 非基元反应：速率方程与计量系数无关

**初始速率法：**
- 实验设计思想：保持其他浓度不变，改变一个反应物浓度
- 通过对比实验确定各反应物的级数
- 例题演示（配合课堂例题1）

#### 4.2.2 反应级数初步（10 min）

**反应级数概念：**
- 总级数 = m + n
- 常见：0级、1级、2级
- 级数可以是分数、负数（基础班点到为止）

**积分法判断级数（概念层面）：**
- 思路：假设某级数，看浓度-时间数据是否符合该级数的积分形式
- 只讲思路，不讲具体推导
- 一级反应：ln[A] vs t 呈直线
- 二级反应：1/[A] vs t 呈直线

#### 4.2.3 Arrhenius方程与活化能（15 min）

**Arrhenius方程：**
- 指数形式：$k = A\exp\left(-\frac{E_a}{RT}\right)$
- 对数形式：$\ln k = \ln A - \frac{E_a}{RT}$
- 两点法：$\ln\frac{k_2}{k_1} = \frac{E_a}{R}\left(\frac{1}{T_1} - \frac{1}{T_2}\right)$

**活化能物理意义：**
- 活化分子的最低能量与反应物平均能量之差
- 能垒概念（配合示意图）
- Ea越大，反应越慢

**lnk vs 1/T 作图：**
- 直线关系
- 斜率 = -Ea/R
- 截距 = lnA
- 基础班只讲概念，不要求作图计算

#### 4.2.4 基元反应与复杂反应概念（5 min）

**基元反应：**
- 一步完成的反应
- 速率方程可直接由计量系数写出
- 分子数概念（单分子、双分子、三分子）

**复杂反应：**
- 多步完成的反应
- 速率方程由实验确定
- 反应机理的概念（点到为止）

**催化剂：**
- 改变反应路径，降低Ea
- 不改变ΔH和K
- 同等程度加快正逆反应

### 4.3 课堂例题（20 min）

**例题1：初始速率法确定速率方程（⭐⭐）**
- 参照专题页例题1
- 通过对比实验数据确定m、n和k
- 预计用时：8 min

**例题2：Arrhenius方程求活化能（⭐⭐）**
- 给定两个温度下的k值
- 用两点法计算Ea
- 预计用时：7 min

**例题3：反应级数概念判断（⭐⭐）**
- 根据速率方程判断总级数
- 区分基元反应和非基元反应
- 预计用时：5 min

### 4.4 即时演练（3 min）

**快速检测题：**
1. 判断：基元反应的速率方程可以直接由化学方程式写出。（对/错）
2. 某反应速率方程为 v = k[A]²[B]，总反应级数为____。
3. 催化剂能改变反应的____，但不能改变____和____。

### 4.5 本节结论总结（5 min）

1. 反应速率 = k[反应物]^级数，速率方程由实验确定
2. 基元反应速率方程可直接由计量系数写出，非基元反应不行
3. Arrhenius方程描述温度对速率常数的影响：$k = A\exp(-E_a/RT)$
4. 催化剂降低活化能，不改变ΔH和平衡常数K

### 4.6 易错点强调（2 min）

- 反应级数 ≠ 计量系数（非基元反应）
- 活化能 ≠ 反应热（ΔH）
- 催化剂不改变平衡位置

### 主板书
```
化学动力学初步

一、反应速率
   v = (1/ν_i)(d[i]/dt)
   单位：mol·L⁻¹·s⁻¹

二、速率方程（实验确定！）
   v = k[A]^m[B]^n
   基元反应：m,n = 计量系数
   非基元反应：m,n ≠ 计量系数

三、Arrhenius方程
   k = A·exp(-Ea/RT)
   ln(k₂/k₁) = (Ea/R)(1/T₁ - 1/T₂)

四、催化剂
   ↓Ea，不改变ΔH和K
```

## 七、教学反思预设

### 可能的问题
1. 学生可能混淆反应级数和计量系数
2. 学生可能不理解为什么非基元反应速率方程不能由方程式写出
3. Arrhenius方程的对数运算可能出错

### 应对策略
1. 反复强调：只有基元反应才能直接写
2. 用具体反例说明（如H₂ + I₂ → 2HI）
3. 板书展示对数运算步骤
