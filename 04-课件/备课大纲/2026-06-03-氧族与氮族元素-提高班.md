# 零、教学逻辑来源与深度边界

## 0.1 来源表

| 来源 | 核心借鉴点 | 在本节中的映射 |
|------|-----------|--------------|
| 质心元素化学第一轮（硫与氮） | 族内递变规律、含氧酸结构对比、氧化还原性质递变 | 课时1硫化合物结构-性质关联、课时2氮氧化物氧化还原序列 |
| 普化原理第15章 | 主族元素化学通论、含氧酸缩合规律、歧化反应热力学判据 | 课时1 S₂O₃²⁻歧化讲解、课时3磷含氧酸缩合规律 |
| 第二轮总体备课框架 | 3课时分配、深化应用层级定位、真题穿插策略 | 整体课时划分、37届Q5/38届Q1穿插位置 |

## 0.2 讲到为止（深度边界）

| 话题 | 讲到什么程度 | 不展开什么 |
|------|------------|-----------|
| 氮氧化物平衡 | 知道NO + O₂ → NO₂的平衡特征，能写方程式 | 不推导平衡常数随温度的变化、不计算平衡组成 |
| 磷含氧酸缩合 | 知道正→焦→偏的脱水缩合规律，能写通式 | 不引入群论分析P₄O₁₀的Td对称性 |
| 过二硫酸盐 | 知道S₂O₈²⁻是强氧化剂，能写与Mn²⁺的反应 | 不展开过氧键均裂机理、不讨论链式氧化动力学 |
| 肼的氧化还原 | 知道N₂H₄是强还原剂，能写与O₂/H₂O₂的反应 | 不画完整电位图、不讨论催化分解机理 |
| 硫循环工业流程 | 知道接触法制硫酸的三步反应 | 不展开催化剂机理、不计算转化率优化 |

## 0.3 禁止提前展开

- 禁止引入化学平衡常数计算解释NOₓ转化（学生尚未系统学习平衡深化）
- 禁止用分子轨道理论详细分析O₂/S₂的电子结构（超出本轮深度）
- 禁止讨论磷化合物的对称性群论分析（属于结构化学深化内容，第三轮处理）
- 禁止展开硫代硫酸盐的定量滴定分析（分析化学专题内容）

## 0.4 结论策略

- **硫化合物**：以"结构决定性质"为主线，SO₃²⁻三角锥 vs SO₄²⁻正四面体的结构差异决定其还原性和稳定性差异
- **氮化合物**：以"氧化态梯度"为主线，从NH₃(-3) → N₂(0) → NO(+2) → NO₂(+4) → HNO₃(+5)构建氧化还原认知框架
- **磷化合物**：以"缩合规律"为主线，正磷酸 → 焦磷酸 → 偏磷酸的脱水缩合体现含氧酸通性
- **综合训练**：通过37届Q5的NO/H₂S反应，强化氧化还原方程式配平与元素推断的综合能力

# 一、教学定位

## 1.1 课时安排

| 课时 | 时长 | 主题 | 内容范围 |
|------|------|------|----------|
| 第1课时 | 45 min | 硫的化合物 | H₂S、SO₂、SO₃、H₂SO₄、硫代硫酸盐、过二硫酸盐；SO₃²⁻/SO₄²⁻结构对比 |
| 第2课时 | 45 min | 氮的化合物 | NH₃、N₂H₄、HN₃、NOₓ、HNO₃；浓/稀HNO₃还原产物差异 |
| 第3课时 | 45 min | 磷的化合物 + 综合训练 | P₄、PH₃、P₄O₁₀、磷的含氧酸；37届Q5切片训练 |

## 1.2 教学目标

| 目标层级 | 具体描述 |
|----------|----------|
| 知识目标 | 掌握硫、氮、磷重要化合物的基本性质、特征反应和结构特点；能正确书写核心反应方程式 |
| 能力目标 | 能根据氧化还原规律推断反应产物（如浓/稀HNO₃与金属反应）；能运用结构-性质关联解释化合物性质差异 |
| 方法目标 | 建立"族内递变 + 氧化态梯度 + 结构对比"三维分析框架；培养从实验事实归纳规律的思维能力 |
| 应试目标 | 能独立完成37届Q5级别的氧化还原综合推断题；能在38届Q1级别情境化推断题中准确定位氧族/氮族元素 |

## 1.3 重难点

| 类型 | 内容 | 突破策略 |
|------|------|----------|
| **重点** | 硫代硫酸盐与酸反应的歧化方程式 | 实验事实切入：Na₂S₂O₃ + HCl → 浑浊（S）+ 刺激性气体（SO₂），再配平方程式 |
| **重点** | 浓/稀HNO₃还原产物差异 | 对比讲解：浓→NO₂（+4）、稀→NO（+2）、极稀→NH₄⁺（-3），用氧化态变化解释 |
| **重点** | 磷的含氧酸缩合规律 | 类比讲解：正磷酸H₃PO₄ → 脱水 → 焦磷酸H₄P₂O₇ → 继续脱水 → 偏磷酸HPO₃ |
| **难点** | N₂H₄的二元弱碱性质与还原剂双重身份 | 归谬讲解：若N₂H₄只是一元碱，则与HCl反应产物不符实验事实；同时展示与O₂反应的氧化还原方程式 |
| **难点** | 37届Q5中NO与H₂S反应的氧化还原配平 | 推导讲解：先确定氧化产物（S）和还原产物（N₂），再用电子守恒配平 |

# 二、知识脉络与授课路线

## 2.1 路线总览

```
课时1：硫的化合物（45min）
├── 引入：氧族元素通性回顾（5min）
│   └── 从O→S→Se→Te的递变：非金属性减弱，含氧酸酸性减弱，还原性增强
├── H₂S的性质（8min）
│   ├── 弱酸性（二元弱酸，分步电离）
│   ├── 强还原性：与O₂、Cl₂、SO₂反应
│   └── 特征反应：Cl₂ + H₂S → S↓ + 2HCl（卤素与氧族联系）
├── SO₂与SO₃（10min）
│   ├── SO₂：V型结构，既有氧化性又有还原性
│   ├── SO₃：平面三角形，酸性氧化物
│   └── 催化氧化：2SO₂ + O₂ ⇌ 2SO₃（点到为止，不展开平衡计算）
├── 硫酸与硫代硫酸盐（17min）
│   ├── 浓H₂SO₄的强氧化性、脱水性、吸水性
│   ├── SO₃²⁻（三角锥）vs SO₄²⁻（正四面体）结构对比
│   ├── S₂O₃²⁻的歧化反应：S₂O₃²⁻ + 2H⁺ → S↓ + SO₂↑ + H₂O
│   └── 过二硫酸盐S₂O₈²⁻的强氧化性（选讲，点到为止）
└── 课时小结：硫的氧化态图谱（5min）

课时2：氮的化合物（45min）
├── 引入：氮族元素通性回顾（5min）
│   └── 从N→P→As→Sb→Bi的递变：金属性增强，氢化物稳定性减弱
├── NH₃与N₂H₄（12min）
│   ├── NH₃：三角锥结构，弱碱性，配位能力
│   ├── N₂H₄（联氨/肼）：二元弱碱，强还原剂
│   └── N₂H₄与O₂/H₂O₂的反应（火箭燃料应用，数值冲击）
├── HN₃（叠氮酸）（5min）
│   └── 不稳定，易爆，弱酸性，与金属反应生成叠氮化物
├── 氮氧化物NOₓ（10min）
│   ├── NO：无色，遇O₂变红棕，不成盐氧化物
│   ├── NO₂：红棕色，二聚为N₂O₄，与水反应生成HNO₃ + NO
│   └── N₂O₃、N₂O₅：酸性氧化物（点到为止）
├── HNO₃的性质（10min）
│   ├── 浓HNO₃ + Cu → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O
│   ├── 稀HNO₃ + 3Cu → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O
│   ├── 极稀HNO₃可还原为NH₄⁺（选讲）
│   └── 钝化现象：Fe/Al遇浓HNO₃（冷）
└── 课时小结：氮的氧化态图谱（3min）

课时3：磷的化合物 + 综合训练（45min）
├── P₄与PH₃（8min）
│   ├── P₄：正四面体结构，白磷的剧毒与自燃性
│   ├── PH₃（膦）：三角锥结构，极毒，弱碱性（弱于NH₃）
│   └── 制备：P₄ + 3OH⁻ + 3H₂O → PH₃↑ + 3H₂PO₂⁻
├── P₄O₁₀（7min）
│   ├── 最强干燥剂之一，与水反应生成H₃PO₄
│   └── 结构特点：P=O双键与P-O-P桥键（点到为止，不展开群论）
├── 磷的含氧酸（15min）
│   ├── 正磷酸H₃PO₄：三元中强酸，无氧化性
│   ├── 焦磷酸H₄P₂O₇：两分子正磷酸脱水缩合
│   ├── 偏磷酸HPO₃：多分子脱水缩合，玻璃状固体
│   └── 缩合规律通式：nH₃PO₄ → H_{n+2}P_nO_{3n+1} + (n-1)H₂O
├── 综合训练：37届Q5切片（12min）
│   ├── 题目呈现：NO与H₂S反应体系
│   ├── 分析思路：氧化还原推断 → 确定产物 → 配平方程式
│   └── 学生板演 + 教师点评
└── 课堂总结与下节预告（3min）
```

# 六、课后追踪

## 6.1 必做

- [ ] 完成38届Q1（炼丹元素推断）独立作答，下节课前交
- [ ] 书写并记忆以下核心方程式（各3遍）：
  - S₂O₃²⁻ + 2H⁺ → S↓ + SO₂↑ + H₂O
  - Cu + 4HNO₃(浓) → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O
  - 3Cu + 8HNO₃(稀) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O
  - 2NH₄NO₃ → 2N₂↑ + O₂↑ + 4H₂O（高温）
  - P₄ + 5O₂ → P₄O₁₀
- [ ] 绘制硫和氮的氧化态-化合物对应图谱

## 6.2 选做

- [ ] 查阅资料：为什么浓HNO₃使Fe/Al钝化？（提示：致密氧化膜）
- [ ] 思考：N₂H₄作为火箭燃料的优势是什么？（从能量密度和产物清洁度分析）
- [ ] 尝试推导：焦磷酸H₄P₂O₇与正磷酸H₃PO₄的酸性强弱比较

## 6.3 暂不要求

- [ ] 氮氧化物平衡热力学的完整推导（等化学平衡专题后）
- [ ] 过二硫酸盐氧化机理的详细讨论（等动力学专题后）
- [ ] 磷化合物群论对称性分析（等结构化学深化后）
- [ ] 硫循环工业制硫酸的完整流程计算（等工业流程专题后）
