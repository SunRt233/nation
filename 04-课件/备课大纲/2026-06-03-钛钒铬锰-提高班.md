## 零、教学逻辑来源与深度边界

### 0.1 来源表

| 来源类型 | 具体来源 | 在本课中的映射 |
|---------|---------|--------------|
| 教材提炼 | 质心元素化学第一轮（钪钛钒） | TiO₂金红石结构、TiCl₄水解、钒多价态颜色序列 |
| 教材提炼 | 质心元素化学第一轮（铬） | Cr³⁺稳定性、Cr₂O₇²⁻/CrO₄²⁻互变、Cr(VI)毒性 |
| 总体框架 | 第二轮总体备课框架 | 课时分配、深度标记、例题选择、易错点预警 |
| 真题导向 | 37届Q3（LiMn₂O₄合成）、36届Q5（铁化合物ABCD推断） | Mn多价态应用、过渡金属颜色信号推断 |

### 0.2 讲到为止

- **钛**：掌握TiO₂（金红石型结构，6:3配位）、TiCl₄（易水解产生白色烟雾）、钛白粉制备（TiCl₄ + O₂ → TiO₂ + 2Cl₂），不展开金红石能带结构与光催化机理
- **钒**：掌握V₂O₅催化氧化SO₂、钒的多价态颜色序列（V²⁺紫→V³⁺绿→VO²⁺蓝→VO₂⁺黄），不展开钒氧化还原电对的完整电位图
- **铬**：掌握Cr³⁺（稳定，绿色，八面体配位）、Cr₂O₇²⁻（强氧化性，橙色，酸性条件稳定）、CrO₄²⁻（黄色，碱性条件稳定），Cr(VI)有毒，废液需FeSO₄还原处理；不展开Cr(III)配合物的配位场分裂能计算
- **锰**：掌握Mn²⁺（稳定，淡粉色）、MnO₄⁻（强氧化性，紫色）、MnO₂（催化，黑色）；KMnO₄在不同pH下的还原产物差异（酸性→Mn²⁺、中性→MnO₂、碱性→MnO₄²⁻）；不展开MnO₂催化反应的机理

### 0.3 禁止提前展开

- 禁止引入过渡金属配位场理论的完整MO分析
- 禁止展开多酸化学（同多酸/杂多酸）
- 禁止讨论金红石TiO₂的能带结构与光催化机理
- 禁止画出钒氧化还原电对的完整电位图（只记颜色序列）
- 禁止展开MnO₂催化H₂O₂分解或KClO₃分解的机理

### 0.4 结论策略

- **钛** → 用"Ti⁴⁺电荷高、半径小→极易水解→TiCl₄遇水冒烟"一句话收束
- **钒** → 用"价态越高，颜色越浅（紫→绿→蓝→黄）"口诀记忆
- **铬** → 用"酸性橙Cr₂O₇²⁻，碱性黄CrO₄²⁻，Cr³⁺绿色最稳定"收束
- **锰** → 用"酸性Mn²⁺、中性MnO₂、碱性MnO₄²⁻"三句话记忆KMnO₄还原产物

## 一、教学定位

### 1.1 课时安排

| 项目 | 内容 |
|-----|------|
| 总课时 | 2课时 × 45 min = 90 min |
| 课时1 | 钛 + 钒 + 铬（45 min） |
| 课时2 | 锰 + 综合训练（45 min） |
| 课型 | 新授课（第二轮深化） |

### 1.2 教学目标

| 目标层级 | 具体内容 |
|---------|---------|
| 知识目标 | 掌握TiO₂（金红石型）、TiCl₄（易水解）、钛白粉制备；掌握钒的多价态颜色序列和V₂O₅催化氧化；掌握Cr³⁺/Cr₂O₇²⁻/CrO₄²⁻的性质与互变；掌握Mn²⁺/MnO₄⁻/MnO₂的性质和KMnO₄在不同pH下的还原产物 |
| 能力目标 | 能根据颜色信号推断过渡金属离子（V/Cr/Mn）；能写出KMnO₄在酸/中/碱性条件下的还原反应方程式；能解释Cr(VI)废液处理原理；能判断TiCl₄水解产物 |
| 素养目标 | 建立"价态-颜色-性质"三位一体的过渡金属分析框架；体会工业催化（V₂O₅接触法、MnO₂催化）与元素性质的关联 |

### 1.3 重难点

| 类型 | 内容 | 突破策略 |
|-----|------|---------|
| 重点 | 钒的多价态颜色序列（V²⁺紫→V³⁺绿→VO²⁺蓝→VO₂⁺黄） | 口诀记忆+实例驱动 |
| 重点 | KMnO₄在不同pH下的还原产物差异 | 对比表+方程式强化 |
| 重点 | Cr₂O₇²⁻/CrO₄²⁻的酸碱互变与氧化性 | 实验事实切入（橙色↔黄色） |
| 难点 | TiCl₄遇水剧烈水解产生白色烟雾（TiO₂·nH₂O） | 联系SiCl₄水解类比 |
| 难点 | KMnO₄在碱性条件下还原为MnO₄²⁻（绿色），学生易与MnO₂混淆 | 强调"碱性→锰酸根绿色" |
| 难点 | 37届Q3中LiMn₂O₄合成方程式的Mn价态判断 | 先确定Mn平均价态，再拆分为Mn²⁺/Mn³⁺/Mn⁴⁺ |

## 二、知识脉络与授课路线

### 2.1 路线总览

```text
课时1：钛 + 钒 + 铬（45 min）
├─ 导入（3 min）："钛合金用于航空航天，但钛的化合物你熟悉吗？"
├─ 钛（10 min）
│  ├─ TiO₂：金红石型结构（6:3配位），白色颜料（钛白粉）
│  ├─ TiCl₄：易水解，TiCl₄ + 2H₂O → TiO₂ + 4HCl（白色烟雾）
│  └─ 钛白粉制备：TiCl₄ + O₂ → TiO₂ + 2Cl₂（气相氧化法）
├─ 钒（12 min）
│  ├─ 钒的多价态：+2/+3/+4/+5
│  ├─ 颜色序列：V²⁺(紫)→V³⁺(绿)→VO²⁺(蓝)→VO₂⁺(黄)
│  ├─ V₂O₅：橙红色，催化氧化SO₂（接触法制硫酸）
│  └─ 易错点：颜色必须记住，竞赛常考颜色信号推断
├─ 铬（17 min）
│  ├─ Cr³⁺：稳定，绿色，[Cr(H₂O)₆]³⁺八面体
│  ├─ Cr₂O₇²⁻：强氧化性，橙色，酸性条件稳定
│  ├─ CrO₄²⁻：黄色，碱性条件稳定
│  ├─ 互变：2CrO₄²⁻ + 2H⁺ ⇌ Cr₂O₇²⁻ + H₂O（橙↔黄）
│  ├─ Cr(VI)有毒，废液用FeSO₄还原处理
│  └─ 重要方程式：Cr₂O₇²⁻ + 6Fe²⁺ + 14H⁺ → 2Cr³⁺ + 6Fe³⁺ + 7H₂O
└─ 课时1小结（3 min）："Ti水解→V颜色→Cr价态互变"

课时2：锰 + 综合训练（45 min）
├─ 导入（3 min）："KMnO₄是实验室最常用的氧化剂，但你知道它在不同pH下产物不同吗？"
├─ 锰的化合物（17 min）
│  ├─ Mn²⁺：稳定，淡粉色，d⁵半满构型
│  ├─ MnO₄⁻：强氧化性，紫色
│  ├─ MnO₂：黑色，催化（H₂O₂分解、KClO₃分解）
│  └─ KMnO₄在不同pH下的还原产物：
│     ├─ 酸性：MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
│     ├─ 中性：MnO₄⁻ + 2H₂O + 3e⁻ → MnO₂↓ + 4OH⁻
│     └─ 碱性：MnO₄⁻ + e⁻ → MnO₄²⁻（绿色）
├─ 易错点回顾（5 min）
│  ├─ KMnO₄酸性→Mn²⁺，中性→MnO₂，碱性→MnO₄²⁻
│  ├─ Cr(VI)有毒，废液需FeSO₄还原
│  ├─ TiCl₄遇水剧烈水解
│  └─ 钒颜色序列口诀
├─ 真题切片（15 min）
│  ├─ 37届Q3：LiMn₂O₄合成方程式（Mn多价态应用）
│  └─ 36届Q5：铁化合物ABCD推断（过渡金属颜色信号）
└─ 课时2小结（5 min）："Mn价态最多变，颜色信号要记牢"
```

## 六、课后追踪

### 6.1 必做

- 整理KMnO₄在酸性/中性/碱性条件下的还原产物对比表（含方程式）
- 默写钒的多价态颜色序列（V²⁺→V³⁺→VO²⁺→VO₂⁺）
- 写出TiCl₄水解、V₂O₅催化氧化SO₂、Cr₂O₇²⁻氧化Fe²⁺的三个核心方程式
- 解释为什么Cr(VI)废液需要用FeSO₄还原处理（写出离子方程式）

### 6.2 选做

- 查阅资料：钛白粉的其他制备方法（硫酸法vs气相氧化法）
- 思考：为什么Mn²⁺特别稳定？（提示：d⁵半满构型）
- 分析LiMn₂O₄中Mn的价态分布（提示：平均+3.5，实际为Mn²⁺和Mn⁴⁺混合）

### 6.3 暂不要求

- 过渡金属配位场理论的完整MO分析
- 多酸化学（同多酸/杂多酸）
- 金红石TiO₂的能带结构与光催化机理
- 钒氧化还原电对的完整电位图
- MnO₂催化反应的机理

## 七、下节衔接

- 下节内容：过渡金属（二）——铁、钴、镍、铜、锌（第二轮）
- 衔接点：
  - 本节Cr₂O₇²⁻氧化Fe²⁺ → 下节Fe²⁺/Fe³⁺的氧化还原性质
