# 专题产出总览看板

> 用于跨专题查看当前课件体系的真实进度。
>
> 本页采用“**半自动 Dataview**”模式：
> - 人工维护：当前重点专题、当前状态、备注
> - 自动聚合：同专题的备课大纲 / 新授课 / 加工页 / 试点产出证据
>
> 当前使用口径（2026-06-19）：
> - 本页查看重点仍以 `备课大纲 + 学生讲义` 为主
> - `PPT / 教具 / 工具卡 / 评分表` 仅作为历史样板信息保留，不再作为当前推进状态判断依据
>
> 使用规则：
> - 新专题实跑或老专题有重大推进后，同步更新本页顶部总表。
> - 若某专题只有文件落盘、但未完成加工页或正式产物，不要写成“已完成”。
> - 若上游资料尚未补齐“备课抽料资产标记”，备注中应写明“资料已入库，抽料接入待补”。

## 查看顺序建议

1. 先看本页顶部总表判断专题当前在哪一阶段。
2. 再看对应专题的 Dataview 自动证据是否齐全。
3. 若自动证据齐全但状态判断仍有疑问，再跳转对应加工页或备课大纲真相源。
4. 若要核对近期系统变化，再看 `00-首页/工作日志/2026-06-17` 与 `00-首页/工作日志/2026-06-18`。
