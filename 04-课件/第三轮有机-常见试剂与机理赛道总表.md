# 第三轮有机-常见试剂与机理赛道总表

> 用途：看到试剂先快速切赛道，避免一上来机理就走错。

---

## 一、条件信号 → 机理赛道

| 条件/试剂 | 第一赛道 | 高频题眼 |
|:---|:---|:---|
| `HX`（普通） | 亲电加成 | 马氏规则、碳正离子 |
| `Br2` / `Cl2` | 卤鎓离子或自由基卤代 | anti 加成 / 卤代选择性 |
| `BH3/H2O2` | 协同加成 | 反马氏 + syn |
| `ROOR` + `HBr` | 自由基链反应 | 反马氏自由基加成 |
| `RMgX` / `RLi` | 强亲核/强碱构碳 | `1,2` 加成、羧化、兼容性 |
| Gilman | 软亲核 | `1,4` 加成、温和构碳 |
| `NaBH4` / `LiAlH4` | 负氢加成/还原 | 羰基还原 |
| `LDA` / `NaH` / `BuLi` | 去质子化 → 碳负离子 | 烯醇负离子、烷基化、缩合 |
| `hv` / `AIBN` | 自由基引发 | 链式机理 |
| 强酸 + 好离去基 | 碳正离子赛道 | `SN1 / E1 / 重排` |
| 强碱 + 二级/三级底物 | 协同消除 | `E2` |
| 芳卤 + 邻/对位 EWG + `Nu-` | SNAr | Meisenheimer 复合物 |
| 芳卤 + 强碱 / 液氨 | 苯炔 | 消除-加成 |
| 肟 | Beckmann | 反式迁移，酰胺/内酰胺 |
| 酮 + 过酸 | B-V | 插氧，酯/内酯 |
| 邻二醇 + 酸 | Pinacol | `1,2` 迁移、扩环 |
| 烯丙基芳基醚 | Claisen | 协同 `[3,3]` |
| `1,5-二烯` | Cope | 协同 `[3,3]` |

---

## 二、最常用的“看到就警觉”

| 看到什么 | 立刻警觉什么 |
|:---|:---|
| `ROOR` | 规则可能改写，不是普通离子路径 |
| `RMgX / RLi` | 活泼氢与保护基冲突 |
| 二级底物 | SN/E 竞争 |
| 环己烷 E2 | 必做 `trans-diaxial` |
| 羰基旁有离去基 | 不要只按加成处理 |
| 肟 / 过酸 / 邻二醇 | 重排模板题 |
| `hv / AIBN` | 不是普通离子箭头 |
| 强碱芳卤 | 可能走苯炔，不一定是 SNAr |

---

## 三、试剂不是反应名，而是“路径提示”

| 试剂 | 不要只背成什么 | 更好的口径 |
|:---|:---|:---|
| `PPh3` | 一个名字 | 亲核/配体/Wittig 来源提示 |
| `NBS` | 烯丙位溴代反应式 | 烯丙/苄位自由基路径 |
| `LDA` | 强碱三个字 | 选择性去质子化工具 |
| `NaNH2 / 液氨` | 一种碱条件 | 可能进入苯炔赛道 |
| `mCPBA` | 一个氧化剂 | 看到酮时要防 B-V |
| `SOCl2` / `PBr3` | 一个转化反应 | 看离去基活化与后续 SN/E |
