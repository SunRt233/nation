## 二、最常见的“一号按钮”

| 模块 | 一号按钮 |
|:---|:---|
| 电子效应 | 主导稳定化是诱导、共轭还是超共轭 |
| 活性中间体 | 条件最先导向哪类中间体 |
| SN/E | 底物几级，试剂更亲核还是更碱 |
| 加成 | 先分反应中心，再谈规则 |
| 芳香 | 先分机理，再判位点 |
| 羰基/缩合 | 谁有羰基亲电中心，谁能做供体 |
| 重排 | 模板是什么，谁迁移 |
| 周环 | 类型是什么，几电子，热还是光 |
| 自由基 | 链增长关键步是什么 |
| 合成 | 最值得切哪根键 |
| 特殊体系 | 这到底是哪一类骨架 |

---

## 三、最常见的“不要急着做”

| 看到什么 | 不要急着做什么 | 先做什么 |
|:---|:---|:---|
| 芳香题 | 直接背定位表 | 先分机理 |
| 羰基题 | 一上来背人名反应 | 先分加成/取代/缩合/分流 |
| 重排题 | 一上来凭名字猜产物 | 先认模板和迁移规则 |
| 自由基题 | 只写引发步 | 先写链增长关键步 |
| SN/E 题 | 看到二级底物就秒答 | 并列看底物/试剂/溶剂/温度 |
| 周环题 | 先找中间体 | 先分类型和电子数 |
| 糖类题 | 先背名称 | 先看异头碳是否自由 |
| 高分子题 | 被长链吓住 | 先找重复单元断点 |

---

## 四、十句课中口令

1. 先分题型，再分路径。  
2. 先找中间体，再画箭头。  
3. 先看共轭碱，再谈酸性。  
4. 先看底物，再看试剂。  
5. 先分机理，再判位点。  
6. 先看供体/受体，再说缩合。  
7. 先认模板，再问谁迁移。  
8. 先数电子，再判热光。  
9. 先写链增长，再说自由基。  
10. 先断键，再谈路线。
