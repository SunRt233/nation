# 活性中间体识别卡

> 用途：机理题第一步，先由条件反推最可能的中间体。

## 一、条件 → 中间体

| 条件信号 | 第一联想 |
|:---|:---|
| 强酸 + 好离去基 | 碳正离子 |
| 强碱 / 金属化 | 碳负离子 |
| `hv / ROOR / AIBN` | 自由基 |
| 重氮 / Simmons-Smith | 卡宾 / 类卡宾 |
| 强碱芳卤 | 苯炔 |

## 二、中间体 → 常见命运

| 中间体 | 高频命运 |
|:---|:---|
| 碳正离子 | 重排 / 捕获 / E1 |
| 碳负离子 | 加成 / 烷基化 / 缩合 / 消除 |
| 自由基 | 链增长 / 终止 / 抽氢 |
| 卡宾 | 环丙烷化 / 插入 |
| 苯炔 | 消除-加成 / 异构位点进攻 |

## 三、最常见误区

- 一上来只画箭头，不先认赛道。  
- 看到能重排的条件却不防碳正离子。  
- 把苯炔和 SNAr 混成一类。  

## 四、十秒口令

```text
先看条件，
再认中间体，
再看命运，
最后才画箭头。
```
