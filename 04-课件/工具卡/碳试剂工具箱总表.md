# 碳试剂工具箱总表

> 用途：合成题里快速判断该用哪类“带碳进去”的试剂。

## 一、先问三件事

1. 我要做的是 `1,2` 还是 `1,4`？  
2. 底物旁边有没有 `OH / NH / COOH` 这类活泼氢？  
3. 我要的是强进攻，还是温和选择性进攻？  

## 二、主工具表

| 工具 | 最常用途 | 第一步判断 | 高频警报 |
|:---|:---|:---|:---|
| `RMgX` | 羰基 `1,2` 加成、制醇、羧化 | 普通强亲核构碳 | 怕活泼氢 |
| `RLi` | 更强碱/更强亲核、卤锂交换 | 需要更强金属化能力 | 更容易副反应 |
| Gilman | 共轭 `1,4` 加成、温和构碳 | 想保留羰基时优先想 | 不要按 Grignard 去猜 |
| 炔负离子 | 增碳、生成炔基醇或炔烃骨架 | 底物是否允许 SN2 或加成 | 也怕活泼氢 |
| 烯醇负离子/等价物 | 缩合构碳 | 是否有 α-H/活泼亚甲基 | 容易混成普通亲核加成 |
| 叶立德 | 烯烃构建 | 目标是不是适合断成羰基 + 叶立德 | 别只记成“Wittig 名字” |

## 三、最短对照

| 场景 | 更常用谁 |
|:---|:---|
| 普通醛、酮制醇 | `RMgX` |
| 需要更强金属化 | `RLi` |
| α,β-不饱和羰基保留羰基 | Gilman |
| 目标是烯烃 | 叶立德 |
| 目标是 1,3-二羰基延伸 | 烯醇负离子/等价物 |

## 四、最常见误区

- 把 `RLi` 当成“更强的 Grignard”就结束。  
- 看到烯酮还在优先写 `RMgX` 的 `1,2` 产物。  
- 不查活泼氢兼容性。  

## 五、十秒口令

```text
先判 1,2 还是 1,4，
再查活泼氢，
最后选强进攻还是温和进攻。
```
