# 重排模板一页表

> 用途：看到重排题时，先认模板、再问谁迁移、最后核产物。

---

## 一、先认模板

| 模板 | 优先联想 | 高频产物 |
|:---|:---|:---|
| 邻二醇 + 酸 | Pinacol | 酮 / 扩环酮 |
| 肟 | Beckmann | 酰胺 / 内酰胺 |
| 酮 + 过酸 | Baeyer-Villiger | 酯 / 内酯 |
| 烯丙基芳基醚 | Claisen | 邻烯丙基酚骨架 |
| `1,5-二烯` | Cope | 双键重排产物 |
| α-硅基醇 + 碱 | Brook | Si-O 转移 / 极性翻转 |
| α-卤代酮 + 碱 | Favorskii | 缩环羧酸衍生物 |

---

## 二、再问“谁迁移”

| 反应 | 迁移规则 |
|:---|:---|
| Pinacol | 看离去后形成的碳正离子与 `1,2` 迁移收益 |
| Beckmann | 与 `OH` 反式的基团迁移 |
| B-V | 迁移能力更强的基团迁移 |
| Claisen / Cope | 协同 `[3,3]`，不是自由选边 |
| Brook | Si 从 C 迁到 O |
| Favorskii | 缩环/骨架重组，不是普通取代 |

---

## 三、最后核产物

| 反应 | 最后一定核什么 |
|:---|:---|
| Pinacol | 羰基位置、是否扩环 |
| Beckmann | 酰胺中 N 连的是谁 |
| B-V | 氧插到了哪边 |
| Claisen / Cope | 双键和骨架连接方式 |
| Brook | Si-O 是否形成 |
| Favorskii | 是否缩环、是否成羧酸衍生物 |

---

## 四、最容易混的两组

### Beckmann vs B-V

| 比较点 | Beckmann | B-V |
|:---|:---|:---|
| 起始模板 | 肟 | 酮 + 过酸 |
| 判据 | 反式迁移 | 迁移能力 |
| 产物 | 酰胺/内酰胺 | 酯/内酯 |

### Claisen / Cope vs 离子型重排

| 比较点 | Claisen / Cope | 离子型重排 |
|:---|:---|:---|
| 是否协同 | 是 | 不一定 |
| 是否强调六元过渡态 | 是 | 不一定 |
| 是否先找中间体 | 通常否 | 往往要 |

---

## 五、最常见误区

- 一看到“重排”就直接猜产物。  
- Beckmann 忘看反式构型。  
- B-V 忘记是“插氧”不是普通氧化。  
- Claisen / Cope 误写成离子机理。  
- Favorskii 误判成普通取代。  

---

## 六、十秒口令

```text
先认模板，
再问谁迁移，
最后核产物官能团和环尺寸。
```
