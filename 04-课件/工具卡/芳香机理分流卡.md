# 芳香机理分流卡

> 用途：芳香题先分 SEAr / SNAr / 苯炔 / 杂环位点，不再一上来背定位。

## 一、先分四路

| 题干信号 | 第一机理入口 |
|:---|:---|
| 硝化/卤化/F-C | SEAr |
| 芳卤 + `Nu-` + 邻/对位 EWG | SNAr |
| 芳卤 + 强碱 / 液氨 | 苯炔 |
| 杂环体系 | 杂环位点题 |

## 二、最短抓手

| 路线 | 中间体/关键体 | 常问什么 |
|:---|:---|:---|
| SEAr | σ 络合物 | 邻对位/间位 |
| SNAr | Meisenheimer 复合物 | 负电荷能否稳定 |
| 苯炔 | 苯炔 | 是否混合物 |
| 杂环 | 芳香性损失最小位点 | `α / β / 3` 位等 |

## 三、最常见误区

- 芳香题一律套定位表。  
- 把 SNAr 和苯炔混成一类。  
- 杂环位点直接照搬苯环。  

## 四、十秒口令

```text
先分机理，
再看中间体，
最后判位点和产物数目。
```
