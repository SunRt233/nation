# 周环四问法工具卡

> 用途：周环题快速分型并落到允许性、区域和立体结果。

## 一、四问法

1. 是电环化、环加成，还是 σ 迁移？  
2. 是 `4n` 还是 `4n+2`？  
3. 是热条件还是光条件？  
4. 最后考区域、立体，还是两者一起？  

## 二、三类母型

| 看到什么 | 常见类型 |
|:---|:---|
| 共轭多烯开/关环 | 电环化 |
| 两个 π 体系拼环 | 环加成 |
| σ 键在共轭体系里搬家 | Claisen / Cope 型迁移 |

## 三、最常见落点

| 模块 | 高频落点 |
|:---|:---|
| 电环化 | 对旋 / 旋光，立体结果 |
| D-A | 区域选择、endo/exo、构型传递 |
| `[2+2]` | 热禁阻 / 光允许 |
| Claisen / Cope | 协同 `[3,3]`，六元过渡态 |

## 四、最常见误区

- 一上来去找中间体。  
- 只背热/光口诀，不画端基旋转。  
- 把 Claisen/Cope 误写成离子重排。  

## 五、十秒口令

```text
先分类型，
再数电子，
再判热光，
最后落到区域和立体。
```
