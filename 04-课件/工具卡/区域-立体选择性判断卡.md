# 区域-立体选择性判断卡

> 用途：选择性题先拆成“位置”和“面”两类问题。

## 一、先拆问题

| 题目在问什么 | 类型 |
|:---|:---|
| 加到哪一位 | 区域选择性 |
| 从哪一面进攻 | 立体选择性 |
| 哪个异构体更多 | 常常两者都有 |

## 二、区域选择性常见入口

| 场景 | 第一判断 |
|:---|:---|
| 烯烃加成 | 马氏 / 反马氏 |
| 共轭体系 | `1,2 / 1,4` |
| 芳香反应 | 邻对位 / 间位 / 杂环位点 |

## 三、立体选择性常见入口

| 场景 | 第一判断 |
|:---|:---|
| SN2 | 背面进攻 → 翻转 |
| E2 | `anti-periplanar` |
| Br2 加成 | anti |
| 硼氢化 | syn |
| 电环化 | 对旋 / 旋光 |

## 四、最常见误区

- 区域和立体混成一句话。  
- 只答“反马氏”却漏掉 `syn`。  
- 只答“anti”却没说位置分流。  

## 五、十秒口令

```text
先问位置还是面，
再看中间体或过渡态，
最后补位阻、电子效应和热力学修正。
```
