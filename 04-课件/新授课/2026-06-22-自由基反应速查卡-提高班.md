# 自由基反应 · 速查卡

> 基础班你背过 HBr/过氧化物反马氏加成是个"例外"——本卡把自由基从**零散例外题合集**升级到**链式单电子机理**的统一判断系统。

---

## 一、自由基反应 · 四问判断法

```
拿到自由基题 → 四问导航
    │
    ├─ 问 1：有没有自由基引发信号？
    │   ├─ hv（光照）、Δ（加热）、过氧化物（ROOR, BPO, AIBN）
    │   ├─ NBS（N-溴代丁二酰亚胺）→ Br· 的自由基来源
    │   └─ Bu₃SnH + AIBN → Sn· 自由基链引发
    │
    ├─ 问 2：链增长关键步是什么？
    │   ├─ 自由基取代：X· + R–H → H–X + R· → R· + X₂ → R–X + X·
    │   ├─ 自由基加成：Br· + C=C → C–C· → C–C· + H–Br → C–C–H + Br·
    │   └─ 自由基环化：C=C–(CH₂)ₙ–C· → 环化 → 新环 + 新自由基
    │
    ├─ 问 3：哪个自由基或过渡态更有利？
    │   ├─ 自由基稳定性：3° > 2° > 1° > CH₃·（超共轭越多越稳定）
    │   ├─ 烯丙基/苄基自由基：共振稳定→特别稳定→优先在该位点反应
    │   └─ Baldwin 规则：5-exo-trig 通常有利
    │
    └─ 问 4：最终考的是位点、区域还是环化方式？
        ├─ 卤代题 → 哪个 H 被取代？（位点选择性）
        ├─ 加成题 → Br 加在哪端？（区域选择性）
        └─ 环化题 → 5-exo vs 6-endo？（环化方式）
```

## 二、链式反应三阶段

```
自由基链反应 = 统一的机理框架，贯穿所有自由基题型

┌─────────────────────────────────────────────────┐
│  阶段 1：链引发（Initiation）                      │
│    X₂ + hv/Δ → 2 X·                              │
│    ROOR + Δ → 2 RO·                              │
│    AIBN + Δ → 2 (CH₃)₂C(CN)· + N₂               │
│    └─ 关键：产生第一个自由基（需要能量输入）        │
│─────────────────────────────────────────────────│
│  阶段 2：链增长（Propagation）                     │
│    X· + R–H → H–X + R·  （夺氢步，产生碳自由基）   │
│    R· + X₂ → R–X + X·   （卤代步，再生卤自由基）    │
│    └─ 关键：每一步消耗一个自由基又产生一个新自由基    │
│    └─ 循环进行 → 少量自由基可转化大量底物            │
│─────────────────────────────────────────────────│
│  阶段 3：链终止（Termination）                     │
│    2 X· → X₂                                     │
│    2 R· → R–R（偶联产物，常作副产物识别信号）       │
│    R· + X· → R–X                                 │
│    └─ 关键：两个自由基结合 → 链断裂 → 反应停止      │
└─────────────────────────────────────────────────┘
```

**推断题信号**：看到 hv / Δ + 过氧化物 / AIBN / NBS → 先写引发步（均裂），再写增长步（夺氢+卤代），最后写终止步（自由基偶联）。

## 三、自由基稳定性与选择性

### 3.1 碳自由基稳定性顺序

```
自由基稳定性（由超共轭 + 共振决定）：

  PhCH₂· ≈ CH₂=CH–CH₂·  >  R₃C·  >  R₂CH·  >  RCH₂·  >  CH₃·
  苄基 ≈ 烯丙基            3°       2°        1°        甲基

  解释：
    ├─ 3° > 2° > 1°：更多的 C–H σ 键与自由基单电子轨道超共轭
    ├─ 烯丙基/苄基：单电子离域到 π 体系 → 共振稳定
    └─ 稳定性越高 → 越容易生成 → 决定了反应位点
```

### 3.2 Cl₂ vs Br₂ 卤代选择性

| 维度 | Cl₂ / hv | Br₂ / hv |
|:---|:---|:---|
| **反应活性** | 高（Cl· 极活泼） | 低（Br· 较温和） |
| **选择性** | 差（各位置产物比例接近统计值） | 好（强烈倾向 3° > 2° > 1°） |
| **过渡态** | 早过渡态（反应物-like）→ 选择性差 | 晚过渡态（产物-like）→ 选择性好 |
| **Hammond 假说** | 放热反应→早 TS→ TS 对自由基稳定性不敏感 | 吸热反应→晚 TS→ TS 充分反映产物自由基稳定性 |
| **典型比例（丙烷）** | 1°:2° ≈ 45:55（接近统计 3:1 的 45:55） | 1°:2° ≈ 3:97（极强偏向 2° 位点） |

**记忆口诀**："溴慢挑食，氯快乱打"——Br· 活性低选择性高，Cl· 活性高选择性差。

## 四、过氧化物效应 · HBr/ROOR 反马氏加成

```
正常 HBr 加成（离子机理）：H⁺ 先加到烯烃→C⁺→Br⁻ 捕获→马氏产物
HBr/ROOR 加成（自由基机理）：Br· 先加到烯烃→C·→HBr 夺氢→反马氏产物！

机理详解：
  引发：  ROOR + Δ → 2 RO·  →  RO· + HBr → ROH + Br·
  增长 1：Br· + CH₂=CH–R → Br–CH₂–CH·–R（Br 加到端位→生成更稳的 2° 自由基）
  增长 2：Br–CH₂–CH·–R + HBr → Br–CH₂–CH₂–R + Br·
  终止：  2 Br· → Br₂  /  2 R· → R–R

为什么是反马氏？
  └─ 马氏规则基于 C⁺ 稳定性（离子机理）
  └─ 自由基加成中，Br· 优先加到烯烃端位→生成更稳定的碳自由基
  └─ 通常 2° R· > 1° R· → Br 加到端位（1° Br 产物）→ 表现反马氏！
```

**推断题信号**：HBr + ROOR（过氧化物）+ 烯烃 → 产物中 Br 在端位（反马氏）→ 不是"例外"，是自由基链机理的自然结果。

## 五、NBS · 烯丙位与苄位自由基溴代

```
NBS = N-溴代丁二酰亚胺（N-Bromosuccinimide）

NBS 的关键结构特征：
  └─ N–Br 键非常弱 → 微量 Br₂（NBS 中总含痕量 Br₂）→ 光照均裂→ Br·

NBS 反应条件：NBS + CCl₄（溶剂）+ hv 或 AIBN（引发剂）

NBS 溴代机理：
  引发： Br₂ + hv → 2 Br·（或 AIBN 热引发）
  增长 1：Br· + R–CH₂–CH=CH₂ → HBr + R–CH·–CH=CH₂（夺烯丙位 H！）
  增长 2：R–CH·–CH=CH₂ + Br₂ → R–CHBr–CH=CH₂ + Br·
  注意： 生成的 HBr + NBS → Br₂ + 丁二酰亚胺（补 Br₂，维持链）

为什么只取代烯丙位？
  └─ 烯丙位 C–H 键的 BDE 显著低于普通烷烃 C–H 键
  └─ 烯丙位 H 被夺后→烯丙基自由基（共振稳定）→极有利于生成
  └─ 同理适用于苄位（Ph–CH₂–R → Ph–CH·–R，苄基自由基共振稳定）
```

**推断题信号**：NBS + hv/AIBN + 含烯丙基/苄基的底物 → Br 取代在烯丙位或苄位，不动双键本身！

## 六、自由基环化 · Baldwin 规则

```
自由基环化 = 分子内的自由基加成到 C=C 或 C≡C

Baldwin 规则命名：N-exo/endo-trig/dig

  5-exo-trig：
    ├─ 5 = 环化后形成五元环
    ├─ exo = 断裂的键在环外
    ├─ trig = 受体是 C=C（trigonal，sp² 碳）
    └─ 最有利的模式！→ 动力学优势

  6-endo-trig：
    ├─ 6 = 六元环
    ├─ endo = 断裂的键在环内
    └─ 通常不如 5-exo 有利，但在特定底物中可竞争

  5-exo-dig：
    └─ dig = 受体是 C≡C（digonal，sp 碳）→ 也可发生

环化选择性规律：
  ├─ 5-exo > 6-exo > 6-endo > 5-endo（通常情况下）
  ├─ 五元环形成速率最快（过渡态环张力适中 + 轨道路径最短）
  └─ 产物中优先形成五元环（或六元环）
```

**推断题信号**：底物中有远端 C=C + 自由基引发条件（AIBN + Bu₃SnH / n-Bu₃SnH）→ 碳自由基生成后分子内环化到 C=C → 给出环状产物。

## 七、BDE 键能表 · 选择性判断工具

```
BDE（键解离能）= 均裂一根键需要的能量

BDE 越小 → 键越容易均裂 → 自由基越容易在该位点生成

常见键的 BDE (kJ/mol) 相对顺序：
┌──────────────────────────────────────────┐
│  N–Br (NBS中)              ≈ 弱         │ ← 最易均裂
│  O–O (ROOR中)              ≈ 150        │
│  C–I                       ≈ 210        │
│  C–Br                      ≈ 280        │
│  烯丙位 C–H                ≈ 370        │
│  苄位 C–H                  ≈ 370        │
│  3° C–H                    ≈ 390        │
│  2° C–H                    ≈ 410        │
│  1° C–H                    ≈ 420        │
│  CH₃–H                     ≈ 440        │
│  C=C π 键                  ≈ 270 (π)    │ ← π 键比 σ 键弱
└──────────────────────────────────────────┘

选择性判断三步：
  1. 找出所有可能的反应位点
  2. 比较各位点的 BDE（BDE 越小的位点→越容易被夺氢/均裂）
  3. 如果有共振稳定因素（烯丙基/苄基）→优先在该位点
```

## 八、常见自由基条件与试剂速查

| 条件/试剂 | 产生什么自由基 | 用途 | 推断题标志 |
|:---|:---|:---|:---|
| **Cl₂ + hv** | Cl· | 烷烃氯代（选择性差） | 光照+Cl₂ |
| **Br₂ + hv** | Br· | 烷烃溴代（选择性好） | 光照+Br₂ |
| **NBS + hv/AIBN** | Br·（慢速稳定释放） | 烯丙位/苄位溴代 | NBS + CCl₄ + 光照 |
| **HBr + ROOR** | Br·（过氧化物引发） | 反马氏加成 | HBr + 过氧化物 |
| **AIBN + Δ** | (CH₃)₂C(CN)· | 自由基链引发剂 | AIBN + 加热 |
| **Bu₃SnH + AIBN** | Bu₃Sn· | 自由基还原/环化引发 | Bu₃SnH + AIBN |
| **BPO（过氧化苯甲酰）** | PhCOO· | 通用自由基引发剂 | BPO + Δ |

## 九、常见易错点

| 错误 | 纠正 |
|:---|:---|
| 自由基机理只用"鱼钩箭头"标一步 | 必须写完整链式三阶段（引发→增长→终止），多步传链是自由基机理的核心特征 |
| 反马氏加成是"例外" | 反马氏不是例外——自由基加成中 Br· 先加到端位（生成更稳的 C·），自然给出反马氏产物。是两种机理（离子 vs 自由基）的正常结果 |
| NBS 是给双键加 Br₂ | NBS 不做双键加成！NBS 提供低浓度 Br·→夺烯丙位 H→烯丙基溴代。双键+Br₂ 加成是离子型反应，根本不同赛道 |
| Cl₂ 和 Br₂ 选择性差不多 | Cl₂ 选择性极差（接近随机统计），Br₂ 选择性很强（几乎只在 3° 位反应）。Hammond 假说解释：Cl 放热→早 TS→不挑位点；Br 吸热→晚 TS→充分区分自由基稳定性 |
| 自由基反应和离子反应是互斥的 | 同一体系中可能同时有离子和自由基路径！关键是条件：有 ROOR/hv/AIBN → 自由基；有强酸/极性溶剂 → 离子 |
| 烯丙位自由基只有一种产物 | 不对称烯丙基底物→夺氢后产生两种共振式→Br· 可以进攻任一端→可能产生两种烯丙基溴代产物（常考！） |

## 十、自查清单

- [ ] 四问法能默推（引发信号→链增长→自由基稳定性→位点/区域/环化）
- [ ] 链式反应三阶段（引发/增长/终止）能独立写出完整步骤
- [ ] 自由基稳定性顺序：苄基 ≈ 烯丙基 > 3° > 2° > 1° > CH₃·
- [ ] Cl₂（低选择性）vs Br₂（高选择性）差异的 Hammond 解释
- [ ] HBr/ROOR 反马氏加成的完整链机理（Br· 先加成 → 更稳 C· → 反马氏产物）
- [ ] NBS 的特殊性：微量 Br₂→Br·→夺烯丙位 H→烯丙基溴代（不是双键加成！）
- [ ] Baldwin 规则：5-exo-trig 通常最有利
- [ ] BDE 表的核心相对顺序（N–Br < O–O < 烯丙位 C–H < 3° C–H < 2° C–H < 1° C–H）
- [ ] 六种常见自由基条件/试剂的引发信号（Cl₂+hv / Br₂+hv / NBS / HBr+ROOR / AIBN / Bu₃SnH）
- [ ] NBS 不对称底物→两种烯丙位产物的共振判断
