# 配合物与配位化学深化 · 速查卡

> Round 3 你学会了"画 d 轨道分裂"——本卡把配位化学从**画图层**升级到**轨道级别标注层**：给磁性/光谱/结构数据，独立推断电子构型+成键图像+反应性。

## 一、配位化学判断总流程

```
拿到配合物问题 → 四步推断
    │
    ├─ 1. 从化学式提取信息
    │   └─ 氧化态 → dⁿ → 配体类型（σ给体/π给体/π受体）
    │
    ├─ 2. 从配体场推断裂分
    │   ├─ σ 给体（NH₃, en）→ 基础 Δo
    │   ├─ π 给体（X⁻, OH⁻）→ 推高 t₂g → Δo↓
    │   └─ π 受体（CO, CN⁻）→ 拉低 t₂g → Δo↑
    │
    ├─ 3. 从磁性/光谱锁定自旋态
    │   └─ μ_eff = √[n(n+2)] BM → 反推未成对电子数 n → HS/LS
    │
    └─ 4. 从 CFSE+JT 预测结构和反应性
        └─ CFSE 决定几何偏好，JT 预测畸变方向
```

## 二、配体三分类与光谱化学序列的轨道解释

> **核心觉悟**：光谱化学序列不是"背"出来的——是轨道相互作用的结果。

```
配体三分类：

  σ 给体（NH₃, en, H₂O）
  └─ 只给 σ 电子对 → 基础 Δo（中等场）

  π 给体（X⁻, OH⁻, O²⁻）
  └─ 填满的 p 轨道 + 金属 d 形成 π 键 → 推高 t₂g
      → t₂g 与 eg 间距缩小 → Δo↓（弱场）

  π 受体（CO, CN⁻, PR₃）
  └─ 金属 d → 空的配体 π* 反馈 → 拉低 t₂g
      → t₂g 与 eg 间距扩大 → Δo↑（强场）
```

**光谱化学序列（从弱场到强场）**：
I⁻ < Br⁻ < S²⁻ < SCN⁻ < Cl⁻ < NO₃⁻ < F⁻ < OH⁻ < H₂O < NCS⁻ < NH₃ < en < NO₂⁻ < PPh₃ < CN⁻ < CO

**为什么 CO 最强？** → 反馈 π 键（M→CO π*）拉低 t₂g，Δo 大幅增加。

## 三、八面体 σ 成键 MO 图 · 轨道级别标注

> **2025 决赛新要求**：不仅要画 d 轨道分裂，还要标注每个 MO 的金属/配体轨道来源。

```
                         ┌── a₁g*  (金属 4s - 配体 LGO(a₁g) 反键)
                         ├── t₁u*  (金属 4p - 配体 LGO(t₁u) 反键)
         ─ ─ ─ ─ ─ ─ ─ ─┼── eg*   (dₓ₂₋ᵧ₂, d_z² - 配体 LGO(eg) 反键) ← 这才是"d轨道"
    Δo                   │
         ─ ─ ─ ─ ─ ─ ─ ─┼── t₂g   (d_{xy}, d_{xz}, d_{yz} 非键)       ← π 键修正在此
                         │
         ─ ─ ─ ─ ─ ─ ─ ─┼── eg    (dₓ₂₋ᵧ₂ + LGO(eg) → eg σ)
                         ├── t₁u   (4p + LGO(t₁u) → t₁u σ)
                         └── a₁g   (4s + LGO(a₁g) → a₁g σ)
```

**轨道级别标注模板**："dₓ₂₋ᵧ₂ + N 2pₓ (LGO) → eg σ 键"——不是"dₓ₂₋ᵧ₂→eg"，别忘了配体贡献！

## 四、CFSE + JT 畸变 · 结构预测速查

### 4.1 CFSE 决定结构偏好

| dⁿ | 八面体 CFSE | 四面体 CFSE | 偏好 |
|:---|:---:|:---:|:---|
| d⁸ Ni²⁺ | −1.2Δo (最大) | −0.36Δt | **强八面体偏好**——解释 Ni²⁺ 在尖晶石中抢八面体位 |
| d⁹ Cu²⁺ | −0.6Δo | −0.27Δt | 八面体 → 但 JT 畸变！ |
| d¹⁰ Zn²⁺ | 0 | 0 | 无偏好——配体决定 |

### 4.2 JT 畸变方向速判

| dⁿ 构型 | JT 类型 | 结构变化 | 实例 |
|:---|:---|:---|:---|
| d⁴ HS (t₂g³eg¹) | 弱 JT（eg 简并） | 拉长八面体 | Mn³⁺ |
| d⁷ LS (t₂g⁶eg¹) | 弱 JT（eg 简并） | 拉长八面体 | Co²⁺ LS |
| d⁹ (t₂g⁶eg³) | **强 JT** | 严重拉长→近平面四方 | Cu²⁺（几乎所有 Cu²⁺ 配合物都是拉长八面体！） |

**口诀**：eg 电子不对称→拉长；t₂g 电子不对称→拉长但幅度小。d⁹ Cu²⁺ 拉长最显著。

## 五、磁性→构型推断 · 完整链

```
μ_eff 实验值 → n（未成对电子数）→ S → 自旋态 → 几何构型

公式：μ_eff = √[n(n+2)] BM  （仅自旋，忽略轨道贡献）
      μ_eff ≈ √[4S(S+1)] BM

速查表：
  n=1 (S=½) → μ ≈ 1.73
  n=2 (S=1) → μ ≈ 2.83
  n=3 (S=3/2) → μ ≈ 3.87
  n=4 (S=2) → μ ≈ 4.90
  n=5 (S=5/2) → μ ≈ 5.92

判断流程：
  μ_eff=4.90, 配体=NH₃, d⁶ Fe²⁺
  → n=4 → S=2 → 四个未成对电子 → t₂g⁴eg² → 高自旋
  → 八面体（6个NH₃配位）
```

## 六、SCO/SMM · 2025 热点概念

| 概念 | 全称 | 核心图像 | 典型体系 |
|:---|:---|:---|:---|
| **SCO** | 自旋态交叉（Spin Crossover） | 温度/压力/光照 → HS↔LS 转变 | Fe²⁺ N₆ 配合物 |
| **SMM** | 单分子磁体（Single Molecule Magnet） | 负各向异性 D → 磁矩沿易轴 → 低温磁化强度缓慢弛豫 | 直线形 Co²⁺、Mn₁₂ 簇 |

**直线形 Co²⁺ SMM 的轨道解释**：
- 直线形场：d_z² 能量最低（沿 z 轴无配体）→ 基态
- d_xz/d_yz 其次 → d_{xy}/dₓ₂₋ᵧ₂ 最高
- d_z² 基态 → 大轨道角动量 → 强各向异性 → SMM 行为

## 七、反式效应 · 取代反应预测

### 7.1 反式效应序列（Pt²⁺ 四方平面）

```
CN⁻ > CO > PR₃ > I⁻ > Cl⁻ > NH₃ > OH⁻ > H₂O
  ← 强反式效应（加速 trans 位取代）      弱反式效应 →
```

### 7.2 反式效应 vs 反式影响

| | 反式效应（trans effect） | 反式影响（trans influence） |
|:---|:---|:---|
| **性质** | **动力学**——加速 trans 配体的取代速率 | **热力学**——削弱 trans 配体的键强度 |
| **关注** | 反应速率 | 键长/键能/振动频率 |
| **应用** | 设计合成路线（先加哪个配体） | 解释结构数据 |

### 7.3 合成路线设计

**题**：从 [PtCl₄]²⁻ 合成 cis-PtCl₂(NH₃)(NO₂)

**速解**：
1. Cl⁻ 反式效应 > NH₃ → 先加 NH₃（避免 NH₃ 被后续取代）
2. [PtCl₄]²⁻ + NH₃ → [PtCl₃(NH₃)]⁻
3. Cl⁻（trans to Cl）的反式效应使对位 Cl 不稳定 → NO₂⁻ 取代对位 Cl
4. 最后加第二个 Cl 或控制 NO₂⁻ 取代 position → **cis 产物**

## 八、18e⁻ + 催化循环 · Wilkinson 加氢

```
Wilkinson 加氢完整循环（每步验证 18e⁻）：

[RhCl(PPh₃)₃]  Rh(I) d⁸, 16e⁻（不饱和→可氧化加成）
    │
    ├─ 1. 氧化加成：+H₂ → [RhCl(H)₂(PPh₃)₃]
    │   Rh(III) d⁶, 18e⁻ ✓ （金属氧化态+2，电子数+2）
    │
    ├─ 2. 配体解离：−PPh₃ → [RhCl(H)₂(PPh₃)₂]
    │   Rh(III) d⁶, 16e⁻（空出配位点给烯烃）
    │
    ├─ 3. 烯烃配位：+烯烃 → [RhCl(H)₂(烯烃)(PPh₃)₂]
    │   Rh(III) d⁶, 18e⁻ ✓
    │
    ├─ 4. 迁移插入：→ [RhCl(H)(烷基)(PPh₃)₂]
    │   烯烃插入 Rh-H → Rh-烷基（氧化态不变）
    │
    └─ 5. 还原消除：→ [RhCl(PPh₃)₂] + 烷烃
        Rh(I) d⁸, 14e⁻ → +PPh₃ → 回到 [RhCl(PPh₃)₃] 16e⁻
```

**三步基元反应**：氧化加成 → 迁移插入（转金属化）→ 还原消除

## 九、取代机理 D/Iₐ/I_d 速判

| 机理 | 全称 | 速率定律特征 | 立体化学结果 |
|:---|:---|:---|:---|
| **D** | 解离（Dissociative） | $v = k[\text{ML}_n]$（与进入配体浓度无关） | 可能外消旋 |
| **I_d** | 交换解离 | 键断裂 > 键形成 → 近 D | 近 D |
| **Iₐ** | 交换缔合 | 键形成 > 键断裂 → 近 A | 近 A |

**惰性 vs 不稳定**：惰性 = 动力学取代慢（d³ Cr³⁺，d⁶ LS Co³⁺——CFSE 大→活化能高）。这是**动力学**概念，与热力学稳定性（$K_f$）无关！

## 十、稀土配合物 · 3d vs 4f 对比

| 维度 | 3d 过渡金属 | 4f 稀土 |
|:---|:---|:---|
| **价轨道** | 3d 参与成键 | 4f 收缩，几乎不参与成键 |
| **Δ（配体场分裂）** | 200-400 kJ/mol | < 1 kJ/mol（极小！） |
| **磁性** | 由 dⁿ 决定，受配体场影响 | 由 fⁿ 决定，几乎不受环境影响 |
| **光谱** | d-d 跃迁（宽峰） | f-f 跃迁（线状，Laporte 禁阻但微弱允许） |
| **典型应用** | 催化、磁性材料 | Eu³⁺/Tb³⁺ 发光（天线效应）、Gd³⁺ MRI 造影剂 |

## 十一、例题速练

### 例 1：磁性反推构型

**题**：某配合物 μ_eff = 0 BM，组成为 [Ni(CN)₄]²⁻。推断几何构型和自旋态。

**速解**：μ=0 → n=0 → 全部电子配对。Ni²⁺ d⁸ → CN⁻ 强场 → 四方平面（d_{xy}²d_{xz}²d_{yz}²d_z²²dₓ₂₋ᵧ²⁰）→ dₓ₂₋ᵧ² 空 → 所有电子配对 ✓。

### 例 2：反式效应设计合成

**题**：从 [PtCl₄]²⁻ 合成 trans-PtCl₂(NH₃)₂。

**速解**：Cl⁻ 反式效应 > NH₃ → 先加 NH₃ 会导向 cis。要 trans：先利用 Cl⁻ 的高反式效应——或者先加 Cl⁻ 再取代，或用 NH₃ 饱和 → 热力学控制 → trans 更稳定。
