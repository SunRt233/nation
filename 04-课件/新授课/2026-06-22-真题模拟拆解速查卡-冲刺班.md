# 真题模拟拆解 · 速查卡

> 本卡不是"再讲一遍化学"——是"考场上的判断武器库"。所有化学原理你都会，本卡只教你怎么在 3 小时限时压力下**不丢不该丢的分**。

---

## 一、给材料学习题 · 五步拆解法

> 近年决赛最难题型——给你一段从未见过的机理/数据，要求现场推理。**你不是在"回忆知识"，是在"使用知识"。**

```
拿到给材料题 → 五步闭环
    │
    ├─ 1. 定位（2 min）
    │   ├─ 这段材料属于哪个领域？（有机机理/晶体结构/热力学/动力学/配合物）
    │   ├─ 材料中哪个词/反应/结构是你认识的？（锚点）
    │   └─ ⚠️ 不要试图先读懂全部材料——先找到你认识的东西！
    │
    ├─ 2. 提取（3 min）
    │   ├─ 圈出所有反应条件（温度、溶剂、催化剂、当量比）
    │   ├─ 圈出所有结构变化（哪个键断了/哪个键生了/哪个原子动了）
    │   └─ 把材料中的"长句"翻译成化学符号
    │
    ├─ 3. 类比（2 min）
    │   ├─ 这和你学过的哪个反应机理最像？
    │   ├─ 如果完全没见过——退到"电子效应"基本判断：
    │   │   └─ 谁富电子（亲核端）？谁缺电子（亲电端）？
    │   └─ 不要因为"没见过"就慌——没见过是正常的，出题人就是要考你没见过的
    │
    ├─ 4. 推理（5 min）
    │   ├─ 从锚点出发，用电子效应/平衡/动力学逐步推进
    │   ├─ 每推一步，回去看一眼材料——有没有条件被我漏了？
    │   └─ 推到"死胡同"时：回头检查第一步的类比是不是合理
    │
    └─ 5. 验证（3 min）
        ├─ 我的推理结果能解释材料中所有的现象吗？
        ├─ 副产品/副反应——材料中提到了吗？我的推理能解释吗？
        └─ 写答案前，最后问一遍："如果我是出题人，我想考什么？"
```

## 二、考场时间分配策略

| 阶段 | 时间 | 操作 | 铁律 |
|:---|:---:|:---|:---|
| **通览** | 5 min | 翻一遍全部题目，标难度（★易/★★中/★★★难） | 不急着做题——先知道"哪里有便宜可捡" |
| **第一遍** | 90 min | 做所有★和★★题，★★★只写思路不深算 | 遇到卡壳超过 3 min 的题，跳！标问号 |
| **第二遍** | 60 min | 回头做★★★题，按剩余分值排序 | 只攻"跳一跳够得着"的——完全没思路的放弃 |
| **检查** | 25 min | 验算（有效数字！单位！正负号！）+ 补漏 | 不改已经写的答案——除非有确凿错误证据 |

**考场三不：** ① 不跟一道题死磕超过 5 min ② 不在计算题上省有效数字检查 ③ 不让"这题我好像见过"的感觉替代仔细审题

## 三、常见命题套路与反套路

| 出题人套路 | 学生常见中招 | 反套路 |
|:---|:---|:---|
| 给熟悉的反应但反常条件 | 按标准条件推产物 → 全错 | 条件（温度/溶剂/浓度）是第一关键词！ |
| "已知……求……"中塞无用数据 | 把所有数据都用上 → 绕晕 | 先写目标量和已知量的关系式，多余数据不理 |
| 前一问的结论是后一问的条件 | 前一问做错 → 后一问连锁全错 | 计算题每问独立验算！用常识判断（pH 不能 20，产率不能 200%） |
| "给材料"题的材料故意多给一段 | 读完所有材料再做题 → 时间不够 | 先看问题再读材料——问题告诉你"材料中哪段有用" |
| 最后一问是"解释为什么" | 只写结论不写推理过程 | "因为A→B→C所以D"——每一步都写，过程分 > 结论分 |

## 四、快速验算三原则

1. **量纲检查**：算出来的值的单位对吗？mol vs mmol、kJ vs J、dm³ vs cm³
2. **数量级检查**：pH=20？ΔG=10⁶ kJ/mol？产率=200%？→ 绝对算错了
3. **极限情况检查**：设温度为 0 K 或 ∞，公式还合理吗？

## 五、"解释为什么"题的标准话术

> 竞赛中"解释题"的分值不亚于计算题。不要只写结论。

| 问法 | 答法模板 |
|:---|:---|
| "为什么 A 比 B 更稳定？" | A 的（共振/超共轭/电荷分散/……）使（中间体/过渡态/产物）更稳定 → 能量更低 |
| "为什么反应走路径 1 而非路径 2？" | 路径 1 的过渡态因（位阻更小/电荷更匹配/轨道重叠更好）→ 活化能更低 → 动力学有利 |
| "为什么改变条件产物变了？" | 条件改变使（热力学控制↔动力学控制）→ 不同条件下最稳定产物不同 |
| "为什么这个机理不对？" | 该机理预测（某个中间体/某个选择性）→ 但实验观察到（相反证据）→ 排除 |

## 六、例题速练

### 例 1：给材料学习——陌生催化循环

**材料**：某 Pd 催化偶联反应，催化循环含 Pd(0)→Pd(II)→Pd(0)，其中氧化加成步骤的速率与底物中 C–X 键的 BDE 相关。

**问**：为什么 Ar–I 比 Ar–Br 反应更快？

**速解**：氧化加成 = Pd(0) 插入 C–X 键 → 需要断 C–X 键 → BDE(C–I) < BDE(C–Br) → Ar–I 的氧化加成活化能更低。不要因为"没见过这个循环"就慌——问的是 BDE，你学过。

### 例 2：计算题——单位陷阱

**材料**：某电池反应在 298K 的 E° = 1.10V，求 ΔG°（kJ/mol）。

**速解**：ΔG° = −nFE°。关键：n=2（看方程式！），F=96485 C/mol，1J=1C·V。ΔG° = −2×96485×1.10 = −212267 J/mol = −212 kJ/mol。⚠️ 经常有人忘了 n 或单位换算。

## 八、自查清单（考前一天过一遍）

- [ ] 给材料学习五步法（定位→提取→类比→推理→验证）能默写
- [ ] 考场时间分配表记住了——第一遍不卡题
- [ ] 有效数字规则：加减看小数点后位数、乘除看有效位数、pH 看小数点后位数
- [ ] 单位陷阱清单：kJ↔J、dm³↔cm³、kPa↔Pa、mol↔mmol
- [ ] "解释为什么"题的话术模板会套用
- [ ] 验算三原则：量纲→数量级→极限
- [ ] R=8.314 时 p/Pa, V/m³；R=8.31 时 p/kPa, V/dm³
- [ ] 排水集气扣水蒸气压
- [ ] 混合后浓度要重算（体积变化！）
- [ ] 题号与答题纸对应——答错位置 = 0 分
