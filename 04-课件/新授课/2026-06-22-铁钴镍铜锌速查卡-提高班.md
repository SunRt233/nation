# 铁钴镍铜锌 · 速查卡

> 基础班你背过"Fe²⁺浅绿/Fe³⁺黄棕/Cu²⁺蓝/Zn²⁺无色"——本卡把 Fe/Co/Ni/Cu/Zn 从**颜色记忆**升级到**价态竞争+配合物稳定化+歧化平衡**的推断系统。

---

## 一、铁 · 价态与颜色

```
Fe²⁺ vs Fe³⁺——竞赛最高频的氧化还原对：

  Fe²⁺：浅绿色，还原性——被 Cl₂/H₂O₂/MnO₄⁻ 氧化
  Fe³⁺：黄棕色，氧化性——被 Cu/I⁻/H₂S 还原

  E°(Fe³⁺/Fe²⁺) = +0.77V
    ├─ 可氧化：Cu → Cu²⁺（+0.34V）、I⁻ → I₂（+0.54V）、H₂S → S（+0.14V）
    └─ 不能氧化：Cl⁻ → Cl₂（+1.36V）、Br⁻ → Br₂（+1.07V）
```

### 1.1 铁的特征反应

| 反应 | 方程式 | 现象/用途 |
|:---|:---|:---|
| **Fe³⁺ + SCN⁻** | Fe³⁺ + nSCN⁻ → [Fe(SCN)ₙ]³⁻ⁿ | **血红色**——Fe³⁺ 最特征鉴定（F⁻/PO₄³⁻ 干扰！） |
| **Fe²⁺ + [Fe(CN)₆]³⁻** | Fe²⁺ + K₃[Fe(CN)₆] → | 滕氏蓝（Turnbull's blue） |
| **Fe³⁺ + [Fe(CN)₆]⁴⁻** | Fe³⁺ + K₄[Fe(CN)₆] → | 普鲁士蓝（Prussian blue） |
| **Fe²⁺ + OH⁻** | Fe²⁺ + 2OH⁻ → Fe(OH)₂↓（白）→ 绿 → 棕 | 白色→灰绿→红棕（空气氧化：4Fe(OH)₂+O₂+2H₂O→4Fe(OH)₃） |

### 1.2 普鲁士蓝 vs 滕氏蓝

```
历史争论：普鲁士蓝（Fe³⁺ + [Fe(CN)₆]⁴⁻）vs 滕氏蓝（Fe²⁺ + [Fe(CN)₆]³⁻）

现代表征结论（XRD/Mössbauer）：
  二者结构相同！都是 Fe³⁺/Fe²⁺ 与 CN⁻ 的立方骨架
  框架：Fe(III)-N≡C-Fe(II) 的无限网络

关键：生成过程中发生了电子转移——滕氏蓝中 Fe²⁺ 被 [Fe(CN)₆]³⁻ 氧化
```

## 二、钴 · 价态反转

```
钴的核心矛盾：Co²⁺ vs Co³⁺——谁的稳定性更高？取决于配体！

  水溶液中：
    Co²⁺（粉红，稳定）>>>> Co³⁺（强氧化剂，不稳定）
    E°(Co³⁺/Co²⁺) = +1.84V —— Co³⁺ 能氧化 H₂O→O₂！

  配合物中（配体 = NH₃, CN⁻ 等）：
    [Co(NH₃)₆]³⁺ 稳定 >>> [Co(NH₃)₆]²⁺（后者易被空气氧化）
    配体场效应反转了价态稳定性！

  口诀：水溶液中 Co²⁺ 稳，氨配合物中 Co³⁺ 稳
```

| 物种 | 颜色 | 稳定性 | 关键信息 |
|:---|:---|:---|:---|
| **[Co(H₂O)₆]²⁺** | 粉红 | 水中稳定 | d⁷ 高自旋 |
| **[Co(H₂O)₆]³⁺** | 蓝色 | 水中极不稳定 | 强氧化剂，氧化 H₂O |
| **[Co(NH₃)₆]²⁺** | 土黄色 | 易被空气氧化 | → [Co(NH₃)₆]³⁺ |
| **[Co(NH₃)₆]³⁺** | 橙黄色 | 配合物中稳定 | d⁶ 低自旋——配位场稳定化能大 |
| **CoCl₄²⁻** | 蓝色 | 无水/有机溶剂 | 四面体配合物（区别于八面体粉红色） |

**维生素 B₁₂**：Co 与咕啉环配位——天然有机金属化合物的罕见例子（点到为止）。

## 三、镍 · 配合物与电池

```
Ni²⁺：绿色（[Ni(H₂O)₆]²⁺，d⁸ 八面体）

  与 NH₃：[Ni(H₂O)₆]²⁺ + 6NH₃ → [Ni(NH₃)₆]²⁺（蓝色）
  与 en：[Ni(H₂O)₆]²⁺ + 3en → [Ni(en)₃]²⁺（紫色）
  与 DMG：Ni²⁺ + 2DMG → Ni(DMG)₂↓（鲜红色沉淀——Ni²⁺ 特征鉴定！）

Ni 的工业角色：
  ├─ 镍镉电池：Cd + 2NiO(OH) + 2H₂O ⇌ Cd(OH)₂ + 2Ni(OH)₂
  ├─ 镍氢电池：MH + NiO(OH) ⇌ M + Ni(OH)₂
  └─ Raney Ni：催化加氢（H₂ 在 Ni 表面解离）
```

## 四、铜 · 歧化与稳定化

```
Cu⁺ vs Cu²⁺——热力学决定命运：

  Cu⁺ 在水溶液中歧化：2Cu⁺ → Cu²⁺ + Cu（K ≈ 10⁶！）
    原因：Cu²⁺ 的水合能极大（电荷高、半径小）→ 补偿了第二电离能

  Cu⁺ 稳定化策略：
    ├─ 沉淀：CuCl↓（白色）、CuI↓（白色）
    ├─ 配合物：[Cu(CN)₂]⁻、[Cu(NH₃)₂]⁺（无色）
    └─ 固态：Cu₂O（红色）、Cu₂S（黑色）

  Cu²⁺：蓝色，在水溶液中稳定
    ├─ + NH₃ → [Cu(NH₃)₄]²⁺（深蓝色——特征！）
    ├─ + OH⁻ → Cu(OH)₂↓（蓝色胶状）→ 加热→CuO↓（黑色）
    └─ + I⁻ → CuI↓ + I₂（Cu²⁺ 氧化 I⁻ 同时自身沉淀为 CuI）
```

### 4.1 铜的冶炼

```
火法炼铜：黄铜矿(CuFeS₂) → 冰铜(Cu₂S·FeS) → 粗铜 → 电解精炼
湿法炼铜：CuO + H₂SO₄ → CuSO₄ + H₂O → Fe + CuSO₄ → FeSO₄ + Cu

印刷电路板蚀刻：FeCl₃ + Cu → FeCl₂ + CuCl₂
  → Fe³⁺ 氧化 Cu——绿色的 FeCl₃ 溶液蚀刻铜箔
```

## 五、锌 · 两性与防护

```
Zn 的两性（弱——远弱于 Al）：

  与酸：Zn + 2H⁺ → Zn²⁺ + H₂↑
  与碱：Zn + 2OH⁻ + 2H₂O → [Zn(OH)₄]²⁻ + H₂↑

  Zn(OH)₂↓（白色）：
    ├─ 加酸：Zn(OH)₂ + 2H⁺ → Zn²⁺ + 2H₂O
    └─ 加碱：Zn(OH)₂ + 2OH⁻ → [Zn(OH)₄]²⁻

  与 Al 对比：
    ├─ Al 两性更强（Al(OH)₃ 在 pH~4-10 沉淀）
    └─ Zn 两性较弱（Zn(OH)₂ 在 pH~6-12 沉淀）
```

### 5.1 牺牲阳极原理

```
Zn 比 Fe 活泼 → 优先被腐蚀 → 保护铁

  E°(Zn²⁺/Zn) = −0.76V < E°(Fe²⁺/Fe) = −0.44V
  → Zn 优先失去电子 → Zn → Zn²⁺ + 2e⁻
  → Fe 上的电子不被消耗 → Fe 得到阴极保护

  应用：镀锌铁（白铁）、船体牺牲阳极、管道保护
```

## 六、五元素对比速查

```
颜色速记：
  Fe²⁺ 浅绿  Fe³⁺ 黄棕  → SCN⁻ 血红色（Fe³⁺ 鉴定）
  Co²⁺ 粉红  Co³⁺ 蓝（水合）→ 配合物中 Co³⁺ 更稳（价态反转！）
  Ni²⁺ 绿    → DMG 鲜红色沉淀（Ni²⁺ 鉴定）
  Cu²⁺ 蓝    Cu⁺ 无色（水溶液中歧化！）
  Zn²⁺ 无色  → 两性（弱）

价态竞争核心：
  Fe：Fe²⁺(还原性) ↔ Fe³⁺(氧化性)——电势+0.77V 居中
  Co：水溶液 Co²⁺ 稳，配合物 Co³⁺ 稳——配体场反转
  Cu：水溶液 Cu²⁺ 稳，Cu⁺ 必须靠沉淀/配合物稳定化
  Zn：只有 +2
```

## 七、常见易错点

| 错误 | 纠正 |
|:---|:---|
| "Cu⁺ 和 Cu²⁺ 一样稳定" | Cu⁺ 在水中**歧化**（K≈10⁶）！只有 CuCl/CuI/[Cu(CN)₂]⁻ 等才能稳定 Cu⁺ |
| "普鲁士蓝和滕氏蓝结构不同" | XRD/Mössbauer 证明**二者结构相同**——都是 Fe(III)-NC-Fe(II) 网络 |
| "Co³⁺ 在任何条件下都是强氧化剂" | 在 [Co(NH₃)₆]³⁺ 中 Co³⁺ **稳定**——配体场改变了相对稳定性 |
| "Zn(OH)₂ 和 Al(OH)₃ 两性一样强" | **Al 的两性强得多**——Zn(OH)₂ 只溶于强碱且溶解度小 |
| "Fe³⁺ + I⁻ → FeI₃" | **氧化还原！** 2Fe³⁺ + 2I⁻ → 2Fe²⁺ + I₂——Fe³⁺ 氧化 I⁻ |
| "Cu²⁺ + I⁻ → CuI₂" | **氧化还原+沉淀！** 2Cu²⁺ + 4I⁻ → 2CuI↓ + I₂——Cu²⁺ 氧化 I⁻ |
| "Fe(OH)₂ 是红棕色沉淀" | Fe(OH)₂ 是**白色**！变红棕是**被空气氧化**成 Fe(OH)₃ |
| "Ni²⁺ 溶液是蓝色的" | Ni²⁺ 水合离子是**绿色**！[Ni(NH₃)₆]²⁺ 才是蓝色 |

## 八、例题速练

### 例 1：Fe 价态推断

**题**：浅绿色溶液 A 通 Cl₂ → 变黄棕色溶液 B。B + KSCN → 血红色。A 和 B 分别是什么？

**速解**：A = Fe²⁺（浅绿），B = Fe³⁺（黄棕）。Cl₂ 氧化：2Fe²⁺ + Cl₂ → 2Fe³⁺ + 2Cl⁻。Fe³⁺ + SCN⁻ → 血红色配合物（特征鉴定）。

### 例 2：Cu⁺ 歧化

**题**：为什么 Cu₂SO₄ 在水中不存在而 Ag₂SO₄ 稳定？

**速解**：Cu⁺ 在水中歧化——2Cu⁺ → Cu²⁺ + Cu（K≈10⁶）。Cu²⁺ 的水合能极大补偿了第二电离能。Ag⁺ 的电离能较高 + Ag²⁺ 水合能不够大 → Ag⁺ 不歧化。

### 例 3：Co 价态反转

**题**：为什么 CoCl₂ 溶液呈粉红色，但加过量 NH₃ 后久置变橙黄色？

**速解**：[Co(H₂O)₆]²⁺（粉红）→ 加 NH₃ → [Co(NH₃)₆]²⁺（土黄）→ 被空气中 O₂ 氧化 → [Co(NH₃)₆]³⁺（橙黄）。配体从 H₂O 变 NH₃ → 配位场增强 → Co³⁺ 被稳定化。

## 九、自查清单

- [ ] Fe²⁺(浅绿/还原性) ↔ Fe³⁺(黄棕/氧化性)，E°=+0.77V
- [ ] Fe³⁺ + SCN⁻ → 血红色（F⁻/PO₄³⁻ 干扰）
- [ ] 普鲁士蓝 = 滕氏蓝（结构相同——Fe(III)-NC-Fe(II) 网络）
- [ ] Fe(OH)₂ 白色→灰绿→红棕（空气氧化过程）
- [ ] Co²⁺（粉红）水溶液稳定 vs Co³⁺（蓝）强氧化剂
- [ ] 配体场反转：[Co(NH₃)₆]³⁺ 稳定——配合物中 Co³⁺ 更稳
- [ ] Ni²⁺ + DMG → 鲜红色沉淀（Ni²⁺ 特征鉴定）
- [ ] Cu⁺ 歧化（K≈10⁶）——CuCl/CuI/[Cu(CN)₂]⁻ 稳定 Cu⁺
- [ ] Cu²⁺ + NH₃ → [Cu(NH₃)₄]²⁺ 深蓝色（特征）
- [ ] Zn 两性（弱），牺牲阳极原理（Zn→Zn²⁺+2e⁻ 保护 Fe）
- [ ] 印刷电路板蚀刻：Fe³⁺ + Cu → Fe²⁺ + Cu²⁺
- [ ] Cu²⁺ + I⁻ → CuI↓ + I₂（不是复分解！氧化还原+沉淀）
