## 一、本节要解决的问题

1. 为什么 ΔG=ΔH−TΔS？这个公式是怎么来的？还能推出什么？
2. 温度变了，平衡常数怎么变？（不只背勒夏特列）
3. 沸点升高、凝固点降低、渗透压——为什么都能从同一个"化学势"推出来？

### 学习目标

- 能从 dU=TdS−pdV 出发，独立推导到 van't Hoff 方程
- 能区分 van't Hoff 等温式/积分式/微分式，并知道各用在什么场景
- 能判断温度题应该用 van't Hoff 还是 Clausius-Clapeyron

## 二、推导链总地图

> **核心原则**：只需要记住 `dU = TdS − pdV`，其余全部能推出来。

```text
dU = TdS − pdV          ← 联合一二定律（唯一需要记住的起点）
    │
    ├─ Legendre 变换 ────────────────────────────────────────┐
    │   H = U + pV        →  dH = TdS + Vdp                  │
    │   A = U − TS        →  dA = −SdT − pdV                 │
    │   G = H − TS        →  dG = −SdT + Vdp                 │
    │                                                         │
    ├─ 全微分条件 → Maxwell 关系 ─────────────────────────────┤
    │   (∂T/∂V)_S = −(∂p/∂S)_V                               │
    │   (∂T/∂p)_S =  (∂V/∂S)_p                               │
    │   (∂S/∂V)_T =  (∂p/∂T)_V                               │
    │   (∂S/∂p)_T = −(∂V/∂T)_p                               │
    │                                                         │
    ├─ Gibbs-Helmholtz 方程 ──────────────────────────────────┤
    │   [∂(G/T)/∂T]_p = −H/T²                                │
    │   → 变温 ΔG 计算（竞赛最爱考）                           │
    │                                                         │
    ├─ 化学势 ────────────────────────────────────────────────┤
    │   μ_B = (∂G/∂n_B)_{T,P,n_{j≠B}}                        │
    │   μ = μ° + RTln(p/p°)   （理想气体）                    │
    │   μ = μ° + RTln a       （非理想通式）                   │
    │       │                                                  │
    │       ├─→ van't Hoff 等温式：ΔG = ΔG° + RTlnQ           │
    │       │   → 平衡时 ΔG = 0：ΔG° = −RTlnK                │
    │       │                                                  │
    │       ├─→ van't Hoff 微分式：dlnK/dT = ΔH°/RT²         │
    │       │   → 积分式：ln(K₂/K₁) = −(ΔH°/R)(1/T₂−1/T₁)   │
    │       │                                                  │
    │       ├─→ Clapeyron：dp/dT = ΔS/ΔV                     │
    │       │   → Clausius-Clapeyron：ln(p₂/p₁) = −(ΔH/R)(1/T₂−1/T₁) │
    │       │                                                  │
    │       └─→ 依数性（化学势相等→三项统一推导）              │
    │           沸点升高：ΔT_b = K_b·m                         │
    │           凝固点降低：ΔT_f = K_f·m                       │
    │           渗透压：Π = cRT                                │
    │                                                          │
    └─ 综合推断：ΔH(T) → ΔS(T) → ΔG(T) → K(T) → 反应方向     │
         └─ 电化学接口：ΔG° = −nFE°                           │
```

## 三、核心公式速查

### 3.1 四大基本公式

| 热力学函数 | 定义 | 微分式 | 自然变量 |
|:---|:---|:---|:---|
| 内能 U | — | dU = TdS − pdV | S, V |
| 焓 H | H = U + pV | dH = TdS + Vdp | S, p |
| Helmholtz 自由能 A | A = U − TS | dA = −SdT − pdV | T, V |
| Gibbs 自由能 G | G = H − TS | dG = −SdT + Vdp | T, p |

> **记忆法**：四大公式的自然变量都是"共轭对"——S↔T, V↔p。每个公式都是两项，符号交替。

### 3.2 Maxwell 关系（4 条）

| 来源 | Maxwell 关系 | 口诀 |
|:---|:---|:---|
| dU | (∂T/∂V)_S = −(∂p/∂S)_V | T对V = −p对S |
| dH | (∂T/∂p)_S = (∂V/∂S)_p | T对p = V对S |
| dA | (∂S/∂V)_T = (∂p/∂T)_V | S对V = p对T |
| dG | (∂S/∂p)_T = −(∂V/∂T)_p | S对p = −V对T（最重要） |

> **用法**：竞赛中主要用 dG 这条——**(∂S/∂p)_T = −(∂V/∂T)_p**，用于把难算的熵变换成好算的体积膨胀系数。

### 3.3 van't Hoff 方程（一个公式，三种用法）

| 形式 | 公式 | 用途 |
|:---|:---|:---|
| **等温式** | ΔG = ΔG° + RTlnQ | 判断任意状态下的反应方向 |
| **微分式** | dlnK/dT = ΔH°/RT² | 定性判断：吸热→升温→K↑ |
| **积分式** | ln(K₂/K₁) = −(ΔH°/R)(1/T₂−1/T₁) | 定量计算：已知 T₁ 的 K₁，求 T₂ 的 K₂ |

> ⚠️ **积分式的符号陷阱**：右边是 **−(ΔH°/R)(1/T₂−1/T₁)**，注意负号！吸热反应 ΔH°>0，升温 T₂>T₁，1/T₂<1/T₁，括号为负，负负得正 → ln(K₂/K₁)>0 → K₂>K₁ ✓

### 3.4 Clausius-Clapeyron 方程

$$ \ln\frac{p_2}{p_1} = -\frac{\Delta_{vap}H}{R}\left(\frac{1}{T_2} - \frac{1}{T_1}\right) $$

> **与 van't Hoff 积分式的对比**：形式完全一样，但 van't Hoff 用 K 和 ΔH°，Clausius-Clapeyron 用 p 和 Δ_vapH。**判断用哪个**：问的是"平衡常数随温度变化"→ van't Hoff；问的是"蒸气压/沸点"→ Clausius-Clapeyron。

## 四、van't Hoff vs Clausius-Clapeyron 快速判断

| 题目问法 | 用哪个 | 公式中的变量 |
|:---|:---|:---|
| "升温后 K 变为多少" | van't Hoff | K, ΔH° |
| "某温度下 ΔG 是多少" | van't Hoff 等温式 | ΔG, Q |
| "某高山上的沸点" | Clausius-Clapeyron | p, Δ_vapH |
| "加压后相变温度" | Clapeyron | p, T, ΔV |
| "依数性" | 化学势相等 → μ_A(l,T,p,x_A)=μ_A(g,T,p) | x_A, T |

## 五、常见易错点

| # | 易错点 | 正确认知 |
|:---:|:---|:---|
| 1 | 把 van't Hoff 积分式的符号搞反 | 先判断"吸热升温→K变大"来验证符号 |
| 2 | 温度题一律套勒夏特列 | 勒夏特列只能定性！定量必须用 van't Hoff |
| 3 | 分不清 van't Hoff 和 Clausius-Clapeyron | 看对象：K 相关→van't Hoff；p 相关→Clausius-Clapeyron |
| 4 | 标准态与非标准态混淆 | ΔG° 是标准态（各物质 p=1 bar）；ΔG 是实际状态——差一个 RTlnQ |
| 5 | 化学势相等≠浓度相等 | 沸点升高不是因为液相化学势变了，是因为加了溶质→μ_A(l)降低了→需要更高 T 让 μ_A(g) 也降低来匹配 |
| 6 | ΔH° 当成常数 | 严格说 ΔH° 随 T 变（ΔC_p≠0），但竞赛中除非明确给 ΔC_p 数据，否则默认常数 |

## 六、例题速练

### 例 1：van't Hoff 积分式
> 某反应 ΔH° = +50 kJ/mol，300K 时 K=2.0。求 350K 时的 K。

**解**：吸热升温 → K 应变大。
$$\ln\frac{K_2}{2.0} = -\frac{50000}{8.314}\left(\frac{1}{350} - \frac{1}{300}\right)$$
$$= -6014 \times (-0.000476) = +2.86$$
$$K_2 = 2.0 \times e^{2.86} = 2.0 \times 17.5 = 35$$

### 例 2：Clausius-Clapeyron
> 水在 100°C 时蒸气压为 760 Torr，Δ_vapH = 40.7 kJ/mol。求 90°C 时的蒸气压。

**解**：降温 → 蒸气压应变小。
$$\ln\frac{p_2}{760} = -\frac{40700}{8.314}\left(\frac{1}{363} - \frac{1}{373}\right)$$
$$= -4895 \times (0.002755 - 0.002681) = -4895 \times 0.000074 = -0.362$$
$$p_2 = 760 \times e^{-0.362} = 760 \times 0.696 = 529 \text{ Torr}$$

### 例 3：依数性
> 0.1 mol 蔗糖溶于 500 g 水。水的 K_b=0.512 K·kg/mol。求沸点升高。

**解**：质量摩尔浓度 m = 0.1/0.5 = 0.2 mol/kg
$$\Delta T_b = K_b \cdot m = 0.512 \times 0.2 = 0.102 \text{ K}$$
沸点 = 100.102°C
