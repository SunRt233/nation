# 元素化学深度与结构推断综合 · 速查卡

> Round 1/2 你学了全部元素化学——本卡的核心不是"再讲一遍元素"，而是**给你一个陌生物质也能执行推断的结构化框架**。2025 初赛/决赛的元素化学新形式：不给熟悉物质，给陌生物质+数据→推断。

---

## 一、陌生物质推断 · 五步法

```
拿到陌生物质推断题 → 五步结构化操作
    │
    ├─ Step 1：物理性质（1 min）
    │   ├─ 颜色 → 暗示元素（Cu²⁺蓝绿、Cr³⁺绿紫、MnO₄⁻紫、Co²⁺粉红）
    │   ├─ 磁性 μ_eff → 未成对电子数 n → dⁿ 构型 → 缩小范围
    │   └─ 状态（气体/液体/固体）→ 共价 vs 离子
    │
    ├─ Step 2：结构数据（3 min）
    │   ├─ 晶胞参数 → 晶系判断（先看角、再看边）
    │   ├─ 投影图 → 化学式 → 配位数 → 结构类型
    │   └─ 键长/键角 → 成键类型（单/双/多重键）+ 杂化方式
    │
    ├─ Step 3：反应性（3 min）
    │   ├─ 与水/酸/碱的反应 → 金属性/非金属性
    │   ├─ 氧化还原行为 → 可变氧化态？
    │   └─ 配位行为 → 易形成配合物？（d 区特征）
    │
    ├─ Step 4：热力学判据（3 min）
    │   ├─ ΔG°/E°/K → 稳定性判断
    │   ├─ Born-Haber 循环 → 氧化态是否可及
    │   └─ Latimer/Frost 图 → 歧化/归中判断
    │
    └─ Step 5：综合推断 + 方程式（5 min）
        ├─ 以上四条线索交汇 → 锁定元素身份
        ├─ 写出所有相关方程式
        └─ 验证：我推的元素能解释所有数据吗？
```

**核心原则**：不要一上来就猜元素名——先收数据、再收线索、最后交汇锁定。

## 二、做题前先分型：三类元素推断题

| 题型 | 特征 | 第一动作 |
|:---|:---|:---|
| **化学史材料题** | 古文描述+操作+物质名（如"雄黄""砒霜""胆矾"） | 三步翻译法（见 §七） |

## 三、新维度 1：多酸 POM + Zintl 相

### 3.1 多酸 POM（Polyoxometalate）

**Keggin 型结构**：[XM₁₂O₄₀]ⁿ⁻

```
中央 XO₄ 四面体（X = P⁵⁺, Si⁴⁺, etc.）
  └─ 被 12 个 MO₆ 八面体包围（M = W⁶⁺, Mo⁶⁺, V⁵⁺）
      └─ 每 3 个 MO₆ 共边成一组 → 4 组围绕中央四面体
```

**n 的确定方法**：从 X 和 M 的氧化态反推
- 例：H₃PW₁₂O₄₀ → P⁵⁺ + 12W⁶⁺ = +77；40O²⁻ = −80 → 净 −3 → n = 3 ✓

### 3.2 Zintl 相

**核心概念**：电正性金属（Na/K）+ 电负性金属/半金属（Sn/Pb/Sb）→ 形成含**金属阴离子簇**的化合物

- [Sn₉]⁴⁻：带帽四方反棱柱
- Wade 规则：骨架电子对数 = 顶点数 + 1 → 决定多面体类型

## 四、新维度 2：稀土/锕系系统

### 4.1 镧系收缩的三条后果

```
4f 轨道不参与成键（收缩在内部）
    │
    ├─ 后果 1：Zr⁴⁺(r=72pm) ≈ Hf⁴⁺(r=71pm) → Zr/Hf 分离极其困难
    │
    ├─ 后果 2：Y³⁺(r=90pm) ≈ Ho³⁺(r=90.1pm) → Y 和重稀土共生
    │
    └─ 后果 3：从 La 到 Lu 离子半径缓慢减小 → 性质渐变（不是剧变）
```

### 4.2 两个异常氧化态

| 异常 | 电子构型 | 原因 |
|:---|:---|:---|
| **Ce⁴⁺** | [Xe]4f⁰ | 空 f 轨道 → 类似于惰性气体构型 → 异常稳定 |
| **Eu²⁺** | [Xe]4f⁷ | 半满 f 轨道 → 类似于 Gd³⁺ 的稳定构型 |
| **Tb⁴⁺** | [Xe]4f⁷ | 同上——半满稳定 |

### 4.3 锕系特殊物种

- **UO₂²⁺（铀酰离子）**：直线形 O=U=O → U(VI)，非常稳定，是铀化学的核心物种

## 五、新维度 3：铂族金属

| 配合物 | 金属 | dⁿ | e⁻ 数 | 功能 |
|:---|:---|:---:|:---:|:---|
| **Wilkinson 催化剂** RhCl(PPh₃)₃ | Rh(I) | d⁸ | 16e⁻ | 烯烃加氢 |
| **Vaska 配合物** IrCl(CO)(PPh₃)₂ | Ir(I) | d⁸ | 16e⁻ | O₂ 可逆加合 |

**共同特征**：都是 d⁸、16e⁻、平面四方 → 可氧化加成（变成 d⁶、18e⁻ 八面体）

**反式效应序列（Pt²⁺）**：CN⁻ > CO > PR₃ > I⁻ > Cl⁻ > NH₃ > OH⁻ > H₂O

## 六、新维度 4：非经典化学键

| 类型 | 实例 | 关键结构特征 | 为什么"非经典" |
|:---|:---|:---|:---|
| **3c-2e 键** | B₂H₆（B–H–B 桥键） | 2 个电子被 3 个原子共享 | 打破"2c-2e"定式 |
| **Be–Be 键** | [Be₂R₂]（第 38 届决赛） | Be(I) 而非 Be(II) | 碱土金属的金属-金属键 |
| **主族多重键** | R–P=P–R, R–Si=Si–R | 需要大位阻阻止二聚/聚合 | 主族多重键的热力学不稳定性 |
| **M–M 多重键** | Cr–Cr 四重键（σ+2π+δ） | δ 键锁定重叠构象 | δ 键的存在+重叠构象的刚性 |

## 七、新维度 5：热力学判据三合一

### 7.1 三种判据各自用途

```
Born-Haber 循环 → 判断陌生物质的氧化态是否"可及"
    │  例：从电离能和晶格能判断 CaCl 是否存在 → 不存在（歧化为 Ca+CaCl₂）
    │
Latimer 图 → 判断多氧化态物种的歧化/归中
    │  例：E°(右) > E°(左) → 物种歧化
    │
Ellingham 图 → 判断最低还原温度和碳热还原可行性
    │  例：ΔG° = 0 的温度 = 最低还原温度
    │
Frost 图 → 多物种的稳定性一目了然
    └─ 斜率 = E°；"凸点"=歧化，"凹点"=归中
```

### 7.2 速判口诀

- **Born-Haber**："晶格能大才能活"——MX₂ 的晶格能 ≈ 2×MX 的晶格能，所以 MX 歧化
- **Latimer**："右边比左边正→歧化"
- **Frost**："凸点歧化、凹点归中"

## 八、化学史推断策略 · 三步翻译法

```
古文描述 → 三步翻译 → 现代化学方程式

Step 1：圈出操作词（1 min）
  "煅" = 高温加热（通常有气体产物）
  "淬" = 热 → 冷（急冷，可能改变晶型）
  "飞" = 升华/挥发
  "点" = 少量加入引发反应（催化/成核）
  "淋" = 用液体浇→溶解/反应
  "养" = 长时间加热/保温

Step 2：定位关键物质（2 min）
  雄黄 = As₄S₄（或 AsS）
  砒霜/砒石 = As₂O₃
  胆矾 = CuSO₄·5H₂O
  绿矾 = FeSO₄·7H₂O
  朱砂 = HgS
  密陀僧 = PbO

Step 3：按操作写方程式（3 min）
  煅 + 物质A → 物质B + （气体？）
  淬 + 物质B → 物质C
  ...
```

## 九、常见易错点

| 错误 | 纠正 |
|:---|:---|
| 一上来就猜"这应该是 Fe"| 先走五步流程——物理→结构→反应→热力学→方程，最后锁定 |
| 把镧系收缩当成"4f 参与成键" | 4f 在 5s²5p⁶ 内部，**不参与成键**——收缩来自核电荷递增 |
| Born-Haber 只用来算晶格能 | Born-Haber 的真正用途是**判断氧化态是否稳定**——CaCl 为什么不存在？因为 2×U(CaCl) < U(CaCl₂) |
| 化学史题试图逐字翻译古文 | 圈出操作词和物质名——其余描述可能是文学的（不直接对应化学） |
| Ce⁴⁺ 氧化性强所以不稳定 | 恰恰相反——Ce⁴⁺ 因为 [Xe]4f⁰ 的惰性气体构型而异常**稳定** |

## 十、例题速练

### 例 1：五步推断法实战

**题**：某绿色晶体 A 溶于水呈绿色，加 NaOH 产生灰绿色沉淀，沉淀在空气中逐渐变棕。A 的磁性 μ_eff ≈ 3.87 BM。推断 A。

**速解**：
1. 物理：绿色 + μ=3.87 → n=3 → d³ Cr³⁺ 或 d⁷ Co²⁺ HS
2. 反应：加 NaOH→灰绿色沉淀→空气中变棕 → Cr³⁺（Cr(OH)₃ 灰绿→CrO₂⁻→但不是棕）
   等等！灰绿→棕 → Fe²⁺→Fe³⁺ 转换（Fe(OH)₂ 白→绿→棕）？d⁶ Fe²⁺ HS μ≈4.90 不是 3.87。
   重新来：μ=3.87 → n=3。d³ Cr³⁺ 的沉淀是 Cr(OH)₃ 灰绿，在空气中不变棕。
   这不对……让数据说话：μ=3.87 → n=3 确认。绿色晶体+有颜色变化→可能是 Ni²⁺(d⁸, μ≈2.83)？不匹配。
   **重新思考**：绿色+μ≈3.87 → d⁷ Co²⁺ HS 或 d³ Cr³⁺。Cr³⁺ 在碱性中 Cr(OH)₃ 灰绿→空气中氧化→CrO₄²⁻ 黄色。不是棕。
   Co²⁺ + NaOH → Co(OH)₂ 蓝/粉→空气中氧化→Co(OH)₃ 棕黑。μ=3.87 → d⁷ HS Co²⁺ n=3 ✓。
   → **A = CoCl₂·6H₂O 或 CoSO₄·7H₂O**。

### 例 2：化学史翻译

**题**：古文："取砒霜……以炭火煅之……有白烟如雪飞腾"→写方程式。

**速解**：
- 砒霜 = As₂O₃
- 煅 + 炭 → C 还原
- 白烟飞腾 → As₄O₆ 升华（As₂O₃ 易升华）
- 方程式：As₂O₃ + C →（可能的还原产物：As₄↑ + CO/CO₂，但"白烟"说明是 As₂O₃ 升华而非 As 蒸气）
- 更准确：煅烧 As₂O₃ → 升华→"飞"；加入炭 → 2As₂O₃ + 3C → 4As↑ + 3CO₂↑
