## 一、本节要解决的问题

1. 给你一组浓度-时间数据，怎么判断反应级数？——不只代公式，要选对工具。
2. 给你一个多步反应机理，怎么推导总速率方程？——稳态近似和平衡假设，什么时候用哪个？
3. 催化剂为什么能加速反应？——三条本质一次性讲清，不再死记"催化剂不改变平衡"。

### 学习目标

- 能写出零/一/二级反应的积分速率方程和线性关系
- 能利用半衰期与初始浓度的关系确定反应级数
- 能使用稳态近似从多步机理推导总速率方程（≤3 步）
- 能区分稳态近似和平衡假设的适用条件
- 能处理对峙反应 A ⇌ B 的动力学
- 能由 Arrhenius 推导表观活化能 Ea_app
- 能解释催化剂的三条本质

## 二、工具判断总流程图

```text
拿到动力学题目
    │
    ├─ 给 c-t 数据？
    │   ├─ 积分法：分别作 c~t / lnc~t / 1/c~t 图 → 哪个成直线？
    │   └─ 半衰期法：t₁/₂ 随 c₀ 怎么变？→ 反推 n
    │
    ├─ 给反应机理？
    │   ├─ 有快速可逆步骤 + 后续是 RDS？→ **平衡假设**
    │   └─ 中间体持续生成/消耗？→ **稳态近似**
    │
    ├─ 是可逆反应？
    │   └─ 对峙反应：v = k₁[A] - k₋₁[B]；平衡时 K = k₁/k₋₁
    │
    ├─ 问活化能？
    │   ├─ 单步：Arrhenius 两点法 ln(k₂/k₁) = -(Ea/R)(1/T₂-1/T₁)
    │   └─ 多步：表观活化能合成 Ea_app = f(Ea₁, Ea₂, Ea₋₁)
    │
    └─ 问催化剂？
        └─ 三条本质：改变机理 → 降低 Ea → 不改变 K
```

## 三、积分速率方程三级速查

### 3.1 公式并排

| 级数 | 微分式 | 积分式 | 线性关系 | k 的单位 |
|:---:|:---|:---|:---|:---:|
| **零级** | v = k | c = c₀ − kt | **c ~ t**（直线，斜率=−k） | mol·L⁻¹·s⁻¹ |
| **一级** | v = k[A] | ln(c₀/c) = kt | **ln c ~ t**（直线，斜率=−k） | s⁻¹ |
| **二级** | v = k[A]² | 1/c − 1/c₀ = kt | **1/c ~ t**（直线，斜率=+k） | L·mol⁻¹·s⁻¹ |

### 3.2 直线判断法操作

```
给一组 c-t 数据 →
    作 3 张图：c~t / lnc~t / 1/c~t →
        哪张图点最接近直线 → 就是哪个级数
```

> ⚡ **陷阱**：二级反应的 1/c~t 图斜率是 **+k**（正斜率！），不是 −k。零级和一级的斜率才是负的。

### 3.3 半衰期速判

$$ t_{1/2} \propto c_0^{1-n} $$

| 级数 | t₁/₂ 公式 | 与 c₀ 的关系 |
|:---:|:---|:---|
| **零级** | t₁/₂ = c₀/(2k) | **正比** c₀（c₀ 加倍→t₁/₂ 加倍） |
| **一级** | t₁/₂ = ln2/k | **无关**！（c₀ 怎么变，t₁/₂ 都不变） |
| **二级** | t₁/₂ = 1/(kc₀) | **反比** c₀（c₀ 加倍→t₁/₂ 减半） |

> ⚡ **一级反应 t₁/₂ 与 c₀ 无关**——这是最快捷的判断依据！看到"半衰期恒定"直接判一级。

## 四、稳态近似 vs 平衡假设——决策树

### 4.1 什么时候用哪个？

```
机理中有中间体 I
    │
    ├─ 第一步是快速可逆（k₋₁ ≫ k₂）且第二步是 RDS？
    │   └─ YES → **平衡假设**
    │       ① K = k₁/k₋₁ → [I] = K[A]
    │       ② v = k₂[I] = k₂K[A]（简化！）
    │
    └─ 中间体在多个步骤中持续生成和消耗？
        └─ **稳态近似**
            ① d[I]/dt = 生成速率 − 消耗速率 ≈ 0
            ② 解出 [I] = 表达式
            ③ 代入总速率方程
```

### 4.2 两种方法对比

| 特征 | 稳态近似 | 平衡假设 |
|:---|:---|:---|
| **条件** | 中间体浓度低且近似恒定 | 前一步快速可逆 + 后续是 RDS |
| **操作** | d[I]/dt ≈ 0 → 解方程 | K = k₁/k₋₁ → 代 [I] |
| **结果复杂度** | 较复杂（分母含 k₋₁+k₂） | 简化（分母消失） |
| **何时失效** | 中间体浓度不低时 | k₋₁ 不大时（预平衡不成立） |

### 4.3 经典例题：H₂ + I₂ → 2HI

```
机理：
  ① I₂ ⇌ 2I·        （快速可逆，K = k₁/k₋₁）
  ② I· + H₂ → HI + H· （慢，RDS）
  ③ H· + I₂ → HI + I· （快）

平衡假设推导：
  [I·] = √(K[I₂])
  v = k₂[I·][H₂] = k₂√K [H₂][I₂]^(1/2)

→ 实验测得 v = k[H₂][I₂] → 该机理被否决！

（正确机理涉及 I· + H₂ ⇌ HI + H· 快速可逆 + H· + I₂ → HI + I· RDS）
```

## 五、对峙反应 A ⇌ B

### 5.1 核心公式

$$ v = -\frac{d[A]}{dt} = k_1[A] - k_{-1}[B] $$

$$ \text{平衡时：}\ K = \frac{[B]_{eq}}{[A]_{eq}} = \frac{k_1}{k_{-1}} $$

### 5.2 由动力学数据求 k₁ 和 k₋₁

```
方法一（标准法）：
  ① 测平衡浓度 → K = [B]eq/[A]eq
  ② 测初始速率 → v₀ = k₁[A]₀ → k₁
  ③ k₋₁ = k₁/K

方法二（积分法，一级对峙）：
  ln([A]−[A]eq) 对 t 作图 → 斜率 = −(k₁+k₋₁)
  结合 K = k₁/k₋₁ → 解出 k₁ 和 k₋₁
```

## 六、表观活化能合成

### 6.1 核心概念

> 多步机理中，Arrhenius 公式的"Ea"是表观活化能——它是各基元步骤 Ea 的**代数和**。

### 6.2 以对峙反应 A ⇌ I → P 为例

$$ E_a^{\text{app}} = E_{a1} + E_{a2} - E_{a,-1} $$

| 极限情况 | 表观 Ea | 物理含义 |
|:---|:---|:---|
| k₋₁ ≫ k₂（预平衡） | Ea₁ + Ea₂ − Ea₋₁ | 三步都贡献 |
| k₂ ≫ k₋₁（第一步不可逆） | Ea₁ | 只有第一步影响 |
| 放热预平衡（Ea₋₁ 大） | **可能为负！** | 升温反而降速——少见但竞赛考过 |

### 6.3 Arrhenius 两点法（单步用）

$$ \ln\frac{k_2}{k_1} = -\frac{E_a}{R}\left(\frac{1}{T_2} - \frac{1}{T_1}\right) $$

> ⚠️ **符号验证**：升温（T₂>T₁）→ 1/T₂−1/T₁ 为负 → −Ea/R × 负 = 正 → ln(k₂/k₁)>0 → k₂>k₁ ✓

## 七、催化剂三条本质

| # | 本质 | 详解 |
|:---:|:---|:---|
| ① | **改变反应机理** | 催化剂参与反应→形成中间体→再生。不出现于总反应方程式，但出现在机理中 |
| ② | **降低活化能** | 新路径的 Ea' < 原路径 Ea → Arrhenius：k 指数级增大。**同等降低正逆反应 Ea** |
| ③ | **不改变平衡常数** | K = k₁/k₋₁，正逆速率同等增大→K 不变。**ΔG°=−RTlnK 也不变** |

> ⚡ **一句口诀**：催化剂改变路径、降低能垒、但不移动平衡。
>
> ⚠️ **常见混淆**：催化剂不改变 ΔG° 和 K，但能让你**更快到达**平衡位置。

### 均相 vs 多相催化速辨

| 特征 | 均相催化 | 多相催化 |
|:---|:---|:---|
| 催化剂相态 | 与反应物同相 | 与反应物异相（通常固体） |
| 机理 | 形成中间体配合物 | 表面吸附→反应→脱附 |
| 实例 | I⁻ 催化 H₂O₂ 分解 | Pt 催化加氢、Fe 催化合成氨 |
| 分离 | 难 | 易（过滤） |

## 八、常见易错点

| # | 易错点 | 正确认知 |
|:---:|:---|:---|
| 1 | 看到数据就代公式，不分题型 | 先判断：是 c-t 数据→积分法/半衰期法？是机理→稳态近似/平衡假设？是可逆→对峙反应？ |
| 2 | 反应式系数 = 反应级数 | 只有**基元反应**才能直接从系数写级数！非基元反应的级数必须由实验确定 |
| 3 | 半衰期法只背 t₁/₂ 公式 | 核心是 **t₁/₂ ∝ c₀¹⁻ⁿ**——记这一个关系就能判断任何级数 |
| 4 | 稳态近似和平衡假设混用 | 预平衡 + 后续 RDS → 平衡假设（简化）；中间体持续生成消耗 → 稳态近似 |
| 5 | 对峙反应中 K = k₁/k₋₁ 和热力学 K 是两个东西 | 它们是**同一个 K**！动力学平衡 = 热力学平衡 |
| 6 | Ea_app 就是最大那个 Ea | 表观活化能是**代数和**——可能小于任何单个 Ea，甚至可能为负 |
| 7 | 催化剂"降低反应热" | 催化剂不改变 ΔH！正逆反应活化能同等降低→ΔH = Ea(正)−Ea(逆) 不变 |

## 九、例题速练

### 例 1：半衰期法判断级数
> 某反应 A→P：c₀=0.1M 时 t₁/₂=100s；c₀=0.05M 时 t₁/₂=200s。判断级数。

**解**：c₀ 减半（0.1→0.05），t₁/₂ 加倍（100→200）
→ t₁/₂ ∝ 1/c₀ → **二级反应** ✓
→ k = 1/(t₁/₂ × c₀) = 1/(100×0.1) = 0.1 L·mol⁻¹·s⁻¹

### 例 2：稳态近似推导
> 机理：A ⇌ I → P（k₁=0.1, k₋₁=0.5, k₂=2.0 s⁻¹）。用稳态近似推导速率方程。

**解**：
d[I]/dt = k₁[A] − k₋₁[I] − k₂[I] ≈ 0
→ [I] = k₁[A]/(k₋₁+k₂) = 0.1[A]/(0.5+2.0) = 0.04[A]
→ v = k₂[I] = 2.0 × 0.04[A] = **0.08[A]**（表观一级，k_app=0.08 s⁻¹）

**对比**：若用平衡假设（k₋₁≫k₂ 不成立！）：[I]=K[A]=(0.1/0.5)[A]=0.2[A] → v=0.4[A]
→ 平衡假设高估了 5 倍！**因为 k₋₁ 不大，预平衡不成立。**

### 例 3：表观活化能
> 同上机理，Ea₁=50, Ea₋₁=30, Ea₂=20 kJ/mol。求表观 Ea。

**解**：极限情况判断——k₂=2.0 ≫ k₋₁=0.5？不！k₋₁+k₂=2.5，两者可比。
需要更精确处理（竞赛中通常取极限）。若 k₋₁≫k₂ 极限：
Ea_app = 50 + 20 − 30 = **40 kJ/mol**
