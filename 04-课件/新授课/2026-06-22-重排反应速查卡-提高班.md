# 重排反应 · 速查卡

> 基础班你学过几个重排的人名和产物——本卡把重排从**散乱的人名表**升级到**模板识别→迁移规则→产物核对**的四家族判断系统。

---

## 一、重排反应 · 四家族分类

```
重排反应 → 先分家族
    │
    ├─ 家族 1：碳正离子型重排
    │   └─ 缺电子中心生成 → 1,2-迁移 → 稳定化
    │   └─ W-M / Pinacol / Semipinacol / Tiffeneau-Demjanov / Wolff
    │
    ├─ 家族 2：向缺电子杂原子迁移
    │   └─ N/O 上构建离去基 → R 迁移到 N/O → 后续水解
    │   └─ Beckmann / Hofmann / Curtius / Lossen / Schmidt / BV
    │
    ├─ 家族 3：[3,3]-σ 协同迁移
    │   └─ 环状过渡态 → 同时断键成键 → 无离子中间体
    │   └─ Claisen / Cope / oxy-Cope
    │
    └─ 家族 4：阴离子/硅驱动与缩环型
        └─ 碳负离子或硅驱动迁移 + 环丙酮/Favorskii 缩环
        └─ Brook / Favorskii / 频哪醇逆重排
```

---

## 二、家族 1：碳正离子型重排

**统一机理**：缺电子中心生成 → 1,2-迁移（R/H/Ar 带电子对迁移到缺电子中心）→ 新 C⁺ 稳定化

| 反应 | 缺电子中心怎么来的 | 迁移基团 | 产物 | 碳数变化 |
|:---|:---|:---|:---|:---:|
| **W-M** | 醇+酸→失水→C⁺ | H/R/Ar | 重排烯烃/醇 | 不变 |
| **Pinacol** | 邻二醇+酸→C⁺-OH | R（与 OH 反式的一侧） | 酮 | 不变 |
| **Semipinacol** | 邻二醇（一侧有 LG）+酸 | R | 酮 | 不变 |
| **Tiffeneau-Demjanov** | 胺+NaNO₂→失N₂→C⁺ | R（环扩大） | 扩环酮 | 环 ↑1C |
| **Wolff** | 重氮酮→失N₂→卡宾 | R | 缩环/Wolff 重排产物 | 可能−1 |

**迁移能力顺序**：Ph > Me > H > R（3° > 2° > 1°）
- 苯基迁移最快（π 参与稳定过渡态）
- H 迁移比 Me 慢（但比大烷基快）

**Pinacol 的秘诀**：哪个 OH 走？→ 形成更稳定 C⁺ 的一侧 OH 被质子化→离去→另一侧 R 迁移过来。

---

## 三、家族 2：向缺电子杂原子迁移

**统一机理**：N/O 上构建好离去基 → R 从 C 迁移到 N/O → 异氰酸酯/酯 → 水解

| 反应 | N/O 的前体 | 碳数变化 | 产物 | 特别提示 |
|:---|:---|:---:|:---|:---|
| **Beckmann** | 肟+酸 | **不变** | 酰胺 | 迁移基团与 OH **反式**（反式迁移） |
| **Hofmann** | 酰胺+Br₂/OH⁻ | **−1** | 胺（少一个 C） | 异氰酸酯→水解→脱羧→胺 |
| **Curtius** | 酰基叠氮→失N₂ | **−1** | 胺 | 温和条件（与 Hofmann 比） |
| **Lossen** | 异羟肟酸+碱 | **−1** | 胺 | 类似 Hofmann |
| **Schmidt** | 羧酸+HN₃ | **−1** | 胺 | N₂ 是驱动力 |
| **B-V** | 酮+m-CPBA | **不变** | 酯 | O 插入→R 迁移到 O→酯。迁移能力：H > Ph > 3° > 2° > 1° > Me |

**Beckmann 迁移规则**：反式迁移！与 OH 在反式位置的 R 迁移→这是立体化学决定的。

---

## 四、家族 3：[3,3]-σ 协同迁移

```
Claisen 重排：烯丙基乙烯基醚→γ,δ-不饱和羰基化合物
  └─ [3,3] 环状过渡态 → 无离子中间体
  └─ 热反应 → 同面-同面 6e⁻（4n+2）→ 允许

Cope 重排：1,5-二烯→异构化的 1,5-二烯
  └─ 与 Claisen 同机理但无 O
  └─ oxy-Cope：3-羟基-1,5-二烯→烯醇→酮
  
Claisen vs Cope 速判：看到烯丙基+O→Claisen；纯碳体系→Cope
```

## 五、家族 4：阴离子/硅驱动与缩环型

| 反应 | 驱动因素 | 关键中间体 | 产物 |
|:---|:---|:---|:---|
| **Brook** | Si–O 键极强（比 Si–C 强） | O–Si→C–Si 迁移 | α-硅醇→硅醚 |
| **Favorskii** | α-卤代酮+碱 | **环丙酮**中间体 | 缩环酯（−1C） |
| **逆 Pinacol** | 酮+强还原条件 | C⁺-OH→脱氧→逆迁移 | 邻二醇（Pinacol 逆过程） |

**Brook 重排口诀**："硅是大号质子"→O–Si→C–Si，驱动力是 Si–O 键的强度。

## 六、四家族对比表

| 维度 | 家族 1（C⁺型） | 家族 2（杂原子） | 家族 3（[3,3]） | 家族 4（阴离子/硅） |
|:---|:---|:---|:---|:---|
| **中间体** | C⁺ | 异氰酸酯/四面体 | 无（协同TS） | C⁻ 或 硅迁移态 |
| **条件标志** | 酸催化 | 碱/酸/叠氮 | Δ（加热） | 碱/硅试剂 |
| **碳数变化** | 不变（Wolff 可能−1） | Beckmann 不变，其余−1 | 不变（重排） | Favorskii −1 |
| **立体化学** | C⁺ 外消旋可能 | Beckmann 反式迁移 | 椅式 TS→立体专一 | 取决于具体反应 |

## 七、常见易错点

| 错误 | 纠正 |
|:---|:---|
| Beckmann 迁移是等同两侧的 | 迁移基团必须与 OH **反式**——这是立体化学决定，产物完全取决于肟的 E/Z 构型 |
| Hofmann 和 Curtius 的产物一样 | 对——都是失羰基→得胺。但条件不同：Hofmann 用 Br₂/OH⁻（强碱），Curtius 用酰基叠氮（中性→可兼容敏感底物） |
| Claisen 重排是芳香重排 | Claisen 分脂肪族和芳香族两种：芳香 Claisen=烯丙基苯醚→邻烯丙基酚（环上 Claisen）；脂肪族 Claisen=本卡所指 |
| Pinacol 重排两侧对称→产物单一 | 不对称 Pinacol：生成更稳 C⁺ 的 OH 被质子化→反式 R 迁移。判断标准：哪侧的 C⁺ 更稳定？ |

## 八、例题速练

### 例 1：Pinacol 重排产物

**题**：Ph(Me)C(OH)–C(OH)(Me)Et + H⁺ → 主产物？

**速解**：
1. 两侧 OH 如何选择？→ 比较 C⁺ 稳定性：左侧 C⁺(Ph,Me)→苄基C⁺（稳定）；右侧 C⁺(Me,Et)→2° C⁺（一般）
2. 左侧 C⁺ 更稳→左侧 OH 质子化→离去→C⁺ 在左侧
3. 右侧 Me 反式迁移→产物 Ph(Me)(Et)C–C(=O)Me
4. **主产物**：酮在右侧，因为左侧 C⁺ 更稳定→左侧 OH 走→右侧 R 迁

### 例 2：Beckmann 构型决定

**题**：(E)-PhC(=NOH)CH₃ + H⁺ → ？

**速解**：
1. (E) 构型：OH 和 CH₃ 在同侧
2. 与 OH 反式的是 Ph→Ph 迁移到 N
3. 产物：CH₃C(=O)NHPh（N-苯基乙酰胺）

如果是 (Z) 构型→OH 和 Ph 在同侧→CH₃ 反式迁移→PhC(=O)NHCH₃
