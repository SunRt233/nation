# 方程式书写专项（基础班新授课）

## 1. 课堂引入与本节问题

### 1.1 竞赛中的方程式书写

化学竞赛中，方程式书写题通常占 **15-25分**。但很多学生会"知道反应但写不对"——产物错误、配平错误、介质判断错误、电荷不守恒……

**本节核心问题：**
> 如何建立系统方法，杜绝"凭感觉写方程式"？

**本节课目标：**
- 掌握"介质优先"的配平思维
- 熟练使用离子-电子法（半反应法）配平氧化还原方程式
- 掌握离子方程式书写规则

### 1.2 学习目标

- **本节讲到哪里为止**：
  - 能用“先介质、后氧化态、再半反应”的流程处理基础氧化还原配平
  - 能区分酸性/碱性/中性条件下常见氧化剂的典型还原产物
  - 能处理常见非氧化还原反应与基础离子方程式书写
  - 能判断一道题更适合离子-电子法还是观察法
- **后续深入学习**：
  - 多元素同时变价与复杂耦合配平 → 第二轮
  - 电化学电池总反应与半反应联立书写 → 与电化学专题联动深化
  - 有机氧化还原与非水体系配平 → 第三轮/专项训练

### 1.3 本节使用方式

- 这张页的课堂价值在于把“会背反应”升级成“会按流程写反应”。建议全程反复强调统一步骤：先判介质，再判氧化态，再拆半反应，最后查原子、电荷、得失电子是否同时守恒。
- 讲题时不要急着给答案，建议固定追问三句：
  1. 谁被氧化、谁被还原？
  2. 题目环境是酸性、碱性还是中性？
  3. 这题适合离子-电子法，还是只需观察法/守恒法？
- 学生课后复盘时，最好自己整理一张“常见氧化剂在酸/碱/中性介质中的还原产物表”，再配一张“离子方程式可拆/不可拆清单”。

## 2. 知识内容

### 2.1 氧化态判定与介质判断

#### 2.1.1 氧化态判定（复习）

| 规则 | 内容 |
|-----|------|
| 规则1 | 单质中元素氧化态为0 |
| 规则2 | 离子化合物中：氧化态 = 离子电荷 |
| 规则3 | 共价化合物中：电负性大的原子得电子 |
| 规则4 | H通常为+1（金属氢化物中为-1） |
| 规则5 | O通常为-2（过氧化物中为-1，超氧化物中为-1/2，OF₂中为+2） |

#### 2.1.2 介质判断原则

**介质判断是配平的第一步，决定产物形态。**

| 判断依据 | 酸性介质 | 碱性介质 | 中性介质 |
|---------|---------|---------|---------|
| 题目明确说明 | "酸性条件"、"稀H₂SO₄" | "碱性条件"、"NaOH" | "中性"、"水溶液" |
| 反应物特征 | 含H⁺、强酸 | 含OH⁻、强碱 | 无强酸强碱 |
| 产物提示 | 生成CO₂ | 生成CO₃²⁻ | 视情况而定 |

> **核心纪律（必须牢记）：**
> - **碱性条件绝不允许出现游离H⁺**
> - **酸性条件绝不允许出现游离OH⁻**

### 2.2 离子-电子法配平步骤

#### 2.2.1 配平口诀

```
介质定 → 产物定 → 原子平 → 电荷平
```

#### 2.2.2 详细步骤

**Step 1：写出半反应**
分别写出氧化半反应和还原半反应的骨架。

**Step 2：配平原子**
- 用H₂O配平O原子
- 酸性条件：用H⁺配平H原子
- 碱性条件：用OH⁻配平H原子（或先在酸性条件下配平，再加OH⁻中和）

**Step 3：配平电荷**
用电子（e⁻）配平两边的电荷。

**Step 4：合并半反应**
使氧化半反应失去的电子数 = 还原半反应得到的电子数，然后相加消去电子。

**Step 5：检查守恒**
- 原子守恒：左右两边各元素原子数相等
- 电荷守恒：左右两边总电荷相等

#### 2.2.3 酸性介质示例：MnO₄⁻ → Mn²⁺

```
① 写出骨架：
   MnO₄⁻ → Mn²⁺

② 配平Mn：已平

③ 配平O（加H₂O）：
   MnO₄⁻ → Mn²⁺ + 4H₂O

④ 配平H（加H⁺）：
   MnO₄⁻ + 8H⁺ → Mn²⁺ + 4H₂O

⑤ 配平电荷（加e⁻）：
   MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
   （左边电荷：-1 + 8 - 5 = +2，右边电荷：+2，守恒）
```

#### 2.2.4 碱性介质处理技巧

**方法：先在酸性条件下配平，再转化为碱性条件。**

以MnO₄⁻ → MnO₄²⁻为例：

```
① 酸性条件下配平：
   MnO₄⁻ + e⁻ → MnO₄²⁻
   （已平，无需H⁺和H₂O）

② 由于无H⁺参与，直接就是碱性条件下的式子
```

再以MnO₄⁻ → MnO₂为例：

```
① 酸性条件下配平：
   MnO₄⁻ + 4H⁺ + 3e⁻ → MnO₂ + 2H₂O

② 两边各加4OH⁻（中和H⁺）：
   MnO₄⁻ + 4H⁺ + 4OH⁻ + 3e⁻ → MnO₂ + 2H₂O + 4OH⁻

③ 合并H⁺ + OH⁻ → H₂O：
   MnO₄⁻ + 4H₂O + 3e⁻ → MnO₂ + 2H₂O + 4OH⁻

④ 化简（两边减去2H₂O）：
   MnO₄⁻ + 2H₂O + 3e⁻ → MnO₂ + 4OH⁻
```

### 2.3 常见氧化剂在不同介质中的还原产物

| 氧化剂 | 酸性介质 | 中性/弱碱性 | 强碱性介质 |
|-------|---------|-----------|-----------|
| KMnO₄ | Mn²⁺（无色） | MnO₂↓（棕黑） | MnO₄²⁻（绿色） |
| K₂Cr₂O₇ | Cr³⁺（绿色） | — | — |
| HNO₃(浓) | NO₂ | — | — |
| HNO₃(稀) | NO | — | — |
| H₂SO₄(浓) | SO₂ | — | — |
| H₂O₂ | H₂O | — | — |
| Cl₂ | Cl⁻ | — | Cl⁻ / ClO⁻（歧化） |

**记忆要点：**
- MnO₄⁻：酸性降5价（Mn⁷⁺→Mn²⁺），中性降3价（Mn⁷⁺→Mn⁴⁺），碱性降1价（Mn⁷⁺→Mn⁶⁺）
- Cr₂O₇²⁻：只在酸性条件下作氧化剂（Cr⁶⁺→Cr³⁺，每个Cr得3e⁻，整体得6e⁻）

>| 氧化剂 | 条件 | 还原产物 | 得电子数 |

>| MnO₄⁻ | 中性/弱碱 | MnO₂ | 3e⁻ |
>| MnO₄⁻ | 强碱性 | MnO₄²⁻ | 1e⁻ |
>| Cr₂O₇²⁻ | 酸性 | Cr³⁺ | 6e⁻（整体） |
>| HNO₃(浓) | — | NO₂ | 1e⁻ |
>| HNO₃(稀) | — | NO | 3e⁻ |
>| H₂O₂ | — | H₂O | 2e⁻ |
>| Fe³⁺ | — | Fe²⁺ | 1e⁻ |
>| Cu²⁺ | — | Cu | 2e⁻ |
>| Ag⁺ | — | Ag | 1e⁻ |

### 2.4 非氧化还原反应观察法

对于非氧化还原反应（复分解反应、酸碱中和、沉淀反应等），使用观察法配平：

1. **写出反应物和产物的化学式**
2. **从最复杂的化合物开始**，配平其组成元素
3. **逐步配平其他元素**
4. **检查守恒**

**示例：**
$$\text{CaCO}_3(\text{s}) + 2\text{HCl}(\text{aq}) \rightarrow \text{CaCl}_2(\text{aq}) + \text{H}_2\text{O}(\text{l}) + \text{CO}_2(\text{g})$$

### 2.5 离子方程式书写规则

#### 2.5.1 拆分原则

| 物质类型 | 处理方式 | 示例 |
|---------|---------|------|
| 强酸 | 拆成H⁺和酸根 | HCl → H⁺ + Cl⁻ |
| 强碱 | 拆成金属阳离子和OH⁻ | NaOH → Na⁺ + OH⁻ |
| 可溶性盐 | 拆成离子 | NaCl → Na⁺ + Cl⁻ |
| 弱酸 | 保留化学式 | CH₃COOH |
| 弱碱 | 保留化学式 | NH₃·H₂O |
| 难溶物 | 保留化学式 | CaCO₃、BaSO₄ |
| 单质 | 保留化学式 | Fe、Cl₂ |
| 氧化物 | 保留化学式 | CO₂、MnO₂ |
| 水 | 保留化学式 | H₂O |

#### 2.5.2 离子方程式书写步骤

1. 写出完整的化学方程式
2. 将可溶性强电解质拆成离子
3. 删去两边相同的离子（旁观离子）
4. 检查原子和电荷守恒

**示例：**

化学方程式：
$$\text{AgNO}_3(\text{aq}) + \text{NaCl}(\text{aq}) \rightarrow \text{AgCl}(\text{s}) + \text{NaNO}_3(\text{aq})$$

拆分为离子：
$$\text{Ag}^+ + \text{NO}_3^- + \text{Na}^+ + \text{Cl}^- \rightarrow \text{AgCl}(\text{s}) + \text{Na}^+ + \text{NO}_3^-$$

删去旁观离子（Na⁺、NO₃⁻）：
$$\text{Ag}^+(\text{aq}) + \text{Cl}^-(\text{aq}) \rightarrow \text{AgCl}(\text{s})$$

## 3. 课堂例题

### 例题1：KMnO₄在不同介质中氧化Na₂SO₃（⭐⭐）

**题目：** 写出KMnO₄在不同介质中氧化Na₂SO₃的离子方程式。

**解答：**

**（1）酸性介质：**

还原半反应：$\text{MnO}_4^- + 8\text{H}^+ + 5\text{e}^- \rightarrow \text{Mn}^{2+} + 4\text{H}_2\text{O}$

氧化半反应：$\text{SO}_3^{2-} + \text{H}_2\text{O} \rightarrow \text{SO}_4^{2-} + 2\text{H}^+ + 2\text{e}^-$

合并（还原×2，氧化×5）：

$$2\text{MnO}_4^-(\text{aq}) + 5\text{SO}_3^{2-}(\text{aq}) + 6\text{H}^+(\text{aq}) \rightarrow 2\text{Mn}^{2+}(\text{aq}) + 5\text{SO}_4^{2-}(\text{aq}) + 3\text{H}_2\text{O}(\text{l})$$

**（2）中性介质：**

$$2\text{MnO}_4^-(\text{aq}) + 3\text{SO}_3^{2-}(\text{aq}) + \text{H}_2\text{O}(\text{l}) \rightarrow 2\text{MnO}_2(\text{s}) + 3\text{SO}_4^{2-}(\text{aq}) + 2\text{OH}^-(\text{aq})$$

**（3）碱性介质：**

$$2\text{MnO}_4^-(\text{aq}) + \text{SO}_3^{2-}(\text{aq}) + 2\text{OH}^-(\text{aq}) \rightarrow 2\text{MnO}_4^{2-}(\text{aq}) + \text{SO}_4^{2-}(\text{aq}) + \text{H}_2\text{O}(\text{l})$$

**要点：** 同一氧化剂在不同介质中还原产物不同，强调"介质优先"原则。

---

### 例题2：Cr₂O₇²⁻酸性条件下氧化Fe²⁺（⭐⭐）

**题目：** 写出酸性条件下K₂Cr₂O₇氧化FeSO₄的离子方程式。

**解答：**

还原半反应：$\text{Cr}_2\text{O}_7^{2-} + 14\text{H}^+ + 6\text{e}^- \rightarrow 2\text{Cr}^{3+} + 7\text{H}_2\text{O}$

氧化半反应：$\text{Fe}^{2+} \rightarrow \text{Fe}^{3+} + \text{e}^-$

合并（还原×1，氧化×6）：

$$\text{Cr}_2\text{O}_7^{2-}(\text{aq}) + 6\text{Fe}^{2+}(\text{aq}) + 14\text{H}^+(\text{aq}) \rightarrow 2\text{Cr}^{3+}(\text{aq}) + 6\text{Fe}^{3+}(\text{aq}) + 7\text{H}_2\text{O}(\text{l})$$

**要点：**
- 1个Cr₂O₇²⁻含2个Cr，每个Cr从+6降到+3，得3e⁻，**整体得6e⁻**
- 注意系数关系：Cr₂O₇²⁻ : Fe²⁺ = 1 : 6

---

### 例题3：Al与NaOH溶液反应（⭐⭐）

**题目：** 写出Al与NaOH溶液反应的化学方程式和离子方程式。

**解答：**

**化学方程式：**

$$2\text{Al}(\text{s}) + 2\text{NaOH}(\text{aq}) + 6\text{H}_2\text{O}(\text{l}) \rightarrow 2\text{Na}[\text{Al(OH)}_4](\text{aq}) + 3\text{H}_2(\text{g})$$

**离子方程式：**

$$2\text{Al}(\text{s}) + 2\text{OH}^-(\text{aq}) + 6\text{H}_2\text{O}(\text{l}) \rightarrow 2[\text{Al(OH)}_4]^-(\text{aq}) + 3\text{H}_2(\text{g})$$

**要点：**
- Al是两性金属，可与强碱反应
- 碱性条件下产物是**四羟基合铝酸盐** [Al(OH)₄]⁻，不是AlO₂⁻（偏铝酸根是简化写法）
- Al从0价升到+3价，H从+1价降到0价

---

### 例题4：Cl₂与NaOH反应（歧化反应）（⭐⭐）

**题目：** 写出Cl₂与不同浓度NaOH溶液反应的方程式。

**解答：**

**（1）冷稀碱条件下：**

$$\text{Cl}_2(\text{g}) + 2\text{OH}^-(\text{aq}) \rightarrow \text{Cl}^-(\text{aq}) + \text{ClO}^-(\text{aq}) + \text{H}_2\text{O}(\text{l})$$

**（2）热浓碱条件下：**

$$3\text{Cl}_2(\text{g}) + 6\text{OH}^-(\text{aq}) \xrightarrow{\Delta} 5\text{Cl}^-(\text{aq}) + \text{ClO}_3^-(\text{aq}) + 3\text{H}_2\text{O}(\text{l})$$

**要点：**
- 歧化反应中，同一元素（Cl）既被氧化又被还原
- 温度影响产物：冷稀碱→ClO⁻（Cl从0到+1），热浓碱→ClO₃⁻（Cl从0到+5）
- 这是卤素在碱性条件下的特征反应

---

### 例题5（拓展）：Zn与NaOH反应（⭐⭐⭐）

**题目：** 写出Zn与NaOH溶液反应的离子方程式。

**解答：**

$$\text{Zn}(\text{s}) + 2\text{OH}^-(\text{aq}) + 2\text{H}_2\text{O}(\text{l}) \rightarrow [\text{Zn(OH)}_4]^{2-}(\text{aq}) + \text{H}_2(\text{g})$$

**要点：** Zn也是两性金属，与Al类似，产物为四羟基合锌酸盐。

---

### 例题6（拓展）：Cu与浓HNO₃反应（⭐⭐⭐）

**题目：** 写出Cu与浓HNO₃反应的化学方程式。

**解答：**

$$\text{Cu}(\text{s}) + 4\text{HNO}_3(\text{浓})(\text{aq}) \rightarrow \text{Cu(NO}_3)_2(\text{aq}) + 2\text{NO}_2(\text{g}) + 2\text{H}_2\text{O}(\text{l})$$

**离子方程式：**

$$\text{Cu}(\text{s}) + 4\text{H}^+(\text{aq}) + 2\text{NO}_3^-(\text{aq}) \rightarrow \text{Cu}^{2+}(\text{aq}) + 2\text{NO}_2(\text{g}) + 2\text{H}_2\text{O}(\text{l})$$

**要点：** 浓HNO₃的还原产物是NO₂（红棕色气体），稀HNO₃的还原产物是NO（无色气体）。

## 5. 本节形成的核心结论

1. **介质判断是配平的第一步**，决定产物形态。同一氧化剂在不同介质中还原产物可能完全不同。

2. **离子-电子法四步口诀：介质定 → 产物定 → 原子平 → 电荷平**。按步骤进行，杜绝"凑系数"。

3. **碱性条件绝不允许出现游离H⁺，酸性条件绝不允许出现游离OH⁻**。这是配平的"红线"。

4. **歧化反应中同一元素既被氧化又被还原**，需要分别写出氧化和还原半反应。

5. **配平后必须检查原子和电荷守恒**。这是验证配平是否正确的最后关卡。

## 6. 易错点预警

| 错误想法 | 为什么错 | 正确理解 |
|---------|---------|---------|
| "配平就是凑系数" | 缺乏系统方法，容易出错，且无法验证 | 使用离子-电子法，按"介质定→产物定→原子平→电荷平"四步进行 |
| "碱性条件也可以写H⁺" | 违反基本化学事实，碱性溶液中H⁺浓度极低 | 碱性条件只能用OH⁻和H₂O配平，绝不允许出现游离H⁺ |
| "Cr₂O₇²⁻得3个电子" | 忽略了1个Cr₂O₇²⁻含2个Cr原子 | Cr₂O₇²⁻整体得6e⁻（每个Cr从+6到+3，得3e⁻，2个Cr共得6e⁻） |
| "MnO₄⁻总是还原成Mn²⁺" | 忽略了介质对产物的影响 | 酸性→Mn²⁺，中性→MnO₂，碱性→MnO₄²⁻，必须根据介质判断 |
| "Al与NaOH反应生成AlO₂⁻" | AlO₂⁻是简化写法，不够准确 | 正确产物是[Al(OH)₄]⁻（四羟基合铝酸根） |
| "离子方程式中所有物质都拆" | 不了解拆分规则 | 只有强酸、强碱、可溶性盐才能拆，其余保留化学式 |

### 主板书框架

```
方程式书写专项（1）——氧化还原配平

一、配平口诀
   介质定 → 产物定 → 原子平 → 电荷平

二、MnO₄⁻在不同介质中的还原产物
   ┌─────────┬─────────┬─────────┐
   │  酸性   │  中性   │  碱性   │
   ├─────────┼─────────┼─────────┤
   │  Mn²⁺  │ MnO₂↓  │ MnO₄²⁻ │
   │ 降5价  │ 降3价  │ 降1价  │
   └─────────┴─────────┴─────────┘

三、离子-电子法步骤
   1. 写出半反应
   2. 配平原子（H₂O配O，H⁺/OH⁻配H）
   3. 配平电荷（加e⁻）
   4. 合并半反应
   5. 检查守恒

四、核心纪律
   碱性条件：禁止出现H⁺
   酸性条件：禁止出现OH⁻

五、Cr₂O₇²⁻氧化Fe²⁺
   Cr₂O₇²⁻ + 6Fe²⁺ + 14H⁺ → 2Cr³⁺ + 6Fe³⁺ + 7H₂O
   （注意：1个Cr₂O₇²⁻整体得6e⁻）
```
