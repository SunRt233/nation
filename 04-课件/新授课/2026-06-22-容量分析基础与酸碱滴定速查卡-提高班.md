# 容量分析基础与酸碱滴定 · 速查卡

> 基础班你背过"SP=计量点、EP=终点、Et=终点误差"——本卡把酸碱滴定从**术语记忆**升级到**基准物质→指示剂推导→滴定曲线直觉→林邦误差→双指示剂法**的定量计算链。

---

## 一、基准物质与标准溶液

```
基准物质五项要求：
  ├─ 组成确定（与化学式一致，如 Na₂CO₃ 而非 NaOH）
  ├─ 纯度 ≥ 99.9%
  ├─ 性质稳定（不吸湿、不风化、不被空气氧化）
  ├─ 定量反应（与待标定物质按确定计量比反应）
  └─ 摩尔质量大（减小称量相对误差！）

常见基准物质分类：
  标定酸：Na₂CO₃（无水）、Na₂B₄O₇·10H₂O（硼砂，M=381.4→称量误差小！）
  标定碱：KHP（邻苯二甲酸氢钾）、H₂C₂O₄·2H₂O
  标定 EDTA：Zn（纯金属）、CaCO₃
  标定 KMnO₄：Na₂C₂O₄、H₂C₂O₄·2H₂O
  标定 AgNO₃：NaCl（基准）

标准溶液 = 已知准确浓度的溶液
  ├─ 直接配制法：基准物质 → 定容（准确称量 + 容量瓶）
  └─ 间接配制法（标定）：近似配制 → 用基准物质标定 → 计算准确浓度
```

## 二、酸碱指示剂 · 从背到推导

```
指示剂为什么是区间而非一个点？

  HIn ⇌ H⁺ + In⁻（指示剂自身是弱酸/弱碱）
    酸式色              碱式色

  Ka = [H⁺][In⁻]/[HIn] → [H⁺] = Ka·[HIn]/[In⁻]

  人眼分辨极限：
    [In⁻]/[HIn] ≥ 10 → 看到碱式色 → pH ≥ pKa + 1
    [In⁻]/[HIn] ≤ 0.1 → 看到酸式色 → pH ≤ pKa − 1

  变色范围：pH = pKa ± 1（不是恰好 pKa！）
```

| 指示剂 | pKa | 变色范围 | 酸式色→碱式色 | 适用场景 |
|:---|:---:|:---|:---|:---|
| **甲基橙** | 3.4 | 3.1–4.4 | 红→黄 | 强酸强碱滴定 |
| **甲基红** | 5.0 | 4.4–6.2 | 红→黄 | 强酸强碱 / 弱碱滴定 |
| **酚酞** | 9.1 | 8.0–10.0 | 无色→粉红 | 弱酸强碱滴定 |
| **百里酚酞** | 10.0 | 9.4–10.6 | 无色→蓝 | 多元酸第二计量点 |

**选择铁律**：变色范围必须落入滴定的 pH 突跃范围！

## 三、滴定曲线 · 三种对比

```
三种滴定曲线的核心差异：

  强酸强碱（0.1M NaOH → 0.1M HCl）：
    曲线形状：起始低→平缓→突跃(4.3→9.7)→平缓
    计量点 pH = 7.00
    甲基橙/酚酞均可——突跃范围够宽！

  弱酸强碱（0.1M NaOH → 0.1M HAc）：
    曲线形状：起始较高→缓冲区平缓→突跃窄(7.7→9.7)→平缓
    计量点 pH ≈ 8.72（NaAc 水解！）
    只能选酚酞——甲基橙的变色范围（3.1-4.4）完全在突跃之外！

  多元酸（H₃PO₄）：
    第一计量点 pH ≈ 4.70（甲基橙/甲基红）
    第二计量点 pH ≈ 9.66（百里酚酞）
    分步滴定条件：cKa₁ ≥ 10⁻⁸ 且 Ka₁/Ka₂ ≥ 10⁵
```

### 浓度对突跃的影响

```
浓度增大 10 倍 → 突跃范围大约增加 2 个 pH 单位

  0.01 M：突跃 5.3 → 8.7（仅酚酞可用）
  0.1 M： 突跃 4.3 → 9.7（甲基橙/酚酞都可用）
  1 M：   突跃 3.3 → 10.7（更宽）

→ 浓度越低，突跃越窄，指示剂选择越受限！
```

## 四、终点误差 · 林邦公式

```
Et = (终点 − 计量点) / 计量点

弱酸滴定的林邦误差公式：
  Et = (10^ΔpH − 10^−ΔpH) · √(Kw / Csp·Ka)

  其中 ΔpH = pH_ep − pH_sp（终点 pH − 计量点 pH）

ΔpH 符号规则（最易错！）：
  ΔpH > 0 → 滴定过头 → Et > 0（正误差）
  ΔpH < 0 → 滴定不够 → Et < 0（负误差）

实例：0.1M NaOH 滴定 0.1M HAc，SP pH=8.72，酚酞终点 pH=9.0
  → ΔpH = +0.28 → 滴定略微过头 → Et 为正
```

## 五、双指示剂法 · V1/V2 判组成

```
原理：Na₂CO₃ 是二元碱，两个终点对应两步反应

  第一终点（酚酞变色，pH≈8.3）：
    CO₃²⁻ + H⁺ → HCO₃⁻（只走到一半！）

  第二终点（甲基橙变色，pH≈3.8）：
    HCO₃⁻ + H⁺ → H₂CO₃ → CO₂↑ + H₂O

  V1 = 酚酞终点消耗 HCl 体积（对应 CO₃²⁻ → HCO₃⁻）
  V2 = 甲基橙终点额外消耗 HCl 体积（对应 HCO₃⁻ → CO₂）

V1/V2 判组成速查：
  ├─ V1 = V2 ≠ 0 → 纯 Na₂CO₃
  ├─ V1 > V2 → Na₂CO₃ + NaOH（V1 = NaOH全量 + Na₂CO₃一半）
  ├─ V1 < V2 → Na₂CO₃ + NaHCO₃（V2 = Na₂CO₃一半 + NaHCO₃全量）
  ├─ V1 = 0, V2 > 0 → 纯 NaHCO₃
  └─ V2 = 0, V1 > 0 → 纯 NaOH
```

## 六、常见易错点

| 错误 | 纠正 |
|:---|:---|
| "计量点 pH 一定是 7" | 强酸强碱才是 7！弱酸强碱计量点偏碱（NaAc 水解→pH≈8.72） |
| "指示剂变色发生在 pKa" | 变色是**区间** pH=pKa±1——人眼看到的是 [In⁻]/[HIn] 从 10:1 到 1:10 |
| "弱酸滴定也可以用甲基橙" | 弱酸滴定突跃在碱性区（7.7-9.7）→ 甲基橙变色在酸性区 → **不行！** |
| "双指示剂法第一终点 = 全中和" | 酚酞终点 CO₃²⁻ 只走到 HCO₃⁻——**只中和了一半！** |
| "基准物质只要纯度高就行" | 还需组成确定、性质稳定、摩尔质量大——NaOH 纯度高但不能做基准物质（吸湿+吸收CO₂） |
| "浓度增大时突跃变小" | **浓度越大突跃越大**——因为过量一滴导致的 pH 变化更剧烈 |
| "V1 < V2 一定是 Na₂CO₃+NaHCO₃" | 也可能有 Na₂CO₃+惰性杂质——需结合质量和摩尔质量验证 |

## 七、例题速练

### 例 1：基准物质判断

**题**：NaOH、Na₂CO₃（无水）、KHP、浓 HCl 中哪些能做基准物质？分别说明理由。

**速解**：
- NaOH：❌ 吸湿+吸收 CO₂→组成不确定
- Na₂CO₃（无水）：✅ 组成确定、稳定、M=106→摩尔质量较大
- KHP：✅ 组成确定、稳定、M=204.2→摩尔质量大（优选！）
- 浓 HCl：❌ 挥发性—组成不确定、浓度不稳定

### 例 2：指示剂选择

**题**：0.1M NaOH 滴定 0.1M HCOOH（Ka=1.8×10⁻⁴）→ 计量点 pH≈8.3，应选什么指示剂？

**速解**：计量点 pH=8.3，突跃约 7.5→9.5。酚酞（8.0-10.0）在突跃范围内 ✅。甲基橙（3.1-4.4）在突跃之外 ❌。

### 例 3：双指示剂法判组成

**题**：混合碱 0.5000g，酚酞终点消耗 0.2000M HCl 15.00mL，甲基橙终点再消耗 25.00mL。判断组成。

**速解**：V1=15.00 < V2=25.00 → Na₂CO₃ + NaHCO₃。Na₂CO₃ 对应 2×V1=30.00mL（CO₃²⁻→HCO₃⁻ 和 HCO₃⁻→CO₂），NaHCO₃ 对应 V2−V1=10.00mL。
