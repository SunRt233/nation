# 银金汞钼钨 · 速查卡

> 基础班你背过"Ag⁺=氧化性适中/Au=王水可溶/Hg=唯一液态金属"——本卡把 Ag/Au/Hg/Mo/W 从**孤立事实**升级到**歧化对比+配位驱动氧化还原+多酸定性概念**的判断系统。

---

## 一、银 · 分步配位

```
Ag⁺ 的核心化学：氧化性适中 + 难溶盐 + 分步配位

  E°(Ag⁺/Ag) = +0.80V —— 介于 Cu²⁺/Cu（+0.34V）和 Hg²⁺/Hg（+0.85V）之间
```

### 1.1 Ag⁺ 与 NH₃ 的分步反应

```
Ag⁺ + NH₃·H₂O（少量）→ AgOH↓（白色）→ 立即分解→Ag₂O↓（棕色）
AgOH/Ag₂O + NH₃·H₂O（过量）→ [Ag(NH₃)₂]⁺ + OH⁻（无色溶液）

与 Cu²⁺ 对比：
  Cu²⁺ + NH₃（少量）→ Cu(OH)₂↓（蓝色）→ 过量 NH₃ → [Cu(NH₃)₄]²⁺（深蓝）
  Ag⁺ + NH₃（少量）→ AgOH↓ → Ag₂O↓ → 过量 NH₃ → [Ag(NH₃)₂]⁺（无色）
```

### 1.2 卤化银 · 感光性

| AgX | 颜色 | 溶解性 | 特殊性质 |
|:---|:---|:---|:---|
| **AgF** | 无色 | **易溶！** | 唯一可溶卤化银（F⁻ 水合能大+晶格能较小） |
| **AgCl** | 白色 | 难溶，溶于 NH₃ | [Ag(NH₃)₂]⁺ 可溶 |
| **AgBr** | 浅黄 | 更难溶，微溶于 NH₃ | **见光分解**：2AgBr → 2Ag + Br₂（照相底片原理） |
| **AgI** | 黄色 | 最難溶，不溶于 NH₃ | 溶于 S₂O₃²⁻/CN⁻；人工降雨晶核 |

### 1.3 银镜反应

```
RCHO + 2[Ag(NH₃)₂]⁺ + 3OH⁻ → RCOO⁻ + 2Ag↓ + 4NH₃ + 2H₂O

条件：碱性 + 过量 NH₃——试管内壁形成银镜
用途：醛基检验（Tollen 试剂）
注意：银氨溶液不可久置——生成爆炸性 Ag₃N！
```

## 二、金 · 王水原理

```
Au 为什么在普通酸中不溶？

  Au → Au³⁺ + 3e⁻：E° = +1.50V（极高的氧化电势！）
  → HNO₃（+0.96V）不足以氧化 Au

王水的秘密：HNO₃ 氧化 + HCl 配位降低电势

  Au + HNO₃ + 4HCl → HAuCl₄ + NO↑ + 2H₂O

  机制分解：
    HNO₃ 提供氧化能力：Au → Au³⁺ + 3e⁻
    HCl 提供 Cl⁻ 配位：Au³⁺ + 4Cl⁻ → [AuCl₄]⁻
    → Cl⁻ 配位降低 Au³⁺/Au 的有效电势 → 反应可以进行！

  一句话：HNO₃ 负责"推一把"（氧化），HCl 负责"接住"（配位）
```

| 金化合物 | 氧化态 | 关键信息 |
|:---|:---:|:---|
| **HAuCl₄** | +3 | 王水溶金产物——Au(III) 以 [AuCl₄]⁻ 存在 |
| **AuCl₃** | +3 | 二聚体 Au₂Cl₆（Cl 桥连） |
| **AuCN** | +1 | 氰化法提金：4Au + 8CN⁻ + O₂ + 2H₂O → 4[Au(CN)₂]⁻ + 4OH⁻ |

## 三、汞 · 不歧化的反例

```
Hg₂²⁺ 不歧化 vs Cu⁺ 歧化——过渡金属最经典对比！

  Cu⁺：2Cu⁺ → Cu²⁺ + Cu（K≈10⁶，强烈歧化！）
  Hg₂²⁺：Hg₂²⁺ 稳定存在，不歧化为 Hg²⁺ + Hg

  热力学解释（电势图）：
    酸性：Hg²⁺ ──+0.92V──→ Hg₂²⁺ ──+0.79V──→ Hg
    因为 E°(右) < E°(左) → Hg₂²⁺ 不歧化！

    Cu²⁺ ──+0.15V──→ Cu⁺ ──+0.52V──→ Cu
    因为 E°(右) > E°(左) → Cu⁺ 歧化！

  口诀：铜歧化、汞不歧化——电势图一看便知
```

### 3.1 汞的核心化合物

| 化合物 | Hg 价态 | 关键性质 |
|:---|:---:|:---|
| **Hg₂Cl₂**（甘汞） | +1 | 白色沉淀，不歧化——用作参比电极（甘汞电极） |
| **HgCl₂**（升汞） | +2 | 白色，剧毒——可被 SnCl₂ 还原：2HgCl₂ + SnCl₂ → Hg₂Cl₂↓(白) + SnCl₄ |
| **Hg₂Cl₂ + NH₃** | +1 | Hg₂Cl₂ + 2NH₃ → Hg↓(黑) + Hg(NH₂)Cl↓(白)——**变黑**！推断题经典信号 |
| **HgS**（辰砂） | +2 | 黑色沉淀——只溶于王水或 Na₂S 溶液 |
| **HgO** | +2 | 红色（粒状）或黄色（粉状）——加热分解：2HgO → 2Hg + O₂ |

**Hg₂Cl₂ + NH₃ 变黑**：这是区分 Hg₂²⁺ 和 Hg²⁺ 的特征反应——Hg₂Cl₂(白) + NH₃ → Hg(黑，金属汞) + Hg(NH₂)Cl(白，氨基氯化汞)。

### 3.2 汞的毒性

```
两种中毒途径：
  ├─ Hg 蒸气：液态汞在室温下饱和蒸气压远超安全标准
  │   └─ 体温计打破 → 撒硫粉（Hg + S → HgS，降低蒸气压）
  └─ 有机汞：CH₃Hg⁺ 脂溶性 → 生物富集 → 血脑屏障穿透
      └─ 水俣病（Minamata disease）：甲基汞中毒的经典案例
```

## 四、钼钨 · 多酸化学

```
Mo 和 W 的最重要特征：形成多酸（同多酸 + 杂多酸）

  同多酸：同种含氧酸缩合
    例：七钼酸铵 (NH₄)₆Mo₇O₂₄（Mo₇O₂₄⁶⁻）
       偏钨酸铵 (NH₄)₁₀W₁₂O₄₁（W₁₂O₄₁¹⁰⁻）

  杂多酸：不同种含氧酸缩合
    例：磷钼酸 H₃PMo₁₂O₄₀（PW₁₂O₄₀³⁻——Keggin 结构）
       硅钨酸 H₄SiW₁₂O₄₀

  判断口诀：只含一种中心原子→同多酸；含两种以上中心原子→杂多酸
```

### 4.1 Mo/W 的核心化合物与用途

| 化合物 | 关键性质 | 应用 |
|:---|:---|:---|
| **MoS₂** | 层状结构（类似石墨） | 加氢脱硫（HDS）催化剂、固体润滑剂 |
| **MoO₃** | 酸性氧化物（与 CrO₃ 同族） | Mo 的冶炼中间体 |
| **WO₃** | 黄色 | 电致变色材料 |
| **WC** | 极硬 | 硬质合金（切削工具） |
| **磷钼酸铵** | (NH₄)₃PMo₁₂O₄₀ 黄色沉淀 | PO₄³⁻ 的检测——生成黄色沉淀→还原→钼蓝 |

**Mo/W 与 Cr 的同族对比**：

```
CrO₄²⁻（黄）/ Cr₂O₇²⁻（橙）——简单含氧酸
MoO₄²⁻ → Mo₇O₂₄⁶⁻（七钼酸）——易缩合形成多酸
WO₄²⁻ → W₁₂O₄₁¹⁰⁻ / W₁₂O₄₀⁸⁻——更复杂的多酸

趋势：Cr → Mo → W，缩合倾向递增（Mo/W 的原子半径大→可容纳更高配位数）
```

## 五、贵金属与难熔金属对比速查

```
Ag/Au/Hg vs Mo/W——两类完全不同的过渡金属后续元素：

贵金属（ds 区）：Ag(4d¹⁰5s¹)、Au(5d¹⁰6s¹)、Hg(5d¹⁰6s²)
  ├─ 电离能极高→自然界以单质存在
  ├─ 强配位倾向（软酸——与软碱 S²⁻/CN⁻ 结合强）
  └─ 惰性电子对效应：Hg 中 6s² 很"懒"→Hg₂²⁺ 中 Hg–Hg 共价键稳定

难熔金属（d 区）：Mo(4d⁵5s¹)、W(5d⁴6s²)
  ├─ 极高熔点（W: 3422°C，所有金属中最高！）
  ├─ 易形成多酸（高氧化态 + 半径大→高配位数）
  └─ 催化应用（MoS₂ HDS、固氮酶 FeMo-co）
```

## 六、常见易错点

| 错误 | 纠正 |
|:---|:---|
| "王水是 HNO₃ 和 HCl 的简单混合酸" | 王水的威力来自 **HNO₃ 氧化 + Cl⁻ 配位** 协同——不是简单混合 |
| "Hg 在室温下无害" | Hg 的饱和蒸气压在 20°C 时远超安全标准——**蒸气剧毒！** |
| "Hg₂Cl₂ 和 HgCl₂ 性质一样" | Hg₂Cl₂（甘汞，+1，无毒沉淀）/ HgCl₂（升汞，+2，剧毒可溶）——完全不同！ |
| "AgF 也难溶于水" | **AgF 易溶！** 其他 AgX 难溶——F⁻ 的水合能补偿了晶格能 |
| "银镜反应在酸性条件下进行" | 银镜反应需要**碱性条件**——[Ag(NH₃)₂]OH（Tollen 试剂） |
| "同多酸和杂多酸只差一个元素" | 同多酸 = 同种中心原子缩合（如 Mo₇O₂₄⁶⁻），杂多酸 = 含两种以上中心原子（如 PW₁₂O₄₀³⁻） |
| "MoO₃ 和 CrO₃ 一样是强氧化剂" | MoO₃ 氧化性**远弱于** CrO₃——Cr(VI)→Cr(III) 的还原趋势比 Mo(VI)→Mo(IV) 强得多 |
| "Hg₂²⁺ 加 NH₃ 只产生白色沉淀" | Hg₂Cl₂ + NH₃ → **Hg(黑)** + Hg(NH₂)Cl(白)——**变黑是特征信号！** |

## 七、例题速练

### 例 1：王水原理

**题**：为什么 Au 不溶于浓 HNO₃ 但溶于王水？写出王水溶金的方程式并解释 HCl 的作用。

**速解**：Au³⁺/Au 的 E°=+1.50V > HNO₃ 的氧化电势。HCl 提供 Cl⁻ 配位：Au³⁺ + 4Cl⁻ → [AuCl₄]⁻——配位降低了 Au³⁺/Au 的有效电势，使 HNO₃ 氧化 Au 成为可能。Au + HNO₃ + 4HCl → HAuCl₄ + NO↑ + 2H₂O。

### 例 2：歧化对比

**题**：用标准电极电势解释为什么 Cu⁺ 歧化而 Hg₂²⁺ 不歧化。

**速解**：
- Cu：Cu²⁺ → Cu⁺（+0.15V），Cu⁺ → Cu（+0.52V）。E°(右) > E°(左) → Cu⁺ 歧化（K≈10⁶）
- Hg：Hg²⁺ → Hg₂²⁺（+0.92V），Hg₂²⁺ → Hg（+0.79V）。E°(右) < E°(左) → Hg₂²⁺ 不歧化

### 例 3：Ag⁺ 分步反应推断

**题**：某无色溶液 A 逐滴加入氨水 → 先产生棕色沉淀 → 继续加氨水 → 沉淀溶解变无色。A 是什么？

**速解**：A = AgNO₃。Ag⁺ + NH₃·H₂O → AgOH↓（白，立即分解为 Ag₂O↓ 棕色）→ 过量 NH₃ → [Ag(NH₃)₂]⁺（无色）。区别于 Cu²⁺（蓝色→蓝色沉淀→深蓝溶液）。
