## 一、本节要解决的问题

1. 为什么 CO 是强场配体而 Cl⁻ 是弱场？光谱化学序列是背出来的还是推出来的？
2. 看到配合物的颜色/磁性，怎么反推 dⁿ 构型和几何构型？
3. KMnO₄ 为什么是深紫色而 [Mn(H₂O)₆]²⁺ 是淡粉色？

### 学习目标

- 能从配体的 σ/π 性质推导光谱化学序列
- 能计算任意 dⁿ 构型在 Oh/Td 场中的 CFSE
- 能判断哪些 dⁿ 构型发生 Jahn-Teller 畸变
- 能用 Method A/B 计数 18e（含 NO、Cp 等特殊配体）
- 能区分 d-d 跃迁和 CT 跃迁，并用选律解释颜色差异

## 二、判断总流程图

```text
给定配合物
    │
    ├─ 1. 配体分类 → 判断场强 ──────────────────────────────┐
    │   σ-donor → Δo 中等                                      │
    │   π-acceptor → Δo 大（t₂g↓）                              │
    │   π-donor → Δo 小（t₂g↑）                                │
    │                                                          │
    ├─ 2. dⁿ 构型 → 高/低自旋 → CFSE ─────────────────────────┤
    │   d⁴~d⁷：Δo > P → 低自旋；Δo < P → 高自旋               │
    │   d¹~d³, d⁸~d¹⁰：只有一种自旋态                           │
    │                                                          │
    ├─ 3. JT 畸变判断 ────────────────────────────────────────┤
    │   基态简并电子 → JT畸变                                   │
    │   e_g 简并 → 强畸变（拉长八面体）                         │
    │   t₂g 简并 → 弱畸变（通常可忽略）                          │
    │                                                          │
    ├─ 4. 磁矩 → 验证自旋态 ──────────────────────────────────┤
    │   μ = √[n(n+2)]  BM（仅自旋）                             │
    │                                                          │
    ├─ 5. 颜色判断 ───────────────────────────────────────────┤
    │   淡色 → d-d 跃迁（自旋禁阻+宇称禁阻）                    │
    │   浓色 → CT 跃迁（无选律限制）                             │
    │                                                          │
    └─ 6. 18e 计数 ───────────────────────────────────────────┘
        Method A（离子法）：金属 dⁿ + 配体给出电子
        Method B（中性法）：金属 0 价 + 中性配体电子
```

## 三、配体三分类 → 光谱化学序列

| 配体类型 | 电子性质 | 对 Δo 的影响 | 代表配体 | 轨道机制 |
|:---|:---|:---:|:---|:---|
| **π-acceptor** | 空的 π* 接受 d 电子 | **Δo ↑↑** (最大) | CO, CN⁻, PR₃ | M→L 反馈 π 键 → t₂g 降低 → Δo 增大 |
| **σ-donor only** | 只有 σ 给电子 | Δo 中等 | en, NH₃, H₂O | 纯 σ 键 → eg 升高 |
| **π-donor** | filled p/π 推高 t₂g | **Δo ↓** (最小) | F⁻, Cl⁻, Br⁻, I⁻ | L→M π 给电子 → t₂g 升高 → Δo 减小 |

### 光谱化学序列（从强到弱）

$$ \text{CO} \approx \text{CN}^- > \text{PR}_3 > \text{en} > \text{NH}_3 > \text{H}_2\text{O} > \text{F}^- > \text{Cl}^- > \text{Br}^- > \text{I}^- $$

> **记忆法**：含 π 键的分子（CO/CN⁻/PR₃）→ 最强；纯 σ 给体（胺/水）→ 中等；卤素阴离子 → 最弱。
> **为什么 CO 用 C 配位而不是 O？** CO 的 HOMO（5σ）电子云偏向 C → C 端给电子能力强于 O 端。

## 四、CFSE 速算表

### Oh 场（八面体）

| dⁿ | 高自旋 CFSE | 低自旋 CFSE | 未成对电子 (HS/LS) |
|:---:|:---:|:---:|:---:|
| d¹ | −0.4 Δo | — | 1 / — |
| d² | −0.8 Δo | — | 2 / — |
| d³ | −1.2 Δo | — | 3 / — |
| d⁴ | −0.6 Δo | −1.6 Δo + P | 4 / 2 |
| d⁵ | 0 | −2.0 Δo + 2P | 5 / 1 |
| d⁶ | −0.4 Δo | −2.4 Δo + 2P | 4 / 0 |
| d⁷ | −0.8 Δo | −1.8 Δo + P | 3 / 1 |
| d⁸ | −1.2 Δo | — | 2 / — |
| d⁹ | −0.6 Δo | — | 1 / — |
| d¹⁰ | 0 | — | 0 / — |

> **Td 场换算**：Δt ≈ 4/9 Δo。Td 场总是高自旋（Δt 太小，很少超过成对能 P）。

### 高低自旋判断

$$ \Delta_o > P \Rightarrow \text{低自旋} \quad \quad \Delta_o < P \Rightarrow \text{高自旋} $$

> 只有 **d⁴~d⁷** 才需要判断。d¹~d³ 强制高自旋（电子先填满 t₂g），d⁸~d¹⁰ 强制低自旋（t₂g 已满）。

## 五、Jahn-Teller 畸变速判

> **核心原理**：基态有简并电子 → 畸变消除简并 → 降低能量。

| dⁿ 构型 | Oh 场基态 | JT 畸变？ | 畸变方向 |
|:---:|:---|:---:|:---|
| d⁰ | t₂g⁰ e_g⁰ | ❌ 无 | — |
| d¹ | t₂g¹ e_g⁰ | ⚠️ 弱（t₂g简并） | 通常忽略 |
| d² | t₂g² e_g⁰ | ⚠️ 弱 | 通常忽略 |
| d³ | t₂g³ e_g⁰ | ❌ 无（t₂g半满，无简并） | — |
| d⁴ HS | t₂g³ e_g¹ | ✅ **强**（e_g简并） | **拉长**（最常见） |
| d⁴ LS | t₂g⁴ e_g⁰ | ⚠️ 弱 | 通常忽略 |
| d⁵ HS | t₂g³ e_g² | ❌ 无（e_g半满） | — |
| d⁵ LS | t₂g⁵ e_g⁰ | ⚠️ 弱 | — |
| d⁶ HS | t₂g⁴ e_g² | ❌ 无 | — |
| d⁶ LS | t₂g⁶ e_g⁰ | ❌ 无（t₂g全满） | — |
| d⁷ HS | t₂g⁵ e_g² | ❌ 无 | — |
| d⁷ LS | t₂g⁶ e_g¹ | ✅ **强**（e_g简并） | **拉长** |
| d⁸ | t₂g⁶ e_g² | ❌ 无 | — |
| d⁹ | t₂g⁶ e_g³ | ✅ **强**（e_g简并） | **拉长**（Cu²⁺经典案例） |
| d¹⁰ | t₂g⁶ e_g⁴ | ❌ 无 | — |

> ⚡ **考场判断口诀**：e_g 不对称填充 → JT 强畸变。**d⁴HS / d⁷LS / d⁹** 三个构型必考。
> **Cu²⁺ (d⁹)**：所有 Cu²⁺ 配合物都是拉长八面体——两个轴向配体比赤道面配体更远。

## 六、18 电子规则速算

### 计数公式

$$ \text{VE} = \text{金属 d 电子} + \text{配体给出电子} - \text{配合物电荷} $$

### Method A（离子法）——推荐竞赛用

1. 确定金属氧化态 → 数 d 电子数
2. 每个配体按"给电子数"加到金属
3. 减去配合物总电荷

### Method B（中性法）

1. 金属按 0 价 → 数 d 电子数（周期表位置）
2. 配体按中性形式给电子
3. 直接加和

### 常用配体电子计数

| 配体 | 给电子数 | 备注 |
|:---|:---:|:---|
| CO, PR₃, NR₃ | 2e | 经典 σ 给体 |
| H, Cl, CH₃, OH | 1e（Method A）/ 1e | X 型配体 |
| η⁵-Cp | 6e（Method A）/ 5e | 芳香性配体 |
| η³-allyl | 4e（Method A）/ 3e | |
| η²-C₂H₄ | 2e | 烯烃 π 配合物 |
| NO（直线） | 3e | NO⁺ → M |
| NO（弯曲） | 1e | NO⁻ → M |

> **16e 也稳定**：平面四边形 d⁸（Pt²⁺/Pd²⁺/Au³⁺）→ D₄h 场中 b₁g*(dx²−y²) 空着 → 16e 最稳定。

## 七、d-d 跃迁 vs CT 跃迁

| 特征 | d-d 跃迁 | CT 跃迁（电荷转移） |
|:---|:---|:---|
| **本质** | 电子在 d 轨道之间跃迁 | 电子从金属→配体（MLCT）或配体→金属（LMCT） |
| **自旋选律** | ΔS=0（自旋禁阻则弱） | 同 d-d |
| **宇称选律** | **禁阻**（g→g）→ 弱 | **允许**（g↔u）→ 强 |
| **ε 数量级** | 1~100 L/mol·cm | 10³~10⁵ L/mol·cm |
| **颜色** | **淡**（如 [Mn(H₂O)₆]²⁺淡粉） | **浓**（如 KMnO₄深紫） |
| **典型案例** | [Cu(H₂O)₆]²⁺ 淡蓝 | MnO₄⁻ 深紫（LMCT: O→Mn） |

### 选律详解

| 选律 | 规则 | 违反后果 |
|:---|:---|:---|
| **自旋选律** | ΔS = 0（自旋多重度不变）| 自旋禁阻 → 强度降 ~100 倍 |
| **宇称选律 (Laporte)** | g ↔ u（必须有宇称变化）| Oh 场中 d-d 都是 g→g → 禁阻！|

> **为什么 d-d 跃迁还能看到颜色？** 振动耦合使对称中心瞬间消失 → 宇称选律部分放松 → 弱吸收仍可见。

## 八、常见易错点

| # | 易错点 | 正确认知 |
|:---:|:---|:---|
| 1 | 光谱化学序列靠死记 | CO 强是因为反馈 π 键（t₂g↓），Cl⁻ 弱是因为 π-donor（t₂g↑）——**轨道解释比顺序重要** |
| 2 | 所有 d⁴~d⁷ 都要判断高低自旋 | d⁴/d⁵/d⁶/d⁷ 才需要。够不够过 P？比较 Δo 和 P |
| 3 | CFSE 只和 dⁿ 有关 | CFSE 取决于 dⁿ + 场类型 (Oh/Td) + 高低自旋 → 三个变量 |
| 4 | JT 畸变是"偶发的" | 对于 d⁹ Cu²⁺，JT 畸变是**必然的**——基态 e_g 简并必须消除 |
| 5 | 18e 计数中 NO 固定给 3e | NO 直线配位 → 3e（NO⁺）；NO 弯曲配位 → 1e（NO⁻）。看配位几何 |
| 6 | 颜色越深 = 浓度越大 | KMnO₄ 深紫不是因为浓，是因为 CT 跃迁（LMCT）宇称允许——ε 本身就高 1000 倍 |
| 7 | 磁矩公式用 μ = √[n(n+2)] | 这个公式**只含自旋**。含轨道角动量贡献的修正值略高，竞赛用自旋公式已足够 |

## 九、综合推断流程（配合物题五步法）

```
磁矩 μ → 未成对电子数 n
    │
    ├─→ n = 0/1/2/3/4/5 → dⁿ 构型候选
    │
    ├─→ n + 配体场强 → 高/低自旋确认
    │
    ├─→ dⁿ + 自旋态 → CFSE 计算
    │
    ├─→ 颜色深浅 → d-d vs CT → 验证配体场强
    │
    └─→ 几何构型 + JT 畸变 → 最终结构
```

### 例题速练

> **题**：某配合物 μ=3.87 BM，配体为 NH₃，深蓝色。推断金属离子和构型。

**解**：
- μ=3.87 → √[n(n+2)]=3.87 → n≈3（三个未成对电子）
- NH₃ 是 σ-only 中等场 → 无法确定高/低自旋，但 n=3 对应：
  - d³（Oh, 总是高自旋, t₂g³）✓
  - d⁷ HS（Oh, t₂g⁵e_g²）→ n=3 ✓
- 颜色深蓝 → 可能 CT 跃迁或强 d-d
- 结合 n=3 的可能：Cr³⁺(d³) 或 Co²⁺(d⁷ HS)
- Cr³⁺ 配合物通常绿/紫，Co²⁺ 配合物常为蓝/粉
- → **Co²⁺(d⁷ HS)**，Oh 场
