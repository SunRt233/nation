## 二、8 族地图——全览

#### 2.0.1 条件像指纹，人名只是标签

这一讲最重要的不是多记几个名字，而是先学会看条件：

- `酸 / 离去基 / 迁移` 常在提醒你看重排族。
- `碱 / α-H / 羰基` 常在提醒你看缩合或极性翻转。
- `hν` 常在提醒你别再用基态直觉。
- `Pd / Ni / 硼酸 / 有机锌 / 炔铜盐` 常在提醒你进入偶联族。

所以第四轮更稳的打法是：**先读条件，再归机理族，最后才补上人名。**

  │ 1,2-迁移 │ 重排     │ 缩合     │ 极性翻转 │

## 三、八族逐一速查

> 📖 每个反应的基础机理已在 Round 3 详细讲授。本节只做**归类+统一机理+族内对比**。如需回顾单个反应的细节→参见对应的 Round 3 讲义。

### 第 1 族：碳正离子 1,2-迁移重排 🟡

**统一机理**：
```
缺电子中心生成 → 相邻基团 1,2-迁移 → 更稳定的碳正离子/羰基
```

**族内对比**——5 种"缺电子中心的制造方式"：

| 反应 | 制造缺电子中心的方法 | 迁移后变成什么 |
|:---|:---|:---|
| **Wagner-Meerwein** | 醇 + H⁺ → −H₂O → C⁺ | 烯烃（−H⁺）或醇（+H₂O） |
| **Pinacol** | 邻二醇 + H⁺ → −H₂O → C⁺-OH | 酮（−H⁺） |
| **Semipinacol** | α-羟基醇 → C⁺-O（环状） | 扩环酮 |
| **Tiffeneau-Demjanov** | β-氨基醇 + HNO₂ → C⁺-N₂⁺ | 扩/缩环酮 |
| **Wolff** | α-重氮酮 → Δ/−N₂ → 卡宾 | 烯酮 → 羧酸衍生物 |

**一眼识别**："底物有 OH/NH₂/重氮 + 酸或 Δ 或 HNO₂"

---

### 第 2 族：缺电子氮重排 🟢

**统一机理**：
```
N 上构建离去基团 → 缺电子氮 → R 基从 C 迁移到 N → 异氰酸酯 R-N=C=O → 水解 → 少一个碳的胺
```

**关键例外**：**Beckmann 重排碳数不变**（肟→酰胺，R 迁移后保留在产物中，不经过异氰酸酯）。

**族内对比**：

| 反应 | 底物 | 关键试剂 | 产物 | 碳数变化 |
|:---|:---|:---|:---|:---:|
| **Beckmann** | 肟 R₂C=NOH | H⁺/PCl₅ | RCONHR'（酰胺） | **不变** |
| **Hofmann** | RCONH₂ | Br₂ + NaOH | RNH₂ | **−1** |
| **Curtius** | RCON₃ | Δ | RNH₂ | **−1** |
| **Lossen** | RCONHOH | Δ/碱 | RNH₂ | **−1** |
| **Schmidt**（羧酸） | RCOOH + HN₃ | H⁺ | RNH₂ | **−1** |
| **Schmidt**（酮） | R₂CO + HN₃ | H⁺ | RCONHR' | 不变 |

**一眼识别**："从羰基化合物出发→产物是胺（少一个碳）= Hofmann/Curtius/Lossen/Schmidt。Beckmann 是肟→酰胺（碳数不变）"

---

### 第 1 族 vs 第 2 族——关键对比

| 维度 | 第1族（碳正离子 1,2-迁移） | 第2族（缺电子氮重排） |
|:---|:---|:---|
| **中间体** | 碳正离子 R₃C⁺ | 缺电子氮→异氰酸酯 R-N=C=O |
| **迁移终点** | 碳（骨架重排） | 氮（变成胺/酰胺） |
| **驱动力** | 碳正离子稳定性↑ | N₂/CO₂ 离去 |
| **典型条件** | H⁺/酸催化 | Br₂+NaOH 或 HN₃ 或 Δ |

### 第 3 族：碳负离子缩合 🔵

**统一机理**：
```
α-C 去质子化 → 烯醇负离子 → 亲核进攻亲电羰基 → β-羟基/β-酮酯（→ 可选脱水）
```

**族内对比**——按"亲核组分 × 亲电组分"矩阵：

| 反应 | 亲核组分 | 亲电组分 | 产物 |
|:---|:---|:---|:---|
| **Aldol** | 醛/酮 α-C | 醛/酮 C=O | β-羟基醛/酮 |
| **Claisen** | 酯 α-C | 酯 C=O | β-酮酯 |
| **Dieckmann** | 二酯 α-C（分子内） | 酯 C=O（分子内） | 环状 β-酮酯 |
| **Knoevenagel** | 活泼亚甲基（丙二酸酯等） | 醛/酮 C=O | α,β-不饱和羰基 |
| **Perkin** | 酸酐 α-C | 芳醛 C=O | 肉桂酸衍生物 |
| **Stobbe** | 琥珀酸酯 α-C | 醛/酮 C=O | 亚烷基琥珀酸酯 |
| **Michael** | 烯醇负离子 | α,β-不饱和羰基（C=C） | 1,5-二羰基 |
| **Mannich** | 烯醇/烯胺 | 亚胺离子（C=N⁺） | β-氨基羰基 |

**一眼识别**："有羰基 α-H + 碱 = 缩合族。Aldol=醛+醛，Claisen=酯+酯，Michael=共轭加成"

---

### 第 4 族：极性翻转（Umpolung）🟣

**统一机理**：
```
正常亲电的羰基碳 → 临时转化为亲核中心 → 与亲电试剂反应 → 恢复羰基
```

**族内对比**：

| 反应 | 翻转策略 | 亲核等价物 | 进攻谁？ |
|:---|:---|:---|:---|
| **安息香缩合** | CN⁻/噻唑盐→酰基阴离子等价物 | 氰醇/Breslow 中间体 | 另一分子醛 |
| **Stetter 反应** | 同上（噻唑盐） | Breslow 中间体 | α,β-不饱和羰基 |
| **Corey-Seebach** | 1,3-二噻烷 → Li 化 → 亲核碳 | 二噻烷负离子 | RX（烷基化）→ 水解→醛 |

**第 3 族 vs 第 4 族——关键对比**：

| 维度 | 第3族（缩合） | 第4族（Umpolung） |
|:---|:---|:---|
| **羰基角色** | 亲电的（被攻） | 亲核的（攻别人） |
| **活性物种** | 烯醇负离子（来自羰基旁边） | 酰基阴离子等价物（来自羰基本身） |
| **一眼识别** | 碱 + 羰基 | CN⁻ 或二噻烷 + 羰基 |

### 第 5 族：协同周环反应 🔴

**统一机理**：
```
环状过渡态 + 芳香性过渡态（4n+2 热允许）→ 立体专一性产物。无任何离子中间体！
```

**族内对比**：

| 反应 | 周环类型 | 电子数 | 热条件 | 特征产物 |
|:---|:---|:---:|:---|:---|
| **Diels-Alder** | [4+2] 环加成 | 6π | Δ | 六元环烯 |
| **1,3-偶极环加成** | [3+2] 环加成 | 6π | Δ | 五元杂环 |
| **Alder-ene** | ene 反应 | 6π | Δ | 烯烃+H 移位 |
| **Cope 重排** | [3,3]-σ 迁移 | 6π | Δ | 1,5-二烯异构化 |
| **Claisen 重排** | [3,3]-σ 迁移 | 6π | Δ | γ,δ-不饱和羰基 |
| **电环化** | 电环化 | 4π/6π | Δ/hν | 环状产物 |

**第 5 族 vs 前 4 族——根本区别**：

| 维度 | 第 1-4 族 | 第 5 族 |
|:---|:---|:---|
| **中间体** | 离子（C⁺/C⁻/N⁺） | 无！协同一步走 |
| **条件** | 酸/碱 | 加热或光照 |
| **立体化学** | 不一定专一 | **高度专一** |
| **溶剂效应** | 极性溶剂影响大 | 影响小 |
| **一眼识别** | 有酸/碱试剂 | 只有加热（Δ）或光照（hν） |

### 第 6 族：氧化重排/插氧 🟠

**统一机理**：
```
过氧酸 O-插入 → 重排 → 酯/酚/α-羟基酮
```

| 反应 | 底物 | 氧化剂 | 产物 | 关键选择性 |
|:---|:---|:---|:---|:---|
| **Baeyer-Villiger** | 酮 | m-CPBA | 酯/内酯 | 迁移能力：H > 3° > 2° ≈ Ar > 1° > CH₃ |
| **Dakin** | 邻/对羟基芳醛 | H₂O₂ | 酚 | 需要 OH 在邻/对位 |
| **Rubottom** | 硅基烯醇醚 | m-CPBA | α-羟基酮 | 氧化后自动水解硅醚 |

**一眼识别**："过氧酸（m-CPBA, RCO₃H）+ 酮/醛/硅醚"

### 第 7 族：光化学反应 ⚪

> 📖 基于 ABOC Ch.10 §10.2-10.3

#### 7.1 激发态基础

光化学反应的核心在于**分子吸收光子后进入激发态**，激发态的电子分布与基态完全不同——你无法用基态的化学直觉来推断光化学产物。

**三种电子态**：

| 参数 | 基态 S₀ | 单线态 S₁ | 三线态 T₁ |
|:---|:---:|:---:|:---:|
| 电子自旋 | 配对 ↑↓ | 配对 ↑↓ | 平行 ↑↑ |
| 寿命 | — | 10⁻⁹–10⁻⁶ s | **10⁻⁶–10⁻³ s**（更持久！） |
| 能量 | 最低 | 较高 | 中等 |

> **关键认知**：三线态 T₁ 寿命比单线态 S₁ 长 1000 倍以上——大部分光化学反应发生在 T₁ 态（有足够的时间发生化学反应）。

**光物理跃迁路径**：
```
S₀ →[hν]→ S₁（激发）
S₁ →[荧光]→ S₀ + hν（快，~10⁻⁸s）
S₁ →[ISC, 系间窜越]→ T₁（自旋翻转，通过自旋-轨道耦合）
T₁ →[磷光]→ S₀ + hν（慢，~10⁻³s）
T₁ →[化学反应]→ 产物 ← 这是我们要关注的路！
```

#### 7.2 Norrish Type I（α-断裂）

**底物要求**：酮（R-CO-R'），无特殊限制。

**机理**（分三步理解）：
```
Step 1: C=O 吸收 hν → n→π* 激发 → 氧原子具有自由基性质（双自由基）
Step 2: α-断裂（C-C 键均裂）→ 酰基自由基 R-C•=O + 烷基自由基 •R'
Step 3: 两条路——
   路 A: R-C•=O → R• + CO（脱羰，竞赛最常考！）
         然后 R• + •R' → R-R'（自由基重组）
   路 B: 两个自由基直接重组 → R-CO-R'（回到起始物）
         或与溶剂（H 供体）反应 → R-CHO + R'-H
```

**竞赛考点**：Norrish I 的**降碳策略**——酮 → 醛（少一个碳），因为脱了 CO。这是热反应做不到的。

**一眼识别**：酮 + hν + 产物碳数少 1 → Norrish I。

#### 7.3 Norrish Type II（γ-H 夺取——最重要！）

**底物要求**：酮必须有至少一个 **γ-H**（C=O 起第三个碳上有 H）。

**机理**（六元环过渡态是关键）：
```
Step 1: C=O 吸收 hν → n→π* 激发态（O 有自由基性质）
Step 2: 激发态的 O 夺取 γ-位 H → 六元环过渡态（最稳定！）
        → 形成 1,4-双自由基
Step 3: 双自由基的两条出路——
   路 A（环化）: C2-C5 成键 → 环丁醇（四元环）
   路 B（断裂）: C2-C3 断裂 → CH₂=CH-R' + CH₂=C(OH)-R
                                      ↓
                                 烯醇互变异构 → CH₃-CO-R
```

> **为什么六元环过渡态是关键？** γ-H 与 C=O 的距离天然形成六元环——六元环过渡态是热力学最有利的环大小（无角张力）。

**一眼识别**：酮有 γ-H + hν → Norrish II。产物要么是环丁醇（记住了！），要么是烯烃+酮。

**如果酮没有 γ-H**：只能走 Norrish I——这可以用于区分两兄弟。

#### 7.4 Paternò-Büchi 反应（[2+2] 光化学环加成）

```
C=O（激发态）+ C=C →[hν]→ 氧杂环丁烷（四元环醚）
```

**核心理解**：基态 C=O 和 C=C 的 [2+2] 环加成是 **Woodward-Hoffmann 禁阻**的（4n 电子体系热禁阻）。但光照下 C=O 被激发到三线态——变成了**双自由基**——[2+2] 变成了分步反应！不再受 WH 规则约束。

> **热 vs 光**：热反应 [2+2] = 禁阻；光化学 [2+2] = 允许（因为机理变了——从协同变成了分步双自由基路径）。

#### 7.5 di-π-methane 重排

```
1,4-二烯 →[hν]→ 双自由基重排 → 乙烯基环丙烷
```

**应用**：从安全易得的 1,4-二烯直接构建含环丙烷的复杂骨架——热反应无法实现的选择性。

#### 7.6 光氧化还原催化（进阶·竞赛前沿）

> ⚠️ 光氧化还原催化 ≠ 普通光化学！区别在于：光化学是**底物自己吸收光**；光氧化还原是**催化剂吸收光→转移电子（SET）**。

**核心催化剂识别**：

| 催化剂 | 特点 | 竞赛中出现形式 |
|:---|:---|:---|
| **Ru(bpy)₃²⁺** | 经典，红色，水溶性好 | 橙红色溶液 + hν |
| **Ir(ppy)₃** | 更强还原性（E* = −1.73V） | 黄绿色发光 + hν |
| 曙红 Y（有机染料） | 廉价替代，近年流行 | 红色有机染料 + hν |

**还原淬灭循环**（竞赛最常考）：
```
① Ru(II) + hν → *Ru(II)（激发态，星号表示激发态）
② *Ru(II) + 给体（Et₃N）→ Ru(I) + 给体•⁺（SET，还原淬灭）
③ Ru(I) + 底物 → Ru(II) + 底物•⁻（单电子还原底物，催化剂再生）
④ 底物•⁻ → 自由基中间体 → 产物
```

**竞赛识别铁律**：反应条件中有 **hν + Ru/Ir 配合物（或曙红 Y）** → 必须写 **SET（单电子转移）机理**，绝不能写成极性机理（C⁺/C⁻）！

**一眼识别**："hν/光照→光化学族。酮→Norrish（有 γ-H=Norrish II, 无=Norrish I），醛/酮+烯烃=Paternò-Büchi→氧杂环丁烷，hν+Ru/Ir→光氧化还原（SET 机理）"

### 第 8 族：金属催化交叉偶联 🔵

> 📖 基于 Clayden Ch.40 + ABOC Ch.8

#### 8.1 核心概念：Pd 在两个氧化态之间穿梭

**催化循环的本质**：
```
Pd(0) ⇄ Pd(II)
  ↑       ↑
d¹⁰     d⁸
亲核性   亲电性
14e/16e  16e

Pd(0) ——氧化加成（+2, 配位数+2）——→ Pd(II)
Pd(II)——还原消除（−2, 配位数−2）——→ Pd(0)
```

> **核心认知**：整个催化化学就是让 Pd 在 Pd(0) 和 Pd(II) 之间来回跳——Pd(0) 负责"抓"卤代物（氧化加成），Pd(II) 负责"放"产物（还原消除）。

#### 8.2 催化循环四步曲（通用骨架）

```
Step ① 氧化加成（Oxidative Addition）
   Pd(0) + R-X → R-Pd(II)-X
   Pd 氧化态从 0 → +2，配位数从 14e/16e → 16e
   底物活性：R-I > R-Br >> R-Cl >>> R-F
   可以用 R-OTf（三氟甲磺酸酯）
   绝对不能用有 β-H 的烷基卤！→ β-H 消除比赛太快！

Step ② 转金属化（Transmetallation）
   R-Pd(II)-X + R'-M → R-Pd(II)-R' + M-X
   有机金属试剂 R'-M 将 R' 转给 Pd
   通常是最慢的一步（速率决定步）
   Suzuki 中碱的作用：R-B(OH)₂ + OH⁻ → R-B(OH)₃⁻
                                    ↑ "ate"络合物，更亲核！

Step ③ 还原消除（Reductive Elimination）
   R-Pd(II)-R' → R-R' + Pd(0)
   Pd 氧化态从 +2 → 0，配位数从 16e → 14e
   两个配体必须顺式排列！反式→不反应！
   形成新的 C-C 或 C-N 键——整个偶联的"目的步骤"

Step ④ β-H 消除（β-Hydride Elimination）[Heck 专用]
   Pd-CH₂-CH₂-R → Pd-H + CH₂=CH-R（syn 消除）
   只有 Heck 反应走这条路（取代了②转金属化+③还原消除）
   然后碱脱 HX 再生 Pd(0)：H-Pd(II)-X + Base → Pd(0) + Base·HX
```

#### 8.3 Suzuki 偶联——最温柔、最常用

```
Ar-B(OH)₂ + Ar'-Br →[Pd(0)/碱]→ Ar-Ar' + B(OH)₃ + Br⁻
```

**完整催化循环**：
```
① Pd(0) + Ar'-Br → Ar'-Pd(II)-Br（氧化加成）
② Ar-B(OH)₂ + NaOH → Ar-B(OH)₃⁻（碱活化→"ate"络合物）
③ Ar'-Pd(II)-Br + Ar-B(OH)₃⁻ → Ar'-Pd(II)-Ar + B(OH)₃ + Br⁻（转金属）
④ Ar'-Pd(II)-Ar → Ar-Ar' + Pd(0)（还原消除→催化剂再生！）
```

**为什么是"最温柔的"？**
- 硼酸毒性低、对空气/水稳定（工业上最爱）
- 条件温和、官能团耐受好（–OH, –NH₂, –CHO 都能忍）
- 碱可以用 Na₂CO₃（弱碱就行！）

**配体**：PPh₃（经典）/ SPhos/XPhos（大位阻配体→加速氧化加成）

#### 8.4 Negishi 偶联——反应性最高

```
R-ZnX + R'-X →[Pd(0)]→ R-R' + ZnX₂
```

**特点**：有机锌试剂反应性比硼酸高，对缺电子卤化物比 Suzuki 更好。官能团耐受性好（–COOR, –CN, –COR 都能忍）。

**催化循环**：氧化加成 → 转金属（R-ZnX 直接把 R 给 Pd）→ 还原消除。**没有碱活化步骤**——有机锌本身就足够亲核。

#### 8.5 Heck 反应——唯一不用转金属的偶联

```
Ar-X + CH₂=CH-R →[Pd(0)/碱]→ Ar-CH=CH-R + HX
```

**Heck 与其他偶联的根本区别**：**没有转金属化！** 用迁移插入 + β-H 消除替代。

**完整催化循环**：
```
① Pd(0) + Ar-X → Ar-Pd(II)-X（氧化加成）
② 烯烃配位 + 迁移插入（碳钯化 carbopalladation）：
   Ar-Pd(II)-X + CH₂=CH-R → Ar-CH₂-CH-Pd(II)-X
                                    |
                                    R
   Pd 几乎总是加到取代少的一端（电子效应）
③ β-H 消除（syn 消除——H 和 Pd 必须同侧）：
   Ar-CH₂-CH-Pd(II)-X → Ar-CH=CH-R + H-Pd(II)-X
④ 碱再生：H-Pd(II)-X + Et₃N → Pd(0) + Et₃NH⁺X⁻
```

**区域选择性规律**：
- **缺电子烯烃**（丙烯酸酯 CH₂=CH-COOR）：Pd 加到 **CH₂ 端** → 反式（E）产物
- **富电子烯烃**（烯基醚 CH₂=CH-OR）：Pd 加到 **CH(OR) 端** → 1,1-双取代产物

> **为什么不用烷基卤？** CH₃CH₂-Br → 氧化加成后 → β-H 消除立即发生 → 乙烯 + H-Pd-Br → 偶联来不及做！所以 Heck 的 R-X 必须是**没有 sp³ β-H** 的（Ar-X, CH₂=CH-X, CH₂=CH-CH₂-X 等）。

#### 8.6 Sonogashira 偶联——炔烃专属

```
RC≡CH + Ar-X →[Pd(0)/CuI/碱]→ RC≡C-Ar + HX
```

**CuI 的作用**：末端炔烃在 CuI/碱下形成**炔基亚铜** RC≡CCu——比末端炔烃本身更亲核，转金属更快。

**并行双催化循环**：
```
Pd 循环：                      Cu 循环：
① Pd(0) + Ar-X → Ar-Pd-X     a) RC≡CH + CuI →[碱]→ RC≡CCu
② Ar-Pd-X + RC≡CCu           b) RC≡CCu 进入 Pd 循环的转金属步骤
   → Ar-Pd-C≡CR + CuX
③ Ar-Pd-C≡CR → Ar-C≡CR + Pd(0)
```

**应用**：烯二炔（enediyne）抗生素合成 → Bergmann 环化——这是天然产物化学的经典路线。

#### 8.7 Buchwald-Hartwig 胺化——C-N 偶联

```
Ar-X + R₂NH →[Pd(0)/碱]→ Ar-NR₂ + HX
```

**与传统 SₙAr 的对比**（非常重要！）：

| 维度 | Buchwald-Hartwig | SₙAr |
|:---|:---|:---|
| **离去基团活性** | **I > Br > Cl** >>> F | **F > Cl > Br > I**（完全相反！） |
| **是否需要吸电子基** | 不需要 | 需要邻/对位 EWG |
| **机理** | Pd 催化氧化加成→还原消除 | 加成-消除（Meisenheimer 中间体） |
| **大位阻底物** | 可以（大位阻配体反而有利） | 困难（位阻抑制加成） |

**催化循环**：
```
① Pd(0) + Ar-X → Ar-Pd(II)-X（氧化加成）
② 胺配位：Ar-Pd(II)-X + R₂NH → Ar-Pd(II)-NHR₂（X 离去）
③ 碱脱 H：Ar-Pd(II)-NHR₂ + t-BuO⁻ → Ar-Pd(II)-NR₂ + t-BuOH
④ 还原消除：Ar-Pd(II)-NR₂ → Ar-NR₂ + Pd(0)
```

**关键配体**：大位阻双膦配体（**Xantphos**、BINAP）——大位阻有利于还原消除（给产物腾出空间）。

#### 8.8 五大偶联一眼速查

```
Pd 催化剂 + 两种底物
    │
    ├── 亲核组分含 B（硼酸）──────→ Suzuki（加碱激活）
    ├── 亲核组分含 Zn（有机锌）────→ Negishi（不用碱）
    ├── 亲核组分是烯烃（无有机金属）→ Heck（走 β-H 消除！）
    ├── 亲核组分是炔 + CuI ────────→ Sonogashira（双催化）
    └── 亲核组分是胺 ──────────────→ Buchwald-Hartwig（C-N 键）
```

(1) 环己酮 + m-CPBA →

(3) PhCHO + CH₃COCH₃ + NaOH →

### 例题 4（⭐⭐⭐⭐ 光化学实战）：Norrish I vs II 区分

**题目**：以下两个酮分别光照，写出主要产物并判断属于 Norrish I 还是 Norrish II：
(1) Ph-CO-CH₃（苯乙酮）
(2) Ph-CO-CH₂-CH₂-CH₃（苯丁酮）

**解答**：
(1) 苯乙酮：分子中**没有 γ-H**（C=O 起第三个碳上只有 3 个 H，但那是 α-H 旁边的 CH₃ 的 H——不是在 γ 位）→ 只能走 **Norrish I**：
    Ph-CO-CH₃ →[hν]→ Ph-CO• + •CH₃ → Ph• + CO + •CH₃ → Ph-CH₃（甲苯）

(2) 苯丁酮：分子中**有 γ-H**（CH₂-CH₂-CH₃ 的末端 CH₃ 的 H 离 C=O 隔三个键）→ 走 **Norrish II**：
    六元环 TS → 1,4-双自由基 → 环化路径：环丁醇衍生物 / 断裂路径：Ph-CO-CH₃ + CH₂=CH₂

**教学要点**：Norrish I 和 Norrish II 的区分完全取决于 γ-H 的有无——不是两种"不同"的反应，而是同一种光激发态在不同结构下的两种走向。这个例题帮学生建立"结构→路径选择"的思维。

### 例题 5（⭐⭐⭐⭐ 偶联实战）：催化循环书写

**题目**：(1) 写出 Pd(PPh₃)₄ 催化下，Ph-B(OH)₂ + p-Br-C₆H₄-NO₂ → 产物的完整催化循环（Suzuki 偶联）
(2) 如果要合成 Ph-CH=CH-COOEt，你应该用 Heck 反应还是 Suzuki 偶联？为什么？

**解答**：
(1) Suzuki 催化循环：
- ① Pd(PPh₃)₄ → Pd(PPh₃)₂ + 2 PPh₃（配体解离，产生活性 14e Pd(0)）
- ② Pd(PPh₃)₂ + p-Br-C₆H₄-NO₂ → p-O₂N-C₆H₄-Pd(II)-Br(PPh₃)₂（氧化加成）
- ③ Ph-B(OH)₂ + NaOH → Ph-B(OH)₃⁻ Na⁺（碱活化）
- ④ p-O₂N-C₆H₄-Pd(II)-Br + Ph-B(OH)₃⁻ → p-O₂N-C₆H₄-Pd(II)-Ph + B(OH)₃ + Br⁻（转金属）
- ⑤ p-O₂N-C₆H₄-Pd(II)-Ph → p-O₂N-C₆H₄-Ph + Pd(PPh₃)₂（还原消除→再生）
- 产物：4-硝基联苯

(2) 用 **Heck 反应**——因为目标产物是 Ar-CH=CH-COOEt（芳基取代烯烃），这正是 Heck 的专长（Ar-X + CH₂=CH-COOEt）。Suzuki 需要先制备 Ar-CH=CH-B(OH)₂（乙烯基硼酸）再偶联，路线更长。Heck 一步搞定。

**教学要点**：这道题的关键是区分"什么时候用 Suzuki，什么时候用 Heck"——看目标产物：如果是 Ar-Ar' 联芳基→Suzuki；如果是 Ar-烯烃→Heck。

### 例题 6（⭐⭐⭐⭐⭐ 决赛综合）：第38届决赛——多族联合分析

**题目片段**：第38届决赛有机题涉及以下转化→判断每个步骤属于哪一族？写出关键中间体或催化循环步骤。

(1) 邻硝基甲苯 → von Richter 重排产物（CN⁻, Δ）→ 苯甲酸
(2) 某酮 → hν → Norrish II 产物（给出环丁醇和断裂产物两路）
(3) Michael 加成 + Claisen 缩合串联

**分析要点**：
- **von Richter**：芳环上硝基被 CN⁻ 亲核进攻→N₂ 离去→重排→苯甲酸。**归入第2族**（N 上构建离去基团→缺电子 N→重排）——虽然反应不在本课第2族表中，但按机理族归类！中间体为芳环-N=C（异腈类中间体）。
- **Norrish II**：hν=第7族。六元环 TS→γ-H 夺取→1,4-双自由基→环化（环丁醇）+ 断裂（烯烃+烯醇→酮）两路产物。需要写出六元环过渡态的结构。
- **Michael+Claisen 串联**：第3族+第3族（同一族内串联！因为两者都是碳负离子缩合——先 Michael 加成形成 1,5-二羰基，再分子内 Claisen 缩合成环）。

**教学要点**：这道决赛题完美展示了"归族思维"的力量——von Richter 不在任何教科书的"标准人名反应列表"中，但通过分析中间体类型（缺电子 N→重排），立刻归入第2族，统一机理解释。这就是本课的核心能力。

## 八、本节形成的核心结论

1. **30+ 人名反应 = 8 个机理族**。不是 30 个独立反应，而是 8 种中间体类型下的不同"实现方式"。

2. **第 1/2/3/4 族 = 离子型**（C⁺/N⁺/C⁻/Umpolung）——需要酸或碱催化。第 5 族 = 协同型——只需要加热或光照。第 6 族 = 氧化型——需要过氧酸。

3. **条件决定族**：H⁺→第1族，Br₂+NaOH→第2族，碱→第3族，CN⁻→第4族，Δ→第5族，m-CPBA→第6族，hν→第7族，Pd→第8族。

4. **第 1 族 vs 第 2 族**：都发生迁移，但第 1 族是骨架重排（碳→碳），第 2 族是变成胺/酰胺（碳→氮）。

5. **第 3 族 vs 第 4 族**：都构建 C-C 键，但第 3 族是羰基被攻（亲电），第 4 族是羰基攻别人（亲核翻转）。

6. **第 5 族 vs 前 4 族**：完全没有离子中间体——这是根本区别。如果你写的机理中有 C⁺ 或 C⁻——那就不可能是第 5 族。

7. **第 7 族（光化学）= 激发态化学**。Norrish I/II 的核心是激发态 C=O 的双自由基性质——六元环过渡态（Norrish II）决定选择性。光氧化还原催化 ≠ 普通光化学——催化剂吸光→SET（单电子转移），底物不吸光！识别标志：hν + Ru/Ir/曙红 Y → SET 机理。

8. **第 8 族（金属偶联）= Pd(0)↔Pd(II) 的穿梭**。催化循环四步曲（氧化加成→转金属化→还原消除）是通用骨架，但 Heck 是唯一的例外——没有转金属化，用迁移插入+β-H 消除替代。五大偶联的差异全在第二步（转金属化）和是否存在 β-H 消除。

9. **Suzuki 是"最温柔的"**（硼酸稳定、无毒、碱激活"ate"络合物），**Heck 是"最不同的"**（唯一的 β-H 消除路径），**Buchwald-Hartwig 是"最容易被误判的"**（C-N 偶联 vs SₙAr——离去基团活性完全相反！）。

## 九、⚠️ 易错点预警

| 错误想法 | 为什么错 | 正确理解 |
|:---|:---|:---|
| "Beckmann 和 Hofmann 差不多" | Beckmann=肟→酰胺（碳数不变），Hofmann=酰胺→胺（碳数−1），机理不同 | Beckmann 不经过异氰酸酯→碳数不变；Hofmann 经过异氰酸酯→碳数−1 |
| "Claisen 缩合和 Claisen 重排是同一个反应" | 名字都叫 Claisen，但一个是缩合（第3族），一个是重排（第5族） | Claisen 缩合=酯+酯→β-酮酯；Claisen 重排=烯丙基乙烯基醚→γ,δ-不饱和羰基 |
| "有碱就一定是缩合族" | 碱+Br₂=Hofmann 重排（第2族），碱+H₂O₂=Dakin（第6族） | 看完整条件，不只看碱 |
| "所有重排都是第1族" | Beckmann/Curtius 重排=第2族（缺电子氮），Cope/Claisen 重排=第5族（周环） | 重排≠一定是碳正离子重排——中间体类型决定族 |
| "周环反应也需要酸/碱" | 周环反应的条件只有加热或光照——不需要任何催化剂 | 如果题目条件中有酸或碱→优先考虑离子型族（1-4） |
| "Umpolung 就是安息香缩合" | 安息香缩合只是 Umpolung 策略之一，还有 Stetter 反应和 Corey-Seebach 二噻烷法 | Umpolung=第4族=羰基极性翻转。三种翻转策略各有不同的亲核等价物 |
| "光化学=加热的另一种形式" 🔥 | 以为 hv 只是提供能量 | 光化学是**激发态反应**——电子分布与基态完全不同 | 产物选择性不能用基态规律推断！[2+2] 热禁阻→光允许 |
| "Norrish I 和 Norrish II 差不多" 🔥 | 都叫 Norrish | Norrish I=α-断裂（所有酮都可），Norrish II=γ-H 夺取（必须有 γ-H）→机理路径完全不同！ | 没有 γ-H 的酮只能走 Norrish I（脱羰）；有 γ-H 的酮走 Norrish II（环化或断裂） |
| "光氧化还原=光化学" 🔥 | 都有 hv | 光化学=底物直接激发；光氧化还原=**催化剂吸收光→SET**（底物不吸收光） | 看到 hv+Ru/Ir→写 SET 机理，不要写成底物激发态 |
| "Suzuki 和 Heck 机理一样" 🔥 | 都是 Pd 催化偶联 | Suzuki=氧化加成→转金属→还原消除；Heck=氧化加成→**迁移插入→β-H 消除** | Heck 没有转金属化！Heck 产物有双键（来自 β-H 消除），Suzuki 产物是饱和碳 |
| "偶联反应中 Pd 是反应物" 🔥 | Pd 出现了所以被消耗 | Pd 是**催化剂**——Pd(0) 氧化成 Pd(II) 后又还原消除回到 Pd(0) | 催化循环的每一步都要写 Pd 的氧化态变化：0→+2→0 |
| "还原消除随时都能发生" 🔥 | 以为只要 Pd(II) 上有两个配体就行 | 两个配体必须**顺式排列**！反式配体不反应（需要先异构化） | 看配体的空间排列——反式配合物加热也不还原消除 |
| "Buchwald-Hartwig 和 SₙAr 差不多" 🔥 | 都是 Ar-X + 胺 → Ar-N | 机理完全不同！B-H 走 Pd 催化循环，SₙAr 走 Meisenheimer 中间体 | 最关键的区分：**离去基团活性完全相反**——B-H 是 I>Br>Cl，SₙAr 是 F>Cl>Br>I |
| "Heck 反应中 Pd 会加到你期望的位置" 🔥 | 按马氏规则推 | Heck 的区域选择性：缺电子烯烃 Pd 加在取代少的碳；富电子烯烃 Pd 加在取代基一端的碳 | 用"电子效应决定碳钯化方向"，不可以用马氏规则推 |
