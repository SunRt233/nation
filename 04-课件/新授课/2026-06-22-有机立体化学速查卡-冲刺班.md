# 有机立体化学与构象分析 · 速查卡

> Round 3 你学会了"辨认 R/S、识别顺/反"——本卡把立体化学从**辨认层**升级到**预测层**：给定底物结构和反应条件，预测主产物和选择性。

---

## 一、立体化学判断总流程

```
拿到有机反应题 → 先判：产物选择性是"热力学"还是"动力学"决定的？
    │
    ├─ 可逆条件（高温/酸或碱催化/长时间）
    │   └─ 热力学控制 → 最稳定产物为主
    │       └─ 比产物稳定性：多取代烯 > 少取代；e键大基团 > a键
    │
    └─ 不可逆条件（低温/动力学控制/快速反应）
        └─ 动力学控制 → 过渡态能量最低的路径为主
            │
            ├─ 亲核加成到羰基 → Felkin-Anh 或 Cram 螯合
            │   └─ 先找底物最大的基团 → 定构象 → 定进攻面
            │
            ├─ 烯烃加成 → 立体专一性（syn/anti）
            │   ├─ H₂/Pd → syn 加成
            │   ├─ Br₂ → anti 加成（溴鎓离子中间体）
            │   └─ mCPBA → syn 环氧化
            │
            ├─ E2 消除 → 反式共平面（a,a-双直立）
            │   └─ 先画椅式构象 → 找 Br a 键 → 找邻位 H a 键且反式
            │
            └─ 周环反应 → WH 规则
                ├─ DA 反应：syn 立体专一（顺式亲双烯体→顺式产物）
                └─ 电环化：热/光 + 共轭/间旋——查 WH 表
```

---

## 二、Felkin-Anh 模型 · 三步预测法

> 手性中心 α 位羰基的亲核加成——竞赛最高频立体化学预测题型。

```
R¹         O
  \       //
   C — C        ← 亲核试剂 Nu⁻ 从哪一面进攻？
  /       \
R²        R³

步骤：
1. 找最大基团 R_L（通常是位阻最大的取代基）
2. R_L 垂直于 C=O（与羰基形成 ~90° 二面角）
3. Nu⁻ 从最小基团那一侧进攻（避开 R_L 的位阻 + 利用 σ*_C-R_L 的反键轨道）
   → 主产物：Nu 与 R_L 反式
```

**一句话**：Felkin-Anh = "最大基团⊥羰基 + 亲核走反面"

---

## 三、Cram 螯合模型

> 当底物含能螯合的基团（如 α-羟基、α-烷氧基）+ 金属反离子（Li⁺/Mg²⁺/Zn²⁺）

```
步骤：
1. 金属离子螯合羰基 O 和 α-杂原子 O
2. 形成刚性五元螯合环 → 锁定构象
3. Nu⁻ 从小基团一侧进攻 → 主产物与 Felkin-Anh 可能不同！
```

**区分 Felkin-Anh vs Cram**：反应中有 Li⁺/Mg²⁺ + α-OR/OH → 优先考虑螯合模型。无金属离子 → Felkin-Anh。

## 四、A 值速查表（环己烷取代基 e 键偏好）

> A 值 = −RT lnK_eq（a⇌e 平衡），A 值越大，越"锁定"e 键。

| 取代基 | A 值 (kJ/mol) | 构象偏好 | 对反应的影响 |
|:---|:---:|:---|:---|
| t-Bu | > 20 | **绝对 e 键**——环被锁定 | E2 必须从特定位置消去 |
| i-Pr | ~9.2 | 强 e 键 | 较大选择性 |
| Me | ~7.5 | 中等 e 键 | 可翻转，选择性中等 |
| OH | ~2.5 (质子溶剂) / ~4 (非质子) | 弱 e 键 | 氢键溶剂相关 |
| F | ~0.6 | 几乎无偏好 | — |

## 五、立体电子效应 · 四大效应速查

| 效应 | 核心图像 | 典型场景 | 一句话 |
|:---|:---|:---|:---|
| **端基效应** | 杂原子孤对 → σ*_C–O 反键轨道 | 糖的端基碳：OR 倾向于 a 键而非 e 键 | "孤对→反键σ* 的稳定化 > 位阻" |
| **反式共平面** | H–C 与 C–LG 呈 180° 二面角 | E2 消除的立体要求 | "不反式不消除" |
| **Bürgi-Dunitz 角** | 亲核进攻 C=O 的 ~107° 角 | 所有羰基加成反应 | "Bürgi-Dunitz 轨迹" |
| **邻位孤对效应** | 邻位杂原子孤对稳定碳正离子 | 缩醛水解、糖化学 | "O⁺—C—O ↔ O—C—O⁺ 共振" |

## 六、常见易错点

| 错误 | 纠正 |
|:---|:---|
| Felkin-Anh 中把 C=O 碳上的取代基当"最大基团"选 | 最大基团选的是**手性中心**上的取代基，不是羰基碳的 |
| E2 消除忘了环翻转——认为 Br 向下就永远向下 | 环翻转后 a↔e 互换！先检查是否能翻转（t-Bu 锁定→不能翻） |
| DA 反应的 syn 加成写成 anti | DA 是**协同同面**——亲双烯体的顺反关系保留到产物 |
| 把 A 值当成"取代基一定 e 键" | A 值只说**偏好**——室温下 Me 的 e/a 比约 80:20，不是 100:0 |
| 氢键改变 A 值被忽略 | OH 在非质子溶剂 A~4，在质子溶剂 A~2.5——氢键竞争改变构象偏好 |

## 七、例题速练

### 例 1：Felkin-Anh 预测

**题**：(R)-2-甲基-3-戊酮 + MeLi → 主产物构型？

**速解**：
1. 手性中心：C2（连 Me, Et, C=O, H）
2. 最大基团：Et（比 Me 大）→ Et ⊥ C=O
3. MeLi 从 H 一侧进攻（最小基团侧）→ 产物 C2：原 R → 新立体中心 S
4. **主产物为 S 构型**

### 例 2：E2 立体选择性

**题**：反-1-Br-4-t-Bu-环己烷 + t-BuOK → 产物？

**速解**：
1. t-Bu 锁定 e 键（向下）
2. Br 反式 → Br 必须 a 键（向上）
3. E2 需要 Br (a↑) 与 H (a↓) 反式共平面 → 只能消 C2/C6 上的 a-H
4. **产物为单一烯烃**（C1=C2 双键，t-Bu e 键向下）
