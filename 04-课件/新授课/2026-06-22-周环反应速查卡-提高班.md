# 周环反应 · 速查卡

> 基础班你背过 Diels-Alder 是 [4+2] 环加成——本卡把周环从**零散的人名反应**升级到**先分类型→数电子→判热/光→落立体**的四问判断系统。

---

## 一、周环反应 · 四问判断法

```
拿到周环题 → 四问导航
    │
    ├─ 问 1：这是什么类型？
    │   ├─ 电环化：一根键断或成，环状共轭体系 ↔ 线性共轭体系
    │   ├─ 环加成：两根键同时断/成，两分子 → 一环（D-A 是 [4+2]）
    │   └─ σ 迁移：一根 σ 键在共轭体系内迁移位置
    │
    ├─ 问 2：参与反应的是多少电子？
    │   ├─ 电环化：数 π 电子数（看开的那个方向的共轭 π 电子）
    │   ├─ 环加成：数总 π 电子数（两个组分 π 电子之和）
    │   └─ σ 迁移：看反应类型标记 [i,j]（如 [3,3]-σ 迁移）
    │
    ├─ 问 3：热条件还是光条件？
    │   ├─ 热（Δ）：基态反应 —— 用基态 HOMO 判断
    │   └─ 光（hν）：激发态反应 —— 允许性规则翻转！
    │
    └─ 问 4：区域和立体结果是什么？
        ├─ 电环化：对旋（disrotatory）vs 顺旋（conrotatory）
        ├─ D-A：endo vs exo + 区域选择性
        └─ σ 迁移：同面（suprafacial）vs 异面（antarafacial）
```

---

## 二、电环化反应 · 热/光规则

```
电环化 = 一个共轭多烯分子，分子内成键关环（或开环）

判断三步法：
  1. 数 π 电子数（只看在开的那个方向参与的 π 电子）
  2. 查热/光条件
  3. 对照规则表：

┌──────────────────────────────────────────┐
│  π 电子数    热 (Δ)       光 (hν)        │
│──────────────────────────────────────────│
│  4n         顺旋         对旋            │
│              (con)       (dis)           │
│──────────────────────────────────────────│
│  4n+2       对旋         顺旋            │
│              (dis)       (con)           │
└──────────────────────────────────────────┘
```

**记忆口诀**："热 4n 顺，热 4n+2 对"——光条件反过来！

**例题速推**：1,3-丁二烯（4π）加热 → 顺旋关环 → 环丁烯
- 4π = 4n（n=1）→ 热顺旋 → 末端 p 轨道同向旋转关环

**1,3,5-己三烯（6π）加热** → 对旋关环 → 环己二烯
- 6π = 4n+2（n=1）→ 热对旋 → 末端 p 轨道对向旋转关环

## 三、环加成反应 · Diels-Alder

### 3.1 Diels-Alder = [4+2] 环加成

```
热允许（4n+2=6e⁻，基态 HOMO–LUMO 同面-同面匹配）

必要条件：
  ├─ 二烯体：必须是 s-cis 构象（s-trans 不能反应！）
  ├─ 亲二烯体：带吸电子基（EWG）更好（降低 LUMO）
  └─ 二烯体：带供电子基（EDG）更好（抬高 HOMO）
```

### 3.2 D-A 区域选择性：FMO 判断

```
富电子二烯 + 缺电子亲二烯体 → 区域选择性来源于 HOMO/LUMO 系数匹配

"大系数对大系数，小系数对小系数" → 决定哪种取向更有利

实例：
  1-甲氧基丁二烯 + 丙烯醛
  └─ OMe 抬高二烯 HOMO → C1 系数大
  └─ CHO 降低亲二烯体 LUMO → β-C 系数大
  └─ 大-大匹配 → "邻位"产物（1,2-取代环己烯）
```

### 3.3 Endo 规则

```
endo 过渡态 ≠ "取代基向里"
endo = 亲二烯体上的不饱和基团（EWG）位于二烯体系上方（次级轨道相互作用）

次级轨道相互作用：
  └─ 亲二烯体的 EWG（如 C=O）的 π* 与二烯体 C2/C3 的 p 轨道有重叠
  └─ 这种额外的稳定化使 endo 过渡态能量更低 → endo 产物 = 动力学产物

典型考题信号：
  ├─ 环戊二烯二聚 → endo 产物（二聚环戊二烯）
  ├─ 马来酸酐 + 环戊二烯 → endo 加成物
  └─ 题目画两个产物让你选 → endo 是 EWG 在"下方"的那个
```

### 3.4 D-A 反应 · 立体专一性

| 二烯体构型 | 亲二烯体构型 | 产物立体化学 |
|:---|:---|:---|
| (E,E)-二烯 | cis-亲二烯体 | cis 取代基在环同侧 |
| (E,E)-二烯 | trans-亲二烯体 | trans 取代基在环对侧 |
| (E,Z)-二烯 | 任意 | 注意 Z 端的相对构型保留 |

## 四、σ 迁移反应 · [3,3] 与 [1,5]

### 4.1 σ 迁移 = σ 键在共轭 π 体系内移动位置

```
命名：[i,j]-σ 迁移
  i = 迁移起点的原子编号（从断裂 σ 键端点数起）
  j = 迁移终点的原子编号

常见类型：
  [1,5]-σ 迁移：H 或 R 从 C1 迁移到 C5（共轭二烯体系上）
  [3,3]-σ 迁移：Cope 重排、Claisen 重排
```

### 4.2 Cope 重排 vs Claisen 重排

| 反应 | 底物 | 产物 | 关键特征 |
|:---|:---|:---|:---|
| **Cope** | 1,5-二烯 | 异构化 1,5-二烯 | 纯碳体系，无杂原子 |
| **oxy-Cope** | 3-羟基-1,5-二烯 | 烯醇→酮 | O 加速（氧负离子版更快） |
| **Claisen** | 烯丙基乙烯基醚 | γ,δ-不饱和羰基 | 有 O 参与，产物是羰基化合物 |
| **芳香 Claisen** | 烯丙基苯醚 | 邻烯丙基酚 | [3,3] 后烯醇化→芳构化驱动 |

**Cope/Claisen 协同过渡态**：六元椅式过渡态 → 立体化学由椅式构象决定。

### 4.3 [1,5]-H 迁移与 [1,5]-R 迁移

```
[1,5]-H 迁移：
  ├─ 热允许（同面迁移，6e⁻体系 = 4n+2）
  └─ 常见于共轭二烯体系中的 H 迁移

[1,5]-R 迁移（如 [1,5]-C 迁移）：
  ├─ 热允许，但比 H 迁移慢（C 比 H 大→空间要求高）
  └─ 同面迁移保留构型
```

## 五、[2+2] 环加成 · 热禁阻 / 光允许

```
[2+2] 热反应 = 禁阻（4e⁻ = 4n → 热同面-同面禁阻）
[2+2] 光反应 = 允许（激发态 HOMO → 同面-同面允许）

常见 [2+2] 光反应：
  ├─ 烯烃光二聚 → 环丁烷
  ├─ Paterno-Büchi 反应（烯烃 + 羰基 → 氧杂环丁烷）
  └─ 题目出现 hν + 两个烯烃 → 先查是否 [2+2] 光环加成
```

## 六、1,3-偶极环加成（基础识记）

```
1,3-偶极体 + 亲偶极体 → [3+2] 环加成 → 五元杂环

常见 1,3-偶极体：
  ├─ 叠氮化物（R–N₃）
  ├─ 重氮化合物（R₂C=N₂）
  ├─ 腈氧化物（R–C≡N⁺–O⁻）
  ├─ 硝酮（R₂C=N⁺(R)–O⁻）
  └─ 臭氧（O₃）——臭氧化反应 = 1,3-偶极环加成

[3+2] = 6π 电子体系 = 4n+2 → 热允许
```

**推断题信号**：看到五元杂环产物、臭氧化、叠氮+炔（"Click"反应）→ 可能是 1,3-偶极环加成。

## 七、Woodward-Hoffmann 规则 · 课堂速记版

```
WH 规则的核心一句话：
  热反应中，同面-同面相互作用的 4n+2 电子体系是允许的；
  光反应中，同面-同面相互作用的 4n 电子体系是允许的。

┌──────────────┬────────────────┬────────────────┐
│  电子数      │  热 (Δ)        │  光 (hν)       │
│──────────────┼────────────────┼────────────────│
│  4n          │  禁阻/异面     │  允许(同面)    │
│  4n+2        │  允许(同面)    │  禁阻/异面     │
└──────────────┴────────────────┴────────────────┘

记忆公式：
  电环化：热(4n顺, 4n+2对)　光翻转
  环加成：热(4n+2允许, 4n禁阻)　光翻转
  σ迁移：热(同面 4n+2 允许, 异面 4n 允许)　光翻转
```

## 八、常见易错点

| 错误 | 纠正 |
|:---|:---|
| 不问类型直接套规则 | 第一步永远是分类型——电环化数 π 电子、环加成数总 π 电子、σ 迁移看标记 |
| 不看热/光条件就写产物 | 热和光结论翻转！题目的 Δ 或 hν 不是装饰，是决定性的 |
| s-trans 二烯体做 D-A | s-trans 构象的二烯体不能发生 D-A——两端太远，无法同时成键。必须先异构化成 s-cis |
| 把 Claisen/Cope 当普通重排 | Claisen/Cope 是协同 [3,3]-σ 迁移——无离子中间体，过渡态是六元环状。与 Beckmann/Hofmann 不同族 |
| endo 规则死背"取代基向里" | endo 是次级轨道相互作用的结果——亲二烯体的不饱和基团（C=O, CN）与二烯体系有额外的轨道重叠稳定化 |
| 忘记 D-A 的立体专一性 | 二烯体和亲二烯体的构型在产物中保留——(E,E)-二烯+cis-亲二烯体→cis 取代产物 |

## 九、自查清单

- [ ] 四问法能默推（类型→电子数→热/光→区域/立体）

- [ ] WH 规则总表（热/光 × 4n/4n+2）能默列
- [ ] 常见推断题信号词：Δ / hν / s-cis / endo/exo / [3,3] / 1,3-dipolar
