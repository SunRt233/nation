## 一、本节要解决的问题

1. 为什么大多数稳定金属有机配合物满足 18 电子？为什么有些只有 16 电子也稳定？

## 二、知识内容

### 2.1 18 电子规则：不只是"9 个轨道填满"

#### 2.1.1 基础层：9 个价轨道 × 2 = 18

过渡金属有 9 个价轨道（1 个 s + 3 个 p + 5 个 d），每个可容纳 2 个电子 → 填满 = 18 电子 = **下一周期稀有气体的电子构型**（有效原子序数规则 EAN）。

> 例如：Cr(CO)₆：Cr(0) = d⁶ = 6e + 6×CO(每个给 2e) = 6+12 = **18e** ✓

#### 2.1.2 MO 层面解释：为什么 π 受体配体需要 18e？

这是初赛讲义第5讲的核心补充——不是所有配合物都需要 18e！

在八面体场中，配体与金属的 MO 相互作用：

```
金属 d 轨道 ───→ t₂g（非键/弱 π）
              → eg*（σ 反键，高能，不填）

π 受体配体（CO、CN⁻、PR₃）：
  空的 π* 轨道接受金属 d 电子反馈
    ↓
  t₂g 能级进一步降低（反馈 π 键稳定化）
    ↓
  t₂g（6e）填满 + 配体 σ 给电子（12e）= 18e 最稳定 ✓
```

**π 给体配体（Cl⁻、I⁻、OH⁻）**：
- 填满的 p 轨道与金属 t₂g 形成 π 键 → **推高** t₂g
- t₂g 升到接近 eg* → 电子可以从 t₂g 跑到 eg* → 不必填满 6e
- 结果：< 18e 也稳定（如 [TiCl₆]²⁻ = 12e）

> 🔑 **核心洞察**：
> - CO 型配体 → **π 受体** → 拉低 t₂g → **必须 18e**
> - Cl⁻ 型配体 → **π 给体** → 推高 t₂g → **不必 18e**
> - NH₃ 型配体 → **纯 σ 给体** → 中间情况

### 2.2 16 电子规则：为什么 Pt²⁺/Pd²⁺ 喜欢 16e？

#### 2.2.1 D₄h 平面四方场的 d 轨道分裂

在八面体场（O_h）中 d 轨道分裂为 t₂g(3) + eg(2)。在**平面四方场（D₄h）**中进一步分裂为 4 组：

```
d轨道在 D₄h 中的分裂（从低到高）：
  d_{xz}, d_{yz} → 最低（与配体 σ 作用最弱）
  d_z²         → 中等
  d_{xy}       → 较高（在 xy 平面内，与配体直接对顶）
  d_{x²−y²}    → 最高（b₁g*）！指向四个配体 → 强 σ 反键
```

**关键**：b₁g*（d_{x²−y²}）能量太高，电子不填它。剩下的 8 个 d 轨道能容纳 16 个电子 → **16e 最稳定**。

#### 2.2.2 Zeise 盐：历史上第一个金属有机配合物

**K[PtCl₃(C₂H₄)]·H₂O**（Zeise 盐，1827 年）

- **合成**：K₂[PtCl₄] + C₂H₄ → K[PtCl₃(C₂H₄)] + KCl
- **结构**：Pt²⁺ 四方平面，C₂H₄ 垂直于平面配位
- **成键**（Dewar-Chatt-Duncanson 模型）：
  - C₂H₄ 的 π 电子 → Pt²⁺ 空 d 轨道（σ 给体）
  - Pt²⁺ 填满的 d 电子 → C₂H₄ 的 π* 空轨道（**反馈 π 键**）
  - 双重作用 → C=C 键被削弱、拉长

**电子计数**：
- Pt²⁺ = d⁸ = 8e
- 3×Cl⁻ = 3×2e = 6e（每个 Cl⁻ 给一对电子）
- C₂H₄ = 2e（η²-配位，π 电子对）
- 总计 = 8 + 6 + 2 = **16e** ✓

### 2.3 电子计数 6 条规则

| # | 规则 | 说明 | 示例 |
|:---:|:---|:---|:---|
| 1 | **中性配体给 2e** | CO、PR₃、NH₃、H₂O、C₂H₄ 等都按 2e 算 | CO → 2e |
| 2 | **卤阴离子给 2e**（作为配体时） | Cl⁻、Br⁻、I⁻ 各给一对电子 | [PtCl₄]²⁻：4×Cl⁻ = 8e |
| 3 | **共价键按键数给电子** | M—CH₃：C 给 1e（σ 键的一个电子）；M=CR₂：卡宾给 2e | M—H：H 给 1e |
| 4 | **螯合配体数原子点数** | 每个配位原子给 2e，不论是不是螯合 | en（乙二胺）= 2 个 N → 4e；EDTA = 6 个配位原子 → 12e |
| 5 | **NO 特殊处理** | 线性 NO = 3e（NO⁺ 等瓣体）；弯曲 NO = 1e（NO⁻ 等瓣体） | 看 M—N—O 键角：~180° = 3e，~120° = 1e |
| 6 | **桥连配体分边计** | μ₂-CO：每边各给 2e（总共 4e 分给两个金属） | Fe₂(CO)₉：3 个 μ₂-CO + 6 个端 CO |

**ηⁿ 标注法**：
- η¹ = 配体通过 1 个原子配位（如 η¹-C₃H₅ = σ-烯丙基，给 1e）
- η³ = 配体通过 3 个原子配位（如 η³-C₃H₅ = π-烯丙基，给 3e）
- η⁵ = 配体通过 5 个原子配位（如 η⁵-C₅H₅ = Cp，给 5e）
- η⁶ = 配体通过 6 个原子配位（如 η⁶-C₆H₆ = 苯，给 6e）

> 🔑 **电子计数口诀**：中性配体 2e，卤素阴离子 2e，共价键数键，螯合数原子，ηⁿ 就是 n。

### 2.4 等瓣类似（Isolobal Analogy）

#### 2.4.1 核心思想

有机化学中的分子碎片（CH₃、CH₂、CH）和金属有机碎片（Mn(CO)₅、Fe(CO)₄、Co(CO)₃）如果**前线轨道的数目、对称性和电子数相同**，它们就是**等瓣的**——可以互相替换来预测配合物结构！

#### 2.4.2 关键等瓣对应表

| 有机碎片 | 价电子数 | 金属有机等瓣体 | 电子数验证 |
|:---|:---:|:---|:---|
| **CH₄** | 8 | — | 8 = 8 ✓ |
| **CH₃·** | 7 | Mn(CO)₅ | Mn: 7e(填满 d⁷?) — 验证：Mn(0)=d⁷=7e + 5CO=10e = **17e**（差 1e 到 18e → 需要 1e 成键 → ≈ CH₃· 需要 1e 成键） |
| **CH₂** | 6 | Fe(CO)₄ | Fe(0)=d⁸=8e + 4CO=8e = **16e**（差 2e → ≈ CH₂ 需要 2e） |
| **CH** | 5 | Co(CO)₃ | Co(0)=d⁹=9e + 3CO=6e = **15e**（差 3e → ≈ CH 需要 3e） |
| **C** | 4 | Ni(CO)₂ | Ni(0)=d¹⁰=10e + 2CO=4e = **14e**（差 4e → ≈ C 需要 4e） |

> 💡 **记忆法**：从 Mn(CO)₅ 开始，每往右移一个金属（Mn → Fe → Co → Ni），减少一个 CO 配体。17e → 16e → 15e → 14e，对应 CH₃· → CH₂ → CH → C。

#### 2.4.3 等瓣拆解三步法

```
Step 1: 拆分目标配合物 → 写出各金属碎片的电子数
Step 2: 将每个金属碎片映射为有机等瓣体
Step 3: 用有机碎片重新拼装 → 用有机化学直觉预测结构
```

**案例 1：Ir₄(CO)₁₂ —— 为什么是四面体？**

1. 拆分：4 个 Ir(CO)₃ 碎片
2. Ir(CO)₃：Ir(0)=d⁹=9e + 3CO=6e = **15e** → 差 3e 到 18e
3. 等瓣对应：15e ≈ CH（需要 3e 成键）
4. 4 个 CH 拼起来 = 四面体烷 (CH)₄ = C₄H₄（四面体）
5. **预测**：Ir₄(CO)₁₂ 也应该是四面体！✓（实验证实：Td 对称性）

**案例 2：Os₅(CO)₁₉ —— 为什么是螺环？**

1. 拆分：Os₅(CO)₁₉ = [Os(CO)₃]·4[Os(CO)₄]
2. Os(CO)₃ = 15e → ≈ CH；Os(CO)₄ = 16e → ≈ CH₂
3. 拼装：CH·4CH₂ → 一个 CH 连着四个 CH₂ → **螺[2.2]戊烷骨架**
4. **预测**：Os₅(CO)₁₉ 是螺环结构！✓

**案例 3：Mn₂(CO)₁₀ —— 为什么 Mn—Mn 单键？**

1. Mn(CO)₅ = 17e → ≈ CH₃·
2. 两个 CH₃· 偶合 = CH₃—CH₃（C—C 单键）
3. **预测**：两个 Mn(CO)₅ 通过 Mn—Mn 单键连接 = (CO)₅Mn—Mn(CO)₅ ✓

> 🔑 等瓣类似的核心力量：用你已经非常熟悉的有机化学直觉（CH₃ 怎么连、CH₂ 怎么连）来预测你不熟悉的金属配合物结构！

## 三、课堂例题

### 例题 1（⭐⭐ 基础）：电子计数

**题目**：计算以下配合物的价电子数，并判断是否满足 18 电子规则。
1. Fe(CO)₅
2. [Co(NH₃)₆]³⁺
3. (η⁵-C₅H₅)₂Fe（二茂铁）
4. [PtCl₄]²⁻

**解答**：

| 配合物 | 金属电子 | 配体电子 | 总电子 | 18e? |
|:---|:---|:---|:---|:---:|
| Fe(CO)₅ | Fe(0)=d⁸=8e | 5×CO=10e | **18e** | ✓ |
| [Co(NH₃)₆]³⁺ | Co³⁺=d⁶=6e | 6×NH₃=12e | **18e** | ✓ |
| (η⁵-Cp)₂Fe | Fe²⁺=d⁶=6e | 2×η⁵-Cp=2×5e=10e? 等等——Fe²⁺=6e + Cp⁻(6e)×2=12e | **18e** | ✓ |
| [PtCl₄]²⁻ | Pt²⁺=d⁸=8e | 4×Cl⁻=8e | **16e** | ✗（但 16e 对 D₄h Pt²⁺ 稳定！） |

> 💡 重新验算二茂铁：Fe²⁺ = d⁶ = 6e，每个 Cp⁻ 给 6e（η⁵ 配位），总计 6+6+6 = **18e** ✓

---

### 例题 2（⭐⭐ 基础）：等瓣类似预测

**题目**：已知 Re₂(CO)₁₀ 的结构与 Mn₂(CO)₁₀ 相似。用等瓣类似解释为什么它们是二聚体。

**解答**：
- Re(CO)₅ = Re(0)=d⁷=7e + 5CO=10e = **17e**
- 17e ≈ CH₃·（甲基自由基，也需要 1 个电子成键）
- 两个 CH₃· 偶合 = CH₃—CH₃
- ∴ 两个 Re(CO)₅ 通过 Re—Re 单键二聚 = (CO)₅Re—Re(CO)₅ ✓

---

### 例题 3（⭐⭐⭐ 拓展）：解释 16e 稳定性

**题目**：Pd(PPh₃)₄ 是重要的催化前体（用于 Suzuki 偶联等）。计算它的价电子数，并解释为什么它是 16e 配合物在溶液中会解离一个 PPh₃。

**解答**：
- Pd(0) = d¹⁰ = 10e
- 4×PPh₃ = 4×2e = 8e
- 总计 = **18e**（四配位时满足 18e）

但在溶液中：Pd(PPh₃)₄ ⇌ Pd(PPh₃)₃ + PPh₃
- Pd(PPh₃)₃ = 10e + 3×2e = **16e**（三配位，14e? 等等）

重新算：Pd(0)=d¹⁰=10e + 3PPh₃=6e = **16e**。但 Pd 是三配位不太对...实际上 Pd(PPh₃)₃ 是 14e，真正的催化活性物种是 Pd(PPh₃)₂ = 14e...让我再检查。

实际上：
- Pd(PPh₃)₄：Pd(0)=d¹⁰=10e + 4×2e=8e = **18e**
- 解离一个 PPh₃ → Pd(PPh₃)₃ = 10e + 6e = **16e**（三配位时只有 14e，但会通过溶剂配位或 agostic interaction 补到 16e）
- 真正的催化活性物种是 Pd(0)L₂ = 10e + 4e = 14e，但 14e Pd(0) 是极活泼的——实际有溶剂分子或底物配位补到 16e

> 🔑 核心点：Pd 和 Pt 的第二/第三行过渡金属倾向 16e（D₄h），所以 18e → 16e 的解离是催化循环的关键第一步——腾出空配位点让底物上来。

## 四、本节形成的核心结论

1. **18 电子规则不是铁律**——π 受体配体（CO/PR₃）需要 18e，π 给体配体（Cl⁻/I⁻）可以 < 18e。
2. **16 电子规则**适用于 d⁸ 平面四方配合物（Pt²⁺/Pd²⁺/Rh⁺/Ir⁺）——b₁g*（d_{x²−y²}）空着是最稳定状态。
3. **Zeise 盐**是 16e Pt²⁺ 的经典案例——Dewar-Chatt-Duncanson 模型解释 C₂H₄ 与 Pt 的双重成键（σ 给体 + π 反馈）。
4. **电子计数 6 条规则**：中性配体 2e，卤素阴离子 2e，共价键数键，螯合数原子，NO 特殊，桥连分边。ηⁿ 标注 = 给 n 个电子。
5. **等瓣拆解三步法**：拆分金属碎片 → 映射有机等瓣体 → 用有机化学直觉预测结构。Mn(CO)₅ ≈ CH₃·、Fe(CO)₄ ≈ CH₂、Co(CO)₃ ≈ CH。

## 五、⚠️ 易错点预警

| 错误想法 | 为什么错 | 正确理解 |
|:---|:---|:---|
| "所有稳定配合物都是 18e" | π 给体配体推高 t₂g → 不必 18e；D₄h 场 Pt²⁺/Pd²⁺ 16e 最稳定 | 看配体类型（π 受体/给体）和几何构型 |
| "CO 只是 σ 给体" | CO 同时是强 π 受体——反馈 π 键才是它成为强场配体的原因 | CO 的 C 端孤对电子 σ 给金属，空的 π* 接受金属 d 反馈 → π 酸 |
| "数电子时 Cl⁻ 给 1e" | 作为**配体**时 Cl⁻ 给 2e（一对孤对电子）。作为共价 M—Cl 时才是 1e（按共价键算） | 配体 Cl⁻ = 2e；M—Cl 共价键 = 1e。看 Cl 是配位还是成共价 σ 键 |
| "η⁵-Cp 给 5e" | η⁵-C₅H₅⁻（环戊二烯阴离子）给 **6e**，不是 5e | Cp⁻ 是 6π 电子芳香体系 → 给 6e。η⁵ 描述的是配位的碳原子数（5 个），不是电子数 |
| "等瓣类似就是背对照表" | 等瓣类似是**推理工具**，关键是 17e → 差 1e → ≈ CH₃·（差 1e） | 掌握"缺几个电子到 18e → 等价于有机碎片缺几个电子到 8e"的逻辑 |
