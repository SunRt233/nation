# 芳香反应 · 速查卡

> 基础班你背过"定位规则：NH₂→o/p、NO₂→m"——本卡把芳香化学从**定位规则碎片**升级到**反应类型识别→中间体稳定性比较**的判断系统。

---

## 一、芳香反应 · 四问判断法

```
芳香底物 + 试剂 → 先分机理赛道
    │
    ├─ 问 1：是不是 SEAr（亲电芳香取代）？
    │   └─ 条件：亲电试剂 + 催化剂（Lewis酸/质子酸）+ 苯环有给电子基
    │   └─ 这是最常考的芳香机理（~80%的芳香题）
    │
    ├─ 问 2：是不是 SNAr（亲核芳香取代）？
    │   └─ 条件：强吸电子基（o/p-NO₂）+ 强亲核试剂 + 加热
    │   └─ 特征：Meisenheimer 复合物中间体（加成的σ配合物带负电荷）
    │
    ├─ 问 3：是不是苯炔机理？
    │   └─ 条件：无吸电子基的卤苯 + 极强碱（NaNH₂/NH₃）+ 高温高压
    │   └─ 特征：消除-加成→苯炔中间体→产物可能"异常"定位（苯炔两侧都可加成）
    │
    └─ 问 4：是不是杂环位点题？
        └─ 底物是吡啶/吡咯/呋喃/吲哚等→杂环的亲电/亲核取代各有偏好位点
```

## 二、SEAr 定位规则 · 从 σ 配合物理解

### 2.1 取代基分类

| 取代基 | 定位 | 活化/钝化 | 电子效应组合 | σ 配合物的故事 |
|:---|:---:|:---:|:---|:---|
| –NH₂, –NHR, –NR₂ | o/p | **强活化** | +M（孤对共轭给电子）>> −I | o/p 位进攻的 σ 配合物→N 孤对与正电荷共轭→大幅稳定 |
| –OH, –OR | o/p | **活化** | +M > −I | 同上，但 O 电负性高→M 效应弱于 N |
| –R（烷基）| o/p | **弱活化** | +I + 超共轭 | 烷基给电子稳定 σ 配合物（但效果弱） |
| –X（卤素）| o/p | **钝化**（！） | −I > +M | o/p 位的 σ 配合物→卤素孤对共轭→部分稳定；但 —I 效应使总稳定性不如苯 |
| –NO₂ | m | **强钝化** | −I + −M | o/p 位进攻→σ配合物有邻位正电荷与 NO₂ 相邻→极度不稳定→只能走 m |
| –CF₃, –CN, –SO₃H | m | **钝化** | −I（仅诱导） | 没有孤对电子→只有吸电子→o/p 位特别不稳定 |

### 2.2 卤素的特殊地位

**卤素是唯一"钝化的 o/p 导向基"**——这个特殊身份竞赛最爱考！
- Br 的 —I 使总环电子密度下降（钝化）→反应比苯慢
- 但 Br 的孤对电子通过 +M 共轭→o/p 位 σ 配合物比其他位略稳定→定位仍选 o/p

## 三、SNAr · Meisenheimer 复合物

```
SNAr = 加成-消除机理（需要强吸电子基活化）

条件铁三角：
  1. 芳环上有强吸电子基（通常是 o/p-NO₂）：吸收负电荷→稳定中间体
  2. 强亲核试剂：OH⁻, OR⁻, NR₂⁻, CN⁻
  3. 好离去基：F > Cl > Br > I（反常！速率与 CX 键强无关——第一步加成是决速步）

中间体：Meisenheimer 复合物
  └─ Nu⁻ 加成到与吸电子基共轭的位置→负电荷被吸电子基离域→稳定

  F 比 I 快的原因：F 的强 —I 使加成步的过渡态更稳定（不是断 CX 键那步）
```

---

## 四、苯炔机理

```
NaNH₂/NH₃(l) + 卤苯（无吸电子基）→ 苯炔中间体

机制：E2型消除-加成
  Step 1：NH₂⁻ 夺取邻位 H → 苯炔（C≡C 三键在环内？不，是高度张力的"额外 π 键"）
  Step 2：NH₂⁻（或 NH₃）加到苯炔→产物

定位异常：苯炔两侧都可加成→产物可能是"预期"和"非预期"的混合物
  例：p-Cl-PhCH₃ + NaNH₂/NH₃ → p-CH₃-PhNH₂ + m-CH₃-PhNH₂
  └─ 苯炔中间体→NH₂⁻ 可以从两端加成→得到对位和间位两种产物
```

**SNAr vs 苯炔区分口诀**：有吸电子基→SNAr（加成-消除）；无吸电子基+NaNH₂→苯炔（消除-加成）

## 五、杂环位点选择速查

| 杂环 | 亲电取代偏好位点 | 亲核取代偏好位点 | 原因 |
|:---|:---:|:---:|:---|
| **吡咯** | 2（α）位 | — | N 孤对在芳香六电子体中→环富电子→亲电取代极快 |
| **呋喃** | 2（α）位 | — | 同上，但 O 电负性高→反应性低于吡咯 |
| **噻吩** | 2（α）位 | — | S 的 3p 孤对共轭→反应性介于呋喃和苯之间 |
| **吡啶** | 3（β）位 | 2,4 位 | N 的吸电子使环缺电子→亲电取代慢（需剧条件）→亲核取代快（N 活化 o/p） |
| **吲哚** | 3 位 | — | 吡咯环优先反应（比苯环快得多） |

## 六、Friedel-Crafts 反应精要

| 反应 | 试剂 | 产物 | 局限与陷阱 |
|:---|:---|:---|:---|
| **F-C 烷基化** | RCl + AlCl₃ | 烷基苯 | ⚠️ C⁺ 重排！→产物常常是重排后的烷基。多烷基化（产物更活泼→继续反应）。**不能在钝化苯环上发生**（强钝化→不反应） |
| **F-C 酰基化** | RCOCl + AlCl₃ | 芳酮 | ✅ 无重排（酰基正离子 RC≡O⁺ 共振稳定）。✅ 单取代（产物被钝化→停止）。但也不能在强钝化苯环上发生 |

## 七、常见易错点

| 错误 | 纠正 |
|:---|:---|
| "卤素是钝化基团所以是 m 定位" | **卤素是钝化的 o/p 导向基**——竞赛最高频陷阱 |
| "SNAr 中 C–F 键最强所以 F 最难取代" | **F 是最快被取代的卤素**——SNAr 的决速步是加成不是断键！ |
| 看到卤苯+NaNH₂ 就写 SNAr | 如果没有强吸电子基→是苯炔机理，不是 SNAr |
| F-C 烷基化用 1° 卤代烷+AlCl₃ 得到直链烷基苯 | **1° 卤代烷的 C⁺ 会重排！** → 产物是重排后的支链烷基 |
| 杂环定位只靠"电子密度" | 还要看 σ 配合物稳定性——吡咯的 2 位和 3 位 σ 配合物稳定性差异很大 |

## 八、例题速练

### 例 1：机理赛道判断

**题**：p-NO₂-PhCl + NaOH/H₂O/加热 → p-NO₂-PhOH。什么机理？

**速解**：
1. 有强吸电子基（p-NO₂）→不是苯炔
2. OH⁻ 是强亲核→SNAr（加成-消除）
3. Cl 是离去基→产物 p-NO₂-PhOH ✓

### 例 2：苯炔定位异常

**题**：p-Cl-PhOCH₃ + KNH₂/NH₃(l) → 两种产物。分别是什么？为什么？

**速解**：
1. 没有强吸电子基→苯炔机理
2. 消除 HCl→苯炔中间体（OCH₃ 在间位？不——在 4 位→苯炔在 2,3 或 3,4？需要具体分析）
3. NH₂⁻ 可以从苯炔两侧加成→得到 p-OCH₃-PhNH₂ + m-OCH₃-PhNH₂
4. 对位为主（OCH₃ 的 +M 效应使其对位更稳）

## 九、自查清单

- [ ] 四问判断法（SEAr/SNAr/苯炔/杂环位点）能默推
- [ ] SEAr 定位规则——从 σ 配合物稳定性理解而非死记
- [ ] 卤素="钝化的 o/p 导向基"——这一特殊身份必记
- [ ] SNAr 的条件铁三角（吸电子基+强亲核+好离去基）

- [ ] 强钝化苯环不能做 F-C 反应
- [ ] 杂环亲电/亲核取代的偏好位点（吡咯α/吡啶β/吲哚3）
