## 一、课堂引入与本节问题

### 1.1 课堂切入

- **现象/问题/情境**：
  - 为什么把锌片和铜片插入稀硫酸中，单独都不反应，但用导线连起来就能产生电流？
  - 为什么高锰酸钾在酸性条件下氧化性极强，而在中性/碱性条件下明显减弱？
  - 为什么锂电池的电压是 3.7 V，而干电池只有 1.5 V？这些数值能算出来吗？

- **学生已有认知**：
  - 高中已学过氧化还原反应、化合价升降、金属活动性顺序
  - 知道原电池能把化学能转化为电能，但可能不清楚"电势"是什么、为什么不同金属电压不同

- **本节要解决的问题**：
  1. 电子转移如何被"拆分"成外电路电流？原电池的结构是什么样的？
  2. 如何判断一个氧化还原反应能否自发进行？标准电极电势 $E^{\ominus}$ 怎么查、怎么用？
  3. 浓度、pH 改变时，氧化还原能力如何变化？Nernst 方程怎么代入计算？
  4. ln 形式和 lg 形式的 Nernst 方程有什么区别、怎么切换？

### 1.2 学习目标

- **本节讲到哪里为止**：
  - 能完成简单氧化态判定和常见酸性半反应配平
  - 能查 $E^{\ominus}$ 表判断氧化剂/还原剂强弱和反应方向
  - 能用 Nernst 方程处理基础浓度/pH 影响，掌握 ln/lg 切换
  - 能进行简单的化学电源电动势计算
  - 能读懂 Latimer 图并进行简单计算（选讲）

- **后续深入学习**：
  - Butler-Volmer 方程（电极动力学）→ 第三轮/物理化学
  - Pourbaix 图的完整绘制 → 第二轮
  - 超电势的定量计算 → 第二轮
  - SEI 膜机理 → 决赛/前沿
  - 活度系数修正 → 第二轮/分析化学

### 2.1 氧化态与半反应——电子转移的"账本"

**从问题出发**：$\mathrm{Na_2S_4O_6}$（连四硫酸钠）中硫的化合价是多少？

如果按高中方法，设 S 为 $x$，则 $2(+1) + 4x + 6(-2) = 0$，得 $x = +2.5$。但硫不可能有半个电子！这说明"化合价"在复杂化合物中不够用，我们需要引入**氧化态（oxidation state）**——它是一种形式上的 bookkeeping 工具，允许出现平均值。

**关键规则**（课堂精简版）：
1. 单质中元素的氧化态为 0
2. 离子化合物中，氧化态 = 离子电荷
3. 共价化合物中，电负性大的原子"拿走"电子对
4. O 通常为 -2（过氧化物中为 -1）；H 通常为 +1（金属氢化物中为 -1）
5. **氧化态可以是非整数**——它只反映平均情况

> [!info]- 完整推导与教材对照（课后复习用）
>
> **教材来源**：宋天佑《无机化学》§10.1、[[教学逻辑提炼-周坤无机新课-酸碱理论与电化学-第一轮]]
>
> **完整推导/补充内容**：
> 氧化态的本质是"假设所有化学键都是离子键"时，某原子所带的电荷数。对于 $\mathrm{S_4O_6^{2-}}$，实际结构是两个 S 为 +5（端基，连 3 个 O），两个 S 为 0（中间，S-S 键），平均为 +2.5。
>
> **与教材的对照说明**：
> - 教材中通常称"氧化数"，与高中"化合价"概念基本一致，但竞赛中更强调其 bookkeeping 属性
> - 周坤笔记中强调：氧化态判定是后续半反应配平的前提，但**不要把配平当成电化学的主线**

**半反应——把电子转移拆开看**：

任何氧化还原反应都可以拆成两个**半反应**：
- **还原半反应**（得电子）：$\mathrm{Cu^{2+} + 2e^- \longrightarrow Cu}$
- **氧化半反应**（失电子）：$\mathrm{Zn \longrightarrow Zn^{2+} + 2e^-}$

配平半反应的**离子-电子法**（酸性介质）：
1. 写出未配平的骨架
2. 配平除 O、H 外的所有原子
3. 用 $\mathrm{H_2O}$ 配平 O，用 $\mathrm{H^+}$ 配平 H
4. 用电子 $e^-$ 配平电荷

**示例**：配平 $\mathrm{MnO_4^- \longrightarrow Mn^{2+}}$（酸性）
- 骨架：$\mathrm{MnO_4^- \longrightarrow Mn^{2+}}$
- 配平 O：右侧加 $4\mathrm{H_2O}$ → $\mathrm{MnO_4^- \longrightarrow Mn^{2+} + 4H_2O}$
- 配平 H：左侧加 $8\mathrm{H^+}$ → $\mathrm{MnO_4^- + 8H^+ \longrightarrow Mn^{2+} + 4H_2O}$
- 配平电荷：左侧 $(-1)+(+8)=+7$，右侧 $+2$，需加 $5e^-$ 到左侧

$$\mathrm{MnO_4^- + 8H^+ + 5e^- \longrightarrow Mn^{2+} + 4H_2O}$$

**阶段性小结**：氧化态是 bookkeeping 工具，半反应是电子转移的账本。配平的目的不是"做对题目"，而是**准确记录每个原子得失多少电子**——这是后续查表判断和计算的基础。

### 2.2 原电池——让电子走外电路

**从问题出发**：锌片放入 $\mathrm{CuSO_4}$ 溶液，锌表面会析出红色铜，溶液蓝色变浅。电子直接从锌转移给 $\mathrm{Cu^{2+}}$，没有电流。怎么让电子"被迫"走外电路？

**Daniel 原电池装置**：

```
      外电路（导线 + 电流计）
   ┌─────────────────────────────┐
   │                             │
  (-)                           (+)
   Zn │ Zn²⁺(1 M) ║ Cu²⁺(1 M) │ Cu
      │  盐桥                       │
      └─────────────────────────────┘
```

**核心结构**（必须掌握）：
| 部件 | 作用 | 注意 |
|:---|:---|:---|
| **负极（阳极）** | 发生氧化反应，失电子 | 电子流出的一极 |
| **正极（阴极）** | 发生还原反应，得电子 | 电子流入的一极 |
| **外电路** | 电子从负极流向正极 | 电流方向与电子流相反 |
| **盐桥** | 维持两半电池电荷平衡，离子迁移 | 阳离子移向正极，阴离子移向负极 |
| **半电池** | 电极 + 电解质溶液 | 标准态时浓度为 1 mol/L |

**电子流向与离子迁移**：
- 电子流：$\mathrm{Zn \to Cu}$（外电路，负极到正极）
- 电流方向：$\mathrm{Cu \to Zn}$（与电子流相反）
- 盐桥中：$\mathrm{K^+}$ 移向正极区（补充被消耗的阳离子），$\mathrm{Cl^-}$ 移向负极区（中和多余的阳离子）

**电池符号写法**：
$$(-)\ \mathrm{Zn|Zn^{2+}(1.0\ M)\|\|Cu^{2+}(1.0\ M)|Cu}\ (+)$$

规则：
- 负极在左，正极在右
- "|" 表示相界面，"||" 表示盐桥
- 溶液浓度/气体分压必须标注
- 固体、纯液体不写进电池符号

**对比表格**：

| 维度 | 原电池 | 电解池 | 常见陷阱 |
|:---|:---|:---|:---|
| 能量转化 | 化学能 → 电能（自发） | 电能 → 化学能（非自发） | 电解池阴极=还原反应，但连接电源**负极** |
| 反应自发性 | $E_{\text{池}} > 0$，自发 | 需外加电压 > $E_{\text{池}}$ | 把电解池的阴极当成"负极" |
| 电极命名 | 负极/正极（按电势高低） | 阴极/阳极（按反应类型） | 原电池负极=阳极（氧化），正极=阴极（还原） |
| 盐桥 | 需要 | 不需要（通常直接连通） | — |

> 🔷 **拓展**（选讲）：浓差电池——两电极相同，但电解质浓度不同。$E^{\ominus}_{\text{池}} = 0$，电动势完全来自浓度差：$E = \dfrac{0.0592}{n}\lg\dfrac{c_{\text{浓}}}{c_{\text{稀}}}$。

### 2.3 标准电极电势 $E^{\ominus}$——查表判断的"海拔高度"

**从问题出发**：怎么知道 $\mathrm{Zn}$ 和 $\mathrm{Cu^{2+}}$ 的反应能不能自发？能不能定量比较不同氧化剂的强弱？

**核心概念**：
- 标准电极电势 $E^{\ominus}$ 是**相对值**——以标准氢电极（SHE）为基准：$\mathrm{2H^+ + 2e^- \rightleftharpoons H_2}$，规定 $E^{\ominus}_{\text{SHE}} = 0\ \mathrm{V}$
- $E^{\ominus}$ 是**还原电势**：数值越正，该电对的氧化态越容易被还原（氧化性越强）
- $E^{\ominus}$ 是**强度性质**：与物质的量无关，不能直接加减

**"海拔高度"类比**：
- 把每个电对想象成一座山，$E^{\ominus}$ 就是山顶的海拔
- 海拔高的山（$E^{\ominus}$ 大）上的氧化态"喜欢"往下走（被还原）
- 电子从低海拔（$E^{\ominus}$ 小，易氧化）流向高海拔（$E^{\ominus}$ 大，易还原）

**查表判断三步法**：

| 步骤 | 操作 | 检查点 |
|:---|:---|:---|
| 1 | 写出两个半反应的还原形式 | 确认查的是**还原电势** |
| 2 | 比较 $E^{\ominus}$：大者为正极（还原），小者为负极（氧化） | 不要颠倒 |
| 3 | 计算 $E^{\ominus}_{\text{池}} = E^{\ominus}_{\text{正}} - E^{\ominus}_{\text{负}}$ | $E^{\ominus}_{\text{池}} > 0$ 则反应自发 |

**常用标准电极电势表（精简版）**：

| 电对（还原形式） | $E^{\ominus}\ /\ \mathrm{V}$ | 说明 |
|:---|:---:|:---|
| $\mathrm{F_2 + 2e^- \rightleftharpoons 2F^-}$ | +2.87 | 最强氧化剂之一 |
| $\mathrm{MnO_4^- + 8H^+ + 5e^- \rightleftharpoons Mn^{2+} + 4H_2O}$ | +1.51 | 酸性 $\mathrm{KMnO_4}$ 强氧化性来源 |
| $\mathrm{Cl_2 + 2e^- \rightleftharpoons 2Cl^-}$ | +1.36 | 卤素中氧化性较强 |
| $\mathrm{Cr_2O_7^{2-} + 14H^+ + 6e^- \rightleftharpoons 2Cr^{3+} + 7H_2O}$ | +1.33 | 酸性 $\mathrm{K_2Cr_2O_7}$ |
| $\mathrm{Br_2 + 2e^- \rightleftharpoons 2Br^-}$ | +1.07 | — |
| $\mathrm{Fe^{3+} + e^- \rightleftharpoons Fe^{2+}}$ | +0.77 | 常用氧化还原指示 |
| $\mathrm{I_2 + 2e^- \rightleftharpoons 2I^-}$ | +0.54 | — |
| $\mathrm{Cu^{2+} + 2e^- \rightleftharpoons Cu}$ | +0.34 | — |
| $\mathrm{2H^+ + 2e^- \rightleftharpoons H_2}$ | 0.00 | 标准氢电极 |
| $\mathrm{Pb^{2+} + 2e^- \rightleftharpoons Pb}$ | -0.13 | — |
| $\mathrm{Fe^{2+} + 2e^- \rightleftharpoons Fe}$ | -0.44 | — |
| $\mathrm{Zn^{2+} + 2e^- \rightleftharpoons Zn}$ | -0.76 | 常用负极材料 |
| $\mathrm{Al^{3+} + 3e^- \rightleftharpoons Al}$ | -1.66 | — |
| $\mathrm{Na^+ + e^- \rightleftharpoons Na}$ | -2.71 | 强还原剂 |
| $\mathrm{Li^+ + e^- \rightleftharpoons Li}$ | -3.04 | 最强还原剂之一 |

**$E^{\ominus}$ 与 $\Delta G^{\ominus}$、$K$ 的关系**（必记）：

$$\Delta G^{\ominus} = -nFE^{\ominus}_{\text{池}}$$

$$\lg K = \frac{nE^{\ominus}_{\text{池}}}{0.0592}$$（25°C 时）

**数量级直觉**：
| $E^{\ominus}_{\text{池}}$ | $\lg K$（$n=1$） | 反应彻底程度 |
|:---:|:---:|:---|
| +0.2 V | 3.4 | $K \approx 10^3$，反应较完全 |
| +0.4 V | 6.8 | $K \approx 10^7$，反应很完全 |
| +0.6 V | 10.1 | $K \approx 10^{10}$，基本进行到底 |

> [!info]- 完整推导与教材对照（课后复习用）
>
> **教材来源**：宋天佑《无机化学》§10.3、[[教学逻辑提炼-周坤无机新课-酸碱理论与电化学-第一轮]]
>
> **完整推导**：
> 从电功与 Gibbs 自由能的关系出发：
> - 电池做的最大电功 = $-nFE_{\text{池}}$
> - 可逆过程中，$\Delta G = w_{\text{max}} = -nFE_{\text{池}}$
> - 标准态下：$\Delta G^{\ominus} = -nFE^{\ominus}_{\text{池}}$
> - 结合 $\Delta G^{\ominus} = -RT\ln K$，得：
>   $$\ln K = \frac{nFE^{\ominus}_{\text{池}}}{RT} = \frac{nE^{\ominus}_{\text{池}}}{0.02569}$$
>   $$\lg K = \frac{nE^{\ominus}_{\text{池}}}{0.0592}$$（25°C，$\dfrac{RT\ln 10}{F} = 0.05916\ \mathrm{V}$）
>
> **与教材的对照说明**：
> - 教材中常先讲 $\Delta G^{\ominus} = -nFE^{\ominus}$，再推导与 $K$ 的关系
> - 周坤笔记强调：第一轮不需要完整推导，但**必须记住数量级直觉**——$E^{\ominus}_{\text{池}} \approx 0.2\text{--}0.4\ \mathrm{V}$ 时反应已较完全

**阶段性小结**：$E^{\ominus}$ 是热力学判据，回答"能不能"，不回答"快不快"。$E^{\ominus}_{\text{池}} > 0 \iff \Delta G^{\ominus} < 0 \iff$ 反应自发。

### 2.4 Nernst 方程——浓度与 pH 改变电势

**从问题出发**：标准电极电势表上的数据都是在 1 mol/L、1 bar、25°C 下测的。如果 $\mathrm{[Zn^{2+}] = 0.01\ M}$，锌电极的电势还等于 -0.76 V 吗？

**Nernst 方程（两种形式，必须掌握）**：

**通用形式（ln）**：
$$E = E^{\ominus} - \frac{RT}{nF}\ln Q$$

**25°C 实用形式（lg）**：
$$E = E^{\ominus} - \frac{0.0592}{n}\lg Q$$

**ln 与 lg 的切换**（重点！）：

$$\frac{RT}{nF}\ln Q = \frac{RT\ln 10}{nF}\lg Q = \frac{0.05916}{n}\lg Q \approx \frac{0.0592}{n}\lg Q$$

其中 $\dfrac{RT\ln 10}{F} = 0.05916\ \mathrm{V}$（25°C），竞赛中常取 0.0592。

> 💡 **记忆口诀**："RT 除以 nF，ln 变 lg 乘 2.303，约等于 0.0592"

**Q 的书写规则**：
- Q 是**反应商**，形式与平衡常数 $K$ 相同
- **固体、纯液体**活度取 1，**不写进 Q**
- **气体**用分压（单位 bar）
- **溶液**用浓度（单位 mol/L）

**电池形式的 Nernst 方程**：
$$E_{\text{池}} = E^{\ominus}_{\text{池}} - \frac{0.0592}{n}\lg Q$$

其中 $Q$ 是总反应的反应商。

**电极形式的 Nernst 方程**（对单个半反应）：
$$E = E^{\ominus} + \frac{0.0592}{n}\lg\frac{[\text{Ox}]}{[\text{Red}]}$$

> ⚠️ 注意符号！电池形式是"减"，电极形式是"加"——因为电极形式把还原态放分母，氧化态放分子。

**pH 对电势的影响——数值冲击**：

以 $\mathrm{MnO_4^- + 8H^+ + 5e^- \rightleftharpoons Mn^{2+} + 4H_2O}$ 为例：

$$E = E^{\ominus} + \frac{0.0592}{5}\lg\frac{[\mathrm{MnO_4^-}][\mathrm{H^+}]^8}{[\mathrm{Mn^{2+}}]}$$

假设 $[\mathrm{MnO_4^-}] = [\mathrm{Mn^{2+}}] = 1\ \mathrm{M}$：

| pH | $[\mathrm{H^+}]$ | $\lg[\mathrm{H^+}]^8$ | $E\ /\ \mathrm{V}$ |
|:---:|:---:|:---:|:---:|
| 0 | 1 | 0 | +1.51（标准态） |
| 1 | $10^{-1}$ | -8 | +1.42 |
| 3 | $10^{-3}$ | -24 | +1.22 |
| 5 | $10^{-5}$ | -40 | +1.03 |
| 7 | $10^{-7}$ | -56 | +0.83 |

**结论**：pH 从 0 升到 7，$\mathrm{MnO_4^-}$ 的氧化性从 +1.51 V 骤降到 +0.83 V，**几乎减半**！这就是为什么酸性 $\mathrm{KMnO_4}$ 是强氧化剂，而中性/碱性条件下明显变弱。

> [!info]- 完整推导与教材对照（课后复习用）
>
> **教材来源**：宋天佑《无机化学》§10.4、[[教学逻辑提炼-周坤无机新课-酸碱理论与电化学-第一轮]]
>
> **完整推导**：
> 从 $\Delta G = \Delta G^{\ominus} + RT\ln Q$ 出发：
> - $-nFE = -nFE^{\ominus} + RT\ln Q$
> - 两边除以 $-nF$：
>   $$E = E^{\ominus} - \frac{RT}{nF}\ln Q$$
> - 25°C 时，$\dfrac{RT}{F} = 0.02569\ \mathrm{V}$，$\dfrac{RT\ln 10}{F} = 0.05916\ \mathrm{V}$
> - 因此：
>   $$E = E^{\ominus} - \frac{0.0592}{n}\lg Q$$
>
> **周坤笔记特别提示**：
> - W2 中专门强调 Nernst 方程 ln vs lg 形式等价：$\dfrac{RT}{nF}\ln = \dfrac{0.05916}{n}\lg$
> - 考试中两种形式都可能出现，必须熟练切换
> - 不要死记"0.0592"，要知道它来自 $RT\ln 10/F$

**沉淀/络合对 $E$ 的影响**（选讲）：

如果氧化态或还原态被沉淀/络合消耗，其有效浓度降低，电势会改变。

**示例**：$\mathrm{Cu^{2+} + 2e^- \rightleftharpoons Cu}$，$E^{\ominus} = +0.34\ \mathrm{V}$。但 $\mathrm{Cu^{2+}}$ 与 $\mathrm{I^-}$ 生成 CuI 沉淀（$K_{\text{sp}}$ 很小），实际 $[\mathrm{Cu^{2+}}]$ 极低，使得 $E$ 大幅升高，足以氧化 $\mathrm{I^-}$。

#### 🔬 深入拓展：温度变化与 Nernst 方程（必读！）

> 📌 **教材来源**：宋天佑《无机化学》§10.4、[[教学洞察-Nernst方程]]

##### 为什么不能无脑套 0.0592？

**核心事实**：$0.0592 = \dfrac{RT\ln 10}{F}$ 只在 **25°C（298.15 K）** 成立！

| 温度 | $RT\ln 10 / F$ | 教材场景 |
|:---|:---:|:---|
| 25°C（298.15 K） | **0.05916** | 普化教材默认值 |
| 37°C（310.15 K） | **0.0651** | 生化教材（人体温度） |
| 0°C（273.15 K） | 0.0542 | 低温条件 |
| 100°C（373.15 K） | 0.0740 | 高温条件 |

> 💡 **经典开场冲突**（Zchem 教学法）："为什么有的教材 Nernst 方程系数是 0.0591，有的是 0.0651？不是实验误差，不是印刷错误——普化用 25°C，生化用 37°C 人体温度。"学生从此记住「温度必须保留，不能无脑套 0.0592」。

**非 25°C 的 Nernst 计算**：

**例题**：计算 50°C 时，$[\mathrm{Zn^{2+}}] = 0.10\ \mathrm{M}$ 的锌电极电势。

$$E = E^{\ominus} - \frac{RT}{nF}\ln\frac{1}{[\mathrm{Zn^{2+}}]}$$

$T = 323.15\ \mathrm{K}$，$R = 8.314\ \mathrm{J \cdot mol^{-1} \cdot K^{-1}}$，$F = 96485\ \mathrm{C \cdot mol^{-1}}$

$$\frac{RT}{F} = \frac{8.314 \times 323.15}{96485} = 0.02784\ \mathrm{V}$$

$$E = -0.76 - \frac{0.02784}{2}\ln\frac{1}{0.10} = -0.76 - 0.01392 \times (-2.303) = -0.76 + 0.0321 = -0.728\ \mathrm{V}$$

> ⚠️ **高频错误（~45%）**：所有题都用 0.0591 不查温度。口诀：「298 用 0.0591，其他温度用 RT/nF」。

##### E° 与平衡常数 K 的定量关系

从 $\Delta G^{\ominus} = -nFE^{\ominus}_{\text{池}}$ 和 $\Delta G^{\ominus} = -RT\ln K$ 出发：

$$\ln K = \frac{nFE^{\ominus}_{\text{池}}}{RT}$$

25°C 时：

$$\lg K = \frac{nE^{\ominus}_{\text{池}}}{0.0592}$$

**数值直觉**（25°C）：

| $n$ | $E^{\ominus}_{\text{池}}$ | $\lg K$ | $K$ | 反应程度 |
|:---:|:---:|:---:|:---:|:---|
| 1 | +0.10 V | 1.7 | ~50 | 不够彻底 |
| 1 | +0.20 V | 3.4 | ~2.5×10³ | 基本完全 |
| 1 | +0.30 V | 5.1 | ~1.3×10⁵ | 很彻底 |
| 2 | +0.10 V | 3.4 | ~2.5×10³ | 基本完全 |
| 2 | +0.20 V | 6.8 | ~6.3×10⁶ | 非常彻底 |

> 💡 **判据**：$E^{\ominus}_{\text{池}} \gtrsim 0.2\ \mathrm{V}$ 时反应基本可视为完全（$K > 10^3$）。

##### 浓差电池

当两个半电池的电极相同、电解质相同，仅**浓度不同**时，也能产生电势——没有化学反应，纯粹靠浓度差驱动。

**示例**：$\mathrm{Zn|Zn^{2+}(0.010\ M)\|\|Zn^{2+}(1.0\ M)|Zn}$

$$E^{\ominus}_{\text{池}} = E^{\ominus}(\mathrm{Zn^{2+}/Zn}) - E^{\ominus}(\mathrm{Zn^{2+}/Zn}) = 0$$

但 $E_{\text{池}} \neq 0$！用 Nernst 方程：

$$E_{\text{池}} = 0 - \frac{0.0592}{2}\lg\frac{0.010}{1.0} = -\frac{0.0592}{2} \times (-2) = +0.059\ \mathrm{V}$$

> 浓差电池的电势来自熵效应——高浓度侧 Zn²⁺ 倾向于"稀释"，推动电子流动。放电终点是两侧浓度相等。

### 2.6 电解池——外加电压驱动非自发反应

> 📌 **教材来源**：宋天佑《无机化学》§10.5、华彤文《普通化学原理》§10.6

**与原电池的本质区别**：

| | 原电池（Galvanic cell） | 电解池（Electrolytic cell） |
|:---|:---|:---|
| **驱动力** | 自发反应（$\Delta G < 0$） | 外加电源强迫非自发反应（$\Delta G > 0$） |
| **能量转换** | 化学能 → 电能 | 电能 → 化学能 |
| **负极** | 氧化（阳极），电势低 | 还原（阴极），接电源负极 |
| **正极** | 还原（阴极），电势高 | 氧化（阳极），接电源正极 |

> ⚠️ **命名陷阱**：原电池的负极 = 阳极（氧化），正极 = 阴极（还原）。但电解池的"负极"接电源负极 = 阴极（还原），和原电池**恰好相反**！学生常混淆——口诀：「原电池负氧，电解池负还」。

**分解电压与超电势**：

- **理论分解电压**：由 Nernst 方程计算的最小外加电压
- **实际分解电压** > 理论分解电压，差值 = **超电势**

超电势来源：电极极化（活化极化 + 浓差极化）+ 溶液内阻

> 🔷 **超电势的定量计算**属于第二轮/电极动力学内容。本轮只建立概念：实际需要的电压总是大于理论值。

**电解应用举例**：

| 应用 | 阳极反应 | 阴极反应 | 意义 |
|:---|:---|:---|:---|
| 电解饱和食盐水 | $2\mathrm{Cl^- \to Cl_2 + 2e^-}$ | $2\mathrm{H_2O + 2e^- \to H_2 + 2OH^-}$ | 氯碱工业 |
| 电解氧化铝（冰晶石熔融） | $2\mathrm{O^{2-} \to O_2 + 4e^-}$（碳阳极消耗） | $\mathrm{Al^{3+} + 3e^- \to Al}$ | 铝的工业冶炼 |
| 电解精炼铜 | 粗铜溶解：$\mathrm{Cu \to Cu^{2+} + 2e^-}$ | 纯铜沉积：$\mathrm{Cu^{2+} + 2e^- \to Cu}$ | 获得 99.99% 纯铜 |
| 水的电解 | $2\mathrm{H_2O \to O_2 + 4H^+ + 4e^-}$ | $2\mathrm{H_2O + 2e^- \to H_2 + 2OH^-}$ | 制氢 |

**电解计算（法拉第定律）**：

$$Q = It,\quad n_e = \frac{Q}{F} = \frac{It}{96485}$$

> 💡 电解计算的核心：先算通过的电子物质的量，再根据电极反应的化学计量关系换算产物的量。

**例题**：以 2.00 A 电流电解 $\mathrm{CuSO_4}$ 溶液 30 min，阴极析出多少克铜？

$$n_e = \frac{It}{F} = \frac{2.00 \times 30 \times 60}{96485} = 0.0373\ \mathrm{mol}$$

$$\mathrm{Cu^{2+} + 2e^- \to Cu},\quad n(\mathrm{Cu}) = \frac{n_e}{2} = 0.01865\ \mathrm{mol}$$

$$m(\mathrm{Cu}) = 0.01865 \times 63.55 = 1.18\ \mathrm{g}$$

### 2.7 Latimer 图（选讲/点到为止）

**Latimer 图**是元素不同氧化态之间标准电极电势的直观表示。

**Mn 在酸性介质中的 Latimer 图**：

$$\mathrm{MnO_4^- \xrightarrow{+0.56\ V} MnO_4^{2-} \xrightarrow{+2.26\ V} MnO_2 \xrightarrow{+0.95\ V} Mn^{3+} \xrightarrow{+1.51\ V} Mn^{2+} \xrightarrow{-1.18\ V} Mn}$$

**读图规则**：
- 箭头上的数字是**从左到右**的还原电势
- 判断歧化：若右侧电势 > 左侧电势，中间氧化态不稳定，会歧化

**计算中间电势**（重要！不能直接加减 $E^{\ominus}$）：

已知：$\mathrm{MnO_4^- \to MnO_2}$（$E^{\ominus}_1 = +1.70\ \mathrm{V}$，$n_1 = 3$）
$\mathrm{MnO_2 \to Mn^{2+}}$（$E^{\ominus}_2 = +1.23\ \mathrm{V}$，$n_2 = 2$）

求：$\mathrm{MnO_4^- \to Mn^{2+}}$ 的 $E^{\ominus}_3$（$n_3 = 5$）

**必须通过 $\Delta G^{\ominus} = -nFE^{\ominus}$ 转换**：
$$\Delta G^{\ominus}_3 = \Delta G^{\ominus}_1 + \Delta G^{\ominus}_2$$
$$-n_3FE^{\ominus}_3 = -n_1FE^{\ominus}_1 - n_2FE^{\ominus}_2$$
$$E^{\ominus}_3 = \frac{n_1E^{\ominus}_1 + n_2E^{\ominus}_2}{n_3} = \frac{3 \times 1.70 + 2 \times 1.23}{5} = +1.51\ \mathrm{V}$$

> ⚠️ **常见陷阱**：$E^{\ominus}$ 不能直接算术平均！必须用 $\Delta G^{\ominus}$ 过渡。

### 2.8 🔬 深入拓展：Frost 图与 Pourbaix 图（Weller Ch5）

**Frost 图**与 **Pourbaix 图**是 Latimer 图的两个延伸工具——三者同出一源（标准电极电势数据），但回答不同层次的问题：

| 工具 | 回答的问题 | 核心原理 |
|:---|:---|:---|
| Latimer 图 | 各氧化态间的电势是多少？ | 线状还原电势链 |
| **Frost 图** | 哪个氧化态稳定？会歧化吗？ | $\nu E^{\ominus}$ 对氧化数 $N$ 作图 |
| **Pourbaix 图** | 在给定的 pH 和 $E$ 下，哪个物种稳定？ | pH-E 二维相图 |

#### 2.8.1 Frost 图

**构造方法**：从元素单质（$N=0$）开始，将各步 $\nu E^{\ominus}$ 对氧化数 $N$ 累加：

$$\text{纵坐标} = \sum_{i=1}^{N} \nu_i E^{\ominus}_i$$

**核心判据**：
- **凸形点**（点在连线上方）→ 能量高，倾向于歧化
- **凹形点**（点在连线下方）→ 能量低，歧化产物倾向于反歧化（归中）
- **斜率** = 该步的还原电势 $E^{\ominus}$——斜率越陡，氧化性越强

**经典实例**：Mn 的 Frost 图（酸性溶液）

| 氧化态 | $N$ | $\nu E^{\ominus}$（V）| 纵坐标（V·e⁻）|
|:---|:---:|:---:|:---:|
| Mn(0) | 0 | 0 | 0 |
| Mn²⁺ | +2 | 2×(−1.18) | −2.36 |
| Mn³⁺ | +3 | 1×(1.51) | −0.85 |
| MnO₂ | +4 | 1×(0.95) | +0.10 |
| MnO₄²⁻ | +6 | 2×(1.70) | +3.50 |
| MnO₄⁻ | +7 | 1×(0.56) | +4.06 |

- **Mn³⁺ 处于凸形**（−0.85 高于 Mn²⁺→MnO₂ 连线）→ 水溶液中自发歧化：$2\mathrm{Mn^{3+}} + 2\mathrm{H_2O} \to \mathrm{MnO_2} + \mathrm{Mn^{2+}} + 4\mathrm{H^+}$
- **MnO₄²⁻ 也在凸形** → 酸性条件下歧化为 MnO₄⁻ + MnO₂

> 💡 **记忆口诀**：Frost 凸起 = 不稳定 = 歧化；Frost 凹陷 = 稳定 = 归中产物。

#### 2.8.2 Pourbaix 图

**基本结构**：以 pH 为横轴、电极电势 $E$ 为纵轴的二维相图，每个区域标注该条件下热力学最稳定的物种。

**三种边界类型**（均从 Nernst 方程导出）：

| 边界类型 | Nernst 形式 | 斜率 | 含义 |
|:---|:---|:---:|:---|
| 水平线 | $E = E^{\ominus}$ | 0 | 仅涉及 e⁻，与 pH 无关（如 $\mathrm{Fe^{3+}/Fe^{2+}}$） |
| 竖直线 | $\mathrm{pH} = \mathrm{p}K_a$ | ∞ | 仅涉及 H⁺，与 $E$ 无关（如 $\mathrm{Fe^{3+}}$ 水解为 $\mathrm{Fe(OH)_3}$） |
| 斜线 | $E = E^{\ominus} - \frac{0.059m}{n}\mathrm{pH}$ | −0.059m/n | 同时涉及 e⁻ 和 H⁺（如 $\mathrm{MnO_2/Mn^{2+}}$） |

**水的稳定区**（虚线）：
- 上线：$E(\mathrm{O_2/H_2O}) = 1.23 - 0.059\,\mathrm{pH}$
- 下线：$E(\mathrm{H^+/H_2}) = 0 - 0.059\,\mathrm{pH}$
- 两线之间：水热力学稳定；上线以上：水被氧化出 O₂；下线以下：水被还原出 H₂

**竞赛应用**：
1. 判断某 pH 和电位下某元素的稳定氧化态
2. 判断金属在特定水环境中的腐蚀倾向
3. 推断氧化还原滴定中控制 pH 的必要性

> 📎 **三图关系总结**：Latimer 图（数字链）→ 加和 → Frost 图（一维曲线）→ 叠加 pH 维度 → Pourbaix 图（二维平面）。三者是决赛氧化还原分析的完整工具箱。

**教材关联**：《Inorganic Chemistry》第6版（Weller）§5.12–5.14；[[07-资料提炼/书籍提炼/提炼-无机化学第6版-第5章-氧化还原反应180-212]]

## 三、课堂例题

### 例题 1（⭐⭐ 基础）：查表判断反应方向

**题目**：查标准电极电势表，判断下列反应在标准状态下能否自发进行：

(1) $\mathrm{2Fe^{3+} + 2I^- \longrightarrow 2Fe^{2+} + I_2}$

(2) $\mathrm{Br_2 + 2Fe^{2+} \longrightarrow 2Br^- + 2Fe^{3+}}$

**分析提示**：
- 第一步：写出两个半反应的还原形式，查出 $E^{\ominus}$
- 第二步：$E^{\ominus}$ 大者为正极（被还原），小者为负极（被氧化）
- 第三步：计算 $E^{\ominus}_{\text{池}} = E^{\ominus}_{\text{正}} - E^{\ominus}_{\text{负}}$

**完整解答路径**：

**(1)** 相关电对：
- $\mathrm{Fe^{3+} + e^- \rightleftharpoons Fe^{2+}}$，$E^{\ominus} = +0.77\ \mathrm{V}$
- $\mathrm{I_2 + 2e^- \rightleftharpoons 2I^-}$，$E^{\ominus} = +0.54\ \mathrm{V}$

$E^{\ominus}(\mathrm{Fe^{3+}/Fe^{2+}}) > E^{\ominus}(\mathrm{I_2/I^-})$，故：
- 正极（还原）：$\mathrm{Fe^{3+} + e^- \to Fe^{2+}}$
- 负极（氧化）：$\mathrm{2I^- \to I_2 + 2e^-}$

$$E^{\ominus}_{\text{池}} = 0.77 - 0.54 = +0.23\ \mathrm{V} > 0$$

**结论**：反应(1)能自发进行。$\mathrm{Fe^{3+}}$ 是氧化剂，$\mathrm{I^-}$ 是还原剂。

**(2)** 相关电对：
- $\mathrm{Br_2 + 2e^- \rightleftharpoons 2Br^-}$，$E^{\ominus} = +1.07\ \mathrm{V}$
- $\mathrm{Fe^{3+} + e^- \rightleftharpoons Fe^{2+}}$，$E^{\ominus} = +0.77\ \mathrm{V}$

$E^{\ominus}(\mathrm{Br_2/Br^-}) > E^{\ominus}(\mathrm{Fe^{3+}/Fe^{2+}})$，故：
- 正极（还原）：$\mathrm{Br_2 + 2e^- \to 2Br^-}$
- 负极（氧化）：$\mathrm{2Fe^{2+} \to 2Fe^{3+} + 2e^-}$

$$E^{\ominus}_{\text{池}} = 1.07 - 0.77 = +0.30\ \mathrm{V} > 0$$

**结论**：反应(2)能自发进行。$\mathrm{Br_2}$ 是氧化剂，$\mathrm{Fe^{2+}}$ 是还原剂。

### 例题 2（⭐⭐ 基础）：Nernst 方程代入计算

**题目**：计算下列电池的电动势（25°C）：

$$\mathrm{Zn|Zn^{2+}(0.01\ M)\|\|Cu^{2+}(1.0\ M)|Cu}$$

已知：$E^{\ominus}(\mathrm{Zn^{2+}/Zn}) = -0.76\ \mathrm{V}$，$E^{\ominus}(\mathrm{Cu^{2+}/Cu}) = +0.34\ \mathrm{V}$

**分析提示**：
- 先算标准态电动势 $E^{\ominus}_{\text{池}}$
- 再写总反应，确定反应商 $Q$
- 代入 Nernst 方程：$E_{\text{池}} = E^{\ominus}_{\text{池}} - \dfrac{0.0592}{n}\lg Q$

**完整解答路径**：

**Step 1**：标准态电动势
$$E^{\ominus}_{\text{池}} = E^{\ominus}_{\text{正}} - E^{\ominus}_{\text{负}} = 0.34 - (-0.76) = +1.10\ \mathrm{V}$$

**Step 2**：总反应
$$\mathrm{Zn + Cu^{2+} \longrightarrow Zn^{2+} + Cu}$$

**Step 3**：反应商 $Q$
$$Q = \frac{[\mathrm{Zn^{2+}}]}{[\mathrm{Cu^{2+}}]} = \frac{0.01}{1.0} = 0.01$$
（固体 Zn 和 Cu 活度取 1，不写进 Q）

**Step 4**：Nernst 方程（$n = 2$）
$$E_{\text{池}} = 1.10 - \frac{0.0592}{2}\lg(0.01) = 1.10 - 0.0296 \times (-2) = 1.10 + 0.059 = +1.16\ \mathrm{V}$$

**结论**：由于 $[\mathrm{Zn^{2+}}]$ 降低，反应推动力增大，电动势从 +1.10 V 升至 +1.16 V。

> 💡 **课堂任务**：学生跟随板书推导，5 min。重点检查 Q 的书写（固体是否遗漏）和 lg 计算。

### 例题 3（⭐⭐ 基础）：pH 对氧化性的影响

**题目**：计算 $\mathrm{MnO_4^-}$ 在 pH = 5 时的电极电势（假设 $[\mathrm{MnO_4^-}] = [\mathrm{Mn^{2+}}] = 1.0\ \mathrm{M}$，25°C）。

已知：$E^{\ominus}(\mathrm{MnO_4^-/Mn^{2+}}) = +1.51\ \mathrm{V}$

**分析提示**：
- 先写出还原半反应
- 确定 $n$ 和 $Q$ 的表达式
- 注意 $[\mathrm{H^+}]$ 要代入 pH = 5 的值

**完整解答路径**：

**Step 1**：半反应
$$\mathrm{MnO_4^- + 8H^+ + 5e^- \rightleftharpoons Mn^{2+} + 4H_2O}$$

**Step 2**：$n = 5$，$Q$ 的表达式
$$Q = \frac{[\mathrm{Mn^{2+}}]}{[\mathrm{MnO_4^-}][\mathrm{H^+}]^8}$$

**Step 3**：代入数值
- $[\mathrm{Mn^{2+}}] = [\mathrm{MnO_4^-}] = 1.0\ \mathrm{M}$
- pH = 5，$[\mathrm{H^+}] = 10^{-5}\ \mathrm{M}$

$$Q = \frac{1.0}{1.0 \times (10^{-5})^8} = \frac{1}{10^{-40}} = 10^{40}$$

**Step 4**：Nernst 方程
$$E = E^{\ominus} - \frac{0.0592}{5}\lg Q = 1.51 - \frac{0.0592}{5}\lg(10^{40})$$
$$E = 1.51 - 0.01184 \times 40 = 1.51 - 0.474 = +1.04\ \mathrm{V}$$

**结论**：pH 从 0 升到 5，$\mathrm{MnO_4^-}$ 的电极电势从 +1.51 V 降到 +1.04 V，氧化性大幅减弱。

> 💡 **课堂任务**：学生独立计算，3 min。教师强调 $[\mathrm{H^+}]^8$ 的指数不要漏掉。

### 例题 4（⭐⭐⭐ 拓展）：化学电源电动势计算

**题目**：某 $\mathrm{Zn-I_2}$ 二次电池，正极区发生反应：$\mathrm{2Cu^{2+} + I_2 + 4e^- \rightleftharpoons 2CuI}$，$E^{\ominus} = +0.70\ \mathrm{V}$。电池表示为：

$$(-)\ \mathrm{Zn|ZnSO_4(1.0\ M)\|\|CuCl_2(1.0\ M)|I_2(s)}\ (+)$$

以 200 mA 恒电流放电 482 s 后，计算电池电动势。

已知：$E^{\ominus}(\mathrm{Zn^{2+}/Zn}) = -0.76\ \mathrm{V}$

**分析提示**：
- 先算转移电子的物质的量：$n_e = \dfrac{It}{F}$
- 分别计算正极区和负极区放电后的浓度变化
- 对两个电极分别用 Nernst 方程，再相减

**完整解答路径**：

**Step 1**：转移电子数
$$n_e = \frac{It}{F} = \frac{0.200\ \mathrm{A} \times 482\ \mathrm{s}}{96485\ \mathrm{C/mol}} = 1.00 \times 10^{-3}\ \mathrm{mol} = 1.00\ \mathrm{mmol}$$

**Step 2**：负极浓度变化
$$\mathrm{Zn \longrightarrow Zn^{2+} + 2e^-}$$
生成 $\mathrm{Zn^{2+}}$：$\dfrac{1.00}{2} = 0.50\ \mathrm{mmol}$

假设溶液体积为 1.0 L：
$$[\mathrm{Zn^{2+}}] = 1.0 + 0.50 = 1.5\ \mathrm{M}$$

负极电势：
$$E_{\text{负}} = E^{\ominus}(\mathrm{Zn^{2+}/Zn}) + \frac{0.0592}{2}\lg[\mathrm{Zn^{2+}}]$$
$$E_{\text{负}} = -0.76 + \frac{0.0592}{2}\lg(1.5) = -0.76 + 0.005 = -0.755\ \mathrm{V}$$

**Step 3**：正极浓度变化
$$\mathrm{2Cu^{2+} + I_2 + 4e^- \longrightarrow 2CuI}$$
消耗 $\mathrm{Cu^{2+}}$：$\dfrac{1.00 \times 2}{4} = 0.50\ \mathrm{mmol}$

$$[\mathrm{Cu^{2+}}] = 1.0 - 0.50 = 0.50\ \mathrm{M}$$

正极电势（$\mathrm{I_2}$ 和 CuI 是固体，活度取 1）：
$$E_{\text{正}} = E^{\ominus} + \frac{0.0592}{4}\lg[\mathrm{Cu^{2+}}]^2$$
$$E_{\text{正}} = 0.70 + \frac{0.0592}{4}\lg(0.50)^2 = 0.70 + \frac{0.0592}{4} \times (-0.602) = 0.70 - 0.009 = +0.691\ \mathrm{V}$$

**Step 4**：电池电动势
$$E_{\text{池}} = E_{\text{正}} - E_{\text{负}} = 0.691 - (-0.755) = +1.45\ \mathrm{V}$$

**结论**：放电后电动势约为 +1.45 V。

> 🔷 **拓展例题**（可选）：Latimer 图计算——已知 $\mathrm{MnO_4^- \to MnO_2}$ 的 $E^{\ominus} = +1.70\ \mathrm{V}$（$n=3$），$\mathrm{MnO_2 \to Mn^{2+}}$ 的 $E^{\ominus} = +1.23\ \mathrm{V}$（$n=2$），求 $\mathrm{MnO_4^- \to Mn^{2+}}$ 的 $E^{\ominus}$（$n=5$）。
>
> **答案**：$E^{\ominus}_3 = \dfrac{3 \times 1.70 + 2 \times 1.23}{5} = +1.51\ \mathrm{V}$。注意不能直接算术平均！

## 六、本节形成的核心结论

1. **电极电势是热力学判据**：$E^{\ominus}$ 越正，氧化态氧化性越强；$E^{\ominus}_{\text{池}} > 0 \iff \Delta G^{\ominus} < 0 \iff$ 反应自发。但它只回答"能不能"，不回答"快不快"。

2. **非标准态必须用 Nernst 方程**：只要题目出现具体浓度、分压、pH 偏离标准态，立刻从 $E^{\ominus}$ 切换到 $E = E^{\ominus} - \dfrac{0.0592}{n}\lg Q$。ln 形式和 lg 形式等价：$\dfrac{RT}{nF}\ln = \dfrac{0.0592}{n}\lg$。

3. **原电池的核心是电子转移的空间分离**：负极氧化失电子，正极还原得电子，盐桥维持电荷平衡。电池电动势 $E_{\text{池}} = E_{\text{正}} - E_{\text{负}}$。

4. **$E^{\ominus}$ 不能直接加减**：必须通过 $\Delta G^{\ominus} = -nFE^{\ominus}$ 转换。Latimer 图计算中间电势时尤其注意。

**这些结论来自哪些问题？**

| 结论 | 回答了本节哪个问题 | 对应 KP/专题 |
|:---|:---|:---|
| 结论一 | 如何判断反应能否自发？ | [[标准电极电势]] |
| 结论二 | 浓度/pH 改变时氧化性怎么变？ | [[Nernst方程]] |
| 结论三 | 原电池为什么能产生电流？ | [[原电池]] |
| 结论四 | Latimer 图怎么算中间电势？ | [[Latimer图]] |

## 七、⚠️ 易错点预警

| 错误想法 | 为什么错 | 正确理解 |
|:---|:---|:---|
| "$E^{\ominus}$ 可以直接加减" | $E^{\ominus}$ 是强度性质，与物质的量无关 | 必须通过 $\Delta G^{\ominus} = -nFE^{\ominus}$ 转换后再加减 |
| "Nernst 方程中固体也要写进 Q" | 固体、纯液体活度取 1 | 只有溶液浓度和气体分压进 Q |
| "ln 形式和 lg 形式系数相同" | $\ln x = 2.303\lg x$，系数不同 | $\dfrac{RT}{nF}\ln Q = \dfrac{0.0592}{n}\lg Q$，记住 0.0592 来自 $RT\ln 10/F$ |
| "任何温度都用 0.0592" 🔗 | **~45% 发生率**——0.0592 只在 298K 有效！ | 换温度必须用 $RT/nF$；口诀「298 用 0.0592，其他温度用 RT/nF」 |
| "n 取总反应配平系数即可" 🔗 | **~35% 发生率**——n 是半反应转移电子数 | n = 氧化剂半反应中的电子数 = 还原剂半反应中的电子数（配平后相等） |
| "原电池负极就是阴极" | 命名体系不同 | 原电池：负极/正极（按电势）；电解池：阴极/阳极（按反应类型）。口诀「原电池负氧，电解池负还」 |
| "$E^{\ominus}_{\text{池}} > 0$ 就一定反应很快" | $E^{\ominus}$ 是热力学判据，与速率无关 | 热力学可行 ≠ 动力学快速，实际速率还受活化能、催化剂等影响 |
| "电池符号中溶液浓度可以省略" | 标准态必须标注浓度 | 非标准态更要标注，否则无法计算 |
| "Q 产物写分母、反应物写分子" 🔗 | **~30% 发生率**——把 Q 写反了 | Q 的写法同 K：产物在上（分子）、反应物在下（分母）；纯固纯液不写入 |

## 十、下节衔接

**下节课**：[[定量化学分析]]

**衔接点**：
- 本节已经建立了两件事：一是**用热力学量判断反应能否发生**，二是**用浓度/pH 修正实际条件下的反应驱动力**。
- 下节转入定量分析时，问题会从“能不能发生”进一步推进到“**到底进行了多少、怎样判定终点、怎样控制误差**”。
- 最自然的桥梁不是“电化学换题型”，而是“定量语言继续升级”：
  - 在本节里，我们已经看到浓度进入 Nernst 方程会直接改变电势；
  - 在下节里，浓度测定本身会成为主角，滴定终点、计量点、误差传播都会决定结论是否可信。
- 可以直接向学生抛出过桥问题：
  - 既然 `E` 会随浓度变化，那实验里怎样才能知道某个离子浓度到底是多少？
  - 既然理论上可自发，实验上又怎样判断“反应已经恰好完成”而不是“只完成了大半”？

### 10.1 第二轮去向

- 第二轮最自然的两条承接线是：
  - 元素化学线：把 `E° / Nernst / 氧化态变化` 落到具体元素体系，如 `Mn / Cr / Fe / Cu`
  - 分析化学线：把电势、终点、浓度判断推进到滴定、条件电势和误差控制
- 边界可直接说清：
  - 第一轮：会查表、会判断方向、会代 Nernst
  - 第二轮：进入条件电势、元素体系细节、与沉淀/配位/滴定联动

### 10.2 学生讲义配套建议

- 建议补一张“电化学判断卡”：
  - `谁是氧化剂 / 谁是还原剂`
  - `何时只看 E°`
  - `何时必须切 Nernst`
- 真题入口优先建议接：
  - `E° 查表判断`
  - `浓度 / pH 对电势影响`
