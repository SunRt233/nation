# 物化综合计算 · 速查卡

> 本卡的核心思想：**物化不是一堆散公式——是一条计算链**。从题目给的数据出发，沿链走到目标量。每条链上的每一环你都学过，本卡只教你怎么把它们串起来。

---

## 一、物化计算链总地图

```
题目给的任意数据 → 沿链走到目标量
    │
    ├─ 热力学链
    │   E° ──→ ΔG° = −nFE° ──→ lnK = −ΔG°/RT ──→ K ──→ 平衡浓度
    │   ΔH_f° ──→ ΔH_r° ──→ ΔG° = ΔH° − TΔS° ──→ 同上
    │   p,T 数据 ──→ Clapeyron/Clausius-Clapeyron ──→ ΔH_vap/ΔH_sub
    │
    ├─ 电化学链
    │   E ──→ E° (Nernst) ──→ ΔG ──→ K ──→ …
    │   E°₁, E°₂ ──→ E°_cell = E°_cathode − E°_anode
    │   电流/时间 ──→ Q = It ──→ n(e⁻) ──→ n(产物) ──→ m(产物)
    │
    ├─ 动力学链
    │   k₁, k₂, T₁, T₂ ──→ Arrhenius ──→ Ea ──→ 或 A
    │   浓度-时间数据 ──→ 积分速率方程 ──→ k ──→ 反应级数
    │   k(T₁), k(T₂) ──→ ln(k₂/k₁) = −(Ea/R)(1/T₂−1/T₁)
    │
    └─ 跨链桥梁（最高频！）
        E° ←→ ΔG° ←→ K ←→ 平衡浓度 ←→ Ksp ←→ 溶解度
        ΔH° ←→ van't Hoff ←→ K(T) ←→ 平衡移动
        Ea ←→ Arrhenius ←→ k(T) ←→ 半衰期
```

## 二、核心公式速查

### 2.1 热力学七公式

| # | 公式 | 什么时候用 | 易错点 |
|:---:|:---|:---|:---|
| 1 | ΔG° = −RT lnK | 从 K 求 ΔG°，或反之 | K 必须用标准态！ |
| 2 | ΔG° = −nFE° | 从 E° 求 ΔG° | n = 转移电子数，别漏！ |
| 3 | ΔG = ΔH − TΔS | 从热数据求自发性 | ΔS 单位 J/K·mol，ΔH 单位 kJ/mol——注意单位！ |
| 4 | ΔG = ΔG° + RT lnQ | 非标准态的自发性 | Q 的写法与 K 完全一样，但用实际浓度 |
| 5 | ln(K₂/K₁) = −(ΔH°/R)(1/T₂ − 1/T₁) | van't Hoff——温度对K的影响 | 假设 ΔH° 不随 T 变 |
| 6 | dP/dT = ΔS/ΔV = ΔH/(TΔV) | Clapeyron——相界线斜率 | ΔV = V_终 − V_初 |
| 7 | ln(P₂/P₁) = −(ΔH_vap/R)(1/T₂ − 1/T₁) | Clausius-Clapeyron——蒸气压 vs T | 假设 ΔH_vap 不随 T 变，气体视为理想气体 |

### 2.2 电化学五公式

| # | 公式 | 什么时候用 |
|:---:|:---|:---|
| 1 | E = E° − (RT/nF) lnQ | Nernst 方程——非标态电势 |
| 2 | E = E° − (0.0592/n) lgQ | 298K 简化版（最常用！） |
| 3 | E°_cell = E°_cathode − E°_anode | 电池标准电动势 |
| 4 | Q = It = nF | 电量 ↔ 物质的量 |
| 5 | 比容量 (mAh/g) = nF/(3.6M) | 电池材料计算 |

### 2.3 动力学五公式

| # | 公式 | 什么时候用 |
|:---:|:---|:---|
| 1 | k = A·e^(−Ea/RT) | Arrhenius——求 k 或 Ea |
| 2 | ln(k₂/k₁) = −(Ea/R)(1/T₂ − 1/T₁) | 两个温度下的 k 比 |
| 3 | 0 级：[A] = [A]₀ − kt | c-t 线性关系判断 |
| 4 | 1 级：ln([A]₀/[A]) = kt | 最常用！t₁/₂ = ln2/k |
| 5 | 2 级：1/[A] − 1/[A]₀ = kt | t₁/₂ = 1/(k[A]₀) |

## 三、最高频跨链题型

### 题型 1：E° → K → 溶解度

```
已知 E°(Ag⁺/Ag) 和 E°(AgCl/Ag) → 求 Ksp(AgCl)

路径：E°(AgCl/Ag) = E°(Ag⁺/Ag) + (RT/F)ln[Ag⁺]
     当 [Cl⁻]=1 时，[Ag⁺]=Ksp
     → E°(AgCl/Ag) = E°(Ag⁺/Ag) + (0.0592)lgKsp
```

### 题型 2：ΔH° → K(T₁) → K(T₂) → 平衡转化率

```
已知 ΔH° 和 K(298) → 求 K(373) → 求转化率

路径：van't Hoff → K(373) → ICE 表 → 转化率
```

### 题型 3：BDE 热化学循环

```
已知 ΔH_f°(各物质) + BDE(某键) → 求另一 BDE

路径：ΔH_r° = ΣΔH_f°(产物) − ΣΔH_f°(反应物)
     ΔH_r° = ΣBDE(断键) − ΣBDE(成键)
     → 等式联立求未知 BDE
```

## 四、计算常见自杀行为

| 错误 | 为什么致命 | 怎么避免 |
|:---|:---|:---|
| kJ 和 J 混用 | ΔG = ΔH − TΔS 中 ΔH/kJ + ΔS/J·K⁻¹ → 全错 | 统一到 J 或统一到 kJ，养成标注单位的习惯 |
| n（电子数）漏乘 | ΔG° = −nFE° 中 n=1 而非 2 → 结果差一倍 | 先写半反应，再数电子数 |
| Q 的写法忘了固体 | K = [产物]/[反应物]，固体不写 → Q 也一样 | 写 K 表达式时先划掉所有 (s) 和 (l) |
| 平衡浓度 vs 初始浓度 | 代公式的是平衡浓度不是初始浓度 | ICE 表！每次都画 |
| ln 和 lg 混用 | Nernst: 0.0592 对应 lg，0.0257 对应 ln | 298K 用 0.0592/n × lgQ |
| 忘了体积变化 | 混合后浓度 = n_总/V_总 ≠ (c₁V₁+c₂V₂)/(V₁+V₂)…其实这个公式就是对的 | 重点是 n_总/V_总，不是直接平均浓度 |

## 五、计算链构建口诀

> **"给了什么 → 沿什么链 → 到目标量"**
>
> E° → ΔG° → lnK → K → 平衡浓度
>
> ΔH° → van't Hoff → K(T₂) → 转化率
>
> k(T₁),k(T₂) → Arrhenius → Ea → 机理判断

## 六、例题速练

### 例 1：E° 链

**题**：E°(Cu²⁺/Cu) = 0.34V，E°(Cu²⁺/Cu⁺) = 0.16V。求 E°(Cu⁺/Cu)。

**速解**：ΔG° = −nFE°。Cu²⁺→Cu: ΔG°₁ = −2F(0.34)；Cu²⁺→Cu⁺: ΔG°₂ = −1F(0.16)。Cu⁺→Cu: ΔG°₃ = ΔG°₁ − ΔG°₂ = −2F(0.34) + 1F(0.16) = −0.52F。n=1 → E° = 0.52V。

### 例 2：van't Hoff 链

**题**：反应 ΔH° = −50 kJ/mol，K(298)=10⁵。求 K(373)。

**速解**：ln(K₂/K₁) = −(ΔH°/R)(1/T₂−1/T₁) = −(−50000/8.314)(1/373−1/298) = 6013×(0.002681−0.003356) = 6013×(−0.000675) = −4.06 → K₂/K₁ = e^(−4.06) ≈ 0.017 → K(373) ≈ 1.7×10³。放热反应升温 K 减小 ✓。

## 八、自查清单

- [ ] 计算链总地图能默画（热力学链 + 电化学链 + 动力学链 + 跨链桥梁）
- [ ] ΔG° = −RTlnK = −nFE° 的双向转换熟练
- [ ] Nernst 298K 简化式 0.0592/n × lgQ 熟记
- [ ] van't Hoff 不混方向——升温 K 增大的是吸热反应
- [ ] Arrhenius 两个温度比 k 不列错 1/T₁ 和 1/T₂ 顺序
- [ ] 积分速率方程三条（0/1/2 级）能默写
- [ ] 半衰期公式：0 级 t₁/₂=[A]₀/2k、1 级 t₁/₂=ln2/k、2 级 t₁/₂=1/k[A]₀
- [ ] ICE 表每道平衡题必画
- [ ] kJ↔J 换算不混（1 kJ = 1000 J）
- [ ] 混合后浓度 = n_总/V_总，不是直接加浓度
