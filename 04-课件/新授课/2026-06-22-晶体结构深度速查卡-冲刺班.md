# 晶体结构深度与推断 · 速查卡

> Round 3 你学会了"看图识结构"——本卡把晶体推断从**识别层**升级到**反推层**：给一张投影图+几组数据，独立还原三维结构并推断缺陷→性质。

---

## 一、投影图→3D 还原 · 三步法

```
拿到一张陌生晶向投影图 → 三步还原
    │
    ├─ Step 1：识别投影方向（2 min）
    │   ├─ [100] → 矩形网格，原子排列最"正"
    │   ├─ [110] → 矩形但有 2 重对称轴沿对角线
    │   ├─ [111] → 六边形排列（立方晶系沿体对角线看）
    │   └─ [112] → 锯齿形层状排列——非主晶轴，2024 CeO₂ 就考了这个！
    │
    ├─ Step 2：数原子（3 min）
    │   ├─ 角 = 1/8，棱 = 1/4，面 = 1/2，体 = 1
    │   └─ ⚠️ 投影图中的"一个点"可能对应 z 轴不同高度的多个原子！
    │       └─ 技巧：先标每个原子的 z 坐标（分数坐标第 3 个数），再判断是否重叠
    │
    └─ Step 3：推配位数和连接方式（3 min）
        ├─ 从投影图量最近邻距离 → 配位数
        └─ 多面体连接稳定性：共顶点 > 共棱 > 共面
```

**一句话**：投影方向定→原子数清→连接方式明→化学式出

## 二、晶系判断 · 7 行速查表

> **铁律：先看角、再看边。**

| 晶系 | 轴长关系 | 轴角关系 | 一眼识别 |
|:---|:---|:---|:---|
| 立方 | a = b = c | α = β = γ = 90° | 全部相等+全部正交 |
| 四方 | a = b ≠ c | α = β = γ = 90° | 两个相等+全部正交 |
| 正交 | a ≠ b ≠ c | α = β = γ = 90° | 都不等但全部正交 |
| 六方 | a = b ≠ c | α = β = 90°, γ = 120° | 两个相等+一个 120° |
| 三方 | a = b = c | α = β = γ ≠ 90° | 全部相等但不正交 |
| 单斜 | a ≠ b ≠ c | α = γ = 90°, β ≠ 90° | 一个角不 90° |
| 三斜 | a ≠ b ≠ c | α ≠ β ≠ γ ≠ 90° | 全不相等+全不正交 |

**速判练习**：给 a=5.0, b=5.0, c=5.4, α=β=γ=90° → **四方**（a=b≠c, 全正交）

## 三、五大复杂结构 · 从多面体连接一眼识别

| 结构类型 | 通式 | 多面体连接方式 | 一眼识别口诀 |
|:---|:---|:---|:---|
| **钙钛矿** | ABX₃ | BX₆ 八面体**全部共顶点**→三维骨架；A 填 12 配位空隙 | "八面体共顶点 = 钙钛矿" |
| **尖晶石** | AB₂X₄ | BX₆ 八面体共棱成链 + AX₄ 四面体填隙 | "八面体共棱 + 四面体填隙" |
| **萤石** | AX₂ | AX₈ 立方体全部共棱 | "立方体共棱 = 萤石型" |
| **金红石** | AX₂ | AX₆ 八面体共棱成链 + 共顶点连链 | "八面体共棱成链 + 共顶点" |
| **NaCl 型** | AX | AX₆ 八面体全部共棱→三维 | "八面体全部共棱 = NaCl" |

## 四、缺陷化学 · Kröger-Vink 记号速查

### 4.1 四种缺陷类型

| 缺陷类型 | 描述 | 实例 | Kröger-Vink |
|:---|:---|:---|:---|
| **Schottky** | 等量阴阳离子空位 | NaCl | $V'_\text{Na} + V^\bullet_\text{Cl}$ |
| **Frenkel** | 空位 + 间隙对 | AgCl | $V'_\text{Ag} + \text{Ag}^\bullet_i$ |
| **非化学计量** | 金属缺位 | Fe₁₋ₓO | $V''_\text{Fe} + 2h^\bullet$ |
| **氧空位** | 还原产生 | CeO₂₋ₓ | $V_\text{O}^{\bullet\bullet} + 2e'$ |

### 4.2 Kröger-Vink 记号三要素

```
V_M^{''}  →  V = 空位(Vacancy), M = 位置(金属位), '' = 有效电荷(-2)
A_B^\bullet  →  A = 占据原子, B = 位置, • = 有效电荷(+1)
e'  →  电子(有效电荷 -1)
h^\bullet  →  空穴(有效电荷 +1)
```

**主题记法**：上标 = 与完美晶格相比的**有效电荷**（不是实际电荷！）
- `×` = 电中性（如 $V_\text{O}^\times$ 在 CeO₂ 中实际不存在——氧空位总是带正电）
- `•` = 正有效电荷（缺负离子 → 位置带正电）
- `′` = 负有效电荷（缺正离子 → 位置带负电）

## 五、XRD 指标化 · 立方晶系完整执行

### 5.1 执行流程

```
给一组 sin²θ 值 → 五步指标化
    │
    ├─ 1. 计算 sin²θ 的比值 → 化为最小整数比
    │   └─ 例：sin²θ 比 = 3:4:8:11:12 → ×3 = 无整数 → 这不是 FCC！
    │       实际：3:4:8:11:12 → 全 ×1 = 3,4,8,11,12
    │       → 对应 h²+k²+l² = 3,4,8,11,12
    │
    ├─ 2. 指标化：给每个峰分配 (hkl)
    │   └─ 3=1²+1²+1²→(111); 4=2²+0²+0²→(200); 8=2²+2²+0²→(220)
    │
    ├─ 3. 判断晶格类型（系统消光）
    │   ├─ P：无限制——所有 hkl 都出现
    │   ├─ I：h+k+l = 偶数才出现
    │   ├─ F：h,k,l 全奇或全偶才出现
    │   └─ C：h+k = 偶数才出现
    │
    ├─ 4. 计算晶胞参数 a
    │   └─ 1/d² = (h²+k²+l²)/a², Bragg: λ = 2d sinθ
    │       → a = λ√(h²+k²+l²)/(2 sinθ)
    │
    └─ 5. 验证：用算出的 a 反推其他峰的 θ → 与实验一致 ✓
```

### 5.2 系统消光速查

| 晶格类型 | 消光条件 | 出现的最低 5 个 h²+k²+l² |
|:---|:---|:---|
| **P（简单）** | 无限制 | 1,2,3,4,5（全部出现） |
| **I（体心）** | h+k+l = 奇数消光 | 2,4,6,8,10 |
| **F（面心）** | 奇偶混合消光 | 3,4,8,11,12 |
| **C（底心）** | h+k = 奇数消光 | 1,2,3,4,5（但 (100) 等消光） |

## 六、缺陷→性质推理链

```
氧空位浓度 → 离子电导率 → 应用

CeO₂₋ₓ：
  CeO₂ ──(还原)──→ CeO₂₋ₓ + x/2 O₂↑
  O²⁻ 离开晶格 → 留下 V_O^{••} + 2e'
  V_O^{••} 可在晶格中跳跃迁移 → 氧离子导体
  → SOFC（固体氧化物燃料电池）电解质

YSZ（Y₂O₃ 稳定的 ZrO₂）：
  Y₂O₃ ──(掺杂入ZrO₂)──→ 2Y'_Zr + 3O_O^× + V_O^{••}
  每 2 个 Y³⁺ 取代 Zr⁴⁺ → 产生 1 个氧空位（电荷补偿）
  → 人为制造氧空位 → 稳定的氧离子导体
```

## 七、常见易错点

| 错误 | 纠正 |
|:---|:---|
| 投影图中把所有"点"当独立原子 | 不同 z 高度的原子在投影中重合！先读分数坐标第三个数 |
| 晶系判断先看 a,b,c 是否相等再看角 | **先看角！**α=β=γ=90° 是"正交系"的前提条件 |
| XRD 把 sin²θ 比直接当做 h²+k²+l² | sin²θ 比 = 归一化后的比 → 需要找最小整数比才能对应 |
| Kröger-Vink 上标当实际电荷 | 上标是**有效电荷**（相对完美晶格），不是实际电荷 |
| 看到 BCC → 一定是体心立方结构 | BCC = 晶格类型（I），不是结构类型。NaCl 是 FCC 晶格但结构不是"面心立方结构" |

## 八、例题速练

### 例 1：投影图→化学式

**题**：某立方晶体 [100] 投影图显示：顶点和面心有原子 A，棱心和体心有原子 B。求化学式。

**速解**：A: 8×1/8(角)+6×1/2(面)=1+3=4；B: 12×1/4(棱)+1×1(体)=3+1=4 → **AB**。这是 NaCl 型的标准占位。

### 例 2：XRD 指标化

**题**：某立方晶体前 4 条衍射线 sin²θ 比 = 3:4:8:11。判断晶格类型。

**速解**：h²+k²+l² = 3(111), 4(200), 8(220), 11(311)。出现 3 和 4——混合奇偶 → 不是 I（必须全偶），不是 P（缺 1,2）。3=全奇，4=全偶，8=全偶，11=全奇——全部全奇或全偶 → **F（面心立方）**。
