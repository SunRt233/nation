# 高等有机机理与轨道理论 · 速查卡

> Round 3 你学会了"写箭头推机理"——本卡把机理判断从**叙述层**升级到**证据层**：给定实验数据，判断哪个机理对、哪个机理错。

---

## 一、机理判断工具箱总览

```
怀疑某机理是否正确 → 用六种证据筛
    │
    ├─ 1. 化学合理性（必要但不充分）
    │   └─ 箭头推得通？中间体稳定？过渡态合理？
    │
    ├─ 2. Hammett 方程（ρ 值告诉你：过渡态在累积正电荷还是负电荷）
    │   └─ lg(k/k₀) = ρσ → ρ>0：过渡态富负电；ρ<0：过渡态富正电
    │
    ├─ 3. 动力学同位素效应 KIE（k_H/k_D 告诉你：H/D 是否参与决速步）
    │   ├─ 一级 KIE (k_H/k_D ≈ 2-8)：C–H/C–D 键在决速步中断裂
    │   └─ 二级 KIE (k_H/k_D ≈ 1.0-1.4)：杂化变化但不断 C–H 键
    │
    ├─ 4. 标记实验（同位素标记告诉你：这个原子从哪来到哪去）
    │   └─ ¹⁸O 标记 → 追踪酯水解的断键位置（烷氧断裂 vs 酰氧断裂）
    │
    ├─ 5. 中间体捕获（如果能截获中间体 → 机理路线被证实）
    │   └─ 交叉实验：两个不同底物 → 产物交叉 → 中间体为自由基/离子对
    │
    └─ 6. FMO/轨道对称性（WH 规则：周环反应的"允许/禁阻"判断）
        └─ 热反应：同面-同面 4n+2 允许；光反应：同面-同面 4n 允许
```

## 二、Hammett 方程 · 速判法

### 2.1 核心公式

$$\lg\frac{k}{k_0} = \rho\sigma \quad \text{或} \quad \lg\frac{K}{K_0} = \rho\sigma$$

- **σ**：取代基的电子效应参数（σ>0 = 吸电子，σ<0 = 给电子）
- **ρ**：反应的敏感度（ρ>0 = 过渡态富负电荷，ρ<0 = 过渡态富正电荷）
- **|ρ| 越大**：反应对该取代基效应越敏感

### 2.2 从 ρ 值反推机理

| ρ 值 | 过渡态特征 | 典型反应 |
|:---:|:---|:---|
| ρ < −3 | 强烈富正电荷 | SN1（碳正离子中间体累积正电荷） |
| ρ ≈ −1 ~ −2 | 中等富正电荷 | 亲电芳香取代的过渡态 |
| ρ ≈ 0 | 电荷变化不大 | 自由基反应（自由基中间体几乎电中性） |
| ρ ≈ +1 ~ +2 | 中等富负电荷 | SN2、亲核芳香取代 |
| ρ > +3 | 强烈富负电荷 | 碳负离子中间体、E1cB |

### 2.3 σ⁺ 和 σ⁻

- **σ⁺**：当反应中心能通过共轭与给电子基相互作用时用（如 SN1 中碳正离子被 p-OMe 共轭稳定）
- **σ⁻**：当反应中心能通过共轭与吸电子基相互作用时用（如苯酚电离中 p-NO₂ 共轭稳定酚氧负离子）
- **出现非线性 Hammett 图 → 机制改变**（如改用 σ⁺ 才成直线 → 过渡态累积正电荷，需共轭给电子稳定）

## 三、动力学同位素效应（KIE）· 速判法

| 类型 | k_H/k_D 范围 | 物理含义 | 典型场景 |
|:---|:---:|:---|:---|
| **一级 KIE** | 2 ~ 8（最大~7） | C–H/C–D 键在**决速步**中断裂 | E2（C–H 断裂在决速步）、HAT（氢原子转移） |
| **一级 KIE ~1** | ≈ 1 | C–H 断裂**不在**决速步 | SN1（决速步是 C–LG 断裂，H 不参与） |
| **二级 α-KIE** | 1.1 ~ 1.4 | 决速步中碳从 sp³→sp²（S_N2 过渡态） | SN2 反应 |
| **二级 β-KIE** | 0.9 ~ 1.1 | 超共轭参与稳定过渡态 | 碳正离子形成（C–D 比 C–H 给超共轭更弱 → 逆 KIE < 1） |

**速判口诀**：一级>2 = 断 C–H；一级≈1 = H 不关键；二级 1.1-1.4 = 杂化变；二级<1 = 超共轭

## 四、WH 规则 · 周环反应速查

| 反应类型 | 电子数 | 热反应 | 光反应 |
|:---|:---:|:---|:---|
| 电环化（丁二烯→环丁烯） | 4n | **间旋**（conrotatory） | **同旋**（disrotatory） |
| 电环化（己三烯→环己二烯） | 4n+2 | **同旋**（disrotatory） | **间旋**（conrotatory） |
| 环加成 [2+2] | 4n | **热禁阻**（光允许） | 热禁阻 |
| 环加成 [4+2] (DA) | 4n+2 | **热允许**（同面-同面） | 光禁阻 |
| σ 重排 [3,3] (Cope/Claisen) | 4n+2 | **热允许**（椅式过渡态） | — |

**一句话记 WH**：热反应 4n+2 允许（同面），4n 禁阻（或需间旋）。光反应反过来。

## 五、FMO 快速判断法

> 不需要算轨道系数，用以下规则即可判断大多数竞赛题。

```
DA 反应（正常电子需求）：
├─ 二烯 HOMO（给电子基抬升） + 亲双烯体 LUMO（吸电子基降低）
├─ 给电子基在二烯上 → 加速反应
└─ 吸电子基在亲双烯体上 → 加速反应

逆电子需求 DA：
├─ 二烯上有吸电子基 → 二烯 LUMO 降低
├─ 亲双烯体上有给电子基 → 亲双烯体 HOMO 抬升
└─ 此时二烯 LUMO + 亲双烯体 HOMO 相互作用
```

## 六、常见机理判断误区

| 错误 | 纠正 |
|:---|:---|
| "箭头推得通 → 机理成立" | 箭头推得通只是**必要不充分**——很多错误机理也能推通 |
| Hammett ρ≈0 → 无电荷变化 | ρ≈0 也可能是两种相反效应恰好抵消——需结合其他证据 |
| KIE 大 → 一定是断 C–H | 一级 KIE 只在 C–H 断在**决速步**时才大。C–H 断在快步骤中 → KIE≈1 |
| 看到自由基产物 → 一定走自由基机理 | 也可能是离子机理后发生 SET（单电子转移）——检查是否有自由基捕获证据 |
| WH 规则用于所有环化反应 | WH 只用于**协同**周环反应——分步环化不走 WH |

## 七、例题速练

### 例 1：Hammett 判机理

**题**：某取代苯甲酸酯的碱性水解，ρ = +2.5。判断决速步的过渡态特征。

**速解**：ρ > 0 且 |ρ| 较大 → 过渡态累积负电荷。这符合 B_AC2 机理（OH⁻ 进攻 C=O，四面体中间体带负电荷）。若 ρ 接近 0 → 可能是自由基机理或决速步不涉及电荷变化。

### 例 2：KIE 判机理

**题**：某反应中 k_H/k_D = 1.0，但产物中有 C–H 断裂发生。如何解释？

**速解**：C–H 断裂不在决速步中。可能机理：决速步是 C–LG 断裂（如 SN1），C–H 在后续快步骤中失去。或 C–H 断裂在决速步之前的平衡中（前平衡）。

### 例 3：交叉实验

**题**：Ar¹–N₂⁺ + Ar²–H → 产物中有 Ar¹–Ar¹ 和 Ar²–Ar²，无 Ar¹–Ar²。说明什么？

**速解**：无交叉产物 → 不是自由基机理（自由基会有交叉偶联）。可能机理：芳基正离子直接亲电取代，或重氮盐先分解为芳基正离子再加成。
