# 氢与卤素 · 速查卡

> 基础班你背过"卤素=F₂/Cl₂/Br₂/I₂"——本卡把氢卤从**事实记忆**升级到**族内递变规律驱动**：不用背每个反应，用递变趋势推断产物。

---

## 一、氢的三种成键形态

```
H 在化合物中扮演三种角色：
    │
    ├─ H⁺（质子/酸）
    │   └─ HCl/H₂SO₄ 等质子酸 → H⁺ 传递
    │   └─ H₃O⁺ 在水溶液中实际存在形式
    │
    ├─ H⁻（氢负离子/强还原剂）
    │   └─ 离子型氢化物：NaH, CaH₂, LiAlH₄
    │   └─ H⁻ + H₂O → H₂↑ + OH⁻（极强的还原性！）
    │   └─ LiAlH₄：有机合成的"万能还原剂"
    │
    └─ 共价 H（H–X 键）
        └─ H₂O, NH₃, CH₄, HCl 等
        └─ 极性变化：H–F（强极性）→ H–I（弱极性），酸性递升
```

**氢化物三分类**：离子型（NaH）+ 共价型（CH₄）+ 金属型（PdHₓ，H 在金属晶格间隙）

## 二、卤素族内递变规律

| 性质 | F₂ | Cl₂ | Br₂ | I₂ | 递变方向 → |
|:---|:---|:---|:---|:---|:---|
| **状态/颜色** | 淡黄气体 | 黄绿色气体 | 红棕色液体 | 紫黑色固体 | 颜色加深、状态趋于凝聚 |
| **氧化性** | **最强** | 强 | 中 | 弱 | F₂ > Cl₂ > Br₂ > I₂ |
| **还原性 X⁻** | 极弱 | 弱 | 中 | **最强** | I⁻ > Br⁻ > Cl⁻ > F⁻ |
| **X–X 键能** | 159 | 243 | 193 | 151 (kJ/mol) | Cl₂ 最大（F₂ 反常弱→孤对排斥） |
| **HX 键能** | 569 | 431 | 366 | 298 | HF 最强→最难解离 |
| **HX 酸性** | 弱酸 | 强酸 | 强酸 | **最强酸** | HI > HBr > HCl > HF |

**核心矛盾**：F₂ 键能反常弱（孤对排斥）→F₂ 是实验室可用的最强氧化剂；但 HF 键能最强→F⁻ 是极弱的还原剂。

---

## 三、卤素含氧酸 · 反向规律

```
酸性递增：HClO < HClO₂ < HClO₃ < HClO₄
  └─ 中心 Cl 氧化态↑ → O 原子数↑ → 电子密度↓ → H⁺ 更易解离

氧化性递减：HClO > HClO₂ > HClO₃ > HClO₄
  └─ Cl–O 键越来越强 → 越不容易断裂 → 氧化性反而减弱！

一句话：酸性从左到右递增，氧化性从左到右递减——不要搞混！
```

**HClO 的特殊性**：虽然 Cl 氧化态最低（+1），但氧化性最强——因为 Cl–O 键最弱、最容易断裂释放活性氧。

## 四、卤素互化物与拟卤素

| 类别 | 定义 | 实例 |
|:---|:---|:---|
| **卤素互化物** | 不同卤素原子间的化合物 | ICl, BrF₃, IF₇（电负性低的卤素在中心） |
| **拟卤素** | 行为类似卤素的基团 | (CN)₂（氰）、(SCN)₂（硫氰） |

**拟卤素判断标准**：①可形成 X⁻ 离子（CN⁻, SCN⁻）②可形成 X₂ 二聚体 ③可形成 HX 酸 ④难溶盐类似卤化银

## 五、元素电势图 · 歧化判断

```
酸性条件 Cl 的电势图（简版）：
  ClO₃⁻ ──+1.21V──→ HClO₂ ──+1.65V──→ HClO ──+1.63V──→ Cl₂ ──+1.36V──→ Cl⁻

歧化判断：E°(右) > E°(左) → 中间物种歧化
  例：HClO 两侧：E°(左→HClO)=+1.65V, E°(HClO→Cl₂)=+1.63V
      1.63 < 1.65 → HClO 不歧化（但加热或光照下可歧化—动力学因素）
```

**推断题信号**：Cl₂ 黄绿色气体 + 使湿润 KI-淀粉试纸变蓝（I⁻→I₂）→ 锁定卤素

## 六、常见易错点

| 错误 | 纠正 |
|:---|:---|
| "F₂ 键能最强所以最稳定" | F₂ 键能反常弱（159 vs Cl₂ 243 kJ/mol）——F 原子太小→孤对排斥 |
| "含氧酸氧化性随氧化态升高而增强" | **反向规律！** HClO 氧化性 > HClO₄——Cl–O 键越弱越容易断 |
| "HF 是强酸" | HF 是弱酸（pKa~3.2）——H–F 键太强+HF 分子间氢键稳定 |
| "卤化银都难溶" | AgF **易溶**！AgCl/AgBr/AgI 难溶——F⁻ 水合能大+晶格能较小 |

## 七、例题速练

### 例 1：歧化判断

**题**：Cl₂ 在 NaOH 中歧化为 Cl⁻ 和 ClO₃⁻（加热）还是 ClO⁻（冷）？

**速解**：
- 冷：Cl₂ + 2OH⁻ → Cl⁻ + ClO⁻ + H₂O
- 热：3Cl₂ + 6OH⁻ → 5Cl⁻ + ClO₃⁻ + 3H₂O
- ClO⁻ 在热条件下进一步歧化→ClO₃⁻

### 例 2：推断题信号

**题**：某气体黄绿色、使湿润 KI-淀粉试纸变蓝、溶于 NaOH 产生漂白性溶液。是什么？

**速解**：黄绿色气体 → Cl₂。Cl₂ + 2KI → 2KCl + I₂（使淀粉变蓝）。Cl₂ + 2NaOH → NaCl + NaClO（NaClO 有漂白性）。
