# 有机结构基础与电子效应 · 速查卡

> 基础班你背过"诱导效应=吸电子/给电子"——本卡把电子效应从**名词层**升级到**判断语言层**：酸碱性、稳定性、活性、区域选择性，全部统一到"谁稳定了谁+轨道怎么匹配"。

---

## 一、电子效应判断总流程

```
拿到有机比较题 → 三问定位
    │
    ├─ 1. 题目到底在比较什么？
    │   └─ 酸性？碱性？稳定性？反应活性？区域选择性？
    │
    ├─ 2. 结构中主导的稳定化来自哪里？
    │   ├─ 诱导效应（σ骨架传递，随距离衰减）
    │   ├─ 共轭效应（π体系传递，不随距离衰减）
    │   ├─ 超共轭效应（σ→π/σ→p/σ→σ*，稳定碳正离子/自由基）
    │   └─ 场效应（空间直接传递，不依赖键）
    │
    └─ 3. 什么时候要从"电子富贫"升级到 HOMO/LUMO？
        └─ 当比较的是反应速率/区域选择性→光凭电子富贫不够→需要轨道系数和能级匹配
```

---

## 二、四大电子效应速查

| 效应 | 传递方式 | 衰减 | 典型场景 | 判断口诀 |
|:---|:---|:---|:---|:---|
| **诱导效应** | σ 键骨架 | 随距离快速衰减 | 卤代酸酸性：CF₃COOH > CH₂FCOOH > CH₃COOH | "电负性差异沿σ传，隔三个碳基本没了" |
| **共轭效应** | π 体系 | 不衰减（整个共轭体系共享） | p-NO₂-PhO⁻ → 负电荷分散到 NO₂ → 酚酸性增强 | "孤对/空p→π→共振→稳定化" |
| **超共轭效应** | σ→p/σ→π*/σ→σ* | 中等 | C⁺稳定性：3°>2°>1°——C-H σ → 空 p | "C-H/C-C σ给空p→碳正离子的命根子" |
| **场效应** | 空间直接 | 取决于距离和方向 | 邻位取代基的酸性影响（不经过键） | "不过键、走空间、距离近才显著" |

## 三、酸碱性 · 不背 pKa 表，用稳定化判断

```
HA ⇌ H⁺ + A⁻

酸性强 = 共轭碱 A⁻ 稳定
├─ A⁻ 负电荷被分散 → A⁻ 稳定 → HA 酸性强
│   ├─ 诱导吸电子：CF₃COO⁻（F 吸电子→分散负电荷） > CH₃COO⁻
│   ├─ 共轭离域：PhO⁻（负电荷分散到苯环） > ROH（烷氧负离子电荷集中）
│   └─ 场/溶剂：更大的原子/基团→电荷更分散→更稳定
│
碱性強 = 共轭酸 BH⁺ 稳定，或碱本身孤对电子活泼

有机酸强度顺序（直觉版，不背数值）：
  磺酸 > 羧酸 > 酚 > 醇 > 炔 > 烯 > 烷
  └─ 共轭碱：RSO₃⁻(共振) > RCOO⁻(共振) > PhO⁻(共轭) > RO⁻ > RC≡C⁻ > RC=C⁻ > R⁻
```

**关键区别**：醇酸性 vs 酚酸性——PhO⁻ 的负电荷通过共轭分散到苯环（尤其是 o/p 有吸电子基时），RO⁻ 的负电荷集中在 O 上。因此酚的酸性远强于醇（pKa ~10 vs ~16）。

## 四、碳正离子/碳负离子/自由基稳定性速判

| 中间体 | 稳定性顺序 | 主导效应 | 一句话 |
|:---|:---|:---|:---|
| **碳正离子** | 3° > 2° > 1° > CH₃⁺ | 超共轭（C–H σ→空 p）+ 诱导给电子 | "越多烷基越稳——超共轭是命根子" |
| **碳负离子** | 1° > 2° > 3° | 诱导吸电子 + 共轭离域 | "越多烷基越不稳——烷基给电子→负电荷更集中" |
| **自由基** | 3° > 2° > 1° > CH₃• | 超共轭（类似C⁺但弱） | "趋势同C⁺但差异较小——超共轭+离域双贡献" |

**烯丙基/苄基特殊稳定**：三种中间体在烯丙位/苄位都额外稳定——因为 p 轨道与 π 体系共轭（共振）。

## 五、前线轨道 · 从"电子富贫"到"轨道匹配"

```
反应发生的前提：亲核体 HOMO 与亲电体 LUMO 能级接近 + 轨道重叠有效

判断升级路径：
  初级判断（基础班）：谁富电子（亲核）？谁缺电子（亲电）？
  升级判断（提高班）：HOMO/LUMO 能级差 + 轨道系数大小 → 区域选择性

典型应用：
  DA 反应：二烯 HOMO + 亲双烯体 LUMO（正常电子需求）
           二烯 LUMO + 亲双烯体 HOMO（逆电子需求，二烯有吸电子基时）
  Michael 加成：烯醇负离子 HOMO + α,β-不饱和羰基 LUMO（β位系数大→β位进攻）
```

**Bent 规则速用**：s 成分越高的杂化轨道→电负性越大→键角越大。解释：环丙烷 C–H（sp²·²⁸）比普通烷烃 C–H（sp³）更"酸"——C 的 s 成分更高→C 更电负→C–H 极性更大→H 更易离去。

## 六、常见易错点

| 错误 | 纠正 |
|:---|:---|
| 比较酸性只看"有没有吸电子基" | 要看吸电子基**作用在共轭碱的哪个位置**——o/p 位共轭效应+诱导效应，m 位只有诱导效应 |
| 碳正离子稳定性=碳负离子稳定性反过来 | 不是完全反过来——共轭稳定两者都有效（烯丙基C⁺和烯丙基C⁻都比普通稳定） |
| 共轭效应=共振效应 | 共轭=π体系电子离域；共振=结构用多个 Lewis 式描述。不是同一个概念，但密切相关 |
| 场效应不重要 | 邻位取代基对酸性的影响中，场效应有时比诱导效应更显著（特别是带电基团） |

## 七、例题速练

### 例 1：酸性比较

**题**：比较 p-NO₂-PhOH、m-NO₂-PhOH、PhOH 的酸性。

**速解**：酸性来自共轭碱稳定性。p-NO₂：—I（诱导吸电子）+ —M（共轭吸电子——NO₂ 的 π* 接受苯环负电荷）→ 最强。m-NO₂：只有 —I（间位不参与共轭）→ 中等。PhOH：无取代基 → 最弱。顺序：p-NO₂ > m-NO₂ > PhOH。

### 例 2：HOMO/LUMO 判区域选择性

**题**：为什么 Michael 加成中亲核试剂优先进攻 β 位而非羰基碳？

**速解**：α,β-不饱和羰基的 LUMO 在 β 位系数最大（C=C 的 π* 与 C=O 的 π* 共轭→β 位轨道系数最大）→亲核试剂 HOMO 优先与 β 位重叠→β 位进攻。这是**轨道控制**，不是电荷控制（虽然 β 位也更正）。

## 八、自查清单

- [ ] 诱导/共轭/超共轭/场效应四种效应各能一句话解释
- [ ] 酸性比较不背 pKa——能通过"共轭碱稳定性"推理
- [ ] 碳正离子(C⁺) vs 碳负离子(C⁻) vs 自由基的稳定性顺序不混淆

- [ ] 从"电子富贫"升级到"HOMO/LUMO 能级+系数"的场景判断
- [ ] Bent 规则的基本含义——s 成分↑→电负性↑→键角↑
