## 一、课堂引入与本节问题

### 1.1 课堂切入

**回忆第一轮**：你们学会了用初始速率法判断反应级数 n，用 Arrhenius 两点法计算活化能 Ea。这些工具都只关心"初始时刻"。

**问题升级**：如果我说"这个反应进行到 10 min 时还剩多少 A？"——初始速率法回答不了。如果你面对的是"这个 3 步机理能否解释实验观测到的总反应级数？"——光会质量作用定律也不够。

**本节要解决的问题**：
1. 如何从浓度-时间数据直接判断反应级数？（积分法·半衰期法）
2. 给定一个多步反应机理，如何推导出总速率方程？（稳态近似）
3. 正反应和逆反应同时存在时，动力学怎么处理？（对峙反应）
4. 催化剂到底做了什么，让反应更快但不改变平衡？（活化能深化）

### 1.2 学习目标

- **本节讲到哪里为止**：积分速率方程→半衰期法→稳态近似→对峙反应→表观活化能→催化本质
- **后续深入学习**：过渡态理论（Eyring方程）、碰撞理论定量推导、酶催化、链式反应爆炸极限

## 二、模块一：积分速率方程——从微分到积分

> 📚 教材：Atkins §20A-20B、物理化学第5版 §11.2

### A. 为什么要"积分"？

第一轮用的微分速率方程 v = k[A]ⁿ 描述了"这一刻的速率"，但实际实验中我们测量的是**浓度随时间的变化**——不同时间的 [A] 值。把微分式积分，就得到 [A] 和 t 的直接关系。

### B. 零级/一级/二级反应的积分形式

<details>
<summary>📐 完整推导（按需展开）</summary>

**零级反应**（n=0）：
$$v = -\frac{d[A]}{dt} = k \quad \Rightarrow \quad \int_{[A]_0}^{[A]} d[A] = -k\int_0^t dt \quad \Rightarrow \quad \boxed{[A] = [A]_0 - kt}$$

**一级反应**（n=1）：
$$v = -\frac{d[A]}{dt} = k[A] \quad \Rightarrow \quad \int_{[A]_0}^{[A]} \frac{d[A]}{[A]} = -k\int_0^t dt \quad \Rightarrow \quad \boxed{\ln\frac{[A]_0}{[A]} = kt}$$

或写成：$\ln[A] = \ln[A]_0 - kt$（线性形式）

**二级反应**（n=2）：
$$v = -\frac{d[A]}{dt} = k[A]^2 \quad \Rightarrow \quad \int_{[A]_0}^{[A]} \frac{d[A]}{[A]^2} = -k\int_0^t dt \quad \Rightarrow \quad \boxed{\frac{1}{[A]} - \frac{1}{[A]_0} = kt}$$

或写成：$\frac{1}{[A]} = \frac{1}{[A]_0} + kt$（线性形式）

</details>

### C. 结果速查表

| 级数 | 积分速率方程 | 线性关系 | 斜率 | 截距 | k 的单位 |
|:---:|:---|:---|:---|:---|:---|
| **0** | [A] = [A]₀ - kt | **[A] ~ t** | -k | [A]₀ | mol·L⁻¹·s⁻¹ |
| **1** | ln[A] = ln[A]₀ - kt | **ln[A] ~ t** | -k | ln[A]₀ | s⁻¹ |
| **2** | 1/[A] = 1/[A]₀ + kt | **1/[A] ~ t** | +k | 1/[A]₀ | L·mol⁻¹·s⁻¹ |

### D. 直线判断法——最可靠的级数确定方法

**操作流程**：
1. 实验测一组 c-t 数据（至少 4-5 个时间点）
2. 分别作 [A]~t、ln[A]~t、1/[A]~t 三张图
3. 哪张图成直线 → 就是哪个级数
4. 从直线斜率求 k（一级/二级）或直接从截距和斜率求 k（零级）

**例题**：N₂O₅ 在 CCl₄ 中 318K 分解，测得以下数据：

| t/min | 0 | 20 | 40 | 60 | 80 |
|:---:|:---:|:---:|:---:|:---:|:---:|
| [N₂O₅]/M | 0.500 | 0.352 | 0.248 | 0.175 | 0.123 |

**判断步骤**：
1. 先试 lnc~t：ln(0.500)=-0.693, ln(0.352)=-1.044, ln(0.248)=-1.394, ln(0.175)=-1.743, ln(0.123)=-2.096 → 与 t 完美线性 → **一级反应**
2. k = -斜率 = -(-1.044+0.693)/20 = 0.0176 min⁻¹（或直接用完整数据线性拟合）

> ⚠️ **陷阱**：竞赛题常常给你"看似线性的非一级数据"——如果只给 2-3 个点，光看 c~t 下降趋势可能误判为零级。**必须至少尝试两种画法**来确认。

## 三、模块二：半衰期——更快捷的级数判断工具

> 📚 教材：Atkins §20B、普通化学原理 §7.3

### E. 半衰期公式

半衰期 t₁/₂ 指反应物浓度降到初始浓度一半所需的时间。令 [A] = [A]₀/2，代入积分速率方程：

| 级数 | 半衰期公式 | t₁/₂ 与 [A]₀ 的关系 |
|:---:|:---|:---|
| **0** | $t_{1/2} = \frac{[A]_0}{2k}$ | **正比于 [A]₀**（初始浓度加倍 → t₁/₂ 加倍） |
| **1** | $t_{1/2} = \frac{\ln 2}{k}$ | **与 [A]₀ 无关**（任何初始浓度，t₁/₂ 相同！） |
| **2** | $t_{1/2} = \frac{1}{k[A]_0}$ | **反比于 [A]₀**（初始浓度加倍 → t₁/₂ 减半） |

### F. 半衰期法的竞赛应用

**通式**：$t_{1/2} \propto [A]_0^{1-n}$

**使用场景**：当没有完整的 c-t 数据，只知道"两个不同初始浓度下的半衰期"时，直接反推 n。

**例题**：某反应 A→P，[A]₀=0.1M 时 t₁/₂=100s；[A]₀=0.05M 时 t₁/₂=200s。判断级数。

- t₁/₂ 与 [A]₀ 成反比（0.05→200s 是 0.1→100s 的加倍）→ **二级反应**（t₁/₂ ∝ 1/[A]₀）

**另一个例题**：某反应 [A]₀=0.2M 时 t₁/₂=50s；[A]₀=0.1M 时 t₁/₂ 还是 = 50s → **一级反应**（t₁/₂ 与 [A]₀ 无关）。

## 四、模块三：稳态近似——从机理到速率方程

> 📚 教材：Atkins §20D、物理化学第5版 §12.1-12.2

这是本课**最核心的工具升级**。竞赛中大量的动力学大题都是"给机理→推导速率方程→与实验对比→判断机理是否正确"。

### G. 稳态近似的定义与操作

**定义**：反应中间体 I 的浓度在整个反应过程中几乎不变（生成速率 ≈ 消耗速率）：

$$\frac{d[I]}{dt} \approx 0$$

**操作流程**（3 步法）：
1. 找出机理中的所有**中间体**（不出现在总反应中的物种）
2. 对每个中间体写出 d[I]/dt = 0 的方程
3. 解出 [I] 的表达式，代入 RDS 的速率方程

### H. 经典示例：A ⇌ I → P 机理

**机理**：
- 步骤1：A ⇌ I（k₁ 正向, k₋₁ 逆向，快速可逆）
- 步骤2：I → P（k₂，慢，RDS）

**稳态近似推导**：
$$\frac{d[I]}{dt} = k_1[A] - k_{-1}[I] - k_2[I] \approx 0$$
$$[I] = \frac{k_1[A]}{k_{-1} + k_2}$$
$$v = k_2[I] = \frac{k_1 k_2}{k_{-1} + k_2}[A]$$

**极限情况讨论**：
- 若 k₋₁ ≫ k₂（第一步快速可逆，I 主要回 A）：$v \approx \frac{k_1}{k_{-1}}k_2[A] = K \cdot k_2[A]$
- 若 k₂ ≫ k₋₁（I 一生成就转化为 P）：$v \approx k_1[A]$（第一步是速率瓶颈）

### I. 平衡假设——稳态近似的"简化版"

**适用条件**：第一步**快速可逆**，第二步是**RDS**（慢）。

**操作**：直接用平衡条件 [I] = K[A]（其中 K = k₁/k₋₁），而不是解完整的稳态近似方程。

**对比**：

| | 稳态近似 | 平衡假设 |
|:---|:---|:---|
| 假设 | d[I]/dt = 0 | 前步快速可逆，K = k₁/k₋₁ |
| 适用条件 | 任何有中间体的机理 | 仅当前步可逆 + 后续是 RDS |
| 结果 | [I] = k₁[A]/(k₋₁+k₂) | [I] = (k₁/k₋₁)[A] |
| 精度 | **更通用** | 更简单（是稳态近似的极限情况） |

### J. 竞赛经典题：H₂ + I₂ → 2HI 的机理验证

**实验事实**：速率方程 v = k[H₂][I₂]（总反应二级）

**提出机理**：
1. I₂ ⇌ 2I·（k₁, k₋₁，快速平衡，K = k₁/k₋₁）
2. I· + H₂ → HI + H·（k₂，慢，**RDS**）
3. H· + I₂ → HI + I·（k₃，快）

**推导过程**：
- 中间体：I· 和 H·（两个！）
- RDS = 步骤2 → v = k₂[I·][H₂]
- [I·] 来自步骤1的平衡假设：K = [I·]²/[I₂] → [I·] = √(K[I₂])
- [H·] 用稳态近似：d[H·]/dt = k₂[I·][H₂] - k₃[H·][I₂] ≈ 0 → [H·] = k₂[I·][H₂]/(k₃[I₂])
- 代入：v = k₂[I·][H₂] = k₂√(K[I₂])[H₂] = **k₂√K · [H₂][I₂]^{1/2}**

**等等——推导结果是 v ∝ [H₂][I₂]^{1/2}，但实验是 v ∝ [H₂][I₂]！机理有问题！**

<details>
<summary>🔬 机制解析</summary>

上面的"经典例子"实际上**暴露了 H₂+I₂ 反应的真实争论**：

- 早期（1890s-1960s）认为 H₂+I₂ 就是简单的双分子基元反应 H₂+I₂→2HI（一步，v=k[H₂][I₂]）
- 1967年 Sullivan 用光化学方法证明：该反应**确实经过碘原子机理**，但实验速率方程 v=k[H₂][I₂] 是因为 k₃ ≫ k₂ 且 [I₂] 足够大，使得第3步非常快——实际上它既不是简单基元也不是简单稳态近似能完整描述的
- **教学上**：此题常用于展示"机理推导必须与实验一致"——如果推导结果 v ∝ [H₂][I₂]^{1/2} 与实验 v ∝ [H₂][I₂] 不符，说明**机理需要修正**（例如考虑第3步也可能是RDS的一部分，或引入更复杂的碘原子动力学）

</details>

### K. 决策树：什么时候用稳态近似 vs 平衡假设？

```
机理中有中间体？
  ├── 是 → 前一步是否快速可逆 + 后续是否有明确的 RDS？
  │     ├── 是 → 平衡假设（更简单）
  │     └── 否 → 稳态近似（更通用）
  └── 否 → 基元反应，直接用质量作用定律
```

## 五、模块四：对峙反应与平行反应

> 📚 教材：Atkins §20C、物理化学第5版 §12.3-12.4

### L. 对峙反应 A ⇌ B —— 动力学和热力学的"交汇点"

**定义**：正反应和逆反应同时以可观速率进行的反应。

**微分方程**（正逆向均为一级）：
$$\frac{d[A]}{dt} = -k_1[A] + k_{-1}[B] = -k_1[A] + k_{-1}([A]_0 - [A])$$

**关键结论**：
- 平衡时 d[A]/dt = 0 → k₁[A_eq] = k₋₁[B_eq] → $$\boxed{K = \frac{k_1}{k_{-1}} = \frac{[B]_{eq}}{[A]_{eq}}}$$
- 动力学和热力学在此**统一**——平衡常数可以从动力学速率常数得到

**由动力学数据求 k₁ 和 k₋₁**：
- 方法1：测平衡浓度得 K → 测初始速率得 k₁ → k₋₁ = k₁/K
- 方法2：由积分式 ln{([A]₀-[A_eq])/([A]-[A_eq])} = (k₁+k₋₁)t → 斜率得 k₁+k₋₁，结合 K 解出两者

### M. 平行反应——选择性由温度调控

A 同时生成 B 和 C（k₁ → B, k₂ → C）。

$$\frac{[B]}{[C]} = \frac{k_1}{k_2}$$

**与合成化学的连接**：如果两路径的活化能不同（Ea₁ ≠ Ea₂），改变温度可以调控选择性：
- 低温 → 以活化能低的路径为主（动力学控制）
- 高温 → 两个路径都变快，选择性下降，且热力学更稳定的产物占比增大

这是第一轮和第三轮专题2/6讲过的"动力学控制 vs 热力学控制"的**定量基础**。

## 六、模块五：催化与活化能深化

> 📚 教材：Atkins §20E-20F、物理化学第5版 §12.5-12.6

### N. 表观活化能——多步机理的 Ea 怎么合成？

**问题**：多步机理的总速率常数 k_app 是各步骤 k 的某种组合（乘积/比值）。k_app 的 Arrhenius 活化能（表观活化能 Ea_app）与各步骤的 Ea 有什么关系？

**关键公式**（以 A ⇌ I → P 为例）：

由稳态近似：$k_{app} = \frac{k_1 k_2}{k_{-1} + k_2}$

- 若 k₋₁ ≫ k₂：$k_{app} \approx \frac{k_1 k_2}{k_{-1}}$ → $E_{a,app} = E_{a,1} + E_{a,2} - E_{a,-1}$
- 若 k₂ ≫ k₋₁：$k_{app} \approx k_1$ → $E_{a,app} = E_{a,1}$

**操作规则**：$k = A e^{-E_a/RT}$，所以 $k_{app} = \prod k_i^{n_i}$ 时，$E_{a,app} = \sum n_i E_{a,i}$（乘积→加和，除法→相减）。

### O. 催化本质三条——每个化竞生必须能完整说出

| 本质 | 具体含义 | Arrhenius 解释 |
|:---|:---|:---|
| **改变机理（提供新路径）** | 催化剂参与反应，形成中间体，开辟一条不同的反应路径 | 新的"机理"对应新的 Ea_app 表达式 |
| **降低活化能** | 新路径的速控步活化能比非催化的单步低 | k_cat/k_uncat = exp[-(Ea_cat-Ea_uncat)/RT] ≫ 1 |
| **不改变平衡** | 催化剂同等程度降低正逆反应的 Ea → K 不变 | 正反向 Ea 降低量相同 → k₁/k₋₁ 不变 |

**实例——H₂O₂ 分解**：
- 非催化：2H₂O₂ → 2H₂O + O₂，Ea ≈ 75 kJ/mol（慢）
- I⁻ 催化：H₂O₂ + I⁻ → H₂O + IO⁻（Ea₁低），H₂O₂ + IO⁻ → H₂O + O₂ + I⁻（Ea₂低）
- 总反应不变，但分步后每步 Ea 都低得多 → 速率大幅提升

> 🔷 **与第四轮的衔接——为什么"催化剂不改变平衡"在竞赛中很重要？**
> 
> 39届初赛曾考过：某催化反应的表观活化能比非催化反应低，但催化剂不能改变 ΔG 或 K——因为 K 由反应物和产物的 Gibbs 自由能差决定，催化剂**同时加速正逆反应**，不改变这个差。这和本课 §L 对峙反应中 K = k₁/k₋₁ 是一脉相承的。

## 七、本节形成的核心结论

### 动力学工具速查表

| 工具 | 输入 | 输出 | 使用场景 |
|:---|:---|:---|:---|
| **积分速率方程 + 直线判断** | c-t 数据 | 级数 n + 速率常数 k | 有完整的浓度-时间测量 |
| **半衰期法** | 至少两个不同 [A]₀ 下的 t₁/₂ | 级数 n | 只有半衰期数据（无完整 c-t） |
| **稳态近似** | 反应机理（含中间体）| 总速率方程 + k_app 表达式 | 需要从机理推导到实验可验证的方程 |
| **平衡假设** | 快速可逆 + RDS 的机理 | 简化速率方程（[I] = K[A]） | 稳态近似的快捷替代 |
| **对峙反应公式** | k₁ 和 k₋₁（或 K + k₁）| 浓度-时间曲线 + 平衡浓度 | 正逆反应都不可忽略 |
| **表观活化能合成** | 各步骤的 Ea | 总反应的 Ea_app | 多步机理的 Arrhenius 分析 |

### 从实验数据到反应机理的完整流程

```
①实验测 c-t 数据
    ↓
②直线判断/半衰期 → 确定级数 n → 确定速率方程形式
    ↓
③提出反应机理（基于化学直觉）
    ↓
④稳态近似推导机理的速率方程
    ↓
⑤对比推导结果与实验速率方程
    ├── 一致 → 机理可能正确（但不唯一）
    └── 不一致 → 机理需要修正
    ↓
⑥Arrhenius → 活化能 → 催化分析
```

## 八、课堂例题

### 例题1：直线判断法（必做）
> 某反应 A→P 测得：
> | t/s | 0 | 100 | 200 | 300 | 400 |
> | [A]/M | 1.00 | 0.50 | 0.33 | 0.25 | 0.20 |
> 判断级数并求 k。

**提示**：试 1/[A]~t → 1/1.00=1.00, 1/0.50=2.00, 1/0.33=3.03, 1/0.25=4.00, 1/0.20=5.00 → 完美线性！→ **二级反应**，k = 斜率 = (2.00-1.00)/100 = 0.010 L·mol⁻¹·s⁻¹。

### 例题2：半衰期法（必做）
> 相同温度下，[A]₀=0.200M 时 t₁/₂=45s；[A]₀=0.100M 时 t₁/₂=90s。求 n 和 k。

**答案**：t₁/₂ 与 [A]₀ 成正比（半衰期减半 → 初始浓度加倍）→ t₁/₂ ∝ [A]₀ → **零级反应**。k = [A]₀/(2t₁/₂) = 0.200/(2×45) = 2.22×10⁻³ mol·L⁻¹·s⁻¹。

### 例题3：稳态近似（必做）
> 机理：① A + B ⇌ C（k₁, k₋₁，快速可逆）② C → P（k₂，RDS）。推导总速率方程。

**答案**：K = [C]/([A][B]) → [C] = K[A][B]。v = k₂[C] = k₂K[A][B] → **总反应二级**（对 A 和 B 各一级）。注意——不能直接用平衡假设得出"与实验速率方程形式一致=机理正确"——这只是必要条件，不是充分条件。

### 例题4：表观活化能（选做）
> A ⇌ I → P 机理：Ea₁=80, Ea₋₁=60, Ea₂=40 kJ/mol。若 k₋₁ ≫ k₂，求 Ea_app。

**答案**：Ea_app = Ea₁ + Ea₂ - Ea₋₁ = 80 + 40 - 60 = **60 kJ/mol**。

## 九、易错点预警

| 易错点 | 正确理解 |
|:---|:---|
| 积分速率方程中的 k 单位忘记随级数变化 | k₀: mol·L⁻¹·s⁻¹, k₁: s⁻¹, k₂: L·mol⁻¹·s⁻¹。做题时先检查 k 的单位是否与级数一致 |
| 半衰期 = 反应进行到一半的时间 → 任何级数都是"一半" | 零级和二级的半衰期**依赖于初始浓度**——不能像一级那样直接调表 |
| "稳态近似 = 中间体浓度 = 0" | 稳态近似是 d[I]/dt ≈ 0（生成=消耗），不是 [I] = 0！中间体浓度虽小但不是零 |
| RDS 后的步骤"不影响总速率" | RDS 后的步骤不影响速率方程的**形式**，但会影响 k_app 的数值（如果涉及可逆步骤） |
| 催化剂降低 Ea → 放热反应变得更放热 | 催化剂降低的是 **Ea**（活化能），不是 **ΔH**（反应焓）。放热还是放那么多热，只是到达彼岸更快 |
| 对峙反应 t → ∞ 时 [A] = 0 | 对峙反应的最终浓度是 [A_eq]（平衡浓度），不是 0——动力学平衡 ≠ 反应完全 |
