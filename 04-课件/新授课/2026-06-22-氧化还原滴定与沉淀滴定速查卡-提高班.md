# 氧化还原滴定与沉淀滴定 · 速查卡

> 基础班你背过"氧化还原滴定用氧化还原反应"——本卡把氧化还原/沉淀滴定从**方法记忆**升级到**方法选择+计量比判断+条件控制+指示剂逻辑**的定量决策系统。

---

## 一、氧化还原滴定总览

```
方法选择三维决策：

  反应是否定量完成？（K 足够大）
  ├─ 反应速度是否够快？（否则需加热/催化剂）
  └─ 终点能否准确判断？（指示剂/自指示剂/电位法）

三种核心方法对比：
  ┌──────────┬──────────────┬──────────────┬──────────────┐
  │          │  碘量法       │  KMnO₄ 法    │  K₂Cr₂O₇ 法  │
  ├──────────┼──────────────┼──────────────┼──────────────┤
  │ 氧化剂   │ I₂（直接）    │ MnO₄⁻        │ Cr₂O₇²⁻      │
  │ 还原产物 │ I⁻           │ Mn²⁺         │ Cr³⁺         │
  │ 指示剂   │ 淀粉（蓝→无）│ 自指示剂      │ 二苯胺磺酸钠  │
  │ 条件     │ 中性/弱酸性   │ 强酸性！      │ 酸性         │
  │ 特点     │ 可逆性好      │ 无需外加指示剂│ 基准物质稳定  │
  └──────────┴──────────────┴──────────────┴──────────────┘
```

## 二、碘量法 · 计量比是核心

```
直接碘量法：I₂ 直接氧化待测物
  I₂ + 2e⁻ → 2I⁻
  例：I₂ + Sn²⁺ → 2I⁻ + Sn⁴⁺（测定 Sn²⁺）

间接碘量法：I⁻ 先被氧化 → 生成 I₂ → Na₂S₂O₃ 返滴定 I₂
  核心方程式：I₂ + 2S₂O₃²⁻ → 2I⁻ + S₄O₆²⁻

  ⚠️ 计量比 I₂ : S₂O₃²⁻ = 1 : 2（不是 1:1！最易错！）

  为什么不是 1:1？
    半反应：I₂ + 2e⁻ → 2I⁻
           2S₂O₃²⁻ → S₄O₆²⁻ + 2e⁻
    → 1 个 I₂ 接收 2 个 e⁻ → 需要 2 个 S₂O₃²⁻ 提供 2 个 e⁻

应用场景判断：
  直接碘量法：测 Sn²⁺、S₂O₃²⁻、SO₃²⁻（还原性物质）
  间接碘量法：测 Cu²⁺、KMnO₄、H₂O₂、Cl₂（氧化性物质→氧化 I⁻→I₂→Na₂S₂O₃ 滴定）
```

## 三、KMnO₄ 法 · 酸性条件是铁律

```
KMnO₄ 法的三大特征：

  ① 酸性条件是必须的！
     MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
     非酸性条件 → MnO₂ 棕色沉淀 → 终点无法判断！

  ② 自指示剂（无需外加指示剂！）
     MnO₄⁻ 紫色 → Mn²⁺ 淡粉色（几乎无色）
     终点：过量一滴 KMnO₄ 使溶液呈淡粉色（30s 不褪）

  ③ 典型滴定反应：
     KMnO₄ 滴定 Fe²⁺：
       5Fe²⁺ + MnO₄⁻ + 8H⁺ → 5Fe³⁺ + Mn²⁺ + 4H₂O
       计量比 Fe²⁺ : MnO₄⁻ = 5 : 1

     KMnO₄ 滴定 H₂C₂O₄：
       5H₂C₂O₄ + 2MnO₄⁻ + 6H⁺ → 10CO₂ + 2Mn²⁺ + 8H₂O
       计量比 H₂C₂O₄ : MnO₄⁻ = 5 : 2
       ⚠️ 需加热（~70°C）——室温反应太慢！
```

## 四、沉淀滴定 · 两法对比

```
沉淀滴定的核心问题：如何判断 Cl⁻ 刚好被 Ag⁺ 沉淀完？

方法1：莫尔法（Mohr）——直接滴定
  指示剂：K₂CrO₄（5%）
  滴定剂：AgNO₃
  原理：分步沉淀！
    Ag⁺ + Cl⁻ → AgCl↓（白色，Ksp=1.8×10⁻¹⁰，先沉淀）
    终点时：2Ag⁺ + CrO₄²⁻ → Ag₂CrO₄↓（砖红色，Ksp=1.1×10⁻¹²，后沉淀）
  条件：中性或弱碱性（pH 6.5~10.5）
    酸中：CrO₄²⁻ → Cr₂O₇²⁻（失去指示剂功能）
    强碱中：Ag⁺ → Ag₂O↓

方法2：Volhard 法——返滴定
  指示剂：Fe³⁺（铁铵矾）
  滴定剂：NH₄SCN（返滴定剂）
  原理：返滴定！
    ① 先加过量 AgNO₃（已知量）→ Ag⁺ + X⁻ → AgX↓
    ② 再用 NH₄SCN 滴定剩余 Ag⁺ → AgSCN↓（白色）
    ③ 终点：SCN⁻ + Fe³⁺ → FeSCN²⁺（血红色）
  条件：必须在酸性条件下（HNO₃ 介质）！
    防止 Fe³⁺ 水解：Fe³⁺ + 3H₂O → Fe(OH)₃↓
```

### 两法对比速查

| | 莫尔法 | Volhard 法 |
|:---|:---|:---|
| **指示剂** | K₂CrO₄ | Fe³⁺ |
| **滴定方式** | 直接滴定 | 返滴定 |
| **pH 条件** | 中性/弱碱性（6.5-10.5） | **酸性！**（HNO₃ 介质） |
| **终点颜色** | 白色→砖红色 | 白色→血红色 |
| **可测离子** | Cl⁻、Br⁻（不能测 I⁻，AgI 吸附 CrO₄²⁻） | Cl⁻、Br⁻、I⁻ 均可 |
| **干扰** | Ba²⁺、Pb²⁺（与 CrO₄²⁻ 沉淀） | 强氧化剂（氧化 SCN⁻） |

## 五、返滴定逻辑模板

```
返滴定计算通式（"总量 − 剩余量 = 反应量"）：

  n(待测物) = n(Ag⁺ 总) − n(Ag⁺ 剩余)
            = c(AgNO₃)·V(AgNO₃) − c(NH₄SCN)·V(NH₄SCN)

实例：测定 Cl⁻
  ① 加 50.00 mL 0.1000 M AgNO₃（过量）
  ② 用 0.0800 M NH₄SCN 返滴定，消耗 25.00 mL
  → n(Cl⁻) = 0.1000×50.00 − 0.0800×25.00 = 3.00 mmol
  → c(Cl⁻) = 3.00 mmol / V(样品)
```

## 六、常见易错点

| 错误 | 纠正 |
|:---|:---|
| "I₂ + S₂O₃²⁻ 计量比是 1:1" | **1:2！** I₂ 得 2e⁻，每个 S₂O₃²⁻ 只给 1e⁻ |
| "KMnO₄ 可以在任何 pH 下滴定" | **必须在强酸性条件下！** 否则生成 MnO₂ 干扰终点 |
| "KMnO₄ 滴定需要外加指示剂" | KMnO₄ 是**自指示剂**——自身紫色即指示终点 |
| "莫尔法和 Volhard 法条件一样" | 莫尔法→中性/弱碱；Volhard 法→**强酸性**（防 Fe³⁺ 水解） |
| "莫尔法可以直接测 I⁻" | AgI 强烈吸附 CrO₄²⁻→终点提前→**不能直接用莫尔法测 I⁻** |
| "Volhard 法返滴定用 NH₄SCN 滴 Ag⁺" | 注意是**返滴定**——先加过量 AgNO₃，再滴剩余！ |
| "碘量法中淀粉指示剂一开始就加" | 间接碘量法：淀粉应在**近终点**（淡黄色）时加——过早加会吸附 I₂ |
| "KMnO₄ 滴定 H₂C₂O₄ 在室温即可" | 需要**加热至 ~70°C**——室温反应太慢！ |

## 七、例题速练

### 例 1：碘量法计量比

**题**：用 0.1000 M Na₂S₂O₃ 滴定 I₂ 溶液，消耗 24.50 mL。计算 I₂ 的物质的量。

**速解**：I₂ + 2S₂O₃²⁻ → 2I⁻ + S₄O₆²⁻，计量比 1:2。n(I₂) = ½·n(S₂O₃²⁻) = ½ × 0.1000 × 24.50 = 1.225 mmol。

### 例 2：KMnO₄ 法条件

**题**：为什么 KMnO₄ 滴定 Fe²⁺ 必须在酸性条件下进行？写出酸性和中性条件的方程式对比。

**速解**：酸性：MnO₄⁻ + 5Fe²⁺ + 8H⁺ → Mn²⁺ + 5Fe³⁺ + 4H₂O（产物单一，终点清晰）。中性：MnO₄⁻ + 3Fe²⁺ + 4H⁺ → MnO₂↓ + 3Fe³⁺ + 2H₂O（MnO₂ 棕色沉淀干扰终点判断）。

### 例 3：Volhard 法返滴定

**题**：测定某溶液中 Cl⁻ 浓度：取 25.00 mL 样品，加 50.00 mL 0.1000 M AgNO₃，再用 0.0800 M NH₄SCN 返滴定，消耗 20.00 mL。求 c(Cl⁻)。

**速解**：n(Ag⁺总) = 0.1000×50.00 = 5.00 mmol；n(Ag⁺剩余) = 0.0800×20.00 = 1.60 mmol；n(Cl⁻) = 5.00−1.60 = 3.40 mmol；c(Cl⁻) = 3.40/25.00 = 0.136 M。
