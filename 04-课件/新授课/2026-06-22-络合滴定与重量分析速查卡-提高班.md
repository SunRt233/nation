# 络合滴定与重量分析 · 速查卡

> 基础班你背过"EDTA 是六齿配体"——本卡把络合滴定+重量分析从**名词记忆**升级到**EDTA 1:1 计量本质→K'f 条件控制→金属指示剂封闭/僵化→重量分析换算因数→沉淀条件选择**的定量判断系统。

---

## 一、EDTA · 为什么总是 1:1

```
EDTA（乙二胺四乙酸）= 六齿配体

  配位原子：4 个 O（四个羧基）+ 2 个 N（两个胺基）= 6 个配位点

  与金属离子配位时：
    ├─ 一个 EDTA 分子占据 Mⁿ⁺ 的全部 6 个配位点（八面体）
    ├─ 形成 5 个五元螯合环 → 螯合效应 → 特别稳定
    └─ 无论 Mⁿ⁺ 的电荷是多少 → 配位比始终 M : EDTA = 1 : 1

  → 所有 EDTA 滴定计算的计量比都是 1:1！
  → 这是络合滴定计算的"宪法"——不要被金属价态迷惑
```

### EDTA 的酸效应

```
EDTA 是四元酸 H₄Y，在不同 pH 下存在形式不同：

  pH < 2：H₄Y 为主
  pH 2-3：H₃Y⁻
  pH 3-6：H₂Y²⁻
  pH 6-10：HY³⁻
  pH > 10：Y⁴⁻（完全去质子化——配位能力最强！）

  → pH 越低，EDTA 越"懒得配位"（质子竞争）
  → 大多数 EDTA 滴定在 pH ≈ 10（NH₃-NH₄Cl 缓冲液）进行
```

## 二、条件稳定常数 K'f

```
Kf（绝对稳定常数）= [MY]/[M][Y]——理想条件下

但 Y⁴⁻ 的实际浓度取决于 pH！
  → αY(H) = [Y']/[Y⁴⁻]（酸效应系数，pH 越低→αY(H) 越大）
  → K'f = Kf / αY(H)（条件稳定常数）

物理意义：
  K'f ≥ 10⁸ → 可以准确滴定（滴定突跃 ≥ 0.2 pM 单位）
  K'f < 10⁸ → 不能准确滴定

pH 对滴定的影响（以 Ca²⁺ 为例）：
  pH = 6：αY(H) ≈ 10⁴·⁶⁵ → K'f ≈ 10¹⁰·⁷/10⁴·⁶⁵ ≈ 10⁶ → ❌ 不能滴定
  pH = 10：αY(H) ≈ 10⁰·⁴⁵ → K'f ≈ 10¹⁰·⁷/10⁰·⁴⁵ ≈ 10¹⁰·² → ✅ 可以滴定
```

## 三、金属指示剂 · 封闭 vs 僵化

```
金属指示剂原理（以 EBT = 铬黑 T 为例）：

  M-In + EDTA → M-EDTA + In（游离指示剂变色）
   (色1)            (无色)      (色2)

  EBT：M-In 为酒红色 → 终点时游离 In⁻ 为蓝色
        → 终点颜色变化：酒红 → 蓝

封闭现象：M-In 太稳定（K(MIn) > K(MY)）
  → EDTA 无法从 M-In 中夺取 M → 终点不变色！
  例：Al³⁺、Fe³⁺ 封闭 EBT——需加掩蔽剂（三乙醇胺/KCN）

僵化现象：M-In 难溶或动力学惰性
  → 置换反应太慢 → 终点拖尾、变色不敏锐
  例：Ni²⁺-EBT 有时僵化——需加热

封闭 vs 僵化速判：
  封闭 = 热力学问题（M-In 太稳定）→ 掩蔽
  僵化 = 动力学问题（反应太慢）→ 加热
```

| 金属指示剂 | 适用 pH | 颜色变化 | 适用离子 |
|:---|:---:|:---|:---|
| **EBT（铬黑 T）** | 7–11 | 酒红→蓝 | Ca²⁺、Mg²⁺、Zn²⁺ |
| **XO（二甲酚橙）** | < 6 | 红→黄 | Bi³⁺、Pb²⁺、Zn²⁺ |
| **PAN** | 2–10 | 红→黄 | Cu²⁺、Zn²⁺、Cd²⁺ |

## 四、掩蔽与解蔽

```
掩蔽：加掩蔽剂与干扰离子形成稳定配合物，使其不与 EDTA 反应

  常用掩蔽剂：
    KCN：掩蔽 Cu²⁺、Ni²⁺、Zn²⁺、Cd²⁺（剧毒！必须在碱性条件下使用）
    三乙醇胺：掩蔽 Al³⁺、Fe³⁺、Sn⁴⁺
    F⁻：掩蔽 Al³⁺、Fe³⁺（生成 AlF₆³⁻/FeF₆³⁻）
    NH₄F：掩蔽 Ca²⁺（生成 CaF₂↓ 沉淀掩蔽）

解蔽：将被掩蔽的离子重新释放出来
  例：Zn(CN)₄²⁻ + 4HCHO + 4H₂O → Zn²⁺ + 4HOCH₂CN + 4OH⁻
      （甲醛解蔽 Zn²⁺——破坏 CN⁻ 配合物）
```

## 五、重量分析 · 换算因数

```
重量分析的基本流程：

  样品溶液 → 沉淀剂 → 沉淀形式 → 过滤洗涤 → 烘干/灼烧 → 称量形式 → 称重

换算因数 F = 待测组分的摩尔质量 / 称量形式的摩尔质量
           ×（化学计量系数比）

关键原则：
  ├─ 称量形式摩尔质量应尽量大 → 减小称量相对误差
  ├─ 沉淀形式 ≠ 称量形式！（如 Fe(OH)₃ 沉淀 → Fe₂O₃ 称量）
  └─ 组成必须确定（如 BaSO₄ 既是沉淀形式也是称量形式）

实例：
  测定 Fe：沉淀形式 = Fe(OH)₃，称量形式 = Fe₂O₃
  F = 2×M(Fe) / M(Fe₂O₃) = 2×55.85 / 159.7 = 0.6994
  → m(Fe) = m(Fe₂O₃) × 0.6994
```

### 沉淀条件 · 晶形 vs 无定形

| | 晶形沉淀（BaSO₄型） | 无定形沉淀（Fe(OH)₃型） |
|:---|:---|:---|
| **沉淀条件** | 热、稀、慢加、搅拌、陈化 | 热、浓、快加、搅拌、**不陈化** |
| **粒径** | 大晶体，易过滤 | 细小胶状，不易过滤 |
| **纯度** | 高（共沉淀少） | 较低（吸附杂质多） |
| **陈化** | ✅ 必须陈化（小晶溶解→大晶长大） | ❌ 不陈化（会吸附更多杂质） |
| **口诀** | 热稀慢搅陈 | 热浓快搅不过夜 |

**陈化原理**：小晶体溶解度 > 大晶体（Kelvin 效应，定性）→ 小晶体溶解 → 溶质在大晶体表面析出 → 晶体长大变纯。

## 六、常见易错点

| 错误 | 纠正 |
|:---|:---|
| "EDTA 与 M²⁺ 配位比是 1:2" | **永远是 1:1！** 一个 EDTA 占满全部 6 个配位点，与金属电荷无关 |
| "任何 pH 下都可以用 EDTA 滴定" | 必须先算 K'f！pH 太低→αY(H) 太大→K'f 太小→**不能滴定** |
| "封闭和僵化是同一种现象" | 封闭=热力学（K(MIn)太大），僵化=动力学（反应太慢）——处理方法不同 |
| "沉淀形式 = 称量形式" | **不一定！** Fe(OH)₃ 沉淀→Fe₂O₃ 称量→换算因数 2Fe:1Fe₂O₃ |
| "所有沉淀都需要陈化" | 无定形沉淀**不能陈化**——越陈化吸附杂质越多 |
| "称量形式摩尔质量小好" | **摩尔质量越大越好！** 同样称量误差下，待测组分相对误差更小 |
| "EDTA 滴定时用甲基橙做指示剂" | 络合滴定用**金属指示剂**（EBT/XO），不是酸碱指示剂 |

## 七、例题速练

### 例 1：EDTA 计量计算

**题**：用 0.0100 M EDTA 滴定 25.00 mL 水样中的 Ca²⁺+Mg²⁺（总硬度），消耗 EDTA 18.50 mL。计算水总硬度（以 mg CaCO₃/L 表示）。

**速解**：M:EDTA = 1:1 → n(Ca+Mg) = 0.0100 × 18.50 = 0.1850 mmol。换算为 CaCO₃：m = 0.1850 × 100.09 = 18.52 mg。硬度 = 18.52 / 0.02500 = 741 mg/L（以 CaCO₃ 计）。

### 例 2：换算因数

**题**：测定某矿物中 Al 含量：样品 0.5000g 溶解后，Al³⁺ 沉淀为 Al(OH)₃，灼烧后得 Al₂O₃ 0.2038g。求 Al 的质量分数。

**速解**：F = 2×M(Al)/M(Al₂O₃) = 2×26.98/101.96 = 0.5292。m(Al) = 0.2038 × 0.5292 = 0.1079g。w(Al) = 0.1079/0.5000 = 21.58%。

### 例 3：指示剂封闭判断

**题**：用 EBT 作指示剂、EDTA 滴定 Zn²⁺ 时，溶液中有少量 Al³⁺。终点不变色。为什么？如何解决？

**速解**：Al³⁺ 封闭 EBT——Al-EBT 比 Al-EDTA 更稳定 → EDTA 无法夺取 Al³⁺，EBT 始终呈 Al-EBT 的酒红色。解决：加 NH₄F 掩蔽 Al³⁺（Al³⁺+6F⁻→AlF₆³⁻）。

## 八、自查清单

- [ ] EDTA 六齿配位结构（4O+2N），M:EDTA=1:1 永不变
- [ ] 酸效应系数 αY(H) → K'f = Kf/αY(H) → K'f ≥ 10⁸ 才可滴定
- [ ] pH 越低→αY(H) 越大→K'f 越小→滴定越困难
- [ ] 金属指示剂变色原理：M-In + EDTA → M-EDTA + In（游离色）
- [ ] 封闭（热力学，K(MIn)太大→掩蔽）vs 僵化（动力学，反应慢→加热）
- [ ] 掩蔽剂选择：KCN（Cu/Ni/Zn）、三乙醇胺（Al/Fe）、F⁻（Al/Fe）
- [ ] 换算因数 F = M(待测组分)/M(称量形式) × 化学计量系数
- [ ] 称量形式摩尔质量越大→称量相对误差越小
- [ ] 晶形沉淀：热稀慢搅陈；无定形沉淀：热浓快搅不陈化
- [ ] 陈化使小晶体溶解、大晶体长大——提高纯度
