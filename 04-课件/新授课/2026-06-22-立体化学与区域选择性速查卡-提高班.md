# 立体化学与区域选择性 · 速查卡

> 基础班你学会了"辨认 R/S、识别顺/反"——本卡把选择性从**辨认层**升级到**判断层**：先分是"位置"还是"面"的问题，再找哪个过渡态/中间体决定了结果。

---

## 一、选择性判断总流程

```
拿到有机反应题 → 先问两个问题
    │
    ├─ 是"位置"问题 → 区域选择性
    │   ├─ 亲电加成：Markovnikov vs 反 Markovnikov
    │   ├─ 共轭加成：1,2 vs 1,4
    │   ├─ 芳香取代：o/p vs m
    │   └─ 消除反应：Saytzeff vs Hofmann
    │
    └─ 是"面"问题 → 立体选择性
        ├─ 加成反应：syn vs anti
        ├─ 消除反应：syn vs anti（E2 反式共平面要求）
        ├─ SN2：构型翻转（Walden 翻转）
        └─ 周环：同旋 vs 间旋、同面 vs 异面
```

**核心原则**：先分位置还是面，再看过渡态/中间体，最后用电子/位阻修正。

---

## 二、区域选择性速判

### 2.1 Markovnikov vs 反 Markovnikov

| 规则 | 适用 | 中间体/过渡态 | 口诀 |
|:---|:---|:---|:---|
| **Markovnikov** | HX 加成到不对称烯烃 | 碳正离子中间体→H 加在氢多的碳上（生成更稳 C⁺） | "H 加在氢多的一侧" |
| **反 Markovnikov** | HBr + 过氧化物 | 自由基链→Br• 先加到端碳（生成更稳自由基） | "过氧化物→反马氏→只对 HBr 有效!" |
| **硼氢化-氧化** | BH₃ + H₂O₂/OH⁻ | 四中心过渡态→B 加在端碳（位阻小）→H 加在内碳 | "硼氢化→反马氏→syn 加成→醇" |

### 2.2 共轭加成：1,2 vs 1,4

```
α,β-不饱和羰基 + 亲核试剂 → 两个位点

  动力学控制（低温/强亲核/不可逆）→ 1,2 加成（直接攻羰基）
  热力学控制（高温/弱亲核/可逆）→ 1,4 加成（共轭加成，产物更稳定）
  
  一句话：低温得1,2（快），高温得1,4（稳）
```

### 2.3 芳香取代定位

| 已有取代基 | 定位 | 活化/钝化 | 原因 |
|:---|:---:|:---:|:---|
| –NH₂, –OH, –OR | o/p | 活化 | 孤对电子共轭给电子→稳定 σ 配合物 |
| –R（烷基）| o/p | 弱活化 | 超共轭/诱导给电子 |
| –X（卤素）| o/p | **钝化**（但定位仍是 o/p！） | —I 钝化 > +M 活化→净钝化，但 o/p 仍最稳 |
| –NO₂, –CN, –CF₃ | m | 钝化 | 吸电子→o/p 位 σ 配合物特别不稳定→只能走 m |
| –NR₃⁺, –CF₃ | m | 钝化 | 强吸电子 |

**o/p vs m 的本质**：σ 配合物中间体的稳定性——哪个位置的 σ 配合物最稳定（正电荷被最好地分散），哪个位置就被优先取代。

## 三、立体选择性速判

### 3.1 syn vs anti 加成

| 反应 | 立体化学 | 原因 |
|:---|:---:|:---|
| **H₂/Pd（催化氢化）** | syn | H₂ 和烯烃都在催化剂表面同一侧 |
| **Br₂ 加成** | anti | 溴鎓离子中间体→Br⁻ 从反面进攻 |
| **mCPBA 环氧化** | syn | 协同机理→O 从一面加 |
| **硼氢化-氧化** | syn | 四中心过渡态→B 和 H 同侧加 |
| **羟汞化-去汞** | anti（总结果） | 汞鎓离子→H₂O 反式进攻→去汞后总 anti |

### 3.2 E2 消除的立体要求

```
E2 的刚性要求：H–C 与 C–LG 呈 ~180° 二面角（反式共平面）

环己烷体系判断口诀：
  1. 先画椅式构象
  2. 找离去基团——它在 a 键还是 e 键？
  3. 只有 a 键的离去基团才有邻位 a-H 与之反式 → 能消
  4. 如果离去基团在 e 键 → 需环翻转 → 检查翻转后是否能 a
  5. 如果 t-Bu 锁定环 → 只有一种椅式 → 只能消特定 H
```

### 3.3 SN2 的 Walden 翻转

SN2 = 背面进攻 → 构型翻转。如果离去基团和亲核试剂优先级不同→绝对构型 R↔S 翻转。

## 四、选择性的五维控制框架

> 当区域选择性和立体选择性冲突时，用五维判断谁主导。

| 维度 | 控制什么 | 典型表现 |
|:---|:---|:---|
| **电子** | 区域选择性为主 | 马氏规则（C⁺稳定性）、SEAr 定位（σ配合物稳定性） |
| **立体/位阻** | 立体选择性为主 | E2 反式共平面、SN2 背面进攻、硼氢化端位加B |
| **动力学 vs 热力学** | 1,2 vs 1,4、Saytzeff vs Hofmann | 低温=动力学=快路径；高温=热力学=稳产物 |
| **溶剂** | 影响中间体稳定性 | 极性溶剂稳定 C⁺→倾向 SN1/E1 |
| **轨道** | 周环反应的立体专一性 | WH 规则决定同旋/间旋 |

## 五、常见易错点

| 错误 | 纠正 |
|:---|:---|
| 反 Markovnikov = 所有 HX + 过氧化物都行 | **只有 HBr！** HCl/HI 的链增长步热力学不利 |
| 卤素是钝化基团所以定位是 m | 卤素 —I（钝化）和 +M（活化）竞争→净钝化但定位仍是 o/p! |
| E2 消除只看 Br 的位置 | 要先看环能否翻转——翻转后 a↔e 互换，E2 只能在 a,a 反式时发生 |
| 1,2 vs 1,4 只取决于温度 | 也取决于亲核试剂的硬软和可逆性——硬亲核倾向 1,2，软亲核倾向 1,4 |

## 六、例题速练

### 例 1：区域选择性判断

**题**：CH₃CH=CH₂ + HBr + 过氧化物 → ？为什么是反 Markovnikov？

**速解**：过氧化物→自由基链机理。Br• 先加到端碳（生成 2° 自由基比 1° 自由基稳定）→然后 H 从 HBr 提取→H 加在内碳→反马氏产物 CH₃CH₂CH₂Br。

### 例 2：立体化学判断

**题**：反-1-Br-2-Me-环己烷 + t-BuOK → 产物是什么？为什么只有一个？

**速解**：
1. t-BuOK = 大位阻强碱→E2
2. 画椅式：Br e 键（向下），Me e 键（向下）
3. Br 在 e 键→不能直接消→需要环翻转！
4. 翻转后：Br a 键（向上），Me a 键（向上）
5. 找 Br 邻位 a-H：C2 上的 H 是 a（向下）→与 Br（向上）反式→可消
6. **单一产物**：C1=C2 双键

- [ ] 能先分"位置"（区域选择性）还是"面"（立体选择性）问题
- [ ] Markovnikov vs 反 Markovnikov 的三种情况（HX/过氧化物/硼氢化）
