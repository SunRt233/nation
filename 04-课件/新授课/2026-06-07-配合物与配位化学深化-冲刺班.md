### 2.1 开场：光谱化学序列的秘密——为什么 CO 是强场？

**经典错误**："因为 C 电负性低，所以 CO 给电子能力强，所以是强场。"

**真相**：CO 是强场的根本原因是**反馈 π 键（back-bonding）**——金属的填满 d 轨道（t₂g）向 CO 的空 π* 轨道反馈电子，拉低了金属 t₂g 能级→Δo 增大。

这是本课的核心图景：**光谱化学序列不是背出来的——它来自轨道相互作用。**

---

### 2.2 配体三分类：σ 给体 / π 给体 / π 受体

每种配体对 Δo 的影响取决于它与金属 d 轨道的**σ 和 π 两种相互作用**：

| 配体类型 | σ 作用 | π 作用 | 对 t₂g 的影响 | 对 Δo 的影响 | 代表配体 |
|:---|:---|:---|:---|:---|:---|
| **σ 给体**（only） | 给 σ 电子→形成 eg σ 键 | 无 π 作用 | t₂g 不变（非键） | Δo = **中等** | NH₃, en, RNH₂ |
| **π 给体** | 给 σ 电子 | **填满的配体 π→金属 t₂g**（给 π 电子） | **推高** t₂g | Δo **减小** ⬇ | F⁻, Cl⁻, Br⁻, I⁻, OH⁻, H₂O, O²⁻ |
| **π 受体** | 给 σ 电子 | **金属 t₂g→空的配体 π\***（反馈 π） | **拉低** t₂g | Δo **增大** ⬆ | CO, CN⁻, PR₃, N₂, NO⁺, C₂H₄ |

**由此推出完整光谱化学序列**：

$$\text{CO} \approx \text{CN}^- > \text{NO}_2^- > \text{PR}_3 > \text{en} > \text{NH}_3 > \text{py} \approx \text{H}_2\text{O} > \text{F}^- > \text{OH}^- > \text{Cl}^- > \text{Br}^- > \text{I}^-$$

| ← 强场（Δo 大，倾向低自旋） | 中间场 | 弱场（Δo 小，倾向高自旋）→ |
|:---|:---|:---|

**记忆技巧**："CO≈CN⁻ 最强场（反馈 π 拉 t₂g），卤素最弱场（填满 p 推 t₂g），NH₃ 在中间（纯 σ 给体无 π）。"

### 2.3 八面体 ML₆ σ 成键 MO 图：轨道级别标注

> ⚠️ 这是 2025 决赛新题型要求——不是"d 轨道分裂成 t₂g 和 eg"，而是"dₓ₂₋ᵧ₂ + N 2pₓ → eg σ 成键轨道"。

#### 2.3.1 准备工作：金属和配体的轨道

**金属提供 9 个价轨道**（第一过渡系）：
- 5 个 3d：d_z², dₓ₂₋ᵧ², d_{xy}, d_{xz}, d_{yz}
- 1 个 4s
- 3 个 4p

**6 个配体各提供 1 个 σ 型轨道**（指向金属，如 NH₃ 的孤对电子轨道，Cl⁻ 的填满 p_z）→ 共 6 个配体 σ 轨道。

#### 2.3.2 按对称性分类：配体群轨道（LGO）

6 个配体 σ 轨道按对称性分为三组：

| 对称性标记 | 组成 | 与金属哪个轨道匹配？ |
|:---|:---|:---|
| **a₁g** | 所有 6 个配体同相位（全对称） | 金属 **4s**（球形对称） |
| **t₁u** | 三重组（沿 x, y, z 各一组，对面反相） | 金属 **4p**（pₓ, p_y, p_z 的对称性） |
| **eg** | 二重组（dₓ₂₋ᵧ² 型/d_z² 型对称组合） | 金属 **3dₓ₂₋ᵧ²** 和 **3d_z²** |

**t₂g（d_{xy}, d_{xz}, d_{yz}）**——配体的 σ 轨道没有匹配对称性→**非键**（在仅有 σ 相互作用时）。

#### 2.3.3 逐步构建 MO 图

```
能量 ↑
      │  4p ───→ t₁u*（反键，主要金属 4p 特征）
      │  4s ───→ a₁g*（反键，主要金属 4s 特征）
      │  3d ─┬─→ eg*（反键，主要金属 dₓ₂₋ᵧ² + d_z² 特征）
      │      │        ↕ Δo = eg* - t₂g 的间距
      │      ├─→ t₂g（非键，纯金属 d_{xy}, d_{xz}, d_{yz}）
      │      │
      │      ├─→ eg（成键，主要配体特征 + 金属 dₓ₂₋ᵧ²/d_z²）
      │ 配体 ├─→ t₁u（成键，主要配体特征 + 金属 4p）
      │ LGO  ├─→ a₁g（成键，主要配体特征 + 金属 4s）
```

**轨道级别标注示例——[Co(NH₃)₆]³⁺**（d⁶ 低自旋）：

| MO | 能量级别 | 主要轨道来源 | 标注写法（2025 真题模式） |
|:---|:---|:---|:---|
| eg* | 反键（LUMO） | Co 3dₓ₂₋ᵧ² + N 2s/2pₓ（反相位） | Co dₓ₂₋ᵧ² − N 2pₓ → eg σ* |
| t₂g | 非键（HOMO） | 纯 Co 3d_{xy}, 3d_{xz}, 3d_{yz} | Co d_{xy}, d_{xz}, d_{yz} → t₂g（非键） |
| eg | 成键 | N 2s/2p（同相位）+ Co 3dₓ₂₋ᵧ²/d_z² | N 2pₓ + Co dₓ₂₋ᵧ² → eg σ |
| t₁u | 成键 | N 2p + Co 4p | N 2p + Co 4p → t₁u σ |
| a₁g | 成键 | N 2s + Co 4s | N 2s + Co 4s → a₁g σ |

**关键理解**：Δo = eg* − t₂g 的间距。t₂g 是非键（纯金属 d）→能量不变；eg* 是反键 → 能量被推高。**配体 σ 给电子能力越强→eg* 推得越高→Δo 越大。**

### 2.4 π 键修正→光谱化学序列完整解释

上节的 MO 图只有 σ 键。现在加入 π 相互作用：

#### π 给体配体（Cl⁻, Br⁻, I⁻, OH⁻, H₂O）

配体有**填满的 p 轨道**（如 Cl⁻ 的 3pₓ, 3p_y），对称性与金属 t₂g 匹配：

```
配体填满 p(π) ──→ 金属 t₂g（给电子）
                         ↓
                  推高 t₂g 能级
                         ↓
                  Δo = eg* − t₂g(↑) → Δo 减小！
```

**结论**：π 给体配体 → t₂g 升高 → Δo ↓ → **弱场配体**。

#### π 受体配体（CO, CN⁻, PR₃）

配体有**空的 π\* 轨道**（如 CO 的 2π*），对称性与金属 t₂g 匹配：

```
金属 t₂g ──→ 配体空 π*（反馈 π 键）
       ↓
拉低 t₂g 能级
       ↓
Δo = eg* − t₂g(↓) → Δo 增大！
```

**结论**：π 受体配体 → t₂g 降低 → Δo ↑ → **强场配体**。

**CO 反馈 π 键的三个关键事实**：
1. CO 的 HOMO 是 5σ（C 上的孤对）→ 给 σ 电子到金属（σ 给体）
2. CO 的 LUMO 是空的 2π* → 接受金属 t₂g 反馈电子（π 受体）
3. 反馈 π 键 → C≡O 键级降低（红外 ν_CO 从 2143 cm⁻¹ 降至 ~2000 cm⁻¹）— **实验可测！**

### 2.5 CFSE + JT 效应深度：从计算到结构预测

#### 2.5.1 CFSE 速查公式

**八面体场**：
$$\text{CFSE} = (-0.4n_{t_{2g}} + 0.6n_{e_g})\Delta_o$$

**四面体场**：
$$\Delta_t \approx \frac{4}{9}\Delta_o$$
$$\text{CFSE}_\text{tet} = (-0.6n_e + 0.4n_{t_2})\Delta_t$$

| dⁿ | 八面体 HS CFSE | 八面体 LS CFSE | 四面体 CFSE | 八面体偏好？ |
|:---:|:---:|:---:|:---:|:---:|
| d¹ | −0.4 Δo | — | −0.27 Δo | 八面体略优 |
| d² | −0.8 Δo | — | −0.53 Δo | 八面体略优 |
| d³ | −1.2 Δo | — | −0.36 Δo | **八面体强偏好**（Cr³⁺ 永远是八面体！） |
| d⁴ | −0.6 Δo | −1.6 Δo + P | −0.18 Δo | HS 时八面体偏好弱；LS 时八面体极强偏好 |
| d⁵ | 0 | −2.0 Δo + 2P | 0 | HS 时无 CFSE 差→可能四面体 |
| d⁶ | −0.4 Δo | −2.4 Δo + 2P | −0.27 Δo | LS 时八面体极强偏好（Co³⁺ 惰性！） |
| d⁷ | −0.8 Δo | −1.8 Δo + P | −0.53 Δo | HS 时八面体偏好中等 |
| d⁸ | −1.2 Δo | — | −0.36 Δo | **八面体强偏好**（Ni²⁺） |
| d⁹ | −0.6 Δo | — | −0.18 Δo | 八面体偏好弱（Cu²⁺ 也常见四方平面） |
| d¹⁰ | 0 | — | 0 | 无 CFSE→配体决定几何 |

**竞赛应用**：CFSE 大小 = 八面体偏好的强度。d³(Cr³⁺)和 d⁸(Ni²⁺)的 CFSE 差最大→强烈偏好八面体。这对理解**尖晶石正/反分布**至关重要。

**尖晶石正/反的 CFSE 判据**：
- **正尖晶石**：A²⁺ 占四面体位，B³⁺ 占八面体位
- **反尖晶石**：A²⁺ 占八面体位，B³⁺ 一半四面体一半八面体
- 判据：哪个离子的八面体 CFSE > 四面体 CFSE → 那个离子抢占八面体位

**例题**：NiFe₂O₄ 为何是反尖晶石？
- Ni²⁺(d⁸)：八面体 CFSE = −1.2Δo，四面体 CFSE = −0.36Δt ≈ −0.16Δo → ΔCFSE = −1.04Δo（巨大！）
- Fe³⁺(d⁵)：CFSE = 0（HS d⁵ 在任何场中都是 0）
- 结论：Ni²⁺ 必须抢占八面体→反尖晶石 Fe³⁺[Ni²⁺Fe³⁺]O₄

#### 2.5.2 Jahn-Teller 效应的轨道驱动力

**JT 定理**：非线性分子如果在基态有轨道简并→一定会发生结构畸变来消除简并。

**哪些电子构型必发生 JT 畸变？**

| dⁿ | 八面体基态 | e_g 占据 | JT 畸变？ | 畸变方向 |
|:---:|:---|:---:|:---:|:---|
| d⁴ HS | t₂g³ e_g¹ | 不对称（d_z² 或 dₓ₂₋ᵧ²） | ✅ **必畸变** | 通常拉长（z 轴） |
| d⁷ LS | t₂g⁶ e_g¹ | 不对称 | ✅ **必畸变** | 通常拉长 |
| d⁹ | t₂g⁶ e_g³ | 不对称（一个轨道 2e⁻，另一个 1e⁻） | ✅ **必畸变** | **拉长**（Cu²⁺ 经典案例） |
| d⁸ | t₂g⁶ e_g² | 对称 | ❌ 无 JT | 但可能变成四方平面（极端 JT） |

**为什么 Cu²⁺(d⁹) 的八面体总是拉长的？**

e_g 轨道：d_z²（1e⁻）和 dₓ₂₋ᵧ²（2e⁻）→ z 轴方向电子密度比 xy 平面少→z 轴配体受到的屏蔽较小→z 轴键缩短倾向。但 e_g 反键占据不对称→系统通过**拉长 z 轴键**来降低能量（z 轴拉长→d_z² 能量降低→电子优先占据 d_z²→dₓ₂₋ᵧ² 只有一个电子→总能量最低）。

**拉长 vs 压扁**：d⁹ Cu²⁺ → **几乎总是拉长**（z 轴键长 > xy 平面）。因为拉长使 e_g 分裂后能量降低更多。

### 2.6 磁性Ⅰ：从 μ_eff 反推一切

#### 2.6.1 基础：自旋-only 公式

$$\mu_\text{eff} = \sqrt{n(n+2)} \text{ BM}$$

其中 n = 未成对电子数。注意：此公式忽略轨道角动量贡献——对大多数第一过渡系 dⁿ 配合物足够准确。

| n | μ_eff (BM) | 典型例子 |
|:---:|:---:|:---|
| 0 | 0（反磁） | d⁶ LS Co³⁺, d⁸ 四方平面 Ni²⁺, Pt²⁺ |
| 1 | 1.73 | d¹ Ti³⁺, d⁵ LS Fe³⁺, d⁷ LS Co²⁺ |
| 2 | 2.83 | d² V³⁺, d⁴ LS Mn³⁺, d⁸ Ni²⁺ 八面体 |
| 3 | 3.87 | d³ Cr³⁺, d⁷ HS Co²⁺ |
| 4 | 4.90 | d⁴ HS Mn³⁺, d⁶ HS Fe²⁺ |
| 5 | 5.92 | d⁵ HS Fe³⁺, Mn²⁺ |

#### 2.6.2 磁性数据→完整推理链

**推理链**：μ_eff 实验值 → n → 结合配体场强度 → 判断 HS/LS → 结合氧化态 → 推断几何构型

**例题**：某配合物 μ_eff = 4.90 BM，组成为 [Fe(NH₃)₆]²⁺。

- μ = 4.90 → n = 4（μ ≈ √24 = 4.90）
- Fe²⁺ = d⁶。d⁶ HS = 4 u.e. ✓，d⁶ LS = 0 u.e. ✗
- NH₃ 是中间场配体 → Fe²⁺ 在 NH₃ 场中通常是 HS（因为 Fe²⁺ 的 Δo 不够大）
- d⁶ HS 八面体 → t₂g⁴ e_g² → 正八面体（无 JT，e_g 对称占据）
- 结论：高自旋八面体 [Fe(NH₃)₆]²⁺

**进阶例题**：某配合物 μ_eff = 0 BM，组成为 [Ni(CN)₄]²⁻。

- μ = 0 → 反磁 → n = 0
- Ni²⁺ = d⁸。d⁸ 八面体永远有 2 个未成对电子（e_g 对称占据）→ 不可能是八面体
- CN⁻ 是强场 → 四方平面几何→d 轨道分裂：d_{xy} < d_{xz},d_{yz} < d_z² < dₓ₂₋ᵧ²
- d⁸ 四方平面：填满 d_{xy}, d_{xz}, d_{yz}, d_z² → 全部配对 → 反磁 ✓
- 结论：四方平面 [Ni(CN)₄]²⁻（经典案例）

### 2.7 磁性Ⅱ：SCO 与 SMM（2025 竞赛热点）

#### 2.7.1 自旋态交叉（SCO = Spin Crossover）

**定义**：某些配合物（尤其是 d⁴-d⁷ 第一过渡系）在外界条件（温度、压力、光照）改变时，发生高自旋↔低自旋的可逆转变。

**经典体系**：Fe²⁺ N₆ 配合物（如 [Fe(phen)₂(NCS)₂]）

| 温度 | 自旋态 | μ_eff | Fe-N 键长 | 颜色 |
|:---|:---|:---:|:---:|:---|
| 低温（< 180 K） | LS (¹A₁) | ~0 BM（反磁） | ~1.97 Å | 深紫色 |
| 高温（> 220 K） | HS (⁵T₂) | ~5.0 BM | ~2.18 Å | 白色/浅黄 |

**SCO 的判据**：
- μ_eff 随温度渐变（不是突跃→是渐变转变）
- 键长随温度明显变化（HS Fe-N ≈ 2.18 Å > LS ≈ 1.97 Å）
- 存在热滞回线（T↑ 和 T↓ 的转变温度不同——合作效应）

**2025 竞赛要点**：SCO 的驱动因素不是 Δo 的温度变化（Δo 随温度变化很小），而是**振动熵**——HS 态有更多的低频振动模式（键更弱更长→力常数更小→振动频率更低→振动熵更大）→高温时熵项 −TΔS 使 HS 更稳定。

#### 2.7.2 单分子磁体（SMM = Single Molecule Magnet）

**定义**：单个分子（而非三维磁有序晶体）在低温下表现出磁化强度的缓慢弛豫——即每个分子像一个"纳米磁铁"。

**SMM 的两个必要条件**：
1. **大的基态自旋 S**（多未成对电子）
2. **负的磁各向异性 D**（易轴型各向异性→磁矩优先沿某个方向排列）

**能量势垒**：$U_\text{eff} = |D|S^2$（整数 S）或 $|D|(S^2-1/4)$（半整数 S）

**2025 决赛真题——直线形 Co²⁺ SMM**：

Co²⁺ = d⁷。在直线形配体场中（两个配体沿 z 轴）：
- d_z²：最强 σ 反键→能量最高
- d_{xz}, d_{yz}：π 作用→中间能量
- d_{xy}, dₓ₂₋ᵧ²：δ 对称性→能量最低（几乎不与配体作用）

d 轨道分裂（从低到高）：d_{xy}, dₓ₂₋ᵧ² < d_{xz}, d_{yz} < d_z²

d⁷ 填充：d_{xy}² dₓ₂₋ᵧ²² d_{xz}² d_{yz}² d_z²¹ → S = 3/2

Co²⁺ 有大的轨道角动量贡献（不像第一过渡系其他离子）→极强的磁各向异性→优秀 SMM 候选。

**2025 真题电子分布几率分析**（42.8%/41.2%/16.0%）：
- Co²⁺ d⁷ 在直线形场：基态 d_{xz}² d_{yz}² d_z²¹ + 低激发态（d_{xz} 或 d_{yz} 的一个电子跃迁到 d_z² 或更高）
- Boltzmann 分布：$p_i = g_i e^{-E_i/kT} / \sum_j g_j e^{-E_j/kT}$
- 三个低能态热分布→42.8% + 41.2% + 16.0% = 100%

### 2.8 惰性 vs 不稳定：热力学≠动力学

> ⚠️ 竞赛最常见的混淆概念——"配合物稳定=配体交换慢"← **完全错误！**

| 概念 | 衡量指标 | 含义 | 典型例子 |
|:---|:---|:---|:---|
| **热力学稳定** | 形成常数 $K_f$（大=稳定） | 平衡时配合物的**浓度占比** | [Co(NH₃)₆]³⁺ K_f ≈ 10³⁵（非常稳定）|
| **动力学惰性** | 配体交换速率 $k$（小=慢=惰性） | 配体**多快被换掉** | Cr³⁺ 配合物→配体交换 t₁/₂ 以小时计 |
| **动力学不稳定** | $k$ 大=快 | 配体交换快 | Cu²⁺ 配合物→配体交换几乎是瞬时的 |

**Taube 分类法**：

| 电子构型 | 惰性/不稳定 | 轨道解释 |
|:---|:---|:---|
| **d³**（Cr³⁺, V²⁺） | **惰性** | CFSE = −1.2Δo，配体解离损失 CFSE 巨大→活化能高 |
| **d⁶ LS**（Co³⁺, Fe²⁺ LS） | **惰性** | CFSE = −2.4Δo（最大！），配体解离损失 CFSE 巨大 |
| **d⁸ 四方平面**（Pt²⁺, Pd²⁺, Au³⁺） | **惰性** | 四方平面 dₓ₂₋ᵧ² 为空（LUMO）→轴向没有配体遮挡→S_N2 机理但活化能仍高 |
| d¹, d² | 不稳定 | CFSE 小→配体交换活化能低 |
| d⁴, d⁵, d⁶ HS | 不稳定 | CFSE 不大→活化能低 |
| d⁷, d⁸, d⁹, d¹⁰ | 不稳定 | CFSE 小或无（d¹⁰ CFSE=0） |

**关键理解**：惰性的轨道解释 → CFSE 对活化能的贡献。配体解离（D 机理）→中间体配位数降低→CFSE 改变→CFSE 损失越大→活化能越高→越惰性。

### 2.9 反式效应（trans effect）：指挥 Pt²⁺ 取代反应

#### 2.9.1 概念区分

| 概念 | 定义 | 性质 |
|:---|:---|:---|
| **反式效应（trans effect）** | 配体 T 加速其 **trans 位**配体的**取代反应速率** | **动力学**概念 |
| **反式影响（trans influence）** | 配体 T 削弱其 **trans 位** M-L 键的**键强度** | **热力学**概念 |

两者有关联但不同——反式效应大的配体通常反式影响也大，但不是严格对应。

#### 2.9.2 Pt²⁺ 反式效应序列

$$\text{CN}^- \approx \text{CO} > \text{PR}_3 \approx \text{H}^- > \text{CH}_3^- > \text{C}_6\text{H}_5^- > \text{I}^- > \text{Cl}^- > \text{NH}_3 \approx \text{py} > \text{OH}^- > \text{H}_2\text{O}$$

| ← 强反式效应（加速 trans 位取代） | 弱反式效应 → |
|:---|:---|

**物理本质**：强反式效应配体（如 CO, CN⁻, PR₃）→既是强 σ 给体又是 π 受体→与金属形成强键→削弱 trans 位的 M-L 键→降低 trans 位配体解离的活化能→加速取代。

#### 2.9.3 反式效应指导合成设计——经典例题

**任务**：从 [PtCl₄]²⁻ 出发，合成 cis-PtCl₂(NH₃)(NO₂)。

**分析**：目标是顺式（NH₃ 和 NO₂⁻ 相邻）。需要利用反式效应来控制取代顺序。

**路线 A**（先加 NH₃，后加 NO₂⁻）：
```
[PtCl₄]²⁻  +  NH₃  →  [PtCl₃(NH₃)]⁻  +  Cl⁻
```
Cl⁻ 的反式效应 > NH₃ → trans 到 Cl⁻ 的 Cl 更容易被取代 → 再加 NO₂⁻：
```
[PtCl₃(NH₃)]⁻  +  NO₂⁻  →  trans-PtCl₂(NH₃)(NO₂)  ← 反式产物！不是我们想要的。
```

**路线 B**（先加 NO₂⁻，后加 NH₃）：
```
[PtCl₄]²⁻  +  NO₂⁻  →  [PtCl₃(NO₂)]²⁻  +  Cl⁻
```
NO₂⁻ 的反式效应 > Cl⁻ → trans 到 NO₂⁻ 的 Cl 优先被取代 → 再加 NH₃：
```
[PtCl₃(NO₂)]²⁻  +  NH₃  →  cis-PtCl₂(NH₃)(NO₂)  ✓ 顺式产物！
```

**结论**：利用反式效应序列→先加弱反式效应配体占据→再加目标配体取代其 trans 位→得到想要的异构体。

### 2.10 配体取代机理：D / I_d / Iₐ

#### 2.10.1 三种极限机理

| 机理 | 全称 | 决速步 | 中间体 | 速率定律 | 立体化学 |
|:---|:---|:---|:---|:---|:---|
| **D**（解离） | Dissociative | M-L 键断裂 | 5 配位中间体（可被检测） | $v = k[\text{ML}_n]$（与进入配体浓度无关） | 通常外消旋化 |
| **I_d**（交换解离） | Interchange Dissociative | 键断裂 > 键形成 | 无稳定中间体（紧密离子对） | $v = k[\text{ML}_n]$ | 部分保留构型 |
| **Iₐ**（交换缔合） | Interchange Associative | 键形成 > 键断裂 | 无稳定中间体 | $v = k[\text{ML}_n][\text{L}']$ | 可能反转 |
| **A**（缔合） | Associative | M-L' 键形成 | 7 配位中间体 | $v = k[\text{ML}_n][\text{L}']$ | 构型反转常见 |

#### 2.10.2 如何从实验数据判断机理？

**关键实验**：
1. **速率定律**：$v = k[\text{配合物}]$ → D 或 I_d；$v = k[\text{配合物}][\text{L}']$ → Iₐ 或 A
2. **进入配体浓度影响**：改变 [L'] 不影响速率 → D/I_d；改变 [L'] 影响速率 → Iₐ/A
3. **离去配体的反式效应**：强反式效应配体加速反应 → 支持 D/I_d（反式效应主要影响键断裂）
4. **立体化学结果**：构型完全反转 → A；外消旋化 → D

**八面体配合物的一般趋势**：
- **d³, d⁶ LS**（惰性配合物）→ 倾向 **D 或 I_d**（CFSE 大→配体不容易加）
- **d⁸ 四方平面**（Pt²⁺）→ 倾向 **Iₐ 或 A**（轴向空位→第五配体容易接近）
- **大多数第一过渡系 M²⁺** → **I_d**（配体交换快）

### 2.11 18e⁻ 规则 + 催化循环：以 Wilkinson 加氢为例

#### 2.11.1 18e⁻ 计数规则速查

| 配体 | 给电子数 | 备注 |
|:---|:---:|:---|
| CO | 2 | 端基；桥连=1（但要均分） |
| PR₃ | 2 | 和 CO 类似 |
| Cl⁻（端基） | 1 | 形成 M-Cl σ 键 |
| H⁻ | 1 | 金属氢化物 |
| η²-C₂H₄ | 2 | 每个 π 键给 2e⁻ |
| η⁵-Cp | 5 | C₅H₅⁻ = 6π e⁻ 芳香体系 |
| η⁶-C₆H₆ | 6 | 苯 |
| NO（直线形） | 3 | NO⁺ 等电子体→给 3e⁻ |
| NO（弯曲形） | 1 | NO⁻ 等电子体→给 1e⁻ |

**计数公式**：总 e⁻ = 金属价电子数 + Σ 配体给电子数 − 配合物正电荷（或 + 负电荷绝对值）

#### 2.11.2 催化循环三步基本反应

| 反应 | 描述 | 电子数变化 | 氧化态变化 | 典型底物 |
|:---|:---|:---|:---|:---|
| **氧化加成** | M + A−B → M(A)(B) | +2e⁻ | M→M⁺² | H₂, RX, ArX |
| **迁移插入** | M(R)(L) → M−C(R)−L | 不变 | 不变 | CO, 烯烃 |
| **还原消除** | M(A)(B) → M + A−B | −2e⁻ | M→M⁻² | 两个 cis 配体 |

#### 2.11.3 Wilkinson 加氢完整循环（Ph₃P)₃RhCl + H₂ + 烯烃 → 烷烃）

```
Step 1: H₂ 氧化加成
  RhCl(PPh₃)₃ + H₂ → RhCl(H)₂(PPh₃)₃
  Rh¹(16e⁻) + H₂(2e⁻) → Rh³(18e⁻)
  氧化态: Rh(+1) → Rh(+3)

Step 2: PPh₃ 解离
  RhCl(H)₂(PPh₃)₃ → RhCl(H)₂(PPh₃)₂ + PPh₃
  Rh³(18e⁻) → Rh³(16e⁻)（空出一个配位点）

Step 3: 烯烃配位
  RhCl(H)₂(PPh₃)₂ + C₂H₄ → RhCl(H)₂(PPh₃)₂(η²-C₂H₄)
  Rh³(16e⁻) + 2e⁻ → Rh³(18e⁻)

Step 4: 迁移插入（氢迁移到烯烃）
  RhCl(H)₂(PPh₃)₂(η²-C₂H₄) → RhCl(H)(C₂H₅)(PPh₃)₂
  Rh³(18e⁻) → Rh³(16e⁻)（乙基本身占一个配位点，给 1e⁻）

Step 5: 还原消除
  RhCl(H)(C₂H₅)(PPh₃)₂ → RhCl(PPh₃)₂ + C₂H₆
  Rh³(16e⁻) → Rh¹(14e⁻) + 产物乙烷

Step 6: PPh₃ 重新配位
  RhCl(PPh₃)₂ + PPh₃ → RhCl(PPh₃)₃
  Rh¹(14e⁻) + 2e⁻ → Rh¹(16e⁻)（回到催化剂初始态）
```

**每步验证 18e⁻**：催化剂始终在 16e⁻（活性态）和 18e⁻（稳定态）之间切换——从不违反 18e⁻ 规则。

### 2.12 稀土配合物化学：3d vs 4f 全面对比

#### 2.12.1 核心差异

| 维度 | 3d（第一过渡系） | 4f（镧系） |
|:---|:---|:---|
| **价轨道** | 3d（外围，参与成键） | 4f（内层，被 5s²5p⁶ 屏蔽） |
| **配体场分裂** | Δo ≈ 200-400 kJ/mol | Δo < 1 kJ/mol（极小！） |
| **磁性** | 磁矩 ≈ 自旋-only（轨道角动量大部分猝灭） | 磁矩 = 自旋+轨道耦合（总角动量 J 决定） |
| **光谱** | d-d 跃迁，宽带（振动加宽），摩尔吸光系数 ε ≈ 10-100 | f-f 跃迁，线状光谱，ε ≈ 0.1-10（Laporte 禁阻） |
| **配位数** | 最常见 6（八面体），也有 4,5 | 常见 8-10（离子大+4f 不参与成键→配位数由空间决定） |
| **几何构型** | 由配体场决定 | 由配体空间排斥决定（无配体场导向） |
| **氧化态** | 变化多（−2 到 +7） | 几乎全是 +3（Ce⁴⁺ 和 Eu²⁺ 例外） |

#### 2.12.2 镧系收缩

从 La 到 Lu，4f 电子逐渐填充→核电荷增加但 4f 屏蔽不完全→有效核电荷缓慢增加→离子半径缓慢减小。

**两个重要后果**：
1. Y³⁺ 的离子半径 ≈ Ho³⁺→Y 和重镧系共生→分离困难
2. Zr⁴⁺/Hf⁴⁺ 离子半径几乎相同（镧系收缩的"后遗症"）→最难分离的元素对之一

#### 2.12.3 天线效应（Antenna Effect）

**问题**：f-f 跃迁 Laporte 禁阻→稀土离子自身吸光能力极弱（ε < 1）。

**解决方案**：在配体上设计"天线"——配体含强吸收发色团（如芳香环）→配体吸收 UV/Vis 光→能量转移给稀土离子的 f 轨道→稀土发光。

```
光 ──→ 配体（高 ε）吸收 ──→ 系间窜越到 T₁ ──→ 能量转移 ──→ Ln³⁺ f 轨道 ──→ 发光
```

**经典体系**：
- **Eu³⁺ 配合物**：红色发光（⁵D₀ → ⁷F₂, ~612 nm）—用于荧光免疫分析
- **Tb³⁺ 配合物**：绿色发光（⁵D₄ → ⁷F₅, ~545 nm）—用于时间分辨荧光

#### 2.12.4 MRI 造影剂——Gd³⁺

- Gd³⁺ = 4f⁷（半满壳层）→ 7 个未成对电子 → S = 7/2 → 巨大磁矩 → 高效缩短水质子的 T₁ 弛豫时间
- [Gd(DOTA)(H₂O)]⁻ 是典型 MRI 造影剂（DOTA = 大环多胺羧酸配体）
- **关键设计原则**：Gd³⁺ 有 9 个配位点——8 个被 DOTA 占据 + 1 个水分子配位→水分子与 bulk 水交换→T₁ 缩短

#### 2.12.5 Ce⁴⁺/Ce³⁺ 氧化还原催化

- Ce⁴⁺ 是强氧化剂（E° Ce⁴⁺/Ce³⁺ ≈ +1.72 V）——但受配体和介质影响极大
- Ce³⁺ → Ce⁴⁺ 氧化可用于有机合成（CAN = (NH₄)₂Ce(NO₃)₆ 是常用试剂）

### 2.13 综合演练：2025 决赛 Co²⁺ SMM 完整分析

> 本节以 2025 决赛第 2 题为蓝本，展示如何将本课所有知识串联解决一道完整题目。

**题目背景**：某直线形 Co²⁺ 配合物 [Co(NR₂)₂]（R = 大位阻基团），表现出单分子磁体行为。μ_eff = 5.2 BM（室温），远大于自旋-only 值 3.87 BM。

**分析步骤**：

**Step 1：确定 d 轨道分裂（直线形场）**

两个 N 配体沿 z 轴。按对称性：
- d_z²：与配体 σ 轨道直接对顶（沿 z 轴）→最强 σ 反键→能量最高
- d_{xz}, d_{yz}：与配体 p_x/p_y 形成 π 键→中间能量
- d_{xy}, dₓ₂₋ᵧ²：δ 对称性→与配体无任何 σ 或 π 重叠→非键→能量最低

**轨道级别标注**：
- Co d_z² + N 2p_z/2s → σ（成键）/ σ*（反键）——反键轨道 ≈ LUMO
- Co d_{xz} + N 2p_x → π / π*（π 相互作用）
- Co d_{yz} + N 2p_y → π / π*（π 相互作用）
- Co d_{xy} → 非键（无配体轨道匹配）
- Co dₓ₂₋ᵧ² → 非键（无配体轨道匹配）

**Step 2：电子填充（Co²⁺ = d⁷）**

从低到高填充：d_{xy}² dₓ₂₋ᵧ²² d_{xz}² d_{yz}² d_z²¹

S = 1/2（一个未成对电子在 d_z²）——但磁矩 5.2 BM >> 1.73 BM！

**Step 3：为什么 μ_eff 这么大？**

Co²⁺ 的轨道角动量贡献！第一过渡系中 Co²⁺ 的轨道角动量猝灭不完全→实际的磁矩远超自旋-only 值。

对于 Co²⁺ 直线形配合物：基态 $^4\Phi$（S = 3/2 加上轨道角动量 L = 3）→ μ_eff = g_J√[J(J+1)] BM。实际的基态中 L 不完全猝灭→磁矩显著大于自旋-only。

**Step 4：SMM 行为分析**

- 基态 S = 3/2（Co²⁺ 高自旋 d⁷）
- 直线形→强轴向各向异性→**负的零场分裂参数 D**（易轴型）
- U_eff = |D| × (9/4 − 1/4) = 2|D|
- Co²⁺ 的大轨道角动量→D 可以非常大（几十 cm⁻¹）→SMM 性能好

**Step 5：电子分布几率（42.8% / 41.2% / 16.0%）**

Co²⁺(d⁷) 在直线形场中的低能态：
- 基态：d_{xz}² d_{yz}² d_z²¹（二重简并——电子在 d_{xz}/d_{yz} 的分布有对称等价态）→ 42.8%
- 第一激发态：d_{xz}¹ d_{yz}² d_z²² → 41.2%
- 第二激发态：d_{xz}² d_{yz}¹ d_z²² → 16.0%

由 Boltzmann 分布：$p_i = g_i \exp(-E_i/kT) / Z$。三个态在室温（~300 K）下几乎等概率占据→热平均磁矩。

### 例题 2（⭐⭐⭐ 基础）：π 给体 vs π 受体——解释 Δo 差异

**题目**：[CoF₆]³⁻ 是高自旋（μ = 5.0 BM），[Co(NH₃)₆]³⁺ 是低自旋（μ = 0 BM）。用 π 键分析解释为什么 F⁻ 比 NH₃ 的场更弱。

**解答**：
- F⁻：σ 给体 + **π 给体**（填满的 2p→Co 3d t₂g）→推高 t₂g → Δo 减小→小于成对能 P→HS
- NH₃：σ 给体 only（无 π 给体也无 π 受体）→t₂g 非键→Δo 较 F⁻ 大→大于 P→LS
- 核心：F⁻ 的 π 给体效应降低了 Δo

### 例题 3（⭐⭐⭐⭐ 拓展）：磁性→构型推断

**题目**：某配合物组成为 [NiX₄]²⁻（X 为卤素），μ_eff = 3.2 BM。判断几何构型并解释。

**分析**：
- Ni²⁺ = d⁸。自旋-only μ = √[n(n+2)]
- μ = 3.2 → 接近 n=2 (μ=2.83) 但偏大，接近 n=3 (μ=3.87) 但偏小
- 轨道角动量贡献使 μ 略大于自旋-only→n=2
- d⁸ 四面体：e⁴ t₂⁴ → n=2 ✓（四面体 Ni²⁺ 永远是 HS，因为 Δt 太小）
- d⁸ 四方平面：全部配对→n=0→μ=0 ✗
- **结论**：四面体 [NiX₄]²⁻。卤素是弱场 π 给体→Δ 小→倾向四面体。

### 例题 4（⭐⭐⭐⭐ 拓展）：反式效应合成设计

**题目**：从 [PtCl₄]²⁻ 出发，设计合成路线制备以下两个异构体：
(1) cis-PtCl₂(NH₃)₂（顺铂类似物）
(2) trans-PtCl₂(NH₃)₂

**解答**：

**(1) cis-PtCl₂(NH₃)₂**（两个 NH₃ 相邻）：
- 利用反式效应：Cl⁻ 的反式效应 > NH₃ → 先加 NH₃ 后，trans 到 Cl⁻ 的 Cl 更容易被第二个 NH₃ 取代
- 路线：先加 NH₃ → [PtCl₃(NH₃)]⁻ → 再加 NH₃ → trans 到第一个 NH₃ 的 Cl 被取代？不对，trans 到 Cl⁻ 的 Cl 被取代→得到 cis！
- 因为 Cl⁻ 的反式效应 > NH₃→每个 NH₃ 取代 trans 到 Cl⁻（不是 trans 到 NH₃）的 Cl→两个 NH₃ 在 cis 位

**(2) trans-PtCl₂(NH₃)₂**（两个 NH₃ 在对位）：
- NH₃ 的反式效应弱→如果先加 NH₃，第二个 NH₃ 不会取代 trans 到 NH₃ 的 Cl
- 但实际上可以通过 Cl⁻→NH₃ 逐步取代得到 trans：第一个 NH₃ 取代后，trans 到 Cl⁻ 的 Cl 被第二个 NH₃ 取代→得 trans

## 五、本节形成的核心结论

1. **σ 成键 MO 图 = 理解一切配合物的钥匙**：金属 4s(a₁g) + 4p(t₁u) + 3d(e_g) + 配体群轨道 → a₁g/t₁u/eg 成键 + t₂g 非键 + 反键。Δo = eg* − t₂g。2025 要求的是**轨道级别标注**——写出具体哪个轨道和哪个轨道形成什么键。

2. **π 键修正完整解释光谱化学序列**：π 给体推高 t₂g（Δo↓）→弱场；π 受体拉低 t₂g（Δo↑）→强场。CO 是强场因为反馈 π 键拉低 t₂g——不是因为 C 电负性低。

3. **CFSE 决定结构偏好**：d³(Cr³⁺)和 d⁸(Ni²⁺)八面体 CFSE 最大→强烈偏好八面体→解释尖晶石正/反分布和配合物惰性。

4. **JT 畸变 = 基态简并 + 不对称振动**：d⁴ HS, d⁷ LS, d⁹ → e_g 不对称占据→必畸变。d⁹ Cu²⁺ 几乎总是拉长八面体。

5. **磁性数据 = 结构的指纹**：μ_eff 实验值 → 未成对电子数 n → 结合配体场强度 → HS/LS → 几何构型。SCO/SMM 是 2025 热点——理解自旋态交叉的熵驱动和 SMM 的各向异性势垒。

6. **反式效应 = 合成路线设计的关键**：强反式效应配体（CO, CN⁻）→加速 trans 位取代→利用序列控制取代顺序→定向合成顺/反异构体。

7. **18e⁻+催化循环 = 三步舞**：氧化加成(+2e⁻)→迁移插入(不变)→还原消除(−2e⁻)。Wilkinson 加氢 = 六步循环，每步验证 18e⁻。

8. **稀土与 3d 完全不同**：4f 不参与成键→配体场极小→光谱线状→磁性由 J 决定→配位数由空间决定→天线效应弥补吸光弱→Gd³⁺ MRI→Ce⁴⁺ 氧化催化。

## 六、⚠️ 易错点预警

| 错误想法 | 为什么错 | 正确理解 |
|:---|:---|:---|
| "CO 是强场因为 C 电负性低，给电子能力强" | CO 给 σ 电子能力其实中等，强场的根本原因是反馈 π 键拉低 t₂g | CO 强场 = σ 给体 + π 受体。反馈 π 键→金属 t₂g→CO π*→拉低 t₂g→Δo↑ |
| "所有 d⁸ 配合物都是四方平面" | d⁸ 八面体（如 [Ni(H₂O)₆]²⁺）是 HS，有 2 个未成对电子 | 强场+大 Δo→四方平面（[Ni(CN)₄]²⁻）；弱场→八面体（[Ni(H₂O)₆]²⁺） |
| "配合物稳定=不容易发生配体交换" | 热力学稳定（Kf 大）≠动力学惰性（k 小） | [Co(NH₃)₆]³⁺ Kf≈10³⁵ 且惰性（巧合）——但 [Ni(CN)₄]²⁻ Kf 也大却是动力学不稳定的 |
| "CFSE 最大的离子一定八面体" | 还要看配体场强度和成对能 | d⁴ LS Cr²⁺ CFSE = −1.6Δo+P——若 P 很大→可能 HS 四面体 |
| "JT 畸变 = 结构坏了" | JT 畸变是基态简并体系自发降低对称性来稳定自身的自然结果 | JT 畸变是电子结构的必然表现——不是"缺陷"，是正常现象 |
| "f 轨道和 d 轨道差不多" | 4f 被 5s²5p⁶ 完全屏蔽，几乎不参与成键 | 配体场对 4f 的影响 <1 kJ/mol（对 3d 是 200-400 kJ/mol）→完全不同的化学 |
| "反式效应 = 反式影响" | 反式效应=动力学（加速取代）；反式影响=热力学（削弱键） | 两者相关但不同——竞赛中通常考查反式效应（合成设计） |
