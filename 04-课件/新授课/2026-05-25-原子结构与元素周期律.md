## 一、课堂引入与本节问题

### 1.1 课堂切入

- **现象/问题/情境**：
  1. 电子绕核做圆周运动，为什么不辐射能量掉进原子核？
  2. 氢原子光谱为什么是离散的线状谱，而不是连续的光带？
- **学生已有认知**：高中已学过"核式结构+电子层排布"，知道 1s²2s²2p⁶... 的写法，但不理解"为什么这样排"。
- **本节要解决的问题**：
  1. 从经典物理的矛盾出发，为什么必须放弃"行星轨道"模型？
  2. 电子排布遵循什么规则？为什么 4s 先于 3d？Cr、Cu 为什么是特例？
  3. 怎样从电子排布推断元素在周期表中的位置，并用它解释性质趋势？

### 1.2 本节深度边界

- **本节讲到哪里为止**：
  - 能按能级顺序写常见元素（尤其 Z≤36）的基态电子排布（含 Cr、Cu 特例）
  - 能写出 Fe²⁺/Fe³⁺、Cu⁺ 等简单离子的电子排布，并判断失电子顺序
  - 能用价电子结构解释半径、电离能、电负性的基础趋势及主要反常（ⅡA/ⅢA、ⅤA/ⅥA）
  - 能用电离能突变（I₂→I₃ 剧增）推断元素族属
- **本节暂不展开**：
  - 波函数的具体数学形式（薛定谔方程不解）→ 后续在 [[原子轨道]] 深化中处理
  - 钻穿效应的定量计算（Slater 规则）→ 第二轮结构化学
  - 交换能的量子力学解释 → 第二轮或更高阶
  - 镧系收缩的定量数据、相对论效应 → 高阶无机化学

> 📷 **图片策略**：本专题类型为 `structure + calculation`，以公式、对比表格和能级图为主，轨道图像（s 球形、p 哑铃形）仅作辅助理解。

## 二、知识内容

### 2.1 从经典矛盾到量子力学模型

**从问题出发**：卢瑟福核式模型成功解释了 α 粒子散射实验，但经典电磁理论预言——做圆周运动的电子会不断辐射能量，最终坠入原子核。稳定的原子何以存在？

**关键推理/观察**（课堂精简版）：
```
经典矛盾：电子绕核圆周运动 → 加速运动 → 辐射电磁波 → 能量损失 → 坠入原子核
                ↓
氢原子光谱实验：只在特定波长出现离散谱线（不是连续光带）
                ↓
波尔模型（过渡桥梁）：能量量子化，E = hν = ΔE，能级跃迁解释特征谱线
                ↓
测不准原理：不能同时精确知道电子的位置和动量 → "轨道"概念被否定
                ↓
量子力学模型：薛定谔方程的解给出波函数 ψ，|ψ|² 代表电子在空间某处出现的概率
                ↓
"轨道"的新定义：波函数的特定解，代表概率分布，不是电子运动的轨迹
```

> [!info]- 完整推导与教材对照（课后复习用）
>
> **教材来源**：[[质心直播课-L2原子结构_笔记]]（约 24:33 ~ 01:07:43）、[[宋天佑《无机化学》§5.1~5.3]]
>
> **完整推导/补充内容**：
> ```
> 1. 氢原子光谱的里德伯公式：1/λ = R_H (1/n₁² - 1/n₂²)
>    - n₁=1（赖曼系，紫外），n₁=2（巴耳末系，可见光）
>    - 离散谱线 → 能量只能取分立值
>
> 2. 波尔能级公式：E_n = -13.6 eV × Z²/n²
>    - 成功解释了氢原子光谱，但老师强调"fundamentally incorrect"
>    - 错误根源：仍保留了经典轨道的概念
>
> 3. 测不准原理：Δx · Δp ≥ ℏ/2
>    - 若精确知道电子位置，动量完全不确定
>    - 从根本上否定了"行星轨道"的可能性
>
> 4. 薛定谔方程（黑箱化处理）：
>    - 方程形式：Ĥψ = Eψ
>    - 不需要求解，只需知道：特定解 ψ 对应一个原子轨道
>    - |ψ|² 是概率密度，电子云是 |ψ|² 的空间分布
> ```
>
> **与教材的对照说明**：
> - 教材中通常先讲量子数再讲模型发展史；本讲义采用"历史发展线+实验驱动"策略，按 Dalton → Thomson → Rutherford → Bohr → 量子力学的顺序展开
> - 已提炼资料中 [[教学逻辑提炼-质心-原子结构与元素周期律-第一轮]] 明确指出：波尔模型只作为"过渡桥梁"，必须强调其 fundamentally incorrect

**阶段性小结**：
- "轨道"不是电子绕核运动的轨迹，而是波函数的特定解，代表概率分布
- 我们不需要解薛定谔方程，只需要用**量子数**作为"轨道标签"来定位和使用它

### 2.2 量子数：给轨道贴标签

**从问题出发**：既然轨道是波函数的解，怎样给不同的轨道命名和区分？

**关键推理/观察**（课堂精简版）：

| 量子数 | 符号 | 取值范围 | 物理意义 |
|:---:|:---:|:---|:---|
| 主量子数 | n | 1, 2, 3, ... | 电子层（能层），决定主要能量 |
| 角量子数 | l | 0, 1, ..., n-1 | 轨道形状（0=s, 1=p, 2=d, 3=f） |
| 磁量子数 | m_l | -l, ..., 0, ..., +l | 轨道空间取向 |
| 自旋量子数 | m_s | +½ 或 -½ | 电子自旋方向 |

**判断规则**：
- l 必须 **< n**（如 n=2 时，l 只能是 0 或 1，不能是 2）
- |m_l| 必须 **≤ l**
- 同一轨道中两个电子的 m_s 必须 **相反**（Pauli 不相容原理）

> [!info]- 完整推导与教材对照（课后复习用）
>
> **教材来源**：[[宋天佑《无机化学》§5.4]]、[[教学逻辑提炼-质心-原子结构与元素周期律-第一轮#五、第一轮认知台阶总表]]
>
> **完整推导/补充内容**：
> ```
> 量子数的物理来源（定性理解）：
> - n 来自薛定谔方程的径向解，决定电子能量的主要部分
> - l 来自角向解，决定轨道的角动量和形状
> - m_l 来自角动量在磁场中的空间量子化
> - m_s 是电子内禀属性，与轨道运动无关
>
> 轨道数计算：
> - 给定 n 和 l，m_l 有 (2l+1) 个取值 → 即该亚层有 (2l+1) 个轨道
> - 每个轨道最多容纳 2 个电子（m_s = ±½）
> - 第 n 层最多容纳电子数：Σ(2l+1)×2 = 2n²
> ```
>
> **与教材的对照说明**：
> - 教材中可能详细展开量子数取值规则的推导；本轮只要求"会用"，不要求"会证"
> - 复杂的不可能量子数组合判断放到第二轮专项训练

**阶段性小结**：
- 量子数是"给轨道定位的坐标系统"，本轮先会用即可
- 给定 n、l，就能写出轨道符号（如 n=3, l=2 → 3d）；m_l 决定该亚层有几个轨道

### 2.3 电子排布：三种顺序

**从问题出发**：知道了有哪些轨道，电子是怎样填充进去的？为什么书写顺序和填充顺序不一样？离子失电子又是按什么顺序？

**关键推理/观察**（课堂精简版）：

**填充顺序（Aufbau 原理）**：按能级高低填充（n+l 规则，能级图）
```
能级顺序：1s < 2s < 2p < 3s < 3p < 4s < 3d < 4p < 5s < 4d < 5p < 6s < 4f < 5d < 6p ...
记忆要点：n+l 小的能级低；n+l 相同时 n 小的能级低
```

**书写顺序（化学式/离子式中）**：按主量子数 n 递增书写
> Fe: **[Ar] 3d⁶ 4s²**（n=3 写在 n=4 前面）

**失电子顺序（电离）**：先失 **n 最大**的轨道电子
> Fe²⁺: **[Ar] 3d⁶**（失去 4s²，不是 3d）
> Fe³⁺: **[Ar] 3d⁵**（再失去 1 个 3d，半满更稳定）

**记忆口诀**：
- **书写看 n，填充看能，失电子看层数**

> [!info]- 完整推导与教材对照（课后复习用）
>
> **教材来源**：[[宋天佑《无机化学》§5.5]]、[[教学逻辑提炼-质心-原子结构与元素周期律-第一轮#五、第一轮认知台阶总表]]
>
> **完整推导/补充内容**：
> ```
> 1. 能级顺序的定性解释（钻穿效应 + 屏蔽效应）：
>    - 4s 电子钻穿能力强，感受到的有效核电荷更大 → 4s 能量低于 3d
>    - 填充时先填 4s，但书写时按 n 排序写 3d 在前
>
> 2. 电子排布三原则：
>    - 能量最低原理（Aufbau）：电子优先占据能量最低的轨道
>    - Pauli 不相容原理：同一轨道最多 2 个电子，自旋相反
>    - Hund 规则：等价轨道（同一亚层）上电子优先单独占据不同轨道，且自旋平行
>
> 3. 半满/全满特例（稳定性修正）：
>    - Cr (Z=24): [Ar] 3d⁵ 4s¹（3d 半满 + 4s 半满，比 3d⁴ 4s² 更稳定）
>    - Cu (Z=29): [Ar] 3d¹⁰ 4s¹（3d 全满，比 3d⁹ 4s² 更稳定）
>    - 注意：半满全满稳定对 d 区显著，主族元素影响小，不要过度推广
> ```
>
> **与教材的对照说明**：
> - 教材中可能给出钻穿效应的定量推导（径向分布函数）；本轮只要求定性理解"4s 先于 3d 是实验事实"
> - Cr、Cu 特例在第一轮作为"稳定性规则"记住即可，深层交换能解释放到后续轮次

**对比表格**：

| 场景 | 顺序规则 | 示例（Fe, Z=26） | 常见陷阱 |
|:---|:---|:---|:---|
| 填充顺序 | 按能级高低（n+l 规则） | 先填 4s²，再填 3d⁶ | 以为按周期表顺序填 |
| 书写顺序 | 按主量子数 n 递增 | [Ar] **3d⁶ 4s²** | 把填充顺序直接写成书写顺序 |
| 失电子顺序 | 先失 n 最大的轨道 | Fe²⁺: [Ar] 3d⁶（失 4s²） | 以为"先填的后失" |

> 🔷 **拓展**（选讲，教师根据学生接受度取舍）：
> Cu⁺ 的电子排布：[Ar] 3d¹⁰（Cu 先失 4s¹，不是 3d）。这再次验证"失电子看 n 最大"。

### 2.4 周期律趋势：结构如何决定性质

**从问题出发**：同周期从左到右、同族从上到下，原子半径、电离能、电负性为什么有规律变化？为什么有些"规律"会反常？

**关键推理/观察**（课堂精简版）：

**同周期（从左 → 右）**：

| 性质 | 趋势 | 核心原因 | 反常点 |
|:---|:---|:---|:---|
| 原子半径 | ↓ 减小 | 有效核电荷 Z* 增大，电子被拉得更紧 | — |
| 第一电离能 I₁ | ↑ 增大（总体） | Z* 增大，更难失电子 | **ⅡA > ⅢA**（p¹ 易失）；**ⅤA > ⅥA**（p³ 半满稳定） |
| 电负性 χ | ↑ 增大 | 吸引电子能力增强 | F 最大（4.0） |
| 金属性 | ↓ 减弱 | 失电子能力减弱 | — |

**同族（从上 → 下）**：

| 性质 | 趋势 | 核心原因 |
|:---|:---|:---|
| 原子半径 | ↑ 增大 | 电子层数增加 |
| 电离能 | ↓ 减小 | 外层电子离核更远，受核吸引减弱 |
| 金属性 | ↑ 增强 | 更易失电子 |

> [!info]- 完整推导与教材对照（课后复习用）
>
> **教材来源**：[[宋天佑《无机化学》§6.1~6.3]]、[[教学逻辑提炼-质心-原子结构与元素周期律-第一轮#五、第一轮认知台阶总表]]
>
> **完整推导/补充内容**：
> ```
> 电离能反常的微观解释：
>
> 1. ⅡA > ⅢA（如 Mg > Al）：
>    - Mg: [Ne] 3s²，失去的是全满的 3s 电子，需要较高能量
>    - Al: [Ne] 3s² 3p¹，失去的是单个的 3p 电子，且 3p 能量略高于 3s
>    - 结果：Al 的第一电离能反而低于 Mg
>
> 2. ⅤA > ⅥA（如 P > S, N > O）：
>    - ⅤA 族：np³ 半满构型（如 N: 2p³，P: 3p³），半满稳定
>    - ⅥA 族：np⁴，失去一个电子即可达到半满（np³）
>    - 结果：ⅥA 族失去第一个电子"更容易"，第一电离能反而较低
>
> 3. 电离能突变推断族属：
>    - 若 I₂ → I₃ 剧增（如 Mg: I₁=738, I₂=1451, I₃=7733 kJ/mol）
>    - 说明失去 2 个电子后达到稳定结构 → 价电子数为 2 → ⅡA 族
> ```
>
> **与教材的对照说明**：
> - 教材中可能用 Slater 规则定量计算有效核电荷；本轮只要求定性比较
> - 电离能突变推断族属是竞赛常见题型，需重点掌握

**阶段性小结**：
- 周期律不是"需要背诵的规律"，而是电子排布的"自然结果"
- 遇到性质比较题，**第一步先写电子排布** → 判断周期/族/区 → 调用趋势或解释反常

## 三、课堂例题

> 例题分两层：**⭐⭐ 基础**（必做，覆盖核心概念）和 **⭐⭐⭐ 拓展**（选做，覆盖综合应用）。

### 例题 1（⭐⭐ 基础）：量子数判断

**题目**：判断下列各组量子数是否可能存在于原子中：
(1) n=2, l=2, m_l=0  (2) n=3, l=1, m_l=-1  (3) n=4, l=0, m_l=1

**分析提示**：
- 第一步：检查 l < n？（l 最大为 n-1）
- 第二步：检查 |m_l| ≤ l？
- 注意：同一轨道中两个电子的 m_s 必须相反

**完整解答路径**：
```
(1) n=2, l=2: l 必须 < n，2 < 2 不成立 → 不可能
(2) n=3, l=1: 1 < 3 ✓；m_l=-1: |-1| ≤ 1 ✓ → 可能（对应 3p 轨道）
(3) n=4, l=0: 0 < 4 ✓；m_l=1: |1| ≤ 0 不成立 → 不可能
```

> 💡 **课堂任务**：先独立判断 2 min，再与同桌核对。教师随后投影标准答案，强调"l < n"和"|m_l| ≤ l"两个硬约束。

---

### 例题 2（⭐⭐ 基础）：电子排布与离子推断

**题目**：写出 Fe（Z=26）的基态电子排布式，以及 Fe²⁺ 和 Fe³⁺ 的电子排布式。

**分析提示**：
1. 基态 Fe：按 Aufbau 填充（先 4s 后 3d），书写时按 n 排序（3d 在 4s 前）
2. Fe²⁺：先失哪个轨道的电子？（提示：失电子看 n 最大）
3. Fe³⁺：3d⁵ 有什么特殊稳定性？（提示：半满稳定）

**完整解答路径**：
```
Fe (Z=26):
  填充顺序：1s² 2s² 2p⁶ 3s² 3p⁶ 4s² 3d⁶
  书写顺序：[Ar] 3d⁶ 4s²

Fe²⁺:
  先失 n 最大的 4s 电子 → [Ar] 3d⁶

Fe³⁺:
  再失 1 个 3d 电子 → [Ar] 3d⁵（半满稳定）
```

> 💡 **课堂任务**：学生板演 → 全班订正 → 教师强调"三种顺序"陷阱。特别注意 Fe³⁺ 的 3d⁵ 半满稳定性。

### 例题 3（⭐⭐ 基础）：元素推断（综合）

**题目**：某元素基态原子价电子排布为 3d² 4s²。
(1) 该元素的原子序数是多少？
(2) 它位于周期表第几周期、第几族？
(3) 它有多少个未成对电子？

**分析提示**：
- [Ar] 核心有多少个电子？（Ar 的 Z=18）
- 最高 n = ? → 周期数
- 价电子总数 = ? → 族数（过渡金属：s + d 电子数）
- 3d² 按 Hund 规则如何排布？（先单独占据，自旋平行）

**完整解答路径**：
```
(1) 原子序数：
    [Ar] 核心 = 18 个电子
    价电子 = 3d² 4s² = 4 个电子
    Z = 18 + 4 = 22（钛 Ti）

(2) 周期与族：
    最高 n = 4 → 第 4 周期
    价电子总数 = 2 + 2 = 4 → 第 IVB 族（或按新 IUPAC 为第 4 族）

(3) 未成对电子：
    3d² 按 Hund 规则：↑ __ __ __ __ （两个电子分占不同轨道，自旋平行）
    4s²：↑↓ （成对）
    未成对电子数 = 2
```

> 💡 **课堂任务**：独立 5 min 完成 → 投影解析 → 反思"族数判断"陷阱（过渡金属族数 = s + d 电子数，不是只看 d）。

### 例题 4（⭐⭐⭐ 拓展）：电离能突变推断族属

**题目**：某元素的第一至第四电离能（kJ/mol）分别为：580、1815、2740、11600。推断该元素位于周期表哪一族，并写出其价电子结构。

**分析提示**：
- 观察 I₁→I₂→I₃→I₄ 的变化幅度，哪里出现"剧增"？
- 剧增点说明失去了"价电子"后达到稳定结构
- 价电子数 = 剧增前失去的电子数

**完整解答路径**：
```
电离能数据：I₁=580, I₂=1815, I₃=2740, I₄=11600

分析突变点：
  I₁→I₂：1815/580 ≈ 3.1 倍（正常递增）
  I₂→I₃：2740/1815 ≈ 1.5 倍（正常递增）
  I₃→I₄：11600/2740 ≈ 4.2 倍（剧增！）

结论：
  - I₃→I₄ 剧增 → 失去 3 个电子后达到稳定结构
  - 价电子数 = 3 → ⅢA 族
  - 价电子结构：ns² np¹
```

> 💡 **课堂任务**：独立 4 min → 同桌讨论 → 教师投影解析。强调"剧增"的判断标准（通常 >3 倍即可认为显著）。

## 四、即时演练

> 课堂上留出 3-5 分钟的快速检测，学生独立完成，教师立即扫一眼正确率。

**演练题**：
1. 写出 Cu（Z=29）的基态电子排布式（注意特例）。
2. 比较 Mg 和 Al 的第一电离能大小，并用电子排布解释。

**自检答案**：
- (1) Cu: [Ar] 3d¹⁰ 4s¹（半满/全满稳定特例，不是 3d⁹ 4s²）
- (2) Mg > Al。Mg 为 [Ne] 3s²（全满稳定），Al 为 [Ne] 3s² 3p¹（p¹ 电子能量较高且易失），故 Al 的第一电离能反而低于 Mg。

## 五、本节形成的核心结论

> 在学生经历问题、推理、例题后再出现本节总结。用 2-4 条一句话结论概括。

1. **元素性质由电子排布决定**。电子排布遵循 **Aufbau 原理 + Pauli 不相容原理 + Hund 规则**。
2. **离子失电子先失 n 最大的轨道**（如 Fe 先失 4s，不是 3d）。书写时按 n 排序、填充时按能级高低——**三种场景三种顺序**。
3. 题目要求"推断元素"或"比较性质"时，**第一步先写电子排布** → 判断周期/族/区 → 调用周期律比较。
4. 周期律的反常点（ⅡA>ⅢA、ⅤA>ⅥA）可用**电子排布的稳定性**（全满/半满）解释，不是死记硬背。

## 六、⚠️ 易错点预警

| 错误想法 | 为什么错 | 正确理解 |
|:---|:---|:---|
| "离子失电子先失最后填的电子" | 填充顺序记忆迁移 | 先失 **n 最大**的轨道电子（Fe²⁺ 失 4s²，不是 3d） |
| "3d 在 4s 后面写" | 书写习惯混淆 | 书写按 **n 排序**：3d 在 4s **前面**（n=3 < n=4）✅ |
| "Hund 规则可有可无" | 忽略细节 | 决定**未成对电子数**和**磁性**（如 3d² 有 2 个未成对电子） |
| "全满/半满总是稳定" | 过度推广 | 对 **d 区**显著（Cr、Cu 特例），主族元素影响小 |
| "过渡金属族数 = 价电子数" | 主族规则迁移 | 过渡族 = **s + d** 电子数（如 3d²4s² → 第 4 族） |
| "电离能只看趋势，不看反常" | 机械记忆 | ⅡA/ⅢA、ⅤA/ⅥA 的反常必须用电子排布解释 |
