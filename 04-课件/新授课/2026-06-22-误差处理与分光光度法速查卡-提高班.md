# 误差处理与分光光度法 · 速查卡

> 基础班你背过"系统误差可校正、随机误差统计处理"——本卡把误差处理+分光光度法从**概念分类**升级到**误差源→数据可靠性判断→A=εbc 定量计算→标准曲线法→线性范围判断**的完整分析链。

---

## 一、三类误差 · 一刀切开

```
三种误差的本质区别：

  系统误差（可校正）：
    ├─ 来源：方法缺陷、仪器不准、试剂不纯、操作习惯
    ├─ 特征：固定方向（总是偏大/偏小）、重复测定不消失
    ├─ 例子：天平两臂不等长、砝码磨损、指示剂变色点≠计量点
    └─ 处理：校准仪器、空白试验、对照试验、回收试验

  随机误差（统计处理）：
    ├─ 来源：环境温度波动、读数视差、仪器噪声
    ├─ 特征：方向随机、符合正态分布、重复测定可以减小
    ├─ 例子：滴定管读数最后一位估计值、天平最后一位波动
    └─ 处理：多次测量取平均值、用标准偏差衡量离散程度

  过失误差（剔除！）：
    ├─ 来源：操作失误——溶液溅出、读错刻度、记录错误
    ├─ 特征：明显偏离正常值——离群值
    └─ 处理：直接剔除，不能纳入统计！
```

## 二、准确度 vs 精密度 · 靶心图

```
准确度（Accuracy）= 测量值与真实值的接近程度 → 用误差衡量
精密度（Precision）= 平行测定结果的一致性 → 用偏差/标准偏差衡量

靶心图四象限：
  ┌─────────────────┬─────────────────┐
  │ 高精密度 高准确度 │ 高精密度 低准确度 │
  │   •••  (理想)    │    •••  (系统误差) │
  │   (集中且准确)    │   (集中但偏了)    │
  ├─────────────────┼─────────────────┤
  │ 低精密度 高准确度 │ 低精密度 低准确度 │
  │  • • • (罕见)   │ •  • • (最差)    │
  │  (分散但均值准)   │  (又散又偏)      │
  └─────────────────┴─────────────────┘

核心结论：
  高精密度是高准确度的必要条件，但不是充分条件！
  → 系统误差存在时：精密度可以很高，但准确度一定低
  → 随机误差大时：精密度低，但平均值可能碰巧接近真实值（不可靠！）
```

## 三、有效数字运算 · 快速回顾

```
加减法：结果的小数位数 = 各数据中小数位数最少者
  23.45 + 1.2 + 0.678 = 25.3（1.2 只有 1 位小数→结果也只保留 1 位）

乘除法：结果的有效数字位数 = 各数据中有效数字位数最少者
  0.1023 × 2.5 = 0.26（2.5 只有 2 位有效数字→结果也只保留 2 位）

注意：
  ├─ "0" 在数字前（定位零）不是有效数字：0.0032 = 2 位
  ├─ "0" 在数字后（测量零）是有效数字：3.200 = 4 位
  └─ 科学计数法消除歧义：3.2×10⁻³（2 位）vs 3.200×10⁻³（4 位）
```

## 四、Lambert-Beer 定律

```
A = εbc

  A = 吸光度（Absorbance，无单位）——仪器直接读出
  ε = 摩尔吸收系数（L·mol⁻¹·cm⁻¹）——物质固有属性
  b = 光程（cm）——吸收池厚度，通常 1 cm
  c = 浓度（mol·L⁻¹）

三个变量互算：
  已知 ε, b, c → A = εbc（正向计算）
  已知 A, ε, b → c = A/εb（求浓度）
  已知 A, b, c → ε = A/bc（求摩尔吸收系数）

适用条件（三个前提）：
  ├─ 入射光为单色光（实际上只能近似）
  ├─ 稀溶液（c < 0.01 M —— 分子间无相互作用）
  └─ 均匀体系（无散射、无荧光）

最佳测量条件：
  ├─ 选择 λ_max 作为测定波长 → 灵敏度最高 + ε 变化最小
  └─ A ≈ 0.434 时浓度相对误差最小（结论，不推导）
```

## 五、标准曲线法

```
标准曲线法步骤：

  ① 配制系列标准溶液（4-6 个浓度梯度）
  ② 在 λ_max 处测定每个标准溶液的吸光度 A
  ③ 作 A-c 标准曲线（应为过原点的直线）
  ④ 测定未知样品在相同条件下的 A
  ⑤ 由标准曲线查得或回归方程求得未知浓度

线性范围：通常 A = 0.2 ~ 0.8
  ├─ A < 0.2：透射比太大→读数误差大
  └─ A > 0.8：可能偏离线性（浓度太高）

标准曲线不过原点怎么办？
  ├─ 参比溶液选择不当（空白实验可消除）
  ├─ 显色剂有吸收（选适当参比）
  └─ 只要线性好，仍可用（用回归方程而非过原点直线）
```

## 六、Lambert-Beer 定律偏离

```
两种偏离原因：

  化学因素（高浓度时）：
    ├─ 分子间缔合/解离 → 有效浓度 ≠ 实际浓度
    ├─ 配合平衡移动 → 吸光物种浓度不是总浓度
    └─ 溶剂-溶质相互作用改变 ε

  光学因素（仪器限制）：
    ├─ 非单色光 → ε 随波长变化 → 不再是常数
    ├─ 杂散光 → 表观吸光度偏低
    └─ 散射光 → 表观吸光度偏高

一句话：Lambert-Beer 定律是"稀溶液+单色光"的理想模型
           → 超出线性范围就不能直接套公式！
```

## 七、误差与分光光度法联合判断速查

```
数据分析完整判断链：

  实验数据 → 先判误差类型
    ├─ 三份平行结果都偏高 → 系统误差（试剂/方法问题）
    ├─ 三份平行结果分散但均值对 → 随机误差大（精密度差）
    └─ 一个数据明显离群 → 过失误差（剔除）

  分光光度法数据 → 先看线性范围
    ├─ A = 1.5 → 超出线性范围！稀释重测
    ├─ A = 0.05 → 灵敏度不足！增大浓度或光程
    └─ r² = 0.95 → 线性差！检查标准溶液和测量条件
```

## 八、常见易错点

| 错误 | 纠正 |
|:---|:---|
| "多测几次可以消除系统误差" | 系统误差是**方向固定的**——测 100 次还是一样偏！只能靠校准/空白/对照 |
| "精密度高准确度就一定高" | 系统误差存在时，打靶全在 9 环（精密度高）但靶心在别处（准确度低） |
| "A 值越大测得的浓度越准" | A 太大→超出线性范围→**偏离 Lambert-Beer 定律**→不准！ |
| "标准曲线一定过原点" | 空白/背景/显色剂影响可使截距不为零——只要线性好就可以用 |
| "有效数字运算时先四舍五入再算" | **先按完整数字算完，最后一步再四舍五入！** |
| "ε 是常数任何条件下都可用" | ε 仅在 λ_max 和稀溶液条件下为常数——换波长/高浓度都会变！ |
| "过失误差也要纳入统计分析" | **过失误差必须剔除！** 那是操作失误，不是"误差" |

## 九、例题速练

### 例 1：误差类型判断

**题**：某学生用 NaOH 滴定 HCl，四次结果为 0.1021, 0.1023, 0.1020, 0.1022 M，但真实值为 0.0998 M。分析误差类型。

**速解**：精密度高（四次结果极为接近），但准确度低（偏离真值 ~2.4%）。说明存在**系统误差**——可能是 NaOH 浓度标定有偏差、HCl 体积读数系统偏大，或基准物质出了问题。

### 例 2：Lambert-Beer 定律计算

**题**：某溶液在 525 nm 处 ε=2.8×10³ L·mol⁻¹·cm⁻¹，使用 2 cm 吸收池，测得 A=0.56。求溶液浓度 c。

**速解**：c = A/(εb) = 0.56/(2.8×10³ × 2) = 1.0×10⁻⁴ mol·L⁻¹。

### 例 3：标准曲线法

**题**：标准曲线数据：c(mM): 0.020, 0.040, 0.060, 0.080；A: 0.105, 0.210, 0.308, 0.415。未知样品 A=0.35，求 c。

**速解**：数据接近线性，斜率 ≈ 5.2 A/(mM)。c ≈ 0.35/5.2 ≈ 0.067 mM。或用回归方程精确计算。
