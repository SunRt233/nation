## 二、知识内容

### 2.1 高分子物理核心：三个必会概念

#### 2.1.1 数均分子量 Mₙ 与重均分子量 Mw

高分子不是单一分子量，而是**分子量分布**。两个最常用的平均值：

| 名称 | 公式 | 物理意义 | 竞赛考点 |
|:---|:---|:---|:---|
| **数均分子量** Mₙ | $M_n = \frac{\sum N_i M_i}{\sum N_i}$ | 按**分子个数**加权平均 | **渗透压法**测量 |
| **重均分子量** Mw | $M_w = \frac{\sum N_i M_i^2}{\sum N_i M_i}$ | 按**质量**加权平均 | **光散射法**测量 |
| **多分散指数** PDI | $\text{PDI} = M_w / M_n$ | ≥ 1，越接近 1 分布越窄 | PDI = 1 为单分散（如天然蛋白质） |

> 💡 **一句话记忆**：Mₙ 是"人均收入"（按人头平均），Mw 是"户均收入"（按财富加权）。Mw ≥ Mₙ 恒成立，因为大分子在 Mw 中权重更大。

**Carothers 方程与凝胶点**：缩聚反应中，当反应程度 $p \to 1$ 时，$\bar{X}_n \to \infty$（数均聚合度爆炸）。凝胶点 = 反应混合物突然失去流动性的那一刻。

#### 2.1.2 立构规整性（Tacticity）

| 类型 | 结构特征 | 催化体系 | 性质 |
|:---|:---|:---|:---|
| **等规** (isotactic) | 所有取代基在锯齿链同侧 | **Ziegler-Natta**（TiCl₄/AlEt₃） | 高结晶度、高熔点（~165°C）、高强度 |
| **间规** (syndiotactic) | 取代基交替排列 | **金属茂催化剂**（如 Cp₂ZrCl₂/MAO） | 中等结晶度、透明度好 |
| **无规** (atactic) | 取代基随机分布 | **自由基聚合** | 无定形、软而黏 |

> 🔑 **竞赛判断口诀**：Ziegler-Natta → 等规；金属茂 → 间规；自由基 → 无规。

#### 2.1.3 热塑性 vs 热固性

| 类型 | 结构 | 加热行为 | 典型例子 | 根源 |
|:---|:---|:---|:---|:---|
| **热塑性** | 线形或支化链 | 加热软化、冷却硬化（可逆） | PE、PP、PS、PVC、PET | 链间只有范德华力/氢键，无交联 |
| **热固性** | 三维交联网络 | 加热不熔、高温分解（不可逆） | 酚醛树脂（电木）、环氧树脂、脲醛树脂 | 链间共价交联 → 整块材料就是一个分子！ |

> 💡 热固性塑料一旦成型就无法重新塑形——因为它**不是一个一个的分子链，而是一整张共价键网络**。

### 2.2 人工高分子三分类（口诀 + 速查表）

> **三分类口诀**：
> - 橡胶：官能团少、有双键
> - 纤维：官能团最多，一般缩聚
> - 塑料：最多变，一般较高饱和度

| 类别 | 结构特征 | 典型代表 | 聚合类型 | 竞赛常见考点 |
|:---|:---|:---|:---|:---|
| **橡胶** | 含 C=C 双键（可硫化交联） | 天然橡胶（顺-1,4-聚异戊二烯）、丁苯橡胶、氯丁橡胶（Neoprene） | 加聚（二烯烃 1,4-加成） | 硫化 = 硫桥交联 → 弹性；EPDM = 乙丙三元橡胶（第三单体提供交联点） |
| **纤维** | 强分子间氢键、高结晶度 | 尼龙-66、涤纶（PET）、腈纶（PAN）、Kevlar | **缩聚**为主 | 尼龙-66 = 己二胺 + 己二酸；PET = 对苯二甲酸 + 乙二醇 |
| **塑料** | 种类最多，饱和烃链为主 | PE、PP、PS、PVC、PMMA（有机玻璃）、PC（聚碳酸酯） | 加聚为主，也有缩聚 | LDPE vs HDPE（支化度不同）；PP 立构规整性决定性质 |

**常见塑料速查**：
- **PE**（聚乙烯）：$n\text{ CH}_2\text{=CH}_2 \to [\text{CH}_2\text{--CH}_2]_n$，加聚，最常见塑料
- **PP**（聚丙烯）：$n\text{ CH}_2\text{=CHCH}_3 \to$ 等规 PP（Z-N 催化）→ 高结晶度、耐 130°C
- **PS**（聚苯乙烯）：$n\text{ CH}_2\text{=CHPh} \to$ 透明硬脆，发泡 = 泡沫塑料
- **PVC**（聚氯乙烯）：$n\text{ CH}_2\text{=CHCl} \to$ 硬质（不加增塑剂）/ 软质（加增塑剂）
- **PET**（聚对苯二甲酸乙二酯）：**缩聚**，对苯二甲酸 + 乙二醇 → 涤纶纤维/饮料瓶
- **尼龙-66**：**缩聚**，己二胺 + 己二酸 → 酰胺键 $\text{--CONH--}$

### 2.3 生物高分子速查

#### 2.3.1 多肽与蛋白质

**固相多肽合成（SPPS）**：反复两步循环——
1. **DCC 缩合**：氨基被保护的氨基酸 + 羧基被保护的氨基酸 $\xrightarrow{\text{DCC}}$ 肽键
2. **TFA 脱保护**：三氟乙酸脱去 Boc 保护基
3. 重复 1 → 2，逐次延长肽链

> 竞赛考点：知道 DCC = 缩合剂（活化羧基），TFA = 脱保护剂即可。

#### 2.3.2 DNA 与 RNA

| 组分 | DNA | RNA |
|:---|:---|:---|
| 五碳糖 | **脱氧核糖**（2'-H） | **核糖**（2'-OH） |
| 碱基 | **A** / **G** / **C** / **T** | **A** / **G** / **C** / **U** |
| 结构 | 双螺旋（**A=T 2个氢键**，**G≡C 3个氢键**） | 通常单链 |
| 稳定性 | G≡C 对比 A=T 对更稳定（3 个氢键 > 2 个） | — |

> 🔑 **G-C 对为什么比 A-T 对稳定？** 因为 G≡C 之间有 **3 个氢键**，而 A=T 只有 **2 个**。DNA 双螺旋的"胶水"就是碱基对之间的氢键 + π-π 堆积。

#### 2.3.3 多糖

| 多糖 | 糖苷键类型 | 来源 | 竞赛考点 |
|:---|:---|:---|:---|
| **直链淀粉** | α-1,4 | 植物淀粉（~20%） | 螺旋空穴恰好容纳 I₂ → **遇碘变蓝**（需痕量 I⁻ 参与形成 I₃⁻ 或 I₅⁻） |
| **支链淀粉** | α-1,4 + α-1,6 分支 | 植物淀粉（~80%） | 遇碘变**紫红**（分支干扰螺旋形成） |
| **纤维素** | **β-1,4** | 植物细胞壁 | 链间强氢键 → **不溶于水**，但可溶于 **铜氨溶液**（Schweizer 试剂）→ 人造丝 |
| **几丁质** | β-1,4（N-乙酰葡萄糖胺） | 虾蟹壳、真菌细胞壁 | 单体是 **N-乙酰葡萄糖胺**（不是葡萄糖！） |

> 💡 **为什么纤维素不溶于水而直链淀粉可以？** 关键在糖苷键构型：β-1,4 使纤维素链完全伸展，链间形成密集氢键网络；α-1,4 使直链淀粉卷成螺旋，链间氢键较少。

### 2.4 聚合机理与课堂引入案例

#### 2.4.1 四大聚合机理

| 机理 | 特征 | 典型例子 |
|:---|:---|:---|
| **自由基聚合** | 引发剂 → R·→ 链增长 → 偶合/歧化终止 | PE（高压法）、PS、PMMA |
| **阳离子聚合** | 酸引发 → 碳正离子链增长 | 异丁烯 → 丁基橡胶 |
| **阴离子聚合** | 碱/有机金属引发 → 碳负离子链增长 | 丁二烯 + BuLi → **活性聚合**（无终止！PDI≈1） |
| **配位聚合** | Ziegler-Natta / 金属茂 → 立构规整 | 等规 PP、HDPE |

#### 2.4.2 🔥 课堂引入案例（让高分子"活"起来）

| 现象 | 化学原理 | 一句话解释 |
|:---|:---|:---|
| **502 胶水遇空气就干** | 氰基丙烯酸酯 + 空气中痕量水 → **阴离子聚合**（水 = 引发剂！） | 502 不需要严格无水——水就是引发剂 |
| **尿不湿为什么能吸那么多水** | 聚丙烯酸钠（交联聚电解质）→ 渗透压驱动吸水 → 凝胶 | 交联高分子 = 一张巨大的"分子网"兜住水 |
| **嫦娥四号国旗月球不坏** | 聚酰亚胺（Kapton）→ 芳杂环刚性骨架 → 耐高温+耐辐射 | 月球表面 200°C 温差，普通塑料早化了 |
| **淀粉遇碘为什么变蓝** | 直链淀粉螺旋空穴恰好容纳 I₂ → 蓝黑配合物 | 这是超分子主客体识别！螺旋 = 主体，I₂ = 客体 |
| **DNA 双螺旋的"胶水"** | A=T（2个氢键）、G≡C（3个氢键）→ 碱基配对特异性 | 为什么 G-C 对比 A-T 对更稳定？多一个氢键！ |

## 三、课堂例题

### 例题 1（⭐⭐ 基础）：分子量计算

**题目**：某聚合物样品由 100 个分子量为 10,000 的分子和 100 个分子量为 100,000 的分子组成。计算 Mₙ 和 Mw，以及 PDI。

**分析提示**：
- Mₙ：按分子个数平均 → (100×10000 + 100×100000) / 200
- Mw：按质量加权 → 分子量大的权重更大

**完整解答路径**：
- Mₙ = (100×10⁴ + 100×10⁵) / 200 = (10⁶ + 10⁷) / 200 = 1.1×10⁷ / 200 = **55,000**
- Mw = (100×(10⁴)² + 100×(10⁵)²) / (100×10⁴ + 100×10⁵) = (10⁹ + 10¹¹) / 1.1×10⁷ ≈ **91,818**
- PDI = 91818/55000 ≈ **1.67**（分布较宽）

---

### 例题 2（⭐⭐ 基础）：单体反推

**题目**：写出 PET（聚对苯二甲酸乙二酯）和尼龙-66 的单体与聚合类型。

**解答**：
- **PET**：单体 = **对苯二甲酸** + **乙二醇**；聚合类型 = **缩聚**（酯交换，脱去乙二醇/甲醇）
- **尼龙-66**：单体 = **己二胺** + **己二酸**；聚合类型 = **缩聚**（脱去水，形成酰胺键）

> 💡 从重复单元反推单体的口诀：**找到官能团连接点 → 补上脱去的小分子 → 还原为单体**。

---

### 例题 3（⭐⭐⭐ 拓展）：立构规整性与催化剂

**题目**：为什么用 Ziegler-Natta 催化剂得到的聚丙烯是等规的，而用自由基聚合得到的是无规的？

**解答**：
- Z-N 催化剂（TiCl₄/AlEt₃）是**异相催化剂**——活性中心在晶体表面，单体只能以特定取向配位和插入 → 每个单体的甲基都朝同一方向 → **等规**
- 自由基聚合中，链末端是平面型 sp² 碳自由基——单体可以从平面两侧以相同概率进攻 → 甲基随机取向 → **无规**
- 金属茂催化剂（均相）通过配体空间位阻控制单体插入方向 → **间规**

## 四、本节形成的核心结论

1. **分子量不是单一值**：高分子有分布，Mₙ（按个数）< Mw（按质量），PDI = Mw/Mₙ ≥ 1。
2. **立构规整性由催化剂决定**：Ziegler-Natta → 等规、金属茂 → 间规、自由基 → 无规。
3. **热塑性 vs 热固性的根源在交联**：线形链 → 热塑性（可逆）；三维交联网络 → 热固性（不可逆）。
4. **多糖的性质由糖苷键构型决定**：α-1,4 → 螺旋（遇碘变蓝）；β-1,4 → 伸直链（强氢键、不溶于水）。
5. **从重复单元反推单体**：找到官能团连接点 → 补上脱去的小分子 → 还原为单体。

## 五、⚠️ 易错点预警

| 错误想法 | 为什么错 | 正确理解 |
|:---|:---|:---|
| "Mₙ 和 Mw 差不多" | Mw 对大分子更敏感，分布越宽差越多 | 单分散样品 Mₙ = Mw；真实高分子 Mw > Mₙ |
| "热塑性塑料加热就分解" | 只有热固性才加热不熔。热塑性加热软化是可逆的 | 热塑性 = 线形链，加热 → 链可以滑动 → 软化（物理变化） |
| "所有淀粉遇碘都变蓝" | 只有直链淀粉的空穴能容纳 I₂ 形成蓝色配合物 | 支链淀粉的分支干扰螺旋 → 遇碘紫红；纤维素完全不变色 |
| "PET 是加聚产物" | PET 含酯键（—COO—），是典型的缩聚产物 | 从重复单元看：有酯键/酰胺键 → 缩聚；只有 C—C → 加聚 |
| "502 是自由基聚合" | 502（氰基丙烯酸酯）被水引发 → 阴离子聚合 | 水就是引发剂！这是极少数不需要严格无水的阴离子聚合 |
