# 钛钒铬锰 · 速查卡

> 基础班你背过"Ti +4、Cr +3/+6、Mn +2/+4/+7"——本卡把 Ti/V/Cr/Mn 从**价态列表**升级到**颜色信号→价态判断→反应产物**的三位一体推理系统。

---

## 一、钛 · 水解驱动

```
Ti⁴⁺ 极小 + 电荷极高 → 极化能力极强 → 极易水解

  TiCl₄ + 2H₂O → TiO₂ + 4HCl↑（白色烟雾！）
  TiCl₄ + O₂ → TiO₂ + 2Cl₂（气相氧化法 → 钛白粉）
```

| 化合物 | 结构/性质 | 一句话 |
|:---|:---|:---|
| **TiO₂** | 金红石型——6:3 配位（Ti:O） | 最重要白色颜料（钛白粉） |
| **TiCl₄** | 无色液体，遇湿空气发烟 | 水解产生 TiO₂·nH₂O 白色烟雾 |
| **TiOSO₄** | 硫酸氧钛 | Ti⁴⁺ 在溶液中实际以 TiO²⁺（钛酰离子）存在 |

**类比记忆**：TiCl₄ 水解 ⇔ SiCl₄ 水解——都是高电荷小阳离子的氯化物，遇水都剧烈水解产生"冒烟"效果。

---

## 二、钒 · 颜色序列

```
钒的多价态颜色序列（核心记忆项！）：

  V²⁺(紫) → V³⁺(绿) → VO²⁺(蓝) → VO₂⁺(黄)
   +2        +3        +4         +5

  口诀：二紫三绿四蓝五黄
  规律：价态越高 → d 电子越少 → 颜色越浅（紫→绿→蓝→黄）
```

| 化合物 | 价态 | 颜色 | 关键性质 |
|:---|:---:|:---|:---|
| **V₂O₅** | +5 | 橙红色固体 | 催化氧化 SO₂→SO₃（接触法制硫酸） |
| **VO₂⁺** | +5 | 黄色 | 酸性溶液中稳定 |
| **VO²⁺** | +4 | 蓝色 | 最稳定的钒氧离子 |
| **V³⁺** | +3 | 绿色 | 还原性——可被 KMnO₄ 氧化 |
| **V²⁺** | +2 | 紫色 | 强还原性 |

**推断题信号**：溶液颜色随氧化态变化在紫/绿/蓝/黄之间切换 → 锁定钒！

## 三、铬 · 酸碱互变

```
铬的三种核心形态：

  Cr³⁺（绿色，最稳定）
    ├─ [Cr(H₂O)₆]³⁺ 八面体，d³ 构型
    └─ 在碱中：Cr³⁺ + 3OH⁻ → Cr(OH)₃↓（灰绿胶状）
         Cr(OH)₃ + OH⁻ → CrO₂⁻（亚铬酸根，绿色）+ 2H₂O

  CrO₄²⁻（黄色，碱性条件稳定）
    ⬇ 加酸
  Cr₂O₇²⁻（橙色，酸性条件稳定）
  
  互变口诀：加酸变橙，加碱变黄
  2CrO₄²⁻ + 2H⁺ ⇌ Cr₂O₇²⁻ + H₂O
```

| 反应 | 方程式 | 意义 |
|:---|:---|:---|
| **Cr(VI) 氧化 Fe²⁺** | Cr₂O₇²⁻ + 6Fe²⁺ + 14H⁺ → 2Cr³⁺ + 6Fe³⁺ + 7H₂O | Cr(VI) 废液处理原理 |
| **Cr(VI) 氧化乙醇** | 2Cr₂O₇²⁻ + 3C₂H₅OH + 16H⁺ → 4Cr³⁺ + 3CH₃COOH + 11H₂O | 酒驾检测原理（橙→绿） |
| **CrO₅ 鉴定** | Cr₂O₇²⁻ + 4H₂O₂ + 2H⁺ → 2CrO₅（蓝色，乙醚萃取）+ 5H₂O | Cr 的特征鉴定（蓝色过氧化铬） |

**Cr(VI) 毒性**：强氧化性 + 致癌——废液必须用 FeSO₄ 还原为 Cr³⁺ 后再沉淀处理。

## 四、锰 · pH 决定产物

```
KMnO₄ 在不同 pH 下的还原产物——核心分水岭！

  酸性：MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O（淡粉色，几乎无色）
  中性：MnO₄⁻ + 2H₂O + 3e⁻ → MnO₂↓ + 4OH⁻（黑色沉淀）
  碱性：MnO₄⁻ + e⁻ → MnO₄²⁻（绿色溶液）

  一句话：酸性→Mn²⁺，中性→MnO₂↓，碱性→MnO₄²⁻
```

### 4.1 经典例题速判

```
KMnO₄ + Na₂SO₃ 在不同条件下的产物：

  酸性：2MnO₄⁻ + 5SO₃²⁻ + 6H⁺ → 2Mn²⁺ + 5SO₄²⁻ + 3H₂O
        现象：紫色褪去→几乎无色

  中性：2MnO₄⁻ + 3SO₃²⁻ + H₂O → 2MnO₂↓ + 3SO₄²⁻ + 2OH⁻
        现象：紫色褪去→黑色沉淀

  碱性：2MnO₄⁻ + SO₃²⁻ + 2OH⁻ → 2MnO₄²⁻ + SO₄²⁻ + H₂O
        现象：紫色→绿色
```

### 4.2 锰的核心化合物

| 化合物 | Mn 价态 | 颜色 | 关键性质 |
|:---|:---:|:---|:---|
| **Mn²⁺** | +2 | 淡粉色 | d⁵ 半满 → 特别稳定（与 Fe³⁺ 同电子构型） |
| **MnO₂** | +4 | 黑色 | 催化剂（H₂O₂ 分解、KClO₃ 分解） |
| **MnO₄²⁻** | +6 | 绿色 | 仅在强碱中稳定——加酸即歧化 |
| **MnO₄⁻** | +7 | 紫色 | 实验室最强氧化剂之一 |
| **Mn₂O₇** | +7 | 暗红色油状 | **爆炸性！** 浓 H₂SO₄ + KMnO₄ → Mn₂O₇（油状，遇有机物爆炸） |

**Mn²⁺ 稳定性解释**：d⁵ 半满构型 → 配位场稳定化能 + 交换能最大 → Mn²⁺ 难以被氧化或还原。

## 五、四元素对比速查

```
价态递变规律：

  Ti: +4 为主（TiO₂, TiCl₄），+3 罕见
  V:  +2/+3/+4/+5 四价态齐全——颜色序列是核心记忆项
  Cr: +3 最稳定，+6 强氧化（Cr₂O₇²⁻/CrO₄²⁻ 互变）
  Mn: +2(Mn²⁺)/+4(MnO₂)/+6(MnO₄²⁻)/+7(MnO₄⁻)——价态跨度最大！

颜色速记：
  Ti⁴⁺ 无色 | V²⁺紫 V³⁺绿 VO²⁺蓝 VO₂⁺黄
  Cr³⁺绿 CrO₄²⁻黄 Cr₂O₇²⁻橙 | Mn²⁺淡粉 MnO₂黑 MnO₄²⁻绿 MnO₄⁻紫
```

## 六、常见易错点

| 错误 | 纠正 |
|:---|:---|
| "KMnO₄ 在任何条件下都还原成 Mn²⁺" | 酸性→Mn²⁺，中性→MnO₂↓，碱性→MnO₄²⁻——产物完全不同！ |
| "Cr³⁺ 是黄色的" | Cr³⁺ 是**绿色**！CrO₄²⁻ 才是黄色，Cr₂O₇²⁻ 是橙色 |
| "TiCl₄ 水解和 SiCl₄ 水解产物一样" | TiCl₄ → TiO₂（不生成原酸），SiCl₄ → H₄SiO₄（原硅酸）→ 脱水→SiO₂ |
| "V₂O₅ 是催化剂所以不参与反应" | V₂O₅ 是催化剂但价态在 +4/+5 间循环——V₂O₅ + SO₂ → V₂O₄ + SO₃；V₂O₄ + O₂ → V₂O₅ |
| "Mn₂O₇ 稳定" | Mn₂O₇ **极不稳定**——油状液体，遇有机物即爆炸！ |
| "CrO₅ 是 Cr 的常规氧化态" | CrO₅ 是过氧化铬，O 以 O₂²⁻ 存在，Cr 仍是 +6——只在 H₂O₂ + 酸条件下瞬时存在 |

## 七、例题速练

### 例 1：KMnO₄ 还原产物判断

**题**：KMnO₄ 分别与 Na₂SO₃ 在 (a) 稀 H₂SO₄、(b) 水、(c) NaOH 溶液中反应——写出方程式并描述现象。

**速解**：
- (a) 酸性：2MnO₄⁻ + 5SO₃²⁻ + 6H⁺ → 2Mn²⁺ + 5SO₄²⁻ + 3H₂O——紫色褪去→无色（淡粉）
- (b) 中性：2MnO₄⁻ + 3SO₃²⁻ + H₂O → 2MnO₂↓ + 3SO₄²⁻ + 2OH⁻——紫色褪去→棕黑色沉淀
- (c) 碱性：2MnO₄⁻ + SO₃²⁻ + 2OH⁻ → 2MnO₄²⁻ + SO₄²⁻ + H₂O——紫色→绿色

### 例 2：钒的颜色推断

**题**：某溶液加 Zn 粒还原→紫色→放置变绿→再加 KMnO₄→蓝色→继续加→黄色。推断可能是什么元素。

**速解**：Zn 强还原→V²⁺（紫）→空气氧化→V³⁺（绿）→KMnO₄ 氧化→VO²⁺（蓝）→继续氧化→VO₂⁺（黄）。钒的颜色序列经典推断题。

### 例 3：Cr 的酸碱互变

**题**：黄色溶液 A + 酸 → 变橙；橙色溶液 B + 碱 → 变黄。A 和 B 是什么？写出方程式。

**速解**：A = CrO₄²⁻（黄），B = Cr₂O₇²⁻（橙）。2CrO₄²⁻ + 2H⁺ ⇌ Cr₂O₇²⁻ + H₂O。

## 八、自查清单

- [ ] TiCl₄ 水解方程式（白色烟雾原因）
- [ ] 钒颜色序列：V²⁺(紫)→V³⁺(绿)→VO²⁺(蓝)→VO₂⁺(黄)，规律：高价→颜色浅
- [ ] V₂O₅ 催化 SO₂ 的氧化还原循环（V⁵⁺⇌V⁴⁺）
- [ ] Cr³⁺(绿) / CrO₄²⁻(黄，碱性) / Cr₂O₇²⁻(橙，酸性) 三者互变
- [ ] Cr(VI) 废液处理：Cr₂O₇²⁻ + Fe²⁺ + H⁺ → Cr³⁺ + Fe³⁺
- [ ] CrO₅ 蓝色过氧化铬——Cr 的特征鉴定
- [ ] KMnO₄ 酸性→Mn²⁺，中性→MnO₂↓，碱性→MnO₄²⁻
- [ ] Mn²⁺ 的 d⁵ 半满稳定性解释
- [ ] MnO₄²⁻（绿）在酸中歧化：3MnO₄²⁻ + 4H⁺ → 2MnO₄⁻ + MnO₂↓ + 2H₂O
- [ ] Mn₂O₇ 爆炸性——KMnO₄ + 浓 H₂SO₄ 的危险产物
