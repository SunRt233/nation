## 一、课堂引入与本节问题

### 1.1 课堂切入

**问题**：第一轮学了晶体场理论——配体被当作**点电荷**，把 d 轨道推成 t₂g 和 e_g 两组。但你有没有想过一个问题：

> 如果配体只是点电荷，为什么 **CO 是强场配体**而 **Cl⁻ 是弱场配体**？两者都带负电（或中性），为什么场强差距这么大？

**现象冲击**：
- [Cr(H₂O)₆]³⁺ 吸收黄光 → 透射紫色（淡紫），Δo ≈ 17400 cm⁻¹
- [Cr(CN)₆]³⁻ 吸收蓝光 → 透射黄色，Δo ≈ 26600 cm⁻¹
- 同样是 Cr³⁺，换一个配体，Δo 变化了 **50%**！晶体场理论的"点电荷"无法解释这个差距。

**学生已有认知**：第一轮已学——Oh 场 d 轨道分裂为 t₂g/e_g、光谱化学序列（死记版）、高自旋/低自旋的定性判断、配合物命名与异构。

**本节要解决的问题**：
1. 为什么不同配体的场强不同？如何从**轨道相互作用**推导光谱化学序列？
2. CFSE（晶体场稳定化能）如何**定量计算**？它如何解释配合物的磁性？
3. 哪些 dⁿ 构型会发生 Jahn-Teller 畸变？为什么 d⁹ Cu²⁺ 总是拉长八面体？
4. d-d 跃迁和 CT 跃迁有何区别？为什么 KMnO₄ 深紫而 [Mn(H₂O)₆]²⁺ 淡粉？
5. 18 电子规则如何系统计数？如何处理 NO、Cp 等特殊配体？

### 1.2 学习目标

- **本节讲到哪里为止**：
  - 能从 σ/π 性质解释光谱化学序列，不再死记
  - 能定量计算任意 dⁿ 构型在 Oh/Td 场中的 CFSE（含高低自旋）
  - 能判断 Jahn-Teller 畸变构型及畸变方向
  - 能区分 d-d 跃迁和 CT 跃迁，用选律解释颜色深浅
  - 能用 Method A/B 计数 18e（含 NO、Cp、烯烃等特殊配体）
  - 能用 IR 峰数判断配合物异构体
- **后续深入学习**：AOM 数学推导（系数 1/4、3/4）、Fischer/Schrock 卡宾、等瓣相似、金属多重键、反式效应定量动力学（后续在第四轮/决赛处理）

### 2.1 从晶体场到配体场——配体的三分类

> 核心认知升级：晶体场理论说"配体是点电荷"→ 配体场理论说"**配体有轨道，与金属 d 轨道发生相互作用**"。

**从问题出发**：为什么 CO 是强场配体而 Cl⁻ 是弱场配体？

**晶体场理论的局限**：

```
晶体场理论：配体 = 点电荷 → 静电排斥 → d 轨道分裂
            ↓ 问题：无法解释不同配体的 Δo 差异
            ↓ CO（中性分子）的场强 >> Cl⁻（阴离子）的场强
            ↓ 静电模型预测：负电荷越大 → 场强越大 → 与实际相反！
```

**配体场理论的核心**：配体不是点电荷，**配体轨道与金属 d 轨道发生共价相互作用**。根据 π 对称性，配体分为三类：

| 类型 | 代表配体 | π 作用方向 | 对 t₂g 的影响 | 对 Δo 的影响 | 场强 |
|:---|:---|:---|:---|:---|:---:|
| **σ-only donor** | NH₃, en（乙二胺）, H₂O | 无 π 轨道可参与 | t₂g 不变（非键） | Δo 仅由 σ 决定 | 中 |
| **π-acceptor**（π 受体）| CO, CN⁻, NO⁺, PR₃ | 金属 d → 配体 π*（反馈 π） | t₂g **下降**（稳定化） | Δo **增大** | **强场** |
| **π-donor**（π 给体）| Cl⁻, Br⁻, I⁻, F⁻, OH⁻, O²⁻ | 配体 filled p → 金属 d | t₂g **上升**（去稳定化） | Δo **减小** | **弱场** |

**关键理解**：

```
σ-donor（如 NH₃）:
  配体孤对电子 → 金属空轨道（σ 给体）
  → e_g↑（σ* 反键）  t₂g 不变
  → Δo = e_g能量 - t₂g能量（中等）

π-acceptor（如 CO）:
  配体孤对电子 → 金属空轨道（σ 给体）→ e_g↑
  + 金属 d(t₂g) → 配体 π*（反馈 π 键）→ t₂g↓
  → Δo = e_g↑ - t₂g↓ → **Δo 显著增大**

π-donor（如 Cl⁻）:
  配体孤对电子 → 金属空轨道（σ 给体）→ e_g↑
  + 配体 filled p → 金属 d(t₂g)（π 给体）→ t₂g↑
  → Δo = e_g↑ - t₂g↑ → **Δo 减小（两种作用接近抵消或反向）**
```

> [!info]- 完整推导：AOM 视角下的 Δo 表达式（课后复习用）
>
> **教材来源**：Atkins Ch7 §7.6-7.7、[[教学逻辑提炼-Zchem-过渡元素的结构化学-第三轮深化]] §四 台阶 1-4
>
> **角重叠模型（AOM）的核心思想**：
> σ 相互作用和 π 相互作用对 d 轨道能量的贡献是**加和**的。在八面体场中：
> - σ 相互作用：使 e_g 上升 3eσ（每配体贡献 1eσ，6 个配体共 6eσ，其中 e_g 分到 3eσ）
> - π 相互作用：使 t₂g 能量变化 ±4eπ（每配体贡献 1eπ，6 个配体共 6eπ，t₂g 分到 4eπ）
>
> ```
> Δo = 3eσ + 4eπ（π-acceptor，eπ > 0，因为反馈 π 使 t₂g 稳定）
> Δo = 3eσ − 4eπ（π-donor，eπ < 0，因为 π 给体使 t₂g 去稳定）
> Δo = 3eσ          （σ-only donor，eπ = 0）
> ```
>
> **与教材的对照说明**：
> - Atkins Ch7 §7.7 从 MO 理论导出相同的能级图，但用群论语言；本讲义简化为 σ/π 贡献加和
> - 结构化学基础 §9.3 给出了 ML₆ 的定性 MO 图（σ-only、σ+π），与本讲义的配体三分类一致
> - 本课不要求学生掌握 eσ/eπ 系数的数学推导（1/4、3/4 因子放决赛）

**阶段性小结**：
> 配体的场强 = σ 给电子（抬 e_g）+ π 作用（移 t₂g）。π-acceptor 降 t₂g → Δo 最大（强场）；π-donor 抬 t₂g → Δo 最小（弱场）；σ-only 在中间。

### 2.2 CO 配位的轨道图像——反馈 π 键的精髓

> 这是整节课理论升级的核心：从"CO 是强场配体"到"CO 为什么是强场配体"。

**从问题出发**：CO 的偶极矩指向 C←O（O 更电负），但配位时却是 **C 端朝向金属**。为什么不指向 O？为什么 C 配位后 Δo 那么大？

**CO 的轨道结构**：

```
CO 分子轨道排布（价层）：
  σ(2s)² σ*(2s)² π(2p)⁴ σ(2p)² π*(2p)⁰

  HOMO: σ(2p)——电子云偏向 C 端！！
  LUMO: π*(2p)——两个简并的空轨道，可接受电子

关键：HOMO 的电子云在 C 端 > O 端 → C 端是更好的 σ 给体
     LUMO 是 π*，可接受金属 d 电子的反馈 → 形成反馈 π 键
```

**CO 与金属的双重成键**：

```
       σ 给体：        反馈 π 键：
    CO(σ) → M(dσ)    M(dπ) → CO(π*)

        C≡O              C≡O
        ↓σ                ↑π反馈
        M                 M

  σ 键：CO 的 HOMO 电子 → 金属空轨道（与所有配体一样）
  π 键：金属 filled dπ → CO 空 π*（这是 CO 独有的！）
```

**反馈 π 键的三个后果**：

| 后果 | 机理 | 实验证据 |
|:---|:---|:---|
| **Δo 大幅增大** | π 反馈使 t₂g 下降 → e_g 与 t₂g 间距拉大 | [Cr(CO)₆] 是无色的（d-d 跃迁在紫外区，Δo 极大） |
| **C≡O 键削弱** | 电子进入 π* 反键轨道 → C-O 键级下降 | IR：自由 CO ν=2143 cm⁻¹ → [Cr(CO)₆] ν≈2000 cm⁻¹（红移 ~140 cm⁻¹） |
| **M-C 键增强** | π 反馈提供额外成键 → M-C 具有部分双键性质 | M-C 键长比预期短，键能比预期大 |

> [!info]- 完整推导：反馈 π 键的 MO 图（课后复习用）
>
> **教材来源**：Atkins Ch7 §7.8、Atkins Ch20 §20.5、[[教学逻辑提炼-Zchem-过渡元素的结构化学-第三轮深化]] §四 台阶 3
>
> **CO 与金属的完整 MO 相互作用**：
>
> ```
> 金属侧              配体侧（CO）
> ───────              ──────────
> 
> e_g*  ←──σ*──  CO-σ(2p) HOMO  ← σ 给体作用
> 
> t₂g   ──π反馈→  CO-π* LUMO    ← π 反馈作用
>        （金属 filled d → CO 空 π*）
> ```
>
> **M-C-O 键的共振描述**：
> ```
> M⁻—C≡O⁺  ←→  M=C=O  ←→  M≡C—O⁻
> (σ-only)    (反馈π弱)   (反馈π强)
> ```
> 反馈程度增大时，C≡O 键级下降，ν(CO) 红移更大。
>
> **IR 光谱作为反馈程度的探针**：
> - 自由 CO：2143 cm⁻¹
> - Ni(CO)₄（Ni⁰，富电子）：2057 cm⁻¹（反馈强 → C≡O 削弱多）
> - [Mn(CO)₆]⁺（Mn⁺，缺电子）：2090 cm⁻¹（反馈弱 → C≡O 削弱少）
> - [V(CO)₆]⁻（V⁻，极富电子）：1858 cm⁻¹（反馈极强）
>
> **与教材的对照说明**：
> - Atkins Ch7 §7.8 给出了 σ-donor + π-acceptor 的完整 MO 能级图（Fig. 7.22）
> - Atkins Ch20 §20.5 讨论了不同金属氧化态对 CO 伸缩频率的影响
> - 结构化学基础 §9.4 以 M-CO 为例讲解反馈 π 键，内容简洁但核心到位

**阶段性小结**：
> CO 的秘诀 = σ 给体（所有配体都有）+ **π 反馈**（CO 独有优势）。π 反馈使 t₂g↓ → Δo↑ → CO 是最强场配体之一。这也解释了为什么 CO 的 IR 伸缩频率随金属富电子程度红移。

> 🔷 **拓展**（选讲）：CN⁻ 与 CO 是等电子体，也是强 π-acceptor。但 CN⁻ 的 σ 给电子能力强于 CO（带负电），π 受电子能力弱于 CO（负电不利于接受电子）。所以 CN⁻ 的场强 ≈ CO。

### 2.3 光谱化学序列的轨道推导——从死记到理解

**从问题出发**：第一轮你背过"I⁻ < Br⁻ < Cl⁻ < F⁻ < OH⁻ < H₂O < NH₃ < en < CN⁻ < CO"。现在你能**自己推导**这个顺序吗？

**推导逻辑链**：

```
配体的 σ/π 性质 → 对 t₂g/e_g 的影响方向 → Δo 的相对大小 → 场强排序
```

**逐级推导**：

| 配体 | σ 给体 | π 性质 | 对 t₂g | 对 Δo | 场强位置 |
|:---|:---:|:---|:---|:---|:---:|
| **CO, CN⁻** | ✅ | **强 π-acceptor** | t₂g↓↓ | Δo **最大** | 最右（强场） |
| **PR₃** | ✅ | 中 π-acceptor | t₂g↓ | Δo 大 | 强场 |
| **en（乙二胺）** | ✅ | **σ-only**（无 π） | t₂g 不变 | Δo 中等偏上 | 中-强 |
| **NH₃** | ✅ | σ-only | t₂g 不变 | Δo 中等 | 中 |
| **H₂O** | 弱 | 弱 π-donor（很弱） | t₂g 微↑ | Δo 中等偏下 | 中-弱 |
| **F⁻** | 弱 | 弱 π-donor | t₂g↑ | Δo 较小 | 弱场 |
| **Cl⁻** | 中 | **中 π-donor** | t₂g↑↑ | Δo 小 | 更弱 |
| **Br⁻** | 中 | 中强 π-donor | t₂g↑↑ | Δo 更小 | 更弱 |
| **I⁻** | 中 | **强 π-donor** | t₂g↑↑↑ | Δo **最小** | 最左（弱场） |

**完整的推导序列**：

```
CO ≈ CN⁻ > PR₃ > en > NH₃ > H₂O > F⁻ > OH⁻ > Cl⁻ > Br⁻ > I⁻
 ───────     ──   ──────────────   ─────────────────────────────
 π-受体      π-受体  σ-only域       π-给体域（逐级增强）
(最强场)    (强场)   (中场)          (弱场 → 最弱场)

口诀："受体强场在右，给体弱场在左，σ-only 站中间"
```

**自我检验**：不看笔记，给以下配体排序并说明理由：
- PPh₃（σ-only + 弱 π-acceptor？）
- py（吡啶，σ-only，类似 NH₃ 但稍弱）
- NCS⁻（两可配体，N 端 ≈ NH₃ 场强，S 端 ≈ Cl⁻ 场强）

> [!info]- 完整推导与教材对照（课后复习用）
>
> **教材来源**：Atkins Ch7 §7.8（光谱化学序列）、[[专题-配位化学#spectrochemical-series]]
>
> **序列的精细调节规律**：
> 1. **同族配体**：第 2 周期 < 第 3 周期 < 第 4 周期 ≈ 第 5 周期（π-donor 能力递增，因为 p 轨道更大）→ F⁻ > Cl⁻ > Br⁻ > I⁻
> 2. **同周期配体**：负电荷越大 → σ 给电子越强 → 场强微增（如 OH⁻ > F⁻？否！F⁻ 更小更靠近金属，实际 Δo 类似）
> 3. **螯合效应**：en（双齿）> 2NH₃（两个单齿）——因为螯合环限制了配体运动，使配位更紧密
>
> **两可配体的场强差异**：
> - NCS⁻（N 配位）≈ NH₃ 场强——N 端是 σ-only donor
> - NCS⁻（S 配位）≈ Cl⁻ 场强——S 端有 filled p（π-donor）
> - NO₂⁻（N 配位，硝基）> NO₂⁻（O 配位，亚硝酸根）
>
> **与教材的对照说明**：
> - Atkins Ch7 Table 7.3 给出了实验测得的光谱化学序列（f 因子）
> - 本讲义从 σ/π 性质推导序列，教材侧重实验数据；两者互补
> - 结构化学基础 §9.2 仅列出序列（I⁻ < Br⁻ < Cl⁻ < F⁻ < OH⁻ < H₂O < NH₃ < NO₂⁻ < CN⁻ < CO），未解释原因——本课填补了这个缺口

**阶段性小结**：
> 光谱化学序列的本质 = **配体 π 性质对 t₂g 能量的调节**。π-acceptor → t₂g↓ → Δo↑（强场右端）；π-donor → t₂g↑ → Δo↓（弱场左端）；σ-only → t₂g 不动（中场）。

### 2.4 CFSE 定量计算——从定性到定量

> 第一轮你知道"强场配体 → 低自旋 → 配对能 vs Δo"。本轮你学会**算出 CFSE 的具体数值**，用它解释磁性、稳定性和反应活性。

**从问题出发**：[Fe(H₂O)₆]³⁺ 的磁矩是 5.9 BM，[Fe(CN)₆]³⁻ 的磁矩只有 ~2.3 BM。为什么同样的 Fe³⁺（d⁵），磁矩差别这么大？CFSE 是多少？

**CFSE 公式回顾**：

```
八面体场（Oh）：
  CFSE = (−4Dq) × n(t₂g) + (6Dq) × n(e_g) + P × (配对额外代价)

  其中：一个 t₂g 电子稳定 −4Dq
        一个 e_g 电子去稳定 +6Dq
        Dq = Δo/10（以 Δo = 10 Dq 为基准时，1 Dq = Δo/10）

四面体场（Td）：
  CFSE = (4Dq) × n(e) + (−6Dq) × n(t₂) + P × (配对额外代价)

  注意：Td 场中 e 组能量低（−6Dq），t₂ 组能量高（+4Dq）——与 Oh 场符号相反！
  且 Δt ≈ 4/9 Δo（四面体场分裂能约为八面体的 44%）
```

> ⚠️ **高频陷阱**：Dq 单位——不同教材定义不同。Atkins 用 Δo = 10Dq，此时 t₂g = −4Dq，e_g = +6Dq。结构化学基础用 Dq = Δo/10，一致。**做题时先确认题目用的 Dq 定义**。

**完整的 d¹–d¹⁰ CFSE 表（Oh 场，单位 Dq）**：

| dⁿ | 构型 | 高自旋 | CFSE (HS) | 低自旋 | CFSE (LS) |
|:---:|:---|:---|:---:|:---|:---:|
| d¹ | t₂g¹ | t₂g¹ | −4 | — | −4 |
| d² | t₂g² | t₂g² | −8 | — | −8 |
| d³ | t₂g³ | t₂g³ | −12 | — | −12 |
| d⁴ | t₂g³ e_g¹ | t₂g³ e_g¹ | **−6** | t₂g⁴ | −16+P |
| d⁵ | t₂g³ e_g² | t₂g³ e_g² | **0** | t₂g⁵ | −20+2P |
| d⁶ | t₂g⁴ e_g² | t₂g⁴ e_g² | **−4** | t₂g⁶ | −24+2P |
| d⁷ | t₂g⁵ e_g² | t₂g⁵ e_g² | **−8** | t₂g⁶ e_g¹ | −18+P |
| d⁸ | t₂g⁶ e_g² | t₂g⁶ e_g² | −12 | — | −12 |
| d⁹ | t₂g⁶ e_g³ | t₂g⁶ e_g³ | −6 | — | −6 |
| d¹⁰ | t₂g⁶ e_g⁴ | t₂g⁶ e_g⁴ | 0 | — | 0 |

> 💡 **观察规律**：
> - CFSE 最大（最稳定）的构型：d³ 和 d⁸（−12 Dq）、d⁶ 低自旋（−24+2P）
> - CFSE = 0 的构型：d⁵ 高自旋、d¹⁰（对称填满）
> - 高自旋⇌低自旋的竞争取决于 Δo vs P（配对能）——**当 CFSE(LS) − CFSE(HS) > P × (额外配对数)** 时，低自旋有利

**Td 场 CFSE**（通常不需记整张表，会算即可）：

| dⁿ | 构型 | CFSE（Dq, 以 Δt = 4/9 Δo） |
|:---:|:---|:---|
| d¹ | e¹ | −6 × (4/9) = −2.67 Dq（按 Δo 的 Dq） |
| d² | e² | −12 × (4/9) = −5.33 Dq |
| d⁶ | e⁴ t₂² | −24+8 = −16 × (4/9) = −7.11 Dq |
| d⁷ | e⁴ t₂³ | −24+12 = −12 × (4/9) = −5.33 Dq |

> ⚠️ **关键**：Td 场的 CFSE 始终小于 Oh 场（因为 Δt < Δo），所以 dⁿ 离子通常优先占据 Oh 位（如尖晶石中的 CFSE 驱动占位——回顾 [[2026-06-06-晶体结构深化-提高班]] 的尖晶石正/反式判断）。

**CFSE → 磁性**：

```
磁矩（自旋-only 公式）： μ = √[n(n+2)]  BM
  其中 n = 未成对电子数

实例对比：
  [Fe(H₂O)₆]³⁺：Fe³⁺ d⁵ 高自旋 → n=5 → μ = √35 ≈ 5.92 BM  ← 接近实测
  [Fe(CN)₆]³⁻：Fe³⁺ d⁵ 低自旋 → n=1 → μ = √3 ≈ 1.73 BM   ← 实测 ~2.3 BM
                                                    ↑ 轨道角动量贡献
```

> 🔷 **拓展**：实测磁矩偏离自旋-only 公式的情况——轨道角动量贡献。d⁴-d⁷ 八面体配合物中，t₂g 轨道的轨道角动量未完全淬灭，导致 μ_eff > μ_spin-only。如 [Co(H₂O)₆]²⁺（d⁷ 高自旋，n=3）自旋-only = 3.87 BM，实测 ~5.0 BM（轨道贡献 ~1.1 BM）。

> [!info]- 完整推导：轨道角动量对磁矩的贡献（课后复习用）
>
> **教材来源**：Atkins Ch7 §7.14、结构化学基础 §9.6
>
> **为什么 d⁴-d⁷ 有轨道贡献**：
> - t₂g 轨道（dxy, dxz, dyz）之间可以通过绕轴旋转互相转化 → 轨道角动量不完全淬灭
> - e_g 轨道（dz², dx²-y²）形状差异大，不能通过旋转互变 → 轨道角动量完全淬灭
> - 因此，t₂g 电子数不均衡的构型（d⁴ hs t₂g³、d⁵ ls t₂g⁵、d⁶ hs t₂g⁴、d⁷ ls t₂g⁶）有轨道贡献
>
> **Kotani 公式**（考虑自旋-轨道耦合的精确磁矩）：
> 对于 T 基态（t₂g 不对称占据），μ_eff 随温度变化，需查 Kotani 图。
>
> **与教材的对照说明**：
> - Atkins Ch7 §7.14 给出了完整磁矩表（Table 7.5），包含自旋-only 与实测对比
> - 结构化学基础 §9.6 以 [Fe(H₂O)₆]²⁺ vs [Fe(CN)₆]⁴⁻ 为例说明高/低自旋的磁性差异

**阶段性小结**：
> CFSE 是配合物稳定性的关键驱动力。d³ 和 d⁸ 总是 −12 Dq（无需担心自旋态）；d⁴-d⁷ 的 CFSE 取决于高/低自旋选择——这是 Δo 与 P 的竞争。磁性测量是判断高/低自旋最直接的实验手段。

### 2.5 Jahn-Teller 效应——"为什么不均匀填 e_g 就会畸变？"

> 第一轮你听说过 Jahn-Teller 效应。本轮你用**电子排布**精确判断：哪些构型畸变？向哪个方向畸变？畸变后能级怎么变？

**Jahn-Teller 定理（简化版）**：

> 在**简并电子基态**的非线性分子中，分子会发生几何畸变以消除简并、降低能量。

**对八面体配合物的含义**：
- 如果 e_g 轨道**不均匀占据**（1 个或 3 个电子）→ 系统不稳定 → 发生畸变 → e_g 分裂 → 能量降低
- e_g 均匀占据（0/2/4 个电子）→ 不需要畸变

**会发生 JT 畸变的 dⁿ 构型**：

| dⁿ | 构型（Oh） | e_g 占据 | 是否 JT？ | 畸变方向 | 典型实例 |
|:---:|:---|:---:|:---:|:---|:---|
| d⁴ 高自旋 | t₂g³ e_g¹ | 1 | ✅ **是** | 拉长（z 轴键变长） | [Cr(H₂O)₆]²⁺ |
| d⁷ 低自旋 | t₂g⁶ e_g¹ | 1 | ✅ **是** | 拉长 | [Co(NO₂)₆]⁴⁻ |
| d⁹ | t₂g⁶ e_g³ | 3（等效于 1 空穴） | ✅ **是** | **拉长**（绝大多数） | [Cu(H₂O)₆]²⁺ |
| d³ 高自旋 | t₂g³ | 0 | ❌ 否 | — | [Cr(H₂O)₆]³⁺ |
| d⁵ 高自旋 | t₂g³ e_g² | 2 | ❌ 否 | — | [Fe(H₂O)₆]³⁺ |
| d⁶ 低自旋 | t₂g⁶ | 0 | ❌ 否 | — | [Co(NH₃)₆]³⁺ |
| d⁸ | t₂g⁶ e_g² | 2 | ❌ 否 | — | [Ni(H₂O)₆]²⁺ |

**JT 畸变的能级图（以 d⁹ Cu²⁺ 拉长八面体为例）**：

```
Oh（正八面体）          D4h（拉长八面体，z 轴键长 > x,y 轴）

  e_g  ───        →    ┌─ b₁g (dx²-y²)——x,y 方向配体更近 → 能量更高
        ───            └─ a₁g (dz²)———z 方向配体更远 → 能量更低

  t₂g  ──────     →    ┌─ b₂g (dxy)
        ──────         ├─ e_g (dxz, dyz)——z 轴拉长使含 z 的轨道下降
        ──────         └─ (分裂成二组)

能量增益：E_JT ≈ δ₁（e_g 分裂能的一半左右每电子）
```

> 💡 **记忆口诀**："四高一七低 还有九——e_g 上 1 或 3 个电子"（d⁴ hs、d⁷ ls、d⁹）

**为什么 Cu²⁺ 几乎总是拉长八面体而不是压缩？**

- d⁹ = t₂g⁶ e_g³ → e_g 上 3 个电子（即 1 个空穴）
- 空穴在 dz² 和 dx²-y² 之间分配
- **拉长**（z 轴键变长）：dz² 能量下降 → 空穴进入 dx²-y² → dx²-y² 能量上升但 dz² 下降更多 → **净能量收益更大**
- **压缩**（z 轴键变短）：dz² 能量上升 → 空穴进入 dz² → dx²-y² 下降 → 净能量收益较小
- 拉长八面体在**能量上更有利**——所以 Cu²⁺ 配合物几乎总是拉长八面体或极端情况（平面四边形）

**二阶 Jahn-Teller 效应（d⁰ 体系）**：

> 与"一阶 JT"不同——这里基态本身不简并，但基态与低激发态能量接近时发生耦合，导致畸变。

| 类型 | 适用 | 畸变来源 | 实例 |
|:---|:---|:---|:---|
| **一阶 JT** | e_g 不均匀占据（d⁴ hs / d⁷ ls / d⁹） | 简并基态 → 畸变消除简并 | Cu²⁺ 拉长八面体 |
| **二阶 JT** | d⁰ 体系（Ti⁴⁺, V⁵⁺, Nb⁵⁺） | 空 t₂g 基态与低激发态混合 → 偏离中心 | BaTiO₃ 铁电体（Ti⁴⁺ d⁰ 偏心） |

> 🔷 **拓展**：二阶 JT 效应是 **BaTiO₃ 铁电性**的根源。Ti⁴⁺ 是 d⁰，在 Oh 场中基态是"空"的（无 d 电子），但 Ti⁴⁺ 稍微偏离八面体中心 → 空的 t₂g 轨道与 O²⁻ 的 filled 2p 混合 → 能量降低 → Ti⁴⁺"冻"在偏心位置 → 产生永久电偶极矩 → 铁电性。这也是 [[2026-06-06-晶体结构深化-提高班]] 中钙钛矿容忍因子的"下一层"物理原因。

> [!info]- 完整推导：JT 畸变的群论分析（课后复习用）
>
> **教材来源**：Atkins Ch7 §7.10、结构化学基础 §9.5
>
> **JT 活性模式的对称性判断**：
> - 对于 Oh 配合物，JT 活性模式属于 e_g 振动对称性（两个分量：Q_θ 和 Q_ε）
> - Q_θ 模式：z 轴拉长 + x,y 轴压缩（或反之）
> - Q_ε 模式：x 轴和 y 轴不对称变化
> - 大多数 Cu²⁺ 配合物走 Q_θ 模式（拉长 z 轴）
>
> **JT 能的计算**：
> E_JT = −½ × δ₁（每个 e_g 电子获得的稳定化能）
> 对于 d⁹ Cu²⁺：e_g³ → 畸变后 a₁g² b₁g¹ → 能量降低 δ₁/2
> 对于 d⁴ hs Cr²⁺：e_g¹ → 畸变后 a₁g¹ → 能量降低 δ₁/2
>
> **与教材的对照说明**：
> - Atkins Ch7 §7.10 以 Cu²⁺ 为例给出 JT 畸变的完整讨论，包括能级图和选律
> - 结构化学基础 §9.5 讨论了 JT 效应的条件和 Cu²⁺ 的畸变方向
> - 二阶 JT 效应在本科教材中通常仅作简要提及，详见 West《Solid State Chemistry》Ch5

**阶段性小结**：
> JT 畸变的核心判据：e_g 轨道是否**不均匀占据**。d⁴ 高自旋（e_g¹）、d⁷ 低自旋（e_g¹）、d⁹（e_g³ = 1 空穴）→ JT 活性。Cu²⁺ 总是拉长八面体是因为拉长在能量上比压缩更有利。d⁰ 体系（Ti⁴⁺ 等）走二阶 JT → 偏离中心 → 铁电性。

### 2.6 电子光谱——配合物颜色的科学

> 第一轮你知道"配合物颜色来自 d-d 跃迁"。本轮你学会：d-d 跃迁弱在哪里？CT 跃迁强在哪里？选律如何限制？

**d-d 跃迁基础**：

```
d-d 跃迁 = t₂g 电子吸收光子 → 跳到 e_g
  吸收能量 = Δo = hc/λ
  吸收波长 λ = hc/Δo
  透射光的颜色 = 吸收光的互补色

实例： [Cu(H₂O)₆]²⁺ 吸收 ~800 nm（红光/近红外）→ 透射蓝绿色（浅蓝）
       [Cu(NH₃)₄]²⁺ 吸收 ~600 nm（橙黄）→ 透射蓝紫色（深蓝）
       ← NH₃ 场强 > H₂O → Δo 更大 → 吸收波长更短 → 透射更深
```

**互补色速查**：

| 吸收波长 (nm) | 吸收颜色 | 透射（观察到的）颜色 |
|:---:|:---|:---|
| 400-435 | 紫 | 黄绿 |
| 435-480 | 蓝 | 黄 |
| 480-490 | 绿蓝 | 橙 |
| 490-500 | 蓝绿 | 红 |
| 500-560 | 绿 | 紫红 |
| 560-580 | 黄绿 | 紫 |
| 580-595 | 黄 | 蓝 |
| 595-650 | 橙 | 绿蓝 |
| 650-780 | 红 | 蓝绿 |

**选律双重限制——为什么 d-d 跃迁那么弱？**

d-d 跃迁受到**两道"禁令"**的限制，导致摩尔消光系数 ε 仅 1–100 L·mol⁻¹·cm⁻¹（颜色淡）：

| 选律 | 内容 | d-d 跃迁违反？ | ε 惩罚 | 如何"部分突破" |
|:---|:---|:---:|:---:|:---|
| **自旋选律** | ΔS = 0（跃迁不改变自旋多重度） | 高自旋 d⁵ → e_g 跃迁需翻转自旋 → 违反！ | 10⁻² ~ 10⁻³ 倍 | 自旋-轨道耦合（重原子效应） |
| **宇称选律**（Laporte）| g↔u（只允许 g→u 或 u→g） | Oh 场中 t₂g(g) → e_g(g) → **g→g 禁阻**！ | 10⁻² ~ 10⁻³ 倍 | ① 振动耦合破坏对称中心 ② Td 场无对称中心 |

**两者的叠加效果**：
```
d-d 跃迁（Oh 场，高自旋 d⁵）：
  自旋禁阻（10⁻³）+ 宇称禁阻（10⁻³）→ ε < 1（几乎无色！）
  → 所以 [Mn(H₂O)₆]²⁺（d⁵ 高自旋）是极淡的粉色——几乎看不到颜色

d-d 跃迁（Oh 场，d¹ [Ti(H₂O)₆]³⁺）：
  自旋允许（ΔS=0）+ 宇称禁阻（10⁻³）→ ε ~10-50（淡色）
  → 淡紫色，能看到

Td 场 d-d 跃迁（如 [CoCl₄]²⁻）：
  自旋允许 + 宇称允许（Td 无对称中心）→ ε ~100-500（较浓）
  → 深蓝色！比 Oh 配合物颜色浓得多
```

**CT 跃迁——颜色浓烈的原因**：

```
CT 跃迁 = 电荷转移跃迁（Ligand-to-Metal 或 Metal-to-Ligand）
  → 自旋允许（ΔS=0 通常满足）
  → 宇称允许（配体轨道 g/u 与金属轨道 u/g 互补）
  → ε = 1000–50000（比 d-d 强 100–10000 倍！）
  → 颜色极浓（颜色深）
```

| CT 类型 | 方向 | 实例 | 颜色 | ε |
|:---|:---|:---|:---|:---:|
| **LMCT** | 配体 → 金属 | MnO₄⁻（O²⁻→Mn⁷⁺）、CrO₄²⁻ | **深紫、深黄** | 10000–50000 |
| **MLCT** | 金属 → 配体 | [Fe(phen)₃]²⁺、[Ru(bpy)₃]²⁺ | **血红、橙红** | 5000–20000 |

**d-d 与 CT 对比总结**：

| 特征 | d-d 跃迁 | CT 跃迁 |
|:---|:---|:---|
| **本质** | 金属 d 轨道之间 | 配体 ↔ 金属之间 |
| **自旋选律** | 高自旋 d⁵ 禁阻，其余可能允许 | 通常允许 |
| **宇称选律** | Oh 场禁阻（g→g），Td 场允许 | 允许（L→M 或 M→L） |
| **ε (L·mol⁻¹·cm⁻¹)** | 1–100（淡色） | 1000–50000（浓色） |
| **实例** | [Ti(H₂O)₆]³⁺ 淡紫 | MnO₄⁻ 深紫 |
| **溶剂效应** | 不明显 | 可显著（溶剂极性影响 CT 能量） |

> 💡 **口诀**："d-d 淡（两道禁令），CT 浓（无禁令）。浅色 → d-d，深色 → CT。高自旋 d⁵ 近无色。"

> [!info]- 完整推导：Tanabe-Sugano 图的使用（课后复习用）
>
> **教材来源**：Atkins Ch7 §7.12-7.13、结构化学基础 §9.7
>
> **Tanabe-Sugano 图是做什么的**：
> - 横轴 Δ/B，纵轴 E/B——给出了 dⁿ 体系**所有可能电子态**的能量随配体场强的变化
> - 用于指认 d-d 吸收光谱中的各条谱带（哪个基态→哪个激发态）
>
> **实际使用步骤**：
> 1. 由配合物磁矩确定基态自旋多重度
> 2. 确定 Δo/B（由最低能量吸收带）
> 3. 在 T-S 图上垂直画线，读出各激发态能量
> 4. 将计算值与实验吸收峰比对
>
> **本课不展开 T-S 图指认**，但学生应知道它的存在和用途。决赛阶段会系统训练 T-S 图指认。
>
> **与教材的对照说明**：
> - Atkins Ch7 Fig. 7.26-7.31 给出了 d²–d⁸ 的 T-S 图
> - 结构化学基础 §9.7 以 d² 和 d⁷ 为例讲解了 T-S 图使用
> - 本课要求掌握选律判断和 d-d/CT 区分，T-S 图指认放决赛

**阶段性小结**：
> d-d 跃迁受**自旋选律（ΔS=0）和宇称选律（Laporte g↔u）**双重限制 → 颜色淡；CT 跃迁无这些限制 → 颜色浓。Td 配合物的 d-d 跃迁比 Oh 配合物颜色更浓（宇称选律允许）。d⁵ 高自旋（如 Mn²⁺、Fe³⁺）的 d-d 跃迁双禁阻 → 近无色。

### 2.7 18 电子规则深化——从初步计数到系统应用

> 第一轮你知道"过渡金属配合物倾向于 18 电子"。本轮你学会用两种方法系统计数，处理 NO、Cp、烯烃等"特殊"配体。

**两种计数方法**：

| | Method A（离子法 / 氧化态法） | Method B（中性法 / 共价法） |
|:---|:---|:---|
| **思路** | 把所有配体当闭壳层离子，金属氧化态 = 形式电荷 | 所有组分当电中性，直接数价电子 |
| **金属贡献** | dⁿ 电子数（按氧化态数） | 价电子总数（s + d） |
| **配体贡献** | 每个配体给 2e（Lewis 碱） | 每个配体按中性贡献（如 CO 给 2e，Cp 给 5e） |
| **总电荷调整** | 配合物总电荷 — 配体形式电荷代数和 → 金属氧化态 | 无需调整 |
| **适合场景** | 配合物带电荷、氧化态明确的体系 | 金属有机（中性配体多）、低价态金属 |

**以 Fe(CO)₅ 为例**：

```
Method A（离子法）：
  Fe(0) → 8 个价电子（3d⁶4s²）→ d⁸ → 8e
  5 CO × 2e = 10e
  总 = 8 + 10 = 18e ✓

Method B（中性法）：
  Fe 中性原子：8 个价电子 → 8e
  5 CO 中性分子 × 2e = 10e
  总 = 8 + 10 = 18e ✓
  （两者一致，因为 Fe(0) 正好是中性原子）
```

**以 [CoCp₂]⁺ 为例**（检验两种方法是否等价）：

```
Method A（离子法）：
  [CoCp₂]⁺ 总电荷 +1
  2 Cp⁻ × (−1) = −2 → Co 氧化态 = +1 − (−2) = +3
  Co(III) = d⁶ → 6e
  2 Cp⁻ × 6e = 12e
  总 = 6 + 12 = 18e ✓

Method B（中性法）：
  Co 中性原子：9e（3d⁷4s²）
  2 Cp 中性自由基 × 5e = 10e
  配合物电荷 +1 → 减去 1e
  总 = 9 + 10 − 1 = 18e ✓
```

**特殊配体的电子计数**：

| 配体 | 配位模式 | Method A（离子法）贡献 | Method B（中性法）贡献 | 说明 |
|:---|:---|:---:|:---:|:---|
| CO | η¹-C | 2e | 2e | σ 给体 + π 受体 |
| CN⁻ | η¹-C | 2e | 1e（中性 CN 自由基） | 负电荷体现在氧化态中 |
| **NO（直线）** | η¹-N, M–N–O ≈ 180° | **3e**（NO⁺ 等电子于 CO） | 3e（NO 中性 11e 给 3e） | NO⁺ 与 CO 等电子体 |
| **NO（弯曲）** | η¹-N, M–N–O ≈ 120° | **1e**（NO⁻） | 1e | N 上有孤对，视为 X 型配体 |
| **Cp（η⁵）** | η⁵-C₅H₅ | **6e**（Cp⁻） | 5e（Cp 中性自由基） | 芳香性 6π 电子体系 |
| Cp（η³） | η³-C₅H₅ | 4e | 3e | 较少见 |
| **C₂H₄（η²）** | η² | 2e | 2e | π 电子作为 σ 给体 |
| C₄H₆（η⁴-丁二烯） | η⁴ | 4e | 4e | 两个 π 键 = 4e |
| C₆H₆（η⁶） | η⁶ | 6e | 6e | 三个 π 键 = 6e |
| PR₃ | η¹-P | 2e | 2e | 类似 CO 但 π 受体较弱 |
| H（氢负离子） | η¹ | 2e（H⁻） | 1e（H 中性自由基） | 常见于金属氢化物 |
| halide（X⁻） | η¹ | 2e（X⁻） | 1e（X 中性自由基） | 卤素配体 |

> ⚠️ **NO 配体的关键区分**：直线 NO（M–N–O ≈ 180°）→ 3e 给体（视为 NO⁺）；弯曲 NO（M–N–O ≈ 120°）→ 1e 给体（视为 NO⁻）。**先看键角再计数！**

**18e 规则的例外——为什么有些配合物稳定存在？**

| 类型 | 电子数 | 为什么稳定 | 实例 |
|:---|:---:|:---|:---|
| **平面四边形 d⁸** | 16e | dz² 填满、dx²-y² 是 LUMO——填满所有成键+非键轨道 = 16e | PtCl₄²⁻ (16e)、[Ni(CN)₄]²⁻ (16e) |
| **强 π-donor 配合物** | 12-14e | F⁻ 等 π-donor 推高 t₂g → t₂g 接近非键 → 不需要填 t₂g 就能稳定 | TiF₆²⁻ (12e)、WF₆ (12e) |
| **超 18e** | 19-20e | 配体是弱 π-acceptor，不能有效降低 d 能量 → d 电子多于 18e | Cp₂Ni (20e)、[CoCp₂] (19e) |

**判断逻辑**：
```
平面四边形 d⁸（Pt²⁺, Pd²⁺, Ni²⁺, Au³⁺, Rh⁺）→ 大概率 16e
强 π-donor 配体（F⁻, O²⁻, NR₂⁻）+ 高氧化态金属 → 可能 12-14e
大环 Cp 配合物（第一行过渡金属）→ 可能 19-20e
其他体系 → 默认追求 18e
```

> [!info]- 完整推导：18e 规则的 MO 解释（课后复习用）
>
> **教材来源**：Atkins Ch20 §20.4-20.5、[[教学逻辑提炼-Zchem-过渡元素的结构化学-第三轮深化]] §四 台阶 7
>
> **为什么是 18？**
> 过渡金属有 9 个价轨道（1 个 s + 3 个 p + 5 个 d）。形成八面体配合物时：
> - 6 个 σ 成键轨道（a₁g + e_g + t₁u）→ 可容纳 12e
> - 3 个非键/弱 π 轨道（t₂g）→ 可容纳 6e
> - 共计 18e → 填满全部成键 + 非键轨道 → 热力学稳定
>
> **16e 平面四边形的 MO 图像**：
> - dx²-y² 指向配体 → 强 σ* 反键（LUMO）
> - dz² 在 z 方向 → 非键（被填满）
> - 剩下 8 个成键/非键轨道 → 16e 填满
>
> **12-14e 的解释**：
> 强 π-donor（如 F⁻）→ t₂g 被推高，接近反键 → 不填 t₂g → 只填 σ 成键轨道 → 12e
>
> **与教材的对照说明**：
> - Atkins Ch20 §20.4 从 MO 理论推导了 18e 规则的来源
> - Atkins Ch20 Table 20.3 给出了常见配体的电子计数（Method A 和 Method B）
> - 结构化学基础并未系统展开 18e 规则，本课补充完整

**阶段性小结**：
> 18e 规则的两大计数法：离子法（Method A，氧化态 + 2e/配体）和中性法（Method B，直接价电子加和）。关键是特殊配体的计数——NO 直线 3e/弯曲 1e、Cp η⁵=6e/5e、C₂H₄=2e。16e 平面四边形和 12-14e 强 π-donor 是常见例外，均有合理的 MO 解释。

### 2.8 IR 鉴定配合物异构体——对称性决定峰数

> 配合物的 IR 光谱中，CO 伸缩振动（~1700-2150 cm⁻¹）是鉴定异构体的"指纹区"。

**核心原理**：分子对称性越高 → IR 活性振动模式越少 → CO 伸缩峰越少。

**同分异构体的 IR 峰数速查表**：

| 配合物类型 | 点群 | CO 伸缩 IR 活性峰数 | 口诀 |
|:---|:---:|:---:|:---|
| **trans-[M(CO)₄L₂]** | D₄h | **1 峰**（Eᵤ） | 反式高对称，单峰 |
| **cis-[M(CO)₄L₂]** | C₂ᵥ | **4 峰**（2A₁ + B₁ + B₂） | 顺式低对称，四峰 |
| **fac-[M(CO)₃L₃]** | C₃ᵥ | **2 峰**（A₁ + E） | 面式 C₃ 轴，两峰 |
| **mer-[M(CO)₃L₃]** | C₂ᵥ | **3 峰**（2A₁ + B₁） | 经式无 C₃，三峰 |
| [M(CO)₆] | Oₕ | 1 峰（T₁ᵤ） | 最对称，单峰 |
| [M(CO)₅L] | C₄ᵥ | 3 峰（2A₁ + E） | 单取代降对称 |

**实例判断**：

```
题目：某配合物 Mo(CO)₃(NCMe)₃ 的 IR 在 CO 伸缩区显示 2 个峰。
      判断它是 fac 还是 mer 异构体？

解答：
  fac-[Mo(CO)₃(NCMe)₃] → C₃ᵥ → 2 峰（A₁ + E）
  mer-[Mo(CO)₃(NCMe)₃] → C₂ᵥ → 3 峰（2A₁ + B₁）
  2 峰 → fac 异构体 ✓
```

> 💡 **核心口诀**："对称性越高，峰越少——反式 1、面式 2、经式 3、顺式 4。"

> [!info]- 完整推导：群论方法数 IR 峰数（课后复习用）
>
> **教材来源**：Atkins Ch20 §20.8、[[教学逻辑提炼-Zchem-过渡元素的结构化学-第三轮深化]] §四 台阶 10
>
> **怎么数**：
> 1. 确定分子的点群
> 2. 确定 CO 伸缩振动的可约表示（Γ_CO）
> 3. 约化为不可约表示之和
> 4. 减去平动（T）和转动（R）→ 得到真正的振动模式
> 5. 检查哪些不可约表示有 IR 活性（与 x, y, z 有相同变换性质）
> 6. 计数
>
> **本课不要求完整群论推导**，但学生应记住上表的峰数结论并会使用。
>
> **与教材的对照说明**：
> - Atkins Ch20 §20.8 给出了金属羰基配合物的 IR 光谱实例
> - 群论约化表的详细使用放决赛或物理化学深化

**阶段性小结**：
> 对称性决定 IR 峰数——这是"结构决定光谱"的完美案例。peak count = f(symmetry)：fac（C₃ᵥ, 2 峰）、mer（C₂ᵥ, 3 峰）、trans（D₄h, 1 峰）、cis（C₂ᵥ, 4 峰）。

### 2.9 配合物结构推断综合流程

> 将本节所有工具串联成一个完整推断链。

```text
实验数据 → 推断流程 → 结论

① 元素分析  →  配合物组成（中心金属 + 配体 + 外界离子）

② 磁矩 μ_eff → 未成对电子数 n = ?
     ↓  μ = √[n(n+2)]（自旋-only）
     ↓  注意：d⁴-d⁷ 可能有轨道角动量贡献，实测值略高于自旋-only

③ n → dⁿ 电子构型 → 高自旋 / 低自旋？
     ↓  dⁿ 构型 = (金属价电子数) − (氧化态)
     ↓  例如 Fe³⁺ = 26 − 3 = 23 → d⁵

④ 高/低自旋 + 配体种类 → Δo 的相对大小
     ↓  低自旋 → Δo > P（配对能）→ 配体在光谱化学序列右端
     ↓  高自旋 → Δo < P → 配体在光谱化学序列左端

⑤ Δo → 光谱化学序列中的配体位置验证
     ↓  由颜色和吸收波长可估算 Δo ≈ hc/λ_max

⑥ 颜色（浓/淡）→ d-d（淡，ε < 100）vs CT（浓，ε > 1000）
     ↓  浓色 + 高氧化态金属 + O²⁻/S²⁻ 配体 → LMCT
     ↓  浓色 + 低氧化态金属 + π-acceptor 配体 → MLCT

⑦ 18e 计数验证稳定性
     ↓  16e 平面四边形？→ d⁸ + 强场（Pt²⁺, Ni²⁺, Pd²⁺）
     ↓  12-14e？→ 强 π-donor（F⁻, O²⁻）+ 高氧化态
     ↓  超 18e？→ Cp 配合物、弱 π-acceptor

⑧ 如有 IR 数据 → 峰数 → 异构体判断
     ↓  trans-1 / cis-4 / fac-2 / mer-3

最终结论：配合物的完整结构（中心金属氧化态 + 配体排列 + 几何构型 + 自旋态）
```

### 例题 2（⭐⭐ 基础）：CFSE 计算与磁性判断

**题目**：
(1) 计算 [Fe(H₂O)₆]²⁺ 和 [Fe(CN)₆]⁴⁻ 的 CFSE（单位 Dq，忽略配对能 P）。
(2) 预测两者的磁矩（自旋-only），并指出哪个的实测磁矩更可能偏离自旋-only 公式。
(3) 已知 [Fe(H₂O)₆]²⁺ 的实测 μ = 5.3 BM，解释为什么大于自旋-only 值。

**分析提示**：
- Fe²⁺ = d⁶
- H₂O 是弱场（高自旋），CN⁻ 是强场（低自旋）
- d⁶ 高自旋 = t₂g⁴ e_g²，d⁶ 低自旋 = t₂g⁶
- t₂g 不对称占据 → 轨道角动量贡献

<details>
<summary>完整解答路径（点击展开）</summary>

(1) CFSE 计算：
- [Fe(H₂O)₆]²⁺：d⁶ 高自旋 → t₂g⁴ e_g²
  CFSE = 4 × (−4Dq) + 2 × (6Dq) = −16Dq + 12Dq = **−4Dq**
- [Fe(CN)₆]⁴⁻：d⁶ 低自旋 → t₂g⁶
  CFSE = 6 × (−4Dq) = **−24Dq**（不考虑 P）

(2) 磁矩预测：
- [Fe(H₂O)₆]²⁺：d⁶ hs → n = 4 → μ = √(4×6) = √24 ≈ **4.90 BM**
- [Fe(CN)₆]⁴⁻：d⁶ ls → n = 0 → μ = **0 BM**（抗磁性）

偏离更大的：[Fe(H₂O)₆]²⁺（d⁶ hs，t₂g⁴ 不对称 → 轨道贡献）

(3) 实测 5.3 BM > 4.90 BM 的原因：
t₂g⁴ 中有 1 个空穴 → t₂g 电子可绕轴旋转 → 轨道角动量不完全淬灭 → 额外顺磁贡献
</details>

### 例题 4（⭐⭐⭐ 拓展）：综合推断——从实验数据推结构

### 例题 5（⭐⭐⭐ 拓展）：18e 计数 + IR 异构体判断

**题目**：某配合物 Mn(CO)₃(NO)(PPh₃) 的 IR 在 CO 伸缩区显示 2 个峰。已知 NO 以直线方式配位。
(1) 用 Method A 和 Method B 分别计数该配合物的价电子数。
(2) 判断该配合物的几何构型（fac 还是 mer？）。
(3) 预测 CO 伸缩频率相对于自由 CO（2143 cm⁻¹）的变化趋势（红移/蓝移），并解释原因。

**分析提示**：
- NO 直线配位 → 3e 给体（Method A）
- IR 峰数 → 对称性 → fac/mer
- Mn 的氧化态由电荷平衡决定

<details>
<summary>完整解答路径（点击展开）</summary>

(1) 电子计数：

Method A（离子法）：
- 配合物总电荷：0（中性配合物）
- NO 直线 → NO⁺（等电子于 CO）→ 氧化态考虑：NO⁺ + 3CO + PPh₃ = 全部中性 → Mn(0)
- Mn(0) = d⁷（3d⁵4s² → 7 价电子）
- 3 CO × 2e = 6e
- 1 NO（直线）× 3e = 3e
- 1 PPh₃ × 2e = 2e
- 总 = 7 + 6 + 3 + 2 = **18e** ✓

Method B（中性法）：
- Mn 中性原子：7e
- 3 CO × 2e = 6e
- 1 NO（直线中性）× 3e = 3e
- 1 PPh₃ × 2e = 2e
- 总 = 7 + 6 + 3 + 2 = **18e** ✓

(2) IR 判断几何构型：
- 类似 M(CO)₃L₃（3CO + NO + PPh₃ = 5 配体，但在这里配位数 = 5 或 6？）
- 实际上是五配位 Mn(CO)₃(NO)(PPh₃) → 总配位数 = 5
- 对五配位三羰基配合物，fac-like（CO 在赤道面）→ C₃ᵥ or Cₛ → **2 峰**
- mer-like（2CO 赤道 + 1CO 轴向）→ Cₛ → **3 峰**
- 显示 2 峰 → **fac 型**排布 ✓

(3) CO 伸缩频率预测：
- Mn(0) 是富电子金属（d⁷，低氧化态）
- 强烈的 M→CO π 反馈 → CO 的 π* 被占据 → C≡O 键削弱
- ν(CO) 将**显著红移**，预计在 1900–2000 cm⁻¹ 区域（自由 CO 2143 cm⁻¹）
- 与 [Mn(CO)₆]⁺（缺电子，ν≈2090）相比，红移更大
</details>

## 四、即时演练

> 3 分钟快速检测，覆盖本节最核心的决策分叉点。

**演练题**：

(1) 从 σ/π 性质判断：为什么 en（乙二胺）的场强介于 NH₃ 和 CN⁻ 之间？

(2) [Mn(H₂O)₆]²⁺ 的磁矩应接近哪个值？
A. 0 BM　B. 1.73 BM　C. 5.92 BM　D. 4.90 BM

(3) 从以下两种配合物中选出颜色更浓的一个并说明理由：
[Cr(H₂O)₆]³⁺ 淡紫 vs [CrO₄]²⁻ 深黄

**自检答案**：
- (1) en 是 σ-only donor（无 π 作用）→ 场强在 σ-only 范围内。但 en 是双齿配体，螯合效应使配位更紧密 → 场强略高于 NH₃。CN⁻ 是 π-acceptor → 场强更大。
- (2) **C. 5.92 BM**。Mn²⁺ = d⁵，H₂O 弱场 → 高自旋 → n=5 → μ≈5.92
- (3) [CrO₄]²⁻ 更浓。CrO₄²⁻ 是 CT 跃迁（O→Cr LMCT，ε > 1000），[Cr(H₂O)₆]³⁺ 是 d-d 跃迁（宇称禁阻，ε ~10–50）。

## 五、本节形成的核心结论

> 以下 5 条是本节必须带走的知识，课后应能独立复述和推导。

1. **配体场理论**：配体不是点电荷——σ 给体抬 e_g，π 受体降 t₂g（Δo↑），π 给体抬 t₂g（Δo↓）。光谱化学序列的本质是 π 作用对 t₂g 的调节。

2. **CFSE 定量**：Oh 场 CFSE = (−4Dq)×n(t₂g) + (6Dq)×n(e_g)，d³/d⁸ 始终 −12Dq，d⁴-d⁷ 取决于 Δo vs P。Td 场 Δt ≈ 4/9Δo，CFSE 更小。

3. **Jahn-Teller 效应**：e_g 不均匀占据（d⁴ hs / d⁷ ls / d⁹）→ 拉长畸变。d⁶ ls / d³ / d⁸ 无 JT。二阶 JT（d⁰）→ 偏心 → 铁电性。

4. **电子光谱双选律**：自旋选律（ΔS=0）+ 宇称选律（Laporte g↔u）→ d-d 跃迁弱（淡色）；CT 跃迁无限制 → 强（浓色）。高自旋 d⁵ 双禁阻 → 近无色。

5. **18e 规则系统计数**：Method A（离子法，2e/配体）和 Method B（中性法）等价但数法不同。NO 直线 3e / 弯曲 1e，Cp η⁵=6e，C₂H₄=2e。16e 平面四边形 d⁸ 和 12-14e 强 π-donor 是常见例外。

| 1. 配体三分类 → 光谱序列 | 为什么 CO 是强场而 Cl⁻ 是弱场？ | [[配体场理论]]、[[光谱化学序列]] |
| 2. CFSE 定量 → 磁性 | 如何定量计算配合物稳定性和磁性？ | [[晶体场理论#CFSE]] |

## 六、⚠️ 易错点预警

| 错误想法 | 为什么错 | 正确理解 |
|:---|:---|:---|
| "π 键一定增大 Δo" | π-donor（如 Cl⁻）的 filled p 推高 t₂g → Δo 反而**减小** | π-acceptor（CO, CN⁻）增大 Δo；π-donor（Cl⁻, F⁻）减小 Δo |
| "CFSE 的 Dq 跟 Δo 的关系可以随便用" | 不同教材定义不同：Δo = 10Dq（Atkins），Δo = Dq（某些教材） | 先确认题目或教材的 Dq 定义 |
| "d⁴ 一定发生 JT 畸变" | d⁴ **高自旋**才 JT（e_g¹），d⁴ 低自旋（t₂g⁴）无 JT | 先判断高/低自旋（取决于配体场强） |
| "d-d 跃迁自旋禁阻 → 完全看不到颜色" | 自旋-轨道耦合可使自旋禁阻**部分**放松（重原子效应），ε 虽小但不为零 | 双禁阻（自旋+宇称）→ ε < 1，极淡；单禁阻 → ε ~10–100，淡色 |
| "Method A 和 Method B 总是一致" | 对于带电配合物，两种方法的**中间数**不同，但最终总电子数应一致 | 若结果不一致 → 配体计数或氧化态算错了 |
| "18e 是硬规则" | 16e 平面四边形 d⁸ 和 12-14e 强 π-donor 是稳定例外 | 18e 是经验规则而非法律，理解例外背后的 MO 原因 |
| "Td 场的 CFSE 比 Oh 大" | Δt ≈ 4/9 Δo → Td 的 CFSE 总是更小 | dⁿ 离子优先占据 Oh 位（CFSE 驱动力） |
