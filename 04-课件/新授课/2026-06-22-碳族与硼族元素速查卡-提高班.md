# 碳族与硼族元素 · 速查卡

> 基础班你背过"碳族=C/Si/Ge/Sn/Pb"——本卡把碳族/硼族从**事实列表**升级到**惰性电子对效应+缺电子结构**双线驱动的判断系统。

---

## 一、碳族 · 惰性电子对效应

```
碳族从上到下（C → Si → Ge → Sn → Pb）：

  惰性电子对效应：6s² 电子对越来越"懒"——不参与成键
  
  后果：
  C：+4 为主（CO₂, CO₃²⁻），CO 中 +2 是例外
  Si：+4 为主（SiO₂, 硅酸盐），+2 不稳定
  Ge：+4 > +2
  Sn：+4 和 +2 都常见（SnCl₂/SnCl₄）
  Pb：+2 >> +4（PbO 稳定，PbO₂ 强氧化剂！）
```

**PbO₂ 的强氧化性**：Pb(IV)→Pb(II) 的还原趋势极强——PbO₂ + 4HCl → PbCl₂ + Cl₂↑ + 2H₂O。这是惰性电子对效应最直观的表现。

---

## 二、碳族核心化合物速查

| 元素 | 氧化物 | 特征性质 | 推断信号 |
|:---|:---|:---|:---|
| **C** | CO/CO₂ | CO 可燃（蓝色火焰），CO₂ 使石灰水变浑 | CO₂ 无色无味不支持燃烧 |
| **Si** | SiO₂ | 不与除 HF 外的酸反应；SiO₂+4HF→SiF₄↑+2H₂O | 白色固体+只溶于 HF |
| **Sn** | SnO/SnO₂ | Sn²⁺ 还原剂（SnCl₂ 还原 Fe³⁺→Fe²⁺） | SnCl₂ 白色固体，易水解 |
| **Pb** | PbO/PbO₂/Pb₃O₄ | Pb₃O₄（铅丹）= 2PbO·PbO₂；PbO₂ 强氧化剂 | Pb₃O₄+HNO₃→PbO₂↓（棕色） |

**CO 的特殊性**：C 在 CO 中为 +2——是碳的罕见低价态。CO 是极强的 σ 给体+π 受体配体（金属羰基配合物）。

---

## 三、硅酸盐结构速查

```
硅酸盐的基本单元：SiO₄⁴⁻ 四面体

  岛状：孤立 SiO₄⁴⁻（橄榄石 Mg₂SiO₄）→ Si:O = 1:4
  链状：SiO₃²⁻（辉石）→ Si:O = 1:3
  层状：Si₂O₅²⁻（云母/黏土）→ Si:O = 2:5
  架状：SiO₂（石英/长石）→ Si:O = 1:2

  口诀：共顶点越多→Si:O 比越大→从岛到架
```

---

## 四、硼族 · 缺电子化学

```
硼族（B/Al/Ga/In/Tl）：

  B：缺电子（只有 3 个价电子→6 电子构型）→ Lewis 酸
  Al：两性金属 + 缺电子（AlCl₃ 是 Lewis 酸催化剂）
  Ga/In/Tl：惰性电子对效应回归（Tl⁺ 稳定）
```

### 4.1 硼的核心化学

| 化合物 | 结构 | 关键性质 |
|:---|:---|:---|
| **B₂H₆** | 两个 B–H–B 3c-2e 桥键 | 缺电子桥键——打破 2c-2e 定式 |
| **H₃BO₃** | 平面三角形 | Lewis 酸（不是质子酸！）——B(OH)₃ + H₂O → B(OH)₄⁻ + H⁺ |
| **硼砂** | [B₄O₅(OH)₄]²⁻ | 水解呈碱性——Na₂B₄O₇+7H₂O→4H₃BO₃+2NaOH |

**硼酸的特殊酸式**：H₃BO₃ 不是通过解离 H⁺ 表现酸性，而是通过接受 OH⁻（Lewis 酸性）→B(OH)₄⁻ + H⁺。因此它的酸性来自水的自解离平衡被拉动。

---

## 五、铝的两性

```
Al 的两性 = 既能与酸反应也能与碱反应

  与酸：2Al + 6H⁺ → 2Al³⁺ + 3H₂↑
  与碱：2Al + 2OH⁻ + 6H₂O → 2[Al(OH)₄]⁻ + 3H₂↑

  Al(OH)₃ 的两性：
  ├─ 酸中：Al(OH)₃ + 3H⁺ → Al³⁺ + 3H₂O
  └─ 碱中：Al(OH)₃ + OH⁻ → [Al(OH)₄]⁻（铝酸根）
```

**推断题信号**：白色胶状沉淀+加过量碱溶解 → Al(OH)₃（Zn(OH)₂ 也有类似行为——Zn 也是两性！）

---

## 六、常见易错点

| 错误 | 纠正 |
|:---|:---|
| "Pb 的稳定氧化态是 +4" | **Pb²⁺ 最稳定**——惰性电子对效应使 6s² 不参与成键 |
| "CO 中 C 的氧化数是 +4" | CO 中 C 是 +2——不要只记得 C=+4 |
| "硼酸是质子酸" | H₃BO₃ 是 Lewis 酸（接受 OH⁻），不是解离 H⁺ |
| "Al₂O₃ 与所有酸反应" | Al₂O₃ 不与 HNO₃（钝化——表面形成致密氧化膜）有效反应 |
| "硅酸=SiO₂·xH₂O" | 硅酸通常以胶体或凝胶形式存在，没有确定的化学式 |

## 七、例题速练

### 例 1：惰性电子对效应

**题**：PbO₂ + HCl → ？为什么不是复分解？

**速解**：PbO₂ 中 Pb(IV) 有强氧化性（惰性电子对效应→Pb(IV)→Pb(II) 趋势强）。与 HCl 发生氧化还原：PbO₂ + 4HCl → PbCl₂ + Cl₂↑ + 2H₂O。PbO₂ 将 Cl⁻ 氧化为 Cl₂，自身还原为 Pb²⁺。

### 例 2：硼的缺电子

**题**：为什么 BCl₃ 是 Lewis 酸而 CCl₄ 不是？

**速解**：B 只有 3 个价电子→BCl₃ 中 B 只有 6 个电子→缺电子→可接受孤对电子（Lewis 酸）。C 有 4 个价电子→CCl₄ 中 C 有 8 个电子→八隅体饱和→不是 Lewis 酸。

## 八、自查清单

- [ ] 惰性电子对效应：C/Si→+4 为主；Sn→+4/+2 均可；Pb→+2 最稳
- [ ] PbO₂ 的强氧化性（惰性电子对效应的直接体现）
- [ ] 硅酸盐四类结构：岛(SiO₄⁴⁻)/链(SiO₃²⁻)/层(Si₂O₅²⁻)/架(SiO₂)
- [ ] 硼烷 B₂H₆ 的 3c-2e 键（缺电子桥键）
- [ ] 硼酸 H₃BO₃ 是 Lewis 酸，不是质子酸
- [ ] Al 的两性——酸中 Al³⁺、碱中 [Al(OH)₄]⁻
- [ ] Al(OH)₃ 与 Zn(OH)₂ 都是两性——加过量碱都溶解
- [ ] CO 中 C=+2（罕见低价态）+ CO 是强配体
- [ ] SnCl₂ 是还原剂（Sn²⁺→Sn⁴⁺）
- [ ] SiO₂ 只与 HF 反应（不与 HCl/H₂SO₄/HNO₃ 反应）
