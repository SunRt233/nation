## 一、课堂引入与本节问题

### 1.1 课堂切入

**问题**：第一轮你学了 ΔG = ΔH − TΔS，知道 ΔG < 0 反应自发。但你想过吗——

> 这个公式是**推导出来的**，不是拍脑袋定义的。而且它只是"冰山一角"——热力学有一条完整的推导链，从**一条公式**出发，可以推出整个化学平衡、相变、溶液性质的理论。

**现象冲击**：
- 你知道沸腾需要 100°C——但如果去珠峰顶上（0.33 atm），水 71°C 就沸腾了。**你能算出来吗？**
- 你知道升温有利于吸热反应——但你**能定量算出** N₂+3H₂⇌2NH₃ 在 700K 的平衡常数比 298K 小了多少倍吗？
- 你知道 ΔG = −RTlnK——但这个公式**怎么从热力学定律推导出来**？

**学生已有认知**：第一轮已掌握 ΔU/ΔH/ΔS/ΔG 的概念、盖斯定律、键能计算、K 表达式、Q vs K 判断、勒夏特列原理。

**本节要解决的问题**：
1. 热力学的四大基本公式（dU/dH/dA/dG）如何从 dU=TdS−pdV 一条公式推导出来？
2. Maxwell 关系和 Gibbs-Helmholtz 方程是什么？有什么用？
3. 化学势如何作为"桥梁"连接热力学和化学平衡？
4. van't Hoff 方程如何定量描述温度对平衡常数的影响？
5. Clausius-Clapeyron 方程如何从热力学推导出来？依数性又如何统一推导？

### 1.2 学习目标

- **本节讲到哪里为止**：
  - 能从 dU=TdS−pdV 推导全部热力学关系
  - 能区分"普化表达式"和"物化微分表达式"
  - 能用 van't Hoff 方程定量计算变温 K
  - 能用 Clapeyron/Clausius-Clapeyron 处理相变条件
  - 能从化学势推导依数性
- **后续深入学习**：统计热力学熵、完整 Debye-Hückel、逸度系数实验测定、不可逆过程热力学（决赛/第四轮）

## 二、知识内容

> 📷 **图片策略**：本专题类型为 `calculation + derivation`，以板书推导和公式对比表为主，PPT 展示 Maxwell 关系 4 条和 van't Hoff 三形式对比。

### 2.1 组合定律：dU = TdS − pdV——一切热力学的起点

> 第一轮你学过热一律和热二律是"分开的两句话"。本轮你把它们**合并成一条公式**。

**从问题出发**：能不能把热一律和热二律写成一条微分式？

**关键推理**：

```
热一律：dU = δq + δw
  对于可逆过程 + 只有体积功：δq = δq_rev，δw = −p dV

热二律：dS = δq_rev / T  →  δq_rev = T dS

合并：dU = T dS − p dV    ← 这就是"组合定律"（Combined Law）
```

**这条公式的威力**：
- U 的自然变量是 S 和 V：U=U(S,V)
- 所有其他热力学函数（H, A, G）都通过 Legendre 变换从 U 导出
- 所有 Maxwell 关系都从这条公式的全微分条件导出
- **只需要记住 dU=TdS−pdV，其余全部可以推导**

> [!info]- 完整推导：为什么 δw = −p dV 只在可逆时成立（课后复习用）
>
> **教材来源**：Atkins Ch2 §2.4-2.5、物理化学 §1.6
>
> **可逆体积功 vs 不可逆体积功**：
> - 可逆膨胀：体系压强 p 与外压 p_ex 始终相差无穷小 → δw_rev = −p_ex dV = −p dV
> - 不可逆膨胀（如向真空膨胀）：p_ex = 0 → w = 0，但 p dV ≠ 0 → δw ≠ −p dV
> - 因此 dU = TdS − pdV 只在**可逆过程**严格成立
> - 但因为 U 是状态函数，两个状态之间的 ΔU 与路径无关，所以可以用可逆路径的 dU=TdS−pdV 积分来计算任何过程的 ΔU
>
> **与教材的对照说明**：
> - Atkins Ch2 Table 2.3 给出了 dU = TdS − pdV 的推导步骤
> - 本课采用的符号约定：体系吸热 q>0，体系对外做功 w<0（与 Atkins 一致）

**阶段性小结**：
> dU = TdS − pdV 是热力学推导的"第一性原理"。U 是 S 和 V 的函数——改变 S（传热）和改变 V（做功）就是改变 U 的两种方式。

### 2.2 Legendre 变换——从 U 到 H, A, G

> 如果 U 的自然变量是 S 和 V……但实验中我们控制的是 T 和 p！怎么把自变量换成好用的？

**从问题出发**：U 的自变量是 S 和 V，但 S 不好测、V 不好控。能不能定义新的热力学函数，让自变量变成好控制的变量（T, p）？

**Legendre 变换的思想**：用"减去共轭变量乘积"的方式**换掉一个自变量**。

```
定义焓 H：换掉 V（→ p）
  H = U + pV          →  dH = dU + pdV + Vdp = TdS + Vdp
  自然变量：S, p       →  H = H(S, p)

定义 Helmholtz 自由能 A：换掉 S（→ T）
  A = U − TS          →  dA = dU − TdS − SdT = −SdT − pdV
  自然变量：T, V       →  A = A(T, V)

定义 Gibbs 自由能 G：换掉 S 和 V（→ T 和 p）
  G = H − TS = U + pV − TS  →  dG = −SdT + Vdp
  自然变量：T, p       →  G = G(T, p)
```

**四大基本公式汇总**：

| 函数 | 定义 | 微分式 | 自然变量 | 适用的过程条件 |
|:---|:---|:---|:---|:---|
| **U**（内能） | — | **dU = TdS − pdV** | S, V | 恒 S 恒 V（孤立体系） |
| **H**（焓） | H = U + pV | **dH = TdS + Vdp** | S, p | 恒 S 恒 p |
| **A**（Helmholtz） | A = U − TS | **dA = −SdT − pdV** | T, V | 恒 T 恒 V |
| **G**（Gibbs） | G = H − TS | **dG = −SdT + Vdp** | T, p | 恒 T 恒 p（最常见！） |

> 💡 **记忆法**：dU=TdS−pdV → dH 把 −pdV 变成 +Vdp（H=U+pV 的微分），dA 把 TdS 变成 −SdT（A=U−TS 的微分），dG 两个都变。**每次变换"换掉一个自变量、留下它的共轭变量"**。

**为什么 Gibbs 自由能最重要？**
- 化学反应通常在**恒温恒压**下进行 → G 的自然变量正好是 T 和 p
- 恒温恒压下：dG = −SdT + Vdp 中 dT=dp=0 → dG=0 是平衡条件
- ΔG < 0 是自发过程的判据（恒 T 恒 p 且只有体积功时）

> [!info]- 完整推导：Legendre 变换的普遍形式（课后复习用）
>
> **教材来源**：Atkins Ch3 §3.3-3.6、物理化学 §2.1-2.4
>
> **Legendre 变换的数学原理**：
> 给定函数 f(x,y)，其全微分为 df = u dx + v dy（其中 u=∂f/∂x, v=∂f/∂y）
> 如果想用 u 替代 x 作为自变量，定义新函数 g = f − ux
> 则 dg = df − u dx − x du = −x du + v dy → 自变量从 (x,y) 变成了 (u,y)
>
> 应用到热力学：
> - U(S,V): dU = TdS − pdV → H = U − (−p)V = U + pV → dH = TdS + Vdp（换 V→p）
> - U(S,V): dU = TdS − pdV → A = U − TS → dA = −SdT − pdV（换 S→T）
> - A(T,V): dA = −SdT − pdV → G = A − (−p)V = A + pV → dG = −SdT + Vdp（再换 V→p）
>
> **与教材的对照说明**：
> - Atkins Ch3 从数学上完整介绍了 Legendre 变换，本课取结论性的对比表
> - 物理化学 §2.1-2.4 逐步推导四个基本公式，与 Zchem 教师的讲授顺序一致
> - 本课不要求掌握 Legendre 变换的严格数学证明，但要求学生能写出定义式并验证微分式

**阶段性小结**：
> U→H→A→G 是一条变换链。H 把 V 换成 p（恒压反应用 H），A 把 S 换成 T（恒温反应用 A），G 两个都换（恒温恒压反应用 G）。化学反应用 G 是因为实验通常在恒温恒压下进行。

### 2.3 Maxwell 关系 + Gibbs-Helmholtz 方程

> 四大基本公式不只是"好看"——它们的全微分条件导出一组极其有用的偏导数关系。

**Maxwell 关系（简述）**：

全微分 df = M dx + N dy 满足 ∂M/∂y = ∂N/∂x（二阶混合偏导相等）。对四大基本公式逐一应用：

```
对 dU = TdS − pdV：  ∂T/∂V)_S = −(∂p/∂S)_V   （1）
对 dH = TdS + Vdp：  ∂T/∂p)_S =  (∂V/∂S)_p   （2）
对 dA = −SdT − pdV： ∂S/∂V)_T =  (∂p/∂T)_V   （3）
对 dG = −SdT + Vdp： ∂S/∂p)_T = −(∂V/∂T)_p   （4）
```

**Maxwell 关系的用途**：把不方便测量的量（如 ∂S/∂V）转化为方便测量的量（如 ∂p/∂T）。

**最常用的两条**：
- (3)：∂S/∂V)_T = (∂p/∂T)_V — 用于推导理想气体内能只依赖温度
- (4)：∂S/∂p)_T = −(∂V/∂T)_p — 用于计算恒温下 ΔG 随压力的变化

**Gibbs-Helmholtz 方程**：

从 dG = −SdT + Vdp 可知 (∂G/∂T)_p = −S。代入 G = H − TS：

```
G = H − TS = H + T(∂G/∂T)_p

整理得：(∂(G/T)/∂T)_p = −H/T²    ← G-H 方程

对于反应：∂(ΔG/T)/∂T)_p = −ΔH/T²
代入 ΔG = −RTlnK：d lnK / dT = ΔH° / RT²   ← 这就是 van't Hoff 方程！
```

> 💡 **G-H 方程的重要性**：它把 **ΔG 的温度依赖**与 **ΔH** 联系起来。一旦你知道 ΔH，就可以求出任何温度下的 ΔG（进而求出 K）。

> [!info]- 完整推导与教材对照（课后复习用）
>
> **教材来源**：Atkins Ch3 §3.7-3.9（Maxwell）、§3.11（G-H 方程）；物理化学 §2.5-2.7
>
> **Maxwell 关系推导细节**：
> 以 dU = TdS − pdV 为例：
> - M = T(S,V), N = −p(S,V)
> - ∂M/∂V)_S = ∂T/∂V)_S
> - ∂N/∂S)_V = −∂p/∂S)_V
> - 全微分条件：∂T/∂V)_S = −∂p/∂S)_V ✓
>
> **Maxwell 关系的记忆法（"Thermodynamic Square"）**：
> 以 U→p→H→V→G→T→A→S→U 画一个正方形和两条对角线，从每个顶角顺时针/逆时针读取偏导关系。详见 Atkins Ch3 Fig 3.15。
>
> **与教材的对照说明**：
> - Atkins Ch3 §3.7 给出了 Maxwell 关系的完整推导
> - 物理化学 §2.5-2.6 以 (∂S/∂V)_T 为例展示了 Maxwell 关系的应用
> - 本课不要求记住全部 4 条 Maxwell 关系，但要求会推导和应用常用的 2 条

**阶段性小结**：
> Maxwell 关系 = 全微分条件 × 四大基本公式 → 4条偏导数恒等式。G-H 方程 = (∂(G/T)/∂T) = −H/T² → 关键应用：已知 ΔH，求任何温度下的 ΔG 和 K。

### 2.4 化学势——热力学与化学平衡的桥梁

> 为什么化学反应会达到平衡？因为**反应物和产物的化学势相等了**。化学势是"物质逃逸倾向"的量化。

**从问题出发**：纯物质的 Gibbs 自由能随 n 线性增加。但混合物中，1 mol A 的 Gibbs 自由能贡献不是简单的 G_A(T,p)，因为混合物有熵效应。怎么定义混合物中每种物质的"有效 Gibbs 自由能"？

**化学势定义**：

```
μ_B = (∂G/∂n_B)_{T, p, n_{j≠B}}

开放体系 dG  = −SdT + Vdp + Σ μ_B dn_B
             ↑恒温恒压时只剩这一项↑
     dG = Σ μ_B dn_B（恒T恒p）
```

**化学势的物理意义**：
- 化学势 = 恒温恒压下向体系中加入 1 mol 物质 B 时 Gibbs 自由能的增量
- **化学势是"物质从当前环境逃逸的倾向"**——μ 越高，越倾向于从该环境离开
- 平衡条件：反应物和产物的加权化学势相等：Σ ν_B μ_B = 0

**理想气体化学势**：

```
对于纯理想气体，恒温下：dG = Vdp = (RT/p) dp → ∫dG = RT ∫dp/p

  μ(p) = μ° + RT ln(p/p°)    ← 标准化学势 + 压力修正项

其中 μ° = 标准化学势（1 bar 时的化学势）
     RT ln(p/p°) = 压力偏离标准态的"化学势惩罚/奖励"
```

**从化学势到 van't Hoff 等温式**：

```
反应 aA + bB → cC + dD

ΔG = Σ ν_B μ_B = c·μ_C + d·μ_D − a·μ_A − b·μ_B
    = Σ ν_B μ_B° + RT Σ ν_B ln(p_B/p°)
    = ΔG° + RT ln Q

其中 ΔG° = Σ ν_B μ_B°（标准反应 Gibbs 自由能）
     Q = Π(p_B/p°)ν_B

平衡时 ΔG=0, Q=K°:    ΔG° = −RT ln K°    ← 核心公式
```

> 💡 **这条推导链的价值**：ΔG° = −RTlnK 不是"另一个公式"——它是从 dU=TdS−pdV → 四大公式 → G → 化学势 → 理想气体 μ(p) → 一路推导出来的**必然结论**。

**逸度与活度——非理想体系的修正**：

| | 理想体系 | 非理想体系 | 修正 |
|:---|:---|:---|:---|
| **气体** | μ=μ°+RTln(p/p°) | p 偏离理想 → **逸度 f=φp** | μ=μ°+RTln(f/p°) |
| **溶液** | μ=μ°+RTln(x) | x 偏离理想 → **活度 a=γx** | μ=μ°+RTln a |
| **电解质** | — | 离子间静电作用 → a=γ·c/c° | γ < 1（稀溶液）；Debye-Hückel 极限公式 |

> ⚠️ **关键理解**：逸度和活度不是"另一套理论"——它们只是**保持化学势形式统一**的修正因子。μ=μ°+RTln(f/p°) 与 μ=μ°+RTln(p/p°) 形式上完全一样。

> [!info]- 完整推导：化学势的名称由来和热力学定义群（课后复习用）
>
> **教材来源**：Atkins Ch6 §6.1-6.3、物理化学 §3.1-3.5
>
> **化学势的多种等价定义**：
> μ_B = (∂U/∂n_B)_{S,V,n_j} = (∂H/∂n_B)_{S,p,n_j} = (∂A/∂n_B)_{T,V,n_j} = (∂G/∂n_B)_{T,p,n_j}
> 四个偏导数在各自恒定条件下给出相同值——因为 U/H/A/G 之间只差 Legendre 变换。
>
> **化学势名称的由来**：
> Gibbs 用"势"（potential）这个词类比电势和重力势——物质从高 μ 处"流动"到低 μ 处，就像电流从高电势流到低电势。化学反应就是"化学势驱动的流动"。
>
> **与教材的对照说明**：
> - Atkins Ch6 §6.1 从混合 Gibbs 自由能引入化学势概念
> - 物理化学 §3.1-3.5 给出了化学势的完整推导链条
> - Zchem 教师的教学策略："把化学势理解为物质的分压贡献"——这在第一轮够用，第三轮需要完整的偏微分定义

**阶段性小结**：
> 化学势 μ = (∂G/∂n_B) 是连接热力学（G）和化学平衡（K）的桥梁。μ=μ°+RTln(p/p°) → Σνμ → ΔG=ΔG°+RTlnQ → 平衡时 ΔG°=−RTlnK。一条链下来，化学平衡的定量理论已经是热力学的必然推论。

### 2.5 van't Hoff 方程——温度如何改变平衡

> 你知道升温有利于吸热反应——但**定量上**，平衡常数随温度变化多少？

**三层递进**：

| 形式 | 公式 | 条件 | 用途 |
|:---|:---|:---|:---|
| **等温式** | ΔG = ΔG° + RT ln Q | 恒 T | 判断任意 Q 下的反应方向 |
| **微分式** | d lnK / dT = ΔH° / RT² | ΔH° 可以是 T 的函数 | 定性：吸热 → K↑；放热 → K↓ |
| **积分式** | ln(K₂/K₁) = −(ΔH°/R)(1/T₂ − 1/T₁) | **ΔH° ≈ 常数** | 已知 T₁ 的 K₁ → 求 T₂ 的 K₂ |

**微分式的推导（从 G-H 方程一步即得）**：

```
G-H 方程：∂(ΔG°/T)/∂T = −ΔH°/T²
代入 ΔG° = −RTlnK：∂(−RlnK)/∂T = −ΔH°/T²

→ d lnK / dT = ΔH° / RT²    ✓
```

**积分式的推导**：

```
假设 ΔH° 在 T₁→T₂ 范围内近似为常数：
∫ d lnK = (ΔH°/R) ∫ dT/T²
ln K₂ − ln K₁ = −(ΔH°/R)(1/T₂ − 1/T₁)

ln(K₂/K₁) = −(ΔH°/R)(1/T₂ − 1/T₁)
```

**ΔC_p ≠ 0 时的温度修正（初步）**：

如果 ΔC_p 在温度范围内变化不可忽略，则需要先求 ΔH°(T)：

```
ΔH°(T) = ΔH°(298) + ∫_{298}^{T} ΔC_p dT

再代入积分式：lnK(T) = lnK(298) + ∫_{298}^{T} (ΔH°(T)/RT²) dT
```

本课只要求学生理解修正的必要性，不要求完整计算 ΔC_p≠0 的积分。

> [!info]- 完整推导与教材对照（课后复习用）
>
> **教材来源**：Atkins Ch6 §6.4-6.5、物理化学 §3.8-3.10
>
> **van't Hoff 方程的历史**：
> Jacobus van't Hoff（第一位诺贝尔化学奖得主，1901年）在 1884 年出版的《Études de dynamique chimique》中首次提出了这个方程。它与 Arrhenius 方程（k=Ae^{−Ea/RT}）在形式上惊人地相似——温度对平衡（热力学）和速率（动力学）的影响都来自同一指数因子 e^{−E/RT}。
>
> **lnK vs 1/T 图的斜率**：
> 以 lnK 对 1/T 作图，斜率 = −ΔH°/R：
> - 吸热反应（ΔH°>0）：斜率为负 → 1/T↓（T↑）→ lnK↑ → K 随升温增大
> - 放热反应（ΔH°<0）：斜率为正 → 1/T↓（T↑）→ lnK↓ → K 随升温减小
>
> **与教材的对照说明**：
> - Atkins Ch6 §6.4-6.5 给出了 van't Hoff 方程的详细推导和应用例题
> - 物理化学 §3.8-3.10 以合成氨为经典案例，展示了 ΔH° 和 ΔG° 随温度的变化
> - Zchem 教师对 ΔC_p≠0 仅做定性介绍，定量积分公式来自 Atkins

**阶段性小结**：
> dlnK/dT = ΔH°/RT² ——吸热反应 K 随 T↑ 递增，放热反应 K 随 T↑ 递减。积分式 ln(K₂/K₁) = −(ΔH°/R)(1/T₂−1/T₁) 用于计算任意温度下的 K（前提：ΔH°≈常数）。

### 2.6 Clapeyron 方程——相变的"热力学定律"

> 相变（沸腾、熔化、升华）的条件是什么？Clapeyron 方程精确描述了相变线上 p 与 T 的关系。

**从化学势推导 Clapeyron 方程**：

```
两相平衡条件：μ_α(T,p) = μ_β(T,p)

沿相变线移动一小步：dμ_α = dμ_β
  −S_α dT + V_α dp = −S_β dT + V_β dp
  (S_β − S_α) dT = (V_β − V_α) dp

→ dp/dT = ΔS/ΔV = ΔH/(TΔV)     ← Clapeyron 方程（最一般形式）
```

**Clapeyron 方程对各类相变的预测**：

| 相变类型 | ΔV | ΔH | dp/dT 符号 | 实例/解释 |
|:---|:---|:---|:---:|:---|
| **固→液**（熔化） | V_l > V_s（通常） | >0 | **正**（通常） | 熔点随压力升高而升高 |
| 固→液（水例外） | V_l < V_s（水！） | >0 | **负** | 冰的熔点随压力升高而**降低**（滑冰原理） |
| **液→气**（蒸发） | V_g ≫ V_l | >0 | **正** | 沸点随压力升高而升高 |
| **固→气**（升华） | V_g ≫ V_s | >0 | **正** | 升华温度随压力升高而升高 |

**Clausius-Clapeyron 近似（气-液/气-固平衡）**：

```
气-液（或气-固）平衡：ΔV = V_g − V_l ≈ V_g = RT/p

Clapeyron: dp/dT = ΔH/(T·RT/p) = pΔH/(RT²)

→ d ln p / dT = ΔH / RT²     ← Clausius-Clapeyron 方程

积分（ΔH≈常数）：ln(p₂/p₁) = −(ΔH/R)(1/T₂ − 1/T₁)
```

**应用三例**：

1. **高原沸点**：Δ_vapH(H₂O)=40.7 kJ/mol，正常沸点 373K(1 atm)。珠峰顶 0.33 atm → ln(0.33/1)=−(40700/8.314)(1/T₂−1/373) → T₂≈344K=71°C
2. **压力锅原理**：2 atm 下水的沸点 → ln(2)=−(40700/8.314)(1/T₂−1/373) → T₂≈394K=121°C
3. **冷冻干燥**：冰的 Δ_subH≈51 kJ/mol，273K 时 p≈611 Pa。要在 −50°C 升华 → p≈4 Pa（高真空）

> [!info]- 完整推导：Clapeyron 方程的积分解（课后复习用）
>
> **教材来源**：Atkins Ch4 §4.3-4.4、物理化学 §4.1-4.3
>
> **不假设 ΔH 为常数的积分**：
> ln(p₂/p₁) = −(1/R)∫_{T₁}^{T₂} (ΔH/T²) dT
> 如果 ΔH(T) = ΔH(T₁) + ΔC_p(T−T₁)，则可以解析积分。但在竞赛范围内，通常假设 ΔH 为常数。
>
> **与 Antoine 方程的关系**：
> Clausius-Clapeyron 方程的对数线性形式 ln p = A − B/T 在窄温度范围内成立。工程上常用的 Antoine 方程 log p = A − B/(T+C) 增加了参数 C 来纠正宽温度范围的偏离。
>
> **与教材的对照说明**：
> - Atkins Ch4 §4.3-4.4 给出了 Clapeyron 和 Clausius-Clapeyron 方程的完整推导
> - 物理化学 §4.1-4.3 以水的相图为例展示了相变线的热力学分析

**阶段性小结**：
> Clapeyron: dp/dT=ΔH/(TΔV)（最一般形式，对任何相变成立）。Clausius-Clapeyron: dlnp/dT=ΔH/(RT²)（气-液/气-固限用）。水的 dp/dT 在固-液线上是负的——因为冰的体积大于水。

### 2.7 依数性的统一推导——一切从化学势出发

> 沸点升高、凝固点降低、渗透压——四个公式背后是同一条推导链：**溶质降低了溶剂的化学势**。

**统一推导链**：

```
① 溶质加入 → 溶剂摩尔分数 x_A < 1
② 溶剂化学势 μ_A(l) = μ_A°(l) + RT ln x_A  ← 因为 x_A<1，μ_A 下降
③ 为了保持两相平衡 μ_A(l) = μ_A(g) 或 μ_A(l) = μ_A(s)
   → 必须调整 T 或 p 来补偿 μ_A 的下降
④ 四种调整方式 → 四种依数性
```

**1. 蒸气压降低（Raoult 定律）**：

```
μ_A(l) = μ_A°(l) + RT ln x_A
μ_A(g) = μ_A°(g) + RT ln(p_A/p°)
平衡时相等 → p_A = x_A · p_A*    （Raoult定律，p_A* 为纯溶剂蒸气压）
Δp = p_A* − p_A = (1−x_A)p_A* = x_B · p_A*
```

**2. 沸点升高**：

```
气-液平衡：μ_A(g,T,p) = μ_A(l,T,p,x_A)
展开化学势 → Δ_vapG° + RT ln(p_A/p°) = RT ln x_A
结合 Clausius-Clapeyron → ΔT_b = K_b · m_B
其中 K_b = RT_b² M_A / Δ_vapH_A   ← 只与溶剂有关！
```

**3. 凝固点降低**：

```
液-固平衡：μ_A(s,T,p) = μ_A(l,T,p,x_A)
类似推导 → ΔT_f = K_f · m_B
其中 K_f = RT_f² M_A / Δ_fusH_A
```

**4. 渗透压**：

```
半透膜两侧：纯溶剂 μ_A*(T,p) = 溶液 μ_A(T,p+Π,x_A)
展开 → −S_A dT + V_A dp 中 dT=0 → V_A Π = −RT ln x_A ≈ RT x_B
稀溶液 x_B ≈ n_B/n_A → Π V = n_B RT → **Π = cRT**（van't Hoff 形式）
```

**四个依数性的对比**：

| 依数性 | 公式 | K 的表达式 | 只取决于 | 常见值（水） |
|:---|:---|:---|:---|:---:|
| **蒸气压降低** | Δp = x_B · p_A* | — | 溶剂的 p* | p*(H₂O,298K)=3.17 kPa |
| **沸点升高** | ΔT_b = K_b · m | RT_b² M_A / Δ_vapH_A | 溶剂 | K_b(H₂O)=0.512 K·kg/mol |
| **凝固点降低** | ΔT_f = K_f · m | RT_f² M_A / Δ_fusH_A | 溶剂 | K_f(H₂O)=1.86 K·kg/mol |
| **渗透压** | Π = cRT | — | —（普适） | 0.1M 溶液 Π≈2.4 atm |

> 💡 **关键理解**：四个依数性**不是四个不同的理论**——它们都来自同一个物理图像：溶质降低了溶剂的化学势 → 体系为了恢复平衡调整 T 或 p。

> [!info]- 完整推导：依数性的热力学证明（课后复习用）
>
> **教材来源**：Atkins Ch6 §6.6-6.8、物理化学 §3.6-3.7
>
> **渗透压的精确推导**：
> 从 μ_A*(T,p) = μ_A(T,p+Π,x_A) 出发：
> μ_A*(T,p) = μ_A°(T) + ∫_p^{p+Π} V_A dp ≈ μ_A°(T) + V_A Π
> μ_A(T,p+Π,x_A) = μ_A°(T) + V_A Π + RT ln x_A
> 等式 → V_A Π = −RT ln x_A
> 对稀溶液 −ln x_A = −ln(1−x_B) ≈ x_B ≈ n_B/n_A
> → Π = (n_B RT)/(n_A V_A) ≈ n_B RT / V = cRT
>
> **电解质溶液的修正**：
> 对于电解质（如 NaCl → Na⁺ + Cl⁻），van't Hoff 因子 i > 1：
> ΔT_f = i K_f m（如 NaCl i≈2，CaCl₂ i≈3）
> 这是 Arrhenius 在 1884 年提出电离理论的实验依据之一。
>
> **与教材的对照说明**：
> - Atkins Ch6 §6.6-6.8 给出了依数性的完整热力学推导
> - 物理化学 §3.6-3.7 以渗透压为重点，给出了精确的 V_A Π 展开

**阶段性小结**：
> 依数性 = 溶质降低溶剂化学势 × 四种平衡调整方式。ΔT_b=K_b·m, ΔT_f=K_f·m, Π=cRT 来自同一条化学势推导链。K_b 和 K_f 只取决于溶剂性质，不依赖溶质种类——这正是"依数性"（colligative properties）名称的由来。

### 2.8 热力学综合推断流程

> 将本节所有工具串联成完整的"数据→结论"推断链。

```text
已知数据：Δ_fH°(298K), S°(298K), C_p(T)

① ΔH°(298) = Σ ν_B Δ_fH°_B(298)          ← 标准反应焓
② ΔS°(298) = Σ ν_B S°_B(298)              ← 标准反应熵
③ ΔCp° = Σ ν_B C_p,B°                      ← 热容差

④ ΔH°(T) = ΔH°(298) + ∫ΔC_p° dT           ← 任意温度的反应焓
⑤ ΔS°(T) = ΔS°(298) + ∫(ΔC_p°/T) dT     ← 任意温度的反应熵

⑥ ΔG°(T) = ΔH°(T) − TΔS°(T)              ← 任意温度的 ΔG°
   或由 G-H 方程从 298K 积分得到

⑦ K(T) = exp(−ΔG°(T) / RT)                ← 任意温度的平衡常数

⑧ Q vs K(T) → 判断反应方向                 ← 等温式
⑨ 若涉及相变 → Clapeyron 方程给出 p(T)     ← 相变条件
⑩ 若涉及溶液 → 依数性给出 ΔT/Π            ← 溶液性质

终极公式三角：
  ΔG° = −RTlnK = −nFE°
  热力学    平衡    电化学
```

### 例题 1（⭐⭐ 基础）：四大基本公式推导

**题目**：已知 dU = TdS − pdV，推导 dH、dA、dG 的微分式，并写出各自对应的自然变量。

**分析提示**：
- H = U + pV，对两边取微分
- 将 dU = TdS − pdV 代入
- 同理推导 A = U − TS 和 G = H − TS

<details>
<summary>完整解答路径（点击展开）</summary>

dH = d(U + pV) = dU + pdV + Vdp = (TdS − pdV) + pdV + Vdp = TdS + Vdp → 自然变量 (S,p)

dA = d(U − TS) = dU − TdS − SdT = (TdS − pdV) − TdS − SdT = −SdT − pdV → 自然变量 (T,V)

dG = d(H − TS) = dH − TdS − SdT = (TdS + Vdp) − TdS − SdT = −SdT + Vdp → 自然变量 (T,p)
</details>

### 例题 2（⭐⭐ 基础）：van't Hoff 方程应用

**题目**：反应 N₂(g) + 3H₂(g) ⇌ 2NH₃(g)，已知 298K 时 ΔH° = −92.2 kJ/mol，K_p = 6.0×10⁵。
(1) 求 700K 时的 K_p（工业合成氨温度）。
(2) 解释为什么工业合成氨选择 700K 而非 298K 操作（尽管 298K 的 K 更大）。
(3) 如果 ΔC_p ≠ 0，定性说明实际 K_p(700K) 比 (1) 的结果偏大还是偏小？（NH₃ 的 C_p 大于 N₂ 和 H₂）

**分析提示**：
- 积分式：ln(K₂/K₁) = −(ΔH°/R)(1/T₂−1/T₁)
- 工业条件需要考虑速率和平衡的权衡

<details>
<summary>完整解答路径（点击展开）</summary>

(1) ln(K₇₀₀/6.0×10⁵) = −(−92200/8.314)(1/700 − 1/298)
    = (11087)(0.001429 − 0.003356) = (11087)(−0.001927) = −21.36
    K₇₀₀/6.0×10⁵ = e^{−21.36} = 5.3×10^{−10}
    → K₇₀₀ = 6.0×10⁵ × 5.3×10^{−10} = **3.2×10^{−4}**

(2) 尽管 298K 的 K 极大（6×10⁵），但**反应速率太慢**（N₂ 的 N≡N 键能 945 kJ/mol）。工业选择 700K 是为了获得足够的反应速率（动力学因素），同时用高压（~200 atm）和高浓度 N₂/H₂ 使 Q 远小于 K、用催化剂降低活化能。这是**热力学+动力学+工程**的综合优化。

(3) ΔC_p = 2C_p(NH₃) − C_p(N₂) − 3C_p(H₂)。因为 NH₃（四原子分子）的 C_p 大于 N₂ 和 H₂（双原子分子），ΔC_p < 0。这意味着 ΔH° 在高温下更负（放热更多）。更负的 ΔH° 意味着 dlnK/dT 的斜率更陡 → **K 下降得更快**，实际 K_p(700K) 比 (1) 中假设 ΔC_p=0 的结果还要小。
</details>

### 例题 3（⭐⭐ 基础）：Clausius-Clapeyron——高原沸点

**题目**：水的 Δ_vapH = 40.7 kJ/mol，正常沸点 100°C。
(1) 求拉萨（海拔 3650 m，大气压 ~0.65 atm）水的沸点。
(2) 为什么高压锅可以"把饭煮熟"？2.0 atm 压力锅内的水温是多少？
(3) 从 Clausius-Clapeyron 方程定性解释：为什么 Δ_vapH 大的液体沸点随海拔变化更剧烈？

<details>
<summary>完整解答路径（点击展开）</summary>

(1) ln(0.65/1) = −(40700/8.314)(1/T₂ − 1/373)
    ln(0.65) = −0.4308 = −4895(1/T₂ − 0.002681)
    1/T₂ = 0.002681 − 0.4308/4895 = 0.002681 − 0.0000880 = 0.002593
    T₂ = **386 K = 113°C**
    （注：实际拉萨沸点 ~90°C，因为这里用 Δ_vapH 常数近似有偏差）

(2) ln(2/1) = 0.6931 = −4895(1/T₂ − 0.002681)
    1/T₂ = 0.002681 − 0.6931/4895 = 0.002681 − 0.0001416 = 0.002539
    T₂ = **394 K = 121°C** ✓

(3) ln(p₂/p₁) = −(Δ_vapH/R)(1/T₂−1/T₁)
    对给定的 Δp，Δ_vapH 越大 → |ln(p₂/p₁)| 越大 → |1/T₂−1/T₁| 越大 → T 变化更大
    即**蒸发焓大的液体对压力更敏感**（如水的 Δ_vapH=40.7 > 乙醇的 38.6 > 乙醚的 26.0 kJ/mol）
</details>

### 例题 4（⭐⭐⭐ 拓展）：热力学综合推断

**题目**：石灰石分解反应 CaCO₃(s) ⇌ CaO(s) + CO₂(g)，已知：
- Δ_fH°(CaCO₃) = −1207 kJ/mol, S°(CaCO₃) = 93 J/(mol·K)
- Δ_fH°(CaO) = −635 kJ/mol, S°(CaO) = 40 J/(mol·K)
- Δ_fH°(CO₂) = −394 kJ/mol, S°(CO₂) = 214 J/(mol·K)

(1) 求 298K 时的 ΔG° 和 K_p。
(2) 求 p_CO₂ = 1 atm（即分解产物 CO₂ 压力等于大气压）时的最低分解温度。
(3) 空气中 CO₂ 含量约 0.04%（p_CO₂ ≈ 4×10⁻⁴ atm）。常温下 CaO 是否会被空气中的 CO₂ 转化为 CaCO₃？

<details>
<summary>完整解答路径（点击展开）</summary>

(1) ΔH° = (−635−394) − (−1207) = +178 kJ/mol
    ΔS° = (40+214) − 93 = +161 J/(mol·K) = 0.161 kJ/(mol·K)
    ΔG°(298) = 178 − 298×0.161 = 178 − 48.0 = **+130 kJ/mol**
    K_p(298) = exp(−130000/(8.314×298)) = exp(−52.5) ≈ **1.6×10⁻²³**
    结论：常温下几乎不分解 ✓

(2) 分解条件：ΔG = ΔG° + RT ln(p_CO₂) ≤ 0，即 ΔG° ≤ −RT ln(p_CO₂)
    当 p_CO₂ = 1 atm 时，需要 ΔG° ≤ 0
    ΔG°(T) ≈ ΔH°(298) − TΔS°(298) = 178000 − T×161 ≤ 0
    T ≥ 178000/161 = **1106 K ≈ 833°C**

(3) p_CO₂ = 4×10⁻⁴ atm → Q = 4×10⁻⁴
    ΔG(298) = ΔG°(298) + RT ln Q = 130000 + 8.314×298×ln(4×10⁻⁴)
    = 130000 + 2477×(−7.82) = 130000 − 19370 = +110600 J/mol > 0
    结论：常温下自发方向是 CaO + CO₂ → CaCO₃（ΔG>0 表示分解方向非自发，即化合方向自发！）→ 生石灰在空气中会**逐渐变质**为 CaCO₃ ✓
</details>

### 例题 5（⭐⭐⭐ 拓展）：依数性综合

**题目**：某蛋白质的摩尔质量通过渗透压法测定。将 0.150 g 蛋白质溶于水制成 10.0 mL 溶液，在 25°C 测得渗透压为 2.36 kPa。
(1) 求该蛋白质的摩尔质量。
(2) 同一浓度的该蛋白质水溶液，其凝固点降低和沸点升高分别是多少（K_f=1.86, K_b=0.512 K·kg/mol）？
(3) 为什么用渗透压法测蛋白质摩尔质量比凝固点降低法更精确？

<details>
<summary>完整解答路径（点击展开）</summary>

(1) Π = cRT → c = Π/(RT) = 2360/(8.314×298) = 0.952 mol/m³ = 9.52×10⁻⁴ mol/L
    10.0 mL 中含 n = 9.52×10⁻⁴ × 0.0100 = 9.52×10⁻⁶ mol
    M = m/n = 0.150/9.52×10⁻⁶ = **1.58×10⁴ g/mol ≈ 15.8 kDa**

(2) 质量摩尔浓度 m ≈ 9.52×10⁻⁴ mol / 0.0100 kg = 0.0952 mol/kg
    ΔT_f = 1.86×0.0952 = **0.177°C**（极难准确测量！）
    ΔT_b = 0.512×0.0952 = **0.0487°C**（几乎测不到！）

(3) 凝固点降低 0.177°C 很难用普通温度计准确测量，过冷现象也会干扰。而渗透压 2.36 kPa ≈ 18 mmHg，用压力计可精确测量。且渗透压对稀溶液的响应比凝固点降低灵敏得多（Π∝c 而 ΔT_f 的绝对数值很小）。
</details>

## 五、本节形成的核心结论

1. **热力学是一棵"推导树"**：从 dU=TdS−pdV（热一+热二的组合定律）出发 → Legendre 变换 → H/A/G → 四大基本公式 → Maxwell 关系 → G-H 方程 → 化学势 → 平衡/相变/依数性。不需要记忆 100 条公式。

2. **化学势是统一框架**：μ = (∂G/∂n_B) → μ=μ°+RTln(p/p°) → ΔG=ΔG°+RTlnQ → ΔG°=−RTlnK。化学平衡、相变条件（μ_α=μ_β）、依数性（μ 下降→T/p 调整）全都是"化学势相等"的应用。

3. **van't Hoff 方程定量描述温度效应**：dlnK/dT=ΔH°/RT²（微分式）→ ln(K₂/K₁)=−(ΔH°/R)(1/T₂−1/T₁)（积分式）。吸热反应 K↑，放热反应 K↓。

4. **Clapeyron 方程描述相变**：dp/dT=ΔH/(TΔV)（普遍）；Clausius-Clapeyron dlnp/dT=ΔH/(RT²)（气-液/气-固限用）。水的固-液线斜率为负（V_ice > V_water）。

5. **依数性来自同一推导链**：ΔT_b=K_b·m, ΔT_f=K_f·m, Π=cRT——全都可以从"溶质降低溶剂化学势"一条推导链得到。K_b 和 K_f 只取决于溶剂。

## 六、⚠️ 易错点预警

| 错误想法 | 为什么错 | 正确理解 |
|:---|:---|:---|
| "Legendre 变换就是换符号" | Legendre 变换不仅换符号，还换**自变量**。dU=TdS−pdV 中自变量是 (S,V)，dG=−SdT+Vdp 中自变量是 (T,p) | Legendre 变换 = "减共轭积"换自变量（如 G=U−TS+pV 同时换 S→T 和 V→p） |
| "dU=TdS−pdV 对所有过程都成立" | 只有在**可逆**且**只有体积功**时每个增量才是严格可逆路径的增量 | 但 U 是状态函数，ΔU 与路径无关——可以用可逆路径的积分计算任何过程的 ΔU |
| "H 的定义是 ΔH=Q_p" | ΔH=Q_p 是**恒压无其他功时的结论**，不是定义。H 的定义是 H=U+pV | 先有定义 H=U+pV，才推导出恒压下 dH=δQ_p |
| "van't Hoff 积分式总是可以用的" | 积分式假设 ΔH° 在 T₁→T₂ 范围内**为常数**。如果温度跨度大或 ΔC_p 不可忽略，需要先积分 ΔH°(T) | 先判断 ΔC_p 是否可忽略（通常 ΔT < 200K 时 ΔC_p 修正约 5-10%） |
| "Clausius-Clapeyron 对所有相变都成立" | C-C 方程假设**ΔV≈V_g=RT/p**，只适用于气-液或气-固平衡 | 固-液平衡用 dp/dT=Δ_fusH/(TΔ_fusV)（完整 Clapeyron），不能近似 |
| "依数性只取决于溶质的量，与种类完全无关" | "依数性"来自 ideal dilute solution 近似——只对非电解质稀溶液严格成立 | 电解质需乘以 van't Hoff 因子 i；浓溶液中活度系数 γ 偏离 1 |
| "G-H 方程和 van't Hoff 方程是两回事" | van't Hoff 方程就是 G-H 方程代入 ΔG°=−RTlnK 的结果 | dlnK/dT = ΔH°/RT² 是从 ∂(ΔG°/T)/∂T = −ΔH°/T² 代入一步得到的 |
