## 一、课堂引入与本节问题

### 1.1 课堂切入

**问题情境**：

> 实验室有两瓶无色溶液，一瓶是Na₂SO₃，一瓶是Na₂S₂O₃。向其中分别滴加稀盐酸，一瓶产生无色刺激性气体，另一瓶既产生刺激性气体又出现黄色浑浊。你能解释现象差异吗？写出对应的离子方程式。

**思考方向**：
- 为什么SO₃²⁻遇酸只放出SO₂，而S₂O₃²⁻遇酸既放出SO₂又析出S？
- S₂O₃²⁻中硫的氧化态是多少？这个氧化态稳定吗？
- 这体现了硫元素的什么核心规律？

### 1.2 本节深度边界

| 必须掌握 | 点到为止 | 暂不展开 |
|----------|----------|----------|
| 硫/氮/磷重要化合物的性质与反应 | NO + O₂ → NO₂的平衡特征（不写K） | 氮氧化物平衡热力学的完整推导 |
| 浓/稀HNO₃还原产物的区分 | 过二硫酸盐是强氧化剂（不写机理） | 磷化合物的群论对称性分析 |
| S₂O₃²⁻歧化反应的理解与配平 | 肼是强还原剂（不画电位图） | 硫循环工业制硫酸的完整流程计算 |
| 磷的含氧酸缩合规律 | P₄O₁₀的结构特点（不展开Td对称性） | 过二硫酸盐的详细氧化机理 |

## 二、知识内容

### 2.1 硫的化合物（第1课时，45 min）

#### 从问题出发

**问题**：氧族元素从O→S→Se→Te，非金属性递减。硫作为氧的"下一行邻居"，其化合物在结构和性质上与氧有哪些相似与差异？

#### 2.1.1 H₂S：弱酸性与强还原性

**关键观察**：
- H₂S水溶液呈弱酸性（二元弱酸，分步电离：Kₐ₁ ≈ 1.3×10⁻⁷）
- 具有臭鸡蛋气味，有毒
- **强还原性**：能使KMnO₄褪色、能使I₂还原为I⁻

**核心方程式**：

$$
\ce{Cl2 + H2S -> S↓ + 2HCl}
$$

$$
\ce{2H2S + SO2 -> 3S↓ + 2H2O}
$$

> **结构-性质关联**：S-H键键能（363 kJ/mol）小于O-H键（463 kJ/mol），H₂S更易失电子，还原性强于H₂O。

#### 2.1.2 SO₂与SO₃

| 性质 | SO₂ | SO₃ |
|------|-----|-----|
| 结构 | V型（sp²杂化，含孤对电子） | 平面三角形（sp²杂化，无孤对电子） |
| 氧化态 | S为+4（中间态） | S为+6（最高价） |
| 氧化还原性 | **既有氧化性又有还原性** | 只有氧化性（较弱） |
| 典型反应 | 使品红褪色（漂白性） | 与水剧烈反应生成H₂SO₄ |
| 酸性氧化物 | 是 | 是（酸性更强） |

**SO₂的氧化还原双重性**：
- 还原性（主要）：$\ce{SO2 + Cl2 + 2H2O -> H2SO4 + 2HCl}$
- 氧化性（次要）：$\ce{SO2 + 2H2S -> 3S↓ + 2H2O}$

> **催化氧化**（点到为止）：$\ce{2SO2 + O2 <=>[V2O5][△] 2SO3}$，不展开平衡计算。

#### 2.1.3 硫酸与硫代硫酸盐

**浓H₂SO₄的三大特性**：

| 特性 | 本质 | 典型实例 |
|------|------|----------|
| **强氧化性** | S为+6，可被还原 | $\ce{Cu + 2H2SO4(浓) ->[△] CuSO4 + SO2↑ + 2H2O}$ |
| **脱水性** | 按H:O=2:1夺水 | 使蔗糖炭化、使纸张变黑 |
| **吸水性** | 形成水合物 | 作干燥剂（不能干燥NH₃、H₂S等） |

**核心突破：S₂O₃²⁻的歧化反应**

> **实验事实**：Na₂S₂O₃溶液中滴加稀盐酸，溶液变浑浊（生成S单质），同时产生刺激性气体（SO₂）。

**S₂O₃²⁻中硫的氧化态分析**：
- 设两个S的氧化态分别为x和y，则 x + y = +4
- 实验事实表明产物为S（0价）和SO₂（S为+4价）
- 因此一个S从+2→0（被还原），另一个S从+2→+4（被氧化）
- 平均氧化态为+2，实际为**不对称结构**（中心S为+5，端基S为-1，但中学阶段记平均+2即可）

**离子方程式**：

$$
\ce{S2O3^{2-} + 2H+ -> S↓ + SO2↑ + H2O}
$$

> [!info]- 完整推导与教材对照
> **氧化态法配平**：
> 1. S₂O₃²⁻中S平均氧化态为+2
> 2. 产物S氧化态为0，SO₂中S氧化态为+4
> 3. 一个S：+2 → 0，得2e⁻；另一个S：+2 → +4，失2e⁻
> 4. 电子得失相等，S₂O₃²⁻系数为1，S和SO₂系数各为1
> 5. 电荷守恒：左边-2+2=0，右边0，已平衡
> 6. 原子守恒：补H₂O
>
> **教材对照**：人教版《化学反应原理》将S₂O₃²⁻结构描述为[SSO₃]²⁻，中心S类似SO₄²⁻中的S，端基S具有还原性。与酸反应时，端基S被"释放"为单质S，中心S转化为SO₂。

**过二硫酸盐S₂O₈²⁻**（点到为止）：
- 含过氧键（-O-O-），强氧化剂
- 典型反应：$\ce{S2O8^{2-} + 2Mn^{2+} + 8H2O -> 2SO4^{2-} + 2MnO4^- + 16H+}$（需Ag⁺催化）

#### 2.1.4 结构对比：SO₃²⁻ vs SO₄²⁻

| 对比项 | SO₃²⁻ | SO₄²⁻ |
|--------|-------|-------|
| 中心原子杂化 | sp³ | sp³ |
| 空间构型 | **三角锥**（含1对孤对电子） | **正四面体**（无孤对电子） |
| S-O键长 | 较长 | 较短 |
| 稳定性 | 较弱，易被氧化 | 稳定，难被氧化 |
| 还原性 | **强**（可被O₂、Cl₂等氧化） | **极弱**（S已达最高价+6） |

> **结论**：孤对电子的存在使SO₃²⁻结构不稳定，S原子更易被氧化；SO₄²⁻对称性高，S处于最高价，化学性质稳定。

#### 阶段性小结

**硫的氧化态图谱**：

```
氧化态:  -2      0      +4      +6
        H₂S ──→ S ──→ SO₂ ──→ SO₃/H₂SO₄
         ↑      ↑      ↑        ↑
       还原性  单质   两性    氧化性
       最强   稳定   氧化     为主
```

**🔷 拓展**：硫代硫酸盐在碘量法中的应用

$$
\ce{2S2O3^{2-} + I2 -> S4O6^{2-} + 2I-}
$$

- S₂O₃²⁻在此反应中作还原剂，但产物不是S和SO₂，而是连四硫酸根S₄O₆²⁻
- 这是**定量反应**，用于测定I₂的含量（碘量法基础）
- 注意：S₂O₃²⁻与I₂反应≠与酸反应，条件不同产物不同

### 2.2 氮的化合物（第2课时，45 min）

#### 从问题出发

**问题**：氮的氧化态从-3（NH₃）到+5（HNO₃），跨度极大。如何利用"氧化态梯度"理解氮化合物的性质递变？

#### 2.2.1 NH₃与N₂H₄

**NH₃的核心性质**：
- **弱碱性**：$\ce{NH3 + H2O <=> NH4+ + OH-}$（K_b ≈ 1.8×10⁻⁵）
- **还原性**：$\ce{4NH3 + 5O2 ->[Pt][△] 4NO + 6H2O}$（催化氧化，工业制硝酸基础）
- **配位性**：孤对电子可与金属离子配位，如$\ce{[Cu(NH3)4]^{2+}}$

**N₂H₄（联氨/肼）的双重身份**（难点）：

> **问题**：N₂H₄是几元碱？若为一元碱，与HCl反应的产物是什么？若为二元碱呢？

**实验事实**：N₂H₄与过量HCl反应生成N₂H₆Cl₂，说明是**二元弱碱**。

**电离方程式**：

$$
\ce{N2H4 + H2O <=> N2H5+ + OH-} \quad K_{b1} \approx 8.5\times10^{-7}
$$

$$
\ce{N2H5+ + H2O <=> N2H6^{2+} + OH-} \quad K_{b2} \approx 8.9\times10^{-16}
$$

**强还原性**（火箭燃料应用）：

$$
\ce{N2H4 + O2 -> N2 + 2H2O} \quad \Delta H = -622 \text{ kJ/mol}
$$

$$
\ce{N2H4 + 2H2O2 -> N2 + 4H2O}
$$

> **数值冲击**：N₂H₄燃烧热极高，产物为N₂和H₂O，无污染，是理想的高能清洁燃料。

#### 2.2.2 HN₃（叠氮酸）

- **不稳定，易爆**，撞击或加热易分解
- **弱酸性**：$\ce{HN3 <=> H+ + N3-}$（Kₐ ≈ 1.9×10⁻⁵）
- 与金属反应生成叠氮化物（如NaN₃，汽车安全气囊产气剂）
- **点到为止**：不展开N₃⁻的分子轨道分析

#### 2.2.3 NOₓ系列

| 氧化物 | 颜色 | 关键性质 | 与水的反应 |
|--------|------|----------|------------|
| NO | **无色** | 不成盐氧化物，遇空气变红棕 | 不反应 |
| NO₂ | **红棕色** | 强氧化性，有毒 | $\ce{3NO2 + H2O -> 2HNO3 + NO}$ |
| N₂O₄ | **无色** | NO₂的二聚体，低温稳定 | — |

**NO与NO₂的转化**：

$$
\ce{2NO + O2 -> 2NO2} \quad \text{（无色→红棕，实验鉴别特征）}
$$

$$
\ce{2NO2 <=> N2O4} \quad \text{（放热，低温有利于N₂O₄）}
$$

#### 2.2.4 HNO₃的性质（核心重点）

**浓/稀HNO₃与Cu反应对比**（必须掌握）：

| 条件 | 浓HNO₃ | 稀HNO₃ |
|------|--------|--------|
| 反应方程式 | $\ce{Cu + 4HNO3(浓) -> Cu(NO3)2 + 2NO2↑ + 2H2O}$ | $\ce{3Cu + 8HNO3(稀) -> 3Cu(NO3)2 + 2NO↑ + 4H2O}$ |
| N的还原产物 | **NO₂**（红棕色） | **NO**（无色，遇空气变红棕） |
| N的氧化态变化 | +5 → +4（降1） | +5 → +2（降3） |
| 氧化性强弱 | **强**（浓度高，氧化性强） | **较强**（浓度低，还原产物还原性更强） |
| 反应剧烈程度 | 剧烈 | 较缓和 |

**氧化态解释**：
- 浓HNO₃中，HNO₃浓度高，氧化能力强，但反应速率快，N只被还原到+4（NO₂）
- 稀HNO₃中，HNO₃浓度低，氧化能力相对弱，但反应中N可被进一步还原到+2（NO）
- **极稀HNO₃**与活泼金属反应，N可被还原到-3（NH₄⁺）：$\ce{4Zn + 10HNO3(极稀) -> 4Zn(NO3)2 + NH4NO3 + 3H2O}$

**钝化现象**：
- Fe、Al、Cr遇**冷浓HNO₃**表面形成致密氧化膜，阻止进一步反应
- **加热时钝化膜被破坏**，反应继续进行

#### 2.2.5 硝酸盐热分解规律

| 金属活泼性 | 分解产物类型 | 实例 |
|------------|--------------|------|
| 活泼金属（K→Na） | 亚硝酸盐 + O₂ | $\ce{2NaNO3 ->[△] 2NaNO2 + O2↑}$ |
| 中等活泼金属（Mg→Cu） | 金属氧化物 + NO₂ + O₂ | $\ce{2Cu(NO3)2 ->[△] 2CuO + 4NO2↑ + O2↑}$ |
| 不活泼金属（Hg→Ag） | 金属单质 + NO₂ + O₂ | $\ce{2AgNO3 ->[△] 2Ag + 2NO2↑ + O2↑}$ |
| NH₄NO₃ | 特殊分解 | 低温：$\ce{NH4NO3 ->[△] N2O + 2H2O}$；高温/撞击：$\ce{2NH4NO3 -> 2N2↑ + O2↑ + 4H2O}$ |

> [!info]- 完整推导与教材对照
> **NH₄NO₃分解的两种途径**：
> 1. **低温（170-240℃）**：质子转移反应，N₂O是产物
>    - NH₄⁺中的H转移至NO₃⁻的O上，形成HNO₃和NH₃，随后反应生成N₂O
>    - $\ce{NH4NO3 -> N2O + 2H2O}$
> 2. **高温或撞击**：氧化还原反应，N的自身氧化还原
>    - NH₄⁺中N（-3）被氧化，NO₃⁻中N（+5）被还原，最终都变为N₂（0）
>    - $\ce{2NH4NO3 -> 2N2↑ + O2↑ + 4H2O}$（爆炸性分解）
>
> **教材对照**：人教版高中化学必修2提到NH₄NO₃受热分解，但不区分温度；竞赛层面需要掌握温度对产物的影响。

#### 阶段性小结

**氮的氧化态图谱**：

```
氧化态:  -3     0      +2      +4      +5
        NH3 ──→ N2 ──→ NO ──→ NO2 ──→ HNO3
         ↑      ↑       ↑       ↑        ↑
       还原性  稳定   不成盐  酸性氧化  强氧化
       最强                    物       性酸
```

### 2.3 磷的化合物（第3课时前段，30 min）

#### 从问题出发

**问题**：磷与氮同族，但P₄分子结构、PH₃碱性、磷含氧酸缩合规律都与氮有明显差异。为什么同族元素会有这些差异？

#### 2.3.1 P₄与PH₃

**P₄（白磷）**：
- **正四面体结构**，键角60°（张力大，不稳定）
- **剧毒**，约50 mg可致死
- **自燃**（空气中约30℃自燃），需保存在水中
- 红磷、黑磷是更稳定的同素异形体

**PH₃（膦）**：
- **三角锥结构**（类似NH₃）
- **极毒**，大蒜气味
- **弱碱性**：碱性远弱于NH₃（因P电负性小于N，孤对电子更易给出？不，实际上PH₃碱性弱于NH₃，因P-H键极性小，且P原子半径大，孤对电子分散）
- **强还原性**：$\ce{PH3 + 2O2 -> H3PO4}$

**制备**：

$$
\ce{P4 + 3OH- + 3H2O -> PH3↑ + 3H2PO2-}
$$

> 注意：此反应中P既被氧化又被还原（歧化），生成PH₃（P为-3）和H₂PO₂⁻（P为+1）。

#### 2.3.2 P₄O₁₀

- **最强干燥剂之一**，吸水能力极强
- 与水反应：$\ce{P4O10 + 6H2O -> 4H3PO4}$
- 结构特点（点到为止）：含P=O双键和P-O-P桥键，不展开Td对称性分析

#### 2.3.3 磷的含氧酸缩合规律（核心重点）

**缩合规律**：正磷酸 → 脱水 → 焦磷酸 → 继续脱水 → 偏磷酸

| 名称 | 化学式 | 结构特点 | 形成方式 |
|------|--------|----------|----------|
| **正磷酸** | H₃PO₄ | 单个PO₄四面体 | 基础单元 |
| **焦磷酸** | H₄P₂O₇ | 两个PO₄四面体共用一个O | 2H₃PO₄ → H₄P₂O₇ + H₂O |
| **偏磷酸** | HPO₃ | 多个PO₄四面体共用两个O，形成链状或环状 | nH₃PO₄ → (HPO₃)_n + nH₂O |

**缩合通式**：

$$
\ce{nH3PO4 -> H_{n+2}P_nO_{3n+1} + (n-1)H2O}
$$

- n=1：H₃PO₄（正磷酸）
- n=2：H₄P₂O₇（焦磷酸）
- n→∞：(HPO₃)_n（偏磷酸，玻璃状固体）

> **类比记忆**：硅酸的缩合规律类似，H₄SiO₄ → 多硅酸。主族元素含氧酸的脱水缩合是普遍规律。

**PO₄³⁻结构**：正四面体，与SO₄²⁻类似，但P为+5价，无孤对电子，结构稳定。

#### 阶段性小结

**磷含氧酸缩合链**：

```
H3PO4 ──脱水──→ H4P2O7 ──继续脱水──→ (HPO3)n
(正磷酸)         (焦磷酸)              (偏磷酸)
  n=1            n=2                  n→∞
```

## 三、课堂例题

### ⭐⭐ 基础题

#### 基础1：Cu与浓/稀HNO₃的反应

**题目**：写出Cu与浓HNO₃、稀HNO₃反应的化学方程式，并比较还原产物的差异。解释为什么浓HNO₃的还原产物是NO₂，而稀HNO₃是NO。

**解答**：

$$
\ce{Cu + 4HNO3(浓) -> Cu(NO3)2 + 2NO2↑ + 2H2O}
$$

$$
\ce{3Cu + 8HNO3(稀) -> 3Cu(NO3)2 + 2NO↑ + 4H2O}
$$

**解释**：浓HNO₃氧化能力强，但反应中NO₃⁻浓度高，生成的NO来不及进一步反应就逸出为NO₂；稀HNO₃中NO₃⁻浓度低，反应生成的NO有机会与未反应的HNO₃接触被氧化为NO₂，但由于体系中HNO₃少，最终主要产物为NO。从氧化态角度，浓HNO₃中N从+5降到+4（NO₂），稀HNO₃中N从+5降到+2（NO）。

---

#### 基础2：S₂O₃²⁻与酸反应的歧化机理

**题目**：向Na₂S₂O₃溶液中滴加稀盐酸，观察到黄色浑浊和刺激性气体。写出离子方程式，并标出电子转移。

**解答**：

$$
\ce{S2O3^{2-} + 2H+ -> S↓ + SO2↑ + H2O}
$$

**电子转移分析**：
- S₂O₃²⁻中S平均氧化态为+2
- 产物S中S氧化态为0（得2e⁻）
- 产物SO₂中S氧化态为+4（失2e⁻）
- 电子得失相等，属于**歧化反应**

---

#### 基础3：磷含氧酸的缩合关系

**题目**：正磷酸H₃PO₄加热脱水可生成焦磷酸H₄P₂O₇，继续脱水生成偏磷酸HPO₃。写出缩合反应方程式，并说明n分子正磷酸缩合的通式。

**解答**：

$$
\ce{2H3PO4 ->[△] H4P2O7 + H2O}
$$

$$
\ce{nH3PO4 -> (HPO3)n + nH2O} \quad (n \geq 3)
$$

**通式**：$\ce{nH3PO4 -> H_{n+2}P_nO_{3n+1} + (n-1)H2O}$

- n=1：H₃PO₄
- n=2：H₄P₂O₇
- n→∞：(HPO₃)_n

---

#### 基础4：NH₄NO₃热分解

**题目**：NH₄NO₃在不同条件下分解产物不同。写出低温分解和高温/撞击分解的方程式，并说明产物差异的原因。

**解答**：

**低温（170-240℃）**：

$$
\ce{NH4NO3 ->[△] N2O + 2H2O}
$$

**高温或撞击**：

$$
\ce{2NH4NO3 -> 2N2↑ + O2↑ + 4H2O}
$$

**原因**：低温下主要是质子转移和分子内重排；高温下发生氧化还原反应，NH₄⁺中N（-3）与NO₃⁻中N（+5）发生归中反应生成N₂（0），同时O被释放为O₂。

### ⭐⭐⭐ 拓展题

#### 拓展1：37届Q5切片（NO/H₂S反应）

**题目**（简化）：某混合气体含NO和H₂S，在一定条件下反应生成N₂和S。写出配平的化学方程式，并计算反应中电子转移总数。

**解答**：

**步骤1：确定氧化产物和还原产物**
- H₂S中S为-2价，被氧化为S（0价），失2e⁻/S
- NO中N为+2价，被还原为N₂（0价），得2e⁻/N（即4e⁻/N₂）

**步骤2：电子守恒配平**
- 2个S失4e⁻，1个N₂得4e⁻
- 因此2H₂S对应2NO

**配平方程式**：

$$
\ce{2NO + 2H2S -> N2 + 2S↓ + 2H2O}
$$

**电子转移**：每2 mol NO参与反应，转移4 mol电子。

#### 拓展2：38届Q1切片（炼丹元素推断）

**题目**（简化）：某化合物A加热分解生成气体B和固体C。B能使品红溶液褪色，C溶于水得蓝色溶液。推断A、B、C可能的化学式，并写出分解方程式。

**解答**：

**推断过程**：
1. B能使品红褪色 → B含SO₂（硫的化合物特征）
2. C溶于水得蓝色溶液 → C含Cu²⁺（铜盐特征）
3. A加热分解生成SO₂和Cu的化合物 → A可能为CuSO₄·5H₂O或CuSO₃

**最可能答案**：
- A = CuSO₄·5H₂O（五水硫酸铜，蓝色晶体）
- B = SO₃（或SO₂，取决于分解条件）
- C = CuO（黑色）或CuSO₄（白色，无水）

**分解方程式**：

$$
\ce{CuSO4·5H2O ->[△] CuSO4 + 5H2O} \quad \text{（低温脱水）}
$$

$$
\ce{CuSO4 ->[强热] CuO + SO3↑} \quad \text{（高温分解）}
$$

> 注：若A为CuSO₃，则$\ce{CuSO3 ->[△] CuO + SO2↑}$，但CuSO₃不稳定，常见的是CuSO₄·5H₂O。

<details>
<summary>参考答案</summary>

</details>

$$
\ce{P4 + 3OH- + 3H2O -> PH3↑ + 3H2PO2-}
$$

P的氧化态变化：P₄（0）→ PH₃（-3，被还原）和H₂PO₂⁻（+1，被氧化）。

</details>

## 五、本节形成的核心结论

| 主题 | 核心结论 |
|------|----------|
| **硫化合物** | 结构决定性质：SO₃²⁻（三角锥，含孤对电子）还原性强；SO₄²⁻（正四面体，S最高价）稳定。S₂O₃²⁻遇酸歧化生成S+SO₂。 |
| **氮化合物** | 氧化态梯度驱动性质递变：NH₃(-3)强还原 → N₂(0)稳定 → NO(+2)不成盐 → NO₂(+4)强氧化 → HNO₃(+5)强氧化性酸。浓HNO₃→NO₂，稀HNO₃→NO。 |
| **磷化合物** | 含氧酸缩合规律：nH₃PO₄ → H_{n+2}P_nO_{3n+1} + (n-1)H₂O。正→焦→偏，脱水程度递增。 |
| **综合方法** | 元素化学分析三维框架：**族内递变**（同族上下对比）+ **氧化态梯度**（同一元素不同价态）+ **结构对比**（空间构型决定稳定性）。 |

## 六、⚠️ 易错点预警

| 易错点 | 错误表现 | 正确理解 |
|--------|----------|----------|
| **浓/稀HNO₃还原产物** | 认为稀HNO₃还原产物也是NO₂ | 浓→NO₂，稀→NO，极稀→NH₄⁺ |
| **S₂O₃²⁻与酸反应** | 只写SO₂不写S，或配平错误 | S₂O₃²⁻ + 2H⁺ → S↓ + SO₂↑ + H₂O（歧化） |
| **N₂H₄的碱性** | 认为N₂H₄是一元弱碱 | N₂H₄是**二元弱碱**，可形成N₂H₆²⁺ |
| **钝化条件** | 认为Al/Fe遇任何HNO₃都钝化 | 仅**冷浓HNO₃**钝化，加热或稀HNO₃不钝化 |
| **NH₄NO₃分解** | 认为只有一种分解方式 | 低温→N₂O+2H₂O；高温→N₂+O₂+2H₂O |
| **Cu与浓HNO₃计量关系** | 配平时Cu与HNO₃比例记错 | Cu + 4HNO₃(浓) → Cu(NO₃)₂ + 2NO₂ + 2H₂O（1:4） |
| **Cu与稀HNO₃计量关系** | 配平时Cu与HNO₃比例记错 | 3Cu + 8HNO₃(稀) → 3Cu(NO₃)₂ + 2NO + 4H₂O（3:8） |
| **PH₃碱性** | 认为PH₃碱性强于NH₃ | PH₃碱性**弱于**NH₃（P电负性小但半径大，孤对电子分散） |
