# 元素推断综合训练

**课型**：新授课（方法课+实战课）　**轮次**：第二轮　**时长**：90 min（2课时×45 min）

**前置知识**：主族元素化学、过渡金属通性、氧化还原、配合物基础

**对应备课大纲**：[[04-课件/备课大纲/2026-06-03-元素推断综合训练-提高班]]

## 一、课堂引入与本节问题

> **问题情境**：你拿到一道竞赛推断题，题目给出5-6条实验现象，你能快速提取有效信号、建立完整推理链吗？

**学生常见问题**：
- 知识都有，但面对题目时"无从下手"
- 能猜出答案，但写不出"因为…所以…"的推理过程
- 答案对了，但过程分丢了一大半

**本节核心任务**：把已学的零散元素化学知识，通过"信号法"工具化，训练"现象→推断→验证"的完整推理链。

## 二、知识内容

### 2.1 颜色信号法

**从问题出发**：看到溶液颜色，你能确定是什么离子吗？

**关键推理/观察**：
- 颜色是最直观的信号，但多数颜色**不具有唯一性**
- 必须结合其他信号交叉验证

| 颜色 | 可能离子 | 唯一性 | 需验证信号 |
|------|---------|--------|-----------|
| 蓝色 | Cu²⁺ | ✓ 较唯一 | 加NaOH→蓝色沉淀；加NH₃→深蓝 |
| 紫色 | MnO₄⁻、Ti³⁺ | ⚠ 需区分 | 强氧化性→MnO₄⁻；还原性→Ti³⁺ |
| 绿色 | Cr³⁺、Ni²⁺、Fe²⁺（淡绿） | ⚠ 三种可能 | 加NaOH行为不同；氧化后颜色变化 |
| 黄色 | CrO₄²⁻、Fe³⁺（黄棕） | ⚠ 需区分 | 酸碱性：CrO₄²⁻↔Cr₂O₇²⁻；加SCN⁻→Fe³⁺ |
| 橙色 | Cr₂O₇²⁻ | ✓ 较唯一 | 酸性条件稳定；加碱变黄 |
| 粉红色 | Co²⁺、Mn²⁺（淡粉） | ⚠ 需区分 | 加NH₃：Co²⁺→土黄/[Co(NH₃)₆]²⁺；Mn²⁺变化不明显 |
| 血红色 | Fe³⁺+SCN⁻ | ✓ 特征配合物 | 加F⁻或PO₄³⁻褪色（掩蔽） |

> [!info]- 完整推导与教材对照
> **Cu²⁺的蓝色**：水合铜离子[Cu(H₂O)₄]²⁺的d-d跃迁吸收橙红光，透射蓝绿光。
> **Cr³⁺的绿色**：[Cr(H₂O)₆]³⁺的d-d跃迁。加NaOH生成灰蓝色Cr(OH)₃沉淀，溶于过量NaOH生成绿色[Cr(OH)₄]⁻。
> **Fe²⁺与Fe³⁺颜色差异**：Fe²⁺为淡绿色（[Fe(H₂O)₆]²⁺），Fe³⁺为黄棕色（水解产物）。
> 对照：[[钛铬锰铁钴镍铜锌基础元素化学]]、[[过渡金属通性]]

**阶段性小结**：颜色信号是"第一印象"，但**单一颜色不能定案**。看到绿色溶液，第一反应应该是"三种可能"，然后找其他信号排除。

**对比表格：易混淆颜色信号**

| 信号组合 | 推断结论 | 排除依据 |
|---------|---------|---------|
| 绿色溶液 + 加NaOH→灰绿色沉淀→空气中变红棕 | Fe²⁺ | Cr³⁺沉淀不变色；Ni²⁺沉淀为浅绿色 |
| 绿色溶液 + 加NaOH→灰蓝色沉淀→溶于过量NaOH | Cr³⁺ | Fe²⁺沉淀不溶于过量NaOH（两性弱） |
| 黄色溶液 + 加SCN⁻→血红色 | Fe³⁺ | CrO₄²⁻无此反应 |
| 黄色溶液 + 加酸→变橙色 | CrO₄²⁻ | Fe³⁺酸化后颜色不变 |

🔷 **拓展**：Xe化合物的颜色信号
- XeF₂（白色固体）、XeF₄（白色固体）、XeF₆（白色固体）—— 固体颜色无区分度
- 但XeF₂水解产物含Xe（气体）+ O₂，XeF₆水解剧烈，可作为辅助信号
- 稀有气体化合物极少见，一旦涉及通常是"排除法"的终点

### 2.2 沉淀信号法

**从问题出发**：沉淀的颜色变化和溶解性，能告诉你什么？

**关键推理/观察**：
- 沉淀的"生成→转化→溶解"全过程都是信号
- 特别关注"颜色渐变"和"两性"行为

| 沉淀特征 | 推断指向 | 关键验证 |
|---------|---------|---------|
| 白色沉淀，溶于氨水 | AgCl | 加HNO₃重新析出；不溶于稀酸 |
| 白色→灰绿→红棕（空气中） | Fe²⁺→Fe(OH)₂→Fe(OH)₃ | 加KSCN→血红色确认Fe³⁺ |
| 白色沉淀，不溶于酸 | BaSO₄、AgCl、PbSO₄ | 加Na₂S：Pb²⁺→黑色PbS；Ag⁺→黑色Ag₂S |
| 黑色沉淀 | CuS、PbS、Ag₂S、HgS | 加稀HNO₃：CuS溶；HgS不溶（需王水） |
| 白色沉淀，溶于过量NaOH | Al(OH)₃、Zn(OH)₂、Cr(OH)₃ | 加NH₃：Zn²⁺溶；Al³⁺不溶（区分Al³⁺/Zn²⁺） |
| 黄色沉淀，溶于稀酸 | AgBr | 加Na₂S₂O₃溶→Ag(S₂O₃)₂³⁻ |
| 黄色沉淀，不溶于稀酸 | AgI、PbI₂ | 加KCN：AgI溶；PbI₂不溶 |

> [!info]- 完整推导与教材对照
> **Fe²⁺→Fe³⁺的沉淀转化链**：
> Fe²⁺ + 2OH⁻ → Fe(OH)₂↓（白色）
> 4Fe(OH)₂ + O₂ + 2H₂O → 4Fe(OH)₃（红棕）
> 这是推断Fe²⁺的**最强信号组合**之一：白色→灰绿→红棕的渐变过程几乎唯一。
> **Al³⁺的两性验证**：
> Al³⁺ + 3OH⁻ → Al(OH)₃↓（白色）
> Al(OH)₃ + OH⁻ → [Al(OH)₄]⁻（无色）
> 对照：[[主族元素化学]]、[[硼族元素]]

**阶段性小结**：沉淀信号比颜色信号更可靠，因为沉淀涉及"生成+性质"双重信息。尤其是**颜色渐变过程**（如Fe²⁺的氧化）和**两性行为**，往往是唯一性信号。

**对比表格：白色沉淀的区分**

| 沉淀 | 稀HCl | 稀HNO₃ | 氨水 | NaOH过量 | Na₂S |
|------|-------|--------|------|---------|------|
| AgCl | 不溶 | 不溶 | 溶 | 不溶 | 黑色Ag₂S |
| BaSO₄ | 不溶 | 不溶 | 不溶 | 不溶 | 不溶 |
| Al(OH)₃ | 溶 | 溶 | 不溶 | 溶 | — |
| Zn(OH)₂ | 溶 | 溶 | 溶（配位） | 溶 | 白色ZnS |
| PbSO₄ | 不溶 | 不溶 | 不溶 | 微溶 | 黑色PbS |

🔷 **拓展**：卤化银的溶解性梯度与摄影化学
- AgCl（白，Ksp=1.8×10⁻¹⁰）→ AgBr（淡黄，Ksp=5.0×10⁻¹³）→ AgI（黄，Ksp=8.3×10⁻¹⁷）
- 溶解性递减对应感光度递增，这是胶片摄影的化学基础
- 推断题中若出现"黄色沉淀，感光变黑"→AgBr

### 2.3 气体信号法

**从问题出发**：气体的颜色、气味、与空气/试剂的反应，如何转化为推断依据？

**关键推理/观察**：
- 气体的"生成+性质"是双重信号
- 特别关注"遇空气变色"和"特征气味"

| 气体特征 | 推断指向 | 关键验证 |
|---------|---------|---------|
| 无色，遇空气变红棕 | NO → NO₂ | 通入水中得HNO₃；加KI→I₂（棕） |
| 臭鸡蛋气味 | H₂S | 使Pb(Ac)₂试纸变黑；燃烧生成SO₂ |
| 使品红褪色，加热恢复 | SO₂ | 酸性氧化物；加BaCl₂→BaSO₃（溶于酸） |
| 使澄清石灰水变浑浊 | CO₂ | 过量CO₂→澄清（Ca(HCO₃)₂） |
| 支持燃烧，使带火星木条复燃 | O₂ | 与NO混合→红棕 |
| 不助燃，使燃着木条熄灭 | N₂、CO₂等 | 需结合其他信号区分 |
| 黄绿色，有刺激性 | Cl₂ | 使KI淀粉试纸变蓝 |
| 无色，使湿润红色石蕊试纸变蓝 | NH₃ | 加HCl→白烟（NH₄Cl） |

> [!info]- 完整推导与教材对照
> **NO→NO₂的推断链**：
> 2NO + O₂ → 2NO₂（红棕）
> 3NO₂ + H₂O → 2HNO₃ + NO
> 若题目描述"无色气体遇空气变红棕，通入水中得无色溶液，该溶液能使KI变棕"，则信号链为：
> NO → NO₂ → HNO₃ → I₂
> 这是推断含氮化合物的**经典信号组合**。
> **H₂S的推断链**：
> H₂S + Pb²⁺ → PbS↓（黑）+ 2H⁺
> 2H₂S + SO₂ → 3S↓（黄）+ 2H₂O
> 对照：[[氮族元素]]、[[氧族元素]]

**阶段性小结**：气体信号的可靠性高，因为气体的性质（颜色变化、气味、与试剂反应）往往具有唯一性。**"遇空气变红棕"是NO的指纹级信号**。

**对比表格：无色气体的区分**

| 气体 | 空气/O₂ | 石灰水 | 品红 | KI淀粉 | 燃烧 |
|------|---------|--------|------|--------|------|
| NO | 变红棕 | — | — | — | — |
| N₂ | 无变化 | 无变化 | 无变化 | 无变化 | 熄灭 |
| CO₂ | 无变化 | 浑浊 | 无变化 | 无变化 | 熄灭 |
| NH₃ | 无变化 | 无变化 | — | — | — |
| H₂S | 变SO₂（燃烧） | 无变化 | — | — | 淡蓝火焰 |

🔷 **拓展**：氮的氧化物家族的推断逻辑
- NO（无）→ NO₂（红棕）→ N₂O₄（无，低温）→ N₂O₅（白固）
- 推断题中若出现"红棕色气体，低温颜色变浅"→NO₂/N₂O₄平衡
- 若出现"无色气体，与O₂反应后体积减半"→2NO + O₂ → 2NO₂（体积比3:2）

### 2.4 价态变化法

**从问题出发**：某元素在反应中既被氧化又被还原，说明什么？

**关键推理/观察**：
- **中间价态可歧化**：若某元素在反应中既升价又降价，则反应前处于中间价态
- **既可升价又可降价**：说明该元素不是最高价态，也不是最低价态

| 价态特征 | 推断逻辑 | 典型实例 |
|---------|---------|---------|
| 歧化反应 | 反应前为中间价态 | Cl₂ + 2OH⁻ → Cl⁻ + ClO⁻ + H₂O（Cl：0→-1和+1） |
| 归中反应 | 高低价态→中间价态 | 2H₂S + SO₂ → 3S + 2H₂O（S：-2和+4→0） |
| 既可氧化又可还原 | 处于中间价态 | Fe²⁺可被氧化为Fe³⁺，也可被还原为Fe |
| 只升不降 | 处于最低价态 | S²⁻、I⁻、Br⁻等还原剂 |
| 只降不升 | 处于最高价态 | MnO₄⁻、Cr₂O₇²⁻、HNO₃等氧化剂 |

> [!info]- 完整推导与教材对照
> **Cl₂的歧化**：
> Cl₂ + 2NaOH → NaCl + NaClO + H₂O（冷稀碱）
> 3Cl₂ + 6NaOH → 5NaCl + NaClO₃ + 3H₂O（热浓碱）
> 同一元素（Cl，0价）既被氧化又被还原，说明0价是中间价态。
> **S₂O₃²⁻的歧化**：
> S₂O₃²⁻ + 2H⁺ → S↓ + SO₂↑ + H₂O
> S的平均价态为+2，歧化为0（S）和+4（SO₂），+2是中间价态。
> 对照：[[氧化还原]]、[[卤素]]、[[氧族元素]]

**阶段性小结**：价态变化法是**逻辑推断法**，不依赖实验现象，而是通过氧化还原方程式的配平和分析确定元素身份。当题目给出"某物质既可作氧化剂又可作还原剂"时，立即想到"中间价态"。

**价态数轴可视化**：

```
Cl: -1 ——— 0 ——— +1 ——— +3 ——— +5 ——— +7
          ↑中间价态，可歧化
          Cl₂ + OH⁻ → Cl⁻ + ClO⁻

S: -2 ——— 0 ——— +2 ——— +4 ——— +6
               ↑中间价态，可歧化
               S₂O₃²⁻ + H⁺ → S + SO₂

N: -3 ——— 0 ——— +1 ——— +2 ——— +3 ——— +4 ——— +5
                    ↑中间价态
                    NO既可被氧化为NO₂，也可被还原为N₂O
```

🔷 **拓展**：Mn的价态变化推断
- Mn²⁺（淡粉）→ MnO₂（棕黑，+4）→ MnO₄²⁻（绿，+6）→ MnO₄⁻（紫，+7）
- 推断题中若出现"淡粉色溶液被氧化，先变棕黑，后变紫"→Mn²⁺的逐步氧化
- 若出现"紫色溶液被还原，先变绿，后变棕黑"→MnO₄⁻的逐步还原（与介质有关）

### 2.5 配合物生成法

**从问题出发**：加入某试剂后颜色突变，说明什么？

**关键推理/观察**：
- 配合物生成常伴随**显著颜色变化**
- 颜色突变是"指纹级"信号，往往具有唯一性

| 原始状态 | 加入试剂 | 变化后状态 | 推断 |
|---------|---------|-----------|------|
| Cu²⁺（浅蓝） | 过量NH₃ | 深蓝 [Cu(NH₃)₄]²⁺ | Cu²⁺确认 |
| Fe³⁺（黄棕） | SCN⁻ | 血红色 [Fe(SCN)]²⁺ | Fe³⁺确认 |
| Fe³⁺（黄棕） | F⁻或PO₄³⁻ | 褪色（掩蔽） | Fe³⁺确认（反向验证） |
| Co²⁺（粉红） | 过量NH₃ | 土黄→[Co(NH₃)₆]²⁺（土黄） | Co²⁺确认 |
| Ni²⁺（绿） | 过量NH₃ | 蓝紫 [Ni(NH₃)₆]²⁺ | Ni²⁺确认 |
| Ag⁺（无） | NH₃ | 先沉淀后溶解 [Ag(NH₃)₂]⁺ | Ag⁺确认 |
| Zn²⁺（无） | NH₃ | 先沉淀后溶解 [Zn(NH₃)₄]²⁺ | Zn²⁺确认 |

> [!info]- 完整推导与教材对照
> **Cu²⁺的氨配合物**：
> Cu²⁺ + 2NH₃·H₂O → Cu(OH)₂↓（蓝）+ 2NH₄⁺
> Cu(OH)₂ + 4NH₃ → [Cu(NH₃)₄]²⁺（深蓝）+ 2OH⁻
> 颜色变化：浅蓝→蓝色沉淀→深蓝溶液，这是Cu²⁺的**三阶段确认信号**。
> **Fe³⁺的掩蔽反应**：
> [Fe(SCN)]²⁺ + 6F⁻ → [FeF₆]³⁻ + SCN⁻（褪色）
> 若题目要求"在不破坏Fe³⁺的情况下消除血红色"，应加F⁻或PO₄³⁻。
> 对照：[[配合物基础]]、[[过渡金属通性]]

**阶段性小结**：配合物生成法是**确认法**——当其他信号已初步指向某离子时，配合物反应可作为"最终确认"。尤其是Cu²⁺+NH₃和Fe³⁺+SCN⁻，是竞赛题中最常出现的确认反应。

**对比表格：氨水作为配体的区分作用**

| 离子 | 加少量NH₃·H₂O | 加过量NH₃·H₂O | 推断 |
|------|--------------|--------------|------|
| Cu²⁺ | 蓝色沉淀 | 深蓝溶液 | Cu²⁺ |
| Fe³⁺ | 红棕沉淀 | 不溶 | Fe³⁺ |
| Fe²⁺ | 白色→灰绿→红棕 | 不溶 | Fe²⁺ |
| Al³⁺ | 白色沉淀 | 不溶 | Al³⁺ |
| Zn²⁺ | 白色沉淀 | 无色溶液 | Zn²⁺ |
| Ag⁺ | 棕褐色沉淀 | 无色溶液 | Ag⁺ |
| Co²⁺ | 蓝绿色沉淀 | 土黄色溶液 | Co²⁺ |
| Ni²⁺ | 浅绿色沉淀 | 蓝紫色溶液 | Ni²⁺ |

🔷 **拓展**：EDTA配合物与推断
- EDTA与大多数金属离子形成1:1稳定配合物，颜色与游离离子相近
- 推断题中若出现"加入EDTA后某干扰离子被掩蔽"，通常是消除Al³⁺、Fe³⁺等对指示剂的干扰
- 例如：用铬黑T滴定Mg²⁺时，需先加三乙醇胺掩蔽Fe³⁺、Al³⁺

### 2.6 推理链整合方法

**从问题出发**：单一信号不可靠，如何整合多个信号进行交叉验证？

**关键推理/观察**：
- **信号优先级**：唯一性信号 > 组合信号 > 单一信号
- **最少信号数**：至少3个独立信号交叉验证，才能确定元素身份
- **推理链格式**：因为[现象A]，所以[推断X]；又因为[现象B]，与推断X一致/矛盾，所以[修正/确认]

**信号优先级决策树**：

```
拿到题目
  ↓
提取所有信号（颜色/沉淀/气体/价态/配合物）
  ↓
是否有"指纹级"唯一性信号？
  ├─ 是（如"遇空气变红棕"→NO）→ 以此为核心，找验证信号
  └─ 否（如"绿色溶液"→三种可能）→ 列出所有可能，逐一排除
        ↓
对每个可能，检查与其他信号是否矛盾
  ├─ 矛盾 → 排除
  └─ 一致 → 保留
        ↓
保留选项中，找"确认级"信号（如配合物反应）
  ↓
确定元素身份，写出关键方程式
```

**完整推理链示例**：

> **题目**：某绿色溶液A，加NaOH生成灰绿色沉淀B，空气中放置后变红棕C；过滤后取滤液加BaCl₂生成白色沉淀D，不溶于稀硝酸。推断A中可能含有什么离子？
>
> **推理链**：
> 1. 因为A为绿色溶液，所以可能含Cr³⁺、Ni²⁺或Fe²⁺（颜色信号，非唯一）。
> 2. 因为加NaOH生成灰绿色沉淀B，且B在空气中变红棕C，所以B为Fe(OH)₂，C为Fe(OH)₃（沉淀信号，Fe²⁺的指纹级信号）。
> 3. 因为滤液加BaCl₂生成白色沉淀D且不溶于稀硝酸，所以D为BaSO₄，滤液中含SO₄²⁻（沉淀信号，唯一性）。
> 4. 综合以上，A中含Fe²⁺和SO₄²⁻，即A为FeSO₄溶液。
> 5. 关键方程式：
>    - Fe²⁺ + 2OH⁻ → Fe(OH)₂↓
>    - 4Fe(OH)₂ + O₂ + 2H₂O → 4Fe(OH)₃
>    - Ba²⁺ + SO₄²⁻ → BaSO₄↓

> [!info]- 完整推导与教材对照
> **推理链的评分标准**：
> 竞赛评分中，推断题的分数通常按"信号提取→逻辑推理→结论确认→方程式"分配。
> 以38届Q1（22分）为例，评分点大致分布：
> - 信号正确提取：每个有效信号1-2分
> - 逻辑推理完整：推理链无跳跃，每步有依据
> - 结论正确：元素身份正确
> - 关键方程式：每个正确方程式1-2分
> 对照：[[04-课件/备课大纲/2026-06-03-元素推断综合训练-提高班]]

**阶段性小结**：推理链整合是"方法的方法"——把五种信号法组织成完整的逻辑链条。核心原则是：**每个推断必须有现象作为证据，每个结论必须有验证作为支撑**。

## 三、课堂例题

### ⭐⭐ 基础题

#### 例题1：Cu²⁺的三阶段确认

> 某蓝色溶液A，加入NaOH生成蓝色沉淀B，加热后B变为黑色C；向A中加入过量氨水，得深蓝色溶液D。推断A中含有什么离子？写出相关方程式。

**信号提取**：
- 蓝色溶液 → Cu²⁺（初步）
- 加NaOH→蓝色沉淀 → Cu(OH)₂（验证）
- 加热变黑 → CuO（验证）
- 加过量NH₃→深蓝溶液 → [Cu(NH₃)₄]²⁺（确认）

**推理链**：
1. 因为A为蓝色溶液，所以初步推断含Cu²⁺。
2. 因为加NaOH生成蓝色沉淀，且加热变黑色，所以沉淀为Cu(OH)₂，分解产物为CuO，与Cu²⁺一致。
3. 因为加过量NH₃得深蓝色溶液，所以生成了[Cu(NH₃)₄]²⁺，这是Cu²⁺的确认级信号。
4. 综合以上3个独立信号，确定A中含Cu²⁺。

**关键方程式**：
- Cu²⁺ + 2OH⁻ → Cu(OH)₂↓
- Cu(OH)₂ → CuO + H₂O（加热）
- Cu²⁺ + 4NH₃ → [Cu(NH₃)₄]²⁺

---

#### 例题2：Fe²⁺→Fe³⁺的氧化链

> 某淡绿色溶液A，加入NaOH生成白色沉淀B，迅速变为灰绿色C，最终在空气中变为红棕色D；向A中加入KSCN无明显变化，再加入氯水后呈血红色。推断A中含有什么离子？写出相关方程式。

**信号提取**：
- 淡绿色溶液 → Fe²⁺（初步，但Cr²⁺、Ni²⁺也可能）
- 白色→灰绿→红棕沉淀 → Fe(OH)₂→Fe(OH)₃（指纹级信号）
- 加KSCN无变化 → 无Fe³⁺（排除Fe³⁺）
- 加氯水后血红色 → Fe³⁺生成（确认Fe²⁺被氧化）

**推理链**：
1. 因为A为淡绿色溶液，所以可能含Fe²⁺、Cr³⁺或Ni²⁺。
2. 因为加NaOH生成白色沉淀B，且B在空气中逐步变为红棕色D，所以B为Fe(OH)₂，D为Fe(OH)₃。这是Fe²⁺的指纹级信号，排除Cr³⁺和Ni²⁺（它们的氢氧化物不会在空气中变色）。
3. 因为加KSCN无血红色，所以A中无Fe³⁺，进一步确认A中为Fe²⁺而非Fe³⁺。
4. 因为加氯水后呈血红色，所以Fe²⁺被氧化为Fe³⁺，与Fe²⁺的推断一致。
5. 综合以上4个信号，确定A中含Fe²⁺。

**关键方程式**：
- Fe²⁺ + 2OH⁻ → Fe(OH)₂↓
- 4Fe(OH)₂ + O₂ + 2H₂O → 4Fe(OH)₃
- 2Fe²⁺ + Cl₂ → 2Fe³⁺ + 2Cl⁻
- Fe³⁺ + SCN⁻ → [Fe(SCN)]²⁺（血红色）

---

#### 例题3：NO的推断链

> 某无色气体A，遇空气立即变为红棕色B；将B通入水中得无色溶液C，C能使KI-淀粉溶液变蓝。推断A、B、C分别是什么？写出相关方程式。

**信号提取**：
- 无色气体，遇空气变红棕 → NO→NO₂（指纹级信号）
- 红棕气体通入水得无色溶液 → HNO₃（验证）
- 溶液使KI-淀粉变蓝 → I₂生成，说明溶液有氧化性（确认HNO₃）

**推理链**：
1. 因为A为无色气体，遇空气变红棕色B，所以A为NO，B为NO₂。这是NO的指纹级信号，几乎唯一。
2. 因为B（NO₂）通入水中得无色溶液C，所以C为HNO₃（3NO₂ + H₂O → 2HNO₃ + NO）。
3. 因为C能使KI-淀粉变蓝，所以C将I⁻氧化为I₂，与HNO₃的强氧化性一致。
4. 综合以上3个信号，确定A=NO，B=NO₂，C=HNO₃。

**关键方程式**：
- 2NO + O₂ → 2NO₂
- 3NO₂ + H₂O → 2HNO₃ + NO
- 6I⁻ + 2NO₃⁻ + 8H⁺ → 3I₂ + 2NO↑ + 4H₂O

---

#### 例题4：Al³⁺+SO₄²⁻的组合推断

> 某无色溶液A，加入BaCl₂生成白色沉淀B，B不溶于稀硝酸；过滤后向滤液中加入NaOH生成白色沉淀C，C溶于过量NaOH。推断A中含有什么离子？

**信号提取**：
- 加BaCl₂→白色沉淀，不溶于稀HNO₃ → BaSO₄，所以A含SO₄²⁻（唯一性信号）
- 滤液加NaOH→白色沉淀→溶于过量NaOH → Al(OH)₃的两性，所以A含Al³⁺（唯一性信号）

**推理链**：
1. 因为A加BaCl₂生成白色沉淀B且不溶于稀硝酸，所以B为BaSO₄，A中含SO₄²⁻。
2. 因为滤液加NaOH生成白色沉淀C，且C溶于过量NaOH，所以C为Al(OH)₃，A中含Al³⁺。
3. 两个信号相互独立，无矛盾。
4. 综合以上，A中含Al³⁺和SO₄²⁻，可能为Al₂(SO₄)₃溶液。

**关键方程式**：
- Ba²⁺ + SO₄²⁻ → BaSO₄↓
- Al³⁺ + 3OH⁻ → Al(OH)₃↓
- Al(OH)₃ + OH⁻ → [Al(OH)₄]⁻

### ⭐⭐⭐ 拓展题

#### 拓展1：37届初赛Q5（NO/H₂S反应，13分）

> **题目大意**：无色气体A与另一气体B混合，在催化剂作用下反应生成C和D；A遇空气变红棕色；B有臭鸡蛋气味，能使Pb(Ac)₂试纸变黑；C为无色液体；D为淡黄色固体。推断A、B、C、D，并写出反应方程式。

**信号提取与推理**：
- A：无色气体，遇空气变红棕 → **NO**（指纹级）
- B：臭鸡蛋气味，使Pb(Ac)₂变黑 → **H₂S**（指纹级）
- C：无色液体 → **H₂O**
- D：淡黄色固体 → **S**

**反应方程式**：
- 2NO + 2H₂S → N₂ + 2S + 2H₂O（催化剂，需配平验证）
- 或：2NO + H₂S → N₂O + S + H₂O（视条件而定）

> **评分要点**：
> - 正确推断A=NO（2分）：必须写出"遇空气变红棕→NO"的推理
> - 正确推断B=H₂S（2分）：必须写出"臭鸡蛋味+PbS黑色→H₂S"的推理
> - 正确推断C=H₂O、D=S（各1分）
> - 正确配平方程式（4分）：需体现氧化还原配平过程
> - 推理过程完整（3分）：无跳步，有"因为…所以…"格式

#### 拓展2：38届初赛Q1（炼丹元素推断，22分）

> **题目大意**：某古代炼丹术记载："取绿矾（A）煅烧，得赤色固体B和刺激性气体C；C通入水中得溶液D，D能使品红褪色；B加炭高温还原得黑色固体E；E溶于稀硫酸得淡绿色溶液F；F加KSCN无变化，加氯水后呈血红色。"推断A-F，写出关键方程式。

**信号提取与推理**：
- A：绿矾 → **FeSO₄·7H₂O**（常识+颜色信号）
- 煅烧：2FeSO₄·7H₂O → Fe₂O₃ + SO₂ + SO₃ + 14H₂O（或简化为Fe₂O₃ + SO₂）
- B：赤色固体 → **Fe₂O₃**（颜色信号）
- C：刺激性气体 → **SO₂**（气体信号：刺激性+后续使品红褪色）
- D：SO₂水溶液 → **H₂SO₃**（气体信号验证）
- E：B加炭还原的黑色固体 → **Fe**（或Fe₃O₄，需进一步验证）
- F：淡绿色溶液 + KSCN无变化 + 氯水后血红色 → **Fe²⁺**（颜色+配合物信号）

> **关键难点**：E的确定
> - 若E为Fe，溶于稀H₂SO₄得FeSO₄（淡绿色），与F一致
> - 若E为Fe₃O₄，溶于稀H₂SO₄得Fe²⁺+Fe³⁺混合，加KSCN应有血红色，与"F加KSCN无变化"矛盾
> - 所以E为Fe，不是Fe₃O₄

**关键方程式**：
- 2FeSO₄·7H₂O → Fe₂O₃ + SO₂↑ + SO₃↑ + 14H₂O（煅烧）
- Fe₂O₃ + 3C → 2Fe + 3CO↑（高温还原）
- Fe + H₂SO₄ → FeSO₄ + H₂↑
- 2Fe²⁺ + Cl₂ → 2Fe³⁺ + 2Cl⁻
- Fe³⁺ + SCN⁻ → [Fe(SCN)]²⁺

> **评分要点**：
> - 每个物质推断正确（各2分，共12分）
> - E的排除推理（3分）：必须说明为什么E不是Fe₃O₄
> - 方程式正确（每个2分，共4分）
> - 推理链完整（3分）：绿矾→煅烧产物→还原→溶解→氧化的全链条

## 五、本节形成的核心结论

1. **五种信号法**：颜色、沉淀、气体、价态变化、配合物生成——覆盖95%以上的元素推断情境。

2. **信号优先级**：唯一性信号（如NO遇空气变红棕）> 组合信号（如Fe²⁺的沉淀氧化链）> 单一信号（如绿色溶液）。

3. **最少验证原则**：至少3个独立信号交叉验证，才能确定元素身份。

4. **推理链格式**：因为[现象]，所以[推断]；又因为[现象]，所以[验证/确认]。

5. **过程分意识**：每个推断必须有现象作为证据，每个结论必须有验证作为支撑——不要跳步，不要想当然。

6. **常见指纹级信号**（看到即确定）：
   - 遇空气变红棕 → NO
   - 臭鸡蛋味 → H₂S
   - 白色→灰绿→红棕 → Fe²⁺
   - 血红色+SCN⁻ → Fe³⁺
   - 蓝色+过量NH₃→深蓝 → Cu²⁺

## 六、⚠️ 易错点预警

| 易错点 | 错误表现 | 正确做法 |
|--------|---------|---------|
| **绿色溶液直接定案** | 看到绿色就写Cr³⁺ | 先列三种可能（Cr³⁺/Ni²⁺/Fe²⁺），再找其他信号排除 |
| **忽略沉淀的渐变过程** | 只写"生成红棕沉淀" | 必须写出"白色→灰绿→红棕"的完整过程，这是Fe²⁺的指纹 |
| **气体信号不完整** | 只写"生成NO" | 必须写"无色气体遇空气变红棕"的完整现象 |
| **配合物反应漏写配位数** | [Cu(NH₃)]²⁺ | 正确为[Cu(NH₃)₄]²⁺（四氨合铜(II)） |
| **价态变化不说明理由** | "因为歧化，所以是Cl₂" | 必须说明"Cl为0价，既可升为+1（ClO⁻）又可降为-1（Cl⁻），是中间价态" |
| **跳步丢过程分** | "因为蓝色，所以是Cu²⁺" | "因为蓝色溶液，初步推断Cu²⁺；加NaOH生成蓝色沉淀，与Cu²⁺一致；加过量NH₃得深蓝色溶液，确认[Cu(NH₃)₄]²⁺生成，所以是Cu²⁺" |
| **混淆Al³⁺和Zn²⁺** | 两者都溶于过量NaOH | 加氨水：Zn²⁺溶（[Zn(NH₃)₄]²⁺），Al³⁺不溶——这是区分信号 |
| **Fe₃O₄的干扰** | 黑色固体溶于酸得Fe²⁺+Fe³⁺ | 若后续加KSCN无变化，则排除Fe₃O₄，应为Fe |
