# 有机合成与金属有机 · 速查卡

> 基础班你背过 Grignard 试剂"加水就炸"——本卡把有机合成从**零散的单步反应**升级到**断键→选试剂→查兼容性**的三步路线设计系统。

---

## 一、合成路线设计 · 三步法

```
拿到目标分子 → 三步法
    │
    ├─ 第一步：断键——从产物往回找最值得切的键
    │   ├─ 切在官能团旁边（α 位）→ 找熟悉的逆合成子
    │   ├─ 切 C=C 双键 → Wittig / 消除 / 缩合来源
    │   └─ 切在 "天然断键标记" 处（1,3-二羰基、α,β-不饱和羰基）
    │
    ├─ 第二步：选试剂——哪个 organometallic 最合适？
    │   ├─ 做 C–C 键：RMgX / RLi / R₂CuLi / Pd 偶联
    │   ├─ 做 C=C 键：Wittig / Horner–Wadsworth–Emmons
    │   └─ 做官能团转化：氧化/还原/保护
    │
    └─ 第三步：查兼容性——官能团会不会打架？
        ├─ 有活泼氢（OH, NH, SH, COOH）→ RMgX/RLi 会淬灭 → 需要保护或多加当量
        ├─ 有 α,β-不饱和羰基 → RMgX 给 1,2-加成，Gilman 给 1,4-加成
        └─ 多步路线 → 保护基顺序：先保护再反应，最后脱保护
```

## 二、制醇网络 · 四线汇总

```
在合成题中，看到目标分子里有 —OH → 先查四线制醇网络

┌─────────────────────────────────────────────────┐
│  路线 1：Grignard/RLi + 羰基                      │
│    RMgX + 甲醛 → 1° 醇                           │
│    RMgX + 醛   → 2° 醇                           │
│    RMgX + 酮   → 3° 醇                           │
│    RMgX + 酯   → 3° 醇（两次加成，两个相同 R）      │
│─────────────────────────────────────────────────│
│  路线 2：还原                                    │
│    NaBH₄  → 醛→1°醇，酮→2°醇（温和，不还原酯）     │
│    LiAlH₄ → 醛/酮/酯/酸→醇（强力，全还原）          │
│    DIBAL-H → 酯→醛（部分还原，低温可控）           │
│─────────────────────────────────────────────────│
│  路线 3：水合 / 硼氢化-氧化                       │
│    烯烃 + H₂O/H⁺       → 醇（马氏规则）            │
│    烯烃 + BH₃→H₂O₂/OH⁻ → 醇（反马氏规则）          │
│─────────────────────────────────────────────────│
│  路线 4：取代                                    │
│    RX + OH⁻ → 醇（SN2，仅 1°/2° 卤代烃）         │
└─────────────────────────────────────────────────┘
```

**推断题信号**：题目要求从给定原料合成指定醇 → 先画出目标醇、标出 α-C、反问"这根 C–C 键能用 Grignard 建吗？"

## 三、Organometallic 工具箱 · 三线对比

### 3.1 核心碳亲核试剂

| 维度 | RMgX（Grignard） | RLi（有机锂） | R₂CuLi（Gilman） |
|:---|:---|:---|:---|
| **活性** | 高 | 很高 | 中等（较温和） |
| **1,2 vs 1,4** | 1,2-加成 | 1,2-加成 | **1,4-加成**（共轭加成） |
| **对 α,β-不饱和酮** | 给 1,2-产物（进攻 C=O） | 给 1,2-产物 | 给 1,4-产物（进攻 β-C） |
| **酸性敏感度** | 极敏感→质子淬灭 | 极敏感→质子淬灭 | 敏感→质子淬灭 |
| **溶剂** | Et₂O / THF | Et₂O / THF / 己烷 | Et₂O / THF |
| **与酯反应** | 两次加成→3° 醇 | 两次加成→3° 醇 | 不反应（温和） |
| **与 CO₂** | 羧酸盐→羧酸 | 羧酸盐→羧酸 | 一般不反应 |

### 3.2 1,2 vs 1,4 选择速判

```
α,β-不饱和羰基化合物 + 碳亲核试剂：

  RMgX / RLi ────→ 1,2-加成（进攻 C=O）
                     └─ 原因：C=O 碳更缺电子（硬-硬匹配）

  R₂CuLi ────────→ 1,4-加成（共轭加成，进攻 β-C）
                     └─ 原因：Cu 的 d 轨道参与→软-软匹配
  
  口诀："格氏上羰基，铜锂走共轭"
```

## 四、Wittig 反应 · 双键构建

```
Wittig 反应 = 磷叶立德 + 醛/酮 → 烯烃 + Ph₃P=O

断键价值：合成中看到目标分子的 C=C 双键 → 可以切回 Wittig 前体

  R₂C=O  +  Ph₃P=CR'R"  ──→  R₂C=CR'R"  +  Ph₃P=O

稳定磷叶立德（R'=EWG，如 COOR）：
  ├─ 反应较慢，但 E-选择性高
  └─ 适合合成热力学稳定的 E-烯烃

非稳定磷叶立德（R'=alkyl）：
  ├─ 反应快，但 Z-选择性
  └─ 适合合成 Z-烯烃

Horner–Wadsworth–Emmons (HWE) = Wittig 升级版：
  └─ (EtO)₂P(=O)CHR + 碱 → 磷酸酯负离子 → 与醛/酮反应
  └─ E-选择性极高，副产物水溶性→易分离
```

**推断题信号**：产物中有 C=C 双键且原料中无此双键 → 很可能走 Wittig 或 HWE。

## 五、Pd 偶联反应 · 最低入门

```
Pd 偶联 = 过渡金属催化的 C–C 成键，比 Grignard 更温和、兼容性更好

┌─────────────────────────────────────────────────┐
│  Suzuki：R¹–B(OH)₂ + R²–X → R¹–R²               │
│    ├─ 硼酸 + 卤代物，Pd(0) 催化，碱（Na₂CO₃）      │
│    └─ 优点：硼酸无毒、稳定、官能团兼容性好          │
│─────────────────────────────────────────────────│
│  Negishi：R¹–ZnX + R²–X → R¹–R²                  │
│    ├─ 有机锌 + 卤代物，Pd(0) 催化                  │
│    └─ 优点：比 Suzuki 更快，适合大位阻偶联          │
│─────────────────────────────────────────────────│
│  Heck：R–X + CH₂=CHR' → R–CH=CHR'                │
│    ├─ 卤代物 + 烯烃，Pd(0) 催化，碱                │
│    └─ 产物是取代烯烃（在烯烃 C 上引入 R 基）        │
│─────────────────────────────────────────────────│
│  共同特点：                                       │
│    ├─ 都需要 Pd(0) 催化剂（Pd(PPh₃)₄ 最常用）      │
│    ├─ 三步催化循环：氧化加成→转金属化→还原消除       │
│    └─ 官能团兼容性远好于 Grignard/RLi              │
└─────────────────────────────────────────────────┘
```

**初赛层面**：知道 Suzuki/Negishi/Heck 的存在和简单底物→产物模式即可，不要求写完整催化循环。

## 六、保护基策略 · 兼容性管理

### 6.1 什么时候需要保护基？

```
保护基三问：
  Q1：这条路线中，有没有官能团比目标反应更活泼？
  Q2：更活泼的官能团会不会消耗试剂或给出副反应？
  Q3：如果会 → 先保护，再做目标反应，最后脱保护。

典型冲突场景：
  ├─ 有 —OH 但要用 RMgX 做 Grignard → OH 的质子淬灭 RMgX → 保护 OH！
  ├─ 有 —NH₂ 但要氧化 → NH₂ 比目标位更容易被氧化 → 保护 NH₂！
  └─ 有 —C≡CH 但要做 RLi 反应 → 端炔 H 也是酸性质子 → 保护或选别的路线
```

### 6.2 常用保护基速查

| 保护目标 | 保护基 | 保护条件 | 脱保护条件 | 稳定性 |
|:---|:---|:---|:---|:---|
| —OH → 缩醛/缩酮 | 二氢吡喃 (DHP) | DHP + H⁺ | H₃O⁺ | 对碱稳定 |
| —OH → 硅醚 | TMSCl / TBDMSCl + 碱 | TMSCl + Et₃N | TBAF / H⁺ | TBDMS 比 TMS 稳定 |
| —OH → 醚 | BnBr + NaH | BnBr + 碱 | H₂, Pd/C | 对酸碱均较稳定 |
| —C=O → 缩醛 | HOCH₂CH₂OH + H⁺ | 乙二醇 + 酸 | H₃O⁺ | 对碱/还原剂稳定 |
| —NH₂ → 酰胺 | Ac₂O / Boc₂O | 酰化试剂 + 碱 | H⁺ / TFA | Boc 比 Ac 更好脱 |

### 6.3 保护基选择原则

```
选保护基的三句话：
  1. 保护条件不破坏已有官能团
  2. 在后续反应条件下保护基稳定
  3. 脱保护条件不破坏目标产物

实例：
  TMS 保护羟基 → 做 Grignard 反应 → TBAF 脱保护
  └─ TMS 保护条件温和（TMSCl + Et₃N，不破坏大多数官能团）
  └─ TMS 对 RMgX 稳定（但在强碱下可能脱落→TBDMS 更稳妥）
  └─ TBAF 脱保护极温和，不破坏其他官能团
```

## 七、逆合成分析 · 断键速判

```
逆合成断键的三类 "高价值目标"：

1. C–C 单键旁边的官能团：
     R–C–OH → 切→ R⁻ + C=O（等价于 Grignard + 羰基）
     └─ 醇、胺、醚等官能团的 α-C 是最常切的键

2. C=C 双键：
     C=C → 切→ C=O + Ph₃P=CR'（Wittig 前体）
     └─ 双键处逆向 → 醛/酮 + 磷叶立德

3. 天然断键标记：
     O    O
     ‖    ‖
   R–C–CH₂–C–R' → 切→ 酯 + 酯缩合（Claisen 缩合）
     └─ 1,3-二羰基 = "此处可切"的信号
```

**推断题信号**：题目说"从不超过 X 个碳的原料出发"→ 限制原料碳数 → 必须做断键分析，把大分子切回小分子原料。

## 八、常见易错点

| 错误 | 纠正 |
|:---|:---|
| Grignard 对所有底物都加成 1,2-位 | 对 α,β-不饱和羰基，RMgX 走 1,2-位（进攻 C=O），R₂CuLi 走 1,4-位（共轭加成）——试剂决定了加在哪里 |
| Grignard 能和任何官能团共存 | RMgX 遇 OH、NH、SH、COOH 等活泼氢直接淬灭——必须先保护这些官能团！ |
| Wittig 反应就是"醛变烯" | Wittig 是醛/酮 + 磷叶立德→烯烃，断键价值在于逆向：烯烃可以切回酮+叶立德。同时区分 E/Z 选择性取决于叶立德类型 |
| 保护基没什么讲究 | 保护基是"暂驻组"——必须考虑三件事：保护条件不破坏原料、后续反应中稳定、脱保护条件不破坏产物 |
| RMgX + 酯只加一次 | RMgX 进攻酯两次——第一次加成→酮中间体→酮比酯更活泼→立即第二次加成→3° 醇（含两个相同的 R） |
| 碳数不够就做不出来 | 逆合成就是解决"碳数不够"的——断在能由小分子构建的键处，用 Grignard/Wittig/缩合反应 "买入" 碳 |

## 九、自查清单

- [ ] 合成三步法能默推（断键→选试剂→查兼容性）
- [ ] 制醇四线（Grignard/还原/水合/取代）能默列并写出代表性条件
- [ ] RMgX / RLi / R₂CuLi 三线对比表（活性/1,2 vs 1,4/酸性敏感度）
- [ ] α,β-不饱和羰基：RMgX→1,2-位 vs Gilman→1,4-位
- [ ] Wittig 的断键价值（C=C→回推 C=O + Ph₃P=CR'）
- [ ] Suzuki/Negishi/Heck 三偶联的基本模式（不要求写催化循环）
- [ ] 保护基三问（更活泼？会打架？要保护？）
- [ ] 保护基五类（缩醛/硅醚/苄醚/缩羰基/酰胺）的保护/脱保护条件
- [ ] 逆合成三种高价值断键（α-官能团/C=C / 1,3-二羰基）
- [ ] 常见推断题信号：碳数限制 / 逆推 / 保护基 / 活泼氢冲突
