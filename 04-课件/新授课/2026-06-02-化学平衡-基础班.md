## 一、课堂引入与本节问题

### 1.1 课堂切入

- **现象/问题/情境**：恒温恒容条件下，向已达平衡的反应体系中充入惰性气体（如 Ar），容器内总压明显增大。那么，化学平衡会不会移动？
- **学生已有认知**：在热力学初步中已学习过 $Q/K$ 判断方向、$\Delta G^{\circ}=-RT\ln K^{\circ}$、标准平衡常数表达式。
- **本节要解决的问题**：
  1. 如何判断一个反应是否达到平衡？若未达平衡，向哪个方向进行？
  2. 改变温度、压强、浓度或充入惰性气体时，平衡如何移动？$K$ 本身变不变？
  3. 已知 $K$ 和起始条件，如何计算平衡时各物质的量和转化率？

### 1.2 本节深度边界

- **本节讲到哪里为止**：
  - 能熟练写出 $Q$ 表达式并与 $K$ 比较判断方向
  - 会用 ICE 表求简单气相反应的转化率和平衡组成
  - 能正确判断恒压充惰性气体、恒容充惰性气体、改变压强、改变浓度时平衡是否移动及移动方向
  - 知道 $K_{\mathrm{sp}}$ 和 $K_{\mathrm{稳}}$ 是平衡常数的特例，能写表达式
- **本节暂不展开**：活度与非理想平衡常数、复杂多重平衡联动、van't Hoff 方程完整定量计算、恒压/恒容温度变化的完整定量比较（后续轮次处理）

## 二、知识内容

> 📷 **图片策略**：本专题类型为 `calculation`，默认 `images_priority: low`。

### 2.1 反应方向判断：$Q$ 与 $K$ 的比较

> 基础必讲层。呈现课堂需要使用的核心结论、公式、表格，同时保留完整推导供课后复习。

**从问题出发**：给定一个反应和一组起始浓度（或分压），如何判断反应会向哪个方向进行？

**关键推理/观察**（课堂精简版）：

反应商 $Q$ 的定义与平衡常数 $K$ 形式相同，但代入的是**任意时刻**的浓度或分压：

$$Q = \prod_i \left(\frac{p_i}{p^{\circ}}\right)^{\nu_i} \quad \text{或} \quad Q = \prod_i \left(\frac{c_i}{c^{\circ}}\right)^{\nu_i}$$

比较 $Q$ 与 $K$：

| 关系 | 反应方向 | 体系状态 |
|:---|:---|:---|
| $Q < K$ | 正向自发进行（向右） | 反应物过剩，产物不足 |
| $Q = K$ | 达平衡（净反应 = 0） | 动态平衡 |
| $Q > K$ | 逆向自发进行（向左） | 产物过剩，反应物不足 |

> [!info]- 完整推导与教材对照（课后复习用）
>
> **教材来源**：[[教学逻辑提炼-周坤无机新课-热力学与化学平衡-第一轮]]
>
> **完整推导**：
> 由 van't Hoff 等温式：
> $$\Delta G = \Delta G^{\circ} + RT\ln Q = -RT\ln K^{\circ} + RT\ln Q = RT\ln\frac{Q}{K^{\circ}}$$
>
> 当 $Q < K^{\circ}$ 时，$\ln(Q/K^{\circ}) < 0$，$\Delta G < 0$，正向自发；
> 当 $Q = K^{\circ}$ 时，$\Delta G = 0$，达平衡；
> 当 $Q > K^{\circ}$ 时，$\Delta G > 0$，逆向自发。
>
> **与教材的对照说明**：
> - 教材中常将 $Q$ 称为"反应商"，$K^{\circ}$ 称为"标准平衡常数"
> - 竞赛中通常默认使用标准平衡常数（无量纲），但经验平衡常数 $K_p$、$K_c$ 仍广泛使用
> - 注意区分：$Q$ 用任意态数据，$K$ 用平衡态数据

**阶段性小结**：$Q/K$ 判断是平衡问题的第一性原理。拿到题目先写 $Q$ 表达式，再与 $K$ 比较。

### 2.2 温度对 $K$ 的影响与 Le Chatelier 原理

**从问题出发**：升温时平衡常数 $K$ 本身会不会变？Le Chatelier 原理的"减弱改变"到底是什么意思？

**关键推理/观察**（课堂精简版）：

**温度是唯一改变 $K$ 的因素**：
- 吸热反应（$\Delta H > 0$）：升温 → $K$ 增大 → 平衡向正向（吸热方向）移动
- 放热反应（$\Delta H < 0$）：升温 → $K$ 减小 → 平衡向逆向（吸热方向）移动

**Le Chatelier 原理**：改变条件（浓度/压强/温度）→ 系统向**减弱改变的方向**移动 → 建立新平衡。

> ⚠️ **本质提醒**：Le Chatelier 是定性近似，本质是比较 $Q$ 与 $K$。
> - 浓度/压强改变 → 只改变 $Q$，不改变 $K$
> - 温度改变 → 改变 $K$ 本身
> - 催化剂 → 不改变 $Q$ 也不改变 $K$，只加快达到平衡的速度

**对比表格**：

| 改变条件 | 平衡移动方向 | $K$ 是否改变 | $Q$ 如何变化 | 本质 |
|:---|:---|:---:|:---|:---|
| 增大反应物浓度 | 正向 | 不变 | $Q$ 减小（$<K$） | 改 $Q$ |
| 增大总压（压缩体积） | 向气体分子数减少方向 | 不变 | $Q$ 变化 | 改 $Q$ |
| 升高温度（$\Delta H > 0$） | 正向 | **改变**（$K$ 增大） | 不变 | 改 $K$ |
| 加入催化剂 | **不移动** | 不变 | 不变 | 不改 $Q$ 也不改 $K$ |
| 恒容充惰性气体 | **不移动** | 不变 | 不变 | 各组分分压不变 |
| 恒压充惰性气体 | 向气体分子数增多方向 | 不变 | $Q$ 变化 | 等效于减压 |

> 🔷 **拓展**（选讲）：van't Hoff 方程定性形式
> $$\ln\frac{K_2}{K_1} = -\frac{\Delta H^{\circ}}{R}\left(\frac{1}{T_2}-\frac{1}{T_1}\right)$$
> 吸热反应 $\Delta H^{\circ} > 0$，升温（$T_2 > T_1$）→ 右侧为负 → $K_2 > K_1$。

### 2.3 $K_p$ 与 $K_c$ 的换算

**从问题出发**：气相反应中，题目给的是分压数据，但 $K$ 用浓度表示，如何换算？

**关键推理/观察**（课堂精简版）：

由理想气体状态方程 $p_i = c_i RT$，代入标准平衡常数定义可得：

$$K_p = K_c(RT)^{\Delta n}$$

其中 $\Delta n = \sum \nu_{\text{产物(g)}} - \sum \nu_{\text{反应物(g)}}$（仅计气体）。

**记忆口诀**：$\Delta n > 0$（气体增多），$K_p > K_c$（因为 $RT > 1$）；$\Delta n < 0$（气体减少），$K_p < K_c$。

> [!info]- 完整推导与单位检查（课后复习用）
>
> **推导**：
> 对于反应 $a\mathrm{A(g)} + b\mathrm{B(g)} \rightleftharpoons c\mathrm{C(g)} + d\mathrm{D(g)}$：
> $$K_p = \frac{(p_\mathrm{C}/p^{\circ})^c(p_\mathrm{D}/p^{\circ})^d}{(p_\mathrm{A}/p^{\circ})^a(p_\mathrm{B}/p^{\circ})^b}$$
> 由 $p_i = c_i RT$：
> $$K_p = \frac{(c_\mathrm{C}RT/p^{\circ})^c(c_\mathrm{D}RT/p^{\circ})^d}{(c_\mathrm{A}RT/p^{\circ})^a(c_\mathrm{B}RT/p^{\circ})^b} = K_c(RT)^{(c+d)-(a+b)} = K_c(RT)^{\Delta n}$$
>
> **单位检查**：
> - 标准 $K_p^{\circ}$ 和 $K_c^{\circ}$ 均无量纲（已除以标准态）
> - 经验 $K_p$ 和 $K_c$ 可能有单位，竞赛中通常使用标准平衡常数

### 2.4 ICE 表与转化率计算

**从问题出发**：已知 $K$ 和起始条件，怎么算出平衡时各物质有多少？反应物转化了多少？

**关键推理/观察**（课堂精简版）：

ICE 表 = Initial（初始）→ Change（变化）→ Equilibrium（平衡）

以反应 $\mathrm{N_2O_4(g) \rightleftharpoons 2NO_2(g)}$ 为例，设起始 $\mathrm{N_2O_4}$ 分压为 $p_0$，转化率为 $\alpha$：

| | $\mathrm{N_2O_4}$ | $\mathrm{NO_2}$ |
|:---|:---|:---|
| I（初始）| $p_0$ | $0$ |
| C（变化）| $-p_0\alpha$ | $+2p_0\alpha$ |
| E（平衡）| $p_0(1-\alpha)$ | $2p_0\alpha$ |

代入 $K_p$ 表达式：
$$K_p = \frac{(p_{\mathrm{NO_2}})^2}{p_{\mathrm{N_2O_4}}} = \frac{(2p_0\alpha)^2}{p_0(1-\alpha)} = \frac{4p_0\alpha^2}{1-\alpha}$$

解此方程即可求 $\alpha$。

**转化率定义**：
$$\alpha = \frac{\text{转化量}}{\text{起始量}} \times 100\%$$

> [!info]- ICE 表完整步骤与检查清单（课后复习用）
>
> **步骤**：
> 1. 写出配平的化学方程式
> 2. 确定初始量（浓度或分压）
> 3. 设未知数（$x$ 或 $\alpha$），按计量比写变化量
> 4. 写出平衡量 = 初始量 + 变化量
> 5. 代入 $K$ 表达式，建立方程求解
> 6. 验证：回代检查 $Q = K$
>
> **检查清单**：
> - 变化量符号正确（反应物减少为负，产物增加为正）
> - 变化量按化学计量比分配
> - 代入的是平衡量，不是初始量
> - 舍去不合理的根（浓度为负、转化率 > 100%）

### 2.5 多重平衡规则与 $K_{\mathrm{sp}}$、$K_{\mathrm{稳}}$ 入口

**从问题出发**：如果总反应是几个分反应的加和，总反应的 $K$ 怎么求？沉淀溶解和配位反应的平衡常数有什么特殊名字？

**关键推理/观察**（课堂精简版）：

**多重平衡规则**：
- 反应相加 → $K$ 相乘：$K_{\text{总}} = K_1 \times K_2$
- 反应相减 → $K$ 相除：$K_{\text{总}} = K_1 / K_2$
- 系数变 $n$ 倍 → $K$ 变 $n$ 次方：$K' = K^n$

**本质**：$\Delta G^{\circ}$ 具有加和性，$\Delta G^{\circ} = -RT\ln K^{\circ}$，故 $K$ 表现为乘除关系。

**$K_{\mathrm{sp}}$（溶度积）**：
$$\mathrm{AgCl(s) \rightleftharpoons Ag^+(aq) + Cl^-(aq)}$$
$$K_{\mathrm{sp}} = [\mathrm{Ag^+}][\mathrm{Cl^-}]$$
（纯固体不写入表达式）

**$K_{\mathrm{稳}}$（稳定常数/累积稳定常数 $\beta_n$）**：
$$\mathrm{Ag^+(aq) + 2NH_3(aq) \rightleftharpoons [Ag(NH_3)_2]^+(aq)}$$
$$\beta_2 = K_{\mathrm{稳}} = \frac{[[\mathrm{Ag(NH_3)_2}]^+]}{[\mathrm{Ag^+}][\mathrm{NH_3}]^2}$$

> **统一规律**：$K_{\mathrm{sp}}$ 和 $K_{\mathrm{稳}}$ 本质上都是平衡常数，都遵循 $Q/K$ 判断规则，都只与温度有关。

## 三、课堂例题

### 例题 1（⭐⭐ 基础）：$K$ 表达式书写与 $Q/K$ 判断

**题目**：对于反应 $\mathrm{N_2(g) + 3H_2(g) \rightleftharpoons 2NH_3(g)}$：
(1) 写出标准平衡常数 $K^{\circ}$ 的表达式；
(2) 某时刻测得 $p_{\mathrm{N_2}} = 1.0\ \mathrm{atm}$，$p_{\mathrm{H_2}} = 3.0\ \mathrm{atm}$，$p_{\mathrm{NH_3}} = 2.0\ \mathrm{atm}$（标准态 $p^{\circ} = 1\ \mathrm{atm}$），已知该温度下 $K^{\circ} = 0.5$，判断反应方向。

**分析提示**：
- 第一步：确认化学方程式已配平，产物在分子、反应物在分母
- 第二步：$Q$ 表达式中各分压要除以标准态
- 第三步：比较 $Q$ 与 $K$ 的大小

**完整解答路径**：

(1) $K^{\circ} = \dfrac{(p_{\mathrm{NH_3}}/p^{\circ})^2}{(p_{\mathrm{N_2}}/p^{\circ})(p_{\mathrm{H_2}}/p^{\circ})^3}$

(2) 计算 $Q$：
$$Q = \frac{(2.0/1.0)^2}{(1.0/1.0)(3.0/1.0)^3} = \frac{4}{1 \times 27} = 0.148$$

比较：$Q = 0.148 < K^{\circ} = 0.5$，反应**正向自发进行**。

> 💡 **课堂任务**：学生 2 分钟独立完成，教师抽答检查 $Q$ 表达式书写是否正确

---

### 例题 2（⭐⭐ 基础）：Le Chatelier 原理与条件辨析

**题目**：反应 $\mathrm{2NO_2(g) \rightleftharpoons N_2O_4(g)}$（无色）$\quad \Delta H < 0$（放热）

判断下列操作对平衡的影响（正向/逆向/不移动），并说明原因：
(1) 恒温恒容下充入 $\mathrm{Ar}$ 气；
(2) 恒温恒压下充入 $\mathrm{Ar}$ 气；
(3) 恒温下将容器体积压缩为原来的一半；
(4) 升高温度。

**分析提示**：
- 充惰性气体时，先判断是恒容还是恒压
- 恒容：各组分分压不变；恒压：总体积膨胀，各组分分压降低
- 温度改变：$K$ 本身变化，需结合 $\Delta H$ 符号判断

**完整解答路径**：

(1) **恒温恒容充入 Ar：不移动**
- 各组分分压 $p_i = n_iRT/V$ 不变（$n_i$、$V$、$T$ 均未变），$Q$ 不变。

(2) **恒温恒压充入 Ar：逆向移动（向 $\mathrm{NO_2}$ 方向）**
- 总体积 $V$ 增大，各组分分压降低，等效于减压。
- 向气体分子数增多方向移动（$1 \to 2$，逆向）。

(3) **体积压缩为一半：正向移动（向 $\mathrm{N_2O_4}$ 方向）**
- 各组分分压加倍，$Q$ 变化，向气体分子数减少方向移动（$2 \to 1$，正向）。

(4) **升高温度：逆向移动（向 $\mathrm{NO_2}$ 方向）**
- 放热反应，升温 $K$ 减小，$Q > K$，逆向移动。
- 现象：体系颜色加深（$\mathrm{NO_2}$ 为红棕色）。

> 💡 **课堂任务**：学生分组讨论 (1)(2) 的区别，教师板书对比

### 例题 3（⭐⭐ 基础）：ICE 表与转化率计算

**题目**：在 700 K 时，反应 $\mathrm{H_2(g) + I_2(g) \rightleftharpoons 2HI(g)}$ 的平衡常数 $K_c = 50.3$。将 $1.00\ \mathrm{mol}$ 的 $\mathrm{H_2}$ 和 $1.00\ \mathrm{mol}$ 的 $\mathrm{I_2}$ 充入 $1.00\ \mathrm{dm^3}$ 的密闭容器中，求：
(1) 平衡时各物质的浓度；
(2) $\mathrm{H_2}$ 的平衡转化率。

**分析提示**：
- 先列 ICE 表，注意变化量按计量比 $1:1:2$
- 反应前后气体分子数不变（$\Delta n = 0$），压强变化不影响平衡位置
- 转化率 = 转化量 / 起始量

**完整解答路径**：

设平衡时 $\mathrm{H_2}$ 消耗了 $x\ \mathrm{mol \cdot dm^{-3}}$：

| | $\mathrm{H_2}$ | $\mathrm{I_2}$ | $\mathrm{HI}$ |
|:---|:---|:---|:---|
| I（初始）| $1.00$ | $1.00$ | $0$ |
| C（变化）| $-x$ | $-x$ | $+2x$ |
| E（平衡）| $1.00 - x$ | $1.00 - x$ | $2x$ |

代入平衡常数表达式：
$$K_c = \frac{[\mathrm{HI}]^2}{[\mathrm{H_2}][\mathrm{I_2}]} = \frac{(2x)^2}{(1.00-x)(1.00-x)} = 50.3$$

$$\frac{2x}{1.00-x} = \sqrt{50.3} = 7.09$$

$$2x = 7.09 - 7.09x \quad \Rightarrow \quad 9.09x = 7.09 \quad \Rightarrow \quad x = 0.780\ \mathrm{mol \cdot dm^{-3}}$$

(1) 平衡浓度：
- $[\mathrm{H_2}] = [\mathrm{I_2}] = 1.00 - 0.780 = 0.22\ \mathrm{mol \cdot dm^{-3}}$
- $[\mathrm{HI}] = 2 \times 0.780 = 1.56\ \mathrm{mol \cdot dm^{-3}}$

(2) $\mathrm{H_2}$ 的平衡转化率：
$$\alpha = \frac{x}{1.00} \times 100\% = \frac{0.780}{1.00} \times 100\% = 78.0\%$$

**验证**：回代 $Q = \dfrac{(1.56)^2}{(0.22)(0.22)} = 50.2 \approx K_c$（舍入误差允许）。

> 💡 **课堂任务**：学生跟随教师同步填写 ICE 表，独立完成代数求解

### 例题 4（⭐⭐⭐ 拓展）：$K_p$ 与转化率、$K_p/K_c$ 换算综合

**题目**：反应 $\mathrm{N_2O_4(g) \rightleftharpoons 2NO_2(g)}$ 在 325 K 时 $K_p = 0.14$（单位：atm）。起始时只有 $\mathrm{N_2O_4}$，分压为 $1.00\ \mathrm{atm}$，求：
(1) 平衡时各物质的分压；
(2) $\mathrm{N_2O_4}$ 的转化率；
(3) 若改用 $K_c$ 计算，$K_c$ 等于多少？（$R = 0.0821\ \mathrm{L \cdot atm \cdot mol^{-1} \cdot K^{-1}}$）

**分析提示**：
- 设转化率 $\alpha$，用 ICE 表建立 $K_p$ 表达式
- 注意 $K_p = \dfrac{4p_0\alpha^2}{1-\alpha}$ 的推导
- $K_p = K_c(RT)^{\Delta n}$，$\Delta n = 2-1 = 1$

**完整解答路径**：

(1)(2) 设 $\mathrm{N_2O_4}$ 转化率为 $\alpha$：

| | $\mathrm{N_2O_4}$ | $\mathrm{NO_2}$ |
|:---|:---|:---|
| I | $1.00$ | $0$ |
| C | $-1.00\alpha$ | $+2.00\alpha$ |
| E | $1.00(1-\alpha)$ | $2.00\alpha$ |

$$K_p = \frac{(2.00\alpha)^2}{1.00(1-\alpha)} = \frac{4\alpha^2}{1-\alpha} = 0.14$$

$$4\alpha^2 = 0.14(1-\alpha) \quad \Rightarrow \quad 4\alpha^2 + 0.14\alpha - 0.14 = 0$$

用求根公式：
$$\alpha = \frac{-0.14 + \sqrt{0.0196 + 2.24}}{8} = \frac{-0.14 + 1.50}{8} = 0.170$$

（舍去负根）

- 平衡分压：$p_{\mathrm{N_2O_4}} = 1.00(1-0.170) = 0.83\ \mathrm{atm}$，$p_{\mathrm{NO_2}} = 2.00 \times 0.170 = 0.34\ \mathrm{atm}$
- 转化率：$\alpha = 17.0\%$

(3) $K_c$ 换算：
$$K_p = K_c(RT)^{\Delta n} \quad \Rightarrow \quad K_c = \frac{K_p}{(RT)^{\Delta n}} = \frac{0.14}{(0.0821 \times 325)^1} = \frac{0.14}{26.7} = 5.2 \times 10^{-3}\ \mathrm{mol \cdot dm^{-3}}$$

> 💡 **课堂任务**：教师板书分步演算，学生记录关键步骤；$K_c$ 换算部分可课后独立完成

## 四、即时演练

**演练题**（3 分钟独立完成）：

反应 $\mathrm{PCl_5(g) \rightleftharpoons PCl_3(g) + Cl_2(g)}$ 在某温度下达平衡。判断下列操作对平衡的影响：
(1) 恒温恒容下充入 $\mathrm{N_2}$ 气；
(2) 恒温恒压下充入 $\mathrm{N_2}$ 气；
(3) 恒温下缩小容器体积；
(4) 恒容条件下再充入 $\mathrm{Cl_2}$。

**自检答案**：
- (1) 不移动（各组分分压不变）
- (2) 正向移动（等效于减压，向分子数增多方向）
- (3) 逆向移动（加压，向分子数减少方向）
- (4) 逆向移动（$Q > K$）

---

## 五、本节形成的核心结论

1. **结论一**：判断反应方向的核心流程是"写 $Q$ → 比 $K$ → 定方向"。$Q < K$ 正向，$Q = K$ 平衡，$Q > K$ 逆向。
2. **结论二**：温度是唯一改变 $K$ 的因素。吸热反应升温 $K$ 增大，放热反应升温 $K$ 减小。浓度/压强改变只改 $Q$ 不改 $K$。
3. **结论三**：恒容充惰性气体 → 各组分分压不变 → 平衡不移动；恒压充惰性气体 → 总体积膨胀 → 等效于减压 → 向分子数增多方向移动。
4. **结论四**：ICE 表是平衡计算的标准工具：I（初始）→ C（变化，按计量比）→ E（平衡），代入 $K$ 求解。

## 六、⚠️ 易错点预警

| 错误想法 | 为什么错 | 正确理解 |
|:---|:---|:---|
| "充惰性气体 = 加压 = 平衡移动" | 恒容充惰气时各组分分压不变，$Q$ 不变 | 必须区分恒容 vs 恒压：恒容不移动，恒压等效于减压 |
| "升温总是让平衡向吸热方向移动，所以 $K$ 不变" | 温度改变的是 $K$ 本身，不只是移动方向 | 吸热升温 $K$ 增大，放热升温 $K$ 减小 |
| "ICE 表变化量随便设，不用管计量比" | 变化量必须按化学计量数配平 | 如 $1:3:2$ 的反应，变化量应为 $-x$、$-3x$、$+2x$ |
| "$K_p$ 与 $K_c$ 换算时 $\Delta n$ 符号搞反" | $\Delta n$ = 生成物气体分子数 - 反应物气体分子数 | 记忆：$\Delta n > 0$ 则 $K_p > K_c$ |
| "纯固体/纯液体写入 $K$ 表达式" | 纯凝聚相活度近似为 1，不写入表达式 | 多相反应中只写入气态和溶液态物质 |
| "转化率与 $K$ 是一回事" | $K$ 只与温度有关，转化率还与起始条件有关 | 同一反应、同一温度，不同起始浓度可能转化率不同 |

## 附录：板书速查（学生课后整理用）

```
化学平衡工具箱
├── 方向判断：写 Q → 比 K → 定方向
│   Q < K：正向    Q = K：平衡    Q > K：逆向
├── 温度影响：吸热↑K增，放热↑K减（唯一改K的因素）
├── 条件辨析：
│   ├── 恒容充惰气 → 分压不变 → 不移动
│   ├── 恒压充惰气 → 体积分压降 → 向分子多方向移动
│   └── 加压 → 向分子少方向移动
├── ICE表：I(始) → C(变，按计量比) → E(平)
├── 转化率 α = 转化量 / 起始量
├── Kp/Kc 换算：Kp = Kc(RT)^Δn，Δn = 产物气体系数和 - 反应物气体系数和
└── 多重平衡：加→乘，减→除，系数→指数
```
