# 活性中间体与反应机理基础 · 速查卡

> 基础班你学过"写箭头推机理"——本卡把机理判断从**会写箭头**升级到**条件→中间体→命运→产物**的四步判断系统。看到条件先判断最可能生成哪类中间体，再用中间体的"性格"解释一切。

---

## 一、机理判断 · 四步流程

```
拿到有机反应题 → 四步机理判断
    │
    ├─ Step 1：条件最像在生成哪类中间体？（2 min）
    │   ├─ 酸 + 醇/烯 → C⁺（碳正离子）
    │   ├─ 强碱 + 活泼H → C⁻（碳负离子/烯醇负离子）
    │   ├─ 自由基引发剂 + hν/Δ → R•（自由基）
    │   ├─ CHX₃/CH₂N₂ + 碱 → :CR₂（卡宾）
    │   └─ 卤苯 + 强碱(NaNH₂) → 苯炔
    │
    ├─ Step 2：这个中间体稳定吗？会不会重排或继续链增长？（2 min）
    │   ├─ 稳定性顺序 → 决定区域选择性
    │   └─ 重排倾向 → 可能的副产物
    │
    ├─ Step 3：这是中间体控制还是过渡态/速控步控制？（1 min）
    │   └─ 中间体控制 = Hammond 假说：中间体稳定→过渡态像中间体→活化能低
    │
    └─ Step 4：中间体的最终命运是什么？（3 min）
        ├─ C⁺ → 被亲核试剂捕获 / 消除 β-H / 重排
        ├─ C⁻ → 亲核进攻 / 消除 / 电子转移
        ├─ R• → 偶联 / 夺取 H/X / 加成到 π 键 / β-断裂
        ├─ :CR₂ → 插入 / 加成到 π 键 / 环丙烷化
        └─ 苯炔 → 加成（两侧都可）
```

## 二、五大中间体速查表

| 中间体 | 电子结构 | 生成条件 | 典型命运 | 稳定性顺序 |
|:---|:---|:---|:---|:---|
| **碳正离子 C⁺** | sp², 空 p | 酸/R–X 解离/重氮分解 | 被亲核捕获/消除/重排 | 3° > 苄基≈烯丙基 > 2° > 1° > CH₃⁺ |
| **碳负离子 C⁻** | sp³, 孤对（或 sp² 共轭） | 强碱+活泼H | 亲核进攻/消除/单电子转移 | 1° > 2° > 3°（但烯丙基/苄基C⁻共轭稳定） |
| **自由基 R•** | sp²-ish, 单电子 p | 引发剂+hν/Δ | 偶联/夺取/加成/β-断裂 | 苄基≈烯丙基 > 3° > 2° > 1° > CH₃• |
| **卡宾 :CR₂** | sp², 孤对+空p（单线态）或两单电子（三线态） | CHX₃+碱 / CH₂N₂+hν | 插入/环丙烷化/重排 | 取决于取代基和多重态 |
| **苯炔** | 张力三键+芳香性 | 卤苯+强碱 | 加成（两侧） | — |

## 三、碳正离子（C⁺）深度

### 3.1 稳定性三支柱

```
C⁺ 稳定性决定因素：
  ├─ 超共轭：C–H/C–C σ → 空 p（3° > 2° > 1°）
  ├─ 共轭：π 体系 → 空 p（烯丙基C⁺、苄基C⁺）
  └─ 杂原子邻位：O/N 孤对 → 空 p（氧鎓/亚胺正离子的稳定化）
```

### 3.2 C⁺ 的三种命运

| 命运 | 条件 | 产物 | 例子 |
|:---|:---|:---|:---|
| **被亲核捕获** | 有亲核试剂（H₂O, X⁻, ROH） | 加成/取代产物 | SN1 → 醇 |
| **消除 β-H** | 无强亲核/高温 | 烯烃（E1） | 醇+酸+Δ→烯 |
| **重排** | 邻位有可迁移基团（H/R/Ar） | 重排产物 | Pinacol/W-M 重排 |

**Hammond 假说用于 C⁺**：放热步→过渡态像反应物；吸热步→过渡态像产物（C⁺）。因此 C⁺ 越稳定→形成 C⁺ 的过渡态能量越低→反应越快。

## 四、碳负离子（C⁻）深度

### 4.1 稳定性反转

```
C⁻ 稳定性顺序：1° > 2° > 3°（与 C⁺ 完全相反！）

  原因：烷基是给电子基→C⁻ 上负电荷被烷基推得更集中→越不稳定

  ⚠️ 例外：烯丙基C⁻ 和苄基C⁻ 通过共轭离域→异常稳定
        烯醇负离子（C⁻ α 位连 C=O）→共轭离域到 C=O →大幅稳定
```

### 4.2 pKa 速查——烯醇负离子的酸性

```
羰基 α-H 的 pKa：
  醛 α-H ~17
  酮 α-H ~20
  酯 α-H ~25
  1,3-二酮（β-二酮）α-H ~9（两个羰基共轭→烯醇负离子大幅稳定！）
  活泼亚甲基（两个吸电子基之间）~10-13

  碱的选择：NaOH(pKa~15.7)→可脱质子醛/酮/β-二酮α-H；
            LDA (pKa~36)→可脱质子任何羰基α-H（动力学烯醇）
```

## 五、自由基（R•）深度

### 5.1 自由基链反应三步

```
引发：R–X + hν/Δ → R• + X•（或引发剂分解）
链增长：R• + 反应物 → 产物 + 新R•
链终止：R• + R• → R–R（偶联）

自由基稳定性的"两面性"：
  ├─ 类似于C⁺：3° > 2° > 1°（超共轭）
  └─ 苄基/烯丙基自由基异常稳定（π共轭→自旋离域）
```

### 5.2 BDE（键解离能）速判

```
BDE 越小 → 键越容易均裂 → 该自由基越稳定

BDE(C–H) 大小顺序（kJ/mol）：
  PhCH₂–H (~370) < CH₂=CHCH₂–H (~370) < 3°C–H (~400) < 2°C–H (~410) < 1°C–H (~420) < CH₃–H (~440)

  自由基链反应的方向：朝着生成更稳定自由基的方向→BDE 梯度下降
```

## 六、卡宾（:CR₂）深度

### 6.1 单线态 vs 三线态

| 维度 | 单线态卡宾 | 三线态卡宾 |
|:---|:---|:---|
| **电子排布** | σ²（孤对） + p⁰（空） | σ¹（孤对） + p¹（单电子） |
| **生成条件** | CHX₃ + 碱（热/碱条件） | CH₂N₂ + hν（光化学条件） |
| **对 C=C 加成** | **stereospecific syn**（协同） | **非 stereospecific**（分步，双自由基） |
| **插入反应** | 协同插入 C–H | 分步→夺取 H→再偶联 |

### 6.2 Simmons-Smith 环丙烷化

```
CH₂I₂ + Zn(Cu) → ICH₂ZnI（类卡宾试剂）
  └─ 与烯烃→syn 环丙烷化（stereospecific）
  └─ 保留了烯烃的立体化学：顺式烯烃→顺式环丙烷
```

---

## 七、苯炔深度

```
苯炔 = 高度张力环中的"额外 π 键"（在 σ 骨架的侧面重叠）

生成方式：
  ├─ 卤苯 + NaNH₂/NH₃(l)（经典）
  └─ 邻二卤苯 + BuLi（温和）

反应性：
  ├─ 亲核加成：Nu⁻ 加到苯炔→苯基负离子→质子化
  ├─ 亲电加成：E⁺ 加到苯炔→苯基正离子→被亲核捕获
  └─ 环加成：[4+2] DA 反应（苯炔是非常好的亲双烯体）

关键判断：苯炔两侧都可被加成→产物可能是一对区域异构体
```

## 八、常见易错点

| 错误 | 纠正 |
|:---|:---|
| 认为 C⁻ 稳定性和 C⁺ 一样（3°最稳） | **C⁻ 稳定性顺序完全相反：1° > 2° > 3°**——烷基给电子使负电荷更集中 |
| 看到条件就写 C⁺ 路径 | 先判断：酸/极性条件→C⁺；强碱条件→C⁻；hν+引发剂→R• |
| C⁺ 重排总是发生 | 重排需要邻位有可迁移基团且迁移后 C⁺ 更稳定→不是所有 C⁺ 都会重排 |
| 自由基链反应看到 Br₂ 产率就会好 | 自由基链中 Br₂ 太活泼→副反应多→选择性差→AIBN+Bu₃SnH 是更好选择 |
| 卡宾只有一种反应模式 | 单线态和三线态卡宾立体化学结果完全相反——判断之前先分多重态 |

## 九、例题速练

### 例 1：条件→中间体

**题**：(CH₃)₃CBr + Ag⁺/H₂O → (CH₃)₃COH + (CH₃)₂C=CH₂。什么中间体？为什么两种产物？

**速解**：
1. Ag⁺ 拉走 Br⁻ → C⁺（碳正离子）
2. C⁺ 的命运：被 H₂O 捕获→(CH₃)₃COH；消除 β-H→(CH₃)₂C=CH₂
3. 3° C⁺ 稳定性好→不重排
4. **两种产物 = C⁺ 的两种命运（捕获 vs 消除）**

### 例 2：自由基链判断

**题**：CH₃CH₂CH=CH₂ + HBr + AIBN/Δ → 主产物是 CH₃CH₂CH₂CH₂Br（反 Markovnikov）。为什么？

**速解**：
1. AIBN+Δ→自由基引发剂→**自由基链反应**
2. Br• 加到端碳→生成 2° 自由基（CH₃CH₂CH•–CH₂Br）→ 比 1° 自由基稳定
3. 自由基从 HBr 提取 H→产物
4. **自由基加成=反 Markovnikov**——因为决定区域选择的是自由基稳定性不是 C⁺ 稳定性
