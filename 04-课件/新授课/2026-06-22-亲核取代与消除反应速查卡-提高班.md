# 亲核取代与消除反应 · 速查卡

> 基础班你背过"SN2=一步、SN1=两步"——本卡把 SN/E 从**四条定义**升级到**竞争判断系统**：同一底物换试剂/溶剂/温度，为什么会从 SN2 转到 E2？

---

## 一、SN/E 竞争 · 五问统一判断法

```
拿到底物+试剂+条件 → 五问导航
    │
    ├─ 问 1：底物几级？
    │   ├─ 1° → SN2 或 E2（取决于试剂，几乎不走 SN1/E1）
    │   ├─ 2° → 四者都可能（最复杂！试剂+溶剂+温度决定）
    │   └─ 3° → SN1/E1 或 E2（几乎不走 SN2——位阻太大）
    │
    ├─ 问 2：试剂更亲核还是更碱？
    │   ├─ 强亲核 + 弱碱（I⁻, RS⁻, CN⁻）→ SN2
    │   ├─ 强碱 + 弱亲核（t-BuOK, LDA）→ E2
    │   └─ 既是强亲核又是强碱（OH⁻, OR⁻）→ 需看后续问题
    │
    ├─ 问 3：溶剂在帮哪边？
    │   ├─ 极性非质子（DMSO, DMF, CH₃CN）→ 帮 SN2（裸阴离子亲核性强）
    │   └─ 极性质子（H₂O, ROH）→ 帮 SN1/E1（稳定C⁺）+ 帮 E2（碱性↑）
    │
    ├─ 问 4：温度在推哪边？
    │   ├─ 低温 → SN2 > E2（SN2 ΔS‡ 更负→低温有利）
    │   └─ 高温 → E2 > SN2（E2 产物更多→熵增有利消除）
    │
    └─ 问 5：有没有几何/重排/邻基参修正？
        ├─ E2 需要反式共平面（不满足→不走E2）
        ├─ C⁺ 可能重排（SN1/E1 需检查重排产物）
        └─ 邻基参与→构型保留（看似SN1实为两次SN2）
```

## 二、SN1 vs SN2 核心对比

| 维度 | SN1 | SN2 |
|:---|:---|:---|
| **动力学** | 一级：v = k[底物] | 二级：v = k[底物][Nu] |
| **决速步** | C–LG 断裂→C⁺ | C–Nu 成键 + C–LG 断裂（协同） |
| **底物偏好** | 3° > 2° > 1° | 1° > 2° > 3°（位阻！） |
| **立体化学** | 外消旋化 | **构型翻转**（Walden 翻转） |
| **溶剂偏好** | 极性质子（稳定C⁺） | 极性非质子（裸阴离子亲核强） |
| **重排风险** | 有（C⁺可重排） | 无 |
| **离去基团** | 好 LG 必需 | 好 LG 必需 |

---

## 三、E1 vs E2 核心对比

| 维度 | E1 | E2 |
|:---|:---|:---|
| **动力学** | 一级：v = k[底物] | 二级：v = k[底物][碱] |
| **决速步** | C–LG 断裂→C⁺（同 SN1） | C–H 断裂 + C–LG 断裂 + C=C 形成（协同） |
| **底物偏好** | 3° > 2° | 3° > 2° > 1°（但 1° 也可） |
| **产物取向** | Saytzeff（热力学稳定烯烃） | Saytzeff（通常）或 Hofmann（大位阻碱） |
| **立体要求** | 无刚性要求 | **反式共平面**（H–C 与 C–LG ~180°） |
| **重排风险** | 有（C⁺可重排） | 无 |

---

## 四、SN2 vs E2 竞争速判（最常见场景）

| 底物 | 试剂特征 | 主路径 | 原因 |
|:---|:---|:---:|:---|
| 1° | 强亲核弱碱（I⁻, RS⁻） | SN2 | 弱碱不利于E2；1°底物 SN2 极快 |
| 1° | 强碱弱亲核（t-BuOK） | E2 | 大位阻碱不利SN2；1°也可E2 |
| 2° | 强亲核弱碱 | SN2 ≈ E2 | 试剂+温度+溶剂微调 |
| 2° | 强碱弱亲核 | E2 | 碱促进消除 |
| 3° | 任何强碱 | E2 | 3°位阻太大几乎不走SN2 |
| 3° | 弱碱/中性 | SN1/E1 | 无强碱时走单分子路径 |
| 烯丙基/苄基 | 任何 | SN2 加速 | 共轭稳定过渡态→SN2异常快 |

## 五、邻基参与（Neighboring Group Participation）

```
邻基参与 = 分子内的 SN2——看起来像 SN1 但构型保留

必要条件：
  1. 邻位有含孤对电子的基团（S, O, N, Br, COO⁻…）
  2. 能形成 3 元或 5 元环中间体（最稳定）

典型例子：
  硫芥子气：ClCH₂CH₂SCH₂CH₂Cl
    └─ S 的孤对电子→分子内 SN2 进攻→三元环锍盐中间体
    └─ 然后 Cl⁻ 从背面开环→总结果：构型保留（两次SN2=一次翻转+一次翻转=保留！）

  邻基参与速判：
  ├─ 产物构型保留 + 底物是手性 → 很可能有邻基参与
  ├─ 反应速率异常快（比预期 SN1 快）→ 邻基参与加速
  └─ 有 3 元/5 元环中间体可形成→检查邻位是否有 S/O/N/Br
```

## 六、离去基团与亲核性/碱性

### 6.1 离去基团能力顺序

```
好离去基（弱碱、稳定负离子）：
  TsO⁻ > I⁻ > Br⁻ > Cl⁻ > F⁻ > OH⁻ > NH₂⁻

  TsO⁻ 最好（负电荷高度离域在磺酸基上）
  OH⁻ 和 NH₂⁻ 是坏离去基——需要先质子化（变成 H₂O 和 NH₃）才能走
```

### 6.2 亲核性 vs 碱性

| 亲核性强 | 亲核性弱 | 碱性强 | 碱性弱 |
|:---|:---|:---|:---|
| I⁻, RS⁻, CN⁻ | F⁻, ROH | t-BuO⁻, LDA, NH₂⁻ | I⁻, ROH |

**亲核性≠碱性**：I⁻ 亲核性很强但碱性很弱（HI 是强酸→I⁻ 是极弱碱）。大位阻碱（t-BuOK, LDA）碱性强但亲核性弱（位阻挡住背面进攻）。

## 七、常见易错点

| 错误 | 纠正 |
|:---|:---|
| 认为 SN2 和 E2 是互斥的 | 同一反应中 SN2 和 E2 常同时发生→主产物由反应条件微调 |
| OH⁻ 一定是好亲核试剂 | OH⁻ 既是亲核又是碱——结果是 SN2 和 E2 的混合物，除非底物是 1°（SN2 为主） |
| 3° 底物绝对不走 SN2 | 对——位阻太大。但烯丙基 3° 因为共轭可能通过 SN2'（烯丙基型 SN2）反应 |
| 极性溶剂帮 SN1 所以一定不利 SN2 | 极性非质子溶剂（DMSO, DMF）非常有利 SN2——它们溶剂化阳离子但不溶剂化阴离子→阴离子"裸"着→亲核性极强 |

## 八、例题速练

### 例 1：五问判断

**题**：(CH₃)₃CBr + CH₃ONa/CH₃OH → 主产物？

**速解**：
1. 底物 3° → 不走 SN2
2. CH₃O⁻ = 强碱+中等亲核 → 可能 E2
3. CH₃OH = 极性质子溶剂 → 帮 SN1/E1 + 也帮碱的 E2
4. 3°底物+强碱 → **E2 为主**
5. CH₃O⁻ 不是大位阻碱 → Saytzeff 取向→主产物 (CH₃)₂C=CH₂

### 例 2：邻基参与

**题**：(R)-CH₃CHBrCH₂SCH₃ + NaOH → 产物构型？

**速解**：产物构型保留（R→R）→ 邻基参与！S 孤对电子→分子内 SN2 形成三元环锍盐→OH⁻ 背面开环→两次翻转=构型保留。如果只写常规 SN2→产物应该翻转，但实验结果是保留→邻基参与是唯一解释。

## 九、自查清单

- [ ] 五问判断法能默推（底物级数→试剂→溶剂→温度→几何修正）
- [ ] SN1/SN2/E1/E2 四种机理的核心差异表（动力学/立体/底物/溶剂）能默列
- [ ] 1° 底物 SN2 vs E2 的试剂依赖关系
- [ ] 2° 底物为什么最复杂——需要通过试剂+溶剂+温度微调
- [ ] 3° 底物的判断流程：有强碱→E2；无强碱→SN1/E1
- [ ] 邻基参与的三元/五元环条件 + 构型保留的判断
- [ ] 亲核性 ≠ 碱性（I⁻ 是好亲核但是弱碱；t-BuOK 是强碱但亲核弱）
- [ ] E2 反式共平面——环己烷先画椅式、找 a-H
- [ ] 极性非质子溶剂（DMSO/DMF）加速 SN2 的原因
- [ ] 离去基团能力顺序：TsO⁻ > I⁻ > Br⁻ > Cl⁻
