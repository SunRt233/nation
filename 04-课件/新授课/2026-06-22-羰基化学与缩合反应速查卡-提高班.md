# 羰基化学与缩合反应 · 速查卡

> 基础班你背过"Aldol=醛+碱→β-羟基醛"——本卡把羰基化学从**背人名/背试剂**升级到**供体-受体-离去基-后续分流**四层统一框架。

---

## 一、羰基反应四层框架

```
羰基的反应 → 四层判断
    │
    ├─ 1. 亲核加成（醛/酮 → 醇）
    │   └─ Nu⁻ 进攻 C=O 碳 → 四面体中间体 → 质子化→醇
    │   └─ 没有离去基→加成即终点（除非后续脱水）
    │
    ├─ 2. 酰基取代（羧酸衍生物 → 另一衍生物）
    │   └─ Nu⁻ 进攻 → 四面体中间体 → LG⁻ 离去 → 取代产物
    │   └─ 有离去基→加成-消除=净取代
    │
    ├─ 3. α 位缩合（烯醇/烯醇负离子→C–C 键构建）
    │   └─ α-H 酸性 + 碱 → 烯醇负离子（亲核）→ 攻另一 C=O
    │   └─ 这是"羰基从受体变供体"的关键
    │
    └─ 4. 极性翻转（Umpolung）
        └─ 羰基碳从亲电翻转为亲核
        └─ CN⁻/噻唑盐/二噻烷 = 三种翻转策略
```

---

## 二、亲核加成 vs 酰基取代 · 一图区分

```
         O                            O⁻
        ║                           /
  R — C — G  +  Nu⁻  →  R — C — G  →  产物？
                                    / \
                                   Nu
  
  G = H/R（无离去基）→ 只能走到四面体中间体 → 质子化 → 醇（亲核加成）
  G = Cl/OR/NR₂（好离去基）→ 四面体中间体 → G⁻ 离去 → 回到 C=O（酰基取代）
  
  核心判断：加成后有没有好离去基？
  ├─ 有 → 酰基取代（加成-消除）
  └─ 没有 → 亲核加成（停留在醇）
```

**酰基取代活性顺序**（从快到慢）：酰氯 > 酸酐 > 酯 > 酰胺
- 酰氯：Cl⁻ 极好离去基 → 瞬间完成
- 酰胺：NH₂⁻ 极坏离去基 → 需要酸催化（质子化 NH₂→NH₃ 好离去基）

## 三、α 位缩合 · 供体×受体矩阵

### 3.1 核心人物：烯醇负离子

```
碱 + 羰基化合物（有 α-H）→ 烯醇负离子

  动力学烯醇：LDA/低温 → 位阻小的 α 位去质子 → 较少取代的烯醇
  热力学烯醇：RO⁻/高温 → 更稳定的（更多取代的）烯醇
  
  提高班重点：交叉 Aldol 必须用动力学烯醇（预先制好）→ 否则混合物
```

### 3.2 缩合反应矩阵

| 反应 | 供体（亲核） | 受体（亲电） | 产物 | 可控？ |
|:---|:---|:---|:---|:---:|
| **Aldol** | 醛/酮 α 位 | 醛/酮 C=O | β-羟基羰基 | 交叉 Aldol 需预制备烯醇 |
| **Claisen** | 酯 α 位 | 酯 C=O | β-酮酯 | 交叉 Claisen 需预制备 |
| **Dieckmann** | 分子内 Claisen | — | 环状 β-酮酯 | 5/6 元环最优 |
| **Knoevenagel** | 活泼亚甲基 | 醛/酮 | α,β-不饱和酯/腈 | 弱碱（哌啶）催化 |
| **Michael** | 烯醇负离子/烯胺 | α,β-不饱和羰基 | 1,5-二羰基 | 供体酸性强即可 |
| **Mannich** | 烯醇/烯胺 | 亚胺正离子 | β-氨基羰基 | 需预制亚胺正离子 |
| **Perkin** | 酸酐+碱（当量烯醇） | 芳醛 | 肉桂酸 | 高温、碱催化 |
| **Stobbe** | 丁二酸酯+t-BuOK | 醛/酮 | 亚烷基丁二酸 | 丁二酸酯 α 位酸性更强 |

## 四、活泼亚甲基工具箱

```
活泼亚甲基 = 两个吸电子基中间的 –CH₂–（酸性大幅增强 pKa~10-13）

三大经典试剂：
  ├─ 乙酰乙酸乙酯（CH₃COCH₂COOEt）：pKa~11
  │   └─ 经过 SN2 烷基化→水解→脱羧→甲基酮（"丙酮当量"）
  │
  ├─ 丙二酸二乙酯（CH₂(COOEt)₂）：pKa~13
  │   └─ 经过 SN2 烷基化→水解→脱羧→取代乙酸（"乙酸当量"）
  │
  └─ Knoevenagel：活泼亚甲基+醛/酮→α,β-不饱和化合物
      └─ 丙二酸/氰乙酸酯/丙二腈 等均可做供体
```

**三步经典法**：①碱去质子→烯醇负离子 ②SN2 烷基化→C–C 键 ③水解+脱羧→目标产物

## 五、Cannizzaro 与 Beckmann/BV

| 反应 | 底物 | 条件 | 产物 | 机理本质 |
|:---|:---|:---|:---|:---|
| **Cannizzaro** | 无 α-H 的醛（ArCHO, HCHO） | 浓 NaOH | 醇 + 酸（歧化） | 氢负离子转移（分子间） |
| **Beckmann** | 肟 | 酸 | 酰胺/内酰胺 | N 上 OH→好离去基→R 迁移到 N |
| **Baeyer-Villiger** | 酮 | m-CPBA | 酯 | O 插入→R 迁移→酯 |

**Cannizzaro 交叉版**：ArCHO + HCHO（过量）→ HCHO 被氧化成 HCOO⁻（甲酸），ArCHO 被还原成 ArCH₂OH——因为 HCHO 更适合做氢负离子供体。

## 六、Umpolung（极性翻转）

```
正常的羰基碳 = 亲电（δ⁺）
Umpolung 后的羰基碳 = 亲核（δ⁻ → 攻别的羰基）

三种翻转策略：
  ├─ CN⁻ 催化（Benzoin 缩合）：氰醇→"酰基负离子"当量
  ├─ 噻唑盐催化（Stetter 反应）：NHC 催化→"酰基负离子"当量
  └─ 二噻烷（Corey-Seebach）：保护+H⁺夺取→二噻烷负离子

翻转后 C–C 键构建：原来是"烯醇攻羰基"→现在是"羰基攻羰基"
```

## 七、常见易错点

| 错误 | 纠正 |
|:---|:---|
| 交叉 Aldol 加碱直接混合两个醛/酮 | 两种醛/酮都有 α-H→四种产物！必须预制备动力学烯醇 |
| 羧酸+胺→酰胺（直接加热） | 羧酸的 OH 不是好离去基——先变成酰氯/酸酐/酯再与胺反应 |
| Claisen 缩合用 NaOH | 酯在 NaOH 中水解比缩合快——Claisen 必须用醇钠（NaOEt/EtOH）|
| 乙酰乙酸乙酯的水解产物=丙酮 | 需要水解**再脱羧**才得甲基酮——中间经过 β-酮酸（易脱羧） |
| Mannich 反应与 Aldol 混为一谈 | Mannich 的亲电伙伴是亚胺正离子（不是 C=O）→产物含 N |

## 八、例题速练

### 例 1：交叉 Aldol 可控策略

**题**：如何从环己酮和 PhCHO 定向制备 PhCH=环己酮（α,β-不饱和酮）？

**速解**：
1. 环己酮 + LDA/THF/−78°C → 动力学烯醇锂（预制备）
2. 缓慢加入 PhCHO → 混合 Aldol → β-羟基酮
3. 酸催化脱水 → α,β-不饱和酮
4. 关键：不能用 NaOH/EtOH 直接混合（环己酮自身 Aldol 也会发生）

### 例 2：活泼亚甲基经典三步

**题**：从丙二酸二乙酯合成 PhCH₂CH₂COOH。

**速解**：
1. NaOEt/EtOH → 丙二酸二乙酯烯醇负离子
2. + PhCH₂Br → PhCH₂CH(COOEt)₂（SN2 烷基化）
3. 酸水解 + 加热脱羧 → PhCH₂CH₂COOH（取代丙二酸→β-酮酸→脱羧）
