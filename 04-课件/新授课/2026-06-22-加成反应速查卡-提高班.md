# 加成反应 · 速查卡

> 基础班你背过"马氏规则：H 加在氢多的碳上"——本卡把加成从**背规则**升级到**底物类型×试剂类型×中间体/过渡态共同决定**的判断系统。

---

## 一、加成反应三分流

```
看到 C=C / C≡C / C=O → 加成反应 → 先分三大类
    │
    ├─ 亲电加成（烯烃/炔烃 + 亲电试剂）
    │   └─ 中间体：碳正离子 / 溴鎓离子 / 汞鎓离子
    │   └─ 规则：Markovnikov（通过 C⁺ 稳定性解释）
    │
    ├─ 亲核加成（C=O / α,β-不饱和体系 + 亲核试剂）
    │   ├─ 直接加成到 C=O：醛 > 酮 > 酯（位阻+电子递升）
    │   └─ 共轭加成：1,2 vs 1,4（动力学 vs 热力学）
    │
    └─ 协同加成（无离子中间体）
        ├─ 硼氢化：四中心过渡态→syn 加成→反马氏取向
        ├─ 催化氢化：H₂ 在金属表面 syn 加成
        └─ 卡宾/类卡宾加成：stereospecific（单线态syn；三线态非stereospecific）
```

---

## 二、亲电加成速判表

| 试剂 | 中间体 | 区域选择性 | 立体化学 | 特殊点 |
|:---|:---|:---|:---:|:---|
| **HX** | 碳正离子 | Markovnikov | 非立体专一 | 只有 HBr+过氧化物→反马氏（自由基） |
| **H₂O/H⁺** | 碳正离子 | Markovnikov | 非立体专一 | 醇→可能重排 |
| **Br₂/Cl₂** | 卤鎓离子 | — | **anti** | Br₂ 水→溴醇（H₂O 开环） |
| **Hg(OAc)₂/H₂O** | 汞鎓离子 | Markovnikov | anti（总结果） | NaBH₄ 去汞→醇，无重排 |
| **BH₃/H₂O₂** | 四中心 TS | **反 Markovnikov** | **syn** | B 加端碳→氧化变 OH |

## 三、亲核加成到 C=O · 反应活性递升

```
羰基活性顺序（从低到高）：

  酰胺 < 酯 < 酸酐 < 醛 < 酮 < 酰氯
  
  原因：
  ├─ 位阻：酮 > 醛（酮两边都有烷基→位阻大）
  ├─ 电子：给电子基（酰胺的 NH₂ 共轭给电子）→C=O 碳电正性降低→反应性降低
  └─ 离去基：酰氯 Cl⁻ 好离去基→加成-消除易；酰胺 NH₂⁻ 坏离去基→需酸催化
```

**醛 vs 酮 vs 酯**：
- 醛：没有位阻+没有给电子共轭 → 最快
- 酮：两个烷基给电子+位阻 → 比醛慢
- 酯：OR 共轭给电子（+M 效应显著）→ C=O 碳电正性进一步降低 → 更慢

## 四、共轭加成 · 1,2 vs 1,4

```
α,β-不饱和羰基 + 亲核试剂 → 竞争

条件判断表：
  │ 条件              │ 主路径  │ 原因                        │
  │:------------------|:--------|:----------------------------|
  │ 强亲核/低温        │ 1,2     │ 动力学控制→直接攻羰基更快    │
  │ 弱亲核/高温        │ 1,4     │ 热力学控制→共轭产物更稳定    │
  │ RLi / RMgX        │ 1,2     │ 硬亲核→羰基碳               │
  │ R₂CuLi (Gilman)   │ 1,4     │ 软亲核→β 碳                 │
  │ NaBH₄             │ 1,2     │ 不可逆氢负离子转移            │
  │ CN⁻/RS⁻           │ 1,4     │ 软亲核+可逆                  │
```

**硬软亲核试剂（HSAB）判断**：硬亲核→1,2；软亲核→1,4。这是因为羰基碳是硬亲电中心，β 碳是软亲电中心。

## 五、炔烃的特殊加成

```
炔烃加成 = 烯烃加成的"两倍"——加两次

H-C≡C-H + HX（过量）→ 二卤代物（两次 Markovnikov）
H-C≡C-H + H₂O/Hg²⁺ → 烯醇→酮（Markovnikov，酮式更稳定）

端炔的特殊性：
  ≡C–H 有酸性（pKa~25）→ 可被 NaNH₂ 去质子→炔负离子
  炔负离子 = 碳负离子 → 亲核试剂 → 可做 SN2 烷基化→增长碳链
  
  → 端炔是"亲核的炔"——基础班炔烃只做亲电加成受体，提高班要会把它当亲核试剂用
```

## 六、硼氢化反应精要

```
BH₃ + 烯烃 → 烷基硼烷 → H₂O₂/OH⁻ → 醇

三步理解：
  1. BH₃ 是亲电试剂（B 缺电子）→B 加在端碳（位阻小）
  2. 四中心过渡态→B 和 H 从同一侧加→syn 加成
  3. H₂O₂/OH⁻ 氧化：C–B → C–OH（构型保留，B 变 OH 不翻转）
  
总结果：反 Markovnikov + syn 加成 → 相当于"反马氏水合"
```

## 七、常见易错点

| 错误 | 纠正 |
|:---|:---|
| 所有 HX+过氧化物都反马氏 | **只有 HBr**——HCl 链增长吸热（BDE太大），HI 链增长虽可行但 I• 不稳定 |
| 炔烃水合=烯烃水合 | 炔烃水合经由烯醇中间体→互变异构→羰基化合物（酮为主） |
| 1,2 和 1,4 只取决于温度 | 还取决于亲核试剂的硬软——RLi（硬）→1,2；R₂CuLi（软）→1,4 |
| 硼氢化后用酸水解得醇 | 不对——BH₃ 水解得烷烃（B–C→C–H），需要 H₂O₂/OH⁻ 氧化才能得到醇 |
| 羰基反应活性只看位阻 | 电子效应（共轭给电子）也关键——酰胺的 NH₂ 给电子→C=O 活性大幅下降 |

## 八、例题速练

### 例 1：亲电加成与重排

**题**：CH₂=CHCH(CH₃)₂ + HCl → 主产物？

**速解**：
1. Markovnikov 加成→H 加在端碳→C⁺ 在 C2（与两个烷基相连→较稳）
2. C2 的 C⁺ 可能重排——检查：邻位 C3 有 Me 可从 2°→3° C⁺ → 重排！→C⁺ 迁移到 C3（3° C⁺）
3. Cl⁻ 加到 C3→主产物是重排产物：CH₃CHClC(CH₃)₂（经过甲基迁移）

⚠️ 不检查重排是亲电加成最常见的失分原因！

### 例 2：共轭加成判断

**题**：CH₂=CHCOCH₃ + CH₃Li（1当量）→ 主产物？

**速解**：CH₃Li = 硬亲核→倾向 1,2 加成→产物 CH₂=CHC(OH)(CH₃)₂。如果用 (CH₃)₂CuLi（Gilman 试剂）→软亲核→1,4 加成→CH₃CH=CHOCH₃→互变→CH₃CH₂COCH₃。

## 九、自查清单

- [ ] 亲电加成/亲核加成/协同加成三大类的中间体/过渡态区分
- [ ] Markovnikov 规则的机理解释（C⁺稳定性）——不是背规则
- [ ] 只有 HBr+过氧化物→反马氏（自由基链），HCl/HI 不行
- [ ] 卤素加成=anti（卤鎓离子），硼氢化=syn（四中心 TS）
- [ ] 羰基活性顺序（位阻+电子双因素）能解释
- [ ] 1,2 vs 1,4 的三种控制因素：温度、硬软亲核、可逆性
- [ ] 炔烃端炔的酸性→炔负离子→亲核试剂→增长碳链
- [ ] 硼氢化-氧化=反马氏醇（不是酸水解！）
- [ ] 亲电加成要检查重排——C⁺ 可能迁移
- [ ] 烯丙基/苄基底物→SN2'（烯丙基型取代）可能发生
