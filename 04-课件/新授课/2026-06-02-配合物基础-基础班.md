## 一、课堂引入与本节问题

### 1.1 课堂切入

- **现象/问题/情境**：高中化学中你已经接触过银氨溶液 $\mathrm{[Ag(NH_3)_2]^+}$、铜氨溶液 $\mathrm{[Cu(NH_3)_4]^{2+}}$，也知道它们叫"络离子"。但如果问你：
  - $\mathrm{[Co(NH_3)_5Cl]Cl_2}$ 为什么能电离出 2 个 $\mathrm{Cl^-}$ 而不是 3 个？
  - 为什么 $\mathrm{[PtCl_2(NH_3)_2]}$ 的两种异构体，一种能抗癌，另一种不能？
  - 为什么 $\mathrm{Fe^{2+}}$ 与不同配体结合时，有的有磁性、有的没有？

- **本节要解决的问题**：
  1. 如何系统命名配合物？命名规则背后的逻辑是什么？
  2. 配位数与几何构型如何对应？为什么 CN=4 时既有四面体也有平面四方形？
  3. 配合物有哪些异构现象？如何判断和计数？
  4. 晶体场理论如何解释配合物的颜色、磁性、高低自旋？

### 1.2 本节深度边界

### 2.1 配合物的基本概念与命名规则

> 基础必讲层。从学生熟悉的 $\mathrm{Ag(NH_3)_2^+}$ 出发，拆解"中心原子 + 配体 + 配位数 + 配位原子 + 外界"五要素。

**从问题出发**：为什么 $\mathrm{[Co(NH_3)_5Cl]Cl_2}$ 溶液中加 $\mathrm{AgNO_3}$ 只产生 2 份 $\mathrm{AgCl}$ 沉淀？

**关键推理**：
```
[Co(NH3)5Cl]Cl2 → [Co(NH3)5Cl]2+ + 2Cl-
     ↑内界↑        ↑外界↑
内界 = 中心离子 Co3+ + 配体(5NH3 + 1Cl-) → 配离子
外界 = 2Cl-，自由存在，遇 Ag+ 立即沉淀
```

**五要素速记**：

| 要素 | 定义 | 实例（以 [Co(NH₃)₅Cl]Cl₂ 为例）|
|:---|:---|:---|
| 中心原子/离子 | 被配体围绕的原子/离子 | Co³⁺ |
| 配体 | 提供孤电子对的分子或离子 | NH₃（中性）、Cl⁻（阴离子）|
| 配位原子 | 配体中直接与中心原子成键的原子 | N（来自 NH₃）、Cl（来自 Cl⁻）|
| 配位数 | 直接与中心原子相连的配位原子总数 | 6（5 个 N + 1 个 Cl）|
| 外界 | 方括号外的离子 | 2Cl⁻ |

> ⚠️ **高频陷阱**：配位数 ≠ 配体数！多齿配体（如 en 乙二胺，双齿）时，配位数 = 齿数 × 配体数。

**命名规则**（课堂精简版）：

| 步骤 | 规则 | 检查清单 |
|:---|:---|:---|
| 1 | **配体排序**：先无机后有机；同类型先阴离子后中性分子；同类按配位原子字母序 | ☐ 顺序无误 |
| 2 | **配体数目**：二、三、四...（复杂配体用 bis, tris）| ☐ 词头正确 |
| 3 | **中心原子**：名称 + 氧化态（罗马数字）| ☐ 氧化态计算正确 |
| 4 | **内外界**：阴离子名称在前、阳离子名称在后 | ☐ 词尾"化"/"酸"正确 |
| 5 | **特殊符号**：μ = 桥联，ηⁿ = n 齿配位 | ☐ 希腊字母标注正确 |

**常见配体命名速查**：

| 配体 | 命名 | 配体 | 命名 |
|:---|:---|:---|:---|
| H₂O | 水 | NH₃ | 氨（ammine）|
| CO | 羰基 | Cl⁻ | 氯（chloro）|
| CN⁻ | 氰（cyano）| OH⁻ | 羟（hydroxo）|
| en (乙二胺) | 乙二胺 | NO₂⁻（N 配位）| 硝基 |
| ONO⁻（O 配位）| 亚硝酸根 | SCN⁻（S 配位）| 硫氰酸根 |

**命名实例**：

| 化学式 | 系统命名 |
|:---|:---|
| [Co(NH₃)₆]Cl₃ | 三氯化六氨合钴(III) |
| K₃[Fe(CN)₆] | 六氰合铁(III)酸钾 |
| [PtCl₂(NH₃)₂] | 二氯·二氨合铂(II) |
| [CrCl₂(H₂O)₄]Cl | 氯化二氯·四水合铬(III) |
| [Fe(CO)₅] | 五羰基合铁(0) |

> [!info]- 完整命名规则与教材对照（课后复习用）
> 
> **教材来源**：[[配合物#命名规则]]、[[教学逻辑提炼-周坤无机新课-晶体配合物与气体-第一轮]]
> 
> **完整规则补充**：
> 1. 配体为中性分子时，名称不变（NH₃ → 氨，H₂O → 水）
> 2. 配体为阴离子时，词尾变为 -o（Cl⁻ → chloro，OH⁻ → hydroxo）
> 3. 两可配体需标注配位原子：-N 表示 N 配位，-O 表示 O 配位
> 4. 桥联配体用 μ 表示，如 μ-OH 表示羟桥
> 5. 氧化态计算：配离子总电荷 = 中心离子电荷 + 配体电荷代数和
> 
> **与教材的对照说明**：
> - 教材中命名规则分散在§14.1和§14.2，本讲义整合为五步流程
> - 已提炼资料中的命名流程图（source_extract B4-8）可直接用于板书

**阶段性小结**：
> 命名核心逻辑 = **先阴后阳、配体排序、词尾变化**。记住：配体顺序错了，整个命名就错了。

### 2.2 配位数与几何构型

> 用 VSEPR 类比迁移到配合物空间构型，但强调配合物构型还受配体场和 d 电子数影响。

**从问题出发**：为什么 $\mathrm{[Ni(CN)_4]^{2-}}$ 是平面四方形，而 $\mathrm{[NiCl_4]^{2-}}$ 是四面体？

**关键推理**：配位数决定"大致框架"，但 d 电子数和配体场强决定"精确构型"。

| 配位数 | 常见构型 | 杂化类型 | 典型实例 | 备注 |
|:---:|:---|:---|:---|:---|
| 2 | 直线形 | sp | [Ag(NH₃)₂]⁺ | d¹⁰ 金属常见 |
| 3 | 平面三角形 | sp² | [HgI₃]⁻ | 较少见 |
| 4 | 正四面体 | sp³ | [NiCl₄]²⁻、ZnCl₄²⁻ | 弱场/d¹⁰ 倾向 |
| 4 | 平面四方形 | dsp² | [Ni(CN)₄]²⁻、PtCl₄²⁻ | d⁸ 强场倾向 |
| 5 | 三角双锥 | dsp³ / sp³d | Fe(CO)₅ | — |
| 5 | 四方锥 | — | [Ni(CN)₅]³⁻ | 较少见 |
| 6 | 八面体 | d²sp³ / sp³d² | [Co(NH₃)₆]³⁺、[FeF₆]³⁻ | 最常见 |

> ⚠️ **常见陷阱**：d⁹ 的平面四方形**不是 dsp²**，而是拉长八面体的极限。看到 Cu²⁺ + 平面 → 先想姜泰勒效应（后续提及）。

**判断口诀**：
- CN=2 → 直线（d¹⁰ 如 Ag⁺、Au⁺）
- CN=4 + d⁸ + 强场 → 平面四方形（如 [Ni(CN)₄]²⁻）
- CN=4 + 弱场/d¹⁰ → 四面体（如 [NiCl₄]²⁻、ZnCl₄²⁻）
- CN=6 → 八面体（绝大多数情况）

> [!info]- 构型判断的完整逻辑（课后复习用）
> 
> **教材来源**：[[配合物#几何构型]]、[[专题-配位化学#comparison-table]]
> 
> **补充说明**：
> - 配位数 5 的构型在竞赛中较少直接考查，了解即可
> - 配位数 6 的八面体是异构体最丰富的构型，也是晶体场理论的主要研究对象
> - 4d/5d 金属由于 Δ 更大，更容易形成低自旋和平面四方形构型

### 2.3 配合物的异构现象

> 将有机化学中的顺反概念迁移到配合物，建立空间判断能力。

**从问题出发**：为什么 $\mathrm{[PtCl_2(NH_3)_2]}$ 有两种异构体，一种抗癌，一种不抗癌？

**异构体分类总览**：

```
配合物异构
├── 结构异构（连接方式不同）
│   ├── 电离异构：内外界离子交换
│   ├── 水合异构：H₂O 在内界 vs 外界
│   ├── 配位异构：配体在阴阳离子间分配不同
│   └── 键合异构：两可配体用不同原子配位
└── 立体异构（空间排列不同）
    ├── 几何异构（顺/反、面/经）
    └── 旋光异构（镜像不可重叠）
```

**结构异构速查**：

| 类型 | 特征 | 典型实例 |
|:---|:---|:---|
| 电离异构 | 内外界离子可交换 | [Co(NH₃)₅Br]SO₄（紫红）vs [Co(NH₃)₅SO₄]Br（红）|
| 水合异构 | H₂O 在内界 vs 外界 | [Cr(H₂O)₆]Cl₃（紫）vs [CrCl(H₂O)₅]Cl₂·H₂O（亮绿）|
| 键合异构 | 两可配体不同原子配位 | [Co(NO₂)(NH₃)₅]²⁺（黄褐，N 配位）vs [Co(ONO)(NH₃)₅]²⁺（红褐，O 配位）|

**立体异构——几何异构**：

| 类型 | 适用配合物 | 判断方法 |
|:---|:---|:---|
| 顺/反 (cis/trans) | 平面四方 MA₂B₂、八面体 MA₄B₂ | 相同配体相邻 vs 相对 |
| 面/经 (fac/mer) | 八面体 MA₃B₃ | 三同配体共面 vs 共经线 |

**顺铂与反铂（重要实例）**：
- **顺式(cis-)**：棕黄色，偶极矩 μ ≠ 0，易溶于水（0.2577 g/100g H₂O），是**抗癌药物（顺铂）**
- **反式(trans-)**：淡黄色，偶极矩 μ = 0，难溶于水（0.0366 g/100g H₂O），无抗癌功能

**旋光异构判断口诀**：
> **"无 σ 无 i 无 S₄ 则 Chiral"**
> - σ = 对称面（mirror plane）
> - i = 对称中心（inversion center）
> - S₄ = 四重反轴（improper rotation）
> - 若三种对称元素都没有 → 该配合物有手性，存在旋光异构体

**八面体配合物立体异构体速查**：

| 类型 | 立体异构体数 | 对映体对数 | 记忆提示 |
|:---|:---:|:---:|:---|
| Ma₆ / Ma₅b | 1 | 0 | 全同或单取代 |
| Ma₄b₂ | 2 | 0 | 顺/反 |
| Ma₃b₃ | 2 | 0 | 面式(fac)/经式(mer) |
| Ma₄bc | 2 | 0 | — |
| Ma₃b₂c | 3 | 0 | — |
| **Ma₂b₂c₂** | **5** | **1对** | **核心考点** |
| Ma₂b₂cd | 6 | 1对 | — |

> [!info]- 异构体枚举方法：固定-变换法（课后复习用）
> 
> **教材来源**：[[配合物异构]]、[[教学逻辑提炼-周坤无机新课-晶体配合物与气体-第一轮]]
> 
> **固定-变换法步骤**：
> 1. 确定配位数和构型（通常是八面体）
> 2. 固定一个配体在顶点，变换其对位（trans）配体
> 3. 对每种构型检查 σ/i/S₄ → 判断手性
> 4. 对照速查表验证总数
> 
> **示例**：Ma₂b₂c₂ 型八面体的 5 种异构体
> - 先固定 a 在顶点，另一个 a 可以在对位或邻位
> - 两个 a 在对位时，b 和 c 的排列有 2 种
> - 两个 a 在邻位时，b 和 c 的排列有 3 种，其中 1 种有对映体
> - 总计：5 种（含 1 对对映体）

### 2.4 晶体场理论初步

> 从静电模型出发，建立 d 轨道分裂的物理图像。这是本节课的理论核心。

**从问题出发**：为什么 $\mathrm{[Fe(H_2O)_6]^{2+}}$ 有顺磁性而 $\mathrm{[Fe(CN)_6]^{4-}}$ 是反磁性的？两者都是 Fe²⁺ (d⁶) 啊！

**晶体场理论的基本假设**：
1. 配体视为**点电荷/点偶极**（纯静电模型）
2. 中心金属的 **d 轨道在配体静电场作用下发生能级分裂**
3. d 电子在分裂后的轨道上重新排布

**关键推理：八面体场的 d 轨道分裂**

```
能量 ↑
      ____ ____  eg  (+0.6Δo)  ← dx²-y², dz² 指向配体 → 能量升高
      |  | |  |
      
      Δo  ← 分裂能
      
      ____ ____ ____  t2g  (-0.4Δo) ← dxy, dxz, dyz 避开配体 → 能量降低
      |  | |  | |  |
```

**推导逻辑（课堂板书版）**：
1. 6 个配体沿 ±x、±y、±z 轴接近中心离子
2. **dₓ²₋ᵧ²** 和 **dᵤ²** 的电子云极大值正好指向配体 → 受到最大静电排斥 → 能量升高 **+3/5 Δo**
3. **dₓᵤ**、**dₓᵤ**、**dᵤᵤ** 的电子云极大值指向配体之间 → 避开配体 → 能量降低 **-2/5 Δo**
4. **重心不变**：2×(+3/5) + 3×(-2/5) = 0

> 💡 **记忆口诀**：八面体 eg 高（像"鹅哥"飞得高），t2g 低（像"踢二哥"蹲得低）。

**四面体场的 d 轨道分裂**：
- 分裂方向与八面体**相反**：t₂ ↑，e ↓
- 分裂能 $\Delta_t \approx \frac{4}{9}\Delta_o$（小得多）
- **四面体总是高自旋**（Δt 太小，无法克服配对能）

**光谱化学序列（必背）**：
$$\mathrm{I^- < Br^- < Cl^- < F^- < OH^- < C_2O_4^{2-} < H_2O < SCN^- < NH_3 < en < NO_2^- < CN^- < CO}$$

| 场强分类 | 配体 | 电子效应 | 典型配合物 | 自旋倾向 |
|:---|:---|:---|:---|:---|
| 弱场（Δ小）| I⁻、Br⁻、Cl⁻、F⁻ | σ给电子 + π给电子（Δ↓）| [FeF₆]⁴⁻ | 高自旋 |
| 中强场 | H₂O | σ给电子 | [Fe(H₂O)₆]²⁺ | 高自旋（d⁶时）|
| 强场 | NH₃、en | σ给电子 | [Co(NH₃)₆]³⁺ | 低自旋 |
| 强场 | CN⁻、CO | σ给电子 + π受电子（Δ↑）| [Fe(CN)₆]⁴⁻ | 低自旋 |

**高自旋 vs 低自旋：Δ 与 P 的竞争**

```
确定中心离子 dⁿ 组态
        ↓
确定配体在光谱化学序列中的位置
        ↓
估算 Δo（八面体场分裂能）
        ↓
比较 Δo 与 P（成对能）
        ↓
    Δo > P → 低自旋 (LS) → 电子优先成对填入 t2g
    Δo < P → 高自旋 (HS) → 电子优先分占各轨道（Hund 规则）
```

> 📌 **竞赛高频**：d⁴-d⁷ 是高低自旋的"决策区间"，必考。

**dⁿ 电子排布速查（八面体场）**：

| dⁿ | 弱场(HS)电子排布 | CFSE(HS) | 强场(LS)电子排布 | CFSE(LS) | 自旋状态 |
|:---:|:---|:---:|:---|:---:|:---:|
| d¹ | t2g¹ | -0.4Δo | — | — | — |
| d² | t2g² | -0.8Δo | — | — | — |
| d³ | t2g³ | -1.2Δo | — | — | — |
| d⁴ | t2g³eg¹ | -0.6Δo | t2g⁴ | -1.6Δo | Δ<P HS, Δ>P LS |
| d⁵ | t2g³eg² | 0 | t2g⁵ | -2.0Δo | Δ<P HS, Δ>P LS |
| d⁶ | t2g⁴eg² | -0.4Δo | t2g⁶ | -2.4Δo | Δ<P HS, Δ>P LS |
| d⁷ | t2g⁵eg² | -0.8Δo | t2g⁶eg¹ | -1.8Δo | Δ<P HS, Δ>P LS |
| d⁸ | t2g⁶eg² | -1.2Δo | — | — | — |
| d⁹ | t2g⁶eg³ | -0.6Δo | — | — | — |
| d¹⁰ | t2g⁶eg⁴ | 0 | — | — | — |

> [!info]- 晶体场理论的完整推导与教材对照（课后复习用）
>

> 
> **具体数值实例**：
> - $\mathrm{FeF_6^{3-}}$（弱场）：$\Delta_o = 13700\,\mathrm{cm^{-1}}$，$P = 30000\,\mathrm{cm^{-1}}$，$\Delta_o < P$ → 高自旋
> - $\mathrm{Fe(CN)_6^{3-}}$（强场）：$\Delta_o = 34250\,\mathrm{cm^{-1}}$，$\Delta_o > P$ → 低自旋
> - $\mathrm{Ti(H_2O)_6^{3+}}$：$\Delta_o = 20400\,\mathrm{cm^{-1}}$（对应吸收波长约 500 nm，呈紫色）
> 
> **与教材的对照说明**：
> - 教材中分裂能推导从球形场出发，本讲义简化为"指向配体 vs 避开配体"的空间分析
> - CFSE 定量计算（含成对能 P）放到第二轮，本轮只要求定性理解

> 🔷 **拓展**（选讲，教师根据学生接受度取舍）：
> **Jahn-Teller 效应（现象级提及）**：
> - 当 eg 轨道电子不对称占据时（如 d⁹ 的 eg³），配合物会发生几何畸变以消除简并
> - 典型例子：Cu²⁺ (d⁹) 的配合物通常是**拉长的八面体**（4 短键 + 2 长键）
> - **关键认知**：d⁹ 的平面四方形不是 dsp² 杂化，而是拉长八面体的极限

## 三、课堂例题

### 例题 1（⭐⭐ 基础）：配合物命名

**题目**：写出下列配合物的系统命名：
(1) $\mathrm{[Co(NH_3)_5Cl]Cl_2}$
(2) $\mathrm{K_3[Fe(CN)_6]}$
(3) $\mathrm{[PtCl_2(NH_3)_2]}$

**分析提示**：
- 第一步：拆内外界，确定配离子电荷
- 第二步：配体排序——先阴离子(Cl⁻)后中性(NH₃)，同类按字母序
- 第三步：中心原子氧化态用罗马数字
- 注意陷阱：配体数词头用"二""六"，配体间用"·"隔开

**完整解答路径**：
```
(1) [Co(NH3)5Cl]Cl2
    - 内界：[Co(NH3)5Cl]2+，外界：2Cl-
    - 配体：Cl⁻（氯）+ 5NH₃（氨）
    - 排序：先阴离子后中性 → 氯·五氨
    - 中心：Co 氧化态 = +3（因为 5×0 + (-1) + x = +2 → x = +3）
    - 命名：氯化一氯·五氨合钴(III)
    
(2) K3[Fe(CN)6]
    - 内界：[Fe(CN)6]3-，外界：3K+
    - 配体：6CN⁻（氰）
    - 中心：Fe 氧化态 = +3（6×(-1) + x = -3 → x = +3）
    - 命名：六氰合铁(III)酸钾
    
(3) [PtCl2(NH3)2]
    - 无外界
    - 配体：2Cl⁻（氯）+ 2NH₃（氨）
    - 排序：先阴离子后中性 → 二氯·二氨
    - 中心：Pt 氧化态 = +2（2×(-1) + 2×0 + x = 0 → x = +2）
    - 命名：二氯·二氨合铂(II)
```

> 💡 **课堂任务**：学生动笔 3 min，教师巡视，点名 2 人板演，集体纠错。

### 例题 2（⭐⭐ 基础）：配位数与构型判断

**题目**：判断下列配合物中中心离子的配位数和几何构型：
(1) $\mathrm{[Fe(CN)_6]^{4-}}$
(2) $\mathrm{[Ni(CO)_4]}$
(3) $\mathrm{[Pt(en)_2]^{2+}}$

**分析提示**：
- 配位数 = 直接与中心离子相连的配位原子总数（多齿配体要乘齿数）
- CN=6 → 八面体；CN=4 + d⁸ + 强场 → 平面四方形；CN=4 + 弱场/d¹⁰ → 四面体

**完整解答路径**：
```
(1) [Fe(CN)6]4-
    - CN = 6（6 个 CN⁻，每个单齿）
    - 构型：八面体
    
(2) [Ni(CO)4]
    - CN = 4（4 个 CO，每个单齿）
    - Ni 为 d¹⁰，CO 虽为强场但 d¹⁰ 无 CFSE 偏好
    - 构型：四面体
    
(3) [Pt(en)2]2+
    - en = 乙二胺，双齿配体
    - CN = 2 × 2 = 4（2 个 en × 2 个配位原子）
    - Pt²⁺ 为 d⁸，en 为强场配体
    - 构型：平面四方形
```

> 💡 **课堂任务**：学生口答，教师追问"为什么 [Pt(en)₂]²⁺ 不是四面体？"引出 d⁸ + 强场 → 平面四方。

### 例题 3（⭐⭐ 基础）：异构体判断

**题目**：
(1) 平面四方形 $\mathrm{[PtCl_2(NH_3)_2]}$ 有几种几何异构体？画出顺式和反式。
(2) 八面体 $\mathrm{[Co(NH_3)_4Cl_2]^+}$ 有几种几何异构体？
(3) 八面体 $\mathrm{[Co(en)_3]^{3+}}$ 是否有旋光异构体？为什么？

**分析提示**：
- 几何异构：相同配体相邻(cis) vs 相对(trans)
- 旋光异构：检查是否有 σ/i/S₄；无则 Chiral
- en 是双齿配体，[M(AA)₃] 型通常有旋光异构

**完整解答路径**：
```
(1) [PtCl2(NH3)2]（平面四方形 MA2B2）
    - 2 种几何异构体：
      cis-：两个 Cl 相邻（夹角 90°），μ ≠ 0，棕黄色，抗癌活性
      trans-：两个 Cl 相对（夹角 180°），μ = 0，淡黄色，无抗癌活性
    
(2) [Co(NH3)4Cl2]+（八面体 MA4B2）
    - 2 种几何异构体：
      cis-：两个 Cl 相邻（夹角 90°），紫色
      trans-：两个 Cl 相对（夹角 180°），绿色
    
(3) [Co(en)3]3+（八面体 M(AA)3）
    - 有旋光异构体！
    - 理由：[Co(en)3]3+ 无对称面、无对称中心、无 S4 轴
    - 存在 Δ（右旋）和 Λ（左旋）两种对映体
```

> 💡 **课堂任务**：学生画图 4 min，教师用 PPT/3D 模型展示标准构型，强调 cis/trans 的判断方法。

### 例题 4（⭐⭐⭐ 拓展）：命名 + 构型 + 异构 + 晶体场综合题

**题目**：配合物 $\mathrm{[Co(en)_2Cl_2]Cl}$ 的系统命名是什么？它有多少种立体异构体？Co³⁺ 为 d⁶，判断其自旋状态和 CFSE（以 Δo 为单位）。

**分析提示**：
- 命名：拆内外界 → 配体排序 → 中心原子氧化态
- 异构：八面体 M(AA)₂b₂ 型 → 先判几何异构 → 再判旋光异构
- 晶体场：d⁶ + en（强场）→ 低自旋 → t2g⁶ → 计算 CFSE

**完整解答路径**：
```
Step 1：命名
- 内界：[Co(en)2Cl2]+，外界：Cl-
- 配体排序：Cl⁻（阴离子）先于 en（中性）
- 配体名称：二氯·二(乙二胺)
- Co 氧化态：2×0 + 2×(-1) + x = +1 → x = +3
- 命名：氯化二氯·二(乙二胺)合钴(III)

Step 2：立体异构体
- 八面体 M(AA)2b2 型
- 几何异构：
  * cis-：两个 Cl 相邻 → 有旋光性（无 σ/i/S4）→ 1 对对映体
  * trans-：两个 Cl 相对 → 有对称中心 → 无旋光性 → 1 种
- 总计：3 种立体异构体（cis-的 1 对对映体 + trans-的 1 种）

Step 3：自旋状态
- Co3+ 为 d6
- en 是强场配体（光谱化学序列右侧）
- Δo > P → 低自旋
- 电子排布：t2g^6 eg^0

Step 4：CFSE
- CFSE = 6 × (-0.4Δo) + 0 × (+0.6Δo) = -2.4Δo
- （本轮仅要求以 Δo 为单位的 CFSE，成对能 P 的修正放到第二轮）
```

> 💡 **课堂任务**：学生独立完成 5 min，教师投影完整解答，强调"命名→构型→异构→电子结构"的五步法。

## 五、本节形成的核心结论

1. **结论一**：配合物命名遵循"先阴后阳、配体排序、词尾变化"三原则——配体顺序错了，整个命名就错了。
2. **结论二**：配位数决定大致框架，d 电子数和配体场强决定精确构型——CN=4 + d⁸ + 强场 → 平面四方形；CN=4 + 弱场/d¹⁰ → 四面体。
3. **结论三**：八面体是异构体最丰富的构型——Ma₂b₂c₂ 有 **5 种立体异构体**（含 1 对对映体）；手性判断口诀是 **"无 σ 无 i 无 S₄ 则 Chiral"**。
4. **结论四**：晶体场理论的核心是 **d 轨道分裂** 与 **CFSE 稳定化能**——分裂能 Δ 与配对能 P 的竞争决定高/低自旋；光谱化学序列是竞赛必背的场强排序工具。

## 六、⚠️ 易错点预警

| 错误想法 | 为什么错 | 正确理解 |
|:---|:---|:---|
| 配位数 = 配体数 | 忽略了多齿配体 | 配位数 = 齿数 × 配体数；en（双齿）× 2 = 4 |
| 命名时配体顺序混乱 | 没有掌握"先阴后阳、字母序"规则 | 先无机后有机，先阴离子后中性，同类按配位原子字母序 |
| d⁹ 平面四方形 = dsp² 杂化 | 混淆了价键理论和晶体场理论 | d⁹ 平面四方形是**拉长八面体的极限**，先想姜泰勒效应 |
| 四面体也有几何异构 | 四面体的四个顶点等价 | 四面体 MA₂B₂ 只有一种构型，所有顶点通过对称操作等价 |
| 所有 d⁴-d⁷ 都有 HS/LS 之分 | 忽略了 4d/5d 金属 | 4d/5d 金属 Δo 很大，几乎总是低自旋 |
| 高自旋 = 顺磁，低自旋 = 反磁 | 忽略了 dⁿ 的多样性 | d⁵ LS（t2g⁵）仍有 1 个未成对电子，是顺磁的 |
| 四面体场也用 eg/t2g 符号 | 四面体无对称中心 | 四面体场记作 e 和 t₂，不加 g |

## 附录：板书速查（学生课后整理用）

```
【配合物五要素】
中心原子 + 配体 + 配位原子 + 配位数 + 外界

【命名流程】
拆内外界 → 配体排序（先阴后中，字母序）→ 数目标注 → 中心原子(罗马数字) → 词尾

【构型速查】
CN=2: 直线(sp)        CN=4+d⁸+强场: 平面四方(dsp²)
CN=4+弱场/d¹⁰: 四面体(sp³)   CN=6: 八面体(d²sp³/sp³d²)

【异构判断口诀】
无σ 无i 无S₄ → Chiral

【d轨道分裂图】
        eg↑  (+0.6Δo)     [dx²-y², dz²]
        Δo
        t2g↓ (-0.4Δo)     [dxy, dxz, dyz]

【高/低自旋判断】
d⁴-d⁷: 查光谱化学序列 → 估Δ → 比较Δ vs P → Δ>P: LS, Δ<P: HS

【光谱化学序列（必背）】
I⁻ < Br⁻ < Cl⁻ < F⁻ < OH⁻ < H₂O < NH₃ < en < CN⁻ < CO
```
