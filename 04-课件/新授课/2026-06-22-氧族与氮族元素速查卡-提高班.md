# 氧族与氮族元素 · 速查卡

> 基础班你背过"浓 HNO₃→NO₂、稀 HNO₃→NO"——本卡把氧族/氮族从**事实清单**升级到**结构-性质-氧化态梯度**的关联推理。

---

## 一、硫化合物 · 结构决定性质

```
硫的氧化态从 -2 到 +6 → 性质递变

  H₂S (−2) ──→ S (0) ──→ SO₂ (+4) ──→ SO₃/H₂SO₄ (+6)
  还原性极强        中间态         还原性           氧化性
```

| 化合物 | 结构 | 关键性质 | 一句话 |
|:---|:---|:---|:---|
| **SO₃²⁻** | 三角锥（有一对孤对） | **还原性**——易被氧化成 SO₄²⁻ | "孤对电子=还原性的来源" |
| **SO₄²⁻** | 正四面体（无孤对） | 稳定——难被还原 | "无孤对=稳定" |
| **S₂O₃²⁻** | SO₄²⁻ 中的一个 O 被 S 取代 | **歧化**：S₂O₃²⁻+H⁺→S↓+SO₂↑ | "酸性条件下必歧化" |
| **S₂O₈²⁻** | 两个 SO₄ 通过 –O–O– 桥连 | **极强氧化性**（O–O 键易断） | "过氧桥=氧化性发动机" |
| **H₂S** | 弯曲形 | 弱酸性 + 强还原性 + 沉淀重金属 | "臭鸡蛋+使 Pb(Ac)₂ 试纸变黑" |

**S₂O₃²⁻ 歧化**：在酸中产生 S 沉淀和 SO₂ 气体——这是推断题的经典信号（沉淀+气体同时产生）。

---

## 二、氮氧化物 · 氧化态梯度

```
N 的氧化态从 -3 到 +5：

  NH₃(−3) ← N₂H₄(−2) ← NH₂OH(−1) ← N₂(0) → N₂O(+1) → NO(+2) → N₂O₃(+3) → NO₂/N₂O₄(+4) → N₂O₅/HNO₃(+5)
  
  还原剂                          中性                    酸性条件下氧化剂增强 →
```

| 物种 | 关键性质 | 颜色与状态 |
|:---|:---|:---|
| **NO** | 自由基——易被 O₂ 氧化→NO₂ | 无色气体，遇空气立即变红棕色 |
| **NO₂** | 自身二聚：2NO₂ ⇌ N₂O₄ | 红棕色气体（⇌ 无色 N₂O₄） |
| **N₂O** | 惰性——不助燃、不与 O₂ 反应 | 无色，"笑气" |

---

## 三、硝酸还原产物 · 浓度与金属活性双重控制

```
HNO₃ + 金属 → 还原产物取决于两个因素：

  浓 HNO₃ → NO₂（+4）——还原程度浅（浓硝酸氧化性集中在自身）
  稀 HNO₃ → NO（+2）——还原程度深（H₂O 参与→可以更深还原）

  活泼金属（Zn, Mg）→ 可能更深还原至 N₂O(+1)/N₂(0)/NH₄⁺(−3)
  
  口诀：浓 HNO₃ 得 NO₂，稀 HNO₃ 得 NO，极稀 HNO₃+活泼金属得 NH₄⁺
```

---

## 四、磷的含氧酸缩合

```
正磷酸 H₃PO₄ → 焦磷酸 H₄P₂O₇ → 偏磷酸 (HPO₃)ₙ → 多聚磷酸

  缩合规律：正→焦→偏 = 脱水缩合（每步失去 H₂O）
  
  关键：缩合程度越高→P–O–P 键越多→酸性越强
        P–O–P 键在水中可水解→焦磷酸在水中缓慢回到正磷酸
```

**磷的三种同素异形体**：白磷 P₄（四面体，剧毒，在空气中自燃）→ 红磷（链状，无毒）→ 黑磷（层状，最稳定）

## 五、推断题信号速查

| 信号 | 指向 | 验证方法 |
|:---|:---|:---|
| 臭鸡蛋气味气体 | H₂S | 使 Pb(Ac)₂ 试纸变黑 |
| 刺激性气味+使品红褪色 | SO₂ | 褪色后加热→恢复（SO₂ 褪色可逆） |
| 红棕色气体 | NO₂/Br₂ | NO₂ 通入水中→无色溶液+HNO₃；Br₂ 水溶液呈橙黄色 |
| 酸中产生沉淀+气体 | S₂O₃²⁻ 歧化 | S↓（黄色）+ SO₂↑ |
| 棕色环实验 | NO₃⁻ | Fe²⁺+浓 H₂SO₄→[Fe(NO)]²⁺ 棕色 |
| 白烟（NH₄Cl） | NH₃ + HCl | NH₃ 使湿润红色石蕊试纸变蓝 |
| 钼蓝（磷钼酸铵还原） | PO₄³⁻ | 钼酸铵+HNO₃→黄色沉淀→还原剂→蓝色 |

## 六、常见易错点

| 错误 | 纠正 |
|:---|:---|
| "浓 HNO₃ 还原产物是 NO" | 浓 HNO₃→**NO₂**；稀 HNO₃→**NO**。浓硝酸还原程度浅！ |
| SO₂ 使品红褪色=Cl₂ 漂白 | SO₂ 褪色可逆（加热恢复），Cl₂ 的漂白不可逆（氧化破坏） |
| "N₂O₅ 是硝酸的酸酐所以稳定" | N₂O₅ 极不稳定——易分解为 NO₂+O₂ |
| 白磷和红磷只是颜色不同 | 结构完全不同：白磷 P₄（四面体），红磷链状聚合物 |
| S₂O₃²⁻ 在酸中稳定的 | **酸性下必歧化**：S₂O₃²⁻+2H⁺→S↓+SO₂↑+H₂O |

## 七、例题速练

### 例 1：HNO₃ 还原产物

**题**：Cu + 稀 HNO₃ → ？Cu + 浓 HNO₃ → ？

**速解**：
- 稀：3Cu + 8HNO₃ → 3Cu(NO₃)₂ + 2**NO**↑ + 4H₂O
- 浓：Cu + 4HNO₃ → Cu(NO₃)₂ + 2**NO₂**↑ + 2H₂O

### 例 2：硫代硫酸钠推断

**题**：Na₂S₂O₃ 溶液中加入稀 HCl → 产生黄色沉淀 + 刺激性气体。是什么？

**速解**：S₂O₃²⁻ 在酸中歧化 → S↓（黄色）+ SO₂↑（刺激性，使品红褪色）。这是鉴定 S₂O₃²⁻ 的经典反应。
