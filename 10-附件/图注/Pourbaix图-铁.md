# Pourbaix图——铁

![[mineru/03-教材书籍/无机化学第6版Weller/无机化学第6版Weller/无机化学第6版氧化还原反应180-212_images/3184c42e89576d427dd4c3f7db24097f37b59511836d3476c742ab5be29f211f.jpg]]

> **来源**：Weller《无机化学》第6版 §5.14，图5.11

---

## 图中关键信息

- **横轴**：pH（0–14）
- **纵轴**：$E$ / V（vs SHE）
- **稳定区**：
  - 低 pH + 低 $E$：Fe²⁺(aq)
  - 低 pH + 高 $E$：Fe³⁺(aq)
  - 高 pH + 低 $E$：Fe(OH)₂(s)
  - 高 pH + 高 $E$：Fe(OH)₃(s) → 实际为 Fe₂O₃·nH₂O（铁锈）
- **水稳定线**（虚线）：
  - 上虚线：O₂/H₂O（$E^\circ = +1.23\ \mathrm{V}$）
  - 下虚线：H₂O/H₂（$E^\circ = 0\ \mathrm{V}$）

## 教学要点

| 要点 | 说明 |
|:---|:---|
| 铁的腐蚀区 | 在水稳定线之间且 Fe²⁺/Fe³⁺ 稳定区，铁易被氧化 |
| 钝化区 | Fe(OH)₃ 覆盖表面形成保护膜，阻止进一步腐蚀 |
| 阴极保护 | 将铁电位降至水稳定线以下，使其处于热力学稳定区 |

- **决赛**：结合 Nernst 方程计算边界线的精确 $E$ 值
