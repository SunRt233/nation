## 四、📚 资料提炼进度跟踪

> 监控「计划提取题数」vs「实际提取题数」的进度。

```dataview
TABLE WITHOUT ID
  link(file.name, title) as 提炼页,
  source_type as 来源类型,
  year as 年份,
  review_status as 审核状态,
  problems_planned as 计划题数,
  problems_extracted as 已提取,
  round(problems_extracted / problems_planned * 100, 1) + "%" as 进度,
  join(knowledge_points, "、") as 涉及知识点
FROM "07-资料提炼"
SORT review_status ASC, file.mtime DESC
```

## 五、🕐 最近任务时间线

> 自动抽取 `00-首页/工作日志/` 下最近日志文件中的 H2 标题，形成时间线。

```dataviewjs
const tasks = [];
const pages = dv.pages('"00-首页/工作日志"')
  .where(p => p.file.name !== "工作日志")
  .sort(p => p.file.name, "desc")
  .array()
  .slice(0, 10);

for (const page of pages) {
  const content = await dv.io.load(page.file.path);
  const lines = content.split("\n");
  for (const line of lines) {
    const m = line.match(/^##\s*📅\s*(\d{4}-\d{2}-\d{2})\s+(.+?)\s*$/);
    if (m) {
      tasks.push({ date: m[1], title: m[2].trim(), source: page.file.link });
    }
  }
}

if (tasks.length === 0) {
  dv.paragraph("⚠️ 未在 `00-首页/工作日志/` 下找到符合 `## 📅 YYYY-MM-DD ...` 格式的标题。");
} else {
  dv.table(
    ["日期", "任务标题", "来源"],
    tasks.slice(0, 10).map(t => [t.date, t.title, t.source])
  );
}
```

## 六、⚠️ 踩坑速查（最近 3 条）

> 自动抽取 [[经验沉淀]] 中标了 🔥 的 H3 条目。

```dataviewjs
const exp = await dv.io.load("00-首页/经验沉淀.md");
const lines = exp.split("\n");

const tops = [];
let currentCategory = "";
for (const line of lines) {
  const cat = line.match(/^##\s+([一二三四五六七八九十]+)、(.+?)\s*$/);
  if (cat) {
    currentCategory = cat[2].trim();
    continue;
  }
  const m = line.match(/^###\s+(\S+)\s+(.+?)\s*🔥\s*$/);
  if (m) {
    tops.push({ code: m[1], title: m[2].trim(), category: currentCategory });
  }
}

if (tops.length === 0) {
  dv.paragraph("⚠️ 未在 [[经验沉淀]] 中找到 🔥 标记的条目。");
} else {
  dv.table(
    ["编号", "问题", "分类"],
    tops.slice(0, 3).map(t => [t.code, t.title, t.category])
  );
}
```
