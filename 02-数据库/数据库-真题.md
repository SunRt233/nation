# 数据库 · 真题

> 本文件是所有已入库真题的统一操作台。支持按届次、模块、题型、难度、知识点交叉筛选。
>
> 💡 **使用提示**：按届刷题请直接看「一、按届次汇总」；备课按模块选题请看「二、按模块统计」；找某知识点考了哪些题请看「五、按知识点交叉筛选」。

---

## 一、按届次汇总

```dataview
TABLE WITHOUT ID
  link(file.name, title) as 题目,
  year as 年份,
  module as 模块,
  question_type as 题型,
  difficulty as 难度,
  teaching_level as 层次,
  join(knowledge_points, "、") as 核心知识点
FROM "04-题库/真题"
SORT year DESC, file.name ASC
```

## 二、按模块统计

```dataview
TABLE WITHOUT ID
  count(file.name) as 题数,
  average(difficulty) as 平均难度,
  min(difficulty) as 最低难度,
  max(difficulty) as 最高难度,
  join(map(rows, (r) => link(r.file.name, r.title)), "、") as 题目列表
FROM "04-题库/真题"
GROUP BY module
SORT count(file.name) DESC
```

## 五、按知识点交叉筛选

> 点击下方任意知识点链接，即可跳转到该知识点页面查看其关联真题。

```dataview
TABLE WITHOUT ID
  knowledge_points as 知识点,
  count(file.name) as 出现次数,
  join(map(rows, (r) => link(r.file.name, r.title)), "、") as 关联题目
FROM "04-题库/真题"
FLATTEN knowledge_points
GROUP BY knowledge_points
SORT count(file.name) DESC
LIMIT 30
```

## 七、数据质量检查

### 7.1 缺少 knowledge_points 的题目（需补全）

```dataview
TABLE WITHOUT ID
  link(file.name, title) as 题目,
  year as 年份,
  module as 模块,
  question_type as 题型,
  difficulty as 难度
FROM "04-题库/真题"
WHERE !knowledge_points OR length(knowledge_points) = 0
SORT year DESC, file.name ASC
```
