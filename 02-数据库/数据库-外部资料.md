## 五、知识点入库追踪

> 检查提炼页中提到的知识点是否已在 `03-知识点/` 下创建对应文件。

```dataviewjs
// 获取所有提炼页中提到的 knowledge_points
const提炼Files = await app.vault.getMarkdownFiles()
  .filter(f => f.path.startsWith("07-资料提炼/"));

const提炼KP = new Set();
for (const f of 提炼Files) {
  const content = await app.vault.read(f);
  const fm = app.metadataCache.getFileCache(f)?.frontmatter;
  if (fm && fm.knowledge_points) {
    for (const kp of fm.knowledge_points) {
      提炼KP.add(kp);
    }
  }
}

// 获取所有已创建的知识点文件名
const kpFiles = await app.vault.getMarkdownFiles()
  .filter(f => f.path.startsWith("03-知识点/"));
const kpNames = new Set(kpFiles.map(f => f.basename));

// 找出尚未创建的知识点
const missing = [];
for (const kp of 提炼KP) {
  if (!kpNames.has(kp)) {
    missing.push(kp);
  }
}

if (missing.length === 0) {
  dv.paragraph("✅ 所有提炼页涉及的知识点均已创建文件。");
} else {
  dv.paragraph("⚠️ 以下知识点在提炼页中被提及，但尚未在 `03-知识点/` 下创建文件：");
  dv.list(missing.map(m => `[[${m}]]（待创建）`));
}
```

## 六、真题入库追踪

> 检查提炼页中计划的题目是否已拆分为 `04-题库/真题/` 下的独立文件。

```dataviewjs
const提炼Files = await app.vault.getMarkdownFiles()
  .filter(f => f.path.startsWith("07-资料提炼/"));

let totalPlanned = 0;
let totalExtracted = 0;

for (const f of 提炼Files) {
  const fm = app.metadataCache.getFileCache(f)?.frontmatter;
  if (fm) {
    totalPlanned += fm.problems_planned || 0;
    totalExtracted += fm.problems_extracted || 0;
  }
}

const pct = totalPlanned > 0 ? Math.round(totalExtracted / totalPlanned * 100) : 0;

dv.paragraph(`**真题提取总进度**：${totalExtracted} / ${totalPlanned} 题（${pct}%）`);
```
