## 一、库存总览（低频参考）

```dataview
TABLE WITHOUT ID
  subject as 模块,
  length(rows) as 总数,
  length(filter(rows, (r) => r.status = "已填充")) as 已填充,
  length(filter(rows, (r) => r.status = "初稿")) as 初稿,
  length(filter(rows, (r) => r.status = "骨架")) as 骨架,
  round(length(filter(rows, (r) => r.status = "已填充")) / length(rows) * 100, 1) + "%" as 完成率,
  round(average(map(filter(rows, (r) => r.importance >= 4), (r) => r.importance)), 1) as "高优重要度均分"
FROM "03-知识点"
WHERE type = "知识点"
GROUP BY subject
SORT 完成率 DESC
```

## 二、待填充高优清单（低频参考）

> 当前大面积待填充问题已不是主矛盾。
> 这一表更适合做监控或异常回查；若真要启动补缺，请同步核对 [[活跃任务]] 与 [[知识库待补充清单]]。

```dataview
TABLE WITHOUT ID
  link(file.name, title) as 知识点,
  subject as 模块,
  submodule as 子模块,
  status as 状态,
  importance as 重要度,
  difficulty as 难度
FROM "03-知识点"
WHERE type = "知识点"
  AND status != "已填充"
  AND importance >= 4
SORT importance DESC, difficulty DESC
LIMIT 30
```

## 三、二维矩阵：难度 × 重要度

> 右上象限（高难+高重要）是竞赛核心考点；左下象限（低难+低重要）可适度延后。

### 4.1 高难度 + 高重要度（核心竞赛考点）

```dataview
TABLE WITHOUT ID
  link(file.name, title) as 知识点,
  subject as 模块,
  status as 状态,
  length(filter(file.inlinks, (f) => contains(string(f.folder), "04-题库/真题"))) as 关联真题数
FROM "03-知识点"
WHERE type = "知识点"
  AND difficulty >= 4
  AND importance >= 4
SORT importance DESC, difficulty DESC
```

### 4.2 低难度 + 高重要度（基础必拿分）

```dataview
TABLE WITHOUT ID
  link(file.name, title) as 知识点,
  subject as 模块,
  status as 状态,
  mastery as 掌握度
FROM "03-知识点"
WHERE type = "知识点"
  AND difficulty <= 2
  AND importance >= 4
SORT importance DESC
```

## 五、按题型覆盖查缺

> 以下列出有 `problem_types` 定义但关联真题数为 0 的知识点，可能是「有题型设计但无真题链接」。

```dataview
TABLE WITHOUT ID
  link(file.name, title) as 知识点,
  subject as 模块,
  problem_types as 设计题型,
  length(filter(file.inlinks, (f) => contains(string(f.folder), "04-题库/真题"))) as 关联真题数
FROM "03-知识点"
WHERE type = "知识点"
  AND length(problem_types) > 0
  AND length(filter(file.inlinks, (f) => contains(string(f.folder), "04-题库/真题"))) = 0
SORT importance DESC
LIMIT 15
```

## 六、最近更新

```dataview
TABLE WITHOUT ID
  link(file.name, title) as 知识点,
  subject as 模块,
  status as 状态,
  mastery as 掌握度,
  updated as 更新日期
FROM "03-知识点"
WHERE type = "知识点"
  AND updated >= date(today) - dur(7 days)
SORT file.mtime DESC
LIMIT 10
```
