# 专题：取代与消除竞争

> 本专题对应考纲条目：[[36-取代反应]]、[[38-消除反应]]、[[44-脂肪族亲核取代反应]]
> 核心知识点：[[SN1反应]]、[[SN2反应]]、[[E1反应]]、[[E2反应]]

---

## 一、四种反应路径总览

| 反应 | 类型 | 底物偏好 | 试剂要求 | 产物特征 |
|------|------|---------|----------|---------|
| SN2 | 双分子取代 | 1°, 2° | 强亲核试剂 | 构型翻转 |
| SN1 | 单分子取代 | 3°, 烯丙基, 苄基 | 弱亲核试剂 | 外消旋 |
| E2 | 双分子消除 | 2°, 3° | 强碱 | 反式共平面消除 |
| E1 | 单分子消除 | 3° | 弱碱/高温 | Zaitsev产物为主 |

## 二、竞争判断矩阵

### 2.1 按底物分类

| 底物 | 强亲核/弱碱 | 强碱/低温 | 强碱/高温 |
|------|-----------|----------|----------|
| CH₃-X | SN2 | E2（极少见） | E2 |
| 1°-R-X | SN2 | E2 | E2 |
| 2°-R-X | SN2为主 | E2为主 | E2 |
| 3°-R-X | SN1 | E2 | E1 |

### 2.2 按试剂分类

- **强亲核/弱碱**（I⁻, Br⁻, RS⁻, N₃⁻）：SN2 有利
- **强亲核/强碱**（HO⁻, RO⁻）：1°→SN2；2°/3°→E2竞争
- **弱亲核/强碱**（t-BuO⁻, LDA）：E2 有利
- **弱亲核/弱碱**（H₂O, ROH）：SN1/E1（3°底物）

### 2.3 按溶剂分类

- **极性质子溶剂**（H₂O, EtOH）：SN1/E1 有利（3°）
- **极性非质子溶剂**（DMSO, DMF）：SN2/E2 有利
- **非极性溶剂**：反应速率普遍降低

## 三、E2 的立体化学要求

### 3.1 反式共平面（Anti-periplanar）

- H 和 LG 必须处于反式共平面（二面角 180°）
- 这是 E2 发生的几何前提
- 环己烷体系中要求 H 和 LG 均为直立键（a键）

### 3.2 Zaitsev 规则 vs Hofmann 规则

| 规则 | 条件 | 产物 |
|------|------|------|
| Zaitsev | 通常条件 | 取代基多的烯烃（热力学稳定） |
| Hofmann | 大位阻碱（t-BuO⁻） | 取代基少的烯烃（位阻控制） |

## 四、解题决策树

```
1. 看底物
   ├─ 1° / CH₃ → 几乎总是 SN2（强碱条件下 E2）
   ├─ 3° → SN1 或 E1（强碱下 E2）
   └─ 2° → 看试剂和条件

2. 看试剂（亲核性 vs 碱性）
   ├─ 强亲核 + 弱碱 → SN2
   ├─ 弱亲核 + 强碱 → E2
   └─ 弱亲核 + 弱碱 → SN1 / E1

3. 看溶剂
   ├─ 极性质子 → 有利于 SN1/E1
   └─ 极性非质子 → 有利于 SN2/E2

4. 看温度
   ├─ 低温 → 取代有利（动力学控制）
   └─ 高温 → 消除有利（热力学控制）
```

## 五、常见试剂分类表

| 试剂 | 亲核性 | 碱性 | 典型反应 |
|------|--------|------|---------|
| I⁻, Br⁻ | 强 | 弱 | SN2 |
| Cl⁻ | 中等 | 弱 | SN2 |
| N₃⁻ | 强 | 弱 | SN2 |
| RS⁻ | 强 | 中等 | SN2 |
| HO⁻ | 中等 | 强 | SN2/E2 |
| RO⁻ | 中等 | 强 | SN2/E2 |
| t-BuO⁻ | 弱 | 很强 | E2（Hofmann） |
| LDA | 弱 | 极强 | E2 |

## 六、竞赛高频考点

1. **环己烷体系的 E2**：必须 a,a-反式共平面
2. **桥头碳的 SN2**：由于背面进攻受阻，几乎不发生
3. **SN1 中的重排**：碳正离子中间体可发生 1,2-氢/烷基迁移
4. **溶剂解反应**：溶剂本身作为亲核试剂（SN1）

## 七、典型例题与备课调用指南

> **说明**：初赛直接以"SN1/SN2/E1/E2 机理判断"为主考点的题目出现频率较低，更多以**"反应条件改变 → 产物变化"**的形式嵌入有机合成题中。以下提供两类素材：自编入门例题（用于课堂即时练习）+ 真题调用指南（用于巩固与拔高）。

### 入门例题 1：判断主要产物（课堂练习，5 min）

**题目**：写出下列反应的主要产物，并判断机理类型（SN1/SN2/E1/E2）。

(1) 1-溴丙烷 + NaOH(稀)/H₂O, 25 °C → ?
(2) 2-溴丙烷 + NaOH(浓)/EtOH, 80 °C → ?
(3) 叔丁基溴 + NaCN/DMSO, 25 °C → ?
(4) 环己基溴 + t-BuOK/t-BuOH, 70 °C → ?

**预期解答路径**：
1. 先看底物级数 → 定候选机理
2. 再看试剂（亲核性 vs 碱性）→ 定取代/消除偏好
3. 最后看溶剂/温度 → 定最终路径

**答案**：(1) SN2, 1-丙醇  (2) E2, 丙烯  (3) SN1, 叔丁基氰（实际主要产物是消除产物，因CN⁻碱性也不弱；此处为教学简化） (4) E2, 环己烯（大位阻碱）

## 八、备课时间线建议（45 min 专题课）

| 时间段 | 内容 | 形式 |
|:---:|:---|:---|
| 0-8 min | **概念引入**：用"同样的底物，换试剂就换产物"制造认知冲突 | 演示实验 / 案例对比 |
| 8-20 min | **核心对比表讲解**：SN2 vs SN1（底物、试剂、溶剂、立体化学） | 板书 / PPT |
| 20-28 min | **竞争判断矩阵**：按底物分类的速查表 + 试剂分类表 | 学生默背 + 抽查 |
| 28-38 min | **入门例题 1 课堂练习**：4 小题，学生当场写 + 互评 | 纸笔练习 |
| 38-45 min | **决策树实战**：给一道陌生底物，学生口述判断过程 | 口头答辩 |

> **分层策略**：
> - **基础班**：重点讲 SN2 vs E2（初赛最常考），SN1/E1 简略带过
> - **提高班**：四种机理全讲，强调"竞争"而非"非此即彼"
> - **竞赛班**：增加离子对效应、邻基参与、溶剂解等细节，引入决赛真题
