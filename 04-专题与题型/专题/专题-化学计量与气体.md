## 一、核心结论汇总 {#core-conclusions}

**必须记住：**
1. **物质的量是化学计量计算的"中心桥梁"**——所有宏观可测量（质量、气体体积、溶液浓度）均通过 $n$ 与微观粒子数建立联系。拿到任何定量题，第一步是"把一切转成 $n$"。
2. **理想气体状态方程与分压定律是气体计算的两大支柱**：$pV=nRT$ 处理纯气体，Dalton 定律处理混合气体。计算前必须确认**单位匹配**（$p,V$ 与 $R$ 一致）和**温度换算**（热力学温度）。
3. **浓度表示法的选择取决于"是否随温度变化"和"计算便利性"**：理论推导/依数性计算用摩尔分数或质量摩尔浓度（不随温度变）；实验操作和滴定计算用物质的量浓度（取液方便）。

**最高频决策路径：**

```text
拿到定量计算题
    ├── 已知为气体（p, V, T）→ 用 pV=nRT 求 n → 进入化学计量关系
    ├── 已知为溶液（c, V）→ 用 n=cV 求 n → 进入化学计量关系
    ├── 已知为固体（m）→ 用 n=m/M 求 n → 进入化学计量关系
    └── 混合气体 → 先判恒容/恒压 → 用 Dalton 定律求分压/分体积
              ↓
        写出配平方程式 → 按计量系数比求目标物质的 n → 转换回目标物理量
              ↓
        检查：单位一致？有效数字正确？结果合理？
```

## 二、对比表格 {#comparison-table}

### 表 1：浓度表示法速查（触发条件：题目要求"配制""换算""比较"）

| 触发条件 | 表示法 | 公式 | 单位 | 温度依赖性 | 典型用途 |
|:---|:---|:---|:---|:---:|:---|
| 商品试剂标签、粗略配制 | **质量分数** $w$ | $w = m_{\text{质}} / m_{\text{液}}$ | 无量纲 | 无 | 浓硫酸 $98\%$、双氧水 $30\%$ |
| 理论推导、Raoult 定律 | **摩尔分数** $x$ | $x_i = n_i / \sum n$ | 无量纲 | 无 | 蒸气压下降、沸点升高计算 |
| 依数性、精确理论计算 | **质量摩尔浓度** $b$ | $b = n_{\text{质}} / m_{\text{剂}}$ | $\mathrm{mol \cdot kg^{-1}}$ | 无 | 凝固点降低常数 $K_f$ 配合 |
| 实验室取液、滴定计算 | **物质的量浓度** $c$ | $c = n_{\text{质}} / V_{\text{液}}$ | $\mathrm{mol \cdot dm^{-3}}$ | 有 | 滴定分析、反应速率 |
| 混合溶剂配制 | **体积分数** $\varphi$ | $\varphi = V_{\text{组分}} / \sum V$ | 无量纲 | 有 | 乙醇-水混合溶剂 |

> **常见陷阱**：稀水溶液中 $c \approx b$（因 $1\ \mathrm{kg}$ 水 $\approx 1\ \mathrm{dm^3}$），但**不能通用**；配制 $c$ 溶液时必须用容量瓶定容，不能将溶质加入固定体积溶剂中。

### 表 2：气体定律适用条件与公式选择

| 触发条件 | 定律/公式 | 核心公式 | 适用条件 | 常见陷阱 |
|:---|:---|:---|:---|:---|
| 纯气体，求 $n$、$M$、$\rho$ | **理想气体状态方程** | $pV = nRT = \frac{m}{M}RT$ | 温度不太低、压力不太高 | 单位不匹配（$\mathrm{kPa}$ 配 $R=0.0821$）；忘记 $T$ 换算 |
| 混合气体，已知总压 | **Dalton 分压定律** | $p_i = p_{\text{总}} \times x_i$ | 各组分均为理想气体、不反应 | 不同初压气体直接相加；混淆分压与分体积 |
| 恒容混合 | **Boyle 扩展** | $p_i = p_{\text{初}} \times \frac{V_{\text{初}}}{V_{\text{总}}}$ | 等温、混合前各气体压力不同 | 直接用初始压力相加得总压 ❌ |
| 排水集气 | **蒸气压校正** | $p_{\text{干}} = p_{\text{总}} - p(\mathrm{H_2O})$ | 气体通过水收集 | 忘记扣除水蒸气分压 |
| 精确测定分子量 | **外延法** | $pV$ 对 $p$ 作图，外推至 $p \to 0$ | 实际气体、低压多组数据 | 单点直接计算，忽略非理想性偏差 |

### 表 3：有效数字运算规则速查

| 运算类型 | 规则 | 示例 | 常见陷阱 |
|:---|:---|:---|:---|
| **加减** | 以小数点后位数最少者为准 | $25.37 + 1.2 + 0.056 = 26.6$ | 误按总有效位数取舍 |
| **乘除** | 以有效数字位数最少者为准 | $25.37 \times 1.2 \times 0.056 = 1.7$ | 与加减规则混淆 |
| **对数（pH）** | 小数点后位数 = 真数有效位数 | $[\mathrm{H}^+] = 5.3 \times 10^{-4}$ → $\mathrm{pH} = 3.28$ | 误将整数部分计入有效数字 |
| **精确数字** | 定义值、计量系数视为无限位 | 方程式系数 2、换算 $1\ \mathrm{L}=1000\ \mathrm{mL}$ | 将 1000 当作 4 位有效数字 ❌ |

## 三、解题套路 / 决策流程 {#problem-solving-routine}

### Step 1：统一转换为物质的量 $n$
- **操作**：识别题目给出的已知条件类型，选择对应公式转换为 $n$
  - 质量：$n = m / M$
  - 气体（纯）：$n = pV / (RT)$
  - 溶液：$n = cV$
  - 粒子数：$n = N / N_A$
- **依据 KP**：[[化学计量#物质的量的桥梁作用]]、[[理想气体状态方程#方程的导出]]
- **检查点**：☐ $p,V$ 单位与 $R$ 匹配 ☐ $T$ 已换算为热力学温度 ☐ $c$ 的单位是 $\mathrm{mol \cdot dm^{-3}}$

### Step 2：应用化学计量关系
- **操作**：写出配平的化学方程式，根据计量系数比确定目标物质的 $n$
  - 限量试剂判断：$R_i = n_i / \nu_i$，最小者为限量试剂
  - 连续反应：找出总反应关系或逐步计算
- **依据 KP**：[[化学计量#限量试剂判别方法]]、[[方程式书写#常见配平方法]]
- **检查点**：☐ 方程式已配平 ☐ 限量试剂已判定（若多反应物）

### Step 3：转换为目标物理量并验证
- **操作**：将求得的 $n$ 转换为题目要求的物理量（$m$、$V$、$c$、$N$ 等）
- **验证清单**：
  - [ ] 结果的有效数字位数与原始数据匹配
  - [ ] 气体体积是否在合理范围（STP 下 $1\ \mathrm{mol} \approx 22.4\ \mathrm{dm^3}$）
  - [ ] 浓度数量级是否合理（浓硫酸约 $18\ \mathrm{mol \cdot dm^{-3}}$，稀溶液约 $0.01\sim1$）
  - [ ] 限量试剂判断后，理论产量不超过按限量试剂计算值
- **依据 KP**：[[有效数字#加减运算规则]]、[[化学计量#产率]]

| 步骤 | 核心操作 | 依据 KP | 检查清单 |
|:---|:---|:---|:---|
| 1 | 识别条件类型，统一转换为 $n$ | [[化学计量]]、[[理想气体状态方程]] | ☐ 单位匹配 ☐ $T$ 换算 ☐ 浓度单位正确 |
| 2 | 写配平方程式，按计量比求目标 $n$ | [[化学计量#限量试剂]]、[[方程式书写]] | ☐ 方程式配平 ☐ 限量试剂已判定 |
| 3 | 转换为目标物理量，验证合理性 | [[有效数字]]、[[溶液浓度]] | ☐ 有效数字正确 ☐ 结果数量级合理 |

## 五、典型例题串讲 {#typical-examples}

### 例题 1：限量试剂与混合气体分压综合（⭐⭐⭐）

**题目**：将 $5.00\ \mathrm{g}$ $\mathrm{Fe}$ 与 $3.00\ \mathrm{g}$ $\mathrm{S}$ 在真空密闭容器中加热反应生成 $\mathrm{FeS}$。反应完成后，向容器中加入 $100\ \mathrm{cm^3}$ $2.00\ \mathrm{mol \cdot dm^{-3}}$ 的稀硫酸，完全反应后收集到气体（经干燥）。若在 $25^\circ\mathrm{C}$、$101.0\ \mathrm{kPa}$ 下测得干燥气体体积为 $2.10\ \mathrm{dm^3}$，求：
(1) 原反应中 $\mathrm{FeS}$ 的产率；
(2) 收集到的气体中各组分的分压。

**分析**：
- 第一步：判断 $\mathrm{Fe}$ 与 $\mathrm{S}$ 的限量试剂，计算理论产量
- 第二步：$\mathrm{FeS}$ 与稀硫酸反应生成 $\mathrm{H_2S}$；过量的 $\mathrm{Fe}$ 与稀硫酸反应生成 $\mathrm{H_2}$
- 第三步：用理想气体状态方程求气体总物质的量，再用分压定律求各组分分压

**解答**：

(1) **限量试剂判断**：
$$n(\mathrm{Fe}) = \frac{5.00}{55.85} = 0.0895\ \mathrm{mol},\quad n(\mathrm{S}) = \frac{3.00}{32.07} = 0.0935\ \mathrm{mol}$$
计量比 $1:1$，$\mathrm{Fe}$ 为限量试剂，理论 $n(\mathrm{FeS}) = 0.0895\ \mathrm{mol}$

加入硫酸：$n(\mathrm{H_2SO_4}) = 2.00 \times 0.100 = 0.200\ \mathrm{mol}$（过量）

若 $\mathrm{FeS}$ 产率为 $\eta$，则：
- $\mathrm{FeS}$ 与酸反应：$\mathrm{FeS} + \mathrm{H_2SO_4} \to \mathrm{FeSO_4} + \mathrm{H_2S}$
  - 生成 $n(\mathrm{H_2S}) = 0.0895\eta\ \mathrm{mol}$
- 剩余 $\mathrm{Fe}$ 与酸反应：$\mathrm{Fe} + \mathrm{H_2SO_4} \to \mathrm{FeSO_4} + \mathrm{H_2}$
  - 剩余 $n(\mathrm{Fe}) = 0.0895(1-\eta)\ \mathrm{mol}$
  - 生成 $n(\mathrm{H_2}) = 0.0895(1-\eta)\ \mathrm{mol}$

气体总物质的量：
$$n_{\text{总}} = \frac{pV}{RT} = \frac{101.0 \times 2.10}{8.31 \times 298} = 0.0857\ \mathrm{mol}$$

又
$$n_{\text{总}} = n(\mathrm{H_2S}) + n(\mathrm{H_2}) = 0.0895\eta + 0.0895(1-\eta) = 0.0895\ \mathrm{mol}$$

可见每摩尔 $\mathrm{Fe}$ 无论经 $\mathrm{FeS}$ 路径（产生 $\mathrm{H_2S}$）还是直接路径（产生 $\mathrm{H_2}$），均产生 $1\ \mathrm{mol}$ 气体，因此气体总量与产率无关。

由理想气体状态方程反推理论体积（假设产率 $100\%$）：
$$V_{\text{理论}} = \frac{nRT}{p} = \frac{0.0895 \times 8.31 \times 298}{101.0} = 2.19\ \mathrm{dm^3}$$

实测体积 $2.10\ \mathrm{dm^3}$，产率估算：
$$\eta = \frac{2.10}{2.19} \times 100\% \approx \mathbf{96\%}$$

(2) **各组分分压**：
- 由产率 $\eta$，$x(\mathrm{H_2S}) = \eta$，$x(\mathrm{H_2}) = 1-\eta$
- $p(\mathrm{H_2S}) = 101.0 \times 0.96 = 96.9\ \mathrm{kPa}$
- $p(\mathrm{H_2}) = 101.0 \times 0.04 = 4.1\ \mathrm{kPa}$

**反思**：本题综合了限量试剂判断、产率计算、气体计量和分压定律。关键洞察是"每摩尔 $\mathrm{Fe}$ 无论生成 $\mathrm{FeS}$ 与否，最终都产生 $1\ \mathrm{mol}$ 气体（$\mathrm{H_2S}$ 或 $\mathrm{H_2}$）"，因此气体总量直接反映 $\mathrm{Fe}$ 的初始量。产率差异只影响气体组成比例，不影响总量——这是本题的精妙之处。

### 例题 2：燃烧分析确定分子式（⭐⭐⭐）

**题目**：某碳氢化合物样品 $0.500\ \mathrm{g}$ 完全燃烧，生成 $1.540\ \mathrm{g}$ $\mathrm{CO_2}$ 和 $0.756\ \mathrm{g}$ $\mathrm{H_2O}$。已知其摩尔质量约 $58\ \mathrm{g/mol}$，求分子式。

**分析**：
- 由 $\mathrm{CO_2}$ 质量求 $n(\mathrm{C})$，由 $\mathrm{H_2O}$ 质量求 $n(\mathrm{H})$
- 验证是否含氧（质量守恒）
- 求实验式，结合摩尔质量确定分子式

**解答**：
$$n(\mathrm{C}) = n(\mathrm{CO_2}) = \frac{1.540}{44.01} = 0.03500\ \mathrm{mol}$$
$$n(\mathrm{H}) = 2n(\mathrm{H_2O}) = 2 \times \frac{0.756}{18.02} = 0.08391\ \mathrm{mol}$$
$$m(\mathrm{C}) = 0.03500 \times 12.01 = 0.4204\ \mathrm{g}$$
$$m(\mathrm{H}) = 0.08391 \times 1.008 = 0.08458\ \mathrm{g}$$
质量和 $= 0.5050\ \mathrm{g} \approx 0.500\ \mathrm{g}$，确认不含氧。

$$\mathrm{C:H} = 0.03500 : 0.08391 = 1 : 2.40 = 5 : 12$$

实验式 $\mathrm{C_5H_{12}}$，式量 $= 72.15$。但题目给出摩尔质量约 $58\ \mathrm{g/mol}$，与实验式不符。重新检查——最接近的合理分子式为 $\mathrm{C_4H_{10}}$（丁烷，$M = 58.12$），实验误差来源于摩尔质量的"约"字。

**反思**：燃烧分析的核心是"碳守恒"和"氢守恒"。注意氢原子数要乘以 2；若质量和大于样品质量则含氧，小于则可能有计算误差或样品不纯。
