# 专题：有机合成逆合成分析

> 本专题对应考纲条目：[[51-有机合成]]
> 核心知识点：[[有机合成]]、[[逆合成分析]]、[[保护基化学]]

---

## 一、逆合成分析基本思想

### 1.1 核心概念

- **逆合成（Retrosynthesis）**：从目标分子（TM）出发，逆向推导出简单易得的起始原料
- **合成子（Synthon）**：逆合成分析中的虚拟结构单元
- **等价试剂（Equivalent）**：合成子的实际反应试剂

### 1.2 常用断开策略

| 策略 | 适用情况 | 典型转化 |
|------|---------|---------|
| 官能团转化（FGI） | 目标官能团难以直接引入 | 羧酸 → 醛 → 醇 |
| 官能团添加（FGA） | 需要引入活化基团 | 在目标位点引入羰基再还原 |
| 碳-碳键断开 | 构建碳骨架 | 羰基化合物的 α,β-断开 |

## 二、常见 C-C 键构建反应

### 2.1 亲核加成类

| 反应 | 合成子 | 产物类型 |
|------|--------|---------|
| Grignard + 醛/酮 | 碳负离子 + 亲电羰基 | 醇 |
| 羟醛缩合 | 烯醇负离子 + 亲电羰基 | β-羟基羰基/α,β-不饱和羰基 |
| Wittig 反应 | 磷叶立德 + 羰基 | 烯烃 |
| 氰化物加成 | CN⁻ + 羰基 | α-羟基腈 |

### 2.2 亲核取代类

| 反应 | 合成子 | 产物类型 |
|------|--------|---------|
| 炔钠烷基化 | 炔负离子 + 卤代烷 | 高级炔烃 |
| 丙二酸酯合成 | 烯醇负离子 + 卤代烷 | 取代乙酸 |
| 有机铜锂试剂 | R₂CuLi + 卤代烷/酰卤 | 酮/烷烃 |

### 2.3 偶联反应（金属有机）

| 反应 | 催化剂 | 产物类型 |
|------|--------|---------|
| Suzuki 偶联 | Pd(0) | 联芳基/烯基芳基 |
| Heck 反应 | Pd(0) | 芳基烯烃 |
| Sonogashira 偶联 | Pd/Cu | 芳基炔烃 |

## 三、官能团引入与转化

### 3.1 官能团引入顺序原则

1. **先引入稳定、耐受性好的官能团**
2. **敏感官能团后引入或在最后脱保护**
3. **考虑官能团的正交保护策略**

### 3.2 常见官能团转化路径

```
烷烃 → 卤代烷 → 格氏试剂/有机锂 → 醇/酮/酸
                    ↓
              烯烃 ← 消除
                    ↓
              环氧/二醇 ← 氧化
```

## 四、保护基策略

### 4.1 羟基保护

| 保护基 | 保护条件 | 脱保护条件 |
|--------|---------|-----------|
| 甲基醚 (Me) | NaH, MeI | BBr₃ 或 HI |
| 苄基醚 (Bn) | NaH, BnBr | H₂/Pd 或 Na/NH₃ |
| 硅醚 (TMS, TBS) | R₃SiCl, 碱 | TBAF 或 HF |
| 缩醛/缩酮 | 酮/醛 + 醇/二醇 | 酸水溶液 |

### 4.2 羰基保护

| 保护形式 | 保护条件 | 脱保护条件 |
|---------|---------|-----------|
| 缩醛/缩酮 | 乙二醇, 酸催化 | 酸水溶液 |
| 硫缩醛/硫缩酮 | 乙二硫醇, BF₃ | Hg²⁺ 或 Raney Ni |

### 4.3 氨基保护

| 保护基 | 保护条件 | 脱保护条件 |
|--------|---------|-----------|
| Boc | (Boc)₂O, 碱 | TFA |
| Cbz (Z) | Cbz-Cl, 碱 | H₂/Pd 或 HBr/AcOH |

## 五、竞赛合成题解题策略

### 5.1 分析目标分子

1. **碳骨架**：数碳原子数，判断需要构建几个 C-C 键
2. **官能团**：识别所有官能团及其位置关系
3. **立体化学**：注意手性中心、顺反异构等

### 5.2 逆合成切断

1. **优先切断 C-C 键**：尤其是带有官能团的碳附近
2. **利用已知反应的逆过程**：如羟醛缩合的逆过程是切断 α,β-C-C 键
3. **简化到常见起始原料**：≤6 个碳的常见化合物

### 5.3 常见起始原料

- 乙醇、乙醛、丙酮、乙酸、苯、甲苯
- 环己烷、环己烯、环己酮
- 丙二酸二乙酯、乙酰乙酸乙酯

## 六、真题策展（按难度梯度）

> **备课建议**：逆合成分析是决赛核心能力，初赛几乎不直接考查完整合成设计。本专题适合第三轮有机深化阶段讲授，需配合大量纸笔推演。

### ⭐⭐⭐ 提高：C-C键构建方法学
- [[题-35决理-1-8-有机新试剂与合成应用]] —— 决赛2021，新试剂在合成中的选择性应用（难度4）
- **备课要点**：引导学生从"试剂记忆"转向"合成子识别"——看到试剂即想到它代表的合成子

### ⭐⭐⭐⭐ 拔高：碎片化反应与官能团转化
- [[题-36决理-2-10-Wharton碎片化反应]] —— 决赛2022，α,β-环氧酮的碎片化，逆合成中C-C键断开的经典案例（难度5）
- **备课要点**：Wharton碎片化是"官能团转化(FGI)+碳骨架重排"的典型，训练学生逆向识别碎片化前体

### ⭐⭐⭐⭐ 拔高：杂原子参与的ene反应
- [[题-37决理-2-8-杂原子参与的 ene 反应]] —— 决赛2023，杂原子ene反应构建C-X键的策略（难度5）
- **备课要点**：ene反应可视为"协同版的烯丙位取代"，逆合成中识别ene产物的前体模式

### ⭐⭐⭐⭐ 综合：碳正离子化学与合成
- [[题-38决理-9-碳正离子化学]] —— 决赛2024，碳正离子的重排/捕获在合成中的应用（难度5）
- **备课要点**：碳正离子重排常导致"预期之外的碳骨架"，逆合成时需考虑重排可能性

### ⭐⭐⭐ 官能团保护实战
- [[题-36决理-1-7-卤化氢前体在合成中的应用]] —— 决赛2022，卤化氢等价体在敏感官能团存在下的选择性反应（难度4）
- **备课要点**：当直接引入官能团会破坏已有官能团时，必须考虑保护基或等价试剂策略

### 课堂推演示范（教师演示，15分钟）
**目标分子**：某萜类天然产物中间体（含1个季碳手性中心）

```
步骤1：数碳 → 15碳，判断需构建3个C-C键
步骤2：识别官能团 → 酮、醚、双键
步骤3：优先切断 → α,β-不饱和酮附近（羟醛缩合逆过程）
步骤4：保护基检查 → 酮在Wittig反应前需保护为缩酮
步骤5：起始原料 → 环己酮 + 丙酮 + 甲醛（均≤6碳）
```

**逆合成分析检查清单（学生自查用）**：
- [ ] 碳原子数守恒（产物 = 原料之和）
- [ ] 官能团耐受性检查（反应条件不会破坏已有官能团）
- [ ] 保护基使用合理（不脱保护即进行下一步）
- [ ] 立体化学考虑（手性中心的构型控制策略）
- [ ] 起始原料≤6碳常见化合物
