# 1）皂化反应的原理 02:02:20

● 反应物: 油脂（如牛油中的甘油三硬脂酸酯）与碱（氢氧化钠或氢氧化钾）

○ 使用NaOH时生成钠盐（如硬脂酸钠 $C_{17}H_{35}CO_{2}Na$ ），是固体肥皂主要成分  
- 使用KOH时生成钾盐，用于制造液体肥皂

● 分子特性: 同时具有极性羧酸根基团和非极性长链烷基，赋予其表面活性剂特性

# 2）肥皂的成分与问题 02:03:19

# ● 主要成分:

- 普通肥皂：以硬脂酸钠为主  
- 高级肥皂：含更高比例的棕榈酸钠 $C_{15}H_{31}CO_2Na$ （来自棕榈油）

- 十六烷酸（棕榈酸） $C_{16}H_{32}O_2$   
○ 十八烷酸（硬脂酸） $C_{18}H_{36}O_{2}$

# 3）磺酸根替代缩酸根的原因 02:04:02

# - 溶解度问题:

- 传统肥皂（羧酸盐）的钙/镁盐溶解度极低

# - 改进方案:

现代常用苯磺酸钠盐（结构：苯环连接长链+磺酸根）  
○ 磺酸盐在硬水中溶解度更高，去污效果更稳定
