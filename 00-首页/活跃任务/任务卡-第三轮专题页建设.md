# 第三轮专题页建设

为有机 / 结构 / 物化深化建立第三轮专题页骨架，按 `[[模板-专题]]` v1.7 新建。

本轮新增骨架：`有机结构基础与电子效应 / 立体化学与区域选择性 / 活性中间体与反应机理基础 / 加成反应 / 羰基化学与缩合反应 / 重排反应 / 芳香反应`。
