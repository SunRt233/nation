### 🔴 高优（影响日常使用）

1. **断链检测脚本降噪**

   - 优化：增强占位符过滤规则，把路径前缀链接、`.base` 后缀、图片示例链接加入白名单
   - 工作量：~30 分钟改脚本 + 重跑基线

## 三、首页 2.0 提案（操作台化）

> 详见与 Agent 的对话记录，或参考以下核心思路：

### 设计目标
把"静态导航页"变成"动态操作台"，按当前两种工作状态切换：

```
┌─────────────────────────────────────────┐
│  [📊 状态快照]  [📚 备课模式]  [🔧 建设模式]  │
├─────────────────────────────────────────┤
│                                         │
│  备课模式（默认显示）                      │
│  ├── 快速入口：新授课 / 复习课 / 习题课 模板  │
│  ├── 数据库直达：课件.base / 真题.base      │
│  ├── 本周计划：Dataview 拉取最近标记         │
│  └── 热点专题：最近真题高频考点             │
│                                         │
│  建设模式（折叠/二级）                     │
│  ├── 资料导入管线：导入 → 提炼 → 并入      │
│  ├── 缺口看板：决赛 11/32 + 实验模块       │
│  └── 快速工具：断链检测 / 健康检查 / 模板升级 │
│                                         │
└─────────────────────────────────────────┘
```

### 关键改动点
1. **顶部增加"今日操作区"**：把最常用的 4-6 个链接做成卡片按钮（模板、数据库、最近工作日志、真题索引）
2. **状态栏用 Dataview 动态拉取**：今日待办数、本周备课数、最近更新文件
3. **双模式 Tab 切换**：用 Markdown 折叠块或 Obsidian Callouts 组织，避免一屏信息过载
4. **保留底部精简导航**：把现有的长列表导航压缩成"系统文件 / 数据库 / 索引 / 教材提炼"四列快捷链接
