## 🕐 最近 3 次任务（DataviewJS 自动，详见 [[工作日志]]）

> 自动抽取 `00-首页/工作日志/` 下所有文件中 `## 📅 YYYY-MM-DD <title>` H2 标题的最近 3 条。
> 净增 KP / 改动明细需要点进具体日期的日志文件查看。

```dataviewjs
// 扫描 00-首页/工作日志/ 下的所有文件，提取 H2 标题
const folder = "00-首页/工作日志";
const files = app.vault.getMarkdownFiles().filter(f => f.path.startsWith(folder + "/"));

const allTasks = [];
for (const file of files.sort((a, b) => b.stat.mtime - a.stat.mtime)) {
  const content = await app.vault.read(file);
  const lines = content.split("\n");
  for (const line of lines) {
    const m = line.match(/^##\s*📅\s*(\d{4}-\d{2}-\d{2})\s+(.+?)\s*$/);
    if (m) {
      allTasks.push({ date: m[1], title: m[2].trim(), mtime: file.stat.mtime });
    }
  }
}

// 按修改时间倒序，取最近 3 条
allTasks.sort((a, b) => b.mtime - a.mtime);
const top3 = allTasks.slice(0, 3);

if (top3.length === 0) {
  dv.paragraph("⚠️ 未在 `00-首页/工作日志/` 中找到符合 `## 📅 YYYY-MM-DD ...` 格式的标题。");
} else {
  dv.table(
    ["日期", "标题"],
    top3.map(t => [t.date, t.title])
  );
}
```

## ⚠️ 踩坑速查 Top 5（DataviewJS 自动，来自 [[经验沉淀]] 🔥 标记）

> 自动抽取 [[经验沉淀]] 中标了 🔥 的 H3 条目（高频踩坑）的前 5 条。
> 加 🔥/去 🔥 即可调整本表，无需改本文件。完整说明见原条目。

```dataviewjs
const exp = await dv.io.load("00-首页/经验沉淀.md");
const lines = exp.split("\n");

const tops = [];
let currentCategory = "";
for (const line of lines) {
  // 一级分类标题：## 一、Frontmatter / YAML 类
  const cat = line.match(/^##\s+([一二三四五六七八九十]+)、(.+?)\s*$/);
  if (cat) {
    currentCategory = cat[2].trim();
    continue;
  }
  // 🔥 标记的 H3 条目：### 1-1 全角冒号导致 Edit 失败 🔥
  const m = line.match(/^###\s+(\S+)\s+(.+?)\s*🔥\s*$/);
  if (m) {
    tops.push({ code: m[1], title: m[2].trim(), category: currentCategory });
    if (tops.length >= 5) break;
  }
}

if (tops.length === 0) {
  dv.paragraph("⚠️ 未在 [[经验沉淀]] 中找到 🔥 标记的 H3 条目。");
} else {
  dv.list(tops.map(t => `**${t.code}** ${t.title}（${t.category}）`));
}
```
