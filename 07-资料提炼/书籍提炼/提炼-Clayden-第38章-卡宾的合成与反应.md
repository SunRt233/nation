# 提炼：Clayden 第38章——卡宾的合成与反应（Carbene Synthesis and Reactions）

---

## 一、核心概念与定义

### 1.1 卡宾（Carbene）
- **定义**：包含只具有六个价电子的碳原子的中性物种，通式为 $:CR_2$。
- **与碳阳离子的区别**：卡宾鎓离子（carbenium ions，即碳阳离子如 $^+CH_3$）同样具有六个价电子，但卡宾是**中性**的，不带电荷。
- **与自由基的类比**：卡宾和自由基一样，是尤其活泼的物种，但卡宾的电子状态更为复杂。

### 1.2 单线态与三线态卡宾
- **三线态卡宾（Triplet）**：
  - 具有两个未成对电子，可被 ESR 观察到。
  - 键角较大（130–150°），电子排斥较小。
  - 典型代表：$:CH_2$、$:CHPh$、$:CHR$、$:CPh_2$。
- **单线态卡宾（Singlet）**：
  - 所有电子都成对，不可被 ESR 观察到。
  - 键角较小（100–110°），$sp^2$ 轨道中孤对电子排斥较大。
  - 典型代表：$:CCl_2$、$:CHCl$、$:C(OMe)_2$、N-杂环卡宾。
- **电子结构**：
  - 三线态：一个电子在 p 轨道，一个电子在 $sp^2$ 轨道（洪特规则）。
  - 单线态：两个电子均成对进入 $sp^2$ 轨道，p 轨道空置。
- **取代基效应**：邻位具有孤对电子的富电子取代基（如 Cl、OR、NR₂）可稳定单线态——孤对电子与卡宾的空 p 轨道相互作用，降低能量，激励 p 轨道电子在 $sp^2$ 轨道中成对。

### 1.3 卡宾体（Carbenoid）与类卡宾
- **卡宾体（Carbenoid）**：由重氮化合物的金属催化分解得到的与过渡金属键合的卡宾，仍保留与金属的络合。
- **类卡宾**：如锂类卡宾（lithium carbenoid，$R_2C(Li)X$）、锌类卡宾（zinc carbenoid，Simmons-Smith 试剂），具有一个可接受电子的离去基团和一个可贡献电子的金属。

### 1.4 N-杂环卡宾（NHC）
- 卡宾碳被包含进五元环内，被两个邻位给电子氮原子以及大体积 N-取代基所稳定。
- 1991 年首次制得结晶样品，X-射线显示卡宾 C 键角约 $102^{m o}$，$^{13}$C NMR 确认缺电子性。
- 现代 Grubbs II 和 Hoveyda-Grubbs 催化剂的关键配体。

## 二、重要反应与机理

### 2.1 卡宾的制备方法

#### 2.1.1 由重氮化合物失去 $N_2$（line ~1836）
- **重氮甲烷 $CH_2N_2$**：光解或热解产生 $:CH_2$。
  - 重氮甲烷可甲基化羧酸（质子化 → 重氮阳离子 → $S_N2$ 取代 → 甲酯 + $N_2$）。
  - 毒性大、易爆，沸点仅 $-24^{\rm o}C$，需稀溶液使用。
- **重氮羰基化合物**：比重氮烷烃稳定得多，是更实用的卡宾来源。
  - 制备方法1：酰氯 + 重氮甲烷 → 重氮酮。
  - 制备方法2：羰基化合物 + 对甲苯磺酰叠氮 $(TsN_3)$ + 碱。
  - 热分解或光分解产生卡宾，氮气的失去在能量上补偿了不稳定卡宾的形成。

#### 2.1.2 由对甲苯磺酰腙得到卡宾（Bamford-Stevens 反应）（line ~2126）
- 对甲苯磺酰腙 → 碱催化消除对甲苯亚磺酸根 $(Ts^-)$ → 短暂产生重氮化合物 → 热分解为卡宾。
- 安全性优于直接使用重氮化合物。

#### 2.1.3 通过 $\alpha$ 消除形成卡宾（line ~2157）
- **氯仿 + 强碱**：碱夺取质子 → 碳阴离子 → 失去 $Cl^-$ → 二氯卡宾 $:CCl_2$。
  - 氢氧根、烷氧基阴离子足以促使反应；更强碱（LDA）可使二卤烷烃去质子。
- **三氯乙酸钠脱羧**：$Cl_3CCO_2^- \xrightarrow{\Delta} :CCl_2 + CO_2 + Cl^-$，约 $80^{\rm o}C$ 完成，避免使用强碱。
- **锂类卡宾**：偕二溴代烷 + $t$-$BuLi$ → 卤素-金属交换 → $R_2C(Li)Br$，低温稳定，约 $-100^{\rm o}C$ 分解为卡宾。

#### 2.1.4 通过阳离子去质子形成卡宾（line ~2236）
- 咪唑鎓阳离子 + 强碱（如 $NaH$）→ 去质子 → N-杂环卡宾。
- 邻位孤对电子通过离域稳定卡宾中心。

#### 2.1.5 金属催化分解（line ~2102）
- 铜、铑等过渡金属催化重氮羰基化合物分解，形成金属-卡宾配合物（卡宾体）。
- 钨、铬等可形成稳定、可分离的 Fischer 卡宾。

### 2.2 卡宾与烯烃反应——环丙烷化（line ~2464）

#### 2.2.1 单线态卡宾的协同加成
- **机理**：$[1+2]$ 环加成（周环反应），卡宾以"侧向（sideways-on）"方式接近烯烃。
  - 卡宾的空 p 轨道（LUMO）与烯烃的 $\pi$ 键（HOMO）相互作用；
  - 或卡宾 $sp^2$ 孤对电子（HOMO）与烯烃 $\pi^*$（LUMO）相互作用。
- **立体化学**：**立体专一性**——烯烃几何结构在产物中保留（Z → 顺式环丙烷，E → 反式环丙烷）。
- 卡宾在成键同时旋转 $90^{\rm o}$，使原来卡宾的碳达到四面体排列。

#### 2.2.2 三线态卡宾的分步加成
- **机理**：自由基加成 → 双自由基（三线态）中间体 → 自旋翻转（spin-flipping，需碰撞）→ 成键。
- **立体化学**：**非立体专一性**——中间体自由旋转打乱了起始原料的立体化学。
- 稀释于惰性溶剂（如全氟丙烷）中可增加单线态→三线态翻转机会，降低立体专一性。

#### 2.2.3 Simmons-Smith 反应（line ~2756）
- **试剂**：$Zn/Cu$ 偶 + $CH_2I_2$（或二乙基锌替代）。
- **本质**：锌类卡宾对烯烃的加成，**不释放游离卡宾**。
- **立体化学**：表现如单线态卡宾——立体专一性 + 立体选择性。
- **螯合导向**：烯丙型醇的羟基与锌配位，使亚甲基立体选择性地加成到羟基同面，速率提高 >100 倍。
- **局限**：转移 $CH_2$ 效果好；取代亚甲基（$RCH:$ 或 $R_2C:$）效果较差。

### 2.3 卡宾插入 C–H 键（line ~2854）
- **分子内插入**：优先发生，尤其当卡宾与 C–H 键处于合适位置时。
  - 单线态卡宾：协同插入，立体化学保持。
  - 三线态卡宾：两步自由基路径（理论上），但实验证据较少。
- **选择性插入**：铑催化重氮酮可选择性地插入到特定 C–H 键（如五原子外的 C–H），形成环戊烷。
- **合成价值**：在未官能化中心产生新键，非常有用。

### 2.4 卡宾插入 O–H 和 N–H 键（line ~3069）
- **机理**：卡宾先进攻杂原子孤对电子 → 形成内盐（叶立德）→ 质子转移 → 中性产物。
- **应用**：制取活泼叶立德；铑催化的 N–H 插入用于碳青霉烯类抗生素合成。

### 2.5 卡宾的重排反应

#### 2.5.1 1,2-氢迁移（line ~2866）
- 有 $\beta$-氢的卡宾会极其迅速地发生氢向卡宾中心的 1,2-迁移，给出烯烃。
- 实质是对邻位 C–H 键的"插入"。
- 用非常强的碱（如苯基钠）处理伯卤代烷时，$\alpha$ 消除形成的卡宾迅速重排，导致消除产物氘标记位置异常。

#### 2.5.2 Wolff 重排（line ~2961）
- **底物**：$\alpha$-重氮酮（由酰氯 + 重氮甲烷制得）。
- **机理**：加热或光解 → 卡宾形成 → 烷基向卡宾中心迁移 → 烯酮。
- **产物**：烯酮不稳定，水解为羧酸；醇解为酯。
- **Arndt–Eistert 同系化**：羧酸 → 酰氯 → 重氮酮 → Wolff 重排 → 多一个碳的羧酸（链增长一个 $CH_2$）。

### 2.6 氮宾（Nitrene）及相关重排（line ~3027）
- **氮宾**：卡宾的氮类似物，$R\!-\!N:$，同样具有六个电子。
- **Curtius 重排**：酰基叠氮 → 加热 → 氮宾 → 烷基迁移 → 异氰酸酯 → 水解为胺（少一个碳）。
- **Hofmann 重排**：伯酰胺 + $Br_2$/碱 → 氮宾 → 重排 → 异氰酸酯 → 胺（少一个碳）。
- **Lossen 重排**：异羟肟酸的对甲苯磺酰化 → 氮宾 → 重排。

### 2.7 烯烃复分解（Olefin Metathesis）（line ~3115）
- **Grubbs 催化剂**：Ru(II)-卡宾配合物（如 $[Ru]=CHPh$）。
- **机理**：$[2+2]$ 环加成 → 金属代环丁烷（metallacyclobutane）→ 逆 $[2+2]$ → 新卡宾配合物 + 新烯烃。
- **关环复分解（RCM）**：双烯 → 环状烯烃 + 乙烯（气体离去，驱动反应）。
- **交叉复分解**：分子间反应，需电子/空间性质差异大的搭档。
- **烯-炔复分解**：烯烃 + 炔烃 → 双烯（Diels-Alder 前体）。
- **催化剂演进**：Grubbs I → Grubbs II（NHC 配体）→ Hoveyda-Grubbs（分子内络合）。
- 2005 年诺贝尔奖：Chauvin、Schrock、Grubbs。

## 三、关键结论与规则

### 3.1 卡宾的结构与形成
1. **卡宾以何种自旋态形成取决于前体**：由所有电子成对的分子（如 $\alpha$ 消除）形成时，必为**单线态**；光解可将单线态转化为三线态。
2. **大多数卡宾反应太快，来不及自旋翻转**——即使三线态是基态，也可能以单线态反应。
3. **单线态基态的卡宾**具有邻位给电子取代基（Cl、OR、NR₂），孤对电子离域稳定单线态。

### 3.2 卡宾的反应性
4. **卡宾是强亲电试剂**，但选择性比碳阳离子低——可与任何 HOMO 反应（孤对电子、C=C、甚至 C–H）。
5. **单线态卡宾**：协同机理，立体专一性。
6. **三线态卡宾**：分步自由基机理，非立体专一性。
7. **Simmons-Smith 反应**：锌类卡宾，不释放游离卡宾，立体专一 + 立体选择。

### 3.3 合成应用
8. **环丙烷化**：卡宾对烯烃加成是制取环丙烷最重要的方法。
9. **C–H 插入**：在未官能化中心建键，分子内版本更可靠。
10. **Arndt–Eistert 同系化**：羧酸扩链一个碳的标准方法。
11. **烯烃复分解**：构建 C=C 键的革命性方法，关环、交叉、烯-炔三种模式。

## 八、摘录的原始段落（附页码/位置标记）

### 8.1 卡宾定义与结构
> "卡宾是包含只具有六个价电子的碳原子的中性物种。"（line ~1968）

> "三线态卡宾具有两个未成对电子，分别填入一个 sp 和一个 p 轨道中；而单线态卡宾则在非键 $sp^2$ 轨道中填入一对电子，并空置 p 轨道。"（line ~2376）

> "大多数卡宾在三线态下稳定，这是因为将 p 轨道上的电子带下到 $sp^2$ 轨道所获得的能量不足以弥补同一轨道中两个电子间的排斥。"（line ~2415）

### 8.2 卡宾的形成
> "卡宾太不稳定，因此必须要使它们在它们要反应的化合物的存在下形成。"（line ~2226）

> "由所有电子均配对的分子得到时，不论三线态在能量上是否更稳定，它也一定以单线态形成。"（line ~2442）

### 8.3 环丙烷化
> "单线态卡宾的协同加成是一个周环反应，通过 Chapter 34 您应当能识别它是一个 $[1 + 2]$ 环加成。"（line ~2621）

> "在卡宾加成中缺乏立体专一性，必定表明所涉及的是三线态卡宾，但具有立体专一性却不一定表明卡宾是单线态。"（line ~2619）

### 8.4 Simmons-Smith 反应
> "Simmons-Smith 反应的机理表现出是一个由金属到烯烃的卡宾转移，此过程不释放任何游离卡宾。"（line ~2776）

> "锌原子在过渡态中与羟基的配位不仅能解释立体选择性，而且能解释速率的增加。"（line ~2796）

### 8.5 复分解
> "大多数步骤是可逆的，而整个反应是不可逆的一步所驱动的——这步是乙烯以气体失去。"（line ~3185）

## 九、竞赛题型线索

### 9.1 机理书写题
- 卡宾对烯烃的协同加成（单线态）vs 分步加成（三线态）。
- Simmons-Smith 反应的金属转移机理。
- Wolff 重排的完整机理（重氮酮 → 卡宾 → 烯酮）。
- 烯烃复分解的 $[2+2]$ 环加成/逆环加成机理。

### 9.2 产物预测题
- 给定烯烃和卡宾前体，预测环丙烷产物的立体化学（注意单线态/三线态）。
- 给定羧酸，用 Arndt-Eistert 反应预测同系化产物。
- 给定双烯，预测关环复分解产物。

### 9.3 立体化学分析题
- 判断卡宾加成是否具有立体专一性（推断单线态/三线态）。
- Simmons-Smith 反应中羟基导向的立体选择性分析。
- C–H 插入反应中立体中心的构型保持。

### 9.4 合成设计题
- 利用卡宾环丙烷化构建环丙烷结构（天然产物如除虫菊酯）。
- 利用 Arndt-Eistert 反应进行羧酸同系化。
- 利用关环复分解构建大环或中环。

## 十、存疑或待验证之处

1. **三线态卡宾 C–H 插入的实验证据**：本章提到"所观察到的三线态卡宾对 C–H 键的插入非常少"，两步机理的立体化学结果"从未被核实"——竞赛中如涉及，应以单线态协同机理为主。

2. **"类宾"概念的边界**：本章将 Simmons-Smith 试剂、过酸环氧化、锍叶立德统一为"类宾"——这一概念在竞赛中的考查深度如何？

3. **复分解催化剂的详细结构**：Grubbs II 和 Hoveyda-Grubbs 的具体配体结构是否要求记忆？竞赛中通常只需知道 Ru 卡宾催化剂和反应类型。

4. **N-杂环卡宾的催化机理**：作为 Grubbs II 配体时，NHC 通过什么方式增加催化剂活性？本章未深入讨论。
