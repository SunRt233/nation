# 提炼：Clayden 第37章——自由基反应（Radical Reactions）

---

## 一、核心概念与定义

### 1.1 自由基（Radical）
- **定义**：含有未成对电子的原子或分子物种。键发生**均裂（homolysis）**时产生：每个原子带走一个成键电子。
- **与异裂对比**：异裂（heterolysis）产生离子；均裂产生自由基。
- **表示法**：用点（•）标记未成对电子，如 Cl•、Br•、R•。

### 1.2 键的强度与均裂
- 键的强度（键解离能, BDE）对自由基反应至关重要——这是自由基反应选择性的核心决定因素。
- 弱键易均裂：O–O（~213 kJ/mol）、Br–Br（192）、I–I（151）、N–O（~150）等。
- 对比：H–Cl 的异裂 ΔG = +1347 kJ/mol（极不利），而均裂 ΔG = +431 kJ/mol（高温可行）。

### 1.3 自由基的稳定性
- **共轭稳定**：自由基可被邻近的 π 体系离域稳定（如烯丙基自由基、苄基自由基）。
- **吸电子基稳定**：通过 SOMO 能量降低稳定自由基（如 •CCl₃）。
- **给电子基稳定**：通过填充 SOMO 稳定（如烷氧基自由基邻近的碳自由基）。
- **空间位阻**：大位阻降低自由基活性（如稳定的持久自由基 TEMPO）。

> 任何能稳定阴离子或阳离子的基团，也能稳定自由基。

## 二、自由基的形成方法

| 方法 | 条件/试剂 | 实例 |
|:---|:---|:---|
| **热均裂** | 加热弱键 | 过氧化物（RO–OR）、AIBN |
| **光均裂** | hν 断裂弱键 | Br₂ → 2 Br•；Cl₂ → 2 Cl• |
| **攫取（Abstraction）** | 自由基从分子上攫取原子 | Cl• + R–H → HCl + R• |
| **加成** | 自由基加到 π 键上 | Br• + C=C → C–C–Br |
| **消除** | 自由基断裂 β-键 | RO• → R'• + 羰基化合物 |

## 三、重要反应与机理

### 3.1 自由基-自由基反应（非链式）

#### 3.1.1 二聚（偶联）
- 两个自由基结合形成新的 σ 键。
- **频哪醇偶联（Pinacol Coupling）**：醛/酮在还原条件下（如 Mg/Hg）发生自由基二聚，形成邻二醇。

#### 3.1.2 McMurry 反应
- Ti(0) 促进的羰基化合物还原偶联，产物为烯烃（频哪醇中间体脱水）。
- 用于合成对称或不对称烯烃，特别是大环烯烃。

#### 3.1.3 酮醇反应（Acyloin Condensation）
- 酯在 Na 作用下发生类频哪醇偶联，产物为 α-羟基酮（酮醇）。

> 以上三种反应都是**例外**——大多数自由基反应是链式反应。

### 3.2 自由基链式反应

链式反应三步曲：

| 步骤 | 描述 | 特征 |
|:---|:---|:---|
| **引发（Initiation）** | 产生第一个自由基 | 热/光均裂、AIBN 分解 |
| **增长（Propagation）** | 自由基反应后再生新自由基 | 链长可达 10⁶ 步 |
| **终止（Termination）** | 两个自由基结合或歧化 | 消耗自由基，结束链 |

#### 3.2.1 HBr 对烯烃的自由基加成（反马氏加成）
- **过氧化物效应**：ROOR → 2 RO• → 攫取 HBr 中的 H → Br• → 加成到烯烃较少取代端 → 形成较稳定的碳自由基 → 从 HBr 攫取 H → 产物 + Br•（链增长）。
- **区域选择性**：Br• 加成到烯烃的较少取代碳（位阻较小），形成更稳定的碳自由基中间体。
- **立体化学**：若烯烃有立体中心，加成可能保持或翻转，取决于中间体构型。

#### 3.2.2 烷烃的氯代
- Cl₂ + hν → 2 Cl•
- Cl• 从烷烃上攫取 H，形成烷基自由基 + HCl
- 烷基自由基 + Cl₂ → 氯代烷 + Cl•（链增长）
- **选择性差**：伯/仲/叔氢的相对活性约为 1 : 3.8 : 5（氯代）

![[mineru/中文版clayden-chinese37-401000-1132_images/4fcb1034562c529c813d6253b525c64ef658993d74d8f32e6399aa27e8af5fcd.jpg]] — 环己烷氯代（line ~996）

#### 3.2.3 烷烃的溴代（选择性更高）
- Br• 活性低于 Cl•，因此更"挑剔"，选择性更好。
- 伯/仲/叔氢的相对活性约为 **1 : 80 : 1600**（溴代）
- 键强度解释：形成较弱的 H–Br 键（366 kJ/mol）vs H–Cl（431），使过渡态更接近产物，自由基稳定性差异被放大（Hammond 假设）。

#### 3.2.4 烯丙型溴代（NBS）
- **N-溴代琥珀酰亚胺（NBS）** 提供低浓度的 Br₂，维持低浓度的 Br•，避免加成竞争。
- 选择性攫取烯丙位氢，形成稳定的烯丙基自由基 → 与 Br₂ 反应得烯丙基溴。
- 烯丙基自由基通过共振稳定，因此反应专一于烯丙位。

#### 3.2.5 选择性逆转：用 H 取代 Br（锡氢化物还原）
- **试剂**：Bu₃SnH + AIBN（引发剂）
- **机理**：AIBN → R• → 攫取 Sn–H 中的 H → Bu₃Sn• → 攫取烷基溴中的 Br → R• → 从 Bu₃SnH 攫取 H → 还原产物 + Bu₃Sn•
- **选择性**：C–Br 键最弱（280 kJ/mol），优先被还原；C–Cl 也可，但较慢；C–F 不反应。
- AIBN 只需催化量（0.02–0.05 eq），Bu₃SnH 稍过量（1.2 eq）。

### 3.3 用自由基形成 C–C 键

#### 3.3.1 CCl₃• 对烯烃的加成
- BrCCl₃ + hν → Br• + •CCl₃
- •CCl₃ 加成到烯烃较少取代端（位阻+稳定性）→ 新自由基 → 攫取 BrCCl₃ 中的 Br → 产物 + •CCl₃（链增长）
- 可发生分子内环化：若烯烃链上另有双键，自由基可加成成环。

![[mineru/中文版clayden-chinese37-401000-1132_images/abf49bf770bfc80be4c31e3d64d40bf1dc0adca1935e5a1895c4e191fc9edc2e.jpg]] — 自由基环化反应（line ~1299）

#### 3.3.2 硼烷 + 氧产生烷基自由基
- R₃B + O₂ → R•（烷基自由基）+ 硼酸酯
- R• 可发生共轭加成到 α,β-不饱和羰基化合物上。
- **优势**：避免使用有毒的锡试剂。

### 3.4 前线轨道效应：亲电自由基 vs 亲核自由基

- **SOMO（单占分子轨道）能量**决定自由基的亲电/亲核性质：
  - **高能 SOMO** → 亲核自由基（如烯醇自由基、烷氧基自由基）
  - **低能 SOMO** → 亲电自由基（如 •CCl₃、酰基自由基）
- **前线轨道相互作用**：
  - 亲核自由基倾向于加成到亲电烯烃（如丙烯酸酯，LUMO 能量低）
  - 亲电自由基倾向于加成到亲核烯烃（如醋酸乙烯酯，HOMO 能量高）

#### 3.4.1 交替共聚
- 醋酸乙烯酯（亲核单体）+ 丙烯酸甲酯（亲电单体）在自由基引发下形成严格交替共聚物。
- 机理：亲核自由基 → 加成亲电烯烃 → 亲电自由基 → 加成亲核烯烃 → 循环。

![[mineru/中文版clayden-chinese37-401000-1132_images/eb070f2d2c25159dd4d9330ff2779d98f33c45f2a6cb15fa97a2035dbec931ac.jpg]] — 交替共聚反应（line ~1510）

### 3.5 自由基反应性模式与极性试剂的对比

| 底物 | 极性亲核试剂 | 自由基 |
|:---|:---|:---|
| α,β-不饱和羰基 | 1,4-加成（Michael） | 1,4-加成（共轭加成） |
| 羰基 C=O | 亲核加成 | 通常不反应 |
| O–H 键 | 被强碱去质子 | 被自由基攫取 H |
| 卤代烷 R–X | SN2（碳上取代） | 攫取 X（不是碳） |

> 自由基是**软物种**：反应由键强度和前线轨道驱动，不受电荷极化主导。

## 四、关键结论

1. **键强度是自由基反应选择性的核心**：形成和断裂的键的相对强度决定反应方向和区域选择性。
2. **链式反应三步曲**：引发 → 增长（循环）→ 终止。增长步骤数可达 10⁶ 步。
3. **溴代选择性远高于氯代**：Br• 更"软"、更挑剔，过渡态更接近产物，自由基稳定性差异被放大。
4. **锡氢化物还原是温和的去卤方法**：Bu₃SnH/AIBN 体系可选择性还原 C–Br/C–I，C–Cl 较慢，C–F 不反应。
5. **前线轨道理论解释亲电/亲核自由基**：SOMO 能量高低决定自由基与不同烯烃的匹配性。
6. **分子内自由基反应比分子间更有效**：成环反应在合成复杂环系（如天然产物）中极具价值。
7. **硼烷/氧体系提供无金属替代**：避免锡的毒性，适用于共轭加成。

## 九、存疑或待验证之处

1. **溴代选择性的定量数据**：本章节中伯/仲/叔氢溴代相对活性约为 1:80:1600，但不同教材给出的数值略有差异，教学中应注明数据来源。
2. **NBS 机理的精确描述**：NBS 维持低 Br₂ 浓度的机理在竞赛中常以简化版出现，是否与 Clayden 的描述完全一致需对照。
3. **McMurry 反应的立体化学**：Ti(0) 还原偶联中产物的 E/Z 选择性控制因素，本章节未深入讨论。
4. **分子内环化的区域选择性**：5-exo-trig vs 6-endo-trig 的竞争，本章涉及但未用 Baldwin 规则系统总结（Baldwin 规则见 [[Baldwin规则]]）。
