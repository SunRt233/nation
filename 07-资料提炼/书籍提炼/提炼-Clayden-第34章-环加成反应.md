# Clayden 第34章：周环反应1——环加成 资料提炼

---

## 一、核心概念与定义

### 1.1 周环反应（Pericyclic Reaction）
- **定义**：电子绕一个环移动的反应，任何中间体上都没有正负电荷，**没有中间体**，一步协同完成。
- **三类**：环加成（本章）、电环化（Ch35）、σ重排（Ch35）。
- **核心特征**：协同机理、环状过渡态、轨道对称性控制。

### 1.2 环加成反应（Cycloaddition）
- **定义**：两个共轭π体系之间，形成用于连接两个试剂各端的两根新σ键的一步成环反应。所有箭头起始于π键并绕一个环移动。
- **通式**：[m+n] 环加成，数字代表参与反应的每个组分的**原子数目**（有时也用电子对数标号）。

### 1.3 前线轨道理论（FMO）核心要点
- Diels-Alder 反应中：二烯的 HOMO ($\psi_2$) 与亲双烯体的 LUMO ($\pi^*$) 对称性匹配 → 热允许。
- 二烯的 LUMO ($\psi_3$) 与亲双烯体的 HOMO ($\pi$) 也匹配（次要贡献）。
- EDG 在二烯 → 升高 HOMO 能量；EWG 在亲双烯体 → 降低 LUMO 能量 → 反应更快。

### 1.4 芳香过渡态
- Diels-Alder 过渡态包含六个离域的π电子，符合芳香性特征，具有一些苯的特殊稳定性。
- 可将其看作一个留有全部π键、但丢失了两根σ键的苯环。

## 二、重要反应与机理

### 2.1 Diels-Alder 反应 [4+2] 环加成

#### 基本特征
- **反应物**：共轭二烯（diene）+ 亲双烯体（dienophile）
- **产物**：环己烯衍生物
- **条件**：加热（通常 100-150°C），无需溶剂或催化剂
- **限制**：二烯必须能采取 **s-cis 构象**；亲双烯体需有共轭吸电子基团活化

#### 立体化学
- **立体专一性（syn 加成）**：亲双烯体的 cis/trans 构型在产物中保留。
- **二烯的立体化学**：
  - 顺式,顺式-二烯 → 产物中两组取代基在环同侧
  - 反式,反式-二烯 → 产物中两组取代基也在环同侧（看似顺式）
  - 顺式,反式-二烯 → 产物中取代基在环的不同侧

#### endo 规则（内型规则）
- **定义**：亲双烯体上的吸电子取代基倾向于在产物的 **endo 位**（桥位，靠近新形成的双键）。
- **原因**：endo 过渡态中，亲双烯体的π*轨道（LUMO）与二烯的后部存在额外的**次级轨道相互作用（成键相互作用）**，降低过渡态能量。
- **动力学 vs 热力学**：endo 是**动力学产物**（形成更快），exo 是**热力学产物**（更稳定）。
- **例外**：分子内 D-A 反应中，endo 规则可能被破坏——通常倾向于外型产物，由空阻因素支配。

#### 区域选择性
- **"邻位/对位"规则**：
  - EDG 在二烯 1-位 → 与亲双烯体 EWG **邻位**
  - EDG 在二烯 2-位 → 与亲双烯体 EWG **对位（1,4-位）**
- **FMO 系数解释**：不对称二烯的 HOMO 在一端系数较大；不对称亲双烯体的 LUMO 在β-碳（连 EWG）系数较大。最大系数相连 → 主要产物。
- **记忆法**：Diels-Alder 反应是邻对位定位的，具有芳香过渡态的环加成反应。

#### 水溶剂效应
- 水对 D-A 反应有**很大的加速作用**（相对速率可达 700 倍）。
- 水还能提高 **endo 选择性**（如从 80:20 提高到 96:4）。
- 原因：不溶于水的试剂被水的作用聚集在油滴中，被迫靠得很近。这类反应有时被称作"在水上 (on water)"的反应。

#### Lewis 酸催化
- Lewis 酸（如 SnCl₄）与亲双烯体上的 EWG 结合 → 进一步降低 LUMO 能量 → 催化反应。
- 还能**增加 LUMO 在不同碳上轨道系数的差异** → 提高区域选择性。
- 例：某反应在 120°C 下区域选择性 71:29；在 SnCl₄ 催化下，低于 25°C 即可反应，选择性提高到 93:7。

#### 分子内 Diels-Alder 反应（IMDA）
- 二烯和亲双烯体在同一分子内。
- **立体选择性**：通常倾向于外型产物（空阻因素支配），但若有共轭羰基，可通过折叠使羰基与二烯后部重叠 → 得到较不稳定的顺式环交点（内型）。
- **区域选择性**：短链连接时可能被迫采取"错误"取向。
- 产物中正在形成的环必须始终采取**船式构象**。

#### 特殊变体
- **[4+3] 环加成**：氧烯丙基阳离子（oxyallyl cation）与双烯的环加成。三个原子、两个电子的烯丙型阳离子 + 双烯 → 七元环。电子总数为六，与 D-A 相同。被允许（一个 $(4q+2)_s$ 组分）。
- **捕获活泼中间体**：苯炔可被 D-A 捕获（如蒽 + 苯炔 → 笼状结构）。

### 2.2 [2+2] 环加成

#### 一般规则
- **热同面-同面禁阻**（4e = 4n，轨道对称性不匹配）。
- **光同面-同面允许**：激发态 HOMO（原 LUMO）相位改变。

#### 光化学 [2+2] 环加成
- 烯烃吸收 UV 光 → 激发态 → 与另一烯烃基态组合 → 对称性匹配 → 成键。
- **产物稳定性**：即使得到有张力的四元环，产物也不会自发回到起始原料，因为产物失去了π键，没有低能空轨道可以吸收光。
- **区域选择性**：光化学 [2+2] 中，真正重要的是 **HOMO/HOMO 和 LUMO/LUMO 相互作用**。烯烃 LUMO 轨道系数的大小正好与其 HOMO 的相反 → 产物取向与热反应中 HOMO/LUMO 预料的**相反**。
- 反应在每个组分内部都是立体专一性的，但没有内型规则。

#### 热 [2+2] 环加成——烯酮例外
- **烯酮（ketene, R₂C=C=O）**和**异氰酸酯（isocyanate, R-N=C=O）**的 [2+2] 是**热允许**的。
- **原因**：正交π体系 → 可通过 **2s+2a 途径**（一个组分同面、一个异面）。烯酮中心的 sp 碳具有与第一个烯烃成直角的额外π键（C=O），允许 HOMO 的两个轨道与 LUMO 及额外的 p 轨道同时成键。
- **产物**：环丁酮（烯酮 + 烯烃）、β-内酰胺（烯酮 + 亚胺 / 异氰酸酯 + 烯烃）。

#### Staudinger 合成
- 烯酮 + 亚胺 → β-内酰胺。
- 二氯烯酮可由二氯乙酰氯在碱作用下经 ElcB 消除制得。
- 环戊二烯上最亲核的原子添加到烯酮最亲电的原子上，[2+2] 环加成比 D-A 更快。

#### 掩蔽烯酮（Masked Ketene）
- 硝基烯烃和氰乙醇酯类化合物可作为烯酮等价物。
- 先进行 [2+2] 环加成，再通过 Nef 反应（NaOH）或还原将硝基转化为酮。

#### 异氰酸酯 [2+2]
- 氯代硫酰异氰酸酯（chlorosulfonyl isocyanate）可与简单烯烃反应 → β-内酰胺。
- 硫酰基可在温和条件下水解去除。
- 富电子烯烃（如烯醇醚）反应可能变为**分步机理**（立体选择性而非立体专一性）。

### 2.3 1,3-偶极环加成 [3+2]

#### 基本概念
- **1,3-偶极体**：三原子、四电子的"双烯"等价物。具有亲核端和亲电端。
- **亲偶极体（dipolarophile）**：通常为烯烃（简单烯烃或缺电子烯烃均可）。
- 1,3-偶极体既可用 HOMO 又可用 LUMO，取决于亲偶极体是缺电子还是富电子。

#### 硝酮（Nitrone）
- 亚胺的 N-氧化物。四个电子来源于：N=C 双键的两个π电子 + 氧原子上孤对电子的两个电子。
- 产物中的 N-O 键可通过还原（LiAlH₄、Zn/HOAc、H₂/Ni）选择性断裂 → 得到 1,3-氨基醇。
- 没有内型规则（无共轭基团跨空间相互作用）。

#### 氧化腈（Nitrile Oxide）
- 具有一根三键的直线型 1,3-偶极体。
- 可由肟氯化后碱处理（γ-消除）或硝基烷烃脱水（PhNCO）制得。
- 与炔烃环加成 → **异噁唑**（稳定芳杂环）。

#### 叠氮-炔环加成
- 叠氮化物 + 炔烃 → 三唑（Click 化学）。

#### 生物素合成中的分子内 [3+2]
- 硝基烷烃经 PhNCO 脱水 → 氧化腈 → 分子内 [3+2] 环加成 → 笼状结构 → LiAlH₄ 还原 → 氨基醇。

### 2.4 Alder 烯反应（Ene Reaction）

#### 基本特征
- 将 D-A 反应中双烯的一根双键替换为 C-H 键。
- **产物**：只形成一根新的 C-C σ 键 + 双键迁移 + H 转移。
- **六电子过程**：三个组分——一个 π2（烯的 C=C）、一个 σ2（C-H 键）、另一个 π2（enophile 的 C=C）。
- 三个 $(4q+2)_s$ 组分 → 总和为三（奇数）→ 反应被允许。

#### 羰基烯反应（Carbonyl Ene Reaction）
- 亲烯体为醛/酮的 C=O。
- Lewis 酸催化可降低羰基 LUMO 能量 → 加速反应。
- **薄荷醇工业生产**：分子内羰基烯反应是核心步骤之一。β-蒎烯衍生物在 ZnBr₂ 催化下发生分子内羰基烯反应 → 异蒲勒醇（isopulegol）→ 氢化得薄荷醇。
- 立体化学来源于过渡态构象的全椅式排列，甲基采取平伏位点。

#### 一般烯反应
- 简单烯烃作为 ene 组分效果不佳（除 β-蒎烯等张力烯烃外）。
- 通常需要 Lewis 酸催化或分子内反应（或二者皆有）。

### 2.5 OsO₄ 双羟基化——[3+2] 环加成视角
- OsO₄ 与烯烃的反应是 **[3+2] 环加成**（1,3-偶极环加成）。
- 可将 OsO₄ 看作偶极体（锇有大量轨道容纳四根双键）。
- 协同加成 → 两个 OH 的相对立体化学必定保持 **syn**。
- Os 由 Os(VIII) 被还原到 Os(VI) —— 这是对 C=C 双键非常专一的氧化反应。
- 可通过共氧化剂（NMO 或 Fe(III)）使 Os 只做催化剂。
- OsO₄ 与缺电子烯烃和富电子烯烃反应几乎一样好（不同于 m-CPBA 或 Br₂ 的亲电加成）。

### 2.6 臭氧解——两次 [3+2] 环加成 + 一次逆 [3+2]

#### 机理三步曲
1. **第一次 [3+2]**：臭氧（1,3-偶极体）+ 烯烃 → 初级臭氧化物（molozonide，极不稳定）。
2. **逆 [3+2]**：初级臭氧化物分解 → 醛 + 羰基氧化物（carbonyl oxide，新的 1,3-偶极体）。
3. **第二次 [3+2]**：羰基氧化物 + 醛 → 二级臭氧化物（ozonide，第一个稳定产物）。

#### 臭氧化物分解
- 还原条件（Me₂S 或 Ph₃P）→ 两分子醛/酮。
- 氧化条件（H₂O₂）→ 羧酸。
- 强还原条件（NaBH₄）→ 醇。

## 三、关键结论与规则

### 3.1 Woodward-Hoffmann 规则（正式表述）

**版本一**：
> 在热周环反应中，$(4q+2)_s$ 和 $(4r)_a$ 组分的数目之和必须是**奇数**。

**版本二**：
> 在一个允许的热周环反应中：带有 2, 6, 或 10 个电子的同面组分的数目 + 带有 0, 4, 或 8 个电子的异面组分的数目 = 一个奇数。

- **组分**：作为一个单元参与周环反应的一根键或一条轨道。双键 = π2 组分，双烯 = π4 组分。
- **s** = 同面（suprafacial），两端在同侧形成新键。
- **a** = 异面（antarafacial），两端在对面形成新键。
- 不计入的组分：$(4q+2)_a$ 和 $(4r)_s$ 可以有任意多。

### 3.2 环加成选择规则总结

| 电子数 | 热反应（同面-同面）| 光反应（同面-同面）|
|:---:|:---:|:---:|
| **4n**（如 [2+2]）| **禁阻** | **允许** |
| **4n+2**（如 [4+2]）| **允许** | **禁阻** |

### 3.3 芳香过渡态与允许性
- 电子总数为 $(4n+2)$ 的环加成反应，如果是同面的，则往往被允许。
- 它们被认为具有"芳香过渡态"——与芳香性对 $(4n+2)$ 电子的需求明显有联系。

### 3.4 Diels-Alder 关键结论
1. 协同 [4+2] 环加成，一步完成，无中间体。
2. endo 规则——EWG 倾向于 endo 位（动力学控制，次级轨道相互作用）。
3. 区域选择性——"邻对位规则"，由 FMO 系数决定。
4. 立体专一——syn 加成，cis/trans 构型保留。
5. EDG 在二烯，EWG 在亲双烯体——反应最快。
6. 水溶剂可大幅加速反应并提高 endo 选择性。
7. Lewis 酸催化可降低温度、提高区域选择性。

### 3.5 [2+2] 关键结论
1. 普通烯烃热 [2+2] 同面-同面禁阻。
2. 光化学 [2+2] 允许（激发态 HOMO 相位改变）。
3. 烯酮和异氰酸酯是热允许例外（2s+2a 途径，正交π体系）。
4. 光化学 [2+2] 区域选择性由 LUMO-LUMO 相互作用决定，与热反应相反。

### 3.6 1,3-偶极环加成关键结论
1. [3+2] 环加成，六电子，热允许。
2. 1,3-偶极体既亲电又亲核，可用 HOMO 或 LUMO。
3. 产物中的 N-O 键可还原断裂 → 氨基醇（合成价值）。
4. 没有内型规则。

## 八、摘录的原始段落（附页码/位置标记）

### 8.1 关于协同机理的本质（lines 37-97）
> "电子绕一个环移动，任何中间体上都没有正负电荷——事实上，根本就没有中间体。这类反应被称为周环反应。"
> "每个箭头都直接指向下一个箭头，最后一个箭头又与第一个相连。方才的图中，电子顺时针旋转，然而，如果要画电子逆时针旋转，也不会有任何区别。"
> "Diels–Alder 反应进行得如此顺利的原因在于，其过渡态包含六个离域的π电子，符合芳香性的特征，具有一些苯的特殊稳定性。"

### 8.2 关于 s-cis 构象要求（lines 143-177）
> "丁二烯由于空间因素，一般倾向于处在两根双键彼此尽可能地远离的 s-反式构象中。绕中间σ键旋转的能垒很小（室温下大约 30 kJ mol⁻¹）。"
> "始终处于 s-反式构象的环状双烯，无法采取 s-顺式构象，因而也根本无法进行 Diels–Alder 反应。"

### 8.3 关于 endo 规则（lines 413-450）
> "内型产物没有外型产物稳定，但不可逆的 Diels–Alder 反应却倾向于形成它——它一定是反应的动力学产物。它形成得更快，这是由于亲双烯体中缺电子的羰基与双烯体后部正在形成的π键之间存在成键相互作用。"

### 8.4 关于水溶剂效应（lines 646-662）
> "水，大多数有机反应最没希望的溶剂，对于 Diels–Alder 反应有很大的加速作用。即使只在有机溶剂中添加一些水，都能加速反应。"
> "不溶于水的试剂，会被水的作用聚集在油滴中，被迫靠得很近。水，确切地说，并不是一种溶剂——它几乎是一种反溶剂！"

### 8.5 关于分子内 D-A 的例外（lines 664-696）
> "当双烯体和亲双烯体已经是同一分子的两部分时，通过跨空间的成键相互作用将它们持在一起就不那么重要了，因而通常倾向于外型产物。"
> "分子内的 Diels–Alder 反应可能给出内型产物，也可能相反！请准备好，可能得到外型、内型产物或它们的混合物。"

### 8.6 关于区域选择性的 FMO 解释（lines 698-787）
> "在轨道术语中，这意味着双烯的 HOMO 发生变形，与质子反应的一端有更大的系数。"
> "当不对称双烯与不对称亲双烯在一个 Diels–Alder 反应中结合时，反应本身会变得不对称。它仍然是协同的，但在过渡态中，每个组分中最大轨道系数的两端会更率先成键。"

### 8.7 关于 Lewis 酸催化（lines 847-860）
> "Lewis 酸通常能通过进一步降低亲双烯体 LUMO 的能量，来催化反应的发生。它还有另一个优势：它会增加 LUMO 在不同碳上轨道系数的差异，可能因此增加区域选择性。"

### 8.8 Woodward-Hoffmann 规则正式表述（lines 875-920）

## 九、竞赛题型线索

### 9.1 产物预测型
- 给定二烯和亲双烯体，预测 D-A 产物（考虑 endo/exo、区域选择性、立体化学）。
- 给定烯酮 + 烯烃/亚胺，预测 [2+2] 产物。
- 给定 1,3-偶极体 + 亲偶极体，预测 [3+2] 产物。

### 9.2 机理判断型
- 判断反应是协同环加成还是分步离子型/自由基机理。
- 区分 D-A 和 Ene 反应（产物是否成环、新 σ 键数目）。
- 判断 [2+2] 是热允许还是光允许。

### 9.3 立体化学型
- 根据二烯的 cis/trans 构型判断产物立体化学。
- endo/exo 产物的判断（尤其注意分子内 D-A 的例外）。
- 光化学 [2+2] 的立体专一性。

### 9.4 区域选择性型
- 应用"邻对位规则"判断 D-A 主要产物。
- 光化学 [2+2] 的区域选择性（与热反应相反）。
- FMO 系数分析不对称反应物的取向。

### 9.5 合成设计型
- 逆 D-A 切断：识别六元环中的 D-A 产物特征（六元环 + 环内双键 + 环外共轭基团）。
- 逆 [2+2] 切断：环丁烷/β-内酰胺切成两个烯烃组分。
- 利用 1,3-偶极环加成 + N-O 键还原断裂策略构建氨基醇。
- 利用掩蔽烯酮策略构建环丁酮。

### 9.6 条件选择型
- 水溶剂 vs 有机溶剂对 D-A 速率和选择性的影响。
- Lewis 酸催化对 D-A 区域选择性的提升。
- 光化学条件 vs 热条件对 [2+2] 允许性的决定。

## 十、存疑或待验证之处

1. **水加速 D-A 的定量数据**：文中给出的相对速率 700 倍和 endo:exo 96:4 的数据来自特定例子，是否可推广到一般 D-A 反应？
2. **分子内 D-A 的 endo 例外**：文中提到"可能得到外型、内型产物或它们的混合物"，但缺乏系统性的预测规则。
3. **光化学 [2+2] 的 LUMO-LUMO 解释**：文中说"快速得出结论的最简单方法，是画出与 HOMO/LUMO 预料的相反的结果"——这是否总是成立？需要更多例子验证。
4. **富电子烯烃与异氰酸酯的分步机理**：文中提到烯醇醚与氯代硫酰异氰酸酯的反应是"分步"的，但如何判断其他体系的协同/分步边界？
