## 二、知识精讲要点（备课可抽料）

### 2.1 d 区元素概述（各讲通用）【→学生讲义：钛钒铬锰 · d区概述】

过渡元素共有 **4 个特征**：
1. **多种氧化数** — 最高氧化数 = ns + (n-1)d 电子总和
2. **金属活泼性** — 同族自上而下活泼性降低（与主族相反）
3. **形成配合物倾向强** — 有空 d 轨道接受孤对电子
4. **水合离子/酸根有色** — 有未成对 d 电子的离子显色

**课堂信号点**：Sc³⁺(d⁰)、Zn²⁺(d¹⁰) 无色；Cu²⁺(d⁹) 蓝色、Cr³⁺(d³) 紫色/绿色

### 2.2 钛的特殊性（第六讲可用）【→学生讲义：钛钒铬锰 · Ti】

- **"第三金属""二十一世纪金属"** — 强度接近钢但密度仅 4.54 g/cm³，耐高温(800K性能不变)
- **钝化现象**：φ°(Ti²⁺/Ti) = -1.63 V（热力学活泼），但表面致密氧化层 → 常温极稳定，不与王水、强碱反应
- **课堂对比**：Ti 和 Si 的相似性 → 都因空 d 轨道易形成配离子（TiF₆²⁻）
- **TiO²⁺ + H₂O₂ → [TiO(H₂O₂)]²⁺ 橘黄色** → 钛的定性检验/比色分析

### 2.3 钒的多价态与颜色（第六讲可用）【→学生讲义：钛钒铬锰 · V】

钒的五种氧化态在水溶液中有特征颜色，是课堂的绝佳记忆点：

| 氧化态 | 存在形式 | 颜色 |
|:---|:---|:---:|
| V(II) | V²⁺ | 紫色 |
| V(III) | V³⁺ | 绿色 |
| V(IV) | VO²⁺ | 蓝色 |
| V(V) | VO₂⁺（酸性）/VO₄³⁻（碱性） | 黄色/无色 |

**钒酸根的缩合**（与磷酸根类比）：VO₄³⁻ → V₂O₆(OH)³⁻ → V₃O₉³⁻ → V₁₀O₂₈⁶⁻，随 pH 降低缩合度增大，颜色由无色→淡黄→深红

### 2.4 Cr(Ⅲ) 的性质——与 Al(Ⅲ) 的异同（第七讲核心）【→学生讲义：钛钒铬锰 · Cr】

**相似性**：
- 氧化物两性：Cr₂O₃ + H⁺ → Cr³⁺；Cr₂O₃ + OH⁻ → CrO₂⁻
- 氢氧化物两性：Cr(OH)₃ → [Cr(OH)₄]⁻
- 易形成矾：K₂SO₄·Cr₂(SO₄)₃·24H₂O
- 双水解：Cr³⁺ + S²⁻/CO₃²⁻ → Cr(OH)₃↓

**不同点**：
- 颜色：Cr³⁺ 有色（配体不同→颜色不同），Al³⁺ 无色
- 还原性：Cr(III) 在碱中可被 H₂O₂ 氧化为 Cr(VI)；Al(III) 不表现还原性
- 配合物：Cr³⁺(d³) 易形成内轨型配合物（动力学惰性，配体交换慢），Al³⁺ 不能

**课堂核心**：Cr(OH)₃ + OH⁻ → [Cr(OH)₄]⁻（绿色）再 + H₂O₂ → CrO₄²⁻（黄色）—— 三步颜色变化极其直观

### 2.5 Cr(VI) 的缩合平衡与沉淀（第七讲核心）【→学生讲义：钛钒铬锰 · Cr】

**CrO₄²⁻ / Cr₂O₇²⁻ 平衡**：
- 2CrO₄²⁻(黄) + 2H⁺ ⇌ Cr₂O₇²⁻(橙) + H₂O，K=4.2×10¹⁴
- pH > 6：CrO₄²⁻ 为主；pH < 1：Cr₂O₇²⁻ 为主
- 加入 Ag⁺/Pb²⁺/Ba²⁺ 形成难溶铬酸盐 → 平衡移动：

| 离子 | 沉淀 | 颜色 | 用途 |
|:---|:---|:---:|:---|
| Ag⁺ | Ag₂CrO₄ | 砖红色 | 沉淀滴定指示剂 |
| Pb²⁺ | PbCrO₄ | 橙黄色 | 黄色颜料（铬黄）|
| Ba²⁺ | BaCrO₄ | 黄色 | 检验 CrO₄²⁻ |

**卤素类比**：除碱金属/铵/镁的铬酸盐易溶外，其余均难溶（← 与卤化物的溶解规律对比）

### 2.6 Mn 的五种氧化态（第八讲核心）【→学生讲义：钛钒铬锰 · Mn】

Mn 的氧化态及特征颜色/反应：

| 氧化态 | 代表物 | 颜色 | 性质 | 稳定性 |
|:---:|:---|:---:|:---|:---:|
| +2 | Mn²⁺ | 淡粉红 | d⁵半充满稳定 | 酸性中很稳定 |
| +3 | Mn³⁺ | 酒红 | 歧化：2Mn³⁺ → MnO₂ + Mn²⁺ | 仅强酸中稳定 |
| +4 | MnO₂ | 黑色 | 强氧化性（酸性） | 中性稳定 |
| +6 | MnO₄²⁻ | 绿色 | 歧化：3MnO₄²⁻ + 2H₂O → 2MnO₄⁻ + MnO₂ + 4OH⁻ | 仅碱性稳定 |
| +7 | MnO₄⁻ | 紫红 | 最强氧化性 | 中性/碱性慢分解 |

**KMnO₄ 还原产物与 pH 的关系**（竞赛必考点）：

| 介质条件 | 还原产物 | 颜色变化 | φ°/V |
|:---|:---|:---:|:---:|
| 强酸性 (pH<1) | Mn²⁺ | 紫红→无色 | 1.51 |
| 中性/微碱性 | MnO₂ | 紫红→棕黑 | 0.59 |
| 强碱性 | MnO₄²⁻ | 紫红→绿 | 0.56 |

### 2.7 Mn²⁺ 的鉴定（第八讲可用）【→学生讲义：钛钒铬锰 · Mn】

**方法**：用 NaBiO₃ 或 (NH₄)₂S₂O₈（Ag⁺催化）将 Mn²⁺ 氧化为 MnO₄⁻：
- 2Mn²⁺ + 5NaBiO₃ + 14H⁺ → 2MnO₄⁻(紫红) + 5Bi³⁺ + 5Na⁺ + 7H₂O
- 2Mn²⁺ + 5S₂O₈²⁻ + 8H₂O → 2MnO₄⁻ + 10SO₄²⁻ + 16H⁺

### 2.8 Fe/Co/Ni 氢氧化物还原性递变（第九讲核心）【→学生讲义：铁钴镍铜锌 · Fe/Co/Ni】

**三大特征递变**：

| 性质 | Fe | Co | Ni |
|:---|:---:|:---:|:---:|
| M(OH)₂ 颜色 | 白色 | 粉红色 | 绿色 |
| M(OH)₂ 被空气氧化 | 极易（瞬间→红棕Fe(OH)₃） | 缓慢（→棕褐Co(OH)₃） | 不被氧化 |
| M(OH)₃ 氧化性 | 弱（不氧化Cl⁻） | 强（氧化Cl⁻→Cl₂） | 强（氧化Cl⁻→Cl₂） |
| M³⁺/M²⁺ φ°/V | 0.771 | 1.92 | — |

**课堂演示思路**：Fe(OH)₂ → Fe(OH)₃ 颜色变化极快 → 制备时需隔绝空气

### 2.9 CoCl₂ 的变色性质（第九讲课堂可用）【→学生讲义：铁钴镍铜锌 · Fe/Co/Ni】

- CoCl₂·6H₂O (粉红) ⇌ CoCl₂·2H₂O (紫红) ⇌ CoCl₂·H₂O (蓝紫) ⇌ CoCl₂ (蓝色)
- **课堂信号点**：CoCl₂ 用作变色硅胶的湿度指示剂——干燥时蓝色，吸湿后粉红色

### 2.10 铜族 vs 碱金属对比（第十讲可用）【→学生讲义：铁钴镍铜锌 · Cu】

| 比较项 | 铜族(IB) | 碱金属(IA) |
|:---|:---|:---|
| 价电子构型 | (n-1)d¹⁰ns¹ | ns¹ |
| 金属活泼性 | 不活泼（Cu/Ag/Au） | 极活泼 |
| 与水反应 | 不反应 | 剧烈反应 |
| 与非氧化性酸 | 不反应 | 剧烈反应 |
| 常见氧化态 | +1,+2,+3 | +1 |
| 离子颜色 | Cu²⁺蓝、Ag⁺无色 | 均无色 |
| 配合物倾向 | 极强 | 弱 |

### 2.11 Cu(I) vs Cu(II) 的稳定性（第十讲核心）【→学生讲义：铁钴镍铜锌 · Cu】

**口诀**：高温、固态 → Cu(I) 稳定；水溶液 → Cu(II) 稳定

- Cu₂O + H₂SO₄ → CuSO₄ + Cu↓（歧化）
- Cu²⁺ + Cu + 2Cl⁻(浓热) → 2[CuCl₂]⁻（逆歧化，用于制备 CuCl）
- 2Cu²⁺ + 4I⁻ → 2CuI↓(白) + I₂（分析化学中定量测铜）

**[\Cu(NH₃)₂]⁺ 除 O₂**：4[\Cu(NH₃)₂]⁺ + O₂ + 8NH₃·H₂O → 4[\Cu(NH₃)₄]²⁺(蓝) + 4OH⁻ + 6H₂O

### 2.12 锌族元素与碱土金属对比（第十讲可用）【→学生讲义：铁钴镍铜锌 · Zn族】

| 性质 | 锌族(ⅡB) | 碱土金属(ⅡA) |
|:---|:---|:---|
| 次外层电子 | 18e（强极化力） | 8e |
| 金属活泼性 | Zn > Cd < Hg | 从上到下增强 |
| 与非氧化性酸 | Zn溶、Cd难溶、Hg不溶 | 均溶 |
| 氢氧化物 | Zn(OH)₂两性 | Mg(OH)₂中强碱 |
| 配合物 | 易形成 | 难形成 |
| 熔点 | 低（低熔点金属） | 较高 |

**汞的特殊性**：
- 常温唯一液态金属
- 与 Au 形成金汞齐（用于提取黄金）
- Hg₂²⁺(亚汞离子) 存在 → 区别于 Zn/Cd

### 2.13 Hg 的两种价态与鉴定（第十讲可用）【→学生讲义：银金汞钼钨 · Hg】

- Hg₂Cl₂（甘汞，白色，难溶）：Hg₂²⁺ + 2Cl⁻ → Hg₂Cl₂↓
- HgCl₂（升汞，白色，可溶）：Hg²⁺ + 2Cl⁻ → HgCl₂（共价分子）
- **鉴定反应**：SnCl₂ 还原 Hg²⁺ 分两步：
  1. 2HgCl₂ + SnCl₂ → Hg₂Cl₂↓(白) + SnCl₄
  2. Hg₂Cl₂ + SnCl₂（过量）→ 2Hg↓(黑) + SnCl₄

## 四、例题精选（含解析）

### 例1 [V₂O₅ 的氧化-还原-缩合综合] ⭐⭐⭐⭐

**来源**：第六讲 典型例题

**教学价值**：⭐⭐⭐⭐
- V(V) → V(IV) → V(III) → V(II) 四步还原，颜色变化丰富
- 多钒酸根的 pH 依赖缩合
- 钒在工业催化中的应用（接触法制硫酸的催化剂）

### 例2 [Cr(III) 配合物的动力学惰性] ⭐⭐⭐⭐

**来源**：第七讲 典型例题

**教学价值**：⭐⭐⭐⭐
- [Cr(H₂O)₆]³⁺ (紫) / [CrCl₂(H₂O)₄]⁺ (绿) / [CrCl(H₂O)₅]²⁺(浅绿) 三种水合异构体
- 配体交换极慢（d³ 构型，动力学惰性）
- **课堂信号点**：无水 CrCl₃ 极难溶，需痕量还原剂促使溶解（破坏 d³ 惰性）

### 例3 [KMnO₄ 滴定与条件控制] ⭐⭐⭐

**来源**：第八讲 典型例题（Fe²⁺ 滴定）

**教学价值**：⭐⭐⭐⭐
- KMnO₄ 自身做指示剂（紫红→无色，不需额外指示剂）
- Mn²⁺ 的自催化效应（初始反应慢，Mn²⁺ 生成后加速）
- 酸度控制（不能用 HCl，Cl⁻ 会被氧化）

### 例4 [Fe³⁺/Co³⁺/Ni³⁺ 氧化性比较] ⭐⭐⭐⭐

**来源**：第九讲 典型例题

**教学价值**：⭐⭐⭐⭐⭐
- Fe(OH)₃ + HCl → FeCl₃ + H₂O（酸碱中和）
- Co(OH)₃ + 6HCl → 2CoCl₂ + Cl₂↑ + 6H₂O（氧化还原）
- **课堂核心**：从电极电势直观解释 φ°(Fe³⁺/Fe²⁺)=0.771V < φ°(Cl₂/Cl⁻)=1.36V → 不氧化 Cl⁻；φ°(Co³⁺/Co²⁺)=1.92V > 1.36V → 氧化 Cl⁻

### 例5 [Cu(I) 与 Cu(II) 的转化] ⭐⭐⭐

**来源**：第十讲 典型例题（Cu²⁺ + I⁻ → CuI + I₂）

**教学价值**：⭐⭐⭐⭐
- Cu²⁺ 氧化 I⁻ 的定量反应 → 碘量法测铜
- CuI 沉淀推动反应进行 → φ°(Cu²⁺/CuI) 上升至 0.86V > φ°(I₂/I⁻)=0.536V
- **课堂信号点**：沉淀生成可以改变氧化还原方向！
