## 一、这讲真正提供了什么

第4讲 `配合物` 是初赛讲义中最"宽广"的讲次——它从酸碱理论讲起，一路延伸到配位场理论的分子轨道解释。表面上看是一本配合物的"小百科全书"，但仔细对读之后，**这一讲真正提供的不是更多配合物知识，而是一条从酸碱理论→配合物推断→成键解释的完整认知链**。

现有知识库对配合物的覆盖密度在初赛讲义所有主题中可能是最高的：[[配合物]]（命名/概念/异构）、[[配合物杂化轨道理论]]（VBT→几何/磁性）、[[配合物几何构型]]（CN 1-9 构型表）、[[配合物异构]]（六大异构类型）、[[晶体场理论]]（d 轨道分裂/HS-LS/CFSE/光谱化学序）、[[晶体场分裂与磁性]]。但第4讲提供了七样现有系统缺乏或组织方式不同的东西：

### 1. 酸碱理论作为配合物的"母语"（§4.1）

这是第4讲最容易被误判为"重复内容"但实际上最有教学价值的部分。表面上 §4.1 讲的是 Brønsted 酸碱 + Lewis 酸碱，但第4讲做了一件现有 KB 没做的事：**把酸碱理论作为配合物化学的统一前置语言**。

- **诱导效应→酸性→配位倾向的因果链**（习题4.1-4.7）：从 Bent 规则解释诱导效应来源 → 吸电子诱导使酸性增强 → 酸性强弱反映配体的给电子能力 → 配位键的形成倾向。这是把有机化学的电子效应语言翻译成配位化学的判断工具。
- **Lewis 酸碱的 LUMO/HOMO 解释**（例4.8-4.9）：BF₃ + NH₃ 不是简单的"酸碱加合"——讲义从 B 的 LUMO 和 N 的 HOMO 的能级匹配出发，解释了为什么 BF₃ 的 Lewis 酸性弱于 BCl₃（F 的 2p→B 的 2p 共轭使 LUMO 升高更多）。**这比现有 KP 中"Lewis 酸=电子对受体"的定义深了一个层次。**
- **I₂ 在不同溶剂中的颜色变化**（例4.9）：用 Lewis 酸碱 + 荷移跃迁（charge transfer）统一解释——己烷中紫色（纯 I₂ 跃迁）、苯中红色（弱 Lewis 碱→给-受体跃迁）、乙醚中红棕色（强 Lewis 碱→更大蓝移→补色红移）。**这是"配合物颜色"在进入晶体场理论之前的最自然入口。**

> 现有 [[酸碱理论]] 和 [[Lewis酸碱理论]] 是独立的知识点页，缺少"酸性/碱性/配位倾向"的统一叙述。第4讲提供了这个"统一语言层"。

### 2. 一套竞赛难度的配合物推断技术链（全讲核心）

第4讲最值钱的部分不是概念讲解，而是**超过 20 道推断型例题构成的"技术流"训练体系**。这些题目的共同模式是：

```
化学式/反应信息 → 氧化态 → d电子数 → 配位数 → 几何构型 → 磁性 → 完整结构
```

这与第3讲的"五工具链式推断"一脉相承，但更聚焦于配合物特有的推断节点：

**最值得保留的推断题（详见 §五）**：

- 例4.15（V 配合物——氧化数 + 配位数变化的判别）
- 例4.25（钌红 [Ru₃O₂(NH₃)₁₄]Cl₆——氧化态分配 + d-pπ 多中心键的推断）
- 例4.26（Rh(PPh₃)₃Cl + 烯烃催化循环——VBT 杂化 + 18e⁻ + 磁性 + 催化中间体）
- **例4.27（★ 全讲最完整的推断链）**：VI₃ → VI₂ → VOI₂ → V(bipy)₃I₂ → V(bipy)₃ → LiV(bipy)₃ → V(bipy)₃I — 从失重数据反推化学式 → 从磁矩验证 dⁿ → 从反应条件判断氧化态变化
- 例4.32（棕色环 [Fe(NO)(H₂O)₅]²⁺——从磁矩反推 NO 的配位形式（NO⁺/NO/NO⁻）和 Fe 的氧化态）
- 习题4.33（FeCl₂ + CN⁻ + CO → 两种八面体异构体——磁矩 0 + 18e⁻ + 对称性约束 → 结构）
- 例4.34（Pt 抗癌药合成——反位效应驱动的五步合成路线推断）
- 习题4.35（IrCl₃ + L → fac/mer 异构 + 空气氧化 + 对称性推理）

> **这些题与现有 KP 例题的互补关系**：KP 例题偏"概念检验"（判断 CN、命名、HS/LS），第4讲例题偏"多步推断"（给你不完整信息，让你把氧化态/dⁿ/几何/磁性/对称性五个节点串起来推出结构）。

### 3. 定性推出 d 轨道裂分型式的方法（§4.3.2 例4.29）

现有 [[晶体场理论]] 已经给出了八面体场和四面体场的完整分裂图，但第4讲提供了一样更底层的东西：**"从配体位置直接推出分裂型式"的推理方法**。

例4.29 的方法论：
- **平面正方形**：配体在 x, y 正负半轴 → d_(x²−y²) 受影响最大 → d_(xy) 其次 → d_(xz)/d_(yz) 等价（受影响最小） → d_(z²) 略有不同（xOy 面上有小环） → 裂分型式 "2111"
- **四方锥**：同平面正方形，但 d_(z²) 受影响更多 → 仍为 "2111"
- **三角双锥**：轴向配体在 z 轴，赤道面 3 配体在 xOy → d_(xz)/d_(yz) 等价，d_(x²−y²)/d_(xy) 等价 → d_(z²) 受影响最大 → 裂分型式 "221"

> **这个方法的独到之处**：它不是让学生死记"八面体裂成 t₂g+e_g、四面体裂成 e+t₂"，而是教他们"给我任何配体排布，我都能推"。这对竞赛中遇到陌生配位构型（如三棱柱、加冠多面体）极其有用。

### 4. 反位效应的初赛级讲解（§4.3.3）

现有 KB 中没有独立的"反位效应"KP。第4讲提供了：
- **完整的反位效应顺序**（F⁻/H₂O/OH⁻ < NH₃ < py < Cl⁻ < Br⁻ < I⁻ < SO₃²⁻ < PR₃ < H⁻/NO/CO/CN⁻/C₂H₄）
- **顺铂合成的反位效应逻辑**（为什么从 PtCl₄²⁻ 开始，用 2 当量 NH₃ 得到顺铂，而不能反过来）
- **反位效应强的配体特征**：强 π 受体 或 体积特别小
- **例4.34**（Pt 抗癌药的完整五步合成推断——反位效应每一步都在起作用）

### 5. 配位场理论对光谱化学序的 σ/π 解释（§4.4）

第4讲声明 §4.4 "不在初赛要求范围内"，但它提供的 MO 解释恰好是**理解光谱化学序本质的最短桥**：

- **只考虑 σ 成键的定性 MO 图**：配体群轨道（A₁g + T₁u + Eg）与金属 s/p/d 的对称性匹配 → t₂g 为非键 → Δ₀ 出现
- **π 给体配体（如 I⁻）**：填充的 π 轨道与 t₂g 作用 → t₂g 能量升高 → Δ₀ **减小** → **弱场**
- **π 受体配体（如 CN⁻, CO）**：空的 π* 轨道与 t₂g 作用 → t₂g 能量降低 → Δ₀ **增大** → **强场**
- **中间配体（如 H₂O, NH₃）**：π 相互作用很弱 → Δ₀ 居中

> 这个解释的价值在于：它把光谱化学序从"需要死记的顺序表"变成了"可以用 π 给体/受体逻辑理解和预测的分类"。

### 6. 反馈 π 键的价键理论入口（§4.3.1 例4.24）

第4讲在 VBT 部分引入了一个简洁的反馈 π 键图示：过渡金属卡宾配合物中 M=CH₂ 的双键性质来自 σ 给电子（C→M）+ π 反馈（M→C）。这比 [[配合物杂化轨道理论]] 中目前的内容更直观，是"反馈 π 键"进入课堂的最自然入口。

### 7. 酸碱理论中的电子效应与空间效应（§4.1.1）

§4.1.1 的三个小节（诱导效应、超共轭效应、空间效应）虽然表面上是"有机化学内容"，但第4讲把它们放在配合物讲次的开头，意图明确：**这些是理解"配体为什么这样配位"的语言**。

- 习题4.1：用 Bent 规则解释诱导效应的来源
- 例4.3：丁二酸 vs 戊二酸的 pKₐ（诱导效应 + 链长）
- 例4.4：氰基甲烷的酸性递变（共轭效应 + 空间限制）
- 例4.7：超强碱的位阻驱动（质子化前排斥 → 质子化后六元环氢键稳定）

## 三、第4讲的核心方法树

### 3.1 第一层：配合物推断的标准工作流

第4讲在反复训练一套"拿到配合物信息怎么推结构"的稳定工序：

```
第一步：化学式/反应信息层
  → 从化学式和反应条件推断元素组成
  → 从失重/增重数据计算化学式（如例4.27的方程联立）
  → 从质量分数验证推测

第二步：氧化态层
  → 计算中心原子的氧化态
  → 在对称结构中分配氧化态（如例4.25中 x+2y-4=6）
  → 结合氧化态判断可能的配体形式（如 NO 可以是 NO⁺/NO/NO⁻）

第三步：d 电子层
  → 由氧化态 + 原子序数 → dⁿ 组态
  → 结合磁性数据验证 dⁿ（μ = √(n(n+2)) → 反推 n）
  → 如果是 d⁴-d⁷ → 判断 HS/LS

第四步：配位数 + 几何层
  → 从配体数和配位模式确定配位数
  → 从 CN + dⁿ + 配体性质 → 判断几何构型
  → 从对称性约束（如 C₂、S₄）进一步限制几何

第五步：成键解释层（如需要）
  → VBT：杂化→内外轨→磁性
  → CFT：d 轨道分裂→CFSE→稳定性
  → LFT：σ/π 作用→光谱化学序的位置
  → 反馈 π 键、d-pπ 多中心键（解释异常键长）
```

### 3.2 第二层：排除法——"先列约束再试答案"

第4讲与前几讲一致地强调排除法：

- 例4.32（棕色环）：先列出 NO 三种可能配位形式（NO⁺给 3e⁻ / NO 给 2e⁻ / NO⁻ 给 1e⁻）→ 分别计算 Fe 的氧化态和 dⁿ → 逐一检验磁矩 → 只有 NO⁺ + Fe(I) d⁷ 高自旋满足 μ ≈ 3.8
- 习题4.33：磁矩 = 0 → 全部配对 → Fe(II) d⁶ 低自旋 + 18e⁻ → 排除所有高自旋可能
- 例4.29：不记分裂图 → 从"哪个 d 轨道离配体最近"直接推理

### 3.3 第三层：最该纠正的几个误区

1. **"配合物磁性只看 d 电子数"** —— 不够。还要看：配体场强（HS/LS）、几何构型（Δ₀ vs Δt vs Δsq）、是否存在金属-金属键。
2. **"氧化态一定是个整数"** —— 在多核配合物中可能需要分配。例4.25 中 Ru₃ 的总电荷 +6，方程 x+2y-4=6 推出 x=4, y=3。
3. **"光谱化学序只能死记"** —— 第4讲 §4.4 给出的 π 给体/受体逻辑可以理解和预测：I⁻（π 给体→弱场）、CN⁻/CO（π 受体→强场）、H₂O/NH₃（π 中性→中间）。
4. **"配位数 = 直接相连的原子数"** —— 讲义指出配位数在配合物中应理解为"配位电子对数"。例4.15 中 V 的配位数按电子对算是 7。
5. **"Jahn-Teller 畸变任何 dⁿ 都会发生"** —— 只有 e_g 上有奇数个电子时（d⁴ HS, d⁷ LS, d⁹）畸变才明显。

## 四、分节提炼 · 高增量板块

### 4.1 `§4.1` 酸碱理论→配合物的统一语言（★ 本讲最高增量之一）

#### 核心可复用的教学框架

**从 Brønsted 酸性到 Lewis 碱性到配位能力的因果链**：

```
吸电子诱导效应 → H⁺ 更易离去 → Brønsted 酸性增强
                          ↓
          共轭碱的给电子能力减弱 → Lewis 碱性减弱
                          ↓
          配体的配位能力减弱（软硬酸碱在调控）
```

- 习题4.1：Bent 规则→杂化 s 成分→电负性→诱导效应→酸性/碱性
- 例4.3：丁二酸 vs 戊二酸的 pKₐ——链长越短→诱导效应越强→第一级电离容易（pKₐ₁ 小）但第二级电离困难（因为—COO⁻ 对—COOH 的给电子诱导 / 场效应）

**Lewis 酸碱的 LUMO/HOMO 解释**（比"电子对受体/给体"深一层）：

- 例4.8（BF₃ + NH₃）：BF₃ 的 LUMO（B 的 2p + F 的 2p 共轭→反键性质）+ NH₃ 的 HOMO（N 的孤对）→ 形成配位键。BX₃ 的 Lewis 酸性：BCl₃ > BF₃（Cl 的 3p 与 B 的 2p 能级不匹配→LUMO 中 B 2p 成分更多→能量更低→更容易接受电子）。
- 例4.9（I₂ 在不同溶剂中的颜色）：己烷（无 Lewis 碱性）→ π_g*→σ_u* 紫色；苯（弱 Lewis 碱）→ 给-受体跃迁→蓝移→补色红色；乙醚（强 Lewis 碱）→ 荷移跃迁→更大蓝移→补色红棕色。**这是"配合物颜色"在晶体场之前的最自然入口。**

#### 对现有系统的回流价值

最适合回流到 [[酸碱理论]] 或 [[Lewis酸碱理论]]：
- 在现有定义之后，补入"酸碱→配位倾向"的衔接段
- 加入 Lewis 酸碱的 LUMO/HOMO 能级解释（比纯定义式讲解更有竞赛价值）
- 加入 I₂ 在不同溶剂中的颜色变化作为"HOMO-LUMO→颜色"的直观案例

### 4.2 `§4.3.2` 晶体场理论的"定性推出"方法 + 反位效应（★ 本讲最高增量之二）

#### 定性推出 d 轨道裂分型式的三步法（例4.29）

**第一步：设坐标系，定位配体**
- 平面正方形：4 配体在 ±x, ±y
- 四方锥：4 底面配体在 ±x, ±y，1 顶配体在 +z
- 三角双锥：2 轴向在 ±z，3 赤道在 xOy 面 120° 分布

**第二步：判断每个 d 轨道受配体影响的程度**
- d_(x²−y²)：瓣指向 x, y 轴 → 配体在 x, y 轴时受影响**最大**
- d_(z²)：沿 z 轴的瓣 + xOy 面的"甜甜圈" → z 轴有配体时受影响大，仅在 xOy 面时受影响较小
- d_(xy)：瓣在 x, y 轴之间的 45° 方向 → 受影响**次之**
- d_(xz), d_(yz)：瓣在 xz/yz 面 → 通常受影响**最小**且**等价**

**第三步：按受影响程度排序，得裂分型式**
- 平面正方形：d_(x²−y²) ≫ d_(xy) > d_(z²) > d_(xz)=d_(yz) → "2111"（具体轨道归属暂不明确）
- 四方锥：同平面正方形 → "2111"
- 三角双锥：d_(z²) ≫ d_(x²−y²)=d_(xy) > d_(xz)=d_(yz) → "221"

> 教学价值：这个方法训练的不是"背诵"，而是"空间想象力 + 静电推理"。学生学会之后，遇到任何陌生配位构型都能自己推出来。

#### 反位效应（trans effect）

**完整顺序**（从弱到强）：
```
F⁻, H₂O, OH⁻ < NH₃ < py < Cl⁻ < Br⁻ < I⁻, SCN⁻, NO₂⁻, SC(NH₂)₂, Ph⁻ 
  < SO₃²⁻ < PR₃, AsR₃, SR₂, CH₃⁻ < H⁻, NO, CO, CN⁻, C₂H₄
```

**两个特征**：强反位效应的配体要么具有很强的 π 受体性质（CO, CN⁻, C₂H₄），要么体积特别小（H⁻）。

**顺铂合成的反位效应逻辑**：
- 从 PtCl₄²⁻ 开始：Cl⁻ 的反位效应 > NH₃ → 第一当量 NH₃ 取代后，其对位的 Cl⁻ 被活化 → 第二当量 NH₃ 选择性地取代这个被活化的 Cl⁻ → 得到顺式 PtCl₂(NH₃)₂
- 如果反过来（从 Pt(NH₃)₄²⁻ 开始加 Cl⁻），NH₃ 的反位效应弱 → 不会得到顺铂

**例4.34（Pt 抗癌药的合成推断）**是反位效应最好的综合训练：
1. K₂PtCl₄ + 过量 KI → K₂PtI₄（I⁻ 容易离去，为后续取代做准备）
2. + 2 CH₃NH₂ → 顺式 Pt(CH₃NH₂)₂I₂（反位效应：I⁻ > CH₃NH₂ → 第二当量取代对位 I⁻）
3. + HClO₄ → [Pt₂(μ-I)₂(CH₃NH₂)₂I₂]（双核，两个 I 桥连）
4. + NH₃ → Pt(NH₃)(CH₃NH₂)I₂
5. + Ag₂CO₃ + 丙二酸 → 目标产物（Ag⁺ 去 I⁻，CO₃²⁻ 去质子）

### 4.3 `§4.4` 配位场理论对光谱化学序的 σ/π 解释（★ 中等增量）

这节虽标注"不在初赛要求范围内"，但它提供的 MO 解释是把光谱化学序从"死记表"变成"可理解可预测的分类"的关键。

#### 核心逻辑（三层）

**第一层：只考虑 σ 成键的 MO 图**
- 6 个配体的 σ 群轨道对称性：A₁g + T₁u + E_g
- 金属 4s → A₁g；4p → T₁u；3d → E_g + T₂g
- 相同对称性的轨道组合：成键 MO（主要配体特征）+ 反键 MO（主要金属特征）
- T₂g 的 d 轨道（d_(xy), d_(xz), d_(yz)）没有相同对称性的配体群轨道可匹配 → **非键**
- 分裂能 Δ₀ = E(e_g*) − E(t₂g) 自然出现

**第二层：π 给体→弱场**
- π 给体配体（I⁻, Br⁻, Cl⁻）有**填充的** π 对称性轨道
- 这些轨道具有 T₂g 对称性 → 与金属的 t₂g 非键轨道作用
- 成键 π MO（更低的能量）+ 反键 π* MO → t₂g 的能量被**推高**
- 结果：Δ₀ **减小** → 弱场

**第三层：π 受体→强场**
- π 受体配体（CO, CN⁻, 膦）有**空的** π* 轨道
- 同样具有 T₂g 对称性 → 与金属的 t₂g 轨道作用
- 金属 t₂g 电子部分"反馈"到配体 π*（反馈 π 键）→ t₂g 能量**降低**
- 结果：Δ₀ **增大** → 强场

**分类口诀**：
```
I⁻/Br⁻/Cl⁻ → 有孤对给 π → t₂g 升高 → Δ₀ 小 → 弱场
H₂O/NH₃   → π 作用弱   → t₂g 不动 → Δ₀ 中 → 中间场
CN⁻/CO    → 有空 π* 接 → t₂g 降低 → Δ₀ 大 → 强场
```

#### 对现有系统的回流价值

最适合回流到 [[晶体场理论]] 的光谱化学序部分：
- 在现有"光谱化学序表"之后，加一个"为什么是这个顺序？——σ/π 配位场解释"板块
- 这个板块同时服务第二轮（理解光谱化学序）和第四轮（配位场理论的 MO 处理）
