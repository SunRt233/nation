## 一、这讲真正提供了什么

这讲是初赛讲义中最"实用"的讲次之一。表面上它覆盖了 Lewis 结构式 → 价键理论 → 杂化 → VSEPR → 离域键 → Bent 规则 → 分子间力 → 对称性 八块内容，现有知识库对每一块都已有独立的 KP 覆盖。但仔细对读之后，**这讲提供了六样现有系统缺乏或不够深的东西**：

### 1. 一套"程序化"的 Lewis 结构推断工具箱（§3.1）

现有 [[Lewis结构式]] 讲的是概念和规则。第3讲提供的是一套**可以当口诀传的操作手册**：

- **键数计算公式**：$n_{成键数} = \frac{1}{2}\left(\sum n_{希望得到的电子数} - \sum n_{价电子数}\right)$
  - 这不是死记的公式——它的逻辑是："每个原子希望达到稀有气体构型需要多少电子"减去"实际有多少价电子"，差额就是要通过成键补足的电子数，除以 2 = 键数
  - 例3.1 中用它秒算 N₈ 链状分子的键数：$8 \times (8-5) / 2 = 12$ 根键
- **形式电荷三规则**（口诀化）：
  - (a) 价层电子数计数规则——每成一根共价键，原子"多一个电子"；配位键受体多 2 个电子
  - (b) 形式电荷与键数的关系——成键数比特征键数多 1 → 形式电荷 +1；少 1 → −1
  - (c) 形式电荷总和 = 物种净电荷（校验用）
- **共振式稳定性排序**（五条，按重要度降序）：稀有气体结构 > 形式电荷绝对值 ≤ 1 > 电负性匹配 > 键数最多 > 正负电荷相邻

> 这套工具箱的独到之处在于：它不是让学生"凭感觉画 Lewis 式"，而是给了一条**可校验、可逆推**的机械路径。这对基础班学生尤其重要——他们最缺的不是知识，是"拿到一个陌生分子，第一步往哪走"。

### 2. 一套竞赛难度的结构推断题链（全讲核心）

第3讲是整套讲义中**例题密度和质量最高的讲次之一**。超过 15 道例题和习题覆盖了从基础 Lewis 式到决赛级多面体对称性推断的全谱。这些题目共同的特点是：**不是简单判断"这是什么构型"，而是需要综合运用 Lewis → VSEPR → 杂化 → 离域键 → 对称性 五件工具进行链式推断**。

最值得保留的题目包括（详见 §五）：

- 例3.1（N₈ 链状分子——键数公式的实战）
- 例3.4/3.5（N₅⁻ 和 S₂N₂ 的键级计算——共振式的量化出口）
- 例3.7（ClO₂ 自由基——"不出现形式电荷"约束下的 Lewis 式推断）
- 例3.10（Zintl 相 As₄⁴⁻——从化学式反推团簇结构 + 等电子体思维）
- 例3.16（I₅⁺ 的 N/W 异构体——VSEPR + 杂化 + 形式电荷的综合判断）
- 例3.23（Xe 稀有气体化合物——VSEPR + 对称性 + 氧化态的三层推断）
- 例3.24（HBS₂ → B₈S₆(S₂)₄ 的 6 步结构推断链——全讲最能代表"竞赛真实难度"的一道题）
- 例3.35（Al₈C₁₆N₈H₄₈——欧拉公式 + S₄ 对称性 → 多面体结构，决赛级）

> **这些题目和现有 KP 中的例题形成互补**：KP 例题偏"概念检验"（判断量子数是否有效、比较电离能大小），第3讲例题偏"工具链综合"（给你不完整信息，让你自己把五个工具串起来推出结构）。

### 3. Bent 规则的无机/结构化学应用（§3.5.2）

现有 [[Bent规则]] 放在有机化学模块下，侧重解释酸性（环丙烷 > 丙烷 > 乙烯）和格氏试剂制备。第3讲提供了 Bent 规则的**物理推导**和**无机分子键角/键长预测应用**：

- 推导核心：电负性差异 → 电子偏向 → s/p 成分重新分配 → 键角/键长变化
- 例3.27：H₂O₂ 的 H—O—O 键角 < H₂O 的 H—O—H 键角（因为 O 比 H 电负性大，O—O 侧 p 成分更多 → 键角更小）
- 习题3.28：NH₃ vs NF₃ vs NCl₃ 的键角排序，H₂O vs H₂S vs H₂Se
- 习题3.29：E(CF₃)₃（E = P, As, Sb）的结构数据异常判断

还有一个非常微妙的点：第3讲中 Bent 规则的描述和现有 KP 的表述**方向恰好相反**：
- 现有 KP 说"s 成分更多分配给电负性**大**的取代基"（把低能 s 轨道给更需要电子的原子）
- 第3讲 §3.5.2 的推导说"指向低电负性原子/基团的杂化轨道 s 成分增加"——这个差异需要辨析，见 §四 4.3 的详细讨论

### 4. 离域键的"二聚倾向"分析框架（§3.5.1）

现有 [[离域π键]] 有定义和判断步骤。第3讲最有价值的增量是一个对比：

| 分子 | 离域键类型 | 二聚倾向 | 原因 |
|:---|:---|:---|:---|
| ClO₂ | $\pi_3^5$（含单电子） | **不容易**二聚 | 单电子在离域体系中稳定化 |
| NO₂ | $\pi_3^4$（不含单电子） | **容易**二聚 | 单电子留在 N 的 sp² 杂化轨道上，未进入离域体系 → 活泼 |

> 这个对比的价值在于：它把"离域键类型"和"化学性质（二聚倾向）"直接挂钩——学生不只是学会画 $\Pi_m^n$，而是学会用离域键解释"为什么这个分子稳定/不稳定"。NO₂ ⇌ N₂O₄ 的平衡也是竞赛常考题。

### 5. 分子对称性作为结构推断工具（§3.7）

现有 [[分子对称性初步]] 是作为"配位光谱桥梁"出现的（旋转轴/镜面 → 选律 → d-d 跃迁）。第3讲把对称性放在完全不同的用途上：**作为结构推断的约束条件**。

- 例3.23：利用"沿轴向有四次轴"推断 $[Pt_2F_{11}]^-$ 中两个 PtF₄ 面的相对旋转角（全重叠 vs 全交错）
- 例3.35：利用"除一条 S₄ 外没有其他独立对称元素"推断 Al₈C₁₆N₈H₄₈ 的多面体结构 → 欧拉公式 → 四边形/六边形数目 → 排布方式
- 三个对称元素存在定理（若有一个 C₂ 与主轴垂直，则必有 n 个...）

> 这一块直接服务第四轮冲刺——例3.35 就是 2018 年 CChO 决赛真题的变体。它让学生明白：**对称性不是"分类标签"，而是"结构约束"**。

### 6. 价键理论的 σ/π/δ 键图示（§3.2）

目前库里 [[共价键]] 覆盖了 σ 和 π 键，但第3讲提供了一张非常直观的**σ/π/δ 键的"头碰头/肩并肩/面对面"对比图**（`fbc758f7...`），这是讲义中唯一一张把三者放在一起、可以直接进课堂的配图。

## 三、第3讲的核心方法树

### 3.1 第一层：结构判断的标准工作流

第3讲其实在教一套"拿到陌生分子/离子怎么写结构"的稳定工序：

```
第一步：Lewis 层
  → 键数公式算总键数
  → 画骨架（经验：电负性低的在中心，H/F 永远在末端）
  → 分配孤对电子（满足八隅体优先）
  → 标注形式电荷（三规则校验）
  → 如有多个合理 Lewis 式 → 进入共振分析

第二步：VSEPR 层
  → 算价层电子对数 = (中心原子价电子 + 配位原子贡献) / 2
  → 确定电子对构型（直线/三角形/四面体/三角双锥/八面体/五角双锥）
  → 扣孤对电子 → 得分子实际构型
  → 按排斥顺序放孤对电子（lp-lp > lp-bp > bp-bp）
  → 在三角双锥构型中：电负性大的基团放轴向

第三步：杂化 + 轨道层
  → 由构型反推杂化类型
  → 判断是否有未杂化 p 轨道 → 可能形成离域 π 键
  → 判断离域键类型（$\Pi_m^n$）：共面 → 平行 p → m < 2n

第四步（如需要）：进阶判断
  → Bent 规则：电负性差异 → 键角的精细比较
  → 对称性：是否有特殊对称约束
  → 等电子体：有没有已知结构的等电子体可参考
```

### 3.2 第二层：排除法比"猜"更稳

第3讲反复传递的一个元思维是：

> "遇到可能纠结的情况时，如果能用题目条件予以排除则最好，否则就要尽可能合理且多地运用基本理论说服自己：各个答案分别有什么合理之处，又有什么不合理之处？"

典型例子：
- 例3.1（N₈）：键数公式限定键数 → 对称性约束排列 → 形式电荷选最优共振式
- 例3.22（KPO₂ 六元环）：两个候选结构 → 离域键条件排除左图 → 氧化稳定性排除三价 P
- 例3.35（多面体）：欧拉公式算面数 → S₄ 对称性约束排布 → 由等电子体辅助

### 3.3 第三层：最该纠正的几个误区

1. **"杂化轨道理论可以预测构型"** —— 错了。讲义原文说"杂化轨道理论是用来**解释**分子几何构型的，并不能**预测**构型。"预测靠 VSEPR，解释靠杂化。
2. **"形式电荷 = 真实电荷"** —— 错了。形式电荷是人为分配电子的记账结果，不代表该原子上真的有那么多电荷。
3. **"共振式就是分子的真实结构"** —— 错了。共振杂化体才是，任何一种单独共振式都不是。
4. **"看到不饱和就一定是烯/炔"** —— 错。环也是一种不饱和度来源。例3.17（C₃F₄ 的多种异构体）是最好的纠偏材料。
5. **"三角双锥构型中孤对电子随便放"** —— 错。放在赤道面（与更多键成 90°）比放在轴向（与更多键成 120°）排斥更小 → 孤对电子优先放赤道面。

## 四、分节提炼 · 高增量板块

### 4.1 `§3.1` Lewis 结构工具箱（★ 本讲最高增量之一）

#### 核心可复用的操作口诀

**键数公式**（适合做课堂"秒算卡"）：
$$n_{成键数} = \frac{1}{2}\left(\sum n_{希望得到的电子数} - \sum n_{价电子数}\right)$$

> 逻辑：每个原子希望 8 电子（H 希望 2），总数减去价电子总数 = 需要通过成键"补足"的电子数，除以 2 = 键数。含配位键时，受体"多得 2 个电子"、给体不变。

**形式电荷三规则**（适合做课堂"一句总结"）：
1. 成键数 = 特征键数（族数）→ 形式电荷 = 0
2. 成键数 > 特征键数 → 每多 1 根 → 形式电荷 +1（自己的电子被迫分给别人了）
3. 成键数 < 特征键数 → 每少 1 根 → 形式电荷 −1（从别人那里多得了电子）
4. 校验规则：所有原子的形式电荷之和 = 物种的净电荷

**共振式稳定性五条**（按重要度排序，适合做课堂速查卡）：
1. 满足稀有气体结构（八隅体）
2. 形式电荷绝对值 ≤ 1
3. 电负性高的原子不承受正形式电荷，电负性低的原子不承受负形式电荷
4. 键数尽可能多（键能高）
5. 正负形式电荷相邻（静电吸引）

**等电子体口诀**："通式相同（非 H 原子数和电子总数相等）→ 结构相似"

#### 对现有系统的回流价值

最适合回流到 [[Lewis结构式]]：
- 在现有 Lewis 式书写规则后，加一个"快速推断工具箱"板块（键数公式 + 形式电荷三规则 + 共振排序）
- 这恰好是第一轮基础班最需要的——不是更深的 MO 理论，而是"拿到题先做什么"

### 4.2 `§3.4` VSEPR 的精细规则（★ 中等增量）

现有 [[VSEPR理论]] 有完整的构型表。第3讲增加了两条课堂教学中非常实用的规则：

**排斥力排序（6 级）**：
```
孤对-孤对 > 孤对-重键 > 孤对-单键 > 重键-重键 > 重键-单键 > 单键-单键
```

**三角双锥中的位置规则**（目前库里不够明确）：
- 孤对电子优先放赤道面（与 2 个 90° + 2 个 120° 的键相邻），而不是轴向（与 3 个 90° + 1 个 180° 的键相邻）
- 电负性大的原子/基团优先放轴向——因为电负性小的原子使成键电子对更像孤对电子，而这种"像孤对的键对"应该放赤道面

> 例3.15（CH₂SF₄）是这条规则的完美示范：F 电负性大 → 放轴向，CH₂ 放赤道面。

### 4.3 `§3.5.2` Bent 规则的双向表述辨析（★ 本讲最高增量之二）

这是一个**需要特别辨析的微妙点**。

**第3讲的表述**（§3.5.2 例3.26 的推导）：
> "指向低电负性原子/基团的杂化轨道的 s 成分增加，反过来则 p 轨道成分增加。"

推导逻辑：设 XAY₃，χ_X > χ_A > χ_Y。Y-A 键的电子偏向 A → 该电子能量受 A 指向 Y 的杂化轨道影响较大 → s 轨道能量更低 → s 成分增加。反过来，X-A 键的电子偏向 X（远离 A）→ A 指向 X 的杂化轨道 p 成分增加。

**现有 KP [[Bent规则]] 的表述**：
> "将更多的 s 成分分配给与电负性较大的取代基成键的轨道。"

这两条看起来矛盾，但其实说的是**同一件事的不同侧面**：
- 第3讲说的是：**中心原子 A 的哪个方向的杂化轨道 s 成分更多**——指向"低电负性"取代基的方向（因为该方向电子更靠近 A，s 轨道（低能）更"值得"用在电子云密度更高的地方）
- 现有 KP 说的是：**电子更偏向哪个原子**——电负性大的原子需要更多电子密度，所以 s 成分更多的轨道应该指向电负性大的原子来"稳住"更多电子

**实际上两者一致**，关键在于"从谁的视角出发"：
- 从中心原子的能量优化出发 → s 成分给"电子更靠近自己"的方向 → 那个方向的取代基电负性低（电子推给中心） → 第3讲的表述
- 从"给电负性大的原子更多电子"出发 → s 成分给那个方向 → 现有 KP 的表述

> **教学建议**：不纠缠两种表述的表面差异。统一口径为——
> **"s 成分越高的轨道，越靠近原子核、能量越低、越适合承载更多电子密度。因此，在电负性差异大的键中，s 成分更多分配给电子云更密集的方向。"**
> 这个统一口径在例3.27（H₂O₂ vs H₂O）和习题3.28（NF₃ vs NCl₃）中都能用。

### 4.4 `§3.5.1` 离域键的性质解释出口（★ 中等增量）

现有 [[离域π键]] 的 §三讲的是"三步判断法"（共面→平行p→m<2n），但缺少"判断完之后做什么"——即解释化学性质。第3讲补上了这个缺口。

**ClO₂ vs NO₂ 的对比**（最适合做课堂"并排讲解"）：

| | ClO₂ | NO₂ |
|:---|:---|:---|
| 中心原子杂化 | sp²（一个放孤对，两个与 O 成 σ） | sp²（一个放单电子，两个与 O 成 σ） |
| 未杂化 p 轨道 | 含 1 个单电子 | 空的 |
| O 的 p 轨道 | 各含 1 对孤对电子 | 各含 1 对孤对电子 |
| 垂直方向 p 电子总数 | 3（Cl p¹ + 两个 O p²） | 4（两个 O p² + 两个 O p²） |
| 离域键类型 | $\pi_3^5$ | $\pi_3^4$ |
| 单电子在哪 | **在离域体系中**（稳定化） | **在 N 的 sp² 杂化轨道上**（未参与离域） |
| 二聚倾向 | 不容易（单电子被离域稳定） | 容易（单电子活泼，N₂O₄ 存在） |

> 这个对比表的妙处在于：它让离域键从一个"分类标签"变成了一个**可以预测化学行为的工具**。

CO₂ 的双 $\pi_3^4$ 分析也值得保留——从 C 的 sp 杂化出发，两个正交的未杂化 p 轨道各与两个 O 形成一套 $\pi_3^4$，在共振式中表现为 $O=C=O \leftrightarrow ^+O\equiv C-O^-$。

#### 对现有系统的回流价值

最适合回流到 [[离域π键]]：
- 在现有 §三"判断方法"之后，新增一个"离域键与化学性质"板块
- 放入 ClO₂/NO₂ 对比表 + CO₂ 双 π₃⁴ 分析

### 4.5 `§3.7` 分子对称性作为结构约束（★ 中等增量）

现有 [[分子对称性初步]] 是服务于配位光谱的。第3讲把对称性放到完全不同的语境：**结构推断的边界条件**。

**三个对称元素存在定理**（适合做课堂速查卡）：
1. 若有一个 C₂ 轴与主轴 C_n 垂直 → 必有 n 个 C₂ 轴与主轴垂直，相邻夹角 = 基转角/2
2. 若有一个镜面通过主轴 C_n → 必有 n 个镜面通过主轴，相邻夹角 = 基转角/2
3. C_{2n}、σ_h、i 中任意两个可推出第三个

**例3.35（2018 CChO 决赛改编）的完整推断链**：
```
AlMe₃ + MeNH₂ → (AlMe)₈(NMe)₈（等电子于 C₁₆Me₁₆）
  → 每个原子 3 根键
  → 欧拉公式：V - E + F = 2 → 16 - 24 + F = 2 → F = 10
  → 四边形 x 个 + 六边形 y 个 = 10
  → 4x + 6y = 3 × 16（每条棱被 2 个面共享，3 键/原子）
  → x = 6, y = 4
  → S₄ 对称性 → 两条异面垂直的抽象链 → 三个四边形共棱 → 添加 4 根键得六边形
```

> 这道题的教学价值是巨大的：它把**欧拉公式 + 对称性约束 + 化学直觉**三个维度绑在了一道题里，是典型的"决赛级结构推断"思维训练。

#### 对现有系统的回流价值

可以在 [[分子对称性初步]] 中增补"对称性作为结构推断约束"板块，或者回流到 [[专题-分子结构基础]] 的竞赛拓展部分。

### 4.6 `§3.6` 分子间作用力的特殊案例

现有 [[分子间力]] 和 [[氢键]] 已覆盖主体内容。第3讲提供了几个值得保留的细节：

1. **HF 三聚**（例3.33）：25°C 时 HF 蒸气表观摩尔质量 58 g/mol（≈ 3 × 20），说明气态 HF 以三聚体 (HF)₃ 为主——这个例子是"氢键不是只在液体中"的最佳教学素材。
2. **非常规氢键**：双氢键（BH₃NH₃ 中的 B—H⋯H—N）、C—H⋯O 等
3. **稀有气体溶解度**（例3.31）：Xe > Kr > Ar > Ne > He，因为原子半径增大 → 电子云更弥散 → 与水分子之间的诱导力更强

## 十、当前结论

第3讲 `分子结构` 是这套初赛讲义中**最实用、最接近"课堂教学手册"**的讲次之一。现有知识库对每一块知识都已有独立 KP 覆盖（Lewis、VSEPR、杂化、MO、离域键、Bent、分子间力、对称性），但第3讲真正提供的不是新知识，而是六样"让知识可操作"的东西：

1. **Lewis 结构推断工具箱**（键数公式 + 形式电荷规则 + 共振排序）——给基础班学生一条"先算后画"的机械路径
2. **竞赛难度结构推断题链**（15+ 道题，覆盖全谱）——给提高班/冲刺班提供"五工具链式推断"的实战训练
3. **Bent 规则的无机应用**（键角/键长预测 + 两种表述辨析）——把 Bent 规则从有机化学模块拉回到结构化学主线
4. **离域键的"性质解释"出口**（ClO₂ vs NO₂ 二聚对比）——让离域键从分类标签变成化学行为预测工具
5. **对称性作为结构推断约束**（欧拉 + 对称性 → 多面体）——给对称性一个不同于"光谱选律"的实用定位
6. **分子间力特殊案例**（HF 三聚、稀有气体溶解度）——丰富课堂举例
