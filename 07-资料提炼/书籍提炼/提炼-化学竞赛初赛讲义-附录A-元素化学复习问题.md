## 一、这讲真正提供了什么

附录A 与前面12讲有本质区别。它不是"讲授某种新方法或新理论的教学单元"，而是**一份按元素族分类的系统复习清单**，约250道问题覆盖：

| 元素族 | 问题数 | 核心覆盖 |
|:---|:---:|:---|
| 碱金属和碱土金属 | 5 | Li反常性、臭氧化物稳定性、微溶盐、钾钠区别 |
| 卤族元素 | 12 | I₂溶剂颜色、氢卤酸酸性、拟卤素、次卤酸根、高卤酸氧化性 |
| 氧族元素 | ~33 | O₂磁性、H₂O₂制备/Fenton、硫氧化物/酸、连二硫酸盐、Se/Te |
| 氮族元素 | ~40 | N₂配合物、液氨、联氨/羟胺、氮氧化物、硝酸/亚硝酸、P₄/AlP、磷含氧酸、As/Sb/Bi |
| 碳族元素 | 33 | CO活泼性、Si/Ge/Sn/Pb单质+氧化物+卤化物+硫化物 |
| 硼族元素 | 24 | 硼砂/硼酸/硼烷、Al/Ga/In/Tl特性 |
| B族元素 (Cu/Ag/Au, Zn/Cd/Hg) | 32 | Cu(I/II/III)、Ag配合物、Hg(I/II)、Zn/Cd |
| 钒钛铬 | 18 | Ti冶炼、V多色、Cr(III/VI)转化、铬酸盐 |
| 钼钨 | 3 | 初赛不要求 |
| 锰 | 10 | Mn氧化物/氢氧化物/含氧酸盐 |
| 铁钴镍 | 17 | 氧化物颜色、配合物、Ni鉴定、羰基配合物 |

**附录A真正提供的不是新知识，而是三样东西：**

### 1. 一份"初赛元素化学全覆盖自检表"

这250道题覆盖了初赛可能涉及的元素化学事实的绝大部分。它的定位不是"学新东西"，而是"学完后检查有没有漏"。任何一个学生如果能流畅回答这250题中的80%，初赛元素化学部分基本没问题。

### 2. 一份隐含的"命题人视角"

仔细读题会发现，很多问题不是在问"X的化学式是什么"，而是在问"为什么X和Y不一样"——这恰好是初赛元素推断题的起点。例如：

- "Li和水反应比同族其他碱金属低，但还原电势也比其他碱金属都要低，为什么？"
- "为什么一氧化碳和氮气是等电子体，键级都为3，但一氧化碳活泼得多？"
- "为什么硒酸氧化性比硫酸强？"

这些都是初赛推断题中"反常现象→结构/原理解释"的经典模式。

### 3. 一份"复习时不跑偏"的边界标定

附录A明确标注了"初赛原则上不要求"的内容（如钼、钨），以及严宣申教材中超出初赛范围但值得了解的边缘知识。这实际上起到了"复习边界标定"的作用。

## 二、本附录的内部结构

附录A按元素族组织，每一族内部的题目没有严格排序，但遵循大致的"单质→氧化物→氢氧化物→卤化物→含氧酸/盐→配合物→特殊性质"路线。

**典型的问题类型分布**：

| 问题类型 | 占比 | 示例 |
|:---|:---:|:---|
| 事实回忆型 | ~50% | "XX是什么颜色？"、"XX如何制备？"、"XX的化学式如何？" |
| 原理解释型 | ~20% | "为什么X比Y更稳定？"、"为什么X的酸性/氧化性更强？" |
| 方程式书写型 | ~15% | "X和Y反应，方程式如何？"、"X加热分解，方程式如何？" |
| 结构/性质关联型 | ~10% | "画出X的结构"、"X的结构有什么特征？" |
| 应用/鉴定型 | ~5% | "如何鉴定X？"、"X常用于做什么？" |

## 五、高价值问题簇（★ 优先回流）

### 5.1 题簇A："反常现象→原理"型（初赛推断题原型）

这类问题不是简单的事实回忆，而是"观察到反常，解释为什么"。它们与第6讲推断技术的"先列硬约束→结构决定性质"一脉相承：

- **Li反常**：反应活性低但还原电势也低 → 水合能/原子化能/电离能的平衡
- **CO vs N₂**：等电子体但CO活泼得多 → 前线轨道：CO的HOMO是C孤对（σ给体），LUMO是π*（π受体）
- **H₃PO₃是二元酸**：结构中有P-H键，不是P-OH
- **BX₃ Lewis酸性：BCl₃ > BF₃**：π共轭压倒σ诱导（已在第4讲回流中覆盖）
- **硒酸氧化性>硫酸**：Se(VI)不如S(VI)稳定，更容易被还原（4p→3d可参与）
- **Hg₂Cl₂光解/歧化**：Hg(I)的稳定性来源是Hg-Hg键
- **SnCl₂还原性 > PbCl₂**：惰性对效应 → Pb(II)比Pb(IV)稳定，但Sn(II)→Sn(IV)更容易

### 5.2 题簇B："特征颜色/沉淀"型（初赛鉴定题基础）

初赛元素推断题中，"颜色"是最高频的硬约束。附录A中对颜色的覆盖极其密集：

| 元素 | 关键颜色信息（摘录） |
|:---|:---|
| Cr | Cr³⁺(绿/紫)、CrO₄²⁻(黄)、Cr₂O₇²⁻(橙)、CrO₃(红)、CrO₂Cl₂(红) |
| Mn | MnO₄⁻(紫红)、MnO₄²⁻(绿)、MnO₂(黑棕)、Mn²⁺(浅粉) |
| Fe | Fe²⁺(浅绿)、Fe³⁺(黄棕)、Fe(CN)₆⁴⁻(黄)、Fe(CN)₆³⁻(红棕) |
| Co | Co²⁺(粉红)、CoCl₂无水(蓝)→吸水(粉红) |
| Ni | Ni²⁺(绿)、Ni(丁二酮肟)₂(鲜红) |
| Cu | Cu²⁺(蓝)、Cu₂O(红)、CuO(黑)、CuI(白) |
| V | V²⁺(紫)、V³⁺(绿)、VO²⁺(蓝)、VO₂⁺(黄) |
| S | 各种硫化物沉淀颜色（ZnS白、CdS黄、Sb₂S₃橙、SnS棕、PbS黑等） |

### 5.3 题簇C："结构画图"型（需图片辅助）

附录A中明确要求学生"画出结构"的问题，这些是最需要图片辅助的：

1. P₄O₁₀、P₄O₆ 等7+种磷氧化物（碳族/氮族）
2. SF₄、S₂F₂、S₂Cl₂ 等9+种硫卤化物（氧族）
3. S₄N₄、(SN)x 等硫氮化物（氧族）
4. 硼砂阴离子 [B₄O₅(OH)₄]²⁻（硼族）
5. H₃PO₂、H₃PO₃、H₃PO₄、H₄P₂O₇（氮族）
6. P₄S₃、P₄S₇、P₄S₁₀（氮族）
7. 五卤化磷的固相结构（PCl₅ = [PCl₄]⁺[PCl₆]⁻, PBr₅ = [PBr₄]⁺Br⁻）
8. CuSO₄·5H₂O 中的氢键网络（B族）
