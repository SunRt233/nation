# 提炼：Clayden 第40章——金属有机化学（Organometallic Chemistry）

---

## 一、核心概念与定义

### 1.1 18电子规则（18 Electron Rule）
- **定义**：若过渡金属配合物中心金属的价壳层具有18个电子（填满1个s、5个d和3个p轨道），则配合物稳定。
- **电子来源**：金属本身的价电子 + 所有配体贡献的电子。
- **族号与价电子**：新族号1–18直接给出金属价电子数（如Pd为10e，Cr为6e）。
- **例外**：Ti、Zr、Ni、Pd、Pt可形成稳定的16电子配合物（平面方形几何，空出一个高能轨道）。
- **实例**：
  - Cr(苯)(CO)₃：6 + 6 + 2 + 2 + 2 = 18e
  - Pd(PPh₃)₄：10 + 4×2 = 18e
  - (MeCN)₂PdCl₂：8(Pd²⁺) + 4×2 = 16e（稳定且活泼）

### 1.2 η数字（Hapto Number）与配体类型
- **η数字**：表示配体与金属成键时涉及的原子数目。
- **σ配合物**：金属与配体单个原子形成σ键（如R–MgBr，η¹）。
- **π配合物**：金属位于π键中央，与多个碳原子成键（如烯烃配合物，η²）。
- **常见配体电子贡献**：

| 配体类型 | η | 形式电荷 | 贡献电子数 |
|:---|:---:|:---:|:---:|
| 卤素、烷基、芳基 | η¹ | –1 | 2 |
| 烯烃 | η² | 0 | 2 |
| π-烯丙基阳离子 | η³ | +1 | 2 |
| π-烯丙基阴离子 | η³ | –1 | 4 |
| 共轭双烯 | η⁴ | 0 | 4 |
| 环戊二烯基（Cp） | η⁵ | –1 | 6 |
| 苯环 | η⁶ | 0 | 6 |

### 1.3 氧化态的确定
- **简单规则**：若非金属与Cl、AcO、Me等阴离子配体形成σ键，金属为(0)氧化态；提供孤对电子的中性配体（如PPh₃）不影响氧化态。
- **二茂铁的两种计数法**：Fe(II) + 2 Cp⁻（各6e）= 18e；或 Fe(0) + 2 Cp•（各5e）= 18e。

### 1.4 反馈成键（Back-bonding）
- **σ给电子**：配体充满的spⁿ轨道与金属空的dsp轨道重叠，形成σ键。
- **π反馈**：金属充满的d轨道与配体空的π*轨道重叠，减少金属电子密度。
- **证据**：CO配合物中C–O键长增加、红外伸缩频率下降。

## 二、金属有机化学的基本反应步骤

### 2.1 氧化加成（Oxidative Addition）
- **定义**：金属插入X–Y键，氧化态升高2（如M(0) → M(II)），配位数增加2。
- **底物范围**：
  - 非极性：H₂、R–H、R₃Si–H
  - 亲电/极性：H–X、R–X、X–X
- **Vaska配合物的机理差异**：
  - H₂：顺式加成，协同机理
  - MeI：反式加成，类Sₙ2机理 + 离子重组
- **要求**：起始金属通常低氧化态（0或I），络合不饱和（<18e）。

### 2.2 还原消除（Reductive Elimination）
- **定义**：氧化加成的逆反应，M(II) → M(0)，释放X–Y中性产物，形成新的σ键。
- **几何要求**：两个配体必须**顺式**排列。
- **实例**：顺式二甲基钯配合物加热释放乙烷；反式双膦配合物不反应。

### 2.3 迁移插入（Migratory Insertion）
- **定义**：一个配体从金属迁移到另一个不饱和配体上，后者插入金属–配体键。
- **可逆性**：逆过程为β-负氢消除（或脱羰基化）。
- **驱动力**：新配体（L）加成恢复络合饱和。
- **常见插入基团**：CO（→酰基）、烯烃（→烷基）、炔烃（→烯基）。
- **立体化学**：迁移基团保持构型。

### 2.4 β-负氢消除（β-Hydride Elimination）
- **定义**：金属烷基的β-H迁移到金属上，形成烯烃配合物。
- **要求**：
  - 金属需有空位（16e配合物有利）
  - 金属与β-H必须处于**syn**构型
- **后果**：Pd–Et等配合物极不稳定，迅速消除乙烯。
- **Pd催化中的底物选择**：R²-X中的R²必须无sp³碳上的β-H（常用烯基、芳基、烯丙基、苄基卤）。

## 三、重要反应与机理

### 3.1 Heck反应（line ~5711）
- **定义**：烯烃与卤代烃/三氟甲磺酸酯的钯催化偶联，形成新的C=C键。
- **底物要求**：
  - R¹：芳基、烯基、无β-H的烷基
  - X：Br、I、OTf
  - 烯烃：单/双取代，富电子/缺电子均可
  - 碱：Et₃N、NaOAc、Na₂CO₃（弱碱即可）
- **催化循环**：
  1. Pd(0)对R¹–X氧化加成 → Pd(II)配合物
  2. 烯烃络合 + 碳钯化反应（carbopalladation）→ 烷基钯
  3. β-负氢消除 → 产物烯烃 + HPd(II)X
  4. 碱消除HX → 再生Pd(0)
- **区域选择性**：
  - 缺电子烯烃：新C–C键在烯烃另一端（远端）
  - 富电子烯烃（如烯基醚）：进攻发生在取代基一端，得1,1-双取代产物
- **立体化学**：β-负氢消除为**syn消除**，控制双键几何。
- **Pd(0)原位生成**：Pd(OAc)₂可被胺、膦、烯烃、有机金属试剂还原。

### 3.2 交叉偶联反应总览（line ~6031）
- **通用机理**：氧化加成 → 转金属反应 → 还原消除
- **卤代烃组分（R²–X）**：必须无β-H，X = I, Br, (Cl), OTf, OPO(OR)₂
- **有机金属组分（R¹–M）**：M = MgX, ZnX, Cu, SnR₃, SiR₃, ZrCp₂Cl, AlMe₂, B(OR)₂, BF₄⁻；R¹可含β-H

### 3.3 Stille偶联（line ~6111）
- **有机金属组分**：有机锡烷（R¹SnR₃）
- **特点**：
  - 广泛适用于sp²–sp²和sp²–sp偶联
  - 几何构型保持
  - 可与羰基化反应结合（CO气氛）
- **转金属为慢步骤**，CO插入可在转金属前发生。

### 3.4 Suzuki偶联（line ~6187）
- **有机金属组分**：硼酸或硼酸酯（R¹B(OR)₂）
- **特点**：
  - 最常用的交叉偶联反应
  - 需额外碱（NaOEt、NaOH、K₂CO₃）
  - 碱通过形成"酸根"（ate）配合物加速转金属
  - 空阻容忍度高，芳基–芳基偶联优异
  - 芳杂环偶联良好
- **立体专一性**：双键几何在偶联中保持。

### 3.5 Negishi偶联（line ~6093）
- **有机金属组分**：有机锌试剂（R¹ZnX）
- **特点**：反应条件温和，官能团容忍性好。

### 3.6 Sonogashira偶联（line ~6271）
- **底物**：末端炔烃 + 芳基/烯基卤
- **共催化剂**：CuI
- **机理**：末端炔烃在碱/CuI下形成炔基亚铜，与Pd(II)中间体转金属，还原消除得产物。
- **条件温和**：常室温进行，适用于热敏感底物。
- **应用**：烯二炔抗生素合成、Bergmann环化前体。

### 3.7 Buchwald–Hartwig胺化（line ~6517）
- **定义**：芳基/烯基卤与胺的钯催化C–N偶联。
- **机理**：氧化加成 → 胺配位 → 碱消除H–I → 还原消除形成Ar–N键。
- **特点**：
  - 可取代传统SₙAr反应
  - 离去基团活性：I > Br > Cl >> F（与SₙAr相反）
  - 无需邻对位吸电子基团
  - 大空阻配体（如xantphos）促进反应
- **工业应用**：伊曲康唑等药物合成。

### 3.8 烯丙基钯化学（π-Allyl Pd Chemistry）（line ~6317）
- **离去基团**：乙酸根最常用，也可为OCO₂R、OPO(OR)₂、Cl、Br、OPh等。
- **催化循环**：
  1. Pd(0)与烯丙基化合物络合，氧化加成到C–X键
  2. 形成π-烯丙基阳离子配合物（η³，16e）
  3. 亲核试剂进攻（软亲核试剂最佳）
  4. 再生Pd(0)
- **立体化学**：两次翻转 = **净构型保持**（Pd从离去基团反面进攻，Nu从Pd反面进攻）。
- **区域选择性**：亲核试剂进攻π-烯丙基配合物**空阻较小**的一端。
- **特殊底物**：
  - **烯基环氧**：开环生成烷氧基碱，无需外加碱，1,4-加成
  - **碳酸一烯丙酯**：脱羧生成烷氧基阴离子，活化亲核试剂
- **Tsuji–Trost反应**：格氏试剂等硬亲核试剂先转金属再还原消除，得**构型翻转**产物。

### 3.9 Wacker氧化（line ~6646）
- **定义**：末端烯烃氧化为甲基酮的Pd(II)催化反应。
- **机理**：
  1. 烯烃与Pd(II)络合
  2. 水分子亲核进攻（钯氧化，oxypalladation）
  3. β-负氢消除释放烯醇
  4. 烯醇→酮互变异构
- **共氧化剂**：CuCl₂/O₂体系将Pd(0)重新氧化为Pd(II)。
- **区域选择性**：水进攻更多取代的一端（马氏水合氧化）。

### 3.10 羰基化反应（Carbonylation）（line ~5589）
- **OXO过程（氢甲酰基化）**：烯烃 + CO + H₂ → 醛
- **催化循环**：烯烃络合 → 金属氢化（迁移插入）→ CO络合 → 羰基插入 → 与H₂还原消除
- **应用**：工业上由丙烯制取丁醛。

## 四、关键结论与规则

### 4.1 电子计数与稳定性
1. **18电子规则**：满足18e的配合物稳定；<18e为络合不饱和，具反应活性。
2. **16电子例外**：Ni、Pd、Pt族金属可形成稳定16e平面方形配合物。
3. **氧化态确定**：与阴离子配体成键时金属非零氧化态；中性配体不影响。

### 4.2 基本反应步骤
4. **氧化加成**：M(0) → M(II)，配位数+2，需络合不饱和金属。
5. **还原消除**：M(II) → M(0)，需顺式配体，形成新σ键。
6. **迁移插入**：构建碳链的主要方式，可逆，被新配体加成驱动。
7. **β-负氢消除**：syn几何要求，16e配合物有利，是Pd催化循环的终止/再生步骤。

### 4.3 Pd催化反应的选择性
8. **Heck反应区域选择性**：缺电子烯烃远端进攻；富电子烯烃近端进攻。
9. **交叉偶联底物限制**：R²–X必须无sp³ β-H；R¹–M可自由选择。
10. **π-烯丙基化学**：软亲核试剂→构型保持；硬亲核试剂（经转金属）→构型翻转。
11. **Suzuki偶联需碱**：碱促进"酸根"配合物形成，加速转金属。

### 4.4 与SₙAr的对比
12. **Buchwald–Hartwig vs SₙAr**：
    - 离去基团：B-H中I > Br > Cl；SₙAr中F > Cl > Br > I
    - 区域要求：B-H无限制；SₙAr需邻对位吸电子基

### 9.1 18电子规则
> "如果一个配合物满足了18电子规则，这就意味着处在配合物中心的金属在价壳层中具有18电子的稀有气体构型，配合物便会是稳定的。"（line ~5289）

> "所谓的铂族金属Ni、Pd和Pt在催化过程中都尤其重要……它们稳定的16电子构型，会通过让配合物采取平面方形几何构型，来得到一个空出的高能轨道。"（line ~5306）

### 9.2 基本反应步骤
> "氧化加成反应……过渡金属的形式氧化态升高2……络合配体的数目同样会增加2。"（line ~5463）

> "若要发生还原消除，两个配体必须彼此顺式。这是因为整个过程是协同的。"（line ~5519）

> "迁移过程是可逆的，由于过程中金属失去了一个配体，因而整个过程会被新配体(L)加成以产出络合饱和配合物的过程所驱动。"（line ~5529）

### 9.3 Heck反应
> "Heck反应可将烯烃与卤代烃或三氟甲磺酸酯R¹-X偶合形成新的烯烃……碱不需要很强……反应非常肯通融。"（line ~5713）

> "β-负氢消除是分子内反应，发生得非常快……钯化学氧化加成反应的底物常是烯基卤、烯丙基卤或芳基卤，绝不会是乙基或异丙基卤。"（line ~5709）

### 9.4 交叉偶联
> "卤代烃组分(R²X)必须小心选择，因为β-负氢消除会在慢的转金属步骤发生前分解第一个中间体。"（line ~6069）

> "转金属反应往往是这些偶合反应中的慢步骤。"（line ~6165，Stille羰基化）

### 9.5 π-烯丙基化学
> "反应通常伴随在反应中心的构型保持发生。和构型保持的Sₙ2取代一样……这事实上是两次翻转所导致的。"（line ~6359）

> "注：构型保持的产物只会在使用软亲核试剂时得到；如果使用格氏试剂等有机金属做亲核试剂……总体上得到构型翻转产物。"（line ~6376）

### 9.6 Buchwald-Hartwig
> "把这些反应中的胺看作'亲核试剂'非常诱人，但是，很明显在这里亲核性起到的作用很少，扮演钯的配体的能力才是重要的。"（line ~6531）

## 十、竞赛题型线索

### 10.1 机理书写题
- Pd催化偶联的完整催化循环（氧化加成→转金属→还原消除）。
- Heck反应的催化循环（含β-负氢消除和Pd(0)再生）。
- π-烯丙基钯化学的催化循环（两次翻转 = 构型保持）。
- Wacker氧化的机理（钯氧化→β-消除→烯醇→酮）。

### 10.2 产物预测题
- 给定烯烃和卤代烃预测Heck反应产物（注意区域选择性）。
- 给定硼酸和卤代烃预测Suzuki偶联产物。
- 给定烯丙基乙酸酯和亲核试剂预测Tsuji-Trost产物（注意构型保持/翻转）。
- 给定末端烯烃预测Wacker氧化产物（甲基酮）。

- 利用Heck反应构建共轭烯酮。
- 利用Suzuki/Stille偶联立体专一性地构建双烯。

- 利用π-烯丙基钯化学进行烯丙基化（1,4-加成）。

### 10.4 电子计数题
- 给定配合物计算电子数并判断稳定性。
- 确定金属氧化态。

## 十二、存疑或待验证之处

1. **Heck反应的区域选择性**：缺电子烯烃远端进攻、富电子烯烃近端进攻的规律，在复杂底物中是否总是成立？
2. **Suzuki偶联中"酸根"配合物的具体结构**：原文描述较为模糊，转金属的详细机理有待进一步确认。
3. **π-烯丙基钯配合物的真实电子结构**：两种画法（Pd⁺/中性烯丙基 vs Pd/烯丙基阳离子）在机理推导中是否完全等价？
4. **Wacker氧化中CuCl₂的作用**：CuCl₂将Pd(0)氧化为Pd(II)，自身被O₂氧化回Cu(II)——这一共氧化循环的化学计量关系。
5. **Pd(II)到Pd(0)的原位还原**：不同还原剂（胺、膦、烯烃、有机金属）的还原机理差异。
