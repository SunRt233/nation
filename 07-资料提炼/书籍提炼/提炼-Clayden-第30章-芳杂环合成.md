## 3. 核心合成策略（三种主要方法）

### 3.1 离子型环化反应（最常用）

**核心思路**：杂原子（O/N/S）作亲核试剂，羰基碳作亲电试剂，环化构建芳杂环。

| 目标环 | 前体 | 条件/试剂 | 备注 |
|:---|:---|:---|:---|
| **吡咯** | 1,4-二酮 + NH₃/RNH₂ | 酸或加热 | 两侧 C-N 切断 |
| **呋喃** | 1,4-二酮 | 酸催化脱水 | 氧已在分子内 |
| **噻吩** | 1,4-二酮 | P₂S₅ / Lawesson试剂 | C=O → C=S |
| **吡啶** | 1,5-二酮 + NH₃ | 氧化剂（HNO₃/DDQ） | 先成二氢吡啶再氧化 |
| **哒嗪** | 1,4-二酮/酮酯 + 肼 | Br₂脱氢芳构化 | 两个相邻N |
| **吡唑** | 1,3-二酮 + 肼 | 直接芳香性 | 无需氧化 |
| **嘧啶** | 1,3-二羰基 + 脒/胍 | 环化脱水 | 两个分开的N |
| **噻唑** | 硫代酰胺 + α-卤代酮 | 室温 | N进攻酮(硬)，S进攻卤代烃(软) |

> **关键控制**：碱催化 → 羟醛产物（环戊烯）；酸催化 → 呋喃

### 3.3 环修饰（已有环的官能团化）

| 反应类型 | 适用杂环 | 特点 |
|:---|:---|:---|
| **芳香亲电取代** | 吡咯、噻吩、呋喃、吲哚 | 富电子环，2,5-位活性高 |
| **芳香亲核取代** | 吡啶、喹啉、哒嗪、嘧啶 | 缺电子环，N邻/对位有利；常需先氯化(POCl₃) |
| **杂原子邻位锂化** | 噻吩、呋喃 | 强碱(n-BuLi)在杂原子邻位去质子 |

## 4. 重要人名反应详录

### 4.1 Knorr 吡咯合成法

- **试剂**：β-酮酯 + 亚硝酸钠 → 肟 → 还原(Zn/AcOH)原位生成胺 → 与另一分子β-酮酯环化
- **机理**：烯胺形成 → 分子内缩合 → 脱水芳构化
- **特点**：经典的一锅法，产率高

![[mineru/中文版clayden-chinese30-33章787-907_images/c34c1e528a30e3a89d0d2cbf5416e37b68ff54b7988e45fcf45249c968c8f4ad.jpg]]
*Knorr吡咯合成：肟还原+胺与酮二酯环化*

### 4.2 Hantzsch 吡啶合成法

- **试剂**：2分子β-酮酯 + 醛 + 氨
- **产物**：1,4-二氢吡啶 → 氧化 → 吡啶
- **机理**：羟醛缩合 → Michael加成 → 氨进攻羰基 → 亚胺环化
- **应用**：钙通道阻滞剂（非洛地平、氨氯地平）

![[mineru/中文版clayden-chinese30-33章787-907_images/d6ec56315b88f998b347e790a66b768b91279b2cf28588c84c24d8ad7638f69b.jpg]]
*Hantzsch四组分反应*

![[mineru/中文版clayden-chinese30-33章787-907_images/2a639ac009dfe896fe47e411473ebe66667ec47230dd76571f12da2df29da2ce.jpg]]
*Hantzsch吡啶合成交互机理*

### 4.3 Fischer 吲哚合成法（核心）

- **试剂**：苯肼 + 醛/酮 + 酸（HOAc/PPA/ZnCl₂）
- **机理四步**：
  1. 苯腙形成（可分离）
  2. 腙→烯胺异构化
  3. **[3,3]-σ重排**（关键步骤：N-N键断裂，C-C键形成）
  4. 缩醛胺分解 → 逐出NH₃ → 芳香吲哚

![[mineru/中文版clayden-chinese30-33章787-907_images/b58562d1e60b0914825134cbe9cca5d13b0674861899170d6b60734983fbe6ce.jpg]]
*Fischer机理：[3,3]-σ重排*

![[mineru/中文版clayden-chinese30-33章787-907_images/d7b20c406d02459bcfd1fef5dba1f671497f1054a784621d3095e18c3fa989f5.jpg]]
*Fischer吲哚合成交互机理*

- **区域控制**：
  - 醛（单侧烯醇化）→ 明确产物
  - 对称二酮 → 单一苯腙
  - 苯环单邻位空着 → 环化位置确定

### 4.4 Skraup 喹啉合成法

- **经典条件**：苯胺 + 甘油 + 浓H₂SO₄ + 硝基苯（氧化剂），>100°C
- **现代改良**：先制备共轭加合物 → 酸环化 → DDQ氧化
- **机理**：甘油脱水→丙烯醛 → 共轭加成 → 环化 → 二氢喹啉 → 氧化芳构化

## 6. 例题提取

### 例题 1：吡咯合成切断分析
**题目**：设计以简单原料合成 2,5-二甲基吡咯的路线。
**答案**：2,5-己二酮 + 氨（或乙酸铵）→ 加热环化。切断：两侧 C-N 键 → 1,4-二酮 + NH₃。

### 例题 2：Fischer 吲哚的区域选择性
**题目**：环己酮与苯肼在酸催化下发生 Fischer 吲哚合成，产物是什么？若改用 2-甲基环己酮呢？
**答案**：环己酮 → 对称 → 单一产物（四氢咔唑）；2-甲基环己酮 → 不对称 → 可能两种产物，但烯醇化方向由酸/碱控制。

### 例题 3：1,3-偶极环加成区域选择性
**题目**：氧化腈与不对称炔烃（如甲基丙炔酸酯）发生环加成，预测主要区域异构体。
**答案**：HOMO-LUMO 控制，缺电子炔烃的β-碳与氧化腈的O端结合，得3-取代异噁唑-5-甲酸酯为主。

### 例题 4：Hantzsch 吡啶合成的组分计数
**题目**：Hantzsch 吡啶合成需要哪四种组分？写出简化的反应方程式。
**答案**：2分子β-酮酯 + 1分子醛 + 1分子氨 → 1,4-二氢吡啶 → [O] → 吡啶。

## 7. 超纲标注

- **Viagra 完整合成**：竞赛级别通常不需要掌握完整药物合成，但**逆合成切断思路**值得学习
- **现代 Skraup 改良**：使用 DDQ 等现代氧化剂，传统竞赛可能仍侧重经典条件
- **Click 化学**：铜催化叠氮-炔环加成属于前沿方法，国初/决赛可能涉及概念

## 8. 踩坑回流

| 坑点 | 说明 |
|:---|:---|
| **混淆吡唑和咪唑** | 吡唑：1,2-位两个N；咪唑：1,3-位两个N。合成前体不同（吡唑用肼+1,3-二酮，咪唑用α-卤代酮+脒） |
| **Hantzsch产物氧化遗漏** | 直接产物是二氢吡啶，必须氧化才能得芳香吡啶 |
| **Fischer重排方向** | [3,3]-σ重排后，原苯环的邻位变为吲哚的C2/C3侧链，注意编号对应 |
| **噻唑选择性** | N进攻酮羰基（硬-硬），S进攻卤代烃（软-软），不能互换 |
