## 一、这讲真正提供了什么

第5讲 `金属有机化学` 是初赛讲义中与第4讲（配合物）构成"无机化学→金属有机"连贯线的讲次。它篇幅中等（~480行正文 + ~100行习题），但概念密度高——三个核心工具各压缩在一个小节里，且每个工具都配有竞赛真题级例题。

现有知识库中已有 [[18电子规则]]、[[过渡金属催化基础]] 等KP，但第5讲提供了四样现有系统深度不够或完全缺失的东西：

### 1. EAN规则的完整"为什么"（§5.1）

现有 KB 有 18 电子规则的定义，但第5讲提供了：
- **MO 层面的解释**：为什么八面体+强场是前提——π受体配体拉低 t₂g → 填满 6 个 t₂g 电子最稳定 → 18e；弱场下 t₂g 非键性质强 → 不必须填满 → 可 <18e
- **16 电子规则**：平面正方形（D₄h）的 ML₄ MO 图 → 最高占据轨道与最低空轨道间隙大 → 16e 最稳定（铂系配合物的解释）
- **Zeise 盐作为历史起点**：K[PtCl₃(C₂H₄)]·H₂O 的合成、结构、反馈 π 键（d→π*乙烯）——这是整个金属有机化学的起源案例

### 2. 等瓣类似（Isolobal Analogy）（§5.1 例5.4）

这是 KB 中**完全缺失**的结构预测工具：
- **核心思想**：金属有机"分子片"的前线轨道与简单有机基团对应——Mn(CO)₅（17e）≈ CH₃（缺 1e 满壳）→ 两者成键行为相同
- **应用链**：Ir(CO)₃（15e）≈ CH（缺 3e）→ Ir₄(CO)₁₂ = 四面体（类比 (CH)₄ = 四面体烷骨架）
- **等瓣拆解**：Os₅(CO)₁₉ = [Os(CO)₃]·4[Os(CO)₄] ≈ C·4CH₂ → 螺环结构

### 3. 系统性电子计数规则（§5.2）

现有 KB 的电子计数偏零散。第5讲给出了**6条统一规则**：
1. 中性配体（CO、PPh₃）→ 各 2e
2. 卤阴离子配位 → 各 1e
3. 共价键按键数计数（=O → 2e、≡N → 3e、M-M → 每键 1e）
4. 螯合配体补上螯合原子电子（AcO⁻ 螯合 → 3e）
5. NO 特殊（通常 3e 给体，视条件可变）
6. 桥连配体分边计数（CO 桥 → 每侧 1e；CH₃ 桥 → 仅一侧 1e）

加上 ηⁿ 记法（η = hapticity，n 个原子与金属键连）的电子计数实操。

### 4. 配位催化基元反应 + 工业案例（§5.3）

这是第5讲最"有机化学友好"的部分——把金属有机催化拆解为可识别的基元步骤：

| 基元反应 | 正向 | 逆向 | 金属变化 |
|:---|:---|:---|:---|
| **配位/解离** | 配体配位 | 配体解离 | 电子数 +2 / −2 |
| **插入/消除** | 迁移插入（H/烷基迁移到配位烯烃/CO） | β-消除 | 氧化态不变 |
| **氧化加成/还原消除** | 金属氧化态 +2，两个新配体进入 | 金属氧化态 −2，两个配体偶联离去 | 氧化态 ±2 |

**三个完整工业催化循环案例**：
1. **氢化甲酰化**（HCo(CO)₄ 催化）：解离→配位→H 迁移插入→CO 插入→H₂ 氧化加成→还原消除
2. **Wacker 过程**（PdCl₂-CuCl₂）：乙烯→乙醛，含配位→H₂O 促进插入→β-消除
3. **Monsanto 乙酸合成**（[Rh(CO)₂I₂]⁻ + CH₃I）：氧化加成→CO 插入→还原消除→水解；以及 Cativa 法（Ir 版）的面式/经式异构体分析

### 4.3 对 [[金属多重键]] 的直接帮助（🟡 中增量）

现有KP已有四重键结论，但第5讲提供：
- **δ 键的完整 MO 推导**：两个金属沿 z 轴靠近 → d_{z²}→σ、d_{xz}/d_{yz}→π、d_{xy}/d_{x²-y²}→δ
- **σ > π > δ 的能级顺序**：MO 排布 σ π π δ δ δ* δ* π* π* σ*
- **五重键案例**：Ar'CrCrAr' → 键级 = [10−(5×2−10)]/2 = 5

### 4.4 竞赛解题方法的贡献

第5讲提供了金属有机推断题的三种典型推理路径：
1. **EAN反推法**：给质量分数 + EAN约束 → 反推分子式（如例5.1 Zeise盐、例5.16 光催化Pt配合物）
2. **等瓣替换法**：陌生簇合物 → 等瓣为简单有机骨架 → 推出结构（如例5.4 Ir₄(CO)₁₂、Os₅(CO)₁₉）
3. **催化循环填空法**：给循环图 + 已知反应类型 → 推中间体结构 + 氧化态/电子数（如例5.15 Monsanto、例5.16 光催化）

## 六、缺页桥页检查

**需要新建 1 个紧凑桥页：[[等瓣类似]]（Isolobal Analogy）**

原因：
- 等瓣类似是 Hoffman 诺贝尔奖级贡献，是理解金属簇合物结构的核心工具
- 现有 KB 中完全没有这个概念
- 第5讲提供了完整的定义、推导和应用案例，可直接压缩为一个紧凑桥页

建议桥页内容：
- 定义：两个分子片的前线轨道（数目、对称性、能量、电子数）相似 → 成键行为相同
- 经典对应：Mn(CO)₅（17e）≈ CH₃、Ir(CO)₃（15e）≈ CH、Fe(CO)₄（16e）≈ CH₂
- 应用：簇合物结构预测（等瓣替换→简单有机骨架→推出几何）
- 实例：Ir₄(CO)₁₂（四面体）、Os₅(CO)₁₉（螺环）
