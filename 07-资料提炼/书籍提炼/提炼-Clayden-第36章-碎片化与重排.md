# 提炼：Clayden 第36章——碎片化与重排（Fragmentation and Rearrangement）

---

## 一、核心概念与定义

### 1.1 参与（Participation）
- **邻基参与（Neighbouring Group Participation, NGP）**：与反应中心相邻的基团通过分子内亲核进攻形成环状中间体，显著加速反应速率。
- **Anchimeric Assistance**：邻基参与的同义词（希腊文 anchi = 邻近的；mer = 部分）。
- 邻基参与的共同特征：**环状中间体的形成**（三元环至五元环最常见）。
- 邻基只有在**能加速反应时**才会参与——否则反应走普通 SN2 路径。

### 1.2 重排（Rearrangement）
- **定义**：分子骨架发生重组，一个原子或基团从分子中的一个位置迁移到另一个位置的反应。
- **与参与的关系**：当参与基最终与另一个原子成键时，重排得以发生。
- **驱动力**：生成更稳定的中间体/产物（更稳定的碳正离子、释放环张力、形成羰基、恢复芳香性等）。
- **迁移起点（migration origin）**与**迁移终点（migration terminus）**：迁移基团从起点移动到终点。

### 1.3 碎片化（Fragmentation）
- **定义**：通过 C-C 键断裂将分子切为碎片的反应。
- **核心要求**：需要**电子源（推）**和**电子槽（拉）**——极化帮助 C-C 键断裂。
- **1,2,3,4 碎片化模式**：电子源（1）→ C2-C3 键断裂 → 电子槽（4，离去基）。
- 碎片化受**立体化学严格控制**——需要反叠式（anti-periplanar）几何。

### 1.4 关键区分
| 概念 | 特征 | 关系 |
|:---|:---|:---|
| **参与** | 邻基形成环状中间体，可能不改变骨架 | 重排和碎片化的前奏 |
| **重排** | 骨架重组，原子迁移到新位置 | 参与的"特例"——邻基最终落在其他位置 |
| **碎片化** | C-C 键断裂，分子切成碎片 | 与重排竞争，当电子源和电子槽相间而非相邻时发生 |

## 二、重要反应与机理

### 2.1 邻基参与的立体化学证据

#### 2.1.1 构型保持 = 邻基参与的标志
- 饱和碳上立体化学保持的取代反应 → 着眼邻基参与！
- 两次连续的 SN2 = 构型保持（第一次邻基进攻翻转，第二次亲核试剂进攻再翻转）。

#### 2.1.2 环己烷体系的 anti/syn 差异（line ~3137）
- **anti 化合物**：环翻转后乙酸酯基处在参与 OTs 离去的理想位置 → 形成对称中间体 → 构型保持。
- **syn 化合物**：乙酸酯基无法抵达 OTs C-O 键的 σ* 轨道 → 无邻基参与 → 普通 SN2 → 构型翻转 + 反应缓慢。

#### 2.1.3 光学纯 (R)-2-溴代丙酸的水解（line ~3195）
- 浓 NaOH：SN2 → (S)-乳酸（翻转）。
- Ag₂O + 低浓度 NaOH：邻基参与（羧基形成三元环内酯中间体）→ (R)-乳酸（保持）。

### 2.2 可参与的基团（line ~3231）

| 基团 | 参与能力 | 形成的环 | 备注 |
|:---|:---|:---|:---|
| **硫醚（SR）** | 强 | 三元环锍离子 | 比醚好得多（快 600 倍 vs n-PrCl）|
| **醚（OR）** | 中等/弱 | 三元环氧鎓 | OMe 负电性减慢反应；远端 OMe（五元环）快 4000 倍 |
| **酯（OAc）** | 强 | 三元环 | 本章主要例子 |
| **羧基（COOH/COO⁻）** | 强（条件合适时）| 三元环内酯 | 需 Ag⁺ 协助离去基离去 |
| **胺（NR₂）** | 强 | 三元环氮丙啶鎓 | Payne 重排等 |
| **C=C（π 键）** | 强 | 三元环/非经典碳正离子 | 快 10¹¹ 倍于饱和类似物 |
| **苯基（Ph）** | 强 | 苯鎓离子（phenonium ion）| 分子内 Friedel-Crafts 烷基化 |

- **环大小规律**：三元和五元环形成最快；六元环较少；四元或七元以上非常稀少。

### 2.3 碳正离子重排——Wagner-Meerwein 重排（line ~3681）

#### 2.3.1 基本特征
- **定义**：包含 H 或烷基迁移的碳正离子重排。
- **驱动力**：生成更稳定的碳正离子（伯 → 仲 → 叔）。
- **新戊基重排**（line ~3553）：新戊基碘在 Ag⁺/H₂O 条件下，甲基迁移 → 叔碳阳离子 → 水捕获。
- **NMR 证据**：对甲苯磺酸新戊酯在超强酸中直接给出叔碳阳离子光谱，伯阳离子从未被观察到。

#### 2.3.2 分子轨道分析（line ~3661）
- **甲基迁移**：LUMO = 阳离子空 p 轨道；HOMO = 断裂的 C-C σ 键。
- **负氢迁移**：HOMO = C-H σ 键。
- 甲基迁移后若生成伯阳离子则不利；负氢迁移生成叔阳离子 → 热力学控制。

#### 2.3.3 扩环重排（line ~3776）
- 与阳离子相邻的四元环 → 重排为五元环 → 释放环张力。
- 天然产物 α-石竹烯醇的合成应用。

#### 2.3.4 Wagner-Meerwein 重排的判断准则
1. 重排必须很快，使其他反应不能完成。
2. 产物阳离子必须比起始阳离子足够稳定。
3. 捕获阳离子的过程必须可靠。

### 2.4 频哪醇重排（Pinacol Rearrangement）（line ~3854）

#### 2.4.1 机理
- 1,2-二醇在酸中 → 一个 OH 质子化离去 → 碳正离子 → 邻位 C-C 键迁移 → 氧孤对电子稳定正电荷 → 失去质子 → 酮（频哪酮）。
- **"推-拉"机制**：水离去"拉"来迁移基团；氧孤对电子"推"迁移基团。

#### 2.4.2 螺环体系的构建（line ~3897）
- 频哪醇重排可用于形成螺环体系。
- 五元环扩环为六元环的实例。

#### 2.4.3 不对称二醇的选择性（line ~3952）
- 反应倾向于留下更稳定的阳离子。
- 绿色 OH（被两个苯基稳定）比黑色 OH（只被烷基稳定）更容易离去。

### 2.5 半频哪醇重排（Semipinacol Rearrangement）（line ~3976）

#### 2.5.1 定义与特点
- **定义**：离去基不是水，而是其他基团（OTs、卤素、N₂）的频哪醇型重排。
- **优势**：区域选择性更好——通过对甲苯磺酰化空阻较小的 OH 来强制控制离去基。
- **"拉"的来源**：水 → OTs、卤素或 N₂。

#### 2.5.2 典型应用
- **Corey 长叶烯合成**：仲醇在叔醇存在下被对甲苯磺酰化 → 半频哪醇重排 → 扩环酮。
- **2-碘代醇 + AgNO₃**：提高碘的离去性，有利于重排而非环氧形成。
- **佛手柑油烯（bergamotene）合成**：2-碘代醇的半频哪醇重排。

#### 2.5.3 重氮盐的半频哪醇重排 / Tiffeneau-Demjanov 重排（line ~4041）
- 2-氨基醇 → 重氮化 → 烷基重氮盐 → N₂ 离去 → 碳正离子 → 半频哪醇重排。
- **Tiffeneau-Demjanov 重排**：由 2-氨基醇衍生的重氮盐的半频哪醇重排。
- **重氮甲烷扩环**（line ~4076）：环己酮 + CH₂N₂ → 环庚酮（但环戊酮→环己酮会混乱）。

### 2.6 二烯基酮-苯酚重排（Dienone-Phenol Rearrangement）（line ~4088）

- **本质**：频哪醇重排的"逆过程"——由芳香化驱动。
- **驱动力**：质子化的羰基重排为叔碳阳离子 → 迅速失去 H⁺ → 恢复芳香性（苯酚）。
- 雌酮生物合成中的关键步骤。

### 2.7 二苯乙醇酸重排（Benzilic Acid Rearrangement）（line ~4136）

- **底物**：苯偶酰（1,2-二苯基-1,2-乙二酮）+ OH⁻
- **产物**：二苯乙醇酸（benzilic acid）
- **机理**：OH⁻ 进攻羰基 → 四面体中间体 → 芳基迁移（类似半频哪醇重排）→ 断裂 C=O π 键替代离去基。
- **同时具有"推"和"拉"**：一个羰基提供"推"（形成新羰基），另一个羰基提供"拉"（断裂 C=O π 键）。

### 2.8 Favorskii 重排（line ~4174）

#### 2.8.1 机理——环丙酮中间体
- **关键发现**（1944）：两种 α-氯代酮异构体用甲氧基处理给出完全相同的产物 → 经历相同中间体。
- **中间体**：环丙酮（cyclopropanone）。
- **步骤**：碱夺取 α-H → 烯醇负离子 → 分子内 SN2（或氧烯丙基阳离子的对旋电环化）→ 环丙酮 → 亲核试剂进攻 → 开环 → 酯。

#### 2.8.2 开环方向
- 倾向于形成**更稳定的碳负离子**的方向开环。
- 环丙酮和环丁酮非常活泼（角度接近四面体角 108°，而非 sp² 喜欢的 120°）。

#### 2.8.3 环缩小应用
- 环状 2-溴代酮 → Favorskii → 缩环（环己酮 → 环戊基甲酸酯）。
- **立方烷合成**（1964）：两步 Favorskii 重排将五元环缩为四元环。

#### 2.8.4 无法烯醇化的酮的二苯乙醇酸类机理
- 无 α-H 的酮 → 亲核试剂直接进攻羰基 → 类似二苯乙醇酸重排的机理。

### 2.9 Baeyer-Villiger 氧化（line ~4321）

#### 2.9.1 机理
- 过酸（RCO₃H）对羰基加成 → 四面体中间体 → 羧酸根离去与重排协同 → 酯。
- 氧-氧单键非常弱 → 驱动力。

#### 2.9.2 迁移基团优先级（line ~4355）
| R | 迁移产率（%）| 不迁移（%）|
|:---|:---:|:---:|
| Me | 90 | 0 |
| Et | 87 | 6 |
| i-Pr | 33 | 63 |
| t-Bu | 2 | 77 |

- **次序**：叔烷基 > 仲烷基 > Ph > Et > Me（与稳定正电荷能力大致贴合）。

#### 2.9.3 立体化学保持
- 迁移碳的 sp³ 轨道"滑"到新位置 → **立体化学保持**。
- 所有 1,2-迁移中迁移基团都保持立体化学。

#### 2.9.4 与 SN2 的对比
- SN2：使用反键 σ* 轨道（小波节）→ 构型翻转。
- 重排：使用成键 σ 轨道（大波节）→ 构型保持。

### 2.10 Beckmann 重排（line ~4501）

#### 2.10.1 机理
- 肟 OH 被酸转化为离去基 → 水离去 → 烷基迁移到氮上 → 产物阳离子被水捕获 → 酰胺。
- **非环状 Beckmann**：产物阳离子以腈鎓离子（nitrilium ion）形式表达。

#### 2.10.2 反式迁移规则
- **与 OH 处于反式的基团发生迁移**。
- 肟的 E/Z 构型精确决定产物比例。
- 若条件允许异构体平衡 → 电子因素决定（如 Baeyer-Villiger）。

#### 2.10.3 Beckmann 碎片化（line ~4577）
- 当迁移基团能太好地稳定阳离子时（如叔丁基），发生**碎片化**而非迁移。
- 生成**腈 + 碳正离子** → Ritter 反应重组。
- 也被称为"反常 Beckmann"或"二级 Beckmann"——但"Beckmann 碎片化"是更好的名称。

### 2.11 碎片化反应详解（line ~4656）

#### 2.11.1 1,2,3,4 碎片化模式
- **电子源（1）**：孤对电子（OH、OR、NR₂）或 σ 键电子。
- **电子槽（4）**：离去基（H₂O、OTs、N₂ 等）。
- **断裂的键**：2-3 之间的 C-C 键。
- 额外动力：环张力释放（三元环/四元环 C-C 键较弱）。

#### 2.11.2 立体化学控制（line ~4741）
- **反叠式要求**：断裂的 C-C 键必须与离去基处于反叠式（anti-periplanar）。
- 顺式异构体（两个取代基都平伏）→ 可碎片化。
- 反式异构体（OTs 直立）→ 无 C-C 键与 OTs 反叠 → 主要发生 E2 消除。

#### 2.11.3 通过碎片化扩环（line ~4823）
- 6,6-稠合十氢化萘 → 外环是十元环 → 使稠合键作为 2-3 键断裂 → 十元环产物。
- 碎片化对双键几何是**立体专一性**的（类似 E2 消除）。

### 2.12 Eschenmoser 碎片化（line ~4901）

#### 2.12.1 反应
- **底物**：α,β-不饱和酮的环氧 + 对甲苯磺酰肼（TsNHNH₂）。
- **产物**：炔烃 + 酮（在不同位置上）。
- **应用**：扩环（酮炔 → 氢化 → 大环酮）或合成酮炔。

#### 2.12.2 机理
1. 环氧酮 + TsNHNH₂ → 对甲苯磺酰腙。
2. 腙不稳定 → 环氧通过消除开环 → 产生羟基。
3. **1,2,3,4 碎片化**：羟基"推" + Ts⁻/N₂"拉" → 协同断裂。
4. 所有断裂的键彼此平行，在两根双键两侧反叠。

#### 2.12.3 合成应用
- **麝香酮（muscone）/环十五酮（exaltone）合成**。
- **外-西部松小蠹集合素（exo-brevicomin）合成**。
- **保幼激素（juvenile hormone）合成**：两次碎片化控制三根三取代双键的立体化学。

## 三、关键结论与规则

### 3.1 邻基参与
1. **构型保持表明邻基参与的发生**——两次翻转 = 保持。
2. **邻基只有在可以加速反应时才会参与**——否则走普通路径。
3. **三元和五元环的分子内反应通常比分子间反应快**。
4. **参与基团**：硫醚 > 胺 > 酯/羧基 > 醚（需远端）> π 键 > 苯基。

### 3.2 重排
5. **给原子标号**是分析重排反应的第一步。
6. **为了得到更稳定的碳正离子，烷基会发生迁移**（Wagner-Meerwein）。
7. **频哪醇重排是"推-拉"过程**：氧孤对"推"，碳正离子"拉"。
8. **半频哪醇重排比频哪醇重排有更好的区域选择性**——通过选择离去基控制。
9. **在 1,2-迁移中，迁移基团的立体化学保持**。
10. **Beckmann 重排：与 OH 反式的基团迁移**（立体专一性）。

### 3.3 碎片化
11. **碎片化需求推电子和拉电子**——C-C 键需要极化才能断裂。
12. **碎片化受立体化学控制**——反叠式几何是必需的。
13. **碎片化对双键几何是立体专一性的**——类似 E2 消除。

### 3.4 迁移基团优先级
14. **Baeyer-Villiger**：叔 > 仲 > Ph > Et > Me（稳定正电荷能力）。
15. **二烯基酮-苯酚重排**：不迁移的基团对阳离子的稳定能力更重要（CO₂Et 宁愿迁移也不愿邻位留阳离子）。
16. **立体化学同样重要**：迁移基团与离去基团必须反叠（anti-periplanar）。

## 八、摘录的原始段落（附页码/位置标记）

### 8.1 邻基参与的核心段落
> "邻基，只有在它们能提高取代反应的速率时，才会参与进来——除此之外，机理只会遵守普通的 S_N2 路径。"（line ~3229）

> "如果您见到了一个在饱和碳原子立体中心上伴随立体化学保持的取代反应，请着眼于邻基参与！"（line ~3225）

> "不是所有参与都具有孤对电子"——C=C 双键中的 π 电子也可发生参与。（line ~3274）

### 8.2 重排的核心段落
> "当一个基团能从伯碳迁移到仲碳上时，我们说该重排具有一个伯迁移起点和一个仲迁移终点。"（line ~3532）

> "为了得到更稳定的碳正离子，烷基会发生迁移。"（line ~3774）

> "频哪醇重排一个尤其有价值的用途是用于形成螺环体系。"（line ~3897）

> "半频哪醇重排中，仍是羟基'推'迁移基团，而'拉'迁移基团的则由水换成了其他离去基团。"（line ~4008）

### 8.3 碎片化的核心段落
> "碎片化是通过一根 C-C 单键的断裂，导致分子破裂为碎片的反应。"（line ~4657）

> "碎片化需求推电子和拉电子。"（line ~4656）

> "对于反式异构体，在最受欢迎的构象上发生碎片化是不可行的，因为离去基团并不与任何 C-C 键反叠。"（line ~4789）

> "Eschenmoser 碎片化的起始原料是一种 α,β-不饱和酮的环氧。当用对甲苯磺酰肼处理此环氧酮时，碎片化就会发生，引人注目的一件事是产物是一种炔烃。"（line ~4903）

### 8.4 迁移优先级的核心段落
> "次序是，叔烷基是最好的迁移基团，然后是仲烷基，紧跟着的是 Ph，然后是 Et，然后是 Me，这与基团稳定正电荷的能力大致贴合。"（line ~4367，B-V 反应）

> "每种情况中发生迁移的基团都是起始原料中与 OH 处于反式的基团。"（line ~4545，Beckmann）

> "迁移基团与离去基团反叠的要求，在重排反应中是很普遍的。"（line ~4499）

## 九、竞赛题型线索

### 9.1 机理书写题
- 频哪醇/半频哪醇重排的电子推动（注意"推-拉"机制）。
- Beckmann 重排的完整机理（包括腈鎓离子中间体）。
- Favorskii 重排的环丙酮中间体机理。
- Eschenmoser 碎片化的协同机理。

### 9.2 产物预测题
- 给定不对称二醇预测 Pinacol 重排产物（哪个 OH 离去？哪个基团迁移？）。
- 给定肟的构型预测 Beckmann 产物（反式迁移）。
- 给定酮预测 B-V 氧化产物（迁移优先级）。
- 给定环状二醇的非对映体预测碎片化 vs 消除产物（立体化学控制）。

### 9.3 合成设计题
- 利用频哪醇/半频哪醇重排构建螺环或扩环。
- 利用 Beckmann 重排制备内酰胺（尼龙-6）。
- 利用 Eschenmoser 碎片化扩环制备大环酮。
- 利用碎片化控制双键立体化学（如保幼激素合成）。

### 9.4 立体化学分析题
- 判断邻基参与是否发生（基于构型保持/翻转）。
- 判断迁移基团的立体化学保持。
- 判断碎片化产物的双键几何（E/Z）。

## 十、存疑或待验证之处

1. **Favorskii 重排的周环机理**：本章提到烯醇盐失去氯离子给出"氧烯丙基阳离子"，通过对旋电环化关环。这一机理与经典的分子内 SN2 机理的关系需要进一步厘清——竞赛中应以哪种为准？

2. **二苯乙醇酸重排与 Favorskii 重排的区分**：本章强调二者机理不同（C=O 键断裂 vs C-X 键断裂），但无 α-H 的酮的 Favorskii 又遵循二苯乙醇酸类机理。这种"双重机理"在竞赛中的考查深度如何？

3. **Beckmann 碎片化的条件控制**：迁移基团"太好地稳定阳离子"时发生碎片化——这一边界的定量描述不够清晰，竞赛中如何判断？

4. **Eschenmoser 碎片化中的离去基**：本章脚注指出离去基是"对甲苯亚磺酸根（ArSO₂⁻）"而非对甲苯磺酸根——这一细节在竞赛中的重要性？

5. **诺卡酮合成的后续**：本章提到三元环/四元环/六元环碎片化三条路线，但均未给出完整合成——是否需要补充？
