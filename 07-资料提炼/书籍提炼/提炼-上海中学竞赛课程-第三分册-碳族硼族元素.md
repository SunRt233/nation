## 一、来源与定位

### 1.1 材料定位

本讲覆盖 C、Si、Ge、Sn、Pb（碳族）和 B、Al、Ga、In、Tl（硼族），重心在 **碳/硅化学（~45%）和硼/铝化学（~35%）**。特色体现在：

- **碳的同素异形体与石墨层间化合物**——膨胀石墨、氧化石墨的结构-性质关联
- **硅酸盐结构系统分类**——链状/片状/三维网络的硅氧四面体连接方式
- **Ge/Sn/Pb 的氧化还原递变**——Sn(Ⅱ)强还原性、Pb(Ⅳ)强氧化性、惰性电子对效应
- **硼烷的 3c2e 氢桥键**——缺电子化合物的结构特征
- **铝的亲氧性与两性**——铝热反应、氧化膜钝化
- **Ga/In/Tl 的惰性电子对效应**——Tl(Ⅰ)稳定而 Tl(Ⅲ)强氧化性

## 二、知识精讲要点（备课可抽料）

### 2.1 碳的同素异形体与层间化合物（课时1可用）【→学生讲义：碳族与硼族 · 碳同素异形体】

- **石墨结构**：层内 C sp² 杂化，层间范德华力，可导电（离域 π 键）
- **膨胀石墨原理**：天然石墨 + 浓 H₂SO₄ 氧化→氧化石墨（Cₓⁿ⁺(HSO₄)ₙⁿ⁻）→高温加热→层间物挥发膨胀
- **课堂信号点**：膨胀石墨可用于处理石油污染（吸附重油）——**结构决定应用**

### 2.2 碳化物分类（课时1可用）【→学生讲义：碳族与硼族 · 碳化物】

| 类型 | 代表 | 特征 |
|:---|:---|:---|
| 离子型 | CaC₂、Al₄C₃ | 遇水生成 C₂H₂ 或 CH₄ |
| 共价型 | SiC（金刚砂） | 共价晶体，高硬度 |
| 间充型 | WC、MoC | 过渡金属，填充于晶格空隙，高熔点高硬度 |

**课堂对比**：CaC₂ + 2H₂O → Ca(OH)₂ + C₂H₂↑  vs  Al₄C₃ + 12H₂O → 4Al(OH)₃ + 3CH₄↑

### 2.3 碳酸盐溶解性规律（课时1方法可用）【→学生讲义：碳族与硼族 · 碳酸盐】

- 碱金属(除Li)及铵的碳酸盐溶于水
- 难溶碳酸盐 → 酸式碳酸盐溶解度略大（如 CaCO₃ → Ca(HCO₃)₂）
- 易溶碳酸盐 → 酸式碳酸盐溶解度反而小（如 Na₂CO₃ → NaHCO₃），原因是 HCO₃⁻ 有分子间氢键缔合

**课堂方法论**：沉淀溶解对比 → "绝大多数含氧酸盐的酸式盐溶解度 > 正盐，但强碱金属的碳酸盐是反例"

### 2.4 硅与碳的关键差异（课时1可用）【→学生讲义：碳族与硼族 · 硅与碳差异】

| 性质 | C | Si |
|:---|:---|:---|
| 氢化物 | 极多（烷烃） | 仅6种硅烷，Si—Si键弱 |
| 热稳定性 | CH₄ 稳定 | SiH₄ >773K分解 |
| 水解性 | CCl₄ 不水解 | SiCl₄ 遇水强烈水解**
| 还原性 | — | SiH₄ 空气中自燃 |

**核心原因**：Si 有空 3d 轨道，可形成 sp³d 五配位中间体 → SiX₄ 易水解；C 无空轨道，CCl₄ 不水解。

### 2.5 硅酸盐结构系统（课时1深化可用）【→学生讲义：碳族与硼族 · 硅酸盐】

硅酸盐基本单元为 SiO₄⁴⁻ 四面体，连接方式决定结构类型：

| 连接方式 | 结构类型 | 代表矿物 |
|:---|:---|:---|
| 孤立 | 正硅酸根 | 橄榄石 |
| 共用1顶点双聚 | 焦硅酸根 Si₂O₇⁶⁻ | — |
| 共用2顶点成链 | 链状(SiₙO₃ₙ₊₁) | 石棉 |
| 共用3顶点成片 | 片状 | 云母 |
| 共用4顶点成网 | 三维网络 | 沸石分子筛 |

**课堂信号点**：沸石分子筛有均一孔径→可筛分分子→"分子筛"名称由来；A型分子筛晶胞参数1420pm。

### 2.6 Ge/Sn/Pb 的氧化还原递变（课时2可用）【→学生讲义：碳族与硼族 · Ge/Sn/Pb递变】

**惰性电子对效应**（6s²电子难失去）：

| 元素 | 稳定氧化态 | 关键性质 |
|:---|:---|:---|
| Ge | +4 > +2 | Ge(Ⅱ)易歧化 |
| Sn | +4 > +2 | Sn²⁺强还原性（还原 Hg²⁺→Hg₂Cl₂白→Hg黑） |
| Pb | +2 > +4 | PbO₂ 棕黑色强氧化剂（氧化 Mn²⁺→MnO₄⁻） |

**课堂对比实验**：
- SnCl₂ 还原 HgCl₂：白→灰→黑（分步还原，颜色变化直观）
- PbO₂ + HCl → PbCl₂ + Cl₂↑（Pb(Ⅳ)氧化 Cl⁻）

### 2.7 硼烷与 3c2e 键（课时2可用）【→学生讲义：碳族与硼族 · 硼烷与3c2e键】

- B₂H₆（乙硼烷）：4个端基 B—H σ键 + 2个氢桥 3c2e 键
- **课堂关键点**：缺电子原子（B价电子3，价轨道4）→形成多中心键补偿
- 制备：2MnB + 6H⁺ → B₂H₆↑ + 2Mn³⁺（类比 Mg₂Si + HCl → SiH₄）
- 性质：自燃、水解（B₂H₆ + 6H₂O → 2B(OH)₃ + 6H₂）、路易斯酸性

**课堂类比**：B₂H₆的氢桥键 → 有机化学中碳正离子的桥键（跨学科连接）

### 2.8 硼酸的特殊酸性（课时2可用）【→学生讲义：碳族与硼族 · 硼酸酸性】

- **H₃BO₃ 是一元弱酸**（K=5.8×10⁻¹⁰），但酸性源于 **B 接受 OH⁻ 形成 B(OH)₄⁻**，而非自身电离 H⁺
- 加甘油后酸性增强 → 缺电子化合物的配位能力

**课堂方法论贡献**："看原子是否缺电子 → 判断酸性的独特来源"

### 2.9 硼珠反应（课时2/实验可用）【→学生讲义：碳族与硼族 · 硼珠反应】

B₂O₃ 或硼砂与过渡金属氧化物熔融显特征颜色：

| 金属氧化物 | 硼珠颜色 |
|:---|:---|
| CoO | 深蓝色 |
| Cr₂O₃ | 绿色 |
| CuO | 蓝色 |
| MnO | 紫色 |
| NiO | 绿色 |
| Fe₂O₃ | 黄色 |

**课堂用法**：类似于焰色反应，可作为定性鉴定方法

### 2.10 硼与硅的相似性（课时2方法论可用）【→学生讲义：碳族与硼族 · B-Si相似性】

| 对比项目 | 相似性 |
|:---|:---|
| 单质制备 | B₂O₃ + Mg → B；SiO₂ + Mg → Si |
| 与碱反应 | B + NaOH → NaBO₂ + H₂；Si + NaOH → Na₂SiO₃ + H₂ |
| 氢化物制备 | Mg₂B + HCl → B₂H₆；Mg₂Si + HCl → SiH₄ |
| 卤化物水解 | BCl₃ + H₂O → H₃BO₃ + HCl；SiCl₄ + H₂O → H₄SiO₄ + HCl |

**实质**：原子或离子的电场力相近 → 对外层电子的约束力相近

### 2.11 铝的亲氧性与铝热反应（课时2可用）【→学生讲义：碳族与硼族 · Al亲氧性】

- **Al₂O₃ 生成焓 -1669.7 kJ·mol⁻¹** — 比 Fe₂O₃(-824.2)、SiO₂(-910.9) 高得多
- 铝热反应：Fe₂O₃ + 2Al → Al₂O₃ + 2Fe，ΔH=-853.8 kJ·mol⁻¹，温度达3000°C
- **课堂信号点**：野外焊接铁轨（铝热剂的实际应用）

### 2.12 AlCl₃ 的双聚与路易斯酸性（课时2可用）【→学生讲义：碳族与硼族 · AlCl₃双聚】

- 气态 Al₂Cl₆：2个 Al sp³ 杂化，3个共价键 + 1个配位键（双四面体共用一条棱）
- AlCl₃ + Cl⁻ → AlCl₄⁻（路易斯酸接受电子对）

**对比表格**：BF₃ sp² 平面三角形 vs AlCl₃ 双聚体 vs B₂H₆ 氢桥键 → 缺电子化合物的三种补偿方式

### 2.13 Ga/In/Tl 的惰性电子对效应（课时2拓展可用）【→学生讲义：碳族与硼族 · Ga/In/Tl惰性电子对】

- Ga 熔点 29.78°C（在手中熔化），沸点 2403°C（液相范围最大）
- Tl 的 6s² 惰性电子对效应：Tl(Ⅲ)强氧化性，Tl(Ⅰ)稳定
- TlX 与 AgX 相似（难溶，光照分解）；Tl₂SO₄ 与 K₂SO₄ 相似（易溶）

## 三、教学路径参考

### 3.1 与质心讲义的差异

质心元素化学第一轮未覆盖碳族/硼族（只有硫与氮），因此本材料是全新补充，无对比。

### 3.2 课堂路径优化建议

1. **引入碳族时** → 用"CCl₄不水解，SiCl₄遇水冒烟"的反差切入，引出空 d 轨道的核心概念
2. **硼烷部分** → 适合作为"缺电子化合物"的总论案例，与后续 AlCl₃ 双聚体形成对比
3. **惰性电子对效应** → 将 Ge/Sn/Pb + Tl + 氮族的 Bi 串联为一条方法论线索
4. **硼与硅的相似性** → 可作为"对角线规则"的重要补充案例

## 四、例题精选（含解析）

### 例1 [膨胀石墨-结构推断与性质应用] ⭐⭐⭐

**来源**：原书例1

**题干要点**：天然石墨经浓 H₂SO₄ 氧化得氧化石墨（Cₓⁿ⁺(HSO₄)ₙⁿ⁻），高温加热后 H₂SO₄ 等挥发使石墨膨胀，得膨胀石墨。氧化石墨 A 有烯醇式和酮式的互变异构。

**教学价值**：⭐⭐⭐⭐
- 结构-性质关联的典型应用
- 互变异构概念在无机材料中的体现
- 实际应用（石油污染处理、金属腐蚀）

**课堂使用建议**：适合课时1碳化学部分，作为"碳的同素异形体"后的拓展案例。建议用时 8 min。

**关键教学节点**：
1. 氧化石墨与膨胀石墨的关系——本质是层间插入+高温剥离
2. 化学式推导：从结构图数 C/H/O 原子
3. 膨胀石墨吸附石油的原理（多孔结构+大比表面积）
4. 金属腐蚀的电化学解释（石墨作惰性电极，杂质作电解质）

---

### 例2 [沸石分子筛-晶体结构抽象] ⭐⭐⭐⭐

**来源**：原书例2（2006全国决赛）

**题干要点**：沸石 β 笼（切顶八面体，6个正方形+8个正六边形），通过六方柱笼连接。将 β 笼看作"原子"，则结构可简化为金刚石型。

**教学价值**：⭐⭐⭐⭐
- **抽象思维训练**：复杂结构→简化模型
- 硅酸盐结构的具体应用
- 晶体化学的综合素材

**课堂使用建议**：适合课时1硅酸盐讲完后作为拔高题/课后思考。难度较高（决赛级），建议选做。

**关键推理链**：
1. 切顶八面体→6个正方形+8个正六边形（柏拉图多面体变体）
2. 抽象化：β笼→原子，六方柱笼→化学键
3. 金刚石型 = 四面体连接
4. 类似氯化钠排列→晶胞含 8 个 β笼

---

### 例3 [BN 纳米管-等电子体与类比推断] ⭐⭐⭐⭐⭐

**来源**：原书例3（2003全国决赛）

**题干要点**：气体 A 与气体 B 相遇生成白色晶体 C。A 密度约 B 的 4 倍。B 通入 CO₂ 得白色 D。C 高温分解得 E₁，E₁ 高温高压得 E₂。E₁ 和 E₂ 的晶体结构分别与碳的两种同素异形体相似。

**教学价值**：⭐⭐⭐⭐⭐（极高）
- 等电子体原理的灵活运用（BN 与 C₂ 是等电子体）
- 定性推断 + 结构类比
- **BN 的两种晶型**：六方 BN（类石墨，润滑剂）vs 立方 BN（类金刚石，磨料）

**课堂使用建议**：适合课时2作为"综合推断训练"或课后思考题。建议用时 10 min。

**关键推理链**：
1. A + B → 白色晶体 → 类 NH₃ + HCl
2. 密度比 → A 为 BF₃，B 为 NH₃
3. D 为 NH₄HCO₃
4. C 为 F₃BNH₃ → 高温分解 → (BN)ₓ
5. BN 是 C₂ 的等电子体 → 类石墨和类金刚石结构

---

### 例4 [Na-Pb 合金-液氨溶液与 Zintl 相] ⭐⭐⭐⭐

**来源**：原书例4（1992全国初赛改编）

**题干要点**：Na 和 Pb 以 2:5 摩尔比合金溶于液氨得绿色溶液 A，电解在阳极析出铅、阴极析出钠。A 中正负离子个数比 4:1。用 PbI₂ 的液氨溶液滴定 A，绿色褪去，铅全部析出。

**教学价值**：⭐⭐⭐⭐
- Zintl 相/聚阴离子的存在（Pb₉⁴⁻）
- 定量推断 + 电解 + 滴定的综合训练
- 晶体结构（单斜晶系，K₄Pb₉）

**课堂使用建议**：决赛级别难度，适合课后选做。

**核心知识点**：
1. Na₄Pb₉ 的确定（质量比 9.44:1 + 电荷比 4:1）
2. 滴定反应：2Pb²⁺ + Pb₉⁴⁻ → 11Pb
3. 晶胞参数与密度计算

## 六、对应备课对接建议

1. **硅与碳的对比 + 硅烷的性质差异**（§2.4）—— 这是学生最容易混淆的知识点，建议融入课时1认知台阶
2. **惰性电子对效应**（§2.6 + §5.3）—— 可跨 Ge/Sn/Pb + Bi + Tl 统一讲解，建方法论卡片
3. **硼的缺电子化合物**（§2.7-2.8）—— 3c2e键是竞赛高频考点，课堂必须有结构图
4. **铝的亲氧性 + 铝热反应**（§2.11）—— 数值冲击（ΔH=-853.8 kJ·mol⁻¹），课堂效果极佳
5. **对角线规则补充（B-Si）**（§2.10）—— 现有知识库若缺此内容则高价值补充
