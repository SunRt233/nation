## 一、这讲真正提供了什么

这讲表面上是在讲“方程式怎么写”，但它的真实价值并不是把配平再讲一遍，而是把初赛方程式题拆成一套更像**推断题技术树**的系统。

和常规课内口径相比，这一讲的提升主要在：

1. 不把方程式题只当“守恒配平”
2. 强调“产物猜测之前先看体系图景”
3. 把酸碱性、配位、沉淀、水解、歧化、光化学、存在形体都纳入同一问题
4. 把“有数据”和“没数据”的题分成两套打法
5. 把很多元素化学题、推断题、有机边缘题统一收进“方程式书写”视角

一句话说，这讲解决的是：

> **题目没有直接告诉你产物时，怎样用化学语言把反应写出来。**

## 三、第1讲的核心技法树

### 3.1 第一层：先判断这是哪类方程式题

拿到题后，先不要立刻配平，先分类：

1. **纯配平题**
   - 已知反应骨架
   - 只是系数难定
   - 核心工具是 `叠加法`
2. **存在形体题**
   - 产物大致知道
   - 但溶液中的真实形态不清楚
   - 核心工具是 `酸碱 / 配位 / 沉淀 / 水解判断`
3. **图景推断题**
   - 题目提示少
   - 需要先判断最可能的反应逻辑
   - 核心工具是 `化学图景`
4. **数据反推题**
   - 给质量分数、摩尔比、失重、平均式量、元素分析
   - 核心工具是 `设未知量 + 代数约束`
5. **极简信息题**
   - 条件看似不够
   - 实则默认最基本反应或最简单产物
   - 核心工具是 `简单处理`

### 3.2 第二层：再选本题的主工具

本讲给出的工具不是并列堆砌，而是有优先级的：

1. 先看能不能**拆反应**
2. 再看要不要**改物种存在形式**
3. 再看是否必须**先讲清反应图景**
4. 再看是否要**借数据反推**
5. 最后才考虑用“机理与配凑”硬推

### 3.3 第三层：最后才进入守恒配平

这一讲最重要的纠偏是：

> 很多初赛方程式题错，不是错在配平，而是前面的反应骨架、存在形体和化学图景就判错了。

## 四、分节提炼

### 4.1 `§1.1 叠加法`

这一节是全讲最基础、也是最容易马上变成课堂工具的部分。

#### 核心观点

1. 化学方程式本质上也是“等式”，可以进行形式叠加。
2. 当一个反应骨架的未知系数数目多于独立守恒方程数目时，单纯待定系数法往往不够，需要拆成若干更简单反应。
3. 叠加法不仅是配平工具，也是**识别反应由哪些子过程组成**的思维方法。

#### 这节真正教会学生的判断标准

- 若 `元素守恒方程数 + 1 = 未知系数数`，往往可唯一配平
- 若 `元素守恒方程数 + 1 < 未知系数数`，通常需要拆反应
- 若 `元素守恒方程数 + 1 > 未知系数数` 仍难配平，则要怀疑反应骨架错了

#### 代表性例题

- `例1.1`：`BrF3 + H2O`
  - 本质不是“难配平”
  - 而是同时存在：
    - `BrF3 -> HBrO3 + Br2`
    - `BrF3 + H2O -> Br2 + O2`
  - 通过分解成两个简单过程，再叠加得到通解

#### 对备课系统的直接价值

这部分内容可以直接回流到：

- [[专题-方程式书写专项（元素化学版）]]
  - 增补“什么时候必须拆反应”的判断按钮
- 元素化学课堂
  - 补“不是所有题都能一把待定系数”

### 4.2 `§1.2 存在形体的判断`

这一节是全讲最有“竞赛味”的一层，也是对现有系统最值得补强的部分之一。

#### 核心观点

方程式题不是只看元素守恒，而是要写出**题目条件下真实存在的主要物种**。

常见误判点有：

1. 酸碱性错
2. 水解产物没改写
3. 配合物先解离还是先参与反应没想清
4. 沉淀是否继续被氧化/配位/歧化没有继续跟
5. 有机部分是否需要写质子化产物没判断

#### 这节提供的四个课堂按钮

1. **先看介质**
   - 强酸、弱酸、弱碱、强碱、非水体系
2. **再看中心粒子高价是否会水解**
   - 例如 `Ti(IV)` 写成 `TiO^{2+}`
3. **再看生成的酸碱是否会继续反应**
   - 例如 `OH^-` 会不会被缓冲体系消掉
4. **最后看配合物 / 沉淀是否会改变主物种**

#### 代表性例题

- `例1.5`
  - 重点不是氧化还原本身
  - 而是弱碱性条件下 `OH^-` 不能粗暴并列，要进一步改写到 `HSO3^- / SO3^{2-}`
- `例1.6`
  - 重点是根据环境决定是否写 `H^+ / OH^-`
  - 以及 `FeO(OH)`、`H2MoO4` 这类实际存在物
- `例1.7`
  - 重点是配合物、氧化、沉淀连续发生
- `例1.8`
  - 有机边缘题进入无机方程式口径：胺应写成质子化形式

  - 增补“存在形体判断”作为独立子模块

### 4.3 `§1.3 化学图景`

这一节把“方程式题”真正推向了“推断题”。

#### 核心观点

当题目提示少、反应陌生时，最忌讳直接瞎猜产物；应该先判断：

1. 哪个物质最可能被氧化 / 还原 / 歧化
2. 哪个物质更像促进反应的驱动力，而不是主反应物
3. 哪个已知经验能解释题目现象

#### 这节最有价值的思维动作

1. 从“颜色 / 沉淀 / 气体 / 光照 / 溶剂”推反应类型
2. 从常见通性出发：
   - 非金属在碱性下易歧化
   - 某些光敏配合物会发生光还原
   - 复合氧化物可形式拆成简单组分
3. 先看“谁在推动反应”，再看“谁是最终主产物”

#### 代表性例题

- `例1.15`
  - `Ag2SO4` 与 `S`
  - 关键不是一眼猜产物，而是先看：
    - 硫单质歧化
    - `Ag2SO4` 通过沉淀硫离子来拉动反应
- `例1.16`
  - 蓝图生成
  - 关键不是背反应，而是：
    - 先找光敏组分
    - 再判断怎么生成 `Fe(II)`
    - 最后才得到普鲁士蓝
- `例1.17`
  - 镁硼石 + 碳酸氢铵
  - 通过把“复合氧化物”拆成形式组分，迅速确定反应图景
- `例1.18`
  - `ClO3F` 与 `PhLi`
  - 专门提醒学生不要被“氟化剂”三个字带偏

#### 对备课系统的直接价值

这一节最适合回流到：

- [[专题-情境化元素推断综合训练]]
  - 增补“先看图景再写方程式”的分流板
- [[专题-元素化学深度与结构推断综合]]

### 4.4 `§1.4 机理与配凑`

这一节是困难题打法，适合补高阶课堂层，而不适合作为第一轮普遍要求。

#### 核心观点

1. 当体系太复杂、题目数据又不充分时，可以：
   - 先设未知量
   - 再写出合理的骨架
   - 再用比例、守恒、结构信息和简单机理去收束
2. “机理”在这里不是完整箭头推动，而是指：
   - 哪一类后续变化更合理
   - 哪一类氧化还原 / 歧化 / 水解可能发生

#### 代表性例题

- `例1.25`
  - `NaBH4 + NiCl2`
  - 通过设 `Ni_xB_y`，再结合比例和元素守恒求解
- `例1.26`
  - `P4S3 + OH^-`
  - 通过先拆出骨架裂解，再叠加歧化 / 还原水过程求解
- `例1.27`
  - 甲基橙与 `Cl2`
  - 用简单反应路径去推总方程，而不是凭空猜结果

#### 对备课系统的价值

这部分不适合直接塞进基础班页，但适合：

- 给 [[专题-方程式书写专项（元素化学版）]] 增补“困难题升级口径”
- 给元素深度题、真题讲评页提供高阶思路

### 4.5 `§1.5 借助数据写方程`

这一节是“方程式题”和“推断题”的交界层。

#### 核心观点

给定以下信息时，不要只想配平，要想“反推物种”：

- 质量分数
- 元素分析
- 失重
- 平均式量
- 气体摩尔比
- 反应物投料比

#### 这节的典型策略

1. 先把未知化合物设成尽量一般的形式
2. 用最硬的数据先确定元素个数关系
3. 再用价态、常见结构、化学合理性排除不合理解
4. 最后再回写总方程式

#### 代表性例题

- `例1.32`
  - 四氟铵盐热分解
  - 用 `F` 质量分数 + 生成物比例锁定组成
- `例1.33`
  - `Mn / C / H / N` 元素分析反推中性产物
- `例1.34`
  - 利用摩尔质量和钠质量分数锁定 `SHMP`

#### 对系统的直接价值

这一节可以增强：

- 元素推断题
- 晶体 / 结构题中的组成反推
- 真题讲评中“先算式量再讲化学图景”的步骤

### 4.6 `§1.6 问题的简单处理`

这一节看似最轻，实际上很适合作为课堂上的“止损按钮”。

#### 核心观点

当题目给的信息少得异常时，先考虑：

1. 是否只是最简单的化合、分解、沉淀、置换、酸碱或 Lewis 酸碱反应
2. 是否命题人故意让你别过拟合

#### 代表性例题

- `例1.40`
  - `NaF + NaPO3`
  - 信息很少，但恰恰说明反应不应复杂化

#### 对系统的价值

这一节适合补到：

- 方程式专项页的“考试止损提醒”
- 学生讲义中的“先写最简单合理答案”提示框

## 五、本讲提炼出的统一做题流程

建议把这讲压缩成一张课堂可用流程卡：

1. **先分类**
   - 纯配平 / 存在形体 / 图景推断 / 数据反推 / 简单处理
2. **再搭骨架**
   - 先写出最可能的反应主线
3. **再修物种**
   - 看介质、配位、水解、沉淀、酸碱
4. **再叠加或设未知量**
   - 必要时分步处理
5. **最后检查**
   - 元素守恒
   - 电荷守恒
   - 氧化还原电子守恒
   - 物种存在合理性

这个流程的价值在于：

> 它把“写方程式”变成了一种带判断顺序的题型，而不是纯经验堆砌。
