# 普通化学原理（第4版）知识提炼

> 来源：[[06-外部资料导入/书籍/普通化学原理第四版/普通化学原理第4版 1-200|1-200页]] · [[普通化学原理第4版 201-400|201-400页]] · [[普通化学原理第4版 401-500|401-500页]]
> 本书由华彤文、王颖霞、卞江、陈景祖编著，是中国化学奥林匹克「化学原理」+「无机和结构化学」模块的核心参考教材。
> 全书共16章，已提取第2、4-15章（共13章）为独立提炼笔记。

## 第一轮抽料总口径

> 当前知识库执行口径：**第一轮应覆盖《普通化学原理》里大部分直接服务初赛的主线内容。**
> 因此，这本书在知识库里不只是“参考书”，而是第一轮备课与讲义抽料的主来源之一。

### 第一轮主用章节

| 章节 | 第一轮使用方式 | 备注 |
|:---|:---|:---|

## 备课抽料建议

1. 第一轮备课时，优先从本 MOC 对应章节向下钻取，不要先从零散 KP 反推教材。
2. 若专题属于 `气体 / 溶液 / 热力学 / 平衡 / 动力学 / 酸碱 / 电化学 / 原子结构 / 分子结构 / 晶体 / 配位`，默认先查本书对应章节，再查网课逻辑提炼与专题页。
3. 若专题已经明显进入 `元素化学系统轮 / 分析化学系统轮 / 更深物化推导`，再转向第二轮或第三轮资料主链。
