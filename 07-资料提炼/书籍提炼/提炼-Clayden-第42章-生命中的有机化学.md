---

## 一、讲解脉络

> 作者采用 **"代谢功能 → 核心结构 → 立体化学 → 生物合成逻辑 → 天然产物"** 的递进式讲解。

- **切入前置**：以初级代谢全局图 (葡萄糖 → 丙酮酸 → 柠檬酸循环 → ATP → 氨基酸/蛋白质/核酸) 作为框架，快速定位核酸、蛋白质、糖、脂四大类分子的代谢关系。
- **第一模块 (核酸)**：从核苷酸化学结构出发 → DNA/RNA 碱基配对 (A–T / G–C) → 抗病毒核苷药物 (AZT / 拉米夫定) → 环状核苷 (cAMP) 的立体化学控制。
- **第二模块 (蛋白质)**：氨基酸 → 肽键 (酰胺键刚性) → 蛋白质折叠驱动力 → 胶原/维生素 C / 坏血病 → 抗生素 (青霉素、万古霉素) 对细菌细胞壁糖肽合成的干扰。
- **第三模块 (糖类)**：环状糖 (吡喃/呋喃) → 半缩醛平衡 → 异头碳效应 (anomeric effect) → α/β 异头体 → 缩醛作为保护基 → 糖苷 (O/S/N-糖苷)。**本章对糖类的讲解是全书最系统的一章。**
- **第四模块 (脂类)**：甘油酯、磷脂、胶束 → 皂化反应 → 细胞膜结构。
- **第五模块 (生物化学机理)**：大自然如何在没有高温/强酸/强碱的条件下完成反应？→ NADH/NADPH (天然 NaBH₄) → 吡哆醛介导的还原胺化/转氨/脱羧 → 赖氨酸烯胺 / CoA 硫醇酯 (天然烯醇等价物) → 磷酸烯醇丙酮酸 (PEP) → 柠檬酸合成酶催化的羟醛反应 → 莽草酸途径框架。
- **第六模块 (天然产物)**：萜类、甾体、生物碱、聚酮四大类 → 标记实验 (同位素示踪) 阐明生物合成路径 → Robinson 托品酮合成 → Pictet–Spengler 反应 → 脂肪酸生物合成 (乙酰 CoA + 丙二酰 CoA)。
- **教学风格标签**：机理优先 / 类比驱动 (不断将生物化学机理与实验室有机反应做对比)。
