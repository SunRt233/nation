## 二、知识精讲要点（备课可抽料）

### 2.1 联氨 N₂H₄ 的双重身份精讲（课时2可用）【→学生讲义：氧族与氮族 · 氮化合物】

> 现有备课大纲已将 N₂H₄ 列为**难点**，但原书的深度处理可为课堂提供更清晰的讲解路径。

**碱性**（二元弱碱）：
- 第一步：N₂H₄ + H₂O ⇌ N₂H₅⁺ + OH⁻，Kb₁ = 3.0×10⁻⁶（比 NH₃ 弱）
- 第二步：N₂H₅⁺ + H₂O ⇌ N₂H₆²⁺ + OH⁻（更弱）

**还原性**（火箭燃料）：
- N₂H₄ + 4AgBr → 4Ag + N₂↑ + 4HBr（银镜反应变体）
- N₂H₄ + 2H₂O₂ → N₂↑ + 4H₂O（**火箭推进剂反应，放热剧烈**）

**配位能力**（课堂拓展）：
- Co³⁺ + 6N₂H₄ → [Co(N₂H₄)₆]³⁺
- N₂H₄ 中的 N 有孤对电子，可作配体

**制备方法**（竞赛考点）：
- NaClO 氧化过量 NH₃：NH₃ + ClO⁻ → NH₂Cl + OH⁻（快）；NH₃ + NH₂Cl + OH⁻ → N₂H₄ + Cl⁻ + H₂O（慢）
- **副反应**：2NH₂Cl + N₂H₄ → N₂↑ + 2NH₄⁺ + 2Cl⁻（需加明胶或络合剂抑制）

### 2.2 铵盐热分解系统分类（课时2可用）【→学生讲义：氧族与氮族 · 氮化合物】

> 原书给出了比现有备课大纲更系统的铵盐分解类型，可直接作为课堂总结表：

| 酸根类型 | 分解产物 | 示例 |
|:---|:---|:---|
| 非氧化性挥发性酸 | NH₃↑ + HX↑（无固体残留） | NH₄Cl、NH₄HCO₃ |
| 非氧化性不挥发性酸 | NH₃↑ + 酸式盐/游离酸残留 | (NH₄)₂SO₄、(NH₄)₃PO₄ |
| 氧化性酸 | N₂O↑ 或 N₂↑（NH₃ 被氧化） | NH₄NO₃、(NH₄)₂Cr₂O₇ |

### 2.3 亚硝酸及其盐（课时2补充可用）【→学生讲义：氧族与氮族 · 氮化合物】

> 现有备课大纲未涉及亚硝酸盐，但在元素推断和氧化还原中频繁出现。

- HNO₂：一元弱酸（Ka=7.2×10⁻⁴），极不稳定，2HNO₂ → H₂O + NO↑ + NO₂↑
- NaNO₂/KNO₂ 稳定，水溶液呈碱性
- **氧化性**（为主）：2HNO₂ + 2I⁻ + 2H⁺ → I₂ + 2NO↑ + 2H₂O
- **还原性**（遇强氧化剂）：5KNO₂ + 2KMnO₄ + 3H₂SO₄ → 2MnSO₄ + 5KNO₃ + K₂SO₄ + 3H₂O
- **配位反应**：3K⁺ + Co³⁺ + 6NO₂⁻ → K₃[Co(NO₂)₆]↓（**黄色，鉴定 K⁺ 的经典方法**）
- **安全提醒**：NaNO₂ 有毒，0.3 g 急性中毒，3 g 致死；用于食品发色剂但需控制用量

### 2.4 磷含氧酸完整体系（课时3核心可用）【→学生讲义：氧族与氮族 · 磷化合物】

> 现有备课大纲已覆盖正/焦/偏磷酸的缩合规律，但本材料可补充更完整的 5 种体系。

| 名称 | 分子式 | P 氧化数 | 结构特点 | 几元酸 | 酸性 K₁ |
|:---|:---|:---:|:---|:---:|:---:|
| 次磷酸 | H₃PO₂ | +1 | 2 个 H 直接连 P | 一元 | 1.0×10⁻² |
| 亚磷酸 | H₃PO₃ | +3 | 1 个 H 直接连 P | 二元 | 6.3×10⁻² |
| 正磷酸 | H₃PO₄ | +5 | 所有 O 都与 P 双键 | 三元 | 7.1×10⁻³ |
| 焦磷酸 | H₄P₂O₇ | +5 | 2 个正磷酸脱水，P—O—P 桥 | 四元 | 3.0×10⁻² |
| 偏磷酸 | (HPO₃)ₙ | +5 | 环状缩合 | 一元 | 1.0×10⁻¹ |

**课堂方法论要点**：
1. 判断几元酸的关键：**看有几个 H 与 O 相连**（直接连 P 的 H 不能电离）
2. 酸性递变：正磷酸 < 焦磷酸 < 偏磷酸（缩合程度越高酸性越强）
3. **与硫含氧酸的对比**：同样是高氧化态 → 脱水缩合 → 酸性增强

### 2.5 磷卤化物与卤氧化物（课时3可用）【→学生讲义：氧族与氮族 · 磷化合物】

- **PCl₃**：三角锥，sp³ 杂化，遇水冒烟（PCl₃ + 3H₂O → H₃PO₃ + 3HCl）
- **PCl₅**：气态为三角双锥（sp³d），固态为 [PCl₄]⁺[PCl₆]⁻（离子型）
- **POCl₃**（三氯氧磷）：**课堂催化剂案例**——用于制备磷酸酯、增塑剂、阻燃剂
- **制备对比**：PCl₃ + Cl₂ → PCl₅；PCl₃ + Cl₂ + H₂O → POCl₃ + 2HCl

### 2.6 As/Sb/Bi 的鉴别反应（决赛/拓展可用）【→学生讲义：氧族与氮族 · 砷锑铋】

- **马氏试砷法**：As₂O₃ + Zn + HCl → AsH₃ 导入热玻璃管 → 黑色砷镜
- **古氏试砷法**：AsH₃ 通入 AgNO₃ 溶液 → 黑色 Ag 沉淀（可检出 0.005 mg As₂O₃）
- **Sb 的锑镜法**：类似砷镜但**不能用 NaClO 洗**（需用 HNO₃/H₂SO₄）
- **NaBiO₃ 鉴别 Mn²⁺**：5BiO₃⁻ + 2Mn²⁺ + 14H⁺ → 5Bi³⁺ + 2MnO₄⁻ + 7H₂O（**紫红色出现**）

## 四、例题精选（含解析）

### 例1 [NF₃ 极性异常-分子结构综合] ⭐⭐⭐⭐

**来源**：原书例1（2004 初赛 · Cu 催化 NH₃ + F₂）

**题干要点**：Cu 催化下 NH₃ 与 F₂ 反应得铵盐 + 三角锥分子 A（键角 102°，偶极矩 0.78×10⁻³⁰ C·m，对比 NH₃ 键角 107.3°，偶极矩 4.74×10⁻³⁰ C·m）。

**教学价值**：⭐⭐⭐⭐⭐（极高）
- **N—F 键偶极方向与孤对电子偶极方向相反** → 分子极性意外变小
- 从"结构→偶极→反应活性"完整推理链
- NF₄⁺ 正离子的水解产物可调（O₂ vs H₂O₂ 比例调控）
- **课堂核心收获**：直观感受分子极性的"矢量和"概念，打破"电负性大→分子极性大"的朴素认知

**课堂使用建议**：适合课时2中段，在讲完 NH₃ 结构后作为 "反直觉案例" 引人入胜。建议用时 10 min。

**关键教学节点**：
1. 从反应物(NH₃ + F₂)推产物(NH₄F + NF₃)——氧化还原化合价分析
2. 键角 102°（NH₃ 为 107.3°）→ F 的电负性拉回电子，使孤对电子更"胖"，对成键电子排斥更大
3. 偶极矩反常小 → 对比 NH₃ 和 NF₃ 的矢量和
4. NF₃ 质子化放热小于 NH₃ → 因为 N 上电子密度被 F 吸引走

---

### 例2 [NO 生物活性-等电子体-反应历程] ⭐⭐⭐⭐

**来源**：原书例2（1997 初赛 · NO 与超氧离子 O₂⁻）

**题干要点**：NO 与 O₂⁻（超氧离子）反应生成 A，A 被认为与炎症/中风等细胞毁坏有关，A 与 NO₃⁻ 互为异构体，与 CO₂ 一对一结合。

**教学价值**：⭐⭐⭐⭐
- **跨学科连接**：化学 + 生物学（NO 的生物信使功能）
- 等电子体 + 异构体 + 路易斯结构式综合
- 奇电子化合物的成键方式

**课堂使用建议**：适合课时2末尾作为"拓展阅读"或课后思考题。也可在讲 NO 时引入，增加课堂趣味性。

**关键知识点**：
1. NO（15e⁻）+ O₂⁻（13e⁻）= ONO₂⁻（28e⁻）——奇电子配对
2. ONO₂⁻ 与 NO₃⁻ 是异构体（原子同，连接方式不同）
3. ONO₂⁻ + CO₂ → 过氧硝酸酯类中间体
4. 3NO → N₂O + NO₂（加压加热聚合），NO₂ 二聚为 N₂O₄

---

### 例3 [Bi 元素综合-物质推断] ⭐⭐⭐⭐

**来源**：原书例3（1992 决赛 · Bi 的综合性质）

**题干要点**：Bi 的物理性质（熔点 545 K，熔融后体积缩小 → 类似水）、蒸气密度测定推算分子型体、与酸反应、Bi(III) 水解与 Bi(V) 的强氧化性。

**教学价值**：⭐⭐⭐⭐
- 理想气体状态方程测定分子量（物理化学在元素化学中的应用）
- 惰性电子对效应的直观体现（Bi 最稳定 +3 而非 +5）
- 类比 Bi 与水的密度异常（熔融后体积缩小）
- **NaBiO₃ 鉴别 Mn²⁺** 的经典反应

**使用建议**：面试级难度，适合课时3/课后选做，或决赛专题使用。

**关键教学节点**：
1. 沸点蒸气密度 → 推算分子量为 Bi₂（二聚体）
2. 更高温 → 解离为 Bi 原子
3. Bi 只与氧化性酸反应生成 Bi(III) → 惰性电子对效应
4. Bi(V) 在酸性介质中极强氧化性 → Mn²⁺ → MnO₄⁻（紫红色）

### 5.2 NH₃ 与 NF₃ 极性对比

| 项目 | NH₃ | NF₃ |
|:---|:---:|:---:|
| 键角 | 107.3° | 102° |
| 偶极矩 (C·m) | 4.74×10⁻³⁰ | 0.78×10⁻³⁰ |
| N 的孤对电子方向 | 与 N—H 偶极同向 → 叠加 | 与 N—F 偶极反向 → 抵消 |
| 质子化放热 | 大 | 小 |
| 碱性 | 强（Kb=1.8×10⁻⁵） | 极弱 |

### 5.3 NOₓ 系列总结

| 分子式 | N 氧化态 | 颜色 | 性质要点 |
|:---|:---:|:---|:---|
| N₂O | +1 | 无色 | 笑气，与 CO₂ 等电子，直线形 |
| NO | +2 | 无色 | 奇电子化合物，不成盐氧化物 |
| N₂O₃ | +3 | 红棕色 | 酸性氧化物，低温为蓝色液体 |
| NO₂ | +4 | 红棕色 | 极性分子，单电子导致不稳定，二聚为 N₂O₄ |
| N₂O₄ | +4 | 无色(气)/黄(液) | 平面分子，Π₆⁸ 离域 |
| N₂O₅ | +5 | 无色晶体 | 酸性氧化物，→HNO₃ |

### 5.4 As/Sb/Bi 酸碱性递变

```
As₂O₃(两性偏酸) → Sb₂O₃(两性) → Bi₂O₃(碱性)
酸性减弱, 碱性增强
As₂O₅(酸性) → Sb₂O₅(两性偏酸) → Bi₂O₅(强氧化性, 不稳定)
```

### 6.2 新判定为"高ROI但当前未纳入"的内容

1. **亚硝酸盐**（§2.3）—— 现有备课大纲完全未涉及，但亚硝酸盐的氧化还原反应是元素推断中常见素材，建议在课时2末尾补充 5 min
2. **铵盐热分解系统分类表**（§2.2）—— 可做成速查卡并入专题页
3. **几元酸判断方法**（§2.4 方法论）—— 不仅在磷，适用于所有含氧酸，建议做成通用方法论卡片
4. **鉴别反应**（马氏试砷法、古氏试砷法、NaBiO₃ 鉴别 Mn²⁺）—— 适合作为"趣闻/拓展"点缀
