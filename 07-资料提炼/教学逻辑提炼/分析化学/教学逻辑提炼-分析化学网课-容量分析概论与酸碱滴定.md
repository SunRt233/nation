## 三、教学切入点

| 维度 | 内容 |
|:---|:---|
| 从什么现象/问题切入 | "用 NaOH 滴定 HCl，指示剂变色时反应真的恰好完全吗？"——以 SP 与 EP 的不重合为切入点 |
| 学生已有认知 | 高中知道酸碱中和、指示剂变色、pH 计算；知道实验有误差；会算一元弱酸 pH |
| 第一处认知冲突 | 学生认为"指示剂变色 = 反应完全"，不理解 SP 与 EP 的差异及由此产生的系统误差 |
| 作者/教师如何铺垫 | 先定义 SP（理论）和 EP（观察），再强调"多数情况下 EP ≠ SP"，最后引出终点误差 Et 的概念 |
| 核心结论何时出现 | 终点误差概念在 SP/EP 区分后立即出现（约 04:43）；林邦误差公式在酸碱滴定末尾出现（约 02:57:37） |
| 结论呈现策略 | 先定性（EP ≠ SP → 有误差）→ 再定量（Et 定义式）→ 再统一（林邦公式）→ 最后推广（多元酸） |

### 9.3 关键发现

1. **教学逻辑主线清晰**：网课从"滴定分析术语 → 误差评价 → 有效数字 → 活度 → 酸碱平衡 → 滴定曲线 → 终点误差"形成完整链条，适合竞赛学生从高中化学向大学分析化学过渡。

2. **林邦误差公式是核心竞赛工具**：网课在酸碱滴定末尾（约 02:57:37）和 L2 开头（多元酸误差）均围绕林邦公式展开，说明这是竞赛高频考点。第一轮应确保学生掌握一元酸/强酸的林邦公式，多元酸可 deferred。

3. **口诀化记忆法丰富**：基准物质口诀（四碳二草硼砂重铬氯溴碘邻钾）、有效数字规则（四舍六入五成双）等，适合竞赛学生快速记忆，但需核对准确性。

4. **OCR 噪声集中在特定区域**：大量噪声出现在 `<details>` 手机截图块和表格空白填充区，正文公式和概念段落相对清晰。建议后续处理同类 OCR 笔记时优先读取正文段落。

5. **与质心酸碱理论课形成互补**：质心 L5/L6 侧重"酸碱概念 + pH 计算"，本分析化学网课侧重"滴定分析框架 + 误差定量 + 终点误差"，两者合并可形成从"酸碱平衡"到"滴定实操"的完整认知链。
