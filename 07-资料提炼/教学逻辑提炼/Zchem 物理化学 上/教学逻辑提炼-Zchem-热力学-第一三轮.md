# 教学逻辑提炼：Zchem 热力学 · 第一+三轮总版

> 本文件提炼 Zchem 物理化学 上 §1（热力学上）、§2（热力学下）、§3（热力学第二定律（下））三篇笔记的教学思路，为第一轮基础班「热力学初步」专题和第三轮提高班「热力学深化」专题提供教学逻辑来源。
>
> **关键说明**：笔记 2 和笔记 3 存在内容错位（见 [[Zchem物理化学-授课顺序总表#二、跨笔记内容错位与闭坑索引]]）——笔记 3 标题为"热力学第二定律（下）"，实际在约 01:02 起进入大量做题环节，以练代讲。
>
> **特殊价值**：质心热力学提炼已覆盖第一轮基础概念，Zchem 的独特价值在于：**(1) 推导更完整**（热力学四大基本公式、Maxwell 关系、G-H 方程均有完整推导链路）；**(2) 计算题量更多、类型更丰富**（从简单恒压/恒容到复杂循环、相变路径设计、混合熵）；**(3) 普化→物化升级层次分明**（教师反复标注"普化表达式 vs 物化微分表达式"）。
>
> **三篇笔记分工**：
> - 笔记 1（热力学上）≈ 概念建立 + 热一定律 + 计算工具（全部可计算量：Q, W, ΔU, ΔH）
> - 笔记 2（热力学下）≈ 作业切入 → 热二定律 → 熵 → 自由能 → Maxwell → G-H（从方向判据到理论深化）
> - 笔记 3（第二定律下）≈ 概念复习（前半 01:02 前）→ 大量做题（后半 01:02 起）→ 综合应用

### 2.3 笔记 3：热力学第二定律（下）—— 概念复习 + 大量做题

|:---:|:---|:---|:---|:---|

## 三、教学切入点

| 维度 | 内容 |
|:---|:---|
| 从什么现象/问题切入 | **笔记 1**：从"温度到底是什么"这个基本物理问题切入，经由热平衡定义温度，再引出热力学第零定律。**笔记 2**：从学生已做的弹式量热计作业（Q_v → Q_p 换算）切入，通过概念判断题（自由膨胀/Zn+HCl/H₂+Cl₂ 绝热钢瓶/水结冰）制造认知冲突，引出"为什么过程有方向？"→ 热力学第二定律。**笔记 3**：从条件判断题（"C_P,m = C_V,m + R 的成立条件？"）切入，先巩固公式适用范围，再进入做题。 |
| 学生已有认知是什么 | 普化中的热力学：ΔU = Q + W（代数式，非微分式），焓变 = 恒压热效应，自发过程由 ΔH 判断（只会用不会解释为什么有时 ΔH>0 的过程也能自发），理想气体状态方程。**无**全微分、偏导数、可逆过程的严格概念。 |
| 第一处认知冲突是什么 | **笔记 1**："热和功不是状态函数，那什么是状态函数？"→ 引出状态函数与过程量的区分。**笔记 2 最关键的冲突**：绝热钢瓶中 H₂+Cl₂ 反应 → q=0, w=0, ΔU=0，但 ΔH>0（因为放热使 T 升高 → p 升高 → vdp>0）。学生直觉"恒容则 ΔH=0"，教师用 dH=dU+pdV+vdp 的物化微分式打破这个直觉。**笔记 3**：真空膨胀中 ΔG≠0（虽然 ΔU=ΔH=0），学生容易认为"所有状态函数都为零"。 |
| 教师如何铺垫 | **笔记 1**：先花大量时间（约 27 分钟）讲温度、体系、平衡、状态函数这些"看似简单"的概念，确保学生有坚固的概念框架后才引入热一定律。每一组概念都是下一步的前提。**笔记 2**：不直接讲热二，先让学生做 5 道概念判断题 → 暴露认知盲区 → "这些题暴露了你们不会判断过程方向" → 自然引出热二。**笔记 3**："这些公式大家都会背，但知道每个公式的约束条件吗？"→ 条件判断题 → 暴露"条件忽视症" → 复习正确推导。 |
| 核心结论何时出现 | **热一定律**：开头（笔记 1 §10，约 33 分钟处），但之前已充分铺垫所有前置概念。**可逆过程最大功**：通过三种膨胀过程数值对比后归纳（笔记 1 §10.3）。**热机效率上限 η=1-T_c/T_h**：完整推导卡诺循环后得出（笔记 2 §6.2）。**熵是状态函数**：通过卡诺循环证明 Σ(δQ_i/T_i)_R=0 后得出（笔记 2 §6.4）。**ΔG<0 为自发判据**：从自由能定义完整推导后得出（笔记 2 §二.2），但作为"结论"给出，不要求第一轮学生能独立推导。 |
| 结论呈现策略 | **结论后置为主**：绝大多数结论通过"问题→推导→结论"路径得出，不在开头抛结论。例外：绝热可逆公式 TV^(γ-1)=常数直接给出并强调"记住就行，不要每次都推"。**数值对比驱动**：三种膨胀过程的功对比、可逆 vs 不可逆膨胀的熵变对比，用数值而不是用公式制造直觉。 |

| 3 | 讲 H₂+Cl₂ 绝热钢瓶题时（笔记3 §二.7，台阶11） | 左边写已知条件：q=0(绝热), w=0(恒容钢瓶), 反应放热→右边分三行推导：(1)ΔU=q+w=0→(2)dH=dU+pdV+vdp=0+0+vdp=vdp→(3)反应放热→T↑→p↑→vdp>0→ΔH>0 | "这就是物化思维和普化思维的区别。普化告诉你 ΔH=Q_p——恒压热效应。这里不是恒压！是恒容！所以你只能用微分形式 dH=dU+pdV+vdp 去分析。恒容时 pdV=0 但不能说 vdp=0——因为温度变了压强也变了。这道题几乎每年都有人错——它是最经典的'普化直觉失效'案例。" | 分步板书推导 dH 的完整微分——用最经典的"普化直觉失效"题作为 d vs δ、微分 vs 代数的终极区分训练 |
