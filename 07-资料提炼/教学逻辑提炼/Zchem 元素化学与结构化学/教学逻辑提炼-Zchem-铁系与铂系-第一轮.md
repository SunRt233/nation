## 二、原资料讲解顺序

本堂课共分五个大板块：

1. **铁系元素** (08:17-01:56:50)：
   - **铁系元素概述** (10:36-21:29)：族别特征（8B族，d⁶⁻⁸s²）、性质比较（Fe>Co≈Ni活泼性排序、铂系惰性）、氧化态极限（OsO₄唯一稳定高氧化态）、酸反应性（稀酸放H₂、浓酸钝化）、碱反应性、标准电极电势
   - **铁系元素的低价化合物** (21:29-01:09:22)：羰基化合物（Fe(CO)₅三角双锥→Fe₂(CO)₉→Fe₃(CO)₁₂）、二茂铁制备与结构（D₅d/D₅h构象争议）、Ru/Os的三核羰基化合物、Co₂(CO)₈结构（桥羰基平衡）、Ni(CO)₄（Td点群）、18电子规则应用例题
   - **二价铁钴镍化合物颜色** (01:09:22-01:56:30)：水合离子颜色（Fe²⁺浅绿、Co²⁺粉红、Ni²⁺泛绿）、镍配合物颜色（平面四边形[Ni(CN)₄]²⁻黄色）、镍的特征沉淀反应（丁二酮肟鲜红色）、三价铁性质（中强氧化剂、SCN⁻显血红色）、钴的氧化还原（六氨合钴配位反应）、高价氧化物（NiO₂、CoO₂）、高铁酸盐（FeO₄²⁻紫红色）制备与性质

2. **铂系元素** (01:56:50-02:16:56)：
   - **铂系元素概述**：鹅金属特性（Os空气氧化→OsO₄）、六氟合铂(IV)氧的强氧化性（氧化O₂生成O₂⁺[PtF₆]⁻）、键长特性（O₂⁺键长<O₂）
   - **正四价钯和铂**：溶解特性（不溶常见酸，溶于王水）、电子构型与18电子规则、软硬酸碱性质（Pt⁴⁺为罕见的软酸）、稳定性顺序
   - **二价钯和铂配合物**：d⁸平面四边形、蔡思盐（K[Pt(C₂H₄)Cl₃]·H₂O）、电子计数（16电子构型）、顺铂/反铂
   - **反位效应**：配体交换优先级顺序、配体影响顺序

3. **催化循环** (02:16:56-03:08:11)：
   - 基本概念（催化剂参与循环但最终复原）
   - Rh催化循环实例：[Rh(PPh₃)₃Cl]催化烯烃氢化，关键步骤：氧化加成→配体解离→配体交换→插入反应→还原消除
   - Co催化实例：氢甲酰化反应，电子计数（18e→16e变化）
   - Pd交叉偶联：氧化加成→配体交换→还原消除（非经典）
   - 烯烃复分解（2005年诺贝尔奖）：Grubbs催化剂（Ru）与Schrock催化剂

4. **镧系元素** (03:15:40-03:19:38)：
   - 别称稀土元素、+3价存在形态、硬路易斯酸
   - 镧系收缩（4f电子屏蔽效应、离子半径单调递减）
   - 分离方法（离子交换法、洗脱顺序原理）

5. **知识小结** (结尾表格)

## 三、教学切入点

1. **从"贵金属"概念切入（铂系元素）**：
   - 铂金、钯金在珠宝中的应用引出"贵金属为什么贵"（化学惰性→难提炼→稀缺）
   - 王水溶解金/铂的故事性引入（"王水"命由来：能溶解金属之王）

2. **从"颜色记忆"切入（铁系元素）**：
   - "铁绿钴粉镍泛绿"口诀帮助记忆Fe²⁺/Co²⁺/Ni²⁺水合离子颜色
   - 三价铁离子SCN⁻显血红色→限量试剂的颜色变化（教学中常用演示实验）
   - 丁二酮肟镍鲜红色沉淀→特征检测反应（灵敏度高、颜色鲜明）

3. **从"诺贝尔奖故事"切入（催化循环）**：
   - 2005年诺贝尔化学奖（Grubbs、Schrock）烯烃复分解
   - 顺铂抗癌药发现史（从偶然观察到药物设计）
   - 过渡金属催化在现代有机合成中的革命性作用

4. **从"结构-电子规则"切入（羰基化合物）**：
   - 18电子规则类比稀有气体稳定结构→"过渡金属的八隅体规则"
   - Fe(CO)₅三角双锥→二聚体Fe₂(CO)₉（形式上的"配体不足"→金属键补偿）
   - 从18电子到16电子：后过渡金属的特殊性

5. **从"生命元素"切入（铁）**：
   - 血红蛋白中的铁→生物无机化学联系
   - 铁是人体必需的微量元素（缺铁性贫血）
   - 钴在维生素B₁₂中的核心作用

## 五、例题/活动出现顺序

1. **铁系元素性质比较**（Fe/Co/Ni活泼性排序：Fe>Co≈Ni）
2. **铁系与铂系对比**（铂系惰性需王水溶解）
3. **Fe(CO)₅→Fe₂(CO)₉→Fe₃(CO)₁₂结构推导**（18电子规则系统应用）
4. **二茂铁电子计数**（Cp⁻提供6e，Fe²⁺为d⁶，总18e⁻）
5. **Co₂(CO)₈桥羰基平衡的红外光谱推断**（1700cm⁻¹两个峰，强度比2:7→结构比例计算）
6. **镍的四核簇结构推断**[M₄(CO)₁₂]（d⁹构型+3CO→15e→需3个M-M键）
7. **Na+Fe(CO)₅在液氨中反应**（→[Fe(CO)₄]²⁻，Td点群）
8. **CO₂(CO)₈+H₂→HCo(CO)₄**（18电子规则应用，C₄v对称性）
9. **Fe(CO)₅+I₂→Fe(CO)₄I₂**（顺反异构，C₂v点群）
10. **Fe³⁺刻蚀铜板反应**（2Fe³⁺+Cu→2Fe²⁺+Cu²⁺）
11. **Co²⁺氧化为Co³⁺的配位反应**（六氨合钴/六氰合钴体系）
12. **亚硝酸钴钾沉淀的配平与电子计数**（K₃[Co(NO₂)₆]↓）
13. **高铁酸盐制备与酸性分解反应配平**（4FeO₄²⁻+20H⁺→4Fe³⁺+3O₂↑+10H₂O）
14. **六氟合铂(IV)氧的氧化性分析**（O₂⁺[PtF₆]⁻，O₂⁺键级2.5，键长<O₂）
15. **顺铂与反铂结构对比**（抗癌活性差异）
16. **Rh催化烯烃氢化循环分析**（氧化加成→配体解离→插入→还原消除全流程）
17. **Co催化氢甲酰化循环例题**（净效应：RCH=CH₂+CO+H₂→RCH₂CHO）
18. **Pd催化交叉偶联循环例题**（Br⁻和烯烃→偶联产物+HBr）
19. **Grubbs烯烃复分解催化剂**（2005年诺贝尔奖案例）
20. **镧系元素离子交换分离原理**（水合半径→扩散速率→配位常数→洗脱顺序）

## 八、关键教学语言段

1. **铁系与铂系对比**："铁系是中等活泼的金属……铂系元素比较惰性，单质吸收H₂，用作催化剂。"
2. **高价态极限**："除了OsO₄，其它元素没有常见以最高氧化态形式存在的水溶液——四氧化锇是唯一较稳定存在的高氧化态化合物。"
3. **18电子规则在羰基化合物中的普遍性**："绝大多数羰基化合物满足18电子规则……通过聚合形成金属键来满足电子规则。"
4. **二茂铁结构**："传统认为交错式更稳定（类比乙烷构象），但实际两种构型能量接近，不同条件下可分别存在。"
5. **丁二酮肟反应**："需在弱碱性环境（氨水）中进行，生成鲜红色沉淀……是检测镍离子的特征反应，灵敏度高且颜色鲜明。"
6. **三价铁颜色**："强酸性溶液中为淡紫色（高自旋d⁵，d-d跃迁弱）；含氯离子时呈黄色……与二价锰类似，都是几乎无色的浅色。"
7. **三价铁刻蚀铜板**："二价铁不足以还原二价铜→记忆氧化还原电势关系。"
8. **催化循环核心**："氧化加成：金属同时接受两个配体，化合价升2，电子数增2；还原消除：金属释放两个配体，化合价降2，电子数减2。"
9. **反位效应**："在平面四边形配合物中，特定配体对其反位配体稳定性有影响……属于经验总结规律，不适用于预测反应。"
10. **镧系分离原理**："大半径离子（如La³⁺）极化弱→水合层薄→扩散快；小半径离子（如Lu³⁺）极化强→水合层厚→扩散慢。配体与树脂竞争结合Ln³⁺，离子半径越小配位常数越大，Lu³⁺优先被洗脱。"
11. **烯烃复分解的诺贝尔奖背景**："2005年诺贝尔化学奖……Grubbs是有机化学家，更关注反应性和实际应用；Schrock是无机化学家，专注金属-碳双键结构研究。两种不同风格的科学家对同一领域的贡献。"
12. **后过渡金属电子规则**："铂族特性——铂和钯的羰基化合物在普通条件下不常见……四配位二价铂配合物通常为16电子构型（8+8）。"

## 九、核心内容速查

### 9.1 核心定义与公式

| 概念 | 定义/公式 | 备注 |
|:---|:---|:---|
| 铁系元素 | Fe、Co、Ni（8B族，第四周期） | 中等活泼金属 |
| 铂系元素 | Ru、Rh、Pd、Os、Ir、Pt（5、6周期8B族） | 惰性金属，需王水溶解 |
| 标准电极电势（酸性） | φ°(Fe²⁺/Fe)=-0.44V；Co²⁺/Co=-0.28V；Ni²⁺/Ni=-0.23V | Fe>Co≈Ni活泼性 |
| 18电子规则 | M价电子 + 配体贡献电子 = 18 | 类比稀有气体稳定结构 |
| 16电子规则 | 后过渡金属（d⁸）稳定构型为16电子 | 适用于Pt²⁺、Re等 |
| 二茂铁 | Fe(Cp)₂，夹心化合物，Fe为+2价 | Cp⁻提供6e，总18e |
| 高铁酸根 | FeO₄²⁻，紫红色，强氧化性（比KMnO₄强） | 碱性条件制备 |
| 顺铂 | cis-[Pt(NH₃)₂Cl₂]，平面四边形 | 抗癌药物，破坏DNA复制 |
| 反位效应顺序 | CO,CN⁻,C₂H₄ > PR₃ > X⁻ > Py > NH₃ > H₂O,OH⁻ | 配体对对位配体稳定性的影响 |
| 镧系收缩 | 原子序数增加，原子半径递减（4f电子屏蔽弱） | 导致5、6周期同族元素性质接近 |

### 9.2 规律总结（含对比表）

**Fe/Co/Ni基本性质对比**

| 性质 | Fe | Co | Ni |
|:---|:---|:---|:---|
| 电子排布 | [Ar]3d⁶4s² | [Ar]3d⁷4s² | [Ar]3d⁸4s² |
| φ°(M²⁺/M) 酸性 | -0.44V | -0.28V | -0.23V |
| 活泼性 | Fe > Co ≈ Ni | | |

### 9.3 典型例题与解析

**例1. 丁二酮肟检测镍**（见原笔记图）
- 条件：弱碱性（氨水），生成鲜红色沉淀
- 结构：平面四边形，Ni(Ⅱ)与两个丁二酮肟分子配位
- 化学式：Ni(C₄H₇N₂O₂)₂

**例2. 高铁酸盐的制备与分解**（见原笔记图）
- 制备（碱性）：2Fe(OH)₃ + 3ClO⁻ + 4OH⁻ → 2FeO₄²⁻ + 3Cl⁻ + 5H₂O
- 制备（固相）：Fe₂O₃ + 3KNO₃ + 4KOH → 2K₂FeO₄ + 3KNO₂ + 2H₂O
- 酸性分解：4FeO₄²⁻ + 20H⁺ → 4Fe³⁺ + 3O₂↑ + 10H₂O（配平：1个FeO₄²⁻得3e⁻，4个共得12e⁻→3O₂）

**例3. Co₂(CO)₈红外光谱推断结构比例**（见原笔记图）
- 红外数据：1700cm⁻¹两个峰，强度比2:7
- B结构（含2个桥羰基）占多数
- A结构（无桥羰基）占比计算：(0.125/1.125) = 11.1%

**例4. 顺铂抗癌原理**
- 靶点：DNA碱基对中的氮原子
- 机制：氨配体与DNA发生配体交换→破坏双螺旋结构→抑制癌细胞繁殖
- 局限性：无差别破坏所有细胞DNA→严重毒性

**例5. 镧系洗脱顺序**
- 离子交换法原理：Ln³⁺与树脂H⁺/NH₄⁺置换
- 水合半径：La³⁺（大半径→薄水合层→快扩散→后洗脱）
- 配位能力：Lu³⁺（小半径→厚水合层→慢扩散→配位常数大→优先洗脱）

### 9.4 常见误区

1. **Fe²⁺/Co²⁺/Ni²⁺颜色混淆**：Fe²⁺是浅绿色（不是深绿），Co²⁺是粉红色（不是蓝色），Ni²⁺是泛绿色（不是翠绿色）。CoCl₂·6H₂O颜色变化（粉红→紫红→蓝紫）随脱水程度变化，需要注意区分。
2. **铁系与铂系活泼性混淆**：认为所有过渡金属都很活泼→铂系金属实际惰性，需用王水溶解。
3. **羰基化合物中金属氧化态判断错误**：Fe(CO)₅中铁是0价（不是正价），CO是中性配体；二茂铁中铁是+2价。
4. **18电子规则绝对化**：认为所有金属有机化合物都满足18电子→Pt²⁺、Pd²⁺等d⁸构型常为16电子构型。
5. **反位效应适用范围混淆**：反位效应是**经验规律**，适用于Pt²⁺平面四边形配合物，不适用于预测反应方向。
6. **顺铂与反铂活性差异来源不明**：顺铂中两个Cl处于邻位，可同时与DNA两个位点配位；反铂中Cl处于对位，不能同时配位。
7. **镧系收缩"例外"混淆**：原子半径在4f⁷和4f¹⁴时出现突增（屏蔽效应增强），但**Ln³⁺离子半径单调递减无例外**（因已失去5d和6s电子）。
8. **氧化加成/还原消除与配体解离/配位的区别**：前者伴随氧化态变化（±2），后者氧化态不变。
