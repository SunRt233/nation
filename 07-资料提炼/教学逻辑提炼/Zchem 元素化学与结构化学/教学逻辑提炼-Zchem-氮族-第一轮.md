## 九、核心内容速查

### 9.1 核心定义与公式

| 概念 | 定义/公式 | 备注 |
|:---|:---|:---|
| 电负性反常 | 氮族元素中 N 电负性(3.0) > P(2.1)，违背同族递减规律 | 因 N 原子半径小、核电荷对价电子吸引力强 |
| 惰性电子对效应 | 第六周期元素( Tl/Pb/Bi )的 6s² 电子对不易参与成键，低价态更稳定 | 解释 Bi₂O₅ 极不稳定而 Bi₂O₃ 稳定 |
| N≡N 三键键能 | 945 kJ/mol | 单质 N₂ 极惰性的结构原因 |
| Haber-Bosch 循环 | N₂ + 3H₂ ⇌ 2NH₃（高温高压、Fe 催化剂） | 人工固氮核心方法 |
| 氨自电离 | 2NH₃ ⇌ NH₄⁺ + NH₂⁻ | 类比水的自电离 |
| pKa 值 | NH₃：9.25；HN₃：4.75 | 氨为弱碱，叠氮酸为弱酸(与乙酸相当) |
| 多磷酸滴定 | 链状 → 两个滴定终点；环状 → 一个滴定终点 | 通过 OH 比例计算聚合度 n |

### 9.3 典型例题与解析

**例题 1**：PCl₅ 在液态时为三角双锥，在固态时变为何种结构？为什么？
- **解析**：固态转变为 [PCl₄]⁺[PCl₆]⁻ 离子晶体。PCl₄⁺ 为正四面体(P sp³ 杂化，无孤对电子)，PCl₆⁻ 为正八面体(P sp³d² 杂化)。驱动原因：获得更高的对称性（四面体 + 八面体 > 三角双锥）。
- **联想 KP**：[[价态-氧化态-形式电荷]]

**例题 2**：为什么 H₃PO₃ 是二元酸而非三元酸？
- **解析**：H₃PO₃ 结构式中仅有两个 OH 基团可电离出 H⁺，第三个 H 直接与 P 原子成键(P-H 键)，该键在水中不电离。故实为二元酸。
- **联想 KP**：[[磷化学]]

**例题 3**：比较 NH₃、PH₃、AsH₃、SbH₃ 的键角大小并解释原因。
- **解析**：NH₃(107°) > PH₃(93.5°) > AsH₃(92°) > SbH₃(91°)。原因：中心原子电负性递减→成键电子对远离中心原子→孤对电子对成键电子的排斥增强→键角被压缩。
- **联想 KP**：[[氮族元素]]

### 9.4 常见误区

| 误区 | 正确理解 | 涉及知识点 |
|:---|:---|:---|
| "同族元素电负性一定递减" | 氮族中 N > P 为反例，碳族中 C 电负性也最大 | 电负性反常 |
| "H₃PO₃ 是三元酸" | 只有 OH 上的 H 可电离，P-H 键不电离，实为二元酸 | 磷的含氧酸酸性 |
| "浓 HNO₃ 和稀 HNO₃ 都是强氧化剂" | 浓 HNO₃ 以分子态存在氧化性强，稀 HNO₃ 以 NO₃⁻ 存在主要表现酸性 | 硝酸氧化性差异 |
| "PCl₅ 固态是分子晶体" | 固态为 [PCl₄]⁺[PCl₆]⁻ 离子晶体 | 固态结构转变 |
| "NO 在空气中都容易被氧化" | 高浓度 NO 才易氧化(二聚体机理)，低浓度(如尾气)难以氧化 | NO 浓度依赖氧化动力学 |
