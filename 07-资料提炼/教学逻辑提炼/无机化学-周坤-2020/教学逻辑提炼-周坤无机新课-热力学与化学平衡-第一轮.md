## 三、教学切入点

| 维度 | 内容 |
|:---|:---|
| 从什么现象/问题切入 | **"化学热力学研究的目的"**——直接抛出"化学反应有两个核心问题：①有无能量变化（热效应）②反应能否发生、进行到什么程度"。不拖泥带水，2 分钟内建立全局观。 |
| 学生已有认知是什么 | 高中化学的"放热/吸热"概念、化学方程式的计量关系、勒夏特列原理的定性描述。 |
| 第一处认知冲突是什么 | **"燃烧放热多的反应不一定能自发进行"**——如 H₂ + ½O₂ 放热很大但需点燃；NH₄Cl 与 Ba(OH)₂ 吸热却自发。引出"能量不是唯一判据，需要新函数（熵/Gibbs）"。 |
| 作者/教师如何铺垫 | 热力学部分：从"研究目的"→"语言工具"→"守恒律"→"计算方法"→"方向判据"，层层递进。<br>平衡部分：用**上节课作业讲评**作为暖场，自然过渡到"反应进行到什么程度"。 |
| 核心结论何时出现 | **热力学**：ΔU = Q - W（第一定律）在中段出现；Gibbs 判据在末尾出现。<br>**化学平衡**：Kc/Kp 的定义在开头即给出；ΔG° = -RTlnK 在中段作为"桥梁公式"出现。 |
| 结论呈现策略 | **结论后置为主，但关键定义先行**。例如：先定义"平衡常数"再推导其与 ΔG 的关系；先给第一定律公式再解释各项物理意义。 |
