## 二、本批次的核心价值定位

> **Zchem批次Z-F是学而思有机化学基础的最大增量来源。** 学而思15讲完全没有涉及周环反应、重排反应和波谱分析，而这三个板块在竞赛有机中权重极高（周环反应是初赛/决赛常客；波谱分析是结构推求的核心工具；重排反应是有机合成和机理推断的高频考点）。

**三大板块的协同关系**：
```
周环反应（协同机理范式）
    ↓ 对照
离子型反应（SN/E/加成/缩合）—— 学生已在学而思系统学完
    ↓ 补充
重排反应（离子型机理中的骨架重组）—— 与碳正离子/卡宾/氮宾衔接
    ↓ 工具
波谱分析（结构确证手段）—— 为所有反应产物提供表征方法
```

| 5 | 核磁共振（NMR）——核心重点：化学等价（Replacement Test）、信号数判断、化学位移、积分 | **本讲最大篇幅**，结构解析的核心 | 主讲人专业背景发挥最充分 | [[核磁共振]] |

## 五、教学切入点

### 5.1 周环反应

| 维度 | 内容 |
|:---|:---|
| 从什么现象/问题切入 | 从"缩合反应的作业题"切入，先复习离子型反应，然后抛出"如果反应不受溶剂极性影响、没有中间体，会是什么机理？" |
| 学生已有认知是什么 | 已系统学过SN/E/加成/缩合的离子型机理，知道自由基链反应，但从未接触过"协同反应" |
| 第一处认知冲突是什么 | "没有中间体的反应怎么写箭头？"——资料的回答是：用环状过渡态和轨道对称性 |
| 作者/教师如何铺垫 | 用"机理三大类"的框架（离子>>自由基>>协同）降低周环反应的神秘感 |
| 核心结论何时出现 | Diels-Alder的endo规则在前半出现；电环化的顺旋/对旋规则在中段出现；σ迁移的命名规则在后段出现 |
| 结论呈现策略 | 先给定性直觉（四大特点），再用FMO做半定量预测，最后以记忆技巧收束 |

### 5.2 重排反应

| 维度 | 内容 |
|:---|:---|
| 从什么现象/问题切入 | 从"分子内反应为什么比分子间快"切入，用邻基参与建立直觉 |
| 学生已有认知是什么 | 知道碳正离子可以重排（W-M），但不知道系统的分类和驱动力分析 |
| 第一处认知冲突是什么 | "迁移顺序：W-M是越小越快，Pinacol是给电子能力越强越快——为什么相反？" |
| 作者/教师如何铺垫 | 用"动力学驱动vs热力学驱动"的双维度框架，把所有重排装进统一的分析视角 |
| 核心结论何时出现 | 邻基参与的环状中间体概念在开头出现；迁移选择性规律在各人名反应中分别归纳 |
| 结论呈现策略 | 每个人名反应都拆成"结构特征→机理步骤→驱动力→与其他反应的对比"四段 |

### 5.3 波谱分析

| 维度 | 内容 |
|:---|:---|
| 从什么现象/问题切入 | 从"有机化学家的眼睛"切入，强调波谱不是背数据而是推理工具 |
| 学生已有认知是什么 | 可能零散接触过NMR/MS的概念，但缺乏系统的"何时用哪种谱"的判断框架 |
| 第一处认知冲突是什么 | "为什么NMR信号数不等于氢原子数？"——资料用Replacement Test系统回答 |
| 作者/教师如何铺垫 | 主讲人发挥NMR专业背景，从量子力学底层（自旋量子数）建立可信度 |
| 核心结论何时出现 | IR特征峰位在前1/3出现；MS同位素效应在中段出现；NMR信号判断在最大篇幅的后半出现 |
| 结论呈现策略 | 按"信息层级"组织：MS给分子量→IR给官能团→NMR给连接方式→UV给共轭程度 |
