**原顺序一句话概括**：
- 教师用 ΔG=-nFE 从热力学导入 → Nernst方程推导（含活度修正） → 电导/摩尔电导率体系（从定义→极限→迁移数→应用计算） → 可逆电池严格条件 → 极化/超电势 → 腐蚀防护案例。整体逻辑是 **热力学桥梁→Nernst方程→电导实验量→可逆电池→非理想(极化)→应用**。

## 三、教学切入点

| 维度 | 内容 |
|:---|:---|
| 从什么现象/问题切入 | 直接以考纲定位切入："国初电化学考什么？——主要围绕Nernst方程"——教师不给现象引入，而是以应试框架定位，然后立即复习热力学桥梁公式ΔG=-nFE |
| 学生已有认知是什么 | 普化已学Nernst方程简化版(E=E°-0.0591/n lgQ)，知道ΔG与电动势的关系 |
| 第一处认知冲突是什么 | "为什么有的教材Nernst方程系数是0.0591，有的教材是0.0651？——不是实验误差，是温度不同！普化默认298K，生化默认310K(人体温度)" 这个看似简单的问题暴露了学生对"简化公式的前提假设"缺乏意识 |
| 作者/教师如何铺垫 | 先复习ΔG=-nFE和范特霍夫等温式→严格推导E=E°-(RT/nF)lnQ→对比普化简化版→引出"物化保留T、用ln而非lg"→自然引出活度修正 |
| 核心结论何时出现 | **开头即出现**：Nernst方程在28:55就推导出来了（全文约6分半就进入核心公式），后续约1.5小时都在围绕Nernst方程展开：活度→电导率→可逆电池→极化→腐蚀——全是对Nernst方程在不同场景下的应用和修正 |
| 结论呈现策略 | **推导驱动+结论先行**：Nernst方程先推导给出，再用大量例题反复应用。教师风格是"先把核心公式给你，剩下的时间全部用来做各种变形和计算" |
