# ● 电极表面的两种趋势 01:29:14

当金属M与其盐 $\mathbf{M}^{z + }$ 溶液接触时，有两种过程可能发生

$$
\mathbf {M} - \mathbf {M} ^ {z +} + z \mathbf {e} \tag {1}
$$

$$
\mathbf {M} ^ {z +} + z \mathbf {e} - \mathbf {M} \tag {2}
$$

氧化趋势：金属原子失去电子变成离子进入溶液，表示为 $M \rightarrow M^{z+} + ze^{-}$   
○ 还原趋势：溶液中的金属离子获得电子沉积为金属原子，表示为 $M^{Z+} + ze^{-} \rightarrow M$   
- 电势本质：这两种趋势的强弱差异构成了电极电势，金属越活泼氧化趋势越强，溶液浓度越低氧化趋势越明显。

● 电极电势的定义 01:29:46

物理意义：电极电势反映金属与溶液之间的电荷分离程度，本质上是金属表面电子富集程度的量度。  
○ 计算公式：定义为金属电势与溶液电势之差，即 $E=E_{M}-E_{M^{z+}}$ ，单位为伏特(V)。

● 金属电势与溶液电势的关系 01:30:15

$$
\mathbf {M} - \mathbf {M} ^ {z +} + z \mathbf {e} \tag {1}
$$

金属表面的一些原子受到溶剂水分子的吸引，变成溶剂化离子进入溶液，将电子留在金属上

溶解过程：当金属原子溶剂化进入溶液时，电子留在金属表面，导致金属电势低于溶液电势。  
○ 沉积过程：当离子沉积到金属表面时，金属带正电，电势高于溶液（如图01\_31\_00.jpg所示）。  
○ 影响因素：

■ 金属活泼性：活泼金属更易溶解，电势更低  
■ 溶液浓度：稀溶液促进溶解，浓溶液促进沉积

$$
\mathbf {M} ^ {z +} + z \mathbf {e} - \mathbf {M} \tag {2}
$$

溶液中的离子与金属上的电子结合，沉积到金属表面上，致使金属带上正电荷，从而电势高于溶液。

应用规则：

- 电势值越大，氧化型氧化能力越强  
- 电势值越小，还原型还原能力越强  
○ 组成电池时：电势高的为正极，低的为负极

● 电动势计算： $E_{cell} = \varphi$ 正极 - $\varphi$ 负极

\- 对应关系：

○ 氧化能力： $MnO_{4}^{-}>Cr_{2}O_{7}^{2-}>Fe^{3+}$   
- 还原能力：Fe > Zn > Cu

# 十、酸性介质和碱性介质 02:22:01

# 1. 介质对离子存在形式的影响

- 酸性介质特征: $H^{+}$ 浓度需维持 $\geqslant 1\mathrm{mol} / \mathrm{L}$ , 防止 $Fe^{3+}$ 、 $Cu^{2+}$ 等金属离子生成氢氧化物沉淀 (如 $Fe(OH)_{3}$ 、 $Cu(OH)_{2}$ )  
● Ksp控制原理：氢离子浓度需满足 $[H^{+}]\geq\sqrt{K_{sp}}$ ，具体数值可能大于1mol/L  
- 碱表与酸表区别：电极电势表分为酸表（ $\varphi_A$ ）和碱表（ $\varphi_B$ ），同一反应在不同介质中表达式不同（如酸性： $H^+ \rightarrow H_2$ ，碱性： $H_2O \rightarrow H_2$ ）

# 2. 介质对电极电势的影响

\- 变化规律：酸性→碱性时， $H^{+}/H_{2}$ 电势从0V降为负值， $Fe^{3+}/Fe^{2+}$ 、 $Cu^{2+}/Cu$ 等电对电势均降低

● 本质原因：氧化型离子浓度降低（如 $[H^{+}]$ 下降导致得电子能力减弱）  
● 特例反应： $I_{2}/I^{-}$ 等反应电极电势与pH无关  
- Ksp差异影响： $Fe(OH)_{3}$ 的 $K_{sp}$ 远小于 $Fe(OH)_{2}$ ，故三价铁浓度降低更显著

3. 标准与非标准电势关系

● 通式不变原则：任何介质下电极反应通式均为"氧化型 + e → 还原型"  
- 浓度影响：已知标准电势（如 $[H^{+}] = 1M$ ）可求非标准条件（如 $[H^{+}] = 0.5M$ ）下的电势  
- 介质转换关系： $\varphi_0(H_2O / H_2)$ 与 $\varphi_0(H^+ / H_2)$ 存在定量换算关系（后续能斯特方程讲解）

# 十一、配平氧化还原方程式 02:33:19

# 1. 酸性介质配平方法

● 基本步骤：

○ 确定氧化还原对（如 $Bi^{3+}\rightarrow Bi$ ）  
○ 氧原子处理：缺氧侧补 $H_{2}O$ （如 $BiO_{3}^{-}\rightarrow Bi^{3+}$ 需补 $3H_{2}O$ ）  
○ 电荷平衡：补 $H^{+}$ 和 $e^{-}$ （如 $BiO_{3}^{-} + 6H^{+} + 2e^{-} \rightarrow Bi^{3+} + 3H_{2}O$ ）

\- 氯离子处理： $ClO_{3}^{-}$ 还原产物为 $Cl^{-}$ （不写 $Cl_{2}$ ）

2. 碱性介质配平方法

- 禁止出现物质：不能出现 $H^{+}$ 、 $Fe^{3+}$ 等会与 $OH^{-}$ 反应的物种  
● 氧原子处理：用 $OH^{-}$ 补氧（如 $Ag_{2}O + H_{2}O + 2e^{-} \rightarrow 2Ag + 2OH^{-}$ ）  
- 氢处理原则：缺氢时用水补，生成 $OH^{-}$

4. 例题: $MnO_{4}^{-} + H_{2}SO_{3}$ 反应配平

1）酸性条件配平

● 氧化数变化： $Mn(+7\rightarrow+2)$ 降5， $S(+4\rightarrow+6)$ 升2⇒系数比2:5

● 氧平衡：多出的3个氧转化为 $3H_{2}O$

\- 最终形式： $2MnO_{4}^{-} + 5H_{2}SO_{3} \rightarrow 2Mn^{2+} + 5SO_{4}^{2-} + 3H_{2}O + 4H^{+}$

2）碱性条件配平

● 关键区别：用 $OH^{-}$ 补氧（如 $MnO_{4}^{-} + e^{-} + OH^{-} \rightarrow MnO_{4}^{2-} + H_{2}O$ ）  
● 电荷检查：需保证两边电荷总数相等（如左边2个 $MnO_{4}^{-}$ 带2-，右边2个 $MnO_{4}^{2-}$ 带4-，需补 $2OH^{-}$ ）

# 十二、德尔塔G和电池的关系 02:39:30

- 最大非体积功： $\Delta G_{m}$ 表示体系能做的最大非体积功，当全部用于电功时满足 $\Delta G = -nFE$ 关系  
● 电量计算：电量Q = nF，其中F为法拉第常数（96485 C / mol或考试常用96500 C / mol），表示1摩尔电子所带电量  
● 可逆过程关系：电池可逆工作时 $\Delta G = -nFE$ ，标准状态下 $\Delta G^{\circ} = -nFE^{\circ}$   
● 电子转移数：n为电池反应转移的电子数，例如 $Zn + Cu^{2+} \rightarrow Zn^{2+} + Cu$ 中n = 2  
● 方向判据： $\Delta G < 0$ 对应E > 0，两者判据一致  
- 平衡常数关系：通过 $\Delta G^{\circ} = -RT\ln K$ 和 $\Delta G^{\circ} = -nFE^{\circ}$ 可得 $E^{\circ} = \frac{RT}{nF}\ln K$ 0.0591  
● 常用公式转换：298K时 $E^{\circ}=\frac{n}{n}lgK$ （取F=96500）或0.0592/nlgK（取F=96485）

# 十三、应用案例 02:46:41

# 1. 例题: 求反应K值

● 解题步骤：

○ 查标准电极电势表获得E正极和E负极  
○ 计算电池标准电动势 $E^{\circ}=E$ 正极 - E 负极  
○ 代入公式 $E^{\circ}=\frac{0.0591}{n}lgK$ 求K

● 关键点：当 $E^{\circ}=0.2V$ 时，K已达 $10^{3}$ 数量级，说明电动势微小变化会导致K显著变化

2. 例题:通过电池求反应的k 02:48:22

● 特殊方法：将非氧化还原反应（如 $AgCl \rightleftharpoons Ag^{+} + Cl^{-}$ ）拆分为两个半反应：

○ 正极： $Ag^{+} + e^{-} \rightarrow Ag$   
○ 负极： $AgCl + e^{-} \rightarrow Ag + Cl^{-}$

\- 计算要点:

○ 通过电极电势差求 $E^{\circ}$   
○ 注意电子转移数n=1  
- 最终求得 $K_{sp}$ 即为沉淀溶解平衡常数

● 典型结果： $E^{\circ} = -0.58V$ 对应 $K_{sp} = 1.5 \times 10^{-10}$

3. 例题: 浓差电池分析 02:52:52

● 定义：由相同电极材料但电解质浓度不同构成的电池（如氢离子浓度不同的氢电极）

● 能斯特方程应用： $E = -\frac{0.0591}{n} lg \frac{c_{1}}{c_{2}}$

\- 特点:

○ 标准电动势 $E^{\circ}=0$   
○ 实际电动势仅由浓度差决定  
- 可推广到其他非氧化还原体系（如沉淀平衡）

# 十四、能斯特方程

● 推导基础：由 $\Delta G=\Delta G^{\circ}+RT\ln Q$ 和 $\Delta G=-nFE$ 联立得到  
● 标准形式： $E = E^{\circ} - \frac{RT}{nF} \ln Q$   
● 常用简化式：298K时 $E=E^{\circ}-\frac{0.0591}{n}lgQ$   
● 记忆建议：优先掌握标准形式，简化式可通过代入R = 8.314, F = 96500, T = 298K 推导   
● 应用范围：适用于计算任意浓度条件下的实际电池电动势

# 十五、电极电势的测定方式 02:55:27

1. 电池反应与电极反应的关系

● 电势差计算：电池电动势E等于正极电势 $\varphi_{+}$ 减去负极电势 $\varphi_{-}$ ，即 $E=\varphi_{+}-\varphi_{-}$   
● 标准状态表达式： $E = E^{0} - \frac{RT}{nF} \ln Q$ 或简化为 $E = E^{0} - \frac{0.0591}{n} \log Q$   
- 电极反应拆分：将电池反应拆分为两个半反应（正极和负极），正极反应为 $a + n e^{-} \rightarrow c$ ，负极反应为 $d + n e^{-} \rightarrow b$

2. 能斯特方程推导

$\bullet$ 电势分解： $\varphi_{+} - \varphi_{-} = \varphi_{+}^{0} - \varphi_{-}^{0} - \frac{0.0591}{n}\log \frac{[b][d]}{[a][c]}$

- 对数项处理：将反应商 $Q$ 拆分为正极和负极部分，正极对应 $\frac{[a]}{[c]}$ ，负极对应 $\frac{[d]}{[b]}$   
● 符号规则：注意方程中减号的处理，可将负极部分提出负号变为加号形式

3. 电极的能斯特方程

● 通用形式： $\varphi=\varphi^{0}+\frac{0.0591}{n}\log\frac{[氧化型]}{[还原型]}$   
- 氧化型/还原型定义:

○ 氧化型：电极反应中高价态物质（如a和d）  
○ 还原型：电极反应中低价态物质（如c和b）

\- 浓度处理：所有参与反应的物质浓度都应计入，包括不直接参与电子转移的离子（如 $H^{+}$ ）

4. 具体电极反应示例

重铬酸根还原反应：

$\varphi = \varphi^0 +\frac{0.0591}{6} log\frac{[Cr_2O_7^2-][H^+]^{14}}{[Cr^3 + ]^2}$

○ 氧化型: $Cr_{2}O_{7}^{2-}$ 和 $H^{+}$   
○ 还原型： $Cr^{3+}$   
○ 规律： $H^{+}$ 浓度越高氧化能力越强， $Cr^{3+}$ 浓度越高氧化能力越弱

氢电极反应：

$\varphi = \varphi^0 +\frac{0.0591}{2}\log \frac{[H^+]^2}{P_{H_2} / P^0}$

○ 注意：压强需除以标准压强 $P^{0}$ 消除单位
